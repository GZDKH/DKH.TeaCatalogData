# 遠安黃茶 (远安黄茶, Yuǎnān huángchá)

**遠安黃茶 (远安黄茶, Yuǎnān huángchá)**

![Юаньань Хуан Ча (远安黄茶, Yuǎnān huángchá)](images/Юаньань%20Хуан%20Ча%20(远安黄茶,%20Yuǎnān%20Huángchá).png)

![Юаньань Хуан Ча (远安黄茶, Yuǎnān huángchá)-1](images/Юаньань%20Хуан%20Ча%20(远安黄茶,%20Yuǎnān%20Huángchá)-1.png)

**遠安黃茶 (远安黄茶, Yuǎnān huángchá)** — 來自「《茶經》第一縣」(陆羽《茶经》第一县) 的黃茶，湖北省唯一的黃茶代表，也是「中國四大黃茶」之一，與君山銀針、蒙頂黃芽和霍山黃芽齊名。其古名為鹿苑茶 (鹿苑茶，「鹿苑茶」)，得名於南宋時期建於鹿溪山的佛教寺院鹿苑寺 (鹿苑寺)，僧侶於西元1225年首次開始在此種植茶葉。遠安黃茶擁有兩個其他黃茶所沒有的獨特標誌：「環子腳」(环子脚，「環子腳」) — 乾茶葉的環狀捲曲，以及「魚子泡」(鱼子泡，「魚子泡」) — 炒製時在葉面形成的細微隆起。其工藝包含湖北省獨有的悶堆 (闷堆, mēnduī) 技術，在濕麻布下堆積18–24小時並進行三次翻堆，最終乾燥使用松木炭火 (松木炭火)，賦予茶葉獨特的「松煙香」(松烟香, 松煙香) — 細膩高雅的煙燻韻味，使其更接近正山小種而非其他黃茶。

**1. 分類與產地：**

* **類型：** 黃茶 (黄茶, huángchá)，輕發酵茶。屬於「黃小茶」(黄小茶, huáng xiǎo chá) 子類別 — 由一芽一至二葉製成。根據維基百科分類，「遠安鹿苑」是「黃小茶」的經典代表之一。
* **等級：** 「中國四大黃茶」(中国四大黄茶) 之一。國家農產品地理標誌保護產品 (国家农产品地理标志, 2017年)。製作技藝列入湖北省非物質文化遺產名錄 (2009年，2021年確認)。湖北省唯一的黃茶。2017年獲得中國國際茶文化研究會頒發的「中國茶文化之鄉」(中国茶文化之乡) 和「中華文化名茶」(中华文化名茶) 稱號。
* **產地：** 中國湖北省 (湖北) 宜昌市 (宜昌) 遠安縣 (远安县)。遠安位於宜昌東北部，處於鄂西山地向江漢平原的過渡地帶。古代該地區稱為峽州 (峡州)，陸羽在《茶經》中曾提及。
* **地理座標：** 約北緯31°06'，東經111°38'。

**2. 歷史與文化意義：**

* **歷史：**
    * **唐代 (唐, 618–907年) — 《茶經》：** 陸羽 (陆羽) 在《茶經》(《茶经》, 760年) 中將峽州茶評為南方山區最佳：「山南，以峽州上 (山南，以峡州上)。峽州生遠安、夷陵、巴山峽川有洞庭」。據當地傳說，陸羽於751年西行時，在遠安沮河 (沮河) 停留數月，與當地農民交流製茶經驗。遠安以「陸羽《茶經》第一縣」(陆羽《茶经》第一县) 的稱號為榮。
    * **南宋 (南宋, 1127–1279年) — 鹿苑茶誕生：** 關於鹿苑寺 (鹿苑寺) 茶葉的最早記錄可追溯至嘉定六年 (嘉定六年, 1214年)。據其他史料，僧侶於1225年開始在寺院牆邊種植茶樹。產量微薄，但香氣吸引了周圍農民，他們開始從寺院引種茶樹到山坡上種植。
    * **明代 (明, 1368–1644年) — 寺院傳說：** 傳說技藝的完善與名為玉松 (玉松) 的僧侶有關，他在寺院周圍發現了特殊的茶樹，並開發出成為現代遠安黃茶基礎的加工方法。
    * **清代 (清, 1644–1911年) — 宮廷地位：** 乾隆年間 (乾隆, 1736–1796)，鹿苑茶被選為宮廷貢茶 (贡茶)。咸豐《遠安縣志》(咸丰《远安县志》) 記載：「遠安茶以鹿苑為絕品」(远安茶以鹿苑为绝品)。1883年 (光緒九年)，高僧金田 (金田) 造訪鹿苑寺，品嚐茶葉後留下詩作《絕品茶》(《绝品茶》)：「山精石液品超群，一種馨香滿面熏，不但清心明目好，參禪能伏睡魔軍」(山精石液品超群，一种馨香满面熏，不但清心明目好，参禅能伏睡魔军)。
    * **現代：** 1966年 — 遠安黃茶技藝納入高等農業院校教材《製茶學》(《制茶学》)。1982年和1986年 — 兩次獲得商業部頒發的「全國名茶」(全国名茶) 稱號。1985年 — 收錄於《中國名茶研究選集》(《中国名茶研究选集》)。2009年 — 技藝列入湖北省非物質文化遺產名錄。2017年 — 獲得地理標誌保護。

* **名稱：**
    * 「遠安」(远安) — 縣名。字面意思：「遙遠的安寧」。
    * 「黃茶」(黄茶) — 「黃茶」— 按類型分類。
    * 古名：「鹿苑茶」(鹿苑茶，「鹿苑茶」) — 以鹿苑寺命名。「鹿苑」— 佛教地名，指瓦拉納西附近的鹿野苑 (鹿野苑, 彌伽陀婆)，佛陀在此初轉法輪。也稱為「鹿苑毛尖」(鹿苑毛尖，「鹿苑毛尖」)。
    * 民間俗語：「清漆寺的水，鹿苑寺的茶」(清漆寺的水，鹿苑寺的茶) — 鄂西兩大寺院珍品。

* **文化意義：** 遠安黃茶是一款深具佛教色彩的茶葉。它誕生於寺院牆內，被僧侶以詩歌讚頌，刻在寺院庭院的石碑上。「山精石液」不僅是比喻，更指向佛教修行：茶作為禪修時保持清醒的工具 (参禅)。關於嫘祖 (嫘祖, 黃帝軒轅氏之妻) 因心不在焉而將炒製的茶葉在籃中「過度悶黃」，從而創造出悶黃技術的傳說，是黃茶起源神話之一，特別與遠安相關：據傳嫘祖就出生在這一帶。遠安也是蠶神嫘祖的故鄉：茶與絲綢 — 遠安對中華文明的兩大貢獻。

**3. 植物學描述與原料：**

* **品種：** 主要為本地群體種 (本地群体种, 有性繁殖)，具有明顯的蘭花香 (兰花香)。輔助品種包括福鼎大白茶 (福鼎大白茶) 和鄂茶1號 (鄂茶1号)。本地群體種基因多樣性豐富，適應丹霞 (丹霞) 土壤，氨基酸含量高 (春茶≥6.77% — 比標準綠茶高30%)。
* **採摘：** 清明 (清明) 前3–5天至穀雨 (谷雨)，約15天時間。標準：一芽一至二葉。要求：新鮮、嫩度好、勻整、潔淨。不採「魚葉」(鱼叶, 鱗片狀基部葉)、老葉和受損葉片。
* **特殊工序 — 「短茶」(短茶，「短茶」)：** 加工前對採摘的芽葉進行「掰斷」：將過長的葉片縮短，只選取嫩芽和第一片葉子。這是獨特的預處理工序，其他黃茶製作技藝中未見。

**4. 風土條件與栽培特點：**

* **地貌：** 茶葉產區中獨特的丹霞地貌 (丹霞地貌，「丹霞地貌」)：紅色砂岩丘陵，具有特徵性的峭壁和洞穴。北緯31°。海拔40–800公尺。
* **土壤：** 紅黃壤 (红黄壤)，pH值4.5–6.5，由紅砂岩風化形成。富含硒 (Se) 和鋅 (Zn) — 對健康重要的微量元素。疏鬆，排水良好。
* **氣候：** 亞熱帶季風氣候。年平均氣溫16.4°C。相對濕度≥80%。霧日數≥200天/年。晝夜溫差顯著 (有利於氨基酸積累)。
* **水文：** 三條河流 — 沮河 (沮河)、漳河 (漳河) 和西河 (西河) — 環繞該地區。54座水庫提供灌溉。
* **生態：** 森林覆蓋率74.5%。核心區 (鹿苑村, 鹿苑村) 為水源保護區：禁止使用化學肥料和農藥。古茶樹 (樹齡≥30年) 約佔核心區種植面積的40%。鹿苑寺周圍：青獅山 (青狮，「青獅山」) 和白象山 (白象，「白象山」) 作為天然「守護者」；雲門山 (云门山，「雲門山」) 坡上的錦屏峰 (锦屏峰) 為「屏風」；山谷中溪流潺潺，野生蘭花和常綠楠樹 (楠树) 生長其間 — 這一切形成了獨特的微氣候。

**5. 製作工藝：**

遠安黃茶工藝是湖北省唯一包含真正悶黃的技術。五道工序，完全手工製作。主要特點：濕布下堆積悶黃和松木炭火乾燥。

* **攤青 (摊青 — tān qīng)：** 4–6小時。在竹篩上薄攤。輕度萎凋，開始香氣形成。
* **殺青 (杀青 — shā qīng)：** 溫度約180°C。採用「拋炒」(抛炒，「拋炒」) 技法：茶葉在鍋中拋起，確保均勻受熱。鈍化酶活性，固定綠色香氣。
* **初揉 (初揉 — chū róu)：** 形成茶葉初步結構。輕壓 — 不破壞細胞結構。
* **悶堆 (闷堆 — mēn duī)：** 關鍵工序 — 形成「三黃」(三黄, sān huáng)：黃乾茶、黃茶底、黃湯色。將茶葉裝入竹簍 (竹篓)，壓實，上覆濕麻布 (湿麻布)。時間18–24小時。悶堆期間進行三次翻堆 (三次翻堆)，控制黃化程度：目標黃化率 (黄化率) 50–60%。完成標準：茶葉呈黃色，香氣純正，無酸味。這是湖北省唯一使用真正堆積悶黃的黃茶。
* **復揉 (复揉 — fù róu)：** 深化成型 — 形成特徵性的「環子腳」(环子脚，「環子腳」)：茶葉捲曲成環狀螺旋。
* **松木炭火烘焙 (松木炭火烘焙 — sōng mù tàn huǒ hōng bèi)：** 兩個階段：
    * 初烘 (初烘)：約80°C，在松木炭上的烘籠上進行。
    * 足乾 (足干)：約60°C — 長時間烘乾至完全乾燥。
    松木炭 (松木炭) — 非橡木炭、非槐木炭 — 賦予茶葉細膩的「松煙香」(松烟香，松煙香)。這是「入骨的煙香」(烟香入骨) — 精緻而非粗糙，高雅。
* **全程僅用竹木器具：** 工具 — 竹簍、木鏟、竹篩。禁用金屬表面 (避金属氧化) — 防止不良氧化。

**6. 感官特徵：**

* **乾茶外形：** 「環子腳」(环子脚，「環子腳」)：捲曲條件類似環狀或半月形 — 其他黃茶所沒有的獨特形狀。色澤金黃 (金黄色)。表面有「魚子泡」(鱼子泡，「魚子泡」)：高溫炒製時形成的細微隆起 — 真品的診斷特徵。白毫明顯。
* **乾茶香氣：** 多層次。基調為「清香」(清香，清香)。上層為早春茶的「嫩香」(嫩香，嫩香)；主要春茶的「栗香」(栗香，栗香)；本地群體種特有的「蘭花香」(兰花香，蘭花香)；炭火乾燥產生的「松煙香」(松烟香，松煙香)。
* **湯香：** 「清香帶蘭韻」(清香带兰韵，「清香帶蘭韻」)。細膩、多層次、持久。松煙香如輕盈尾韻，而非主導。
* **滋味：** 「醇厚甘涼」(醇厚甘凉) — 濃厚、飽滿、甘甜，具有特徵性的「甘涼」(甘凉, gānliáng) — 清新感和薄荷般回味，黃茶中少見。油潤質感 (滑, huá — 「滑潤」)。無苦澀。回甘甘甜、持久，有蔗糖回甘 (蔗糖回甘)。
* **湯色：** 「黃淨明亮」(黄净明亮) — 黃色、清澈、明亮。「貢芽」(贡芽，宮廷級) 呈淺黃色；「特級」(特级，特級) 呈較深黃色。透明。
* **葉底 (沖泡後茶葉)：** 「嫩黃勻整」(嫩黄匀整) — 嫩黃色、均勻、完整的葉片，展開如「花朵」(成朵)。

**7. 化學成分：**

* **氨基酸：** 春茶≥6.77% — 黃茶中最高含量之一 (比普通綠茶高30%)。L-茶氨酸提供明顯的甘甜和鮮味。
* **多酚類：** 含量適中，在堆積悶黃過程中深度轉化 — 柔和而不失抗氧化活性。
* **GABA (γ-氨基丁酸)：** ≥160毫克/100克 — 卓越指標。GABA是具有抗焦慮和鎮靜作用的神經遞質。此含量是非專門GABA茶中最高的之一。
* **茶多糖：** 含量顯著 — 參與血糖調節。
* **氟：** 15毫克/100克 — 抑制致齲細菌活性90%。
* **微量元素：** 硒 (Se)、鋅 (Zn) — 來自遠安丹霞土壤。
* **維生素：** C、B群。

**8. 保健功效：**

* **抗氧化保護：** 遠安黃茶的多酚類具有顯著的自由基清除能力。
* **神經保護與鎮靜：** GABA (≥160毫克/100克) — 天然抗焦慮劑。有助於減輕焦慮，改善睡眠品質。僧人金田不無道理地指出：「參禪能伏睡魔軍」— 矛盾的是，茶既能鎮靜又能提神 (L-茶氨酸 + 咖啡因 + GABA)。
* **血糖調節：** 茶多糖有助於控制血糖。
* **牙齒健康：** 氟 (15毫克/100克) 抑制90%的致齲細菌活性。
* **溫和的胃部作用：** 「不寒不燥」(不寒不燥，「不寒不燥」) — 遠安茶在熱性上中性 (中醫術語)，適合任何季節飲用。

**9. 沖泡方法：**

* **水溫：** 85–90°C。不超過90°C — 高溫會破壞L-茶氨酸並降低「鮮味」(鲜味)。
* **茶量：** 蓋碗用5克/250毫升水 (1:50比例)；玻璃杯用3克。
* **茶具：** 瓷質蓋碗 (盖碗) — 首選，能展現多層次香氣。玻璃杯 — 觀察「環子腳」展開。
* **沖泡過程 (蓋碗)：**
    1. 用沸水溫杯，倒掉。
    2. 投入5克茶葉。
    3. 潤茶 (润茶)：10秒，倒掉。
    4. 第一泡：1分鐘，倒出。
    5. 後續各泡：每次增加15秒。可泡3–5次。
* **沖泡過程 (玻璃杯)：**
    1. 注水至杯子7/10容量。
    2. 投入茶葉。
    3. 浸泡2分鐘。觀察「環狀」展開。
* **注意事項：** 不用沸水 (>90°C)。新茶建議在陰涼處放置15天以「褪火氣」(褪火气)。開封後72小時內飲完以保持香氣。不要空腹飲用 (單寧刺激黏膜)。

**10. 儲存：**

遠安黃茶是中等嬌嫩的茶：比君山銀針不那麼嬌貴，但需要注意。密封包裝，陰涼乾燥處。冰箱 (0…+5°C) 最佳。正確儲存條件下保質期12–18個月。「不寒不燥」— 中性特質允許比嬌嫩的芽茶稍長時間儲存。對於「黃大茶」(黄大茶，大葉黃茶 — 遠安產品線之一) — 允許陳化形成「陳香」(陈香，陳香)。

    * **二級 (二级，二級)：** 一芽二葉。香氣清純。200–400元。
    * **黃大茶 (黄大茶)：** 大葉，拼配。自然捲曲，鬆散。「陳香」(陈香)。最經濟實惠。
* **仿冒：** 比君山銀針少見，但可能用未經悶黃的綠茶冒充。真品特徵：「環子腳」(環狀捲曲)、「魚子泡」(小泡點)、金黃 (非綠) 色澤、黃 (非綠) 湯色、「甘涼」(清涼甘甜) — 非草青味。

**12. 趣聞軼事：**

* 遠安是「陸羽《茶經》第一縣」：陸羽將峽州 (包括遠安) 列為南方山區茶葉產區之首。據傳說，他於751年在此居住數月。
* 「鹿苑寺」(鹿苑寺，「鹿苑寺」) 之名直接指向瓦拉納西附近的鹿野苑，佛陀在此初轉法輪宣說轉法輪經。這座寺院的茶葉可謂「初轉法輪之茶」。
* 「環子腳」(环子脚，「環子腳») — 形似小環或半月的形狀 — 其他黃茶所沒有。這是特殊揉捻技法和堆積悶黃的結果。
* 「魚子泡」(鱼子泡，「魚子泡») — 乾茶表面的細微隆起 — 真品標誌。它們在高溫炒製 (180°C「拋炒」) 時形成，是葉內細胞液沸騰的結果。
* 松木炭火最終乾燥 — 這一技術使遠安黃茶與福建正山小種 (正山小种，正山小種) 相似。兩種茶都從同一來源 — 燃燒的松木獲得「松煙香」(松烟香)，但屬於完全不同的類別：一個是黃茶，另一個是紅茶。
* 遠安是湖北省唯一的黃茶產地。「悶堆」(闷堆，堆積悶黃) 技術配合濕布下三次翻堆 — 不僅在湖北獨有，在全中國也很罕見。
* 嫘祖 (嫘祖) 傳說：黃帝軒轅氏之妻，絲綢發明者，據傳出生於遠安附近。有一次她炒茶葉時，想起出征的丈夫而分神 — 在籃中過度悶了茶葉。回來品嚐這黃色茶湯後，黃帝驚嘆：「此茶只應天上有！落到人間變成金！」— 並稱之為「黃茶」。據傳說，黃茶作為一個類別就此誕生。
* GABA ≥160毫克/100克 — 非專門茶類中最高的天然含量之一。相比之下：臺灣的專門「GABA茶」經過厭氧處理來提高GABA含量；遠安通過堆積悶黃自然達到高含量。

**13. 與其他黃茶的比較：**

* **君山銀針 (君山银针)：** 兩者都在「四大名茶」之列。君山 — 單芽，島嶼茶，「絲滑」，雙層包紙悶黃68小時，「舞茶」。遠安 — 芽葉，山地茶，「甘涼」，堆積悶黃18–24小時，「環狀茶」。君山 — 外交官；遠安 — 僧侶。
* **霍山黃芽 (霍山黄芽)：** 兩者都是中國中部山區茶。霍山 — 安徽，「栗香」，「乾攤」，礦物特徵。遠安 — 湖北，「蘭花香帶松煙」，濕布堆積悶黃，「甘涼」。霍山 — 知識分子；遠安 — 禪修者。
* **蒙頂黃芽 (蒙顶黄芽)：** 兩者都在「四大名茶」之列，都有古老歷史。蒙頂 — 四川，高山 (1456公尺)，蜜栗香，「三炒三悶紙包」。遠安 — 湖北，中山 (40–800公尺)，蘭花煙香，濕布堆積悶黃。蒙頂 — 浪漫主義者；遠安 — 苦行僧。
* **平陽黃湯 (平阳黄汤)：** 平陽 — 海濱，玉米香，杏香，72小時「九烘九悶」。遠安 — 河流，蘭花香，「甘涼」，18–24小時堆積悶黃配松木炭。平陽 — 複雜多階段「黃茶禮儀」；遠安 — 直接的僧侶之道。

**結語：**

遠安黃茶 — 誕生於佛教寺院寧靜中的茶，在每一杯中都承載著這份寧靜。其「甘涼」(甘凉) 不是比喻，而是生理感受：飲後留在舌上的溫和清新，如雨後山間空氣。其「環子腳」(环子脚) 不是工藝任性，而是手工技藝的視覺見證，在鹿苑村代代相傳，那裡30年的老樹記住了五代茶農的雙手。其「松煙香」(松烟香) 不是缺陷，而是祝福：湖北山脈的氣息，通過松木炭火滲入茶葉。陸羽將遠安列為第一。僧人金田稱其為「山精石液」。八個世紀 — 從南宋僧侶到21世紀湖北工匠 — 這款茶證明：真正的價值無需喧嘩。「不寒不燥」(不寒不燥) — 中庸之道，配得上其「黃」之名。
