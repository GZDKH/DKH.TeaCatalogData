# 霍山黃芽 (霍山黄芽, Huòshān huáng yá)

**霍山黃芽 (霍山黄芽, Huòshān huáng yá)**

![Хошань Хуан Я (霍山黄芽, Huòshān huáng yá)-1](images/Хошань%20Хуан%20Я%20(霍山黄芽,%20Huòshān%20Huáng%20Yá)-1.png)

**霍山黃芽 (霍山黄芽, Huòshān huáng yá)** — 中國四大傳統黃茶之一，也可能是有文獻記載最古老的黃茶：其蹤跡可在司馬遷的《史記》中發現——這是中華文明的基礎文獻之一。這款茶誕生於大別山脈的核心地帶——這座山系分隔著中國南北，而在這個地理位置中蘊藏著它的特性：霍山黃芽是一款處於交匯點的茶，是一座橋樑之茶，北方的礦物質剛勁在此遇見南方的溫柔甘甜。其工藝建立在「攤放黃變」(摊放黄变, 「攤放使其黃變」) 基礎上——這是黃茶中最從容不迫、最具禪意的悶黃方法，茶葉不在堆中「悶」，也不用紙包裹，而是簡單攤放，讓其自然黃變，無壓力無急躁——一至兩天，個別師傅甚至長達十天。招牌特色是「板栗香」(板栗香, 炒栗子香氣)，因此霍山黃芽被稱為「三鮮茶」(三鲜, sān xiān)：香氣鮮、滋味鮮、湯色鮮。

**1. 分類與產地：**

* **類型：** 黃茶 (黄茶, huángchá)，輕發酵茶。屬於「黃芽茶」(黄芽茶, huáng yá chá) 子類別——原料品質最高的等級。
* **類別：** 中國四大傳統黃茶 (中国四大传统黄茶) 之一——與君山銀針、蒙頂黃芽、平陽黃湯並列。歷史宮廷茶。地理標誌保護產品 (2006年)。製作技藝列入安徽省非物質文化遺產名錄。2024年——列入《中國農耕農品記憶索引名》。
* **產地：** 中國安徽省 (安徽, Ānhuī) 六安市 (六安, Lù'ān) 霍山縣 (霍山县, Huòshān Xiàn)。霍山位於大別山脈 (大别山, Dàbié Shān) 中部——這座山系分隔長江與淮河流域，是中國南北的天然分界線。核心產區——大化坪鎮 (大化坪镇, Dàhuàpíng Zhèn)：金雞山 (金鸡山, 「金雞山」)、金雞凼 (金鸡凼)、金竹坪 (金竹坪) 和烏米尖 (乌米尖)，以及磨子潭鎮 (磨子潭镇) 的高山林區。這些地方在茶界被稱為「三金一烏」(三金一乌, sān jīn yī wū)——按地名首字而得名，正是在這裡生產最高品質的茶葉。
* **地理座標：** 約北緯31°，東經116°。

**2. 歷史與文化意義：**

* **歷史：**
    * **西漢 (西汉, 西元前206年—西元8年) — 首次記載：** 司馬遷《史記》載：「壽春之山有黃芽焉，可煮而飲，久服得仙」(寿春之山有黄芽焉，可煮而饮，久服得仙)。霍山當時隸屬壽州 (寿州)，「壽春之山」即今霍山縣境內群山。這是中國文獻中最早的黃茶記載之一——距今2000多年。需要說明的是，當時的「黃芽」可能僅指黃色茶芽，而非經悶黃工藝處理的茶葉。
    * **唐代 (唐, 618–907年) — 宮廷茶地位：** 李肇在《國史補》(《国史补》) 中將「壽州霍山黃芽」(寿州霍山黄芽) 列入十四種宮廷茶名單。書中還記載了著名的外交插曲：「常魯公使西蕃，烹茶帳中。贊普問：'此為何物？'魯公曰：'茶也，滌煩療渴，所謂茶也。'贊普曰：'我亦有此。'命出示，曰：'此壽州者，此顧渚者。'」霍山茶因此在八至九世紀已傳至西藏。唐代黃芽以餅茶 (饼茶, 餅茶) 和小團 (小团, 小團) 形式生產。《膳夫經手錄》(《膳夫经手录》) 記載：「壽州有霍山小團，蓋仿龍牙小餅；數甚少。」
    * **宋代 (宋, 960–1279年) — 重要貿易中心：** 設立「霍山茶場」(霍山茶场)，年銷售量266,154.5斤 (~133噸)。黃芽逐漸從餅茶形式轉為散茶 (散茶)，雖然殺青仍採用蒸青 (蒸青) 工藝。
    * **明代 (明, 1368–1644年) — 鼎盛期與現代工藝誕生：** 黃芽列入皇室貢品。《六安州志》(《六安州志》) 記載：初始貢額200袋茶葉；霍山析縣後 (1496年) 六安25袋，霍山175袋 (87.5%！)，顯示「六安茶」實際主要產自霍山。官員曹琥 (曹琥) 在《注黃芽茶疏》(《注黄芽茶疏》) 中抱怨：「歲貢芽茶僅二十斤……而正德十年 (1515年) 徵收芽茶一千二百斤，細茶六千斤……一斤芽茶索銀一兩，且不易得。」縣令王毗翁 (王毗翁) 親自督製茶葉，留下《黃芽焙茗詩》(《黄芽焙茗诗》)：「露蕊纖纖才吐碧，即防葉老採須忙，家家篝火山窗下，每到春來一縣香」(露蕊纤纤才吐碧，即防叶老采须忙，家家篝火山窗下，每到春来一县香)。明代發生決定性技術轉變：殺青由蒸改為炒 (改蒸为炒)，出現悶黃 (闷黄, 悶黃) 工序——現代意義的黃茶誕生。明代作者許次紓 (许次纾) 在《茶疏》(《茶疏》) 中寫道：「江北則霍山茶最多……山西、陝西人皆用之。南方謂其能消垢膩，去積滯，亦甚寶愛。」
    * **清代 (清, 1644–1911年) — 宮廷「內用茶」：** 霍山黃芽被定為「內用」(内用) ——供皇室成員個人飲用，地位高於一般宮廷貢茶。光緒《霍山縣志》(《霍山县志》)：「南鄉烏米尖、瓜蘆尖諸峰所產，為全縣茶品之冠；製法精工，價倍他鄉。」另一處記載：「茶為山貨之首。上者曰銀針，次曰雀舌，又次曰梅花片……」
    * **1915年 — 巴拿馬博覽會：** 霍山茶「抱兒鐘秀」(抱儿钟秀, 「懷抱嬰兒，鐘靈毓秀」) 品牌獲巴拿馬萬國博覽會金獎——獲獎黃茶中的唯一代表。
    * **1940–1960年代 — 失傳：** 戰爭和經濟動盪導致黃芽生產實際停止。技術未有文字記錄，僅存於少數老茶農記憶中。據某些資料，復原前曾為山東商人生產名為「米茶」(米茶, 「米茶」) 的茶葉。
    * **1971–1972年 — 復原：** 霍山縣茶葉局組織烏米尖考察。三名技術人員與三名70–80歲老茶農共同重建工藝。1972年4月27–30日製作試驗批次14斤 (7公斤)。6斤裝入白鐵罐直接送國務院審評——這是極其罕見的事實：復原第一年即「呈報政府」的茶葉。次年開始在三個點正規生產：金雞山 (主要)、烏米尖和金竹坪。1972年樣品用於國家級外賓接待。產量逐步增長：1973年178公斤，1980年644公斤，1985年3700公斤。
    * **2006年 — 地理標誌。** 2019年——中國茶葉流通協會授予霍山「中國黃茶之鄉」(中国黄茶之乡) 稱號。至2022年霍山茶園面積達206,400畝 (~13,760公頃)。

* **名稱：**
    * 「霍山」(霍山) — 霍山，即霍山縣。「霍」(霍) 字意為「迅速」、「突然」——可能指山勢陡峭。
    * 「黃芽」(黄芽) — 「黃色芽頭」。在最古老的文獻中可能僅指「黃色嫩芽」，但自明代起指經悶黃工藝處理的茶葉。
    * 歷史地名：壽州 (寿州) — 霍山茶在《史記》和《國史補》中以此名出現。

* **文化意義：** 霍山黃芽是少數幾款歷史可由連續文獻鏈證實的茶葉，從西漢至今：司馬遷 → 陸羽 → 李肇 → 許次紓 → 曹琥 → 王毗翁 → 縣志 → 巴拿馬博覽會 → 1972年國家復原。金雞山金雞傳說——關於神奇茶樹散發無法形容的香氣，由一對金雞守護，每年僅在穀雨前黎明雞鳴時顯現一次——是安徽茶文化神話中最詩意的故事之一。霍山位於大別山「茶葉走廊」(茶叶走廊) 上，這條走廊從豫南經皖西延伸——歷史茶葉貿易路線之一。

**3. 植物學描述與原料：**

* **品種：** 主要栽培品種——霍山金雞種 (霍山金鸡种, 「霍山金雞品種」) — 當地群體品種，省級認定。特徵：茶多酚14.9%，胺基酸4.97% — 「雙高」(双高, shuāng gāo) 的特殊組合，同時提供澀味和甘甜。持嫩性強 (持嫩性强)，適應山地條件良好。輔助使用：櫧葉齊 (槠叶齐) 和黃山大葉種 (黄山大叶种) — 增強香氣和變化性。
* **採摘：** 穀雨 (谷雨, ~4月20日) ± 2–3天為主要採摘期。核心產區「三金一烏」因海拔較高，採摘期較晚，接近四月底。總採摘期約一個月，3–4批春茶。
* **採摘標準：** 特一級 (特一级) — 一芽一葉初展 (一芽一叶初展)，金色茸毛，形如「雀舌」。特二級 (特二级) — 一芽一葉至一芽二葉初展 (一芽一叶至一芽二叶初展)。一級 — 一芽二葉 (一芽二叶)。二級 — 成熟「對夾葉」(对夹叶)。
* **原料要求：** 實行「三個一致，四不採」(三个一致，四不采) 原則：形狀、大小、顏色一致；不採開口芽 (开口芽)、蟲傷芽 (虫伤芽)、霜凍芽 (霜冻芽)、紫色芽 (紫色芽)。全程使用竹製工具；嚴禁接觸鐵器 (全程忌铁器防腥, 「全程避免鐵器防止腥味」)。

**4. 風土條件與栽培特點：**

* **地區：** 霍山位於大別山脈 (大别山) 中心，處於安徽、湖北、河南三省交界。主峰白馬尖 (白马尖, 1774公尺)。縣境被西南—東北走向山脈切割；特殊地質現象——「霍山弧」(霍山弧, Huòshān Hú) — 山脈急劇彎曲，形成眾多具有獨特氣候條件的微谷地。冰磧岩 (冰碛岩, 冰川礫岩) — 約6億年前的古老岩石，在核心產區露出，為土壤提供獨特礦物成分。
* **海拔：** 標準原料≥600公尺。核心產區金雞凼 ~720公尺。金雞凼茶園面積僅約3畝 (~0.2公頃)，年產量不足50公斤，解釋了該微產區茶葉的極度稀有和高價格。
* **土壤：** 黃棕壤 (黄棕壤)，由冰磧岩 (冰碛岩) 形成。pH 5.0–6.5。有機質含量 ~2.5%。富含硒 (Se) — 霍山土壤的特色。所謂「烏砂土」(乌沙土, 「黑砂土」) — 砂質成分與黃壤混合。結構疏鬆，排水極佳。
* **氣候：** 亞熱帶與溫帶過渡性氣候。年均溫 ~15.1°C。年降水量1100–1600毫米。相對濕度≥80%。霧雲天數達181天。晝夜溫差8–10°C — 積累芳香物質和胺基酸的重要因子。森林覆蓋率75.1%。佛子嶺 (佛子岭) 和磨子潭 (磨子潭) 水庫位於淠河上游，進一步調節微氣候。
* **特點：** 霍山弧形成眾多微氣候，解釋了口感的顯著變化：金雞山 (金鸡山) 茶 — 更濃厚、更油潤；烏米尖 (乌米尖) 茶 — 更具礦物質感、更嚴謹；磨子潭 (磨子潭) 茶 — 更溫柔、更花香。霍山位於中國東部茶區北緣 (我国东部茶叶产区的北缘)，進一步強化「邊界」特性 — 生長緩慢，萌發較晚，風味物質積累最大化。

**5. 製作工藝：**

霍山黃芽工藝與其他黃茶的黃變方法不同：這裡使用的不是包紙悶黃 (如蒙頂黃芽) 也不是堆積悶黃 (如海馬宮茶)，而是「攤放黃變」(摊放黄变, 「攤放使其黃變」) — 茶葉簡單攤放成薄層，在室溫下緩慢黃變。這是最「禪意」的悶黃方法。完整工序包括：

* **攤放 (摊放 — tān fàng)：** 鮮葉攤放在竹篩上 (竹制簸箕)。時間1–2小時。部分水分蒸發，開始香氣形成。
* **殺青 (杀青 — shā qīng)：** 兩階段炒製：
    * **生鍋 (生锅, 「生鍋」)：** 溫度 ~150°C。快速高溫處理以鈍化酶活性。
    * **熟鍋 (熟锅, 「熟鍋」)：** 溫度 ~130°C。做形 — 使茶葉呈現特有的「雀舌形」(雀舌形, quèshé xíng)：筆直、略微展開的條索。傳統師傅使用青岡木炭 (青杠木炭) 柴火 — 認為這種火給出更純淨的香氣，無煙燻味。
* **初烘 (初烘 — chū hōng)：** 溫度 ~100°C。烘至約70%乾度。
* **攤放黃變 (摊放黄变 — tān fàng huáng biàn)：** 關鍵獨特工序。烘乾的茶葉攤放成薄層，在室溫下放置1–2天。這是「乾悶」(干闷, gān mèn)：茶葉緩慢黃變，無強制加熱和高濕度。發生葉綠素逐漸分解，酯型兒茶素非酶氧化，形成黃色色素和特有柔和度。某些師傅將此工序延長至10天以上，以獲得最大「黃茶特性」深度。其他師傅交替「乾悶」和「濕悶」(湿闷) — 茶葉在殺青後趁熱堆積。
* **復烘 (足火 — zú huǒ)：** 溫度 ~90°C。烘至約90%乾度。
* **二次攤放 (摊放)：** 再次均勻水分和完善黃變。
* **揀剔 (拣剔 — jiǎn tī)：** 去除不合格葉片、茶梗、雜質。
* **足火 (复火 — fù huǒ)：** 溫度100–120°C。烘至完全乾燥。之後茶葉可用「踩筒」(踩筒, cǎi tǒng) 方法裝入竹籃 — 壓實以便緊湊儲存。

**6. 感官特徵：**

* **乾茶外形：** 筆直、略微展開的條索，似雀舌 (形似雀舌, xíng sì quèshé)。大小均勻，聚成整齊「朵狀」(匀齐成朵)。色澤嫩綠帶黃，油潤有光 (嫩绿披毫, nèn lǜ pī háo)。白毫或金毫豐富。
* **乾茶香氣：** 清純持久，明顯的炒栗香 (板栗香, bǎnlì xiāng) — 霍山黃芽的主要香氣特徵。還有淡淡的煮玉米香 (毫香, máo xiāng — 茸毛香氣)、花香和蜜香層次。
* **茶湯香氣：** 「清香持久」(清香持久) — 清純、持久、優雅。栗香為基調；後續沖泡展現花香和果香層次。霍山黃芽香氣更「北方」，更內斂礦物質感，較南方黃茶。
* **滋味：** 「鮮醇濃厚」(鲜醇浓厚) — 鮮爽、醇和、濃郁、油潤。甘甜清爽。特有雙重性：初始澀味 (比蒙頂黃芽或平陽黃湯更明顯) 轉為深沉持久的回甘 (回甘)。苦味極輕。淡淡「清涼感」(清凉感)，歸因於土壤中硒含量高。滋味是四大黃茶中最具「礦物質感」和「結構感」的。胺基酸含量≥5.2%，多酚類≥28%。
* **湯色：** 「黃綠清澈」(黄绿清澈) — 黃綠色，透明，明亮有金色光澤。高等級茶 — 純淨發光。
* **葉底 (沖泡後茶葉)：** 嫩黃明亮，整齊成朵 (嫩黄明亮，匀齐成朵)。完整、柔嫩、飽滿。

**7. 化學成分：**

* **多酚類：** ≥28% 乾物質 — 黃茶中的高含量。長時間「乾燥」黃變將部分酯型兒茶素轉化為更溫和形式，但保留大量原生化合物，解釋了比其他黃茶更明顯的澀味。
* **胺基酸：** ≥5.2% 乾物質。L-茶胺酸為主要成分。提供明顯甘甜和回味中的「鮮味」。金雞種栽培品種胺基酸4.97% — 在原料階段就為柔和度奠定基礎。
* **生物鹼：** 咖啡因 — 標準含量。與L-茶胺酸協同作用提供溫和提神效果。
* **維生素：** 維生素C、B族維生素、維生素E。
* **礦物質：** 鉀、鎂、鋅。氟 — 含量達246毫克/公斤 (高含量，對牙齒和骨骼健康有意義)。硒 (Se) — 冰磧岩霍山土壤的特色。
* **茶多糖 (茶多糖, cháduōtáng)：** 含量豐富，具免疫調節活性。

**8. 健康功效：**

* **脂質代謝支持：** 高多酚含量 (≥28%) 有效促進脂肪分解。效果評估比同等原料綠茶高約1.8倍。
* **牙齒和骨骼健康：** 高氟含量 (246毫克/公斤) 有助牙釉質強化。
* **免疫調節：** 茶多糖激活巨噬細胞 — 免疫系統細胞。
* **抗氧化保護：** 雙重系統 — 多酚類 + 硒 — 提供強大抗氧化活性。
* **溫和提神：** L-茶胺酸 + 咖啡因 — 平靜清醒的經典組合。促進放鬆和減壓而不困倦。
* **胃部溫和作用：** 儘管多酚含量較高，長時間黃變 (1–2天以上) 軟化兒茶素的刺激性。但因殘留單寧，不建議空腹飲用。
* **視力支持：** 傳統中醫認為黃茶對眼睛有益。

**9. 沖泡方法：**

* **水溫：** 80–90°C。建議將水煮沸後晾涼約2分鐘。過熱的水可能「燙傷」嫩芽並引起苦味。
* **茶量：** 3克配150毫升水。
* **茶具：** 玻璃杯 — 觀察湯色和芽葉展開的美感。白瓷蓋碗 — 最大程度展現香氣。
* **過程：**
    1. 用開水溫杯，倒掉。
    2. 投入3克茶葉。
    3. 注入80–90°C水至1/3容量。潤濕所有茶葉，等待30秒。第一泡不要倒掉 — 其中含有最多茸毛香 (毫香) 和栗香；倒掉意味著失去主要的「第一鮮」。
    4. 續水至7/10容量。浸泡1–2分鐘。
    5. 觀察「春筍出土」(春笋出土)：芽頭垂直沉入杯底，如破土而出的竹筍。茶湯應呈黃綠色，透明。
    6. 重複沖泡：可沖3次，每次增加15–20秒。

**10. 儲存：**

最佳 — 鋁箔袋或鐵罐/瓷罐密封包裝。冰箱 (0…+5°C) 或冷凍庫 (−10…−18°C)。室溫 — 陰涼乾燥處，遠離異味；6個月內飲用。茶葉天敵：光線、熱量、濕氣、異味、氧氣。傳統儲存 — 竹製容器；不宜接觸金屬 (特別是鐵製) 器具。

**11. 價格與假冒：**

霍山黃芽是稀有昂貴茶葉，價格範圍很廣。核心產區金雞凼特一級 (~720公尺，面積 ~3畝，年產不足50公斤) — 2000元人民幣/斤 (500克) 起，甚至更高。大化坪特一級 — 800–1500元。一級和二級 — 日常飲用的平價類別。

* **如何避免假冒：**
    * 主要問題：市場上大量「霍山黃芽」實際是未經完整悶黃 (闷黄) 工序的綠茶。真正的黃芽有明顯黃色 (非鮮綠色) 茶葉和湯色，以及帶玉米味的栗香。「綠色」版本 — 更清鮮、尖銳，缺乏「黃茶」柔和度。
    * 外形 — 「雀舌」(雀舌)：筆直、略微展開的條索，茸毛豐富，非捲曲或扁平。
    * 湯色 — 黃綠色 (黄绿)，透明，金黃光澤，非鮮綠色。
    * 從可靠供應商購買，有「國家地理標誌」標識和具體生產鄉鎮標註。
    * 價格過低 — 假冒或未經完整黃變工序的確切標誌。

**12. 趣聞軼事：**

* 霍山黃芽是唯一在司馬遷《史記》中提及 (古名「壽春黃芽」) 的黃茶。這使其成為有文獻記載最長譜系的茶葉之一 — 超過2000年。
* 霍山黃芽被稱為「三鮮茶」(三鲜茶)，因其三重鮮爽：香氣鮮、滋味鮮、湯色鮮。這既是行銷定義，也是感官體驗的準確描述。
* 1972年技術復原時，6斤實驗茶直接送國務院 — 歷史上茶葉在復原第一年即「呈報政府」的罕見案例之一。
* 金雞傳說：金雞山上長著神奇茶樹，由一對金雞守護。這棵樹對大多數人不可見，但每年穀雨前黎明雞鳴時顯現一次，只有一個幸運者能採摘其葉片。一次，一個來遷葬祖先的青年追趕金雞 — 金雞落入溪中，從此溪名「落雞河」(落鸡河)，草坪名「金雞凼」(金鸡凼)。
* 霍山黃芽是唯一獲巴拿馬萬國博覽會金獎 (1915年) 的黃茶。獲獎品牌「抱兒鐘秀」(抱儿钟秀) 至今存在。
* 明代宮廷貢茶配額中，霍山175袋，總數200袋 — 即87.5%的「六安茶」實際產自霍山。這是歷史悖論：茶葉以他人之名聞名 (六安茶, 「六安茶」)，直到霍山析縣後公正才部分恢復。
* 「栗香」(板栗香, 板栗香) — 霍山黃芽的招牌香氣，在所有黃茶中獨樹一幟。由金雞種栽培品種特性和長時間「乾燥」黃變兩個因素結合形成。
* 整個生產過程避免接觸鐵器 (全程忌铁器) — 僅使用竹、木、陶瓷。這是少數幾款將禁用金屬作為現行規則而非博物館細節保持的茶葉之一。

**13. 與其他黃茶比較：**

* **蒙頂黃芽 (蒙顶黄芽)：** 兩者都是芽茶類「黃芽茶」，都是有古老譜系的歷史宮廷茶。蒙頂 — 更甜、更蜜香、劍形，「三炒三悶紙包」工藝。霍山 — 更具礦物質感、更有結構、「雀舌」形和「乾攤」工藝。蒙頂 — 浪漫主義者，有道士吳理真傳說；霍山 — 知識分子，有司馬遷引文。
* **平陽黃湯 (平阳黄汤)：** 平陽 — 海洋性、玉米香、杏黃色、捲曲形。霍山 — 山地性、栗香、黃綠色、筆直形。平陽 — 「九烘九悶」72小時；霍山 — 「乾攤」1–2天 (有時達10天)。平陽 — 順滑包覆；霍山 — 有「骨架」和礦物結構。
* **君山銀針 (君山银针)：** 兩者都是「黃芽茶」，都在「四大」之列。君山 — 油潤、絲滑、針形；霍山 — 更乾爽、澀感、「舌形」。君山 — 湖泊性，洞庭湖濕潤氣候；霍山 — 山地性，溫差劇烈。
* **霍山黃大茶 (霍山黄大茶)：** 同縣「兄長」黃芽。黃大茶 — 大葉黃茶 (一芽四五叶)，焦香、「麵包」特性和鍋巴香 (锅巴香)。當地茶農諺語：「葉大能包鹽，梗長能撐船」(叶大能包盐，梗长能撑船)。黃芽 — 嫩雅、栗香、芽茶；黃大茶 — 粗獷、焦香、民間茶。

**結語：**

霍山黃芽是一款具有其生長山脈特性的茶葉。大別山分隔中國南北，在霍山黃芽茶杯中能聽到兩面的聲音：北方礦物質的直率和南方溫柔的甘甜，首口的澀味和回味的蜜香，栗香的嚴謹和玉米香的溫柔。其工藝是黃茶中最「從容」的：茶葉不包裹、不壓制、不堆悶 — 只是攤放等待，日復一日，直到它按自己的節奏自然黃變。「三鮮茶」— 香氣鮮、滋味鮮、湯色鮮 — 同時也是記憶最長的茶：從司馬遷到巴拿馬博覽會，從西藏帳篷到中國國務院。也許正因如此，《史記》中說：「壽春之山有黃芽焉，久服得仙」。長生不老 — 問題有爭議。但霍山黃芽教導的耐心 — 是完全真實的東西。
