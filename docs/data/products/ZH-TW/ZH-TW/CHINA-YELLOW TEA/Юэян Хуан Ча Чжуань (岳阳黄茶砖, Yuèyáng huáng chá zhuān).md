# 岳陽黃茶磚 (岳阳黄茶砖, Yuèyáng huáng chá zhuān)

**岳陽黃茶磚 (岳阳黄茶砖, Yuèyáng huáng chá zhuān)**
![Юэян Хуан Ча Чжуань (岳阳黄茶砖, Yuèyáng huáng chá zhuān)](images/Юэян%20Хуан%20Ча%20Чжуань%20(岳阳黄茶砖,%20Yuèyáng%20Huáng%20Chá%20Zhuān).png)

![Юэян Хуан Ча Чжуань (岳阳黄茶砖, Yuèyáng huáng chá zhuān)-1](images/Юэян%20Хуан%20Ча%20Чжуань%20(岳阳黄茶砖,%20Yuèyáng%20Huáng%20Chá%20Zhuān)-1.png)

岳陽黃茶磚是中國茶葉傳統中最不尋常且最具矛盾性的茶類之一。這是一種緊壓黃茶，含有「金花」(金花, jīn huā)——有益真菌*Eurotium cristatum*的菌落，此前僅與黑茶(黑茶)相關聯。實質上，岳陽黃茶磚是連接兩個茶葉世界的「橋樑」：黃茶的溫和與清新，以及黑茶特有的深度、長期儲存和「陳化」能力。其配方為「金磚玉葉、杏湯醇厚、陳韻綿長」(金砖玉叶、杏汤醇厚、陈韵绵长)。

**1. 分類與產地：**

* **類型：** 緊壓黃茶 (紧压黄茶, jǐnyā huángchá)。正式歸屬於黃茶 (黄茶, huángchá) 類別，但在工藝上包含黑茶特有的元素——主要是參與*Eurotium cristatum*的「發花」(发花, fā huā)。這使其成為兩個茶類交界處的獨特茶品。
* **類別：** 再加工茶 (再加工茶, zài jiāgōng chá)，以岳陽黃毛茶為基礎。是受地理標誌保護產品「岳陽黃茶」(2014年註冊)的重要子類別。
* **產地：** 中國湖南省 (湖南, Húnán) 岳陽市 (岳阳, Yuèyáng) 君山區 (君山区, Jūnshān Qū)。
* **地理座標：** 東經112°18′–114°09′，北緯28°25′–29°51′(在「岳陽黃茶」生產區域範圍內)。

**2. 歷史與文化意義：**

* **歷史：**

**清代前史：皇室貢茶磚。** 岳陽緊壓黃茶的根源可追溯到清代貢茶 (贡茶, gòngchá) 傳統。據光緒年間 (光绪, Guāngxù, 1875–1908)《巴陵縣志》記載：「君山茶磚，歲貢十八斤」(君山茶砖，岁贡十八斤)。這證明岳陽茶的緊壓形式在19世紀已經存在，可能更早：唐代的「灉湖含膏」(灉湖含膏) 就是緊壓的「茶膏」。

**1980年代：現代工藝。** 現代岳陽黃茶磚的形式確立於1980年代，當時君山茶廠 (君山茶厂, Jūnshān Cháchǎng) 開發了創新的「梯次增濕發酵法」(梯次增湿发酵法, tīcì zēng shī fājiào fǎ)，能夠控制黃茶中的「發花」過程——這項技術此前僅用於黑茶(茯磚茶)。該方法確保了*Eurotium cristatum*在緊壓黃茶內部的可重現形成，以及儲存過程中可控的風味演變。

**21世紀：獲得認可。** 2001年，緊壓黃茶被納入「君山黃金系列」(君山黄金系列)——高端產品線。2014年，隨著「岳陽黃茶」地理標誌的獲得，緊壓形式被正式確立為獨立子類別。2023年，岳陽緊壓黃茶在中國黃茶鬥茶大賽 (中国黄茶斗茶大赛) 中獲得金獎，成為獨立類別。

* **名稱：**
    * 「岳陽」(岳阳) — 洞庭湖畔的城市。
    * 「黃茶」(黄茶) — 黃茶。
    * 「磚」(砖) — 磚。指緊壓形式。
    * 也使用商業名稱：「岳陽茗片」(岳阳茗片 —「岳陽茶片」)、「黃金磚」(黄金砖 —「金磚」)。
* **文化意義：**

岳陽黃茶磚被定位為「可收藏的液態古董」(可收藏的液态古董, kě shōucáng de yètài gǔdǒng) 和「黃茶陳化標杆」(黄茶陈化标杆)。如果散裝岳陽黃茶體現清新與春天，那麼茶磚就是「透過時間稜鏡看到的同樣岳陽特色」。兩種形式——散茶和緊壓茶——的存在創造了岳陽茶人所說的「黃茶品類的完整價值生態鏈」(黄茶品类的完整价值生态链)，其中清新與陳化相互補充。

**3. 植物學描述與原料：**

* **品種/栽培種：** 使用與整個岳陽黃茶系列相同的栽培種：槠葉齊 (槠叶齐)、碧香早 (碧香早)、黃金茶 (黄金茶) 等。但對於茶磚，偏好大葉種，包括桃源大葉 (桃源大叶)。
* **採摘：** 春季、夏季、秋季。緊壓茶允許使用比散茶更晚期和成熟的葉片。
* **採摘標準：** 主要為一芽三至四葉 (一芽三至四叶) — 比散裝岳陽茶 (一芽一葉) 粗糙得多。大型成熟葉片積累更多茶多糖 (茶多糖) 和礦物質，這對形成茶湯體感和陳化潛力至關重要。
* **原料半成品：** 黃大茶 (黄大茶 —「黃色大葉茶」) 類別的黃毛茶，經過標準的「雙悶」處理。因此，茶磚是已完成黃茶的二次加工產品。

**4. 風土與栽培特點：**

岳陽緊壓黃茶的原料來自與散裝岳陽黃茶相同的生產區域：洞庭湖周邊、君山區、臨湘、平江、華容。紅壤和黃壤 (pH 4.0–6.0)，有機質含量≥1.5%，硒含量較高 (0.82 mg/kg)。

但對於茶磚，不僅栽培風土重要，**「陳化風土」**也很重要——緊壓塊內部的微環境。壓製時形成緻密結構，內部建立自己的微氣候：溫度~25±3°C，濕度~70±5%。正是這些條件確保了*Eurotium cristatum*的發育和儲存過程中物質的緩慢轉化——這在散茶中是不可能的過程。

**5. 製作工藝：**

岳陽緊壓黃茶的工藝是多階段過程，結合了經典黃茶處理與從茯磚茶 (茯砖茶) 生產中借鑒(並適應)的壓製和「發花」技術。

**階段I：黃毛茶生產 (黄毛茶)**

標準的「雙悶黃」(双闷黄) 處理：萎凋 → 殺青 → 第一次悶黃 → 揉捻 → 第二次悶黃 → 乾燥。成品毛茶進入二次加工。

**階段II：精製 (精制, jīng zhì)**

毛茶經過篩分 (筛分, shāi fēn)、切軋 (切轧, qiē yà)、風選 (风选, fēng xuǎn)、手工挑揀 (拣剔, jiǎn tī) 和拼堆 (拼堆, pīn duī)。目標是達到均勻性、純淨度和預定的批次特徵。

**階段III：「梯次增濕發酵」(梯次增湿发酵)**

君山茶廠1980年代的創新。精製毛茶分階段增濕，為*Eurotium cristatum*生長創造條件。溫度和濕度分階段控制，升高和降低——因此稱為「梯次」(梯次, tīcì)。這種方法與黑茶中使用的「衝擊式」增濕根本不同：溫和漸進的過程保持黃茶特色 (溫和、甘甜、無苦澀)，同時啟動「金花」生長。

**階段IV：緊壓定型 (紧压定型, jǐnyā dìng xíng)**

準備好的原料經蒸茶 (蒸茶, zhēng chá) 軟化，然後放入模具壓製成磚或餅。壓製密度是關鍵參數：太緊——「花」不會生長；太鬆——磚會散開且無法創造所需的微氣候。

**階段V：發花 (发花, fā huā)**

新壓製的茶磚放入溫度控制的專門房間 (~28–30°C) 和濕度 (~75–85%) 7–15天。茶磚內部開始生長*Eurotium cristatum*菌落——肉眼可見的金黃色圓點，劈開茶磚時可見。這種真菌分泌多種酶：澱粉酶 (將澱粉轉化為糖)、氧化酶 (溫和氧化多酚，去除粗澀味)、纖維素酶。其代謝產物形成獨特的「菌花香」(菌花香, jūn huā xiāng)，散茶中沒有。*Eurotium cristatum*是強勢「主導者」：其存在抑制不良微生物生長，確保產品微生物安全。

**階段VI：烘乾 (烘干, hōng gān)**

茶磚在適中溫度下乾燥至水分含量≤8%，固定形狀並將微生物群「封存」在內部。

**6. 感官特徵：**

* **外觀：** 磚形，表面平整，稜角清晰。表面緻密、光滑，有金毫顯露 (金毫显露)。劈開茶磚時內部可見金黃色點狀物——*Eurotium cristatum*菌落(「金花」)。其豐富程度和亮度是品質指標：特級品「發花」濃密均勻。
* **乾茶香氣：** 主導「酵花香」(酵花香, jiào huā xiāng) — 複雜香調，由茶磚內*Eurotium cristatum*代謝產生。另外還有焦糖香 (焦糖香, jiāo táng xiāng)，在梯次烘乾過程中產生。三年以上樣品出現「陳香」(陈香) — 陳化香氣，帶有乾果和珍貴木材香調。
* **滋味：** 配方為「醇厚濃強」(醇厚浓强, chún hòu nóng qiáng —「醇和厚重，濃郁強勁」)。這比散裝岳陽茶更「厚重」和濃郁的滋味。「骨架」由多酚形成 (≥30.5% — 顯著高於散茶的~24.7%)。「填充」— 多糖的焦糖甜味 (7.2%)。礦物「清新感」— 來自壓製時從湖泊沉積物中「擠出」的元素。回甘持久，有明顯的甜味。
* **湯色：** 深杏黃 (深杏黄) 至琥珀色 (琥珀色) — 比散裝岳陽茶 (杏黃至橙黃) 明顯更深更濃。茶湯透明、明亮。
* **葉底(沖泡後茶葉)：** 褐黃色 (褐黄)，均勻，有光澤。葉片肉厚、柔軟、有彈性 (比散裝岳陽茶明顯更大更厚)。

**7. 化學成分：**

由於*Eurotium cristatum*的作用和壓製，岳陽黃茶磚的化學特徵與散茶有顯著差異。

* **多酚：** ≥30.5% 乾物質 — 高於散裝岳陽黃茶 (~24.7%)。這個矛盾現象的解釋是大葉原料 (黃大茶) 本身含有更多多酚，而*Eurotium cristatum*只是部分修飾它們，並不破壞。
* **多糖：** 7.2% — 非常高的指標，顯著超過散茶標準。正是多糖形成濃厚、包覆性的茶湯體感並帶來焦糖甜味。
* ***Eurotium cristatum*代謝產物：** 澱粉酶、氧化酶、纖維素酶 — 在儲存過程中繼續緩慢轉化茶葉成分的酶。洛伐他汀 (lovastatin) — 天然HMG-CoA還原酶抑制劑，可能參與脂質代謝調節。真菌胞外多糖 — 具有自身生物活性。
* **生物鹼：** 咖啡因 — 乾物質的2.5–4.5% (含量與散茶相當)。
* **胺基酸：** 含量低於細嫩原料散茶 (大葉積累較少胺基酸)，但足以形成甘甜和溫和口感。
* **礦物質：** 硒、鋅、鉀含量較高 — 反映洞庭湖湖泊土壤的地球化學特徵，因壓製時的「礦物擠壓」效應而增強。

**8. 保健功效：**

* **消化支持：** *Eurotium cristatum*分泌澱粉酶和蛋白酶，有助於食物分解。高多糖含量包覆胃黏膜，減少刺激。
* **脂質代謝調節：** 綜合作用：多酚、洛伐他汀 (真菌代謝產物) 和*Eurotium cristatum*胞外多糖 — 脂質調節潛力的三個「支點」。
* **抗氧化作用：** 高多酚含量提供強大的抗氧化保護，可與優質綠茶相媲美。
* **血糖調節：** 茶多糖和多酚共同影響胰島素敏感性。
* **腸道菌群支持：** *Eurotium cristatum*具有益生元潛力，有助於形成有益的腸道環境。
* **溫和提神效果：** 咖啡因與胺基酸結合提供清醒感而無劇烈波動。

**9. 沖泡方法：**

* **水溫：** 95–100°C (沸水) — 緊壓茶需要比散茶更熱的水。
* **茶量：** 5–7克/150毫升。
* **茶具：** 瓷蓋碗 (盖碗) 或紫砂壺 (紫砂壶, zǐshā hú) — 厚壁保溫，緊壓葉展開所需。
* **過程：**
    1. 用茶刀 (茶刀, chá dāo) 或茶針 (茶针, chá zhēn) 從茶磚撬取所需量。盡量不要弄碎葉片。
    2. 沸水溫壺。
    3. 投茶。第一泡 — 沸水，8–10秒，倒掉。這是緊壓葉的「醒茶」(醒茶, xǐng chá)。
    4. 第二泡 — 同樣8–10秒，倒掉。兩次洗茶去除壓製粉塵並「喚醒」茶葉。
    5. 第三泡 (第一泡正式) — 10–15秒。茶湯深杏黃至琥珀色，有焦糖甜味和菌花香調。
    6. 後續各泡 — 時間遞增5–10秒。
    7. 耐泡度 — 8–12泡以上。
    8. **煮飲 (煮饮, zhǔ yǐn)：** 泡完後 (或代替沖泡) 緊壓黃茶很適合煮飲。5–7克/500毫升水，煮沸後再煮3–5分鐘。煮出的茶湯濃厚、甘甜，有深沉的陳化香氣。

**10. 儲存：**

與散裝黃茶的根本區別：岳陽茶磚**不需要冷藏**且**隨時間改善** (越陈越香, yuè chén yuè xiāng —「越陳越香」)。

* **條件：** 乾燥、陰涼、避光、通風的環境。溫度20–30°C。濕度50–70%。避免直射陽光和異味。
* **包裝：** 棉紙或紙板；可無包裝儲存 (茶磚需要「呼吸」)。**不要使用**密封塑膠袋 —「金花」需要最小的氣體交換。
* **儲存演變：**
    * **新茶 (1年內)：** 主導菌花香。滋味明亮，略有澀味。
    * **1–3年：** 香氣加深，出現乾果調。澀味消失，甜味增加。
    * **3–7年：** 成熟「陳香」— 陳化香氣。茶湯濃厚、蜜甜，有焦糖「底韻」。口感絲滑。
    * **7年以上：** 藥香 (药香, yào xiāng)。滋味極其溫和、包覆，有持久溫暖回甘。
* **保存期限：** 在適當條件下無限期。緊壓岳陽黃茶是少數允許多年 (潛在多十年) 陳化的黃茶之一。

**11. 價格與假冒：**

* **價格區間：** 中高端。緊壓岳陽黃茶通常比散裝岳陽黃葉貴，但比君山銀針便宜。陳年樣品 (5年以上) — 高端收藏級別。
* **如何避免假冒：**
    * **「金花」在內部，不在外部。** 真正的*Eurotium cristatum* (金花) 是金黃色圓點，生長在茶磚*內部*，在茶葉表面。劈開時可見。黴菌 (黄曲霉) 是茶磚*表面*的絨毛狀「毛髮」，容易擦掉和吹掉。
    * **茶湯透明，無渾濁。** 深杏黃至琥珀色是正常的。渾濁、深棕色茶湯是黴菌感染或工藝問題的徵象。
    * **香氣純淨，無霉味。** 應該是「菌花香」(菌花香) 帶焦糖調。酸味、霉味或「魚腥味」是次品。
    * **茶磚表面平整、緊實。** 散碎、鬆散的茶磚是壓製問題，這樣的茶無法創造正確的「發花」微環境。
    * **文件證明。** 真正的岳陽黃茶磚應有標識，確認來自「岳陽黃茶」地理標誌保護區。

**12. 有趣事實：**

* **帶有黑茶「花」的黃茶。** 岳陽黃茶磚是黑茶類別之外極少數有意使用*Eurotium cristatum*的茶類之一。「金花」是湖南茯磚茶 (茯砖茶) 的「名片」，被認為在黑茶之外無法重現。岳陽師傅證明了*Eurotium cristatum*也可以「嫁接」到黃茶上 — 在嚴格遵守梯次發酵參數的條件下。

* **「液態古董」。** 緊壓岳陽黃茶是世界上少數可以像生普洱或老白茶一樣收藏和陳化多年的黃茶之一。在貿易中獲得了詩意的別名「液態古董」(液态古董)。

* **復原「灉湖含膏」。** 現代岳陽茶人將緊壓黃茶工藝視為「復原」古代唐代「灉湖含膏」(灉湖含膏) — 曾被認為失傳的緊壓茶。黃毛茶壓製後「發花」— 據茶史學家認為，這是對唐茶長期儲存過程的現代詮釋。

* **兩種形式 — 一個「生態系統」。** 岳陽茶人強調，散裝和緊壓黃茶不是競爭對手，而是彼此的「時間補充」：散茶用於享受清新，茶磚用於時間之旅。它們共同形成黃茶的「完整價值生態系統」。

**13. 與其他茶類比較：**

* **散裝岳陽黃茶 (岳阳黄茶 —散茶)：** 相關產品，但使用更嫩的原料 (一芽一至二葉)。香氣 —「酵香」和「嫩栗香」(發酵香和嫩栗香)。滋味 — 清淡、清新、「甘醇鮮爽」。湯色 — 淺杏黃。儲存 — 冷藏，12–18個月。主要區別：無「金花」且無法長期陳化。

* **安化茯磚茶 (安化茯砖茶, Ānhuà Fúzhuān Chá)：** 因含*Eurotium cristatum*而在工藝上最相近。但這是黑茶：原料是經過渥堆 (渥堆) 的黑毛茶 (3級)。滋味更粗糙、更「厚重」，有明顯的「土味」。湯色紅褐色。岳陽黃茶磚更溫和、更淺、更甜，保持「黃茶」清新感。

* **千兩茶 (千两茶, Qiān Liǎng Chá)：** 湖南「千兩茶」— 巨型柱狀緊壓黑茶。也含微生物群，但組成不同 (較少控制的「發花」)。滋味強勁、「煙燻味」。完全不同的風格世界。

* **老白茶 (老白茶, Lǎo Báichá)：** 在「儲存陳化」理念上相似，但無*Eurotium cristatum*參與。老白茶的演變純粹是氧化性的。岳陽黃茶磚是微生物性的。風味特徵不同：老白茶「蜜棗味」；岳陽茶磚「焦糖菌香」。

* **生普洱 (生普洱, Shēng Pǔ'ěr)：** 也是隨年份改善的緊壓茶。但生普洱按分類是雲南大葉種「曬青」綠茶，無有意「發花」。生普洱的微生物群在儲存過程中自然形成。岳陽茶磚有「受控」的*Eurotium cristatum*。風味世界不同，但「茶隨時間成長」的哲學相同。

**結論：**

岳陽黃茶磚是兩個茶葉世界邊界上大膽而成功的實驗。將黃茶的溫和甘甜與黑茶的深度和時間前景結合，岳陽師傅創造了無直接類比的產品：今天可以飲用的茶 — 二十年後會發現其中完全不同的飲品。對於喜愛「活茶」理念 — 在儲藏室中繼續發展的茶 — 的人來說，岳陽金磚將是發現和長久伴侶。
