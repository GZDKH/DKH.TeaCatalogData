# 君山銀針 (君山银针, Jūnshān yín zhēn)

**君山銀針 (君山银针, Jūnshān yín zhēn)**

**君山銀針 (君山银针, Jūnshān yín zhēn)** —「君主山的銀針」— 這是一款會跳舞的茶。入選中國十大名茶，是黃茶類別的明珠和代表，擁有「國禮茶」(国礼茶) 稱號，並被列入聯合國教科文組織非物質文化遺產名錄（2022年）。但這一切都在杯中展開的奇景面前黯然失色：沖泡時君山銀針的芽頭會三起三落 (三起三落, sān qǐ sān luò)，垂直懸立，如破土而出的竹筍，如操場上的刺刀，如插在金絲中的銀針。毛澤東稱其為「會跳舞的茶」(会跳舞的茶)。這款茶誕生在中國第二大淡水湖洞庭湖中的一個小島上，茶園面積僅307畝（約20公頃），銀針年產量約400公斤。其工藝是黃茶中唯一包含「雙悶黃」(双闷黄, shuāng mēnhuáng) 的：兩個連續的悶黄週期，總時長達68小時，形成特有的金橙色和油潤質感，因此茶葉獲得「金鑲玉」(金镶玉,「金子鑲嵌玉石」) 的美譽。

**1. 分類與產地：**

* **類型：** 黃茶 (黄茶, huángchá)，輕發酵（15–25%）。屬於「黃芽茶」(黄芽茶, huáng yá chá) 子類別 — 原料品質最高的等級。
* **類別：** 中國十大名茶之一 (中国十大名茶, 1959年)。四大傳統黃茶之一。「黃茶之冠」(黄茶之冠)。清朝宮廷貢茶 (贡茶)。工藝被列入湖南省非物質文化遺產名錄（2009年）和中華人民共和國非物質文化遺產名錄（2021年）。2022年 — 被列入聯合國教科文組織人類非物質文化遺產代表作名錄（作為「中國傳統製茶技藝及其相關習俗」檔案的組成部分）。2006年 — 獲得「國禮」(国礼) 地位，由商務部和外交部確認。
* **產地：** 中國湖南省 (湖南, Húnán) 岳陽市 (岳阳, Yuèyáng) 洞庭湖 (洞庭湖, Dòngtíng Hú) 君山島 (君山岛, Jūnshān Dǎo)。君山是洞庭湖中的一個小島（面積不足1平方公裡），洞庭湖是中國第二大淡水湖（面積約2625平方公裡）。該島也稱洞庭山 (洞庭山)。
* **地理座標：** 約北緯29°24'，東經113°00'。

**2. 歷史與文化意義：**

* **歷史：**
    * **唐朝（唐, 618–907年）— 誕生與首次繁榮：** 君山產茶在唐代已有記載。陸羽 (陆羽) 在《茶經》(《茶经》) 中提到：「巴陵君山出茶」。茶葉當時稱為「黃翎毛」(黄翎毛,「黃色羽毛」) — 因其金黃色澤和如鳥絨般的茸毛。據傳說，文成公主 (文成公主) 於641年前往西藏與吐蕃王松贊干布聯姻時，特別選擇了君山茶作為旅途用茶。宋代該茶也稱「白鶴茶」(白鹤茶,「白鶴茶」) — 據傳說有道士化鶴將茶種帶到島上。
    * **清朝（清, 1644–1911年）— 宮廷地位：** 乾隆皇帝 (乾隆) 在南巡途中遊覽洞庭湖時品嚐了君山茶，印象深刻，將其列入宮廷貢品。《巴陵縣志》(《巴陵县志》) 記載：年貢額僅18斤（約9公斤）。清代茶葉分為「尖茶」(尖茶,「尖頭茶」，又稱「貢尖」贡尖) 和「茸茶」(茸茶,「茸毛茶」)。正是「尖茶」— 劍形白毫芽頭 — 成為現代「銀針」的原型。
    * **1952年 — 復興：** 成立君山茶場 (君山茶场)。一群師傅恢復了傳統工藝，特別是「雙悶黃」(双闷黄)。
    * **1956年 — 萊比錫金獎：** 君山銀針獲得德國萊比錫國際博覽會金獎 — 首次國際認可。口號：「茶蓋中華」(茶盖中华)。
    * **1957年 — 正式命名：** 茶葉獲得現代名稱 —「君山銀針」(君山银针)。
    * **1959年 — 十大名茶：** 被列入經典的「中國十大名茶」名單 — 該名單中唯一的黃茶。
    * **1972年 — 聯合國茶：** 君山銀針由中國政府代表團帶到紐約聯合國總部招待各國元首和大使。同年理查德·尼克松在歷史性訪華期間收到君山銀針作為禮品。
    * **2006年 — 國家禮品：** 商務部和外交部正式確定君山銀針為「國禮」(国礼)。該茶被贈送給俄羅斯總統普京。
    * **2022年 — 聯合國教科文組織：** 製作工藝被列入人類非物質文化遺產代表作名錄。

* **名稱：**
    * 「君山」(君山) —「君主之山」。據傳說，舜帝 (舜帝)，傳說中五帝之一，在南巡途中去世。他的妻子 — 娥皇 (娥皇) 和女英 (女英)，堯帝的女兒 — 前去尋找他，在未到達島嶼前溺死在洞庭湖中。她們的眼淚落在竹子上，留下不可磨滅的斑點 — 於是出現了「湘妃竹」(湘妃竹,「湘妃眼淚竹」)。而落在土地上的眼淚孕育了茶樹。「君主之山」就是君主舜帝嚮往的山。
    * 「銀針」(银针) —「銀針」— 因芽頭形狀：筆直、緊實、覆蓋銀白茸毛，形似針狀。
    * 「金鑲玉」(金镶玉,「金鑲玉石」) — 詩意的別稱，因其雙色結構：芽頭內層橙金色，外層因茸毛而呈白色。

* **文化意義：** 君山銀針在中國文化中佔據獨特地位，遠超茶學範疇。君山島是中國文化層次最「密集」的地方之一：這裡有舜帝兩位妻子的陵墓，這裡有秦始皇封山印 (秦始皇封山印)，這裡有柳毅井 (柳毅井) — 著名愛情傳說中龍宮使者的故事發生地，這裡留下了李白、杜甫、白居易、範仲淹的足跡。範仲淹在此寫下《岳陽樓記》(《岳阳楼记》) — 中國散文的經典文本之一，有著傳奇的句子：「先天下之憂而憂，後天下之樂而樂」(先天下之忧而忧，后天下之乐而乐)。來自這個島嶼的茶 — 是浸潤著千年歷史層次的飲品。

**3. 植物學描述與原料：**

* **品種：** 君山群體種 (君山群体种) — 當地群體種群，灌木型 (灌木型)，中葉類 (中叶类)。僅在君山島種植。特徵：芽頭粗壯厚重（百芽重 — 100個芽頭重量約45克），顏色淡綠帶紫暈 (淡绿带紫晕)，茸毛濃密。茶多酚25–30%，胺基酸≥4.3%。耐濕耐寒。
* **採摘：** 清明 (清明, 約4月5日) 前3–4天，持續7–10天。僅春茶首輪 (春茶首轮)。專用單芽 (单芽, dān yá)。芽長25–30毫米，寬3–4毫米，留梗長度約2毫米。
* **採摘標準 —「九不採」(九不采, jiǔ bù cǎi)：**
    1. 雨天不採 (雨天不采)。
    2. 風霜不採 (风霜不采)。
    3. 開口不採 (开口不采)。
    4. 發紫不採 (发紫不采)。
    5. 空心不採 (空心不采)。
    6. 彎曲不採 (弯曲不采)。
    7. 蟲傷不採 (虫伤不采)。
    8. 細瘦不採 (细瘦不采)。
    9. 不合尺寸不採 (不合尺寸不采)。
* **出茶率：** 製作500克（1斤）乾茶需要約40,000個鮮芽。這是世界上原料與成品比例最「費工」的茶之一。

**4. 風土與種植特色：**

* **君山島：** 洞庭湖中的小島（不足1平方公裡）。四面環水 — 絕對的島嶼隔離。土壤為酸性砂質紅壤 (砂质酸性红壤)，疏鬆肥沃，富含礦物質。森林覆蓋率超過90%。
* **氣候：** 中亞熱帶濕潤氣候。年平均氣溫16–17°C。年降水量約1340毫米。3月至9月相對濕度≥80%。春夏湖水蒸發形成持續的雲霧。散射光 (漫射光) 是主要光照類型。溫差：地表溫差顯著，空氣溫差因湖泊熱緩衝而平緩。
* **茶園面積：** 總計307畝（約20.5公頃）。這使君山成為世界上最小的茶葉風土之一。「君山銀針」年產量約400公斤，「君山毛尖」約2000公斤。市場需求超過80,000公斤/年，這意味著：超過99%以「君山銀針」名義銷售的茶並非來自該島。（為擴大生產，計劃在附近山丘增加4600畝：雲霧山2000畝，竹木山1500畝，天井山1100畝。）
* **特色：** 君山島是保護區。嚴格的生態標準。無工業污染。獨特的微氣候 —「島嶼效應」：湖水緩解溫度波動，提供持續濕度和散射光照 — 芽頭緩慢生長、最大程度積累胺基酸的理想條件。

**5. 製作工藝：**

君山銀針工藝是黃茶中最複雜、最耗時的之一：10道工序，72小時，完全手工製作。主要獨特之處 —「雙悶黃」(双闷黄, shuāng mēnhuáng)：兩個連續的悶黃週期，確保15–25%的發酵度。

* **攤青 (摊青 — tān qīng)：** 鮮芽薄攤在竹篩上輕度萎凋。1–2小時。
* **殺青 (杀青 — shā qīng)：** 溫度約80–100°C。短時間精細炒制。芽頭嫩重 — 需特別小心，避免破壞結構和影響「三起三落」能力。
* **攤涼 (摊凉 — tān liáng)：** 冷卻至室溫。
* **初烘 (初烘 — chū hōng)：** 輕度乾燥。傳統上用槐炭 (槐炭, huái tàn) 在竹架上烘焙 — 認為槐木炭火最純淨，能固定香氣。
* **初包 (初包 — chū bāo,「初次包裹」)：** 芽頭用特製牛皮紙 (牛皮纸, niúpí zhǐ) 每500克一包，放入木箱。溫度為室溫或略高。時間40–48小時。這是第一個悶黃週期 — 緩慢可控的黃變。發生非酶性多酚氧化、葉綠素分解、黃色素形成、澀味減輕。
* **復烘 (复烘 — fù hōng)：** 再次乾燥以停止黃變並部分除濕。
* **復包 (复包 — fù bāo,「二次包裹」)：** 第二個悶黃週期 — 約20小時。深化轉化：橙色素增強，形成特有的油潤和「絲滑」口感。兩次悶黃總時間達68小時。正是「雙悶黃」創造了芽頭內層特有的金橙色，使茶獲得「金鑲玉」美譽。
* **足火 (足火 — zú huǒ)：** 達到完全乾燥。傳統上用槐炭烘籠定香 (槐炭烘笼定香)。溫度逐漸降低。
* **精選 (精选 — jīng xuǎn)：** 精細手工分級。淘汰所有不符合標準的芽頭：彎曲、破損、茸毛不足、過小或過大的。

**6. 感官特徵：**

* **乾茶外形：** 筆直緊實的針芽狀 (针芽状)。覆蓋濃密銀白茸毛 (满披银毫)。色澤金黃光亮 (金黄光亮)。內層橙金色，外層因茸毛呈白色：「金鑲玉」(金镶玉,「金鑲玉石」)。大小均勻 (长短大小均匀)。
* **乾茶香氣：** 清香 (清香, qīngxiāng)，帶「毫香」(毫香, 茸毛香氣) — 類似煮玉米。甘甜，帶熟桃和橘皮香調 (熟桃/蜜桔)。
* **茶湯香氣：** 「清純」(清纯) — 清淨純潔。花香調，蜂蜜，淡果香。無草青味，無煙味。香氣是偉大黃茶中最不「激進」的：不是栗香（如霍山），不是玉米香（如平陽），而是「玉香」— 清淨涼爽。
* **滋味：** 「甘醇甜爽」(甘醇甜爽) — 甘甜、柔和、清爽，具油潤質感和「黏稠感」(粘稠感, 茶湯「體感」)。苦澀幾乎不存在。回甘持久、甘甜、包覆性強。滋味是黃茶中最「絲滑」的，得益於雙悶黃。霍山的「三鮮」在這裡轉化為「三嫩」(三嫩)：香氣、滋味和質感的嫩度。
* **湯色：** 「杏黃明淨」(杏黄明净) — 杏黃色，清澈透明，帶金色光澤。明顯比霍山黃芽或蒙頂黃芽更深更「溫暖」— 深度雙悶黃的結果。
* **葉底（沖泡後茶葉）：** 「肥厚勻亮」(肥厚匀亮) — 厚實、均勻、亮澤的黃金色芽頭。嫩軟、飽滿、有彈性。

**7. 化學成分：**

* **多酚類：** 25–30%。雙悶黃（68小時）深度轉化酯化兒茶素 — 澀味降低程度遠超其他黃茶的單次悶黃。
* **胺基酸：** ≥4.3%。L-茶胺酸為主要成分。君山銀針多酚類的抗氧化活性評估為維生素E的18倍。
* **生物鹼：** 咖啡因 + 茶胺酸 — 經典的「溫和興奮劑」，具協同效應。
* **維生素：** C、B群。
* **礦物質：** 鉀、鎂、氟（抑制齲菌 — 抑制致齲細菌）、鋅、硒。
* **茶多糖和黃酮類：** 顯著含量，提供抗氧化、降脂和免疫調節活性。

**8. 保健功效：**

* **抗氧化保護：** 多酚類（25–30%）+ 黃酮類。高效中和自由基活性。
* **代謝支持：** 咖啡因和L-茶胺酸協同刺激代謝，提供溫和持久的提神效果，無緊張感。
* **消化系統健康：** 兒茶素促進脂肪分解。雙悶黃使茶對胃部特別溫和 — 比綠茶溫和得多。
* **牙齒健康：** 氟抑制致齲細菌活性。
* **降脂作用：** 茶多糖和黃酮類有助降低血脂水平。
* **對敏感胃部的溫和性：** 由於雙悶黃的深度轉化，君山銀針特別推薦給胃部敏感人群。

**9. 沖泡方法：**

* **水溫：** 80–90°C。絕不可用沸水 — 否則芽頭「燙傷」，失去「三起三落」能力並產生苦味。
* **茶量：** 3克配150–200毫升水。
* **茶具：** 透明玻璃杯 — 必須。正是在玻璃中展現「銀針之舞」。替代選擇 — 白瓷蓋碗品香。
* **過程：**
    1. 用沸水溫杯，倒掉。
    2. 投入3克茶。
    3. 緩慢注入80–90°C水。第一泡不倒掉 — 其中含有最多「毫香」(毫香) 和果香調。
    4. 觀察「三起三落」(三起三落)：芽頭先浮到表面，然後慢慢沉到底部，再次上升再次下沉 — 最多三次。物理原理：芽頭外層吸水變重（芽頭下沉），然後芽體膨脹，密度減小（芽頭上浮），接著吸水繼續 — 週期重複。「三起三落」不是傳說，而是物理事實，由君山芽頭獨特的厚度和密度決定。
    5. 浸泡2–3分鐘。芽頭垂直立於底部 —「群筍出土」(群笋出土)，「刀槍林立」(刀枪林立)。
    6. 重複沖泡：可沖3–4次，延長時間。
* **注意：** 不要用保溫瓶裝沸水 — 芽頭會「煮熟」，失去形狀、香氣和「舞蹈」能力。

**10. 儲存：**

君山銀針需要特別小心的儲存。傳統方法 —「石灰壇密封」(石灰坛密封)：密封陶瓷容器配生石灰包（吸濕劑），蠟封。現代方法 — 鋁箔袋密封包裝，冰箱（0…+5°C）防氧化。重要：新茶具有殘餘「火氣」(火气) — 建議存放2週後飲用。一些愛好者實行長期陳化 (陈化) — 隨年份增長，滋味變得更圓潤蜜甜。茶的敵人：光、熱、濕、異味、氧氣。

**11. 價格與假冒：**

君山銀針是世界上最昂貴和最常被假冒的茶之一。島上正宗「銀針」年產量約400公斤。市場需求超過80,000公斤。這意味著：每1克正宗茶對應約200克假冒品。

* **價格參考：** 特號/特級 — 2000元/斤（500克）起，高檔禮盒裝可達5000–10,000元。一號 — 1000–2000元。二號 — 500–1000元。
* **如何避免假冒：**
    * 主要威脅：將其他地區綠茶冒充「君山銀針」銷售。有時將福建白毫銀針 (白毫银针) 冒充「君山銀針」— 名稱中都有「銀針」。
    * 正宗君山銀針：內層橙金色（非綠色！），外層銀白色。「金鑲玉」。綠色內芯是未經悶黃綠茶的特徵。
    * 茶湯杏黃色 (杏黄)，非淺綠色。
    * 「三起三落」— 雖不能保證真偽，但正確沖泡時缺乏此現象是警示信號（其他地區芽頭通常密度較小）。
    * 向「君山」牌 (君山牌) 官方經銷商購買 — 湖南君山銀針茶業有限公司。

**12. 趣聞軼事：**

* 君山銀針是經典「中國十大名茶」名單（1959年）中唯一的黃茶。這意味著它代表整個黃茶類別進入國家「殿堂」。
* 毛澤東稱其為「會跳舞的茶」(会跳舞的茶) — 因「三起三落」現象。物理原理：芽頭外層和內層非同步吸水造成密度波動，使芽頭週期性下沉上浮。
* 製作500克茶需要約40,000個芽頭。每個芽頭手工採摘，按9項標準篩選 — 而且是在不足1平方公裡的島上。採摘季節7–10天。這是地球上最「費工」的茶之一。
* 娥皇女英傳說：舜帝的兩位妻子在前往其死亡地點途中溺死洞庭湖。她們在竹子上的眼淚創造了「湘妃竹」(湘妃竹) — 忠貞愛情的象徵。她們在君山土地上的眼淚孕育了茶樹。因此，君山茶 — 字面意思是「愛情眼淚孕育的茶」。
* 2006年君山銀針作為「國禮」贈送給俄羅斯總統普京 — 茶葉作為最高級別外交禮品的少數案例之一。
* 君山島除茶園外，還有：娥皇女英墓 (二妃墓)、秦始皇封山印 (封山印)、柳毅井 (柳毅井) — 中國最著名愛情傳說之一的故事發生地，李白、杜甫、白居易、範仲淹的足跡。君山不僅是茶葉風土，更是中華文明的「樞紐點」之一。
* 「雙悶黃」(双闷黄) — 其他黃茶都沒有的獨特工藝。兩個悶黃週期 — 48 + 20 = 68小時 — 黃茶中最長的發酵時間（對比：蒙頂黃芽約8小時，霍山黃芽1–2天「乾攤」）。

**13. 與其他黃茶比較：**

* **霍山黃芽 (霍山黄芽)：** 兩者都是芽茶類「黃芽茶」，都在「四大名茶」中。霍山 — 山地、礦物質、栗香，「乾攤」1–2天。君山 — 湖島、絲滑、杏香，「雙包悶黃」68小時。霍山 —「知識分子茶」；君山 —「藝術家茶」。
* **蒙頂黃芽 (蒙顶黄芽)：** 兩者都是古代貢茶，有傳奇歷史。蒙頂 — 高山（1456米），蜜香，劍形，「三炒三悶紙包」約8小時。君山 — 湖泊（0米），油潤，針形，「雙悶」68小時。蒙頂 — 浪漫隱士；君山 — 帝國外交官。
* **平陽黃湯 (平阳黄汤)：** 平陽 — 海洋性，玉米香，捲曲，72小時「九烘九悶」。君山 — 湖泊性，桃香，針形，72小時「雙包悶黃」。總時間相同 — 方法完全不同。
* **霍山黃大茶 (霍山黄大茶)：** 截然相反：黃大茶 — 粗放大葉茶，「麵包」香氣，一週堆悶。君山 — 最嫩單芽，「絲綢」口感，雙包悶黃。民間麵包 vs. 帝王絲綢 — 都是黃茶，都偉大，都不可替代。

**結語：**

君山銀針是集美麗、滋味、歷史、神話、外交和物理學於一身的茶。這是唯一在杯中「舞蹈」的茶 — 不是因製茶師的意願，而是因島嶼自然賦予芽頭結構的流體力學定律。這是誕生於兩位女子為愛夫哭泣眼淚的茶 — 後來成為贈送總統和總書記的「國禮」。這是來自李白和秦始皇踏足、範仲淹寫下天下憂樂名句之島的茶 — 307畝茶園年產僅400公斤銀針。「金鑲玉」不僅指芽頭顏色，更指這款承載整個文明的茶。
