# 毛蟹茶 (毛蟹茶, máoxiè chá)

**毛蟹茶 (毛蟹茶, máoxiè chá)**
![毛蟹茶 (毛蟹茶, máoxiè chá)](images/Маосе%20Ча%20%28毛蟹茶,%20Máoxiè%20Chá%29.png)

![毛蟹茶 (毛蟹茶, máoxiè chá)-1](images/Маосе%20Ча%20%28毛蟹茶,%20Máoxiè%20Chá%29-1.png)

毛蟹茶是安溪縣 (安溪, Ānxī) 四大名茶之一，與鐵觀音 (鐵觀音, tiěguānyīn)、本山 (本山, běnshān) 和黃旦 (黃旦, huángdàn) 齊名。此茶屬於「色種」(色種, sèzhǒng) 類別，是安溪主要的外銷烏龍茶：毛蟹與其他色種茶占安溪烏龍茶出口量的70%以上。該品種具有耐寒、高產的特性，並帶有獨特的茉莉花香，在閩南烏龍茶中獨樹一幟。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶，清香型氧化程度15-30%，焙火型可達35-40%）。同一品種亦可製成紅茶（全發酵）和綠茶（不發酵），但傳統上以烏龍茶最為普遍。
* **類別：** 閩南烏龍茶 (閩南烏龍, Mǐnnán Wūlóng)。在安溪傳統中，毛蟹屬於「色種」(色種) 類別——即「非鐵觀音」品種，此類還包括本山、黃旦、大葉烏龍 (大葉烏龍, dàyè wūlóng) 和梅占 (梅占, méizhān)。
* **產地：** 中國福建省 (福建省, Fújiàn Shěng) 安溪縣 (安溪縣, Ānxī Xiàn) 大坪鄉 (大坪鄉, Dàpíng Xiāng) 福美村 (福美村, Fúměi Cūn) 大丘崙 (大丘崙, Dàqiūlún) 地區。
* **地理座標：** 約北緯24°53′，東經117°58′（大坪鄉，核心產區）。

**2. 歷史與文化意義：**

* **歷史：** 毛蟹品種的起源記載於1979年福建省農業科學院茶葉研究所出版的《茶樹品種志》。據記載，西元1907年（光緒三十三年），萍州村 (萍州村, Píngzhōu Cūn) 茶農張加協 (張加協, Zhāng Jiāxié) 前往購買布料途中經過福美村，從當地村民高香處得知有一種生長特別快速的茶樹，種植兩年即可採收。張氏帶回一百多株茶苗種植，因產量高、品質佳，很快在萍州一帶推廣開來。

  民間另有傳說：光緒年間（1875-1908），福美村青年茶農高坑 (高坑, Gāokēng) 在石牆縫隙中發現一株奇特的茶樹，既非鐵觀音，亦非黃旦或本山。他將此樹移植到自家茶園，精心培育並製成茶葉。鄰居品嚐後讚不絕口，發現其帶有獨特的茉莉花香。因葉片邊緣鋸齒深且密，形似大坪大澗溪中的毛蟹腳，故得名「毛蟹」(毛蟹)。

  1949年後，中華人民共和國政府積極推廣毛蟹在安溪全縣的種植。1985年，全國農作物品種審定委員會認定毛蟹為國家品種 (國家品種, guójiā pǐnzhǒng)，編號GS13006-1985。

* **名稱：** 毛 (máo) 意為「絨毛、茸毛」；蟹 (xiè) 即「螃蟹」。名稱反映了品種的兩個形態特徵：芽葉背面密布的白毫 (白毫, báiháo)，以及葉緣深刻、尖銳、下彎如蟹爪的鋸齒。古名茗花 (茗花, Mínghuā，意為「茶中之花」)；民國時期亦稱毛外 (毛外, Máowài)。

* **文化意義：** 毛蟹穩居「安溪四大名茶」(安溪四大名茶) 之列，是該縣主要的外銷茶品。著名茶學家張天福 (張天福, Zhāng Tiānfú)——「中國茶界四大泰斗」之一——稱大坪鄉為「茶海明珠」(茶海明珠, cháhǎi míngzhū)，正是因其為毛蟹的故鄉。此茶出口日本、瑞典等國；在瑞典以「銀毫美人茶」(銀毫美人茶) 的詩意商品名銷售。

**3. 植物學描述與原料：**

* **品種/品系：** 毛蟹 (毛蟹, Máoxiè)，又名茗花 (茗花)。無性繁殖；灌木型 (灌木型, guànmù xíng)，中葉類 (中葉類, zhōngyè lèi)，中芽種 (中芽種, zhōngyá zhǒng)；混倍體。屬於 *Camellia sinensis* var. *sinensis*。樹姿半開展 (半開展, bànkāizhǎn)，分枝密集。葉片橢圓形，平展，尖端銳尖；深綠色；葉片厚實易碎；葉緣鋸齒深刻、尖銳、下彎（重要鑑別特徵）。芽梢粗壯，節間短；葉背及嫩梢密布白毫 (白毫)。開花多但幾乎不結果。

  年生長期約8個月。發芽力強，但持嫩性 (持嫩性) 相對較弱——葉片易老化。因此需頻繁少量採摘。樹冠形成快，扦插易成活，抗旱抗寒，產量高：每畝（約667平方公尺）可產製成烏龍茶200-300公斤。

* **採摘：** 春季（4月至5月初）和秋季（9-10月）為傳統採摘季；夏茶亦可採製，但香氣較淡。「一芽三葉」盛期在4月中旬。
* **採摘標準：** 主要為「一芽二三葉」(一芽二三葉)。手工採摘：以拇指和食指捏住嫩梢中部，彈性折斷。最佳採摘時間為正午至下午3點，此時露水已乾，葉內芳香物質含量最高。
* **原料要求：** 完整嫩梢，成熟度一致，無機械損傷、病蟲害及異味。特級原料中「一芽二葉」比例須達90%以上，白毫特別顯著。

**4. 風土條件與栽培特點：**

* **產區與地形：** 核心產區為安溪縣大坪鄉 (大坪鄉)，包括萍州村 (萍州村，首次種植地)、福美村 (福美村，原產地) 和楓林村 (楓林村)。大坪鄉產量約占全縣毛蟹總產量的60%。除大坪外，毛蟹還種植於虎邱 (虎邱)、城廂 (城廂)、蓬萊 (蓬萊)、魁斗 (魁斗)、金谷 (金谷)、湖頭 (湖頭)、官橋 (官橋)、龍門 (龍門) 等13個以上鄉鎮。縣外則推廣至福建其他縣市，以及廣東、江西、湖南、浙江、湖北、安徽等省。

* **海拔高度：** 500-1200公尺。大坪馬峰山 (馬峰山, Mǎfēng Shān) 山頂達1200公尺，為種植區最高點；大部分茶園位於600-850公尺。

* **氣候：** 中亞熱帶濕潤季風氣候 (中亞熱帶)。年均溫15-18°C，日夜溫差超過10°C。年降雨量1600-1900公釐，相對濕度78-80%，年霧日超過180天，散射光比例約70%。年日照時數約1875小時。涼爽、多霧、散射光充足及顯著的日夜溫差有利於芳香物質和胺基酸的累積。

* **土壤：** 酸性紅壤 (紅壤, hóng rǎng) 和黃壤 (黃壤, huáng rǎng)，pH值4.5-6.0。土層厚度超過1公尺，有機質含量≥2.45%。大坪森林覆蓋率78%，無工業污染。土壤富含硒元素，使該區成為高標準「綠色食品」(綠色食品) 生產基地。

**5. 製作工藝：**

毛蟹烏龍茶的製作工藝基本與安溪鐵觀音類似，但發酵程度略輕：毛蟹在相對較低的氧化度下保持鮮明的花香茉莉韻。關鍵原則是「看天做青」(看天做青, kàn tiān zuò qīng)：製茶師根據溫濕度調整搖青的強度和時間。成形時不使用機械揉捻，僅採用手工或半手工揉捻，以保持葉片完整性。

* **採摘 / 採摘 — cǎizhāi：** 正午時分（12:00-15:00）採摘「一芽二葉」標準嫩梢，此時葉片含水量最低。鮮葉立即送往加工廠。
* **萎凋 / 萎凋 — wěidiāo (晒青 — shàiqīng)：** 將茶葉薄攤於陽光下或用熱風吹拂 (熱風萎凋)；目標是失水約10%。葉片變軟有彈性，開始產生香氣物質。
* **搖青與靜置 / 搖青 — yáoqīng：** 進行4-5輪搖青，中間穿插靜置。機械作用損傷葉緣細胞，啟動部分氧化；過程中形成關鍵芳香化合物——特別是茉莉花香 (類茉莉花香)。這是最需創造性的步驟，要求製茶師準確「解讀」香氣。
* **殺青 / 殺青 — shāqīng (炒青 — chǎoqīng)：** 在約260°C的鍋中加熱，停止酶促反應，固定已形成的香氣特徵。
* **揉捻 / 揉捻 — róuniǎn：** 手工（或半手工）揉捻形成安溪烏龍茶特有的緊結圓粒狀，呈「頭大尾尖」(頭大尾尖) 形態。避免粗暴機械揉捻，以免破壞葉片完整性和茸毛。
* **烘焙與乾燥 / 烘焙 — hōngbèi / 乾燥 — gānzào：** 兩階段進行：初烘 (初烘, chūhōng) 60-80°C，終烘至含水量≤6%。清香型 (清香, qīngxiāng) 僅輕度烘焙；焙火型則增加炭焙 (炭焙, tànbèi) 工序。

**6. 感官特徵：**

* **乾茶外觀：** 緊結的顆粒狀 (顆粒狀)，呈特徵性「頭大尾尖」形態，帶明顯砂綠色 (砂綠色, shālǜ sè)。表面可見白毫（白色茸毛）。落在瓷器表面時發出清脆響聲——緊結度的標誌，傳統形容為「沉重如鐵」(沉重如鐵)。茶梗圓形，有別於鐵觀音的橢圓形茶梗。
* **乾茶香氣：** 純淨、高揚，帶明顯茉莉花香 (茉莉花香, mòli huāxiāng)——品種的標誌性特徵。伴有鮮綠調和淡淡奶油甜香。
* **茶湯香氣：** 花香為主，前幾泡以茉莉花香為主導。隨著葉片舒展，出現果香調並增強蜜甜感。焙火型 (濃香型) 帶堅果和焦糖香；陳化後呈現蜜香 (蜜香)。
* **滋味：** 純淨柔和，具輕盈而明確的醇厚度 (醇厚, chúnhòu)。口感呈現細膩甜潤和鮮爽；回甘 (回甘, huígān) 持久綿長。高品質樣品展現柔和的「觀音韻」(觀音韻) 餘韻——安溪頂級烏龍茶特有的絲絨般深度。相較於鐵觀音，毛蟹茶湯較輕盈透亮，醇厚度稍低，但香氣更高揚穿透。
* **湯色：** 清香型呈青黃色 (青黃, qīnghuáng) 至焙火型的純正金黃色 (金黃, jīnhuáng)。茶湯透亮明澈。
* **葉底（沖泡後茶葉）：** 葉片展開呈現完整葉形——圓形、小巧、中部寬闊、兩端尖細。鋸齒深刻、密集、尖銳，特徵性下彎 (鋸齒下鉤)。葉底色澤從黃綠至橄欖綠；葉片柔軟有彈性。重要鑑別細節：毛蟹葉片向內捲曲，而鐵觀音葉片向外翻捲。

**7. 化學成分：**

毛蟹的化學成分在福建烏龍茶研究中已有詳細記錄，並在品種審定時確認。

* **茶多酚 (茶多酚)：** 含量14.7-20.1%（春季「一芽二葉」標準）。兒茶素為主要成分；總兒茶素含量約5.8%。在「搖青」過程的部分氧化中，部分兒茶素轉化為芳香醛類和醇類，形成茉莉花香特徵。
* **胺基酸：** 3.0-4.2%，其中L-茶胺酸占主要比例。高胺基酸含量源於高山風土的多霧和散射光環境，抑制兒茶素的光合作用並促進含氮化合物累積。
* **生物鹼：** 咖啡因3.2-4.1%（烏龍茶中屬中高水平），茶鹼和可可鹼含量極微。咖啡因與L-茶胺酸的協同作用提供溫和持久的提神效果，無急劇峰值。
* **水浸出物 (水浸出物)：** 特級茶≥45%——顯示茶湯濃郁、萃取充分。
* **維生素：** 維生素C、B群維生素（B₁、B₂）、蘆丁（維生素P）。
* **礦物質：** 鉀、鎂、錳、鋅。氟含量特別高——毛蟹在所有六大茶類中氟含量名列前茅。此外，大坪土壤富含硒，成品茶硒含量較高。
* **精油與芳香化合物：** 芳香成分是毛蟹區別於其他安溪品種的關鍵。以芳樟醇及其氧化物、順式茉莉酮和茉莉酸甲酯為主——這些化合物賦予茉莉花香。長期陳化後，香氣轉向蜜香和堅果香。

**8. 保健功效：**

* **提神醒腦與專注力：** 咖啡因（3.2-4.1%）與L-茶胺酸的結合提供溫和持久的清醒狀態，無咖啡引起的緊張感。支持認知功能和持續專注。
* **抗氧化保護：** 毛蟹的茶多酚（主要是兒茶素）有效中和自由基。安溪烏龍茶的抗氧化活性研究顯示其效果特別顯著。
* **支持脂質代謝：** 茶多酚參與調節膽固醇和三酸甘油酯代謝。傳統上安溪烏龍茶與維持健康體重相關——1979年和1984年在日本「烏龍茶熱潮」期間，毛蟹以「美容茶」和「瘦身茶」品牌銷售。
* **強化牙齒琺瑯質：** 毛蟹的高氟含量有助於形成氟磷灰石——提高琺瑯質抗酸蝕和蛀牙的化合物。
* **支持免疫系統：** 高硒含量刺激免疫蛋白和抗體的產生。
* **溫和助消化：** 烏龍茶多酚及其氧化產物幫助分解脂肪和蛋白質，促進消化——焙火型尤為明顯。
* **幫助減壓：** 功夫茶 (功夫茶) 多次沖泡的儀式結合L-茶胺酸的溫和鎮靜作用，創造有利於正念休息的條件。

**9. 沖泡方法：**

* **水溫：** 95-100°C（滾水）。清香型可降至90-95°C；焙火型和陳年毛蟹必須使用100°C。
* **茶量：** 7克配140毫升（功夫泡，1:20比例）或3-4克配200-250毫升（西式泡法）。
* **茶具：** 瓷蓋碗 (蓋碗, gàiwǎn)——通用選擇，便於觀察湯色和聞蓋香。焙火型適合紫砂壺 (紫砂壺, zǐshā hú)——紫砂透氣性突顯滋味深度。
* **步驟：**
  1. 以沸水溫潤蓋碗或茶壺，倒掉水。
  2. 投入7克乾茶。
  3. 第一泡洗茶 (溫潤泡, wēnrùn pào)——注水後立即倒出。這能「喚醒」緊結的茶葉。
  4. 第一泡：注入沸水，浸泡10-15秒，倒入公道杯 (公道杯)。
  5. 分茶入杯。
  6. 後續沖泡：可達7泡以上，每泡增加5-10秒。優質毛蟹在5-6泡內保持穩定香氣和甜度。

毛蟹也適合冷泡 (冷泡, lěng pào)：5-6克茶配500毫升冷水，冰箱冷藏6-8小時。冷泡茶湯的茉莉花香特別純淨清新，苦澀味幾乎消失。

**10. 儲存方法：**

* **清香型：** 密封包裝（真空或鋁箔袋），冷藏於0-5°C。此條件下可保鮮6-12個月。開封後建議6個月內飲用，避免多酚氧化。
* **濃香型（焙火型）：** 對溫度較不敏感；室溫下密封不透光容器儲存。保質期18-24個月以上。
* **陳年毛蟹 (陳年毛蟹)：** 如同其他安溪烏龍茶，焙火毛蟹可陳化。隨年份增長，香氣轉向蜜香、堅果香和木質香；茶湯呈深琥珀色。
* **茶葉大敵：** 濕氣、陽光直射、異味（茶葉強烈吸附氣味）、高溫。

**11. 價格與防偽：**

* **價格區間：** 毛蟹是最實惠的安溪烏龍茶之一，因品種高產且種植區域廣泛。大宗產品（普通等級）價格遠低於同品質鐵觀音。然而優質毛蟹——高山、春茶、手工製作——可達每斤（500克）500-600元人民幣以上，媲美優質鐵觀音。價格因素：海拔高度、採摘季節（春茶貴於秋茶和夏茶）、手工vs.機製、焙火程度、製茶師和特定產地聲譽。市場上優質批次常以「毛蟹王」(毛蟹王, Máoxiè Wáng) 商品名銷售——「王」(王) 字尾表示精選原料和特別精心製作。

* **如何避免假貨：**
  * 向可追溯來源的商家購買（可追溯至具體村莊/製茶師）。「綠色食品」認證或有機標章是良好指標。
  * 評估外觀：正宗毛蟹有明顯白毫、緊結顆粒和特徵性砂綠色。劣質原料的仿品通常鬆散暗淡。
  * 檢查香氣：純淨、高揚的茉莉花香是真品關鍵標誌。刺鼻「香水味」、化學甜味或霉味是警訊。
  * 評估茶湯：滋味應純淨柔和，無苦澀和「空洞感」。葉底應為完整、不大的葉片，具深刻鋸齒。
  * 對聲稱高山或「有機」來源卻價格離譜低廉的產品保持警覺。

**12. 趣聞軼事：**

* 毛蟹是最「適應性強」的安溪品種之一：同樣原料可製成烏龍茶、紅茶和綠茶。外銷批次中，紅茶毛蟹因茸毛豐富、外觀美觀和濃郁甜味而受重視。
* 毛蟹的特徵之一是所謂「串味」(串味, chuànwèi)，香氣中略帶穿透性的調子，有別於鐵觀音更圓潤平衡的香調。行家將此特點視為品種的標誌。
* 毛蟹是茶類中氟含量的紀錄保持者。傳統中醫認為經常飲用毛蟹可預防蛀牙。
* 1979年和1984年，毛蟹在日本「烏龍茶熱潮」中以「美容茶」(美容茶) 和「減肥茶」(減肥茶) 銷售，在年輕日本消費者中大受歡迎。
* 毛蟹嫩梢在扦插後僅兩年即可產出商品茶葉——是世界上最快進入生產期的茶樹品種之一。

**13. 與其他安溪烏龍茶的比較：**

* **鐵觀音 (鐵觀音, Tiěguānyīn)：** 安溪四大名茶中的「大哥」。茶湯更濃厚沉重；明亮的花香蘭香帶深沉「觀音韻」；葉片較大，茶梗較粗（似鼓槌）。毛蟹較輕盈透亮，香氣更高揚（茉莉vs.蘭花）；茶梗較細，葉底向內捲（鐵觀音向外翻）。鐵觀音是晚生種 (晚生種)，毛蟹是中芽種 (中芽種)，採收期較早。
* **本山 (本山, Běnshān)：** 鐵觀音的近親，約1870年發現。滋味和葉形似鐵觀音，但茶梗較細，節間明顯（似竹枝）。毛蟹與本山不同，白毫顯著，鋸齒呈「蟹爪」狀。
* **黃旦/黃金桂 (黃旦/黃金桂, Huángdàn/Huángjīnguì)：** 安溪品種中最早熟（「清明茶」）。葉片細長，黃綠色；香氣極其高揚穿透（桃子、肉桂）。毛蟹屬中期品種，茉莉花香更圓潤，茶湯醇厚度更高。
* **梅占 (梅占, Méizhān)：** 三洋村的小喬木型 (小喬木型) 品種。用途廣泛：可製烏龍、紅茶和綠茶。葉片比毛蟹更大更壯；香氣具特徵性「梅占串」——尖銳穿透調，有別於毛蟹的茉莉花香。

**結語：**

毛蟹茶是安溪茶界的「勞動模範」，在最佳表現時可達到真正大師作品的水準。實惠的價格、茶樹的高產特性，以及鮮明獨特的茉莉花香，使毛蟹成為尋求花香特徵而無需承擔鐵觀音價格門檻者進入閩南烏龍世界的理想起點。而對於資深品茗者，來自大坪高山茶園的陳年或精心焙火的毛蟹，提醒著我們：偉大的茶葉不僅取決於品種名稱，更重要的是風土和製茶師的技藝。
