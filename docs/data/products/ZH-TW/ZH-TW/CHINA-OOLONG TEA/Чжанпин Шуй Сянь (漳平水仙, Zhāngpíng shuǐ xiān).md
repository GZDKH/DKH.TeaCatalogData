# 漳平水仙 (漳平水仙, Zhāngpíng shuǐ xiān)

**漳平水仙 (漳平水仙, Zhāngpíng shuǐ xiān)**
![Чжанпин Шуй Сянь (漳平水仙, Zhāngpíng shuǐ xiān)](images/Чжанпин%20Шуй%20Сянь%20%28漳平水仙,%20Zhāngpíng%20Shuǐ%20Xiān%29.png)

![Чжанпин Шуй Сянь (漳平水仙, Zhāngpíng shuǐ xiān)-1](images/Чжанпин%20Шуй%20Сянь%20%28漳平水仙,%20Zhāngpíng%20Shuǐ%20Xiān%29-1.png)

漳平水仙是世界上唯一的緊壓烏龍茶，是福建省漳平市的名片。這款茶結合了閩北 (闽北, Mǐnběi) 和閩南 (闽南, Mǐnnán) 烏龍茶的工藝技術，以特色的方形茶餅形式出產，用紙包裝。2022年，其製作技藝被列入聯合國教科文組織非物質文化遺產名錄，成為「中國傳統製茶技藝及其相關習俗」的組成部分。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶，氧化程度約30–50%）。緊壓烏龍茶 (紧压乌龙茶, jǐnyā wūlóng chá) 類別的唯一代表。常經炭焙 (炭焙, tàn bèi) 處理。
* **類別：** 福建名茶 (福建名茶)。國家地理標誌產品 (国家地理标志产品, guójiā dìlǐ biāozhì chǎnpǐn)。別名：水仙茶餅 (水仙茶饼, Shuǐxiān chábǐng)、紙包茶 (纸包茶, zhǐ bāo chá)、四方茶 (四方茶, sìfāng chá)。
* **產地：** 中國 (中国) 福建省 (福建省, Fújiàn shěng) 龍岩市 (龙岩市, Lóngyán shì) 漳平市 (漳平市, Zhāngpíng shì)。生產區域涵蓋9個鄉鎮：南洋鎮 (南洋镇, Nányáng zhèn) — 生產核心（約佔總產量80%），雙洋鎮 (双洋镇, Shuāngyáng zhèn) — 茶餅的歷史發源地，以及赤水鎮 (赤水镇)、新橋鎮 (新桥镇)、吾祠鄉 (吾祠乡)、靈地鄉 (灵地乡)、溪南鎮 (溪南镇)、象湖鎮 (象湖镇) 和永福鎮 (永福镇)。
* **地理座標：** 東經117°10′–117°45′，北緯24°54′–25°47′。生產核心 — 九鵬溪 (九鹏溪, Jiǔpéng xī) 沿岸雲霧區，海拔600–800公尺。

**2. 歷史與文化意義：**

* **歷史：** 漳平的茶業可追溯至元代 (元, 1271–1368)：當時這些地方開始種植茶樹。到明 (明, 1368–1644) 清 (清, 1644–1912) 時期，茶業已達到相當規模，考古發現的明代宜興茶壺直接證明了發達的茶文化。1914年，雙洋鎮茶農鄧觀金 (邓观金, Dèng Guānjīn) 創造了世界獨有的緊壓水仙茶 — 水仙茶餅 (水仙茶饼, Shuǐxiān chábǐng) 技術，結合了閩北和閩南烏龍茶的加工方法。據說在光緒年間 (光绪, 1875–1908)，漳平的泰昌茶莊 (泰昌茶庄, Tàichāng cházhuāng) 茶葉曾獲得巴拿馬博覽會金獎。從1980年代開始了新階段：政府啟動了漳平水仙品牌化計劃。1995年該茶在第二屆中國農業博覽會上獲得金獎，2000年被中國茶葉博物館 (中国茶叶博物馆) 收藏並列入《中國名茶志》(《中国名茶志》)。2005年在中日韓茶文化論壇上獲得「國際五星茶王」稱號。2008年註冊地理標誌集體商標，2009年中華人民共和國農業部正式授予該茶地理標誌保護產品地位。2021年傳統製作技藝列入中華人民共和國第五批非物質文化遺產名錄，2022年11月29日列入聯合國教科文組織非物質文化遺產名錄。

* **名稱：**
  * 「漳平」(漳平) — 生產縣市名稱。地名可詩意解釋為「漳水之濱的平原」：漳指當地河流，平意為「平原、平坦」 — 「漳水之上，千山之中，此地獨平」(漳水之上，千山之中，此地独平)。
  * 「水仙」(水仙) — 字面意思為「水中仙人」或「水仙花」。這是茶樹品種名稱。據說源於方言中「祝」(祝) 音轉為「水」(水)：茶樹在建陽祝桃洞 (祝桃洞) 被發現，「祝桃仙」逐漸演變為「水仙」。
  * 緊壓形式有時加上「餅」(饼, bǐng — 「餅、圓盤」)，雖然形狀實際上是方形茶磚。

* **文化意義：** 漳平水仙是世界上唯一的傳統緊壓烏龍茶，是漳平縣的象徵之一。該茶曾受到知名人士青睞：據當地資料，周恩來總理是水仙茶的愛好者，1956年曾向日本客人推薦此茶。著名相聲 (相声) 表演藝術家姜昆 (姜昆) 為漳平水仙題字「茶香萬裡」(茶香万里)。該茶傳統上在閩西、廣東、廈門和東南亞國家享有盛譽。南洋每年的開採節 (开采节) 伴隨著祭拜「茶聖」陸羽 (陆羽, Lù Yǔ) 的儀式。

**3. 植物學描述與原料：**

* **品種/栽培種：** 福建水仙 (福建水仙, Fújiàn Shuǐxiān)，又稱水吉水仙 (水吉水仙) 或武夷水仙 (武夷水仙)。拉丁學名：*Camellia sinensis* cv. Fujian-shuixian。國家級良種 (国家级良种) 第一等，1985年註冊編號GS13009-1985。48個國家茶樹品種之首，41個半喬木型大葉種之首。屬無性系 (无性系, wúxìngxì)、半喬木型 (半乔木型 / 小乔木型, xiǎo qiáomù xíng)、大葉 (大叶, dà yè)、晚生種 (晚生种, wǎnshēng zhǒng)。歷史上該品種源於福建省建陽 (建阳, Jiànyáng) 小湖鎮 (小湖镇) 大湖村 (大湖村)。植物學特徵：葉片大（比普通小葉種大1.5-2倍）、厚、肉質、革質；節間伸長；主脈寬而平。
* **採摘：** 主要採摘 — 春季（4月至5月初）；也可秋季採摘。春茶因冬季積累的營養物質而更受重視。
* **採摘標準：** 一芽二葉至三葉 (一芽二叶至三叶, yī yá èr yè zhì sān yè) 的嫩梢用於春茶。高等級允許更細的標準 — 一芽一葉。
* **原料要求：** 葉片應健康、無損傷、成熟度均勻、無異味和過度粗糙。按標準，漳平水仙鮮葉應含多酚類不少於25.0%，氨基酸不少於4.0%。

**4. 風土條件與栽培特點：**

* **地區與地形：** 漳平位於福建中南部，閩西 (闽西, Mǐnxī) 東部，閩南「金三角」北緣。地形十分之九為山地 — 「九山半水半分田」(九山半水半分田)。地勢丘陵山地，被山谷、河流和溪流切割。茶園核心沿九鵬溪 (九鹏溪) 延伸，該溪灌溉山坡並形成雲霧區微氣候。
* **海拔高度：** 海拔400至1000公尺；生產核心 — 九鵬溪區600–800公尺。
* **氣候：** 亞熱帶季風氣候。年平均氣溫16.9–20.7°C。降雨量 — 年1450–2100毫米。無霜期 — 251–317天。年平均日照時數 — 約1853小時。霧日數 — 年超過200天。明顯的晝夜溫差有利於氨基酸積累和複雜香氣特徵的形成。生產核心區森林覆蓋率超過70%。
* **土壤：** 以紅黃壤 (红黄壤, hóng huáng rǎng) 為主，酸性反應（pH 4.5–6.0），有機質含量高（≥ 6.01%），富含微量元素 — 硒和鋅。生產核心區是水源保護區：禁止使用化學肥料和農藥，實行「豬-沼-茶」(猪-沼-茶) 生態系統。

**5. 製作工藝：**

漳平水仙工藝結合了閩北（「重發酵」）和閩南（「輕發酵」）烏龍茶技法，包含獨特的木模壓制工序。整個過程使用竹木工具，避免金屬接觸以防止氧化。

* **採青 / 采青 — cǎiqīng：** 晴天採摘頂部嫩梢，迅速運至加工廠，避免過熱和機械損傷。
* **曬青 / 晒青 — shàiqīng：** 相對強烈的日光萎凋 (较重, jiào zhòng) — 漳平工藝的特色。激活葉片酶活性，為香氣發展奠定基礎。
* **晾青 / 晾青 — liàngqīng：** 萎凋後的葉片移至陰涼處「休息」降溫。採用「薄攤多晾」(薄摊多晾, báo tān duō liàng) 原則。
* **做青 / 做青 — zuòqīng（搖青與靜置）：** 形成滋味和香氣的關鍵工序。搖青 (摇青, yáoqīng) 和靜置交替進行。早期階段使用水篩 (水筛, shuǐshāi) 輕搖；後期使用機械搖青機進行更強烈處理。原則：「輕搖多次」(轻摇多次)。正是「搖青薄攤」(摇青薄摊) 方法形成了特色蘭花香。
* **殺青 / 杀青 — shāqīng（炒青 — chǎoqīng）：** 高溫處理停止酶促反應，固定已達到的香氣特徵。
* **揉捻 / 揉捻 — róuniǎn：** 揉捻形成葉片外形，促進細胞壁破壞，提高沖泡時的浸出率。
* **造型 / 造型 — zàoxíng（含造型與定型）：** 獨特工序。將揉捻後的茶葉放入特製木模 (木模, mù mó)，用當地密實的黃椟木 (黄椟木, huáng dú mù) 製成 — 無異味，密度高。標準模具尺寸：6 × 6 × 2.5公分（內部截面約5 × 5公分）。用木槌 (木槌, mù chuí) 壓制，長25公分，方形底面4.5 × 4.5公分。每塊茶餅用食品級濾紙包裝（歷史上用無色無味的毛邊紙 (毛边纸)），起固定形狀作用。紙上常印紅色漢字 — 祝福、吉祥或地名。
* **烘焙 / 烘焙 — hōngbèi（炭焙 — tàn bèi）：** 最終乾燥-焙火在炭爐 (焙笼, bèilóng) 上進行，溫度不超過60°C。緩慢炭焙賦予茶葉深度、穩定性和儲存耐久性。
* **分級 / 分级 — fēnjí：** 成品茶餅按品質分為三個主要等級。

**6. 感官特徵：**

* **乾茶外形：** 緊實的方形茶餅 (方形, fāngxíng)，重量5至15克，整齊包在紙中。表面平整光滑。乾茶色澤 — 深綠帶褐黃色調，似香蕉色 (似香蕉色)。散茶 (散茶, sǎn chá) 形式具有緊結的縱向條索 — 「拐杖形」(拐杖形, guǎizhàng xíng) 或「扁擔形」(扁担形, biǎndan xíng)，特色的「三節色」(三节色, sān jié sè) 莖部和四方梗 (四方梗, sìfāng gěng)。
* **乾茶香氣：** 清純、高揚持久。以天然蘭花香 (兰花香, lánhuā xiāng) 為主 — 「如蘭似桂」(如兰似桂)。新茶更突出清香 (清香, qīngxiāng) 花香調；陳放後發展出桂花香 (桂花香, guìhuā xiāng)。可能帶有淡淡蜜香和果香。
* **茶湯香氣：** 明亮、花香甜美、高揚持久。明顯的天然「蘭花韻」；焙火版本增加堅果和焦糖香調。香氣逐泡遞增展現。
* **滋味：** 飽滿、柔和、醇爽 (醇爽, chún shuǎng) — 「純淨清爽」，同時柔和活潑 (鲜灵活泼, xiānlíng huópō)。苦澀味極少。明顯的回甘 (回甘, huígān) — 持久清晰。香味層次：花香（蘭花、水仙）、蜜香、鮮果和烤果、淡淡香料調。深度品飲時感受到「味中透香」。喉韻柔和包覆 (喉润, hóu rùn)。
* **湯色：** 橙黃 (橙黄) 或金黃 (金黄)，透明清亮 (清澈明亮)。焙火程度增加時色調轉向琥珀色。
* **葉底（沖泡後葉片）：** 葉片肥大、肉厚、柔軟光亮 (肥厚软亮)。特色的「紅邊」(红边, hóng biān) — 葉緣氧化痕跡。主脈寬黃 (主脉宽黄)。色調從黃綠到銅褐色。

**7. 化學成分：**

* **多酚類：** 鮮葉含量 — 不少於25.0%（按標準）。以兒茶素及其部分氧化產物為代表。據中國資料，漳平水仙的多酚類促進脂肪分解的效率比普通烏龍茶高約30%。
* **氨基酸：** 鮮葉含量 — 不少於4.0%。主要成分 — L-茶氨酸 (L-茶氨酸, L-chá'ānjī suān)，提供鮮爽度 (鲜爽度) 和特色的鮮味甜感。氨基酸含量高歸因於九鵬溪霧氣氣候和散射光。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēi jiǎn) — 中等水平，半發酵茶的典型含量。可可鹼 (可可碱, kěkě jiǎn) 和茶鹼 (茶碱, chá jiǎn) — 微量。
* **維生素：** 維生素C、B群維生素（B₁、B₂）、維生素E、維生素K。
* **礦物質：** 鉀（K）、鎂（Mg）、錳（Mn）、鐵（Fe）、氟（F）。該地區土壤富含硒（Se）和鋅（Zn），反映在茶葉的礦物質組成中。
* **精油：** 形成特色持久花香。成分包括橙花醇、芳樟醇、香葉醇及其酯類 — 「蘭花韻」的典型載體。
* **特點：** 據中國資料，漳平水仙的抗氧化活性在中和自由基能力方面比維生素E高18倍。

**8. 保健功效：**

* **溫和提神效果：** 咖啡鹼和L-茶氨酸的結合提供清醒感而無急劇波動：L-茶氨酸緩解咖啡鹼作用，形成平靜專注狀態。
* **抗氧化作用：** 高多酚含量有助保護細胞免受氧化應激，延緩衰老過程。
* **支持脂質代謝：** 茶多酚促進脂肪分解，可能有助降低「壞」膽固醇（LDL）水平。
* **消化：** 刺激消化酶分泌，幫助緩解腹脹。特色：長期規律飲用不傷胃 (久饮多饮而不伤胃)，這是漳平水仙相比許多綠茶的優勢。
* **心血管系統：** 可能有助加強血管壁和維持其彈性。
* **解毒：** 促進代謝產物的溫和排出。
* **認知功能：** L-茶氨酸支持認知活動，改善注意力，有助減少壓力負荷。
* **溫暖作用：** 焙火版本在寒冷季節特別好 — 溫暖身體而不過度負擔。

**9. 沖泡方法：**

* **水溫：** 95–100°C。不建議超過100°C：據中國專家資料，過高溫度破壞茶氨酸，降低茶湯鮮味 (鲜味, xiānwèi)。
* **茶量：** 一塊茶餅（5–8克）配75–100毫升水，功夫茶法。1:15比例（5克配75毫升）被認為是經典。
* **茶具：** 蓋碗 (盖碗, gàiwǎn) 瓷器 — 通用選擇，很好展現花香調。宜興紫砂壺 (宜兴紫砂壶, Yíxīng zǐshā hú) — 適合焙火版本，紫砂增加額外的柔滑感。
* **過程：**
  1. 用開水溫熱茶具，倒掉水。
  2. 將茶餅放入蓋碗或茶壺。如使用整塊茶餅，可輕輕掰開加速展開。讓茶葉在熱茶具中「呼吸」10–15秒。
  3. 洗茶 (润茶, rùn chá)：注水3–5秒立即倒掉。這喚醒茶葉並去除可能的茶末。
  4. 第一泡：注水浸泡15–20秒。將茶湯倒入公道杯 (公道杯, gōngdào bēi) 或直接倒入茶杯。
  5. 續泡：6–8泡，每泡增加5秒。優質漳平水仙可泡8次，逐漸展現滋味層次。
  6. 冷泡法 (冷泡法, lěng pào fǎ)：5克茶配250毫升冷水，冰箱浸泡6小時。此法「鎖住」鮮爽感，產生精緻淡雅茶湯。

**10. 儲存：**

* 存放在乾燥、陰涼、避光處，密封容器中，遠離異味。茶葉的敵人：潮濕、高溫、直射光、強烈氣味。
* 焙火版本因炭焙處理具有更強的儲存穩定性。適當條件下保質期可達36個月。
* 建議新茶在陰暗處放置15天「褪火」(褪火, tuì huǒ) 後首次沖泡。
* 開封後建議7天內飲完，以免香氣散失。
* 陳年版本 (陈年, chénnián) 允許長期儲存：適當條件下香氣演化 — 從花香到桂花香和木香。

**11. 價格與仿冒：**

* **價格區間：** 漳平水仙屬於福建烏龍茶中等偏高價位。參考價格（中國市場資料）：特級 (特级, tèjí) — 每斤（500克）700元人民幣以上；一級 (一级, yījí) — 每斤350–700元；二級 (二级, èrjí) — 每斤160–240元。影響價格因素：採摘季節（春茶較貴）、海拔高度、茶樹年齡、製作師傅和產區聲譽、手工程度、是否陳放。
* **如何避免仿冒：**
  * 向可提供具體鄉鎮和製作師傅資訊的可靠商家購買。尋找地理標誌標識。
  * 評估外觀：真正茶餅平整緊實，無碎屑和茶末。紙包裝整齊，常有紅色印章。
  * 檢查香氣：優質乾茶具有純淨高揚花香，無「化學」香精味、霉味或酸味。
  * 評估茶湯：色澤透明，橙黃或金黃。渾濁暗淡茶湯是品質低劣或儲存不當的徵象。
  * 異常低價需警惕：真正手工漳平水仙的成本使低價傾銷不可能。

**12. 有趣事實：**

* **世界唯一緊壓烏龍茶。** 在烏龍茶的巨大多樣性中 — 從臺灣高山茶到廣東單叢 — 漳平水仙保持絕對獨特：世界上沒有其他烏龍茶以緊壓形式出產。
* **一塊一泡。** 標準6 × 6公分方塊設計使其重量（5–8克）對應一次功夫茶沖泡份量。這使茶葉在儲存、運輸和計量方面極其便利。
* **無金屬工藝。** 整個過程 — 從搖青到壓制 — 專用竹木工具。與金屬接觸被認為對香氣有害，因為會引起不良氧化。
* **形狀演進：從條索到方塊。** 1914年前漳平水仙先是條索形 (条索形)，後為圓球形 (圆球形)。方形壓制 — 鄧觀金的發明 — 成為第三種也是最終形式，一百多年來未曾改變。
* **零農藥殘留。** 2018年對漳平201個茶樣檢測顯示完全無農藥殘留 — 這是持續生態政策和生物植保方法的結果。

**13. 與其他烏龍茶比較：**

* **武夷水仙 (武夷水仙, Wǔyí Shuǐxiān)：** 品種上最接近的「親戚」。武夷版本是散裝岩茶 (岩茶) 重焙火，明顯礦物特徵 (岩韵, yán yùn)，厚重茶體和苔蘚、老木、烤堅果調。漳平水仙相反強調鮮爽和花香；焙火較輕，緊壓形式提供更「聚合」的漸進展開。
* **鐵觀音 (铁观音, Tiě Guānyīn)：** 安溪著名閩南烏龍。清香型 (清香型) 鐵觀音以高花香和奶香著稱；濃香型 (浓香型) 具溫暖堅果特徵。漳平水仙具有水仙品種特有的更「水潤」、油潤茶體，酸味較不明顯。
* **凍頂 (凍頂, Dòng Dǐng)：** 臺灣中焙火烏龍，具堅果、焦糖和花香調。氧化程度與漳平水仙相似，但品種不同（多為青心 (青心)），形狀（球形揉捻無壓制）和風味重點不同 — 凍頂堅果-焦糖特徵更明顯。
* **鳳凰單叢 (凤凰单丛, Fènghuáng Dāncóng)：** 潮州廣東烏龍。雖同為花香方向，單叢展現顯著更高的香氣成分濃度和「單一主導」香氣特徵（一個主導調 — 梔子花、玉蘭、杏仁等），而漳平水仙提供更柔和的「和弦式」香氣。

**14. 漳平水仙的品種與等級：**

按出產形式：
* **水仙茶餅 / 水仙茶饼 — Shuǐxiān chábǐng：** 使用木模和炭焙製成的緊壓方塊。具有更好的儲存穩定性和更深沉「聚合」的滋味。蘭花香明顯。經典且最珍貴的形式。
* **水仙散茶 / 水仙散茶 — Shuǐxiān sǎn chá：** 特色「拐杖形」條索。沖泡更簡便，展現更高鮮爽度 (鲜爽度)，但儲存穩定性較差。

按原料品質：
* **特級 / 特级 — tèjí：** 最高品質芽頭（≥ 95%均勻嫩梢）。香氣 — 持久、高揚、蘭花香。茶湯金黃。參考價格 — 每斤700元起。
* **一級 / 一级 — yījí：** 一芽一葉嫩梢（≥ 90%）。條索緊結。明顯蜜香，滋味柔和飽滿。
* **二級 / 二级 — èrjí：** 一芽二葉嫩梢。香氣純正經典。性價比良好。參考價格 — 每斤160–240元。

**結語：**

漳平水仙是茶中悖論：作為世界唯一緊壓烏龍茶，它同時繼承兩大福建烏龍茶系傳統，卻不完全屬於任何一系。它的小紙方塊是寧靜與精準的化身：製茶師精確稱量，濾紙包裹，木槌壓制，60度炭火焙製。只要用水沖泡這不起眼的茶磚，它就開始訴說 — 關於九鵬溪霧氣山坡，關於鄧觀金的百年技藝，關於蘭花香在緊壓中的耐心醞釀。漳平水仙不會大聲宣揚 — 它逐泡展開，從一泡到下一泡，提供柔和甜美、純淨和深度。這款茶會在那些喜愛中國茶巨大合唱中低調而真實聲音的愛好者中找到知音。

**15. 武夷水仙與閩南水仙的差異：**

|  **特徵**<br/> | **武夷水仙**<br/> | **閩南水仙**<br/> |
|-----|-----|-----|
|  **外觀**<br/> | 較深，帶紅色調，緊結<br/> | 較淺，偏綠，結度較鬆<br/> |
|  **香氣**<br/> | 更濃郁，帶有烘焙、乾果香，「岩韻」<br/> | 較清新，花香，奶香<br/> |
|  **滋味**<br/> | 較濃厚，收斂，帶礦物質味，「岩韻」<br/> | 較柔和，甜潤，帶花香<br/> |
|  **湯色**<br/> | 較深，琥珀紅色<br/> | 較淺，金黃色<br/> |
|  **焙火程度**<br/> | 通常中火或重火，炭焙<br/> | 通常輕火或中火<br/> |
|  **回甘**<br/> | 持久，帶礦物質和辛香味<br/> | 較清淡，帶花香<br/> |
|  **功效**<br/> | 較強，提神，暖身<br/> | 較柔和，清爽<br/> |

**總結：**

水仙是一種多面向且有趣的烏龍茶，為品茶者提供了廣泛的口感和香氣體驗。根據產地、加工技術和焙火程度，它既可以是細膩花香的，也可以是濃郁飽滿，帶有明顯礦物質味的。武夷水仙無疑是岩茶中的瑰寶，是著名「岩韻」的體現。品嚐正宗的水仙，意味著發現中國茶的奇妙世界，感受大自然的和諧，並觸及古老的茶葉栽培傳統。這種茶既適合日常飲用，也適合在想要享受特別且難忘的茶品時的特殊場合。
