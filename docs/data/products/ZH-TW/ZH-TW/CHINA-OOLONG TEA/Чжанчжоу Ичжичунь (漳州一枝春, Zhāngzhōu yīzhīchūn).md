# 漳州一枝春 (漳州一枝春, Zhāngzhōu yīzhīchūn)

**漳州一枝春 (漳州一枝春, Zhāngzhōu yīzhīchūn)**

**漳州一枝春 (漳州一枝春, Zhāngzhōu yīzhīchūn)** — 經典的閩南烏龍茶，由國營漳州茶廠 (漳州茶厂, Zhāngzhōu cháchǎng) 於1956年創製。與流香 (流香, Liúxiāng) 和色種 (色种, Sèzhǒng) 並列，一枝春為漳州三大傳奇茶品之一，是閩南超過百年烏龍茶精製 (精制, jīngzhì) 傳統的活見證。2019年，漳州烏龍茶傳統精製技藝被列入福建省非物質文化遺產名錄。

**1. 分類與產地：**

* **類型：** 烏龍茶 (青茶, qīngchá) — 半發酵茶。氧化程度從中等到中度（約25–40%），使一枝春更接近傳統閩南烏龍茶，具有特色的炭焙 (炭焙, tàn bèi) 工藝。
* **類別：** 閩南烏龍 (闽南乌龙, Mǐnnán wūlóng)。拼配茶 (拼配, pīnpèi) — 最終產品通過不同季節、品種和產地原料的拼配形成，確保批次間風味的穩定性。
* **產地：** 中國福建省 (福建省, Fújiàn shěng) 漳州市 (漳州市, Zhāngzhōu shì)。生產基地為成立於1953年的漳州茶廠。茶葉原料來自漳州市轄區內11個縣市區，重點產區包括芗城區 (芗城区, Xiāngchéng qū)、平和縣 (平和县, Pínghé xiàn) 和華安縣 (华安县, Huá'ān xiàn)。
* **地理座標：** 約北緯24°30′，東經117°38′（漳州市）。主要生產基地位於芗城區小坑頭 (小坑头, Xiǎokēngtóu)；核心高山區為海拔600–800公尺的九鵬溪 (九鹏溪, Jiǔpéngxī) 流域。

**2. 歷史與文化意義：**

* **歷史：** 漳州茶廠成立於1953年，為國營企業，整合了閩南茶葉工藝的分散手工作坊。1956年，茶廠師傅正式創製「一枝春」品牌，依託當時已有百餘年歷史的閩南烏龍茶精製技術。在中華人民共和國早期，茶廠成為全國僅有的三家獲得茶葉出口嘜號 (出口唛号, chūkǒu mà hào) 的企業之一，在國際烏龍茶市場形成中發揮關鍵作用，特別是在日本市場。1960年代，一枝春與成風、飛馬、大前門香煙並列為奢侈品 — 民間俗語「乘風飛馬大前門，流香色種一枝春」(Chéngfēng Fēimǎ Dàqiánmén, Liúxiāng Sèzhǒng Yīzhīchūn) 將茶葉與頂級煙草並列。1970–1990年代，漳州烏龍茶出口量佔福建省烏龍茶總出口量的40%。1987年，一枝春榮獲「福建省優質產品」(福建省优质产品) 稱號。2019年，漳州茶廠傳統烏龍茶精製技藝被列入福建省第六批非物質文化遺產名錄。截至2024年，茶廠年產值超過4000萬元人民幣，產品出口至日本、新加坡、馬來西亞、印尼等東南亞國家。
* **名稱：** 一枝春 (Yīzhīchūn) 字面意思為「一枝春色」— 詩意的意象，源於古典文學表達 一枝春色 (yīzhī chūnsè, 「一枝春色」)，象徵自然復甦的初兆。漢字 漳州 (Zhāngzhōu) 標示產地。因此，完整名稱可譯為「漳州的春之枝」。
* **文化意義：** 一枝春不僅是茶葉，更是整個漳州茶業時代的象徵。對於閩南、潮汕地區以及東南亞華人社群的長輩而言，這款茶體現了古早味 (古早味, gǔzǎo wèi) — 「古早的味道」，對傳統生活方式的懷念。特色包裝 — 紅藍花紋的牛皮紙和50克 (一两, yī liǎng) 手工包裝 — 數十年來保持不變，成為文化符號。漳州也是最古老的工夫茶 (工夫茶, gōngfū chá) 中心之一：據彭光斗 (彭光斗, Péng Guāngdǒu) 在《閩瑣記》(《闽琐记》, Mǐn suǒ jì) 手稿中記載，早在1766年 — 乾隆年間 — 用小茶杯沖泡茶葉的技藝在漳州已普遍流行。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種為福建水仙 (福建水仙, Fújiàn Shuǐxiān) — *Camellia sinensis* cv. Fujian-shuixian。這是半喬木型 (小乔木型, xiǎo qiáomù xíng) 大葉晚生品種，1985年認定為國家品種（編號 GS13009-1985，「華茶9號」）。特點是主幹明顯，葉片大而橢圓，鋸齒深，表面深綠有光澤，葉質厚實。為天然三倍體。拼配時輔助使用黃旦 (黄旦, Huángdān，又稱黃金桂) 和奇蘭 (奇兰, Qílán) — 典型的閩南群體品種。
* **採摘：** 春季（4–5月）— 主要季節，提供氨基酸含量最高的原料；秋季（9–10月）— 第二重要季節，香氣更為突出。夏季採摘也用於大宗產品。
* **採摘標準：** 春季採摘 — 一芽二葉 (一芽二叶, yī yá èr yè)。大宗產品允許「頂梢二至四片成熟葉」標準。關鍵是保持芽葉完整和成熟度均勻。
* **原料要求：** 鮮葉茶多酚含量不少於25%，游離氨基酸不少於4.0%（一枝春原料標準）。葉片完整，無機械損傷、異味和過熟跡象。

**4. 風土條件與栽培特點：**

* **地區與地形：** 漳州市轄區位於福建省東南部：從西北部山區（戴雲山脈和博平嶺支脈）到東南部臺灣海峽沿岸。總面積約12,600平方公裡。地形以丘陵為主：山區茶園海拔400–800公尺，平原茶園位於九龍江 (九龙江, Jiǔlóngjiāng) 和漳江 (漳江, Zhāngjiāng) 流域。
* **海拔高度：** 主要茶園海拔200–600公尺。九鵬溪流域核心高山區海拔600–800公尺，霧日（超過200天/年）創造高散射光條件，促進葉片氨基酸積累。
* **氣候：** 南亞熱帶季風氣候 (南亚热带季风气候, nán yàrèdài jìfēng qìhòu)。山區茶區年均溫16–20°C（市區約21°C）。年降雨量1,450–2,100毫米，集中於3–9月。無霜期251–317天。山區晝夜溫差大，有利於緩慢生長和芳香物質積累。
* **土壤：** 以紅黃壤 (红黄壤, hóng huáng rǎng) 為主，酸性（pH 4.5–6.0）。富含微量元素硒 (Se) 和鋅 (Zn)，為茶湯礦物質風味做出貢獻。核心區茶園位於水源保護區，禁用化學肥料和農藥，實行生態種植。

**5. 製作工藝：**

一枝春為兩階段生產工藝產品。第一階段將鮮葉製成毛茶 (毛茶, máo chá) 半成品，採用經典閩南烏龍茶工藝。第二階段 — 品牌的關鍵環節 — 不同季節、品種和產地的半成品經過精製 (精制, jīngzhì) 工序：拼配、復焙和分級，形成穩定、可識別的風味特徵。正是這項代代相傳的精製技藝被認定為非物質文化遺產。第四代傳承人為林燕騰 (林燕腾, Lín Yànténg)，前漳州茶廠廠長和茶葉生產高級工程師。

工藝關鍵特色 — 全程使用竹木器具，避免茶葉與金屬接觸，防止催化氧化。

* **採摘 / 采摘 — cǎizhāi：** 晨間露水乾後手工採摘頂梢，迅速送廠，避免過熱和機械損傷。
* **攤晾 / 摊晾 — tānliàng：** 鮮葉薄攤於竹篩上，置通風室內。目標失水15–20%，為後續氧化創造條件。葉片變軟有韌性。
* **搖青 / 摇青 — yáoqīng：** 「搖青—靜置」循環（重搖青, zhòng yáoqīng — 此工藝特有的強力搖青）。機械作用破壞葉緣細胞，啟動多酚酶促氧化。動靜交替形成花果香型。通過紅邊 (红边, hóng biān) 出現而中心保持綠色來控制氧化程度。
* **殺青 / 杀青 — shāqīng：** 鍋炒或滾筒200–280°C高溫瞬間鈍化氧化酶，固定香氣特徵。
* **揉捻 / 揉捻 — róuniǎn：** 塑造特色外形（緊結彎曲條索）並破壞細胞壁，確保沖泡時充分浸出。
* **烘乾 / 烘干 — hōnggān：** 穩定含水量至儲存標準（< 6%）。
* **拼配 / 拼配 — pīnpèi：** 技師拼配不同產地、季節和品種原料，達到年度穩定的風味標準。
* **炭焙 / 炭焙 — tàn bèi：** 木炭文火烘焙，溫度不超過60°C。低溫慢烘「封存」香氣，降低澀味，賦予湯色深度和柔滑質感。此工序形成特有的橙黃琥珀色湯色和漳州烏龍茶「溫潤」風格。
* **分級 / 分级 — fēnjí：** 成品茶過篩、按大小分級並去除茶梗和茶末。

**6. 感官特徵：**

* **乾茶外觀：** 條索緊結彎曲 (条索紧结卷曲, tiáosuǒ jǐnjié juǎnqū)，烏褐帶黃 (乌褐带黄, wūhè dài huáng)。葉片大小均勻一致。高等級可見金毫 (金毫, jīn háo)。
* **乾茶香氣：** 清純，以花香為主 — 特級有蘭花香 (兰花, lánhuā)，一級轉為更廣泛的花甜香。背景調：烤杏仁、焦糖、淡奶香。
* **茶湯香氣：** 清香 (清香, qīngxiāng) 花甜伴隨果香層次 — 熟李、杏。焙火版本呈堅果焦糖基調配蜜香餘韻。香氣持久，蓋香明顯。
* **滋味：** 醇厚 (醇厚, chúnhòu)，中等濃度，甜鮮平衡 (鲜爽, xiānshuǎng)。柔和包覆性澀味快速轉為持久回甘 (回甘, huígān)。風味調性：花蜜、烤栗子、熟桃、淡礦物質感。耐泡6–8水，風味從花鮮轉向甜木質調。
* **湯色：** 金黃 (金黄, jīnhuáng)，透明清澈。重焙版本呈深琥珀色。高透明度為正確加工的標誌。
* **葉底：** 展開葉片烏褐帶綠 (乌褐带绿, wūhè dài lǜ)，柔軟有彈性，葉緣平整。葉片均勻度反映拼配和分級品質。

**7. 化學成分：**

* **多酚類：** 鮮葉含量不少於25%（原料標準）。成品茶中大部分兒茶素（主要是EGCG、EGC）部分氧化為茶黃素和茶紅素，形成特有的柔和、不澀口感。研究顯示多酚類抗氧化活性為維生素E的18倍。
* **氨基酸：** 鮮葉不少於4.0%。主要成分為L-茶氨酸 (L-茶氨酸, L-chá ānjīsuān)，負責柔和度、鮮甜味和保持頭腦清醒的放鬆效果。高氨基酸含量源於山區風土和散射光優勢。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēi jiǎn) 不少於2.0%，提供明顯提神效果。還含有微量可可鹼和茶鹼，補充興奮作用。
* **維生素：** 維生素C（抗壞血酸）少量；B族維生素（B₁、B₂、B₃）；維生素E（生育酚）。
* **礦物質：** 鉀 (K)、鎂 (Mg)、錳 (Mn)，以及微量元素硒 (Se) 和鋅 (Zn) — 後兩者為漳州土壤特色，為茶湯礦物質特徵做出貢獻。
* **精油：** 萜烯化合物（橙花醇、香葉醇、芳樟醇）形成花香；炭焙過程額外產生吡嗪和呋喃酮，負責堅果和焦糖調性。

**8. 保健功效：**

* **提神醒腦：** 咖啡鹼（≥ 2.0%）與L-茶氨酸協同作用，提供柔和持久的興奮效果，無咖啡的「神經質」刺激。提高注意力和認知表現。
* **抗氧化保護：** 多酚複合物（兒茶素、茶黃素）中和自由基，延緩氧化應激和細胞老化過程。
* **助消化：** 傳統焙火烏龍茶以刺激消化酶分泌、緩解油膩食物後的飽脹感而著稱。所含多酚有助腸道菌群正常化。
* **心血管系統：** 規律飲用烏龍茶與降低「壞」膽固醇（LDL）和維持血管彈性相關。
* **代謝支持：** 烏龍茶多酚和咖啡鹼刺激產熱和脂質代謝，在均衡飲食框架內可能有助體重控制。
* **口腔健康：** 茶葉氟化物和兒茶素具抗菌作用，抑制致齲菌群發展。
* **抗壓作用：** L-茶氨酸提高大腦α波水平，促進平靜專注狀態。工夫茶的多次慢泡本身就是正念冥想形式。

**9. 沖泡方法：**

* **水溫：** 95–100°C。為充分展現焙火茶的豐富滋味，建議接近沸騰的溫度。
* **茶量：** 工夫法 — 5克配75毫升（1:15比例）；西式法 — 3克配200–250毫升。
* **茶具：** 瓷蓋碗 (盖碗, gàiwǎn) — 萬能選擇，可評估香氣演變；宜興紫砂壺 (紫砂壶, zǐshā hú) — 適合焙火版本，紫砂「柔化」茶湯並突出深度。
* **流程：**
  1. 沸水溫壺溫杯 — 穩定沖泡溫度。
  2. 投茶入蓋碗，加蓋悶10–15秒，聞乾茶香。
  3. 潤茶 (润茶, rùn chá) — 沸水沖入5秒後倒出。此泡「喚醒」茶葉。
  4. 第一泡 — 15–20秒。茶湯已充分。
  5. 過濾分杯。
  6. 續泡：6–8次，每泡增加5–10秒。優質一枝春穩定耐泡6–8水。

* **冷泡：** 1:50比例（如5克配250毫升）。冷水沖泡，冰箱靜置6小時。此法突出清新花香，消除苦澀。
* **注意事項：** 不建議超過100°C的沸水沖泡 — 可能損害精緻的花果香。新茶購買後宜放置10–15天讓火氣 (火气, huǒqì) 散去。開封後7天內飲完以保持香氣新鮮。

**10. 儲存方法：**

* **基本條件：** 密封包裝，乾燥陰涼處（15–25°C），避免直射陽光，遠離異味。
* **茶葉大敵：** 潮濕、高溫、紫外線、異味（香料、香水、清潔劑）。
* **容器：** 密封鐵罐、鋁箔內層真空袋、矽膠密封陶瓷茶罐。
* **焙火版本特點：** 炭焙使含水量極低，儲存期可達1–2年而品質無明顯下降。部分愛好者實行陳化 (陈化, chénhuà)，認為焙火一枝春隨年份增長獲得高雅陳香 (陈香, chénxiāng)。
* **清香版本：** 低溫儲存（0–5°C），類似綠茶。保質期不超過6個月。

**11. 價格與仿冒：**

* **價格定位：** 一枝春歷史上定位中檔價位 — 日常品質茶，強調可負擔性和穩定性。參考價格：二級 — 最經濟實惠，性價比高；一級 — 主要商業產品；特級 — 約700元人民幣/斤（500克）起，具明顯蘭花香和單芽原料。影響價格因素：原料海拔、採摘季節（春茶貴於夏茶）、手工或機械加工、拼配師技藝和焙火程度。
* **防偽方法：**
  * 從漳州茶廠官方經銷商或專賣店購買（截至2024年，漳州、潮州、汕頭有29個銷售點）。
  * 評估葉片均勻度：正宗廠製一枝春分級均勻純淨 — 無茶梗、茶末和雜葉混入。
  * 檢查香氣：正品茶有純淨、非「化學」氣味。警惕不符合烏龍茶特徵的刺鼻香精味。
  * 評估茶湯：透明金黃或琥珀色，無混濁。滋味柔和，有明顯回甘，無不良苦澀或酸味。
  * 注意包裝：正品常保持傳統紙包裝和廠家標識。特級茶異常低價應引起懷疑。

**12. 趣聞軼事：**

* 漳州民諺「乘風飛馬大前門，流香色種一枝春」中，三個茶品牌與1960年代最頂級香煙品牌並列。當時茶葉是奢侈品而非日常消費品。
* 漳州茶廠是福建省唯一保存至今的國營烏龍茶廠。鼎盛期（1980年代）年產量超過500噸。
* 一枝春也生產餅茶 (饼茶, bǐngchá) 形式，這在閩南烏龍茶中不常見，更類似普洱茶和黑茶。緊壓形式便於陳化和「老茶」香氣發展。
* 茶廠手工包裝 — 每兩 (两, liǎng, ≈ 50克) 紙包 — 自1950年代保持至今。內層白竹紙 (白竹纸, bái zhú zhǐ) 吸收多餘水分並「記憶」茶香。
* 一枝春基礎的精製技術純師徒傳承。截至2020年代，傳承譜系已達四代，第五代師傅正在培養中。

**13. 與其他閩南烏龍茶比較：**

* **安溪鐵觀音 (安溪铁观音, Ānxī Tiěguānyīn)：** 最知名的閩南烏龍茶。與一枝春不同，鐵觀音使用同名栽培種而非拼配原料。現代「清香」鐵觀音 — 清淡、綠色，強調新鮮；一枝春經典風格 — 更多焙火，溫潤蜜甜特徵。鐵觀音是特定茶園製茶師的茶；一枝春是拼配師的茶。
* **漳平水仙 (漳平水仙, Zhāngpíng Shuǐxiān)：** 同樣使用水仙栽培種，但製成獨特的濾紙包裝方形緊壓茶。漳平水仙具明亮蘭花香、奶油質感和清新感。一枝春 — 拼配產品，炭焙工藝，更「成熟」特徵。
* **永春水仙 (永春水仙, Yǒngchūn Shuǐxiān)：** 另一閩南水仙類茶，歷史上閩南最早（1857年起）。永春版本 — 散茶、非拼配，蘭花香突出。一枝春經過工業化拼配和炭焙，穩定性和深度更佳，但「新鮮」香氣較弱。
* **白芽奇蘭 (白芽奇兰, Báiyá Qílán)：** 平和縣（漳州）特產。使用同名栽培種，具明亮「穿透性」蘭花香和礦物質感。相比之下，一枝春更圓潤柔和，無奇蘭的尖銳，但焙火層次更豐富。

**13a. 漳州一枝春品種與等級：**

根據DB35/T 943-2009標準，一枝春按感官指標分為三個主要等級：

* **特級 (特级, tèjí)：** 原料以單芽為主。條索緊實厚重，金毫明顯。香氣清純高銳，蘭香突出 (兰香高锐)。滋味濃厚，回甘深長。參考價格700元人民幣/斤起。
* **一級 (一级, yī jí)：** 原料為一芽一葉。條索整齊均勻。香氣清純持久。滋味醇厚 (醇厚, chúnhòu)。最主要的商業產品。
* **二級 (二级, èr jí)：** 條索稍粗。香氣純和 (纯和, chúnhé)。優秀性價比 — 可負擔的日常烏龍茶。

除散茶形式外，還生產**餅茶版本 (饼茶, bǐngchá)**，專為長期陳化和陳香 (陈香, chénxiāng) 發展而設。

**結語：**

漳州一枝春是一款有歷史的茶，這歷史無法與漳州茶廠本身、世代拼配師、炭火爐的低語和竹紙的沙沙聲分離。它不追求單一茶園烏龍茶的鮮明個性，也不與鐵觀音爭奪世界聲譽 — 它的力量在別處。這是穩定性和記憶之茶：每批次都努力精確重現祖輩所知的味道。對漳州、潮汕居民和東南亞華人社群而言，一口一枝春就是回到「古早味」，回到晨間工夫茶儀式，回到50毫升茶杯旁的悠閒談話。柔和、圓潤，兼具炭焙深度和花香細膩 — 這款茶最好的表現不在第一泡，而在第三、四泡，正是這種耐心等待給飲者以充分回報。
