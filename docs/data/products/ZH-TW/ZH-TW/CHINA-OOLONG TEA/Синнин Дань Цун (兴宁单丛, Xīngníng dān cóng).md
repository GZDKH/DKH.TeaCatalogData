# 興寧單叢 (兴宁单丛, Xīngníng dān cóng)

**興寧單叢 (兴宁单丛, Xīngníng dān cóng)**

![Синнин Дань Цун (兴宁单丛, Xīngníng dān cóng)-1](images/Синнин%20Дань%20Цун%20%28兴宁单丛,%20Xīngníng%20Dān%20Cóng%29-1.png)

**興寧單叢 (兴宁单丛, Xīngníng dān cóng)** — 廣東省梅州市 (梅州, Méizhōu) 興寧縣的廣東香型烏龍茶。此茶採用單叢工藝製作，以蜜蘭香 (蜜兰香, mìlán xiāng) 為特色香型，與經典鳳凰單叢相比，搖青較輕，荔枝木炭焙火工藝更為突出。自2021年起，該茶獲得國家地理標誌證明商標 (国家地理标志证明商标, guójiā dìlǐ biāozhì zhèngmíng shāngbiāo) 保護，2022年其製作技藝被列入非物質文化遺產名錄。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶，烏龍茶, wūlóng chá）。氧化程度為中等，約25–50%，後經炭焙處理。
* **類別：** 廣東香型烏龍茶，屬鳳凰單叢 (凤凰单丛, Fènghuáng Dān Cóng) 系列。有時歸類為「梅州單叢」— 基於從潮州引進茶苗發展而成的地方分支。
* **產地：** 中國廣東省 (广东省, Guǎngdōng shěng) 梅州市 (梅州市, Méizhōu shì) 興寧市 (兴宁市, Xīngníng shì)。保護產區涵蓋興寧全境，包括13個鄉鎮和街道委員會；核心產區集中在徑南鎮 (径南镇, Jìngnán zhèn) 和合水鎮 (合水镇, Héshuǐ zhèn)。
* **地理座標：** 約北緯24°09′，東經115°44′（興寧縣中心）；茶山分佈在城南和東南方向海拔300–800公尺的山地帶。

**2. 歷史與文化意義：**

* **歷史：** 興寧種茶歷史悠久。據明代正德年間 (正德, 1506–1521) 書法家祝枝山 (祝枝山) 手抄本《興寧縣志》(兴宁县志) 記載，當地居民早已從事茶葉種植。清代時期，龍北鎮 (龙北镇, Lóngběi zhèn) 的綠茶「官田綠茶」(官田绿茶) 躋身「嘉應州八大名茶」(嘉应州八大名茶) 之列，並出口至東南亞。20世紀茶業經歷轉型：1960–70年代建立了南蛇崗 (南蛇岗茶林场) 等國營茶場。轉折點出現在1985年，從潮州饒平縣 (饶平, Ráopíng) 引進單叢茶苗，當地農民開始從綠茶生產轉向烏龍茶。1990年代，徑南鎮的黃蜂窩茶場 (黄蜂窝茶场) 和寶山茶場 (宝山茶场) 成為新興產業的領頭羊。1997年在北京國際茶展上，興寧產品榮獲金獎，梅州被授予「中國著名單叢茶之鄉」(中国著名单丛茶之乡) 稱號。2021年「興寧單叢茶」品牌獲得國家地理標誌地位；2022年製作技藝被認定為非物質文化遺產；2023年該茶被列入全國名特優新農產品名錄 (全国名特优新农产品名录)。截至2025年，興寧茶園總面積超過4,000畝（≈ 2,670公頃），年產量超過6,000噸，約佔梅州茶產品總價值的75%。

* **名稱：** 興寧 (兴宁) — 地名源於東晉 (东晋, Dōng Jìn, 西元331年) 設縣時期；字面意思為「興盛安寧」。單叢 (单丛) — 技術術語，意指「單獨的樹」或「個別的叢」：歷史上最好的茶葉是從每棵優秀茶樹單獨採摘和加工，保持其獨特的香氣特徵。

* **文化意義：** 興寧是客家 (客家, Kèjiā) 城市，歷史上以「文化之鄉、足球之鄉、華僑之鄉」聞名。茶葉有機融入客家待客文化：以新泡單叢茶招待客人是迎賓儀式的重要組成部分。當地茶文化結合了潮州工夫茶傳統與客家的樸實自然。徑南鎮每年春季茶市成為全廣東茶愛好者的聚集地。浊水村 (浊水村) 興寧單叢茶文化館 (兴宁单丛茶文化馆) 的建立體現了保護和推廣當地茶文化遺產的努力。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種為白葉單叢 (白叶单丛, Bái Yè Dān Cóng，「白葉單叢」)，約佔種植面積的70%。這是中葉型灌木品種（*Camellia sinensis* var. *sinensis*），具有高抗寒性和濃郁的香氣特徵。另外還種植梅占 (梅占, Méi Zhàn) — 從福建引進的早熟高產品種，適合生產濃香型茶葉，以及黃金桂 (黄金桂, Huáng Jīn Guì)。繁殖主要採用傳統扦插法進行無性繁殖；徑南鎮黃蜂窩山保存有百年母樹。
* **採摘：** 四季採摘：春茶（清明 Qīngmíng — 穀雨 Gǔyǔ，3–4月）— 最珍貴的採摘期；夏茶（立夏 Lìxià 後，5–6月）；秋茶（白露 Báilù 前後，9月）；冬季「雪片茶」(雪片茶, xuěpiàn chá) — 小雪 Xiǎoxuě 前採摘（11月），以清冷香氣和耐泡性著稱。
* **採摘標準：** 春茶特級 — 純芽或一芽一葉；一級 — 一芽二葉；大宗茶 — 一芽三葉。採摘在晨露乾後進行。
* **原料要求：** 芽葉成熟度均勻，無機械損傷和異味。葉片完整性對後續搖青過程中形成特色香氣至關重要。

**4. 風土條件與栽培特點：**

* **地區與地形：** 興寧位於廣東省東北部，東江和韓江上游，受蓮花山脈 (莲花山脉) 和羅浮山脈 (罗浮山脉) 影響。地形為山地丘陵：南北部地勢較高（最高峰陽天嶂 陽天嶂，1,107公尺），中部為寧江盆地。茶園分佈在海拔300–800公尺高度帶，主要在南坡和東南坡。
* **海拔高度：** 海拔300–800公尺。最佳地段在500公尺以上，處於穩定雲霧帶。
* **氣候：** 南亞熱帶季風氣候，向中亞熱帶過渡。年平均氣溫20.4°C；年降雨量1,600–1,900毫米；霧日超過180天；日溫差超過10°C；散射光比例約70%。高濕度和豐富霧氣減緩芽葉生長，有利於香氣物質和胺基酸積累。
* **土壤：** 酸性紅黃壤 (红黄壤, hóng huáng rǎng)，pH值4.5–6.5，有機質含量高。特點是硒含量較高：0.018–0.066毫克/公斤，比中國烏龍茶平均水準高1.3倍。硒增強茶葉抗氧化潛力。山區無工業污染確保原料生態純淨。

**5. 製作工藝：**

興寧單叢工藝繼承潮州單叢基本原理，但具有地方特色，師傅們概括為「輕搖重炒」(轻摇重炒, qīng yáo zhòng chǎo)：搖青時間比經典鳳凰單叢短，保持香氣的清新明亮。荔枝木炭焙 (荔枝木炭焙) 起關鍵作用，形成特色「蜜韻」(蜜韵, mì yùn) — 此工序被認定為非物質文化傳統的核心。

* **採摘 / 采摘 — cǎizhāi：** 採摘「一芽二葉」標準的嫩梢，在晨露乾後進行。原料迅速送至加工廠，防止過熱和過早氧化。
* **曬青 / 晒青 — shàiqīng：** 將採摘的葉子攤放在竹篩上，日曬約30分鐘。葉片失去部分水分，變得柔軟，啟動初步生化轉化。
* **搖青 / 摇青 — yáoqīng (碰青 — pèngqīng)：** 4–5個搖青和靜置循環。機械作用破壞葉緣細胞壁，啟動多酚的部分氧化。正是此階段形成花果香氣譜。搖青時間有意縮短，相比鳳凰傳統，以保持清新「響亮」的香調。
* **殺青 / 杀青 — shāqīng：** 滾筒殺青，溫度約260°C。高溫處理停止酶促反應，固定香氣方向。殺青強度是興寧風格的標誌之一。
* **揉捻 / 揉捻 — róuniǎn：** 機械揉捻結合手工整形。揉捻破壞細胞結構，提高浸出性，賦予茶葉特色外形 — 緊結厚實的條索。
* **初焙 / 初焙 — chū bèi：** 荔枝木炭焙火，溫度約90°C。正是荔枝炭 (荔枝木炭, lìzhī mù tàn) 的選擇被認為是興寧單叢「蜜甜」的秘訣。
* **復焙 / 复焙 — fù bèi：** 最終乾燥，溫度60°C，至含水量≤5%。穩定香氣，確保儲存性。

**6. 感官特徵：**

* **乾茶外形：** 條索緊結壯實 (条索紧结壮实, tiáosuǒ jǐnjié zhuàngshí)，烏褐油潤 (乌褐油润)。茶葉均勻，規格一致；也有顆粒形（CTC風格）供大眾市場。
* **乾茶香氣：** 濃郁的蜜蘭香 (蜜兰香, mìlán xiāng) — 茶葉的招牌特徵。蓋碗加熱後展現層次：首先是純淨花香甜味，然後是桃子和杏子韻味，淡淡蜂蜜氣息。
* **茶湯香氣：** 高揚、持久、多層次。以蜜蘭香主題為主，展現芝蘭香 (芝兰香, zhīlán xiāng — 蘭花香氣)，水蜜桃 (水蜜桃, shuǐ mì táo) 果香韻味。焙火茶款呈現溫暖的堅果和焦糖氣息。陳放後顯現蜜甜。
* **滋味：** 濃醇 (浓醇, nóng chún)，鮮爽 (鲜爽, xiān shuǎng) 明顯，由高胺基酸含量提供。茶體飽滿但不厚重。澀味柔和，多酚平衡和諧。回甘 (回甘, huígān) 強烈持久，具有特色「高山韻清涼感」(高山韵清凉感)。優質茶款在數分鐘內給人「甘從喉韻來」的感受。
* **湯色：** 春茶呈金黃瑩亮 (金黄莹亮, jīnhuáng yíngliang) 且晶瑩透澈；秋茶稍溫暖，橙黃色調。
* **葉底（沖泡後茶葉）：** 葉片完整、肥厚、柔軟有彈性。優質單叢的特徵標誌 — 「綠腹紅邊」(绿腹红边, lǜ fù hóng biān)：葉片中心保持橄欖綠色，經氧化的葉緣呈現紅銅色澤。

**7. 化學成分：**

* **多酚類：** 白葉單叢栽培種春季鮮葉含量≥28.3%（乾重計）。半發酵過程中部分兒茶素氧化為茶黃素和茶紅素，形成湯色和茶體。殘留兒茶素與氧化產物的平衡決定了柔和澀味而無粗糙苦味。
* **胺基酸：** 春季原料總含量≥2.8%。L-茶胺酸佔主要部分，負責甜味、「鮮味」般的柔和感和放鬆效果。
* **生物鹼：** 咖啡因約3.8%（高於烏龍茶平均水準），提供明顯提神效果。可可鹼和茶鹼含量微量，補充溫和刺激作用。
* **硒：** 0.018–0.066毫克/公斤 — 興寧風土的獨特特徵。據當地研究，此指標比中國烏龍茶平均水準高約1.3倍。硒是抗氧化酶穀胱甘肽過氧化物酶的關鍵輔因子。
* **維生素：** 維生素C（抗壞血酸）、B群維生素（B₁、B₂）、維生素E（生育酚）、維生素P（芸香苷）。烏龍茶維生素C含量低於綠茶，因氧化過程中部分破壞。
* **礦物質：** 鉀、鎂、錳、鋅、磷。山地酸性土壤提供豐富礦物質譜，體現在回味結構中。
* **精油：** 豐富複雜的萜烯類化合物，包括芳樟醇、香葉醇、橙花醇、吲哚、茉莉酮和水楊酸甲酯。正是萜烯譜負責特色「高香」蜜蘭香氣，區別單叢與其他烏龍茶。荔枝炭焙火增加焦糖和呋喃類化合物。

**8. 保健功效：**

* **提神效果：** 高咖啡因含量（3.8%）結合L-茶胺酸提供溫和而持久的中樞神經系統刺激 — 清醒而無「緊張」感。
* **抗氧化保護：** 多酚和天然硒共同中和自由基。據中國研究人員評估，興寧單叢的自由基清除效率比標準烏龍茶高25% — 主要歸功於硒。
* **消化支持：** 半發酵性質使烏龍茶比綠茶對胃更溫和。兒茶素有助抑制過量脂肪吸收，傳統上在食用油膩食物時備受推崇。
* **心血管系統：** 規律飲用烏龍茶與維持正常膽固醇水準和改善血管彈性相關 — 效果歸因於多酚和茶黃素作用。
* **認知功能：** 咖啡因和L-茶胺酸的協同作用改善注意力集中和工作記憶 — 茶生化學中記錄最完整的協同效應之一。
* **皮膚狀態：** 抗氧化劑（多酚、硒、維生素E）可能延緩光老化過程。
* **代謝支持：** 烏龍茶傳統上被視為有助體重控制的茶類 — 通過刺激產熱和脂肪氧化。
* **正念茶飲：** 工夫茶多次沖泡過程創造停頓，有助減少心理情緒緊張。

**9. 沖泡方法：**

* **水溫：** 95–100°C。建議用沸水充分釋放單叢豐富香氣；較清淡的春茶可降至92–95°C。
* **茶量：** 8克配110毫升（經典工夫比例1:20重量比）。
* **茶具：** 白瓷蓋碗 (盖碗, gàiwǎn) — 最佳選擇：瓷器不「奪香」，可精確控制浸泡。較重焙火茶款可用宜興紫砂壺 (紫砂壶, zǐshā hú)，突出深度和圓潤口感。
* **沖泡過程：**
  1. 用沸水溫熱蓋碗和茶杯。
  2. 將8克茶葉放入溫熱的蓋碗。
  3. 洗茶：注入沸水3秒後立即倒出 — 「溫潤泡」(温润泡, wēnrùn pào)。
  4. 第一泡：3秒後倒出。
  5. 分茶入杯，聞蓋香 — 這是品鑑單叢的關鍵時刻之一。
  6. 後續沖泡：每次增加5秒。優質興寧單叢可沖泡8–12次，頂級茶款可達15次。

* **註：** 高沖 (高冲, gāo chōng) — 從10–15公分高度注水 — 有助「激活」香氣，特別是前三四泡。

**10. 儲存：**

* **最佳條件：** 密封包裝，陰涼乾燥處，避光保存。為最大程度保持蜜蘭香氣，建議冷藏（0–5°C）真空或鋁箔包裝。開封後6個月內飲用。
* **焙火茶款：** 經明顯炭焙處理的茶可室溫保存（乾燥處），甚至隨時間發展 — 1–2年後蜜香加深，香氣更沉穩醇和。
* **茶葉天敵：** 濕氣（最危險因素 — 興寧茶吸濕性強）、異味、直射陽光、高溫。勿與香料、咖啡或香水同處保存。
* **容器：** 密封鐵罐、鋁箔拉鍊袋或真空包裝。陶瓷茶罐需有矽膠密封圈。

**11. 價格與假冒：**

* **價格區間：** 興寧單叢市場按季節、海拔、手工程度和品牌分級。中國內銷市場參考價：特級 — 600元/斤（500克）起；一級 — 200–400元/斤（主要商業段）；大宗茶 — 80–150元/斤。黃蜂窩山高山春茶或合水百年老樹茶在拍賣會上可達更高價格。
* **避免假冒方法：**
  * 向有透明來源和「兴宁單丛茶」地理標誌證明文件的商家購買。
  * 評估乾茶均勻度：正宗興寧單叢為緊結、規格一致、油潤有光澤的條索，無碎末和茶粉。
  * 檢查香氣：天然蜜蘭香純淨持久；「化學」香精味、刺鼻花香或焦糖味是加香或過度焙火的跡象。
  * 評估湯色：金黃琥珀色，晶瑩透澈；渾濁或深褐色湯色表明加工缺陷。
  * 價格過低需警惕：如「高山春季單叢」售價低於150元/斤，很可能是平地或夏茶冒充。

**12. 趣味知識：**

* **「單叢之鄉」在鳳凰山外。** 大眾認知中單叢專指潮州鳳凰山 (凤凰山)。然而興寧是鳳凰山外首個且唯一獲得「單叢茶之鄉」(单丛茶之乡) 稱號的地區（1997年）。這證明了獨立地方風格的認可。
* **荔枝炭作為風味秘訣。** 荔枝木炭焙 (荔枝木炭焙) 不僅是技術，更是文化認同元素：荔枝是廣東標誌性果樹之一，使用其木材象徵性地將茶與土地「連結」。
* **硒風土。** 興寧山地土壤含天然硒，這在茶葉中較為罕見。當地政府將「富硒單叢」(富硒单丛) 定位為具強化抗氧化特性的功能產品。
* **雪片 — 冬季珍品。** 「雪片茶」(雪片茶) — 小雪 (小雪) 前的冬季採摘 — 以獨特「清冷」香氣和耐泡性著稱。此風格在廣東以外幾乎不見。
* **五大香型。** 興寧單叢分類中有五大主要「香型」(香型, xiāngxíng)：黃枝香 (黄枝香, 梔子花香)、芝蘭香 (芝兰香, 蘭花香)、蜜蘭香 (蜜兰香, 蜜蘭香)、玉桂香 (玉桂香, 肉桂香) 和通天香 (通天香,「通天」薑花香) — 最後一種被認為最稀有昂貴。

**13. 與其他廣東單叢比較：**

* **鳳凰單叢 (凤凰单丛, Fènghuáng Dān Cóng)：** 鳳凰山經典潮州單叢。相比興寧單叢，通常搖青更重，香氣更深沉複雜，結構更明顯。興寧單叢更清淡清新，蜜甜更明顯，澀味較輕。
* **嶺頭單叢 (岭头单丛, Lǐngtóu Dān Cóng)：** 潮州饒平縣 (饶平) 單叢，有時稱「白葉單叢」— 實際上與興寧種植的是同一栽培種。但風土差異（饒平海拔較低較溫暖）和工藝形成不同特徵：嶺頭更圓潤，興寧更有礦物感和「清涼」感。
* **芝蘭香單叢 (芝兰香单丛)：** 鳳凰單叢的特定香型子類，具蘭花香特徵。在興寧系列中芝蘭香只是五種香型之一，且當地版本通常比潮州版更溫和甜美，因搖青較輕。
* **梅州奇蘭 (梅州奇兰, Méizhōu Qí Lán)：** 同樣來自梅州地區的烏龍茶，但用從福建引進的奇蘭 (奇兰) 栽培種製作。香氣更偏香料和「福建風格」（肉桂、丁香韻味），不同於單叢的蜜果香調。
* **鳳凰水仙 (凤凰水仙, Fènghuáng Shuǐxiān)：** 大宗廣東烏龍茶，用未分類水仙叢製作，無個別加工處理。香氣複雜度遠不及單叢，但價格更親民。興寧單叢佔據中間位置：比水仙複雜，比頂級鳳凰品種親民。

**結語：**

興寧單叢是茶葉「遷移」並獲得新面貌的生動例子。1985年從潮州引進的茶苗，在含天然硒的山地酸性土壤上，經客家師傅之手，四十年來形成了獨立風格：清新、蜜甜，帶礦物質清涼回味和荔枝炭的溫暖韻味。此茶是認識廣東單叢世界的理想選擇：比頂級鳳凰茶款更親民，但已具備特色香氣複雜性，使每次沖泡都成為發現。興寧單叢在悠閒工夫茶品飲中最能展現 — 當有時間觀察第三四泡時從茶杯升起的那股蜜蘭香波，正是為此而製作這款茶的原因。
