# 漳州流香 (漳州流香, Zhāngzhōu liú xiāng)

**漳州流香 (漳州流香, Zhāngzhōu liú xiāng)**
![Чжанчжоу Лю Сян (漳州流香, Zhāngzhōu liú xiāng)](images/Чжанчжоу%20Лю%20Сян%20%28漳州流香,%20Zhāngzhōu%20Liú%20Xiāng%29.png)

![Чжанчжоу Лю Сян (漳州流香, Zhāngzhōu liú xiāng)-1](images/Чжанчжоу%20Лю%20Сян%20%28漳州流香,%20Zhāngzhōu%20Liú%20Xiāng%29-1.png)

漳州流香是漳州國營茶廠三大經典茶品之一，與色種 (色种, Sèzhǒng) 和一枝春 (一枝春, Yī Zhī Chūn) 並列。這是現代中國罕見的「拼配」烏龍茶典範：來自福建不同地區的原料經過統一的精製工序 (精制, jīngzhì) ——精製、拼配和炭焙，最終誕生出具有獨特「火香」風格和深厚濃郁滋味的茶品。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶，烏龍茶, wūlóngchá）。風格——重發酵 (重发酵, zhòng fājiào)、重搖青 (重摇, zhòng yáo)、炭焙 (炭焙, tànbèi)。發酵程度約為40–60%。
* **類別：** 按精製地點歸類為閩南烏龍 (闽南乌龙, Mǐnnán wūlóng)。然而主要原料來自閩北 (闽北, Mǐnběi)，這使得流香成為連接福建南北烏龍茶傳統的獨特「橋樑」。屬於「漳州傳統茶」類別。
* **產地：** 中國 (中国, Zhōngguó) 福建省 (福建省, Fújiàn shěng)。精製和拼配——漳州市 (漳州市, Zhāngzhōu shì) 芗城區 (芗城区, Xiāngchéng qū) 小坑頭生產基地 (小坑头, Xiǎokēngtóu)。原料產區——武夷山 (武夷山, Wǔyíshān)、安溪 (安溪, Ānxī) 及福建南北其他烏龍茶產區。
* **地理座標：** 漳州：約北緯24°31′，東經117°39′。原料產區（武夷山核心區）：約北緯27°43′，東經117°57′。

**2. 歷史與文化意義：**

* **歷史：**

漳州精製烏龍茶的傳統可追溯至清代。同治年間（同治, 1862–1874），企業家林心傅 (林心傅) 在漳州創辦「林奇苑」(林奇苑) 茶莊，開始從武夷山採購毛茶，然後在漳州進行精製 (精制, jīngzhì) ——拼配、揀剔、復焙、勻堆和包裝。「林奇苑」以「三印水仙」(三印水仙) 品牌生產的產品在東南亞成為福建烏龍茶的象徵。同時期還有張源美 (张源美)、源崇美 (源崇美)、奇苑 (奇苑) 等茶莊，各有其拼配秘方。

1956年，在私營企業國有化過程中，所有主要漳州茶莊合併為國營漳州茶廠 (漳州茶厂, Zhāngzhōu cháchǎng，成立於1953年)。各家茶莊的拼配技藝融為一體，1957年茶廠正式創製「流香」(流香) 產品。1960年代，流香、色種 (色种) 和一枝春 (一枝春) 成為茶廠三大旗艦茶品。當時民間流傳「流香色種一枝春，乘風飛馬大前門」(流香色种一枝春，乘风飞马大前门)，將這些茶品與最知名的香煙品牌並列，突顯其奢侈品地位。

在鼎盛時期的1970–1990年代，漳州烏龍茶出口量達到福建烏龍茶總出口量的40%。茶廠是中國三家擁有自主出口代碼 (唛号, màhào) 的企業之一，其S2系列產品在日本和東南亞廣為人知。1996年，流香在中國科技精品博覽會 (中国科技精品博览会) 上獲得金獎。2019年，漳州傳統烏龍茶精製技藝入選福建省第六批省級非物質文化遺產名錄 (福建省非物质文化遗产)。2022年，流香入選中國茶葉譜系代表品 (中国茶叶谱系代表品) 名錄。2024年，為慶祝茶廠成立70週年，推出紀念版——特級流香 (特级流香)，採用手工紙包裝，延續半世紀傳統。

* **名稱：**

「漳州」(漳州) ——精製地；「流」(流) ——「流動、流淌」；「香」(香) ——「香氣」。字面意思：「來自漳州的流淌香氣」。名稱借用武夷山岩區著名的流香澗 (流香涧, Liúxiāng Jiàn)，這是歷史上重要原料的來源地。這是武夷山茶葉風土的經典「靈氣之地」之一，位於正岩區 (正岩区, zhèngyán qū) 核心地帶。

* **文化意義：**

流香是漳州茶文化和閩南、潮汕工夫茶 (工夫茶, gōngfu chá) 文化的象徵。漳州是最早有文獻記載的工夫茶中心之一：彭光斗《閩瑣記》(闽琐记, Mǐn Suǒ Jì) 記載於1766年（乾隆31年），描述了漳州使用武夷茶的儀式性茶飲。對於該地區世代居民以及馬來西亞、新加坡和印尼的海外華人社群而言，流香一直是「故鄉的味道」——鄉愁和文化認同的物質體現。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種——福建水仙 (福建水仙, Fújiàn Shuǐxiān)，又稱水吉水仙 (水吉水仙) 或武夷水仙 (武夷水仙)。這是半喬木型 (小乔木型, xiǎo qiáomù xíng) 大葉類 (大叶类, dàyè lèi) 晚生種，為天然三倍體。入選中國國家級茶樹良種 (国家级茶树良种) 名錄，註冊號GS13009-1985，「華茶9號」(华茶9号)。學名：*Camellia sinensis* cv. Fujian-shuixian。輔助栽培種——肉桂 (肉桂, Ròuguì) 和閩北菜茶 (菜茶, càichá) 群體種。
* **採摘：** 春茶 (春茶, chūnchá) 為主，能提供最高的氨基酸濃度和香氣深度。春季原料主要用於高等級產品。
* **採摘標準：** 一芽二至三葉 (一芽二至三叶, yī yá èr zhì sān yè)，成熟，無機械損傷。
* **原料要求：** 鮮葉茶多酚含量不少於25%，氨基酸不少於4.0%（按茶廠規格）。葉片應完整、成熟均勻，無異味。

**4. 風土與栽培特點：**

* **產區與地形：** 主要原料來自武夷山 (武夷山, Wǔyíshān) 高山岩石區——著名的丹霞地貌 (丹霞地貌, Dānxiá dìmào) 山體，具有陡峭峽谷、岩壁和狹窄山谷，為茶樹創造獨特的微氣候。補充原料來自福建南部烏龍茶產區（安溪等）。
* **海拔高度：** 海拔600–800公尺（武夷山核心原料區）。
* **氣候：** 濕潤亞熱帶氣候。採摘區年平均氣溫16–20°C，相對濕度≥75%，年霧日超過200天。晝夜溫差顯著。豐富的散射光 (漫射光, mànshèguāng) 有利於葉片中氨基酸和香氣前體物質的積累。
* **土壤：** 丹霞砂岩-礫岩基底上的酸性紅壤 (红壤, hóng rǎng)，pH值4.5–6.0。富含礦物質和有機質。核心岩區的茶園屬於水源保護區，禁用化學肥料和農藥，實行生態種植。

**5. 製作工藝：**

流香是雙重工藝鏈的產物：原料首先在產地按傳統烏龍茶工藝進行初製 (初制, chūzhì)，然後在漳州茶廠進行精製 (精制, jīngzhì)。精製階段被認定為非物質遺產項目，是形成流香特色的關鍵環節。

**初製 (初制, chūzhì)：**

* **採摘 / 采摘 — cǎizhāi：** 採摘「一芽+二三葉」標準的頂梢。採摘的原料迅速送至加工車間。
* **曬青 / 晒青 — shàiqīng：** 比輕發酵烏龍茶更重 (较重, jiào zhòng)。葉片失去大量水分，細胞壁軟化，開始形成香氣基礎。
* **晾青 / 晾青 — liàngqīng：** 葉片在陰涼處「休息」，水分從葉柄向葉緣重新分佈。
* **做青（搖青）/ 做青 — zuòqīng (摇青 — yáoqīng)：** 搖青和靜置的交替循環。重搖重發酵 (重摇重发酵, zhòng yáo zhòng fājiào) 啟動葉緣的強烈氧化，形成花果香型和特徵性的「紅邊」(红边, hóng biān)。
* **殺青 / 杀青 — shāqīng：** 熱處理停止酶促反應，固定達到的氧化程度。
* **揉捻 / 揉捻 — róuniǎn：** 塑造外形，破壞細胞結構以改善沖泡時的浸出。
* **初焙 / 烘焙 — hōngbèi：** 穩定毛茶 (毛茶, máochá) 的含水量。

**精製 (精制, jīngzhì) ——非物質遺產：**

* **人工拼樣 / 人工拼樣 — réngōng pīnyàng：** 師傅手工製作控制樣品，確定批次風格。
* **手工篩分 / 手工筛分 — shǒugōng shāifēn：** 按大小和形狀分離茶葉。
* **揀剔 / 拣剔 — jiǎntī：** 去除茶梗、葉脈和缺陷片段。
* **官堆（拼配）/ 官堆 — guānduī：** 關鍵步驟。師傅將不同山頭、季節、品種和加工程度的原料混合成統一和諧的拼配。拼配配方在師傅與學徒間傳承五代。
* **炭焙 / 炭焙 — tànbèi：** 在不超過60°C的溫度下用木炭進行多次緩慢焙火。正是這一步驟形成了標誌性的「火香」(火香, huǒxiāng) ——流香的名片。
* **勻堆 / 匀堆 — yúnduī：** 最終混合以確保批次均勻性。
* **包裝 / 包装 — bāozhuāng：** 傳統上採用手工包裝，內層白色竹紙，外層印有品牌圖案的牛皮紙。紅色圖案——「紅流香」(红流香, Hóng Liú Xiāng)，等級較高；深色圖案——「黑流香」(黑流香, Hēi Liú Xiāng)。

**6. 感官特徵：**

* **乾茶外形：** 條索緊結捲曲 (条索紧结卷曲, tiáosuǒ jǐnjié juǎnqū)，相當沉重飽滿。色澤烏褐帶黃似香蕉色 (乌褐带黄似香蕉色)。表面油潤，有適度光澤。高等級產品可見金毫 (金毫, jīnháo)。
* **乾茶香氣：** 深沉濃郁的「火香」(火香, huǒxiāng)，帶有木炭和焦糖香調。陳年版本有「陳香」(陈香, chénxiāng) 韻味。背景中有細膩的蘭花香 (兰香, lánxiāng)，特別是特製版本。
* **茶湯香氣：** 強勁、溫暖、多層次。主調為焦糖化、堅果、烘烤香調；展開後顯現果香花香泛音。香氣持久悠長，在空杯壁上留香。
* **滋味：** 醇厚 (醇厚, chúnhòu)，濃度與柔和度平衡。口中有包覆性甜味，類似蔗糖甜感 (蔗糖甜感)，轉為持久的回甘 (回甘, huígān)。有清新感 (鲜爽, xiānshuǎng)，源於高氨基酸含量。澀味極少，無苦味。滋味平衡：「濃淡適中」(浓淡适中)。
* **湯色：** 橙黃 (橙黄, chénghuáng) 至紅橙 (红橙, hóngchéng)，透明明亮，有良好深度。
* **葉底（沖泡後茶葉）：** 棕褐肥壯 (棕褐肥壮)，有彈性，紅邊明顯 (红边显, hóng biān xiǎn)，證明重度氧化。中脈清晰。

**7. 化學成分：**

流香的化學特徵主要由水仙栽培種的高浸出物含量和強化加工（重氧化+多次炭焙）決定。

* **多酚類：** 鮮葉含量≥25%（按茶廠規格）。成品茶中部分兒茶素在氧化和熱處理過程中轉化為茶黃素、茶紅素和更複雜的聚合物。根據研究，福建水仙栽培種（福安，福建樣品）鮮葉一芽二葉多酚含量約17.6%，低於茶廠標準，這是由於樣品採集海拔較低；武夷山高山原料通常顯示更高數值。
* **氨基酸：** 鮮葉≥4.0%（茶廠規格）。L-茶氨酸為主要成分（占總氨基酸含量50%以上）。正是它提供鮮爽感 (鲜爽) 和柔和甜味，並調節咖啡因作用，提供平靜的提神效果。
* **生物鹼：** 咖啡因——鮮葉約4.0%（福安水仙數據）。焙火後成品茶含量略有下降。還含有微量可可鹼和茶鹼。
* **水溶性浸出物：** 水仙栽培種高達50.5%——極高指標，解釋了茶湯的濃度和飽滿度。
* **維生素：** B族維生素（B₁、B₂）、維生素C（焙火時大部分被破壞）、維生素E、維生素K。
* **礦物質：** 鉀、鎂、錳、氟、鋅、微量硒。武夷山丹霞紅土為茶葉提供額外微量元素。
* **精油和香氣化合物：** 形成複雜香氣特徵。炭焙促進美拉德反應和焦糖化，產生呋喃、吡嗪和吡咯化合物，負責「火香」。保留的萜烯醇（芳樟醇、橙花醇、香葉醇）提供花香背景。

**特點：** 水仙的多酚與氨基酸比值 (酚氨比, fēn'ān bǐ) 約5.33，被認為是製作平衡澀味與甜味烏龍茶的最佳比例。

**8. 保健功效：**

* **溫和持續的提神效果：** 咖啡因與L-茶氨酸的結合提供無劇烈起伏的清醒感。L-茶氨酸刺激大腦α波產生，促進專注平靜。
* **抗氧化活性：** 茶多酚（兒茶素及其氧化產物——茶黃素和茶紅素）有效中和自由基，幫助保護細胞免受氧化壓力。
* **支持脂質代謝：** 兒茶素及其轉化產物促進脂肪代謝加速。據資料顯示，重發酵烏龍茶的這種效果比輕發酵烏龍茶更明顯。
* **助消化：** 焙火烏龍茶傳統上被認為是「溫性」茶，對腸胃蠕動有益，溫和刺激消化。特別推薦餐後飲用，尤其是油膩食物後。
* **利尿作用：** 咖啡因和礦物成分促進溫和的利尿效果，幫助排出多餘水分。
* **支持心血管系統：** 定期適量飲用多酚茶與降低「壞」膽固醇（LDL）水平和維持血管彈性相關。
* **強化牙齒琺瑯質：** 茶葉中的氟有助於琺瑯質再礦化；多酚抑制致齲細菌生長。
* **正念茶飲與減壓：** 流香的工夫茶沖泡儀式，通過多次沖泡和觀察香氣展開，是有助於放鬆和情緒舒緩的實踐。

**9. 沖泡方法：**

* **水溫：** 95–100°C。熱水對於充分展現焙火烏龍茶的「火香」和濃厚茶體是必需的。不建議使用超過100°C的沸水——這可能破壞炭焙的細膩泛音。
* **茶量：** 工夫茶法5–8克/100毫升，或歐式沖泡法3–4克/200–250毫升。
* **茶具：** 瓷質蓋碗 (盖碗, gàiwǎn) 或宜興紫砂壺 (紫砂壶, zǐshā hú)。紫砂壺特別推薦用於陳年 (陈年, chénnián) 版本流香——多孔結構增強「陳香」表現，為茶湯增添絲滑感。
* **沖泡過程：**
  1. 用沸水溫熱茶具並倒掉。
  2. 將茶葉放入蓋碗或茶壺。
  3. 快速洗茶 (润茶, rùnchá)：注水3–5秒後倒掉。這能「喚醒」乾茶並去除表面塵埃。
  4. 第一泡：注入95–100°C水，浸泡15–20秒。倒入公道杯 (公道杯, gōngdào bēi)。
  5. 分杯品飲。
  6. 續泡：可沖泡6–10次。每次增加5–10秒浸泡時間。高等級產品可沖泡8–10次。

* **注意：** 新焙火茶建議在陰涼處存放約15天後飲用，以消散過重的火氣 (火气, huǒqì)。開封後建議7天內飲完，以免失去香氣。不建議空腹飲用——單寧可能刺激胃黏膜；最佳時間是餐後一小時。

**10. 儲存：**

* **條件：** 密封包裝，乾燥陰涼處，遠離熱源和異味。理想儲存溫度15–25°C，濕度不超過60%。
* **容器：** 密封鐵罐、真空袋或茶廠原裝紙包裝。陶罐 (陶罐, táoguàn) 也適合。
* **茶葉天敵：** 潮濕、高溫、異味、直射光、氧氣。
* **保質期：** 焙火烏龍茶是最穩定的茶類之一。正確儲存下流香可保持品質2–3年，陳年版本 (陈年, chénnián) 在定期復焙條件下隨年份增加「陳香」韻味。

**11. 價格與仿冒：**

* **價格區間：** 取決於原料等級、採摘季節、拼配技藝和年份。參考價格（茶廠數據）：特級 (特级, 高等級) ——700元/斤（500克）起；一級 (一级) ——中等價位；二級 (二级) ——160–240元/斤，性價比最佳。紀念版和限量版價格可能顯著更高。
* **避免仿冒方法：**
  * 從漳州茶廠官方經銷商或專賣店購買（漳州、潮州、汕頭共29個點）。
  * 檢查原裝包裝：傳統牛皮紙配品牌紅色或深色圖案，內層白色竹紙。品牌「霞漳」(霞漳, Xiázhāng) 或「芝山」(芝山, Zhīshān)。
  * 評估乾茶香氣：真正的流香有純淨深沉的「火香」，無刺鼻化學味。
  * 檢查茶湯：顏色應為橙黃色，透明；滋味濃厚，有持久回甘，無「空洞感」。
  * 過低價格配合聲稱的高等級應引起警惕。

**12. 趣味知識：**

* 漳州茶廠是福建四大國營烏龍茶廠中唯一存續至今的。其餘三家在市場化改革中關閉。
* 漳州烏龍茶精製技藝是中國唯一以「精製」(精制) 而非初製 (初制) 階段為非物質遺產對象的案例。這突顯了拼配和焙火技藝的獨特性。
* 流香是「橋樑茶」：其配方有意結合閩北（濃厚、深沉）和閩南（花香、柔和）烏龍茶優點於一杯，使其在各地區「純正」烏龍茶中獨樹一幟。
* 茶廠至今仍採用手工包裝：女工將每份（一兩——約50克）用白色竹紙和牛皮紙包裝，約30秒完成，保持半世紀不變的方法。
* 流香和其他漳州烏龍茶的出口沿海上絲綢之路 (海上丝绸之路, Hǎishang Sīchóu zhī Lù)，這種茶的味道成為東南亞數百萬華僑的文化標誌。

**13. 與其他烏龍茶比較：**

* **武夷水仙 (武夷水仙, Wǔyí Shuǐxiān)：** 同一栽培種的單一成分岩茶，但無拼配和閩南添加。通常有更明顯的「岩韻」(岩韵, yányùn) 和礦物感。流香因拼配而有更柔和「圓潤」的特徵。
* **武夷肉桂 (武夷肉桂, Wǔyí Ròuguì)：** 明亮、辛香，有特徵性「桂皮」香和更明顯的辛辣感。流香更平靜、深沉、甘甜。
* **鐵觀音傳統 (铁观音传统, Tiěguānyīn chuántǒng)：** 安溪傳統重焙火鐵觀音——風格上最接近的閩南類比。但鐵觀音用同名栽培種製作，有更多花蜜特徵，而流香基於水仙，茶體更濃厚。
* **漳平水仙 (漳平水仙, Zhāngpíng Shuǐxiān)：** 同一栽培種的閩南烏龍茶，製成壓餅形式。風格更輕，花香奶香特徵。流香顯著更「火香」和濃縮。
* **鳳凰單叢 (凤凰单丛, Fènghuáng Dāncóng)：** 廣東烏龍茶，有明顯的花果單一香型。單叢是「單株茶」，而流香是「拼配茶」；兩者理念截然相反，但都在潮汕工夫茶文化中受到重視。

**結語：**

漳州流香是百年漳州茶藝傳統的活化石：從清代商號到唯一存續的國營烏龍茶廠。其特殊價值在於拼配藝術：掌握「官堆」秘訣的第五代傳人，將武夷山岩茶的山野力量與閩南烏龍茶的柔和在一杯中融合，通過多次炭焙直至誕生標誌性的「火香」。

這款茶適合珍視深度與溫暖的人：濃厚、溫暖，有持久甘甜回味和在杯壁上久久不散的香氣。流香不僅是飲品，更是文化記憶的片段，連接漳州與武夷山，福建與東南亞，以及一代代茶藝師傅。
