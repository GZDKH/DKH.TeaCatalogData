# 玉蘭香單叢 (玉兰香单丛, yùlán xiāng dāncóng)

**玉蘭香單叢 (玉兰香单丛, yùlán xiāng dāncóng)**

**玉蘭香單叢 (玉兰香单丛, yùlán xiāng dāncóng)** — 鳳凰單叢十大經典花蜜香型之一，有別於其他「兄弟茶」的強勁與濃烈，以精緻優雅見長。黃梔香以梔子花的辛香取勝，蜜蘭香以濃郁蜂蜜包裹，玉蘭香則另闢蹊徑 — 以輕盈、清涼的玉蘭花香從茶杯中緩緩升起，一泡接一泡地持續綻放。這款茶常被稱為單叢中的「貴族」：其茶湯不苦不澀，如室內樂般輕柔，與其他高香型品種的管弦樂般轟鳴形成鮮明對比。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶，烏龍茶, wūlóngchá）。廣東烏龍（廣東烏龍, Guǎngdōng wūlóng）。發酵程度 — 中等，通常為30–50%。
* **類別：** 鳳凰單叢（鳳凰單丛, Fènghuáng Dāncóng），十大經典花蜜香型之一（十大花蜜香型, shí dà huā mì xiāng xíng）。地理標誌保護產品（2010年 — AQSIQ；2020年 — 納入歐盟-中國保護名錄）。
* **產地：** 中國廣東省（廣東省, Guǎngdōng shěng）潮州市（潮州市, Cháozhōu shì）潮安區（潮安區, Cháo'ān qū）鳳凰鎮（鳳凰鎮, Fènghuáng zhèn），鳳凰山（鳳凰山, Fènghuáng Shān）山脈。最優質的樣品產自烏崬山（烏岽山, Wūdǒng Shān）及周邊高山村落。
* **地理座標：** 約北緯23°55′，東經116°38′（烏崬山區域，鳳凰山頂峰海拔1,497.8公尺）。

**2. 歷史與文化意義：**

* **歷史：** 玉蘭香是鳳凰單叢十大經典香型之一，其分類於1996年由華南農業大學（華南農業大学）戴素賢教授（戴素贤, Dài Sùxián）的研究團隊科學確立。為期三年的研究識別出不同單叢中的104種芳香化合物，確定十大香型各具獨特的揮發性物質特徵。

然而，孕育玉蘭香品系的茶樹歷史更為悠久。母樹最初於1961年通過扦插繁殖。後來鳳凰鎮茶農魏立民（魏立民, Wèi Lìmín）發現該株茶樹生長勢強、發芽能力高且品質穩定。魏氏進行再次扦插，後代迅速在全村推廣，獲得民間稱號「立民種」（立民種, Lìmín zhǒng）— 「立民的品種」。值得注意的是，玉蘭香是少數能在相對較低海拔種植仍能產出高香茶葉的鳳凰單叢之一，這決定了它在烏崬精英山帶之外的廣泛傳播。

更廣泛的背景：鳳凰山茶業已有900多年歷史。傳說最初的茶樹與1278年逃避蒙古追擊的南宋皇帝有關 — 鳳凰鳥為他帶來了茶樹枝條。最早的文獻記載可追溯至明代：弘治年間（弘治, 1488–1505），待詔山（待诏山）的茶葉成為宮廷貢品。「一株一香」（單株采制, dānzhū cǎizhì）的原則在清代同治-光緒年間（1875–1908）形成，當時茶農開始選育和繁殖優異的單株茶樹。

* **名稱：** 玉蘭（Yùlán）— 「玉蘭花」（字面意思為「玉石蘭花」）；香（xiāng）— 「香氣」；單叢（dāncóng）— 「單獨的茶叢」，指個別選育和加工的原則。完整含義：「具有玉蘭花香的單叢」。植物學上，「玉蘭」在華南和臺灣文化語境中通常指白玉蘭/白蘭花（*Michelia × alba*），而非望春玉蘭（*Magnolia denudata*）— 一種具有濃郁甜涼香氣的花朵，在東南亞常由街頭小販販售供佩戴。

* **文化意義：** 玉蘭香在十大經典香型中佔據特殊地位：如果說黃梔香、蜜蘭香和芝蘭香代表鳳凰派的「強勁」與「濃郁」，那麼玉蘭香則代表其「優雅」的一面。這款茶深受潮州工夫茶（潮州工夫茶, Cháozhōu gōngfū chá）愛好者推崇，因其能夠逐漸展開：前幾泡內斂，但每一泡香氣都在增強，可持續十多泡。單叢整體被稱為「茶中香水」（茶中香水, chá zhōng xiāngshuǐ），而玉蘭香是其最精緻的音符。

**3. 植物學描述與原料：**

* **品種/栽培種：** 玉蘭香是從鳳凰水仙（鳳凰水仙, Fènghuáng Shuǐxiān）群體中選育的無性系品種，屬國家標準品種（*Camellia sinensis* var. *sinensis*）。小喬木型（小乔木, xiǎo qiáomù），中葉種，晚芽種（迟芽種, chí yá zhǒng）。葉片橢圓形，具特徵性葉柄彎曲，鋸齒邊緣稀疏而尖銳。芽活力強，產量良好。最著名的亞品系包括：官目石玉蘭香（官目石玉兰香）、金玉蘭（金玉兰, Jīn Yùlán — 「金色玉蘭」，因嫩葉呈黃綠色而得名）、字茅玉蘭（字茅玉兰）、娘仔傘（娘仔伞, Niángzǐ Sǎn — 「女兒傘」，因樹冠形狀而得名）。

* **採摘：** 玉蘭香屬晚熟品種。主要採摘期為晚春，從穀雨（谷雨, 4月20日）到立夏（立夏, 5月初）。秋茶（秋茶, qiūchá）也有採製，但品質較低。採摘嚴格手工進行：等待駐芽（驻芽, zhù yá）出現，然後以「騎馬採」（骑馬采, qímǎ cǎi）手法採摘2–5葉的嫩梢。有三個嚴格禁忌：烈日下不採、雨天不採、露水未乾不採。

* **採摘標準：** 帶駐芽的嫩梢加2–3片成熟葉 — 高級茶的標準。過嫩的嫩梢會產生過多苦味（兒茶素含量高），過老則口感粗糙、纖維質重。

* **原料要求：** 完整嫩梢無機械損傷，成熟度一致。葉片鬆散放置在茶籮（茶罗, cháluó）中，不壓實，立即送至加工廠。玉蘭香的特點：該品種即使在中等海拔（400–600公尺）種植也能產出明顯花香的茶葉，而大多數單叢需要海拔≥800公尺才能表現出「山韻」。

**4. 風土條件與栽培特點：**

* **地區與地形：** 潮州東北部的鳳凰山山脈，最高海拔1,497.8公尺（鳳凰頂, 鳳凰頂）。地形為陡峭山坡、深谷，被山澗切割。鳳凰鎮位於山脈東南坡，當地人稱之為「潮汕屋脊」（潮汕屋脊）。森林覆蓋率85.1%，總綠化率96.4%。

* **種植海拔：** 玉蘭香的最佳區域為800–1,000公尺（烏崬的頂級樣品）。次級區域為400–600公尺，該品種與許多「嬌貴」的單叢不同，在此海拔仍能形成高香茶葉。平原和沖積地不適合種植。

* **氣候：** 南亞熱帶季風氣候。具有三個創造獨特條件的特點：早冬寒冷、春季涼爽綿長、夏季無極端高溫。烏崬山「十日三日晴，百步無平地」（山高高，霧蒙蒙，天無三日晴，地無百步坪）：全年雲霧繚繞，降雨充沛，直射陽光短暫。這些條件減緩嫩梢生長，有利於芳香前體物質和胺基酸的積累。

* **土壤：** 海拔>400公尺為黃壤、紅壤（黄壤, 紅壤），pH 4.5–6.5，土層深厚，有機質含量高（3.8–4.3%），微量元素豐富。礦物營養通過根系運輸至葉片，參與芳香化合物的生物合成。茶園無需人工灌溉 — 山地土壤因雲霧和森林覆蓋保持充足水分。

**5. 製作工藝：**

鳳凰單叢與福建和臺灣烏龍茶有兩個根本區別：條索型（而非球型）揉捻和深度最終焙火加「退火熟化」（退火熟化, tuìhuǒ shúhuà）。玉蘭香等晚熟品種從採摘到成品的整個週期約15天。

* **採摘 / 采摘 — cǎizhāi：** 在駐芽出現時手工採摘嫩梢，多在午後進行。

* **曬青 / 晒青 — shàiqīng：** 鮮葉薄攤於陽光下。目的是初步失水、軟化細胞壁、啟動酶促過程。玉蘭香的曬青適中：該品種需要溫和處理，以免破壞精緻的芳香前體物質。

* **晾青 / 晾青 — liàngqīng：** 將葉片移至通風室內，平衡水分分佈，讓組織在下一階段前「休息」。

* **做青（搖青與靜置交替）/ 做青 — zuòqīng：** 關鍵階段，包括搖青（摇青, yáoqīng）、浪青（浪青, làngqīng）和碰青（碰青, pèngqīng）與靜置的節奏性交替。每個週期加強葉緣氧化，釋放芳香化合物。對於玉蘭香，師傅追求均勻但不過度的氧化，保持玉蘭花香的純淨和透明。正是在此階段形成特徵性的「綠蒂綠腹紅鑲邊」（青蒂綠腹紅镶边, qīngdì lǜfù hóng xiāngbiān），發生決定性的香氣轉化。

* **殺青（殺青）/ 杀青 — shāqīng：** 在圓筒形炒茶機中高溫處理，停止酶促氧化，固定已形成的香氣特徵。

* **揉捻 / 揉捻 — róuniǎn：** 條索型揉捻形成廣東烏龍茶特有的長、緊、直的條索（條索, tiáosuǒ）— 有別於臺灣和閩南烏龍的球形。揉捻適中：過度壓力會破壞葉片完整性，影響單叢美觀。

* **烘焙 / 烘焙 — hōngbèi：** 多階段乾燥和焙火 — 決定香氣深度和穩定性的最終階段。玉蘭香通常比黃梔香或肉桂香焙火更輕：目標是保持透明的花香，而非增強「火韻」深度。

* **退火熟化 / 退火熟化 — tuìhuǒ shúhuà：** 焙火後靜置約15天，讓「火氣」沉澱，香氣成熟。只有經過這個過程才能展現玉蘭香的真正特色。年底會出現「返春」（返春, fǎnchūn）現象：香氣再次增強，滋味更加柔和。

**6. 感官特徵：**

* **乾茶外形：** 條索緊結（條索紧结, tiáosuǒ jǐnjié），粗壯整齊，深褐色（褐色）帶油潤光澤，具特徵性紅色「朱砂紅點」（朱砂紅點, zhūshā hóngdiǎn）。金玉蘭顆粒更大更緊結，呈灰褐色調。

* **乾茶香氣：** 細膩、清涼、純淨。不會從茶荷中「喊叫」— 需溫壺後才展現。可識別的主題：白玉蘭花帶綠色、略帶水感的韻味。

* **茶湯香氣：** 主要特徵是純淨和「空靈」。天然白玉蘭花香（玉兰花香清幽馥郁）：清涼甘甜、輕盈，隨每泡遞增。香氣不壓迫，而是飄浮在茶杯上方，滲透口腔，在空杯底留香（杯底香, bēidǐ xiāng）。正確焙火後無絲毫「火味」。經15天退火後出現淡淡蜂蜜韻味。

* **滋味：** 玉蘭香在單叢中的特色是精緻而不空洞。茶湯不苦不澀（色黄而不苦涩, sè huáng ér bù kǔsè），滋味清淨而不寡薄（味清而不寡薄, wèi qīng ér bù guǎbó）。湯質輕盈絲滑，帶柔和甘甜和明顯回甘（回甘, huígān）。餘韻持久，花蜜香型，具特徵性「喉韻」（喉韵, hóuyùn）。茶葉可沖泡10多次，香氣和滋味保持穩定。

* **湯色：** 淡黃明亮（淡黄明亮, dàn huáng míngliàng），清澈透明。無混濁和紅色調（與氧化程度更深的蜜蘭香或黃梔香不同）。

* **葉底（沖泡後茶葉）：** 完整大葉帶嫩梢。經典特徵：「青蒂綠腹紅鑲邊」（青蒂綠腹紅镶边）。葉片彈性好，油潤有光澤，鋸齒邊緣有明顯紅點。

**7. 化學成分：**

* **多酚類：** 鳳凰單叢總含量可達30%。玉蘭香因晚採和適度氧化，多酚複合物平衡：兒茶素足以構建滋味結構，但無過度澀味。

* **胺基酸：** L-茶胺酸和其他游離胺基酸提供柔和、甘甜和特徵性「透明」質感。高山樣品（>800公尺）因雲霧和散射光條件下生長緩慢，胺基酸含量更高。

* **芳香化合物：** 根據戴素賢團隊的GC/MS分析，玉蘭香的特徵是法呢醇（法呢醇, fǎ ní chún）、吲哚（吲哚, yǐnduǒ）、法呢烯（法呢烯, fǎ ní xī）、芳樟醇及其氧化物（芳樟醇及其氧化物）和香葉醇（香葉醇, xiāngyè chún）的主導地位。正是法呢醇和香葉醇的高含量形成了區別玉蘭香與其他單叢的特徵性清涼花香。總共識別出40多種芳香成分。

* **生物鹼：** 咖啡因 — 中等水平（廣東半發酵烏龍茶的典型水平），可可鹼和茶鹼 — 微量。

* **維生素：** C、B₁、B₂、P（芸香苷）；深度焙火會降低維生素C含量。

* **礦物質：** 鉀、鎂、錳、鋅、氟；鳳凰山古老火山土壤（有機質3.8–4.3%）豐富了礦物質組成。

* **特點：** 玉蘭香與其他單叢一樣，含有相對較高的吲哚 — 低濃度時產生愉悅的茉莉花香，高濃度時則呈現厚重的「香水味」。加工技藝在於將吲哚保持在「花香」範圍內。

**8. 保健功效：**

* **溫和刺激無焦慮：** 咖啡因和L-茶胺酸的協同作用提供持續的專注力和思維清晰，無「咖啡因峰值」和心跳加速。

* **抗氧化保護：** 多酚複合物（兒茶素、茶黃素）中和自由基，減少氧化應激。單叢的多酚含量是烏龍茶中最高的之一。

* **消化支持：** 半發酵茶對胃黏膜的刺激比綠茶溫和。焙火單叢對敏感消化系統特別舒適，有助於餐後脂肪分解。

* **脂質代謝益處：** 兒茶素有助於調節膽固醇水平，降低低密度脂蛋白。

* **心血管系統支持：** 系統性飲用茶多酚與血壓正常化相關。

* **口腔健康：** 氟和多酚具抗菌作用，防止蛀牙和牙菌斑形成。

* **抗輻射特性：** 研究表明茶多酚能減少電磁輻射的有害影響。

* **冥想茶飲：** 潮州工夫茶是中國最古老、最儀式化的茶道傳統之一。玉蘭香的緩慢多泡沖泡和香氣逐漸展開是減壓、回歸當下的修行。

**9. 沖泡方法：**

* **水溫：** 95–100°C。沸水充分展現玉蘭花香；非常嫩的春茶可用95°C。
* **茶量：** 8公克配100–120毫升（工夫泡）。單叢「寧多勿少」— 濃投茶確保香氣飽滿。
* **茶具：** 潮州瓷蓋碗（盖碗, gàiwǎn）是理想選擇：薄瓷不吸香，可精確控制時間。宜興紫砂壺（紫砂壶）可用，但對玉蘭香這類透明花香單叢，蓋碗更佳。茶杯小型（30–50毫升），潮州工夫茶傳統用三個。
* **流程：**
  1. 用沸水溫熱蓋碗、公道杯和茶杯。
  2. 投茶入熱蓋碗，蓋上搖晃2–3次 — 從蓋上聞溫熱乾茶香。
  3. 溫潤泡（溫润泡, wēnrùn pào）：注水3–5秒後倒出。
  4. 第一泡：5–8秒。
  5. 通過公道杯分茶入杯。
  6. 後續沖泡：第2泡5秒，第3–5泡5–8秒，之後每泡增加5秒。可沖10–15泡。
* **注意事項：** 單叢不耐悶泡：稍微「悶」一下就會從精緻純淨變成苦澀。寧可早一秒出湯，不可晚一秒。泡間稍開蓋碗蓋 — 防止茶葉「悶熟」。

**10. 儲存：**

* 密封不透光容器（鋁箔袋真空封裝或密蓋鐵罐）。乾燥陰涼處，避免直射光和異味。
* 焙火單叢儲存性好 — 正確條件下可存2–3年。每6–12個月建議低溫「復烘」（复烘, fù hōng）穩定水分。
* 清香型輕焙火茶 — 冰箱（0–5°C）密封保存，6個月內飲用。
* 特點：年底優質玉蘭香會出現「返春」（返春, fǎnchūn）現象 — 儲存幾個月後香氣自然增強。
* **茶葉天敵：** 濕氣、高溫、光照、異味。勿與香料、香水、清潔用品同存。

**11. 價格與假冒：**

* **價格範圍：** 價格變化很大。烏崬山玉蘭香（海拔>800公尺，老樹）— 根據具體茶樹和師傅，每公斤數千至數萬元人民幣。中山樣品（400–600公尺）— 價格親民得多，每公斤500–2,000元。價格因素：種植海拔、樹齡、季節（春茶比秋茶貴）、師傅聲譽、手工程度。

* **避免假冒方法：**
  * 向有透明來源鏈的商家購買；理想情況是鳳凰茶農直供。
  * 評估外觀：正宗單叢條索整齊挺直，油潤有光澤。細碎、斷裂、粉塵多是低級或假冒茶的標誌。
  * 評估香氣：玉蘭香應具純淨清涼的玉蘭花香，無「化學香水味」、刺鼻煙味或霉味。
  * 檢查茶湯：透明清黃，無混濁。滋味柔和純淨，正確沖泡不苦。假冒和低級「單叢」常第一泡就苦。
  * 注意耐泡性：正宗玉蘭香香氣可持續10多泡。如果第三四泡香氣就消失，品質可疑。

**12. 趣味知識：**

* **「立民種」— 民間育種家：** 玉蘭香是少數獲得「雙重命名」的單叢之一：官方名（按香氣）和民間名（按1960年代繁殖推廣此品系的農民魏立民命名）。這種雙重命名是鳳凰傳統的典型特徵，茶樹按香氣、主人、樹形甚至鄰近石頭命名。

* **不怕山谷的單叢：** 大多數鳳凰單叢只有在800公尺以上才能真正「發聲」。玉蘭香是罕見的例外：其遺傳特性允許即使在中等海拔（400–600公尺）也能形成明顯的高香特徵，使其成為烏崬精英區外最廣泛種植的品系之一。

* **一山104香：** 戴素賢教授（1996）的研究在不同鳳凰單叢中識別出104種揮發性芳香化合物。僅玉蘭香就發現40多種成分 — 比許多香水配方還多。

* **「返春」：** 返春（返春）現象 — 茶葉香氣在製作幾個月後自然增強 — 許多單叢都有，但玉蘭香表現特別明顯。鳳凰茶農說：「春茶冬飲」（春茶冬飲）。

* **茶中香水：** 單叢常被稱為「茶中香水」（茶中香水）。劉仲華院士（劉仲華）品嚐後留下經典評價：「第一杯 — 一見鍾情；第二杯 — 難以忘懷；第三杯 — 不離不棄」。玉蘭香在鳳凰十大「香水音符」中是白玉蘭音符：最安靜、最持久、最私密。

**13. 與其他單叢比較：**

* **黃梔香單叢（黄枝香單丛, Huángzhī Xiāng Dāncóng）：** 梔子花香 — 強勁、辛香、「張揚」。茶湯更厚重濃郁，湯質更深，澀味明顯。玉蘭香是其完全對立面：以精緻代替強勁，以清涼代替辛香。如果黃梔香是管弦樂，玉蘭香就是獨奏長笛。

* **蜜蘭香單叢（蜜兰香單丛, Mìlán Xiāng Dāncóng）：** 最普及和「親民」的單叢，濃郁蜂蜜蘭花香。茶湯更厚更甜，香氣更「溫暖」包裹。玉蘭香更乾淨透明；其甘甜不是蜂蜜式，而是「花香式」，帶特徵性清涼。

* **芝蘭香單叢（芝兰香單丛, Zhīlán Xiāng Dāncóng）：** 紫蘭花香 — 精緻悠長，在「優雅度」上最接近玉蘭香。區別在性格：芝蘭香更辛香深沉，有明顯「老叢韻味」（老丛特韵）；玉蘭香更輕盈清涼，更「透明」。

* **桂花香單叢（桂花香單丛, Guìhuā Xiāng Dāncóng）：** 桂花香 — 溫暖、辛甜、持久。茶湯更厚，有堅果韻味。玉蘭香音調更「冷」，不那麼辛香，更偏花香綠調。

* **鴨屎香單叢（鸭屎香單丛, Yāshǐ Xiāng Dāncóng，又名銀花香 / 銀花香）：** 近年最「時尚」的單叢，濃郁金銀花香。更強勁、甘甜、「厚重」，蜂蜜韻明顯。玉蘭香在風格上是其對立面：鴨屎香以強勢取勝，玉蘭香以寧靜制勝。

**結語：**

玉蘭香單叢是為那些已經超越「響亮」單叢初識階段，在香氣中尋求寧靜的人而生。其他鳳凰烏龍以第一口就震撼，玉蘭香則邀請耐心聆聽：其玉蘭韻不是傾瀉而下，而是漸次顯現 — 一杯接一杯，一泡接一泡，從最初羞澀的花香暗示到第六七泡充滿整個房間的飽滿立體香氣，直到夜晚仍不離喉韻。

這款茶是單叢世界中力量與精緻並非一回事的最佳證明。有時最輕的聲音響得最久。
