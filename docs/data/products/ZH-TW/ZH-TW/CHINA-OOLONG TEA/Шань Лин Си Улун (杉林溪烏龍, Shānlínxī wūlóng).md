# 杉林溪烏龍 (杉林溪烏龍, Shānlínxī wūlóng)

**杉林溪烏龍 (杉林溪烏龍, Shānlínxī wūlóng)**

![Шань Лин Си Улун (杉林溪烏龍, Shānlínxī wūlóng)-1](images/Шань%20Лин%20Си%20Улун%20(杉林溪烏龍,%20Shānlínxī%20Wūlóng)-1.png)

杉林溪烏龍是最具辨識度的臺灣高山烏龍茶之一，生長在南投縣涼爽多霧的森林中。此茶以其特有的「冷香」(冷香, lěng xiāng) 聞名，帶有柳杉和蘭花的香氣、油潤的茶湯質感以及持久回甘的甜美後韻。與阿裡山和梨山並列為臺灣三大最負盛名的高山茶區。

**1. 分類與產地：**

* **類型：** 烏龍茶 (烏龍茶, wūlóng chá) — 半發酵茶。發酵程度輕度，15至30%。焙火通常輕度或中度；傳統競賽風格 (竹山鎮農會) 允許較明顯的焙火 (臺灣評級標準4-5分)。
* **類別：** 臺灣高山烏龍 (高山烏龍, gāoshān wūlóng)。依臺灣分類標準 — 生長海拔1,000公尺以上的茶葉。
* **產地：** 臺灣 (臺灣, Táiwān)，南投縣 (南投縣, Nántóu Xiàn)，竹山鎮 (竹山鎮, Zhúshān Zhèn)，杉林溪山區 (杉林溪, Shānlínxī)。茶區位於阿裡山支脈 (阿里山支脈) 上，處於南投、嘉義 (嘉義, Jiāyì) 和雲林 (雲林, Yúnlín) 三縣交界。部分茶園延伸至鹿谷鄉 (鹿谷鄉, Lùgǔ Xiāng)。
* **地理座標：** 約北緯23°38′，東經120°46′。

**2. 歷史與文化意義：**

* **歷史：** 杉林溪是相對年輕的茶區。在日治時期及戰後初期數十年間，此山區為工業伐木區：山坡遍植日本柳杉 (杉木, shānmù — *Cryptomeria japonica*)，因此得名。至1970年代，森林保護政策和政府鼓勵替代產業導致林業衰落。企業家劉安定 (劉安定) 成立杉林溪遊樂公司，並促成臺灣首條私人山區道路建設，經過著名的「十二生肖彎道」(十二生肖彎道) 從溪頭 (溪頭) 通往杉林溪。正是在1970年代末至1980年代初，農民開始在清理後的林地種植茶樹，主要是青心烏龍品種。首批收成即顯示茶葉具有獨特的「冷」調柳杉香氣，顯然與昔日柳杉林的風土條件和殘存的酸性針葉腐植層有關。至1990年代，杉林溪的「冷香礦物香氣」成為市場轟動，茶葉穩居最受歡迎的高山茶之列。自2000年代起，竹山鎮農會 (竹山鎮農會) 舉辦年度品質競賽 (春季和冬季)，由臺灣茶業改良場 (茶業改良場) 專家評鑑。競賽標準為香氣與滋味70%，外觀20%，湯色10%。特等獎 (特等獎) 茶樣在拍賣會上可達每斤 (600公克) 約30,000臺幣。

* **名稱：** 「杉林溪」(杉林溪) 字面意思為「柳杉林溪流」。漢字杉 (shān) 指柳杉 (俄語傳統中常譯為「中國杉」— *Cunninghamia*，或「日本杉」— *Cryptomeria*)，林 (lín) 為森林，溪 (xī) 為山溪。「烏龍」(烏龍, wūlóng) 意為「黑龍」，為半發酵茶類的總稱。該區也以個別茶區的地方名稱聞名：龍鳳峽 (龍鳳峽, Lóngfèngxiá — 「龍鳳峽谷」)，羊仔彎 (羊仔彎, Yángzǎiwān)，軟鞍 (軟鞍, Ruǎnān)，番仔田 (番仔田, Fānzǎitián)，三層坪 (三層坪, Sāncéngpíng)，草沓仔 (草沓仔, Cǎotàzǎi)。

* **文化意義：** 杉林溪烏龍被視為臺灣高山茶的典範之一，在臺灣茶愛好者和國際市場均享有崇高地位。杉林溪地區同時是熱門旅遊區 — 設有自然公園、觀景臺、穿越柳杉林和竹林的山徑。沿「十二生肖彎道」行駛，品嚐各段路程的茶葉，使杉林溪成為臺灣最具吸引力的茶旅目的地之一。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種為青心烏龍 (青心烏龍, Qīngxīn Wūlóng)，亦稱軟枝烏龍 (軟枝烏龍, Ruǎnzhī Wūlóng) — 「軟枝烏龍」。此為晚熟臺灣本地品種，遺傳上源自福建軟枝烏龍和矮腳烏龍 (矮腳烏龍) 品種。青心烏龍約佔杉林溪地區種植面積90%。該品種富含醇類芳香化合物 (揮發性物質組成的55-70%)，在不同發酵程度下易轉化為各種花香調。少量種植金萱 (金萱, Jīn Xuān) — TTES 12號，以淡奶香聞名，以及翠玉 (翠玉, Cuì Yù) — TTES 13號，具清新香氣。*Camellia sinensis* var. *sinensis*。
* **採摘：** 主要季節為春茶 (春茶, chūnchá；5月) 和冬茶 (冬茶, dōngchá；11月)。另有晚冬「冬片」(冬片, dōngpiàn) — 冬至前後採摘，以特別冷冽純淨的香氣著稱。夏茶和秋茶較不受重視。杉林溪春茶和冬茶芽葉平均生長期超過55天 — 遠長於平地茶區，有利於胺基酸和芳香物質的累積。
* **採摘標準：** 高級茶為「一心二葉」(一心二葉, yī xīn èr yè)，競賽茶中此標準比例≥95%。葉片背面須覆有細白茸毛。
* **原料要求：** 完整幼嫩芽葉，成熟度均勻，鮮嫩，無機械損傷、異味和粗老葉片。特級鮮葉胺基酸含量不少於5.2%。

**4. 風土條件與栽培特色：**

* **地區與地形：** 杉林溪為阿裡山支脈上的廣闊山區，地勢陡峭，峽谷深邃，溪流眾多。森林覆蓋率達93%，山區覆蓋原始和次生柳杉林及竹林，使茶園免受外界污染。主要茶區：龍鳳峽 (海拔~1,800公尺) — 被認為是最高海拔和最負盛名的茶區；羊仔彎 (~1,500公尺) — 有成熟茶樹 (樹齡≥30年) 的區域；軟鞍、番仔田、三層坪、草沓仔 — 中等海拔茶區。
* **生長海拔：** 茶園分布於海拔1,000至1,950公尺。最優質茶園核心區位於1,500-1,800公尺；最高區域 (據某些資料達~2,600公尺) 屬於與福壽山和翠峰的邊界地帶。
* **氣候：** 亞熱帶高山氣候。年平均氣溫低於15°C。日夜溫差≥15°C。年霧日數≥200天，相對濕度≥85%。霧氣從午間開始籠罩茶園，散射陽光並減緩光合作用。葉片生長緩慢導致苦澀味降低 (兒茶素含量較低) 而甜味和香氣增強 (胺基酸和精油含量較高)。冬季涼爽，偶有霜凍。
* **土壤：** 火山岩上的紅黃壤 (紅黃壤)，pH值4.5-5.5，富含鐵和鎂。有機質含量≥3%。陡峭山坡具良好天然排水。柳杉的酸性針葉腐植層有助於形成茶葉特有的「冷」調礦物香氣。

**5. 製作工藝：**

製作風格為「清香」(清香, qīngxiāng) — 「清香」：輕度發酵 (20-30%)，低溫慢速乾燥以保持冷調柳杉香氣，手工揉捻成半球形。竹山農會競賽風格強調較明顯的發酵和焙火以展現茶湯的深度和厚實感。

* **採摘 / 採摘 — cǎizhāi：** 手工採摘「芽+2葉」標準的頂端嫩芽。鮮葉立即送至加工廠以防過熱和不受控制的氧化。
* **日光萎凋 / 日光萎凋 — rìguāng wěidiāo：** 採摘的葉片在戶外攤放20-30分鐘，接受溫和陽光照射以開始失水並啟動酶促過程。
* **室內萎凋 / 室內萎凋 — shìnèi wěidiāo：** 將葉片移至通風室內，攤放在竹製托盤上約4小時。此階段細胞膜部分破損，啟動緩慢氧化並形成香氣基礎。
* **搖青 / 搖青 — yáoqīng：** 三輪溫和搖青 (浪青, làngqīng)，中間讓葉片「靜置」。杉林溪的搖青特別輕柔 — 目標是形成花香特徵而不過度發酵。氧化主要沿葉緣進行，形成特有的「綠葉紅鑲邊」(綠葉紅鑲邊, lǜyè hóng xiāngbiān)。
* **殺青 / 炒青 — chǎoqīng：** 約280°C高溫處理停止酶促過程並固定香氣走向。
* **揉捻 / 揉捻 — róuniǎn：** 機械處理葉片以破壞細胞結構並提高浸出率。
* **初烘 / 初烘 — chūhōng：** 80°C烘乾以穩定形狀。
* **包揉塑形 / 包揉塑形 — bāoróu sùxíng：** 將葉片包在布中反覆手工揉捻，形成特有的半球狀 (半球狀, bànqiú zhuàng)。此階段最為費工，需要技巧和體力。
* **復烘 / 復烘 — fùhōng：** 約60°C低溫烘乾，採用「低溫慢焙」(低溫慢焙, dīwēn mànbèi) 原則以固定「冷香」並穩定含水量至貯存標準。
* **揀剔 / 揀剔 — jiǎntī：** 去除老葉、茶梗和雜質。競賽茶的篩選特別嚴格。

**6. 感官特徵：**

* **乾茶外觀：** 緊結的半球狀顆粒 (半球狀)，大小中等至偏大。色澤墨綠油潤 (墨綠油潤)。高級茶顆粒沉重緊實。芽尖可見淡色茸毛。
* **乾茶香氣：** 主導「冷香」調 (冷香) — 純淨、清涼，帶明顯柳杉針葉香氣。次要層次為蘭花、鈴蘭和淡奶香甜味。春茶更突出花香豆香 (蘭香與嫩豆香)，冬茶則呈現糖甜味和礦物感。
* **茶湯香氣：** 濃郁的花香針葉香譜，伴隨漸增的蜂蜜甜香。蘭花香與柳杉的木質「基調」交織 — 這是杉林溪區別於其他高山茶的標誌性特徵。杯底冷卻後展現持久的奶香焦糖香氣。焙火競賽風格中出現堅果焦糖韻味。
* **滋味：** 飽滿、油潤、順滑 (甘滑醇厚)。明顯的鮮爽感 (鮮)，源於高胺基酸含量。甜味 — 冬茶呈「甘蔗」甜，春茶則較為花香清淡。輕微宜人的苦味迅速轉為持久回甘 (回甘, huígān)，喉韻帶特有薄荷清涼感 (喉韻, hóuyùn)。茶湯厚實，具明顯「膠質感」(膠質顯)。香調：蘭花、柳杉、熟桃、蜂蜜。
* **湯色：** 蜜綠透金黃 (蜜綠透金黃)，高度透明，表面有明顯膠質光暈 (光暈)。
* **葉底 (沖泡後葉片)：** 肥厚、彈性、完整的葉片，完全展開至原有大小。色澤從亮綠至橄欖綠，葉緣帶紅暈。莖梗富有彈性和水分。

**7. 化學成分：**

* **多酚類：** 總多酚含量在烏龍茶中屬中等，因高海拔緩慢生長和輕度發酵而低於平地品種。主要成分為兒茶素：表沒食子兒茶素沒食子酸酯 (EGCG)、表兒茶素沒食子酸酯 (ECG)、表沒食子兒茶素 (EGC)、表兒茶素 (EC)。部分氧化將部分兒茶素轉化為茶黃素類化合物，減輕澀味。
* **胺基酸：** 含量較高 — 高級茶鮮葉不少於5.2% (高於臺灣烏龍茶平均值)。以L-茶胺酸 (茶胺酸) 為主，負責甜味、鮮味和放鬆效果。高胺基酸濃度直接源於低溫、散射光照和大溫差下的緩慢生長。
* **生物鹼：** 咖啡因含量中等 (約佔乾重2-3%)。單寧含量比平地烏龍高約20%，因此不建議空腹飲用。還含有可可鹼和茶鹼。
* **維生素：** C、E、K、B群。由於輕度發酵，維生素C保存優於重發酵茶。
* **礦物質：** 鉀、鎂、錳、鐵、氟 (~15毫克/100克 — 高含量，具顯著防蛀牙活性)、鋅、微量硒。礦物質組成因火山土壤而富含鐵和鎂。
* **精油和揮發性化合物：** 醇類芳香化合物含量高 (芳樟醇、香葉醇、橙花醇、順-3-己烯醇)，佔青心烏龍香氣組成的55-70%。正是這些化合物在輕度發酵下轉化為蘭花和柳杉香氣。

**8. 保健功效：**

* **抗氧化保護：** 多酚類和兒茶素中和自由基，有助延緩細胞老化。據多項研究，高山烏龍的自由基中和效力相當於維生素E的18倍活性。
* **支持新陳代謝和脂質代謝：** 茶多酚促進脂肪分解；據某些資料，高山烏龍的脂解速度比平地同類茶高約30%。規律飲用可能有助降低「壞」膽固醇 (LDL) 並提高「好」膽固醇 (HDL)。
* **提神與鎮靜平衡：** 咖啡因和L-茶胺酸的結合提供溫和、專注的清醒感而無焦慮。L-茶胺酸刺激大腦產生α波，促進平靜專注狀態。
* **支持消化：** 多酚類和單寧溫和刺激消化酶分泌，改善食物吸收。焙火風格對胃部特別溫和。
* **口腔健康：** 高氟含量 (~15毫克/100克) 結合多酚類的抗菌活性抑制致蛀菌活動 (據某些評估達90%)，強化牙釉質。
* **心血管支持：** 多酚類有助強化血管壁並通過抑制LDL膽固醇氧化降低動脈硬化風險。
* **改善肌膚狀況：** 抗氧化劑、維生素C和E有助均勻膚色並維持肌膚彈性。
* **緩解壓力：** 柳杉香氣和高L-茶胺酸含量共同產生放鬆效果，使此茶成為正念茶飲的絕佳選擇。

**9. 沖泡方法：**

* **水溫：** 標準茶品90-95°C；最嫩的特級春茶85-90°C。競賽風格 (焙火) 可承受95-100°C。
* **茶量：** 功夫茶法6-8克/150-200毫升；歐式沖泡3-4克/250毫升。
* **茶具：** 白瓷蓋碗 (蓋碗, gàiwǎn) — 最適合展現「冷香」和評鑑湯色。宜興紫砂壺 (紫砂, zǐshā) — 適合焙火風格，增加柔和度。瓷壺 — 通用選擇。
* **沖泡過程：**
  1. 用沸水溫壺，倒掉水。
  2. 投茶入蓋碗或茶壺。品聞溫熱乾茶香氣。
  3. 潤茶 (潤茶, rùnchá)：注水後3-5秒倒出 — 這是「喚醒」緊結葉片。可重複兩次。
  4. 第一泡：浸泡30-45秒，然後分茶入杯。
  5. 後續沖泡 (2-10泡)：每泡增加10秒。優質杉林溪可沖泡8-10次。
  6. 後期沖泡 (7-10泡) 可提高水溫至95-100°C以充分萃取剩餘物質。

* **建議：** 前幾泡勿過度浸泡 — 過度萃取會產生不必要的澀味，「掩蓋」精緻的針葉香氣。短時間沖泡更能展現層次豐富的滋味結構。

**10. 貯存：**

* **一般條件：** 密封包裝 (真空或鋁箔袋加夾子)，陰涼 (< 20°C)、避光處，遠離異味。
* **清香風格 (清香)：** 理想狀態下冷藏於0-5°C，單獨存放，避免與食品接觸。此類茶對濕氣和異味特別敏感。開封後建議72小時內飲用以保持香氣新鮮度。
* **焙火風格：** 要求較低，可室溫密封保存。隨時間推移，焙火烏龍可能略有「陳化」，香氣趨於柔和。
* **新茶：** 建議讓新茶在陰暗處「醒茶」約一個月，以散去製作過程中殘留的「火氣」(火氣, huǒqì)。
* **茶葉大敵：** 濕氣、高溫、直射陽光、異味。

**11. 價格與仿品：**

* **價格區間：** 杉林溪烏龍屬臺灣高山烏龍的精品級別。價格取決於生長海拔 (越高越貴)、採摘季節 (高海拔冬茶最珍貴，約佔市場70%)、品種 (青心烏龍比金萱貴)、手工程度、競賽評級和特定生產者聲譽。參考價格：海拔1,800公尺特級茶約6,000人民幣 (~$180) /斤 (500克)。競賽特等獎 (特等獎) 可達30,000臺幣 (~$950) /斤。
* **如何避免仿品：**
  * 向有透明來源鏈的專業商家購買。信譽良好的臺灣茶商會提供海拔、季節和生產者資訊。
  * 評估外觀：顆粒應緊實、均勻、墨綠油潤，無粉末和碎片。
  * 檢查香氣：真正的杉林溪具有純淨的「冷」調針葉香氣，而非化學「香水味」或平淡花香。
  * 分析茶湯：色澤應為蜜綠透金黃，透明，表面有膠質光澤。混濁或深色茶湯是低品質或造假的徵象。
  * 對可疑低價保持警惕：真正的高山杉林溪不可能便宜。標示為「杉林溪」的平地烏龍是常見的冒充手段。

**12. 趣聞軼事：**

* **柳杉香氣作為風土印記。** 杉林溪特有的針葉香調並非人工添香，而是風土條件的自然作用：種植在昔日柳杉林砍伐地的茶樹從土壤中吸收芳香化合物。這種「杉木」韻味成為茶葉真實性的主要標誌，也是其與鄰近阿裡山和梨山茶區的區別所在。
* **臺灣首條私人山區道路。** 1970年代劉安定倡議建設的溪頭至杉林溪道路，成為島上首條私人資本投資的山區道路。正是這條路開啟了該區的旅遊業和茶葉種植，將昔日的伐木區轉變為最負盛名的茶區之一。
* **「十二生肖彎道」。** 通往杉林溪的山路經過12個彎道，每個彎道都以中國生肖動物命名。個別茶區沿這些彎道分布，經驗豐富的品茶師能辨別不同路段茶葉的細微風味差異。
* **龍鳳峽 — 該區最高海拔茶園。** 海拔約1,800公尺的龍鳳峽 (「龍鳳峽谷」) 是杉林溪的精品茶區，一年僅允許三次採摘。其茶葉具有最深層的「冷香」和最持久的喉韻。
* **競賽風格 vs.「清香」。** 與許多專注於輕度清香風格的臺灣高山茶不同，杉林溪通過竹山農會競賽系統也發展出更深層的「熟香」風格 (熟香, shúxiāng)，強調發酵和焙火以突出茶湯的厚實和甜美。

**13. 與其他臺灣高山烏龍的比較：**

* **阿裡山烏龍 (阿里山烏龍, Ālǐshān Wūlóng)：** 生長於鄰近嘉義縣海拔1,000-1,600公尺。香氣較為單一，純淨蘭花香而無明顯木質調。滋味溫和優雅，礦物厚實感較少。被視為「入門」高山烏龍，而杉林溪則更複雜多層次。
* **梨山烏龍 (梨山烏龍, Líshān Wūlóng)：** 生長於海拔1,800-2,600公尺，是世界最高海拔烏龍茶之一。香氣精緻，果香花香帶「霜凍」韻味。相比之下，杉林溪具有更厚實的茶體和明顯的針葉木質特徵。梨山更昂貴稀有。
* **凍頂烏龍 (凍頂烏龍, Dòngdǐng Wūlóng)：** 來自鹿谷鄉 (與竹山相鄰地區) 的經典烏龍，生長於較低海拔600-800公尺。傳統上經較重焙火和發酵。滋味溫暖，焦糖堅果香，相比杉林溪缺少清新感和「冷香」。凍頂是臺灣烏龍的歷史「導師」，而杉林溪是其高山「傳人」。
* **大禹嶺烏龍 (大禹嶺烏龍, Dàyǔlǐng Wūlóng)：** 臺灣最高海拔烏龍茶 (2,400-2,600公尺)。產量極其有限，香氣極為精緻空靈，帶礦物透明感。相比之下，杉林溪更易取得，茶體更明顯，針葉特徵更突出，但在「空靈感」和礦物純淨度上略遜一籌。

**結語：**

杉林溪烏龍是一款擁有山林靈魂的茶葉。誕生於曾經生長百年柳杉的山坡上，它吸收了霧氣的清涼、火山土壤的礦物質和成為其標誌的寧靜針葉香氣。油潤順滑，喉韻深長，持久的蘭花杉木香調，這款茶逐漸展現，一泡接一泡，邀請人們進行從容、深思的茶飲體驗。杉林溪是那些已熟悉臺灣高山烏龍並尋求更高層次複雜度的茶友的絕佳選擇：比阿裡山更厚實，比梨山更具「森林感」，比凍頂更清新。每一杯這款茶都是「杉林溪」霧氣繚繞山徑的小小旅程。
