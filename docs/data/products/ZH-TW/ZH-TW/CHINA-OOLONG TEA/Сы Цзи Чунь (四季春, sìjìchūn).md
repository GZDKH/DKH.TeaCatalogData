# 四季春 (四季春, sìjìchūn)

**四季春 (四季春, sìjìchūn)**
![Сы Цзи Чунь (四季春, sìjìchūn)](images/Сы%20Цзи%20Чунь%20(四季春,%20Sìjìchūn).png)

![Сы Цзи Чунь (四季春, sìjìchūn)-1](images/Сы%20Цзи%20Чунь%20(四季春,%20Sìjìchūn)-1.png)

四季春是最具生產力且最易取得的臺灣烏龍茶之一，因其不挑剔的品種特性而廣泛傳播，該品種能夠每年提供六至八次採收。此茶以明亮的花香特徵著稱，以梔子花香為主導，口感溫和清新，使其成為臺灣茶飲產業的基礎原料，也是傳統茶藝中受歡迎的日常烏龍茶。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶）。氧化程度為15–30%，多數為輕發酵（清香型, qīngxiāng xíng — 約20%）或中度發酵（濃香型, nóngxiāng xíng — 約30%）。烘焙通常極輕；製作風格著重保持清新花香。
* **類別：** 臺灣中低海拔烏龍茶。不屬於高山茶（高山茶, gāoshān chá）；主要種植園位於海拔1000公尺以下。商業上常定位為「松柏長青茶」（松柏長青茶, Sōngbǎi Chángqīng Chá）— 「松柏常青茶」。
* **產地：** 臺灣（台灣, Táiwān），南投縣（南投縣, Nántóu Xiàn），名間鄉（名間鄉, Míngjiān Xiāng）— 主要生產中心。也種植於嘉義縣（嘉義縣, Jiāyì Xiàn）和花蓮縣（花蓮縣, Huālián Xiàn）。該品種最初在木柵（木柵, Mùzhà，現今臺北）地區被發現。
* **地理座標：** 約北緯23°50'，東經120°40'（名間茶區核心，八卦山脈南端的柏嶺台地）。

**2. 歷史與文化意義：**

* **歷史：** 四季春是相對年輕的品種，於1980年代初期被發現。來自木柵地區的茶農張文輝（張文輝, Zhāng Wénhuī）在其茶園中注意到一株茶樹，該樹是自然雜交（自然有性繁殖, zìrán yǒuxìng fánzhí）的結果，展現出異常旺盛的生長力和全年發芽特性。最初這株植物被稱為「輝仔茶」（輝仔茶, Huī Zǎi Chá — 「輝的茶」），以發現者命名，也稱為「六季香」（六季香, Liù Jì Xiāng — 「六季之香」）。當苗木被引入南投縣名間鄉時，當地農民高度評價這個新品種的高產量，並給它命名為「四季春」— 「四季之春」。1988年，種植材料被引入福建省（中國大陸）。2000年後，四季春實現了「產業突破」：由於適合機械採收和高產量，它成為臺灣蓬勃發展的茶飲產業（手搖茶飲, shǒuyáo cháyǐn）的主要茶葉原料，據估計佔島上中低海拔茶區面積的30%。

* **名稱：**
    * 「四季」（四季）— 「四季」，指品種全年生長和採收的能力。
    * 「春」（春）— 「春天」，強調茶葉無論何時採收都保持新鮮春天般的香氣特性。
    * 完整名稱「四季之春」— 永恆新鮮的詩意隱喻：每次採收都帶來令人想起最佳春茶的香氣。

* **文化意義：** 四季春在臺灣茶文化中佔據獨特地位。一方面，它是茶產業的「工作馬」— 大眾化、民主化的產品，是奶茶雞尾酒和香味烏龍茶的基礎。另一方面，來自精品農場的優質春茶和冬茶受到行家珍視，因其純淨的花香特徵和和諧的口感。正是四季春成為數百萬人進入臺灣烏龍茶世界的「引路人」，他們首次以冷泡茶飲的形式品嚐到它。此茶與松柏（松柏, Sōngbǎi）茶區和名間鄉深度相關— 臺灣最大的茶葉批發中心，茶店沿街延伸數百公尺。

**3. 植物學描述與原料：**

* **品種/栽培種：** 四季春（四季春, sìjìchūn）— 臺灣地方品種（地方性品種, dìfāngxìng pǐnzhǒng），通過自然雜交產生。**並非**臺灣茶業改良場（TTES）正式註冊的「臺茶」（台茶, Táichá）系列品種。屬於*Camellia sinensis* var. *sinensis*。植物學特徵：灌木型（灌木型, guànmù xíng），小葉種（小葉種, xiǎoyè zhǒng），早生種（早生種, zǎoshēng zhǒng）。葉形為紡錘形（纺锤形, fǎngchuí xíng），兩端尖銳。葉片顏色為淺綠色帶黃色調。葉緣覆蓋細小尖鋸齒。幼芽呈淡紅紫色。植物具有開張樹型（樹型披張, shùxíng pīzhāng），芽葉密集分佈，生長勢強，對病害和寒冷抗性高。幾乎沒有休眠期，確保全年多次採收。

* **採收：** 品種的主要優勢是每年可採收6–8次（某些資料顯示4–5次完整採收加上額外的夏季採收）。最珍貴的是春茶（春茶, chūnchá，3–4月）：一芽一葉，梔子花香明顯，口感清新純淨。冬茶（冬茶, dōngchá，11–12月）因茶湯濃度、甘蔗糖甜味和特有的「冷香」而受到重視。夏秋採收主要用於商業拼配和茶飲。

* **採收標準：** 主要為一芽二三葉（一芽二三葉, yī yá èr-sān yè）。精品批次為一芽二葉，比例不少於95%。大量生產採用機械採收。

* **原料要求：** 與高山烏龍茶相比，對原料的要求較不嚴格，這是該品種商業吸引力的一部分。允許使用較成熟的葉片，但最佳批次從成熟度均勻的嫩芽中挑選。

**4. 風土與栽培特點：**

* **地區與地形：** 生產核心為名間鄉的柏嶺台地（柏嶺台地, Bǎilǐng Táidì），位於八卦山脈（八卦山脈, Bāguà Shānmài）南端。這是丘陵地帶，坡度平緩，有廣闊的梯田種植園。額外的種植區包括松柏坑（松柏坑, Sōngbǎi Kēng）和赤水（赤水, Chìshuǐ）地區，那裡保存著百年茶園。也在嘉義縣和花蓮縣種植。

* **海拔高度：** 海拔200–500公尺— 典型的低中海拔帶。一些農場在海拔800公尺處種植四季春，但大部分產品來自海拔500公尺以下的種植園。

* **氣候：** 亞熱帶，溫暖潮濕。年平均溫度18–22°C。年降雨量超過2000毫米。空氣濕度80%以上。霧日超過200天。名間開闊台地的充足陽光確保茶樹旺盛生長和頻繁採收，但與高山陰涼地區相比，氨基酸積累較少。

* **土壤：** 紅壤和紅黃壤（紅黃壤, hóng huáng rǎng），八卦山台地特有。酸度pH 4.5–6.5。土壤富含鐵和鎂，排水良好，有利於茶樹深根。茶園周圍森林覆蓋率約85%。

**5. 製作工藝：**

四季春工藝遵循經典臺灣烏龍茶製法，重點在於輕發酵和「鎖住」花香。關鍵特點是「低溫慢焙」（低溫慢焙, dīwēn mànbèi），保持精緻花香。製作過程中在對氧化敏感的階段不使用金屬器具。整個過程在技術上比高山烏龍茶更簡單快速，這是由於大量生產的特性。

* **採摘 / 採摘 — cǎizhāi：** 嫩芽手工採摘（精品批次）或機械採摘（大量生產）。採摘的原料立即送到加工廠。

* **日光萎凋 / 日光萎凋 — rìguāng wěidiāo：** 葉片在陽光下攤放約30分鐘，進行初步失水和活化酶促過程。

* **室內萎凋 / 室內萎凋 — shìnèi wěidiāo：** 在通風室內持續約4小時。葉片變得柔軟，形成香氣基礎。

* **搖青 / 浪青 — làngqīng (摇青 — yáoqīng)：** 三輪在竹盤上輕柔搖動，啟動葉緣部分氧化，形成特有的「綠葉紅鑲邊」（綠葉紅鑲邊, lǜyè hóng xiāngbiān）圖案。四季春的搖青強度通常低於凍頂或單叢。

* **殺青 / 炒青 — chǎoqīng (殺青 — shāqīng)：** 約280°C炒制停止酶促過程並固定香氣方向。

* **揉捻 / 揉捻 — róuniǎn：** 揉捻葉片以形成外觀並提高浸出性。

* **初烘 / 初烘 — chūhōng：** 80°C乾燥以穩定。

* **包揉塑形 / 包揉塑形 — bāoróu sùxíng：** 手工布包後壓制，賦予葉片經典半球狀（半球狀, bànqiú zhuàng）形狀，這是臺灣烏龍茶的典型特徵。

* **復烘 / 復烘 — fùhōng：** 60°C烘乾至適合儲存的穩定濕度。採用「低溫慢焙」模式以最大程度保持花香。

* **分級 / 分級 — fēnjí：** 成品茶按顆粒大小、均勻度和品質分級。

**按加工類型分為兩種風格：**
* **清香型（清香型, qīngxiāng xíng）** — 輕發酵（~20%），重點在清新花香。
* **濃香型（濃香型, nóngxiāng xíng）** — 中度發酵（~30%），更濃郁的蜜香口感，甜味明顯。

**6. 感官特徵：**

* **乾茶外觀：** 緊密捲曲的半球狀顆粒（半球狀, bànqiú zhuàng），緊實緊湊。顏色為墨綠油潤（墨綠油潤, mòlǜ yóurùn），帶有該品種特有的淡黃色調。顆粒大小均勻。

* **乾茶香氣：** 濃郁純淨的花香，明顯的梔子花香（梔子花香, zhīzihuā xiāng）主導— 四季春的名片。次要香調：玉蘭香（玉蘭香, yùlán xiāng）、野薑花香（野薑花香, yě jiānghuā xiāng）、草本果香細節。香氣活躍，「迎面而來」— 在沖泡前就能在遠處感受到。

* **茶湯香氣：** 濃郁花香譜系，甜味漸增。熱茶湯中— 飽滿油潤的花果香調。冷茶湯中— 更純淨透明的花香調，重點在梔子花。中度發酵版本出現蜜香和焦糖調。

* **口感：** 鮮爽（鮮爽, xiānshuǎng），甘滑（甘滑, gānhuá）明顯，回甘（回甘, huígān — 回甜）持久，喉韻（喉韻, hóuyùn）清涼。茶湯體感從輕到中等，比高山烏龍茶油潤性和複雜性低，但花香成分更明亮「響亮」。氨基酸含量≥ 4.2%確保明顯的鮮味和鮮味。過度浸泡可能出現輕微苦味，這是由於兒茶素含量相對較高，茶氨酸水平比青心烏龍低。

* **茶湯顏色：** 輕發酵時— 蜜綠透金黃（蜜綠透金黃, mìlǜ tòu jīnhuáng），透明純淨。中度發酵時— 橙黃明亮（橙黃明亮, chénghuáng míngliàng）。

* **葉底（沖泡後葉片）：** 完整有彈性的肥厚葉片，具有特有的「綠葉紅鑲邊」圖案— 綠橄欖色中心和紅褐色氧化邊緣。葉片肥厚軟亮（肥厚軟亮, féihòu ruǎnliàng），經過幾次沖泡後充分展開。

**7. 化學成分：**

* **多酚類（茶多酚, chá duōfēn）：** 總含量為乾重的15–25%。兒茶素總含量約101–121毫克/克（根據臺灣茶業改良場數據），與金萱（台茶12號）相當，略低於青心烏龍（約124毫克/克）。主要兒茶素：表沒食子兒茶素沒食子酸酯（EGCG）、表兒茶素沒食子酸酯（ECG）、表兒茶素（EC）。四季春多酚類顯示高抗氧化活性。

* **氨基酸：** 游離氨基酸總含量≥ 4.2%（春茶）。主要成分— L-茶氨酸（L-茶氨酸, L-chá ānjīsuān），提供口感中的鮮味和鮮味感。四季春的茶氨酸含量明顯低於青心烏龍（青心烏龍），這是由於低海拔種植園陽光充足，導致光合作用過程中茶氨酸加速分解為兒茶素。

* **生物鹼：** 咖啡鹼（咖啡鹼, kāfēi jiǎn）— 乾重的2–4%。沖泡茶湯中約25–55毫克/100毫升（取決於濃度和沖泡方法），約為等量咖啡中咖啡因含量的四分之一。還含有微量可可鹼和茶鹼。

* **維生素：** 維生素C（抗壞血酸）、B群維生素（B₁、B₂、葉酸）。輕發酵批次維生素C含量較高。

* **礦物質：** 鉀、鎂、錳、氟、鐵、鋅— 生理意義的微量含量。名間土壤中鐵和鎂含量高反映在茶葉的礦物質特徵中。

* **精油（揮發性芳香化合物）：** 決定四季春特有的梔子花香。揮發性成分的主要組成：芳樟醇、香葉醇、橙花醇、苯乙酸酯、水楊酸甲酯。正是揮發性物質的獨特平衡使四季春即使在盲品中也容易識別。

* **特點：** 四季春化學特徵的特點是兒茶素含量相對較高，氨基酸水平中等。這提供更「響亮」、清爽的口感，但長時間浸泡可能出現苦澀味。

**8. 健康功效：**

* **抗氧化保護：** 茶多酚含量高確保中和自由基。一些臺灣資料顯示，四季春多酚類的抗氧化效力比維生素E高數倍。

* **提神效果：** 咖啡因和L-茶氨酸的結合提供溫和但持久的提神作用— 提高注意力和工作能力，沒有急劇的「咖啡因峰值」和隨後的下降。

* **代謝支持：** 多酚類促進脂肪分解和加速代謝過程。四季春在臺灣傳統上被認為是適合日常飲用的「輕」茶，作為均衡飲食的一部分。

* **消化幫助：** 有機酸和多酚類刺激消化酶分泌，幫助消化食物。建議餐後飲用；空腹飲用濃茶可能刺激胃黏膜。

* **心血管支持：** 抗氧化成分有助於降低氧化膽固醇水平並維持血管彈性。

* **利尿作用：** 適量咖啡因有助於排出多餘水分和減少水腫。

* **免疫增強：** 茶多酚具有抗菌活性並支持機體免疫功能。

* **心理舒適：** L-茶氨酸促進放鬆而不困倦，降低壓力荷爾蒙水平。四季春冷泡— 獲得溫和舒緩飲品的流行方法，單寧提取最少。

**9. 沖泡方法：**

* **水溫：** 熱泡90–95°C。冷泡（冷萃）— 冷水（~5°C），在冰箱中浸泡4–5小時。

* **茶量：** 功夫茶法：8克/200毫升（茶水比1:25）。日常方法：3–5克/250–300毫升。冷泡：5克/1000毫升。

* **茶具：** 蓋碗（蓋碗, gàiwǎn）瓷器— 展現花香特徵的最佳選擇。宜興紫砂壺（宜興紫砂壺, Yíxīng zǐshā hú）也適合，雖然對於輕烏龍茶瓷器更佳，因為不吸收香氣。玻璃茶具— 觀察葉片展開的好選擇。

* **過程（功夫茶法）：**
    1. 用沸水溫熱蓋碗和茶杯。
    2. 在200毫升蓋碗中放入8克茶。
    3. 用兩次快速沖洗（各5秒）洗茶— 這「喚醒」葉片。
    4. 第一泡：用95°C水沖泡，浸泡45秒，倒出。
    5. 通過茶濾或公道杯（公道杯, gōngdào bēi）將茶湯分到茶杯中。
    6. 後續沖泡：每次增加10秒浸泡時間。
    7. 優質四季春可沖泡5–7次。

* **冷泡：** 5克茶葉加1000毫升冷水，放入冰箱4–5小時。此方法減少單寧和咖啡因提取，產生特別純淨、甜美清爽的飲品，花香明亮— 在炎熱天氣中展現四季春特性的最佳方法之一。

**10. 儲存：**

四季春作為輕烏龍茶，對外界影響敏感，需要小心儲存：

* **容器：** 密封、避光包裝— 多層鋁箔真空袋或密封蓋鐵罐。
* **溫度：** 長期儲存建議冷藏（5–10°C），特別是清香型批次。烘焙版本（濃香型）要求較低，可室溫儲存。
* **茶葉天敵：** 濕氣、熱量、異味和直射光— 香氣和口感劣化的主要因素。
* **期限：** 新鮮四季春最好在6–12個月內飲用。臺灣資料建議新茶在陰暗處放置約15天以「褪火」（褪火, tuìhuǒ）最終乾燥後的火氣，開封後72小時內使用以保持最大香氣。
* **不適合陳放：** 與深色烏龍茶或普洱茶不同，四季春不會隨年份改善。其價值在於新鮮度。

**11. 價格與仿品：**

* **價格類別：** 四季春屬於最易取得的臺灣烏龍茶。價格範圍明顯低於高山茶（阿裡山、梨山、大禹嶺）。價格取決於採收季節（春冬茶較貴，夏秋茶較便宜）、採收方法（手工較貴，機械較便宜）、具體農場和年份。臺灣特級品質批次參考價格— 從600元（≈ 2700盧布）/斤（500克）起；大宗批次價格明顯更低。

* **如何避免仿品：**
    * **向專業銷售商購買**，提供透明的產地和農場資訊。
    * **評估乾茶香氣：** 正宗四季春具有純淨明亮的梔子花香— 無化學香精味和異味。缺乏明顯花香或「人工」感— 警告信號。
    * **檢查外觀：** 顆粒應均勻捲曲、緊實，無過多莖梗和茶末。
    * **測試茶湯：** 顏色純淨透明，無混濁。口感清新滑順，有回甘，無酸敗或陳味。
    * **對異常低價保持警惕：** 過分便宜的「四季春」可能由低質原料製成，或為中國大陸產品（該品種自1988年也在大陸種植）冒充臺灣茶。

**12. 有趣事實：**

* **茶名中的農民姓名。** 在木柵地區和臺灣老一代茶師中，此茶有時仍被稱為「輝仔茶」（輝仔茶）— 「輝的茶」，以發現者張文輝命名。這是少數在非正式名稱中保留具體人名的臺灣茶之一。

* **冷泡之王。** 四季春被認為是冷萃法的最佳茶之一：冷浸泡時其花香展現特別純淨明亮，苦澀味幾乎不出現。正是以冷飲形式首次征服了臺灣大眾消費者。

* **珍珠奶茶產業引擎。** 2000年代臺灣手搖茶飲（手搖茶飲）產業的爆炸性增長很大程度上依賴四季春作為基礎原料。高產量、低價格和明亮香氣「穿透」牛奶和糖的特性，使其成為奶茶和果茶飲品的理想基礎。

* **不要與金萱混淆。** 某些資料錯誤地將四季春與「臺茶」12號（台茶12號），即金萱（金萱, Jīnxuān）等同。這是完全不同的品種：金萱是TTES正式註冊的選育品種，具有明顯的奶香，而四季春是自然起源的地方品種，以梔子花香為主。

* **「四季」非誇張。** 在名間溫和亞熱帶氣候條件下，該品種確實不進入休眠狀態，繼續全年發芽，提供6–8次採收— 這是世界茶樹品種中的記錄之一。

**13. 與其他臺灣烏龍茶比較：**

* **阿裡山烏龍（阿裡山烏龍, Ālǐshān Wūlóng）：** 高山烏龍茶（1000–1500公尺），主要由青心烏龍品種製成。更複雜、油潤深厚的口感，帶奶香和果香調。茶湯更濃厚「厚重」。四季春更輕盈、明亮、花香，價格明顯更易接受。

* **凍頂烏龍（凍頂烏龍, Dòngdǐng Wūlóng）：** 中海拔烏龍茶（500–800公尺），中重度烘焙。口感特徵— 堅果焦糖味，蜜甜和明顯回甘。凍頂在特性上明顯更深厚「溫暖」，而四季春清新「涼爽」。

* **金萱（金萱, Jīnxuān，臺茶12號）：** 臺灣選育品種，以天然奶香著稱。口感溫和奶香，無明顯花香。四季春和金萱常在名間同一種植園種植，但產生完全不同的香氣特徵：四季春的梔子花和野花對比金萱的奶香溫和。

* **文山包種（文山包種, Wénshān Bāozhǒng）：** 輕氧化臺灣烏龍茶，縱向捲曲（非半球形）。包種更優雅精緻，具有鈴蘭和水仙的細膩香氣。四季春在花香上更「響亮」直接，同時更易理解。

* **翠玉（翠玉, Cuìyù，臺茶13號）：** 選育品種，具有特有的茉莉玉蘭香。特徵比四季春更精緻「溫柔」。兩種茶都屬於中低海拔帶，常在同一價格段競爭，但香氣不同：翠玉的茉莉對比四季春的梔子花。

**14. 四季春品種與等級：**

**按採收季節：**
* **春茶（春茶, chūnchá，3–4月）：** 一芽一葉，梔子花香明顯，口感清新明亮。被認為是最佳季節。
* **冬茶（冬茶, dōngchá，11–12月）：** 葉片較厚，多糖含量高，「冷香」和甘蔗糖甜味。第二珍貴季節。
* **夏秋採收：** 主要用於商業批次和茶飲。口感較簡單，澀味較高。

**按等級：**
* **特級（特級, tèjí）：** 一芽二葉比例≥ 95%。顆粒緊實，顏色墨綠帶沙色調。梔子花香強烈持久穿透。價格從600元/斤起。
* **一級（一級, yī jí）：** 以一芽二葉為主。香氣純淨，茶湯蜜黃透明。
* **二級（二級, èr jí）：** 混合採收，包括夏秋葉。口感純淨但較不複雜，耐泡性較低。

**結論：**

四季春是一款矛盾的茶：作為最簡單易得的臺灣烏龍茶之一，它同時具有驚人的表現力和令人難忘的特性。其梔子花香— 明亮、歡快、幾乎大膽— 無法與任何其他茶混淆，正是這種「花香直率」為它贏得了數百萬人的喜愛，從臺灣街頭茶攤到歐洲茶葉精品店。對於剛開始接觸臺灣烏龍茶的人來說，四季春是理想的入門點：沖泡簡單，容錯性高，冷泡效果極佳，同時誠實地展示了什麼是臺灣烏龍茶。對於經驗豐富的茶愛好者，來自名間的優質春茶四季春提醒我們，茶的特性不僅由種植園海拔和價格決定，還由品種、風土和工藝的成功結合決定— 那種神奇的巧合，臺灣人詩意地稱之為「永恆之春」。
