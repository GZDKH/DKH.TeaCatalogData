# 桂花香單叢 (桂花香单丛, guìhuā xiāng dāncóng)

**桂花香單叢 (桂花香单丛, guìhuā xiāng dāncóng)**
![Гуй Хуа Сян Даньцун (桂花香单丛, guìhuā xiāng dāncóng)](images/Гуй%20Хуа%20Сян%20Даньцун%20(桂花香单丛,%20Guìhuā%20Xiāng%20Dāncóng).png)

![Гуй Хуа Сян Даньцун (桂花香单丛, guìhuā xiāng dāncóng)-1](images/Гуй%20Хуа%20Сян%20Даньцун%20(桂花香单丛,%20Guìhuā%20Xiāng%20Dāncóng)-1.png)

桂花香單叢是鳳凰單叢十大經典香型 (十大香型, shí dà xiāngxíng) 之一，其乾茶和茶湯散發出令人驚嘆的秋日桂花林香氣的精確複製。無需添加任何花朵成分——純粹的風土、遺傳學和製茶師技藝的煉金術——這款茶重現了桂花金色花朵甜蜜蜂蜜般、略帶辛香的氣息，如此逼真，以至於初次接觸時幾乎總是引起懷疑。然而這並非香料茶，而是真正的單叢——「孤獨的茶樹」，按照「一樹一味」的原則種植、採摘和加工。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶，青茶, qīngchá）。氧化程度——中度和中高度，大約30–50%，根據製作者風格進行不同強度的最終焙火 (焙火, bèihuǒ)。
* **類別：** 廣東烏龍茶。鳳凰單叢 (凤凰单丛, Fènghuáng Dāncóng) ——十大經典花蜜香型 (十大花蜜香型, shí dà huā mì xiāngxíng) 之一。屬於珍貴名叢 (珍贵名丛, zhēnguì míngcóng) 之列。
* **產地：** 中國廣東省 (广东省, Guǎngdōng shěng) 潮州市 (潮州市, Cháozhōu shì) 潮安區 (潮安区, Cháo'ān qū) 鳳凰鎮 (凤凰镇, Fènghuáng zhèn)，鳳凰山 (凤凰山, Fènghuáng shān) 山脈。桂花香 (桂花香) 母樹生長在烏岽管區 (乌岽管区, Wūdǒng guǎnqū) 李仔坪村 (李仔坪村, Lǐzǎipíng cūn) 的茶園中——鳳凰茶區的高山核心地帶。2010年鳳凰單叢獲得地理標誌產品 (地理标志产品, dìlǐ biāozhì chǎnpǐn) 地位。
* **地理座標：** 約北緯23°52′，東經116°43′（烏岽山，鳳凰山頂峰）。

**2. 歷史與文化意義：**

* **歷史：** 鳳凰單叢的歷史與南宋末代皇帝趙昺 (赵昺, Zhào Bǐng) 的傳說密不可分，據說他在十三世紀末逃避蒙古征服者南下時，曾在烏岽山停留。當地居民向口渴疲憊的皇帝獻上野生茶樹紅葉的茶湯；這飲品解了渴並恢復了體力，皇帝隨後將此樹命名為「宋種」(宋种, Sòng zhǒng——「宋朝品種」)。傳說還說茶是由「鳳凰鳥」(凤凰, 鳳凰) 帶來的，因此有了第二個歷史名稱——「鳥嘴茶」(鸟嘴茶, niǎozuǐ chá——「鳥嘴茶」)。

單叢選育的真實歷史始於清朝。在同治和光緒皇帝統治期間（1875–1908），鳳凰茶農從集體生產轉向「單株採製」(单株采制, dānzhū cǎizhì) 方法——對傑出單株樹木進行個別採摘和加工。每棵樹都被賦予自己的名字。當時鳳凰山上有超過10,000棵樹按個別方案處理——正是這種方法給整個類別命名為「單叢」(单丛,「孤獨的茶樹」)。

桂花香單叢母樹生長在烏岽管區李仔坪村。根據調查結果，其樹齡超過300年（2012年數據）。這棵祖先樹是鳳凰山上現存最古老的桂花香 (桂花香) 克隆之一。1958年森林火災摧毀了烏岽的大部分茶園，但母樹倖存了54株幼苗，這些幼苗被培育到單叢品質標準，形成了現代種群的基礎。現在這個克隆的代表——烏岽桂花 (乌岽桂花)、金桂花 (金桂花,「金桂花」) 等——保持著母樹的遺傳特徵。

1996年，潮州的茶農和學者正式確立了鳳凰單叢的分類，確定了十大主要香型。桂花香與蜜蘭香 (蜜兰香)、黃梔香 (黄栀香)、玉蘭香 (玉兰香)、芝蘭香 (芝兰香)、夜來香 (夜来香)、肉桂香 (肉桂香)、杏仁香 (杏仁香)、柚花香 (柚花香) 和薑花香 (姜花香) 一同進入這一經典。

* **名稱：** 「桂花」(桂花, guìhuā) ——桂花 (*Osmanthus fragrans*) 的花朵，中國文化中最受尊崇的香花植物之一。桂 (guì) ——「肉桂樹」、「桂花」；花 (huā) ——「花朵」。「香」(香, xiāng) ——「香氣」。「單叢」(单丛, dāncóng)：單 (dān) ——「單獨的」；叢 (cóng) ——「叢、群」。全名意為「具有桂花香氣的單叢」。茶之所以如此命名，是因為其茶湯自然重現了盛開桂花特有的甜蜜辛香氣息——無需任何香料添加。

* **文化意義：** 桂花香單叢在十大經典香型中佔據特殊地位：如果蜜蘭香是最普遍和易得的，鴨屎香 (鸭屎香,「鴨糞」) 是最時尚的，那麼桂花香則是最精緻和文學性的之一。桂花在中國文化中與秋天、滿月和詩意孤獨密不可分：其香氣是「四大香花」(四大香花, sì dà xiānghuā) 之一，與蘭花、梅花和蓮花並列。能夠在沒有一片花瓣的情況下重現這種香氣的茶，被潮州茶師視為「天然勝於人工」原則的最高體現。在傳統潮州工夫茶 (工夫茶, gōngfu chá) 中，鳳凰單叢——包括桂花香——是招待客人的主要茶品；這一儀式已被列入中國非物質文化遺產。

**3. 植物學描述與原料：**

* **品種/栽培種：** *Camellia sinensis* var. *sinensis*，鳳凰水仙 (凤凰水仙, Fènghuáng Shuǐxiān, 華茶17號, 华茶17号) 群體。桂花香代表這一多形性群體的克隆選擇 (无性株系, wúxìng zhūxì)。生活形式——半喬木型 (小乔木型, xiǎo qiáomù xíng)，大葉類 (大叶类, dàyè lèi)，中生種 (中生种, zhōngshēng zhǒng)。葉片橢圓形，黃綠色，具明顯光澤；鋸齒稀疏而尖銳。芽和嫩梢淡黃綠色，絨毛稀少。
* **採摘：** 中生種類型——採摘在清明 (清明, Qīngmíng——「清明」，通常4–5月4–5日) 後4–5天，與其他中生種克隆同時：柚花香 (柚花香)、薑花香 (姜花香)、杏仁香 (杏仁香)。最佳採摘時間——晴天，13:00至16:00，當散射的午後光線為後續萎凋創造理想條件。
* **採摘標準：** 當梢頂形成「駐芽」(驻芽, zhùyá) ——成熟信號時。採摘每梢2–5葉 (嫩对夹叶, nèn duìjiā yè)。原則上重要的是不要採摘過嫩的葉子（苦味過多，缺乏茶體）也不能過熟（粗糙，失去香氣）。只能手工採摘——機械採收會損害葉片完整性並破壞後續發酵。
* **原料要求：** 完整、未損傷的梢，葉片發育均勻。單叢在原則上不同於其他烏龍茶的「一樹一味」方法：每棵樹（或克隆群）分別採摘和加工，以保持獨特特性。不同味道樹木的原料混合對單叢級產品是不可接受的；降低標準時茶葉轉入「浪菜」(浪菜, lángcài) 或「水仙」(水仙, shuǐxiān) 類別。

**4. 風土與栽培特點：**

* **地區和地形：** 鳳凰山山脈——陡峭山坡，覆蓋亞熱帶森林，茶園散佈在岩石和樹木之間。綠化覆蓋率達96.4%，森林覆蓋率85.1%。最高點——烏岽山 (乌岽山, Wūdǒng shān)，約1,498公尺。桂花香母樹所在的李仔坪村位於烏岽上部。
* **生長海拔：** 800–1,200公尺的高山原料為頂級品質。海拔越高——「山韻」(山韵, shānyùn) 越明顯——那種難以捉摸但無誤可辨的礦物蜂蜜調，區別真正的鳳凰單叢與平原仿製品。
* **氣候：** 亞熱帶季風海洋性氣候。年平均溫度約20°C，年降水量約1,800毫米。海拔1,000公尺以上——頻繁的霧和雲，顯著的日夜溫差，豐富的露水。這些條件減緩植物生長，有利於香氣前體物質和氨基酸的積累。
* **土壤：** 深厚、排水良好、酸性（pH 4.5–5.5）。烏岽以黃色紅土為主，有機質和微量元素含量高。局部出現特有的「鴨屎土」(鸭屎土, yāshǐ tǔ) ——含白堊（礦物鹽）夾雜物的黃色粘土，這給了著名的鴨屎香其名字。

**5. 製作工藝：**

鳳凰單叢按獨特的廣東工藝加工，結合深度半發酵與每株個別處理。桂花香工藝要求在碰青 (碰青, pèngqīng) 和發酵控制時特別精細——正是這些階段決定茶湯中是否出現特有的桂花香型。

* **採摘 / 采摘 — cǎizhāi：** 手工採摘帶駐芽的梢，在晴天午後時間。
* **日光萎凋 / 晒青 — shàiqīng：** 將採摘的葉子薄攤在竹篩上，置於散射午後光線下。持續時間——根據陽光強度20至40分鐘。葉片失去初始水分，顏色從鮮綠轉為暗橄欖綠，梢尖下垂。不能過度乾燥或燒傷——萎凋品質直接影響香氣純淨度。
* **陰涼萎凋 / 凉青 — liángqīng：** 將葉子移至陰涼遮蔭處平衡溫度，重新分配葉脈和葉片間的水分。這是「被動調和」階段，為活躍階段準備原料。
* **做青 / 做青 — zuòqīng (碰青 — pèngqīng + 静置 — jìngzhì)：** 關鍵且最複雜的階段。廣東傳統使用「碰撞」(碰青) 方法——在竹籃或竹篩中手工拋撒和搖動葉片，與靜置期 (静置) 交替。機械作用破壞葉緣細胞，釋放多酚氧化酶並啟動氧化，但葉片中心保持綠色——因此有「青蒂綠腹紅鑲邊」(青蒂绿腹红镶边, qīngdì lǜfù hóng xiāngbiān) 的公式。對於桂花香，師傅力求達到那種精細的氧化平衡，使桂花調得以顯現——芳樟醇 (芳樟醇，沉香醇) 及其氧化物、β-紫羅蘭酮 (β-紫罗兰酮，β-紫羅蘭酮) 和順式茉莉酮 (顺式茉莉酮，順式茉莉酮) 痕跡——這些是桂花香氣和這一獨特茶樹克隆的特徵成分。循環次數和持續時間——每位師傅的秘密。
* **殺青 / 杀青 — shāqīng：** 在鍋或滾筒中高溫處理。停止酶氧化並固定達到的香氣特徵。
* **揉捻 / 揉捻 — róuniǎn：** 縱向揉捻，形成廣東單叢特有的緊實、筆直、厚重條索 (条索, tiáosuǒ)。不同於閩南烏龍（顆粒）或閩北烏龍（帶狀），廣東風格正是筆直的「繩狀」。
* **解塊 / 松團 — sōngtuán：** 鬆散揉捻團塊以防過熱並確保均勻乾燥。
* **乾燥 / 烘干 — hōnggān：** 在竹篩上用炭火或電烘乾機初步乾燥。
* **分揀 / 分拣 — fēnjiǎn：** 去除粗梗、黃葉和碎片。
* **復焙 / 复焙 — fùbèi：** 最終焙火——將茶「調整」到所需成熟度。輕焙保持花香新鮮；較深焙火增加蜂蜜、焦糖調並提高儲存穩定性。桂花香多採用中度焙火，保持特有的「透明度」和桂花香氣的「香水感」，不被炭火調掩蓋。焙火後茶葉靜置約15天「退火」(退火, tuìhuǒ) 並調和口味。

**6. 感官特徵：**

* **乾茶外觀：** 緊結、筆直、緊實的條索 (条索紧卷, tiáosuǒ jǐnjuǎn)，均勻厚重。顏色——「鱔魚皮色」(鳝鱼皮色, shànyú pí sè) ——特有的黃褐色帶橄欖光澤和油潤光澤，在較深焙火時轉為深褐色 (乌褐色, wūhè sè)。有時可見「朱砂紅點」(朱砂红点, zhūshā hóng diǎn) ——氧化葉緣的痕跡。
* **乾茶香氣：** 純淨、優雅、無誤可辨的桂花 (桂花, guìhuā) 香氣。甜蜜、蜂蜜般，帶輕微辛香和精緻果香底調——讓人聯想到在秋日陽光下乾燥的金色桂花。尾韻——溫暖、「粉狀」，帶成熟杏子調。
* **茶湯香氣：** 前幾泡——明亮、「香水般純淨」的桂花調：金色甜蜜、蜂蜜柔和、杏子和成熟梨的調子。隨著展開——更深層次：花香辛香、檀香細調、若隱若現的香草。最後幾泡——純淨蜂蜜甜蜜帶礦物「山」底調。
* **口感：** 中等茶體，油潤光滑。第一印象——絲綢般甜蜜帶花香「透明度」。中段——飽滿，帶蜂蜜、成熟杏子、輕微辛香調。苦澀味極少且優雅融合。回甘 (回甘, huígān) ——持久純淨，帶特有「山韻」(山韵蜜味, shānyùn mì wèi)：在口腔深處感受到的礦物蜂蜜調。唇舌長時間保持餘香 (唇舌留香, chúnshé liúxiāng)。
* **湯色：** 橙黃至金琥珀色 (橙黄明亮, chénghuáng míngliàng)，清澈透明，帶溫暖蜂蜜色調。
* **葉底（沖泡後葉片）：** 經典單叢公式：「青蒂綠腹紅鑲邊」(青蒂绿腹红镶边, qīngdì lǜfù hóng xiāngbiān)。葉片完整、柔軟、有彈性，中心均勻橄欖綠，邊緣紅褐色氧化邊。葉柄——淺綠色。

**7. 化學成分：**

* **多酚類：** 鳳凰單叢以高總多酚含量著稱——佔乾重22.6–39.1%。主要成分：兒茶素（部分氧化）、茶黃素和茶紅素，形成茶體密度和特有「山」澀味。類黃酮含量——8.3–14.1%。
* **氨基酸：** 總含量——佔乾重1.15–2.96%。L-茶氨酸提供口感柔和並與咖啡因平衡。高山原料（烏岽）因天然霧氣遮蔭而氨基酸含量較高。
* **生物鹼：** 咖啡因——佔乾重2.3–5.3%（顯著差異由克隆和生長條件變化解釋）。可可鹼、茶鹼——痕量。
* **維生素：** 維生素B₁、B₂、C、E——烏龍茶標準含量。維生素C在焙火時部分破壞，但中度烘焙保留顯著部分。
* **礦物質：** 鉀、鎂、錳、鋅、氟、硒。礦物質特徵強烈依賴具體地塊；烏岽高山土壤富含微量元素。
* **精油：** 桂花香的關鍵方面。氣相色譜-質譜分析（GC-MS）顯示沉香醇 (芳樟醇, fāng zhāngnǎo) 及其氧化物、順式茉莉酮 (顺式茉莉酮, shùnshì mòlì tóng)、法呢烯 (法呢烯, fǎní xī) 和新二萜烯的主導地位——這是桂花香氣特徵的組合。正是這些成分與桂花 *Osmanthus fragrans* 香氣化學（β-紫羅蘭酮、沉香醇、沉香醇氧化物、順式-3-己烯醇）的一致性解釋了香氣的驚人相似性。
* **獨特特點：** 鳳凰單叢水浸出物含量極高——佔乾重35.6–49.4%，這解釋了其多次沖泡能力和茶湯密度。

**8. 保健功效：**

* **溫和特性的提神效果：** 高咖啡因含量與L-茶氨酸結合提供無緊張感的清醒——特有的「單叢」活力：清晰專注、平靜能量。
* **抗氧化保護：** 豐富的多酚複合物（兒茶素、茶黃素、類黃酮）——強效自由基中和劑。單叢多酚含量是烏龍茶中最高的之一。
* **消化支持：** 中度發酵烏龍茶傳統上被認為對胃「友好」。在潮州工夫茶傳統中，豐盛餐後供應正是為了幫助消化。
* **降脂作用：** 烏龍茶多酚能抑制胰脂肪酶活性並減少膳食脂肪吸收——定期飲用效果最明顯。
* **增強免疫力：** 類黃酮和兒茶素具有抗菌和抗病毒特性。
* **皮膚健康支持：** 抗氧化劑和維生素E有助於保護皮膚細胞免受氧化應激。
* **護眼和減輕眼疲勞：** 茶中B族維生素和抗氧化劑有助於減少視覺疲勞。
* **冥想實踐：** 鳳凰單叢因其香氣複雜性和持久沖泡系列，非常適合正念茶飲——這種實踐降低壓力水平並促進情緒調節。

**9. 沖泡方法：**

* **水溫：** 95–100°C。鳳凰單叢需要熱水充分釋放香氣潛力。沸水（100°C）——標準。
* **茶量：** 7–8克配100–120毫升（潮州工夫）；5克配150毫升（蓋碗）。傳統潮州方法要求慷慨投茶——茶葉填滿茶壺⅔–¾體積。
* **茶具：** 經典——潮州紫砂壺 (潮州壶, Cháozhōu hú) 或薄壁瓷蓋碗 (盖碗, gàiwǎn)。小茶杯 (若琛杯, Ruòchēn bēi) ——潮州工夫必備。瓷蓋碗適合初次品嚐：不「偷」香氣，能評估花香特徵純淨度。
* **過程：**
  1. 燒開水並充分溫熱所有茶具——茶壺、茶海和茶杯。
  2. 投茶，蓋蓋，搖一兩次並聞蓋香——第一印象往往最鮮明。
  3. 洗茶：注沸水，立即倒出。它「喚醒」茶葉並去除茶末。
  4. 第一泡：5–10秒（慷慨投茶時）。茶湯立即通過茶海分入茶杯。
  5. 後續沖泡：10–15泡及以上，每泡增加5秒。優質老樹桂花香可沖泡15泡以上，每泡都提供新層次——從明亮花香通過蜂蜜飽滿到純淨礦物甜蜜。
  6. 「潮州規則」：前三杯——香氣最強，中間——口感最深，最後——回味最純。

**10. 儲存：**

* **容器：** 密封鐵罐、陶瓷茶罐或鋁箔真空包裝。潮州傳統常用錫罐 (锡罐, xīguàn) ——提供優異的防潮和防異味保護。
* **條件：** 乾燥、陰涼、避光處。輕焙火單叢——可冷藏（0–5°C）密封包裝。中深焙火單叢無需冷藏——室溫（15–25°C）穩定濕度即可。
* **保質期和陳化：** 新鮮桂花香在前6–12個月最佳。但好的焙火茶能陳化：一年儲存後「火氣」消退，顯現更深、「成熟」的蜂蜜甜蜜。一些收藏家陳放單叢3–5年，定期「復火」輕焙。
* **茶敵：** 濕氣、異味（單叢極易吸濕吸味）、直射陽光、劇烈溫差。

**11. 價格與仿冒：**

* **價格類別：** 桂花香屬鳳凰單叢中高價位段。2003年春茶單株可達每公斤9,600元。決定價格因素：樹齡（老叢, lǎocóng——顯著更貴）、生長海拔（烏岽——頂級；山腳——較低）、採摘季節（春茶——最貴）、個別加工（單叢——比浪菜或水仙貴）、焙火技藝。
* **避免仿冒方法：**
  * 從潮州或鳳凰有透明鏈條的商家購買：地區→海拔→克隆→師傅。
  * 評估外觀：真正桂花香——均勻、緊實、光亮「鱔魚色」條索；粗糙、暗淡、大小不一——低等級標誌。
  * 檢查香氣：天然桂花調——純淨、優雅、不刺鼻；人工香料——刺激、「合成香水感」、快速消散。
  * 茶湯：真正單叢——透明、金琥珀色，多泡持久香氣；仿品——略渾濁，香氣2–3泡後消失。
  * 檢查回味：「山韻」(山韵) ——口腔深處礦物蜂蜜調——真正鳳凰產地標誌；香料無法複製。

**12. 有趣事實：**

* 桂花香母樹經歷了1958年毀滅性森林火災——整個地塊僅倖存54株幼苗，成為現代種群祖先。這種「遺傳瓶頸」使每棵桂花香 (桂花香) 樹成為獨特基因庫的珍貴保存者。
* 鳳凰單叢是唯一將「一樹一茶」原則發揮到極致的烏龍茶。製茶師了解地塊每棵樹的特性，根據當季「狀態」加工原料。任何偏離個別方法都會降低產品等級。
* 桂花香香氣如此精確地重現真正桂花氣味，即使經驗豐富的茶專家在盲品時也會懷疑香料添加。氣相色譜分析證實這款茶的關鍵香氣成分（沉香醇及其氧化物、順式茉莉酮、法呢烯）與桂花 *Osmanthus fragrans* 精油相同。
* 到2020年代，鳳凰山上有超過80個註冊克隆系 (品系, pǐnxì) 單叢，其中10個經典香型和25個以上葉形變種。茶農承認：獨特克隆總數如此龐大，即使當地師傅也無法全部列舉。
* 潮州工夫茶有「關公巡城」(关公巡城, Guāngōng xúnchéng——「關公巡城」) 規則：從茶壺連續循環倒茶到所有茶杯，確保茶湯濃度均勻。最後幾滴——「韓信點兵」(韩信点兵, Hánxìn diǎn bīng——「韓信點兵」)：逐滴分配到每杯，因為最後幾滴最香濃。

**13. 與其他單叢比較：**

* **蜜蘭香 (蜜兰香, Mì Lán Xiāng)：** 最普遍和知名類型。蜂蜜蘭花香——更開放、「寬廣」和甜蜜。桂花香——更內斂、「粉狀」，帶桂花特有的辛香深度和精緻優雅。
* **黃梔香 (黄栀香, Huáng Zhī Xiāng)：** 梔子花 (栀子花) 香——明亮、高調、略尖銳。桂花香更柔和「溫暖」，蜂蜜底調更明顯。黃梔香是最「響亮」的單叢之一；桂花香是最「安靜」和精緻的之一。
* **玉蘭香 (玉兰香, Yù Lán Xiāng)：** 玉蘭花香——純淨、清新、「白花」調。桂花香——更溫暖辛香；玉蘭香——更清涼「透明」。兩種都優雅，但玉蘭香是「春天」，桂花香是「秋天」。
* **肉桂香 (肉桂香, Ròuguì Xiāng)：** 肉桂香——辛香、溫暖、「香料」感。不要與同名武夷品種 (武夷肉桂) 混淆。肉桂香單叢——更「深沉」辛辣；桂花香——更花香甜蜜。
* **薑花香 / 通天香 (姜花香 / 通天香, Jiānghuā Xiāng)：** 薑花香——明亮、「穿透」，帶輕微辛辣。最「響亮」的單叢。桂花香——完全相反：不響亮、柔和、包圍感。

**結語：**

桂花香單叢——不會吶喊而是低語的茶。在其不響亮但完美純淨的桂花香中——蘊含鳳凰傳統的全部深度：烏岽霧山上的三百年古樹、了解每個芽頭特性的師傅，以及竹籃中茶葉慢慢展現天賦本質。第一泡散發金色秋夜香氣，中間幾泡——溫暖蜂蜜和成熟杏子，最後幾泡——山石純淨礦物甜蜜。對於那些厭倦「響亮」茶品、尋求安靜但深刻美感的人，桂花香將提供正是其同名桂花所著稱的：不強加但一旦認識就永不忘記的香氣。
