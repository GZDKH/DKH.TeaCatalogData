# 保靖黃金紅茶 (保靖黄金红茶, Bǎojìng huáng jīn hóngchá)

**保靖黃金紅茶 (保靖黄金红茶, Bǎojìng huáng jīn hóngchá)**

![Баоцзин Хуан Цзинь Хун Ча (保靖黄金红茶, Bǎojìng huáng jīn hóngchá)-1](images/Баоцзин%20Хуан%20Цзинь%20Хун%20Ча%20%28保靖黄金红茶,%20Bǎojìng%20Huáng%20Jīn%20Hóngchá%29-1.png)

保靖黃金紅茶是以傳奇品種保靖黃金茶 (保靖黄金茶, Bǎojìng Huángjīn Chá) 為原料製作的紅茶，這是一個來自湖南省西部湘西 (湘西) 山區深處的古老且基因獨特的品種。傳統上此原料用於製作綠茶，然而其極高的胺基酸含量（高達7.47%，是平均值的兩倍）和多酚含量使其成為製作紅茶的絕佳原料，在紅茶中品種的天然甜味得到特別鮮明的展現。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá) — 完全發酵（氧化）。
* **類別：** 地方中國紅茶，工夫紅茶 (工夫红茶, gōngfu hóngchá)。屬於「湖南紅茶」(湖南红茶, 「湖南紅茶」) 系列，為此制定了專門的技術規程——「保靖黃金茶工夫紅茶技術規程」(《保靖黄金茶工夫红茶技术规程》, 「保靖黃金茶工夫紅茶技術規程」)。
* **產地：** 中國湖南省 (湖南省, Húnán Shěng)，湘西土家族苗族自治州 (湘西土家族苗族自治州, Xiāngxī Tǔjiāzú Miáozú Zìzhìzhōu)，保靖縣 (保靖县, Bǎojìng Xiàn)。核心產區——葫蘆鎮 (葫芦镇, Húlu Zhèn)、夯沙鄉 (夯沙乡, Hāngshā Xiāng) 和水田河鎮 (水田河镇, Shuǐtiánhé Zhèn)。歷史「零公裡」——黃金村 (黄金村, Huángjīn Cūn)，位於呂洞山 (吕洞山, Lǚdòng Shān) 山腳。
* **地理座標：** 東經109°12′–109°33′，北緯28°24′–28°36′（根據地理標誌登記資料）。

**2. 歷史與文化意義：**

* **歷史：** 保靖的茶葉傳統可追溯至遠古時代。早期史書《荊州土地記》(《荆州土地记》) 中記載：「武陵七縣通出茶，最好」(武陵七县通出茶，最好)。唐代學者杜佑 (杜佑) 在百科全書式著作《通典》(《通典》, 西元801年) 中記錄，溪州 (溪州, 大致相當於現今保靖地區) 進貢茶芽作為貢品。更早的證據——裡耶 (里耶) 的發現，在36,000片秦簡 (秦简, 西元前三世紀) 中發現了來自遷陵 (迁陵, 現今保靖) 的經濟運輸記錄，推測包括茶葉原料。

關鍵傳說：西元1539年（明嘉靖十八年, 明嘉靖十八年），總督陸杰 (陆杰, Lù Jié) 巡視湘西山區邊防哨所時，他和隨從在魯旗 (鲁旗, 現今葫蘆) 附近的偏僻峽谷中感染了瘴氣 (瘴气, zhàngqì)。一位向姓 (向) 苗族老婦救了虛弱的士兵：她用自家門前百年茶樹的葉子泡茶給病人喝。半小時後熱病退去。陸杰感激地給老婦一兩 (两) 黃金，並將此茶列入宮廷貢品。從此民間流傳：「一兩黃金一兩茶」(一两黄金一两茶)。村莊得名黃金寨 (黄金寨, 「黃金要塞」)，茶葉被稱為黃金茶 (黄金茶, 「黃金茶」)。

近代史：1993–1994年農藝師張湘生 (张湘生, Zhāng Xiāngshēng) 實現突破——首次成功以扦插法 (扦插, qiānchā) 繁殖黃金茶，克服了多年的繁殖「瓶頸」。這為品種的大規模推廣開闢了道路。2009年文化遺產普查中發現2057棵古茶樹（樹幹周長超過30公分），其中718棵可追溯至明代，1339棵至清代。最古老的——「保靖黃金茶樹王」(保靖黄金茶树王) ——超過400年。2010年該茶獲得中華人民共和國農業部農產品地理標誌 (农产品地理标志) 保護。2020年黃金茶園被列入中國重要農業文化遺產 (中国重要农业文化遗产) 第五批名錄。同年品牌納入《中歐地理標誌協定》(中欧地理标志协定)，在27個國家獲得國際法律保護。2025年保靖黃金茶1號 (保靖黄金茶1号) 品種被列入國家骨幹型茶樹品種 (国家骨干型茶树品种) 名錄。

紅茶版本——黃金紅茶——在2010年代與綠茶版本並行積極發展，當時工夫紅茶技術在省級標準中正式化。

* **名稱：** 保靖 (Bǎojìng) — 縣名，字面意思「受保護的安寧」；黃金 (Huáng Jīn) — 「黃金」，指陸杰為神奇茶葉付出一兩黃金的傳說；紅茶 (Hóngchá) — 紅茶，指加工方式。

* **文化意義：** 保靖黃金茶不僅是農業文化，更是湘西苗族和土家族文化的活遺產。黃金村古茶園被宣布為省級文化遺產保護對象，「茶樹王」本身是湖南唯一的活體非物質文化遺產 (活体非物质文化遗产)。當地苗族尊崇「茶花仙子」(茶花仙子) 為守護神，茶文化主題貫穿傳統苗鼓 (苗鼓) 節奏和歌謠創作 (苗歌)。對保靖縣而言茶葉是主導經濟產業：截至2023年種植面積達15.5萬畝（≈ 10,300公頃），茶產業綜合產值231.6億元，品牌價值評估為409.3億元。

**3. 植物學描述與原料：**

* **品種/栽培種：** 保靖黃金茶 (保靖黄金茶) — 獨特的地方品種，在湘西偏僻山區條件下經長期自然選擇形成。屬於群體種 (群体种)，具有高度遺傳多樣性。根據何士華 (何士华) 教授研究，黃金茶古樹屬於喬木型大葉類品種 (乔木型大叶类品种)，與3500萬年前的景谷寬葉木蘭 (景谷宽叶木兰, *Magnolia latifolia*) 化石顯示一定遺傳親緣關係。中大型灌木，葉片橢圓形或披針形，嫩枝柔軟，芽密度高，對不良條件抗性強。關鍵植物學特徵——嫩芽中胺基酸含量極高。
* **採摘：** 春季為主要季節：明前 (明前, míngqián) 早春採摘特別珍貴。黃金茶特點是發芽早 (发芽早) 且芽葉密集、整齊。紅茶也使用夏季原料，多酚含量更高。
* **採摘標準：** 一芽一葉至一芽二葉 (一芽一叶 — 一芽二叶)。頂級批次使用單芽 (单芽)：製作一斤（500克）需要32,000–35,000次手工採摘。
* **原料要求：** 完整、嫩綠、新鮮的葉片，無損傷。手工採摘遵循「十不採」(十不采) 原則，包括拒絕採摘蟲害、過熟、露水濕潤等不合標準的嫩芽。

**4. 風土條件與栽培特點：**

* **海拔高度：** 海拔280–1500公尺。主要優質原料區——500–800公尺。黃金村位於海拔約635公尺。
* **氣候：** 亞熱帶季風山地氣候。年均溫15–17°C，夏季溫和無極端炎熱，冬季無嚴寒。雲霧幾乎是常伴：該地區位於雲貴高原與武當山交匯處。年降雨量1200–1500毫米。顯著的日夜溫差促進芳香化合物和胺基酸積累。
* **土壤：** 形成於母岩——頁岩和砂岩 (板页岩和砂岩)。弱酸性（pH 4.5–5.5），排水良好，深厚，有機質含量高。礦物組成——武陵山 (武陵山) 數億年地質過程的結果——為特有的風味「深度」做出貢獻。
* **生態：** 湘西地區生態環境極其純淨：遠離工業中心，森林覆蓋率高，山泉清澈。保靖茶園通過有機認證面積超過4200畝。許多種植園毗鄰原始森林，形成天然農林系統。

**5. 製作工藝：**

按工夫紅茶標準生產，並根據黃金茶原料高胺基酸含量特點進行調整：

* **採摘 (采摘, cǎizhāi)：** 手工選摘嫩芽。採摘時間——露水散去後的上午時段。
* **萎凋 (萎凋, wěidiāo)：** 自然或複合式。持續12–20小時，失水35–40%。目的——為揉捻準備葉片，啟動初步芳香轉化。對於高胺基酸含量的黃金茶，萎凋過程溫和，溫度適中，以保持甜味基礎。
* **揉捻 (揉捻, róuniǎn)：** 形成緊密捲曲並破壞細胞膜使汁液滲出到葉面。可手工揉捻 (手工揉捻) 或機械揉捻。
* **發酵/氧化 (发酵, fājiào)：** 關鍵步驟。溫度26–30°C，濕度90–95%，持續3–5小時。由於黃金茶原料多酚含量高（達25%）和胺基酸含量高（達7.47%），創造理想平衡：茶黃素提供亮度和活力，茶紅素提供深度和「體感」，未氧化胺基酸提供明顯天然甜味。
* **乾燥 (烘干, hōnggān / 干燥, gānzào)：** 初步乾燥在100–120°C停止氧化，然後在較溫和溫度（60–80°C）最終乾燥以穩定香氣特徵。
* **分級 (分级, fēnjí)：** 按葉片標準、芽頭比例、顆粒大小分類。

**6. 感官特徵：**

* **乾茶外觀：** 緊密捲曲，整齊的條索 (条索紧细)，深棕色或黑色，金毫 (金毫) 豐富。葉片均勻，分級良好。
* **乾茶香氣：** 甜美、蜂蜜味，帶有成熟南方水果香調和淡雅花香背景。表現出特有的「黃金香」(黄金香) ——複雜的甜蜜蜂蜜香氣，是品種的「名片」。
* **茶湯香氣：** 多層次且持久：頂調——蜂蜜和乾果（杏乾、棗子）；中調——焦糖、餅乾、麥芽；底調——精緻的木質香料調。每次沖泡香氣都在演變，展現新的層面。
* **滋味：** 濃厚、飽滿同時驚人地柔和。主導純淨、深沉的甜味 (回甘)，源於原料中創紀錄的胺基酸含量。澀味極少，快速消散於持久溫暖的回甘中。口感「油潤」(油润)，包覆性強。
* **湯色：** 紅琥珀色，明亮透澈，有明顯的金圈 (金圈, jīn quān) ——茶黃素含量高的標誌。
* **葉底（沖泡後葉片）：** 葉片舒展有彈性且均勻；顏色——銅棕色至紅栗色。結構完整，芽葉清晰可辨。

**7. 化學成分：**

* **胺基酸：** 黃金茶的主要「超能力」。新鮮春季原料中游離胺基酸含量達7.47%（某些資料顯示達7.76%），是普通綠茶的兩倍。以L-茶胺酸為主。完全氧化後大部分胺基酸得以保留，確保紅茶的特殊甜味和「鮮味」。
* **多酚：** 新鮮原料含量約20–25%。紅茶中以氧化形式為主：茶黃素（TF）和茶紅素（TR），形成湯色和「體感」。TF/TR平衡決定特有的「活潑」明亮度和同時的風味深度。
* **水溶性物質：** 高達50% — 極高指標，解釋了多次沖泡時湯色的濃郁和持久。
* **生物鹼：** 咖啡因 — 4.3%（原生原料資料）。可可鹼和茶鹼 — 微量。
* **葉綠素：** 含量比對照品種高50%，有助於提高光合作用活性，間接促進次級代謝物積累。
* **揮發性芳香化合物：** 豐富的萜烯、醛類和美拉德反應產物複合體。特有的「黃金香」由芳樟醇、香葉醇、β-紫羅蘭酮和特定胺基酸衍生物組合形成。
* **維生素和礦物質：** B群維生素、抗壞血酸（部分）、鉀、鎂、錳、鋅、硒。

**8. 健康功效：**

* **溫和提神與認知支持：** L-茶胺酸與咖啡因的高含量組合提供「放鬆專注」狀態——清醒無焦慮，改善記憶和注意力。
* **強效抗氧化保護：** 茶黃素、茶紅素和殘留兒茶素具有顯著中和自由基和降低氧化壓力的能力。就抗氧化活性而言，高多酚黃金茶原料製成的紅茶在中國紅茶中佔據高位。
* **代謝支持：** 多酚化合物和咖啡因刺激產熱並促進脂肪分解，使茶成為體重管理的良好伴侶。
* **消化舒適：** 溫和的紅茶傳統上建議餐後飲用。適度的單寧物質有助於胃液分泌正常化。
* **心血管健康：** 研究顯示茶黃素能夠支持血管彈性和血脂正常化。
* **免疫力增強：** 多酚具有適度抗菌和免疫調節作用。
* **延緩細胞老化：** 茶多酚的高抗氧化特性在研究中與延緩細胞年齡相關變化有關。
* **溫暖效應：** 完全氧化的茶在寒冷季節溫暖身體，緩解主觀疲勞感。

**9. 沖泡方法：**

* **水溫：** 90–95°C。
* **茶量：** 4–5克配100–120毫升。頂級芽茶批次稍少（3–4克），因為物質濃度更高。
* **茶具：** 瓷蓋碗 (盖碗) — 最精確展現香氣特徵；宜興紫砂壺 (宜兴紫砂壶) — 更加圓潤油滑；玻璃茶壺 — 欣賞金芽舒展「舞蹈」的美學享受。
* **過程：**
1. 用沸水溫熱茶具並倒掉水。
2. 投茶，蓋上蓋子輕搖 — 聞溫熱茶葉的「黃金」香氣。
3. 洗茶非必需；如需要 — 短暫（1–2秒）過水。
4. 第一泡：8–10秒。從第一杯就能感受到特有的蜂蜜果香甜味。
5. 2–4泡：10–15秒。
6. 從第5泡開始：增加5–10秒。
7. 優質批次可承受7–9次完整沖泡，以穩定性和「耐泡」(耐冲泡) 著稱 — 品種公認優點之一。

**10. 儲存：**

* 密封不透明容器 — 金屬罐、真空鋁箔袋、陶瓷容器。
* 乾燥、陰暗、涼爽處（15–25°C，濕度低於60%），遠離異味。
* 最佳期限 — 6–18個月保持鮮活度。良好乾燥的批次能在2–3年內溫和「圓潤化」。
* 避免潮濕、直射光、溫度變化和與芳香食品鄰近。

**11. 價格與仿冒：**

* **價格類別：** 保靖黃金紅茶是頂級地方產品。價格由採摘標準（芽茶批次最貴）、季節（早春原料最受重視）、來源（古樹vs.年輕種植園）和獲獎證書決定。俗語「一兩黃金一兩茶」反映了這種茶作為奢侈品的歷史認知，儘管現代市場價格當然遠非字面意思的「一兩黃金換一兩茶」。
* **如何避免仿冒：**
1. 向可追溯至保靖具體農場的可信賣家購買。注意地理標誌 (地理标志) 標識。
2. 評估外觀：均勻、緊密捲曲，金毫豐富，無粉塵和雜質。
3. 檢查香氣：應該純淨、蜂蜜甜香，帶特有「黃金」調。無焦糊、霉味、魚腥味。
4. 評估湯色：明亮、紅琥珀色、透澈，有明顯金圈。混濁是警告信號。
5. 測試「耐泡性」：正宗黃金茶以多次沖泡耐久著稱。如果3–4泡後滋味急劇「衰落」 — 可能表示原料替換。

**12. 有趣事實：**

* 保靖黃金茶被稱為「可以喝的文物」(可以拿来喝的文物, 「可以拿來喝的文物」)：黃金村保存著2057棵古茶樹，最古老的超過400年。七個歷史茶園 — 龍頸坳 (龙颈坳)、格者麥 (格者麦)、德讓拱 (德让拱)、庫魯 (库鲁)、團田 (团田)、冷寨河 (冷寨河) 和夯納烏 (夯纳乌) — 作為露天活博物館保存。

* 「黃金茶之母」— 農藝師張湘生 (张湘生)，文化大革命後第一批大學畢業生。從1993年起她專注於單一任務 — 學會扦插繁殖黃金茶。1994年實驗成功：3.16畝幼苗（約500,000株）開啟了品種在整個湘西的大規模推廣。

* 院士劉仲華 (刘仲华, Liú Zhōnghuá)，世界領先茶科學專家之一和中華人民共和國國家獎獲得者，公開讚賞黃金茶並親自每年採購超過100公斤茶葉。據他說，他從未見過胺基酸含量如此高的茶樹品種。

* 保靖位於北緯28° — 所謂「茶葉生產黃金帶」。縣境與著名國家公園「懸浮山」張家界 (张家界, Zhāngjiājiè) 和作家沈從文故鄉「鳳凰古城」鳳凰 (凤凰, Fènghuáng, 「鳳凰城」) 接壤。

* 保靖茶不僅生產紅茶和綠茶版本：當地師傅嘗試用同樣原料進行黑茶、烏龍茶、白茶和黃茶加工。年產量（2020年資料）：綠茶 — 581噸，紅茶 — 309噸，黑茶 — 270噸，整個茶產業鏈綜合價值 — 125億元。

**13. 與其他紅茶比較：**

* **古丈毛尖紅茶 (古丈毛尖红茶)：** 來自鄰近古丈縣 (古丈) 的紅茶，同樣位於湘西。以著名綠茶古丈毛尖原料製作。結構上是「親緣」產品，但沒有黃金茶創紀錄的胺基酸含量。古丈紅茶通常稍微「乾燥」，甜味較少。

* **湖南紅茶 (湖南红茶, 總體類別)：** 涵蓋湖南省紅茶的傘形品牌，包括湖紅 (湖红, Hú Hóng)。傳統湖南紅茶使用標準品種製作，具有更均勻的「工業」特徵。黃金紅茶以獨特品種特性帶來的特殊甜味和「油潤」質感脫穎而出。

* **滇紅 (滇红, Diān Hóng)：** 雲南大葉種阿薩姆型紅茶。滇紅 — 強勁、飽滿，帶明顯巧克力、胡椒和麝香調。黃金紅茶 — 更柔和、更甜，香氣特徵更精緻，胺基酸含量顯著更高。

* **金駿眉 (金骏眉, Jīn Jùn Méi)：** 武夷山（福建）頂級紅茶，用野生茶樹純芽製作。兩種茶都是芽茶，蜂蜜甜味，金黃湯色。差異在於風土：金駿眉帶有武夷山「岩韻」礦物調，黃金紅茶帶有湘西果香深度和特有「黃金」甜味。

**結論：**

保靖黃金紅茶是具有特殊血統的紅茶。背後有四個多世紀古茶樹的生命，一兩黃金的傳說，胺基酸含量創紀錄的獨特品種，湘西山區及其苗族土家族文化和生態原始性。在茶杯中它提供了大多數紅茶難以同時達到的：深沉蜂蜜甜味而無絲毫膩味，飽滿「油潤」體感，持久的黃金乾果香調 — 以及多次沖泡的驚人「耐久性」，這正是黃金茶獲得「黃金」聲譽的原因。
