# 梅占紅茶 (梅占红茶, méi zhàn hóngchá)

**梅占紅茶 (梅占红茶, méi zhàn hóngchá)**
![Мэй Чжань Хун Ча (梅占红茶, méi zhàn hóngchá)](images/Мэй%20Чжань%20Хун%20Ча%20(梅占红茶,%20Méi%20Zhàn%20Hóngchá).png)

![Мэй Чжань Хун Ча (梅占红茶, méi zhàn hóngchá)-1](images/Мэй%20Чжань%20Хун%20Ча%20(梅占红茶,%20Méi%20Zhàn%20Hóngchá)-1.png)

梅占紅茶是採用中國最具多用途性的品種之一——梅占 (梅占) 「梅花奪魁」製作的紅茶。這個來自安溪的傳奇品種是「安溪六大名茶」(安溪六大名茶) 之一，以「茶界變色龍」聞名：可製作出優質的烏龍茶、紅茶、綠茶和白茶。然而，正是在紅茶的表現中，梅占展現了其最亮麗的一面——高雅的「蘭花香」(兰花香) 和深厚、濃郁、甜潤帶辛香的滋味，帶有梅花的韻味。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá)，全發酵茶。
* **類別：** 品種紅茶／地方紅茶，採用梅占品種製作。也可歸類為高等級白琳工夫 (白琳工夫) 的原料。
* **產地：** 梅占品種原產於福建省安溪縣 (安溪县, Ānxī Xiàn)——具體來自蘆田鎮 (芦田镇, Lútián Zhèn) 三洋村 (三洋村, Sānyáng Cūn)，銀瓶山。作為紅茶，梅占主要在兩個地區生產：**武夷山 (武夷山)** ——用於製作高級紅茶（包括「金梅冰芽」——金梅冰芽）；**福鼎 (福鼎)** ——傳統上作為白琳工夫的優質原料；以及安溪本地和福建其他產茶區。
* **地理座標：** 約北緯25°03′，東經117°57′（安溪，品種發源地）。

**2. 歷史與文化意義：**

* **歷史：** 梅占品種有超過200年的歷史。關於其起源有兩個主要傳說。第一個：約1810年（嘉慶十五年）三洋村農民楊奕糖 (杨奕糖) 遇到一位帶著茶苗的流浪老人，用粥招待了他；老人感激之餘贈送了三株茶苗。楊將它們種在「玉樹厝」(玉树厝) 旁，兩三年後茶樹繁茂。當地舉人 (举人) 楊飛文 (杨飞文) 見這種茶的花葉如梅花瓣般綻放，散發臘梅香氣 (腊梅)，便命名為「梅占」——「梅花奪魁」。第二個傳說：1821年（道光元年）西坪王氏族人在蘆田見到一棵無名茶樹，不知其名，注意到門楣上的題字「梅占百花魁」（「梅花在百花中稱王」）——於是取名「梅占」。

  1960年代，梅占開始在中國各產茶省份推廣。1985年，國家農作物品種審定委員會授予其**國家品種** (国家品种, GS13004-1985) 地位。在鐵觀音熱潮期間（1990-2000年代），許多安溪茶農砍掉梅占，改種鐵觀音。倖存的老樹 (老枞, lǎo cóng)——50-100+年——成為製作高級紅茶的珍貴資源。

* **名稱：** 「梅」(梅) ——梅花；「占」(占) ——「佔據」、「奪取首位」。完整含義：「梅花在百花中奪魁」(梅占百花魁) ——指這種茶樹的花朵像梅花花瓣，香氣如臘梅。

* **文化意義：** 梅占是「安溪六大名茶」(安溪六大名茶) 之一，與鐵觀音、黃金桂、本山、毛蟹、大葉烏龍並列。清末「福建八大詩人」之一、蘆田人林鶴年 (林鹤年) 寫道：「種梅三萬株，終老吾何悔」(种梅三万株，终老吾何悔)。

**3. 植物學描述與原料：**

* **品種／栽培種：** 梅占 (梅占)，又稱大葉梅占 (大叶梅占，「大葉梅占」)、高腳烏龍 (高脚乌龙，「高腳烏龍」)。國家品種（GS13004-1985），*Camellia sinensis*。無性系 (无性系)，小喬木型 (小乔木型)，中葉類 (中叶类)，中芽種 (中芽种)。植株高大（可達1.6公尺），主幹直立，主枝明顯，節間較長。葉片——長橢圓形，深綠色，厚實，邊緣內捲。梅占是適應性冠軍：「百變茶青」(百变茶青)。適合製作烏龍茶、紅茶、綠茶和白茶。
* **原料化學成分（春茶，一芽二葉）：** 胺基酸——3.6%；茶多酚——27.5%；兒茶素——18.1%；咖啡因——4.4%。
* **採摘：** 春茶（最珍貴）、夏茶、秋茶。四月中旬——春茶採摘高峰期。手工採摘。
* **採摘標準：** 一芽一至二葉用於高級等級（包括「金梅」——純芽茶）；一芽二至三葉用於標準等級。
* **原料要求：** 必須嫩採、早採——梅占芽葉持嫩性較差 (持嫩性较差)，需要及時採摘。

**4. 風土與栽培特點：**

* **原產地——安溪蘆田：** 閩南山區。海拔400-1,200公尺。亞熱帶季風氣候，雨量充沛，雲霧繚繞，肥沃的弱酸性紅壤。銀瓶山 (银瓶山, 1,200公尺) ——品種發源地。
* **武夷山：** 梅占紅茶也在武夷山生產，該品種於20世紀初（民國時期）引入。武夷山風土賦予其特有的「岩韻」(岩韵，「岩石韻味」)。
* **福鼎：** 白琳工夫的傳統產區，梅占在此作為優質原料使用。
* **推廣：** 自1960年代起，梅占在全中國栽培——廣東、江西、浙江、安徽、湖南、湖北、江蘇、廣西，以及臺灣。由於高適應性和產量（200-350公斤／畝），成為最廣泛種植的多用途品種之一。

**5. 製作工藝：**

梅占紅茶按標準全發酵紅茶工藝製作：萎凋 → 揉捻 → 發酵 → 乾燥 → 分級。特點——注重「激發」蘭花香氣。

* **採摘 (采摘 — cǎizhāi)：** 手工採摘。早採嫩採 (早采嫩采) ——芽葉易老化。
* **萎凋 (萎凋 — wěidiāo)：** 日光或室內萎凋，12-18小時。控制程度——「不過乾」，保持鮮度並啟動香氣形成。
* **揉捻 (揉捻 — róuniǎn)：** 機械或手工揉捻。形成緊結、細長的條索。梅占因葉肉厚實出汁豐富。
* **發酵／氧化 (发酵 — fājiào)：** 3-5小時，溫度約25-28°C。關鍵指標——出現蘭花梅香。過度發酵產生「平淡」味道；發酵不足則有苦澀味。
* **乾燥 (烘干 — hōnggān)：** 熱風或炭火烘乾。安溪版本有時採用額外烘焙 (提香, tíxiāng — 「提升香氣」)。
* **分級 (分级 — fēnjí)：** 手工最終篩選。

**6. 感官特徵：**

* **乾茶外形：** 緊結細長的條索 (条索紧结细长)。色澤——烏黑油潤 (乌黑油润)，高等級帶金毫。條索勻整，完整。
* **乾茶香氣：** 招牌特色——高銳香氣 (香气高锐)：蘭花香 (兰花香)、梅花香 (梅花香)、蜂蜜、果乾。武夷山版本——額外的礦物質「岩石」韻味。香氣持久，辨識度高。
* **茶湯香氣：** 層次豐富——蘭花、梅花、蜂蜜、焦糖、淡淡辛香。隨沖泡次數增加——甜蜜蜂蜜果香加強。
* **滋味：** 濃厚飽滿，「絲滑」(味厚)。主調——蘭花、梅花、蜂蜜、焦糖、淡淡巧克力韻味。澀味——適中，「絲綢般」。天然甜味。回味——持久，帶蘭花梅香。回甘 (回甘) ——明顯。
* **湯色：** 橙黃至紅寶石色，明亮透澈，高等級帶「金圈」(金圈)。
* **葉底：** 大片完整有彈性的紅銅色葉片。葉緣顯紅邊 (红边显)。

**7. 化學成分：**

* **多酚類：** 原料中27.5%（高於平均），發酵過程中——活躍形成茶黃素和茶紅素，構成濃郁色澤和「體感」。
* **胺基酸：** 3.6%——提供甜味和柔和口感。
* **咖啡因：** 4.4%——中等偏高水平，有明顯提神效果。
* **兒茶素：** 18.1%——發酵時轉化為茶黃素、茶紅素。
* **芳香化合物：** 芳樟醇、橙花醇、香葉醇——形成特有的「蘭花梅香」複合體。

**8. 保健功效：**

* 溫和提神（咖啡因 + L-茶胺酸）。抗氧化作用（高多酚含量）。助消化。溫暖身體。抗菌效果。抗壓力作用。

**9. 沖泡方法：**

* **水溫：** 90-95°C。
* **茶量：** 4-5克／100-120毫升（功夫茶）；3克／200-250毫升（西式沖泡）。
* **茶具：** 瓷蓋碗——完美展現蘭花香氣。
* **沖泡：** 溫杯 → 投茶 → 洗茶（可選）→ 首泡10-15秒 → 5-8泡，每泡增加5-10秒。

**10. 儲存：**

密封容器，乾燥陰涼處，10-25°C，可保存18個月。無需冷藏。

**11. 價格與仿冒：**

梅占紅茶屬中等價位：標準級——200-600元人民幣／500克；老樹 (老枞) ——800-2,000元；「金梅」（純芽）——可達3,000+元。

**如何避免仿冒：**
* 尋找特有的「蘭花梅香」——正宗梅占的標誌。
* 評估條索：細長緊結，烏黑有光澤。
* 湯色——橙黃透亮。混濁——品質低劣的徵象。
* 確認產地：安溪、武夷山、福鼎——主要產區。

**12. 趣聞軼事：**

* **「茶界變色龍」：** 梅占是少數可製作任何茶類的品種之一：烏龍茶（清香型和濃香型）、紅茶、綠茶、白茶。在每種表現中都展現特有的「梅香」特色。
* **「不奪他茶香」：** 在武夷山，梅占被視為拼配的珍貴成分——它豐富拼配茶，而不掩蓋其他品種 (拼茶不夺他茶香)。
* **三萬株茶樹：** 清末蘆田詩人林鶴年，「福建八大詩人」之一，為梅占題詩：「種梅三萬株，终老吾何悔」——「種梅三萬株，終老吾何悔」。
* **鐵觀音熱潮的犧牲品：** 1990-2000年代大規模以鐵觀音替代梅占，導致安溪大部分老茶園消失。倖存的老樹（50-100+年）——珍貴資源。
* **白琳工夫的優質原料：** 歷史上最好的白琳工夫都用梅占製作——這個事實突顯了其卓越的「紅茶」品質。

**13. 與其他紅茶比較：**

* **金駿眉 (金骏眉)：** 純芽茶，採用桐木菜茶。更甜潤、蜜香、「輕盈」。梅占紅茶——更濃厚，蘭花梅香突出，「體感」更強。
* **正山小種 (正山小种)：** 「鼻祖」——帶龍眼香和（煙燻版本）煙味。梅占紅茶——無煙味，香氣譜完全不同（蘭花、梅花）。
* **祁門紅茶 (祁门红茶)：** 「祁門香」——細膩、「內斂」、玫瑰蜜香。梅占——更明亮、「響亮」，蘭花香突出。
* **梅占烏龍 (梅占乌龙)：** 同一品種，烏龍工藝製作。烏龍——更「清香」、花香，回甘明顯。紅茶——更飽滿、濃郁，帶焦糖巧克力深度。

**結語：**

梅占紅茶是一款有個性的茶：明亮、自信、豐富。來自銀瓶山傳奇品種的蘭花梅香，不隱藏不低語——從第一泡開始就全力綻放。這是為那些在紅茶中不僅欣賞柔和，也珍視力量；不僅追求甜味，也渴望深度的人而生的茶。

「種梅三萬株，終老吾何悔」——這位19世紀安溪詩人的詩句今天聽來如預言：梅占，經歷了鐵觀音熱潮並幾乎消失，如今正在回歸——尤其是在紅茶領域，其聲音比以往任何時候都更響亮、更有說服力。
