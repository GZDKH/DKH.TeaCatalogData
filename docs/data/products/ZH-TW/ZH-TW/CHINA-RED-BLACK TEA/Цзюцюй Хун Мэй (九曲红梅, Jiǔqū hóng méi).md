# 九曲紅梅 (九曲紅梅, Jiǔqū hóng méi)

**九曲紅梅 (九曲紅梅, Jiǔqū hóng méi)**

![九曲紅梅 (九曲紅梅, Jiǔqū hóng méi)-1](images/Цзюцюй%20Хун%20Мэй%20(九曲红梅,%20Jiǔqū%20Hóng%20Méi)-1.png)

九曲紅梅是浙江省二十八種名茶中唯一的紅茶，贏得了「萬綠叢中一點紅」的詩意美譽。這款擁有近兩百年歷史的工夫紅茶，誕生於福建紅茶傳統與杭州西郊獨特風土的交匯處。其纖細如魚鉤般彎曲的茶葉、紅寶石色的茶湯，以及宛如紅梅花綻放的香氣，使其與偉大的龍井茶並列成為西湖茶文化的名片。

**1. 分類與產地：**

* **類型：** 中國紅茶（紅茶，hóngchá），完全氧化（發酵）。
* **類別：** 工夫紅茶（工夫紅茶，gōngfū hóngchá）——傳統精工製作的紅茶。屬於中國歷史名茶（歷史名茶）。是杭州十大名茶「九綠一紅」中唯一的紅茶。
* **產地：** 中國，浙江省（浙江省，Zhèjiāng shěng），杭州市（杭州市，Hángzhōu shì），西湖區（西湖區，Xīhú qū），雙浦鎮（雙浦鎮，Shuāngpǔ zhèn）。主要生產村莊：湖埠、雙靈、張余、馮家、靈山、社井、仁橋、上陽、下陽。傳統上以湖埠村大塢山（大塢山）的茶品質最佳。
* **地理座標：** 約北緯30°10′，東經120°05′（雙浦地區，杭州西南郊，錢塘江畔）。

**2. 歷史與文化意義：**

* **歷史：** 九曲紅梅創製於清同治年間（清同治年間，1862-1874），其生產歷史約有一百五十至兩百年。此茶的出現與太平天國起義（太平天國，1850-1864）有關：當戰火蔓延至福建省武夷山地區時，十三戶茶農北遷，定居於大塢盆地——現今的雙浦鎮。他們開墾山坡，種植茶樹，並憑藉製作紅茶的經驗，用當地原料製作新產品。技術逐漸適應當地風土，獨特的紅茶——九曲紅梅由此誕生。

  此茶很快贏得認可。1886年在巴拿馬國際食品博覽會獲金獎。1915年在巴拿馬萬國博覽會（巴拿馬萬國博覽會）再獲金獎。1926年獲費城世界博覽會獎項。1928年獲上海中國國貨展覽會特等獎，新時代「茶聖」吳覺農（吳覺農，Wú Juénóng）在專家評語中寫道：「杭州紅茶色、香、味俱佳，惜乎價格太高」。1929年獲首屆西湖博覽會（首屆西湖博覽會）特等獎，並被列入當時中國十大名茶。

  1937年日軍佔領杭州後，生產陷入衰退。茶園荒廢，至中華人民共和國成立時，該地區處於深度危機中。1950-60年代開始逐步恢復：農民組織合作社，引進新品種和機械化。然而在1990年代，由於西湖龍井的商業成功，地方當局推行「以紅改綠」政策，九曲紅梅瀕臨消失。復興始於2000年代：2000年註冊商標，2003年獲得「浙江綠色產品」認證，2006年和2009年製作技藝分別被列入杭州市和浙江省非物質文化遺產名錄。2016年九曲紅梅作為G20杭州峰會官方用茶，隨後在烏鎮第三、四屆世界互聯網大會（2016、2017）上亮相。

* **名稱：** 名稱由三個語義成分組成。「九曲」——「九個彎曲」——既指武夷山著名的九曲溪（九曲溪），即茶葉創始人的故鄉，也指大塢地區「九曲十八彎」的蜿蜒小溪。「紅」——「紅色」——表示茶類。「梅」——「梅花」——與其特有的紅梅花香氣有關。此茶還有歷史名稱九曲紅、九曲烏龍和龍井紅。

* **文化意義：** 著名佛教高僧、書法家弘一法師（弘一法師，Hóngyī fǎshī），即李叔同（李叔同），以詩句讚美此茶：「白玉杯中瑪瑙色，紅唇舌底梅花香」。九曲紅梅與西湖龍井共同構成西湖茶文化的傳奇組合「一紅一綠」，象徵著杭州茶傳統的完整與平衡。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要使用當地群體種（群體種，qúntǐ zhǒng）*Camellia sinensis* var. *sinensis*——與西湖龍井使用相同的茶樹群體。這是小葉和中葉灌木型（灌木型中小葉種），直立生長，分枝密度中等。葉片長橢圓形，芽和嫩葉黃綠色，密被茸毛。一芽三葉百芽重約71克。植株抗寒性強，新梢生長旺盛。
* **採摘：** 最佳採摘期在穀雨（穀雨，「穀雨」，通常為4月20日）前後，比龍井稍晚。有趣的是，與大多數茶不同，九曲紅梅過早的清明前採摘並非最佳——最適宜的葉片成熟度正好在穀雨時節。
* **採摘標準：** 一芽二葉初展（一芽二葉初展，yī yá èr yè chū zhǎn）。葉片既不能過嫩也不能過老——正是這種成熟度確保了香氣與茶體的特有平衡。
* **原料要求：** 完整、潔淨的葉片，無損傷和粗梗。採摘與加工之間的延遲應降至最低。最佳批次選用樹齡超過百年的老茶樹嫩芽。

**4. 風土與栽培特點：**

* **地形與景觀：** 生產區位於大塢盆地（大塢盆地），四周環繞低山。「九曲十八彎」的地形創造了眾多微氣候生態位。東面以錢塘江（錢塘江）為界，江面蒸發形成早晚持續的雲霧。
* **生長海拔：** 海拔100至500公尺。最佳茶園位於大塢山山坡。
* **年均溫度：** 約16-17°C。亞熱帶季風氣候，冬季溫和，夏季溫暖潮濕。
* **降雨量：** 年降雨量約1400-1500毫米，相對濕度高。
* **土壤：** 砂質酸性土壤（砂質土壤，pH 4.5-5.5），肥沃，排水良好。周圍山林茂密，保護茶園免受風害並提供散射光照。
* **生態：** 2008年，生產區內的雙靈村獲得「江南生態茶村」稱號，證實了自然環境的卓越品質。

**5. 製作工藝：**

九曲紅梅採用經典的工夫紅茶工藝，包括四個主要步驟。獨特之處在於形成特有的纖細彎曲茶葉形狀，這需要特殊的揉捻技巧和發酵的精確控制。

* **萎凋（萎凋，wěidiāo）：** 鮮葉薄攤於竹篾上。萎凋過程中，葉片均勻失水，細胞壁軟化，膨壓降低。青草氣消失，葉片獲得後續揉捻所需的彈性而不破碎。持續時間取決於溫度和空氣濕度。

* **揉捻（揉捻，róuniǎn）：** 萎凋葉經機械揉捻，破壞細胞結構，使細胞液流到表面。正是在這個階段形成九曲紅梅著名的「魚鉤」形狀：纖細、彎曲的茶葉，如銀鉤般能相互勾連成環。揉捻還確保酶的均勻分布以利後續氧化。揉捻之間進行「搓散」（搓散，cuōsàn）——分離黏結的葉片。

* **發酵/氧化（發酵，fājiào）：** 決定紅茶色、味、香的關鍵步驟。揉捻葉在適宜的溫度（通常25-30°C）和濕度條件下攤放。在多酚氧化酶作用下，兒茶素氧化，轉化為茶黃素和茶紅素。葉片逐漸從綠色變為金黃色和紅褐色，出現明顯的花果香。傳統使用自然發酵方法；現代茶廠使用溫濕度可控的自動化發酵室。

* **乾燥（乾燥，gānzào）：** 分兩階段進行。第一階段——「毛火」（毛火，máo huǒ）：高溫快速乾燥以鈍化酶活性並停止氧化。第二階段——「足火」（足火，zú huǒ）：低溫慢速乾燥以徹底去除水分並展現香氣。正是在「足火」階段形成特有的輕微松脂香和蜜甜味。

* **分級（分級，fēnjí）：** 成茶篩分並按等級分類。挑選均勻、一致且金毫比例高的批次。

**6. 感官特徵：**

* **乾茶外形：** 九曲紅梅的標誌是其獨特形狀。茶葉纖細如髮，緊密捲曲成優雅的彎曲，似魚鉤或銀扣；手抓時相互勾連，形成環狀「串」。色澤烏黑油潤（烏潤），密佈金毫（金毫）。
* **乾茶香氣：** 豐富、多層次：明顯的蜂蜜和焦糖香，其後顯現紅梅花香的花香調。高級別茶品帶有細膩的蘭花香（蘭花香）和輕微、幾乎難以察覺的松脂煙燻調。
* **茶湯香氣：** 高揚持久，結合蜜甜、紅梅花香和成熟水果的細微差別。香氣分幾波展現：初泡——明亮的花香；中段——溫暖的蜜糖焦糖香；尾韻——柔和的木質甜香。
* **滋味：** 飽滿濃郁，具有明顯的天然甜味和「絲絨般」的質地。口感中有成熟蜂蜜、紅色水果和焦糖的味道。澀感溫和宜人，迅速轉化為持久的回甘（回甘，huígān）。最佳批次展現特有的「桂圓湯味」——類似龍眼的圓潤甜味。
* **湯色：** 明亮紅豔，晶瑩剔透，純淨清澈，杯緣有特徵性的金圈（金圈，jīnquān）——高茶黃素含量和品質的標誌。
* **葉底（沖泡後的葉片）：** 紅豔明亮，色澤均勻，葉片柔嫩、完整，展開呈小「花朵」狀（紅豔成朵）。最佳等級的葉底柔軟、有彈性，帶有明顯光澤。

**7. 化學成分：**

* **多酚類：** 原料中茶多酚含量約25.6%（以乾物質計）。在完全發酵過程中，大部分兒茶素轉化為茶黃素（茶黃素，cháhuángsù）和茶紅素（茶紅素，cháhóngsù），形成紅色茶湯、「絲絨般」口感和金圈。茶黃素與茶紅素的比例是品質的關鍵指標：茶黃素比例越高，茶湯越明亮「活潑」。氧化多酚產物總含量為乾重的5-15%。
* **胺基酸：** 總含量約3.5%，對紅茶而言相對較高。L-茶胺酸（L-茶胺酸）提供天然甜味、口感柔和度和「鮮爽感」。高胺基酸含量與龍井群體種優質原料有關。
* **生物鹼：** 咖啡鹼（咖啡鹼）——乾重的2-4%，提供提神效果。茶鹼和可可鹼含量較少，補充溫和的刺激作用。
* **兒茶素：** 原料中兒茶素總含量約11.3%。發酵後大部分轉化，但殘留的兒茶素保持抗氧化潛力。
* **揮發性芳香化合物：** 關鍵香氣成分——芳樟醇（芳樟醇，fāngzhāngchún）、香葉醇（香葉醇，xiāngyèchún）、苯甲醛、苯乙醛和水楊酸甲酯。正是它們的組合創造了特有的「梅花-蜂蜜」香氣特徵。
* **維生素：** 維生素C（發酵時部分破壞）、B₁、B₂、P（蘆丁）、PP（菸鹼酸）。
* **礦物質：** 鉀、鎂、鈣、錳、鋅、氟、硒。

**8. 保健功效：**

* **提神與認知效果：** 咖啡鹼和L-茶胺酸的結合提供溫和、持續的清醒感，無劇烈的興奮高峰，改善注意力集中和反應速度。
* **抗氧化保護：** 茶黃素、茶紅素和殘留兒茶素中和自由基，減少細胞氧化壓力。
* **消化支持：** 茶多酚和有機酸溫和刺激胃液分泌和胃腸蠕動，幫助消化油膩食物。溫熱的紅茶對胃部舒適。
* **心血管支持：** 研究顯示，定期適量飲用紅茶可能有助於降低低密度脂蛋白膽固醇並改善血管彈性。
* **抗菌作用：** 多酚類抑制多種病原菌生長，有助於支持免疫力和口腔健康。
* **溫暖效果：** 在中醫飲食學中，紅茶傳統上被認為性溫，推薦給「寒性」體質者、秋冬季節飲用和恢復體力。
* **情緒健康：** L-茶胺酸具有輕微的抗焦慮作用，溫暖甜美的梅花香氣具有放鬆的感官效果。

**9. 沖泡方法：**

* **水溫：** 90-95°C。對於特別細嫩、金毫比例高的春茶可降至85-90°C。
* **茶量：** 4-5克/100-120毫升（工夫泡法）；2-3克/200-250毫升（杯泡或西式泡法）。
* **茶具：** 白瓷蓋碗（蓋碗）100-120毫升——最適合展現香氣和觀察湯色。瓷茶壺也適合。透明玻璃杯可欣賞纖細茶葉的「舞動」。對於成熟緊實的茶品可用宜興紫砂壺。
* **步驟：**
  1. 用開水溫潤蓋碗或茶壺，倒掉水。
  2. 投入茶葉，輕搖茶具讓熱氣激發乾茶香氣——吸聞並品評。
  3. 潤茶（潤茶）：可選——快速沖泡1-2秒以「喚醒」茶葉。
  4. 第一泡：注水，浸泡5-8秒，倒入公道杯（公道杯）。
  5. 後續沖泡：每泡增加3-5秒。
  6. 沖泡次數：優質茶可沖泡6-8次；最佳批次可達10次。
  7. 西式泡法：2-3克/200毫升杯，浸泡3-4分鐘。

**10. 儲存方法：**

九曲紅梅是完全發酵茶，儲存穩定性良好，但需遵守基本規則。茶葉應存放在密封容器中（錫罐或鐵罐內襯鋁袋），避光、防潮、隔絕異味。最佳溫度10-25°C；不需要也不建議冷藏。在適當條件下保質期為18-24個月；此期間茶葉保持新鮮和香氣明亮。某些高火候的緊實批次在「陳放」2-3年後會改善：香氣更深沉，口感更圓潤。開封後建議在2-3個月內飲用。

**11. 價格與防偽：**

九曲紅梅的價格範圍相當寬廣。中國國內市場參考價格：特級——500克700元人民幣起，一級——約400元，二級——約250元。影響價格的因素：風土（大塢山的茶價格更高）、季節和採摘標準（金毫比例）、手工或機械加工、茶樹樹齡（老樹屬高端產品）、獲獎狀況和品牌歸屬。

* **如何避免假貨：**
  1. 向可靠供應商購買，確認產自西湖雙浦地區。
  2. 評估形狀：真正的九曲紅梅具有獨特的纖細「鉤狀」形態，手抓時茶葉相互勾連——這很難仿製。
  3. 香氣應純淨、高雅，帶有梅花和蜂蜜香，無化學刺激味或霉味。
  4. 茶湯明亮紅豔、透明，有明顯金圈；混濁或暗淡的茶湯表示品質低劣。
  5. 「獲獎」或「特級」茶的價格過低——明顯的假貨標誌。

**12. 趣聞軼事：**

* 九曲紅梅是少數幾種以生產著名綠茶西湖龍井的相同茶樹為原料的紅茶之一。因此，「一紅一綠」實際上是來自同一茶園的兩種茶，但加工技術截然相反。

* 靈山地區的民間傳說中有一個美麗的故事：一個名叫阿龍（「小龍」）的男孩意外吞下了一顆神奇的珍珠，變成了黑龍（烏龍）。因不忍離開父母，他沿著小溪游下時每個轉彎都回頭張望——於是形成了「九曲」的蜿蜒小溪。後來在溪畔生產的紅茶得名「九曲烏龍」，之後因其梅花般的香氣改名為「九曲紅梅」。

* 在2016年杭州G20峰會以及烏鎮第三、四屆世界互聯網大會上，此茶被選為官方禮賓用茶，向世界領導人展示杭州紅茶。

* 九曲紅梅製作技藝被列入浙江省非物質文化遺產名錄（2009），強調其作為活態手工傳統的價值。

* 著名佛教大師、啟蒙家弘一法師（李叔同，1880-1942），民國初期最傑出的文化人物之一，留下了獻給此茶的詩句，成為其「名片」。

**13. 與其他紅茶的比較：**

* **正山小種（正山小種，Zhèngshān Xiǎozhǒng）：** 來自福建武夷山的紅茶——孕育九曲紅梅的傳統發源地。小種，特別是煙燻型，具有明顯的松煙香（松煙香）和更飽滿、油潤的茶體。九曲紅梅則優雅精緻得多，「香水般」：香氣更細膩，茶體更輕盈，葉形獨特。

* **祁紅（祁紅，Qí Hóng）——祁門紅茶：** 安徽著名紅茶。祁紅以「祁門香」（祁門香）聞名——蘭花果香，帶玫瑰調。與九曲紅梅相比，其葉形更標準（細緊條索），口感更柔和，「蜜味」較淡。九曲紅梅在形態的視覺奇特性上勝出，香氣特徵傾向於「梅花-蜂蜜」，而祁紅是「玫瑰-蘭花」。

* **滇紅（滇紅，Diānhóng）：** 雲南大葉種*Camellia sinensis* var. *assamica*製成的紅茶。滇紅強勁、飽滿，帶可可和乾果香，金毫顯著。九曲紅梅在性格上完全相反：精緻、優雅，具有纖巧的「銀色」形態和更輕盈的質地，由小葉種製成。

* **九層山紅茶（九層山紅茶）：** 現代臺灣高山紅茶。具有更明顯的果甜味和臺灣風土特有的「高山」清新感。九曲紅梅性格更「經典」，強調梅花香氣和精湛的揉捻技藝。

**結語：**

九曲紅梅是一款矛盾的茶：由戰亂中逃離武夷山的難民創造，在錢塘江畔找到了自己的面貌，經歷了被遺忘的時代，又作為重大國際峰會的禮賓茶重返世界舞臺。其纖細如銀螺旋般捲曲的茶葉、帶金圈的紅寶石茶湯，以及從初泡梅花香到溫暖蜜糖尾韻的多層次香氣——創造出既精緻又深邃的品茶體驗。這款茶非常適合那些在紅茶中欣賞優雅、複雜性和歷史，而非力量與粗獷的人。與其永恆的夥伴——綠茶西湖龍井一起，九曲紅梅構成和諧的組合，體現了南宋古都茶文化的完整性。
