# 正山小種 (正山小种, zhèng shān xiǎo zhǒng)

**正山小種 (正山小种, zhèng shān xiǎo zhǒng)**

![Чжэн Шань Сяо Чжун (正山小种, zhèng shān xiǎo zhǒng)-1](images/Чжэн%20Шань%20Сяо%20Чжун%20(正山小种,%20Zhèng%20Shān%20Xiǎo%20Zhǒng)-1.png)

正山小種是世界上第一款紅茶，是全球所有紅茶（黑茶）的鼻祖。四百多年前在桐木山中偶然誕生，從鄉村製茶師的意外之作，發展成為中國在歐洲的象徵、英國茶文化的啟發者，以及後來祁門、滇紅、阿薩姆、大吉嶺和世界上所有紅茶的基礎。在西方，它以**拉普山小種（Lapsang Souchong）**之名聞名。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá)，完全氧化。按歐洲分類屬於黑茶。發酵度為80–95%。存在兩個主要品種：傳統煙燻 (传统烟熏正山小种, chuántǒng yānxūn) — 用馬尾松 (马尾松, Pinus massoniana) 木材烘乾/煙燻；以及無煙燻/「新工藝」(新工艺正山小种, xīn gōngyì) — 不經煙燻，強調天然蜜果香型。
* **類別：** 中國名茶 (中国名茶)。被認定為世界所有紅茶的鼻祖。自2010年起受地理標誌保護 (地理标志, dìlǐ biāozhì)。
* **產地：** 中國福建省 (福建省, Fújiàn Shěng) 南平市 (南平市, Nánpíng Shì) 武夷山市 (武夷山市, Wǔyíshān Shì) 桐木村 (桐木村, Tóngmù Cūn) — 武夷山國家級自然保護區 (武夷山国家级自然保护区, 565平方公里) 的核心區。桐木是正山小種的歷史發源地和唯一正宗產地。「正山」(正山) 意為「正宗山區」，即桐木及相同海拔帶的鄰近地區。在此範圍外生產的茶稱為「外山」(外山, 「外部山區」)，不能使用「正山」名稱。
* **地理座標：** 東經117°38′6″–117°44′30″，北緯27°41′35″–27°49′00″。
* **別名：** 拉普山小種（Lapsang Souchong）— 歐洲貿易名稱；桐木關小種 (桐木关小种) — 「桐木的小種」；星村小種 (星村小种) — 以歷史上茶葉出口的貿易集鎮命名。在17–18世紀歐洲文獻中稱為BOHEA（來自武夷Wuyi — 「博希」）。

**2. 歷史與文化意義：**

* **歷史：** 正山小種是人類歷史上第一款紅茶，已有400多年歷史。確切創製日期未有記載，但根據綜合資料（鄒新球研究 — 邹新球，《武夷正山小種紅茶》— 《武夷正山小種紅茶》），其起源可追溯至16世紀中後期（約1567–1600年）。廣為流傳的傳說是：明朝末年 (明, 1368–1644)，戰亂期間，一支從江西前往福建的軍隊在廟灣村（庙灣，現為桐木一部分）的製茶作坊過夜。士兵們直接睡在攤放待加工的茶葉上。到了早晨，茶葉開始發酵變紅。為了挽救茶葉，製茶師（據正山堂 — 正山堂 — 江氏祖先的說法）將變色的茶葉揉捻，並用當地馬尾松 (马尾松, 松樹馬松) 的火烘乾。製成的茶葉呈黑色，帶有濃郁的松煙香味，在桐木當地不受歡迎，但當運到45公裡外的星村 (星村) 集市時，一位閩南商人以低價收購。次年他回來時出價高出兩三倍 — 這種特殊茶葉的需求很大。紅茶的歷史由此開始。

* **出口歐洲：** 約1604年，荷蘭商人將正山小種帶到歐洲 — 這是第一款到達西方的中國紅茶。在荷蘭和英國，它最初在藥房作為藥物銷售。1662年，葡萄牙公主凱瑟琳·布拉干薩嫁給英國國王查理二世時，在嫁妝中帶了幾箱「正山小種」— 紅茶由此進入英國宮廷。安妮女王後來在上流社會推廣茶飲，奠定了下午茶傳統的基礎。在17–18世紀，BOHEA（來自「武夷」— Wuyi）一詞在歐洲是「中國茶」的同義詞。英國浪漫主義詩人拜倫在詩作《唐璜》(1819–1824) 中提到了「博希紅茶」(BOHEA)。19世紀，正山小種在中國紅茶出口中的份額達到85%（嘉慶年間, 嘉慶, 1796–1820）。

* **技術竊取：** 1848年，英國植物間諜羅伯特·福瓊（Robert Fortune）受東印度公司委派，偽裝成中國人潛入武夷山，竊取了茶樹樣本和紅茶加工技術的秘密。這一工業間諜行為開啟了印度（阿薩姆、大吉嶺）和斯裡蘭卡的茶業。

* **20世紀 — 危機與拯救：** 在第二次世界大戰和內戰期間，正山小種的生產幾乎停止。1980年代，在出口停滯和面臨完全停產威脅的情況下，傑出的中國茶學家張天福 (张天福, 1910–2017) 向福建省政府呼籲：「正山小種是福建獨特的出口產品，必須保護。」茶葉得以保存。2005年，基於正山小種傳統創製了金駿眉，啟動了「紅茶復興」，使紅茶在中國國內市場重新獲得聲望。

* **名稱：**
  * 「正山」(正山) — 「正宗/真正的山區」。意指茶葉產於正宗產區 — 桐木及具有相似海拔、氣候和傳統的周邊地區。「正」(正) 含有「正確、真正、正宗」之意，與「外山」(外山, 「外來山區」) 相對。
  * 「小種」(小种) — 「小品種」。雙重含義：(1) 茶樹屬於小葉種 (*var. sinensis*)；(2) 產量和產區有限，受高山小氣候限制。
  * 「拉普山小種」(Lapsang Souchong) — 歐洲貿易名稱。「Lapsang」詞源有爭議：可能來自「臘松」(腊松, 「蠟松」?) 或地名；「Souchong」來自「小種」(小种)。

* **文化意義：** 正山小種不僅是茶葉，更是世界級的文化遺產。它是歐洲茶文化傳統的起源，啟發了下午茶的出現，影響了殖民時代的地緣政治（鴉片戰爭、「茶葉競賽」），至今仍是東西方四個世紀茶文化交流的象徵。

**3. 植物學描述與原料：**

* **品種/栽培種：** 當地小葉種茶樹群體 — 奇種 (奇种, Qízhǒng) / 菜茶 (菜茶, Càichá)，*Camellia sinensis* var. *sinensis*。在桐木山中生長數百年的有性繁殖異質群體。每株茶樹基因獨特。小葉型提供更高的氨基酸與多酚比例，確保特有的甜味和龍眼香。
* **採摘：** 由於高海拔位置（寒冷氣候），季節開始較晚：春茶 — 從5月初（約立夏, 立夏）至5月底；夏茶 — 6月底。一年兩季。手工採摘。
* **採摘標準：** 一芽二三葉 (一芽二三叶, yī yá èr sān yè)。高等級為「一芽二葉」；標準級允許較成熟的葉片。與金駿眉（僅芽頭）和銀駿眉（芽頭+1葉）不同，正山小種使用較成熟的原料，提供厚度和「體感」。
* **原料要求：** 完整、潔淨的葉片，無粗老葉柄。採摘後快速送至加工廠。

**4. 風土與栽培特點：**

* **武夷山保護區：** 565平方公裡，聯合國教科文組織世界遺產（1999年）。紅砂岩和火山岩山脈；陡峭峽谷、瀑布、河流、亞熱帶森林，生物多樣性極其豐富。森林覆蓋率96.3%。
* **桐木村：** 「正山」的歷史核心。位於保護區深處，海拔700–1800公尺。茶樹在森林覆蓋下生長，處於半野生和野生狀態，常與竹子和松樹混生。「正山」界限：東至麻栗 (麻栗)，西至掛墩 (挂墩)，南至皮坑 (皮坑)、古王坑 (古王坑)，北至桐木關 (桐木关)。
* **生長海拔：** 海拔700–1800公尺。主要區域為1200–1500公尺。
* **氣候：** 亞熱帶山地氣候。年平均氣溫山腳約18°C，山頂約11°C。降雨量超過2300毫米/年。相對濕度80–85%。霧日超過100天/年。大氣中CO₂濃度0.026%（顯著低於城市水平）。日照時間短，霜期長。這些因素都減緩生長並促進芳香物質和氨基酸的積累。
* **土壤：** 山地紅壤和山地黃壤，弱酸性（pH 4.5–5.5），富含腐殖質，疏鬆，排水良好。土層厚度30–90公分。

**5. 製作工藝：**

正山小種有兩個工藝方向：**傳統煙燻**（在專用建築青樓, 青樓中用松煙烘乾/萎凋）和**無煙燻「新式」**（不使用松煙，強調原料天然蜜果香）。兩種都是正宗桐木產品；區別在於乾燥階段。

* **採摘 (采摘 — cǎizhāi)：** 手工採摘「一芽二三葉」。
* **萎凋 (萎凋 — wěidiāo)：** 關鍵工序，決定茶葉風格。
  * *傳統煙燻：* 萎凋在青樓 (青楼) 中進行 — 二或三層石/木建築。底層用濕潤的馬尾松 (马尾松) 木材生火；煙霧和熱氣通過地板縫隙上升到上層，茶葉攤放在竹製篩子上。溫度約30°C，葉層厚度3–7公分。每20分鐘翻動一次。茶葉同時失水、軟化並吸收松香。
  * *無煙燻：* 在通風良好的室內或露天萎凋（在少有的晴天）。無煙燻。
* **揉捻 (揉捻 — róuniǎn)：** 傳統上用腳揉 (脚揉, jiǎo róu)：製茶師在木桶中用腳揉捻茶葉；現在多使用揉捻機，但手工/腳工揉捻在高端生產商中仍有保留。目的是擠出細胞汁液，為發酵做準備。
* **發酵/氧化 (发酵 — fājiào)：** 揉捻後的茶葉放入竹籃，蓋上濕布，在25–28°C下保持至出現銅紅色和特有果香。製茶師憑感官控制過程。
* **「過紅鍋」(过红锅 — guò hóng guō)：** 正山小種獨有工序，20世紀中期失傳，由梁駿德 (梁骏德) 大師恢復。發酵後的茶葉在熱鍋中短時間炒製，停止發酵並固定香氣。並非所有生產商都採用此工序；多見於高端批次。
* **煙燻和烘乾 (熏焙 — xūn bèi / 烘干 — hōnggān)：**
  * *傳統：* 茶葉回到青樓 — 現在放在下層 (焙架, bèijià)，靠近火源。用慢燃的馬尾松木材煙燻6–8小時。茶葉呈現炭黑色和濃郁的松煙香 (松烟香, sōng yān xiāng)。
  * *無煙燻：* 在竹籃中用炭火烘乾（類似駿眉系列）或熱風烘乾。無松煙。
* **分級 (分级 — fēnjí)：** 手工最終分揀，按大小和品質分級。

**6. 感官特徵：**

煙燻和無煙燻版本的特徵差異顯著。

***傳統煙燻正山小種：***
* **乾茶外觀：** 緊實、緊結的條索，比金駿眉粗大。色澤炭黑帶油潤光澤。茸毛較少（葉片成熟）。
* **乾茶香氣：** 濃郁、複雜 — 松煙香 (松烟香)、桂圓乾 (桂圆干)、話梅、皮革、樟腦。深沉持久。
* **茶湯香氣：** 松煙背景，上層是桂圓甜香、蜂蜜、果乾。隨沖泡次數增加，煙味退去，顯露果甜。
* **滋味：** 厚實、飽滿，具特有的「焦糖絲滑感」。主調是桂圓湯味 (桂圆汤味, guìyuán tāng wèi) — 正宗正山的標誌性特徵。輕微澀感、蜂蜜、果乾。回甘悠長、甘甜，帶「高山」清涼喉韻 (喉韵, hóuyùn)。
* **湯色：** 深琥珀色、紅寶石色，深邃透亮。
* **葉底：** 大片完整葉片，銅紅色，有彈性，光澤。

***無煙燻（新工藝）：***
* **乾茶外觀：** 緊結條索，深棕色帶紅褐色調（非炭黑色）。
* **乾茶香氣：** 花果香、蜜香，帶桂圓和果乾香。無煙味。
* **茶湯香氣：** 純淨、花蜜香、桂圓、蜂蜜、淡薯香 (薯香)。接近駿眉系列風格。
* **滋味：** 比煙燻版柔和，但有明顯「厚度」。桂圓甜、蜂蜜、焦糖、淡果香。澀感較少。
* **湯色：** 金黃琥珀色、橙紅色，透亮。
* **葉底：** 銅紅色、完整、有彈性的葉片。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 乾重的10–20%。兒茶素發酵成茶黃素（0.4–2%）、茶紅素（5–11%）和茶褐素（3–9%），負責湯色、「厚度」和「絲滑感」。
* **氨基酸 (氨基酸)：** 乾重的1.5–3%。L-茶氨酸是主要成分，提供甜味（龍眼香）和放鬆效果。
* **生物鹼：** 咖啡因3–5%。可可鹼、茶鹼含量較少。
* **維生素：** C（部分保留）、B₁、B₂、B₃、E、K。
* **礦物質：** 約30種元素 — 鉀、磷、鈣、鎂、鐵、錳、氟、鋅、銅、硒。
* **精油 (芳香油)：** 約0.02%。煙燻型另含松脂揮發性成分（α-蒎烯、β-蒎烯、莰烯），賦予特有煙香。
* **其他：** 可溶性糖2–4%，果膠1–2%，有機酸約1%。

**8. 保健功效：**

* **溫熱作用：** 完全發酵的紅茶按中醫理論屬「溫性」。適合寒冷季節，特別是煙燻版 — 產生「內在溫暖」感。
* **溫和提神：** 咖啡因和L-茶氨酸的結合提供平穩、持久的提神效果，無焦慮感。
* **助消化：** 溫和刺激胃液分泌，餐後有助消化。
* **抗氧化作用：** 茶黃素和茶紅素是強效抗氧化劑。
* **心血管保健：** 多酚類改善血管彈性。
* **抗菌作用：** 茶多酚和單寧抑制病原菌。
* **抗壓效果：** L-茶氨酸促進放鬆專注。

**9. 沖泡方法：**

* **水溫：** 90–100°C。煙燻正山小種用沸水沖泡效果好；無煙燻版用90–95°C。
* **茶量：** 功夫泡法5–6克/100–120毫升；歐式泡法3–4克/200–250毫升。
* **茶具：** 瓷蓋碗 (盖碗) 100–120毫升是無煙燻版的最佳選擇。煙燻版可用宜興紫砂壺 (宜兴紫砂壶)，能柔化煙味強度。必備公道杯 (公道杯)。
* **沖泡過程：**
  1. **溫壺：** 用沸水沖燙蓋碗、公道杯和茶杯。
  2. **投茶：** 將5–6克茶放入溫熱的蓋碗。
  3. **洗茶 (润茶)：** 快速沖泡2–3秒 — 特別推薦煙燻版（柔化初始煙味）。
  4. **第一泡：** 10–15秒。
  5. **出湯：** 完全倒入公道杯，再分入茶杯。
  6. **續泡：** 5–8泡。每次增加5–10秒。煙燻版的煙味在第3–4泡後消退，顯露果甜。

**10. 儲存：**

* **容器：** 密封、不透光 — 鐵罐、鋁箔袋、陶瓷罐。
* **條件：** 乾燥、陰涼、避光，遠離異味。10–25°C，濕度60%以下。
* **期限：** 最佳風味期12–24個月。煙燻版儲存期更長（可達3年），隨時間煙味柔化，顯露果香基調。部分愛好者專門陳放煙燻正山小種2–5年。
* **注意：** 無需冷藏 — 紅茶在室溫下儲存良好。

**11. 價格與仿冒：**

正宗桐木正山小種價格因等級而異：標準級300–800元/500克；高級800–2000元；老樅 (老枞, lǎo cóng) 收藏級可達3000+元。煙燻型通常比無煙燻高端版便宜。

市場充斥仿冒品和模仿品 — 「外山小種」(外山小种, 保護區外茶葉)、「煙小種」(烟小种, 「煙燻小種」— 人工香精調味的廉價紅茶) 和以「拉普山小種」名義銷售的普通廉價紅茶。

**如何避免仿冒：**

* **檢查產地：** 正宗正山小種僅產自武夷山保護區（桐木）。要求提供生產商和產區資訊。
* **評估龍眼味：** 標誌性特徵是桂圓湯味 (桂圆汤味)。如果缺乏龍眼香而只有「粗糙煙味」，很可能是「外山」或人工煙燻茶。
* **檢查香氣：** 正宗煙燻版有柔和、包覆性的松香，與果甜融合。仿冒品有刺鼻、化學、「焦糊」煙味。
* **評估湯色：** 透亮、深琥珀紅寶石色。混濁、深暗或「髒」的湯色是低品質標誌。
* **警惕異常低價：** 50–100元/500克的正山小種幾乎肯定不是桐木產。

**12. 趣聞軼事：**

* **400年無文獻：** 正山小種的確切創製日期未見任何歷史文獻記載。首次提及「小種」(小种) 作為紅茶品種見於《清代通史》(《清代通史》)：「崇禎十三年[1640年]，紅茶（工夫茶、武夷茶、小種茶、白毫等）開始從荷蘭進入英國」。
* **嫁妝茶：** 1662年，葡萄牙公主凱瑟琳帶了幾箱「正山小種」作為嫁給英國宮廷的結婚禮物。據傳，英國女王每天早晨都以這種茶開始一天。
* **青樓 — 「綠色樓房」：** 萎凋和煙燻茶葉的獨特建築。桐木現存最古老的青樓約100年歷史，但這種結構被認為更加古老。
* **所有紅茶的祖先：** 紅茶發酵技術從桐木傳播：到江西（祁門，1876年 — 通過縣令余干臣），到雲南（滇紅），到印度（通過羅伯特·福瓊，1848年），到斯裡蘭卡，進而傳遍全世界。
* **張天福的拯救：** 1980年代生產幾乎停止時，正是茶界泰斗張天福向福建省政府的呼籲拯救了正山小種免於消失。

**13. 正山小種品種：**

* **傳統煙燻 (传统烟熏正山小种)：** 經典拉普山小種 — 濃郁松煙香。在歐洲市場和「老派」愛好者中受歡迎。
* **無煙燻/「新工藝」(新工艺正山小种)：** 無松煙，強調原料天然蜜果香。自2005年起在中國國內市場受歡迎（與駿眉系列發展同步）。
* **老樅正山小種 (老枞正山小种)：** 採用老樹茶葉（50–100+年）。更深沉、「礦物質」口感，明顯的「高山」清涼感。高端等級。
* **駿眉系列 (骏眉)：** 形式上屬於「新工藝」分支，但按採摘標準獨立成系：金駿眉（純芽頭）、銀駿眉（芽頭+1葉）、銅駿眉/赤甘（芽頭+2葉）。三者都是正山小種傳統的衍生品。
* **外山小種 (外山小种)：** 桐木保護區外生產的茶葉。不是正宗「正山」，但市場上廣泛存在。品質從可接受到低劣不等。

**結語：**

正山小種不僅是茶葉，更是連接四個世紀茶葉歷史的活線索。從桐木山中的偶然和松脂香誕生，它為世界所有紅茶開闢了道路 — 從祁門到阿薩姆，從滇紅到伯爵茶。其煙燻版 — 強勁、深沉、煙香與桂圓甜香交織 — 仍是世界茶葉風味譜中最具辨識度的味道之一。而無煙燻「新工藝」則展現了桐木原料天然的果蜜本性，讓人理解在松煙背後隱藏了數百年的真味。

品嚐真正的正山小種，就是觸摸源頭：觸摸那個茶葉意外發酵和燃燒松香為人類開啟全新飲品世界的時刻。我們至今仍生活在這個世界中。
