# 香草蘭紅茶 (香草蘭紅茶, xiāngcǎo lán hóngchá)

**香草蘭紅茶 (香草蘭紅茶, xiāngcǎo lán hóngchá)**
![香草蘭紅茶 (香草蘭紅茶, xiāngcǎo lán hóngchá)](images/Сянцао%20Лань%20Хун%20Ча%20(香草兰红茶,%20Xiāngcǎo%20Lán%20Hóngchá).png)

![香草蘭紅茶 (香草蘭紅茶, xiāngcǎo lán hóngchá)-1](images/Сянцао%20Лань%20Хун%20Ча%20(香草兰红茶,%20Xiāngcǎo%20Lán%20Hóngchá)-1.png)

香草蘭紅茶是來自海南省的獨特香料紅茶，結合了高品質的茶葉基底與天然香草蘭（香草蘭，*Vanilla planifolia*）萃取物，這種被譽為「世界食品香料之王」的植物。這款茶於1993年發明，是中國香料茶系列中最具特色的產品之一：不同於傳統的花香調味（茉莉、桂花、玫瑰），這裡採用的是熱帶蘭科香草，創造出獨特的香草巧克力風味，帶有焦糖般的甜味。

**1. 分類與產地：**

* **類型：** 再加工茶（再加工茶，zài jiāgōng chá），以紅茶（紅茶，hóngchá）為基底。
* **類別：** 調味紅茶（調味紅茶/添香紅茶）。屬於廣義的「花茶」（花茶）類別——添加天然香料的茶葉。然而，其技術與傳統茉莉花茶根本不同：不採用多次鮮花「窨製」（窨製，xūnzhì——層疊鮮花香氣），而是使用發酵香草莢萃取物的吸附方法。
* **產地：** 中國海南省（海南省，Hǎinán Shěng）。主要生產區域為萬寧市（萬寧市，Wànníng Shì）及鄰近的興隆（興隆，Xīnglóng）地區。海南是中國唯一完全位於熱帶的省份，位於北回歸線以南，為茶樹和香草蘭的栽培創造了條件。
* **地理座標：** 約北緯18°48′，東經110°23′（萬寧/興隆地區）。

**2. 歷史與文化意義：**

* **歷史：** 海南茶葉傳統雖不如大陸古老，但有其特色：島嶼的熱帶氣候允許全年種植大葉種茶樹，1950年代這裡建立了出口紅茶（紅碎茶，hóngsuì chá——CTC茶）生產基地，品質可與印度和錫蘭茶媲美。

  「香草蘭紅茶」的歷史始於1990年代初，當時海南省將發展香草蘭（香草蘭，xiāngcǎo lán）種植園納入國家五年計畫，作為熱帶高效農業的優先發展方向。亞洲開發銀行派遣專家組評估島上香草生產潛力，在萬寧、瓊海、屯昌等縣建立了種植園。

  1993年，海南香聖天然食品有限公司（海南香聖天然食品有限公司，Hǎinán Xiāngshèng Tiānrán Shípǐn Yǒuxiàn Gōngsī）與西南農業大學（西南農業大學，現為西南大學，西南大學）食品科學系合作，開發了基於香草莢萃取物的香料茶生產技術。1993年6月18日，海南省科學技術廳（海南省科學技術廳）舉行了產品官方認證——「海南省香蘭茶成果鑑定會」（海南省香蘭茶成果鑑定會）。該技術被認定為成功，並獲得國家「星火計畫」（星火計畫，Xīnghuǒ Jìhuà）項目地位——這是將科學成就應用於農業的計畫。

  2000年代，「香蘭茶」（香蘭茶，全系列的總稱——紅茶、綠茶和苦丁茶加香草）成為海南茶葉旅遊的象徵之一，也是海口專賣店和三亞度假區的熱門紀念品。到2020年代，由於天然香料茶的趨勢，對該產品的興趣增長。

* **名稱：** 「香草」（香草）——字面意思是「芳香的草」，中文對香草的通稱。「蘭」（蘭）——「蘭花」，指香草在植物學上屬於蘭科（*Orchidaceae*）。「紅茶」（紅茶）——「紅茶」。全名意為「香草蘭花紅茶」。該系列的簡稱為「香蘭茶」（香蘭茶，「香草蘭花茶」）。

* **文化意義：** 香草蘭紅茶反映了海南的熱帶特色：這個位於中國最南端的島嶼，栽培著其他地區無法種植的原料。香草蘭、可可豆、咖啡、椰子——這些都是海南獨特「熱帶風格」的一部分。這款茶成為海南熱帶產品的「名片」之一，與興隆咖啡、胡椒和椰子油齊名。

**3. 植物學描述與原料：**

香草蘭紅茶的生產使用兩種關鍵植物成分：茶葉基底和香草調味劑。

* **茶葉基底——品種/栽培種：** 海南種植的大葉種：雲南大葉種（雲南大葉種，*Camellia sinensis var. assamica*），適應海南氣候，以及海南大葉種（海南大葉種）——當地的大葉品種。也可能使用島上其他栽培種：福鼎大白茶、奇蘭（奇蘭）、福雲6號（福雲6號）。大葉原料適合作為紅茶基底：高多酚含量確保濃郁的茶湯，具有厚實的茶體，能夠承受香草調味而不失茶葉特性。
* **茶葉採摘：** 海南的熱帶氣候允許幾乎全年採摘（3月至11月），但最佳批次是春季（3-4月）。
* **茶葉採摘標準：** 一芽一二葉（一芽一二葉）。大批量生產使用更成熟的葉片。
* **香草蘭——植物學：** *Vanilla planifolia* Andrews（同義詞 *Vanilla fragrans* (Salisb.) Ames）——蘭科（*Orchidaceae*）多年生藤本植物。原產於墨西哥和中美洲；在全球熱帶地區栽培，最大生產國為馬達加斯加、印尼、烏干達。海南自1990年代初在萬寧、屯昌、瓊海和定安等縣種植香草，通常在人工遮蔭棚下。長15-25公分的莢果（果實）含有150-170種芳香化合物；其中主要的是香蘭素（香蘭素，xiānglán sù），提供特有的甜香草巧克力香氣。莢果經過漫長的發酵過程：新鮮果實經過燙漂，然後進行數週的「發汗-乾燥-調理」循環，期間酵素將葡萄糖香蘭素分解為游離香蘭素。

**4. 風土條件與栽培特點：**

海南是中國唯一完全位於熱帶地區的省份，位於北回歸線以南。

* **茶園海拔：** 從海平面到500-800公尺；主要茶園位於島嶼中南部的山麓和丘陵地區（五指山、白沙、瓊中）以及沿海地區（萬寧）。
* **氣候：** 熱帶季風氣候。年平均氣溫22-26°C。全年無霜期。年降雨量1500-2500毫米。5-10月高濕度（>80%）和充沛降雨。條件對茶樹和香草蘭都很理想，香草蘭需要持續的高濕度、散射光和不低於15°C的溫度。
* **土壤：** 紅色磚紅壤（磚紅壤，zhuānhóngrǎng），酸性（pH 5.0-6.5），排水良好，有機質含量高。部分地區（包括海口地區）的火山土壤富含礦物質。
* **特點：** 海南在1990年代經歷了「香草熱潮」，數十公頃土地種植了香草蘭。到該十年末，由於高成本、不穩定的收成和病蟲害，許多種植園被廢棄，但萬寧和興隆地區的部分農場保留下來，仍為香蘭茶生產提供原料。

**5. 製作工藝：**

工藝包括兩個階段：製作紅茶基底和隨後用天然香草萃取物調味。關鍵特點是無法應用傳統的花香「窨製」（窨製）：香草莢不像鮮花那樣釋放香氣（通過精油蒸發），而是通過發酵和濃縮的成分，需要萃取並通過吸附方法應用。

**第一階段——紅茶基底生產：**

* **採摘（採摘——cǎizhāi）：** 一芽一二葉。
* **萎凋（萎凋——wěidiāo）：** 降低葉片膨壓至彈性狀態；在海南的熱帶氣候中，由於空氣溫度較高，這個階段可能比大陸省份更短。
* **揉捻（揉捻——róuniǎn）：** 破壞細胞壁，將汁液帶到表面以均勻氧化。
* **氧化/發酵（發酵——fājiào）：** 完全氧化，形成紅茶特有的紅銅色和果香甜味。
* **乾燥（乾燥——gānzào）：** 停止氧化，固定基本香氣。
* **分級（分級——fēnjí）：** 選擇適合後續調味的部分。

**第二階段——香草萃取物調味：**

* **香草成分準備：** 發酵的香草莢（*Vanilla planifolia*）含有200多種芳香和生物活性化合物。這些物質通過萃取和濃縮方法提取——得到香草萃取物（酊劑或油樹脂），含有香蘭素、香草酸、香草醇、乙醯香蘭素、肉桂酸和數十種其他成分。
* **吸附（吸附，xīfù）：** 成品茶基底用香草萃取物處理，基於現代吸附理論的技術：茶葉作為多孔吸附劑，吸收芳香分子。過程在控制的溫度和濕度條件下進行，以確保香氣均勻分佈。
* **穩定和調整：** 吸附後，茶葉經過乾燥以固定香氣並將濕度降至安全水平。

**6. 感官特徵：**

* **乾茶外觀：** 細緊或中等捲曲；顏色深褐色，有自然光澤。高級別有金黃色芽尖。外觀是典型的優質紅茶。
* **乾茶香氣：** 明亮、獨特的香氣：前調是純淨、甜美的香草，其後是奶油焦糖調和輕微的巧克力味。紅茶的基調（麥芽、乾果）形成深厚的背景。
* **茶湯香氣：** 多層次且持久：溫暖、甜美的香草、牛奶巧克力、焦糖、奶油。隨著冷卻，茶基底的果香和花香顯現。香氣在三泡或更多泡中保持。
* **口感：** 厚實、圓潤，具有明顯的甜味和溫和的澀味。香草成分增加了奶油般的順滑和焦糖餘韻，而不壓制茶葉特性。明顯的回甘（回甘，huígān）。特別適合加牛奶——香草奶油主題得到邏輯發展。作為加冰的冷飲——清爽且具甜點感。
* **茶湯顏色：** 明亮的紅色，透明且有光澤。濃茶湯呈紅寶石色帶琥珀色調。
* **葉底（泡開的葉片）：** 紅銅色，有彈性。高級別的葉片更完整均勻。

**7. 化學成分：**

香草蘭紅茶的化學成分由茶葉基底和香草萃取物的成分組成。

* **多酚（來自茶葉基底）：** 茶黃素和茶紅素，在大葉原料完全氧化過程中形成。形成茶體、顏色和茶湯的抗氧化潛力。
* **氨基酸：** L-茶氨酸（來自茶葉）+ 17種必需氨基酸來自香草萃取物（根據*Vanilla planifolia*莢果的化學分析數據）。
* **生物鹼：** 咖啡因（來自茶葉基底）——提神效果；可可鹼和茶鹼——微量。
* **香草化合物（來自萃取物）：** 香蘭素（香蘭素，xiānglán sù）——關鍵芳香醛（發酵莢果乾重的2-3%）；香草酸（vanillic acid）；香草醇（vanillyl alcohol）；乙醯香蘭素（acetovanillone）；4-羥基苯甲醛；肉桂酸和肉桂醇；愈創木酚；茴香醛和茴香醇——以及數十種次要成分，形成「完整」的香草香氣，與整個發酵莢果的香氣無異。
* **維生素：** 維生素C、維生素B1、B2、維生素E（來自茶葉基底）。
* **礦物質：** 鉀、鎂、鈣、錳、鋅。
* **精油：** 茶葉萜類化合物（芳樟醇、香葉醇）和香草芳香成分的組合，創造出獨特的「混合」香氣。

**8. 保健功效：**

* **提神作用：** 茶葉基底的咖啡因和茶氨酸提供溫和持久的活力，沒有急劇的峰值。
* **抗氧化保護：** 紅茶的茶黃素和茶紅素是強大的抗氧化劑。
* **消化支持：** 紅茶傳統上被認為是「溫性」（溫性）並有益於消化；香草在傳統醫學中也與改善食慾和促進消化有關。
* **溫暖和放鬆作用：** 香草香氣已被證明能降低焦慮水平並創造舒適感——這種效果在芳香療法中廣泛使用。
* **認知功能支持：** 咖啡因、茶氨酸和香草化合物的協同作用有助於改善注意力和情緒。
* **強身作用：** 香草萃取物中的17種氨基酸補充了茶葉基底的氨基酸譜，擴大了營養微量元素的範圍。
* **註：** 香草在傳統中醫和拉丁美洲醫學中用作滋補和強身劑，具有強心和防腐作用。科學數據證實了香蘭素的抗氧化特性。

**9. 沖泡方法：**

* **水溫：** 90-95°C。對於精緻的芽尖批次——85-90°C。
* **茶葉用量：** 4-5克/100-120毫升（功夫泡法）；2-3克/200-250毫升（杯子或茶壺）。
* **茶具：** 瓷蓋碗（蓋碗）或瓷茶壺；玻璃茶具可以欣賞明亮的紅色茶湯。不建議使用紫砂壺——多孔的陶土可能吸收香草香氣並影響其他茶的後續沖泡。
* **過程：**
  1. 用開水預熱茶具並倒掉水。
  2. 放入茶葉並聞香——從乾茶葉就能感受到香草巧克力的香氣。
  3. 洗茶（可選）：快速沖泡1-2秒。
  4. 第一泡：8-15秒。
  5. 後續沖泡：每次增加5-10秒。
  6. 沖泡次數：4-6次；香草香氣保持到第三、四泡，然後茶葉基底繼續發揮作用2-3泡。
* **特別建議：** 香草蘭紅茶是奶茶（奶茶，nǎichá）的絕佳基底——香草焦糖風味與奶油完美和諧。也非常適合冷泡（cold brew）和冰紅茶（冰紅茶，bīng hóngchá）：將濃茶湯倒在冰上——得到清爽的甜點飲料。

**10. 儲存方法：**

密封不透明容器（鋁箔袋、鐵罐），避光、防潮和異味。溫度10-25°C。香料茶對香氣流失敏感，因此最佳飲用期為生產後12-18個月。不建議與氣味強烈的產品一起存放。開封後——在1-2個月內使用以保持香草香氣的完整性。

**11. 價格與仿冒品：**

價格取決於茶葉基底的品質、香草萃取物的等級（天然vs.合成香蘭素）和生產商品牌。天然香草是世界上最昂貴的香料之一（發酵莢果的價格在高峰年份達到每公斤500-600美元），這反映在使用天然萃取物的正宗香草蘭紅茶的成本上。

* **如何避免仿冒品：**
  1. 檢查產地：正宗的香草蘭紅茶在海南用當地原料生產。尋找包裝上的省份和地區標示。
  2. 評估香氣：天然香草萃取物提供複雜、多層次的香氣，帶有巧克力焦糖背景和數十種泛音；合成香蘭素——平淡、單一甜味、「糖果味」，缺乏深度。
  3. 茶湯應該清澈、透明、明亮的紅色；混濁或淡色茶湯是低品質基底的標誌。
  4. 口感：正宗茶保持茶葉和香草特性的平衡；如果香草完全「壓倒」茶葉或感覺到化學刺激——很可能使用了合成香料。
  5. 警惕可疑的低價：天然香草萃取物昂貴，優質香草蘭紅茶不能像普通香料茶那樣便宜。

**12. 趣聞軼事：**

* 香草蘭（*Vanilla planifolia*）是超過25,000種蘭花中唯一果實可食用的品種。其莢果必須經過3至9個月的發酵才能獲得特有香氣：新鮮莢果幾乎沒有氣味。
* 香草蘭紅茶的調味技術與傳統茉莉花茶的「窨製」（窨製）根本不同：香草不是從鮮花「吸入」香氣，而是通過吸附「吸收」濃縮萃取物——這種方法借鑒自現代食品科學。
* 海南是中國唯一可以工業化種植香草蘭的省份：該植物需要年平均溫度不低於20°C、空氣濕度>80%且無霜凍。
* 「香蘭茶」系列包括三個方向和五種產品：紅茶（袋泡和散茶）、綠茶（袋泡和散茶）和苦丁茶加香草（袋泡）。
* 香蘭茶的批量生產得益於國家「星火計畫」（星火計畫）——中國將科學成就應用於農業和食品工業的重要計畫之一。

**13. 與其他香料茶的比較：**

* **茉莉花茶（茉莉花茶，Mòlì Huāchá）——茉莉花茶：** 經典的中國香料茶（通常以綠茶或白茶為基底）。調味方式是用新鮮茉莉花多次「窨製」。風味——輕盈、花香、飄逸。香草蘭紅茶——以紅茶為基底，方向完全不同：溫暖、甜美、甜點感、巧克力香草味。兩種茶代表了香料世界的兩極。
* **桂花紅茶（桂花紅茶，Guìhuā Hóngchá）——桂花紅茶：** 用桂花（*Osmanthus fragrans*）調味的紅茶。風味——果香杏味帶蜂蜜甜味。桂花紅茶在精神上更接近香草蘭紅茶（都是紅茶基底，都是甜的），但香草蘭的香草巧克力方向與桂花的果香花香根本不同。
* **伯爵茶（Earl Grey）：** 西方香料茶——通常是錫蘭或祁門紅茶，用佛手柑（*Citrus bergamia*）油調味。如果伯爵茶是柑橘清新覆蓋茶葉苦味，那麼香草蘭紅茶是奶油般的香草溫暖覆蓋茶葉甜味。兩種茶都非常適合加牛奶，但創造完全不同的氛圍。
* **瑪薩拉茶（Masala Chai）：** 印度香料茶（豆蔻、肉桂、薑、丁香）。瑪薩拉——辛辣、刺激、溫暖；香草蘭——甜美、奶油般、包裹感。兩者都適合加牛奶，但瑪薩拉是「火」，香草蘭是「天鵝絨」。

**結語：**

香草蘭紅茶是中國最獨特和最具特色的香料茶之一，是海南獨特熱帶風土和1990年代科學勇氣的產物。將「食品香料之王」——香草蘭——與厚實、肉感的海南紅茶結合的想法出人意料地成功：香草巧克力香氣不會壓倒茶葉基底，而是補充它，創造出既熟悉又異國情調的飲料。這款茶特別適合那些喜歡甜點風味而不要人工感、重視天然成分並準備發現中國茶世界熱帶一面的人。無論是加牛奶、加冰還是從瓷蓋碗中純飲——香草蘭紅茶每次都帶來溫暖、甜美的愉悅感。
