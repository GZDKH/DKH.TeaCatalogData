# 古丈紅茶 (古丈红茶, Gǔzhàng hóngchá)

**古丈紅茶 (古丈红茶, Gǔzhàng hóngchá)**
![Гучжан Хун Ча (古丈红茶, Gǔzhàng hóngchá)](images/Гучжан%20Хун%20Ча%20%28古丈红茶,%20Gǔzhàng%20Hóngchá%29.png)

![Гучжан Хун Ча (古丈红茶, Gǔzhàng hóngchá)-1](images/Гучжан%20Хун%20Ча%20%28古丈红茶,%20Gǔzhàng%20Hóngchá%29-1.png)

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá) — 全發酵（氧化）茶。
* **類別：** 中國工夫紅茶 (工夫红茶, gōngfū hóngchá)。地理標誌產品 (地理标志产品, dìlǐ biāozhì chǎnpǐn)。
* **產地：** 中國湖南省 (湖南, Húnán) 湘西土家族苗族自治州 (湘西土家族苗族自治州, Xiāngxī Tǔjiāzú Miáozú Zìzhìzhōu) 古丈縣 (古丈县, Gǔzhàng Xiàn)。生產區域涵蓋古陽鎮 (古阳镇)、高峰鎮 (高峰镇)、默戎鎮 (默戎镇)、斷龍山鎮 (断龙山镇)、紅石林鎮 (红石林镇)、坪壩鎮 (坪坝镇) 和岩頭寨鎮 (岩头寨镇)。
* **地理座標：** ≈ 28.6° 北緯，109.9° 東經。

**2. 歷史與文化意義：**

* **歷史：** 古丈縣的茶葉種植有著悠久的歷史。據《湖南茶葉史話》(《湖南茶叶史话》) 記載，早在東漢時期 (东汉, Dōng Hàn, 西元25–220年)，包括現今古丈在內的武陵地區 (武陵) 就以中國主要產茶區之一而聞名。唐朝時期 (唐, Táng, 618–907年)，當地茶葉成為朝廷貢品——此事實在《新唐書·地理志》(《新唐书·地理志》) 中得到證實：「溪州土貢茶芽」(溪州土贡茶芽)。後來，在清朝時期 (清, Qīng)，湖南將軍楊占鰲 (杨占鳌, Yáng Zhànáo)，古丈人，前甘肅提督 (甘肃提督)，回鄉後創立了「綠香園」(绿香园) 茶莊，開始大規模生產紅茶，借鑒了傳統當地「藥茶丸」(药茶丸, yào chá wán) 的技術元素——這是發酵茶的雛形，士兵隨身攜帶以防瘧疾。1957年，古丈綠茶在萊比錫世界博覽會上獲得金獎，1982年被評為中國十大名茶之一。古丈紅茶的現代歷史與福建「正山堂」(正山堂) 公司和多個科研院所進入該地區有關，共同開發了能夠發揮當地小葉種原料潛力的高級紅茶生產技術。2017年12月8日，中國國家質檢總局 (国家质检总局) 正式授予「古丈紅茶」地理標誌產品地位（2017年第108號令）。

* **名稱：** 古丈 (Gǔzhàng) — 地名，位於武陵山脈山區中心的縣名。詞源可追溯至古代行政區劃，自五代時期和土司 (土司) 統治者時代保留至今，當時酉水 (酉水) 沿岸地區是區域茶葉生產和貿易中心。紅茶 (hóngchá) — 字面意思為「紅茶」，指全發酵茶類別。因此，全名意為「來自古丈[縣]的紅茶」。

* **文化意義：** 古丈是名副其實的「茶葉之鄉」(茶叶之乡)：茶園面積超過2.05萬畝（約1367公頃），每位居民平均擁有超過1.3畝茶園。茶葉是縣域經濟的主要產業，總產值超過200億元人民幣（2023年數據）。古丈每年舉辦茶旅文化節 (茶旅文化节)，結合品茶、技藝競賽和生態旅遊。該地區位於張家界和鳳凰之間的「黃金路線」上，使茶葉成為湘西主要旅遊象徵之一。古丈紅茶產品線包括「丈巴紅」(丈巴红)、「老古樹紅茶」(老古树红茶)、「黃金紅茶」(黄金红茶)、「小葉紅茶」(小叶红茶) 等子品牌。

**3. 植物學描述與原料：**

* **品種/栽培種：** 原料基礎主要由當地小葉種群體種 (群体种, qúntǐ zhǒng) *Camellia sinensis* var. *sinensis* 組成，數百年來適應了湘西山區氣候。近年來也並行栽培選育品種：來自鄰近保靖縣的黃金茶 (黄金茶, Huángjīn Chá)，以及從浙江省引進的品種，包括安吉白茶系列無性系。當地小葉種品種具有高度可塑性——可生產綠茶、紅茶、白茶、黃茶和黑茶。
* **採摘：** 春茶採摘（3-4月）是主要的：春季嫩芽提供最高的胺基酸濃度和特有的甜味。夏茶和秋茶採摘也用於紅茶生產，但被認為品質較低。
* **採摘標準：** 根據等級而定：單芽 (单芽, dān yá) 用於最高等級，一芽一葉 (一芽一叶, yī yá yī yè) 用於中等級，一芽二葉 (一芽二叶, yī yá èr yè) 用於大宗產品。
* **原料要求：** 完整、無損的新鮮葉片；優選手工採摘；採摘與開始萎凋之間的最短間隔——不超過4-6小時。

**4. 風土條件與栽培特點：**

* **生長海拔：** 海拔250-1100公尺。主要茶園位於海拔400-800公尺。
* **氣候：** 亞熱帶山地季風氣候，濕潤。年平均氣溫約16°C，極端高溫（>35°C）平均每年僅15天。年平均降水量1476毫米，分佈相對均勻。平均相對濕度81%。年平均日照時數1304小時。生長期（日平均氣溫>10°C）約241天，活動積溫總和約4997°C。
* **土壤：** 山地黃壤 (山地黄壤) 和紫砂土 (紫砂土)，發育於砂岩和板岩上。pH值5.0-6.5。有機質含量≥2.0%。土層厚度≥40公分。土壤富含磷——形成礦物質特徵和濃郁口感的因素。
* **農業技術：** 主要使用有機肥和專用茶葉肥料；化學肥料與有機肥結合使用。超過8000畝獲得有機茶園認證。廣泛採用病蟲害智能監測系統和科學合理的農藥施用。種植密度：單行種植不超過4000株/畝，雙行種植不超過5000株/畝。

**5. 生產工藝：**

古丈紅茶採用工夫紅茶 (工夫红茶) 工藝生產，具有一些地方特色。特點是增加了兩個額外工序——日光烘焙和鐵鍋提香 (铁锅提香)——增強香氣並賦予乾茶葉特有的緊實度。

* **採摘 (采摘, cǎizhāi)：** 在早晨或傍晚時分手工挑選符合標準的嫩芽。
* **萎凋 (萎凋, wěidiāo)：** 薄攤的茶葉失去20-30%水分；採用複合萎凋——短時間日光微凋 (日光微凋) 後在室內完成。持續時間12-18小時。
* **揉捻 (揉捻, róuniǎn)：** 形成葉片結構並釋放細胞汁液。對小葉種原料採用輕壓力。
* **發酵/氧化 (发酵, fājiào)：** 在22-28°C溫度和高濕度（≥90%）條件下控制氧化3-5小時。正是在這個階段形成茶黃素和茶紅素，負責顏色、「體感」和特有甜味。
* **初乾/過紅鍋 (初干 / 过红锅, guò hóng guō)：** 直徑約80公分的鐵鍋加熱至200-220°C；茶葉處理5-10分鐘至含水量20-25%。這個步驟是古丈工藝的招牌特色。
* **日光烘焙 (日光烘焙, rìguāng hōngbèi)：** 額外工序，增強蜂蜜果香調。
* **最終乾燥/鐵鍋提香 (铁锅提香, tiě guō tí xiāng)：** 在鐵鍋上短暫加熱，固定香氣特徵。
* **分級 (分级, fēnjí)：** 按規格分離，篩除粉末和破損碎片，形成商品批次。

**6. 感官特徵：**

* **乾茶外形：** 條索狀；緊實、均勻的捲曲；顏色為深栗色帶青銅光澤。高等級產品有豐富的金黃色茸毫 (金毫, jīn háo)，使茶葉外觀華麗。
* **乾茶香氣：** 純淨、溫暖，帶有明顯的蜂蜜、杏乾香調和淡淡的烤大麥味。來自老樹的批次——額外的深度與可可調性。
* **茶湯香氣：** 層次豐富：前幾泡展現蜂蜜甜香和花香調（山地蜜源花卉），中段——乾果（杏乾、葡萄乾）和焦糖，後段——麵包和堅果調性。
* **滋味：** 醇厚、圓潤，具有特有的「油潤」體感。純淨、不突兀的甜味；適度澀味，迅速轉為持久、溫暖的回甘 (回甘, huígān)。優質批次——明顯的花蜜和成熟桃子調性。
* **湯色：** 從金黃琥珀色（芽茶批次）到濃郁的紅琥珀色；明亮、透明，杯緣有明顯的金圈 (金圈)。
* **葉底（沖泡後茶葉）：** 葉片完全舒展且有彈性；顏色從銅棕色到紅栗色，均勻，無暗斑。葉片彈性證明原料品質和正確發酵。

**7. 化學成分：**

* **多酚類：** 在成品紅茶中，大部分兒茶素轉化為茶黃素 (茶黄素, cháhuángsù) 和茶紅素 (茶红素, cháhóngsù)。根據古丈紅茶分析數據，茶黃素含量為0.7-1.0%（採用改進方法測定可達1%，接近中國工夫紅茶的理論最大值）。高茶黃素含量確保湯色明亮和「澀甜」口感。
* **胺基酸：** 游離胺基酸含量為乾重的3-5%。L-茶胺酸 (L-茶氨酸) 是主要胺基酸，負責柔和、甜美回味和與咖啡因的協同效應（平靜的提神而無焦慮感）。
* **生物鹼：** 咖啡因約為乾重的2.5-4.0%（略低於滇紅等大葉種紅茶）。可可鹼和茶鹼含量微量。
* **維生素：** B群維生素（B₁、B₂、B₆）、維生素C（少量，大部分在發酵過程中被破壞）、維生素PP（菸鹼酸）。
* **礦物質：** 鉀、鎂、錳、鋅、硒（微量）、磷（由於該地區含磷土壤而含量較高）。
* **精油和揮發性化合物：** 萜烯醇類（芳樟醇、香葉醇）、醛類和美拉德反應產物的複合體，形成蜂蜜焦糖香氣，特別是通過鐵鍋雙重加熱而增強。

**8. 健康功效：**

* **溫和提神：** 咖啡因和L-茶胺酸的結合提供持久、平穩的精神狀態，無咖啡特有的「峰谷效應」。
* **抗氧化保護：** 茶黃素和茶紅素表現出明顯的自由基清除能力，支持細胞健康。
* **消化保健：** 發酵多酚對胃黏膜的刺激比綠茶兒茶素溫和；餐後溫熱紅茶有助於舒適消化。
* **心血管支持：** 定期適量飲用紅茶與維持血管彈性相關，這歸功於鉀和多酚類。
* **認知功能：** L-茶胺酸有助於注意力集中和改善腦力工作品質，特別是與咖啡因結合時。
* **溫暖效應：** 紅茶增強末梢血液循環，有助於減少主觀疲勞感，在寒冷季節特別有價值。
* **免疫支持：** 多酚類和微量元素（鋅、錳）在定期飲用時對整體免疫平衡有貢獻。
* **肌膚狀態：** 紅茶抗氧化劑有助於中和氧化壓力，定期飲用可能對肌膚緊緻度和彈性產生積極影響。

**9. 沖泡方法：**

* **水溫：** 90-95°C（純芽等級85-90°C，避免燙傷嫩芽）。
* **茶量：** 4-5克配100-120毫升（工夫泡法）；3-4克配200毫升（杯泡/壺泡歐式方法）。
* **茶具：** 瓷蓋碗 (盖碗, gàiwǎn) — 展現香氣的最佳選擇；宜興紫砂壺 (宜兴紫砂壶) — 獲得更圓潤、「絲滑」的口感；玻璃茶壺 — 視覺享受湯色。
* **沖泡過程：**
  1. 用沸水溫壺並倒掉水。
  2. 投茶；讓茶葉在蓋下「醒茶」15-20秒，吸聞上升的香氣。
  3. 第一泡：8-10秒。對於緊結茶葉可預先洗茶（1-2秒）。
  4. 第2-4泡：10-15秒。
  5. 第5-8泡：每泡增加5-10秒。
  6. 優質古丈紅茶可沖泡6-8次；老樹批次可達10-12次。

**10. 儲存：**

* **容器：** 密封、不透光：鐵罐、鋁箔袋配拉鍊或真空包裝。
* **條件：** 乾燥、陰暗、涼爽處，溫度15-25°C。避免與強烈氣味產品（香料、咖啡）鄰近。
* **保質期：** 最佳飲用期為生產後12-18個月內。優質緊結批次在適當儲存條件下可在2-3年內「圓熟」和改善，獲得更深層的蜂蜜焦糖調性。

**11. 價格與仿冒：**

* **價格區間：** 中國紅茶中等及中高端市場。價格從大宗等級的300-500元人民幣/500克到純芽或老樹競賽級的2000-5000元人民幣/500克不等。影響價格的主要因素：採摘標準（芽vs.葉）、茶園海拔、樹齡、生產者聲譽。
* **如何避免仿冒：**
  1. 從有批次可追溯性的授權經銷商購買；在包裝上尋找地理標誌專用標誌 (地理标志专用标志)。
  2. 評估茶葉：正宗古丈茶具有均勻、緊實的捲曲，無粉末和雜質；金黃茸毫是優質原料的標誌。
  3. 檢查香氣：純淨，無焦糊粗糙、酸味或霉味。
  4. 評估茶湯：應明亮、透明，無混濁懸浮物；杯緣金圈是高茶黃素含量的指標。
  5. 對聲稱為高端等級但價格可疑偏低的產品保持懷疑態度。

**12. 有趣事實：**

* 古丈紅茶的早期雛形是「藥茶丸」(药茶丸)，土家族和苗族戰士以手工方式製作：將新鮮茶葉在衣袋中用手揉搓，利用體溫發酵，然後晾乾。這種「野戰藥箱」被帶到軍事遠征中防治瘧疾和腸胃疾病——實質上是最古老的民間茶葉發酵方法之一。
* 古丈縣位於所謂的「世界茶葉黃金產業帶」(世界茶叶黄金产业带) 上，這一點得到地理和氣候特徵的證實。
* 2023年在第四屆湘西州名優茶評比 (湘西州第四届名优茶评比) 中，三位古丈茶藝師在紅茶類別中獲得一等獎——證明了當地工藝學校的高水準。
* 古丈的「英妹子茶業」(英妹子茶业) 公司正在實驗將紅茶壓製成磚茶 (红茶砖)，其中發展出「金花」(金花, jīn huā) — 有益黴菌培養物 *Eurotium cristatum*，這是湖南黑茶的特徵。
* 古丈紅茶已進入國際市場：「牛角山」(牛角山) 品牌產品在英國銷售，公司積極參加法國茶葉展覽。
* 有趣的是，在技術現代化過程中，一些古丈生產者創造了創新產品——「梔香小葉紅」(栀香小叶红)（帶梔子花香的小葉種紅茶）。這是傳統茶葉適應重視明亮花香與傳統紅茶甜味結合的年輕一代口味的創意方法例子。
* 古丈縣茶園總面積超過2.05萬畝，其中1.65萬畝處於活躍採摘階段。年茶葉產量約14,250噸，其中紅茶約5,700噸，使古丈成為湖南省最大的紅茶產區之一。

**13. 與其他紅茶的比較：**

* **祁門紅茶 (祁门红茶, Qímén Hóngchá)：** 來自安徽祁門縣的經典紅茶。以特有的「祁門香」(祁门香) 著稱——蘭花和蜂蜜調的複雜香氣。古丈紅茶通常表現出更醇厚的體感和明顯的焦糖甜味，而祁門更精緻、「花香」結構。
* **正山小種 (正山小种, Zhèngshān Xiǎozhǒng)：** 福建紅茶鼻祖；經典批次具有松木煙燻香氣。古丈紅茶無煙燻味，更接近「純淨」的蜂蜜果香特徵。
* **滇紅 (滇红, Diānhóng)：** 雲南大葉種 *var. assamica* 紅茶；通常更強勁、「厚實」，具有明顯的巧克力體感。由小葉種原料製成的古丈紅茶更精緻，花蜜調更高，澀味較不強烈。
* **湖南黃金紅茶 (湖南黄金红茶)：** 由鄰近保靖縣種植的黃金茶品種製成的紅茶。具有明亮花香和因品種高胺基酸含量而增強的甜味。由當地小葉種原料製成的古丈紅茶稍微更澀，具有更大的礦物質深度。
* **九層山紅茶 (九层山红茶, Jiǔcéngshān Hóngchá)：** 來自臺灣高山地區的紅茶。以特有的「臺灣式」甜味和淡淡果酸著稱。古丈紅茶具有更「土壤」、礦物質特徵和更溫暖的香氣特徵，這歸因於鐵鍋雙重加熱。

**結論：**

古丈紅茶是一款有個性的紅茶：其中武陵山千年山土、霧谷清涼和師傅「提香」的鐵鍋溫暖交織成統一的蜂蜜焦糖和弦。這款茶特別適合悠閒的晚間茶時光，以及那些在紅茶中欣賞的不是粗獷力量，而是平衡甜味、湯色純淨和持久溫暖回味的人，讓人想起湘西山蜜。
