# 碧螺紅茶 (碧螺红茶, bìluó hóngchá)

**碧螺紅茶 (碧螺红茶, bìluó hóngchá)**
![Було Хун Ча (碧螺红茶, bìluó hóngchá)](images/Було%20Хун%20Ча%20%28碧螺红茶,%20Bìluó%20Hóngchá%29.png)

![Було Хун Ча (碧螺红茶, bìluó hóngchá)-1](images/Було%20Хун%20Ча%20%28碧螺红茶,%20Bìluó%20Hóngchá%29-1.png)

碧螺紅茶是一種紅茶，採用與著名綠茶洞庭碧螺春 (洞庭碧螺春, Dòngtíng Bìluóchūn) 相同的原料和產區，但以全發酵工藝製作。這是一個相對年輕的產品，於2010年代出現，作為合理利用不適合製作精品綠茶原料的方式，並迅速在愛好者中贏得了自己的市場地位。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá) — 全發酵茶。
* **類別：** 中國地方紅茶；工夫紅茶 (工夫红茶, gōngfū hóngchá)。基於歷史風土的現代創新茶品。
* **產地：** 中國江蘇省 (江苏, Jiāngsū)，蘇州市 (苏州市, Sūzhōu shì)，吳中區 (吴中区, Wúzhōng qū)；洞庭山 — 洞庭東山半島 (洞庭东山, Dòngtíng Dōngshān) 和洞庭西山島 (洞庭西山, Dòngtíng Xīshān)，位於太湖 (太湖, Tàihú)。
* **地理座標：** ≈ 31.07° 北緯，120.38° 東經（東山 — 西山地區，蘇州西南部）。

**2. 歷史與文化意義：**

* **歷史：** 洞庭山地區產茶已有千餘年歷史：唐朝 (唐, 618–907) 時期史書中已提及「洞庭茶」，宋朝 (宋, 960–1279) 時期西山的「水月茶」(水月茶) 被列入貢茶 (贡茶) 名單。據王應奎《柳南隨筆》(《柳南随笔》) 記載，茶葉於1699年（康熙38年）獲得現今「碧螺春」(碧螺春) 之名，當時巡撫宋犖 (宋荦) 將當地茶葉進獻給康熙皇帝 (康熙)，此茶原本被稱為俗名「嚇煞人香」(吓煞人香，「令人驚豔的香氣」)。皇帝認為民間名稱不雅，遂賜名「碧螺春」— 「翠綠螺旋的春天」。

  然而，用相同原料製作紅茶是新興現象。對碧螺春紅茶的興趣約始於2010年代，原因有幾個：首先，洞庭山每年在清明 (清明) 和穀雨 (谷雨) 之後採摘的茶葉量增加，這些茶葉不符合精品綠茶碧螺春的要求；其次，手工炒製綠茶的師傅數量減少；第三，紅茶更易儲存和運輸。因此，碧螺春紅茶從副產品轉變為獨立商品。自2019年起，碧螺紅茶價格開始上漲，顯示其形成了自己的市場地位。2025年，吳中區發布了《洞庭山紅茶團體標準》(《洞庭山红茶团体标准》)，確立了生產要求。

* **名稱：** 碧 (bì) — 「翠綠、碧玉色」；螺 (luó) — 「螺旋、蝸牛」（指茶葉特有的螺旋捲曲）；紅茶 (hóngchá) — 「紅茶」。因此，名稱直接表明：這是用碧螺春風土製作的紅茶，承襲了綠茶原型特有的螺旋捲曲。

* **文化意義：** 碧螺紅茶是著名茶區創意多元化的典範：不是綠茶的替代品，而是其「紅色鏡像」，讓相同的風土 — 特別是獨特的茶果複合系統 (茶果复合系统) — 透過完全不同的風味特徵得以展現。這款茶成為新一代洞庭茶農的名片。

**3. 植物學描述與原料：**

* **品種/栽培種：** 洞庭山群體小葉種 (洞庭山群体小叶种, Dòngtíngshān qúntǐ xiǎoyè zhǒng)，屬於 *Camellia sinensis* var. *sinensis*。俗稱包括「柳葉條」(柳叶条，「柳葉條」) 和「醬板頭」(酱板头)。這是製作綠茶碧螺春所用的相同正宗基因庫 — 在太湖條件下經過千餘年栽培自然選育的群體。植株為低矮灌木（0.7–2.5公尺），葉片小而密實，芳香物質含量高。
* **採摘：** 高等級紅茶使用清明 (清明, ≈ 4月5日) 前採摘的原料 — 「明前」(明前)；主要產量來自清明後至穀雨 (谷雨, ≈ 4月20日) 及更晚期間 — 四月底至五月初。
* **採摘標準：** 精品批次 — 一芽一葉或一芽二葉 (一芽一叶 / 一芽二叶, yī yá yī yè / yī yá èr yè)。標準紅茶允許更成熟的採摘 — 一芽二三葉 (一芽二三叶)，包括晚季採摘的「大葉子」(大叶子)。
* **原料要求：** 新鮮完整葉片，無機械損傷。紅茶的關鍵要求 — 在開始控制萎凋階段前不得有「紅頭」(红头)，即茶葉在運輸過程中不應開始自發氧化。

**4. 風土與栽培特色：**

* **海拔高度：** 海拔50–293公尺。主要茶園位於東山斜坡（主峰莫釐峰 (莫厘峰, 293.5公尺)）和西山丘陵。
* **氣候：** 亞熱帶季風氣候，受太湖水體顯著調節 — 長江三角洲最大淡水湖（面積 ≈ 2,250平方公裡）。冬季溫和，夏季不炎熱，年均溫約15.5–16.5°C，年降水量1,000–1,200毫米，空氣濕度75–85%。來自湖面的雲霧是茶園的常伴，提供散射光線，有利於茶葉中胺基酸的積累。
* **土壤：** 黃棕色森林土和紅黃壤 (红黄壤, hónghuáng rǎng)，酸度pH 4.5–6.0；因數百年果樹落葉而富含有機質和磷。
* **農業技術：** 關鍵特色 — 茶果複合系統 (茶果复合系统, chágǔo fùhé xìtǒng)，2020年被中國農業部認定為「中國重要農業文化遺產」。茶樹生長在枇杷 (枇杷, pípa)、楊梅 (杨梅, yángméi)、柑橘 (柑桔, gānjú)、桃樹、李樹、栗樹等果樹蔭下。果樹與茶樹根系交織；落花落果滋養土壤，茶葉吸收果香 — 正是這種方式形成了著名的「花香果味」(花香果味, huāxiāng guǒwèi) 特徵，使洞庭原料區別於中國任何其他產區。森林覆蓋率達80%，提供天然遮蔭。

**5. 製作工藝：**

與綠茶碧螺春四個關鍵步驟（殺青 — 揉捻 — 搓團顯毫 — 乾燥）在一個鍋中30–40分鐘完成不同，紅茶製作是長期過程，需要3–4天，每個階段都需要精確控制溫度、濕度和時間。

* **採摘 (采摘, cǎizhāi)：** 手工採摘，選擇嫩芽；紅茶允許比綠茶碧螺春更成熟的葉片。

* **萎凋 (萎凋, wěidiāo)：** 將新鮮採摘的茶葉薄攤在通風良好、無直射陽光的室內。專業師傅使用特製萎凋架 (萎凋池)，控制溫度以均勻降低濕度。目標是使茶葉柔軟有彈性（握拳時茶葉應有彈性，不折斷，如麵團狀）。定期翻動茶葉，防止過早變紅。持續時間 — 根據天氣條件8–16小時。

* **揉捻 (揉捻, róuniǎn)：** 使用機械滾筒機（與綠茶碧螺春手工捲曲不同）。壓力原則：從輕到重再回到輕，總時長約一小時。目標 — 破壞細胞壁，釋放汁液和酶，形成碧螺春風格特有的緊密螺旋捲曲。過度壓力會導致茶葉破碎。

* **發酵/氧化 (发酵, fājiào)：** 將揉捻後的茶葉置於溫暖潮濕環境（溫度 ~25–28°C，濕度 ≥ 90%）4–8小時。兒茶素氧化成茶黃素 (金黄) 和茶紅素 (红褐)，形成紅琥珀色茶湯和特有的甜蜜蜂蜜果香。發酵停止時機通過感官判斷：茶葉呈現紅銅色澤，帶有明顯果香。

* **乾燥/烘焙 (烘干, hōnggān)：** 在適中溫度（80–110°C）下進行，以固定酵素特徵並將殘餘水分降至 ≤ 6%。兩階段乾燥：初次高溫乾燥停止氧化，然後在較溫和條件下最終「熟成」，發展香氣深度。

* **分級 (分级, fēnjí)：** 將成品茶按等級分類 — 從富含金毫的芽茶到較大葉片的茶葉。

**6. 感官特徵：**

* **乾茶外觀：** 細長、緊密捲曲的「螺旋」，長1–2公分；顏色從深栗色到黑色，散佈金色芽毫 (金毫, jīnháo)。高等級展現豐富絨毛和明顯「青銅」光澤。
* **乾茶香氣：** 明亮、甜美，帶有明顯蜂蜜、乾杏和溫暖焦糖香調；背景有淡雅花香，承襲自與果樹相鄰的環境。
* **茶湯香氣：** 層次豐富：初泡時 — 蜂蜜、荔枝、熟杏；隨著沖泡發展出烤栗子、麵包皮和淡可可香調。特有的「果花香」(花果香) — 正宗洞庭產地的主要標誌。
* **滋味：** 醇厚、圓潤，帶有明顯天然甜味 (回甘, huígān) 和溫和、不粗糙的澀味。茶湯質感「絲滑」，帶有愉悅的溫暖厚度。回甘持久、甜美，帶果香調。
* **湯色：** 紅琥珀色，透明明亮，杯緣有明顯「金圈」(金圈, jīnquān) — 茶黃素含量高的標誌。
* **葉底（沖泡後茶葉）：** 葉片舒展有彈性且均勻；色澤從銅紅到栗色；優質批次茶葉完整，葉脈清晰，無燒焦痕跡。

**7. 化學成分：**

* **多酚類：** 在完全氧化過程中，主要兒茶素（EGCG、EGC、ECG）轉化為茶黃素（TF，約佔乾重0.5–1.5%）和茶紅素（TR，約6–12%）。茶黃素負責「金圈」和茶湯明亮度，茶紅素負責滋味厚度和「體感」。總多酚含量約佔乾重10–15%（低於綠茶碧螺春的20–30%）。
* **胺基酸：** L-茶胺酸是關鍵胺基酸，提供溫和、甜味和放鬆效果。由於小葉種和太湖霧氣微氣候，洞庭原料的胺基酸含量較高。
* **生物鹼：** 咖啡因（約佔乾重2.5–4.0%）、可可鹼、茶鹼。咖啡因與L-茶胺酸的協同作用提供溫和、專注的提神效果，無「跳躍」和隨後的疲勞。
* **維生素：** 微量B群維生素（B₁、B₂、B₆）、維生素E；維生素C在氧化過程中部分破壞（與綠茶保存最大量不同）。
* **礦物質：** 鉀、鎂、錳、鋅、氟、磷 — 源於洞庭土壤的豐富性。
* **揮發性芳香化合物：** 芳樟醇、香葉醇、順式茉莉酮、β-紫羅蘭酮、苯乙醛，以及最終乾燥時形成焦糖蜂蜜香調的美拉德反應產物。洞庭原料的特色 — 果香類萜烯化合物含量較高（來自「吸收」的枇杷和柑橘香氣）。

**8. 健康功效：**

* **溫和提神：** 咖啡因與L-茶胺酸複合物提供持續清醒和專注力提升，無咖啡特有的急劇「峰值」。
* **抗氧化保護：** 茶黃素和茶紅素是強效抗氧化劑，有助中和自由基；某些研究顯示，紅茶的抗氧化活性可與綠茶相媲美。
* **消化支持：** 紅茶傳統上被認為「溫性」（中醫術語），對胃部溫和；適合餐後飲用，有助舒適消化。
* **心血管系統：** 適量飲用紅茶與維持血管彈性和降低「壞」膽固醇（LDL）水平相關。
* **溫暖效果：** 寒冷季節，碧螺紅茶是理想的溫暖飲品，減少主觀疲勞感。
* **免疫支持：** 多酚具有抗菌和抗病毒特性；系統性飲茶增強整體抵抗力。
* **口腔衛生：** 氟化物和多酚有助預防蛀牙和減少口腔病原菌。

**9. 沖泡方法：**

* **水溫：** 90–95°C。沸水（100°C）可能增強澀味；芽茶批次最好使用88–90°C。
* **茶量：** 4–5克配100–120毫升（工夫泡法）；3–4克配200毫升（杯泡法）。
* **茶具：** 蓋碗 (盖碗, gàiwǎn) 瓷器 — 通用選擇；瓷茶壺；宜興紫砂壺 (宜兴紫砂壶) — 如偏好更圓潤、「天鵝絨」般的口感。
* **沖泡過程：**
  1. 用熱水溫熱茶具，倒出。
  2. 投茶；聞「甦醒」乾茶香氣。
  3. 洗茶：快速沖泡1–2秒（新鮮批次非必需，但緊結茶可洗茶）。
  4. 第一泡：8–10秒。
  5. 第2–4泡：10–15秒。
  6. 之後每泡增加5–10秒。
  7. 優質碧螺紅茶可沖泡6–8次，層次展現：從明亮果蜜香調到更深層的焦糖麵包香調。

**10. 儲存：**

* 密封不透光容器（鐵罐、鋁箔袋配單向閥）。避免異味、直射光、潮濕和高溫。
* 最佳條件：15–25°C，濕度 ≤ 60%，陰涼乾燥處。無需冷藏（與綠茶碧螺春不同）。
* 建議飲用期：6–18個月以展現完整香氣；優質緊結批次可「圓熟」至2–3年而不失特色。

**11. 價格與仿冒：**

* **價格區間：** 根據2025年數據，正宗洞庭原料碧螺紅茶價格範圍：
  - 特一級 (特一级, tè yī jí)：≈ 1,500元人民幣/斤（500克）；
  - 特二級 (特二级)：≈ 1,200元人民幣/斤；
  - 一級 (一级)：≈ 750元人民幣/斤；
  - 二級 (二级)：≈ 550元人民幣/斤；
  - 三級 (三级)：≈ 350元人民幣/斤；
  - 普通紅茶：約250元人民幣/斤起。
  相比之下：高等級綠茶碧螺春可達8,000–56,000元人民幣/公斤，因此紅茶版本明顯更實惠。
* **如何避免仿冒：**
  1. **購買可追溯產品：** 尋找「洞庭山」(洞庭山) 標識和吳中區地理標誌 (地理标志)；要求具體茶園資訊。
  2. **評估茶葉：** 正宗碧螺紅茶保持小葉種原料的螺旋捲曲 — 細密緊實；粗大螺旋表明非本地產。
  3. **檢查香氣：** 關鍵標誌 — 果花香 (花果香)，承襲自茶果系統；浙江或雲南仿品缺乏此特徵。
  4. **觀察茶湯：** 茶湯應透明、紅琥珀色，無混濁；杯壁「金圈」是好兆頭。
  5. **對低價保持懷疑：** 如「洞庭」紅茶售價低於200元人民幣/斤，很可能是其他省份原料或秋茶冒充春茶。

**12. 趣聞軼事：**

* 洞庭茶果複合系統 (茶果复合系统) 於2020年入選「中國重要農業文化遺產」(中国重要农业文化遗产)，2022年綠茶碧螺春製作技藝成為聯合國教科文組織非物質文化遺產名錄的組成部分。
* 製作500克精品綠茶碧螺春需要60,000–80,000個芽頭。允許更成熟葉片的紅茶在原料使用上更「經濟」，使其成為環保合理的產品 — 物盡其用。
* 名稱悖論：「碧」(bì) — 「翠綠」，而茶是紅色的。這是對原始綠茶碧螺春和共同風土的有意致敬，而非成品顏色。
* 碧螺紅茶是中國少數用小葉種 (小叶种) 原料製作的紅茶之一，而大多數著名紅茶（滇紅、英紅）用大葉種 (大叶种) 製作。小葉種帶來更精緻、「絲滑」的茶湯特性。
* 在盲品中，專家注意到碧螺紅茶前兩泡的香氣強度可與武夷山紅茶（正山小種 (正山小种)、金駿眉 (金骏眉)）競爭，儘管洞庭茶在多次沖泡的耐泡性上略遜。

**13. 與其他紅茶比較：**

* **正山小種 (正山小种, Zhèngshān Xiǎozhǒng)：** 福建武夷山經典紅茶。以煙松香氣（傳統版本）和更「深沉」、礦物質風味著稱。碧螺紅茶相對展現明亮果花香和承襲自洞庭風土的溫和特質。
* **祁門紅茶/祁紅 (祁门红茶, Qímén Hóngchá)：** 著名安徽工夫紅茶，具特有「蘭花香」(祁门香)。祁門更乾爽、「酒香」；碧螺紅茶更「蜜甜」圓潤，果香明顯。
* **滇紅 (滇红, Diānhóng)：** 雲南大葉種阿薩姆亞種紅茶；濃郁、麥芽味，金毫豐富。碧螺紅茶更精緻細膩，具小葉種原料的「絲滑」質感。
* **九曲紅梅 (九曲红梅, Jiǔqū Hóngméi)：** 杭州（浙江）紅茶，風格上最接近的「鄰居」：同樣小葉種原料，精緻捲曲，細膩風味。主要區別 — 紅梅缺乏洞庭「果花香」，這源於茶果系統。

**結語：**

碧螺紅茶可能是著名洞庭風土在21世紀給茶界最意外的禮物。在綠茶碧螺春數百年來獨領風騷的地方，出現了一款不模仿而是重新詮釋傳統的紅茶。它將茶園東山和西山著名的果花氛圍 — 橙花叢蔭、枇杷和柑橘樹影 — 轉移到完全不同的味覺維度：溫暖、蜜甜、暖心。這是秋冬悠閒茶時光的理想茶品，適合那些珍視溫和、純淨甜味和每一杯中地域感的人們。
