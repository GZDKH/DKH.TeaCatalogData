# 龍脊紅茶 (龙脊红茶, Lóngjǐ Hóngchá)

**龍脊紅茶 (龙脊红茶, Lóngjǐ Hóngchá)**
![Лунцзи Хун Ча (龙脊红茶, Lóngjǐ Hóngchá)](images/Лунцзи%20Хун%20Ча%20%28龙脊红茶,%20Lóngjǐ%20Hóngchá%29.png)

![Лунцзи Хун Ча (龙脊红茶, Lóngjǐ Hóngchá)-1](images/Лунцзи%20Хун%20Ча%20%28龙脊红茶,%20Lóngjǐ%20Hóngchá%29-1.png)

龍脊紅茶是來自龍脊（龍脊，「龍之脊背」）山區的紅茶，產於廣西壯族自治區龍勝縣。原料來自樹齡100至500年的野生及半野生古樹茶樹葉片，這些茶樹生長在著名的龍脊梯田斜坡上——中國最具攝影魅力的景觀之一。此茶是「古樹紅茶」(古树红茶, gǔshù hóngchá) 的珍稀典型，高山雲霧森林的風土條件結合大葉種茶樹，造就了具有深厚絲滑口感和淡雅煙燻味的紅茶。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá) — 全發酵/氧化茶。
* **類別：** 中國地方紅茶；古樹茶 (古树红茶, gǔshù hóngchá)。
* **產地：** 中國，廣西壯族自治區 (广西壮族自治区, Guǎngxī Zhuàngzú Zìzhìqū)；桂林市 (桂林市, Guìlín Shì)；龍勝各族自治縣 (龙胜各族自治县, Lóngshèng Gèzú Zìzhìxiàn)。主要產區——龍脊鎮 (龙脊镇, Lóngjǐ Zhèn) 及鄰近的江底鄉 (江底乡, Jiāngdǐ Xiāng)、龍勝鎮 (龙胜镇, Lóngshèng Zhèn)。
* **地理座標：** ≈ 25.8° 北緯，110.1° 東經（以龍脊鎮為參考點）。

**2. 歷史與文化意義：**

* **歷史：** 龍脊山區的茶葉種植歷史超過千年。據百度百科及地方史料記載，當地壯族 (壮族, Zhuàngzú) 和瑤族 (瑶族, Yáozú) 民眾早在宋朝 (宋朝, Sòng Cháo) 末期就開始種植茶樹，山村居民將野生大葉種茶樹從森林移植到自家庭院。清朝乾隆 (乾隆, Qiánlóng, 1735–1796) 年間，龍脊茶成為貢茶 (贡茶, gòngchá)——龍脊鎮段寨村立有石碑為證。然而在歷史的大部分時期，龍脊茶都是作為綠茶生產。紅茶 (紅茶) 的專門製作始於較晚時期——2010年代初，隨著市場對古樹紅茶興趣的增長。2014年龍勝縣已有13家茶廠運營，生產面積達2000公頃。2015年龍脊茶獲中華人民共和國農業部註冊為農產品地理標誌保護產品 (农产品地理标志, nóngchǎnpǐn dìlǐ biāozhì)。

  至2024年，龍勝縣有超過40家茶企，其中約30家專門生產龍脊茶產品。年產乾茶超過300噸，其中約100噸為龍脊茶（紅茶和黑茶）。全縣茶園總面積超過13,000畝（≈ 870公頃），茶產品總價值達1億元人民幣。傳統傳承人謝福復 (谢福复, Xiè Fùfù) 在推廣龍脊茶品牌方面發揮了關鍵作用，他領導龍脊茶文化旅遊產業有限公司 (龙脊茶文化旅游产业有限公司)，積極將品牌推向龍勝「小產區」之外。

* **名稱：** 「龍脊」(Lóngjǐ) — 字面意思為「龍之脊背」— 山脈和著名梯田的名稱。當地人稱蜿蜒的梯田為躺在雲中的「龍的脊椎」。「紅茶」(hóngchá) — 紅茶。全名為「龍脊紅茶」。

* **文化意義：** 龍脊紅茶與龍勝多民族山區社群文化密不可分。龍脊茶是「龍脊四寶」(龙脊四宝, Lóngjǐ Sì Bǎo) 之一，與水、米、米酒並列。每年舉辦龍脊古茶祭祀大典 (龙脊古茶祭祀大典)，包括儀式供奉、傳統手工製茶展示和茶藝表演——此節慶是該地區非物質文化遺產的一部分。龍脊茶樹並非種植園作物，而是半野生的森林巨樹，採摘葉片需要爬樹——這根本區別於「台地茶」(táidì chá —「梯田/種植園茶」)。

**3. 植物學描述與原料：**

* **品種/栽培種：** 龍脊大葉種 (龙脊大叶种, Lóngjǐ Dàyè Zhǒng)，也稱龍勝龍脊茶 (龙胜龙脊茶)。屬於小喬木大葉種 (小乔木大叶种, xiǎo qiáomù dàyè zhǒng) *Camellia sinensis* 群體。收錄於《中國茶學辭典》(《中国茶学辞典》, Zhōngguó Cháxué Cídiǎn)，列為中國第28個優選茶樹品種。特徵：萌芽力強，芽頭大而厚實，葉片肥厚多肉。樹高可達5–9公尺；龍脊山區最古老的「茶王」(茶王, cháwáng) 樹齡超過500年，樹齡超過150年的茶樹有3000多棵，據某些估計整個景觀區域超過30,000棵。
* **採摘：** 每年僅採摘一次——4至5月（清明 (清明) 和穀雨 (谷雨) 時節）。年度單次採摘是當地生產者的基本原則，確保原料高品質並將對古樹生態系統的影響降至最低。
* **採摘標準：** 一芽二葉 (一芽二叶, yī yá èr yè) — 基本標準。精品批次採用一芽一葉 (一芽一叶, yī yá yī yè)。
* **原料要求：** 葉片僅從古樹茶樹採摘（非灌木種植園）；原料必須新鮮、完整、手工採摘。

**4. 風土條件與栽培特點：**

* **生長海拔：** 800–1000公尺及以上——主要茶園區域。部分野生茶樹分佈在海拔1200公尺處。
* **氣候：** 亞熱帶季風氣候。年平均溫度18.1°C。無霜期314天。年降雨量1500–2400毫米。年平均日照時數僅1223.3小時——對亞熱帶而言偏低，這是由於持續的雲霧覆蓋。相對濕度約82%。晝夜溫差明顯。該地區特點可用「晴日早晚遍野茶，陰雨連天滿山雲」來形容。
* **土壤：** 微酸性（pH 5.8–6.9），深厚疏鬆，有機質含量高。山地森林土壤，經多世紀落葉腐殖質豐富。
* **水資源：** 龍勝縣境內有480多條河流溪澗；主要水系為潯江 (浔江, Xúnjiāng)。地下水源絕對純淨——縣內無大型工業企業，無重金屬排放。
* **農業技術：** 古茶樹並非傳統意義上的栽培——它們在半野生森林環境中生長，不使用化學肥料和合成農藥。山地森林生態系統自然提供病蟲害防護，並通過多世紀落葉腐殖質提供養分。採摘完全手工進行，經常需要爬樹。部分農場正在建立苗圃以保護龍脊品種基因庫，並建立新的密植區域以擴大生產。

**5. 製作工藝：**

龍脊紅茶採用經典工夫紅茶工藝製作，並根據古樹大葉種原料特點進行調整。

* **採摘 (采摘, cǎizhāi)：** 手工採摘古樹茶樹；選取「芽頭 + 1–2葉」標準的嫩梢。
* **萎凋 (萎凋, wěidiāo)：** 鮮葉攤放在竹製托盤上陰乾自然萎凋。持續12–20小時，視天氣而定。龍脊品種大葉厚肉需要比小葉種更長的萎凋時間。目標是將水分降至60–64%，軟化細胞結構。
* **揉捻 (揉捻, róuniǎn)：** 萎凋葉進行揉捻——破壞細胞壁，釋放茶汁。對大葉種原料施用適度壓力，避免折斷葉片。
* **發酵/氧化 (发酵, fājiào)：** 揉捻葉分層放置在溫暖潮濕的房間內。氧化進行至草青味完全消失，出現持久的蜜果香。葉片呈現紅銅色。
* **乾燥/烘焙 (烘干, hōnggān)：** 用熱風或竹架炭火定型。部分生產者採用松木輕度煙燻（類似正山小種工藝），賦予茶葉淡雅煙燻味——這是部分龍脊紅茶批次的特色。
* **分級 (分级, fēnjí)：** 按等級分類，剔除缺陷葉片。

**6. 感官特徵：**

* **乾茶外形：** 大型緊結捲曲茶條，呈現明顯的「螺旋」結構 (紧结, jǐnjié)。色澤深褐帶紅潤。金毫可見但不如小葉種工夫茶豐富——這是大葉種特徵。
* **乾茶香氣：** 深沉溫暖——蜂蜜、堅果、乾果。部分批次帶有淡雅煙燻和松針調，與森林風土相關。
* **茶湯香氣：** 濃郁多層次。第一波——蜜甜帶乾果調（棗、葡萄乾）；第二波——深沉木質堅果調；尾韻——溫暖微煙燻的「森林」調。香氣在杯壁持久留存。
* **滋味：** 醇厚飽滿，質感絲滑——這是古樹大葉種茶的典型特徵。純淨甘甜與溫和澀感及礦物底韻結合。回甘持久溫暖，帶栗子蜜味。滋味明顯比同地區種植園紅茶更「豐滿」深厚。
* **湯色：** 琥珀至紅栗色，透亮明淨，金圈明顯。
* **葉底（沖泡後葉片）：** 大型舒展葉片；色澤銅褐均勻。葉片肥厚有彈性，完整——手工採摘和精心加工的標誌。

**7. 化學成分：**

* **多酚類：** 古樹大葉種原料通常比小葉種含有更高的多酚含量。在完全氧化過程中，兒茶素轉化為茶黃素和茶紅素，形成醇厚茶體和紅琥珀色茶湯。
* **胺基酸：** L-茶胺酸等胺基酸——由於遮陰山地森林生長環境和年度單次採摘（茶樹整年積累營養物質），含量較高。
* **生物鹼：** 咖啡因——大葉種原料含量通常高於小葉種（約3.5–5%）；可可鹼、茶鹼——微量。
* **維生素：** B群維生素，微量維生素C，芸香苷（維生素P）。
* **礦物質：** 鉀、鎂、錳、鋅、硒。龍脊山地森林土壤經多世紀有機腐殖質豐富，確保微量元素含量高。
* **精油和揮發性化合物：** 萜烯醇複合物（芳樟醇、香葉醇、橙花醇）、醛類、美拉德反應產物。古樹森林環境為香氣帶來獨特的「森林」和木質調，這在種植園紅茶中罕見。

**8. 保健功效：**

* **溫和提神：** 咖啡因含量較高結合L-茶胺酸，產生明顯但平衡的提神效果——精神振奮而不焦慮。
* **抗氧化活性：** 高多酚含量提供強大抗氧化保護。
* **消化支持：** 古樹大葉種紅茶傳統上被認為對消化特別有益，因其單寧含量高。
* **溫暖作用：** 在傳統中醫營養學中，古樹紅茶屬於「溫性」食品——溫暖身體，改善氣血循環。
* **認知功能：** L-茶胺酸有助放鬆和提高專注力。
* **礦物質補充：** 由於山地森林土壤的豐富礦物成分，茶葉含有大量鉀、鎂和錳——對神經系統、骨組織和能量代謝重要的元素。
* **皮膚狀態：** 多酚的抗氧化特性結合B群維生素支持皮膚健康，有助減少氧化壓力。
* **免疫支持：** 茶葉中的鋅和硒（得益於山地森林土壤的礦物豐富性）有助維持機體免疫功能。

**9. 沖泡方法：**

* **水溫：** 90–95°C。
* **茶量：** 5–6克配100–120毫升（工夫泡法）。古樹大葉種茶可能需要比小葉種稍多的茶量。
* **茶具：** 瓷質蓋碗 (盖碗) — 最大程度展現香氣。宜興紫砂壺——增加柔和度和深度。玻璃茶壺——可觀察大葉片舒展。
* **沖泡過程：**
  1. 用熱水溫壺。
  2. 投茶，聞熱茶香。
  3. 洗茶——快速沖泡（2–3秒），倒掉。對緊結大葉種茶，洗茶有益：「喚醒」茶葉。
  4. 第一泡：10–12秒。
  5. 第二至四泡：10–15秒。
  6. 第五泡起——每次增加5–15秒。
  7. 優質古樹龍脊紅茶可沖泡8–12次，逐漸展現更深層的堅果和木質調。

**10. 儲存：**

* 密封不透光容器；乾燥、陰暗、涼爽處；溫度15–25°C。
* 最佳飲用期限——12–24個月。由於萃取物含量高，優質古樹原料批次可儲存並「陳化」2–3年，獲得更柔和圓潤的風味。
* 不要冷藏，遠離強烈氣味的食品。

**11. 價格與仿冒：**

* **價格區間：** 正宗古樹原料龍脊紅茶屬於小眾且相對昂貴的產品：標準批次約200–500+元人民幣/500克，「茶王」樹精品批次可達數千元。價格由樹齡、生長海拔、採摘標準和批次規模決定（年度單次採摘產量有限）。
* **避免仿冒方法：**
  1. 向龍勝縣認證生產商購買。具有地理標誌「龍脊茶」(AGI2015-02-1699) 標識是關鍵指標。
  2. 評估茶葉：大型完整緊結茶條——古樹原料特徵；細小破碎多茶末——種植園茶特徵。
  3. 檢查香氣：古樹茶具有深沉「立體」香氣帶森林調；種植園同類產品較平面單調。
  4. 評估葉底：完整肥厚大葉片——正宗古樹茶；細小薄葉片——替代品。
  5. 謹慎對待以「龍脊古树紅茶」品牌銷售但價格可疑偏低的產品——真正產量有限。

**12. 有趣事實：**

* 龍脊紅茶字面上是「樹上採摘的茶」：採茶人需爬上樹幹和枝條數公尺高處採摘嫩芽。這使採摘過程勞動密集且具有一定極限性。
* 龍脊山區保存著樹齡超過500年的「茶王」(茶王)。據當地茶企資料，樹齡超過150年的茶樹至少3000棵，2025年估計整個景觀區域超過30,000棵。
* 清朝時期，當地農民通過石碑見證了龍脊茶作為貢茶的地位，居民潘天紅 (潘天红) 從桂林府背回保護農民自由茶葉貿易權的司法判決石碑——該碑至今仍立在村中（咸豐年間曾被損壞）。
* 龍脊茶是「四寶」(「四宝」)（「龍脊四寶」）之一——與米、水、米酒並列。
* 龍脊紅茶產區與世界著名的龍脊梯田 (龙脊梯田) 重疊，梯田被列入重要農業文化遺產名錄——茶與稻米在同一景觀中共存數世紀。
* 2020年傳統傳承人謝福復開始發展「茶膳」(茶膳, chá shàn) 方向——添加龍脊茶葉的菜餚：茶香竹筒雞、茶葉沙拉。這有助於向每年參觀梯田的遊客（2024年超過115萬人次）推廣茶文化。
* 龍脊茶是中國少數面臨「爬樹」問題的茶類之一——當地採茶人練習攀爬茶樹技能，使龍脊採茶人職業在體能上獨一無二。

**13. 與其他古樹紅茶比較：**

* **滇紅古樹 (滇红古树, Diānhóng Gǔshù)：** 雲南古樹紅茶（*C. sinensis* var. *assamica*）。通常更濃烈，麥芽味，金毫明顯。龍脊紅茶更精緻細膩，「森林」調更明顯，甜度強度較低。
* **宜紅工夫 (宜红工夫, Yíhóng Gōngfū)：** 湖北經典工夫紅茶，基於小葉種。與龍脊明顯不同：條索更細，蜜香更明亮，「深度」和礦物感較弱。宜紅「都市化」精緻；龍脊「山野」自然。
* **九層山紅茶 (九层山红茶)：** 臺灣古樹紅茶。概念相似（野生原料，限量採摘），但風土完全不同，花香調更明顯。

**結論：**

龍脊紅茶是中國最具特色的紅茶之一：每一片茶葉背後不是種植園農業技術，而是人與森林在「龍脊」山區的千年共存。深厚絲滑的滋味帶森林和堅果調，大型肥厚的葉片每年僅採摘一次——這一切使龍脊紅茶成為慢節奏冥想式品茶的理想選擇。它適合尋求雲南古樹紅茶替代品，並希望了解山區廣西鮮為人知但鮮明茶文化傳統的愛好者。
