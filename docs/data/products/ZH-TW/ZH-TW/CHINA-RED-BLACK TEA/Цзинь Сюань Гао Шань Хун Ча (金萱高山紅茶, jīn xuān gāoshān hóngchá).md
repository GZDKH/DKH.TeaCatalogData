# 金萱高山紅茶 (金萱高山紅茶, jīn xuān gāoshān hóngchá)

**金萱高山紅茶 (金萱高山紅茶, jīn xuān gāoshān hóngchá)**
![Цзинь Сюань Гао Шань Хун Ча (金萱高山紅茶, jīn xuān gāoshān hóngchá)](images/Цзинь%20Сюань%20Гао%20Шань%20Хун%20Ча%20(金萱高山紅茶,%20Jīn%20Xuān%20Gāoshān%20Hóngchá).png)

![Цзинь Сюань Гао Шань Хун Ча (金萱高山紅茶, jīn xuān gāoshān hóngchá)-1](images/Цзинь%20Сюань%20Гао%20Шань%20Хун%20Ча%20(金萱高山紅茶,%20Jīn%20Xuān%20Gāoshān%20Hóngchá)-1.png)

金萱高山紅茶是臺灣高山紅茶，採用著名品種金萱 (金萱, Jīn Xuān) 製作，該品種又稱為臺茶12號 (台茶12號, Táichá Shí'èr Hào)。此茶代表了現代臺灣茶藝的典型範例，將育種成就與獨特的山區風土結合，產生具有特色蜂蜜果香和細膩奶香韻味的紅茶。

**1. 分類與產地：**

* **類型：** 紅茶 (紅茶, hóngchá) — 完全發酵（氧化）。按歐洲分類屬於黑茶。氧化程度 — 90–100%。
* **類別：** 臺灣高山紅茶 (台灣高山紅茶, Táiwān Gāoshān Hóngchá)。屬於小葉種 (小葉種, xiǎoyè zhǒng) 紅茶群，有別於大葉種阿薩姆紅茶，如日月潭紅茶 (日月潭紅茶)。
* **產地：** 臺灣 (台灣, Táiwān)。主要產於島中部數個高山茶區，以嘉義縣 (嘉義縣, Jiāyì Xiàn) — 阿裡山 (阿里山, Ālǐshān) 地區，以及南投縣 (南投縣, Nántóu Xiàn) — 杉林溪 (杉林溪, Shānlínxī)、鹿谷 (鹿谷, Lùgǔ) 和梨山 (梨山, Líshān) 地區為主。主要茶園位於海拔1000至1600公尺，最優質原料採自海拔1200公尺以上。
* **地理座標：** 約北緯23°30'，東經120°45'（阿裡山地區，主要產區）。

**2. 歷史與文化意義：**

* **歷史：** 金萱高山紅茶是現代臺灣茶業的產物，其歷史與臺茶12號品種的創造密不可分。該品種由臺灣茶業改良場 (茶業改良場, Cháyè Gǎiliáng Chǎng, TRES) 歷時四十多年研發。該品種於1981年正式註冊，實驗編號2027，商業名稱為「金萱」。最初該品種主要用於生產烏龍茶 — 包種茶 (包種茶) 和臺灣式半球形烏龍茶。然而，自2000年代開始，臺灣茶農開始嘗試將高山金萱原料進行全發酵，力求創造具有獨特風味的紅茶。這種做法成為21世紀初臺灣紅茶產業發展的總體趨勢之一。

* **名稱：**
	* 「金萱」(金萱) — 字面意思為「金色萱草」。此名由TRES首任場長吳振鐸 (吳振鐸, Wú Zhènduó) 以其祖母之名命名。該品種暱稱為「27仔」(二七仔, Èrqī Zǎi)，取自實驗編號2027的末兩位數字。
	* 「高山」(高山) — 「高山」，指原料的高山產地（海拔1000公尺以上）。
	* 「紅茶」(紅茶) — 「紅茶」，定義加工類型 — 完全發酵。

* **文化意義：** 金萱高山紅茶體現了臺灣茶業的創新精神 — 致力於以非傳統工藝發掘熟悉品種的潛力。此茶在臺灣紅茶系列中佔據特殊地位，定位為較澀味大葉種紅茶的溫和、優雅替代品。在臺灣，它也因其甜美的蜂蜜果香和順滑質地而在茶飲產業 (茶飲, cháyǐn) 中廣受歡迎，非常適合冷泡。

**3. 植物學描述與原料：**

* **品種/栽培種：** 金萱 (金萱, Jīn Xuān)，又稱臺茶12號 (台茶12號, Táichá Shí'èr Hào)。屬於 *Camellia sinensis* var. *sinensis* — 小葉種亞種。該品種通過雜交獲得：父系 — 硬枝紅心 (硬枝紅心, Yìngzhī Hóngxīn)，母系 — 臺農8號 (台農8號, Táinóng Bā Hào)。該栽培種的主要植物學和農藝特徵：
	* **樹形：** 橫張型 (橫張型, héngzhāng xíng)，中等高度。
	* **葉片：** 橢圓形，中等大小，肉質厚實，亮綠色有明顯光澤。芽呈綠色帶紫色調，有明顯茸毛。
	* **產量：** 高產 — 比傳統臺灣品種青心大冇 (青心大冇) 和青心烏龍 (青心烏龍) 高20–50%。
	* **抗性：** 相對耐寒，抗枝枯病 (枝枯病, zhīkū bìng)，能很好適應不同海拔和土壤類型。
	* **香氣特徵：** 經適當加工後呈現天然奶香，帶玉蘭花香 (玉蘭花香, yùlánhuā xiāng)。奶香強度取決於生長海拔、採摘季節和加工技藝。在紅茶中，奶香較為細膩，讓位於蜂蜜果香。
	* **中生種** (中生種, zhōngshēng zhǒng)。

* **採摘：** 主要採摘季節 — 春季（3–4月）和冬季（10–11月）。春茶以精緻香氣見長，冬茶以甜度更高著稱。夏茶和秋茶也有生產，但品質被認為較低。
* **採摘標準：** 手工採摘。高級紅茶採用一芽二葉 (一芽二葉, yī yá èr yè)。頂級批次採用「芽加一葉」標準。
* **原料要求：** 僅使用未受損、健康的嫩芽，在乾燥天氣採摘。高山原料是決定茶葉品質和價格等級的關鍵要求。

**4. 風土與栽培特色：**

* **高山：** 金萱紅茶茶園位於海拔1000–1600公尺。阿裡山地區是主要產區，海拔從1000至1400公尺。更高海拔的梨山茶園（1600–2000公尺）和大禹嶺 (大禹嶺, 海拔2000公尺以上) 產出更精緻的原料，但那裡主要種植青心烏龍，金萱較少見。
* **土壤：** 紅色磚紅壤和黃棕色山地森林土，排水良好，富含有機物和礦物質。酸度pH 4.5–5.5，最適合茶樹。磷鉀含量高有助於葉片中芳香化合物的積累。
* **氣候：** 亞熱帶山地氣候，季節性明顯。阿裡山地區年均溫10–14°C，比平地涼爽得多。關鍵氣候因子：顯著的日夜溫差（日夜相差10–15°C）、頻繁的霧氣和雲層（全年霧日超過200天）、高濕度（80–90%）、豐沛降雨（年降雨量2500–3000毫米）。這些條件使茶芽生長緩慢，有利於胺基酸、果膠和精油的積累，賦予茶葉特有的甜味和濃郁的茶湯質地。

**5. 製作工藝：**

金萱高山紅茶的製作遵循經典紅茶工藝，並針對小葉種高山原料的特點進行調整，力求保持品種天然甜味和細膩香氣。

* **採摘 (採摘, cǎizhāi)：** 手工採摘，標準為「一芽二葉」。在晨間露水乾後進行。
* **萎凋 (萎凋, wěidiāo)：** 將採摘的葉片薄攤以失水。採用複合萎凋：初期 — 戶外陰涼或散射光下 (日光萎凋, rìguāng wěidiāo)，然後 — 室內控溫 (室內萎凋, shìnèi wěidiāo)。持續12–18小時或更長。目標是將含水量降至60–65%，使葉片柔軟並啟動初步氧化。
* **揉捻 (揉捻, róuniǎn)：** 萎凋後的葉片用揉捻機或手工揉捻。揉捻破壞細胞壁，釋放細胞液和酶類，激活多酚氧化。對金萱採用溫和揉捻，以保持芽頭完整，避免過度澀味。
* **發酵/氧化 (發酵, fājiào)：** 關鍵步驟。將揉捻後的葉片層疊放置在溫度24–28°C、濕度90–95%的專用室內。持續3–5小時。氧化過程中兒茶素轉化為茶黃素和茶紅素，形成特有的茶湯顏色、滋味和香氣。師傅通過葉片顏色變化（從綠黃色到銅紅色）和香氣（出現果蜜香）來控制過程。
* **乾燥 (烘乾, hōnggān)：** 分階段乾燥以停止氧化並固定品質。初乾在100–110°C下進行15–20分鐘停止酶促反應。復乾在較低溫度（80–90°C）下去除剩餘水分至4–6%。部分製茶師進行最終輕焙 (提香, tíxiāng) 以增強香氣。
* **分級 (分級, fēnjí)：** 成品茶按葉片大小和完整度分級，分離芽頭、完整葉片、碎葉和茶末。

**6. 感官特徵：**

* **乾茶外觀：** 深棕色，近乎黑色，略彎曲的條索狀 (條索狀, tiáosuǒ zhuàng) 茶條。有金黃色和紅褐色芽頭（芽頭），數量是高品質的標誌。葉片整齊，大小均勻。
* **乾茶香氣：** 濃郁多層次 — 以蜂蜜、乾果（李乾、杏乾）、麥芽香為主。有淡雅花香和金萱品種特有的細膩奶香。高山原料增添清新、涼爽的「高調」。
* **茶湯香氣：** 明亮、包覆性強，以蜂蜜果香複合體為主 — 成熟水果、焦糖、麥芽。奶香作為背景存在，在茶湯放涼時尤為明顯。可能出現淡雅花香。
* **滋味：** 醇厚、絲滑，天然甜味明顯，苦味極少。風味中有乾果（李乾、杏乾、葡萄乾）、蜂蜜、麥芽、焦糖調性。澀味輕柔宜人，迅速轉為持久甘甜回味 (回甘, huígān)。茶湯質地順滑、油潤，果膠質 (果膠質, guǒjiāo zhì) 含量高。有時感受到細膩果酸，增加層次感。
* **茶湯顏色：** 從琥珀橙色到濃郁紅琥珀色，明亮透澈，有特色的深度光澤。在良好光線下杯緣呈現金色「光環」。
* **葉底（沖泡後葉片）：** 完整、有彈性、均勻舒展的紅棕色葉片帶青銅色調。芽頭呈金橙色。葉片均勻完整是優質加工的標誌。

**7. 化學成分：**

金萱高山紅茶的化學特徵由小葉種高山原料的完全發酵決定，富含胺基酸和果膠。

* **多酚類：** 完全氧化過程中兒茶素（表沒食子兒茶素沒食子酸酯、表沒食子兒茶素等）轉化為茶黃素（1.5–2.5%）和茶紅素（8–15%），形成茶湯顏色、滋味主體和澀味特性。小葉種原料紅茶的總多酚含量通常低於大葉種阿薩姆品種，這解釋了更溫和的滋味。
* **胺基酸：** 含量比平地紅茶高。L-茶胺酸是主要胺基酸，提供甜味、鮮味和鎮靜效果。高山產地和涼爽氣候有利於胺基酸積累（約佔乾重3–4%）。
* **生物鹼：** 咖啡因約佔乾重2.5–3.5%（每200毫升茶湯約40–60毫克）。可可鹼和茶鹼含量較少。
* **果膠：** 果膠物質含量高是金萱品種的特色，造就茶湯順滑油潤的質地。
* **精油：** 超過300種揮發性芳香化合物，包括芳樟醇、香葉醇、水楊酸甲酯和順式茉莉酮。金萱特有的「奶香」與2-乙醯-1-吡咯啉和γ-十二內酯含量較高有關。
* **維生素：** B₁、B₂、B₆、C（因熱處理含量有限）、E、K。
* **礦物質：** 鉀、鎂、錳、氟、鋅、鐵。高山土壤使茶葉富含礦物元素。

**8. 保健功效：**

* **溫和提神與專注：** 咖啡因與L-茶胺酸的結合提供溫和、持久的提神效果，無劇烈起伏，提高注意力和認知活動。
* **溫暖作用：** 紅茶按傳統中醫理論具有「溫性」(性溫, xìng wēn)，改善末梢血液循環，適合寒冷季節。
* **抗氧化保護：** 茶黃素和茶紅素具有顯著抗氧化活性，保護細胞免受氧化壓力，有助延緩衰老過程。
* **消化支持：** 刺激消化酶分泌，促進脂肪和蛋白質消化。果膠包覆胃黏膜，發揮溫和保護作用。
* **心血管系統：** 規律飲用紅茶可能有助降低低密度脂蛋白膽固醇，改善血管彈性，調節血壓。
* **免疫增強：** 紅茶多酚具有抗菌抗病毒特性，支持機體天然防禦機制。
* **情緒健康：** L-茶胺酸有助提高多巴胺和血清素水平，發揮輕度抗壓抗焦慮作用。茶藝儀式增強放鬆效果。

**9. 沖泡方法：**

* **水溫：** 90–95°C。過熱的水（100°C）可能增強澀味，溫度不足則無法充分釋放香氣。
* **茶量：** 功夫泡法每150毫升水用4–5克茶；歐式泡法每200毫升水用3克茶。
* **茶具：** 瓷蓋碗 (蓋碗, gàiwǎn) 是首選，能充分展現香氣層次。也適合宜興紫砂壺 (紫砂壺, zǐshā hú) 或瓷茶壺。
* **沖泡過程（功夫泡法）：**

	1. 用開水溫熱蓋碗或茶壺。
	2. 投茶，讓茶葉在溫熱茶具中「醒茶」10–15秒，使香氣開始釋放。
	3. 注入90–95°C熱水，快速倒出第一泡（洗茶，5–10秒）。
	4. 第二泡 — 浸泡20–30秒。這是基本時間，可根據口味調整。
	5. 通過茶濾將茶湯倒入茶杯。
	6. 後續各泡 — 每泡增加5–10秒。可沖泡5–7次。

* **冷泡 (冷泡, lěng pào)：** 6克茶配600毫升冷水。放入冰箱6–8小時。冷泡突出天然甜味，減少澀味。

**10. 儲存：**

* **條件：** 乾燥、陰涼、避光處，溫度不超過25°C。遠離直射陽光、濕氣和強烈異味。
* **容器：** 密封不透光罐（鐵罐、瓷罐、有不透光塗層的玻璃罐）或密實鋁箔袋。避免塑膠容器。
* **保存期限：** 最佳期限為製作後12–24個月。紅茶不需冷藏，不同於綠茶和黃茶。在適當條件下可保存3年，但新茶（1年內）香氣最為鮮明。
* **茶葉天敵：** 濕氣、直射光、高溫、異味、氧氣。

**11. 價格與仿冒：**

金萱高山紅茶在臺灣紅茶中屬於「中高端」至「精品」價位。零售價格根據生長海拔、採摘季節、等級和製作者聲譽，每100克從30至80美元不等。阿裡山和杉林溪茶園的茶價格低於梨山或大禹嶺產品。

主要定價因素：生長海拔（越高越貴）、季節（春茶和冬茶較貴）、產地認證、手工製作程度。

**如何避免仿冒：**

* **向可信賣家購買：** 專業臺灣茶店、有透明製作者和產地資訊的經銷商。注意是否有產地證明 (產地證明, chǎndì zhèngmíng)。
* **評估外觀：** 整齊、均勻的茶條，有明顯金色芽頭。碎葉過多、茶末、大小不均是低品質或摻雜的標誌。
* **檢查香氣：** 乾茶應散發純淨、濃郁的蜂蜜果香，無異味、霉味或人工香料。過於濃烈的「奶香」可能是添加香料的標誌。
* **測試茶湯：** 透澈明亮的琥珀紅色茶湯，滋味純淨，回甘持久。混濁茶湯、苦味、平淡滋味是劣質產品的標誌。
* **謹慎對待可疑低價：** 真正的高山金萱紅茶不可能與平地茶同價。如果價格「好得令人難以置信」，應懷疑其來源。

**12. 趣味知識：**

* **以祖母命名：** 「金萱」(金萱) 這個名字是TRES首任場長吳振鐸為紀念其祖母而命名，為這個育種品種增添了植物學界罕見的個人和家庭溫情。
* **奶香迷思：** 金萱作為「奶香烏龍」的世界知名度造成了持久誤解。實際上這個品種的天然奶香非常細膩，只在特定種植和加工條件下才會顯現。市場上絕大多數「奶香烏龍」都是加香茶。在金萱紅茶中，奶香更加細膩，更像是淡雅背景而非主調。
* **萬能冠軍：** 金萱是少數能成功製作四類茶的品種之一：綠茶、烏龍茶（輕發酵和中發酵）、紅茶甚至GABA茶。這種廣泛適應性反映了其卓越的農藝品質。
* **茶飲業寵兒：** 近十年來，金萱成為臺灣茶飲業最受歡迎的基茶品種之一 — 從奶茶到水果冷飲。
* **經濟重要性：** 金萱在臺灣種植面積中佔重要地位，僅次於青心烏龍。主要種植區集中在南投縣和嘉義縣。

**13. 與其他紅茶比較：**

* **日月潭紅茶 (日月潭紅茶, Rìyuètán Hóngchá)：** 採用大葉種臺茶18號（紅玉，紅玉）製作的臺灣紅茶。明顯更澀更濃郁，有特色薄荷和肉桂調性，茶體厚重。金萱高山紅茶則更溫和甜美，有明顯果蜜香和順滑質地。差異源於小葉種（sinensis）和大葉種（assamica）亞種的區別。
* **青心烏龍阿裡山紅茶 (阿里山紅茶)：** 同產區但採用青心烏龍 (青心烏龍) 品種製作。具有更精緻的花香和輕盈茶體，但在甜度和滋味飽滿度上不及金萱。金萱有更明顯的蜂蜜特徵和油潤質地。
* **金駿眉 (金駿眉, Jīn Jùn Méi)：** 桐木 (桐木) 產的中國頂級芽茶紅茶。具有極精緻優雅的滋味，以蜜花香為主，巧克力餘韻。金萱高山紅茶更「溫暖」果香，茶體更厚重，焦糖麥芽調性更明顯。
* **滇紅 (滇紅, Diān Hóng)：** 雲南大葉種紅茶。明顯更澀更濃郁，有明顯香料、巧克力和堅果調性，茶體強勁。金萱高山紅茶更優雅溫和，甜美果蜜特徵。
* **東方美人 (東方美人, Dōngfāng Měirén)：** 臺灣重發酵烏龍茶（60–80%），非紅茶，但因蜜果香常被比較。東方美人有更「香水般」的麝香味（因小綠葉蟬影響），而金萱紅茶更直接，有純淨的焦糖果香。

**結論：**

金萱高山紅茶是新世代臺灣紅茶，將臺灣最成功栽培品種之一的遺傳潛力通過全發酵和高山風土得以展現。其絲滑的蜜果香伴隨細膩奶韻、明亮琥珀紅茶湯和持久甘甜回味，使這款茶成為認識臺灣紅茶世界的絕佳選擇。無論是功夫熱泡還是冷泡都表現出色，既適合有經驗的品茶者，也適合初入優質茶世界的新手。金萱高山紅茶是一款帶來和諧感受的茶：山間陽光的溫暖、雲霧的清新和臺灣茶師的用心 — 盡在每一杯中。
