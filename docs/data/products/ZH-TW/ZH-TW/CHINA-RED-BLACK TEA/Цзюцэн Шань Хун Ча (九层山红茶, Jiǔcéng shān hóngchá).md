# 九層山紅茶 (九层山红茶, Jiǔcéng shān hóngchá)

**九層山紅茶 (九层山红茶, Jiǔcéng shān hóngchá)**
![Цзюцэн Шань Хун Ча (九层山红茶, Jiǔcéng shān hóngchá)](images/Цзюцэн%20Шань%20Хун%20Ча%20(九层山红茶,%20Jiǔcéng%20Shān%20Hóngchá).png)

![Цзюцэн Шань Хун Ча (九层山红茶, Jiǔcéng shān hóngchá)-1](images/Цзюцэн%20Шань%20Хун%20Ча%20(九层山红茶,%20Jiǔcéng%20Shān%20Hóngchá)-1.png)

九層山紅茶——來自貴州省的高山紅茶，誕生於九層山山坡上「低緯度、高海拔、寡日照」的獨特條件下。以蜜糖香與蘭花香的交織聞名——這是將斯裡蘭卡技術革命性地適應中國山地原料的結果。

**1. 分類與產地：**

* **類型：** 全發酵（氧化）茶，屬於紅茶 (红茶, Hóngchá) 類別。發酵程度——完全，延長至48小時的氧化過程，這是此茶的顯著特徵。生產按照國家標準GB/T 13738（紅茶標準）進行。
* **類別：** 貴州高山紅茶。地理標誌產品 (国家地理标志产品)。
* **產地：** 中國，貴州省 (贵州, Guìzhōu)，六盤水市 (六盘水, Liùpánshuǐ)，六枝特區 (六枝特区, Liùzhī Tèqū)，牂牁鎮 (牂牁镇, Zāngkē Zhèn)，半坡村。茶園位於九層山 (九层山) 海拔1000–1500公尺。
* **地理座標：** 約北緯26°，東經105°（26度平行線「黃金產茶帶」區域）。
* **其他名稱：** 產品以S9系列（高級）、S6系列（中檔）和「高山紅茶」(高山红茶) 名義出品。

**2. 歷史與文化意義：**

* **歷史：** 九層山紅茶——年輕但迅速聲名鵲起的茶，其歷史僅有半個多世紀。

	九層山茶業的起源可追溯至1960年代，當時在這片土地上組織了知青農場 (知青农场, zhīqīng nóngchǎng)。正是這些「下鄉知青」(知青) 在山坡上建立了第一批茶園，使用從福建引進的茶苗。

	數十年來，茶園一直是規模有限的地方生產。轉捩點出現在2008年，成立了「六枝特區九層山土特產開發有限公司」(六枝特区九层山土特产开发有限公司)，開始工業化開發該地區的茶葉潛力。

	質的突破發生在2019年，引進了斯裡蘭卡紅茶生產技術專家。將錫蘭方法適應當地高山原料——延長發酵至48小時，在竹籃、木箱和石盤上進行三階段堆積——形成了獨特的香氣特徵：蜜糖香基調配以蘭花香韻。

	到2024年，品牌獲得廣泛認可：在中國好綠茶大會上獲得「金獎茶王」稱號 (中国好绿茶大会金奖茶王)，入選「中國好綠茶地圖」，品牌價值評估為256.7億元人民幣。茶園面積1200畝（約80公頃），年產乾茶約200噸。產品成功通過歐盟標準400多種農藥殘留檢測並出口海外。

* **名稱：**

	* 「九層」(九层) ——「九個層次/級別」。數字九 (九) 在中國數字學中象徵最高點、完整性、極致。九層山因其階梯狀、層次分明的山坡地形而得名。
	* 「山」(山) ——「山」。
	* 「紅茶」(红茶) ——「紅茶」——中國分類中全發酵茶類別（在西方傳統中稱為「黑茶」）。

* **文化意義：** 九層山紅茶——貴州省現代茶業崛起的象徵，該省長期以來一直處於「茶業巨頭」——雲南、福建、浙江的陰影下。六盤水地區，以前以「涼都」(凉都) 和煤炭工業中心聞名，正在經歷「綠色轉型」，高山茶在其中扮演旗艦產品的角色。此外，牂牁 (牂牁) 地區——歷史悠久的地方：漢代時期這裡存在夜郎國 (夜郎国)，茶業是這片土地文化復興戰略的一部分。

**3. 植物學描述與原料：**

* **品種/栽培種：** 使用兩個主要栽培種：

	* **當地群體種 (当地群体种, dāngdì qúntǐ zhǒng)：** 在九層山經過數十年自然選擇形成的本土茶樹群體。屬於 *Camellia sinensis var. sinensis*。特徵是葉片厚實、肉質，果膠含量高，使該品種非常適合深度發酵。茶園保存著樹齡達60年的茶樹——知青時代的遺產。
	* **福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbáichá)：** 來自福建的著名優質栽培種，在引種計劃中引入貴州。春季萌芽早，多酚含量高，金色芽頭豐富。最適合生產香氣濃郁的高級紅茶。

* **採摘：** 春茶（3-4月）被認為最珍貴，夏茶和秋茶原料也有使用。由於高山位置，萌芽比平原地區晚，九層山春茶氨基酸含量較高。
* **採摘標準：** S9系列——僅採單芽或「一芽一葉」(一芽一叶)；S6系列——「一芽二葉」(一芽二叶)；大眾類別——允許「一芽二至三葉」。
* **原料要求：** 生產不使用化學農藥和合成肥料。茶園使用物理防蟲方法——紫外線捕蟲燈 (灭蚊灯)。原料通過歐盟標準481項指標檢測。

**4. 風土與栽培特色：**

九層山的風土——中國西南茶區最令人印象深刻的風土之一，具有三個關鍵特徵：低緯度、高海拔、寡日照 (低纬度、高海拔、寡日照)。

* **生長海拔：** 海拔1000–1500公尺。這比中國大多數紅茶（除了臺灣和部分雲南茶）都要高得多。高山環境減緩芽葉生長，增加芳香物質和氨基酸的積累期。
* **氣候：** 亞熱帶山地氣候，具有明顯的海拔效應。年平均溫度約14°C。年降水量超過1200毫米。雲霧天數超過180天。散射光 (散射光) 比例超過總太陽輻射的70%。日夜溫差超過10°C。這些條件形成所謂的「高山冷韻」(高山冷韵, gāoshān lěng yùn)：生長緩慢、氨基酸組成豐富、香氣特徵精緻。
* **土壤：** 酸性黃壤 (微酸性黄壤, wēi suānxìng huángrǎng)，pH 5.0–6.5。有機物含量≥ 2%。土壤富含硒、鋅等微量元素。酸性環境——茶樹的理想條件。
* **生態：** 森林覆蓋率78.8%。水質符合國家一級標準。完全沒有工業污染。茶園被天然森林環繞，確保生物多樣性和生態平衡。
* **對成分的影響：** 由於高海拔減緩植物生長，春茶氨基酸含量達3.5–5.0%（比低山地區同類產品高20%）。多酚含量≥ 40%，顯著高於紅茶平均水準。

**5. 製作工藝：**

九層山紅茶工藝——斯裡蘭卡紅茶加工學派與中國山地傳統獨特融合的結果。關鍵創新——延長的三階段發酵，釋放高山原料的潛在能力。

* **採摘 (采摘, cǎi zhāi)：** 手工採摘一芽一至二葉。最好在晨露乾後的上午時間。
* **萎凋 (萎凋, wěidiāo)：** 新鮮葉片薄層攤放在竹篩 (竹筛) 上，置於通風室內。持續6–8小時。目的——降低含水量25–30%，提高葉片可塑性並啟動初級酶促過程。
* **揉捻 (揉捻, róuniǎn)：** 機械揉捻，旨在破壞細胞膜並釋放果膠汁 (果胶质, guǒjiāozhì)。由於當地原料果膠含量高，揉捻後的葉片呈現特有的油潤光澤。
* **發酵 (发酵, fājiào)：** 決定九層山紅茶特色的關鍵步驟。從斯裡蘭卡借鑒的長時間控制發酵概念適應當地條件：

	* **第一階段（竹籃，竹篮）：** 揉捻葉放入淺竹籃。溫度25–28°C。自然通風確保溫和的氧化開始。
	* **第二階段（木箱，木盒）：** 部分發酵的葉片轉移到木箱中，溫度保持穩定。增加層厚，減少氧氣接觸——發酵減緩但加深。
	* **第三階段（石盤，石盘）：** 在石盤上的最終階段，確保熱量均勻分佈。總發酵時間達48小時——比大多數中國紅茶（通常4–8小時）長得多。正是這種延長的氧化產生了獨特的香氣組合：蜜糖香 (蜜糖香, mìtáng xiāng) 和蘭花香 (兰花香, lánhuā xiāng)。
* **乾燥/烘焙 (烘干, hōnggān)：** 用燃燒的松木烘乾 (松明火烘焙, sōngmíng huǒ hōngbèi) 至含水量≤ 6.5%。使用松木為香氣帶來最細緻的樹脂調，呼應福建小種的遙遠傳統。

**6. 感官特徵：**

* **乾茶外觀：**
	* *S9系列：* 緊結重實的顆粒 (颗粒紧结重实)，手感沉重，大小均勻。色澤——光澤黑色，散佈金色芽頭 (金毫)。
	* *S6系列和「高山」：* 細秀的條索 (条索细秀)，黑色帶明顯金毫。
* **乾茶香氣：** 濃郁、甘甜，主要是蜂蜜、焦糖和新鮮蘭花香調。S9系列展現額外的新鮮焦糖太妃糖香調。
* **茶湯香氣：** 層次豐富，以蜜糖香 (蜜糖香) 為主導，纏繞優雅的蘭花香 (兰花香)。餘香中——溫暖的焦糖和烤栗子調。深度發酵批次出現焦糖香 (焦糖香)。「冷杯」(冷杯) 保持香氣超過15分鐘——紅茶中的稀有品質。
* **滋味：** 茶體——飽滿、厚重、濃郁（茶多酚≥ 40%）。明顯的甘鮮 (甘鲜, gān xiān)，由高氨基酸含量造成。明顯可感的回甘 (回甘, huí gān)——輕微的初始澀味後，口中充滿持久的蜂蜜餘韻。S9系列展現「七泡留香」現象。
* **湯色：** 紅艷明亮 (红艳明亮)，透明有光澤。最佳批次——側光下帶輕微橙色光暈。
* **葉底（沖泡後葉片）：** 紅亮勻整 (红亮匀整)，鮮豔。芽葉完整、肉厚、柔軟。S9系列——以完整金色芽頭為主。

**7. 化學成分：**

高山產地和延長發酵形成與平原紅茶不同的生化特徵：

* **多酚類：** ≥ 40%——紅茶中最高指標之一。在深度發酵過程中，兒茶素積極轉化為茶黃素（提供湯色亮度和「活性」）和茶紅素（負責茶體和色澤深度）。
* **氨基酸：** 3.5–5.0%，包括高L-茶氨酸含量。正是氨基酸賦予滋味特有的「鮮甜」(鲜甜) 並平衡多酚的澀味。
* **多糖：** 茶多糖提供持久回甘和湯質「厚度」。
* **生物鹼：** 咖啡因、可可鹼、茶鹼。咖啡因含量中等偏高，產生提神效果。
* **硒 (硒)：** 有機，天然來源（來自山地土壤）。具體指標因批次而異，但總體上茶葉富含微量元素。
* **果膠物質：** 由於當地栽培種特性和山地氣候，含量較高。果膠負責湯質的油潤質感。
* **維生素：** C、E、B群、類胡蘿蔔素。
* **礦物質：** 鋅、錳、鉀、鎂、鐵。

**8. 保健功效：**

* **促進脂質代謝：** 高多酚含量（≥ 40%）刺激脂肪酶活性，促進脂肪分解。
* **抗氧化和抗輻射保護：** 硒和多酚的協同作用提供增強的自由基中和能力。
* **心血管保護：** 茶黃素減少膽固醇在血管壁的沉積，有助維護心血管健康。
* **穩定血糖支持：** 多糖和多酚共同調節血糖反應。
* **提神作用：** 咖啡因與L-茶氨酸結合提供「溫和」提神效果——清醒而不緊張，提高注意力集中。
* **改善消化：** 單寧和酶刺激胃液分泌和腸道蠕動。
* **抗炎作用：** 多酚具有明顯的抗炎特性。
* **強化骨組織：** 茶中的錳和氟有助骨骼礦化。

**9. 沖泡方法：**

* **水溫：** 標準系列90–95°C。特級品（S9）——85°C，避免「燙傷」嫩芽並保持精緻的蘭花香。
* **茶量：** 3克配150毫升水（1:50比例）。
* **茶具：**

	* 白瓷蓋碗（白瓷突出湯色並能評估蓋香）。
	* 宜興紫砂壺 (紫砂壶)——更深層、「溫暖」的味道展現。
	* 玻璃茶壺——欣賞湯色之美。

* **過程：**

	1. **溫壺：** 用開水沖洗蓋碗或茶壺。
	2. **投茶：** 放入3克乾茶。吸聞加熱葉片的香氣——「聞香」(闻香, wén xiāng)。
	3. **第一泡：** 注入適溫水，停留5秒，倒入茶海。不洗茶——紅茶第一泡已展現滋味。
	4. **第2–4泡：** 每泡停留8–10秒。正是這些沖泡展現蜜糖-蘭花交響曲。
	5. **第5–8泡：** 每次增加5秒。滋味轉向焦糖和堅果調。
	6. **第8–10泡及以後：** S9系列能保持香氣至10泡；標準批次7–8泡。

**10. 儲存：**

與黑茶不同，九層山紅茶不適合多年陳放，在生產後最初幾個月表現最佳。

* **條件：** 密封、避光容器。儲存溫度0–5°C（冰箱）。家庭條件下可室溫儲存，遠離熱源和光線。
* **期限：** 建議開封包裝在6個月內使用，防止揮發性芳香化合物流失。
* **茶葉天敵：** 光線、濕氣、異味、氧氣。絕對不建議與香料、咖啡、香水一起儲存。

**11. 價格與假冒：**

價格因系列和採摘年份差異很大：

* S9系列（高級，單芽）：每斤（500克）600元以上；50克包裝——300–600元。
* S6系列（中檔，一芽二葉）：每斤200–400元。
* 「高山紅茶」：海拔1300公尺以上批次50克300–600元。
* 大眾類別：100克150–200元起。

**如何避免假冒：**

* **注意標識：** 尋找地理標誌 (地理标志) 和歐盟農藥殘留標準合格證書。
* **評估外形：** S9系列特徵是緊結重實的顆粒，金毫豐富。鬆散、不均勻的茶葉——替代品跡象。
* **檢查香氣：** 正宗九層山紅茶具有純淨的蜜糖-蘭花香，無酸味、霉味或人工香精。
* **測試「冷杯」：** 將茶湯倒入杯中放涼。15分鐘後香氣應仍可感知——正宗茶的特徵。
* **警惕低價：** 48小時發酵和手工採摘的生產週期使茶葉客觀上昂貴。

**12. 有趣事實：**

* **斯裡蘭卡傳承：** 九層山紅茶——少數直接借鑒錫蘭技術並適應當地原料的中國紅茶之一。延長發酵（48小時而非標準4–8小時）——中國非典型做法，產生驚人效果。
* **從煤炭到茶葉：** 六盤水市歷史上是貴州「煤炭之都」。發展高山茶業——中國城市經濟「綠色轉型」的鮮明例子，從礦物開採轉向生態農業。
* **古夜郎國：** 茶園所在的牂牁鎮，以漢代牂牁郡 (牂牁) 命名——與傳說中的夜郎國 (夜郎国) 相關的行政單位之一。這是中國西南最神秘的古代國家之一。
* **三階段發酵：** 使用三種不同材料（竹、木、石）作為發酵容器——中國紅茶中無先例的做法。每種材料都有微妙貢獻：竹——清新，木——柔和，石——礦物感。
* **生態記錄保持者：** 通過歐洲標準481項農藥殘留檢測——中國茶葉中最高指標之一。

**13. 與其他紅茶比較：**

* **滇紅金芽 (滇红金芽)：** 雲南金芽紅茶。滇紅通常更「厚重」，有蜂蜜和巧克力調。九層山以明顯的蘭花調和更持久的回甘著稱，這是高海拔減緩植物生長的結果。
* **祁門紅茶 (祁门红茶)：** 安徽「祁紅」以玫瑰果香聞名 (祁门香)。九層山——更甜更厚重，以蜜糖香基調代替玫瑰花香。
* **錫蘭烏瓦 (Ceylon Uva)：** 技術起源的直接「親戚」。烏瓦展現尖銳、提神的特性和薄荷調。九層山——更柔和、更甜，花香更明顯，這是錫蘭方法適應中國高山原料的結果。
* **正山小種 (正山小种)：** 福建經典，具煙燻特性。九層山沒有煙燻味（雖然松木乾燥可能帶來輕微樹脂調）而更偏果香蜜甜。
* **金駿眉 (金骏眉)：** 福建頂級「金眉」，採用單芽。金駿眉——更精緻花香；九層山S9——更強勁濃郁，茶體更厚重，「高山」礦物感更明顯。

**結論：**

九層山紅茶——遙遠傳統相遇如何產生新奇美妙事物的生動體現。隱藏在貴州雲霧高山中的九層山，接受了斯裡蘭卡技術並將其重鑄為純粹中國和純粹高山的產物——具有蜜糖靈魂和蘭花氣息的紅茶。對於品茶者來說，九層山——邀請認識新一代貴州茶：生態完美、技術創新、風味獨特。每一杯這種湯色紅艷如山間夕陽的茶——證明偉大的茶不僅能在千年歷史之地誕生，也能在雄心、風土和工藝在適當時機相遇的地方誕生。
