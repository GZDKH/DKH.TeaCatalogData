# 昭平紅茶 (昭平红茶, Zhāopíng hóngchá)

**昭平紅茶 (昭平红茶, Zhāopíng hóngchá)**
![Чжаопин Хун Ча (昭平红茶, Zhāopíng hóngchá)](images/Чжаопин%20Хун%20Ча%20(昭平红茶,%20Zhāopíng%20Hóngchá).png)

![Чжаопин Хун Ча (昭平红茶, Zhāopíng hóngchá)-1](images/Чжаопин%20Хун%20Ча%20(昭平红茶,%20Zhāopíng%20Hóngchá)-1.png)

昭平紅茶是來自廣西壯族自治區昭平縣的工夫紅茶，以綠茶品種福雲六號 (福云六号, Fúyún Liùhào) 為原料，採用紅茶工藝製作。該產品成為該地區的標誌性茶品，首次打破了昭平千年綠茶壟斷的局面，並於2013年獲得「昭平茶」(昭平茶) 地理標誌產品保護。

**1. 分類與產地：**

* **類型：** 中國紅茶 (红茶, hóngchá)，全發酵茶。
* **類別：** 工夫紅茶 (工夫红茶, gōngfū hóngchá) — 多階段生產工藝的精製紅茶。廣西地方茶。
* **產地：** 中國廣西壯族自治區 (广西壮族自治区, Guǎngxī Zhuàngzú Zìzhìqū)，賀州市 (贺州市, Hèzhōu Shì)，昭平縣 (昭平县, Zhāopíng Xiàn)。主要產區：昭平鎮、走馬鎮 (走马镇)、南嶺鎮 (南岭镇)，以及象棋山茶園。主要創建品牌的企業是象棋山茶葉有限公司 (象棋山茶叶有限公司)。
* **地理座標：** 約北緯24°10′，東經110°48′。

**2. 歷史與文化意義：**

* **歷史：** 昭平是中國南方最古老的產茶縣之一：明清時期的縣誌中已有當地茶葉的文字記載；20世紀初，縣內茶葉在《昭平縣志》(《昭平县志》) 中有詳細描述。2009年之前，該地區僅以綠茶聞名；1988年「綠茶2號」(绿茶2号) 品種榮獲「陸羽杯」(陆羽杯, Lùyǔ Bēi) 金獎，並被收錄於《中國茶經》(《中国茶经》)。紅茶「昭平紅」於2009年7月由象棋山公司首次成功製作，以福雲六號品種為原料。2010年該新產品榮獲上海國際茶葉展覽會金獎，專家學者陳宗懋 (陈宗懋, Chén Zōngmào) 給予高度評價。2013年「昭平茶」(包括紅茶和綠茶) 獲得中華人民共和國國家質量監督檢驗檢疫總局地理標誌產品保護 (地理标志产品保护) 地位。至2017年，全縣茶園面積達2.15萬畝 (約14,300公頃)，年產乾茶1.16萬噸。
* **名稱：** 「昭平」(昭平) — 地名，字面意思為「光明平坦」；縣名起源於宋代 (12世紀)：1124年 (宣和6年)，徽宗皇帝 (徽宗, Huīzōng) 將龍平縣 (龙平) 改名為昭平，將「招」(招, 「召喚、招募」) 字改為「昭」(昭, 「光明、明亮」)，認為前者不吉利。「紅茶」(红茶) — 紅茶。「工夫」(工夫) 在茶葉製作中意指「精細、費工的工作」，指需要複雜多階段工藝，每個環節都需要特殊技藝 — 從發酵控制到最終精製。
* **文化意義：** 2009年昭平紅的創制成為該縣茶業的轉捩點，打破了多世紀綠茶壟斷的局面。昭平擁有「中國名茶之鄉」(中国名茶之乡, 「中國著名茶葉之鄉」)、「中國有機茶之鄉」(中国有机茶之乡, 「中國有機茶之鄉」) 稱號，並列入「全國重點產茶縣」。該縣同時以「中國長壽之鄉」(中国长寿之乡, 「中國長壽之鄉」) 聞名，在行銷上與當地茶園的生態純淨相結合。

**3. 植物學描述與原料：**

* **品種/栽培種：** 昭平紅的主要栽培種為福雲六號 (福云六号, Fúyún Liùhào)，即「福雲6號」，屬於 *Camellia sinensis* var. *sinensis*。這是小葉和中葉型栽培種，原產福建省，很好地適應了廣西的亞熱帶氣候。也使用當地群體種 (群体种, qúntǐzhǒng) 和其他適應當地的小葉和中葉型栽培種 — 烏牛早 (乌牛早, Wūniú Zǎo)、黃金茶 (黄金茶, Huángjīn Chá)、元宵綠 (元宵绿, Yuánxiāo Lǜ)。
* **採摘：** 昭平以早春開採聞名 — 採摘從2月底開始，因此當地茶葉被稱為「中國大陸早春第一茶」(中国大陆早春第一茶)。春茶 (3-4月) 提供最高品質的原料；夏茶也有採摘，但風味特徵不如春茶。
* **採摘標準：** 一芽一葉或一芽二葉 (一芽一叶, 一芽二叶)。高等級茶品偏好芽頭較多的比例。
* **原料要求：** 完整、新鮮的葉片，無機械損傷和粗梗；採摘與萎凋開始之間的延遲時間最短，以保持酶活性。

**4. 風土與栽培特色：**

* **地形與生態：** 昭平縣位於山區：山地佔總面積87%以上，森林覆蓋率超過87.6%。該地區位於南嶺 (南岭, 「南嶺山脈」) 範圍內，特徵是河谷深切，濕度高，霧氣頻繁。
* **海拔高度：** 主要茶園位於海拔400-1000公尺；象棋山有機茶園約700公尺。
* **氣候：** 亞熱帶季風氣候，溫和濕潤。年平均氣溫19.8-19.9°C；無霜期超過310天；年平均降雨量約2046毫米 — 廣西最多雨的地區之一。年平均日照約1440小時，特徵是經常多雲和高濕度。
* **土壤：** 以黃壤和紅黃壤 (黄壤, huáng rǎng) 為主，酸性，pH 4.5-6.5，由於豐富的森林落葉，有機質含量高。
* **水資源：** 桂江 (桂江, Guìjiāng) 穿越境內，內部水道總長1624公裡。水質清潔，無工業污染 — 該縣因工業存在極少而入選「深呼吸小城百佳」(深呼吸小城)。
* **生態認證：** 「綠色食品」(绿色食品) 認證茶園面積7900畝；16家企業通過有機生產認證或轉換。大型工業缺乏和高森林覆蓋率確保了空氣清潔和無酸雨。

**5. 生產工藝：**

昭平紅按照經典工夫紅茶工藝生產，分為初製 (初制, chūzhì) 和精製 (精制, jīngzhì)。關鍵技術特點是成功克服了用綠茶品種製作紅茶時常見的發酵不均、湯色淡薄、香氣貧乏和明顯苦澀等問題。

* **採摘 (采摘, cǎizhāi)：** 清晨手工採摘「一芽一至二葉」標準。
* **萎凋 (萎凋, wěidiāo)：** 將葉片薄攤在通風室內；水分含量降至約60%，葉片變軟有彈性。持續4-6小時，視空氣濕度而定。
* **揉捻 (揉捻, róuniǎn)：** 機械揉捻破壞細胞壁，使細胞汁液流到表面，確保均勻氧化。葉片呈現特有的緊密螺旋形。
* **氧化/發酵 (发酵, fājiào)：** 在控制溫濕度下進行。兒茶素的酶促氧化形成茶黃素和茶紅素，產生紅色和甜味特徵。福雲六號需要特別仔細的控制，以避免發酵不均。
* **初烘 (初烘, chūhōng)：** 用100-110°C熱風停止氧化。
* **精製 (精制, jīngzhì)：** 包括篩分 (筛分, shāifēn)、風選 (风选, fēngxuǎn)、手工挑揀 (挑拣, tiāojiǎn)、復火 (复火, fùhuǒ) 和拼裝 (拼装, pīnzhuāng)。正是這個階段的多步驟性質賦予了「工夫」— 「精細工作」的名稱。

**6. 感官特徵：**

* **乾茶外形：** 細緊螺旋捲曲 (卷曲成螺, juǎnqū chéng luó)，顆粒均勻緊實。色澤烏潤 (乌润, wūrùn) 有油光，金毫顯露 (金毫显露, jīnháo xiǎnlù)。
* **乾茶香氣：** 清純、高揚持久；以花果香調為主，伴有蜂蜜和溫暖焦糖香。高等級茶品有明顯的花果香 (花果香, huāguǒ xiāng)。
* **茶湯香氣：** 濃郁多層次，甜蜜蜂蜜香帶明顯花香頂調；隨著冷卻出現果乾和淡焦糖香調。
* **滋味：** 醇厚甘爽 (醇厚甘爽, chúnhòu gānshuǎng)，前幾泡甜味明顯，中段柔滑如絲，回甘 (回甘, huígān) 悠長。澀味極輕，餘韻清淨持久。
* **湯色：** 紅豔明亮 (红艳明亮, hóngyàn míngliàng)，杯緣有特徵性金圈 (金圈, jīnquān) — 茶黃素含量高的標誌。
* **葉底 (茶渣)：** 紅勻明亮 (红匀明亮, hóngyún míngliàng)，葉片肥嫩，完整芽頭豐富。

**7. 化學成分：**

* **多酚類：** 發酵過程中90%以上的原始兒茶素轉化為茶黃素 (茶葉「金色」色素，負責湯色亮度和金圈)、茶紅素 (形成紅色深度和滋味「絲滑感」) 和茶褐素。
* **胺基酸：** L-茶胺酸含量顯著，提供滋味柔和甜美，並與咖啡因產生協同「平靜提神」效果。
* **生物鹼：** 咖啡因 (約佔乾重2-4%) 提供提神作用；可可鹼和茶鹼含量微量。
* **維生素：** 維生素C含量顯著 (紅茶比一般認為的更好地保存維生素C，得益於多酚類的保護作用)。
* **礦物質：** 鈣、銅、鈉、磷、鋅和其他微量元素。
* **精油和揮發性芳香化合物：** 芳樟醇、香葉醇、苯甲醛和其他萜烯類和醛類形成特徵性花果香。熱量：每100克乾茶少於1244千焦。

**8. 健康功效：**

* **提神效果：** 咖啡因與茶胺酸結合提供溫和持久的專注力提升，無劇烈「峰值」興奮。
* **抗氧化保護：** 茶黃素和茶紅素具有顯著的自由基清除能力，支持細胞健康。
* **消化支持：** 多酚氧化產物溫和刺激消化酶分泌，促進舒適消化，特別是餐後。
* **心血管系統：** 適量規律飲用紅茶與維持血管彈性和膽固醇水平正常化相關。
* **利尿效果：** 咖啡因與芳香物質結合溫和刺激利尿。
* **溫暖作用：** 紅茶在中醫飲食學中屬於「溫性」飲品；對胃部舒適，推薦寒冷季節飲用。
* **心理情緒放鬆：** 溫暖甜美的香氣和柔和滋味產生明顯的「感官」鎮靜效果，有助緩解緊張。

**9. 沖泡方法：**

* **水溫：** 標準茶品90-95°C；精緻高芽頭等級85-90°C。
* **用茶量：** 工夫泡法4-6克/100-120毫升；歐式泡法2-3克/200-250毫升。
* **茶具：** 瓷蓋碗 (盖碗, gàiwǎn) 100-120毫升容量最佳，能完全展現香氣；瓷壺或玻璃壺可視覺欣賞湯色。緊實茶品可用紫砂壺。
* **沖泡過程：**
  1. 用沸水溫壺，倒掉水。
  2. 投茶，蓋上蓋子幾秒鐘，聞加熱乾茶香氣。
  3. 洗茶 (可選)：注水1-2秒後倒掉 — 這是「喚醒」茶葉。
  4. 第一泡：5-8秒。
  5. 後續各泡：每泡增加3-5秒。
  6. 沖泡次數：優質茶品6-8泡，緊實高芽頭等級可達10泡。

**10. 儲存：**

密封、避光容器 (鐵罐、真空鋁箔袋) 儲存，遠離濕氣、異味和直射陽光。最適溫度10-25°C。最佳品質期：生產後12-24個月。一些高芽頭含量的緊實茶品在適當儲存2-3年後會有良好發展 — 蜂蜜和果香加深，澀味進一步柔化。

**11. 價格與仿冒：**

價格類別：中等。昭平紅比著名紅茶如金駿眉或祁門紅茶便宜得多；基本茶品每100克幾十元起，高等有機等級200-400元。影響價格因素：採摘季節 (春茶較貴)、芽頭比例、有機認證和獲獎地位。

* **如何避免仿冒：**
  1. 檢查產地：真正的昭平紅僅產於昭平縣；尋找包裝上的地理標誌 (地理标志) 標識。
  2. 評估香氣：純淨花蜜香無化學刺激、酸敗或霉味。
  3. 檢查湯色：應透明、紅豔帶金圈；混濁表示發酵或儲存問題。
  4. 品嚐滋味：天然甜味和清淨餘韻；酸味或陳腐味需警惕。
  5. 謹防低價：如果「獲獎」昭平紅售價明顯低於市價 — 很可能是其他地區原料替代。需注意部分福建茶企從昭平採購原料製作金駿眉和正山小種，可能造成混淆。

**12. 趣聞軼事：**

* 昭平紅成為該縣首款紅茶 — 2009年前該地區僅生產綠茶。技術突破在於將紅茶發酵工藝適應綠茶品種福雲六號，需要數年實驗。
* 昭平縣擁有「中國長壽之鄉」(中国长寿之乡) 地位：百歲居民比例超過聯合國「長壽區」標準兩倍以上。當地茶文化與生態純淨形象密不可分。
* 昭平有機茶園認證面積超過3500畝 (約2330公頃)，按有機標準管理的茶園面積達3600畝；該縣是廣西最大的有機茶中心之一。
* 昭平被稱為「中國大陸早春第一茶之鄉」：得益於亞熱帶氣候，採摘從2月底開始 — 比浙江早一個月。
* 除綠茶和紅茶外，該縣也是六堡茶 (六堡茶) 的重要產地和茉莉花茶原料供應地，使其成為廣西最多元化的茶區之一。
* 民間俗語「昭平不平」(昭平不平, 「昭平一點也不平」) 巧妙運用縣名中「平」(平, 「平坦」) 字的含義，反映真實地形：山地佔近90%面積，幾乎沒有平原。
* 昭平是廣西最多雨的地點之一：年平均降雨量超過2000毫米，該縣位於該地區雷暴和暴雨活動中心，為形成豐富霧氣和雲層創造理想條件，這對嫩茶葉發育至關重要。

**13. 與其他紅茶比較：**

* **英紅 (英红, Yīng Hóng)** — 廣東英德縣紅茶。用大葉種栽培種 (英紅九號) 製作，茶體更厚重，麥芽香明顯，「巧克力」餘韻。昭平紅相比更輕盈優雅，以花果香調為主。
* **祁門紅茶 (祁门红茶, Qímén Hóngchá)** — 安徽著名紅茶，「中國十大名茶」之一。祁門以獨特的「祁門香」— 香料蘭花香聞名。昭平紅香氣較簡單，但以明顯甜味、柔和和多次沖泡耐力補償。
* **滇紅 (滇红, Diān Hóng)** — 雲南大葉種阿薩姆栽培種紅茶。滇紅茶體強勁，蜂蜜胡椒香，金毫含量高。昭平紅明顯更精緻輕盈，茶體接近中等。
* **桂紅 (桂红, Guìhóng)** — 廣西紅茶總稱。昭平紅實際上是這個群體的領先代表，為該地區紅茶設定了新的品質標準，獲得學者陳宗懋高度評價。
* **金駿眉 (金骏眉, Jīn Jùn Méi)** — 福建武夷山高端紅茶，僅用芽頭製作。有趣的是，一些福建公司從昭平採購原料製作金駿眉，間接證實了當地茶葉的高品質。然而，真正的昭平紅具有自己的特色 — 更明顯的螺旋捲曲和獨特的地域香氣特徵。

**結論：**

昭平紅茶是創新方法如何在具有多世紀茶葉傳統的地區創造全新產品的雄辯例證。用綠茶品種按紅茶工藝製作，這款茶以純淨明亮的花蜜香、精緻甜味和美麗的紅寶石湯色配金圈令人驚豔。昭平紅非常適合尋求柔和、芳香紅茶而無過度澀味的人 — 來自中國南方最生態純淨角落之一的日常而又獨特的飲品。
