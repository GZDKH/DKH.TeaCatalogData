# 博羅紅茶 (博罗红茶, Bóluó hóngchá)

**博羅紅茶 (博罗红茶, Bóluó hóngchá)**
![Боло Хун Ча (博罗红茶, Bóluó hóngchá)](images/Боло%20Хун%20Ча%20%28博罗红茶,%20Bóluó%20Hóngchá%29.png)

![Боло Хун Ча (博罗红茶, Bóluó hóngchá)-1](images/Боло%20Хун%20Ча%20%28博罗红茶,%20Bóluó%20Hóngchá%29-1.png)

博羅紅茶是產於廣東省博羅縣的紅茶，位於著名的羅浮山與象頭山山脈交匯處。這是著名「柏塘山茶」(柏塘山茶) 的紅茶版本——中國南方少數幾種小葉種山茶之一，其歷史可追溯超過1,700年。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá) — 全發酵（氧化）茶。
* **類別：** 廣東紅茶；基於小葉種山茶群體種 (小叶种山茶, xiǎoyè zhǒng shānchá) 的地方紅茶。該產品與「柏塘山茶」(柏塘山茶) 地理標誌體系相關，於2015年12月4日獲得地理標誌產品保護 (地理标志产品, dìlǐ biāozhì chǎnpǐn) 地位。
* **產地：** 中國廣東省 (广东省, Guǎngdōng Shěng)，惠州市 (惠州市, Huìzhōu Shì)，博羅縣 (博罗县, Bóluó Xiàn)。主要生產區域——柏塘鎮 (柏塘镇, Bǎitáng Zhèn)，惠州最大的茶葉鄉鎮。次要區域——羅浮山 (罗浮山, Luófú Shān) 南坡和象頭山 (象头山, Xiàngtóu Shān) 西坡的鄉鎮。
* **地理座標：** ≈ 北緯23.4°，東經114.1°（柏塘鎮）。

**2. 歷史與文化意義：**

* **歷史：** 博羅是廣東四個最古老的縣之一，於西元前214年秦始皇時期建立。該地區的茶葉傳統可追溯至晉朝 (晋, 265–420)：道教隱士單道開 (单道开, Shàn Dàokāi) 在羅浮山修行時，「時飲茶蘇一二升」——這是嶺南地區飲茶的最早文獻記錄之一。

  唐朝 (唐, 618–907) 時期，李翱 (李翱, Lǐ Áo) 在《解惑》(《解惑》) 中描述了隱士王野人 (王野人)，他在羅浮山坡建立了「草庵茶園」——證明九世紀時已有大規模茶園。宋朝 (宋, 960–1279) 時期，羅浮山茶被列為名茶，蘇東坡 (苏东坡, Sū Dōngpō) 在惠州貶謫期間讚美當地茶園。

  《廣東通志》(《广东通志》，「廣東全誌」) 記載：「茶：羅浮產者佳」(茶，罗浮产者佳)。羅浮山茶 (罗浮山茶) 被列為「嶺南四大名茶」(岭南四大名茶) 之一。

  隨著地區發展，茶葉種植集中在柏塘鎮——羅浮山與象頭山山脈間的天然盆地。農民將野生小葉種茶樹從山坡移植到自己的土地上，逐漸形成獨特的當地群體種——柏塘小葉種 (柏塘小叶种, Bǎitáng Xiǎoyè Zhǒng)。2010年，廣東省農業科學院在柏塘發現了稀有的小葉種紫芽茶 (小叶种紫芽茶, xiǎoyè zhǒng zǐ yá chá)，確認其作為遺傳資源的特殊價值——陸羽 (陆羽) 在《茶經》(《茶经》) 中提到：「茶：紫者上，綠者次」(茶，紫者上，绿者次)。

  歷史上柏塘山茶主要是綠茶。使用相同小葉種原料生產紅茶是相對較新的發展方向，自2010年代積極發展。2015年「柏塘山茶」獲得國家地理標誌產品保護地位，2019年獲得「全國名特優新農產品」(全国名特优新农产品) 認證。到2023年，柏塘茶園面積超過30,000畝（≈ 2,000公頃），年產值達60億元；紅茶在產品組合中佔據重要地位，包括出口至東南亞、香港和澳門。

* **名稱：** 「博」(博) 和「羅」(罗) 是古代地名元素，可追溯至秦朝。據傳說，蓬萊仙山 (蓬莱, Pénglái) 漂洋過海與羅浮山合併——這就是縣名的傳說詞源。「紅茶」(红茶) 即「紅茶」。因此，博羅紅茶即「博羅[縣]的紅茶」。

* **文化意義：** 博羅紅茶是羅浮道茶文化 (罗浮道茶, Luófú Dào Chá) 不可分割的一部分。羅浮山是「十大洞天」(十大洞天) 之一，這裡的茶葉傳統與道教「養生」(养生, yǎngshēng) 修行密不可分。柏塘每年舉辦茶文化節 (柏塘山茶文化节)，包括茶藝表演、鬥茶 (斗茶, dòu chá) 比賽和大師班。2023–2024年，柏塘鎮每年接待超過60,000名遊客，顯示茶葉旅遊的重要性日益增長。柏塘擁有「廣東十大茶鄉」(广东十大茶乡) 和「全國一村一品示範村鎮」(全国一村一品示范村镇) 稱號。

**3. 植物學描述與原料：**

* **品種/栽培種：** 柏塘小葉種 (柏塘小叶种) — *Camellia sinensis* var. *sinensis* 小葉種山地群體種，經過數百年野生茶樹馴化形成。灌木較矮，葉片小而窄，橢圓披針形，長3–10公分，幼芽有明顯絨毛。葉脈清晰。花期8月至12月。部分農場也栽培小葉種紫芽茶 (紫芽茶, zǐ yá chá)，含有較高的花青素。柏塘保存了130多棵老茶樹，其中約30棵樹齡約100年，一棵樹齡約200年。

* **採摘：** 主要採摘季節：春分後的春茶 (春分茶, Chūnfēn Chá) — 最佳批次；清明後茶 (清明茶)；夏茶和秋茶。值得注意的還有冬季「雪片」(雪片, Xuě Piàn) 批次——在小雪至大雪期間採摘的茶，此時芽葉稀少且特別珍貴。主要季節之間採摘「禾花茶」(禾花茶) — 「稻花茶」，與晚稻開花期重疊。

* **採摘標準：** 兩葉一芯 (两叶一芯, liǎng yè yī xīn) — 柏塘茶的標準。高級紅茶等級採用一芽一葉。採摘完全手工：上午採摘的中午加工，下午採摘的晚上加工，確保原料新鮮。

* **原料要求：** 完整、無損的芽葉，芽上有特徵性細白絨毛。葉片必須新鮮、有彈性，無機械損傷和蟲害痕跡。柏塘的生態標準要求最少使用農化產品——許多農場只使用有機肥料。

**4. 風土與栽培特色：**

* **海拔高度：** 海拔200–500公尺。高海拔種植園（如三帽髻山的福波茶園）位於500公尺以上。

* **氣候：** 亞熱帶季風氣候。年平均氣溫22.7°C，年平均降雨量約1,900毫米，無霜期342天。柏塘盆地三面環山，形成多霧、冬季溫和、晝夜溫差明顯的微氣候——有利於芳香物質積累的條件。

* **土壤：** 山地酸性土壤，pH值5.0–5.5，腐殖質層深厚，有機質含量2–3%。基岩為花崗岩和紅土，提供良好的排水和礦化。位於羅浮山和象頭山兩個國家級自然保護區之間（珠江三角洲唯一未受破壞的國家級自然保護區）——確保空氣和水質純淨。

* **農業技術：** 柏塘以小規模家庭茶業為主：鎮上6,000多個農戶幾乎每家都有自己的茶園，面積從1–2畝到10+畝不等。鎮上有60多個茶葉合作社和企業，包括一家省級「農業龍頭企業」(省级农业龙头企业)。按照「無化肥、無農藥」原則種植——柏塘茶定位為「天然、綠色、健康」(天然、绿色、健康)。修剪、除草、有機施肥是主要農業技術措施。

**5. 製作工藝：**

博羅紅茶使用與經典綠茶柏塘山茶相同的小葉種柏塘原料，但採用全發酵工藝。小葉種 *var. sinensis* 原料製成的紅茶具有不同於大葉種的特性：更精緻，具有明顯的花香和較溫和的澀味。

- **採摘 (采摘, cǎizhāi)：** 上午手工採摘嫩芽。
- **萎凋 (萎凋, wěidiāo)：** 自然或複合萎凋（在通風室內）10–16小時。目標是將葉片水分降至60–65%並啟動初期發酵過程。
- **揉捻 (揉捻, róuniǎn)：** 機械揉捻破壞細胞壁。小葉比大葉種更容易快速揉捻，需要更謹慎地控制壓力。
- **發酵/氧化 (发酵, fājiào)：** 在24–28°C溫度和高濕度下進行3–4小時。小葉種原料具有較高的胺基酸與多酚比例，形成特徵性甜美花香風味。
- **乾燥 (烘干, hōnggān / 干燥, gānzào)：** 在100–110°C下固定香氣特徵。部分生產者在80–85°C下進行輕度烘焙以發展焦糖香。
- **分級 (分级, fēnjí)：** 按粒度和品質分級，區分芽茶、葉茶和拼配等級。

個別農場（如光華食品 (光华食品)）也生產基於柏塘原料的調味紅茶：檸檬 (柠檬山茶)、陳皮 (陈皮茶)、荔枝 (荔枝茶)。

**6. 感官特徵：**

* **乾茶外觀：** 緊實、緊結、細小的捲曲；葉片薄而均勻，具有小葉種特有的精緻感。顏色深栗色至黑色，高等級有金色芽尖。

* **乾茶香氣：** 甜美、天然，帶有乾燥野花香、淡蜂蜜味和細膩香料味。香氣不如大葉種廣東紅茶那樣「厚重」——更像是「細長」(细长, xì cháng)，正如柏塘茶的描述。

* **茶湯香氣：** 溫暖、圓潤，明顯的蜂蜜、甜麵包和淡焦糖香。中段沖泡顯現龍眼和乾荔枝果香——廣東風土的印記。紫芽原料批次有額外的漿果香。

* **滋味：** 對小葉種茶而言濃厚 (浓厚, nónghòu)，明顯甜味，溫和澀味，悠長「回甘」(回甘, huígān) 餘韻。「甘、滑、香」(甘、滑、香) 和諧平衡——柏塘茶的經典公式，也適用於紅茶版本。

* **湯色：** 紅琥珀色，明亮透明。比大葉種紅茶稍淺且「柔和」——更接近蜂蜜紅色帶金色光澤。

* **葉底（沖泡後茶葉）：** 細小、均勻、著色均勻的葉片；顏色銅紅至栗色。葉片有彈性，保持小葉種特有結構。

**7. 化學成分：**

* **多酚類：** 小葉種 *var. sinensis* 原料含有適中的多酚（鮮葉中18–25%），發酵後產生茶黃素和茶紅素的和諧比例，無過度澀味。
* **胺基酸：** 相對較高的胺基酸含量（乾重的3–4%），包括L-茶胺酸，解釋了明顯的天然甜味和口感柔和。
* **生物鹼：** 咖啡因——乾重的2.5–3.5%（小葉種典型含量）；可可鹼、茶鹼微量。
* **花青素：** 紫芽批次 (紫芽) 含有較高的花青素，具有抗氧化活性。
* **維生素：** B₁、B₂、P（芸香苷）、微量維生素C。
* **礦物質：** 鉀、鎂、錳、鋅、鐵——反映羅浮山地區富含有機質的山地土壤。
* **精油：** 萜烯醇複合物（芳樟醇、香葉醇），小葉種山茶特有；美拉德反應產物——麥芽酚、糠醛。

**8. 保健功效：**

* 由於咖啡因含量適中且與L-茶胺酸結合，溫和提神並維持專注力——「平靜清醒」的效果。
* 由於茶黃素、茶紅素和（紫芽批次中的）花青素而具有抗氧化活性。
* 促進消化——傳統上廣東茶在用餐期間和餐後飲用，有助於消化油膩食物（在粵菜背景下特別相關）。
* 規律適量飲用可通過類黃酮和多酚維持血管張力。
* 具有溫和的溫暖作用，緩解主觀疲勞感。
* 「羅浮道茶」(罗浮道茶) 傳統上與「養生」(养生，「養生」) 修行相關——維持身體整體平衡。
* 紫芽原料中的花青素具有抗炎特性，有益於視力健康。

**9. 沖泡方法：**

* **水溫：** 90–95°C。
* **茶量：** 4–5克配100–120毫升（小葉種原料比大葉種萃取更快，因此用量稍少）。
* **茶具：** 蓋碗 (盖碗) — 通用選擇；瓷茶壺；潮州砂壺 (潮州砂壶) — 適合廣東背景。玻璃茶壺突出美麗的湯色。
* **過程：**
  1. 用熱水溫壺。
  2. 投茶；欣賞加熱乾茶的香氣。
  3. 通常不需要洗茶——小葉種茶出味快。
  4. 第一泡：5–8秒（小葉展開較快）。
  5. 第二至四泡：8–12秒。
  6. 第五泡起每次增加5–10秒。
  7. 通常6–8泡；高等級可達10泡。

**10. 儲存：**

* 密封容器，避光、防潮、防異味。
* 最適溫度15–25°C，乾燥陰涼處。不需要冷藏。
* 小葉種柏塘原料製成的紅茶最好在生產後12–18個月內飲用。優質批次可「陳化」至2年，增加柔和度和蜂蜜香的深度。

**11. 價格與仿冒：**

* **價格：** 標準博羅紅茶零售價——每500克200至500元。高等級和紫芽原料批次——每斤1,000元以上。柏塘山茶（綠茶）市場參考：標準批次平均價格每斤200–300元，高等級500+元。

* **如何避免仿冒：**
  1. 通過柏塘認證合作社和企業購買（如「福波」(福波)、「三棵松」(三棵松)、「柏塘春」(柏塘春)、「龍頭一號」(龙头一号)），確保批次可追溯。
  2. 評估茶葉：真正的柏塘茶是小葉種，薄而精緻；如果有人向您提供大葉或粗糙葉片的「博羅紅茶」——很可能不是正宗柏塘原料。
  3. 檢查香氣：純淨、天然，無「焦糊」或酸敗味。特徵性「細長」(细长) 香氣是區別特徵。
  4. 評估茶湯：透明、柔和琥珀色，無渾濁。
  5. 謹慎對待極低價格：正宗手工採摘的柏塘小葉種茶由於採摘勞動密集和產量有限，不可能便宜。

**12. 趣聞：**

* 據傳說，羅浮山上最早的茶樹是道士葛洪 (葛洪, Gě Hóng, 284–364) 下棋時隨手潑灑茶水而生：他隨意將剩茶潑向身後，山崖上立即長出茶樹。葛洪是最偉大的道教煉丹家和博物學家之一，《抱朴子》(《抱朴子》) 的作者。

* 2010年，廣東省農業科學院專家在柏塘發現紫芽茶 (紫芽茶)，確認其屬於陸羽在《茶經》中1,200多年前推崇的「紫色」茶。大葉種紫芽茶在雲南有發現，但小葉種極為罕見。

* 柏塘鎮民間被稱為「老虎圩」(老虎圩，「老虎市場」)：當地人喝茶太多，經常飢餓，像「老虎」一樣消耗肉類——市場上的肉類商品早上就售完。

* 博羅紅茶是首批出口至東南亞和香港的廣東紅茶之一，作為調味茶飲料的原料：以此為基礎製作檸檬茶和荔枝茶。

* 柏塘是粵港澳大灣區 (粤港澳大湾区) 連片茶園面積最大的地區：超過30,000畝茶園，36個行政村幾乎都專業種茶。

**13. 與其他紅茶比較：**

* **英德紅茶 (英德红茶, Yīngdé Hóngchá)：** 主要廣東紅茶——來自大葉種栽培種「英紅九號」(英红九号)。英德紅茶「體感」更強，有巧克力、乾玫瑰和肉豆蔻香。博羅紅茶更精緻：花蜜香，具有小葉種優雅和山地風土的礦物純淨感。

* **滇紅 (滇红, Diān Hóng)：** 雲南 *var. assamica* 紅茶。滇紅「大口徑」：胡椒味、蜂蜜味、強勁。博羅紅茶是完全不同的風格：小葉種精緻，更高甜度，較少澀味，更輕的「體感」。

* **祁門紅茶 (祁门红茶, Qímén Hóngchá)：** 安徽小葉種紅茶，具有特徵性「祁門香」(祁门香) — 玫瑰、果香、微煙燻香。在原料類型上是博羅紅茶最接近的類比，但風土特徵不同：祁門更「北方」，帶酸味；博羅更「南方」，熱帶蜂蜜味和圓潤。

* **潮州工夫紅茶 (潮州工夫红茶)：** 另一種廣東紅茶，但來自潮州原料（通常是單叢栽培種）。以花香蘭花特徵著稱。博羅紅茶更「樸實」，山地蜂蜜味，強調「甘、滑、香」。

**結論：**

博羅紅茶是一款迷人、精緻的紅茶，誕生於嶺南兩大山脈交匯處，在擁有千年茶葉歷史的盆地中。小葉種山地原料——與數百年來製作綠茶的柏塘「山地品種」相同——在紅茶形態下出人意料地展現：甜美、圓潤、花蜜香，具有特徵性順滑和「悠長」餘韻。這款茶會引起精緻、優雅紅茶愛好者的共鳴：那些珍視和諧、純淨以及羅浮山道士曾稱為「仙人氣息」的難以捉摸的山地清新感覺的人們。
