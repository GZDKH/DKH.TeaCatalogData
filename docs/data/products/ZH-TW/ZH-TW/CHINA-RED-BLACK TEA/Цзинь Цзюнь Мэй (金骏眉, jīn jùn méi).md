# 金駿眉 (金骏眉, jīn jùn méi)

**金駿眉 (金骏眉, jīn jùn méi)**
![Цзинь Цзюнь Мэй (金骏眉, jīn jùn méi)](images/Цзинь%20Цзюнь%20Мэй%20(金骏眉,%20Jīn%20Jùn%20Méi).png)

![Цзинь Цзюнь Мэй (金骏眉, jīn jùn méi)-1](images/Цзинь%20Цзюнь%20Мэй%20(金骏眉,%20Jīn%20Jùn%20Méi)-1.png)

金駿眉是現代中國紅茶的巔峰之作，於2005年在四百年正山小種傳統基礎上創製而成。這款茶葉僅採用桐木保護區野生茶樹最嫩的芽頭製作，在短短數年內顛覆了中國對紅茶的認知，成為新一代精品紅茶的象徵。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá)，完全氧化。按歐洲分類為黑茶。發酵度為80–90%。
* **類別：** 精品芽頭紅茶。自2013年起，「金駿眉」被北京市高級人民法院認定為通用名稱 (通用名称, tōngyòng míngchēng)——與鐵觀音、碧螺春和大紅袍同等地位。
* **產地：** 中國福建省 (福建省, Fújiàn Shěng) 南平市 (南平市, Nánpíng Shì) 武夷山市 (武夷山市, Wǔyíshān Shì) 桐木村 (桐木村, Tóngmù Cūn)，位於武夷山國家級自然保護區 (武夷山国家级自然保护区) 內。桐木是全世界所有紅茶的歷史發源地：400多年前正山小種 (拉普山小種) 正是在此誕生。
* **地理座標：** 約北緯27°44′，東經117°38′。
* **別名：** 在「駿眉」(骏眉) 系列中有三個等級：金駿眉 (金骏眉，「金色眉毛」)——僅用芽頭；銀駿眉 (银骏眉，「銀色眉毛」)——一芽一葉；銅駿眉 (铜骏眉，「銅色眉毛」)——一芽二葉。

**2. 歷史與文化意義：**

* **歷史：** 金駿眉是中國最年輕的名茶之一。其歷史始於2005年夏天，當時一群北京茶友——張孟江 (张孟江)、閻翼峰 (阎翼峰) 和馬寶山 (马宝山)——向正山茶業 (正山茶业) 董事長江元勳 (江元勋) 提議製作「超越正山小種最高等級的最佳紅茶」。江元勳是正山小種第24代傳承人，他將這項任務交給了一組茶葉大師——江駿生 (江骏生)、江駿發 (江骏发)、梁駿德 (梁骏德)、溫永勝 (温永胜) 等人。第一批試驗茶——約半斤 (250克) 乾茶，由1.5斤 (750克) 鮮芽製成——由梁駿德於2005年6月21-22日製作完成。結果超出預期：茶葉具有紅茶前所未有的蜜果香氣和絲滑甜味。2006年在茶葉泰斗張天福 (张天福) 和駱少君 (骆少君) 指導下，工藝得到完善和穩定。2008年金駿眉正式上市，立即引起轟動，重新喚起了全中國對紅茶的興趣。從2007年到2013年持續了「商標爭議」：最終北京法院裁定「金駿眉」為通用名稱，不得註冊為專有商標。

* **名稱：** 每個漢字都承載著意義：
  * 「金」(金)——「黃金」。據江元勳的說法：指原料的珍貴、芽頭的金黃色澤和茶湯的金琥珀色調。
  * 「駿」(骏)——「駿馬」、「優秀」。有幾種說法：(1) 三位創始大師——江駿生、江駿發、梁駿德——的名字都含有這個字；(2) 原料採集需要「翻越崇山峻嶺」(崇山峻岭, chóngshān jùnlǐng)；(3) 希望茶葉像駿馬一樣「馳騁」市場。
  * 「眉」(眉)——「眉毛」。描述乾燥芽頭的特徵形狀——細長、微彎，像美麗的眉毛。

* **文化意義：** 金駿眉的出現根本改變了中國紅茶市場格局。2005年之前，絕大多數優質中國紅茶都用於出口；國內市場主要偏向綠茶和烏龍茶。金駿眉證明了紅茶同樣可以精緻多層次，引發了「紅茶復興」(红茶复兴) 浪潮。這款茶成為理想的禮品、收藏品和中國紅茶新地位的象徵。

**3. 植物學描述與原料：**

* **品種/栽培種：** 當地野生或半野生小葉種茶樹群體，稱為奇種 (奇种, Qízhǒng) 或菜茶 (菜茶, Càichá)——*Camellia sinensis* var. *sinensis*。這是一個雜合 (種子繁殖) 群體，數百年來在武夷山保護區高山地帶自然生長。每棵茶樹基因獨特，造就了無可複製的香氣特徵。小葉種相比大葉種 (var. *assamica*) 氨基酸含量較高，茶多酚和咖啡因含量適中，這確保了金駿眉特有的甜味和無苦澀感。
* **採摘：** 早春——從4月初 (清明後) 到5月初 (立夏前)。最佳時間為4月中下旬。較晚採摘的芽頭 (6月) 會產生明顯更淡、濃度較低的茶湯。採摘完全手工進行，在乾燥天氣的上午時分。
* **採摘標準：** 僅採未開放、緊實、肥厚的芽頭 (单芽, dān yá)，覆蓋細絨毛。這是金駿眉與絕大多數紅茶的關鍵區別。製作500克成品茶需要60,000到80,000個鮮芽 (據閻翼峰計算——1斤乾茶約需48,000個芽頭)。
* **原料要求：** 芽頭必須完整、無損傷、大小一致，無機械創傷和變黑痕跡。採摘與加工開始之間的延遲時間最短。

**4. 風土條件與栽培特點：**

* **武夷山保護區：** 面積565平方公裡的國家級自然保護區，位於福建和江西省交界處。1999年被聯合國教科文組織列入世界遺產名錄，為自然文化雙重遺產。山脈主要由紅砂岩和火山岩構成；地貌為陡峭峽谷、瀑布、河流和具有特殊生物多樣性的亞熱帶森林。
* **桐木村：** 紅茶製作的歷史中心，位於保護區深處。茶樹在森林覆蓋下的陡峭山坡上呈半野生和野生狀態生長。
* **生長海拔：** 海拔1000–1800公尺。最佳批次來自海拔1200–1500公尺。
* **氣候：** 亞熱帶高山季風氣候。年平均氣溫約11–18°C (依海拔而異)。年降雨量2000–2300毫米。相對濕度80–85%。全年霧日超過100天。冬季溫和，夏季涼爽——最高溫度很少超過33°C。晝夜溫差大有利於氨基酸和芳香化合物的積累。
* **土壤：** 山地紅壤和山地黃壤，酸性 (pH 4.5–5.5)，富含有機質和礦物質，鐵和錳含量高。排水良好，含有風化砂岩和礫石。酸性土壤反應對茶樹最適宜。

**5. 製作工藝：**

金駿眉的工藝基於正山小種傳統，但有根本性創新：完全排除松木煙燻，所有步驟都適應芽頭原料的嬌嫩特性。整個過程手工完成，需要大師的最高技藝。

* **採摘 (采摘 — cǎizhāi)：** 僅手工採摘未開放芽頭。採茶工在陡峭山坡上作業；經驗豐富的工人一天最多只能採摘幾百克鮮葉原料。
* **萎凋 (萎凋 — wěidiāo)：** 將採摘的芽頭薄攤在竹盤上，置於通風良好的室內。關鍵創新——溫濕度控制 (温湿调控, wēn shī tiáokòng)：大師交替使用自然萎凋和溫萎凋，使水分流失約60–65%。持續時間8–14小時，視天氣而定。芽頭變得柔軟、有彈性，開始形成初步香氣。
* **揉捻 (揉捻 — róuniǎn)：** 完全手工，極其精細。目的不是揉緊乾葉，而是輕微破壞細胞壁以實現均勻氧化。壓力最小，動作為輕柔的圓周運動。過度揉捻不可接受：受損芽頭會產生粗糙口感和暗淡色澤。
* **發酵/氧化 (发酵 — fājiào)：** 將揉捻後的芽頭放在托盤或竹籃中，在控制溫度 (約25–28°C) 和濕度 (約90–95%) 下靜置3–5小時。大師通過顏色 (從綠色轉為銅紅色) 和香氣 (出現明顯蜜果香調，發酵階段就有特徵性「蜂蜜香」——正宗桐木金駿眉的獨特標誌) 判斷完成程度。
* **乾燥/炭焙 (炭焙 — tànbèi)：** 在竹籃中用槐木炭 (槐炭, huái tàn) 傳統乾燥。炭火與茶葉間鋪設江西鉛山縣連四紙 (连四纸)。乾燥分兩階段：毛火 (毛火，「初火」)——約110°C持續約1.5小時後冷卻；足火 (足火，「足火」)——約130°C持續約30分鐘。成品茶殘餘水分3–4%。炭焙固定香氣，賦予茶葉純淨、「透明」的口感，無煙味。
* **分級 (分级 — fēnjí)：** 手工最終篩選——去除破損芽頭、雜質。按大小、形狀和顏色統一批次。

**6. 感官特徵：**

* **乾茶外觀：** 細長、緊實、完整的芽頭，具有明顯的「眉毛狀」形態 (海马状，「海馬狀」——江元勳的原創描述)。正宗桐木金駿眉的關鍵特徵——每個芽頭的三色性：金黃色 (來自絨毛)、黃褐色和黑色——三種色調同時存在。完全金黃的芽頭——通常是其他產區茶葉的標誌，而非桐木產。
* **乾茶香氣：** 純淨、濃郁、甜美，具有明顯蜂蜜香調，龍眼 (龙眼)、荔枝、熟桃、花香 (玫瑰、蘭花) 的韻味。精緻的巧克力和麥芽底調。香氣持久，逐漸展開。真品標誌——乾茶中已能感受到蜂蜜香。
* **茶湯香氣：** 深沉、包覆性強。前幾泡——明亮的果蜜複合香 (龍眼、荔枝)。中段——花香調、溫暖焦糖。後段——純淨甜香帶輕微木質調。杯底香 (挂杯香, guà bēi xiāng)——持久、悠長、蜜花香。
* **口感：** 極其柔和、順滑、絲綢般。完全無苦澀和粗糙感。主導——天然花蜜甜味、果香調 (龍眼、荔枝、桃子、杏乾)、輕微麥芽和巧克力韻味。茶體——中等密度但非常圓潤、「油潤」。明顯的「回甘」(回甘, huígān)。餘韻——持久、純淨、蜜果香，喉間有清涼感。正確沖泡下甜味可持續12泡以上。
* **湯色：** 明亮金琥珀色，有時帶橙蜜色調，清澈透明。茶湯冷卻後杯壁可能出現特徵性「金圈」——茶黃素含量高的標誌。
* **葉底 (沖泡後茶葉)：** 完整、有彈性、未開放的芽頭，保持形狀。顏色——均勻的銅紅色帶金色光澤。芽頭彈性好，大小一致。無破碎或變黑片段——品質標誌。

**7. 化學成分：**

金駿眉與大多數紅茶的區別在於主要成分組的有利比例：氨基酸含量較高，茶多酚和咖啡因適中，這得益於使用高山小葉種奇種。

* **多酚類 (茶多酚)：** 乾重的10–20%。在完全發酵過程中，大部分兒茶素轉化為茶黃素 (茶黄素, 0.4–2%) 和茶紅素 (茶红素, 5–11%)——正是它們形成金琥珀色茶湯、「絲綢」口感和形成「金圈」的能力。茶褐素 (茶褐素) 含量3–9%。
* **氨基酸 (氨基酸)：** 乾重的1.5–4%，超過20種。特別重要的是L-茶氨酸 (L-茶氨酸) ——1.5–2.2%，提供明顯甜味和柔和口感，以及放鬆效果。高山產地提高了氨基酸與多酚的比例。
* **生物鹼：** 咖啡因 (咖啡碱) ——乾重的3–5% (每杯約20–60毫克，視用量和萃取時間而定)。還含有少量可可鹼和茶鹼。
* **維生素：** C、B₁、B₂、B₃ (PP)、E、K。儘管經過發酵，維生素C因精細加工模式部分保留。
* **礦物質：** 約30種元素。主要：鉀 (約佔總礦物質的50%)、磷 (約15%)、鈣、鎂、鐵、錳、氟。微量元素：鋅、銅、硒。
* **精油和揮發性芳香化合物 (芳香油)：** 約0.02%——形成獨特的果蜜花香特徵。芳樟醇、香葉醇、苯乙醛、水楊酸甲酯等成分。
* **其他：** 可溶性糖——2–4%，水溶性果膠——1–2%，有機酸——約1%。

**8. 健康功效：**

* **溫和提神和認知支持：** 咖啡因與L-茶氨酸結合提供平穩、持續的精神狀態，無焦慮感——所謂「平靜清醒」效果。改善專注力和認知功能。
* **抗氧化作用：** 茶黃素和茶紅素具有明顯的自由基清除能力，保護細胞免受氧化應激。
* **心血管系統支持：** 多酚化合物促進血管彈性，可能影響低密度脂蛋白膽固醇水平，有助於血壓正常化。茶黃素擴張毛細血管，改善微循環。
* **舒適消化：** 完全發酵的紅茶對胃黏膜作用溫和，刺激蠕動和消化酶分泌。傳統上建議飯後飲用。
* **抗菌和抗炎作用：** 茶多酚和單寧物質抑制病原菌生長，維護口腔健康。
* **鎮靜和抗壓效果：** 高L-茶氨酸含量刺激α腦波產生，促進放鬆專注狀態。
* **溫暖作用：** 完全發酵的紅茶按傳統中醫理論具有「溫」性，適合「寒」體質人群和冬季飲用。

**9. 沖泡方法：**

* **水溫：** 90–100°C。高品質桐木金駿眉「不怕」沸水——正是充分加熱才能展現香氣深度和甜味。對於精細批次或初次品嚐可從85–90°C開始。
* **茶量：** 功夫茶法3–5克/100–120毫升；歐式方法2–3克/200–250毫升。
* **茶具：** 理想選擇——瓷蓋碗 (盖碗) 100–120毫升：中性材質不吸收香氣，可精確控制出湯時間。玻璃蓋碗或茶壺可觀察芽頭舒展的美態。宜興紫砂壺 (宜兴紫砂壶) 也合適，但新壺應專門用於此茶，避免香氣混合。必備公道杯 (公道杯，「茶海」)。
* **沖泡過程：**
  1. **溫壺：** 用沸水沖燙蓋碗、公道杯和茶杯。
  2. **投茶：** 將3–5克芽頭放入溫熱的蓋碗。品評溫熱茶具中的乾茶香氣。
  3. **潤茶 (润茶 — rùn chá)：** 快速出湯1–2秒——「喚醒」芽頭。倒掉茶水。對金駿眉此步驟非必需——許多大師建議跳過，以免失去首泡的濃郁度。
  4. **首泡：** 沿蓋碗壁輕柔注水 (不直接沖芽頭)，避免損傷絨毛。浸泡時間5–10秒。
  5. **出湯：** 完全倒入公道杯，再從公道杯分入茶杯。泡與泡之間不留水。
  6. **續泡：** 8–12泡 (某些批次可達15泡)。每泡增加3–5秒。中段 (4–7泡) 茶葉通常展現最充分。後段可延長至30–60秒。

**10. 儲存：**

* **容器：** 密封、不透光容器——鐵罐、鋁箔袋配拉鍊、陶瓷茶罐。最大限度減少與空氣接觸。
* **條件：** 乾燥、陰涼處，避免直射陽光、熱源和強烈氣味。溫度10–25°C。濕度不超過60%。
* **保質期：** 最佳在12–18個月內飲用。高品質批次在正確儲存下可保持特性2–3年，但新茶更佳。
* **茶葉天敵：** 光線、濕氣、氧氣、高溫、異味。不要與香料、咖啡、香水一起存放。
* **注意：** 與綠茶和黃茶不同，金駿眉無需冷藏，在沒有可靠密封包裝的情況下也不建議冷藏——紅茶在室溫條件下保存良好。

**11. 價格與仿品：**

金駿眉是世界上最昂貴的紅茶之一。正宗桐木金駿眉來自權威生產商 (正山堂, 骏德茶厂) 的價格可達每500克數千元人民幣 (3,000到10,000+元人民幣)。決定高價的因素：

* **極其費工的採摘：** 500克乾茶需要60,000–80,000個芽頭，每個都在陡峭山坡上手工採摘。
* **有限產區：** 正宗原料——僅來自面積565平方公裡的桐木保護區。
* **手工製作：** 所有關鍵步驟由大師手工完成。
* **短暫採摘季：** 每年僅2–3週。
* **高需求：** 金駿眉是中國最受歡迎的禮品和地位象徵茶之一。

**如何避免仿品：**

* **從可靠商家購買：** 有文件證明來源的專業茶店，理想情況下直接從桐木生產商購買。
* **評估芽頭三色性：** 正宗桐木金駿眉——一個芽頭上有金、黃褐、黑三色。完全金黃芽頭——通常是其他產區 (雲南、四川、貴州) 使用大葉種的茶。
* **檢查香氣：** 乾茶——純淨蜂蜜香，無化學刺激、霉味或煙味。蜂蜜香應在每泡中保持。
* **評估茶湯：** 金琥珀色，透明，杯邊有「金圈」。渾濁或深紅茶湯——替代品標誌。
* **警惕異常低價：** 正宗桐木金駿眉不可能便宜。200–500元人民幣/500克的茶幾乎肯定是其他產區原料製作。

**12. 趣聞軼事：**

* **首批半斤：** 歷史上第一批試驗金駿眉 (2005年6月) 不到250克乾茶。第二天大師們試圖重複成功——結果「搞砸了」：純芽頭原料的加工技術極其敏感。直到第三天才獲得穩定結果。
* **2006年全年137斤：** 茶葉存在第二年，金駿眉總產量約68.5公斤。北京茶商孫連泉 (孙连泉) 收購了一半以上——40+公斤——分銷到北京，讓首都精英認識了這款新茶。這成為「淘金熱」的催化劑。
* **一斤48,000個芽頭：** 據參與首批製作的閻翼峰計算，一斤 (500克) 乾金駿眉含約48,000個茶芽。
* **7年名稱官司：** 「金駿眉」商標之爭 (2007–2013) 涉及數十家茶企，將桐木生產商分為三個陣營。最終裁決——認定為通用名稱——重複了鐵觀音和大紅袍的命運。
* **「紅茶復興」推動者：** 金駿眉出現前，許多桐木茶園因國內紅茶需求停滯而轉產烏龍茶。金駿眉的成功阻止了這一過程，刺激了全中國數十種新精品紅茶的出現。

**13. 與其他紅茶比較：**

* **正山小種 (正山小种, Zhèng Shān Xiǎo Zhǒng)：** 金駿眉的直接「祖先」。用成熟葉片 (一芽二三葉) 製作，傳統上用松木煙燻 (煙燻拉普山小種) 或無煙燻。口感更厚重，有明顯焦糖麥芽調和輕微澀感。金駿眉——明顯更精緻甜美。
* **祁門紅茶 (祁门红茶, Qímén Hóngchá)：** 安徽祁門縣著名紅茶。以獨特「祁門香」(祁门香) 聞名——蘭花、蜂蜜、果乾。與金駿眉區別在於更深的紅寶石色茶湯和稍明顯的澀感。用葉片而非芽頭製作。
* **滇紅金芽 (滇红金芽, Diānhóng Jīn Yá)：** 雲南大葉種 (*var. assamica*) 金芽紅茶。外觀似金駿眉 (金黃芽頭)，但差異顯著：更厚重、濃郁，具焦糖巧克力香料特徵，明顯「厚重感」。金駿眉相比之下——更精緻、輕盈，具果蜜優雅特質。
* **銀駿眉 (银骏眉, Yín Jùn Méi)：** 同系列「小弟」——一芽一葉。價格更親民，香氣稍遜，結構稍明顯，輕微澀感。日常飲用的絕佳選擇。

**結語：**

金駿眉——也許是百年茶葉傳統與創新者勇氣結合創造全新茶葉現象的最佳例證。誕生於四百年拉普山小種歷史與桐木大師實驗精神的交匯點，這款茶不僅佔據了「最昂貴紅茶」的地位——它改變了整個範式：證明了紅茶可以像最好的烏龍茶和綠茶一樣複雜、深邃、多維。

每一次沖泡金駿眉——都是杯中冥想：首泡絲綢般蜜甜、逐漸展開的花果香束、純淨無盡的餘韻，讓人一次次回味。這是慢飲、用心品茶的茶——為那些準備在一杯茶中看見整個世界的人而存在。
