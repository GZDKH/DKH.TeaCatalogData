# 南海紅茶 (南海红茶, Nánhǎi hóngchá)

**南海紅茶 (南海红茶, Nánhǎi hóngchá)**
![Наньхай Хун Ча (南海红茶, Nánhǎi hóngchá)](images/Наньхай%20Хун%20Ча%20%28南海红茶,%20Nánhǎi%20Hóngchá%29.png)

![Наньхай Хун Ча (南海红茶, Nánhǎi hóngchá)-1](images/Наньхай%20Хун%20Ча%20%28南海红茶,%20Nánhǎi%20Hóngchá%29-1.png)

南海紅茶是海南紅茶的歷史品牌，與海南省定安縣的南海農場 (南海农场, Nánhǎi Nóngchǎng) 和同名的南海茶廠 (南海茶厂, Nánhǎi Cháchǎng) 密不可分。這是中國出口紅茶史上的標誌性名稱之一：在發展高峰期，南海茶廠躋身中國五大茶葉企業之列，生產著名的「南海CTC紅碎茶」(南海CTC红碎茶)，產品銷往歐美市場。

**1. 分類與產地：**

* **類型：** 紅茶 (红茶, hóngchá) — 完全發酵（氧化）。歷史上南海紅茶主要以CTC技術生產紅碎茶 (红碎茶, hóng suì chá) 格式，也有限量生產工夫紅茶 (工夫红茶, gōngfū hóngchá) 格式。
* **類別：** 海南紅茶；歷史出口品牌。現代時期為復興中的區域產品，具有地理標誌保護 (地理标志, dìlǐ biāozhì) 元素。
* **產地：** 中國海南省 (海南省, Hǎinán Shěng) 定安縣 (定安县, Dìng'ān Xiàn)。南海農場 (南海农场) 位於海南島東北部，處於從沿海平原向中央高原過渡的丘陵地帶。
* **地理座標：** ≈ 19.7° 北緯，110.3° 東經（以定安縣南海農場位置為參考）。

**2. 歷史與文化意義：**

* **歷史：** 南海紅茶的歷史實質上就是海南工業化茶葉種植的簡史。

  1950年代末，在實施國家出口紅茶基地建設計劃過程中，海南建立了幾個大型茶葉國營農場。南海農場 (南海农场) 成立於1958年，成為行業的旗艦之一。最初在其種植園種植海南大葉種 (海南大叶种) 和雲南大葉種 (云南大叶种)，專注於生產出口紅茶。

  1970年代，南海工廠開始掌握CTC技術——對中國而言全新的碎紅茶生產方法。1974-1976年進行了首批實驗生產，1982年工廠從國外進口了全套CTC設備，成為中國第一家建立完整工業化CTC紅茶生產的企業。這一事件成為整個中國茶業史的裡程碑。

  到1979年，海南紅碎茶，首先是南海工廠的產品，獲得了國際聲譽。1985年在英國世界紅茶競賽中，海南紅碎茶獲得金獎。在高峰期的1980年代末至1990年代初，南海茶廠在全國產量排名第五。海南整體是中華人民共和國主要的紅茶出口基地之一：產品供應給歐洲、美洲和東南亞的18個國家。

  然而從1990年代中期開始衰落。世界紅碎茶市場競爭日益激烈（肯亞、斯裡蘭卡、印度增加產量），而中國國內市場轉向綠茶。到1997年，海南碎紅茶出口急劇減少。南海茶廠面臨管理困難和需求下降，逐漸停止生產。

  2010年代，對海南紅茶的興趣開始復甦——但已是以大葉種原料製作的全葉工夫紅茶格式，面向國內市場。2016年鄰近的五指山紅茶 (五指山红茶) 獲得農產品地理標誌 (农产品地理标志) 保護地位，這刺激了整個海南紅茶種植業的發展，包括前南海農場地區。現代意義上的南海紅茶已不再是CTC碎茶，而是延續地區傳統但以新品質格式呈現的全葉或芽茶紅茶。

* **名稱：** 「南」(南) — 「南方」，「海」(海) — 「海洋」：「南海」— 既是南中國海的歷史名稱，也是海南具體農場的名稱。「紅茶」(红茶) — 「紅茶」。南海紅茶字面意思是「南海紅茶」。

* **文化意義：** 南海紅茶是海南茶業工業化過去的象徵，那個紅茶從島上「香飄全世界」(香飘全世界, xiāng piāo quán shìjiè) 的時代。對幾代海南茶業工作者來說，南海工廠是工作和驕傲的地方。現代品牌復興融入海南自由貿易港 (自由贸易港, zìyóu màoyì gǎng) 框架下發展茶業的大戰略：來自海南大葉種原料的紅茶被定位為世界級產品，能與最優秀的印度和錫蘭茶樣品競爭。

**3. 植物學描述與原料：**

* **品種/栽培種：** 歷史上南海農場種植園使用兩個主要栽培種：
  - 雲南大葉種 (云南大叶种, Yúnnán Dàyè Zhǒng) — 從雲南引進的大葉種栽培種 *Camellia sinensis* var. *assamica*。特點是強壯的枝條、大葉片，多酚含量高，非常適合紅茶生產。
  - 海南大葉種 (海南大叶种, Hǎinán Dàyè Zhǒng) — 當地海南大葉種阿薩姆型品種，被認定為國家品種「華茶16號」(华茶16号, GSCT16)。根據研究數據，茶多酚含量約35%，咖啡因約6%，使其成為特別珍貴的遺傳資源。
  現代南海紅茶批次也可能使用半野生和老樹原料，以及先前在海南種植用於綠茶生產但也適合紅茶的福建栽培種（福鼎大白 (福鼎大白)；福雲6號 (福云6号)）。

* **採摘：** 海南的熱帶氣候提供幾乎全年的生長期。主要季節：春季（2-4月）、夏季（5-7月）、秋季（8-10月），以及冬季採摘（11-1月）。歷史上CTC生產使用所有季節採摘；現代工夫紅茶以春季最為珍貴。

* **採摘標準：** CTC用相對成熟的葉片（一芽二至三葉）。現代工夫紅茶用一芽一葉 (一芽一叶, yī yá yī yè) 或一芽二葉 (一芽二叶, yī yá èr yè)。芽茶批次用單芽 (单芽, dān yá)。

* **原料要求：** CTC用體積大、新鮮的葉片，無粗梗，適合機械加工。工夫紅茶用嫩芽明顯的嫩枝，無機械損傷；新鮮、有彈性、高汁液含量的原料。

**4. 風土與栽培特色：**

* **海拔高度：** 海拔50-300公尺。南海農場地區位於從沿海平原向海南中央高原過渡的丘陵地帶。

* **氣候：** 熱帶季風氣候。年平均溫度23.8°C — 中國茶區中最高之一。年平均降雨量1,900-2,000毫米，但分布極不均勻：超過一半集中在8-10月，冬季和早春相對乾燥。年平均濕度約85%。丘陵地帶經常有霧和雲層。熱帶氣候確保茶樹全年生長——除了春、夏、秋季採摘外，還可能有冬季採摘。

* **土壤：** 主要是磚紅壤 (砖红壤, zhuān hóng rǎng)，發育於玄武岩 (玄武岩, xuánwǔyán) 基岩上。土壤深厚、黏重、中等肥力。玄武岩成因確保良好的礦物化，包括鐵、錳、鉀和微量元素含量，對紅茶「體感」和礦物質感的形成有積極影響。

* **農業技術：** 歷史上南海農場實行集約化種植園茶業，機械化採收用於CTC生產。現代面向工夫紅茶的生產者轉向更溫和的做法：手工採摘、限制農化、有機施肥。總體而言，海南茶業正朝生態化和認證方向發展。

**5. 生產工藝：**

歷史上南海紅茶主要是CTC紅碎茶。現代批次也按傳統工夫紅茶工藝生產。以下描述兩種格式。

**CTC格式（歷史）：**
- **萎凋 (萎凋, wěidiāo)：** 縮短，常使用熱風 — 4-8小時將濕度降至65-68%。
- **CTC加工 (CTC切碎, CTC qiēsuì)：** 葉片通過一對不同轉速的滾筒，執行三個操作：壓碎（Crush）、撕裂（Tear）和捲曲（Curl）。葉片細胞結構幾乎完全破壞，確保快速而強烈的發酵。
- **發酵 (发酵, fājiào)：** 由於細胞最大程度破壞 — 加速而強烈，在25-30°C下1-2小時。形成高濃度的茶黃素和茶紅素。
- **乾燥 (干燥, gānzào)：** 快速高溫乾燥。
- **分級 (分级, fēnjí)：** 按粒度 — 從大顆粒（類似BOP）到細粉（Dust）。

**工夫格式（現代）：**
- **採摘 (采摘, cǎizhāi)：** 手工選擇嫩芽。
- **萎凋 (萎凋, wěidiāo)：** 自然，在通風良好的房間，12-18小時。
- **揉捻 (揉捻, róuniǎn)：** 傳統機械揉捻以形成葉片形狀並釋放汁液。
- **發酵 (发酵, fājiào)：** 在25-30°C，3-5小時。高多酚含量的海南大葉種原料產生濃郁、「厚重」的風味特徵。
- **烘乾 (烘干, hōnggān)：** 兩階段：初烘105-115°C，精烘80-90°C以發展蜂蜜焦糖香調。
- **分級 (分级, fēnjí)：** 按葉片品質和芽頭含量。

**6. 感官特徵：**

**CTC格式：**
* **外觀：** 均勻、緊實的深色顆粒，帶油潤光澤。
* **香氣：** 高揚、尖銳、持久，帶麥芽和焦糖香調。
* **滋味：** 濃縮、強勁，具有明顯的「濃強」(浓强, nóng qiáng) 和「鮮爽」(鲜爽, xiān shuǎng)。非常適合加奶和糖飲用：加奶後呈現粉紅色澤和柔和、均勻的甜味。
* **湯色：** 鮮紅、濃郁，帶明顯金圈 (金圈)。
* **葉底：** 細小、均勻紅色顆粒，無大片段。

**工夫格式（現代）：**
* **乾茶外觀：** 緊結、密實的條索；大而肉厚的葉片帶金毫 (金毫)。顏色深栗色至黑色。
* **乾茶香氣：** 蜂蜜香，帶熱帶乾果、龍眼和淡焦糖香調。芽茶批次有明顯的「鮮爽」(鲜爽, xiān shuǎng)。
* **茶湯香氣：** 溫暖、包覆性，從蜂蜜和成熟熱帶水果到麵包焦糖調性和基調中的細膩香料的多層次轉換。
* **滋味：** 厚實、圓潤，具有海南大葉種原料特有的「油潤感」。甜味純淨，澀味適中而柔和。回甘 (回甘) 持久、溫暖，帶蜂蜜尾韻。
* **湯色：** 紅琥珀色，明亮、透明，帶明顯金圈。
* **葉底：** 大片、有彈性的葉片，均勻呈銅紅至栗色；芽頭金橙色。

**7. 化學成分：**

* **多酚類：** 海南大葉種是多酚含量的記錄保持者之一（鮮葉中高達35%）。CTC加工時多酚最大程度氧化，形成高濃度茶黃素（TF，1.5-2.5%）和茶紅素（TR，12-18%），確保濃郁色澤和茶湯「濃強度」。工夫格式中氧化更溫和，TF/TR平衡更柔和。
* **胺基酸：** L-茶胺酸、谷胺酸；總含量約佔乾重2-3%。提供柔和和甜味特徵。
* **生物鹼：** 咖啡因 — 鮮葉中高達6%（中國茶葉中最高指標之一），這是由於大葉種阿薩姆型特性。可可鹼和茶鹼 — 微量。
* **維生素：** B群（B₁、B₂）、P（芸香苷）。維生素C — 鮮葉中高達200+毫克/100克，完全發酵後顯著降低。
* **礦物質：** 鉀、鎂、錳、鐵、鋅。玄武岩土壤富含微量元素。
* **精油：** 芳樟醇、香葉醇；美拉德反應產物 — 麥芽酚、呋喃酮，形成蜂蜜焦糖風味特徵。

**8. 保健功效：**

* 由於高咖啡因含量結合L-茶胺酸，具有明顯提神作用 — 清醒效果而無急躁興奮。
* 強大抗氧化活性：來自大葉種原料的海南紅茶茶黃素和茶紅素是中國紅茶中最濃縮的之一。
* 促進消化：單寧和多酚刺激消化酶分泌，特別是在油膩食物後有效。
* 支持心血管系統：類黃酮改善血管彈性。
* 具有溫暖作用，有助緩解主觀疲勞感。
* CTC格式加奶後成為柔和、舒適的飲品，具有額外營養價值 — 鈣和蛋白質來源。
* 來自玄武岩土壤的高礦物質含量可能豐富飲食中的微量元素。

**9. 沖泡方法：**

**CTC格式：**
* **水溫：** 100°C（沸水）。
* **茶量：** 2-3克/200毫升（CTC萃取很快）。
* **茶具：** 陶瓷或瓷茶壺、帶蓋茶杯、茶壺（茶葉過濾器）。
* **過程：**
  1. 將茶葉放入預熱的茶具中。
  2. 注入沸水，浸泡3-5分鐘。
  3. CTC茶通常不適合多次沖泡 — 1-2次。
  4. 製作「奶茶」(奶茶)：泡濃茶（4-5克/200毫升，5分鐘），加熱牛奶和糖調味。

**工夫格式：**
* **水溫：** 90-95°C。
* **茶量：** 5-6克/100-120毫升。
* **茶具：** 蓋碗 (盖碗)、瓷茶壺、宜興茶壺。
* **過程：**
  1. 用熱水溫熱茶具。
  2. 投茶，讓茶葉「醒茶」15-20秒。
  3. 洗茶 — 快速沖泡（1-2秒）用於緊結條索。
  4. 第一泡：8-12秒。
  5. 第二至四泡：10-15秒。
  6. 之後每泡增加5-10秒；可沖6-10泡。

**10. 儲存：**

* 密封容器，避光、防潮、防異味。
* 最佳條件：15-25°C，濕度不超過60%。
* CTC茶：建議12個月內飲用 — 開封後碎茶更快失去香氣。
* 工夫格式：新茶在前12-18個月最佳；優質大葉種原料批次在2-3年內「圓潤化」。

**11. 價格與仿品：**

* **價格：** 歷史CTC南海紅茶是出口大宗產品 — 價格類別「中下」。現代全葉南海紅茶（工夫格式）— 根據等級和採摘標準，500克200-800元人民幣。芽茶和古樹批次 — 1,000元起。

* **如何避免仿品：**
  1. 確認產地：正宗南海紅茶應來自前南海農場地區或定安縣鄰近種植園的原料。
  2. 評估葉片：海南大葉種原料明顯比福建或浙江紅茶更大更厚實。
  3. 檢查香氣：純淨、蜂蜜熱帶香，無酸敗和霉味。
  4. 評估湯色：明亮紅琥珀色，明顯金圈，透明度。
  5. 謹慎對待標示為「南海CTC」但包裝無生產商標示的產品 — 原CTC工廠已停產，此名稱下可能銷售其他產地茶葉。

**12. 有趣事實：**

* 南海茶廠成為中國第一家建立完整工業化CTC紅茶生產的企業。1982年工廠進口全套CTC設備（滾筒機、乾燥設備），開啟中國茶業史新篇章。在此之前，CTC技術是印度、肯亞和斯裡蘭卡的專利。

* 在1980年代末至1990年代初的發展高峰期，南海農場躋身中國五大茶葉企業。其產品在全國競賽中獲得「特等獎」(特等奖, tèděng jiǎng)。

* 1959年廣東對外貿易局派遣海南考察隊，結果產生《海南島茶葉調查報告》(《海南岛茶叶调查报告》)。報告指出海南大葉種「品質可與世界著名的印度和錫蘭紅茶相媲美」— 這是中國紅茶首次與南亞巨頭並列的案例之一。

* 海南是中國最南端的茶葉省份（北緯18-20°），南海紅茶是世界上最「熱帶」的紅茶之一。這裡的茶樹一年12個月都在生長 — 即使冬季採摘也不停止。

* 「南海」(南海, 「南海」) 這個名稱將這款茶與偉大的海洋歷史聯繫起來：南中國海是最古老的貿易路線之一，茶葉沿此路線數百年來運送到東南亞，再到歐洲。象徵性地，在這片海岸生產的紅茶本身成為出口商品，走了相反的路線 — 從海南到倫敦和紐約。

**13. 與其他紅茶比較：**

* **白沙紅茶 (白沙红茶, Báishā Hóngchá)：** 來自同一海南但不同縣（白沙）和不同風土（隕石坑）的最近「同鄉」。白沙紅茶在礦物質特徵上更「奇特」。南海紅茶是更「經典」的熱帶紅茶，歷史上面向出口。

* **五指山紅茶 (五指山红茶, Wǔzhǐshān Hóngchá)：** 來自同一海南大葉種的高山海南紅茶（高達1,000+公尺）。因高山風土被認為是海南紅茶的「明珠」。南海紅茶更多是丘陵地帶，具有不同的氣候特徵。

* **滇紅 (滇红, Diān Hóng)：** 來自 *var. assamica* 的雲南紅茶 — 原料類型最相近的類比。滇紅更「乾燥」和「胡椒味」；南海紅茶更「濕潤」、熱帶，帶明顯龍眼和焦糖香調。

* **阿薩姆CTC (Assam CTC)：** 南海紅茶CTC版本最相近的國際類比。兩者都來自 *var. assamica*，都面向出口和加奶飲用。阿薩姆CTC在「濃強度」和「麥芽味」上更強；海南的更柔和，帶熱帶甜味。

**結論：**

南海紅茶是一款具有鮮明、戲劇性命運的紅茶。誕生於1950-1980年代工業躍進時期的南中國海岸，它經歷了崛起、世界認可和遺忘 — 現在以全葉工夫紅茶格式獲得新生。強勁的阿薩姆型大葉種原料、熱帶風土、玄武岩土壤和幾乎全年生長期賦予這款茶濃郁的蜂蜜焦糖滋味、油潤質感和溫暖回甘。南海紅茶是歷史之茶、象徵之茶：提醒我們在中國最南端、熱帶島嶼上，生長著能與最優秀阿薩姆和錫蘭茶樣品競爭的紅茶。
