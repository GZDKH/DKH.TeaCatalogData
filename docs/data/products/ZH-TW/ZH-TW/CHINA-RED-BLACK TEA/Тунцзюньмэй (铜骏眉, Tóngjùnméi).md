# 銅駿眉 (銅駿眉, Tóngjùnméi)

**銅駿眉 (銅駿眉, Tóngjùnméi)**
![銅駿眉 (銅駿眉, Tóngjùnméi)](images/Тунцзюньмэй%20(铜骏眉,%20Tóngjùnméi).png)

![銅駿眉 (銅駿眉, Tóngjùnméi)-1](images/Тунцзюньмэй%20(铜骏眉,%20Tóngjùnméi)-1.png)

銅駿眉——「銅色的高貴眉毛」——是2005年誕生於武夷山國家級自然保護區桐木村（桐木村，Tóngmù Cūn）的著名駿眉（駿眉）系列中的第三等級。如果說金駿眉（金駿眉，Jīn Jùnméi，「金色眉毛」）是純芽和精工細作的精髓，銀駿眉（銀駿眉，Yín Jùnméi，「銀色眉毛」）是一芽一葉，那麼銅駿眉則採用更成熟的原料——一芽二至三葉——並經過更深度的發酵。其結果是「眉」系列中最飽滿、濃郁且最耐泡的代表，具有鮮明的蜜糖果香特徵和出色的性價比。在商業流通中，銅駿眉經常以小赤甘（小赤甘，Xiǎo Chìgān——「小紅甜」）和大赤甘（大赤甘，Dà Chìgān——「大紅甜」）的別名銷售，反映了採摘標準和葉片成熟度。

**1. 分類與產地：**

* **類型：** 中國紅茶（紅茶，hóngchá），完全氧化。屬於正山小種（正山小種，Zhèngshān Xiǎozhǒng）——「正宗山區小葉種[茶]」家族，但採用創新的無煙（無煙，wúyān）技術製作，有別於傳統的煙燻小種。
* **類別：** 駿眉系列等級：金（金，「黃金」）→ 銀（銀，「白銀」）→ 銅（銅，「青銅/銅」）。銅駿眉是系列中最實惠的等級，同時擁有與高級「兄弟」不同的獨特鮮明特色。
* **產地：** 中國，福建省（福建省，Fújiàn Shěng），南平市（南平市，Nánpíng Shì），武夷山市（武夷山市，Wǔyíshān Shì），星村鎮（星村鎮，Xīngcūn Zhèn），桐木村（桐木村）。桐木位於武夷山國家級自然保護區（武夷山國家級自然保護區）的核心地帶——該地區被列入聯合國教科文組織世界自然與文化遺產名錄。這裡是所有正山小種紅茶家族的發源地，也是駿眉系列的搖籃。正宗的銅駿眉原料必須來自「正山」（正山，「正確的山」）——包括桐木村及周邊高山地區（麻粟、掛墩、江墩、廟灣坪等）在內的565平方公里保護區範圍。
* **地理座標：** 約北緯27°45′，東經117°40′（桐木村/桐木關）。

**2. 歷史與文化意義：**

* **歷史：** 銅駿眉與金駿眉同時誕生——2005年6月，當製茶大師梁駿德（梁駿德，Liáng Jùndé）和正山茶業（正山茶業）的企業家江元勳（江元勳，Jiāng Yuánxūn）應北京茶葉愛好者——記者閻翼峰（閻翼峰）和收藏家張孟江（張孟江，「佚士茶人」）的建議，嘗試用野生奇種（奇種，qízhǒng——「稀有/奇特品種」）茶樹的純芽製作紅茶。第一批——僅半斤（約250克）——令人驚豔：金黃色的茶湯、蜜糖花香、難以置信的甜度。產品被命名為「駿眉」（駿眉——「高貴/迅捷的眉毛」）：駿（jùn）——紀念大師梁駿德，同時也有「茶中駿馬」之意（取自迅捷駿馬的形象）；眉（méi）——「眉毛」——形容捲曲的茶葉形狀，如同彎曲的眉毛。為區分等級，張孟江、閻翼峰和馬寶山於2005年制定了《駿眉令》（《駿眉令》，「駿眉法令」）：清明採摘的純芽——金（黃金）；穀雨採摘的一芽一葉——銀（白銀）；立夏採摘的一芽二至三葉——銅（青銅）。實際上，由於「銅/青銅」一詞對購買者來說聽起來不夠吸引人，梁駿德大師開始以「小赤甘」和「大赤甘」——「小」和「大紅甜」的商業名稱銷售銅駿眉。這些名稱在市場上紮根。到2009年，駿眉在中國茶業掀起了一場真正的革命，將桐木從一個被遺忘的村莊變成了紅茶愛好者的朝聖中心，並在全中國重新激發了對紅茶的興趣。
* **名稱：** 銅（tóng）——「銅、青銅」——表示系列中的第三等級（在黃金和白銀之後），也指乾茶葉和泡後葉底特有的銅青銅色。駿（jùn）——「高貴的駿馬」（迅捷和純種的形象），同時也是對創造者梁駿德名字的致敬。眉（méi）——「眉毛」——比喻捲曲茶葉的形狀：纖細、彎曲、帶有輕微的「絨毛」。
* **文化意義：** 駿眉系列成為中國「紅茶復興」的象徵。2005年之前，紅茶在國內需求有限（主要產品用於出口）；駿眉的出現徹底改變了這種情況——紅茶變得時尚、受歡迎和有聲望。銅駿眉作為系列中最實惠的代表，在這一趨勢的民主化中發揮了關鍵作用：它讓廣大茶葉愛好者能夠體驗「桐木風格」，而無需支付金駿眉的天價。桐木村——「世界紅茶的發源地」（世界紅茶的發源地）——因駿眉獲得了新生：茶農從摩托車換成了汽車，竹屋變成了石頭房子。

**3. 植物學描述與原料：**

* **品種/栽培種：** 野生群體品種——奇種（奇種，qízhǒng——「奇特品種」），也稱菜茶（菜茶，càichá——「菜園茶」）：在保護區自然條件下生長的小葉種*Camellia sinensis* var. *sinensis*茶樹，生長在竹林中、山澗旁和岩石突出處。許多茶樹覆蓋著厚厚的苔蘚，證明其樹齡達數十年甚至數百年。奇種是未經選育的遺傳異質群體；每棵茶樹都是獨特的，這為成品茶的香氣增添了額外的複雜性。
* **採摘：** 根據《駿眉令》，銅駿眉在立夏（立夏，5月初——夏季開始）前後採摘。實際上，採摘從4月底持續到6月初，在金駿眉和銀駿眉的主要採摘結束後進行。較晚的採摘時間意味著葉片有時間展開並積累更多的多酚和芳香物質。
* **採摘標準：** 1芽+2-3片嫩葉。葉片必須新鮮、嫩（嫩，nèn），無機械損傷。採摘在早晨（7:00-10:00）進行，天氣乾燥，採用「提手採」（提手採，tíshǒu cǎi）方法——不扭轉和擠壓嫩梢。
* **原料要求：** 新鮮採摘的葉片必須立即送到工廠；不允許擠壓、過熱和過早變紅。不使用雨水葉。優選帶有淡綠色或略帶黃色芽的嫩梢；深綠色的被認為品質較差。

**4. 風土與栽培特點：**

* **生長海拔：** 800-1,500公尺；主要茶園位於約1,200公尺的中等海拔。最有價值的原料來自保護區內的「正山」（正山）地區。
* **氣候：** 典型的亞熱帶山地氣候：年平均溫度11-18°C；年降雨量約2,000毫米；年平均濕度80%；霧天數達120天/年。柔和的散射光和充足的水分有助於葉片中氨基酸和精油的積累。
* **土壤：** 酸性（pH 4.5-5.0），深度30-90公分，基於風化石英岩和花崗岩。由於竹葉和森林凋落物的分解而富含有機質。
* **生態：** 森林覆蓋率96.3%。桐木是東亞保存最完好的亞熱帶森林之一的核心。茶樹生長在樹木、竹林和蕨類植物之間；幾乎沒有專門的種植園——茶葉是從分散在山坡上的「野生」茶樹採集的。不使用農藥和化肥；保護區的生態系統提供了天然的害蟲防護。進入該地區受到嚴格限制：桐木對外國公民仍然是封閉區域——這是19世紀的遺產，當時英國「植物獵人」羅伯特·福瓊（Robert Fortune）秘密從這裡帶走了種子和紅茶生產的秘密。

**5. 生產技術：**

銅駿眉採用駿眉系列的創新無煙技術生產，基於正山小種的傳統，但沒有傳統的松木煙燻步驟。與金駿眉的主要區別是發酵程度更深（達70-80%或更高）以及使用更成熟的葉片。

* **採摘（採摘——cǎizhāi）：** 1芽+2-3葉，早晨手工採摘。
* **萎凋（萎凋——wěidiāo）：** 組合式：日光萎凋（日光萎凋，rìguāng wěidiāo）——在竹篾上薄層攤放（不超過2公分），每10-20分鐘翻動一次，直到柔軟並失去光澤；然後——室內萎凋（室內萎凋，shìnèi wěidiāo），使用濕潤的暖空氣。《駿眉令》將此表述為「半陰半陽，晾芽青」（半陰半陽晾芽青）。
* **揉捻（揉捻——róuniǎn）：** 按照「輕推重拉」（輕推重拉）的原則：開始輕柔作用，然後加強以形成緊結並擠出汁液。揉捻的葉片形成球狀（坨，tuó）。
* **氧化（發酵——fājiào）：** 揉捻葉球覆蓋濕布（坨蓋濕布）並在室溫下保持。《駿眉令》規定發酵「七成」（酵七成），但對於原料更成熟的銅駿眉，發酵程度是系列中最高的：葉片呈現深銅紅色，香氣濃郁，帶有蜜糖和果香調。
* **烘焙（烘焙——hōngbèi）：** 低溫、無煙、慢烘（低溫無煙慢烘焙）。無煙是與傳統小種的根本區別。低溫慢烘保留了精緻的芳香化合物並形成特有的「蜜糖」甜味。嚴格規則：「切記莫做隔夜青」（切記莫做隔夜青）——從採摘到烘乾的整個週期必須在一天內完成。
* **分級（分級——fēnjí）：** 去除粗梗，統一顆粒大小。

**6. 感官特徵：**

* **乾茶外觀：** 捲曲比金駿眉和銀駿眉更大更鬆散；葉片更寬，葉片展開明顯。顏色——「半黃九半黑」（半黃九半黑）：整體色調深暗，帶有來自芽毫的個別金青銅色斑點。毫尖明顯少於高級等級。
* **乾茶香氣：** 明顯的蜜糖甜味帶花果香調（花蜜香，huāmì xiāng）。香氣比金駿眉更「濃厚」和「溫暖」，帶有烤地瓜、乾果和輕微焦糖調。
* **茶湯香氣：** 多層次：前幾泡——明亮的花香（蘭花、玫瑰），然後——果香（桃子、杏子），最後——純蜜糖和甜點心。香氣持久性高。
* **口感：** 飽滿（醇厚，chúnhòu）、濃郁，帶有明顯的蜜糖甜味和持久的「回甘」（回甘，huígān）。茶湯比金駿眉更厚重和「濃稠」。正確沖泡時無苦澀味。餘韻——持久、包覆感強，喉部有愉悅的「掛喉感」（掛喉感，guàhóu gǎn）。
* **茶湯顏色：** 琥珀金黃色，透明，帶溫暖的蜜糖色調（湯色澄黃）。比金駿眉的金黃透明茶湯顏色更濃。
* **葉底（泡後茶葉）：** 銅紅色帶青銅光澤，葉片充分展開；可見完整的「芽+2-3葉」嫩梢。質地有彈性、柔軟。

**7. 化學成分：**

* **多酚類：** 由於原料更成熟，含量高於金駿眉。在深度發酵過程中，大部分兒茶素轉化為茶黃素（TF）和茶紅素（TR），形成濃郁的顏色、厚重的「茶體」和天鵝絨般的茶湯質地。
* **氨基酸：** L-茶氨酸——含量適中（低於純芽的金駿眉，但足以確保柔和和甜味）。高海拔產地和竹林遮蔭即使在較成熟的葉片中也能保持良好的茶氨酸水平。
* **生物鹼：** 咖啡因（乾重的2.5-4%——由於成熟葉片的存在，略高於純芽等級）、可可鹼、茶鹼。
* **維生素：** 維生素C（部分保留）、B群維生素、β-胡蘿蔔素。
* **礦物質：** 鉀、錳、鋅、氟、鐵——石英岩土壤上高山武夷山原料的典型組合。
* **精油和揮發性化合物：** 芳樟醇、香葉醇、β-紫羅蘭酮、壬醛——形成特有的花蜜香氣。更成熟的葉片由於烘乾時的美拉德反應帶來額外的「溫暖」香調（麥芽醇、糠醛）。

**8. 保健功效：**

* 由於咖啡因和L-茶氨酸的協同作用，溫和提神並提高注意力；提神效果平穩，無急劇波動。
* 抗氧化作用：茶黃素和殘留的兒茶素中和自由基並支持細胞保護。
* 暖胃助消化——紅茶對胃部有溫和、溫柔的作用（暖胃，nuǎn wèi）；特別適合餐後飲用。
* 促進心血管健康：紅茶多酚支持血管彈性，可能有助於血壓正常化。
* 含有對口腔健康有益的氟和多酚：強化琺瑯質並抑制致齲細菌。
* 具有溫和的利尿作用；有助於排毒。
* 有助於緩解疲勞，恢復精神和體力。
* 豐富的蜜糖香氣和L-茶氨酸成分有助於放鬆和減少焦慮。

**9. 沖泡方法：**

* **水溫：** 90-100°C。與精緻的金駿眉不同，銅駿眉的成熟葉片在接近沸點的溫度下能完美展開；這樣可以更充分地提取厚重的「茶體」和蜜糖深度。桐木大師們的常見做法是用滾開水沖泡。
* **茶量：** 3-5克/100-120毫升（功夫茶）；3-4克/200-300毫升（杯泡或西式）。
* **茶具：** 白瓷蓋碗（蓋碗）100-120毫升——可以欣賞香氣和顏色；瓷器或玻璃茶壺。日常飲用也可使用帶蓋的普通茶杯。
* **過程：**
  1. 用開水預熱蓋碗，倒掉。
  2. 放入茶葉，蓋上蓋子3-5秒——吸入蜜糖「乾香」。
  3. 洗茶：快速沖泡1-2秒，倒掉（可選——可以跳過）。
  4. 第一泡：5-10秒。
  5. 後續沖泡：每次增加5秒；從第6-7泡開始——更明顯地增加（至15-20秒）。
  6. 沖泡次數：8-12泡或更多——由於葉片更厚實、成熟，銅駿眉是所有駿眉等級中最耐泡的。正是在後期沖泡中展現出其標誌性的深層蜜糖甜味。

**10. 儲存：**

儲存在密封不透明容器中（錫罐或陶瓷罐、帶鋁箔層的真空包裝），放在乾燥涼爽處，溫度不超過22-25°C，遠離陽光直射和異味。最佳飲用期限——18-24個月。銅駿眉儲存穩定；在適當條件下可陳放2-3年，期間香氣會變得柔和，口感會獲得額外的圓潤度。不需要冷藏。

**11. 價格與仿冒品：**

銅駿眉是駿眉系列中最實惠的等級。如果正宗桐木金駿眉每公斤價格達到數萬元人民幣，那麼銅駿眉（小赤甘/大赤甘）便宜一個數量級——每500克從幾百到幾千元不等，取決於具體的採摘地點和製茶師。儘管如此，銅駿眉也被大量仿冒：由於「桐木風格」的流行，市場充斥著來自福建其他地區（政和、坦洋、建甌）甚至鄰省的仿製品。

* **如何避免仿冒品：**
  1. 從桐木村的生產商或有透明供應鏈的可靠經銷商購買；理想情況下——標明製茶師姓名（實名制，shímíng zhì）。
  2. 外觀：真正的銅駿眉——「半黃半黑」（不是均勻的黑色也不是亮金色）；捲曲——自然，不是完美均勻；葉片——不太細碎（這不是純芽等級）。
  3. 茶湯：琥珀金黃色，透明，帶蜜糖香氣和甜味，無苦澀。渾濁、深紅色茶湯或明顯的苦味——是假貨的標誌。
  4. 耐泡性：真正的桐木銅駿眉可沖泡8-12次，保持味道和香氣；廉價仿製品在3-4泡後就「空了」。
  5. 標有「桐木關」/「正山」的茶葉價格異常低廉——幾乎肯定是假貨。

**12. 趣聞軼事：**

* 「銅駿眉」這個名稱在零售貨架上幾乎不使用。梁駿德大師首先提出改名：「在它前面是黃金和白銀；銅已經賣不出好價錢了」。於是出現了商業名稱「小赤甘」（小赤甘——來自芽+2葉）和「大赤甘」（大赤甘——來自芽+3片更成熟的葉）。這些名稱成為獨立的商品項目。
* 根據《駿眉令》，駿眉還可以製成「駿眉冰」（駿眉冰——冷凍/冰凍）和「駿眉餅」（駿眉餅——壓製成餅）。這些形式極為罕見，被視為收藏品。
* 500克金駿眉需要50,000-58,000個芽；500克銅駿眉需要的嫩梢數量明顯較少（每個重量更大）。這是價格差距的主要原因。
* 創始大師梁駿德——世代製茶工匠，8歲開始製茶：他的祖母是纏足婦女，無法用腳踩茶（桐木傳統的揉捻方式），於是教會了孫子。2008年，梁駿德離開「正山茶業」公司，創立了自己的工廠「駿德茶廠」（駿德茶廠），成為駿眉系列的標杆生產商之一。
* 桐木居民說江西方言（而不是福建閩南話），大多數家庭的族譜來自江西省：根據家族記載，梁駿德大師是來自貴溪（貴溪，龍虎山腳下的縣城）移民的後代——他的祖先500多年前遷居桐木。

**13. 與其他駿眉系列和小種紅茶的比較：**

* **金駿眉（金駿眉，Jīn Jùnméi）：** 最高等級——清明採摘的純芽。茶湯——金黃透明，「溫暖琥珀」色；香氣——極其精緻，花蜜香帶蘭花（蘭花）香調；口感——絲滑、細膩，強調甜味和「輕盈感」。相比之下，銅駿眉——更「接地氣」、厚重、濃郁，具有明顯的「茶體」和更強的耐泡性。
* **銀駿眉（銀駿眉，Yín Jùnméi）：** 中等級——1芽+1葉，穀雨採摘。介於金的輕盈和銅的飽滿之間的中間特性：香氣——花果香，口感——甜味帶輕微結構感，茶體——中等。
* **正山小種——傳統煙燻（正山小種 傳統煙燻）：** 經典風格，用松木煙燻（松煙，sōngyān）。香氣——煙味、龍眼、乾果；口感——厚重，「煙甜」。銅駿眉沒有煙燻味；其特性——純蜜糖和花香。
* **正山小種——無煙（正山小種 無煙）：** 現代版小種，不經煙燻。風格上最接近銅駿眉，但小種的採摘標準更多樣（從芽+1-2葉到更成熟的原料）；而銅駿眉定位為優質系列的一部分，具有更嚴格的品質和技術控制。
* **老叢紅茶（老叢紅茶，Lǎocóng Hóngchá）：** 來自桐木覆蓋苔蘚的老茶樹的紅茶。具有獨特的「叢味」（叢味）——帶酸味和「海洋」清新感的「苔蘚」味。用老叢製作的銅駿眉結合了兩種特性，但在市場上這類批次被歸為單獨類別。

**結語：**

銅駿眉——駿眉系列的「人民英雄」：它承載著同樣的桐木風土DNA——野生奇種茶樹、山霧、竹林、石英岩土壤——但以更飽滿、濃郁和親民的形式展現出來。其蜜糖深度、青銅溫暖和令人印象深刻的耐泡性使其成為那些欣賞真正武夷山紅茶而不願為「黃金」等級支付過高價格的人的理想日常茶飲。試著用滾開水在白瓷蓋碗中沖泡銅駿眉，觀察香氣如何從花香演變為果香，再到純淨、包覆的蜜糖香。正是在第八到第十泡時，銅駿眉才完全展現——也正是在這一刻，人們才明白為什麼來自桐木的「青銅」比許多其他地方的「黃金」更有價值。
