# 五指山紅茶 (五指山红茶, Wǔzhǐshān hóngchá)

**五指山紅茶 (五指山红茶, Wǔzhǐshān hóngchá)**
![Учжишань Хун Ча (五指山红茶, Wǔzhǐshān hóngchá)](images/Учжишань%20Хун%20Ча%20(五指山红茶,%20Wǔzhǐshān%20Hóngchá).png)

![Учжишань Хун Ча (五指山红茶, Wǔzhǐshān hóngchá)-1](images/Учжишань%20Хун%20Ча%20(五指山红茶,%20Wǔzhǐshān%20Hóngchá)-1.png)

五指山紅茶是中國唯一的熱帶高山紅茶，產於海南島，採用當地特有的大葉種栽培品種海南大葉種製作。此茶的獨特名片是「琥珀湯、奶蜜香」(琥珀汤、奶蜜香，「琥珀色茶湯、奶蜜香氣」)，這個配方已成為品牌的官方標語。該產品自2015年起受國家地理標誌保護，並列入中國-歐盟地理標誌互認清單。

**1. 分類與產地：**

* **類型：** 中國紅茶 (红茶, hóngchá)，全發酵茶。屬於大葉種紅茶 (大叶种红茶, dàyèzhǒng hóngchá)。
* **類別：** 海南省區域性紅茶；有多種商品形式：工夫紅茶 (工夫红茶)、傳統紅碎茶 (红碎茶, hóng suìchá，碎紅茶) 和 CTC。
* **產地：** 中國海南省 (海南省, Hǎinán Shěng) 五指山市 (五指山市, Wǔzhǐshān Shì)。主要生產區域：水滿鄉 (水满乡, Shuǐmǎn Xiāng)、毛陽鎮 (毛阳镇)、南聖鎮 (南圣镇)、番陽鎮 (番阳镇) 等。也在島南部尖峰嶺 (尖峰岭) 山脈地區種植。
* **地理座標：** 北緯18°38′–19°02′，東經109°19′–109°44′（根據證書AGI2015-03-1770的保護區域邊界）。

**2. 歷史與文化意義：**

* **歷史：** 五指山地區的茶葉種植有著悠久的歷史。島上的原住民黎族 (黎族, Lízú) 和苗族 (苗族, Miáozú) 自古就採集野生茶葉用於治療感冒、發燒和消化不良。在《瓊臺志》(《琼台志》，「瓊臺志」，1512年，明代正德年間) 中已記載五指山地區的茶葉生產：「其中最著名的是五指山水滿所產之茶，樹大如抱，味清而妙」。清代時期，水滿茶 (水满茶, Shuǐmǎn Chá) 被列入「南荒貢品」(南荒贡品) 名錄，進貢朝廷。現代工業化紅茶的歷史始於1959年，當時廣東省對外貿易部門在海南建立了紅茶出口基地。《海南島茶葉勘察報告》(《海南岛茶叶勘察报告》，1959年) 指出，當地大葉種栽培品種生產的紅茶品質可與印度和錫蘭同類產品媲美。到1965年，島上新茶園面積達1.5萬畝，年產乾茶350噸。發展高峰期在1988-1993年，海南省成立後國家投資超過1000萬元人民幣發展茶葉產業；到1993年島上有50多個茶場，茶園面積12萬畝，年產量8000噸。正是在這個時期，海南紅茶以「遠航」(远航，「遠航」) 品牌出口，這個名稱由周恩來總理 (周恩来) 親自批准。經過1990年代的衰落後，2000年代開始復興：2015年中華人民共和國農業部授予「五指山紅茶」農產品地理標誌 (农产品地理标志) 地位。2017年該茶在博鰲亞洲論壇上展示。2019年在陳宗懋院士 (陈宗懋) 參與下成立了五指山大葉茶科學研究院，並開設了學術工作站。2020年五指山紅茶列入中國-歐盟地理標誌互認第二批清單。
* **名稱：** 「五指山」(五指山) — 「五指山」，海南島主峰（1867公尺），因五個山峰的特徵輪廓而得名。對黎族人民來說，這是神聖的山峰，宇宙的中心。「紅茶」(红茶) — 紅茶。當地茶葉也以歷史名稱「水滿茶」(水满茶) 聞名：黎語中「水滿」意為「古老的、最高的」。
* **文化意義：** 五指山紅茶是海南的象徵之一，是當地待客文化不可分割的一部分。「老爸茶」(老爸茶，「爸爸茶」) 傳統 — 悠閒地品飲紅茶，常加煉乳或檸檬 — 是島上茶文化日常生活的名片。茶葉成為黎族和苗族山區村莊經濟復興的工具：「公司+合作社+農戶」模式使茶葉種植成為水滿及鄰近村莊數千家庭的主要收入來源。

**3. 植物學描述與原料：**

* **品種/栽培品種：** 主要栽培品種為海南大葉種 (海南大叶种, Hǎinán Dàyè Zhǒng)，又稱「華茶16號」(华茶16号, GSCT16)，1984年被認定為國家一級標準品種。屬於 *Camellia sinensis* var. *assamica*，大葉種，喬木型或半喬木型。特點是葉片厚實肉質，芽頭大，持嫩性強，多酚類和氨基酸含量豐富。五指山野生茶樹可達10公尺以上高度。分為三個生態型：野生型 (野生型)、栽培型 (栽培型) 和「野放型」(野放型, yěfàng xíng) — 野化的種植園茶樹。也使用從雲南引進的品種（阿薩姆大葉種系）。
* **採摘：** 由於熱帶氣候，茶樹幾乎全年生長；一年可採摘9輪，持續11個月。早春採摘（1月底-2月）產出「中國第一早春茶」(中国第一早春茶，「中國第一早春茶」) 產品。春茶和秋茶品質優於夏茶。
* **採摘標準：** 工夫紅茶為1芽1-2葉；CTC和碎茶允許較成熟的葉片。
* **原料要求：** 新鮮完整的葉片，無機械損傷；由於熱帶氣候下酶活性高，採摘後必須儘快加工。

**4. 風土條件與種植特色：**

* **地形與生態：** 五指山是海南中部生態區的核心，屬於熱帶雨林國家公園。森林覆蓋率超過80%，生物多樣性包括3800多個物種。該地區是島上最重要的集水區。
* **種植海拔：** 主要茶園位於海拔300-800公尺；野生茶樹分佈至海拔1200公尺。水滿茶區平均海拔約600公尺。
* **氣候：** 熱帶海洋性季風氣候，兼具山地熱帶雨林特徵。關鍵特點是低緯度（北緯19°以下）結合高海拔，創造出獨特組合：日照時間長、冬季溫和無霜凍、降雨充沛（年降雨量超過2500毫米）和高濕度。雲霧天氣超過200天。年平均氣溫22.4°C；明顯的晝夜溫差有利於芳香物質和氨基酸的積累。該地區因山地環繞而不受颱風影響。
* **土壤：** 磚紅壤 (砖红壤, zhuānhóng rǎng) — 深厚疏鬆，富含鐵和鋁，由熱帶條件下強烈的化學和生物風化作用形成。pH值4.5-6.5。由於豐富的森林落葉，有機質含量高。
* **負離子含量：** 山區茶園空氣中負離子含量達每立方公分12萬個 — 達到6級以上標準，證明大氣環境極其清潔。

**5. 製作工藝：**

標準工藝流程包括從採摘到包裝的十個步驟。工夫紅茶和CTC在揉捻階段有所不同。

* **採摘 (采摘, cǎizhāi)：** 手工採摘「1芽1-2葉」標準。
* **萎凋 (萎凋, wěidiāo)：** 採用「柔光萎凋」(柔光萎凋, róuguāng wěidiāo) 技術，使葉片逐漸均勻失水。葉片變軟，出現淡淡花香。
* **揉捻 (揉捻, róuniǎn)：** 破壞細胞壁以釋放汁液。海南大葉種原料角質層薄、細胞結構嫩，易於揉捻，確保充分出汁和均勻發酵。
* **氧化/發酵 (发酵, fājiào)：** 形成色、香、味的關鍵步驟。大葉種原料中表沒食子兒茶素沒食子酸酯 (L-EGCG) 和其他酯型兒茶素含量高，確保茶黃素和茶紅素的強烈形成，造就特徵性的「琥珀色」茶湯。
* **初烘 — 毛火 (毛火, máohuǒ)：** 熱風快速乾燥以停止發酵。
* **攤涼 (摊凉, tānliáng)：** 平衡葉片水分。
* **復烘 — 二烘 (二烘, èr hōng)：** 將水分含量調至中間水平。
* **攤涼 (摊凉)。**
* **足火 (足火, zúhuǒ)：** 將水分含量調至≤7%，固定香氣。
* **篩分與包裝 (筛分, shāifēn; 装箱, zhuāngxiāng)：** 按等級分類，形成商品批次。

**6. 感官特徵：**

* **乾茶外形：** 緊結肥碩的條索 (条索紧结肥硕, tiáosuǒ jǐnjié féishuò)；棕褐油潤的色澤 (棕褐油润, zōnghè yóurùn)。可見金黃芽頭。
* **乾茶香氣：** 明顯的奶蜜香 (奶蜜香, nǎi mì xiāng) — 五指山紅茶的名片。伴有烤栗子香和淡淡花香背景。
* **茶湯香氣：** 深沉包覆，以奶蜜和溫暖餅乾香為主；冷卻後展現乾果層次。
* **滋味：** 甜醇爽滑 (甜醇爽滑, tiánchún shuǎnghuá)；茶體飽滿圓潤，蜜甜回甘 (回甘) 持久。澀味極少，口感柔和「絲滑」。
* **湯色：** 紅琥珀色 (红琥珀色, hóng hǔpò sè)，明亮透澈 — 「琥珀湯」形象是品牌官方描述的一部分。
* **葉底（沖泡後葉片）：** 肥軟紅亮 (肥软红亮, féi ruǎn hóngliàng)，芽葉完整清晰可見。

**7. 化學成分：**

* **多酚類：** 茶多酚含量≥10%（按GI標準）；由於大葉種原料，酯型兒茶素（L-EGCG、L-ECG、L-EGC）含量顯著高於小葉種同類產品，確保茶黃素（≥0.1%）和茶紅素（≥2.5%）的強烈形成。
* **氨基酸：** 游離氨基酸含量>1.5%；L-茶氨酸為主要氨基酸，確保滋味柔和和「奶香」特色。
* **生物鹼：** 咖啡鹼≥2%乾重；可可鹼和茶鹼含量微量。
* **水浸出物：** ≥34.0% — 表明滋味濃郁和內含物質豐富。
* **灰分：** ≤7.0%，表明原料純淨。
* **礦物質和微量元素：** 鈷、鉬、鋅、硒 — 微量元素豐富的特徵，與熱帶火山土壤礦化有關。
* **維生素：** 含B族維生素、維生素C（即使在完全發酵後仍部分保留）。
* **成分特色：** 海南大葉種的多酚類與氨基酸比例接近最佳，確保茶體飽滿與滋味柔和的平衡 — 這是大葉種紅茶的罕見品質。

**8. 保健功效：**

* **溫和提神作用：** 咖啡鹼與茶氨酸結合，提供平穩持久的精神狀態，無起伏和「咖啡因震盪」。
* **抗氧化保護：** 茶黃素和茶紅素有效中和自由基；大葉種原料提供比小葉種同類產品更高的抗氧化劑濃度。
* **消化支持：** 多酚氧化產物溫和刺激腸蠕動和消化酶分泌；黎族和苗族傳統上用於治療腸道疾病。
* **心血管支持：** 規律適量飲用紅茶與改善血脂狀況和維持血管彈性相關。
* **免疫調節作用：** 茶多酚具有抗病毒和抗菌活性；黎族歷史上使用野生茶預防感冒疾病。
* **溫胃舒適作用：** 紅茶在中醫食療中屬「溫性」食品；推薦給腸胃敏感者。
* **微量元素補充：** 五指山土壤中天然的鈷、鉬和硒含量轉移到茶葉中，有助補充微量營養素缺乏。

**9. 沖泡方法：**

* **水溫：** 標準工夫紅茶90-95°C；精緻芽茶批次85-90°C。
* **茶量：** 工夫法5-6克配100-120毫升；歐式法3克配200-250毫升。
* **茶具：** 瓷蓋碗 (盖碗) 100-120毫升 — 評估香氣的最佳選擇；玻璃茶壺可欣賞湯色。緊實批次可用紫砂壺。
* **沖泡過程：**
  1. 用開水溫熱茶具。
  2. 投茶並聞溫熱乾茶香氣。
  3. 洗茶（可選）：快速注水2-3秒；倒出。
  4. 第一泡：8-10秒。
  5. 後續沖泡：每泡增加5秒。
  6. 沖泡次數：優質批次6-8泡；大葉種原料耐多次沖泡。
* **海南「老爸茶」方式：** 沖泡濃紅茶，加一勺煉乳或檸檬片，配輕食 — 島上日常茶飲的傳統風格。

**10. 儲存：**

密封、避光容器；溫度10-25°C；避免潮濕和異味。大葉種五指山紅茶在18-24個月內保持良好品質。某些芽頭含量高的緊實批次在儲存2-3年後會發展 — 蜜香加深，茶體更加圓潤。在海南熱帶氣候條件下，使用真空包裝或乾燥劑包裝尤為重要。

**11. 價格與仿冒：**

價格區間：中等至中高等。基礎批次80-150元人民幣/100克；優質有機野生原料批次300-600元人民幣以上。價格因素：原料類型（野生比栽培貴）、採摘季節（早春最珍貴）、芽頭比例、加工方式（工夫比CTC貴）和品牌。

* **如何避免仿冒：**
  1. 檢查農產品地理標誌 (农产品地理标志) 標識 — 產品必須在五指山範圍內生產。
  2. 評估香氣：正宗五指山紅茶具有特徵性奶蜜香；缺乏此香調是嚴重疑點。
  3. 控制湯色：真正的「琥珀色」(琥珀色) 是關鍵標誌；過深或混濁的茶湯表明工藝問題。
  4. 品嚐滋味：天然「滑甜」無明顯澀味或苦味。
  5. 警惕替代：由於品牌知名度提高，市場上出現海南其他地區原料冒充五指山產品。

**12. 有趣事實：**

* 五指山紅茶是中國最「南方」的高山紅茶，也是唯一在北緯20度以南生產的紅茶。在低緯度方面，它在中國茶葉中無與倫比。
* 海南大葉種是1984年認定的30個國家標準茶樹品種之一（編號「華茶16號」，GSCT16）。根據1988-1989年調查結果，沿海南三大山脈 — 五指山、黎母山和雅加大嶺發現野生茶樹，其中五指山地區被認為是該品種的起源中心和最大多樣性區域。
* 「遠航」(远航，「遠航」) 品牌是1960-1980年代海南紅茶出口品牌，由周恩來總理親自命名，當時被認為是世界紅茶品牌之一。
* 2017年五指山紅茶是博鰲亞洲論壇參會者的官方用茶 — 海南最大的國際活動。
* 陳宗懋院士，中國茶學界最高權威，稱海南大葉種為「製作紅茶的優秀原料」，並表示「五指山紅茶可以長期保持『紅』色」。
* 水滿茶區是黎族居住地，對他們來說五指山是神聖的宇宙中心。許多茶園位於原儀式森林區域，為當地茶業賦予特殊文化內涵。
* 除經典工夫紅茶外，海南還生產獨特的香蘭紅茶 (香兰红茶) — 添加海南香草莢 (香荚兰, *Vanilla planifolia*) 的紅茶，賦予飲品淡雅的香草奶油尾韻。

**13. 與其他紅茶比較：**

* **滇紅 (滇红, Diān Hóng)** — 雲南紅茶，同樣採用大葉種阿薩姆栽培品種製作。滇紅通常具有更辛辣的胡椒蜜香特徵和「麥芽」厚重感。五指山紅茶更輕盈，甜味更細緻，特徵性奶蜜香在滇紅中沒有類似物。
* **英紅 (英红, Yīng Hóng)** — 廣東紅茶，採用大葉種英紅九號栽培品種。兩者都是大葉種華南紅茶，但英紅更厚重「巧克力味」，而五指山更柔和，強調「滑甜」和琥珀透明湯色。
* **錫蘭高地紅茶** — 在「熱帶+高度」組合方面最接近的國外類似產品。兩者都具有明亮香氣和明顯甜味，但錫蘭茶通常更澀更「柑橘味」，而五指山更柔和、更「奶香」、更「絲滑」。
* **阿薩姆紅茶（印度）** — 植物學上相關（var. *assamica*）。阿薩姆具有強勁、麥芽、「晨茶」特徵；五指山顯著更精緻甜美，無攻擊性澀味。

**結論：**

五指山紅茶是地理位置極其特殊的茶葉：熱帶緯度、山地高度和特有大葉種栽培品種創造出在世界茶業中沒有直接類似物的產品。其「琥珀湯色和奶蜜香氣」不是廣告誇張，而是由獨特風土條件和植物學特性形成的客觀感官特徵。這款茶適合尋求超越常見雲南和福建紅茶的異國體驗的品茶者，也適合被其溫和甜味和完全無苦味所吸引的初學者。
