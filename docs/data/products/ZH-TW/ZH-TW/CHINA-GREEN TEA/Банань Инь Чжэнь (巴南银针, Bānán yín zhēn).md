# 巴南銀針 (巴南银针, Bānán yín zhēn)

**巴南銀針 (巴南银针, Bānán yín zhēn)**
![Банань Инь Чжэнь (巴南银针, Bānán yín zhēn)](images/Банань%20Инь%20Чжэнь%20%28巴南银针,%20Bānán%20Yín%20Zhēn%29.png)

![Банань Инь Чжэнь (巴南银针, Bānán yín zhēn)-1](images/Банань%20Инь%20Чжэнь%20%28巴南银针,%20Bānán%20Yín%20Zhēn%29-1.png)

巴南銀針 (巴南银针, Bānán yín zhēn) 是來自重慶市巴南區的高品質針形綠茶，是這座直轄市茶文化的名片。茶葉種植於白象山 (白象山) 海拔800-1200公尺處，位於明月山脈 (明月山) 的雲霧帶中，採用非物質文化技藝「巴南茶葉製作技藝」(巴南茶叶制作技艺) 生產，該技藝已傳承四代。筆直挺拔的茶條覆蓋銀白色茸毫，沖泡時垂直豎立後緩緩沉降至底部——這一景象如「午後春雨」般美妙，為此茶贏得了「亞太城市市長峰會唯一指定用茶」(2005年)、地理標誌產品 (2011年) 和全國綠色食品博覽會金獎 (2022年) 等殊榮。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，不發酵茶。屬於針形綠茶 (针形绿茶, zhēnxíng lǜchá)。工藝上結合炒制與烘乾，以烘乾為主導——獨創的「初乾以烘代炒」(chūgān yǐ hōng dài chǎo——「初乾以烘代炒」) 方法。

* **類別：** 國家地理標誌商標註冊產品 (国家地理标志商标注册, 2011年)。「中國名牌農產品」(中国名牌农产品)。列入「全國名特優新名錄」(全国名特优新名录)。生產技藝「巴南茶葉製作技藝」(巴南茶叶制作技艺) 列入重慶市非物質文化遺產 (重庆市非物质文化遗产, 2009年)。2004年加工技術獲得國家發明專利 (国家发明专利)。多次榮獲「中茶杯」(中茶杯，第6、7屆競賽一等獎)、「華茗杯」(华茗杯，金獎)、「三峽杯」(三峡杯，金獎) 等獎項。「重慶老字號」(重庆老字号)。亞太城市市長峰會唯一指定用茶 (亚太城市市长峰会唯一指定用茶, 2005年) 和上海世博會指定用茶 (上海世博会指定用茶, 2010年)。「巴南銀針」品牌價值評估為14.1億元人民幣。

* **產地：** 中國重慶市 (重庆市, Chóngqìng Shì) 巴南區 (巴南区, Bānán Qū)。核心產區——二聖鎮 (二圣镇, Èrshèng Zhèn) 白象山 (白象山, Báixiàng Shān)，明月山脈 (明月山脉)。

* **地理座標：** 約北緯29°30′，東經106°42′。

**2. 歷史與文化意義：**

* **歷史：** 巴南的茶葉傳統根深蒂固。陸羽 (陆羽) 在《茶經》(《茶经》，8世紀) 中寫道：「茶者，南方之嘉木也……其巴山峽川有兩人合抱者」(其巴山峡川有两人合抱者)——這是對現今重慶所在地區古老茶業的直接證明。

	**清代——原型創建。** 清朝末期 (19世紀)，巴南地區創製了名為「定心巴渝銀針」(定心巴渝银针，「定心巴渝銀針」) 的茶葉。然而由於20世紀上半葉的戰爭和政治動盪，該技藝失傳。

	**復興與品牌建立 (1980-1990年代)。** 1980年代，重慶二聖茶業公司 (二圣茶业公司，現隸屬重茶集團——重茶集团) 的專家開始恢復失傳的技藝。1990年獲得首個試驗樣品，1995年茶葉正式命名為「巴南銀針」(巴南银针) 並註冊商標。

	**認可與發展 (2000年代至今)。** 2004年生產技術獲國家發明專利保護——確立了關鍵創新：「殺青中度偏嫩、初乾以烘代炒」(殺青中度偏嫩、初乾以烘代炒) (殺青偏向「嫩」度 + 初乾以烘代炒)。2005年成為亞太城市市長峰會唯一指定用茶——此事件將品牌推向國際層面。2009年「巴南茶葉製作技藝」列入重慶市非物質文化遺產名錄。技藝傳承由劉觀祿 (刘观禄) 守護——世代茶農，第四代傳承人。2011年獲得地理標誌。至2017年，巴南區茶園面積達4萬畝 (~2,670公頃)，年產量3,040噸。產品出口丹麥、美國等國家。

* **名稱：**
	* 「巴南」(巴南)——「巴地之南」。巴南區位於重慶南部——該地區歷史上屬於古巴國 (巴国, Bāguó，西元前1千年) 領土。「巴」(巴) 這一地名是重慶地區身份認同的重要標誌，茶名刻意強調這一聯繫。
	* 「銀針」(银针)——「銀針」。描述成品茶條的形狀：筆直、纖細，覆蓋銀白色茸毫，如閃亮的針。在中國茶葉傳統中，「銀針」這一名稱專用於最高品質的茶葉——由單芽或芽帶初展葉製成。

* **文化意義：** 巴南銀針是重慶四大主要茶葉品牌之一 (與永川秀芽、南川古樹茶、雞鳴貢茶並列)。巴南區歷史上是重慶的「茶區」：「以前巴南幾乎家家戶戶都做茶」——傳承人劉觀祿回憶道。白象山上的定心茶園 (定心茶园) 成為重慶近郊主要旅遊景點之一，每年舉辦「採茶節」(采茶节)，吸引數萬遊客。2019年「定心」被中國國際農業合作促進會認定為「中國最美麗茶園」(中国最美丽茶园)。

**3. 植物學描述與原料：**

* **品種/栽培種：** 使用多個 *Camellia sinensis* 品種：
	* **福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbái Chá)** — 主要栽培種，*C. sinensis* var. *sinensis*。提供明顯的芽頭特性、豐富的白毫和細膩的胺基酸風味。
	* **巴渝特早 (巴渝特早, Bāyú Tèzǎo)** — 超早熟無性系品種，列入國家級良種 (国家级良种) 名錄。可在2月底至3月初開始採摘。
	* **雲南大葉種 (云南大叶种)** — 輔助大葉種，用於特定批次。
	鮮葉化學成分數據：多酚類 ≥19.8%，水浸出物 ≥47.4% — 極高指標，證明內含物質豐富。

* **採摘：** 巴南銀針100%原料在清明前 (明前，~4月5日) 採摘。最珍貴的批次來自第一波春茶 (越冬後第一層芽頭)，於2月底至3月初採摘 (得益於巴渝特早品種)。

* **採摘標準：**
	* 單芽級 (单芽级)：僅採第一波春茶的飽滿未開放芽頭。最高等級，價格每斤2000元起。
	* 特級 (特级)：一芽一葉初展 (一芽一叶初展)。
	* 一級 (一级)：一芽一葉。
	* 二級 (二级)：一芽二葉。

* **原料要求：** 芽葉新鮮、均勻、無損傷。手工採摘。當日加工。

**4. 風土與栽培特色：**

* **氣候：** 白象山位於重慶典型的亞熱帶濕潤季風氣候區。年平均氣溫18-20°C。相對濕度 ≥75%。年霧日超過200天。日夜溫差顯著，促進芽葉胺基酸積累。春茶胺基酸含量 ≥3.10% — 這一指標解釋了茶湯鮮爽甘甜的特點。

* **海拔：** 海拔800-1200公尺。核心產區——白象山海拔800-1000公尺高度帶，位於「雲霧帶」(云雾带)。

* **土壤：** 黃壤 (黄壤, huáng rǎng)，pH值4.5-6.0，富含腐殖質和微量元素，包括硒。茶園位於水源保護區 (水源保护区)。禁用化學肥料和農藥；實施「豬—沼—茶」(猪—沼—茶) 生態循環系統。

* **核心產區：** 二聖鎮白象山國家級茶樹良種繁育基地 (白象山国家级茶树良种繁育基地)，巴南區。總面積2萬餘畝 (~1,340公頃)，其中標準化示範園4000畝。茶園配備氣象站監測溫度、降雨、濕度和光照。

**5. 製作工藝：**

巴南銀針採用專利技術 (2004年國家專利) 生產，包含37道工序 (工序)。關鍵創新：「殺青中度偏嫩」(杀青中度偏嫩) — 溫和處理避免過度炒制；「初乾以烘代炒」(初干以烘代炒) — 保持白毫並形成「針形」外觀。工藝結合機械化和手工操作，作為非物質文化遺產傳承。

* **攤放 (摊放 — tān fàng)：** 芽葉薄攤4-8小時。長時間攤放是巴南銀針工藝的特色：此期間水分降低，開始發展栗香，葉片獲得彈性。

* **殺青 (杀青 — shāqīng)：** 滾筒式約150°C或微波殺青。採用原則：「嫩葉老殺，老葉嫩殺」(嫩叶老杀，老叶嫩杀)。總體偏向「嫩」處理 (中度偏嫩)，以保持最大新鮮度。

* **揉捻 (揉捻 — róuniǎn)：** 輕揉以形成基本結構而不破壞茸毫。

* **復揉 (复揉 — fùróu)：** 額外操作以緊實茶條結構。

* **理條提毫 (理条提毫 — lǐtiáo tíháo)：** 關鍵步驟，決定「針形」外觀。茶葉拉直，塑形成細直「針」，同時白毫「提升」(提毫) 至表面，形成銀白光澤。

* **烘乾 (烘干 — hōnggān)：** 熱風乾燥至含水量 ≤5%。此步驟實現專利創新「以烘代炒」— 熱風乾燥代替最終炒制，保持茸毫細膩和香氣純淨。

* **精製·提香 (精制·提香 — jīngzhì · tíxiāng)：** 精細分級 + 最終提香以「提升香氣」並固定栗香特徵。

**6. 感官特徵：**

* **乾茶外形：** 筆直挺拔如針 (挺直似针, tǐngzhí sì zhēn)，覆蓋豐富銀白茸毫 (绿润披毫, lǜ rùn pī háo)。色澤濃綠帶銀白光澤。茶條極其均勻整齊。

* **乾茶香氣：** 栗香 (栗香, lìxiāng) — 主調。伴有清純綠茶新鮮感 (清香, qīngxiāng) 和淡雅「毫香」，類似嫩玉米香 (毫香/玉米香)。

* **茶湯香氣：** 高揚栗香，持久悠長。高等級茶品帶有額外蜜香層次。

* **滋味：** 鮮醇 (鲜醇, xiānchún)，甘爽持久 (甘爽, gānshuǎng)。輕微澀味迅速轉化為甘甜 (微涩速化)。湯體適中，「鮮活」。回甘 (回甘) 明顯持久。

* **湯色：** 黃綠明亮 (黄绿明亮, huánglǜ míngliàng)。

* **葉底 (沖泡後茶葉)：** 嫩綠勻整 (嫩绿匀整)。芽葉完整「鮮活」(芽叶鲜活)。在玻璃杯中沖泡時呈現特有的「針舞」：茶條垂直豎立，然後緩緩沉降至底部——這一景象成為巴南銀針的視覺名片。

**7. 化學成分：**

* **多酚類 (兒茶素)：** ≥19.8%。適中指標，確保溫和口感而無過度澀味——「嫩殺青」的結果。

* **水浸出物：** ≥47.4% — 中國綠茶中最高指標之一，說明可溶性成分極其豐富。

* **胺基酸 (包括L-茶胺酸)：** 春茶中 ≥3.10%。高含量是「鮮爽」甘甜口感的關鍵因素。

* **生物鹼：** 咖啡因 (適中含量)、可可鹼、茶鹼。

* **硒 (Se)：** 0.15-0.35毫克/公斤。白象山土壤含硒，自然積累於茶葉中。

* **維生素：** 維生素C、B群維生素 (B1, B2)、維生素E、維生素K。

* **礦物質：** 鉀、鎂、鋅、氟。

**8. 保健功效：**

* **抗氧化作用：** 多酚類 (~20%) 有效中和自由基。
* **提神效果：** 咖啡因與L-茶胺酸 (≥3.10% 胺基酸) 結合——溫和平衡的提神和專注力提升。
* **脂質代謝支持：** 兒茶素促進脂肪分解和膽固醇水平正常化。
* **硒補充：** 硒含量 (0.15-0.35毫克/公斤) — 對免疫系統和抗氧化防護的額外益處。
* **消化支持：** 多酚類刺激消化過程。
* **清熱作用：** 傳統綠茶「清涼」效果——在重慶炎熱潮濕氣候中特別珍貴。

**9. 沖泡方法：**

* **水溫：** 玻璃杯85-90°C；蓋碗80-85°C。特級茶不超過85°C。過熱 (>90°C) 破壞新鮮感並產生苦澀。

* **茶量：** 3克配150毫升 (玻璃杯，比例1:50)。

* **茶具：** 玻璃杯 (玻璃杯) — 最佳選擇：可觀賞「銀針舞蹈」，茶條在水中垂直豎立後緩緩沉降——巴南銀針因此效果而特別珍貴。白瓷蓋碗 — 最適合聚香。

* **沖泡過程：**
	1. 用熱水溫杯。
	2. 投入茶葉。
	3. 注入熱水 (85-90°C)。不要攪拌——讓「針」自然豎立。
	4. 觀賞「舞蹈」2-3分鐘：茶條垂直上升，然後緩緩沉降。
	5. 首泡2-3分鐘後可品飲。
	6. 蓋碗沖泡：首泡10秒，每次增加5-10秒。可沖泡4次。

**10. 儲存：**

* 密封包裝，避光、防潮、防異味。
* 最佳條件——冰箱0-5°C密封保存。
* 新茶建議在陰涼乾燥處「醒茶」15天以消除「火氣」(火气)。
* 開封後7天內飲用 (!) 以最大程度保持香氣。巴南銀針香氣極其精緻，在空氣中容易失去新鮮感。

**11. 價格與仿冒：**

* **價格區間：** 重慶綠茶高端產品。參考價格：單芽級每斤2000元起；特級每斤700元起；一級400-600元；二級200-400元。

* **防偽方法：**
	* 從官方生產商購買——重茶集團 (重茶集团) 及其子公司二聖茶業公司 (二圣茶业公司)，以及重慶和線上授權銷售點。
	* 檢查包裝上的地理標誌商標「巴南銀針」。
	* 正品茶條筆直均勻，銀毫豐富。仿品常彎曲，茸毫暗淡或缺失。
	* 香氣純正栗香，無異味。湯色透明黃綠。沖泡時「針」應垂直豎立——這是正確成型和真品原料的標誌。
	* 特級茶價格低於每500克400元應引起懷疑。

**12. 趣味知識：**

* **市長和總統的茶。** 2005年巴南銀針成為亞太城市市長峰會唯一指定用茶 (亚太城市市长峰会)。2007年成為重慶直轄市10週年慶典招待用茶和官方禮品。2010年成為上海世博會指定用茶 (上海世博会指定用茶)。

* **37道工序與4代傳承。** 巴南銀針製作技藝包含37道工序 (工序)，作為非物質文化遺產傳承四代。現任傳承人劉觀祿 (刘观禄)：「客人來時，父親總是泡一杯巴南銀針——湯色綠亮透明，每一口都有巴南人的溫暖情懷」。

* **「銀針舞蹈」。** 在玻璃杯中沖泡時，巴南銀針茶條垂直豎立，如微型竹林，5-6分鐘後緩緩沉降。這一視覺效果源於茶條密度 (通過「理條提毫」成型) 與茸毫保持的氣泡之間的精確平衡。觀賞「舞蹈」是品茗的傳統環節。

* **「重慶版斯裡蘭卡」。** 白象山定心茶園 (定心茶园) 因其雲霧繚繞的梯田景觀獲得非官方稱號「重慶版斯裡蘭卡」(重庆版斯里兰卡)。這裡每年舉辦「定心採茶節」(定心采茶节)，設有手工製茶體驗工坊、品茗廳、茶餐廳 (特色菜：「茶香雞」、「綠茶豆腐」)、民宿和露營區。

* **「嫩度」專利。** 2004年技術——「殺青中度偏嫩、初乾以烘代炒」— 成為重慶茶業首個國家發明專利。創新要點：更溫和的殺青保持最多胺基酸，以熱風乾燥替代最終炒制保護白毫不被「燒焦」。

**13. 與重慶其他綠茶比較：**

* **永川秀芽 (永川秀芽)：** 同樣來自重慶，但為捲曲形。巴南銀針為「針形」芽頭，風味更「細膩」。

* **三峽龍井 (三峡龙井)：** 重慶產。扁平，龍井類型。巴南銀針為「針形」，強調茸毫和「銀色」美學。

* **定軍茗眉 (定军茗眉)：** 漢中產。「眉形」，富硒。巴南為不同產區 (重慶)，「針形」。

**13. 與其他「針形」綠茶比較：**

* **永川秀芽 (永川秀芽)：** 同樣來自重慶，但不同區域。扁平直形，「眉形」。巴南為圓針形，更多茸毫。

* **恩施玉露 (恩施玉露)：** 蒸青工藝，「海味」。巴南為炒青工藝，栗香。

* **涇縣特尖 (泾县特尖)：** 安徽產。同為「針形」，但不同省份，不同風土。巴南來自白象山，海拔400-1000公尺。

**總結：**

巴南銀針是一款集三種珍貴品質於一身的茶葉：技藝上的非物質文化遺產、「銀針舞蹈」的視覺詩意，以及建立在豐富胺基酸 (≥3.10%) 和創紀錄高浸出物含量 (≥47.4%) 基礎上的風味特徵。這是來自重慶心臟地帶的茶葉——這座以麻辣火鍋而非精緻綠茶聞名的直轄市，在其南部山坡、白象山雲霧中發現如此細膩純淨的茶葉更顯珍貴。適合那些同樣重視沖泡視覺美學和杯中滋味的茶友。
