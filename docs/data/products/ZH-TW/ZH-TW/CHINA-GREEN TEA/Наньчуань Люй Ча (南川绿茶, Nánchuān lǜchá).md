# 南川綠茶 (南川绿茶, Nánchuān lǜchá)

**南川綠茶 (南川绿茶, Nánchuān lǜchá)**
![Наньчуань Люй Ча (南川绿茶, Nánchuān lǜchá)](images/Наньчуань%20Люй%20Ча%20(南川绿茶,%20Nánchuān%20Lǜchá).png)

![Наньчуань Люй Ча (南川绿茶, Nánchuān lǜchá)-1](images/Наньчуань%20Люй%20Ча%20(南川绿茶,%20Nánchuān%20Lǜchá)-1.png)

南川綠茶 (南川绿茶, Nánchuān lǜchá) —「南川區綠茶」— 來自重慶市直轄市南川區 (南川区, Nánchuān Qū) 的綠茶，生長於金佛山 (金佛山, Jīnfóshān，「金佛山」，2238公尺) 山坡上 — 聯合國教科文組織世界遺產 (自2014年起，屬於「中國南方喀斯特地貌」)、國家自然保護區及中國最大生物多樣性中心之一 (超過5000種植物、500多種脊椎動物)。南川茶葉傳統已有1700多年歷史 — 從巴人 (巴人, Bā rén) 向周朝廷進貢到五代《茶譜》(《茶谱》, Chá Pǔ, 五世紀) 記載：「涪州出三般茶，宾化最上」— 「涪州產三種茶；賓化縣 [=南川] 的最佳」。胺基酸含量 — 4.3% (是一般綠茶的兩倍)，硒 (Se) — 特級品≥0.3毫克/公斤，全年霧日260天。特級品 —「金佛玉翠」(金佛玉翠, Jīnfó Yùcuì，「金佛的翡翠綠」)，5000–10000元人民幣/公斤。四度榮獲「重慶市十大名茶」稱號 (重庆市十大名茶, 自2008年起)。截至2024年 — 134500畝茶園，產業總值 — 12.3億元人民幣。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。工藝 — 炒製後木炭烘乾。有三種形態：**緊直形** (紧直形, jǐn zhí xíng) — 用於特級「金佛玉翠」，**眉形** (眉形, méi xíng) — 用於一級茶，**大宗炒青** (大宗炒青, dàzōng chǎoqīng) — 用於二級茶及基礎產品。

* **類別：** 中國地理標誌產品 (全国农产品地理标志产品, 2012年)。四度榮獲「重慶市十大名茶」(重庆市十大名茶, 自2008年起)。日內瓦國際食品博覽會金獎 (日内瓦国际食品博览会金奖, 1987年 — 南川紅茶)。全國外銷紅茶第一名 (全国外销第一名, 1980年)。該區獲得「全國茶葉基地縣」地位 (全国茶叶基地县, 1970年)。

* **產地：** 中國重慶市直轄市南川區 (南川区)。29個鄉鎮。地理座標：東經106°54′–107°27′，北緯28°46′–29°30′。核心區：北部鄉鎮 — 大觀鎮 (大观镇, Dàguān Zhèn)、興隆鎮 (兴隆镇, Xīnglóng Zhèn)、木涼鎮 (木凉镇, Mùliáng Zhèn)、河圖鎮 (河图镇, Hétú Zhèn)、乾豐鎮 (乾丰镇, Qiánfēng Zhèn) — 海拔750–1200公尺，位於雲霧帶，土壤含硒 (Se)。

* **核心區地理座標：** 約北緯29°10′，東經107°10′。

**2. 歷史與文化意義：**

* **歷史：**

**商周 — 巴人 (商周, ~西元前十一至三世紀)。** 居住在長江上游流域的巴人 (巴人) 向周朝廷進貢茶葉作為貢品。南川位於巴人聚居區 — 世界最古老產茶區之一。

**陸羽與《茶經》(陆羽, 《茶经》, 八世紀)。** 「茶聖」陸羽 (陆羽, Lù Yǔ) 記載：「巴山峡川有两人合抱」— 「巴山峽谷中有需要兩人才能合抱的 [茶樹]」。推測指的是金佛山的古茶樹 — 至今在其山區仍保存約2000株野生茶樹 (*Camellia nanchuanica*)，其中最大的樹齡推測達2700年。

**《茶譜》(《茶谱》, 五世紀/五代)。** 毛文錫 (毛文锡, Máo Wénxī) 記載：「涪州出三般茶，宾化最上」— 「涪州產三種茶；賓化縣 [=南川] 的最佳」。這是該地區作為產茶區最古老的文字記錄之一。

**1920年代 — 出口。** 南川紅茶出口至俄國和日本，綠茶出口至歐洲和美國。該區已融入國際茶葉貿易。

**1970–1987年 — 巔峰期。** 1970年：南川獲得「全國茶葉基地縣」地位 (全国茶叶基地县)。1980年：南川紅茶在中國出口紅茶中排名第一。1984年：商業部第一名。1987年 — 國際認可的巔峰：**日內瓦國際食品博覽會金獎**紅茶。

**1990年 —「金佛玉翠」。** 創立高端產品線「金佛玉翠」(金佛玉翠, Jīnfó Yùcuì，「金佛的翡翠綠」) — 特級綠茶，採用緊直形、手工加工和含硒原料。

**2008–2024年。** 四度榮獲「重慶市十大名茶」稱號 (自2008年起)。地理標誌 (2012年)。截至2024年：134500畝茶園 (~8967公頃)，總價值 — 12.3億元人民幣。詩意別名：「金佛山珍」(Jīnfó Shānzhēn，「金佛山的珍寶」)。

* **名稱：**
	* 「南川」(南川, Nánchuān) —「南川」— 重慶市的行政區，位於金佛山南麓。
	* 「綠茶」(绿茶, Lǜchá) —「綠茶」— 直接指明類別。
	* 特級品：「金佛玉翠」(金佛玉翠) —「金佛的翡翠綠」— 直接引用金佛山和嫩芽的翡翠綠色。

* **文化意義：** 金佛山 —「金佛山」— 得名於夕陽西下時：夏秋傍晚，當陽光照射在多層岩石峭壁上時，山峰閃耀金光，「如金佛放射無數光芒」。聯合國教科文組織世界遺產 (自2014年起)，金佛山屬於「中國南方喀斯特地貌」系列，被認為是地球上最重要的生物多樣性中心之一。來自這座山的茶 —「金佛的聖飲」，融合了佛教象徵「茶禪一味」(茶禅一味，「茶與禪 — 一味」)。金佛山還生長著「五絕」(金佛山五绝)：野生茶樹、方竹 (方竹)、銀杉 (银杉, *Cathaya argyrophylla*)、銀杏和樹杜鵑。

**3. 植物學描述與原料：**

* **品種/栽培種：** 兩個主要品系：**福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbái Chá)** — 標準高產栽培種，和**巴渝特早 (巴渝特早, Bāyú Tèzǎo，「超早巴渝」)** — 為重慶條件培育的無條紋超早熟品系。另外 — 當地群體種 (群体种, qúntǐ zhǒng)，古茶樹的後代。生化特徵：胺基酸 — 4.3%，多酚類 — 30.35%。金佛山境內還保存約2000株野生 *Camellia nanchuanica* — 特有的大葉種，屬國家一級保護植物。

* **等級：**
	* **「金佛玉翠」(金佛玉翠, 特级)：** 一芽一葉初展 (一芽一叶初展)。採用「十大手法」(十大手法, shí dà shǒufǎ) 手工加工。「竹葉形」(竹叶形)。栗香帶「嫩香」。Se ≥0.3毫克/公斤。價格：**5000–10000元人民幣/公斤**。
	* **一級 (一级)：** 一芽二葉。眉形。濃厚持久口感。夏秋茶主體。
	* **二級 (二级)：** 一芽二至三葉。大宗炒青形。

**4. 風土條件與栽培特色：**

* **氣候：** 亞熱帶濕潤季風氣候 (亚热带湿润季风气候)。年平均氣溫 — 15.1°C (金佛山頂夏季僅約17°C，重慶約28°C)。年平均降雨量 — 1185毫米。關鍵指標 — **全年霧日260天** (雾日260天)。相對濕度 — **90%**。散射光為主。日夜溫差 — 超過10°C。這些條件創造了芽葉極慢生長和胺基酸密集累積的環境。

* **海拔：** 600–1200公尺。生產核心 — 750–1200公尺，位於「雲霧帶」(云雾带, yúnwù dài)。金佛山 — 大婁山脈 (大娄山) 東端，主峰風吹嶺 (风吹岭) — 2238公尺 (某些資料為2251公尺)。

* **土壤：** 弱酸性**紫色土** (紫色土, zǐsè tǔ) 和**黃泥土** (黄泥土, huángnítǔ)，pH 4.5–6.5。母岩 — 二疊紀石灰岩。有機質豐富。關鍵特色：**局部硒 (Se) 含量超過1 ppm** — 南川區被認為是重慶最大的硒土壤區 (超過一半領土)。硒 — 大多數茶區缺乏的必需微量元素。

* **水資源：** 河流發源於金佛山，流入長江和烏江。水質優良，部分水源含鍶 (Sr) 達礦泉水標準。

* **生態：** 森林覆蓋率 — 51%。核心區零工業污染。金佛山生物多樣性 (超過5000種植物，包括181種特有種，僅此地存在；500多種脊椎動物) 提供天然生物保護，免受害蟲侵害，特級品農藥需求降至零。

**5. 製作工藝：**

工藝結合特級品手工技法與大宗產品機械化。「金佛玉翠」採用「十大手法」(shí dà shǒufǎ，「十種手法：抓、抖、墊、壓、拉…」)，類似連雲港雲霧茶方法。現代生產整合AI溫度控制炒製。

* **攤放 (摊放, tānfàng)：** 長達8小時 — 明顯長於大多數綠茶。在高濕度 (90%) 下長時間攤放有助於蛋白質的預酶解和游離胺基酸累積。

* **殺青 (杀青, shāqīng)：** 130°C。採用「拋抖法」(抛抖, pāo dǒu) — 高頻翻動防止焦糊，確保均勻固定。

* **揉捻 (揉捻, róuniǎn)：** 輕壓，4–8分鐘。破壞細胞膜以利萃取，形成初步外形。

* **初烘 (初烘, chū hōng)：** 熱風。降低水分至約15–20%。

* **復烘 (复烘, fù hōng)：** 中間階段，平衡水分。

* **木炭烘焙 (木炭烘焙, mùtàn hōngbèi)：** 至含水量≤7%。木炭提供溫和紅外線加熱，增強栗香而無直火的「焦味」。

**「金佛玉翠」**：嚴禁農藥和化肥。AI系統即時監控溫度，根據濕度和葉溫感測器數據調整炒製模式。

**6. 感官特徵：**

* **乾茶外觀：** **特級品**：緊直、緊實茶條，綠色帶豐富茸毛，「如竹葉」(紧直绿润, jǐn zhí lǜ rùn)。**一級**：眉形 (眉形)，緊實。**大宗**：炒青綠色茶末。

* **乾茶香氣：** 栗香 (栗香, lìxiāng) — 主調，溫暖持久。特級品 — 額外的「嫩香」(嫩香, nèn xiāng)，清新略帶乳香。大宗品 — 清香 (清香, qīng xiāng)。

* **茶湯香氣：** 栗香溫暖，分兩階段展開：第一階段 — 明亮「堅果衝擊」，第二階段 — 溫和草本波浪帶輕微甜味。香氣持久性 — 特級品5–6泡。

* **滋味：** 鮮爽 (鲜爽, xiān shuǎng) — 4.3%胺基酸 (是平均值兩倍！) 提供明顯「鮮味」特徵。醇厚 (醇厚, chún hòu) — 高水浸出物含量 (≥35%)。回甘 — 持久，帶長久「乳甜」尾韻。苦澀味 — 因多酚類/胺基酸比例低而極少。

* **湯色：** 嫩綠明亮 (嫩绿明亮, nèn lǜ míng liàng)。透明，光照下帶極淡黃調。

* **葉底 (沖泡後茶葉)：** 黃綠色，均勻，葉片展開成「花朵」(芽叶成朵, yá yè chéng duǒ) — 細心採摘和加工的標誌。

**7. 化學成分：**

* **胺基酸 (氨基酸)：** **4.3%** — 是綠茶平均值 (~2–3%) 的兩倍。主要成分 — L-茶胺酸 (L-茶氨酸)。高含量歸因於三個因素：260天霧日 (散射光抑制茶胺酸光合轉化為兒茶素)、日夜溫差>10°C (低夜溫減緩胺基酸分解代謝) 和含硒土壤 (Se參與胺基酸代謝)。

* **多酚類 (茶多酚)：** 30.35% — 中等偏高。兒茶素特徵平衡：足夠「厚度」而無過度澀味。

* **水浸出物 (水浸出物)：** ≥35% — 茶湯高「濃度」，可多次沖泡。

* **硒 (Se, 硒)：** 特級品≥0.3毫克/公斤。來自金佛山含硒土壤 (局部Se >1 ppm)。硒 — 具抗氧化、免疫調節和甲狀腺保護作用的必需微量元素。

* **咖啡鹼 (咖啡碱)：** 約3.0–4.0% — 中等，綠茶典型值。

* **維生素：** C、B₁、B₂、E、K。維生素C因溫和加工而保存。

* **礦物質：** K、Mg、Mn、Zn、F、Se。鍶 (Sr) — 來自山區水源的微量水平。

**8. 保健功效：**

* **抗氧化作用：** 多酚類 (30.35%) + 硒 — 雙重抗氧化系統。硒是穀胱甘肽過氧化酶的輔因子 — 抗氧化防護的關鍵酶之一。

* **提神效果：** 咖啡鹼 + L-茶胺酸 — 溫和、持久的提神而無焦慮感。

* **免疫支持：** 胺基酸 (4.3%) + 硒。茶胺酸刺激γ-δ T淋巴細胞增殖。硒支持胸腺功能和抗體產生。

* **甲狀腺支持：** 硒 — 脫碘酶成分，將甲狀腺素 (T₄) 轉化為三碘甲狀腺原氨酸 (T₃) 的酶。定期飲用含硒茶對缺硒地區可能有益。

* **脂質代謝支持：** 兒茶素 (主要是EGCG) 促進脂肪氧化和膽固醇譜正常化。

* **認知功能：** L-茶胺酸改善注意力和工作記憶，刺激大腦α波。

* **牙齒保護：** 茶葉中的氟具有防蛀牙作用。

**9. 沖泡方法：**

* **水溫：** 一、二級茶80–85°C。特級「金佛玉翠」— 75°C (嫩料需溫和處理)。

* **茶量：** 3公克配150毫升 (1:50比例)。

* **茶具：**

	* **玻璃杯 (玻璃杯)：** 最適合「金佛玉翠」— 觀察「竹葉」展開。
	* **蓋碗 (盖碗)：** 用於一級和大宗茶 — 控制短時間沖泡。

* **過程 (玻璃杯，特級品)：**
	1. 用開水溫杯，倒掉。
	2. 注入7/10容量的水 (75°C)。
	3. 採用「上投法」(上投法, shàng tóu fǎ — 先水後茶) 投茶。
	4. 第一泡 — 30秒。
	5. 每次後續 — +15秒。
	6. 可沖4–5次。

* **過程 (蓋碗，一級茶)：**
	1. 溫蓋碗和茶海。
	2. 投入3–4公克茶。洗茶 — 5秒，倒掉。
	3. 第一泡 — 15–20秒。
	4. 每次後續 — +5秒。
	5. 可沖6–8次。

**10. 儲存：**

* **條件：** 密封包裝，冰箱0–5°C。最好 — 真空或氮氣包裝。
* **新茶：** 製作後「醒茶」1–2週以穩定香氣。
* **茶敵：** 濕氣、光線、熱度、異味。含硒茶對氧化特別敏感 — 必須嚴格密封儲存。
* **保質期：** 密封包裝在0–5°C下 — 最多18個月。

**11. 價格與仿冒：**

* **價格參考：**
	* **「金佛玉翠」(金佛玉翠, 特級)：** 5000–10000元人民幣/公斤 — 高端水平，可比擬頂級龍井。
	* **一級 (一级)：** 800–2000元人民幣/公斤。
	* **大宗 (二级)：** 平價，日常飲用。

* **如何避免仿冒：**
	* 購買帶有地理標誌「南川綠茶」標記的產品 — 官方保護標誌。
	* 正宗「金佛玉翠」— 緊直、緊實的「竹葉形」茶條，綠色帶茸毛。粗大或鬆散茶條 — 替代品跡象。
	* 檢查香氣：持久栗香帶「嫩香」。無香氣或「草腥」刺激 — 可疑。
	* 評估茶湯：嫩綠明亮透明。渾濁或深色茶湯 — 異常。
	* 可疑低價：「金佛玉翠」不可能遠低於5000元人民幣/公斤。

**12. 趣味知識：**

* **聯合國教科文組織山峰。** 金佛山 —「金佛山」— 聯合國教科文組織世界遺產 (喀斯特地貌，2014年)。超過5000種植物，181種特有種，包括「五絕」(五绝)：野生茶樹、方竹、銀杉、銀杏、樹杜鵑。南川綠茶 — 世界上少數生長在聯合國教科文組織名錄山峰上的茶之一。

* **陸羽與「兩人合抱」。** 「巴山峡川有两人合抱」—《茶經》中對巨大茶樹的描述，推測指金佛山，至今仍生長約2000株野生 *Camellia nanchuanica*，其中最大的樹齡達2700年。

* **「賓化最上」(五世紀)。** 毛文錫在五代《茶譜》(《茶谱》) 中記載南川 (=賓化) 為涪州三個茶區中最佳 — 該地區茶葉最古老的文字證據之一。

* **日內瓦金獎 (1987年)。** 南川紅茶榮獲日內瓦國際食品博覽會金獎 — 國際認可的巔峰。有趣的是，成名的是紅茶，而品牌後來「轉向」綠茶。

* **4.3%胺基酸 —「霧的秘密」。** 胺基酸含量是綠茶平均水平的兩倍 — 260天霧日、90%濕度和含硒土壤的結果。散射光抑制茶胺酸轉化為兒茶素，保持「鮮味」特徵。

* **AI控制 +「十大手法」。** 現代「金佛玉翠」生產結合傳統手工技法 (十大手法，承襲連雲港師傅) 與即時監控炒製溫度的人工智慧 — 中國西南地區最先進的茶葉生產線之一。

* **「金佛山珍」—「金佛山的珍寶」。** 南川綠茶的詩意別名，反映其來自世界級山峰的出身。

**13. 與其他綠茶比較：**

* **蒙頂甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 經典四川綠茶，同樣來自山區 (蒙頂山，1456公尺)。形狀 — 捲曲，香氣 — 栗香。南川綠茶的區別：硒含量更高，特級品形狀更直，260天霧日 (對比蒙頂約300個晴天)。

* **重慶永川秀芽 (重庆永川秀芽, Yǒngchuān Xiùyá)：** 另一著名重慶綠茶。永川的「秀芽」— 針形，清新特徵。南川綠茶更厚重，栗香更明顯，有硒優勢。

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 中國唯一大型蒸青綠茶 (來自鄰近湖北)。同樣是硒茶 (恩施 — 世界最大硒區之一)。區別 — 工藝：恩施玉露蒸青 (蒸青)，而南川炒青 (炒青) 並木炭烘乾，產生更溫暖的「栗香」特徵而非「草本清新」。

* **鳳岡富鋅富硒茶 (凤冈富锌富硒茶, Fènggāng Fùxīn Fùxī Chá)：** 貴州硒鋅茶，「硒」領域的直接競爭對手。兩者都是含硒山區綠茶，但南川綠茶歷史更豐富 (陸羽、《茶譜》) 和聯合國教科文組織風土。

**結論：**

南川綠茶 — 來自「金佛山」的茶，聯合國教科文組織名錄山峰：260天霧日、含硒土壤、4.3%胺基酸、木炭烘乾和人工智慧控制下的「十大手法」。從五代《茶譜》五世紀到1987年日內瓦金獎，從對俄紅茶到每公斤10000元人民幣的綠茶「金佛玉翠」— 一千五百年茶葉傳統，冠以現代別名「金佛山珍」—「金佛山的珍寶」。

這款茶 — 適合尋求有「厚度」綠茶的人：醇厚、鮮爽，明顯「鮮味」特徵和有機硒額外優勢。也適合那些重視手中茶杯是世界公認值得保護山峰上260天霧日結果的人。
