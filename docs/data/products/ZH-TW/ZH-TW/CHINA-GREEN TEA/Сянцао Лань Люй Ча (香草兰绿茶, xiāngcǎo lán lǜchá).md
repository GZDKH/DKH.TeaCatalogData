# 香草蘭綠茶 (香草兰绿茶, xiāngcǎo lán lǜchá)

**香草蘭綠茶 (香草兰绿茶, xiāngcǎo lán lǜchá)**

![Сянцао Лань Люй Ча (香草兰绿茶, xiāngcǎo lán lǜchá)-1](images/Сянцао%20Лань%20Люй%20Ча%20(香草兰绿茶,%20Xiāngcǎo%20Lán%20Lǜchá)-1.png)

香草蘭綠茶 (香草兰绿茶, xiāngcǎo lán lǜchá) — 世界上唯一以天然香草蘭 (*Vanilla planifolia*) 薰香的綠茶，被譽為「世界天然食品香料之王」(世界天然食品香料之王)。此產品誕生於海南島 (海南) — 中國唯一能夠種植香草蘭的熱帶地區，作為中國熱帶農業科學院香料飲料研究所 (中国热带农业科学院香料饮料研究所) 的科學項目。創製於1993年，採用革命性解決方案：取代傳統花卉「窨製」(窨制，以茉莉花或玫瑰花瓣薰香) 技術，使用發酵香草蘭豆莢 — 此產品需要120天發酵過程，含有超過250種芳香化合物。成果是一款具有巧克力香草香氣、「蘭花」甜味和抗焦慮效果的茶品，在中國乃至世界都無類似產品。2024年，其製作技藝被列入萬寧市非物質文化遺產名錄。

**1. 分類與產地：**

* **類型：** 添香型綠茶 (添香型绿茶, tiān xiāng xíng lǜchá)。基茶 — 海南大葉種綠茶。薰香劑 — 天然發酵香草蘭豆莢 (*Vanilla planifolia*)。薰香方法 — 「免花窨製」(免花窨制, miǎn huā yìnzhì) — 25–45 °C低溫吸附，有別於傳統茉莉花瓣「窨製」。

* **類別：** 科學研發新型茶類 (1993)。萬寧市級非物質文化遺產 (万宁市级非遗, 2024)。世界茶葉競賽獲獎者 (2022年世界紅茶大賽金獎 — 紅茶版本「雨林春芽」)。

* **產地：** 中國；海南省 (海南, Hǎinán)；萬寧市 (万宁市, Wànníng Shì)。生產核心 — 興隆熱帶植物園 (兴隆热带植物园, Xīnglóng Rèdài Zhíwùyuán)，位於香料飲料研究所內 (18°44′ 北緯，110°11′ 東經)。興隆 — 東南亞歸僑歷史聚居地，歸僑帶來熱帶作物，包括香草蘭。

* **地理座標：** 18°44′ 北緯，110°11′ 東經 (興隆，萬寧)。


**2. 歷史與文化意義：**

* **歷史：**

香草蘭綠茶是中國最「年輕」的名茶之一：僅有30多年歷史。1993年，海南香聖天然食品有限公司 (海南香圣天然食品有限公司) 與西南農業大學 (西南农业大学) 合作，首次在中國將香草蘭用於茶葉薰香。突破在於取代傳統花卉「窨製」(窨制) — 茶葉吸收新鮮茉莉花或玫瑰花瓣香氣的技術 — 開發出「免花窨製」(免花窨制) 方法：茶葉以發酵香草蘭豆莢薰香，而非花朵。這是根本性創新：香草蘭豆莢不是花朵，而是果實，其香氣 (香草醛和250+伴隨化合物) 不是在開花時形成，而是通過120天發酵過程產生。

同年1993年，產品通過海南省科學技術委員會科技鑑定，填補國內薰香茶類空白 (填补国内空白)。2000年代起，興隆熱帶植物園擴大生產規模，創建產品線：綠茶、紅茶、烏龍茶香草蘭系列。2022年，紅茶版本「雨林春芽」(雨林春芽) 榮獲世界紅茶大賽金獎。2024年，「興隆香草蘭茶製作技藝」(兴隆香草兰茶制作技艺) 被列入萬寧市非物質文化遺產名錄。

* **名稱：** 香草兰 (Xiāngcǎo Lán) — 「香草蘭」— *Vanilla planifolia* 的中文名稱，蘭科熱帶藤本植物，其發酵豆莢是天然香草醛的來源；綠茶 (Lǜchá) — 「綠茶」。完整含義：「香草蘭綠茶」。

* **文化意義：** 香草蘭綠茶是熱帶農業與茶業獨特交匯的產物，僅在海南 — 中國唯一的熱帶島嶼 — 才可能實現。興隆由1950年代從東南亞歸國的華僑建立，成為熱帶作物的「熔爐」：咖啡、可可、胡椒、香草蘭 — 正是這種文化環境孕育了香草蘭與茶結合的創意。興隆植物園是中國最受歡迎的熱帶植物園之一 (42公頃，超過3000種植物)。香料飲料研究所是中國唯一專門研究熱帶香料和飲料作物的科研機構；香草蘭研究自1983年開始。海南被納入「中國自由貿易港」(中国自由贸易港) 計劃，為香草蘭茶出口國際市場開闢前景。


**3. 植物學描述與原料：**

* **基茶 — 海南大葉種 (海南大叶种)：** *Camellia sinensis* var. *assamica*，海南本土品種 — 中國最南端的茶樹栽培品種之一 (北緯18°)。葉片大而肉厚，多酚類 ≥28 %，提供「厚重」基底，能夠「承載」濃郁香草蘭香氣而不失茶葉本色。在海拔400公尺以下的熱帶雨林條件下種植。部分批次另用奇蘭 (奇兰) 品種，帶來花果香調。

* **薰香劑 — 香草蘭 (*Vanilla planifolia*)：** 蘭科熱帶藤本植物，原產墨西哥，1960年代起在海南栽培 (香料飲料研究所)。豆莢長15–25公分，綠色採收後經120天發酵 (殺青 → 乾燥 → 40–50 °C發酵)，過程中β-葡萄糖苷酶將葡萄糖香草醛分解為香草醛和葡萄糖，形成特有巧克力香草香氣及250+伴隨芳香化合物。發酵豆莢切段用於薰香。

* **附加變化：** 存在「香蘭糯米香」(香兰糯米香) 版本，添加西雙版納野生「糯米香葉」(糯米香叶, nuòmǐ xiāng yè) — 創造「香草蘭 + 糯米」雙重香氣。

* **茶葉採收：** 全年採收 (熱帶氣候允許年採10次 — 遠超亞熱帶地區)。特級 — 單芽或一芽一葉，手工採摘。大宗 — 一芽二三葉，可機械採收。海南熱帶茶與大陸同類相比，多酚類含量較高，胺基酸含量較低，成為薰香的理想「厚重」基底。


**4. 風土與栽培特色：**

萬寧位於海南島東南沿海，處於熱帶地區 (北緯18°) — 這是中國唯一能夠露地種植香草蘭的地區。

* **氣候：** 熱帶季風氣候。年均溫 >25 °C；降雨量2200毫米/年；雲霧天數 >200天/年；森林覆蓋率86 %。香草蘭生長需要溫度 ≥25 °C 和遮蔭度 ≥50 %。

* **土壤：** 微酸性紅壤 (pH 5.0–6.5)，富含硒元素。有機質 ≥15 g/kg。

* **特色：** 香草蘭是世界上最勞動密集的農作物之一：每朵花需在開放後4–6小時內人工授粉 (自然界中香草蘭由墨西哥特有的無刺蜂授粉，海南沒有)；豆莢成熟需9個月；發酵需120天。這使天然香草蘭成為最昂貴香料之一 (250–560美元/公斤乾豆莢)。海南研究所開發「單元式熱空氣發酵生香法」(单元式热空气发酵生香法)，達到ISO品質標準，並首創國家香草蘭種苗和產品標準 (《香荚兰种苗》, 《香荚兰》)。海南香草蘭種植園產量超過世界平均水準，得益於獨特的人工遮蔭棚架技術。


**5. 製作工藝：**

香草蘭綠茶生產分兩階段：先製作基礎綠茶，再進行香草蘭薰香。

* **第一階段 — 基礎綠茶製作：** 標準海南工藝：採摘 → 萎凋 → 殺青 (炒製) → 揉捻 → 乾燥。基茶須「厚重」(多酚類 ≥28 %) — 以承受薰香而不失本色。

* **第二階段 — 香草蘭豆莢發酵 (120天)：** 綠色豆莢採收，熱水處理 (殺青，「燙漂」)，然後在40–50 °C發酵約120天。β-葡萄糖苷酶分解葡萄糖香草醛 → 香草醛 + 葡萄糖。發酵豆莢切段。

* **第三階段 — 「免花窨製」薰香 (免花窨制)：**

	* 基礎綠茶與切段發酵香草蘭豆莢混合 (特級品 ≥5 %)。
	* 混合物密封於鐵罐 (铁罐密封)。
	* 25–45 °C靜置5–30天 — 「低溫吸附」(低温吸附, dī wēn xīfù)。茶葉吸附香草蘭豆莢揮發性芳香化合物。溫度至關重要：>45 °C香草醛降解，<25 °C吸附過慢。
	* 取出豆莢，茶葉補充乾燥。

* **與茉莉花「窨製」的關鍵差異：** 茉莉花茶用新鮮花朵薰香，花朵凋謝時「釋放」香氣；香草蘭茶用發酵豆莢 (果實) 薰香，其香氣由120天酵素過程形成。這不是「花香」而是「果香」薰製 — 機制根本不同。


**6. 感官特徵：**

* **乾茶外形：** 緊結卷曲條索 (条索紧结卷曲)，色澤綠潤顯霜 (色泽绿润显霜)。有時可見香草蘭豆莢細片 (褐黑色斑點)。

* **乾茶香氣：** 主調為巧克力香草香 (巧克力香, qiǎokèlì xiāng) — *Vanilla planifolia* 標誌性香氣。背景為「蘭花香」(兰花香) 和草本清香 (草本清香)。香氣持久：冷杯保持 >5分鐘。

* **茶湯香氣：** 濃郁「糖果香」：香草蘭 + 巧克力 + 淡淡綠茶清香。任何其他綠茶都無此香型 — 最接近 (但相距甚遠) 的類比是茉莉花茶，但香型完全不同。冷卻後顯現「奶香」和「焦糖」調 — 香草醛與茶胺基酸在50 °C以下相互作用的結果。

* **滋味：** 醇厚 (醇厚, chúnhòu) — 高多酚基底 (≥28 %) 的結果。回甘 (回甘) 帶香草蘭甜味。順滑油潤 (润滑, rùnhuá)。澀味極輕 — 香草醛掩蓋單寧苦味。

* **湯色：** 黃綠明亮 (黄绿明亮)。

* **葉底 (沖泡後茶葉)：** 嫩勻黃綠 (嫩匀黄绿)；有時可見香草蘭豆莢片段。


**7. 化學成分：**

* **多酚類 (茶多酚)：** ≥28 % (基茶) — 高含量，提供「厚重」基底。

* **香草醛 (香兰素, xiānglán sù)：** 香草蘭豆莢主要芳香化合物 (3-甲氧基-4-羥基苯甲醛)。與250+伴隨芳香分子共同創造「天然香草蘭香韻」，合成香草醛無法複製。

* **胺基酸：** 基茶中存在，提供鮮味。

* **咖啡因：** 綠茶標準含量。

* **維生素：** 維生素C，B群維生素。

* **礦物質：** 硒 (來自海南土壤)，鉀，錳。


**8. 保健功效：**

* **抗焦慮效果：** 香草醛刺激γ-胺基丁酸 (GABA) 活性 — 主要抑制性神經傳導物質。據資料顯示，緩解焦慮效果比普通綠茶高40 %。

* **助消化：** 香草蘭芳香化合物增強胃蛋白酶活性。

* **抗氧化保護：** 多酚類 (≥28 %) 提供綠茶標準抗氧化活性。

* **提神效果：** 咖啡因 + L-茶胺酸 — 溫和提神。

* **助眠：** 香草蘭精油對中樞神經系統有溫和鎮靜效果 — 下午飲用時 (無咖啡因形式 — 冷泡茶)。

* **重要提醒：** 上述功效基於一般資料，非醫療建議。不建議空腹飲用。日用量不超過600毫升 (避免咖啡因過量)。


**9. 沖泡方法：**

* **水溫：** 80 °C (特級品 — 75 °C)。勿用沸水 — 破壞香草醛並產生苦味。

* **茶量：** 3公克配150毫升 (比例1:50)。

* **茶具：** 玻璃杯 (觀察湯色) 或白瓷蓋碗 (聚集香氣)。

* **沖泡：** 80 °C水沖泡，靜置3分鐘。可沖泡3次。

* **冷泡法 (冷泡法, lěng pào fǎ)：** 4公克茶 + 500毫升冷水 → 冰箱，4小時。冷泡香草蘭茶是中國最佳冷泡茶之一：香草蘭巧克力香氣更加鮮明，咖啡因萃取最少。


**10. 儲存方法：**

* **溫度：** 室溫。**勿冷藏！** (綠茶中的例外) — 冰箱凝結水破壞香草蘭香氣，食品異味瞬間被茶葉吸附。這是與大多數需要冷藏的綠茶的根本差異。
* **容器：** 密封，避光。
* **光照：** 完全隔離 — 香草醛對光敏感。
* **保質期：** 18個月內。


**11. 價格與仿品：**

香草蘭綠茶是平價茶品 (相比昂貴茉莉花茶的優勢之一)。特級 — 80–100元人民幣/100公克 (禮盒鐵罐裝)；一級 — 30–50元人民幣/100公克；大眾級 (袋裝) — 20–30元人民幣/66公克。

* **避免仿品方法：**

	* 香氣 — 天然香草蘭巧克力香，持久 (冷杯 >5分鐘)。合成香草醛 — 刺鼻，「單調」，快速消失。
	* 產地 — 興隆，萬寧，海南。品牌：「兴隆」(興隆)，「香圣」(香聖)。
	* 茶中香草蘭豆莢片段 — 天然薰香標誌。


**12. 趣味知識：**

* **世界唯一香草蘭茶：** 無論中國、日本、印度還是斯裡蘭卡，都沒有香草蘭綠茶的類似產品。這直接源於海南獨特地理位置：中國唯一能露地種植香草蘭的熱帶島嶼。

* **「免花窨製」：** 傳統中國「窨製」(窨制) 是花卉薰香：茉莉、玫瑰、桂花。香草蘭茶是首個也是唯一的「果實」窨製：用發酵豆莢而非花瓣薰香。這需要完全重新設計技術：取代6–8小時花香「爆發」— 5–30天緩慢低溫吸附。

* **120天發酵：** 香草蘭豆莢是世界上最「勞動密集」的芳香產品之一。每朵花人工授粉 (清晨，開放後4–6小時內)；豆莢成熟9個月；發酵120天。一批香草蘭茶使用的豆莢，其「生命史」始於一年多前。

* **興隆 — 歸僑村：** 興隆建於1950年代，安置從東南亞 (馬來西亞、印尼、越南) 歸國的華僑。他們帶來熱帶農業文化：咖啡、可可、胡椒、香草蘭。正是這個文化「熔爐」孕育了香草蘭與茶結合的創意。

* **冷泡理想選擇：** 香草蘭巧克力香氣在冷泡時比熱泡更加鮮明，同時咖啡因萃取最少 (約為熱泡的30 %)。這使冷泡香草蘭茶成為熱帶海南的理想夏日飲品 — 更廣泛地說，適合冷茶文化遠比大陸中國發達的整個東南亞地區。

* **檳榔林中的香草蘭：** 海南香草蘭種植的前景方法之一 — 與檳榔 (槟榔, bīnglang) 間作 (间作, jiānzuò)：香草蘭是需要支撐和遮蔭的藤本；檳榔是提供兩者的高大喬木。因此，兩種海南「特產」— 檳榔果和香草蘭 — 在同一種植園生長，節約土地和資源。


**13. 與其他薰香綠茶比較：**

* **茉莉花茶 (茉莉花茶, Mòlì Huāchá)：** 經典茉莉綠茶。薰香 — 新鮮茉莉花瓣 (花卉「窨製」，6–8小時 × 3–7輪)。香氣 — 花香，「清白」；滋味 — 清鮮，「輕盈」。香草蘭茶 — 「深沉」，巧克力香草香；滋味 — 更「厚重」和「糖果感」。技術根本不同：花卉 vs 果實薰香。

* **桂花綠茶 (桂花绿茶, Guìhuā Lǜchá)：** 桂花綠茶。香氣 — 「杏蜜香」。比茉莉花茶更「溫暖」，但無香草蘭茶的巧克力深度。

* **伯爵茶 (Earl Grey)：** 西方對應品 — 佛手柑油薰香紅茶。兩者都是「異國香型薰香茶」，但機制完全不同：伯爵茶用精油 (萃取物)；香草蘭茶用整個發酵豆莢 (天然吸附)。結果 — 香草蘭茶香氣更複雜「深層」，因為豆莢含250+芳香化合物 vs 佛手柑單一主導芳樟醇。此外，香草蘭茶是綠茶，伯爵茶是紅茶，提供根本不同的「背景」風味：清香植物感 vs 麥芽單寧感。


**結語：**

香草蘭綠茶是茶中異數：它誕生於熱帶植物園而非高山；用果實而非花朵薰香；冷飲勝過熱飲；雖是綠茶卻不宜冷藏。這一切都源於兩位「王者」的相遇：海南大葉種茶 (多酚類28 %) 和墨西哥香草蘭 (250+芳香分子，120天發酵)，在中國唯一熱帶島嶼上的東南亞歸僑村落中實現。80 °C沖泡 — 或者更佳，冷水浸泡冰箱4小時：巧克力香草蘭香氣的綻放，是世界上任何其他綠茶都無法企及的。
