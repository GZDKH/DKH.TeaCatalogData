# 黎平香茶 (黎平香茶, Lípíng xiāngchá)

**黎平香茶 (黎平香茶, Lípíng xiāngchá)**
![Липин Сян Ча (黎平香茶, Lípíng xiāngchá)](images/Липин%20Сян%20Ча%20(黎平香茶,%20Lípíng%20Xiāngchá).png)

![Липин Сян Ча (黎平香茶, Lípíng xiāngchá)-1](images/Липин%20Сян%20Ча%20(黎平香茶,%20Lípíng%20Xiāngchá)-1.png)

黎平香茶 (黎平香茶, Lípíng xiāngchá) —「黎平芳香茶」— 產自黎平縣 (黎平县, Lípíng Xiàn) 的大宗綠茶，該縣位於貴州省黔東南苗族侗族自治州 (黔东南苗族侗族自治州, Qiándōngnán Miáozú Dòngzú Zìzhìzhōu)。黎平是貴州省面積最大的縣 (4441 km²)，是侗族 (侗族, Dòngzú) 文化的中心 — 中國「會唱歌」的民族之一，其多聲部歌唱「侗族大歌」(侗族大歌, Dòngzú Dàgē) 於2009年被聯合國教科文組織列入人類非物質文化遺產名錄。該縣的茶葉歷史可追溯至西漢時期：揚雄 (扬雄, Yáng Xióng) 在《方言》(《方言》) 中提到，西南地區稱茶為「蔎」(蔎) — 至今這個詞仍在侗語中保留。在陸羽 (陆羽, Lù Yǔ) 的《茶經》(《茶经》) 中，該地區被納入「黔中茶區」(黔中茶区，「中貴州茶區」)，而在明代永樂年間 (永乐, 1403–1424)，當地官員向皇宮進貢茶葉。黎平香茶的主要競爭優勢是水浸出物 ≥42% (比國家標準GB高5–9%)，確保了「香濃味醇」(xiāng nóng wèi chún，「香氣濃郁，滋味醇和」) 的品質特色。截至2024年 — 269,800畝 (~18,000公頃) 茶園，15,200噸產量，總產值 — 16.2億元人民幣。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，不發酵。捲曲形 (卷曲形, juǎnqūxíng)。生產完全機械化。定位 — 大宗市場 (大宗绿茶, dàzōng lǜchá)：原料 — 一芽二三葉 (比高檔茶葉更成熟)，同時保持極高的水浸出物指標 (≥42%)。

* **類別：** 中華人民共和國地理標誌保護產品 (国家地理标志保护产品)。「中國名茶之鄉」(中国名茶之乡, Zhōngguó Míngchá zhī Xiāng, 2010)。標準：T/LPCX 01-2020《黎平香茶》(《黎平香茶》)。該縣還擁有「全國重點產茶縣」(全国重点产茶县) 和「中國西部最美茶鄉」(中国西部最美茶乡) 等稱號。截至2024年 — 全縣茶葉生產中超過40%屬於「香茶」類別。

* **產地：** 中國，貴州省 (贵州省, Guìzhōu Shěng)，黔東南苗族侗族自治州 (黔东南州)，黎平縣 (黎平县)。該省面積最大的縣。

* **地理座標：** 約北緯26°14′，東經109°08′ (縣中心)。

**2. 歷史與文化意義：**

* **名稱由來。** 「黎平」(黎平) — 縣的歷史名稱，可追溯至永樂年間 (永乐, 1403)，當時設立「黎平府」(黎平府)。「香茶」(香茶) — 字面意思「芳香茶」— 強調栗香花香特色作為主要商業優勢。

* **西漢與《茶經》。** 貴州東南部的茶文化從西漢時期 (西汉, 西元前206年 — 西元9年) 就有記載。在揚雄的語言學著作《方言》中記錄了「蔎」(蔎) 一詞 — 西南地區茶的方言稱謂，至今仍在侗語中保留。陸羽的《茶經》將黎平地區納入「黔中茶區」：「茶之出黔中，生思州、播州、費州、夷州……往往得之，其味極佳」— 「黔中之茶，生於思州、播州、費州、夷州……常常可得，其味極佳」。

* **明代記載。** 在《弘治貴州圖經新志》(弘治《贵州图经新志》, 明弘治, 1488–1505) 中記載：「黎平府，洞茶葉大而味美」— 「黎平府：侗族茶 — 葉大味美」。永樂年間 (永乐, 1413)，黎平府設立後，當地長官司 (长官司, zhǎngguānsī) 將茶葉運送到京城作為朝貢。

* **清代記載。** 《道光黎平府志》(道光《黎平府志》, 1845) 詳細描述了茶葉採摘的季節儀式，按時期分類 (社前 — 最早，穀雨前 — 中等，立夏前 — 最後的優質採摘) 以及侗族和苗族的茶俗。清代詩人顧忳純 (顾忳纯) 寫了《西山茶》(《西山茶》) 詩，序言中描述了黎平「西山」的茶園。

* **現代時期。** 1976年建成「桂花臺萬畝茶場」(桂花台万亩茶场，「桂花臺萬畝茶場」) — 當時該州最大的茶場。2010年該縣獲得「中國名茶之鄉」稱號。2020年實施行業標準T/LPCX 01-2020《黎平香茶》。截至2024年，黎平茶園達269,800畝 (~18,000公頃)，年產量 — 15,200噸，總產值 — 16.2億元人民幣。縣內有300多家茶企業，36家標準化清潔生產工廠和119家茶葉合作社。

* **產業規模。** 黎平縣獲得十多項國家和行業稱號：「中國名茶之鄉」(2010)、「全國重點產茶縣」、「中國十大生態產茶縣」、「茶產業發展示範縣」、「中國西部最美茶鄉」、「中國茶業百強縣」。黎平茶產品在國內外展覽中獲得約百項獎項。縣茶產業總產值接近20億元人民幣，茶葉成為當地民眾增收的支柱產業。

* **文化意義。** 黎平是侗族文化的中心。多聲部歌唱「侗族大歌」(侗族大歌)，據傳起源於春秋時期 (春秋, 西元前770–476年)，2009年被聯合國教科文組織列入人類非物質文化遺產名錄。位於縣內的肇興侗寨 (肇兴侗寨) 是中國最受歡迎的侗族村寨之一。在德順鄉 (德顺乡) 太平山 (太平山) 上保存著野生古茶樹群 — 育種的基因庫。茶是侗族待客不可缺少的部分：油茶「油茶」(油茶, yóuchá)，用米花、花生和香料製作，是侗族和苗族家庭的日常儀式。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要 — **龍井43號** (龙井43号, Lóngjǐng 43 hào)，種植比例超過50%。超早熟無性系品種，由中國農業科學院茶葉研究所 (中国农业科学院茶叶研究所) 培育；適合形成捲曲形。屬於 *Camellia sinensis* var. *sinensis*。其他栽培種：**福鼎大白茶** (福鼎大白茶, Fúdǐng Dàbái Chá) 和**當地群體種** (群体种, qúntǐzhǒng) — 大葉和中葉型，葉厚，果膠含量豐富。多酚類 — ≥14.5%。

* **採摘：** 春茶採摘 — 主要 (3–4月)。夏茶和秋茶採摘 — 輔助。採摘完全機械化。

* **採摘標準和等級 (T/LPCX 01-2020)：**
	* **特級 (特级, tèjí)：** 一芽一葉。捲曲形，栗香。300元人民幣起/500克。
	* **一級 (一级, yījí)：** 一芽二葉。150–300元人民幣/500克。
	* **二級 (二级, èrjí)：** 一芽二葉。80–150元人民幣/500克。

**4. 風土與栽培特點：**

* **氣候：** 該縣位於「北緯26°黃金產茶帶」(北纬26°黄金产茶带, Běiwěi 26° Huángjīn Chánchádài) — 中國亞熱帶茶葉栽培的最佳緯度帶。年平均氣溫 — 15–18°C。降雨量 — 1300–1500毫米。雲霧天數 — 全年超過180天。散射光 — 超過70%。公式：「高海拔、低緯度、寡日照」(gāo hǎibá, dī wěidù, guǎ rìzhào — 「高海拔、低緯度、少日照」) — 正是「寡日照」(直射陽光不足) 減緩了胺基酸向多酚類的轉化，確保了滋味的醇和和創紀錄的水浸出物指標。

* **海拔：** 800公尺以上。茶園佔據山坡，垂直落差300–1200公尺。

* **土壤：** 微酸性黃壤 (微酸性黄壤, wēi suānxìng huángrǎng; pH 4.5–6.5)。有機質含量 ≥2%。土壤富含硒 (Se) 和鋅 (Zn) — 黔東南地球化學特徵。森林覆蓋率 — 76%。

* **核心產區：** 高屯鎮 (高屯镇, Gāotún Zhèn) — 「桂花臺萬畝茶場」(10,000畝)，建於1976年，成為縣工業化茶葉生產的基礎；德順鄉 (德顺乡) — 太平山野生古樹區，育種遺傳材料來源；九潮鎮 (九潮镇) — 高產區，生產最機械化。

* **生態：** 縣境位於雲貴高原向湘桂丘陵過渡地帶。地形 — 主要為中山，有深谷河流。水資源豐富：縣境位於南江 (南江) 及其支流流域。生態環境良好：無重工業，大氣和水體污染極少。黎平縣是貴州省少數幾個獲得有機生產認證的茶葉產區之一。

**5. 製作工藝：**

黎平香茶生產完全機械化，採用「全程機械化清潔生產」(全程机械化清洁生产, quánchéng jīxièhuà qīngjié shēngchǎn) 體系，確保品質穩定和較長保存期。關鍵原則 — 「快速鎖翠」(快速锁翠, kuàisù suǒcuì，「快速鎖定翠綠色澤」)。

* **採摘 (采摘, cǎizhāi)：** 機械化。標準 — 一芽二三葉。

* **攤青 (摊青, tān qīng)：** 鮮葉攤放在通風室內的竹篩上。持續時間 — 4–6小時。蒸發多餘水分，開始形成香氣前體物質。

* **殺青 (杀青, shāqīng)：** 滾筒式，溫度約300°C — 「快速鎖翠」方法。快速鈍化多酚氧化酶，固定翠綠色澤和栗香。

* **揉捻 (揉捻, róuniǎn)：** 機械式。形成捲曲形 (卷曲形) — 決定產品外觀的關鍵工藝操作。茶葉捲成緊密螺旋，類似蝸牛殼。

* **乾燥 (干燥, gānzào)：** 兩段式：初烘 (毛火, máohuǒ) 120°C → 足乾 (足火, zúhuǒ) 80°C。殘留水分 — ≤6.5%。低最終水分確保保存期達24個月。

* 禁止添加劑、香料和色素。

**6. 感官特徵：**

* **乾茶外形：** 捲曲形 (卷曲形)。特級 — 緊細捲曲「如螺」(紧细如螺, jǐnxì rú luó)。色澤 — 墨綠 (墨绿, mòlǜ)，均勻。

* **乾茶香氣：** 栗香 (栗香, lìxiāng) — 主導。清香 (清香) — 特級。花香 (花香) — 春茶明顯。

* **茶湯香氣：** 濃郁栗香帶花香底韻。持久 — 冷杯中保持10分鐘以上。公式「香濃」— 「香氣濃郁」。

* **滋味：** 醇厚 (醇厚, chúnhòu) — 水浸出物 ≥42% 的直接結果：茶湯「厚重」，舌上有重量感。鮮爽 (鲜爽) — 胺基酸含量較高所致。明顯回甘 (回甘)。醇和 — 多酚類14–14.5% (低於綠茶平均值)，苦澀味極少。公式「味醇」— 「滋味醇和」。

* **湯色：** 特級 — 嫩綠，明亮透澈。二級 — 黃綠。

* **葉底 (沖泡後茶葉)：** 嫩黃綠，均勻。芽葉完整，葉質柔軟。

**7. 化學成分：**

* **水浸出物：** ≥42% — 關鍵指標，比國家標準GB/T 14456.1高5–9%。比較：GB綠茶最低標準 — 34%，優質綠茶典型值 — 36–38%。高水浸出物確保茶湯「厚重」和滋味持久。

* **多酚類：** 14–14.5% — 適中水平，確保醇和無明顯澀味。主要成分 — 兒茶素 (EGCG, EGC, ECG)。

* **胺基酸：** 較高 — 「寡日照」(缺乏直射陽光) 公式的結果。海拔>800公尺和雲霧天數>180天減緩光合作用，阻止胺基酸向多酚類轉化。L-茶胺酸佔主導，確保特有的醇甜。

* **咖啡因：** 2–4% (綠茶典型值)。與較高胺基酸結合，提供溫和持久的提神效果。

* **微量元素：** 硒 (Se) 和鋅 (Zn) — 來自黔東南土壤。硒 — 抗氧化劑，參與細胞膜保護；鋅 — 免疫調節。

* **維生素：** 維生素C (300°C快速固定保存)，B群維生素，維生素E。

**8. 保健功效：**

* **抗氧化作用。** 兒茶素 (EGCG) 中和自由基。黔東南土壤中的硒進一步增強抗氧化潛力。

* **提神效果。** 咖啡因與L-茶胺酸平衡，提供溫和清醒而無焦慮。水浸出物 ≥42% 意味著每杯茶中生物活性成分濃度高。

* **微量元素支持。** 硒 — 保護細胞免受氧化壓力，支持甲狀腺功能。鋅 — 免疫調節，組織癒合。

* **消化支持。** 適中多酚類 (14–14.5%) 溫和刺激胃液分泌而不刺激黏膜。

* **心血管支持。** 兒茶素有助降低低密度脂蛋白，正常化血壓。

* **認知支持。** L-茶胺酸刺激大腦α波，改善專注力和學習能力。

* **牙齒保護。** 綠茶中的氟化物通過形成氟磷灰石強化琺瑯質。

* **代謝支持。** 咖啡因與兒茶素結合刺激產熱並促進脂肪氧化，規律飲用可支持健康新陳代謝。

* **抗菌作用。** 兒茶素在口腔中表現抑菌活性，降低蛀牙風險並消除口臭。

**9. 沖泡方法：**

* **水溫：** 標準等級85–90°C。特級 — 80°C。

* **茶量：** 3克/100毫升水 (比例1:25–1:38 — 比大多數綠茶更濃，因定位為「厚重」飲品)。

* **茶具：** 瓷蓋碗 (盖碗) 100–150毫升或玻璃杯 (玻璃杯)。侗族傳統油茶 (油茶) — 鐵鍋。

* **過程：**
	1. 用開水溫杯，倒掉水。
	2. 投入3克茶葉。
	3. 注入適溫水。
	4. 第一泡 — 30秒。
	5. 每次增加10–15秒。
	6. 可沖泡3次。水浸出物 ≥42% 確保即使第三泡仍有厚重茶湯。

* **侗族油茶 (油茶, yóuchá)：** 傳統方法 — 在茶油中炒茶葉，加入陰米 (阴米)、花生、薑、鹽。注入開水，過濾。用小杯盛裝，配脆米。這種方法 — 不是傳統意義的沖泡，而是製作茶「湯」，以完全不同的方式展現黎平香茶的「厚重」特質。

* **注意：** 由於殘留水分低 (≤6.5%) 和機械化乾燥，黎平香茶比許多綠茶更耐運輸和保存 — 即使不冷藏，第一年品質也不會明顯下降 (雖然仍建議冷藏)。

**10. 保存：**

* **條件：** 密封包裝 (真空袋或密蓋罐)。冷藏保存0–5°C。
* **保存期：** 正確保存可達24個月 — 比大多數綠茶 (典型12–18個月) 長得多。原因 — 殘留水分低 (≤6.5%)，通過兩段式機械化乾燥達成。
* **茶葉天敵：** 潮濕、直射陽光、異味、高溫。

**11. 價格與假冒：**

* **價格範圍：** 特級 (特级) — 300元人民幣起/500克。一級 — 150–300元人民幣。二級 — 80–150元人民幣。黎平香茶的主要優勢 — 高水浸出物指標下的可負擔性：這是大宗市場茶，「價格 — 茶湯厚重度」比例是中國最佳之一。

* **如何避免假冒：**
	* 購買帶有地理標誌標識「黎平香茶」的產品。
	* 檢查外形：真正的黎平香茶具有捲曲形 (卷曲形)，緊密捲曲的墨綠色螺旋。假冒品 — 鬆散、不均勻。
	* 評估茶湯：厚重「有體感」的茶湯 — 水浸出物 ≥42% 的直接結果。稀薄茶湯表明原料品質低劣。
	* 檢查香氣：栗香，持久。缺乏栗香 — 固定工藝問題的標誌。
	* 注意價格：二級茶30–50元人民幣/500克令人懷疑。

**12. 有趣事實：**

* **侗族茶與「蔎」字。** 侗語中至今仍使用古老的「蔎」(蔎) 字稱呼茶 — 正是揚雄在《方言》中記錄的兩千多年前的詞。明代記載簡潔地描述：「洞茶葉大而味美」— 「侗族茶 — 葉大味美」。而侗族歌唱 — 「侗族大歌」(侗族大歌) — 被聯合國教科文組織列入遺產名錄。

* **269,800畝 — 茶葉「帝國」。** 黎平茶園面積是貴州最大之一。比較：超過著名安吉白茶全部種植面積 (170,000畝)，與武夷山種植面積相當。

* **水浸出物比國家標準高+5–9%。** 可溶性物質含量 ≥42%，國家標準GB最低門檻34% — 「寡日照」公式的結果：海拔>800公尺，雲霧天數>180天，散射光>70% 減緩胺基酸向多酚類轉化，保持茶湯「厚重」。

* **太平山野生古樹。** 德順鄉太平山 (太平山) 保存著野生茶樹群 — 用於當地群體種選育的基因庫。

* **出口中亞。** 中國綠茶的非典型方向：黎平香茶出口到傳統偏好紅茶的國家。此外，茶葉還銷往北京和上海 — 大都市內銷市場。

* **油茶「油茶」(油茶)。** 侗族茶製作傳統包括在茶油中炒茶葉，加入陰米、花生、薑和鹽 — 製成營養飲品湯，每日飲用。油茶不僅是食物，更是待客儀式：在侗族家中拒絕油茶是嚴重的禮儀違反。

* **「會唱歌」的茶。** 黎平縣是多聲部歌唱「侗族大歌」(侗族大歌) 的搖籃，聯合國教科文組織認定為人類非物質遺產。僅三龍村 (三龙) 就向國家級合唱團輸送了100多人，16人參加國際演出。茶與歌 — 侗族文化的兩大象徵，在縣日常生活中密不可分。

**13. 與其他綠茶比較：**

* **都勻毛尖 (都匀毛尖, Dūyún Máojiān)：** 貴州高檔綠茶，中國「十大名茶」之一。細嫩，價格高。黎平香茶 — 大宗替代品：不如其精緻，但價格實惠得多，茶湯更厚重。

* **湄潭翠芽 (湄潭翠芽, Méitán Cuìyá)：** 另一個貴州品牌，扁平形 (龍井型)。黎平香茶以捲曲形、更厚重茶湯和大宗市場定位區別。

* **碧螺春 (碧螺春, Bìluóchūn)：** 捲曲形，果香花香，茶湯清淡。水浸出物 — 36–38%。黎平香茶 — 明顯更厚重 (≥42%) 且價格便宜數倍。

* **珠茶 (珠茶, Zhūchá，「火藥」)：** 大宗出口綠茶，球形。定位相近 (大宗市場)，但黎平香茶在香氣特色和水浸出物方面更優。

* **雷山銀球茶 (雷山银球茶, Léishān Yínqiú Chá)：** 鄰縣雷山獨特「銀球」，手工成型。高檔，昂貴。黎平香茶 — 同一黔東南州的大宗類似品，但採用機械化工藝。

**結論：**

黎平香茶 — 貴州的「平民」茶：不追求龍井的精緻或碧螺春的雅致，而以可負擔性、穩定品質和比國家標準高5–9%的水浸出物取勝。在這個「大宗」公式背後 — 是侗族的土地，其多聲部歌唱在聯合國教科文組織殿堂中響起，太平山的野生茶樹，明代「貢茶」和269,800畝茶園，年產15,200噸茶葉。栗香、厚重茶湯和誠實價格 — 「香濃味醇」公式，從陸羽《茶經》到T/LPCX 01-2020標準經受時間考驗。這是為那些重視茶杯內容而非行銷的人準備的茶 — 也是為那些知道最誠實的茶往往來自山歌多聲部響起的地方，而「茶」字的發音與兩千年前一樣的人準備的。
