# 康禾茶 (康禾茶, Kānghé chá)

**康禾茶 (康禾茶, Kānghé chá)**
![Канхэ Ча (康禾茶, Kānghé chá)](images/Канхэ%20Ча%20(康禾茶,%20Kānghé%20Chá).png)

![Канхэ Ча (康禾茶, Kānghé chá)-1](images/Канхэ%20Ча%20(康禾茶,%20Kānghé%20Chá)-1.png)

康禾茶 (康禾茶, Kānghé chá) — 歷史悠久的綠茶，產自廣東省河源市東源縣康禾鎮 (康禾镇)，是廣東省唯一經證實連續向四位皇帝進貢的茶葉：康熙、雍正、乾隆和嘉慶 (康熙—雍正—乾隆—嘉庆, 1661–1820)。茶葉歷史可追溯至南宋時期 (南宋, XII–XIII 世紀)，擁有超過 800 年的連續傳統。康禾茶是客家 (客家, kèjiā) 民族的茶葉，其工藝體現了客家茶文化哲學的核心原則：「火功不足不可飲」（「如果火候不足就不能飲用」）。其標誌性特徵是**「炒米香」** (炒米香, chǎomǐ xiāng)，這種香氣源於獨特的「高火萎凋」(高火萎凋, gāohuǒ wěidiāo) 工序 — 廣東省非物質文化遺產 (2012)。2023 年獲得中華人民共和國地理標誌認證。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，形式上為不發酵茶，但工藝包含「做青」(做青, zuòqīng，「形成綠色」— 搖晃以刺激部分發酵) 步驟，賦予茶葉烏龍茶特徵：「綠葉镶边」(lǜyè xiānɡbiān，「綠葉紅邊」) — 葉緣呈現紅色斑點。這使康禾茶成為「邊界」茶 — 介於綠茶和烏龍茶之間。

* **類別：** 中華人民共和國地理標誌證明商標產品 (国家地理标志证明商标, 2023)。廣東省非物質文化遺產 (广东省非物质文化遗产, 2012)。「廣東十大好綠茶」(广东十大好绿茶, 2020)。廣東省唯一擁有四位皇帝「貢茶」地位的茶葉。

* **產地：** 中國廣東省 (广东省)，河源市 (河源市, Héyuán Shì)，東源縣 (东源县, Dōngyuán Xiàn)，康禾鎮 (康禾镇, Kānghé Zhèn)。康禾位於省級自然保護區 (康禾省级自然保护区)，在廣東省東北部。

* **地理座標：** 約北緯 24°00′，東經 115°15′。

**2. 歷史與文化意義：**

* **歷史：**

	**南宋 (XII–XIII 世紀) — 起源。** 根據《河源縣志》(《河源县志》)，康禾在南宋時期就已種植茶葉。史書記載：「康禾諸約，居人生計，多半賴此」(康禾诸约，居人生计，多半赖此) — 「康禾居民的生計一半以上依賴於[茶葉]」。

	**清朝 — 四位皇帝的貢茶 (1661–1820)。** 康熙年間 (康熙, 1661–1722)，仙坑村 (仙坑村) 商人葉庚慶 (叶庚庆) 向皇帝進獻康禾茶。康熙皇帝認可此茶，從此康禾茶成為「貢茶」— 向朝廷進貢的茶葉，連續四個朝代：康熙 → 雍正 → 乾隆 → 嘉慶。這使康禾茶成為廣東省唯一擁有如此長期連續「貢茶」地位的茶葉。

	**現代復興。** 2012 年，生產工藝（包括「高火萎凋」）被列入廣東省非物質文化遺產名錄。2020 年入選「廣東十大好綠茶」。2023 年獲得中華人民共和國地理標誌認證。

* **名稱：**
	* 「康禾」(康禾) — 鎮名。「康」(康) — 「健康/繁榮」，「禾」(禾) — 「穀物/稻米」。地名營造「健康豐收之地」的意象。
	* 「茶」(茶) — 茶。

* **文化意義：** 康禾茶是少數體現客家 (客家) 茶文化的茶葉之一。客家是漢族的一個族群，歷史上從中國北方遷移到南方，為廣東帶來了自己的茶文化哲學，其核心原則是：「火功不足不可飲」— 「如果[加工的]火候不足就不能飲用」。這一原則決定了康禾茶的「高火」特性。仙坑村 (仙坑村) — 「貢茶的搖籃」— 保存了客家「圍龍屋」(围龙屋，「龍形圍屋」) 建築。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要品種：
	* **客家中小葉群體種 (客家中小叶群体种)** — 當地灌木品種，橢圓形葉片，細小鋸齒，絨毛豐富。春茶：多酚類 ≥18.3%，氨基酸含量高。
	* **引進品種：** 青心烏龍 (青心乌龙，「純心烏龍」— 蘭花香)，金萱 (金萱，「金萱」— 奶香)，臺灣軟枝烏龍 (台湾软枝乌龙 — 適合球形揉捻)。

* **採摘：** 三季：**明前茶** (明前茶，清明前 — 最高等級)，**雨前茶** (雨前茶，穀雨前 — 中等級)，**秋茶** (秋茶，秋季 — 大宗)。

* **標準：**
	* **貢芽 (贡芽，「貢茶芽」)：** 單芽或一芽一葉，長度 ≤2.5 公分。清朝「貢茶」的複製品。價格 — 每 500 公克 600 元人民幣起。
	* **特級 (特级)：** 一芽一葉。
	* **一級 (一级)：** 一芽二葉。
	* **大宗茶 (大宗茶)：** 成熟葉片，供茶館使用。

**4. 風土條件與栽培特色：**

* **氣候：** 北緯 24–25°。康禾省級自然保護區。年平均氣溫 — 17–19°C。降雨量 — 1600–1900 毫米/年。雲霧天數 — 超過 180 天。散射光 — 70%。日夜溫差 — 超過 10°C。

* **海拔：** 600–1000 公尺。核心區 — 仙坑村 (仙坑村) 和五指山茶園 (五指山茶园，1000 公尺)。

* **土壤：** 酸性黃壤 (酸性黄壤)，土層厚度 — 超過 1 公尺。富含有機質。禁用化學肥料和農藥。

* **生態：** 森林覆蓋率 — **78.48%**（廣東省茶區中最高之一）。自然病蟲害防護 — 由於保護區的生物多樣性，效果比化學防治高 60%。**茶行間種櫻花樹 (樱花套种)** — 獨特的「茶 + 櫻花」模式。

**5. 製作工藝：**

康禾茶採用客家獨創工藝，關鍵步驟是**「高火萎凋」(高火萎凋, gāohuǒ wěidiāo)** — 廣東省非物質文化遺產。整個過程**完全手工** (全程手工禁用机械)，葉形保持率 ≥95%。

* **採摘 (采摘)：** 明前茶，一芽二葉。
* **高火萎凋 (高火萎凋)：** 關鍵步驟。葉片在**高溫**下萎凋（不同於標準綠茶的溫和萎凋），失水 8–12%。正是這一步驟形成了標誌性的**「炒米香」** (炒米香) — 甜美、溫暖，類似烘烤糯米的香氣。原則：「火功不足不可飲」— 「火候不足就不能飲用」。
* **「做青」(做青 — zuòqīng)：** 搖青 (摇青) 以刺激葉緣部分發酵。這一步驟是綠茶中的**烏龍茶特徵**，創造「綠葉镶边」（「綠葉紅邊」）。
* **「殺青」(杀青)：** 鍋溫 260°C。
* **揉捻 (揉捻)：** 輕揉，成條狀。
* **炭焙 (炭焙)：** 至含水量 ≤6.5%。
* **「輝鍋」— 最終鍋內加熱 (辉锅)：** 定形和「提香」。

**6. 感官特徵：**

* **外觀：** **傳統型 (卷曲形)：** 緊密螺旋，灰白色 (色泽灰白)。**創新型 (扁形)：** 扁平、挺直、綠色。茶底 — 黃綠色，柔軟，**葉緣有紅點** (叶缘红点，「绿叶镶边」)。
* **香氣：** **「炒米香」** (炒米香) — 主要香調，客家工藝的標誌性特徵。引進品種帶有蘭花香 (兰花香，來自青心烏龍)。陳放後呈現蜜香 (蜜香)。
* **滋味：** 濃醇 (浓醇)。回甘持久，具有特色的「高火甜韻」(高火甜韵, gāohuǒ tián yùn)。鮮爽 (鲜爽)。「滑潤」(滑润) — 果膠含量高。
* **湯色：** 黃綠色，明亮清澈。
* **葉底：** 黃綠軟亮 (黄绿软亮)。葉緣紅點明顯 (叶缘红点显) — 「做青」的標記。

**7. 化學成分：**

* **多酚類：** ≥18.3% — 適中水平，確保柔和口感。
* **氨基酸：** 含量高 — 得益於 180+ 天雲霧天氣和保護區生態。
* **氟：** 含量較高 — 保護牙齒琺瑯質。
* **果膠 (叶胶质)：** 含量高 — 提供「滑潤」口感。
* **咖啡因、維生素 (C、B)、礦物質 (K、Mg、F)。**

**8. 保健功效：**

* **抗氧化作用：** 多酚類 ≥18.3%。
* **護齒作用：** 氟含量較高。
* **提神效果：** 咖啡因 + L-茶氨酸。
* **助消化：** 客家傳統上在豐盛餐後飲用康禾茶。

**9. 沖泡方法：**

* **水溫：** 80–85°C。「貢芽」(贡芽) — 75°C。
* **用量：** 3 公克配 150 毫升 (1:50)。
* **茶具：** 宜興紫砂壺 (紫砂壶，濃縮香氣) 或白瓷蓋碗 (觀察湯色和葉底「紅邊」)。
* **過程：** 「溫潤泡」(温润泡，洗茶) — 快速倒出。第一泡 — 30 秒。每次增加 20 秒。可沖泡 3–4 次。

**10. 儲存：**

* 密封包裝，冷藏 0–5°C。開封後 — 3 個月內飲用。

**11. 價格與仿冒：**

* **貢芽 (贡芽)：** 每 500 公克 600 元人民幣起。**特級：** 300–500 元。**一級：** 200–300 元。**大宗** — 供茶館使用。
* 購買時認準「康禾茶」地理標誌標識 (2023)。關鍵測試：「炒米香」— 炒米香氣 — 無法仿製。葉緣紅點 (绿叶镶边) — 正宗「做青」工藝的標記。

**12. 趣味知識：**

* **四位皇帝的貢茶。** 康禾茶是廣東省唯一有文獻證實連續向四位皇帝進貢的茶葉：康熙、雍正、乾隆和嘉慶 (1661–1820)。總計約 160 年的連續供應。

* **「火候不足就不能飲用」。** 客家原則「火功不足不可飲」不是詩意表達，而是技術指導：「高火萎凋」(高火萎凋) 是不可缺少的步驟，沒有這一步驟茶葉就不算完成。這與江南綠茶的「溫和」方法根本不同。

* **帶「紅邊」的綠茶。** 「做青」步驟（搖晃促進部分發酵）是烏龍茶而非綠茶的工藝特徵。結果是「綠葉镶边」（綠葉紅邊）— 視覺標記，使康禾茶成為綠茶和烏龍茶之間的「邊界」茶。

* **茶 + 櫻花。** 康禾茶行間種櫻花樹 (樱花套种) — 獨特的農業生態模式，減少病蟲害並在春季創造驚人的視覺效果：綠色茶行中的粉色花瓣。

* **78.48% 森林。** 康禾的森林覆蓋率是廣東省茶區中最高的之一。該鎮位於省級自然保護區內。

* **800+ 年。** 南宋史書《河源縣志》記載康禾在 XII–XIII 世紀就有茶葉種植 — 800+ 年的連續傳統。

**13. 與其他廣東綠茶的比較：**

* **古勞茶 (古劳茶)：** 同樣來自廣東，歷史悠久，採用超高溫炒制 (300°C)。康禾是「高火」，但更溫和。兩者都處於綠茶和烏龍茶的交界。

* **仁化銀毫 (仁化银毫)：** 廣東產。多毫，挺直。康禾 — 捲曲，具有「火」的特性和蘭花底調。

* **英德綠茶 (英德绿茶)：** 大葉種 (*var. assamica*)。康禾 — 小葉種，客家「火」傳統，皇室貢茶。

**結論：**

康禾茶是客家靈魂與廣東土地相遇的茶葉：「高火萎凋」、「炒米香」、「綠葉紅邊」和「火功不足不可飲」原則 — 這些都是從北方遷移到南方的民族遺產，帶來了自己的「火」茶哲學。從康熙到嘉慶，連續四位皇帝都將此茶作為「貢茶」飲用，原因顯而易見：濃郁的「高火」甜韻、滑潤的口感和蘭花底調創造了與中國其他綠茶都不同的風味特徵。這是一款適合欣賞「邊界」茶的人品飲的茶 — 介於綠茶和烏龍茶之間，介於北方的嚴謹和南方的柔和之間。
