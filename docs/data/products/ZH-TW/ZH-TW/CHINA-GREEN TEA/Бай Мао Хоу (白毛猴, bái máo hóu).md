# 白毛猴 (白毛猴, bái máo hóu)

**白毛猴 (白毛猴, bái máo hóu)**

![Бай Мао Хоу (白毛猴, bái máo hóu)](images/Бай%20Мао%20Хоу%20(白毛猴,%20Bái%20Máo%20Hóu).png)

白毛猴 (白毛猴, bái máo hóu) —「白毛猴」— 歷史名稱，統稱來自福建省的**兩種根本不同的茶**，僅以「白毛猴」這個共同名稱相連：

**(1) 政和白毛猴** (政和白毛猴) — 綠茶 (绿茶) 來自政和縣 (政和县)，福建北部。由茶商**範昌义** (范昌义) 於**1910年**創制。工藝描述為「介于紅茶綠茶之间」—「介於紅茶與綠茶之間」：長時間（16–18小時）萎凋 (萎凋) 伴隨部分酵素氧化，然後殺青、揉捻和乾燥。當地暱稱—「白綠」(白綠，「白綠茶」)。最初出口至廣東、香港和澳門。原料—著名品種**政和大白茶** (政和大白茶) — 與製作白茶「政和白毫銀針」相同的品種。政和自北宋時期即以生產銀針聞名；正是這個縣給了宋徽宗政和年間（1111–1118）的名稱，據傳說，皇帝對當地茶葉如此讚賞，以至於以縣名命名整個統治時期。

**(2) 安溪白毛猴** (安溪白毛猴) — 烏龍茶 (乌龙茶) 來自安溪，福建南部。來自石竹岩山 (石竹岩, 700+ 公尺) 的原創品種，由兄弟**謝駕和謝冰** (谢驾、谢冰) 於清末開發。加工—完整的烏龍茶工藝，包含發酵和「窨制」(窨制，額外的「薰製」)。以「安溪藥茶」(「安溪藥茶」) 聞名，出口至臺灣、日本、東南亞和歐洲。目前幾乎**失傳**—石竹岩山上僅剩7–8棵老樹。

本文專門介紹**政和白毛猴**（綠茶/「白綠茶」），因為正是這種茶在收藏中呈現且更容易獲得。安溪版本在「比較」章節中提及。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，形式上為不發酵茶，然而具有**長時間萎凋**（16–18小時）—「制法介于紅茶綠茶之间」（「工藝介於紅茶與綠茶之間」）。當地暱稱—「白綠」(白綠，「白綠茶」)。重點—在**「保毫」(保毫, bǎo háo) 和「做形」(做形, zuò xíng)**。

* **類別：** 歷史名茶 (历史名茶)。創制於1910年。在《中國茶經》(《中国茶经》) 中分類為綠茶，儘管工藝包含白茶元素（長時間萎凋）甚至輕烏龍（萎凋時部分氧化）。

* **產地：** 中國，福建省 (福建省)，政和縣 (政和县, Zhènghé Xiàn)，福建北部。政和—福建「三大茶縣」之一，與福鼎、建甌並列。座標：約北緯27°22′，東經118°51′。

**2. 歷史與文化意義：**

* **歷史：**

政和—福建北部茶葉歷史最悠久的縣。銀針 (银针) 生產自北宋時期（960–1127）即已開始。據傳說，宋徽宗 (宋徽宗, 1100–1126)，著名的藝術贊助者和《大觀茶論》(《大观茶论》，「大觀茶論」) 的作者，對這個縣的白茶如此讚賞，以至於在1115年將其統治時期重新命名為「政和」(政和，「政治和諧」)—以縣名命名。這是中國歷史上統治者以茶命名統治時期的極罕見案例之一。

在這個千年茶葉基礎上，**1910年**當地茶商**範昌义** (范昌义, Fàn Chāngyì) 創制了新型茶—白毛猴。範昌义開發了獨特工藝，結合白茶的長時間萎凋（16–18小時）與綠茶的「殺青」。結果—「介於紅茶與綠茶之間」的茶，具有毛茸茸、蓬鬆的「蜷縮猴子」形狀和溫和、「香清味醇」的口感。白毛猴迅速征服了廣東、香港和澳門市場—這些地區重視毛茸茸的「毫」茶和溫和口感。

* **名稱：** 白毛 (白毛) —「白毫」（芽和嫩葉上豐富的白色茸毛）；猴 (猴) —「猴子」。彎曲、捲曲的茶葉，密布白毫，像蜷縮的小猴子。當地暱稱—「白綠」（「白綠茶」）—反映工藝的中間特性。特別優秀的批次獲得後綴「王」(王，「王」)。

* **文化意義：** 白毛猴—「混合」茶：誕生於給整個宋朝統治時期命名的縣，在白茶和綠茶傳統的交匯點。其工藝—「介於紅茶與綠茶之間」—反映了福建實驗精神，孕育了中國茶的全部六大類別。

**3. 植物學描述與原料：**

* **品種/栽培種：** **政和大白茶** (政和大白茶, Zhènghé Dàbáichá) —「政和大白茶」。福建最珍貴的栽培種之一：大型、肉質芽葉，密布白毫。灌木型 (灌木型)，中葉類 (中叶类)，中芽種 (中芽种)。正是用這個品種生產著名白茶「政和白毫銀針」(政和白毫银针) 和「政和白牡丹」(政和白牡丹)。

* **採摘：** 清明後 (清明后)，春茶。標準—一芽二三葉 (一芽二三叶)，密布白毫。只採摘乾燥天氣下未受損的嫩芽。嫩芽必須大型、「肥壮」(肥壯，「肥壯結實」)。

* **原料要求：** 絕對不能損傷毫毛—「保毫」(保毫，「保護毫毛」)—主要原則。嫩芽鬆散地放在竹籃中，避免擠壓。

**4. 風土條件與栽培特點：**

* **生長海拔：** 600–900公尺（福建北部山區）。政和縣位於武夷山脈 (武夷山脉) 支脈，福建與浙江交界—省內最「山地」的茶縣之一。

* **氣候：** 亞熱帶季風氣候。年平均溫度約18°C。年降雨量1600–1800毫米。高濕度，頻繁霧氣—一年超過180天。顯著的日夜溫差（>8°C）—刺激胺基酸和芳香物質的積累。雲霧的散射光—確保政和大白茶嫩芽豐富毫毛的關鍵因素之一。

* **土壤：** 紅壤、黃壤 (红壤, 黄壤)，酸性（pH 4.5–5.5）。排水良好，深厚（>60公分），富含有機質和礦物質。母岩—花崗岩和砂岩，提供口感的礦物質「骨架」。典型的福建北部山地土壤—與武夷岩茶生長的土壤相同，儘管海拔和微氣候不同。

* **生態：** 政和—森林覆蓋率高（>70%）的山縣。竹林、松樹和闊葉林在茶園上方形成「綠色天篷」。水資源—閩江流域的溪流和河流。山區無工業污染。

**5. 製作工藝：**

白毛猴工藝—獨特的「混合」，結合白茶、綠茶和部分紅茶元素。主要原則：**「保毫」(保護毫毛)**和**「做形」(塑造「猴子」)**。「內質重萎凋適度」(內在品質依賴適度萎凋)—關鍵公式。

* **萎凋 (萎凋, wěidiāo)：** **16–18小時**—比標準綠茶（2–4小時）長得多。葉子薄層攤放在竹盤或「水筛」(水篩，「水篩」) 上，在陰涼或通風室內。16–18小時內發生部分酵素氧化多酚：失水25–30%，葉子變暗至深綠色，葉柄變皺，毫毛顯現「如銀」(白毫显露, 毫毛如银)，質地「柔軟如棉」(叶质柔软如棉)，葉柄彎曲但不斷裂 (梗折不脆断)。過度萎凋→發紅；萎凋不足→過度「綠色」苦澀。

* **殺青 (杀青, shāqīng)：** 在鍋中140–150°C。先「揚炒」(扬炒, yáng chǎo)，然後「悶炒」(闷炒, mèn chǎo)。直到顏色轉為「青」(青，藍綠色)，葉柄黃綠色，香氣「清香」(清香)，無草腥味。

* **揉捻做形 (揉捻/做形)：** 獨特階段—結合揉捻和塑造「猴子」。師傅交替**「揉搓」**(揉搓, róucuō) 和**「茶團旋轉」**(茶团旋转, chá tuán xuánzhuǎn)：葉子捲成圓形、彎曲形狀，像「蜷縮的小猴子」。操作「輕巧溫和，注意保毫」(操作轻巧，注意保毫)。5–8分鐘。濕潤的「茶團」(湿茶团)—每個50–500克。

* **初烘 (初烘)：** 100–110°C。每個「焙籠」(焙笼，竹製烘乾籃) 放10–15個小「茶團」。至50–60%乾燥度（茶葉不黏手指）。

* **復焙整形 (复焙整形)：** 50–60°C。同時繼續塑形：茶葉「完善」至完整的「小猴」(小猴) 形狀。毫毛不應脫落（溫度不太高），顏色不應「悶」成黃褐色（溫度不太低）。至80–90%乾燥度—降至40–50°C。烘至完全乾燥。

* **揀剔 (拣剔)：** 去除葉柄和有缺陷的茶葉。包裝。

**6. 感官特徵：**

* **乾茶外觀：** 大型、彎曲茶葉，「蜷縮的猴子」(犹如毛猴静伏，「如猴子靜臥安息」)。密布銀白色毫毛。形狀蓬鬆，「肥壮卷曲」(肥壯捲曲，「肥壯捲曲」)。顏色從銀綠色到深綠色帶銀色「霜」。

* **香氣：** 「毫香」(毫香, háo xiāng)—清新、甜美。清香 (清香)。比典型綠茶複雜—帶有「奶香」和「蜜香」層次，由長時間萎凋產生。

* **滋味：** 溫和，「醇和微甘」(醇和微甘, chúnhé wēigān，「和諧溫和，略帶甜味」)。最小澀味。明顯的「回甘」(回甘，回甜)。茶體中等，「絲滑」。

* **湯色：** 「清綠泛黄」(清綠泛黃, qīnglǜ fànhuáng，「清綠帶黃色調」)。某些資料中為「橙黄」(橙黃, chénghúang，「琥珀黃」)—取決於萎凋程度。

* **葉底：** 嫩綠、完整、勻淨、無雜 (嫩绿、完整、匀净、无杂)。嫩芽保持形狀—「保毫」和精心加工的指標。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 中等—長時間萎凋（16–18小時）部分氧化兒茶素，降低澀味。多酚含量白毛猴更接近白茶（15–20%）而非標準綠茶（20–30%）。

* **胺基酸 (氨基酸)：** 較高—政和大白茶，福建最富胺基酸的栽培種之一，加上山地風土（霧氣>180天，日夜溫差>8°C）=豐富的胺基酸譜。L-茶胺酸占主導—負責特有的「絲滑」甜味和「毫香」(毫香，毫毛香氣)。

* **EGCG：** 存在，但以更「溫和」形式—16–18小時萎凋時的部分氧化將部分EGCG轉化為茶黃素和茶紅素，創造出純綠茶所沒有的「雙重」抗氧化譜。

* **咖啡因：** 中等。

* **維生素：** C、B群。

* **礦物質：** 氟、鉀、鎂、鋅。

**8. 保健功效：**

* **抗氧化保護：** 多酚類 + EGCG + 維生素C。長時間萎凋部分將兒茶素轉化為茶黃素，提供「雙重」抗氧化譜（綠茶 + 部分氧化）。

* **溫和提神效果：** 咖啡因 + L-茶胺酸—提神而不緊張。特別溫和—由於多酚類較低和胺基酸較高。

* **消化支持：** 中等兒茶素刺激胃腸蠕動，不刺激黏膜（與更「強烈」的綠茶不同）。

* **認知功能：** L-茶胺酸刺激大腦α波活動。

* **重要：** 所列功效基於一般資料，不構成醫療建議。

**9. 沖泡方法：**

* **水溫：** 75–80°C。嫩「猴子」對沸水敏感—毫毛「燙傷」會使茶湯混濁。

* **茶量：** 3–5克/150–200毫升。

* **茶具：** 玻璃杯或瓷蓋碗—觀察「猴子」在水中展開和銀色毫毛。

* **過程：**
	1. 溫杯。
	2. 投茶。
	3. 洗茶—注水，3秒後倒出。白毛猴建議洗茶—「展開」蓬鬆捲曲的「猴子」並去除第一泡的細毫。
	4. 第一泡—1–2分鐘（75–80°C）。
	5. 3–5泡，每次增加15–20秒。
	6. 觀察葉底：如果嫩芽完整、嫩綠、保持毫毛—茶葉真實且沖泡正確。破碎的嫩芽和脫落的毫毛—過度乾燥或假冒的跡象。

* **特點：** 由於長時間萎凋（16–18小時），白毛猴比標準綠茶更「耐溫」：即使在85°C也不苦澀，反而展現「蜜香」層次。然而為了最大「毫香」(毫香，毫毛香氣)，75–80°C更佳。

**10. 儲存：**

* **溫度：** 0–5°C，密封。白毛猴對濕度特別敏感—茶葉上豐富的毫毛如「海綿」，比大多數綠茶更快吸收大氣濕度和異味。
* **容器：** 密封、不透明。鋁箔 + 真空包裝—理想。開封前—恢復至室溫，避免毫毛凝露。
* **光線：** 完全隔離—毫毛和葉綠素對光敏感。
* **保存期：** 0–5°C下6–8個月。最好在當年採摘年份內飲用。不建議長期存放：白毛猴以新鮮度和「毫香」著稱，隨時間會消散。

**11. 價格與假冒：**

白毛猴—稀有且昂貴的茶：有限產區（政和）、手工勞動、要求高的原料（政和大白茶）、複雜工藝（16–18小時萎凋 + 手工塑造「猴子」）。

* **如何避免假冒：**
	* 形狀—蓬鬆、彎曲的「猴子」，密布白毫。不是扁平、不是「針」。
	* 香氣—「毫香 + 清香」(毫香 + 清香，毫毛香 + 清香)。無「草腥」或「焦糊」味。
	* 茶湯—「清綠泛黄」(清綠泛黃，清綠帶黃色調)。混濁茶湯—過度乾燥的跡象。
	* 葉底—完整、嫩綠的嫩芽，保持毫毛。

**12. 趣聞軼事：**

* **給統治時期命名的縣。** 政和—中國唯一一個宋朝皇帝宋徽宗以其命名整個統治時期（1111–1118）的縣：他對當地白茶如此印象深刻，以至於將時代重新命名為「政和」(政和)。白毛猴—這個千年傳統的延續者。

* **「介於紅茶與綠茶之間」(介于红茶绿茶之间)。** 獨特工藝：16–18小時萎凋（如白茶）+ 殺青和揉捻（如綠茶）。結果—形式上是綠茶，但口感更接近白茶。

* **範昌义 (范昌义, 1910)。** 白毛猴創制者—政和茶商，結合白茶和綠茶傳統為廣東和香港出口市場。

* **兩個白毛猴。** 政和（綠茶/白綠茶，政和大白茶）和安溪（烏龍茶，來自石竹岩山）。安溪白毛猴，由謝駕和謝冰兄弟在清末創制，以「安溪藥茶」(「安溪藥茶」) 聞名，經臺灣出口至日本、東南亞和歐洲。謝冰兄弟致富後，1878年山東發生乾旱時，組織運糧隊響應皇帝救災號召—因此獲得清廷「奉政大夫」(奉政大夫) 頭銜和佩戴「花翎」(花翎，孔雀羽毛) 的權利。目前安溪白毛猴幾乎**失傳**—石竹岩山上僅剩7–8棵「碗口粗」(「碗口粗」，樹幹如碗口粗) 的老樹。

* **「靜臥的猴子」(毛猴静伏)。** 茶葉形狀—名片：蓬鬆、捲曲、覆蓋銀色毫毛，確實像蜷縮成團的小猴子。

* **政和大白茶—萬能品種。** 用同一品種製作：白毫銀針、白牡丹、紅茶「政和工夫」(政和工夫红茶)，和白毛猴。一個栽培種四類茶—記錄。

**13. 與其他政和和福建茶的比較：**

* **政和白毫銀針 (政和白毫银针)：** 來自同一品種（政和大白茶）的白茶。僅芽頭。萎凋更長（24–48小時），無殺青。白毛猴—有殺青和揉捻，芽 + 2–3葉，「綠茶」特性。

* **安溪白毛猴 (安溪白毛猴)：** 來自石竹岩山的烏龍茶。完整烏龍發酵 + 「窨制」。「安溪藥茶」。幾乎失傳。完全不同類型—烏龍，非綠茶。

* **太平猴魁 (太平猴魁)：** 安徽。也是「猴」茶，但扁平、大葉、無毫毛。白毛猴—捲曲、蓬鬆、豐富毫毛。兩者都是綠茶，但來自不同省份，工藝不同。

* **政和白牡丹 (政和白牡丹)：** 來自政和大白茶的白茶。芽 + 1–2葉。無殺青。白毛猴—有殺青和塑造「猴子」。兩者都「毛茸茸」，但不同類別。

**結論：**

白毛猴—具有千年血統的茶：給宋朝皇帝統治時期命名的政和縣，在1910年孕育了這個白茶和綠茶的「混合體」。其公式—16–18小時萎凋 + 殺青 + 塑造「蜷縮猴子」—創造出形式上是綠茶，但精神上是「白綠」(「白綠茶」) 的茶：毫香、溫和口感「介於」類別之間，杯中銀色「猴子」。用同一政和大白茶品種製作四種茶—從白毫銀針到紅茶「工夫」；白毛猴—其中最「過渡」的，白茶與綠茶之間的橋樑茶，誕生於福建實驗精神。
