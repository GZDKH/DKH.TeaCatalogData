# 碧螺春 (碧螺春, bìluóchūn)

**碧螺春 (碧螺春, bìluóchūn)**
![Билочунь (碧螺春, bìluóchūn)](images/Билочунь%20(碧螺春,%20Bìluóchūn).png)

![Билочунь (碧螺春, bìluóchūn)-1](images/Билочунь%20(碧螺春,%20Bìluóchūn)-1.png)

碧螺春 (碧螺春, bìluóchūn) 是中國最偉大的綠茶之一，名列經典的「中國十大名茶」(中国十大名茶) 之中。以其「四絕」(四绝) 而聞名：形美——緊捲的螺旋狀，宛如蝸牛殼；色雅——銀綠色中透著翠綠；香濃——濃郁的花果香調；味醇——清新、鮮爽且帶甘甜。因其無與倫比的細膩與優雅，這款茶贏得了詩意的美稱「茶中仙子」(茶中仙子, chá zhōng xiānzǐ)。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵茶）。屬於炒青綠茶 (炒青绿茶, chǎoqīng lǜchá) 中的螺旋捲曲型。

* **等級：** 名列「中國十大名茶」(中国十大名茶, Zhōngguó shí dà míngchá)。地理標誌產品 (地理标志产品)。2011年洞庭碧螺春手工製作技藝被列入國家級非物質文化遺產名錄，2022年作為「中國傳統製茶技藝及其相關習俗」的組成部分被列入聯合國教科文組織人類非物質文化遺產代表作名錄。生產受國家標準GB/T 18957-2008規範。

* **產地：** 中國江蘇省 (江苏, Jiāngsū) 蘇州市 (苏州, Sūzhōu) 吳中區 (吴中, Wúzhōng)。茶葉產於太湖 (太湖, Tàihú) 中的洞庭山 (洞庭山, Dòngtíng Shān) 島嶼——確切地說，是東山 (洞庭东山, Dòngtíng Dōngshān) 和西山 (洞庭西山, Dòngtíng Xīshān) 山區。生產核心集中在東山鎮 (东山镇, Dōngshān Zhèn) 和金庭鎮 (金庭镇, Jīntíng Zhèn，原西山鎮)。只有來自這一地區的茶葉才有權稱為「洞庭碧螺春」(洞庭碧螺春)。

* **地理座標：** 約北緯31°05′，東經120°22′。

**2. 歷史與文化意義：**

* **歷史：** 洞庭山的茶葉種植傳統可追溯至隋唐時期（6-10世紀），當時當地茶葉已以「洞庭茶」(洞庭茶) 和「嚇煞人香」(吓煞人香, xiàshārénxiāng——字面意思為「香得嚇死人」) 之名聞名。後一個名稱生動地證明了這款茶異常濃郁的香氣如何令當時的人們驚嘆不已。

	茶葉歷史的轉折點與清朝康熙皇帝 (康熙, Kāngxī) 有關。康熙三十八年（1699年），康熙巡視太湖。蘇州知府宋犖 (宋荦, Sòng Luò) 向皇帝進獻當地茶葉。康熙被其品質所征服，但認為民間名稱「嚇煞人香」不雅，遂賜名「碧螺春」(碧螺春)——結合了三個意象：「碧綠的茶湯顏色」(碧)、「螺旋形狀如蝸牛」(螺) 和「春天採摘時節」(春)。從此，碧螺春成為「貢茶」(贡茶, gòngchá) 之一——向皇室進獻的茶葉。

	近代以來，茶葉聲譽持續上升：1915年碧螺春在巴拿馬太平洋萬國博覽會上獲得金獎；1959年入選官方「中國十大名茶」名單。2022年其製作技藝被列入聯合國教科文組織非物質文化遺產名錄。

* **名稱含義：**

	* 「碧」(碧) ——「翠綠」、「碧玉綠」：指茶湯和乾茶葉的顏色。
	* 「螺」(螺) ——「蝸牛」、「螺旋」：描述捲曲茶葉的特徵形狀，宛如微型貝殼。
	* 「春」(春) ——「春天」：強調茶葉專門在早春採摘。

* **文化意義：** 碧螺春是蘇州和整個江南地區 (江南) 的名片，體現了「南方園林」精緻文化的精髓。茶葉與太湖——中國偉大湖泊之一——及其島嶼景觀密不可分，在那裡茶園與果樹數百年來相伴而生。碧螺春經常作為國禮贈送，而洞庭山的春茶採摘是該地區重要的文化活動。

**3. 植物學描述與原料：**

* **品種/栽培種：** 正宗洞庭碧螺春使用當地原生品種——**洞庭山群體種 (洞庭山群体种, Dòngtíngshān Qúntǐzhǒng)**——*Camellia sinensis* var. *sinensis* 有性繁殖品種。這個小葉種具有很高的「持嫩性」(持嫩性, chí nèn xìng)：芽葉長時間保持柔嫩。化學成分特點是多酚與胺基酸比例 (酚氨比, fēn'ān bǐ) 和諧：茶胺基酸含量超過2.5%，確保了明顯的鮮爽和甘甜。正是群體種造就了碧螺春著名的花果香。

* **採摘：** 採摘始於早春。最珍貴的是「明前茶」(明前茶, Míngqián chá)——從春分 (春分, ~3月20日) 到清明 (清明, Qīngmíng, ~4月5日) 期間採摘的茶葉。由完整芽頭或最嫩的「一芽一葉初展」組成，具有最大的細膩度和香氣濃度。從清明到穀雨 (谷雨, Gǔyǔ, ~4月20日) 採摘的茶葉——「雨前茶」(雨前茶, Yǔqián chá)——口感更加厚實濃郁，價格也更為親民。穀雨後採摘的茶葉不再被視為經典碧螺春，歸類為普通炒青綠茶 (炒青)。

* **採摘標準：** 特級——一芽一葉初展 (一芽一叶初展, yī yá yī yè chū zhǎn)。一級——一芽一葉。二級——一芽二葉初展。生產500公克特級 (特级) 乾茶需要6-7萬個芽頭——這是世界上最費工的茶葉之一。

* **原料要求：** 極其嚴格。芽葉必須大小一致、完整、無機械損傷。採摘在清晨進行。新鮮採摘的原料立即進行揀剔 (拣剔, jiǎn tī)：去除有缺陷的葉片、粗糙部分、莖稈、「魚葉」(鱼叶)。加工必須在當天開始。

**4. 風土條件與栽培特色：**

* **氣候：** 洞庭山地區位於亞熱帶季風氣候區，受太湖影響顯著。年平均氣溫15.5-16.5°C，年降雨量1200-1500毫米。湖泊創造了獨特的微氣候：雲霧覆蓋茶園達80%的時間，提供柔和的散射光 (散射光)，被認為是茶樹的理想光照條件。春季晝夜溫差顯著，有利於嫩芽中胺基酸和芳香化合物的積累。

* **海拔高度：** 海拔200-350公尺。主要優質茶園位於洞庭山坡地——生產核心區，產出70%以上的高等級茶葉。

* **土壤：** 弱酸性黃壤 (黄壤, huáng rǎng)，pH值4.5-6.0，疏鬆肥沃，有機質含量高。良好的透水性和通氣性有利於茶樹根系發育。

* **獨特特色——間作系統 (间作, jiānzuò)：** 洞庭山的茶樹歷史上與果樹混植——枇杷 (枇杷, pípá)、楊梅 (杨梅, yángméi)、柑橘 (柑橘, gānjú)。這不僅是裝飾性安排：茶樹吸收落花落果的芳香物質，果樹提供天然遮蔭。結果是著名的「花果香」(花果香, huāguǒ xiāng)，無法通過人工香料複製。正因如此，即使使用相同栽培種和工藝，正宗洞庭碧螺春也無法在其他地區複製。

**5. 製作工藝：**

碧螺春的製作是完全手工過程，遵循「手不離茶，茶不離鍋」(手不离茶，茶不离锅) 的原則。從投料到成品的整個週期約40分鐘一鍋。

* **採摘 (采摘——cǎi zhāi)：** 清晨手工採摘。採摘最嫩的芽頭和一至二片葉子。同時在籃中進行初步分揀。

* **揀剔 (拣剔——jiǎn tī)：** 仔細手工去除損傷、粗糙和不標準的葉片、莖稈和「魚葉」。這一步驟的品質直接決定最終產品的均勻度。

* **攤放 (摊放——tān fàng)：** 將揀剔後的原料薄攤在陰涼通風的室內數小時。期間去除多餘水分，開始形成香氣前體物質。

* **殺青 (杀青——shāqīng)：** 將葉片投入180-200°C的熱鐵鍋 (铁锅) 中。師傅快速翻炒，停止氧化酶活動，固定新鮮綠色香氣。此步驟持續數分鐘，需要絕對精確——稍微過熱會產生焦味，不足則有草腥味。

* **揉捻 (揉捻——róuniǎn)：** 當鍋溫降至70-80°C時，師傅開始揉捻：葉片被滾壓、擠壓和捲曲，開始形成螺旋結構。細胞汁液滲出表面，確保沖泡時快速浸出。

* **搓團顯毫 (搓团显毫——cuō tuán xiǎn háo)：** 關鍵且最具技巧的步驟，賦予碧螺春著名的形狀。在60-65°C溫度下，師傅將葉片聚成小團，用手掌輕柔搓揉，形成緊密螺旋。同時發生「顯毫」——細嫩的銀色茸毛 (白毫, báiháo) 從葉面分離，覆蓋捲曲的螺旋，賦予茶葉特徵性的銀綠外觀。正是這一技法是工藝精湛的標誌：茸毛越密、螺旋越緊，品質越高。

* **文火乾燥 (文火干燥——wénhuǒ gānzào)：** 在50-60°C低溫下最終乾燥。茶葉達到穩定狀態，形成最終香氣。成品含水量不超過7%。

**6. 感官特徵：**

* **乾茶外觀：** 纖細緊捲的螺旋 (条索纤细蜷曲)，宛如微型蝸牛殼 (螺形)。顏色為銀綠隱翠 (银绿隐翠, yínlǜ yǐncuì)。表面密佈細嫩白毫 (白毫密布)。高等級茶葉特點是均勻：每個螺旋大小一致，無碎屑和粗糙片段。

* **乾茶香氣：** 濃郁複雜，具有明顯的花果香調 (花果香馥郁, huāguǒ xiāng fùyù)——枇杷、楊梅、柑橘花香。果香層次之上是嫩芽的清新綠香 (嫩香)。香氣如此濃烈，正是它給了茶葉歷史上的民間名稱「嚇煞人香」——「香得嚇死人」。

* **茶湯香氣：** 高雅清香持久 (清香高雅持久)。以花果香調為主，輔以清新綠色甘甜。杯底留香 (冷杯, lěng bēi) 時顯現蜂蜜和香草韻味，在杯中停留數分鐘。

* **滋味：** 鮮爽 (鲜爽, xiānshuǎng) ——第一印象：明亮「活潑」的鮮爽，源於高胺基酸含量（≥3.5%）。回甘迅速 (回甘迅速, huígān xùnsù) ——第一口後，口中充滿柔和的果香甘甜。茶體中等厚實，柔和圓潤 (醇厚, chúnhòu)。多酚含量（20-24%）提供輕微的結構性澀味，但不粗糙。餘韻悠長，清香怡人，帶果香尾韻。

* **湯色：** 嫩綠清澈 (嫩绿清澈)。在玻璃杯中沖泡時可觀察到「雪浪噴珠」(雪浪喷珠) 效果——白色茸毛從葉片分離，在茶湯中飄浮，創造出如畫般的景象。

* **葉底（沖泡後茶葉）：** 嫩綠柔軟，均勻一致的淺綠色葉片，從螺旋中展開。保持「一芽一葉」形狀。葉片完整無損，色澤均勻。

**7. 化學成分：**

碧螺春的化學成分由早春採摘、小葉種栽培種和洞庭山獨特風土條件決定。以下是春茶的典型指標：

* **多酚類（兒茶素）：** 含量20-24%乾重。主要成分是表沒食子兒茶素沒食子酸酯（EGCG），提供強大的抗氧化潛力。比較研究顯示，碧螺春多酚的抗氧化效力比一般綠茶高約30%，這與兒茶素組成中EGCG比例較高有關。

* **胺基酸：** 含量不少於3.5%乾重。以L-茶胺酸 (茶氨酸, chá'ānjīsuān) 為主，負責鮮爽感、鮮味和溫和放鬆效果。高胺基酸含量配合適中多酚水平，創造出平衡、柔和且帶甘甜的口感，無明顯苦澀。

* **生物鹼：** 咖啡因含量適中（約3.0-4.0%乾重）。咖啡因作用被L-茶胺酸緩和，提供平穩溫和的提神效果。還含有可可鹼和茶鹼。

* **維生素：** 維生素C含量豐富，因為早春採摘和溫和手工炒製最大程度保存了這種不穩定的維生素。B群維生素（B₁、B₂）、維生素E、類胡蘿蔔素（維生素A前體）。

* **礦物質：** 鉀、鎂、鋅、鐵、錳、氟。礦物質組成由洞庭山弱酸性黃壤決定。

* **精油和芳香化合物：** 碧螺春的揮發性芳香複合物極其豐富：芳樟醇、香葉醇、橙花醇、順式茉莉酮等萜烯類化合物形成特徵性花果香調。香氣的獨特性與間作系統——與果樹共生——有關。

* **水溶性糖類和果膠：** 賦予茶湯柔和的「體感」和甘甜感。

**8. 保健功效：**

* **抗氧化作用：** 兒茶素（特別是EGCG）有效中和自由基，減緩氧化壓力和細胞老化過程。
* **提神效果和改善認知功能：** 咖啡因與L-茶胺酸結合，提供溫和平穩的能量提升，無急劇峰值和隨後下降。L-茶胺酸額外促進專注力和平靜集中。
* **清涼解渴作用：** 碧螺春傳統上歸類為「涼性」茶 (性凉, xìng liáng)，推薦在炎熱天氣飲用以解渴和清內熱。
* **改善消化：** 茶多酚刺激消化酶分泌，幫助脂肪分解，緩解餐後飽脹感。
* **強化心血管系統：** 多酚和維生素C有助於降低低密度脂蛋白膽固醇水平，強化血管壁。
* **抗菌作用：** 兒茶素抑制口腔病原菌生長，清新口氣。
* **支持新陳代謝：** 咖啡因和兒茶素激活代謝過程，促進脂肪分解。

* **重要提醒：** 上述功效基於綠茶成分的公開資料，不構成醫療建議。

**9. 沖泡方法：**

* **水溫：** 80-85°C（沸水冷卻約2分鐘）。絕不可用滾燙開水——過熱會損傷最嫩的芽頭，造成苦澀並失去香氣。

* **茶量：** 3公克配150-200毫升水。

* **茶具：** 玻璃杯 (玻璃杯, bōli bēi) 是最佳選擇，可觀賞著名的「雪浪噴珠」效果和螺旋在水中的展開。也可使用白瓷蓋碗 (白瓷盖碗) 以更精確控制香氣。**不建議** 使用紫砂壺 (紫砂壶)——密閉蓋子和多孔壁會「悶住」精緻香氣。

* **沖泡過程（上投法/上投法, shàng tóu fǎ）：**

	1. 用熱水溫杯，倒掉。
	2. 注入80-85°C的水至杯子7/10容量。
	3. 投入3公克茶葉——螺旋開始緩慢下沉，在水中展開「舞蹈」。
	4. 等待茶葉沉底（約30-40秒）。
	5. 第一泡完成——品飲，享受鮮爽和初顯的果香。
	6. 第二、三泡——每次增加10秒。茶葉可沖泡3次。

* **注意：** 「上投法」（先注水後投茶）是碧螺春的經典方法。它防止嫩芽燙傷，並可欣賞茶葉展開。最佳飲用溫度約60°C：此時最能感受甘甜和鮮爽。

**10. 儲存：**

* 密封保存於瓷器、玻璃或鐵罐中，置於陰涼乾燥處，遠離異味。
* 最佳儲存溫度0-5°C（冰箱），單獨存放，避免與有強烈氣味的食品接觸。密封性極其重要：茶葉極易吸濕和吸收異味。
* 避免光照、潮濕和高溫——綠茶的主要「敵人」。
* 開封後建議一個月內飲用完畢以保持最佳鮮度。
* 在適當條件下保質期可達12個月，但為獲得最佳口感體驗，建議在採摘後6個月內飲用。

**11. 價格與仿冒品：**

洞庭碧螺春是中國最昂貴的綠茶之一。價格由幾個關鍵因素決定：採摘時間（明前茶比雨前茶貴數倍）、等級（特級需要6-7萬個芽頭製成500公克）、手工或機械加工，以及洞庭山核心產區的正宗性。核心產區茶園面積有限，造成正宗產品長期供不應求。

價格參考（2024年）：特級明前茶——50公克1200元人民幣起；一級——500公克300-500元人民幣；二、三級——價格顯著較低。

* **如何避免仿冒品：**

	* **向專營蘇州茶葉的可靠商家購買**，檢查地理標誌認證標識。
	* **評估茸毛：** 正宗高等級碧螺春密佈銀色茸毛。但過多「粉狀」茸毛，一觸即脫落，可能表示人工添加。
	* **評估香氣：** 真正的碧螺春散發新鮮果香和花香——自然、豐富，無「香水」或化學氣味。人工香料添加會感到刺鼻且快速消散。
	* **評估茶湯：** 清澈透明，嫩綠色。混濁或暗淡的茶湯值得懷疑。
	* **注意價格：** 如果「洞庭碧螺春」以普通綠茶價格出售——很可能是來自四川、貴州或其他地區的茶葉，雖採用類似工藝製作，但不具備洞庭山的風土特色。

**12. 趣聞軼事：**

* 製作一斤（500公克）特級碧螺春需要採摘和加工6-7萬個單獨芽頭——這是世界茶葉中最費工的指標之一。

* 「雪浪噴珠」(雪浪喷珠, xuělàng pēnzhū) ——沖泡時產生效果的詩意名稱：白色茸毛從展開的螺旋中分離，在綠色茶湯中飄浮，創造出雪花飄落的畫面。

* 洞庭山的間作系統（茶樹與果樹共生）無法人工複製。多次嘗試用香料模仿「果香」都能被識別出與原品的差異。

* 賜名「碧螺春」的康熙皇帝是著名的雅藝鑑賞家。將「嚇煞人香」改名為「碧螺春」是中國美食史上最著名的「皇帝命名」例子之一。

* 碧螺春是少數傳統上使用「上投法」(上投法) 沖泡的中國茶葉之一：先倒水，後投茶。大多數其他綠茶採用相反順序。

**13. 與其他著名中國綠茶的比較：**

* **西湖龍井 (西湖龙井, Xīhú Lóngjǐng)：** 來自浙江省。扁平茶葉，栗豆香氣，「結構性」口感帶明顯鮮味。碧螺春在形狀（螺旋vs扁平）和香氣特徵（花果香vs栗豆香）上完全相反。如果龍井是「建築式」嚴謹，碧螺春則是「繪畫式」柔美。

* **黃山毛峰 (黄山毛峰, Huángshān Máo Fēng)：** 來自安徽省。「雀舌」形茶葉帶白毫，溫和花香，精緻口感。毛峰更柔和細膩，碧螺春更明亮果香，香氣更濃郁。

* **太平猴魁 (太平猴魁, Tàipíng Hóu Kuí)：** 來自安徽省。大型扁平茶葉，蘭花香，深沉草本味。對比鮮明：猴魁是著名綠茶中葉片最大的，碧螺春是最精巧的之一。

* **安吉白茶 (安吉白茶, Ānjí Bái Chá)：** 來自浙江省。來自白化芽葉的綠茶，胺基酸含量創紀錄（6-7%）。安吉白茶是「純粹甘甜和鮮味」無明顯果香，而碧螺春首先是豐富立體的花果香氣。

* **六安瓜片 (六安瓜片, Liù'ān Guā Piàn)：** 來自安徽省。扁平「瓜子」形純葉片茶，無芽頭。口感濃厚草本，帶炒籽香。碧螺春更細膩、輕盈且芳香。

**結語：**

碧螺春是春天在茶杯中的化身：每一個微小螺旋在溫水中展開時，釋放出洞庭山花園的芳香、太湖晨霧的清新和初果的甘甜。這是為那些尋求的不僅是飲品，而是美學體驗的人準備的茶——從凝視玻璃杯中的「雪珠」到悠長的果香餘韻，提醒我們最好的茶葉誕生於自然與人類工藝達到完美平衡的地方。
