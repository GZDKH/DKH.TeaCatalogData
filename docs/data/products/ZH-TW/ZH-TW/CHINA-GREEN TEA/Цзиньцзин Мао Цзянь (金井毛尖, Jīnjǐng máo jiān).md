# 金井毛尖 (金井毛尖, Jīnjǐng máo jiān)

**金井毛尖 (金井毛尖, Jīnjǐng máo jiān)**
![Цзиньцзин Мао Цзянь (金井毛尖, Jīnjǐng máo jiān)](images/Цзиньцзин%20Мао%20Цзянь%20%28金井毛尖,%20Jīnjǐng%20Máo%20Jiān%29.png)

![Цзиньцзин Мао Цзянь (金井毛尖, Jīnjǐng máo jiān)-1](images/Цзиньцзин%20Мао%20Цзянь%20%28金井毛尖,%20Jīnjǐng%20Máo%20Jiān%29-1.png)

金井毛尖是湖南綠茶學派毛尖類 (毛尖, máo jiān —「毛茸茸的嫩尖」) 的代表，以「三綠」(三绿, sān lǜ) 的特色著稱：乾茶綠色、茶湯綠色、葉底綠色。此茶產於湖南省長沙縣金井鎮，是國家級地理標誌產品，其製作技藝已列入湖南省非物質文化遺產名錄。

**1. 分類與產地:**

* **類型：** 綠茶 (绿茶, lǜchá)，不發酵茶。
* **類別：** 湖南地方綠茶；入選「湖南十大名茶」(湖南十大名茶, 2005年)。
* **產地：** 中國湖南省 (湖南省, Húnán shěng) 長沙縣 (长沙县, Chángshā xiàn) 金井鎮 (金井镇, Jīnjǐng zhèn)。核心產區位於金井河 (金井河) 與脫甲河 (脱甲河) 匯流處的丘陵茶園。
* **地理座標：** 約北緯28°40′，東經113°–114°。

**2. 歷史與文化意義:**

* **歷史：** 長沙縣的茶葉傳統可追溯至唐代 (618–907)：唐代文獻已記載長沙地區的茶葉生產。宋代時此地有二十多種名茶，包括仙芝 (仙芝) 和玉津 (玉津)。李時珍 (李时珍, Lǐ Shízhēn) 在《本草綱目》(本草纲目, Běncǎo Gāngmù) 中提及「楚之茶」—湖南省的茶，包括金井的「白露」(白露)，這是當地茶葉知名度的早期證據之一。清代咸豐、同治年間 (1851–1874)，金井和高橋的紅茶特別著名，形成了所謂的長沙–瀏陽–平江「紅茶區」。後來綠茶取代了紅茶，金井毛尖成為綠茶系列的旗艦產品。現代形式的茶葉於**1984年**正式創製，同年獲得農牧漁業部品質獎。**2005年**入選湖南十大名茶。**2008年**「金井」商標 (金井牌) 成為省內茶業首個獲得「中國馳名商標」(中国驰名商标, Zhōngguó chímíng shāngbiāo) 地位的品牌。**2016年**製作技藝列入湖南省非物質文化遺產名錄。

* **名稱：** 金井 (Jīnjǐng) —「金井」，鎮名，與當地天然水源有關。毛尖 (Máo Jiān) —「毛茸茸的嫩尖」—綠茶類別的經典稱謂，指由嫩芽和上層葉片製成，覆蓋白毫 (白毫, bái háo)。

* **文化意義：** 金井鎮的茶葉歷史與該地區的經濟史緊密相連：19世紀金井紅茶經漢口出口至歐洲和俄國，茶葉貿易形成了當地商人階層。20世紀轉向綠茶是對國內市場變化的回應，金井毛尖迅速佔據了「地方旗艦」的地位。今日它是長沙縣茶文化的名片。金井鎮定位為生態清潔區，獲得「中國天然氧吧」(中国天然氧吧) 認證。2023年「金井」品牌在湖南茶葉品牌價值中位居前列。裝在「中國紅」(中国红,「中國紅」) 裝飾瓷器包裝中的茶葉曾在拍賣會上以63,800元人民幣的價格售出二兩 (100克)，因此獲得「綠色金茶」(绿色金茶) 的非正式稱號。

**3. 植物學描述與原料:**

* **品種/栽培種：** 金井茶園栽培數個*Camellia sinensis* var. *sinensis*品種：主要有白毫早 (白毫早, Báiháo Zǎo，早熟且毫毛豐富)、福鼎大白 (福鼎大白, Fúdǐng Dà Bái，發芽能力強)、槠葉齊 (槠叶齐, Zhū Yè Qí) 和湘波綠 (湘波绿, Xiāngbō Lǜ)。大部分茶樹樹齡超過30年。「一芽一葉」標準的百芽重約45克，嫩度保持期比平原茶園延長7–10天。
* **採摘：** 主要採摘期為早春；「明前」(明前, míngqián，清明節前) 等級最為珍貴。嚴格遵循「五不採」(五不采, wǔ bù cǎi) 原則：雨後不採、蟲傷不採、紫色葉不採、過長和畸形芽葉不採。
* **採摘標準：** 特級 (特级, tèjí) —主要為長度不超過2.0公分的完整芽頭；一級—一芽一葉初展 (≥80%)；二級—一芽二葉。
* **原料要求：** 純有機種植—僅使用有機肥料；茶園通過瑞士IMO (Institute for Marketecology) 有機茶標準認證。

**4. 風土條件與栽培特點:**

* **氣候與地形：** 該地區屬亞熱帶季風濕潤氣候。年平均氣溫17.2°C，年降雨量1,300–1,400毫米，無霜期274天。霧日超過180天，提供漫射光照—茶葉中芳香物質和胺基酸積累的關鍵因素之一。
* **海拔高度：** 海拔200–300公尺 (丘陵地帶)。
* **土壤：** 以紫色土 (紫色土, zǐsè tǔ) 為主，pH值5.5–6.5，富含鉀、矽和微量元素。流域森林覆蓋率72–87.6%，負氧離子濃度達城市水準的五十倍。
* **栽培特點：** 茶園以當地泉水灌溉。霧氣、酸性肥沃土壤和溫和溫差的結合形成了特有的「栗香」香氣特徵，使金井毛尖有別於其他湖南毛尖。周圍地區的高森林覆蓋率形成天然生態屏障，阻止污染和害蟲擴散。茶園位於丘陵河間地帶，晨霧在山谷中滯留至上午晚些時候，為茶葉提供額外的漫射光照時間。這減緩了光合作用，促進胺基酸 (主要是L-茶胺酸) 的積累而非兒茶素，形成特有的溫和口感和增強的甜味。

**5. 製作工藝:**

工藝包括六個主要步驟，採用獨特的「適溫提毫」(适温提毫, shìwēn tíháo) 技法，這是金井毛尖的特色：

1. **鮮葉攤放 (鲜叶摊放, xiānyè tānfàng)：** 在竹盤上攤放4–6小時，部分失水並為殺青做準備。
2. **殺青 (杀青, shāqīng)：** 滾筒殺青，溫度150°C，持續5–6分鐘，停止酶促氧化並保持葉片綠色。
3. **揉捻 (揉捻, róuniǎn)：** 三階段「輕—重—輕」過程，持續30–60分鐘；破壞細胞壁並釋放汁液，形成茶湯濃度。
4. **適溫提毫 (适温提毫, shìwēn tíháo)：** 獨特步驟，在嚴格控制的溫度下處理茶葉，固定茶葉表面的白毫並保持香氣清新。正是這一技法確保了成品茶特有的「銀綠色」外觀。
5. **烘乾 (烘干, hōnggān)：** 溫度約80°C，直至含水量≤6.5%。
6. **分級 (分级, fēnjí)：** 按等級和品質分選。

**6. 感官特徵:**

* **乾茶外形：** 茶條細緻均勻，螺旋捲曲 (卷曲似螺)，白毫顯露 (白毫显露)，銀綠色帶翠綠底色。表面光滑有光澤。
* **乾茶香氣：** 清純高香，帶明顯栗香 (栗香, lì xiāng)，特級茶尤為突出。手中溫熱時展現清新花草基調。
* **茶湯香氣：** 持久清純，上層音符帶栗香重點，基調為清新綠意。香氣在數泡中持續。
* **滋味：** 清鮮 (清鲜, qīng xiān)，回甘明顯 (回甘, huígān)。茶體輕盈但不寡淡；胺基酸成分帶來溫和的鮮味成分。正確沖泡時幾乎無苦澀味。
* **湯色：** 嫩綠清澈 (嫩绿清澈, nèn lǜ qīngchè)，透明有亮澤。
* **葉底 (沖泡後茶葉)：** 嫩勻，鮮綠色，芽葉完整度良好。

**7. 化學成分:**

* **多酚類 (兒茶素)：** 茶多酚含量≥30% (一級茶)，提供顯著的抗氧化活性。據生產商數據，金井毛尖多酚類清除自由基的效率是維生素E的18倍。
* **胺基酸 (包括L-茶胺酸)：** 由於霧氣微氣候和遮蔭條件含量較高，形成甜味和口感「體感」。
* **水浸出物：** ≥45% (特級)，表明可溶性物質豐富，浸出率高。
* **生物鹼：** 咖啡因、可可鹼、茶鹼—優質綠茶的標準組合，提供溫和刺激。
* **微量元素：** 茶葉鋅 (Zn) 和硒 (Se) 含量較高—紫色土壤特殊礦物成分的結果。
* **維生素：** B群、維生素C (春季新鮮原料含量顯著)、維生素K。
* **精油：** 栗香和草本芳香化合物，在滾筒殺青和提毫階段形成。
* **氟：** 氟含量有助於強化牙釉質並抑制致齲細菌。

**8. 保健功效:**

1. **顯著抗氧化保護：** 兒茶素中和自由基，延緩細胞老化過程。
2. **溫和提神效果：** 咖啡因和L-茶胺酸的結合提供清醒感而不過度刺激神經系統。
3. **心血管系統支持：** 兒茶素 (特別是EGCG) 有助於減少血管脂質沉積，降低動脈硬化風險。
4. **促進消化：** 多酚類刺激消化酶分泌，有助於餐後脂肪分解。
5. **強化牙釉質：** 氟和兒茶素抑制口腔細菌發展並減少牙菌斑形成。
6. **礦物質補充：** 鋅和硒含量較高，支持免疫功能和皮膚健康。
7. **認知支持：** L-茶胺酸有助於注意力集中和思維清晰，降低焦慮水準。
8. **代謝支持：** 多酚類和咖啡因複合刺激產熱並加速脂肪分解，規律飲用可能有助於維持健康體重。

**9. 沖泡方法:**

* **水溫：** 85°C (特級茶—80°C)。絕對不建議使用85°C以上的沸水：這會破壞L-茶胺酸並引起過度苦澀。
* **茶量：** 玻璃杯法3克配150毫升 (1:50比例)；蓋碗功夫茶法5–6克配120毫升。
* **茶具：** 玻璃杯—觀察茶葉「舞蹈」的理想選擇 (建議上投法—上頭法, 上投法，將茶葉放入已倒好的水中)。白瓷蓋碗適合專注品香。不建議紫砂壺—多孔陶土可能吸收精緻香氣。
* **用水：** 泉水或過濾軟水；高礦化度自來水不宜，可能影響口感。
* **過程：**
  1. 用熱水溫杯或蓋碗。
  2. 上投法：倒入適溫水至7/10容量，然後輕輕投入茶葉。
  3. 首泡30秒，後續每泡增加10秒。
  4. 3–4泡 (玻璃杯) 或5–6泡 (蓋碗，使用5–6克時)。

**10. 儲存方法:**

* **包裝：** 密封包裝 (真空袋或密蓋鐵罐)，必須防異味、避光、防潮。
* **溫度：** 最佳0–5°C (冰箱) 長期儲存；正確條件下保質期可達12個月。開封前需將包裝在密閉狀態下回復至室溫，避免茶葉凝露。
* **開封後：** 一個月內飲用完畢；密封儲存於冰箱。
* **生產商建議：** 新製茶葉建議在暗處放置7天「醒茶」(醒茶, xǐng chá)，讓輕微的火氣散去。

**11. 價格與仿冒品:**

* **價格參考：** 特級 (单芽，完整芽頭，手工處理) —800元人民幣/斤 (500克) 以上；一級—200–500元/斤；二級—100–300元/斤。裝飾瓷器禮盒裝可能價格更高。
* **價格因素：** 採摘季節 (明前比雨前貴)、採摘標準 (完整芽頭比芽葉貴)、手工或機械作業、特定年份產量。
* **避免仿冒品方法：**
  - 向「金井」品牌授權經銷商或金井鎮合作社直接購買。
  - 檢查外觀：正品茶葉條索細緻螺旋捲曲，白毫豐富，銀綠色無褐色雜質。
  - 評估香氣：清純栗香調，無霉味、酸味或煙味。
  - 注意茶湯：應為嫩綠透明，無混濁。
  - 「特級」茶葉價格異常低廉是原料替換的首要徵象。

**12. 趣味知識:**

1. **獨特提毫技法：** 金井毛尖採用的適溫提毫 (适温提毫, shìwēn tíháo) 技術被認為是該茶的獨特技術訣竅。正是這一技法確保了成品茶葉白毫特有的「霜花」光澤—湖南綠茶中的識別標誌。

2. **拍賣記錄：** 「中國紅」(中国红) 瓷器包裝的金井毛尖禮盒 (100克) 曾在拍賣會上以63,800元人民幣售出，因此在當地獲得「綠色金茶樹」(绿色金茶树) 的非正式稱號。

3. **國際有機認證：** 金井茶園多年來通過瑞士IMO組織有機種植標準認證—這在中國地方綠茶中較為罕見。

4. **文學足跡：** 李時珍《本草綱目》中提及的「楚之茶…白露」(楚之茶…白露) 被認為是明代金井地區綠茶知名度的歷史證據之一。

5. **生態品牌：** 金井鎮獲得「中國天然氧吧」認證—茶園負氧離子濃度比城市水準高50倍。這是湖南少數兼具生態度假村地位和工業茶葉生產的茶區之一。

6. **紅茶—被遺忘的前身：** 在金井毛尖出現之前，該鎮主要以紅茶聞名，構成清代出口貿易的基礎。20世紀從紅茶轉向綠茶是整個茶區成功轉型專業化的罕見例子之一。

**13. 與其他毛尖類綠茶比較:**

* **信陽毛尖 (信阳毛尖, Xìnyáng Máo Jiān)：** 河南省。「中國十大名茶」之一。針形筆直 (與金井的螺旋形不同)，栗香明顯帶「熟栗殼」調。在更高溫度下手工炒製 (160–200°C)，茶湯更濃厚飽滿。

* **古丈毛尖 (古丈毛尖, Gǔzhàng Máo Jiān)：** 湖南省 (湘西)。筆直形狀白毫豐富，香氣清新帶花香。在高海拔 (達800公尺) 富硒土壤地區種植。工藝包括八個步驟，手工成形。

* **都勻毛尖 (都匀毛尖, Dūyún Máo Jiān)：** 貴州省。「中國十大名茶」之一。採用小葉種高山品種，海拔1,000公尺以上。形狀為緊捲「鉤形」帶銀毫；口感更清淡甜味明顯，香氣更細膩。多酚含量約20%，明顯低於金井毛尖 (≥30%)。

* **溈山毛尖 (沩山毛尖, Wèishān Máo Jiān)：** 湖南省 (寧鄉)。重要區別：屬於**黃茶** (黄茶) 類，非綠茶。包含「悶黃」(闷黄, mèn huáng) 工序，使茶湯呈橙黃色，帶松脂煙燻香氣—完全不同的風味特徵。

* **白馬毛尖 (白马毛尖, Báimǎ Máo Jiān)：** 湖南省 (雪峰山脈)。以白馬山脈命名，茶園位於山腳與黃金井 (黄金井) 谷地交匯處。茶條更緊實，略彎曲；香氣高雅花香；茶湯更清淡，綠色純淨。1991年獲得「湖南省名茶」稱號。

**結論：**

金井毛尖是現代生產工程精度與湖南丘陵數百年茶葉傳統相結合的茶葉。其名片—「三綠」和栗香配清甜回甘—是紫色土壤富含鋅硒、金井谷地霧氣微氣候和獨特「提毫」技法的獨特風土結果。這是一款適合那些在綠茶中欣賞的不是苦澀禁慾，而是溫和、近似糖果般甜味與清新綠意結合的茶。對綠茶初學者而言，它展現了中國毛尖溫和易飲的一面，對經驗豐富者而言，它提供了重新思考被低估的湖南茶省豐富性的機會。
