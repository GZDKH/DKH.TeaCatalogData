# 徐聞綠茶 (徐闻绿茶, Xúwén lǜchá)

**徐聞綠茶 (徐闻绿茶, Xúwén lǜchá)**
![Сюйуэнь Люй Ча (徐闻绿茶, Xúwén lǜchá)](images/Сюйуэнь%20Люй%20Ча%20(徐闻绿茶,%20Xúwén%20Lǜchá).png)

![Сюйуэнь Люй Ча (徐闻绿茶, Xúwén lǜchá)-1](images/Сюйуэнь%20Люй%20Ча%20(徐闻绿茶,%20Xúwén%20Lǜchá)-1.png)

徐聞綠茶是來自徐聞縣 (徐闻县, Xúwén Xiàn) 的獨特品種綠茶，該縣位於中國大陸最南端，廣東省雷州半島上。這是少數採用蒸青 (蒸青, zhēngqīng) 技術生產的現代中國綠茶之一，與日本茶道傳統相近。此茶以其標誌性的「三綠」(三绿, sān lǜ) 特徵聞名——綠色乾茶葉、綠色茶湯和綠色茶底——以及由於靠近海洋和火山土壤而產生的獨特「海洋」清新韻味。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。主要風格為蒸青 (蒸青绿茶, zhēngqīng lǜchá，蒸製綠茶)；也有炒青 (炒青绿茶, chǎoqīng lǜchá，炒製綠茶) 版本。
* **類別：** 中國地方綠茶。1990年全國首個獲得「綠色食品」(绿色食品, lǜsè shípǐn) 認證的茶葉。入選國家標準「無公害優質食品」(国家无公害优质食品, 1999年)。
* **產地：** 中國廣東省 (广东省, Guǎngdōng Shěng) 湛江市 (湛江市, Zhànjiāng Shì) 徐聞縣 (徐闻县, Xúwén Xiàn)。生產集中在縣境東部和北部山丘地帶。
* **地理座標：** 約北緯20°13′–20°43′，東經109°52′–110°35′。

**2. 歷史與文化意義：**

* **歷史：** 徐聞縣的茶葉種植有四百多年歷史。早在明朝 (明, 1368–1644) 和清朝 (清, 1644–1912) 時期，下橋鎮 (下桥镇, Xiàqiáo Zhèn) 石板嶺 (石板岭, Shíbǎnlǐng) 的當地居民就種植和加工茶葉。這裡保存的約400年樹齡的茶樹被視為徐聞茶的母本基金。20世紀，從東南亞歸國的華僑 (华侨, huáqiáo——中國移民) 對茶業發展做出重大貢獻，帶來了手工炒製技術。1960-1980年代，國有農墾系統 (农垦系统, nóngkěn xìtǒng) 建立了機械化茶園，其中海鷗農場 (海鸥农场, Hǎi'ōu Nóngchǎng) 成為關鍵基地。1983年，這裡建立了首批認證的生態茶園，採用生物植保代替化學農藥。1990年，「雄鷗」(雄鸥, Xióng'ōu) 和「勇士」(勇士, Yǒngshì) 品牌獲得中國首批國家「綠色食品」認證之一。1992年，徐聞綠茶在聯合國環境與發展大會（裡約熱內盧）上作為中國生態清潔食品的典範展示。1995-1997年，該茶兩次在中國農業博覽會 (中国农业博览会, Zhōngguó Nóngyè Bólǎnhuì) 上獲得銀獎。「雄鷗」品牌還獲得「中國名牌農產品」(中国名牌农产品, Zhōngguó Míngpái Nóngchǎnpǐn) 地位，並在2019-2020年獲得「廣東名茶」(广东名茶, Guǎngdōng Míngchá) 稱號。

* **名稱：** 「徐聞」(徐闻) 是該縣的歷史名稱，自西元前111年由漢朝將軍路博德 (路博德, Lù Bódé) 建立時就存在。「綠茶」(绿茶) 字面意思是「綠茶」。因此，全名簡單地意味著「徐聞[縣]綠茶」。

* **文化意義：** 徐聞是漢代海上絲綢之路 (海上丝绸之路, Hǎishang Sīchóu zhī Lù) 的歷史起點。這裡的茶業與農墾文化 (农垦文化, nóngkěn wénhuà) 和雷州半島的「紅土文化」(红土文化, hóngtǔ wénhuà) 緊密交織。現代的「茶與鳳梨」旅遊區 (茶与菠萝创新创业基地) 將茶園與風景如畫的鳳梨田結合，形成獨特的農業旅遊路線。

**3. 植物學描述與原料：**

* **品種/栽培種：** 種植園的主體（約70%）由雲南大葉種 (云南大叶种, Yúnnán Dàyè Zhǒng) 組成——喬木型 (*Camellia sinensis* var. *assamica*)，以厚實肉質葉片著稱，茶多酚含量 ≥ 28.3%。其餘部分為海南大葉種 (海南大叶种, Hǎinán Dàyè Zhǒng)。種植在火山坡地上，不使用化學肥料和農藥。繁殖採用傳統扦插法，保持群體種 (群体种, qúntǐ zhǒng) 的遺傳多樣性和抗性。
* **採摘：** 由於熱帶氣候，徐聞的採摘季節長達8-10個月——比長江流域主要產茶區長得多。早春採摘從1-2月開始，比長江流域茶區早30-70天。
* **採摘標準：** 特級 (特级, tèjí)：純芽或「一芽一葉」(一芽一叶, yī yá yī yè)。一級 (一级, yījí)：「一芽二葉」(一芽二叶, yī yá èr yè)。大宗茶 (大宗茶, dàzōng chá)：「一芽三葉」(一芽三叶, yī yá sān yè)。
* **原料要求：** 幼嫩、均勻的芽葉，無粗老葉片、機械損傷和異味。零農藥殘留是「綠色食品」認證的關鍵指標。

**4. 風土條件與種植特色：**

* **氣候與地形：** 徐聞縣位於熱帶季風氣候區 (热带季风气候, rèdài jìfēng qìhòu)。年平均氣溫23.6°C，年降雨量1864毫米，霧日超過150天，日溫差超過8°C。正是高濕度、豐富霧氣和顯著日夜溫差的結合促進了茶葉中氨基酸的積累：徐聞早春茶的氨基酸含量比中國內陸同類產品高15%。
* **海拔高度：** 種植園主要位於低海拔（海拔200-300公尺）的緩坡火山丘陵上。儘管海拔不高，但海洋鄰近、火山土壤和熱帶霧氣的獨特結合彌補了高度因素的作用。
* **土壤：** 磚紅壤 (砖红壤, zhuānhóng rǎng)，形成於火山岩上，pH值4.5-6.5。土壤富含硒（0.018-0.066毫克/公斤）、礦物元素和有機物質（≥ 3%）。該地區無工業污染。
* **種植特色：** 風土的獨特性由兩個因素的結合決定——火山地質和海洋鄰近，形成了特徵性的風味輪廓，用詩意的公式描述為「陸潤海韻」(陆润海韵, lù rùn hǎi yùn)。茶園分佈在全縣範圍內，但主要生產集中在兩個關鍵區域：
  * **下橋鎮石板嶺** (下桥镇石板岭, Xiàqiáo Zhèn Shíbǎnlǐng) ——400年茶樹母本基金，有機茶業示範區。
  * **海鷗農場** (海鸥农场, Hǎi'ōu Nóngchǎng) ——徐聞蒸青技術的誕生地，「雄鷗」品牌主要生產基地，提供全縣約40%的產量。

**5. 製作工藝：**

徐聞綠茶主要採用蒸青 (蒸青, zhēngqīng) 技術生產，這在現代中國是罕見的——這種方法在唐代 (唐, 618–907) 廣泛使用，並由陸羽 (陆羽, Lù Yǔ) 在《茶經》(茶经, Chájīng) 中描述，後來幾乎完全被炒製取代。海鷗農場從1980年代開始，經過十多年進行了2800多次交叉實驗，才為當地原料開發出最佳蒸青技術。

1. **採摘 (采摘 — cǎizhāi)：** 手工採摘「一芽二葉」標準（主要品種）或「一芽一葉」（特級）的幼嫩芽葉。
2. **蒸汽殺青 (蒸汽杀青 — zhēngqì shāqīng)：** 關鍵步驟——在100°C溫度下用蒸汽固定綠色。高溫短時間蒸汽處理瞬間使酶失活，停止氧化過程。「高溫短時」(高温短时, gāowēn duǎnshí) 方法確保最大限度保持葉綠素和新鮮香氣，同時減少草腥味。
3. **粗揉 (粗揉 — cūróu)：** 初步成型，開始釋放細胞汁液。
4. **中揉 (中揉 — zhōngróu)：** 進一步壓實和整形。
5. **精揉 (精揉 — jīngróu)：** 最終成型——機械化步驟，確保碎茶率低於3%，比手工揉捻提高40%的條索緊直度。
6. **乾燥 (干燥 — gānzào)：** 兩階段：120°C初烘 (初烘, chū hōng)，然後90°C足乾 (足干, zú gān) 至穩定含水量。

除了主要的蒸青風格外，小部分產品以炒青綠茶 (炒青绿茶, chǎoqīng lǜchá) 形式生產——採用傳統鍋炒殺青，產生更濃郁緊實的口感，適合大宗茶。

**6. 感官特徵：**

* **乾茶外觀：** 形狀——條索緊直勻整 (条索紧直匀整, tiáosuǒ jǐnzhí yúnzhěng)。茸毛（毫）表現不明顯。顏色——色澤翠綠光潤 (色泽翠绿光润, sèzé cuìlǜ guāng rùn)，呈濃郁翠綠色帶油潤光澤。
* **乾茶香氣：** 清香 (清香, qīngxiāng) 帶溫暖栗香 (栗香, lìxiāng) 和特徵性的碘海藻鮮香 (海藻鲜香, hǎizǎo xiānxiāng)——這是由海洋鄰近造成的獨特特徵。
* **茶湯香氣：** 相似——清純、新鮮，帶栗香基調和海洋韻味。香氣在冷杯中保持超過20分鐘 (冷杯留香 > 20分钟)。
* **滋味：** 鮮爽 (鲜爽, xiānshuǎng) 明顯，由氨基酸的和諧含量造成。甘滑 (甘滑, gānhuá) 質地，多酚含量適中。微苦無澀 (微苦无涩, wēi kǔ wú sè) 的輕微苦味，無收斂性澀味。回甘持久 (回甘持久, huígān chíjiǔ)。
* **湯色：** 湯色清澈綠亮 (汤色清澈绿亮, tāngsè qīngchè lǜ liàng)。
* **葉底（沖泡後茶葉）：** 葉底嫩勻成朵，黃綠鮮活 (叶底嫩匀成朵，黄绿鲜活)，嫩葉均勻，完整展開如「花朵」，黃綠色澤，生動新鮮。

**7. 化學成分：**

* **多酚類（兒茶素）：** 茶多酚含量佔乾重的25-30%——這是熱帶地區大葉種的典型指標。高兒茶素含量提供顯著的抗氧化特性。
* **氨基酸（包括L-茶氨酸）：** 早春茶含量比內陸同類產品高15%，這解釋了口感的增強「鮮爽」和「甘潤」。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēi jiǎn) 含量符合綠茶標準；可可鹼和茶鹼含量微量。
* **維生素：** 維生素C（含量比一般綠茶高1.2倍），B群維生素。
* **礦物質：** 由於火山紅壤，茶葉硒（Se）含量較高，氟（F）含量也較高——後者提供明顯的防蛀牙作用。含有鉀、錳和其他火山來源的微量元素。
* **精油：** 負責特徵性的「海洋」韻味和冷杯中香氣的持久性。
* **成分特點：** 零農藥殘留——「綠色食品」認證的關鍵指標。大葉種原料的高多酚與熱帶風土的增強氨基酸結合，為綠茶創造了不尋常的平衡：強烈的抗氧化活性但低澀味。

**8. 健康功效：**

* **抗氧化保護：** 高兒茶素含量（25-30%）對自由基提供強大的中和作用。
* **防蛀牙作用：** 氟含量增高在牙釉質上形成保護層；據生產商數據，防蛀牙效果比普通綠茶高30%。
* **代謝支持：** 茶多酚有助於脂肪分解，增高的維生素C含量增強整體代謝過程。
* **溫和提神效果：** 咖啡鹼與L-茶氨酸結合提供平穩、持久的清醒感，無急劇起伏。
* **生態清潔：** 零農藥殘留水平最小化毒性負荷，這對日常飲用特別重要。
* **消化支持：** 適中的多酚水平對餐後胃腸道功能有益。

* **重要提醒：** 所述功效基於成分數據和傳統使用；這是一般資訊，非醫療建議。

**9. 沖泡方法：**

* **水溫：** 標準茶80-85°C；特級 (特级) 75°C，以免損傷嫩芽。
* **茶量：** 比例1:50（約3公克配150毫升）。
* **茶具：** 玻璃杯——理想選擇，可觀察特徵性的「三綠」(三绿)。白瓷蓋碗 (盖碗, gàiwǎn) 也適合。
* **沖泡過程：**
  1. 用熱水溫杯。
  2. 上投法 (上投法, shàngtóu fǎ)：先注入適溫水，然後輕柔放入茶葉。此法保持嫩芽完整。
  3. 第一泡——20秒。
  4. 每次後續沖泡增加10秒。
  5. 茶葉可沖泡3次完整的泡次。

* **飲用建議：** 避免空腹飲用（咖啡鹼可能刺激胃黏膜）；最佳時間——餐後一小時。日飲量不超過600毫升。飲茶與服藥（特別是鐵劑）間隔至少1小時，因為單寧可能降低鐵吸收。

**10. 儲存：**

* 密封包裝，避光、防異味、防潮。
* 最佳溫度——0-5°C（冰箱）；如同大多數綠茶，冷藏對保持新鮮度至關重要，特別是早春採摘。
* 開封後建議3個月內飲用完畢——此後香氣物質明顯減弱。
* 冰箱儲存的包裝開封前需在密封狀態下回溫至室溫，避免茶葉凝結水分。

**11. 價格與仿冒品：**

* **價格參考（市場價格，人民幣每斤/500公克）：**
  * 特級 (特级)：400元起——純芽或「一芽一葉」，最香最鮮。
  * 一級 (一级)：150-300元——「一芽二葉」，明亮綠湯，濃郁鮮味。
  * 大宗茶 (大宗茶)：80元以下——「一芽三葉」，耐泡性高，適合袋泡茶。
* **價格因素：** 採摘季節（早春茶較貴）、採摘標準、品牌（「雄鷗」和「勇士」最知名）、「綠色食品」認證。
* **避免仿冒品方法：**
  * 向可信賴的經銷商和「雄鷗」(雄鸥)、「勇士」(勇士) 品牌授權代理商購買。
  * 檢查「綠色食品」證書和產地標示。
  * 評估特徵性的「三綠」——乾茶、茶湯和葉底都應呈明顯綠色。
  * 注意「海洋」韻味——其他地區的仿品通常缺乏此特色。
  * 價格過低——值得懷疑真實性。

**12. 有趣事實：**

* 徐聞是中國少數工業化實踐蒸青 (蒸青) 的縣之一。這種方法由陸羽在8世紀描述，後來由佛教僧侶傳入日本，在中國本土幾乎完全被炒製取代。徐聞茶農實際上為新原料和氣候復興了古老技術，十年內進行了2800多次實驗。
* 徐聞縣是中國大陸最南端，海上絲綢之路的古代出發港。著名文人蘇軾 (苏轼, Sū Shì) 和偉大戲劇家湯顯祖 (汤显祖, Tāng Xiǎnzǔ) 曾被貶謫至此，後者在此建立了貴生書院 (贵生书院, Guìshēng Shūyuàn)。
* 徐聞茶園位於古火山口 (田洋火山口, tiányáng huǒshānkǒu)，賦予土壤獨特的礦物輪廓，決定了茶葉標誌性的「海洋」風味。
* 由於熱帶氣候，採摘季節長達10個月——這是中國所有產茶區中最長的採摘週期之一。早春茶比長江流域茶葉早1-2.5個月上市。
* 1992年，徐聞綠茶在裡約熱內盧「地球高峰會」上作為中國生態清潔農業的象徵展示——中國唯一獲此殊榮的茶產品。

**13. 與其他綠茶比較：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北省經典蒸青茶。兩種茶都採用蒸青法，但恩施玉露生長在高海拔（600-1200公尺），溫帶亞熱帶氣候，具有更明顯的堅果香氣和細膩草本味。相比之下，徐聞茶以特徵性「海洋」韻味和熱帶大葉種原料的更高多酚含量著稱。
* **日本煎茶 (煎茶, Sencha)：** 技術上最接近的類比——日本蒸製煎茶。然而，日本傳統涉及更長時間蒸製（特別是深蒸），茶樹遮陰和特殊的最終「精揉」成型。徐聞茶的鮮味輪廓較不明顯，但具有更明亮「陽光」的熱帶特性。
* **古勞茶 (古劳茶, Gǔláo Chá)：** 廣東另一知名綠茶，來自鶴山縣 (鹤山)。代表經典炒青綠茶 (炒青)，口感更緊實濃郁。徐聞蒸青茶在香氣上明顯更清新「純淨」。
* **嶗山綠茶 (崂山绿茶, Láoshān Lǜchá)：** 山東綠茶，同樣靠近海洋生長。具有相似的「海洋」韻味，但採用炒製法，在較低年均溫下成熟，採摘季節較短。嶗山茶口感更偏栗香和緊實。

**結論：**

徐聞綠茶是具有鮮明個性的綠茶，結合古老蒸青技術與海邊熱帶火山坡地風土。其「三綠」——翠綠茶葉、透明綠湯和新鮮明亮葉底——悅目怡人，而香氣中獨特的「海洋微風」是中國其他綠茶無法比擬的。這款茶特別適合重視口感清新純淨、尋求經典炒青綠茶替代品、對中國罕見蒸青風格感興趣的茶友。用軟水沖泡，勿過熱——徐聞綠茶將為您帶來天朝南端著名的「陸潤海韻」。
