# 金剛碧綠 (金刚碧绿, jīngāng bì lǜ)

**金剛碧綠 (金刚碧绿, jīngāng bì lǜ)**
![Цзиньган Би Люй (金刚碧绿, jīngāng bì lǜ)](images/Цзиньган%20Би%20Люй%20%28金刚碧绿,%20Jīngāng%20Bì%20Lǜ%29.png)

![Цзиньган Би Люй (金刚碧绿, jīngāng bì lǜ)-1](images/Цзиньган%20Би%20Люй%20%28金刚碧绿,%20Jīngāng%20Bì%20Lǜ%29-1.png)

金剛碧綠 (金刚碧绿, jīngāng bì lǜ) 是中國高山綠茶，屬於複合型加工工藝 (烘炒型, hōngchǎo xíng)，產於河南省商城縣大別山脈主峰金剛台的山坡上。此茶創製於1984年，通過融合龍井和毛尖工藝而成，結合了優雅的扁平外形與高山原料的強勁多酚潛力，是商城縣首個獲得國際獎項的茶葉。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，不發酵茶。按加工方式屬於複合烘炒型 (烘炒型绿茶, hōngchǎo xíng lǜchá) — 在生產過程中結合炒製和熱風乾燥（炭火）的綠茶。
* **類別：** 河南名優綠茶；信陽毛尖 (信阳毛尖, Xìnyáng Máojiān) 傳統的代表。屬於「商城高山茶」(商城高山茶) 系統 — 商城縣高山茶類。
* **產地：** 中國河南省 (河南省, Hénán Shěng) 信陽市 (信阳市, Xìnyáng Shì) 商城縣 (商城县, Shāngchéng Xiàn)。核心產區 — 金剛台山區 (金刚台, Jīngāng Tái) 及鄰近茶區：碧蓮峰 (碧莲峰, Bìlián Fēng，又稱大黃尖 — 大黄尖)、蘇仙石村 (苏仙石, Sūxiānshí)、東河白龍潭 (东河白龙潭) 和殷塆 (殷塆)。
* **地理座標：** 約北緯31°48′，東經115°32′。

**2. 歷史與文化意義：**

* **歷史：** 金剛碧綠是相對年輕、有目的性創製的茶葉。其研發始於1984年，在茶葉大師楊智棟 (杨智栋, Yáng Zhìdòng) 指導下，歷時五年實驗，力求結合兩大綠茶的優點：龍井 (龙井) 的扁平優雅外形和毛尖 (毛尖) 的毫顯香濃本質。到1989年，獨特工藝完成，茶葉進入批量生產。1990年在河南省名茶展覽會上，金剛碧綠被認定為「河南省新名茶」(河南省新名茶)。1992年通過農業部茶葉品質監督檢驗中心認證，確認符合名茶標準。1999年杭州國際茶葉博覽會成為其輝煌時刻，金剛碧綠榮獲金獎 (国际名茶金奖, guójì míngchá jīnjiǎng)，成為商城縣首個獲得國際認可的茶葉。2003年註冊同名商標，2007年成為商城茶葉中首個「河南省著名商標」(河南省著名商标)。2020年金剛碧綠製作技藝被列入信陽市非物質文化遺產 (信阳市非物质文化遗产) 名錄。2023年發佈團體標準 (团体标准, tuántǐ biāozhǔn)，引入官方等級系統。

* **名稱：** 金剛 (金刚) — 「金剛」、「金剛杵」— 指金剛台山 (金刚台)，大別山脈主峰，海拔1584公尺，茶葉生長於其山坡。山名與佛教堅不可摧的象徵意義相關。碧 (碧) — 「翠綠」、「碧玉綠」。綠 (绿) — 「綠色」。因此全名意為「金剛山的翠綠」— 詩意地反映了茶葉的顏色和產地。

* **文化意義：** 金剛碧綠是商城縣茶文化的驕傲，也是更廣泛的信陽地區的驕傲，該地區素以中國十大名茶之一信陽毛尖的故鄉而聞名。商城位於大別山 (大别山, Dàbiéshān) 腹地，河南、安徽兩省交界處 — 這是歷史悠久的茶葉產區，在陸羽《茶經》(《茶经》, Chá Jīng) 中有所記載。此茶象徵著當地茶師在傳統中追求創新的精神：不僅僅是複製著名的毛尖，而是創造全新的東西，結合不同茶葉流派的最佳特點。這種方法 — 融合南方（浙江）扁平優雅與北方（河南）力量和豐滿 — 使金剛碧綠成為河南茶業創新精神的象徵。此茶還與「八最」(八最, bā zuì) 相關：最高海拔、最北緯度、最佳發芽期、最多沖泡次數、最豐富營養成分、最晚上市時間、最鮮明特色、最清爽口感。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要當地栽培種 — 桂花種 (桂花种, Guìhuā zhǒng，字面意思「桂花品種」) — 地方性喬木型群體品種，葉片厚實肉質，多酚含量達25%以上。適合生產濃郁、強勁的綠茶。另外栽培白毫早 (白毫早, Báiháo Zǎo) — 早熟品種，白毫豐富，以及從浙江引進的龍井43 (龙井43, Lóngjǐng 43) — 早發芽，絨毛密集，提升茶湯的鮮爽和明亮度。
* **採摘：** 主要期間 — 春季，品質高峰期在清明–穀雨 (清明–谷雨，4月初–4月底)。由於生長海拔較高，茶葉上市時間比中國東部大多數綠茶晚 — 這是其「八最」之一。
* **採摘標準：** 特級 — 一芽一葉 (一芽一叶, yī yá yī yè)，一級 — 主要為一芽一葉，允許少量一芽二葉，二級 — 一芽二葉。芽葉必須完整，無損傷，剔除病葉、紫葉和粗老葉。

**4. 風土條件與栽培特點：**

* **氣候與地形：** 茶園位於大別山 (大别山) 腹地，金剛台峰及鄰近山峰的山坡上，海拔800–1584公尺。年平均氣溫15.3°C，年降雨量約2200毫米（中國茶區中最高之一）。霧日超過200天，散射光超過70%。這些條件接近形成豐富氨基酸和香氣複合物的理想條件。
* **生長海拔：** 海拔800–1584公尺。金剛台（1584公尺）是整個大別山脈的主峰。
* **土壤：** 基於火山岩的微酸性紅壤 (微酸性红壤, wēi suānxìng hóng rǎng)，pH 4.5–6.0，有機質含量≥15 g/kg。土壤富含鋅和硒。水源來自山泉溪流，水質符合國家一級標準。
* **栽培特點：** 森林覆蓋率達89% — 極高的指標。茶園與杜鵑花 (杜鹃)、野生蘭草 (兰草) 和其他山花共生，豐富了茶葉的香氣特徵。完全禁用化學農藥；使用瓢蟲 (瓢虫, piáochóng) 和其他有益昆蟲進行生物防治。

**5. 製作工藝：**

金剛碧綠工藝是有目的性地融合龍井和信陽毛尖傳統技法的結果，代表了獨特的多步驟過程，具有三次成形和雙重炭火乾燥。

1. **攤放 (鲜叶摊放, xiānyè tānfàng)：** 鮮葉薄攤3–4小時，直至含水量約70%。此階段確保香氣前驅物質的初步發展。
2. **殺青 (杀青, shāqīng)：** 在斜鍋 (斜锅, xié guō) 中於140–160°C進行，交替使用抖 (抖, dǒu) 和悶 (闷, mèn) 技法。組合確保酶活性均勻失活和保持鮮綠色澤。
3. **炒條I — 「炒條」(炒条, chǎotiáo)：** 在80°C下去除殘餘黏性，葉片開始成形。
4. **理條II — 「理條」(理条, lǐtiáo)：** 手工處理使其挺直均勻。
5. **整形III — 「最終整形」(整形, zhěngxíng)：** 最終手工成形，運用抓 (抓, zhuā)、搭 (搭, dā) 和壓 (压, yā) 技法。正是這種「三次」成形技術 (三次成形, sān cì chéngxíng) — 炒條、理條、整形，在三個分別的鍋中進行 — 確保了特有的扁平、挺直、優雅的葉形。
6. **初烘 (初烘, chū hōng)：** 在80°C炭火上進行。炭火避免焦糊味並「封鎖」蘭花香。
7. **復烘 (复烘, fù hōng)：** 在50–60°C下最終穩定含水量並增強香氣特徵。
8. **揀剔篩分 (拣剔筛分, jiǎntī shāifēn)：** 手工去除不合格片段並按規格分級。

「炭火鎖香」(炭火锁香, tànhuǒ suǒ xiāng) 原則 — 在兩個乾燥階段都使用木炭 — 是金剛碧綠的關鍵工藝特色，區別於大多數現代電烘綠茶。

**6. 感官特徵：**

* **乾茶外形：** 扁直挺秀 (扁直挺秀, biǎnzhí tǐngxiù)，肉質緊實，顯毫明顯 (显毫, xiǎn háo)。外形似龍井，但更加肉厚，絨毛豐富 — 「形似龍井，神似毛尖」。
* **乾茶香氣：** 清純高長持久，首現蘭花香 (兰花香, lánhuā xiāng)。
* **茶湯香氣：** 多層次，在連續沖泡中展開：第一泡 — 蘭花香 (兰花香)；第二泡 — 嫩栗香 (嫩栗香, nèn lì xiāng)；第三泡 — 濃香 (浓香, nóng xiāng)；第四泡 — 清香 (清香, qīng xiāng)。冷杯香持續5分鐘以上。
* **滋味：** 鮮爽 (鲜爽, xiānshuǎng)，甘醇 (甘醇, gānchún)。多酚基礎（≥25%）確保緊密的滋味結構，帶有輕微、快速消散的澀味，轉為明顯持久的回甘 (回甘, huígān)。可溶性糖類創造和諧協調。滋味具有明顯層次感 — 綠茶中的稀有品質。
* **湯色：** 嫩綠清亮 (嫩绿清亮, nèn lǜ qīngliàng)。
* **葉底（沖泡後葉片）：** 嫩綠肉厚，富有彈性，呈現特有的「勻整成朵」(匀整成朵, yúnzhěng chéng duǒ)，證明原料品質高和加工精細。

**7. 化學成分：**

* **多酚類 (茶多酚, chá duōfēn)：** 25–28% — 中國名優綠茶中最高值之一。主要由兒茶素（EGCG、ECG等）構成，提供強大抗氧化效果。
* **氨基酸 (氨基酸, ānjīsuān)：** 約4.9%，包括L-茶氨酸。高含量歸因於散射光豐富和多霧的山地風土。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēi jiǎn) — 約3%，可可鹼，茶鹼。
* **水浸出物：** 高，證明豐富的浸出特徵。
* **氟 (氟, fú)：** 10–15 mg/100 g — 相對高含量，有利於牙齒健康。
* **維生素：** C（鮮葉中）、B群、K、E。
* **礦物質：** 鋅、硒、鉀、錳、磷和其他微量元素，由火山土壤來源決定。
* **精油：** 形成特有的蘭花–栗香香氣特徵；其積累得益於茶樹與山花的共生。

**8. 保健功效：**

* **強大抗氧化保護：** 極高的多酚含量（25–28%）提供有效的自由基中和和細胞氧化應激保護。
* **提神效果：** 咖啡鹼（3%）與高含量L-茶氨酸（4.9%氨基酸）結合，提供平穩、持久的提神效果，同時帶來清晰和專注感 — 無急劇波動。
* **支持牙齒和骨骼健康：** 較高的氟含量（10–15 mg/100 g）有利於牙釉質礦化。
* **支持脂質代謝：** 兒茶素積極抑制脂肪吸收和合成，有助於維持健康膽固醇水平。
* **改善消化：** 茶葉在餐後飲用時刺激胃腸道功能。
* **認知支持：** 咖啡鹼和L-茶氨酸的協同作用改善專注力、記憶力和反應速度。
* **整體強化作用：** 維生素、礦物質和生物活性多酚複合物有助於維持免疫力和整體活力。

**9. 沖泡方法：**

* **水溫：** 80–85°C。建議將水煮沸後冷卻約90秒。使用中性或弱酸性礦泉水（鹼性水會影響口感）。
* **茶量：** 3克配150毫升水（1:50比例）。
* **茶具：** 玻璃杯 (玻璃杯, bōli bēi) — 觀察「茶舞」；白瓷蓋碗 (白瓷盖碗, báicí gàiwǎn) — 充分體驗香氣。
* **過程：**
  1. 用熱水溫杯。
  2. 投茶。
  3. 注水至三分之一，搖杯「醒香」(摇香, yáo xiāng)。
  4. 續水至七分滿。
  5. 第一泡浸泡2–3分鐘。
  6. 茶葉可重複沖泡3次，逐漸延長浸泡時間。

**10. 儲存：**

* 密封包裝，完全避光、防潮、防異味。
* 最佳溫度：0–5°C（冰箱）。開封前需在密封包裝中回溫至室溫。
* 保質期 — 在適當儲存條件下12個月。開封後建議儘快飲用。
* 茶葉天敵：潮濕、直射陽光、異味、高溫。

需要記住，高多酚含量的茶不建議餐後立即飲用（單寧可能結合蛋白質並妨礙消化）；最佳間隔約一小時。建議日飲量不超過500毫升。服用鐵劑和抗生素時應間隔至少一小時。明顯貧血者建議適量飲用。

**11. 價格與仿冒：**

* **價格範圍：** 特級 (特级, tèjí) — 完全一芽一葉，90%芽頭比例 — 約550元人民幣/斤（500克）。一級 — 約350元/斤。二級 — 約260元/斤。
* **價格因素：** 採摘季節（早春顯著更貴）、生長海拔、原料等級（芽頭比例）、手工製作、炭火乾燥。金剛台山茶因地處偏遠和成熟較晚，客觀上比平地同類產品昂貴。
* **如何避免仿冒：**
  * 從商城縣可靠商家購買帶有註冊商標「金剛碧綠」的茶葉。
  * 注意外形：真正的金剛碧綠扁直挺秀，顯毫明顯。仿品常有不規則或捲曲外形。
  * 評估香氣：正品具有多層次展開 — 從蘭花香到栗香。
  * 檢查茶湯：應清澈、嫩綠，無渾濁。
  * 聲稱特級但價格可疑偏低 — 可能是平地廉價原料冒充的跡象。

**12. 有趣事實：**

* 金剛碧綠是有意識的創作項目：茶師楊智棟花費五年時間（1984–1989）開發工藝，力求結合龍井的南方優雅與毛尖的北方力量。結果是一款被稱為「外形神似龍井，內質兼具毛尖之神韻」(外形神似龙井，内质兼具毛尖之神韵) 的茶。
* 金剛台山（1584公尺）不僅是大別山脈主峰，也是重要的革命歷史紀念地：內戰期間紅軍基地設在此處。茶園與歷史紀念館毗鄰。
* 金剛碧綠的「八最」(八最) — 這款茶的行銷和文化標語：最高生長海拔、該地區名優綠茶中最北緯度、最佳發芽期、最多沖泡次數、最豐富營養成分、最晚上市時間、最鮮明特色、最清爽口感。
* 多酚含量（25–28%）是中國名優綠茶中最高之一，這歸因於高山風土與當地喬木型「桂花」栽培種厚實葉片的結合。
* 金剛台茶園位於最高森林覆蓋率（89%）和獨特生物多樣性區域：茶樹與野生杜鵑花和山蘭花混生，形成獨特的香氣特徵「花香背景」。

**13. 與其他綠茶比較：**

* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 來自同一信陽市的經典名優綠茶，屬中國十大名茶。以細針狀外形、豐富絨毛、嫩栗香和明顯鮮味著稱。金剛碧綠具有扁平而非針狀外形，更複雜的多層香氣（從蘭花香到栗香）和更高的多酚含量，提供更緊密的茶體和明顯的層次感。
* **西湖龍井 (西湖龙井, Xīhú Lóngjǐng)：** 浙江著名扁形綠茶。金剛碧綠在扁平外形上形式相似，但更加肉厚，顯毫明顯（龍井幾乎無絨毛），具有多層蘭花–栗香（對比龍井的「豆香」）和顯著更高的多酚含量。
* **六安瓜片 (六安瓜片, Liù'ān Guāpiàn)：** 來自鄰近安徽省的著名綠茶，同樣生長在大別山，但在另一側。以獨特的純葉片無芽工藝、扁平「瓜子形」外形和更緊密厚重的滋味著稱。金剛碧綠相反，由嫩芽和葉片製成，具有更鮮爽明亮的滋味和蘭花香。
* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 安徽大葉扁形綠茶。葉片比金剛碧綠大得多，香氣特徵不同（蘭花香存在，但有更多「礦物」基調）。金剛碧綠更緊湊、肉厚，多酚含量更高。

**結論：**

金剛碧綠是茶葉傳統中創新探索能夠產生真正獨特作品的絕佳例證。誕生於大別山最高峰的山坡，被杜鵑花和山蘭花環繞，採用「三次成形」技法在炭火上乾燥，這款茶呈現了優雅扁平外形、多層香氣（從蘭花到栗香）和強勁多酚潛力的罕見結合，賦予滋味深度和層次感。金剛碧綠非常適合尋求經典範疇之外綠茶的愛好者：一款具有個性、歷史和大別山真正山地精神的茶葉。
