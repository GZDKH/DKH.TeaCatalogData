# 香山貢茶 (香山贡茶, Xiāngshān gòngchá)

**香山貢茶 (香山贡茶, Xiāngshān gòngchá)**

![Сяншань Гун Ча (香山贡茶, Xiāngshān gòngchá)](images/Сяншань%20Гун%20Ча%20(香山贡茶,%20Xiāngshān%20Gòngchá).png)

![Сяншань Гун Ча (香山贡茶, Xiāngshān gòngchá)-1](images/Сяншань%20Гун%20Ча%20(香山贡茶,%20Xiāngshān%20Gòngchá)-1.png)

香山貢茶 (香山贡茶, Xiāngshān gòngchá) — 歷史綠茶，產自奉節縣 (奉节县, Fèngjié Xiàn)，位於長江三峽 (长江三峡) 偉大入口處，傳奇白帝城 (白帝城, Báidìchéng) 腳下。奉節為古夔州 (夔州, Kuízhōu)，「三峽門戶」，杜甫 (杜甫) 在此寫下四百餘首詩歌之地，瞿塘峽 (瞿塘峡, Qútáng Xiá) 石崖裂縫「長江咽喉」印於十元人民幣上。唐代時夔州茶列入貢茶 (贡茶) 之列，李肇《國史補》(《国史补》) 中提及「夔州香雨」（「芳香雨夔州」）。陸羽《茶經》(《茶经》) 將夔州納入產茶州縣名錄。經多世紀遺忘後，此茶於1991年復興，至2023年兩度榮獲「中茶杯金獎」(中茶杯金奖)。現代工藝包含28道生產工序，招牌特色為「栗香」(栗香)，由最終乾燥階段精確「火工」(火工, huǒ gōng) 形成。

**1. 分類與產地：**

* **類型：** 綠茶（不發酵，綠茶, lǜchá）。殺青 — 滾筒炒製160–180 °C (滚筒杀青)。最終乾燥 — 組合式：初烘90–100 °C → 手工整形 → 最終提香80–90 °C → 「猛火」(猛火, měng huǒ) 110 °C持續3秒。

* **類別：** 歷史名茶 (历史名茶)。唐代貢茶 — 《新唐書·地理志》(《新唐书·地理志》) 所列夔州雲安郡約20種貢茶之一。全國農業博覽會金獎（1995）。「中茶杯金獎」(中茶杯金奖, 2023)。重慶鬥茶大賽金獎 (重庆斗茶大赛金奖, 2023)。生產分兩種香型：「栗香型」(栗香型, lì xiāng xíng, 主要) 和「清香型」(清香型, qīng xiāng xíng, 較輕火工)。

* **產地：** 中國；直轄市重慶 (重庆市, Chóngqìng Shì)；奉節縣 (奉节县, Fèngjié Xiàn)，古夔州 (夔州)。品質核心 — 白帝鎮 (白帝镇, Báidì Zhèn)，香山寺 (香山寺, Xiāngshān Sì, 「芳香山寺」) 區域，海拔500–900公尺 — 「雲霧帶」。另有新民鎮 (新民镇, Xīnmín Zhèn) 河谷茶園。

* **地理座標：** ~31°00′–31°30′ 北緯，109°00′–109°40′ 東經（奉節境內，三峽咽喉）。


**2. 歷史與文化意義：**

* **歷史：**

夔州 (夔州) — 中國最古老產茶州之一，陸羽《茶經》(《茶经》, 八世紀) 列入產茶地區。《新唐書·地理志》(《新唐书·地理志》) 中夔州雲安郡 (夔州云安郡) 列為帝國茶貢地。唐史學家李肇 (李肇)《國史補》(《国史补》) 列舉著名貢茶，其中包括「夔州香雨」（「夔州芳香雨」），產於香山附近白帝城。

宋代「賓化早春」(「宾化早春」，「賓化早春[茶飲]」) 產自奉節，在京城享有盛名 — 《建炎雜記》(《建炎杂记》) 有記載。「賓化」為奉節古地名之一，「早春」指當地茶葉早熟特性，源於峽谷溫暖微氣候。明代黃一正 (黄一正)《事物紺珠》(《事物绀珠·茶类》) 記錄「夔州香山茶」— 「夔州芳香山茶」，確立香山與茶傳統之聯繫。因此，香山貢茶擁有超過千年連續文獻「族譜」：唐（李肇、陸羽）→ 宋（建炎雜記）→ 明（黃一正）。

清末及二十世紀戰爭期間工藝失傳。1991年開始復興：奉節當局結合歷史工法片段與現代標準，恢復生產。1995年榮獲全國農業博覽會金獎 (中国农博会金奖)。2023年雙重勝利：「中茶杯金獎」(中茶杯金奖) 和重慶鬥茶大賽金獎 (重庆斗茶大赛金奖)。

* **名稱：** 香山 (Xiāngshān) — 「芳香山」— 奉節附近山峰，山坡上有古香山寺；貢茶 (Gòngchá) — 「貢茶」，「進獻皇帝之茶」。完整含義：「芳香山貢茶」。

* **文化意義：** 奉節為詩歌與歷史高度集中之地。此處白帝城 (白帝城) 中，劉備 (刘备) 將繼承人託付諸葛亮 (诸葛亮) — 《三國演義》最戲劇性場景之一。杜甫亦在此度過兩年（766–768），寫下四百餘首詩 — 約佔全部遺作三分之一，包括著名《秋興八首》(秋兴八首)。瞿塘峽 — 三峽入口 — 印於十元人民幣上。李白 (李白) 寫下著名《早發白帝城》(早发白帝城)：「朝辭白帝彩雲間，千裡江陵一日還」(「朝辞白帝彩云间，千里江陵一日还」) — 「晨別白帝彩雲間，千裡江陵一日歸」。香山茶為此文化脈絡延續：誕生於啟發中國最偉大詩人同一山峰之飲品。白帝城在三峽大壩建成後水位上升變為島嶼 — 現矗立江心島上，更添浪漫色彩。


**3. 植物學描述與原料：**

* **品種/栽培種：** 主要 — 福鼎大毫茶 (福鼎大毫茶) 和福鼎大白茶 (福鼎大白茶) — 佔種植70%。兩者均為早熟無性系品種，茸毛豐富，氨基酸含量高，原產福建，但完美適應峽谷山地氣候。正是福鼎大毫豐富茸毛 (毫, háo) 造就特有「銀綠色」乾葉色澤。另有四川中小葉群體種 (四川中小叶群体种) — 當地原生灌木，適應「峽谷門戶」特殊微氣候，及名山早 (名山早, Míngshān Zǎo) — 四川早熟品種，可比清明提前5–7天採摘。

* **採摘：** 春茶 — 清明 (清明, 四月初) 前後。標準 — 一芽一葉初展 (一芽一叶初展)。禁採紫色、病蟲害、受損芽葉。

* **原料要求：** 嫩勻一致，無缺陷嫩芽。


**4. 風土與栽培特色：**

奉節位於重慶東部，瞿塘峽西入口 — 「長江咽喉」。三峽山地地形造就獨特微氣候。

* **生長海拔：** 500–900公尺 — 「雲霧帶」(云雾带, yúnwù dài)。核心 — 香山面向長江山坡。

* **氣候：** 中亞熱帶。年均溫15.2 °C；濕度≥75%；霧日45–52天/年；日夜溫差顯著 — 深峽谷結果，夜間江面冷空氣沿坡上升，日間下沉。此「呼吸」節律刺激氨基酸積累。

* **土壤：** 暗紫泥土 (暗紫泥土, àn zǐ ní tǔ) 和冷沙黃泥土 (冷沙黄泥土, lěng shā huáng ní tǔ)，pH 4.0–6.5。深厚土層，富含有機質。紫色土 — 四川重慶盆地特色土壤類型，富鉀及微量元素。

* **生態：** 茶園按生態原則管理：禁用化肥農藥。長江鄰近提供持續濕潤和溫度極值緩解。面江山坡接受水面反射光，總光照增加10–15%，同時保持高散射光比例。三峽大壩 (三峡大坝, 2006) 建成後水位上升，增強奉節地區濕度和霧氣 — 據當地茶農觀察，改善茶葉種植條件。部分低地種植園被淹沒，但高海拔茶園（500–900公尺）未受影響，反因雲霧增加而受益。


**5. 製作工藝：**

現代香山貢茶工藝包含28道工序 (非遗28道工序)，結合復原歷史工法與電氣設備，避免煙味。

* **攤放 (摊放 — tān fàng)：** 4–8小時 — 較一般更長，發展香氣前驅物。

* **殺青 (杀青 — shāqīng)：** 滾筒炒製 (滚筒杀青) 160–180 °C。

* **揉捻 (揉捻 — róuniǎn)：** 手工滾揉 (手工滚揉, shǒugōng gǔnróu)，約20分鐘。形成緊直條索。

* **初烘 (初烘 — chū hōng)：** 90–100 °C。

* **整形 (整形 — zhěng xíng)：** 「手工搓條定形」(手工搓条定形, shǒugōng cuō tiáo dìng xíng) — 形成特色「緊秀勻直」關鍵步驟 (紧秀匀直)。

* **揀剔 (拣剔 — jiǎn tī)：** 去除不合格部分。

* **足火提香 (足火提香 — zú huǒ tí xiāng)：** 主烘80–90 °C → 最終「猛火」(猛火, měng huǒ) 110 °C持續3秒。此瞬間高溫「衝擊」為「栗香」(栗香) 秘訣。火候控制至關重要：多1–2秒香氣轉為「焦糊」。

* **註：** 電氣設備取代傳統木炭 — 避免外來煙味，保持「栗香」純淨。


**6. 感官特徵：**

* **乾茶外形：** 緊秀勻直條索 (条索紧秀匀直)，鋒苗顯露 (锋苗显露)。色澤「銀綠隱翠」(银绿隐翠, yín lǜ yǐn cuì)。

* **乾茶香氣：** 栗香 (栗香, lì xiāng) — 主導，高揚持久。最終「火工」精確結果。

* **茶湯香氣：** 栗香綠香，濃郁持久。放涼後輕微「麵包」香調。

* **滋味：** 鮮爽 (鲜爽) 同時甘甜清純 (回甘, huígān) — 持久回甘餘味。鮮爽與「厚度」平衡。

* **湯色：** 嫩綠清澈 (嫩绿清澈)，明亮。

* **葉底（沖泡後葉片）：** 黃綠明亮勻整 (黄绿明亮匀整)。


**7. 化學成分：**

* **多酚類 (茶多酚)：** 高含量 — 提供強抗氧化活性。據資料，自由基清除效率為維生素E的18倍。

* **氨基酸 (氨基酸)：** 良好含量 — 提供顯著鮮爽感 (鲜爽)。L-茶氨酸佔主導。峽谷高日夜溫差刺激氨基酸作為抗凍劑積累。

* **咖啡鹼 (咖啡碱)：** 偏高 — 提供顯著提神效果，據資料比綠茶平均高30%。

* **兒茶素 (儿茶素)：** 包括EGCG — 主要多酚成分，負責抗氧化活性和脂質代謝刺激。

* **維生素：** 維生素C、B群維生素、維生素E。

* **礦物質：** 鉀、錳、氟；紫色土微量元素。


**8. 保健功效：**

* **抗氧化保護：** 多酚類 + 維生素C — 全面自由基清除。

* **提神效果：** 高咖啡鹼與L-茶氨酸協同 — 溫和持久提神，無緊張感。

* **消化支持：** 兒茶素刺激胃腸蠕動，加速脂肪分解。

* **認知功能：** L-茶氨酸刺激大腦α波活動。

* **重要提醒：** 所列功效基於一般資料，非醫療建議。不建議空腹飲用。新茶需放置7–15天「退火」。開封後密封保存。


**9. 沖泡方法：**

* **水溫：** 85–90 °C。勿用沸水（>90 °C破壞鮮爽，產生苦澀）。

* **茶量：** 3克配150毫升（1:50比例）。

* **茶具：** 白瓷蓋碗 (白瓷盖碗) — 集中栗香。玻璃杯 — 觀察「芽葉豎立」(芽叶竖立)。

* **流程（蓋碗）：** 洗茶5秒 → 首泡1–2分鐘 → 每泡增加30秒。

* **流程（玻璃杯）：** 80–85 °C水溫沖泡，靜置2分鐘。觀察「芽葉豎立」— 銀綠條索帶白毫垂直緩降，營造「翠玉林立水晶中」效果。可沖3–5泡；栗香在首、二泡最為明顯。


**10. 儲存：**

* **溫度：** 0–5 °C（冰箱），密封。
* **光線：** 完全避光。
* **期限：** 建議6個月內。新茶需放置7–15天「醒茶」退火。


**11. 價格與仿冒：**

香山貢茶為中高端及高端價位茶品。特級 — 800元/斤起；一級400–700元/斤；二級200–400元/斤。

* **避免仿冒方法：**

	* 產地 — 重慶奉節。
	* 栗香 (栗香) — 110 °C「猛火」招牌。缺乏為替代品徵象。
	* 外形 — 緊秀勻直條索，鋒苗顯露 (锋苗显露)。捲曲或無形為非香山。
	* 色澤 — 「銀綠隱翠」(银绿隐翠)。暗淡或黃色為工藝問題。


**12. 趣聞軼事：**

* **十元鈔票上的茶：** 瞿塘峽 (瞿塘峡) 石崖裂縫「長江咽喉」印於十元人民幣背面，距香山茶園僅數公裡。茶葉字面意義「生長在鈔票上」。

* **杜甫與茶：** 偉大詩人在夔州度過兩年（766–768），寫下>400首詩。數首提及當地茶與鹽 — 夔州兩大特產。因此，香山貢茶為少數擁有中國最偉大詩人詩歌「族譜」的茶品之一。

* **白帝與茶：** 白帝城 (白帝城) — 西元223年垂死劉備將繼承人託付諸葛亮之堡壘。此場景為中國文化最知名情節之一。香山茶園位於與堡壘相同山坡 — 俯瞰歷史之茶。

* **28道工序：** 現代香山貢茶工藝包含28道連續生產工序 — 綠茶中最長「茶製程表」之一。比較：標準綠茶經5–8道工序。

* **「猛火」— 110 °C持續3秒：** 最終「猛火」(mēnghuo, 「烈火」) 恰好持續3秒，形成「栗香」。多一兩秒香氣轉「焦糊」。此為中國製茶最精確「火候」技法之一。

* **三峽大壩與茶：** 世界最大水電站「三峽大壩」(三峡大坝, 2006) 建成後，奉節地區水位上升數十公尺。增大水面加強蒸發和霧氣 — 據觀察，改善500–900公尺茶園微氣候。部分低地種植園被淹沒，但高海拔茶園（500–900公尺）未受損，反因雲霧增加而受益。


**13. 與其他重慶四川綠茶比較：**

* **永川秀芽 (永川秀芽, Yǒngchuān Xiùyá)：** 重慶。扁平捲曲形，栗綠香型。較「大眾」平價。香山 — 歷史更深（唐 vs 二十世紀）、更「直」形狀 (紧秀匀直)。

* **蒙頂甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 四川。中國最古老茶品之一（唐代「第一」，據李肇：「蒙頂石花，號為第一」）。形狀 — 捲曲螺旋；香氣 — 「甘露」(甘露)，顯著花香。香山 — 直條，「猛火」栗香型。兩者均為同一地區（四川重慶）唐代貢茶，但來自不同氣候帶：蒙頂 — 1000+公尺，濕潤亞熱帶；香山 — 500–900公尺，峽谷微氣候。蒙頂知名度顯著更高；香山 — 小眾，但具獨特「峽谷」風土。

* **竹葉青 (竹叶青, Zhúyèqīng)：** 四川峨眉山。扁平炒製。清淡「竹綠」香型。香山 — 因「猛火」更「厚重」「栗香」。


**結語：**

香山貢茶 — 來自「三峽門戶」之茶：茶園俯瞰印於十元鈔票瞿塘峽石崖，歷史始於唐代貢茶「夔州香雨」，鄰居為白帝城（劉備託孤處）和杜甫寫下四百首詩之江岸。二十世紀戰爭中失傳，1991年復興，帶著28階段工藝和「猛火」— 110 °C三秒衝擊，將普通綠茶轉化為栗香交響曲。85 °C白瓷蓋碗沖泡 — 讓芳香山香氣為您講述詩人、帝王和在茶園腳下流淌十三世紀之大江的故事。
