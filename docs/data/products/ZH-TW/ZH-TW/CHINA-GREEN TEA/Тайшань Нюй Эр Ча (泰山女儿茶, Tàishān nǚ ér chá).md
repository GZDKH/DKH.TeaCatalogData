# 泰山女兒茶 (泰山女儿茶, Tàishān nǚ ér chá)

**泰山女兒茶 (泰山女儿茶, Tàishān nǚ ér chá)**
![Тайшань Нюй Эр Ча (泰山女儿茶, Tàishān nǚ ér chá)](images/Тайшань%20Нюй%20Эр%20Ча%20%28泰山女儿茶,%20Tàishān%20Nǚ%20Ér%20Chá%29.png)

![Тайшань Нюй Эр Ча (泰山女儿茶, Tàishān nǚ ér chá)-1](images/Тайшань%20Нюй%20Эр%20Ча%20%28泰山女儿茶,%20Tàishān%20Nǚ%20Ér%20Chá%29-1.png)

泰山女兒茶是中國茶葉帶極北地區為數不多的綠茶之一，生長於神聖的泰山腳下。其名稱字面意思為「泰山的女兒茶」，根源可追溯至明代的文學傳統，儘管現代用茶樹葉片製作的茶葉僅從1960年代開始在此地生產。這款茶的招牌特色是濃郁的栗香，因此獲得了「茶中板栗」(茶中板栗, cházhōng bǎnlì) 的民間暱稱。

**1. 分類與產地:**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。屬於炒青 (炒青, chǎoqīng) 子類別——以鍋炒殺青的茶葉。
* **類別：** 山東地方名茶 (山东名茶, Shāndōng míngchá)。中國國家地理標誌產品 (中国国家地理标志产品, Zhōngguó guójiā dìlǐ biāozhì chǎnpǐn)。
* **產地：** 中國山東省 (山东省, Shāndōng shěng) 泰安市 (泰安市, Tài'ān shì)。茶園位於泰山 (泰山, Tàishān) 東坡及山腳下，在泰山國家公園風景區範圍內。
* **地理座標：** 約北緯36.25°，東經117.10°（泰山東坡，茶園種植區）。

**2. 歷史與文化意義:**

**歷史：**

泰山女兒茶有著不尋常的雙重歷史。最初「女兒茶」(女儿茶) 在植物學意義上並非茶葉——這是用青桐 (青桐, qīngtóng) 樹芽製成的飲品，青桐又稱鼠李 (鼠李, shǔlǐ)，生長在泰山山中。這個名稱的首次文字記載可追溯至明代 (明, Míng)：李日華 (李日华, Lǐ Rìhuá, 1565–1635) 在其文集《紫桃軒雜綴》(紫桃轩杂缀, Zǐtáoxuān Zázhuì) 中記錄：「泰山人採青桐芽，製成飲品，稱之為女兒茶」。在縣志《重修泰安縣志》(重修泰安县志, Chóngxiū Tài'ān Xiànzhì) 中也記載，當地居民採集青桐嫩芽，認為是南方茶葉的優質替代品。

在十八世紀曹雪芹 (曹雪芹, Cáo Xuěqín) 的經典小說《紅樓夢》(红楼梦, Hónglóu Mèng) 第六十三回中，賈寶玉 (贾宝玉) 酒後飲用的正是女兒茶，這證明了這種飲品在清代文人階層中的崇高地位。

現代以茶樹 (*Camellia sinensis*) 製茶的歷史始於1960年代。1930年代，馮玉祥 (冯玉祥, Féng Yùxiáng) 將軍隱居泰山時，曾嘗試將南方茶樹移植到山上，但未能成功。真正的突破發生在1966年，作為全國「南茶北引」(南茶北引, nánchá běiyǐn) 計畫的一部分，泰安建立了第一批茶樹種植園。到2000年，茶園面積達到1000多畝（約67公頃），目前已擴展到5000畝（約333公頃）。

2007年，種植園獲得「國家級無公害茶葉種植加工標準化示範區」(国家级无公害茶叶种植加工标准化示范区) 地位。2012年，「泰山女兒茶」品牌被列入首批「好品山東」(好品山东) 名錄。該茶多次在中國國際農業博覽會上獲得金獎，並獲得「山東名牌產品」(山东名牌产品) 稱號。

**名稱：**

* **泰山** (泰山, Tàishān) — 神聖的泰山，中國五嶽 (五岳, Wǔyuè) 之一，聯合國教科文組織世界遺產（自然與文化遺產）。
* **女兒** (女儿, nǚ'ér) — 字面意思為「女兒、少女」。根據流行傳說，乾隆皇帝 (乾隆, Qiánlóng) 遊覽泰山時想品嚐當地茶葉；由於沒有茶樹，官員派遣年輕女子採集青桐芽，用泰山泉水沖泡後獻給皇帝，稱這種飲品為「女兒茶」。
* **茶** (茶, chá) — 茶。

**文化意義：**

泰山女兒茶與神聖山嶽文化密不可分。泰山是五嶽之首，兩千多年來一直是皇帝舉行封禪 (封禅, fēngshàn) 祭祀的聖地。泰山的茶文化傳統是中國最年輕的之一，但建立在深厚的文學和文化基礎上。著名的「四槐樹」(四槐树) 茶館位於登山路上的回馬嶺 (回马岭, Huímǎlǐng) 山口，已成為朝聖者和遊客的聖地。現代生產商開發了產品線：除了經典綠茶外，還生產紅茶 (泰山女儿红茶) 和「金茶」(泰山女儿黄金茶)，這些產品是與山東農業大學茶學系 (山东农业大学茶学系) 合作開發的。

**3. 植物學描述與原料:**

* **品種：** *Camellia sinensis* var. *sinensis*。
* **栽培品種：** 使用國家級茶樹良種 (国家级茶树良种, guójiā jí cháshù liángzhǒng)，專門為北方條件選育。基礎種植包括適應寒冷的小葉品種。也有當地群體種 (群体种, qúntǐ zhǒng) 的種植區域。
* **採摘：** 春茶 (春茶, chūnchá) 是主要且最珍貴的。由於北方位置，茶季較短：茶樹休眠期比南方同類更長，有利於氨基酸和芳香化合物的積累。採摘開始時間比南方省份晚——通常從四月中旬開始。
* **採摘標準：** 主要是一芽一葉或一芽二葉 (一芽一叶或一芽二叶)。頂級批次僅採用嫩頂芽。
* **原料特點：** 葉片明顯比南方綠茶更厚實——這是在日夜溫差大的條件下緩慢生長的結果。這確保了高萃取性和多次沖泡的耐久性。

**4. 風土條件與栽培特色:**

* **海拔高度：** 海拔200–600公尺（泰山東坡及山麓地區）。
* **氣候：** 溫帶季風氣候，大陸性特徵。年平均氣溫約12.9°C（主要茶區）。年降水量約697毫米。無霜期約195天。日夜溫差顯著——這是減緩生長並增強氨基酸積累的關鍵因素。
* **微氣候：** 經常有霧，特別是在清晨時分。山地地形創造自然遮蔭，增加散射光比例——這是合成L-茶氨酸和葉綠素的有利因素。
* **土壤：** 棕色森林土和山地草甸土，弱酸性（pH 5.0–6.5），富含有機質。泰山山坡被森林覆蓋（森林覆蓋率高），確保空氣清潔和土壤肥沃。該地區屬於水源保護區，化學肥料和農藥的使用受到嚴格限制。
* **農業技術：** 種植園按照無公害和有機茶葉標準 (无公害/有机, wúgōnghài/yǒujī) 經營。與山東農業大學的合作提供科學支持：選育抗寒品系、優化修剪、監測植物健康狀況。

**5. 製作工藝:**

泰山女兒茶採用經典炒青綠茶 (炒青绿茶, chǎoqīng lǜchá) 工藝製作，重點在於形成特有的栗香和濃郁但不粗糙的口感。

* **採摘 (采摘, cǎizhāi)：** 清晨手工採摘嫩芽。原料迅速運送到工廠。
* **攤晾 (摊晾, tānliàng)：** 新鮮採摘的葉片在陰涼通風的房間內薄攤2–4小時，以平衡水分並開始產生揮發性芳香化合物。
* **殺青 (杀青, shāqīng)：** 在鍋中（或機械滾筒中）高溫炒製。這是關鍵步驟，停止酶促氧化並奠定栗香基礎。溫度和時間根據葉片濕度和成熟度調整。
* **揉捻 (揉捻, róuniǎn)：** 輕度或中度壓力——足以破壞細胞壁並釋放汁液，但不過度損傷葉片。持續時間10–20分鐘。
* **做形 (做形, zuòxíng)：** 賦予葉片特有的彎曲形狀 (曲卷形, qūjuǎn xíng)。成品茶被描述為「美麗彎曲」——沖泡時葉片沉到底部並展開，像舞蹈的少女。也生產「龍形」(龙形, lóngxíng) ——扁平形狀，以龍井工藝為靈感創造。
* **烘乾 (烘干, hōnggān)：** 在適中溫度下最終乾燥至穩定濕度（≤6.5%）。固定香氣並定型。

**6. 感官特徵:**

* **乾茶外觀：** 優雅彎曲、緊密捲曲的葉片，有明顯的白毫覆蓋。顏色從深綠到橄欖綠，帶有油潤光澤。葉片明顯比典型南方綠茶更厚實。「龍形」版本——扁平、光滑、嫩綠色。
* **乾茶香氣：** 濃郁的栗香 (栗香, lìxiāng) ——茶的主要香氣特徵。層次中有烤堅果、淡淡的烤栗子甜味、新鮮綠葉底調。
* **茶湯香氣：** 純淨、高揚，以栗香為主導。隨著冷卻展現溫和的花香和草本調。香氣持久悠長。
* **口感：** 飽滿、濃郁，有明顯的茶體，但不粗糙。甘甜，帶有平衡的清新感。在正確水溫下苦味極少。特有回甘 (回甘, huígān) ——飲後漸增的甜味。口感逐漸展開：前幾泡清淡，第3–4泡達到濃郁峰值，之後口感平緩消退——這是綠茶中不尋常的特點。
* **湯色：** 透明，從嫩綠到黃綠色。在正確溫度下——清澈明亮，帶有淡淡的珠光光澤。
* **葉底 (叶底, yèdǐ)：** 葉片均勻展開，顏色嫩綠、一致。葉片有彈性、厚實、「肉質」——北方產地的明顯證據。

**7. 化學成分:**

泰山女兒茶的化學成分反映了北方風土特色：長期休眠和緩慢的春季生長促進氨基酸積累增加和兒茶素含量適中。

* **多酚類 (茶多酚, chá duōfēn)：** 綠茶中含量適中——約佔乾重的18–24%。主要兒茶素：EGCG、ECG、EC、EGC。與南方同類相比多酚含量相對較低，解釋了其溫和性和缺乏明顯澀味。
* **氨基酸 (氨基酸, ānjīsuān)：** 含量較高——這是顯著特徵之一。L-茶氨酸 (L-茶氨酸) 提供特有的甜味和鮮味底調。高氨基酸水平是短生長期內長期冬季積累的結果。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēijiǎn) ——含量為綠茶典型水平（佔乾重2.5–3.5%）。含有微量可可鹼和茶鹼。
* **水溶性浸出物 (水浸出物, shuǐ jìnchū wù)：** 不少於45%——高指標，證實了茶湯的濃郁和飽滿。
* **維生素：** 維生素C（抗壞血酸）、B群維生素（B₁、B₂）、維生素E。
* **礦物質和微量元素：** 富含鉀（K）、鋅（Zn）、鐵（Fe）、錳（Mn）、鈉（Na）和硒（Se）。微量元素含量較高——生產商認為這是泰山風土的特色。
* **精油：** 主導芳香成分——吡嗪和呋喃化合物，負責炒製（殺青）時形成的栗香。

**8. 健康功效:**

* **提神作用：** 咖啡鹼與L-茶氨酸結合提供溫和、平穩的提神效果，無劇烈興奮峰值——達到平靜專注和頭腦清晰的狀態。
* **抗氧化保護：** 兒茶素，特別是EGCG，是強效抗氧化劑，有助於中和自由基並減緩細胞氧化壓力。
* **心血管系統支持：** 傳統上認為茶具有軟化血管 (软化血管, ruǎnhuà xuèguǎn) 的能力。多酚有助於正常化膽固醇水平和血壓。
* **改善消化：** 多酚刺激胃液分泌並加速脂肪代謝，使茶成為用餐的良好伴侶。
* **強化牙齒：** 所含氟和兒茶素抑制致齲細菌活性。
* **礦物質補充：** 高鉀、鋅、鐵和錳含量補充日常飲食。
* **抗輻射活性：** 茶多酚具有結合某些重金屬和放射性核素的能力——對使用電子設備的人群有意義。

*注意：功效為一般保健性質。茶不是藥物。*

**9. 沖泡方法:**

* **水溫：** 80–85°C。更熱的水（>85°C）會導致茶湯變黃、出現苦味並失去細膩香氣。建議將水煮沸後冷卻至所需溫度。
* **茶量：** 2–3克配150毫升（歐式方法）；5–7克配蓋碗100–120毫升（功夫茶）。
* **用水：** 軟水，低礦化度。傳統上認為泰山泉水最理想——純淨柔軟，突出栗香甜味。
* **茶具：** 瓷蓋碗 (盖碗) 或白瓷/青白瓷茶杯。玻璃杯——用於視覺享受：葉片緩慢沉到底部並展開，創造如畫景象。

**過程（玻璃杯，上投法 / 上投法）：**

1. 用熱水溫杯後倒掉。
2. 倒入約1/3杯容量的水（80–85°C）。
3. 將茶葉（3克）放在水面上。
4. 等待約1分鐘，直到葉片吸水開始下沉。
5. 細流緩慢加水至杯子7/10容量。
6. 浸泡2–3分鐘後飲用。喝至1/3時加水。
7. 可沖泡4–6次；第3–4泡口感最佳。

**過程（蓋碗，功夫茶）：**

1. 用沸水溫蓋碗。
2. 投入5–7克茶葉，輕搖以釋放香氣。
3. 第一泡：80–85°C，20–30秒，倒掉。
4. 第二泡：15–20秒。
5. 後續沖泡：每次增加5–10秒。
6. 根據葉片厚度可沖5–8泡。

**10. 儲存方法:**

* **溫度：** 最佳——冰箱0–5°C密封包裝保存。這是所有綠茶的標準建議——低溫減緩氧化並保持新鮮度。
* **容器：** 鋁箔真空包裝、鐵罐或帶密蓋的陶瓷容器。避免接觸空氣。
* **茶葉敵人：** 光線、濕氣、高溫、異味。不要與香料、咖啡和其他有香味的產品一起存放。
* **保存期限：** 為獲得最佳口感——生產後6–12個月內飲用。開封後建議在2–4週內飲完，特別是室溫保存時。
* 不建議飲用隔夜茶 (隔夜茶, géyè chá)。

**11. 價格與仿冒品:**

**價格範圍：** 山東綠茶中的中高檔產品。春季首採明前茶 (明前茶, míngqián chá——清明節前）最昂貴。價格從每500克約200元到800+元不等，取決於等級、季節和生產商。在當地市場需求穩定，產量不大，支撐價格。

**如何避免仿冒品：**

* **從可靠來源購買：** 優選有認證和地理標誌地位的公司產品。參考——「泰山女兒」(泰山女儿) 品牌和泰安地區合作社。
* **評估外觀：** 真品具有特有的彎曲形狀，緊實「肉質」葉片，有明顯白毫。仿品通常是薄弱、鬆散捲曲的葉片，缺乏明顯質感。
* **檢查香氣：** 招牌特徵——持久栗香。人工香料會暴露出刺鼻、「化學」或過甜的氣味，很快消散。
* **評估茶湯：** 透明、明亮綠色或黃綠色，無渾濁。渾濁或暗淡的茶湯是陳舊或劣質原料的跡象。
* **對低價警惕：** 北方茶生產成本高（季節短、產量小）。價格明顯低於市場價——值得懷疑：可能用其他地區原料替代。

**12. 有趣事實:**

* **不是茶的茶。** 從明代到二十世紀中期的幾個世紀裡，泰山的「女兒茶」是用鼠李 (鼠李, 鼠李屬, *Rhamnus* spp.) 植物葉片製作的，而不是茶樹。現代茶實際上是古老名稱在新品質中的「重生」。
* **將軍茶農。** 1930年代，中華民國重要軍事領袖之一馮玉祥 (冯玉祥) 隱居泰山時，夢想將南方茶樹移植到北方山區，但當時的技術不允許實現。他的想法在三十年後才得以實現。
* **舞蹈葉片。** 獨特的視覺特色之一：在玻璃杯中沖泡時，葉片先沉到底部，然後從深色變為明亮綠色並展開，創造「舞蹈少女」效果——這是「女兒茶」名稱的另一種解釋。
* **最北方的茶。** 泰山是中國工業化茶葉種植的極北邊界之一（約北緯36°）。更高緯度意味著長期冬季休眠、短暫但密集的生長季節，以及氨基酸與多酚的異常高比例。
* **文學聲譽。** 很少有地方茶能誇耀在中國文學「四大名著」(四大名著, Sì Dà Míngzhù) 之一中被提及。女兒茶在《紅樓夢》中的出現確保了它在民族文化記憶中的地位。

**13. 與其他綠茶的比較:**

* **嶗山綠茶 (崂山绿茶, Láoshān Lǜchá)：** 山東另一著名綠茶，來自青島地區。兩種茶都是「北方」茶，具有特有的葉片厚度和栗香特徵。嶗山茶通常有更明顯的「豆香」（豌豆）調性和稍微更濃郁的茶體。泰山茶更溫和，重點在於純淨的栗香甜味。
* **日照綠茶 (日照绿茶, Rìzhào Lǜchá)：** 山東產量最大的綠茶。在海岸種植，氣候較溫和。特徵相似——栗香、厚實葉片——但日照茶稍微更澀，「山地」特徵較少。
* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南省著名綠茶——按中國標準也是「北方」茶。葉片更細、針狀，明顯清新和花香。泰山女兒茶更濃郁、更「栗香」，花香較少。
* **西湖龍井 (西湖龙井, Xīhú Lóngjǐng)：** 「泰山龍形」版本直接以龍井工藝為導向（扁平形狀）。然而泰山版本葉片明顯更厚，質感更粗糙，栗香主導而非龍井的豆香花香清新。

**結論：**

泰山女兒茶是一款有個性的茶，對北方緯度來說出人意料，因此更加珍貴。它不試圖與南方綠茶傳奇的精緻競爭——相反，它提供了它們所沒有的：深沉溫暖的栗香、厚實「肉質」的葉片、驚人的多次沖泡耐久性，以及獨特的「漸進式」口感動態，在第三到第四泡才達到峰值。這款茶是那些欣賞濃郁、飽滿的無苦味綠茶人士的絕佳選擇，也適合那些想要接觸中國最不尋常茶葉故事之一的人——在這個故事中，神聖山嶽、文學傳說和現代農學融為一體。
