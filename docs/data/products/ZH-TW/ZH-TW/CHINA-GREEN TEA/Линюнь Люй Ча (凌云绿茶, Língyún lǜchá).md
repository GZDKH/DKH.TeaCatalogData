# 凌雲綠茶 (凌雲綠茶, Língyún lǜchá)

**凌雲綠茶 (凌雲綠茶, Língyún lǜchá)**
![凌雲綠茶 (凌雲綠茶, Língyún lǜchá)](images/Линюнь%20Люй%20Ча%20(凌云绿茶,%20Língyún%20Lǜchá).png)

凌雲綠茶（凌雲綠茶，Língyún lǜchá）是產自廣西壯族自治區百色市凌雲縣（凌雲縣，Língyún Xiàn）的綠茶，採用獨特的**凌雲白毫種（凌雲白毫種，Língyún Báiháo Zhǒng）**製成，該品種又稱**華茶26號（華茶26號）**——1965年中華人民共和國農業部認定的首批21個國家級茶樹良種之一。這是小喬木大葉種（小喬木大葉種，xiǎo qiáomù dàyè zhǒng）：茶樹高達6-9公尺，芽頭肥大、肉質豐厚，覆滿白色茸毛——因此得名「白毫」（báiháo，「白色茸毛」）。生化成分令人印象深刻：茶多酚含量**35.6%**，兒茶素**182.92毫克/克**——這些指標可與雲南大葉種茶相媲美，是中國綠茶中最高的之一。凌雲白毫種的獨特之處在於可以製作六大茶類（六大茶類）——「一茶千化」（一茶千化，yī chá qiān huà）。1915年該茶在巴拿馬國際博覽會（巴拿馬國際食品博覽會二等獎）獲得二等獎，2005年獲得中國地理標誌產品保護。至2024年，該縣茶園面積達112,500畝（約7,500公頃），年產量8,326噸，總產值7.16億元人民幣。

**1. 分類與產地：**

* **類型：** 綠茶（綠茶，lǜchá），未發酵茶。有多種形態：**白毫王（白毫王，Báiháo Wáng，「白毫之王」）**——直條針形（針形，zhēnxíng）；**凌螺春（凌螺春，Língluóchūn）**——螺旋形（螺形）；**白毫銀針（白毫銀針，Báiháo Yínzhēn）**——單芽。製作工藝採用獨創的「低溫慢炒三青」（低溫慢炒三青，dīwēn mànchǎo sān qīng）技術，最後復香（復香，fùxiāng）提升香氣。

* **級別：** 國家地理標誌產品（國家地理標誌產品，2005年；商標2016年）。華茶26號——首批21個國家級茶樹良種之一（1965年）。國家級茶樹良種（國家級茶樹良種，1984年）。巴拿馬國際食品博覽會二等獎（巴拿馬國際食品博覽會二等獎，1915年）。多次國際大賽獲獎：紅茶和黃茶類金獎，綠茶類銀獎（第二、三屆國際名茶評比）。凌雲是首批20個全國無公害茶葉生產示範基地縣之一（全國無公害茶葉生產示範基地縣，2001年）。

* **產地：** 中國廣西壯族自治區（廣西壯族自治區，Guǎngxī Zhuàngzú Zìzhìqū），百色市（百色市，Bǎisè Shì），凌雲縣（凌雲縣）和樂業縣（樂業縣，Lèyè Xiàn）。核心產區——岑王老山（岑王老山，Cénwáng Lǎoshān）和青龍山（青龍山，Qīnglóng Shān）山脈，海拔800-1,500公尺。

* **地理座標：** 約北緯24°06′-24°36′，東經106°23′-106°55′。

**2. 歷史與文化意義：**

* **300多年栽培歷史。** 凌雲的茶葉種植記載始於清朝初期（清，1644-1912）。該地區擁有數百年樹齡的野生和半野生茶樹群，岑王老山保護區內的個別野生茶樹高達8.5公尺，據估計樹齡超過1,000年。乾隆年間（乾隆，1736-1795），當地茶葉成為「貢茶」（貢茶，gòngchá）——皇室御用茶，證明其早期聲譽。

* **1915年——巴拿馬國際博覽會。** 在著名的1915年巴拿馬博覽會——20世紀初最大的國際貿易盛會之一——凌雲茶獲得二等獎（二等獎）。這是廣西茶葉首次獲得國際認可，將凌雲白毫推上世界茶葉舞臺。

* **1965年——華茶26號。** 中華人民共和國農業部和中國農業科學院茶葉研究所（中國農業科學院茶葉研究所）對全國茶樹品種進行系統評估。凌雲白毫被列入首批21個國家級茶樹良種，獲得華茶26號註冊編號——是少數入選這一精英名單的廣西品種之一。

* **1984年——國家一級良種。** 中國茶樹良種審定委員會（中國茶樹良種審定委員會）授予凌雲白毫國家級茶樹良種（國家級茶樹良種）一級地位。同年，該品種在全國農業區劃會議上展示，並收錄於《中國茶樹優良品種集》（《中國茶樹優良品種集》）。

* **2005年——地理標誌。** 國家質檢總局（國家質檢總局）批准「凌雲白毫茶」為地理標誌保護產品。

* **文化意義。** 百色是「紅色」（革命）歷史之城：1929年12月，鄧小平（鄧小平，Dèng Xiǎopíng）在此領導百色起義（百色起義，Bǎisè Qǐyì），成為中國共產黨鬥爭史上的關鍵事件。凌雲綠茶是這片「紅色」土地的「綠色」象徵。凌雲是廣西最貧困的縣之一，茶產業成為脫貧攻堅的關鍵工具：至2024年，茶園面積112,500畝，為數萬戶壯族和瑤族（瑤族）少數民族家庭提供就業。凌雲白毫種能夠製作六大茶類，被譽為「一茶千化」（一茶千化）——這種獨特的多功能性在亞洲無與倫比。

**3. 植物學描述與原料：**

* **品種：** **凌雲白毫種（凌雲白毫種，Língyún Báiháo Zhǒng）**，又稱**華茶26號（華茶26號）**——*Camellia sinensis* var. *sinensis*（根據另一分類可能為var. *pubilimba*，待確認）。小喬木型（小喬木，xiǎo qiáomù）。大葉類（大葉類，dàyè lèi），中生種（中生種，zhōngshēng zhǒng）。有性繁殖系（有性繁殖系，yǒuxìng fánzhí xì）。植物學特徵：
  * **高度**——自然生長可達6-9公尺；茶園中通過修剪保持在1-1.5公尺。
  * **樹型**——半張開型（半張開，bàn zhāngkāi），分枝相對稀疏。
  * **葉片**——大型，橢圓形，水平或略下垂。深綠色，無光澤，葉面隆起（葉面隆起）。葉脈明顯。背面（下表面）密被茸毛。
  * **芽頭**——肥大、肉質，覆滿長而密的白色茸毛（茸毫長而密）。一芽三葉百芽重（一芽三葉百芽重）平均99克——顯著高於中葉種。
  * **春梢**——在凌雲於3月中旬開始生長；採摘高峰期（一芽三葉）為3月底至4月初。
  * **花期**——11-12月，結實稀少。

* **生化成分（據中國農科院茶葉研究所數據）：**
  * 氨基酸——**3.36%**。
  * 茶多酚——**35.6%**——綠茶中最高之一。
  * 兒茶素——**182.92毫克/克**——卓越的抗氧化潛力。
  * 咖啡鹼——**4.91%**。
  * 水浸出物——高。

* **等級：**
  * **特級：** 一芽一葉，茸毛≥90%，板栗香（板栗香）。白毫王形態——直條針形。
  * **一級：** 一芽二葉，緊結。
  * **二級：** 成熟葉，大眾產品線。

**4. 風土條件與栽培特點：**

* **地形。** 凌雲是位於雲貴高原與廣西盆地交界處的山區縣。地形為喀斯特（喀斯特，kāsītè）山地、深峽谷、山澗。茶園主要分布在岑王老山和青龍山山脈的坡地上，海拔800-1,500公尺——雲霧帶。典型地貌類型為「峽谷溪澗茶園」：茶樹沿著狹窄峽谷中的山澗種植，獲得自然遮蔭和較高濕度。

* **氣候：** 亞熱帶季風氣候（亞熱帶季風氣候）。年均溫19-23°C。年降雨量1,700-1,800毫米。雲霧天數超過100天/年。日夜溫差大，特別是在1,000-1,500公尺區域。雲霧帶空氣濕度≥90%。

* **土壤：** 發育於岑王老山和青龍山的**火山岩風化土**（火山岩風化土，huǒshān yán fēnghuà tǔ）。酸性（pH 4.5-5.5）。有機質含量3.0-4.5‰。富含礦物元素。因岩石基底排水良好。

* **生態：** 森林覆蓋率**76.7%**——廣西茶區中最高之一。水資源——山澗，一級水質。核心產區——玉洪鄉（玉洪鄉，Yùhóng Xiāng）和加尤鎮（加尤鎮，Jiāyóu Zhèn）。生物多樣性高：喀斯特森林支持豐富的動植物群，茶葉害蟲的天敵充足，可最大限度減少化學防治。

* **規模。** 至2024年：茶園112,500畝（約7,500公頃），年產乾茶8,326噸，總產值7.16億元。凌雲是廣西最大的產茶縣，也是華南重要產茶縣之一。

**5. 製作工藝：**

獨創技術「低溫慢炒三青」（低溫慢炒三青，dīwēn mànchǎo sān qīng）——「低溫慢速三次炒青以最大程度保留茸毛香氣」。全程使用竹木器具加工（全程竹木器具加工，quánchéng zhú mù qìjù jiāgōng）——金屬不接觸茶葉，防止氧化變黑和金屬味。

* **攤青（攤青，tān qīng）：** 攤放時間較長（攤放時間較長）——比大多數綠茶更長。讓肥大、肉質的芽頭均勻失水並形成香氣前體。

* **殺青（殺青，shā qīng）：** 溫度180-200°C。殺青程度偏重（殺青程度偏重）：徹底破壞大葉種高多酚含量（35.6%）原料中的殘餘酶活性，防止褐變。殺青後立即**扇風散熱**（扇風散熱，shàn fēng sàn rè）防悶黃（防悶黃，fáng mèn huáng）——關鍵步驟，防止自發轉化為黃茶類別。

* **揉捻（揉捻，róuniǎn）：** 初步成形——白毫王為直條針形，凌螺春為螺旋形。

* **「慢速三青」（炒乾——三青，chǎo gān — sān qīng）：** 關鍵步驟。低溫、慢速、三次循環：炒製→冷卻→再炒→冷卻→最後炒製。「小火長時間多次慢拋」（小火長時間多次慢拋）。目的——最大程度保留白毫（白毫）並形成深厚的板栗香（板栗香）而不焦糊。這種方法使凌雲綠茶有別於其他大葉種綠茶。

* **復香（復香，fùxiāng）：** 溫和加熱以「提升」香氣並將水分降至≤6%。

**6. 感官特徵：**

* **乾茶外形：** 因形態而異。**白毫王**——直條針形，覆滿白毫，形似銀針（形似銀針，xíng sì yínzhēn）。**凌螺春**——緊結螺旋形。色澤——綠色覆滿白毫（條索緊結、白毫顯露，tiáosuǒ jǐnjié, báiháo xiǎnlù）。五大特色：「色翠、毫多、香高、味濃、耐泡」——「翠綠、毫多、香高、味濃、耐泡」。

* **乾茶香氣：** 板栗香（板栗香，bǎnlì xiāng）——主要且最具特色。清香（清香，qīngxiāng）。「毫香」（毫香，háoxiāng）——細膩的「玉米」香調，為多毫茶葉特有。

* **茶湯香氣：** 板栗香——深沉、溫暖、持久。「毫香」——冷卻時出現的甜香。香氣在杯中持續很長時間，即使沖泡後的葉底幾天後仍保留香氣——罕見特性。

* **滋味：** 鮮爽（鮮爽，xiānshuǎng）。濃醇（濃醇，nóngchún）——茶湯飽滿，結構明顯，這在綠茶中不常見，歸因於高多酚含量（35.6%）。回甘（回甘）——穩定、持久。耐泡性——4-5次完整沖泡不失特色，這也不尋常，可歸因於大葉種原料。

* **湯色：** 嫩綠清澈（嫩綠清澈，nèn lǜ qīngchè）。第二、三泡時略帶黃色調。

* **葉底（沖泡後的茶葉）：** 青橄欖色、勻嫩亮潤（青橄欖色、勻嫩亮潤，qīng gǎnlǎn sè, yún nèn liàng rùn）。葉片大、完整，可見茸毛。在玻璃杯中沖泡時——葉片「站立」下沉，如春筍——視覺效果佳。

**7. 化學成分：**

* **茶多酚（茶多酚）：** **35.6%**——中國綠茶中最高之一，可與雲南大葉種茶（滇綠、普洱生茶）相媲美。高含量歸因於大葉種品種、亞熱帶氣候的強烈日照和刺激酚類化合物合成的火山土壤。

* **兒茶素（兒茶素，ér chá sù）：** **182.92毫克/克**——卓越的抗氧化潛力。主要成分——EGCG（表沒食子兒茶素沒食子酸酯）、EGC、ECG、EC。相比之下：典型的中葉種綠茶含80-120毫克/克兒茶素。

* **氨基酸（氨基酸）：** **3.36%**——適中水平，為南方大葉種特徵。酚氨比（酚氨比）高（約10.6:1），決定了「濃厚、有結構」的風味特徵，具有明顯的澀感和持久的餘韻。

* **咖啡鹼（咖啡鹼）：** **4.91%**——偏高，提供明顯的提神效果。

* **維生素：** 維生素C——因「低溫慢速三青」而得以保留。B族維生素（B₁、B₂、B₃）。維生素E。

* **礦物質：** 鉀、鎂、錳、鋅、鐵、氟。火山基質提供豐富的礦物質譜。

* **獨特特點：** 大葉種、35.6%茶多酚和182.92毫克/克兒茶素與3.36%氨基酸的組合，創造出綠茶中不典型的特徵，接近「普洱型」——強調茶體、結構和耐泡性，而非輕盈和「鮮爽」。

**8. 保健功效：**

* **強大的抗氧化作用。** 兒茶素182.92毫克/克——綠茶中最高濃度之一。EGCG——主要兒茶素——被認為是最強的天然抗氧化劑之一，可中和自由基並延緩細胞老化。

* **明顯的提神效果。** 咖啡鹼4.91%——高於綠茶平均水平。提供活力、改善注意力和工作記憶。與L-茶氨酸結合——無緊張感。

* **支持消化。** 高茶多酚含量（35.6%）刺激消化酶分泌，支持健康的腸道菌群。

* **心臟保護。** 兒茶素和茶多酚參與調節膽固醇水平，減少低密度脂蛋白氧化，維持血管彈性。

* **支持脂質代謝。** EGCG激活產熱和脂肪酸氧化——潛在的代謝支持。

* **抗菌作用。** 兒茶素抑制多種病原菌生長，包括變形鏈球菌（蛀牙）和幽門螺桿菌。

* **增強免疫力。** 茶多酚和維生素C、E複合物具有免疫調節作用。

* **認知支持。** L-茶氨酸刺激大腦α波，改善注意力和放鬆而不鎮靜。

**9. 沖泡方法：**

* **水溫：** 80-85°C。高多酚含量的大葉種原料需要適中的溫度以避免過度苦澀。

* **茶葉用量：** 3克/150毫升（比例1:50）。

* **茶具：** 玻璃杯（觀察茸毛針葉的「舞動」）。蓋碗。瓷壺。

* **步驟：**
  1. 用熱水溫杯，倒掉。
  2. 投入茶葉。
  3. **上投法**（上投法，shàng tóu fǎ）：先注水，後投茶。凌雲白毫的肥大、肉質芽頭緩慢下沉並在杯中「站立」——視覺效果如春筍。
  4. 第一泡——30-45秒。
  5. 倒出茶湯。
  6. 重複沖泡——**4-5次**，每次增加15-30秒。耐泡性——「五大特色」之一（耐泡）：即使第五泡仍保持特色。

**10. 儲存方法：**

* 密封包裝（鋁箔袋真空包裝），冰箱0-5°C。保質期——最長12個月。

* 長期儲存——冷凍（-18°C）真空包裝。

* 茶葉的敵人：濕氣、異味、光線、高溫。廣西亞熱帶氣候——濕度高；不建議在冰箱外儲存。

**11. 價格與防偽：**

* **價格區間：** 特級（白毫王，一芽一葉）——500-1,500元/500克，視生產商和年份而定。凌螺春——200-600元/500克。大眾級——100元/500克起。

* **如何避免假貨：**
  * **地理標誌標記。** 正品標有「凌雲白毫茶」或「凌雲綠茶」。
  * **芽頭大小。** 正宗凌雲白毫——**肥大、肉質**芽頭，**覆滿白毫**。細小芽頭——可能是中葉種替代品。
  * **香氣。** 板栗香（板栗香）+「毫香」（毫香）。無焦味、酸味。
  * **耐泡性。** 正品可沖泡4-5次。兩泡後失味——可能是假貨。
  * **價格。** 特級不可能低於300元/500克。以「凌雲白毫」名義的廉價產品值得懷疑。

**12. 趣聞軼事：**

* **1915年——巴拿馬獎章。** 凌雲白毫是廣西首批獲得國際認可的茶葉之一。1915年巴拿馬博覽會也讓西湖龍井、信陽毛尖等中國名茶聲名遠播。

* **華茶26號——樹高達9公尺。** 小喬木型——對綠茶來說異常高大。中國大多數綠茶產自1-1.5公尺高的中葉種灌木；凌雲白毫自然生長可達三層樓高。

* **「一茶千化」（一茶千化）。** 凌雲白毫是少數（據某些資料，是亞洲唯一）可製作六大茶類的品種：綠茶、紅茶、白茶、黃茶、黑茶和烏龍茶。凌雲紅茶曾獲國際比賽金獎。

* **峽谷茶園。** 凌雲的茶園沿峽谷山澗（峽谷溪澗）分布——獨特的地貌類型，提供自然遮蔭、高濕度和防風保護。

* **兒茶素182.92毫克/克。** 中國綠茶中最高之一——可與傳統上被認為是「兒茶素冠軍」的雲南大葉種茶相媲美。相比之下：典型的龍井含100-130毫克/克兒茶素。

**13. 與其他綠茶的比較：**

* **雲南大葉種綠茶（滇綠，滇綠，Diānlǜ）。** 兩者都是大葉種，茶多酚和兒茶素含量高。滇綠——來自雲南大葉種（*C. sinensis* var. *assamica*）；凌雲——來自凌雲白毫（*C. sinensis* var. *sinensis*，小喬木型）。滇綠——更澀、「粗獷」；凌雲——有明顯的板栗香和「毫」的柔和。兩者都具有強大的抗氧化潛力。

* **桂平西山茶（桂平西山茶，Guìpíng Xīshān Chá）。** 同為廣西綠茶，來自小葉種。西山——輕盈、花香；凌雲——濃厚、板栗香。西山——來自平原丘陵（200-500公尺）；凌雲——來自山地峽谷（800-1,500公尺）。規模不同：凌雲——112,500畝，西山——顯著較小。

* **洞庭碧螺春（洞庭碧螺春，Dòngtíng Bìluóchūn）。** 凌螺春（凌螺春）——凌雲的螺旋形——特意以碧螺春命名。兩者都是螺旋形、多毫。碧螺春——小葉種，花果香；凌螺春——大葉種，板栗香。碧螺春——60,000-80,000芽/公斤；凌螺春——因芽頭大而顯著較少。

* **白沙綠茶（白沙綠茶，Báishā Lǜchá）。** 海南綠茶，獨特地質（隕石坑）。兩者都來自非標準地質區（凌雲的火山岩，白沙的隕石坑）。白沙——熱帶，全年生長；凌雲——亞熱帶，春季高峰。白沙——小葉種；凌雲——大葉種。

**結語：**

凌雲綠茶——來自「紅色」百色的茶葉，誕生於青龍山峽谷的火山土壤上，來自高達九公尺的小喬木茶樹，兒茶素含量182.92毫克/克，擁有1915年巴拿馬獎章。華茶26號品種——中國首批國家級茶樹良種之一——產出具有濃厚板栗香、「毫」的柔和感和可與雲南巨人媲美的抗氧化潛力的茶葉。五大特色——翠綠、毫多、香高、味濃、耐泡——以及一棵樹能製作六大茶類的能力，使凌雲白毫成為中國最多功能且被低估的茶樹品種之一。
