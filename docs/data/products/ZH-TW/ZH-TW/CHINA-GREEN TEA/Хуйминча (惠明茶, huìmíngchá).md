# 惠明茶 (惠明茶, huìmíngchá)

**惠明茶 (惠明茶, huìmíngchá)**
![Хуйминча (惠明茶, huìmíngchá)](images/Хуйминча%20(惠明茶,%20Huìmíngchá).png)

![Хуйминча (惠明茶, huìmíngchá)-1](images/Хуйминча%20(惠明茶,%20Huìmíngchá)-1.png)

惠明茶 (惠明茶, huìmíngchá) — 來自浙江省景寧畲族自治縣 (景宁畲族自治县) 的歷史綠茶，在1915年巴拿馬太平洋萬國博覽會上獲得標誌性勝利——金質獎章 (金质奖章)，因此獲得榮譽稱號「金獎惠明茶」(金奖惠明茶，「獲得金獎的惠明茶」)。這款茶的歷史始於西元861年一位名叫惠明 (惠明) 的佛教僧人，以「三絕」聞名：翠綠色澤 (色翠)、高雅香氣 (香高)、鮮爽滋味 (味鲜)。原料的獨特之處——乳白色帶淡黃色調的芽頭，沖泡時呈現白濁茶湯——為此茶贏得古老別名「白茶」(白茶, báichá)，儘管在工藝上它毫無疑問屬於綠茶類。

**1. 分類與產地：**

* **類型：** 綠茶（不發酵茶）。有兩種形態：螺形 (螺形, luóxíng) 和扁形 (扁形, biǎnxíng)。工藝上屬炒青類。

* **等級：** 國家地理標誌保護產品 (国家地理标志保护产品)。1915年巴拿馬太平洋萬國博覽會金質獎章獲得者 (巴拿马万国博览会金质奖章)。浙江歷史名茶。

* **產地：** 中國浙江省 (浙江, Zhèjiāng) 麗水市 (丽水市, Líshuǐ Shì) **景寧畲族自治縣** (景宁畲族自治县, Jǐngníng Shēzú Zìzhìxiàn) — 中國唯一的畲族自治縣。地理標誌保護區域為整個縣境。核心產區——鶴溪鎮 (鹤溪镇, Hèxī Zhèn) 惠明寺村 (惠明寺村) 和敕木山 (敕木山, Chìmù Shān) 山坡。

* **地理座標：** 約北緯27°58′，東經119°38′。

**2. 歷史與文化意義：**

* **歷史：** 西元861年（唐朝咸通 (咸通) 二年），佛教僧人惠明和尚 (惠明和尚, Huìmíng Héshàng) 在景寧南泉山 (南泉山) 建寺，並在寺廟山坡種植茶樹。以僧人名字命名的「惠明茶」成為該地區最早的「寺院茶」之一。

	1482年（明朝成化 (成化) 十八年），惠明茶被列入皇室貢品 (贡品, gòngpǐn) 名單。

	榮耀巔峰——1915年：惠明茶在舊金山**巴拿馬太平洋萬國博覽會獲得金質獎章**。這一獎項成為中國茶葉在國際舞臺上的標誌性勝利之一，為此茶確立了榮譽稱號「金奖惠明茶」（「獲得金獎的惠明茶」）。

	經過數十年衰落後：1973年開始恢復研究工作。1979年正式恢復生產，茶園得到擴建。今日惠明茶是景寧茶業的旗艦產品，也是畲族文化認同的重要元素。

* **名稱：**

	* 「惠明」(惠明) — 佛教僧人創始者 (惠明和尚) 的名字，他於861年建寺並種植了第一批茶樹。
	* 「茶」(茶) — 「茶」。
	* 古老別名「白茶」(白茶) 與芽頭乳白色和白濁茶湯有關——但這不是工藝術語，而是描述性稱呼。

* **文化意義：** 惠明茶是少數與特定少數民族密不可分的茶葉之一：**畲族 (畲族, Shēzú)** — 中華人民共和國55個官方認定少數民族之一。景寧是中國唯一的畲族自治縣，惠明茶在當地待客文化、儀式和經濟中佔據核心地位。1915年金獎是特別的驕傲：它將惠明茶與其他巴拿馬展覽會偉大獲獎者（茅臺、碧螺春等）並列。

**3. 植物學描述與原料：**

* **品種/栽培種：** 當地原生群體種 *Camellia sinensis* var. *sinensis* 灌木型，中葉種。嫩芽呈黃綠色，茸毛豐富 (芽叶黄绿色，茸毛多)。百芽三葉重約47克。此外，還使用專門栽培種：**景白1號、景白2號 (景白1号、景白2号)** — 當地選育品種，具有改良特性，以及**迎霜 (迎霜, Yíngshuāng)** — 適合生產高品質綠茶的無性系品種。

* **採摘：** 早春採摘。最高等級「精品」(精品，jīngpǐn，「傑作」) — 最嫩的芽頭和嫩梢。「特級」— 一芽一葉。「一級」和「二級」— 一芽一至二葉。

* **原料要求：** 嫩度一致、完整無損的嫩梢。芽頭特有的乳白色澤是當地栽培種的天然特性，而非缺陷。

**4. 風土條件與栽培特點：**

* **海拔高度：** 海拔300-800公尺。核心區域——敕木山山坡和惠明寺周圍。

* **氣候：** 中亞熱帶氣候 (中亚热带)。年平均氣溫≥15.2°C，年降雨量≥1600毫米，年日照時數≥1800小時。雲霧是特色現象。充足的散射光有利於胺基酸積累。

* **土壤：** 酸性砂質黃壤土 (酸性砂质黄壤土) 和「香灰土」(香灰土)，pH值4.5-6.5。土層深度≥0.5公尺。有機質含量≥1.5%。水資源——甌江 (瓯江) 支流。

**5. 製作工藝：**

惠明茶有兩種形態，各有其工藝特點：

**螺形惠明茶 (螺形惠明茶)：**

* **攤放 (鲜叶摊放 — xiānyè tānfàng)：** 短時間攤放萎凋。
* **殺青 (杀青 — shāqīng)：** 鍋炒，運用手工技法「拋、悶、撈、抖」(抛、闷、捞、抖 — pāo, mèn, lāo, dǒu) — 十種傳統手工動作。
* **揉捻 (揉捻 — róuniǎn)：** 冷揉 (冷揉, lěngróu) — 特點：在冷卻後進行揉捻，而非炒制後立即揉捻，這樣能保存更多芳香物質。
* **理條做形 (理条/做形 — lǐtiáo / zuòxíng)：** 形成螺旋結構。
* **提毫整形 (提毫整形 — tí háo zhěngxíng)：** 專門技法，使白毫顯露於表面。
* **提香 (提香 — tíxiāng)：** 最後烘焙，固定香氣。

**扁形惠明茶 (扁形惠明茶)：**

* 類似工序，但在做形階段採用「輝鍋」(辉锅, huīguō，「鍋內拋光」)：茶葉貼著鍋壁壓制拋光，形成扁平整齊的外形。

* **特點：** 機械加工（大批量生產）採用分段乾燥 (分段干燥) 以最大程度保持香氣和色澤。

**6. 感官特徵：**

* **乾茶外形：** 螺形——緊細捲曲，顯毫，色澤翠綠 (条索紧细卷曲，显毫，色泽翠绿)。扁形——條索扁平挺直，色澤嫩綠或深綠 (条索扁平挺直，色泽嫩绿或深绿)。

* **乾茶香氣：** 蘭花香 (兰花香, lánhuā xiāng) — 主導香型。果甜香 (水果甜香, shuǐguǒ tiánxiāng) — 綠茶中的獨特特徵。清香 (清香)。

* **茶湯香氣：** 蘭花果香，持久。「花香」(花香) 與「果香」(果香) 的結合——綠茶中罕見。

* **滋味：** 鮮爽 (鲜爽, xiānshuǎng)、甘甜 (甘, gān)、醇厚 (醇厚, chúnhòu)。游離胺基酸含量3.5-4.5%，其中75-90%為「甜鮮味胺基酸」(甜鲜味氨基酸)，這解釋了其卓越的甜味和鮮味而無粗澀感。品飲公式：「首泡淡雅，二泡鮮濃，三泡甘醇，四泡後餘韻猶存」(首泡淡雅，二泡鲜浓，三泡甘醇，四泡后余韵犹存)。

* **湯色：** 清澈透明，嫩綠帶淡白色 (清澈)，這與乳白色芽頭的特性有關。

* **葉底：** 嫩勻完整。葉片彈性好，鮮活。

**7. 化學成分：**

* **多酚類（兒茶素）：** 含量豐富——提供抗氧化潛力和免疫調節作用。

* **胺基酸：** **3.5-4.5%** — 高於綠茶平均水準。關鍵特點：**胺基酸組成的75-90%** 為「甜鮮味胺基酸」(甜鲜味氨基酸) — L-茶胺酸、谷胺酸等，決定了顯著的甜味和鮮味而無苦澀。這是世界綠茶中最高指標之一。

* **生物鹼：** 咖啡因——適中含量。

* **維生素：** 維生素C、B群維生素。

* **礦物質：** 鉀、鎂、鋅、錳。

**8. 保健功效：**

* **抗氧化和免疫調節作用：** 多酚類和胺基酸複合體。
* **清熱解毒消暑 (解毒，清热消暑)：** 綠茶的傳統功效。
* **明目 (明目)：** 類胡蘿蔔素和維生素C。
* **止渴生津 (止渴生津)：** 刺激唾液分泌。
* **提神醒腦：** 咖啡因和L-茶胺酸。

* **重要提醒：** 以上功效基於公開資料，不構成醫療建議。

**9. 沖泡方法：**

* **水溫：** 約80°C。

* **茶量：** 3克配150-180毫升水（比例1:50-1:60）。

* **茶具：** 玻璃杯或白瓷茶杯。建議使用中性微酸礦泉水 (中性微酸矿泉水)。

* **沖泡步驟：**

	1. 溫杯，倒掉水。
	2. 投茶。
	3. 注入少量水，「潤茶」30秒。
	4. 加水至7/10容量。
	5. 第一泡倒掉 (头汤倒掉，洗茶)。
	6. 第二泡開始品飲。茶葉可沖泡3次完整泡數。

* **注意：** 避免高溫和長時間浸泡——會導致茶湯發黃和失去鮮爽感。最佳品飲：先聞杯中蘭花果香，再觀湯色，最後品味。

**10. 儲存方法：**

* 密封容器，陰涼乾燥處保存。
* 最佳——冰箱0-5°C保存。
* 保存期限——12個月。
* 開封後——1-2個月內飲用完畢。

**11. 價格與防偽：**

惠明茶是來自唯一畲族自治縣的限量生產茶葉。四個等級（精品、特級、一級、二級）提供價格差異化。

* **避免假冒的方法：**

	* **從可信賣家購買**，有景寧縣地理標誌認證。
	* **評估原料特點：** 乳白色芽頭是真品的天然標誌。均勻翠綠無白色的茶葉——可能是其他茶。
	* **評估香氣：** 蘭花與果甜香的結合是招牌特徵。缺乏果香——值得懷疑。
	* **檢查茶湯：** 清澈帶淡白色——特有特徵。
	* **注意價格：** 過低價格——假冒跡象。

**12. 有趣事實：**

* 1915年巴拿馬太平洋博覽會金獎是20世紀初最具聲望的茶葉獎項之一。惠明茶與其他偉大的中國茶葉和酒類（包括茅臺）共享這一榮譽。從此，品牌正式全稱為「金獎惠明茶」(金奖惠明茶，「獲得金獎的惠明茶」)。

* 861年建寺種茶的惠明和尚是中國寺院茶業史上最具體可考的人物之一。「僧人—寺院—茶」的聯繫延續了1160多年。

* 惠明茶是與特定民族群體密不可分的罕見茶例：畲族 (畲族)。畲族是中國南方最古老的民族之一，有自己的茶文化、歌謠傳統和儀式。

* 賦予此茶古老別名「白茶」(白茶) 的乳白色芽頭，不是白化現象（如安吉白茶），而是當地群體種的遺傳特性，表現為早春色素特徵。

* 75-90%的胺基酸組成為「甜鮮味胺基酸」是世界綠茶中最高指標之一。這解釋了其獨特的甜味而無任何苦澀。

**13. 與其他1915年巴拿馬展覽會獲獎茶葉的比較：**

* **碧螺春 (碧螺春)：** 來自江蘇。緊密螺旋，花果香。碧螺春更「精緻」果香；惠明茶蘭花香更突出，原料獨特「白色」特徵。

* **都勻毛尖 (都匀毛尖)：** 來自貴州。同為巴拿馬展覽會獲獎者（不同級別獎項）。都勻更栗香結構性；惠明茶更甜美「蘭花果香」。

* **安吉白茶 (安吉白茶)：** 來自浙江。同為高胺基酸含量的「白色」綠茶，但來自白化芽葉。安吉「純淨甜味」；惠明茶「蘭花果甜」，香氣層次更豐富。

* **開化龍頂 (开化龙顶)：** 來自浙西。針形，蘭花香。龍頂更「綠色」視覺效果（「杯中森林」）；惠明茶更「白色」果香。

**結語：**

惠明茶——擁有千年寺院歷史、世界級金獎和獨特民族認同的茶葉。乳白色芽頭、蘭花果香、卓越甜味（75-90%「甜鮮味胺基酸」）和與畲族文化的聯繫——這一切使其成為浙江最獨特和被低估的綠茶之一。如果龍井是浙江茶的「皇帝」，碧螺春是其「公主」，那麼惠明茶就是「獲得金獎的隱士僧人」：謙遜、千年傳承，每一杯都蘊含乳白色的秘密。
