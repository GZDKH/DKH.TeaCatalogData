# 鶴峰茶 (鹤峰茶, Hèfēng chá)

**鶴峰茶 (鹤峰茶, Hèfēng chá)**

![Хэфэн Ча (鹤峰茶, Hèfēng chá)](images/Хэфэн%20Ча%20(鹤峰茶,%20Hèfēng%20Chá).png)

![Хэфэн Ча (鹤峰茶, Hèfēng chá)-1](images/Хэфэн%20Ча%20(鹤峰茶,%20Hèfēng%20Chá)-1.png)

鶴峰茶是湖北省恩施土家族苗族自治州鶴峰縣綠茶的集合名稱。這是中國中部最大的茶葉產區之一，位於武陵山脈腹地的「黃金茶葉緯度」北緯30°。主要特色是茶葉天然富含硒元素：當地茶葉中這種微量元素的含量比平均水準高出數十倍。2012年，「鶴峰茶」品牌獲得地理標誌產品 (地理标志产品, Dìlǐ Biāozhì Chǎnpǐn) 地位，該縣榮獲「中國茶葉之鄉」稱號。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵。歷史上該縣也生產宜紅茶 (宜红茶, Yíhóng Chá) 類別的紅茶，但現代生產和地理標誌的基礎是綠茶。
* **類別：** 中國區域性綠茶；國家地理標誌產品；湖北省「名茶」(湖北十大名茶)。
* **產地：** 中國，湖北省 (湖北, Húběi)，恩施土家族苗族自治州 (恩施土家族苗族自治州, Ēnshī Tǔjiāzú Miáozú Zìzhìzhōu)，鶴峰縣 (鹤峰县, Hèfēng Xiàn)。生產區域涵蓋9個鄉鎮：鐵爐白族鄉 (铁炉白族乡, Tiělú Báizú Xiāng)、走馬鎮 (走马镇, Zǒumǎ Zhèn)、五裡鄉 (五里乡, Wǔlǐ Xiāng)、燕子鄉 (燕子乡, Yànzi Xiāng)、中營鄉 (中营乡, Zhōngyíng Xiāng)、下坪鄉 (下坪乡, Xiàpíng Xiāng)、鄔陽鄉 (邬阳乡, Wūyáng Xiāng)、太平鄉 (太平乡, Tàipíng Xiāng)、容美鎮 (容美镇, Róngměi Zhèn)。
* **地理座標：** ~30° 北緯，~110° 東經。武陵山脈 (武陵山, Wǔlíng Shān)，湖北東南部。

**2. 歷史與文化意義：**

* **歷史：** 鶴峰的茶文化是中國中部最古老的茶文化之一。西晉《荊州土地記》(《荆州土地记》, *Jīngzhōu Tǔdì Jì*, 三至五世紀) 記載「武陵七縣通出茶」。唐代陸羽 (陆羽, Lù Yǔ) 的《茶經》(《茶经》, *Chájīng*) 中說：「巴山峽川有兩人合抱者」，又說：「山南以峽州上」。鶴峰正位於這些描述區域的核心——武陵山脈深處，在「巴山峽川」之間。

  明清時期，鶴峰茶成為容美土司 (容美土司, Róngměi Tǔsī) 政權下的宮廷貢茶 (贡茶, gòngchá)——土家族當地世襲行政機構。文人顧彩 (顾彩, Gù Cǎi) 於1704年（康熙四十三年）造訪容美土地，在《容美紀遊》(《容美纪游》, *Róngměi Jìyóu*) 中記錄：「諸山皆產茶……上者值錢一串，中者楚省通用，亦謂之湘潭茶；故茶商之來無虛日」。

  1867年，鶴峰首任知州毛峻德 (毛峻德, Máo Jùndé) 在首部《縣誌》中記載：「容美貢茶……種植遍野……最佳者出自署後數株；其味清香甘潤」。1876年，廣東商人將部分生產轉為紅茶工藝，鶴峰成為「宜紅茶」(宜红茶) ——「宜昌紅茶」的重要中心之一，出口至俄國、英國和其他歐洲國家。英國貴族給鶴峰茶起了非正式的綽號「皇后茶」(皇后茶, Huánghòu Chá)。

  1980年代，生產重新定位於高品質綠茶。1995年，茶園面積達到10.5萬畝，產量4,501噸。幾個當地品牌——「官鼎」(官鼎)、「清心翠茗」(清心翠茗)、「綠林翠峰」(绿林翠峰) ——在「陸羽杯」(陆羽杯) 競賽中獲得金獎，佔全國金獎總數的13%。

  2012年7月31日，中華人民共和國國家質量監督檢驗檢疫總局批准「鶴峰茶」地理標誌保護。到2024年，茶園面積達39.5萬畝（約26,330公頃），其中10.9萬畝（約7,270公頃）獲得有機認證——全國最大指標。出口量居湖北省首位；產品銷往摩洛哥、塞內加爾、阿爾及利亞、伊朗、法國等國家。到2024年，「鶴峰茶」品牌價值估計為240.4億元人民幣。

* **名稱：** 鶴峰 (鹤峰) ——「鶴峰」，縣的正式名稱。茶 (茶) ——「茶」。該名稱是區域品牌，統一鶴峰縣境內生產的所有茶葉。

* **文化意義：** 鶴峰是「萬裡茶道」(万里茶道, Wànlǐ Chádào，「偉大茶葉之路」) 的起點，這條古代貿易路線連接中國中部茶區與俄國和歐洲。該縣正努力將偉大茶葉之路遺產列入聯合國教科文組織名錄。土家族 (土家族, Tǔjiāzú) 保持了獨特的茶文化：「四道茶」(四道茶, sì dào chá) 儀式、茶歌、關於茶的諺語和謎語，以及「白鶴井的水、留駕司的茶」(白鹤井的水、留驾司的茶) 傳統。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種——福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbái Chá)、槠葉齊 (槠叶齐, Zhūyè Qí) 和當地群體種 (本地群体种, běndì qúntǐ zhǒng)。植物為灌木型，小葉和中葉類型 (*Camellia sinensis* var. *sinensis*)，具有高耐寒性。芽和嫩葉密被茸毛 (茸毛, róngmáo)；百芽重約45克。
* **採摘：** 春季採摘（3-4月）用於最高等級；允許採摘至秋季用於大眾類別。標準：單芽 (单芽) 用於特級，一芽一葉 (一芽一叶) 用於一級，一芽二葉 (一芽二叶) 用於二級。
* **原料要求：** 嫩綠、均勻的原料，密被茸毛，無粗老葉片和莖桿。禁止使用化學肥料和農藥；種植密度不超過每公頃60,000株。有機認證茶園比例為27.6%，是全國最高指標。

**4. 風土條件與栽培特色：**

* **地形與區域：** 鶴峰位於湖北東南部武陵山脈深處，處於標誌性座標——北緯30°和東經110°的交匯處。該地區以連綿的山脈和深谷、豐富的雲霧和高濕度為特徵。

* **生長海拔：** 海拔500-1,000公尺。生產核心——走馬鎮的木耳山 (木耳山, Mù'ěr Shān，「木耳山」) 和中營鄉麻旺村 (麻旺村, Máwàng Cūn) 的古茶園，那裡保存著百年古樹。這些地區佔總產量的約80%。

* **氣候：** 年平均氣溫12-15.5°C。年降水量1,600-1,900毫米。雲霧天數超過200天。日溫差超過10°C。散射光比例約70%，使芽中氨基酸合成效率提高30%。

* **土壤：** 黃壤和砂質壤土 (黄壤, huáng rǎng)，pH 4.5-6.5，有機物含量≥2.9%。關鍵特色——天然富硒：鶴峰土壤中硒含量比全國平均水準高12倍。森林覆蓋率82.7%（綠化指數88.66%），無工業污染。空氣品質優良天數約350天。

**5. 製作工藝：**

鶴峰茶工藝旨在最大限度保持顏色、氨基酸、硒和香氣，同時對葉片進行最小機械處理。特色是獨創的「陰陽」乾燥制度交替系統，禁止機械揉捻以保持芽的完整性。

* **攤放 (摊放 — tānfàng)：** 將鮮葉薄攤4-8小時。葉片失水，變得有彈性，開始發展芳香化合物。

* **殺青 (杀青 — shāqīng)：** 在120-170°C進行——比許多其他綠茶溫和得多的制度，允許保持細膩的氨基酸特徵和硒的活性。

* **揉捻 (揉捻 — róuniǎn)：** 精細進行，達到成條率≥90% (成条率 ≥ 90%)。不使用機械滾筒——手工或半機械揉捻以保持芽的完整性。

* **初乾 (初干 — chūgān)：** 在110-130°C，穩定形狀並去除主要水分。

* **整形提毫 (整形提毫 — zhěngxíng tí háo)：** 在80-90°C——關鍵階段，使茶葉獲得最終形狀並「提升」表面銀色茸毛，創造鶴峰毛尖的特徵外觀。

* **足乾 (足干 — zúgān)：** 在60-80°C，達到穩定濕度。

* **提香 (提香 — tíxiāng)：** 在90-100°C——短暫焙火以固定和增強香氣。

* **工藝特色：** 獨特的「陰陽火候交替乾燥」(阴阳火候交替干燥) 系統：師傅在整個加工過程中變換高低溫度，獲得深層和多層次的香氣特徵。活性硒保存率≥95%。

**6. 感官特徵：**

* **乾茶外形：** 緊細圓直 (紧细圆直, jǐnxì yuánzhí) 的毛尖型茶條。色澤翠綠顯毫 (翠绿显毫, cuìlǜ xiǎn háo)，油潤有光澤，帶明顯銀色茸毛。

* **乾茶香氣：** 清香 (清香, qīngxiāng) 純正高長，帶蘭花香韻 (兰花香韵, lánhuā xiāng yùn)。炒青 (炒青, 炒製風格) 版本呈現栗香 (栗香, lìxiāng)。陳放後發展蜜香 (蜜香, mìxiāng) 調性。

* **茶湯香氣：** 高長持久，純正。以花香綠色清新為主，帶輕微堅果調性。香氣在5-7泡中保持。

* **滋味：** 鮮爽 (鲜爽, xiān shuǎng) 明亮，因氨基酸含量高（毛尖中≥5%）。茶體醇厚 (醇厚, chúnhòu) 飽滿，茶多酚和氨基酸之間和諧平衡。回甘持久 (回甘持久, huígān chíjiǔ)。總體印象——滋味純淨活潑無苦澀，具礦物深度。

* **湯色：** 嫩綠明亮 (嫩绿明亮, nènlǜ míngliàng)。

* **葉底（沖泡後的葉片）：** 嫩綠勻整，芽葉舒展如蘭 (嫩绿匀整，芽叶舒展如兰)。

**7. 化學成分：**

鶴峰茶以高硒含量和強大多酚特徵的結合在中國綠茶中脫穎而出。

* **茶多酚 (茶多酚)：** ≥18.3%（炒青風格）。主要群體——兒茶素，包括EGCG。根據中國研究，鶴峰茶的茶多酚吸收放射性物質的效率比典型綠茶多酚高35%。
* **氨基酸：** 特級毛尖中≥5%。L-茶氨酸含量高，確保明顯的甜味和滋味柔和。
* **水浸出物 (水浸出物)：** ≥45.2%——極高指標，確保茶體飽滿和多次沖泡的耐久性（5-7泡）。
* **硒 (硒, xī)：** 高達39.31毫克/公斤——而普通綠茶平均含量約0.15毫克/公斤。這是超過260倍的含量，使鶴峰茶成為世界上最富硒的茶之一。硒與茶多酚以有機形式結合，提高其生物利用度。
* **鋅：** 39.31毫克/公斤——免疫功能和組織再生的重要微量元素。
* **生物鹼：** 咖啡因（約2-4%）、可可鹼、茶鹼。
* **維生素：** 維生素C（鮮葉中）、B群維生素、維生素K。
* **精油：** 芳樟醇、香葉醇和其他萜烯化合物，形成花香蘭花特徵。

**8. 功效特性：**

* **抗氧化保護和硒支持：** 多酚和有機硒的獨特結合提供強大的抗氧化協同作用。硒是穀胱甘肽過氧化酶的輔因子——機體抗氧化防護的關鍵酶之一。
* **心血管系統支持：** 兒茶素有助於調節膽固醇合成和維持血管彈性。根據中國研究，鶴峰綠茶的降脂效果比紅茶高25%。
* **輻射防護作用：** 茶多酚具有結合放射性物質（鍶-90、銫-137）的能力。
* **溫和提神效果：** 咖啡因和高濃度L-茶氨酸的結合創造清醒和專注的平衡，無焦慮感。
* **免疫支持：** 生物可利用形式的鋅和硒支持免疫系統功能。
* **消化支持：** 餐後適量飲用有助於舒適消化。
* **認知功能：** L-茶氨酸有助於改善專注力和感知清晰度。

* **重要提醒：** 此資訊僅供一般了解，不構成醫療建議。

**9. 沖泡方法：**

* **水溫：** 80-90°C（沸水需冷卻約3分鐘）。理想用水——泉水或軟質過濾水。
* **茶量：** 3克配150毫升水（比例1:50）。
* **茶具：** 玻璃杯 (玻璃杯) ——觀察芽葉和茸毛展開；白瓷蓋碗 (白瓷盖碗) ——完全控制沖泡。
* **過程：**
  1. 用熱水溫杯後倒掉。
  2. 投茶。
  3. 第一泡——90秒。
  4. 後續沖泡——每次減少15秒。
  5. 因浸出物含量高，茶可沖泡5-7次完整沖泡。

* **注意事項：** 不建議空腹飲用（單寧含量高）。最佳時間——餐後一小時。每日用量不超過600毫升。服藥時至少間隔1小時（單寧酸降低藥物吸收）。

**10. 儲存：**

* 密封、不透明包裝，避光、防潮、防異味。
* 最佳——冰箱（0-5°C）保存硒和新鮮度。
* 開封後——3個月內飲用完畢，防止硒氧化和香氣流失。
* 開啟冷藏包裝前，讓其在密封狀態下回溫至室溫。

**11. 價格與仿冒：**

* **價格範圍：** 鶴峰茶涵蓋廣泛範圍——從大眾炒青到高端毛尖和高山廢棄茶園的「野茶」(野茶, yěchá)。具體價格取決於等級、季節和有機認證。來自老茶園的有機毛尖是最珍貴的類別。
* **價格因素：** 季節（春茶明顯比夏茶貴）、等級、類型（毛尖、炒青、野茶）、有機認證、出口批次。
* **如何避免仿冒：**
  * 購買標有地理標誌「鶴峰茶」(鹤峰茶) 的茶葉。
  * 評估外觀：正宗毛尖——緊細圓直、均勻的茶條，帶明顯茸毛，翠綠色澤。
  * 檢查香氣：真正的鶴峰茶具有純正、清新、高長的香氣，無霉味；炒青風格帶栗香調。
  * 茶湯應為嫩綠明亮，5-7泡滋味不明顯減弱。
  * 「有機」或「富硒」茶的異常低價——值得懷疑。

**12. 有趣事實：**

* 鶴峰是世界最大的有機茶葉種植中心之一：認證有機茶園面積10.9萬畝（約7,270公頃），居全國首位。部分茶園獲歐盟標準認證。

* 鶴峰茶的硒含量——高達39.31毫克/公斤——使其成為地球上最富硒的茶之一。普通綠茶含硒約0.15毫克/公斤。

* 清代鶴峰茶出口英國，在貴族圈獲得「皇后茶」(皇后茶) 的非正式綽號。出口路線經「萬裡茶道」(万里茶道)，該縣聲稱是此路線的「起點」。

* 木耳山 (木耳山) 上有中國最大的連續有機茶園之一——萬畝連片。入選中國30個最美茶園。

* 土家族保持獨特的「四道茶」(四道茶) 茶儀式：第一杯——迎賓白茶，第二杯——蜂蜜甜茶，第三杯——「油茶湯」(油茶汤, yóuchá tāng) 配爆米花和堅果，第四杯——告別「離別茶」。

**13. 鶴峰茶品種：**

* **綠毛尖 (绿毛尖, Lǜ Máojiān)：** 最高等級——單芽或一芽一葉初展。茶條細緊，密被茸毛。滋味鮮爽明亮，突出氨基酸甜味（氨基酸含量≥5%）。適合喜愛細膩、精緻綠茶的品茶者。

* **炒青 (炒青绿茶, Chǎoqīng Lǜchá)：** 中等等級——一芽二葉。炒製風格，栗香明顯，茶體飽滿，多次沖泡耐久性好（多酚≥18.3%）。平衡濃郁度和親民性的日常茶。

* **野茶 (野茶, Yěchá)：** 來自老舊廢棄或高山野生茶樹的原料。具有花果香氣，硒含量更高（高達1.5毫克/公斤）和明顯的礦物深度。限量生產，高收藏價值。

* **毛尖等級：**
  * 特級 (特级)：單芽≥98%，長度≤2.5公分，翠綠顯毫，清香高長，水浸出物≥45.2%。
  * 一級 (一级)：一芽一葉≥90%，緊細成條，滋味鮮醇，多酚≥18.3%。
  * 二級 (二级)：一芽二葉，葉底勻整，耐泡性好，性價比最佳。

**14. 禁忌事項：**

* 不建議空腹飲用——單寧含量高可能引起胃部不適。
* 每日用量不超過600毫升（因咖啡因）。
* 服藥時間隔不少於1小時（單寧酸降低藥物吸收）。
* 儲存時——開封包裝3個月內飲用完畢，防止硒氧化。
* 神經興奮性高、失眠、高血壓患者，以及孕婦和哺乳期婦女——謹慎飲用。

**結語：**

鶴峰茶是一款有著從西晉史書到現代有機認證、從清朝宮廷貢品到摩洛哥港口集裝箱船歷史的茶葉。豐富的天然硒含量、武陵山脈的高山風土、土家族茶文化的活傳統和雄心勃勃的有機發展計劃，使這款茶不僅是美食現象，也是文化現象。在茶杯中，它展現明亮的綠色清新、純正的蘭花香氣和活潑、飽滿的滋味，即使經過五到七次沖泡也不會減弱。對於那些在一杯茶中珍視濃郁度和功能性的人來說，這是理想的選擇。
