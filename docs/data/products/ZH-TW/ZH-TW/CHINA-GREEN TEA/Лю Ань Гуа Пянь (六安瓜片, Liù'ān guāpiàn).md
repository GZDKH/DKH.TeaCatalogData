# 六安瓜片 (六安瓜片, Liù'ān guāpiàn)

**六安瓜片 (六安瓜片, Liù'ān guāpiàn)**
![Лю Ань Гуа Пянь (六安瓜片, Liù'ān guāpiàn)](images/Лю%20Ань%20Гуа%20Пянь%20(六安瓜片,%20Liù'ān%20Guāpiàn).png)

![Лю Ань Гуа Пянь (六安瓜片, Liù'ān guāpiàn)-1](images/Лю%20Ань%20Гуа%20Пянь%20(六安瓜片,%20Liù'ān%20Guāpiàn)-1.png)

六安瓜片是世界上唯一僅用單片葉子製作的綠茶，不含芽頭，不含葉梗。其名稱「六安的瓜子片」描述了加工後葉片形似瓜子的特徵。作為中國十大名茶之一、清代皇室貢茶、國家級非物質文化遺產（2008年認定）以及1971年贈送給季辛吉的國禮茶，六安瓜片以三大獨特之處在綠茶中獨樹一幟：「無芽無梗」(无芽无梗)、「老嫩分炒」(老嫩分炒)和「三昧真火」(三昧真火)——包括傳奇的「拉老火」(拉老火)工藝。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，不發酵茶。屬於特殊的「片茶」(片茶, 「葉片茶」)類別。
* **等級：** 中國十大名茶 (中国十大名茶)。國家級非物質文化遺產 (国家级非物质文化遗产, 2008年)。地理標誌產品。
* **產地：** 中國安徽省 (安徽省)，六安市 (六安市, Liù'ān Shì)，大別山 (大别山, Dàbié Shān) 山脈。主要產區：金寨縣 (金寨县, Jīnzhài Xiàn)——內山區 (内山)；裕安區 (裕安区, Yù'ān Qū)——外山區 (外山)。標準產區——齊山 (齐山 / 齐头山, Qí Shān) 和著名的蝙蝠洞 (蝙蝠洞, Biānfú Dòng)，位於響洪甸鎮齊山村 (响洪甸镇)。
* **地理座標：** ~北緯31°30′，東經115°50′。
* **別名：** 瓜片 (瓜片)——簡稱；片茶 (片茶, 「葉片茶」)；六安片 (六安片)。

**2. 歷史與文化意義：**

* **歷史：** 六安的茶葉歷史源遠流長。早在陸羽 (陆羽) 的《茶經》(《茶经》, 八世紀) 中就提到「廬州六安茶」(庐州六安茶)。明代學者徐光啟 (徐光启) 在《農政全書》(《农政全书》) 中稱「六安州之片茶，為茶之極品」(六安州之片茶，为茶之极品)。清代瓜片成為皇室貢茶 (贡茶)；據傳慈禧太后每月獲得十四兩「齊山雲霧」(齐山云雾)——瓜片的前身。現代形式的六安瓜片約形成於1905年：一說是六安的茶葉評鑑師開始從成品茶中只挑選嫩葉片，去除芽頭和葉梗；另一說是金寨 (齐头山后冲) 的茶農開始分別採摘和炒製不同成熟度的葉片。製成的茶葉形似葵花子，被稱為「瓜子片」(瓜子片, 「瓜子片」)，後簡稱「瓜片」。

  二十世紀，六安瓜片成為革命與外交之茶。周恩來總理 (周恩来) 通過六安的同志認識了這款茶，並終生鍾愛。1971年，亨利·季辛吉首次秘密訪華時，六安瓜片作為國禮茶 (国品礼茶) 贈送給他。在曹雪芹的《紅樓夢》(《红楼梦》) 中，茶被提及超過80次。

* **名稱：** 「六安」(六安)——市名（自漢代起）；「瓜片」(瓜片)——「瓜子片」——描述葉片形狀。

* **文化意義：** 六安瓜片是十大名茶中「最低調」的一款：在海外的知名度不如龍井或碧螺春，但「在十大名茶中品質聲譽最高」(在十大名茶中质誉最高)——據中國國家技術監督局定義。其獨特性是絕對的：世界上沒有其他綠茶僅用單片葉子製作。

**3. 植物學描述與原料：**

* **品種/栽培種：** 生長在大別山的本地 *Camellia sinensis* var. *sinensis* 群體品種。內山區 (内山) 使用富含氨基酸的原生品種。
* **採摘：** 相對於綠茶而言採摘較晚——穀雨 (谷雨, ~4月20日) 前後，±10天。原則：「求壯不求嫩」(求壮不求嫩)——與大多數重視最嫩芽頭的綠茶不同。立夏 (立夏, ~5月6日) 後季節結束：「立夏之後，再無瓜片」(立夏之后，再无瓜片)。
* **採摘標準：** 枝條上的第二、三片葉——成熟、肥厚、「壯」(壮, 強壯)。**無芽無梗**——獨特要求。採摘後進行「扳片」(扳片)——手工將每片葉子從葉梗和相鄰葉片分離，按成熟度分級。
* **「扳片」(扳片)——關鍵工序：** 採摘的枝條手工分解：先摘第三片葉，再摘第二片，然後第一片，最後分離上部嫩梗和下部粗梗/第四片葉。每個等級分別炒製。

**4. 風土條件與栽培特點：**

* **大別山 (大别山)：** 安徽-湖北-河南邊界的雄偉山脈。茶園位於北坡，在霧氣瀰漫的深谷中。森林覆蓋率超過50%。
* **內山區 (内山)：** 金寨——齊山 (齐山)、響洪甸 (响洪甸)、鮮花嶺 (鲜花岭)。海拔300-800公尺。最佳茶葉產自齊山 (齐头山, 804公尺)，特別是蝙蝠洞 (蝙蝠洞) 周圍。當地茶稱為「齊山名片」(齐山名片, 「齊山名片」)——最高等級。
* **外山區 (外山)：** 裕安——石板沖 (石板冲)、石婆店 (石婆店)、獅子崗 (狮子岗)。海拔100-300公尺。較平價的茶，常為機械生產。
* **氣候：** 年均溫約15.3°C。降雨量1,200-1,400毫米。濕度80%以上。山谷創造了霧氣豐富、日照時間短、晝夜溫差大的微氣候。
* **土壤：** 黃棕壤 (黄棕壤)，微酸性（pH約6.5），疏鬆深厚，富含有機質和礦物質。

**5. 製作工藝：**

六安瓜片的製作是世界綠茶中最費工的工藝之一。13道工序，從採摘到成品約需一週。三大獨特之處——「三獨特」(三独特)：

***獨特之一：「無芽無梗」(无芽无梗)：***
* **扳片 (扳片)：** 採摘後手工分解枝條——每片葉子從葉梗分離。葉片按成熟度分級：嫩葉（第1片）、中葉（第2片）、老葉（第3片）。去除芽頭和葉梗。為什麼？無芽頭——沒有「青草味」(青草味)；無葉梗——沒有苦澀（採摘時葉梗已木質化）。結果：「味濃而不苦，香而不澀」(味浓而不苦，香而不涩)。

***獨特之二：「老嫩分炒」(老嫩分炒)：***
* 每個等級（嫩葉、中葉、老葉）分別炒製，溫度和時間不同，因為含水量和葉片結構不同。
* **生鍋 (生锅, 「生鍋」)：** 在滾燙的鐵鍋中初炒（~100°C以上）。鈍化酶活性，開始整形。
* **熟鍋 (熟锅, 「熟鍋」)：** 在較溫和的溫度下二次炒製。最終整形——葉片貼鍋壁，邊緣向後捲曲 (叶缘背卷)，形成特徵性的「瓜子」形狀。

***獨特之三：「三昧真火」(三昧真火)：***
三個連續的火焙階段：
* **拉毛火 (拉毛火, 「拉毛火」)：** 在竹籃上用炭火初焙（~100°C），每2-3分鐘翻動一次。至約80%乾度。葉片呈「瓜子」形，顏色變翠綠。
* **拉小火 (拉小火, 「拉小火」)：** 二次焙火（~120°C）。至約90%乾度。形成香氣。然後「休息」3-5天 (吐绿/回疲, 「吐綠」) 使殘餘水分均勻分佈。
* **拉老火 (拉老火, 「拉老火」)：** 最終也是最壯觀的工序。溫度160-180°C。巨大的竹籃（直徑約1.5公尺）裝3-4公斤茶——兩人抬起籃子置於熾熱炭火上，保持2-3秒，移開，翻茶，再抬起——如此重複約150次。「抬籠要快，翻茶要勻，拍籠要準，腳步要穩，放籠要輕」(抬笼要快，翻茶要匀，拍笼要准，脚步要稳，放笼要轻)。葉片表面出現特徵性的「白霜」(白霜, bái shuāng)——糖類和氨基酸結晶。正是「拉老火」形成了著名的「寶綠」(宝绿) 色澤和瓜片獨特的香氣。

**6. 感官特徵：**

* **乾茶外觀：** 扁平片狀，形似瓜子 (瓜子形)——因此得名「瓜片」。邊緣向後捲曲 (叶缘背卷)。色澤「寶綠」(宝绿, bǎo lǜ)，表面有特徵性白霜 (白霜, bái shuāng)——「拉老火」工藝的痕跡。葉片整齊，大小均勻（等級由片段決定）。
* **乾茶香氣：** 高揚持久，有特徵性的炒栗香 (板栗香, bǎnlì xiāng)——瓜片的招牌香氣，由「三昧真火」形成。附加層次——淡雅花香（蘭花）、「綠色」香氣（新割草）、有時帶輕微煙燻味。
* **茶湯香氣：** 「清香高爽」(清香高爽, qīngxiāng gāoshuǎng)。栗香基調，花香和草本調。香氣持久——在3-5泡中保持。
* **滋味：** 「味濃而不苦，香而不澀」(味浓而不苦，香而不涩)。濃厚「醇厚」(醇厚, chúnhòu)。天然甘甜，有栗子堅果的深度。澀味極少（無芽頭和葉梗）。來自齊山山地土壤的輕微礦物感。回甘悠長甘甜，有「回甘」(回甘, huígān)。回甘明顯，漸強。
* **湯色：** 清澈明亮的黃綠色 (清澈明亮)。「活」色——如山澗清流。
* **葉底（沖泡後葉片）：** 完整扁平有彈性的葉片——無芽頭、無葉梗、無碎片。均勻的黃綠色。瓜片的葉底是評判品質的最佳方式：所有葉片應大小一致、完整、「活」。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 兒茶素含量高，特別是EGCG。成熟葉片（非芽頭！）比嫩芽含有更多多酚類——使瓜片成為「抗氧化」能力最強的綠茶之一。
* **氨基酸 (氨基酸)：** L-茶氨酸——含量中高。山地風土（800公尺以上、雲霧、溫差）有利於氨基酸積累。
* **咖啡因 (咖啡因)：** 含量適中——低於芽茶（芽頭含咖啡因最多；瓜片用葉片製作）。
* **維生素：** C（高含量——綠茶）、B₁、B₂、E、K。
* **礦物質：** 豐富的成分——齊山花崗岩和頁岩土壤。鉀、錳、鋅、鎂、磷。
* **芳香化合物：** 「三昧真火」形成的吡嗪和呋喃複合物——負責栗香。芳樟醇、香葉醇——花香層次。

**8. 健康功效：**

* **高抗氧化潛力：** 成熟葉片（非芽頭）含有更多多酚類——瓜片是綠茶中EGCG含量的「領導者」之一。
* **溫和提神：** 咖啡因 + L-茶氨酸。咖啡因含量低於芽茶——更溫和的提神效果，適合下午茶時間。
* **支持新陳代謝：** 兒茶素加速產熱和脂肪氧化。
* **心血管支持：** 多酚類改善血管彈性，調節膽固醇。
* **舒適消化：** 無芽頭和葉梗減少對胃的刺激。
* **礦物質補充：** 山地土壤提供豐富的礦物質成分。

**9. 沖泡方法：**

* **水溫：** 80-85°C。扁平成熟葉片對高溫的敏感性低於芽茶，但不建議沸水——可能「煮」壞味道。
* **茶量：** 3-5克/150-200毫升。
* **茶具：** 玻璃杯 (玻璃杯)——經典方法：可觀察扁平「瓜子」慢慢沉底展開。瓷蓋碗——更可控的萃取。小容量茶壺。
* **沖泡過程（「二道沖泡法」，二道冲泡法）：**
  1. 用熱水溫杯/蓋碗。
  2. 投入3-5克茶。
  3. 注入約1/3容量的水（80°C），靜置30秒——「醒茶」。
  4. 注滿至全容量。
  5. 浸泡2-3分鐘。觀察「瓜子舞蹈」。
  6. 喝至約2/3時，加入熱水（可用85°C）。
  7. 可沖3-5次。瓜片耐泡——成熟葉片比芽茶萃取更慢。
* **替代方法——功夫泡法 (功夫泡法)：** 4-5克入100-120毫升蓋碗，70-80°C，首泡60秒，之後45-60秒，4-6泡。

**10. 儲存：**

* **容器：** 密封不透光——鐵罐、鋁箔袋配拉鍊。
* **條件：** 冰箱（0-5°C）——強烈建議綠茶冷藏。密封包裝必須（綠茶易吸異味）。室溫下——2-3個月內飲用。
* **保質期：** 冰箱中6-12個月。新鮮瓜片（前3個月）——最佳栗香和新鮮度。葉片上的「白霜」(白霜)——新鮮標誌；如消失——茶葉老化。
* **天敵：** 光線、濕氣、氧氣、高溫、異味。

**11. 價格與假冒：**

六安瓜片作為「中國十大名茶」之一，價格相應較高。標準級（機械採摘，2-3葉）——200-600元/500克。高級（手工 (手工)，第1葉，齊山）——1,000-3,000元/500克。收藏級（蝙蝠洞，手工「拉老火」）——3,000-10,000+元/500克。

**如何避免假冒：**
* **形狀——「瓜子」：** 扁平片狀，無芽頭無葉梗，邊緣向後捲曲。如有芽頭——這不是瓜片。
* **「白霜」(白霜)：** 新鮮瓜片的特徵性霜狀物——「拉老火」的痕跡。無霜——老茶或簡化工藝的標誌。
* **栗香 (板栗香)：** 招牌標識。如香氣「平淡」或「草腥」而無栗子堅果深度——簡化生產。
* **產地：** 正宗瓜片——來自安徽省六安市裕安區 (裕安区) 和金寨縣 (金寨县)。關鍵產區：齊山 (齐山)、蝙蝠洞 (蝙蝠洞)、石頭壩 (石头坝)。
* **警惕四川或湖北的「六安瓜片」：** 常有較便宜的扁平綠茶冒充安徽瓜片銷售。

**12. 有趣事實：**

* **世界唯一「無芽梗」名茶：** 六安瓜片是「中國十大名茶」中唯一（也是世界少數）僅用葉片製作的茶——無芽頭無葉梗。這種「負向選擇」是矛盾之舉：茶界正統重視芽頭；瓜片證明無芽葉片同樣可以偉大。
* **季辛吉與周恩來（1971年）：** 據廣泛流傳的傳說，周恩來總理 (周恩来) 在1971年7月亨利·季辛吉秘密訪問北京期間，將六安瓜片作為國禮贈送給他——為準備尼克森的歷史性訪問。瓜片成為「開啟外交的茶」之一。
* **《紅樓夢》(红楼梦)：** 在曹雪芹的偉大小說中，六安茶被提及超過80次——證明其在清代精英中的受歡迎程度。
* **「拉老火」——中國茶業最壯觀的工序：** 兩人抬起裝茶的巨大竹籃置於熾熱炭火上，翻動放下——重複約150次。這不是流水線——這是需要完美協調的儀式，是茶界最令人印象深刻的視覺場景之一。
* **蝙蝠洞 (蝙蝠洞, 「蝙蝠洞」)：** 瓜片最著名的微產區。茶樹生長在喀斯特洞穴周圍的陡坡上，洞中有蝙蝠過冬。牠們的糞便富含磷和微量元素，當地茶農認為這創造了最佳批次的獨特風味。

**13. 與其他綠茶比較：**

* **頂谷大方 (顶谷大方, Dǐnggǔ Dà Fāng)：** 同樣扁平，同樣安徽產，同樣有栗香——但大方包含芽頭和葉片；瓜片只有葉片。大方是「龍井之祖」；瓜片是「獨立進化分支」。大方更溫和甘甜；瓜片更濃厚飽滿。
* **龍井 (龙井, Lóng Jǐng)：** 同樣扁平，但來自浙江。龍井用芽頭+葉片；瓜片只用葉片。龍井——豆香草香；瓜片——栗香。龍井「清淡」「清新」；瓜片「濃厚」「深沉」。
* **黃山毛峰 (黄山毛峰)：** 安徽「鄰居」。捲曲型（非扁平），白毫豐富。蘭花香。比「栗香濃厚」的瓜片更「花香」「輕盈」。
* **太平猴魁 (太平猴魁)：** 另一款安徽「偉大」綠茶。巨大扁平葉片帶「網紋」。蘭花香草香調。形狀更「奇特」，味道比瓜片更不「濃縮」。
* **碧螺春 (碧螺春, Bì Luó Chūn)：** 來自江蘇——捲曲「螺旋」狀，白毫豐富。花果香調。完全不同風格——「清淡」vs「濃厚」。

**結論：**

六安瓜片是叛逆之茶：在重視嫩芽的世界裡，它刻意拒絕芽頭；在重視捲曲美感的世界裡，它選擇簡單的瓜子形狀；在工藝追求可控極簡的世界裡，它經歷「三昧真火」，包括中國茶業最壯觀的工序。

然而，它仍躋身「中國十大名茶」，曾是周恩來的外交禮品，在《紅樓夢》中被提及超過80次。因為結果不言自明：濃厚飽滿、栗香甘甜的滋味，無法從任何其他原料或任何其他工藝中獲得。

無芽之葉。無情之火。無妥協之味。這就是六安瓜片。
