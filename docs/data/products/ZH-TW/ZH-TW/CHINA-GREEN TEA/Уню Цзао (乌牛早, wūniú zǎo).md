# 烏牛早 (乌牛早, wūniú zǎo)

**烏牛早 (乌牛早, wūniú zǎo)**

![Уню Цзао (乌牛早, wūniú zǎo)](images/Уню%20Цзао%20%28乌牛早,%20Wūniú%20Zǎo%29.png)

![Уню Цзао (乌牛早, wūniú zǎo)-1](images/Уню%20Цзао%20%28乌牛早,%20Wūniú%20Zǎo%29-1.png)

烏牛早是中國最早的綠茶之一，比著名的西湖龍井早整整一個月。其名稱中已蘊含其主要特色：「早」(zǎo)——「早期的」。這款茶擁有超過三百年的栽培歷史，是浙江省永嘉縣的名片，也是受地理標誌保護的國家產品。它常被稱為「江南第一早春茶」(早春江南第一茶)。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵，綠茶，lǜchá）。採用「殺青」(杀青) 處理方式——炒青固定（炒青，炒青）。
* **類別：** 地方名茶（名茶，míngchá），具有地理標誌保護（地理標志產品，dìlǐ biāozhì chǎnpǐn）。國家標準——GB/T 20360-2006「烏牛早茶」。自2004年起受原產地名稱保護（經中華人民共和國國家質量監督檢驗檢疫總局批准）。
* **產地：** 中國浙江省（浙江省，Zhèjiāng Shěng）溫州市（溫州市，Wēnzhōu Shì）永嘉縣（永嘉縣，Yǒngjiā Xiàn）。主要生產區域：烏牛鎮（烏牛鎮，Wūniú Zhèn，現為烏牛街道，烏牛街道）、羅東鄉（罗東鄉）、三江街道（三江街道）及楠溪江（楠溪江，Nánxī Jiāng）下游、甌江（瓯江，Ōu Jiāng）北岸的鄰近地區。
* **地理座標：** 約北緯28.15°，東經120.69°（永嘉縣烏牛地區）。

**2. 歷史與文化意義：**

* **歷史：** 烏牛早擁有超過300年的有文獻記載的栽培歷史。根據地方史料記載，約兩百年前，這是一株野生茶樹，發現於甌北鄉半嶺村（半嶺村）與烏牛鎮嶺下村（嶺下村）之間的長夾嶺（長夹嶺）山脊上。甌北鄉龍頭村（龍頭村）的農民金則洪（金則洪）在新年拜訪親戚時，注意到山坡上一株特別茂盛且早醒的茶樹，便連土挖起移植到自己的田地裡。由於這株植物在春分（春分）節氣就開始萌芽——比其他品種早15天，因此得名「烏牛早」(乌牛早，「烏牛的早茶」)。當地人也稱其為「嶺下茶」(岭下茶，「山嶺下的茶」)。

  數十年來，該品種在羅東和烏牛鄉鎮傳播，成為該地區的主要作物。1930年代，用烏牛早原料製作炒青，經精製後以「天都珍眉」(天都珍眉) 品牌銷往上海，售價比徽州（徽州）同類產品高10銀元。1950年代後，這種原料被用來製作紅毛茶、紅青和炒青。然而，該品種後來在很大程度上失傳，直到**1985年**才得以恢復。**1988年**，「烏牛早龍井」(乌牛早龙井) 產品通過杭州省級新產品評審，正式命名為「永嘉烏牛早」(永嘉乌牛早)。**1994年**，在烏牛鎮建立了80公頃的連片茶園基地。**1995年**，該茶榮獲第二屆中國農業博覽會金獎和香港優質食品展覽會金獎。**2002年**獲得國家「綠色食品」(绿色食品) 認證。**2004年12月13日**，中華人民共和國國家質量監督檢驗檢疫總局批准對「烏牛早茶」實施原產地域保護。目前，永嘉縣擁有近5萬畝（約3333公頃）烏牛早茶園，年產值約30億元人民幣。

* **名稱：** 名稱由三個漢字組成：「烏」(wū)——「黑色、深色」和「牛」(niú)——「牛」組成地名烏牛(乌牛)——永嘉縣一個鎮（現為街道）的名稱，據傳說源於甌江岸邊一塊形似黑牛的岩石；「早」(zǎo)——「早期的」——指出該品種的主要特點：春季極早萌芽。

* **文化意義：** 在永嘉縣流傳著一個美麗的傳說，將茶的起源與觀音菩薩（觀音）聯繫起來。據傳說，在普陀島觀音竹林中生長著一株神聖的茶樹，香氣迷人。有一天，天牛被香氣吸引，偷走了一根枝條逃跑。觀音追趕到風景如畫的甌江岸邊，擊中了牛並將其變成石頭，而被偷的枝條落入山中，成為烏牛茶樹的起源。永嘉縣享有「中國烏牛早茶之鄉」(中国乌牛早之乡) 的榮譽稱號。三江街道的行禪村（行禅村）成為聯合國教科文組織研究的對象，作為農民通過茶葉種植成功脫貧的典型例子。

**3. 植物學描述與原料：**

* **種類：** *Camellia sinensis* (L.) Kuntze var. *sinensis*。
* **品種/栽培種：** 烏牛早 (乌牛早)，也註冊為嘉茗1號 (嘉茗1号，Jiāmíng yī hào)——無性系 (无性系，wúxìngjì) 品種，通過從永嘉縣當地群體中個體選育而成。植物學特徵：灌木型 (灌木型)，中葉類 (中叶类)，**特早生種** (特早生种，tè zǎo shēng zhǒng) 生長期。樹姿半開展 (半开展)，分枝中等密度。葉片橢圓形，綠色，有明顯光澤，表面略凸。芽葉粗壯，肉質。花朵單生或假總狀花序，花冠直徑約35 × 29毫米，6-7片花瓣，4-5片萼片。結果性弱。抗寒性較強。
* **採摘：** 極早期：採摘開始——2月底（約2月25日），結束——不晚於4月5日（清明，清明）。整個產量都是「明前茶」(明前茶，míngqián chá)。整個採摘季節持續約50天。休眠芽在平均日溫穩定在8°C時開始萌動。烏牛早比西湖龍井早30-40天。
* **採摘標準：** 特級 (特级)——一芽一葉初展 (一芽一叶初展)。一級和二級——一芽二葉初展 (一芽二叶初展)。手工採摘，採用「提手採」(提手采) 方法。芽葉應大小均勻，無魚葉 (鱼叶)、葉柄和雜質。500克特級成品茶需要約22,000個鮮芽。
* **原料要求：** 新鮮採摘，完整，無機械損傷和過熱。從茶園到工廠的運輸——盡可能快速，以避免自發氧化的開始。

**4. 風土條件與栽培特點：**

* **生長海拔：** 主要茶園位於楠溪江和甌江沿岸的低山丘陵和緩坡上，海拔50-300公尺。
* **氣候：** 亞熱帶季風氣候。年平均溫度——18.3°C，年溫差——約20°C。高於8°C的活動積溫——5742.5°C。最冷月平均溫度——8.1°C。無霜期——282天。年降水量——約1500-1800毫米，分佈相對均勻。
* **日照：** 年平均日照時數——1798.9小時（平均每天4.9小時），日照百分率——約41%，有利於茶葉中含氮和芳香物質的積累。
* **微氣候：** 東海的鄰近性確保了高空氣濕度。環繞楠溪江谷地的山脈造成頻繁的霧氣和散射光照——這些是優質茶葉的理想條件。春季來得早，這解釋了烏牛早驚人的早熟性。
* **土壤：** 主要為紅壤/黃壤 (红壤/黄壤)，酸性和弱酸性（pH 4.5-5.5），有機質含量高，排水良好。有森林腐殖質的山坡提供天然的土壤肥力。
* **農業技術：** 茶園位於通風良好的坡地。實行生態農業：有機肥料、木屑覆蓋、定期修剪。近年來採用農業技術措施進一步加速萌芽：厚木屑覆蓋層、優化修剪和科學的有機施肥。

**5. 製作工藝：**

烏牛早的製作工藝類似於龍井 (龙井) 茶系的加工，旨在製作扁形 (扁形，biǎnxíng) 綠茶，具有清香和嫩味。關鍵步驟：

* **採摘 (采摘——cǎizhāi)：** 手工採摘，從清晨開始。採摘的原料立即用竹籃運送到工廠，不壓實。

* **攤晾 (摊晾——tānliàng)：** 鮮葉薄攤（約1公斤/平方公尺）在清潔的竹盤上，置於通風陰涼處。持續時間——直到失水20%，通常6-12小時。目的——平衡水分，軟化葉片，初步形成香氣。當葉片手感柔軟，略微變暗並帶有淡淡花香時即為合適。關鍵是不能出現紅變、過熱和葉片受壓。

* **青鍋固定 (青锅——qīngguō)：** 主要的「殺青」(杀青，shāqīng) 階段——在高溫鍋中炒制以鈍化酶活性並停止氧化。同時開始用師傅手掌的特有動作塑形：按壓、滑動和撫平。形成香氣基礎：栗香和堅果香調。

* **輝鍋精製 (辉锅——huīguō)：** 在較低溫度下繼續炒制。葉片最終乾燥、扁平，獲得特有的光澤和光滑度。師傅控制壓力和動作速度，達到形狀均勻。

* **簸片割末 (簸片割末——bǒpiàn gēmò)：** 成品茶過篩去除茶末、碎片和葉片碎屑，然後按等級分類。

**6. 感官特徵：**

* **乾茶外形：** 扁平、光滑、挺直、緊實 (扁平光滑，挺秀匀齐)。芽尖明顯 (芽锋显露)，帶有輕微白毫 (微显毫)。色澤——鮮嫩綠色帶油潤光澤 (嫩绿光润)。芽葉肉質，形似雀舌 (雀舌，quèshé)。特徵：下端（葉柄切口處）可能呈現深褐色調——所謂「黑屁股」(黑屁股)，這是炒制時肉質葉片細胞液滲出造成的；這是烏牛早的品種特徵，而非缺陷。

* **乾茶香氣：** 清純、清新，帶有明顯的栗香 (栗香，lìxiāng)。在手掌中搓揉時散發淡淡的草香和甜香。

* **茶湯香氣：** 高揚、鮮爽、持久 (高鲜持久)。以炒栗子和豆花香 (豆花香) 為主，帶有淡雅花香背景。香氣留存在空杯中，逐漸轉為淡淡甜香。

* **滋味：** 甘醇鮮爽 (甘醇鲜爽)，帶有明顯的鮮嫩感。茶體中等，飽滿。苦澀味極少，快速轉為回甘 (回甘，huígān)。餘韻——悠長、清潔、清爽，帶有清涼礦物質調。

* **湯色：** 嫩綠明亮 (嫩绿明亮)，透明有光澤 (清澈)。隨著冷卻可能略微泛黃。

* **葉底（沖泡後的葉片）：** 鮮亮翠綠，肉質，均勻。葉片完整展開，形成整齊的「朵」(匀齐成朵)。芽葉緊實，手感有彈性，表明營養物質含量高。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 根據國家標準GB/T 20360-2006，成品茶中多酚含量——20.1-29.5%。特早採摘的特點是數值較低（接近20%），這解釋了其柔和性和低澀味。
* **氨基酸 (氨基酸)：** 4.3-5.3%——顯著高於綠茶平均水平（通常2-4%）。L-茶氨酸 (L-茶氨酸) 含量高，賦予明顯的鮮味成分和鎮靜效果。據中國農業科學院（CAAS）數據，鮮葉中氨基氮含量——565.0毫克%。
* **兒茶素 (儿茶素)：** 約103.81毫克/克（據CAAS鮮葉數據）。主要成分：EGCG、EGC、ECG、EC。
* **生物鹼：** 咖啡鹼 (咖啡碱)——綠茶典型的乾重2-4%。可可鹼和茶鹼——微量。咖啡因/茶氨酸比例有利於溫和提神而無明顯興奮。
* **維生素：** 抗壞血酸（維生素C）——特早採摘特有的高含量；B族維生素（B₁、B₂）、維生素E、β-胡蘿蔔素。
* **礦物質：** 鉀、鎂、磷、鋅、錳、氟、硒。
* **精油和芳香化合物：** 關鍵成分：芳樟醇、β-紫羅蘭酮、壬醛、香葉醇、苯甲醛。正是它們形成了特有的栗子花香調。可溶性糖含量高，有助於天然甜味感。
* **成分特點：** 低多酚-氨基酸係數 (酚氨比 < 5)——早期採摘高品質綠茶的經典標誌。

**8. 保健功效：**

* **溫和提神和認知支持：** 咖啡因和L-茶氨酸的良好比例提供持久而溫和的提神效果——改善注意力集中而無緊張感和「咖啡因崩潰」。
* **抗氧化作用：** 兒茶素（特別是EGCG）是強效抗氧化劑，中和自由基並減緩氧化應激。
* **心血管系統支持：** 綠茶多酚有助於降低「壞」膽固醇（LDL）水平並維持血管彈性。
* **血壓調節：** 定期飲用高茶氨酸含量的綠茶與溫和的降壓作用相關。
* **消化支持：** 適度的澀味刺激胃液分泌，有助於餐後消化。
* **免疫力增強：** 維生素C、兒茶素和微量元素（鋅、硒）的複合物支持機體防禦功能。
* **口腔健康：** 所含氟和兒茶素具有抑菌作用，預防蛀牙並清新口氣。
* **抗炎作用：** EGCG和其他多酚降低促炎標誌物水平。

*注意：對咖啡因敏感者以及胃腸道疾病急性期患者應限制飲用或諮詢醫生。*

**9. 沖泡方法：**

* **水溫：** 標準批次80-85°C；高級品種嫩芽75-80°C。不使用沸水——會「燙傷」葉片並加重苦味。
* **茶量：** 3-4克配150-200毫升水（玻璃杯沖泡）；5克配100-120毫升（蓋碗，快沖法）。
* **茶具：** 玻璃杯 (玻璃杯)——最適合觀察芽葉展開和湯色；瓷蓋碗 (盖碗)——更可控和多次沖泡；瓷茶壺——日常飲茶。

* **過程（玻璃杯沖泡）：**

1. 用熱水溫杯，倒掉。
2. 投入3-4克乾茶。
3. 注入80-85°C水至1/3容量，輕搖杯子，讓茶葉「甦醒」30秒。
4. 加水至滿杯。
5. 浸泡1.5-2分鐘後開始品飲。
6. 飲至剩餘1/3時加水。重複2-3次。

* **過程（蓋碗快沖法）：**

1. 用沸水溫蓋碗和公道杯。
2. 投入5克茶，聞溫熱茶葉香氣。
3. 第一泡：80°C水，浸泡15-20秒，倒入公道杯，分入茶杯。
4. 第二、三泡：20-30秒。
5. 後續各泡：每泡增加10-15秒。
6. 沖泡次數：4-6泡（高級品可沖6泡）。

**10. 儲存：**

* 烏牛早是精緻的綠茶，需要嚴格遵守儲存條件。
* **容器：** 密封——抽真空鋁箔袋，放入密封的鐵罐或錫罐中。
* **溫度：** 最佳——冰箱0-5°C（蔬菜室）。室溫儲存僅適用於1-2週內飲用的量。
* **茶葉天敵：** 光線、濕氣、異味、熱量。茶葉積極吸收氣味——與強烈氣味的食品分開儲存。
* **保質期：** 最佳風味——生產後6-12個月內飲用。正確冷藏儲存——18個月內品質無顯著下降。不宜陳放——隨時間失去新鮮度和香氣亮度。
* 不建議頻繁從冰箱取出包裝——冷凝水加速品質劣化。

**11. 價格與仿冒：**

* **價格類別：** 中國綠茶中的中等偏上。價格很大程度上取決於採摘時間：最早批次（2月底-3月初）價格顯著更高。新鮮茶青 (茶青) 在早期採摘高峰期達到400元/公斤以上，一週後下降100+元/公斤。成品特級茶——根據生產商、年份和批次，800-2000+元/公斤。
* **常見仿冒和替代：** 由於烏牛早外觀與西湖龍井非常相似，這個品種經常以昂貴龍井的名義銷售。相反情況也存在：其他地區的茶葉冒充「正宗」永嘉烏牛早。主要造假類型：
  * 以「西湖龍井」品牌銷售烏牛早茶，價格成倍虛高。
  * 用其他省份（四川、貴州）原料替代，這些地方廣泛種植烏牛早作為「早茶」品種。
  * 將去年茶葉重新包裝為當年新鮮「明前」茶。
  * 對低級茶葉加香以模仿栗子香氣。
* **如何區分真正烏牛早與龍井：**
  * **色澤：** 烏牛早——鮮綠帶黃色調，切口處有特徵性深色「尾巴」；龍井——帶「糙米色」(糙米色) 調和葉柄處紅點。
  * **香氣：** 烏牛早——草香栗香，較為清淡；龍井——濃郁豆香栗香帶蘭花調。
  * **時間：** 真正龍井不可能在2月出現在市場上——這肯定是烏牛早。
  * 向可靠商家購買，尋找地理標誌產品 (地理标志产品) 標識和GB/T 20360認證。

**12. 有趣事實：**

* **早熟記錄。** 烏牛早是世界上最早的商業綠茶之一。在有利條件下的某些年份，首批芽葉在2月初至中旬就開始採摘，即在元宵節 (元宵节，燈節) 之前。這使其成為中國所有註冊 *Camellia sinensis* 品種中早期萌芽的絕對記錄保持者。

* **龍井「替身」。** 由於視覺相似性，烏牛早常被稱為「平價龍井」。正因如此，2月底-3月初市場上出現的大部分「早期龍井」實際上是按龍井工藝加工的烏牛早茶。

* **栽培種的全國推廣。** 由於極早熟性，烏牛早栽培種 (嘉茗1号) 成為全中國最廣泛種植的早期品種之一。它被種植在浙江省以外的廣大地區——四川、貴州、湖北、湖南等省，當地用烏牛早原料生產各種綠茶，包括碧螺春和扁形茶。

* **聯合國教科文組織村莊。** 永嘉縣三江街道的茶村行禪村成為聯合國教科文組織研究對象，作為「茶葉」脫貧模式的成功範例：村中幾乎每戶都種植烏牛早，從早春銷售中獲得穩定收入。

* **半公斤茶需22,000個芽。** 生產500克特級烏牛早需要手工採摘超過22,000個精選芽葉——約5公斤鮮葉製1公斤成品。

烏牛早在茶體厚度和香氣深度上不及龍井，但憑藉早期上市和更親民的價格取勝。與碧螺春相比——完全不同的風格：扁平形狀和栗香特徵而非螺旋捲曲和果香。與安吉白茶的共同點是高氨基酸含量和柔和性，但烏牛早在特性和香氣上明顯更「綠」。

**結語：**

烏牛早是一款在二月寒意尚存時就為茶桌帶來春意的茶。它的主要價值不在於追求精緻的複雜性，而在於最初春葉的真誠、純淨的新鮮感。扁平優雅的芽葉在杯中展開為翠綠的茶湯，帶著淡雅的栗香和柔和甘甜的滋味，完全沒有粗糙感。這是珍視自然甦醒時刻的人們的理想茶品——那一口確認春天已來的茶湯。

給它柔軟的水和適中的溫度，不要急躁——它會回報以整潔、清晰、清爽的茶飲體驗。如果您在2月底的茶攤上看到「龍井」，要知道：很可能這就是烏牛早。這並沒有什麼不好——只是春天比往常來得早了一些。
