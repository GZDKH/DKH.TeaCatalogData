# 桐城小花 (桐城小花, Tóngchéng xiǎo huā)

**桐城小花 (桐城小花, Tóngchéng xiǎo huā)**

![Тунчэн Сяо Хуа (桐城小花, Tóngchéng xiǎo huā)](images/Тунчэн%20Сяо%20Хуа%20%28桐城小花,%20Tóngchéng%20Xiǎo%20Huā%29.png)

![Тунчэн Сяо Хуа (桐城小花, Tóngchéng xiǎo huā)-1](images/Тунчэн%20Сяо%20Хуа%20%28桐城小花,%20Tóngchéng%20Xiǎo%20Huā%29-1.png)

桐城小花是來自安徽省的歷史綠茶，屬於「蘭花茶」(兰花茶，「具有蘭花香氣的茶」) 家族。其獨特之處不在於人工香化，而在於天然的蘭花香氣，這種香氣源於與茶樹一同生長在龍眠山坡上的野生蘭花。創製於明代，數百年來一直與中國最優質的茶葉並列，並獲得「貢茶」——皇室貢品的地位。當地俗語「品不減龍井」——「品質不遜於龍井」——準確地傳達了這款茶的特性：謙遜但深度精緻。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵，綠茶，lǜchá）。加工方式——烘青 (烘青，hōngqīng)——殺青後以熱風乾燥（與龍井的炒青不同）。
* **類別：** 歷史名茶 (历史名茶，lìshǐ míngchá)，屬於皖西蘭花茶 (皖西兰花茶，「安徽西部蘭花茶」) 家族。受地理標誌保護：2018年中華人民共和國農業農村部批准了「桐城小花」的農產品地理標誌 (农产品地理标志)。2025年中華人民共和國國家知識產權局認定其為地理標誌保護產品 (地理标志产品)。省級標準——DB34/T 586-2006。
* **產地：** 中國安徽省 (安徽省，Ānhuī Shěng) 桐城市 (桐城市，Tóngchéng Shì)。主要生產區域：大關鎮、呂亭鎮、黃甲鎮、唐灣鎮、青草鎮，以及龍眠街道 (龙眠街道) 和文昌街道 (文昌街道)——共7個行政單位，33個村莊。生產核心——龍眠山 (龙眠山，Lóngmián Shān) 山區，霍山山脈的東南支脈，屬大別山 (大别山，Dàbiéshān) 系統。
* **地理座標：** 北緯30°39′–31°16′，東經116°40′–117°09′（地理標誌保護區域資料）。核心——楊頭 (杨头) 和黃草尖 (黄草尖) 峰區，海拔約800–1000公尺。

**2. 歷史與文化意義：**

* **歷史：** 桐城的茶葉種植傳統可追溯至唐代 (唐，618–907)，但現今稱為「桐城小花」的茶葉創製則歸於明代 (明，1368–1644)。據史料《桐舊集》記載，魯山公——孫晉 (孙晋) 的尊稱，曾任明朝大司馬 (大司马，國防部長)——從公務旅行中帶回特殊的茶種，種植在龍眠山椒園 (椒园) 地區。這個茶園產出的茶，稱為「椒園茶」(椒园茶)，很快便與著名的顧渚 (顾渚) 和蒙頂 (蒙顶) 並提，被列為「貢茶」——皇室貢品。後來，由於沖泡時展開的嫩芽像盛開的花朵，香氣如野蘭花，茶葉獲得詩意的名稱「桐城小花」——「來自桐城的小花」。

  清代詩人姚興泉 (姚兴泉) 在詩集《龍眠雜憶》(《龙眠杂忆》) 中讚頌此茶：「桐城好，谷雨試新鐺，椒園異種分遼薊，石鼎連枝販霍英，活火帶雲烹」——「桐城好：穀雨時節試新鍋，椒園異種遠銷遼薊，石鼎煮霍山嫩芽，活火在雲中烹茶」。《桐城風物記》(《桐城风物记》) 也記載：「品不減龍井」——「品質不遜於龍井」。

  近代史中：**1949年** 開始茶業現代化，引進改良品種和機械化生產。**1986年** 茶葉獲得「安徽省名茶」(安徽省名茶) 稱號。**1991年** 獲星火科技成果博覽會 (星火科技成果博览会) 銀獎。**1993年** 獲中華人民共和國商務部嘉獎。**1998年** 獲國際茶葉展覽會推薦產品稱號。**1999年** 獲中國國際農業展覽會名牌產品地位。**2007年** 獲中茶杯 (中茶杯) 競賽一等獎。**2010和2012年** 在中國國際茶葉展覽會上獲得「安徽十大品牌名茶」(安徽十大品牌名茶) 稱號。**2013年** 茶葉入選「國禮徽茶」(国礼徽茶) 系列。**2015和2017年** 入選國家名優新農產品目錄。**2018年** 註冊農產品地理標誌。**2025年** 獲得地理標誌保護產品 (地理标志产品) 地位。

* **名稱：** 「桐城」(Tóngchéng) ——安徽省城市和縣名，字面意思為「梧桐之城」；「小」(xiǎo) ——「小的」；「花」(huā) ——「花」。名稱指其特有的展開嫩芽形狀，像小小的盛開花朵，以及明顯的花香（蘭花香）。茶葉也稱為「龍眠茶」(龙眠茶，「睡龍茶」——以龍眠山命名) 和「小蘭花茶」(小兰花茶，「小蘭花茶」)。

* **文化意義：** 桐城是安徽省主要文化中心之一，以「桐城派」(桐城派，Tóngchéng Pài) ——18-19世紀最具影響力的散文流派而聞名。該城擁有非正式稱號「文都」(文都，「文化之都」)。桐城小花茶自然融入這一文化傳統：精緻、低調，具有文學聯想。著名的「六尺巷」(六尺巷，「六尺寬的巷子」) 故事——關於相互謙讓的傳說，與清朝宰相張廷玉 (张廷玉)（桐城人）有關——也與當地茶葉相關聯。據傳正是張廷玉稱讚小花：「色澄秋水，味比蘭花」——「色澄如秋水，味似蘭花香」。當地茶道和諧公式為：「龍眠山上茶，紫來橋下水」——「龍眠山上的茶，紫來橋下的水」。

**3. 植物學描述與原料：**

* **種類：** *Camellia sinensis* (L.) Kuntze var. *sinensis*。
* **品種/栽培種：** 傳統基礎——龍眠群體種 (龙眠群体种，lóngmián qúntǐzhǒng) ——當地群體（種子）種植，代表遺傳多樣化的品系混合，數百年來自然適應龍眠山地形。也允許使用從當地群體中選育的無性系品種：舒茶早 (舒茶早，Shūchá Zǎo)、山坡綠 (山坡绿，Shānpō Lǜ) 和其他來自大別山區域的近緣品種，保持特有的「色翠湯清，蘭香甜韻」(色翠汤清，兰香甜韵) 特徵。
* **採摘：** 傳統上——從穀雨 (谷雨，約4月20日) 開始及以後；隨著早熟無性系品種的引進，採摘開始時間提前到清明 (清明，約4月5日) 之前。特級——一芽一葉初展；一級和二級——一芽二至三葉。
* **採摘標準：** 芽葉應肥壯 (壮实)、勻整 (匀整)、茸毛顯露 (茸毛显露)。特級芽長約2.5–3公分。500公克成品茶需要8,000–10,000個芽頭。
* **原料要求：** 新鮮採摘，無機械損傷，無過熱。在晴朗乾燥天氣中採摘。

**4. 風土條件與栽培特點：**

* **生長海拔：** 海拔400至1000公尺。優質生產核心（楊頭、黃草尖、黃嶺）——海拔600–1000公尺。此區域基地種植園約140公頃。
* **氣候：** 亞熱帶濕潤季風氣候。年平均氣溫16°C（山區14.5°C）。降雨量1200–1400毫米/年。四季分明，光照充足。
* **微氣候：** 龍眠山——霍山山脈東南支脈，屬大別山系統——特點是深谷、頻繁雲霧、高濕度。關鍵特色：**野生蘭花 (兰草，lán cǎo) 在茶樹間的山坡上大量生長**，形成天然香化背景。高覆蓋率的山林提供散射光照，有利於胺基酸和芳香物質的積累。
* **土壤：** 砂壤土 (砂壤土，shā rǎng tǔ)，酸性和弱酸性（pH 5.0–6.5），有機質含量高。富含微量元素——鋅和硒。水文系統由四個河流系統（大沙河、掛車河、龍眠河、孔城河）形成，流入菜子湖，再入長江。山溪清澈，富含礦物質。
* **農業技術：** 生態農業：人工除草（無除草劑），有機肥料。高山種植園（楊頭）由合作社按「六統一標準」模式管理（統一種植、管理、採摘、包裝、品牌、銷售）。新種植優先使用龍眠當地群體種子材料，以保持真實特性。

**5. 生產工藝：**

桐城小花採用烘青 (烘青) 工藝——綠茶熱風乾燥技術。工藝旨在最大程度保持天然蘭花香氣，創造特有的「花朵」形狀。傳統和現代生產線在成型階段有所不同。

* **攤放-萎凋 (摊放 — tānfàng)：** 鮮葉在通風室內攤成薄層，以平衡濕度、軟化組織並初步發展香氣。持續時間4–8小時。

* **殺青 (杀青 — shāqīng)：**
  * *傳統方法：* 分兩階段炒鍋——「生鍋」(生锅，shēngguō) 高溫快速鈍化酶活性，「熟鍋」(熟锅，shúguō) 中溫初步成型。
  * *現代方法：* 滾筒或平板殺青機械，溫度可控。

* **理條-整形 (理条 — lǐtiáo)：** 機械或手工將芽葉整理成直條狀，略扁平，似展開的花蕾。此階段存在於現代生產線；傳統工藝中「熟鍋」部分承擔此功能。

* **初烘 (初烘 — chūhōng)：** 熱風 (热风，rèfēng)，溫度100–110°C。葉片乾燥至含水量約20–25%。使用烘乾篩或機械烘乾設備。

* **攤涼 (摊凉 — tānliáng)：** 茶葉攤放，使殘餘水分從葉心重新分佈到表面。持續30–60分鐘。

* **復烘 (复烘 — fùhōng)：** 較低溫度（約80–90°C）至含水量5–7%。

* **剔揀 (剔拣 — tījiǎn)：** 手工或機械去除缺陷葉、梗、茶末。分級：特級 (特级)、一級 (一级)、二級 (二级)、三級 (三级)。

* **提香 (提香 — tíxiāng)：** 中溫（約60–70°C）短時間加熱，「提升」香氣並固定最終風味特徵。正是在此階段蘭花香調最為明顯。

**6. 感官特徵：**

* **乾茶外形：** 芽葉挺直，略扁平，展開——像小小的盛開花朵或展開的蘭花 (形似兰花)。色澤翠綠 (色泽翠绿)，鮮活有光澤 (鲜活)。芽葉完整，茸毛明顯。

* **乾茶香氣：** 清純、清新，有明顯蘭花香調 (兰花香，lánhuā xiāng) ——此茶的名片。野花和鮮草香調。

* **茶湯香氣：** 高揚、清新、持久 (清鲜持久)。主導蘭花香——優雅，非人工香精，而是天然的，如山風吹過野蘭花叢。隨著冷卻顯現蜂蜜和草本底調。

* **滋味：** 濃郁、鮮爽，明顯鮮甜 (鲜醇回甘)。茶體中等，對烘青茶而言較厚實。輕微澀味迅速轉為持久回甘 (回甘)。尾韻清淨，有礦物餘味。清朝宰相張廷玉以「味比蘭花」——「味似蘭花」來定義。

* **湯色：** 嫩綠帶淡黃色調，透明明亮 (嫩绿明亮)。「清澈如秋水」(色澄秋水)。

* **葉底（沖泡後葉片）：** 嫩勻、均勻、翠綠明亮 (嫩匀绿明)。芽葉完全展開，顯示「芽+葉」結構完整性。質地柔軟、有彈性、鮮活。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 安徽高山烘青茶典型水平——18–26%。科學研究比較四個栽培種顯示，用於桐城小花的品種間多酚類、咖啡因和水溶性提取物含量無統計學差異。
* **胺基酸 (氨基酸)：** 含量較高——龍眠山區茶葉特色。當地群體種 (群体种) 和鳧早二號 (凫早二号) 顯示最高胺基酸水平（茶胺酸、天門冬胺酸、精胺酸）。低多酚-胺基酸係數（< 5）確保柔和度和明顯鮮味成分。
* **兒茶素 (儿茶素)：** 主要成分——EGCG、EGC、ECG、EC。有趣的是，不同栽培種兒茶素譜不同：舒茶早含更多EGC、EC和ECG，而當地群體種含更多茶胺酸。
* **生物鹼：** 咖啡因——綠茶典型水平（乾重2–4%）。可可鹼和茶鹼——微量。
* **維生素：** 維生素C含量較高（據中國農科院資料），以及B群維生素、維生素E、β-胡蘿蔔素。
* **礦物質：** 鋅和硒——龍眠山土壤特有微量元素，反映在茶葉礦物質譜中。還有鉀、鎂、錳、氟、磷。
* **精油：** 芳樟醇、香葉醇、β-紫羅蘭酮、順式茉莉酮、苯甲醇——負責特有蘭花香調。茶樹與龍眠山坡野生蘭花（蘭屬）的鄰近，似乎不僅創造嗅覺聯想，也影響種植園微環境中的微生物群和揮發性化合物。

**8. 保健功效：**

* **抗氧化作用：** 兒茶素複合物（EGCG、EGC）保護細胞免受氧化應激和自由基損害。
* **溫和提神：** 咖啡因和L-茶胺酸的平衡比例提供持續專注力，無緊張感和能量「崩潰」。
* **心血管系統支持：** 綠茶多酚有助於膽固醇水平正常化和維持血管彈性。
* **免疫力增強：** 維生素C、鋅、硒和兒茶素複合物支持機體防禦功能，特別是在季節交替時。
* **消化支持：** 適度澀味和多酚刺激腸蠕動和消化液分泌，促進食物消化。
* **口腔健康：** 氟和兒茶素抗菌特性有助於預防蛀牙和保持口氣清新。
* **認知支持：** L-茶胺酸促進大腦α波活動，改善專注工作和創造性思維能力。
* **抗炎作用：** EGCG降低促炎細胞因子水平。

*注意：對咖啡因敏感者和胃腸道疾病急性期患者建議適量飲用。綠茶不宜空腹飲用。*

**9. 沖泡方法：**

* **水溫：** 標準批次80–85°C；特級嫩芽75–80°C。
* **茶量：** 杯泡3–4公克配150–200毫升；蓋碗5公克配100–120毫升。
* **茶具：** 玻璃杯——欣賞展開芽葉「花朵」舞蹈；瓷蓋碗 (盖碗) ——可控多次沖泡；瓷茶壺——日常品茗。據傳統，理想用水來自桐城紫來橋 (紫来桥) 下泉水。

* **過程（杯泡）：**

1. 用開水溫熱玻璃杯，倒掉。
2. 投入3公克乾茶。聞香——乾葉應有蘭花香調。
3. 注入80°C水約1/3容量。輕搖杯子，讓茶葉「甦醒」30–40秒。觀察「花朵」展開。
4. 注滿水至全容量。
5. 浸泡2–3分鐘後開始品飲。
6. 飲至1/3水位，續水。重複2–3次。

* **過程（蓋碗）：**

1. 溫熱蓋碗和茶海。
2. 投入5公克，聞溫熱茶葉香氣。
3. 第一泡：80°C，20–25秒。倒出。
4. 第二、三泡：25–35秒。
5. 後續：增加10–15秒。
6. 沖泡次數：4–6次。

**10. 儲存：**

* **容器：** 密封——鋁箔袋抽真空，放入密封鐵罐或陶瓷容器。
* **溫度：** 最佳0–5°C（冰箱）。短期（2–4週）可室溫陰涼乾燥處。
* **茶葉天敵：** 濕氣、光線、熱度、異味。小花蘭花香特別容易劣化——冷藏對保持香氣至關重要。
* **保質期：** 最佳——生產後6–12個月。正確冷藏可達18個月。不適宜陳放。

**11. 價格與仿冒：**

* **價格區間：** 安徽綠茶中等價位。核心產區（楊頭、龍眠山）特級 (特级) ——600–1500+元/公斤。一般一、二級——200–500元/公斤。價格很大程度取決於生長海拔和採摘時間（明前茶較貴）。
* **價格因素：** 種植園海拔（高山>平地）；採摘時期（明前>雨前）；栽培種（當地群體種>引進無性系）；加工方式（手工>機械）。
* **避免仿冒：**
  * 注意芽葉形狀：真正小花沖泡時展開如小花朵，保持「芽+葉」完整性。粗糙捲曲或破碎葉片是替代品徵象。
  * 香氣：天然蘭花調——細膩、「呼吸感」；人工香化呈現刺鼻、香精味、快速消失的氣味。
  * 湯色：應清澈透明，無混濁。暗淡或黃褐色湯色是陳舊或劣質茶徵象。
  * 價格：可疑低價（所謂「特級」低於150元/公斤）——幾乎確定是替代品。
  * 尋找農產品地理標誌 (农产品地理标志) 標識和桐城市 (桐城市) 產區標示。

**12. 趣聞軼事：**

* **蘭花與茶——天然二重奏。** 桐城小花的獨特性在於其蘭花香不是香化結果，而是天然「鄰居關係」的後果：蘭屬野生蘭花在龍眠山坡茶樹間大量生長。這個生態系統形成數百年，被認為是風土條件的關鍵因素。

* **「不遜於龍井」。** 當地史志《桐城風物記》記載，小花品質不遜於著名的西湖龍井——對謙遜的地方茶而言相當大膽的評價，但得到數百年聲譽的證實。

* **文學之都的茶。** 桐城是桐城派 (桐城派) 的故鄉，該派定義了清帝國散文標準。這款茶以其精緻蘭花特性，成為文人學者的天然伴侶，確立了「知識分子茶」的聲譽。

* **從「貢茶」到「國禮」。** 桐城小花經歷了完整的地位循環：從明代皇室貢品到2013年入選「國禮徽茶」(国礼徽茶) 系列——五個世紀的聲望傳統。

* **四萬茶農。** 桐城茶業涵蓋7個鄉鎮街道，約11,000戶，約40,000茶農。茶園總面積6.75萬畝（約4500公頃），總產量約655噸。包括鄉村旅遊在內的茶業總產值達10.7億元。

**結語：**

桐城小花是一款有山香的茶。不是廣告手冊中抽象的「山野清新」，而是具體的龍眠山：它的雲霧、它的蘭花、它的溪流，帶著透明的水流過砂岩。五個世紀——從孫晉大司馬的椒園到楊頭峰現代合作社——這款茶忠於同一個特性：寧靜優雅、蘭花柔美、純淨甘甜，無一絲虛假音符。

如果桐城派文學主張散文應該「雅潔」，那麼桐城小花就是茶杯中的完美體現。用玻璃杯沖泡，讓「小花朵」在翠綠茶湯中綻放——你就會明白為什麼古代史志敢於將它與龍井相提並論。小花不與偉大者爭辯——它只是在自己的山上綻放，這就足夠了。
