# 高橋銀峰 (高桥银峰, Gāoqiáo yín fēng)

**高橋銀峰 (高桥银峰, Gāoqiáo yín fēng)**
![Гаоцяо Инь Фэн (高桥银峰, Gāoqiáo yín fēng)](images/Гаоцяо%20Инь%20Фэн%20(高桥银峰,%20Gāoqiáo%20Yín%20Fēng).png)

![Гаоцяо Инь Фэн (高桥银峰, Gāoqiáo yín fēng)-1](images/Гаоцяо%20Инь%20Фэн%20(高桥银峰,%20Gāoqiáo%20Yín%20Fēng)-1.png)

高橋銀峰 (高桥银峰, Gāoqiáo yín fēng) 是一種特殊的炒青綠茶，於1959年由湖南省茶葉研究所創制，作為「國慶十週年獻禮」(国庆十周年献礼)。這是湖南省1949年後開發的第一個「新創名茶」(新创名茶)。其名稱詩意精確：芽葉密布銀白茸毫，宛如銀色山峰 (银峰，「銀色山峰」)。1964年詩人、史學家和政治家郭沫若 (郭沫若, Guō Mòruò) 品嚐此茶後，為其題寫讚美詩句，將其與唐宋傳奇名茶並列。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵茶）。特種炒青綠茶 (特种炒青绿茶, tèzhǒng chǎoqīng lǜchá) — 基於科學研發創制的「作者茶」類別。

* **等級：** 中國名茶 (中国名茶, 1989年)。湖南省科學大會獎 (湖南省科学大会奖, 1978年) 得主。多次作為國禮 (国礼) 贈送外國賓客。獲得毛澤東和周恩來讚譽。

* **產地：** 中國湖南省 (湖南, Húnán) 長沙市 (长沙, Chángshā) 長沙縣 (长沙县, Chángshā Xiàn) 高橋鎮 (高桥镇, Gāoqiáo Zhèn)。核心產區為湖南省茶葉研究所 (湖南省茶叶研究所) 實驗茶園，面積約800畝（53公頃），位於玉皇峰 (玉皇峰, Yùhuáng Fēng —「玉皇大帝之峰」) 山腳。

* **地理座標：** 東經113°19′，北緯28°29′。

**2. 歷史與文化意義：**

* **歷史：** 高橋銀峰是一款有確切誕生日期和明確創制目標的茶葉。1959年，為慶祝中華人民共和國成立十週年，湖南省茶葉研究所（成立於1935年 — 中國最古老的茶葉研究機構之一）接受任務，創制一款配得上週年紀念獻禮的「新名茶」。茶葉科學家開發出獨創工藝，關鍵創新步驟為「提毫」(tí háo，「顯露茸毫」)：特殊手法使銀白茸毫不貼伏於葉面（如大多數茶葉），而是「豎立」垂直，營造銀色「山峰」效果。茶葉被命名為「銀峰」(银峰，「銀色山峰」)，地理標識高橋鎮 (高桥，「高橋」) 構成名稱前半部分。

	1964年詩人和政治家郭沫若 (郭沫若, 1892–1978) 品嚐高橋銀峰後，寫下著名詩句：「肯讓湖州誇紫筍，願同雙井鬥紅紗」(肯让湖州夸紫笋，愿同双井斗红纱)。郭沫若將湖南茶與唐代湖州紫筍茶 (紫笋，湖州「紫筍」) 和宋代雙井茶 (双井，「雙井」) 兩大歷史名茶並列。這首詩句成為高橋銀峰的名片。

	1978年獲湖南省科學大會獎。1989年獲「中國名茶」正式地位。多次作為國禮贈送外國代表團。

* **名稱：**

	* 「高橋」(高桥) —「高橋」：研究所所在鎮名。
	* 「銀」(银) —「銀」：覆蓋芽葉的豐富茸毫顏色。
	* 「峰」(峰) —「峰、山頂」：形狀 — 覆蓋銀白茸毫的芽葉宛如雪覆山峰。

* **文化意義：** 高橋銀峰象徵新時代「科學製茶」：這款茶不是由世代農民傳統創制，而是茶葉研究所有目標的科研成果。毛澤東、周恩來的讚譽和郭沫若的詩句確立了其「政治名茶」地位 — 少數同時獲得中華人民共和國多位創建者和文化領袖關注的茶葉之一。

**3. 植物學描述與原料：**

* **品種/栽培種：** 使用數個 *Camellia sinensis* var. *sinensis* 無性系（克隆）栽培種，特別選育以最大化銀白茸毫表現：
	* **福鼎大白茶 (福鼎大白茶, Fúdǐng Dà Bái Chá)** — 主要栽培種。茸毫特別豐富 (茸毛特多)，抗寒性強，適合綠茶生產。
	* **白毫早 (白毫早, Báiháo Zǎo)** — 國家良種。胺基酸含量4.1%。製茶銀毫隱翠 (银毫隐翠)。
	* **槠葉齊 (槠叶齐, Zhūyèqí)** 和 **湘波綠 (湘波绿, Xiāngbōlǜ)** — 省級良種，紅綠茶兼用，香氣清高純正。

* **採摘：** 早春採摘。特級標準為一芽一葉初展 (一芽一叶初展)，芽葉長度約2.5公分。生產1公斤特級乾茶需要12,000–13,000個芽葉。

* **原料要求：** 嫩度一致、完整無損的芽葉。當日採摘當日加工。

**4. 風土條件與栽培特點：**

* **氣候：** 亞熱帶季風濕潤氣候。年均溫16.6°C，年降雨量1441毫米，無霜期275天。研究所基地位於玉皇峰山腳，常有雲霧繚繞。

* **海拔高度：** 玉皇峰山腳丘陵地帶，低中山區。

* **土壤：** 紅壤 (红壤, hóng rǎng)，發育於紫色板頁岩 (紫色板页岩)，富含磷質。pH值約5.0。土層深厚，肥力高。

* **生態系統：** 茶園位於「玉皇峰下河湖掩映」(玉皇峰下河湖掩映) 的風景區。日夜溫差顯著。無工業污染。條件有利於胺基酸和芳香物質積累。

**5. 製作工藝：**

高橋銀峰工藝包含十個步驟 — 比大多數綠茶更複雜。關鍵創新為「提毫」(提毫) 步驟。

* **採摘 (采摘 — cǎi zhāi)：** 手工春季採摘「一芽一葉」標準芽葉。

* **攤青與萎凋 (摊青 — tān qīng)：** 短時間攤放。

* **殺青 (杀青 — shāqīng)：** 溫度120–130°C — 相對溫和的溫度，保持原料嫩度和茸毫。

* **清風 (清风 — qīngfēng)：** 獨特步驟：殺青後進行揚簸 (扬簸, yángbǒ)，去除雜質和細碎物。此工序確保成品純淨。

* **初揉 (初揉 — chūróu)：** 輕度成型。

* **初乾 (初干 — chūgān)：** 預備乾燥。

* **做條 (做条 — zuòtiáo)：** 手工將芽葉塑造成緊密、略彎曲的「針形」。

* **提毫 (提毫 — tí háo)：** **關鍵且最具特色的步驟** — 高橋銀峰創制者的獨創技術。在45–50°C溫度下，師傅用特殊的「搓揉」(搓揉, cuōróu，「搓揉」) 手法將銀白茸毫從葉面「提起」，使其垂直豎立。結果是芽葉覆蓋特有的銀白「霜層」，在儲存和沖泡時不脫落。正是此步驟賦予茶葉「銀峰」之名。

* **攤涼 (摊凉 — tānliáng)：** 中間冷卻。

* **烘焙 (烘焙 — hōngbèi)：** 兩階段：先70°C，後60°C。溫和烘焙固定茸毫並形成最終香氣。

**6. 感官特徵：**

高橋銀峰的經典特徵為「四美」(四美, sì měi)：**形美** (形美)、**香鮮** (香鲜)、**湯清** (汤清)、**味醇** (味醇)。

* **乾茶外觀：** 條索緊細微曲 (条索紧细微曲)。滿披銀毫 (满披银毫) 垂直豎立 — 主要視覺特徵。色澤翠綠隱毫 (翠绿隐毫)，翠綠透過銀白「霜層」顯現。

* **乾茶香氣：** 嫩香持久 (嫩香持久)。清香純正 (清香)。栗香 (栗香)。

* **茶湯香氣：** 清鮮持久，栗綠香型。

* **滋味：** 鮮醇 (鲜醇, xiānchún)，回甘明顯 (回甘)。茶體中等厚度，圓潤。澀味極輕。餘韻清潔爽口。

* **湯色：** 淺綠色，湯清 (汤清) 透明。

* **葉底（沖泡後茶葉）：** 嫩綠均勻的芽葉。完整有彈性。

**7. 化學成分：**

* **多酚類（兒茶素）：** 含量高 — 提供抗氧化潛力。
* **胺基酸（包括L-茶胺酸）：** 含量較高 — 白毫早栽培種含胺基酸4.1%，高於綠茶平均水準。
* **生物鹼：** 咖啡因 — 含量適中。
* **氟：** 含量高 — 有助保護牙釉質。
* **維生素：** 維生素C、B群維生素。
* **礦物質：** 鉀、鎂、磷、鋅、錳、氟。

**8. 保健功效：**

* **抗氧化作用：** 兒茶素中和自由基。
* **提神醒腦 (提神醒脑)：** 咖啡因和L-茶胺酸提供溫和提神效果。
* **降血脂 (降血脂)：** 兒茶素有助降低膽固醇水準。
* **護齒明目 (护齿明目)：** 氟強化牙釉質；類胡蘿蔔素維護眼部健康。
* **增強免疫力：** 維生素礦物質複合物支持免疫功能。

* **重要提醒：** 上述功效基於公開資料，非醫療建議。

**9. 沖泡方法：**

* **水溫：** 80–85°C。

* **茶量：** 3公克配150毫升水（比例1:50–1:60）。

* **茶具：** 玻璃杯或白瓷茶杯 — 便於觀察銀毫和評估湯色清澈度。

* **沖泡過程：**

	1. 溫杯，倒掉溫水。
	2. 投茶。
	3. 注水。首泡10–20秒。
	4. 後續沖泡每次增加5–10秒。可沖泡3–4次。

* **注意：** 建議趁熱飲用，餐後一小時飲用。茶湯最好在沖泡後30分鐘內飲完 — 冷卻氧化後風味下降。

**10. 儲存：**

* 密封容器，陰涼避光處儲存。
* 最佳條件 — 冰箱0–5°C。
* 保存期限 — 最多12個月。
* 開封後 — 1–2個月內飲完。

**11. 價格與仿冒品：**

高橋銀峰產量有限：核心產區為湖南省茶葉研究所800畝實驗茶園。產量不大，使正宗茶葉在湖南省外較為稀少。

* **避免仿冒品方法：**

	* **向可信賣家購買**，最好是湖南省茶葉研究所監製 (湖南省茶叶研究所监制) 標識產品。
	* **評估茸毫：** 特有銀白茸毫應垂直豎立，不脫落。貼伏或缺失茸毫是仿冒跡象。
	* **評估香氣：** 嫩香清鮮，栗香。粗糙或「草腥」氣味值得懷疑。
	* **檢查湯色：** 清澈透明，淺綠色。混濁可疑。
	* **注意價格：** 核心產區正宗高橋銀峰不可能便宜。

**12. 趣聞軼事：**

* 高橋銀峰是少數作為「國慶獻禮」創制的茶葉：1959年為中華人民共和國十週年而開發。這是湖南省1949年後第一個「新創名茶」。

* 郭沫若詩句（1964）：「肯讓湖州誇紫筍，願同雙井鬥紅紗」(肯让湖州夸紫笋，愿同双井斗红纱) —「讓湖州誇耀紫筍茶，我們願與雙井茶爭紅紗」。郭沫若將高橋銀峰與兩大傳奇名茶並列：唐代湖州紫筍茶 (紫笋) 和宋代雙井茶 (双井) — 最高讚譽。

* 「提毫」(提毫) 工藝是湖南研究所的獨創技術，在傳統製茶中無直接類比。銀白茸毫垂直豎立而非貼伏表面 — 這在視覺上區別高橋銀峰與所有其他「有毫」茶類。

* 茶葉獲毛澤東和周恩來讚譽 — 中華人民共和國兩位關鍵領袖。少有茶葉能同時獲得兩位領導人關注。

* 研究所產品通過「無公害」(无公害) 認證，農藥殘留水準符合歐盟標準。

**13. 與其他湖南茶和「有毫」綠茶比較：**

* **古丈毛尖 (古丈毛尖)：** 來自湖南湘西。同為湖南茶，同樣有毫，但針形且耐泡性極強（可沖15次）。古丈更耐泡「厚重」；銀峰更嫩雅「銀白」。

* **君山銀針 (君山银针)：** 來自湖南，但屬黃茶，非綠茶。兩者都「銀白」，都是湖南茶，但工藝和口感屬不同茶類。

* **黃山毛峰 (黄山毛峰)：** 來自安徽。烘青綠茶，「雀舌」形。毛峰更具蘭花香；銀峰更「科學純淨」，茸毫垂直豎立。

* **信陽毛尖 (信阳毛尖)：** 來自河南。同樣有毫，但針形，具明顯「雙鍋」栗香。信陽更厚重「北方」；銀峰更嫩雅「南方」。

**結語：**

高橋銀峰是科學孕育、獻給祖國的茶葉。在湖南省茶葉研究所實驗室為中華人民共和國週年慶典而創制，被郭沫若在一首詩中與唐宋最偉大茶葉並列，獲毛澤東讚譽 — 同時保持謙遜、嫩雅、「銀白」的茶性，其主要美感在於垂直豎立的茸毫，宛如雪覆山峰。其鮮醇滋味和清澈透明湯色不僅是味覺享受，更是科學與工藝如何創制配得上與歷史傳奇「爭紅紗」茶葉的具體體現。
