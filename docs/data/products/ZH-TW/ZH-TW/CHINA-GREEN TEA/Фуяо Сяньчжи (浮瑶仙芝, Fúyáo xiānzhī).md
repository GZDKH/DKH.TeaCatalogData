# 浮瑤仙芝 (浮瑤仙芝, Fúyáo xiānzhī)

浮瑤仙芝是來自江西省景德鎮市浮梁縣的高級綠茶。浮梁是中國茶葉栽培的搖籃之一：唐代時期，這裡集中了帝國最大的茶葉市場，提供了全國茶稅的八分之三。白居易在《琵琶行》中永恆地記錄了浮梁：「商人重利輕別離，前月浮梁買茶去」。「仙芝」之名源於楊貴妃的傳說，她將浮梁茶的滋味比作靈芝的香氣。如今，浮瑤仙芝是浮梁茶品牌的旗艦產品之一，代表著江西省茶業名片「四綠一紅」中的一員。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵茶，綠茶，lǜchá）。按工藝屬烘青綠茶，最後經文火輕烤。
* **類別：** 地方名茶（名茶，míngchá）。浮梁茶品牌於2010年獲得中國農業部頒發的農產品地理標誌認證。浮瑤仙芝是此「傘型」品牌下的主導產品之一；2003年被國務院選為特選禮品茶。
* **產地：** 中國江西省（江西省，Jiāngxī Shěng）景德鎮市（景德鎮市，Jǐngdézhèn Shì）浮梁縣（浮梁縣，Fúliáng Xiàn）。茶園主要分布在佔全縣70%以上的山地和丘陵地區，以瑤里鎮、西湖及周邊山區為中心。
* **地理座標：** 約北緯29.35°，東經117.23°（以浮梁縣為參考）。

**2. 歷史與文化意義：**

* **歷史：** 浮梁的茶葉栽培始於漢代（西元前2世紀至西元3世紀），當時佛教僧侶開始在山坡上種植茶樹。據《中國商業簡史》記載，南北朝時期（5-6世紀）「浮梁茶最好」。唐代（7-10世紀）是鼎盛時期：據《元和郡縣志》記載，浮梁年茶葉交易量達七百萬馱，茶稅超過十五萬貫，相當於全國茶稅收入的八分之三。王敷在《茶酒論》中寫道：「浮梁歙州，萬國來求」。白居易（772-846）於816年在著名的《琵琶行》中提到浮梁，永遠將該縣與繁榮的茶葉貿易聯繫在一起。據宮廷傳說，唐玄宗品嚐浮梁茶後大為讚賞，楊貴妃說：「有靈芝的香和味，只配神仙所用也」，皇帝遂賜名「仙芝」。唐代浮梁茶中還有「嫩蕊」、「福合」和「祿合」。《牡丹亭》作者湯顯祖（1550-1616）評價：「浮梁之茗，冠於天下」。清代，紅茶製作技術傳入浮梁，該縣成為著名的浮梁紅茶產地。1915年，嚴臺村的「天祥」茶號在舊金山巴拿馬太平洋國際博覽會上獲得金獎——「茅臺鎮的酒，嚴臺村的茶」成為流行語。現代，從1991年開始創製浮瑤仙芝（「浮梁」+「瑤里」+「仙芝」的組合名），1992年在農業部展覽會上獲得「最佳產品」獎。1997年浮梁縣獲「中國紅茶之鄉」稱號。2003年浮瑤仙芝被國務院選為禮品茶，同年在上海國際茶文化節獲金獎。2005年該縣成為「全國無公害茶生產示範基地縣」；另一種浮梁綠茶「野蘭芝」在第五屆「中茶杯」獲一等獎。2010年浮梁茶獲地理標誌；至2014年其市場價值評估為9.28億元。2019年浮梁茶入選「中國農業品牌目錄」。

* **名稱：** 浮瑤——「浮梁」和「瑤里」（高品質茶葉產地）的組合詞。仙芝——「神仙靈芝」：仙（xiān）——「神仙、神聖」；芝（zhī）——「靈芝」（*Ganoderma lucidum*）。名稱源於唐玄宗的傳說，傳達「天賜」、崇高滋味的理念。*註：* 某些資料中出現仙枝（xiānzhī，「神仙枝條」）的寫法——這是書寫錯誤；權威資料一致使用仙芝。

* **文化意義：** 浮梁與中國文化的兩大象徵——茶和瓷器密不可分。歷史上的景德鎮「世界瓷都」最初屬於浮梁；該縣理所當然地擁有「瓷都之源，茶國之地」的稱號。這種「瓷茶一體」的結合始於漢代。與廬山雲霧、婺源綠茶、狗牯腦茶並列為「四綠」——江西茶業的名片。

**3. 植物學描述與原料：**

* **品種：** *Camellia sinensis* var. *sinensis*。
* **品種/栽培種：** 適應浮梁縣亞熱帶丘陵和山麓條件的當地群體種（群體種，qúntǐzhǒng）。大規模生產也使用區域化的無性系品種。
* **採摘：** 主要時期——早春。最珍貴的批次——「穀雨尖」：在穀雨（約4月20日）來臨前採摘，此時第一批嫩芽剛剛出現。後續採摘產出「細茶」和「粗茶」。
* **採摘標準：** 一芽一二葉。浮瑤仙芝最高等級——只採嫩芽和第一葉，不含葉柄；標準綠茶允許較成熟的原料。
* **原料要求：** 完整、新鮮採摘、無機械損傷。採摘後去掉葉梗；茶葉迅速送往加工。

**4. 風土與栽培特點：**

* **生長海拔：** 海拔100-800公尺；山地（500公尺以上）佔全縣面積41.7%，山麓（100-500公尺）佔30.6%。最佳茶園位於瑤里鎮周圍的高山區——浮梁（江西）、婺源（江西）、祁門（安徽）和休寧（安徽）縣交界處，海拔600-800公尺。
* **地形：** 丘陵山地：山脈與昌江、東河、西河的河谷交錯，這些河流從南向北貫穿全縣。森林覆蓋率79.4%。
* **氣候：** 中亞熱帶季風氣候。年平均氣溫14-17°C。有效積溫5000-6000°C。年平均降雨量1700-1900毫米——中國茶區最高之一，確保充足的水分。無霜期約247天。日照率約45%。相對濕度約79%。
* **微氣候：** 頻繁的霧氣和雲霧是浮梁山區茶園的特徵。當地諺語說：「晴天早晚遍地霧，陰雨之時滿山雲」。
* **土壤：** 紅壤和黃壤山地土壤，酸性（pH 4.3-5.5），有機質含量極高——達14.5%。這是中國茶區最高的指標之一，確保礦物質豐富和滋味濃郁。
* **生態：** 浮梁縣工業污染極少：經濟歷來以瓷器、茶葉和林業為主，而非重工業。2005年獲「全國無公害茶生產示範基地縣」地位，2007年獲「國家級茶葉標準化示範縣」。水資源豐富：昌江、東河、西河三大河流為茶園提供穩定的水分平衡。

**5. 製作工藝：**

浮瑤仙芝是烘青綠茶，結合手工處理和最後的文火輕烤。工藝旨在保持嫩綠、純淨香氣和最大甜度。

* **採摘（採摘 — cǎizhāi）：** 早晨手工挑選嫩芽。原料快速送至車間。

* **攤放（攤放 — tānfàng）：** 鮮葉薄攤以平衡水分並啟動初步酶促反應，豐富未來的香氣。

* **殺青（殺青 — shāqīng）：** 高溫炒制以滅活氧化酶。形成香氣基礎——純淨、新鮮，帶輕微花香。

* **攤涼（攤涼 — tānliáng）：** 短暫通風以防止「悶熟」。

* **揉捻（揉捻 — róuniǎn）：** 輕柔揉捻釋放細胞汁液，形成緊密、細緻的葉形，確保沖泡時充分萃取。

* **初烘（初烘 — chūhōng）：** 適溫烘焙，降低葉片水分。

* **攤涼與復揉（攤涼 — 復揉 — fùróu）：** 必要時——額外揉捻以獲得更緊的形狀（用於標準等級）。

* **文火輕烤（文火輕烤 — wénhuǒ qīngkǎo）：** 最後的細緻加熱，將水分降至穩定水平（≤6.5%），固定香氣並賦予茶湯純淨度。這一步驟是浮瑤仙芝的標誌：溫和的火候展現細膩的花香和甜味細節，而無「焦味」。

* **揀剔與分級（揀剔 / 分級 — jiǎntī / fēnjí）：** 去除瑕疵葉；按等級分類。

**6. 感官特徵：**

* **乾茶外觀：** 細緊的條索（條索緊細）；顏色——嫩綠色帶豐富白毫（白毫顯露）。葉片勻整，大小一致，有油潤光澤。
* **乾茶香氣：** 純淨、持久，帶細膩花香，類似野生蘭花（蘭花香，lánhuā xiāng）。背景有輕微甜香。
* **茶湯香氣：** 高揚、持久；以蘭花香為主，伴有新鮮綠意的細微背景和柔和蜜香。香氣被描述為「清香持久」。
* **滋味：** 鮮爽醇正。茶體中等，具明顯甜度和圓潤感。苦澀度極低。回甘純淨，有持久的回甘和「甘滑」的口感。
* **湯色：** 黃綠色，透明，明亮（嫩綠 / 嫩黃，清澈無濁）。
* **葉底（沖泡後的葉片）：** 嫩葉和芽完整；顏色——淺黃帶綠（嫩黃明亮顯毫）。茸毛清晰可見。

**7. 化學成分：**

* **茶多酚：** 該地區綠茶典型含量22-30%。主要成分——兒茶素（EGCG、ECG），決定抗氧化活性和輕微澀味。
* **氨基酸：** 含量較高——浮梁茶的特徵，得益於高濕度和豐富降雨刺激L-茶氨酸的積累。具體數據因等級而異，但通常高於江西綠茶的平均水平。
* **水浸出物：** 高指標（通常>40%），源於土壤有機質豐富（達14.5%）和充沛降雨（1700-1900毫米/年），確保茶樹代謝旺盛和葉片中萃取物質的積累。
* **生物鹼：** 咖啡因——該地區綠茶典型含量2.5-4%。可可鹼和茶鹼——微量。
* **維生素：** 維生素C（抗壞血酸）、B群維生素（B₁、B₂）、類胡蘿蔔素。
* **礦物質：** 硒——江西南部土壤天然富含（資料中提及）。還有鉀、鎂、錳、鋅、氟。
* **精油和芳香化合物：** 蘭花香由芳樟醇、香葉醇和乙酸橙花酯形成；柔和的「蜜香」由辛醛和苯甲醛產生。

**8. 保健功效：**

* **抗氧化保護：** 兒茶素，特別是EGCG，中和自由基並減緩細胞氧化過程。
* **認知支持：** L-茶氨酸和咖啡因的組合提供溫和、平穩的提神效果——集中注意力而不緊張。
* **心血管系統支持：** 茶多酚有助於調節膽固醇水平和血壓。
* **助消化：** 溫和刺激腸道蠕動和酶分泌；非常適合搭配清淡餐食。
* **免疫系統支持：** 硒和維生素C增強身體的抗氧化防禦。
* **口腔健康：** 氟和兒茶素抑制致齲細菌的生長。
* **皮膚健康維護：** 抗氧化劑和維生素C有助於防止光老化和膠原蛋白合成。
* **代謝支持：** 茶多酚和咖啡因促進新陳代謝和脂肪氧化，結合均衡飲食可能有助於體重控制。
* **重要提示：** 對咖啡因敏感者應限制下午飲用。不建議空腹飲用。

**9. 沖泡方法：**

* **水溫：** 75-85°C。最高等級浮瑤仙芝——75-80°C；標準綠茶——80-85°C。
* **茶量：** 150-200毫升（玻璃杯）用3-4克或100-120毫升（蓋碗）用4-5克。
* **茶具：** 透明玻璃杯——觀賞茸毫芽葉的「舞蹈」；瓷蓋碗——最大程度展現蘭花香。特別有韻味的是用景德鎮瓷器沖泡：「瓷茶一體」在一杯中相會。
* **步驟：**
  1. 用開水溫杯後倒掉。
  2. 投茶。對於嫩茶——採用「上投法」：先注水至杯子七分滿，然後輕輕加入茶葉。
  3. 無需洗茶。
  4. 第一泡——浸泡1.5-2分鐘（玻璃杯）或20-30秒（蓋碗）。
  5. 倒出；玻璃杯——喝掉三分之一時續水。
  6. 重複沖泡：玻璃杯3-4泡，蓋碗5-6泡（每次增加5-10秒）。

**10. 儲存方法：**

* 存放在密封、不透光的容器中（真空包裝袋、錫罐），避光、防潮、隔熱和異味。
* 最佳溫度——冰箱0-5°C，雙重密封。
* 室溫儲存——涼爽、乾燥、陰暗處；開封後2-3個月內飲用。
* 為獲得最佳品質——生產後6-12個月內飲用。浮瑤仙芝是新鮮茶；陳放對其無益。

**11. 價格與防偽：**

* **價格區間：** 高級早春浮瑤仙芝——800-3,000元/公斤。標準浮梁綠茶——200-600元/公斤。主要價格因素：採摘季節（穀雨尖最貴）、茶園海拔（瑤里山茶更貴）、等級（芽茶比葉茶貴）和具體子品牌。

* **如何避免假貨：**
  * 向有權使用浮梁茶地理標誌品牌的認證企業購買。
  * 評估香氣：正宗浮瑤仙芝具有天然蘭花香，無人工香精。
  * 檢查外觀：細緊條索帶豐富白毫；不勻整、暗淡的葉片是假冒的跡象。
  * 評估茶湯：透明、黃綠色、無渾濁。
  * 價格過低令人懷疑，特別是聲稱「禮品」等級時。

**12. 趣聞軼事：**

* 浮梁是兩大世界珍寶的故鄉：瓷器和茶葉。歷史上的「瓷茶一體」早在漢代就在這裡誕生：僧侶們同時燒製陶瓷和栽培茶樹。

* 白居易《琵琶行》中的詩句——「商人前往浮梁買茶」——進入中國文學的黃金寶庫，成為浮梁茶業的非官方「口號」，而現代的「浮梁買茶節」直接呼應這首詩。

* 唐代浮梁茶稅佔全國的八分之三——這一比例沒有任何其他縣達到過，使浮梁成為中世紀中國最大的茶葉市場。

* 浮梁是兩個偉大茶縣的「母親」：740年從浮梁（當時稱新昌）分出婺源，766年分出祁門。婺源以中國最佳出口綠茶聞名，祁門是「十大名茶」之一祁門紅茶的故鄉。因此，浮梁是兩個茶葉傳奇的共同歷史根源。

* 1915年巴拿馬太平洋博覽會上，嚴臺村的浮梁紅茶與茅臺酒並列獲得金獎——由此產生「茅臺鎮的酒，嚴臺村的茶」的說法。嚴臺和滄溪村的古老茶號建築保存至今，是茶葉貿易的歷史見證。

* 生產浮瑤仙芝的瑤里鎮同時也是景德鎮瓷器的搖籃：正是從這裡開採的高嶺土，給世界帶來了「china」這個詞。今天瑤里是熱門旅遊路線，結合了瓷器和茶葉遺產。

* 湯顯祖（1550-1616）——中國最偉大的劇作家，「中國的莎士比亞」——在關於浮梁學派的文章中寫道：「浮梁之茗，冠於天下，帷清帷馨」，記錄了該地區在16-17世紀之交的聲譽。

**13. 與其他綠茶的比較：**

* **瑤里崖玉（瑤里崖玉，Yáolǐ Yáyù）：** 另一種高級浮梁綠茶，產於瑤里海拔600-1000公尺的山地茶園。崖玉更加細膩，帶銀色茸毫和細緻香氣；浮瑤仙芝稍微厚實和「圓潤」。

* **廬山雲霧（廬山雲霧，Lúshān Yúnwù）：** 「中國十大名茶」之一，與浮瑤同為江西茶。雲霧捲曲，葉片厚實多毫，具強烈「山韻」；仙芝更細膩，蘭花香更明顯，茶體更柔和。

* **婺源綠茶（婺源綠茶，Wùyuán Lǜchá）：** 浮梁茶的歷史「兄弟」（婺源740年前屬於浮梁）。婺源綠茶以極高的水溶性兒茶素和濃郁萃取著稱；浮瑤仙芝更嫩更花香。

* **黃山毛峰（黃山毛峰，Huángshān Máo Fēng）：** 鄰近安徽的綠茶，風土相似（江西安徽交界的山區）。毛峰更多毫，具更細膩的「鈴蘭」香；仙芝稍厚實「圓潤」，具蘭花香。

* **靖安白茶（靖安白茶，Jìng'ān Báichá）：** 江西鄰縣靖安的獨特「白葉」綠茶，由白化品種製成。白茶明顯更輕盈「清涼」，具明顯鮮味；浮瑤仙芝更飽滿，花香更發達，甜度「溫暖」。
