# 鄧村綠茶 (邓村绿茶, Dèngcūn lǜchá)

**鄧村綠茶 (邓村绿茶, Dèngcūn lǜchá)**
![Дэнцунь Люй Ча (邓村绿茶, Dèngcūn lǜchá)](images/Дэнцунь%20Люй%20Ча%20%28邓村绿茶,%20Dèngcūn%20Lǜchá%29.png)

![Дэнцунь Люй Ча (邓村绿茶, Dèngcūn lǜchá)-1](images/Дэнцунь%20Люй%20Ча%20%28邓村绿茶,%20Dèngcūn%20Lǜchá%29-1.png)

鄧村綠茶 (邓村绿茶, Dèngcūn lǜchá) — 來自湖北省宜昌市夷陵區鄧村鄉 (邓村乡) 的高山綠茶，這片土地被「茶聖」(茶圣) 陸羽 (陆羽) 在其《茶經》中譽為最佳：「山南，以峽州上」(山南，以峡州上)。鄧村是中國唯一在鄉鎮級別獲得「中國名茶之鄉」(中国名茶之乡) 稱號的鄉鎮。茶園位於海拔800–1200公尺，坐落在西陵峽 (西陵峡, 長江三峽中的第一峽) 北岸，緊鄰三峽大壩 (三峡大坝)。80,000畝茶園，一年中有180多天籠罩在雲霧中，使鄧村成為「萬裡茶道源頭」(万里茶道源头) 和「世界高品質綠茶核心產區」(世界高品质绿茶核心产区) — 這些稱號由世界綠茶協會授予。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，不發酵茶。生產多種形狀：針形 (针形, zhēnxíng，「鄧村毛尖」— 主要產品)、螺旋形 (卷曲形，「花頂雲霧」)、扁形 (扁形，「丹丘芽」)。主要工藝 — 炒烘結合 (炒烘结合)。

* **等級：** 中華人民共和國地理標誌產品 (国家地理标志产品, 2006年)。AA級綠色食品認證 (AA级绿色食品)。通過歐盟農藥殘留檢測標準。「中國名茶之鄉」(中国名茶之乡) — 全國唯一獲此殊榮的鄉鎮。「萬裡茶道源頭」(万里茶道源头)。「世界高品質綠茶核心產區」(世界高品质绿茶核心产区) — 世界綠茶協會 (評議員小鳥茂夫) 授予的稱號。「世界茶旅小鎮」(世界茶旅小镇)。

* **產地：** 中國湖北省 (湖北省, Húběi Shěng) 宜昌市 (宜昌市, Yíchāng Shì) 夷陵區 (夷陵区, Yílíng Qū) 鄧村鄉 (邓村乡, Dèngcūn Xiāng)。位於西陵峽 (西陵峡) — 「長江三峽」(长江三峡) 第一峽的北岸。鄧村東側為三峽大壩 (三峡大坝)；北側為大老嶺國家森林公園 (大老岭国家森林公园)。

* **地理座標：** 北緯30°57′–31°06′ — 「世界茶樹生長黃金座標」(世界茶树生长黄金坐标)。

**2. 歷史與文化意義：**

* **歷史：** 鄧村的茶葉傳統是中國最古老且記錄最完整的茶葉傳統之一，可追溯至三國時期。

	**三國時期 — 最早記錄 (西元3世紀)。** 在《廣雅》(《广雅》, 西元227–232年) 中記錄了峽州 (峡州) 的茶葉生產技術 — 這個行政區域包括現今鄧村的領土。這是該地區茶葉製作最早的文獻記錄之一。

	**唐代 — 陸羽的最高評價 (8世紀)。** 在《茶經》(《茶经》, 約西元760年) 中，陸羽評價「山南」茶葉，將峽州列為第一：「山南，以峽州上，襄州、荊州次」— 「山南地區中，峽州為上，襄州和荊州次之」。峽州在三個縣生產茶葉：遠安、夷道和夷陵 (夷陵) — 即現今鄧村的領土。後來在宋代，歐陽修 (欧阳修) 稱夷陵為「陸羽茶經第一州」(陆羽茶经第一州) — 「陸羽茶經中的第一茶州」。

	**明代 — 「三峽靈草」(明)。** 在嘉靖年間 (嘉靖, 1522–1566)，鄧村茶獲得詩意的別名「三峽靈草」(三峡灵草) 並成為「貢茶」(贡茶，朝廷貢品茶)。

	**清代 — 「溪茶」與「茶馬古道」。** 清初，來自鄧村的「溪茶」(溪茶，「溪流茶」) 翻山越嶺北運至襄樊 (襄陽)，形成「茶馬古道」(茶马古道) 的陸路段。鄧村也是「宜紅茶」(宜红茶，「宜昌紅茶」) 的核心產區之一，據研究，也是首次描述紅茶「冷後渾」(冷后浑, lěnghòuhún) 現象的地方。

	**現代。** 1956年，品嚐鄧村茶的蘇聯專家建議使用「鄧村綠茶」(邓村绿茶) 這個名稱，並成為正式名稱。2006年獲得地理標誌。2017年建成「三峽茶谷·鄧村茶遺產公園」(三峡茶谷·邓村茶遗产公园)，確立了「萬裡茶道源頭」(万里茶道源头) 的定位。截至2024年，茶園面積80,000畝 (約5,300公頃)，年產乾茶12,000噸，總產值4.6億元人民幣。

* **名稱：**
	* 「鄧村」(邓村) — 鄉鎮名稱。地名與鄧姓 (邓) 相關。
	* 「綠茶」(绿茶) — 「綠茶」。

* **文化意義：** 鄧村是三大茶葉敘事交匯之地：陸羽《茶經》(8世紀)、「萬裡茶道」(万里茶道, 17–19世紀) 和「長江三峽」— 中國最具代表性的景觀形象之一。鄉內設有陸羽紀念碑，每年舉辦「三峽茶文化藝術節」(三峡茶文化艺术节)。鄧村是唯一一個人口約30,000人的鄉鎮，每位居民平均擁有3畝 (約0.2公頃) 茶園。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要品種 — **宜昌大葉種 (宜昌大叶种, Yíchāng Dàyè Zhǒng)**，也稱為「華茶29號」(华茶29号) — 國家認定的前30個國家級茶樹栽培種之一。類型 — 半喬木型 (小乔木型)。葉片 — 橢圓形，14.7 ± 2.4公分，柔軟肥厚。生化成分：多酚類 — 35.8%，胺基酸 — 1.64%，咖啡因 — 5.87%。適合生產香氣濃郁、持久的綠茶。

	鄧村保存有百年以上古茶樹群落 (百年以上古茶树群落)。

* **採摘：** 春茶為主。特級：全芽或一芽帶剛展開的葉片。二、三級：一芽一至二葉和一芽二至三葉。

* **標準 (GB/T 14456 + 地理標誌標準)：** 四個等級：特級、一級、二級、三級。

**4. 風土條件與栽培特色：**

* **氣候：** 高山亞熱帶氣候，受三峽水庫和西陵峽水文氣象效應影響。年平均氣溫14–18°C。降雨量1,144–1,831毫米/年。雲霧天氣超過180天。散射光超過70%。日夜溫差超過8°C。長江水系形成持續蒸發，形成「高山雲霧帶」(高山云雾带)。

* **海拔：** 800–1200公尺。核心區 — 鄧村鄉，生產該區80%的茶葉。

* **土壤：** 花崗岩風化微酸性砂壤土 (花岗岩风化微酸性砂壤土, pH 5.5–6.5)。有機質 ≥1%。富含硒和鋅。水質達國家一級標準。

* **生態：** 森林覆蓋率70%。負離子10,000個/立方公分 (「度假村」標準為1,500+)。完全沒有工業設施。

* **風土關鍵特色：** 鄧村茶的胺基酸含量比綠茶平均值高20%。水浸出物 ≥46.3%。這是由於「高海拔、多雲霧、寡日照」(高海拔、多云雾、寡日照) 的結合 — 貴州-湖北茶區風土的經典公式。

**5. 製作工藝：**

鄧村綠茶採用「三保一高」(sān bǎo yī gāo) 工藝 — 「三個保持，一個追求」：保持翠綠色澤 (保翠绿色泽)，保持茸毫附體 (保茸毫附体)，保持鋒苗完整 (保锋苗完整)，追求香高持久 (求香高持久)。

* **採摘 (采摘)：** 手工 + 大宗等級機械化。
* **攤青 (摊青)：** 竹篩，4–6小時。失水4.5–5.5%。
* **殺青 (杀青)：** 滾筒式，180–200°C，6–8分鐘。完全鈍化酵素。
* **揉捻 (揉捻)：** 「輕—重—輕」(轻—重—轻) 方法，14–20分鐘。
* **初烘 (初烘)：** 100–120°C至80%乾度。
* **復揉理條 (复揉理条)：** 手工塑形 — 形成針形、螺旋形或扁形。
* **足乾提毫 (足干提毫)：** 炭火烘乾，70–80°C。標準：「手捻成粉」(手捻成粉)。
* **精製 (精制)：** 分級，去梗。

禁用化學農藥。採用物理防蟲方法。AA級綠色食品認證。

**6. 感官特徵：**

* **乾茶外觀：** 依產品線而異。**毛尖 (针形)**：緊實筆直的「針」形，茸毫豐富。**花頂雲霧 (卷曲形)**：緊結螺旋。**丹丘芽 (扁形)**：扁平「眉」形，類似龍井。共同特點：色澤綠潤顯毫 (绿润显毫)。

* **乾茶香氣：** 栗香 (栗香) — 主要香調。持久深沉 (幽长持久)。高等級具有嫩香 (嫩香)。冷杯香 ≥10分鐘。

* **滋味：** 醇厚 (醇厚, chúnhòu) — 高多酚含量 (18–23%) 的結果。鮮爽 (鲜爽) — 胺基酸3.3–4.68%。回甘明顯 (回甘明显)。高可溶性糖含量增強「鮮潤感」。

* **湯色：** 黃綠明亮 (黄绿明亮)。

* **葉底：** 黃綠勻整、柔韌有光澤 (黄绿匀整、柔韧有光泽)。

**7. 化學成分：**

* **多酚類：** 18.18–23%。中等範圍，確保醇厚度與柔和度的平衡。
* **胺基酸：** 3.3–4.68% — 比綠茶平均值高20%。決定鮮爽和「鮮潤感」。
* **水浸出物：** ≥46.3% — 最高指標之一。
* **咖啡因：** 約5.87% (宜昌大葉種)。高於平均值。
* **硒、鋅：** 花崗岩土壤天然富集。
* **維生素：** C、B群。**礦物質：** K、Mg、Zn、Se。

**8. 保健功效：**

* **抗氧化作用：** 多酚類18–23% + 硒。
* **支持脂質代謝：** 據生產商資料，餐後飲用可降低脂肪吸收約30%。
* **支持心血管系統：** 兒茶素有助於膽固醇正常化和改善內皮功能。
* **提神效果：** 咖啡因約5.87% — 高於平均值；茶胺酸提供柔和感。
* **微量元素補充：** 硒、鋅。

**9. 沖泡方法：**

* **水溫：** 85–90°C。特級用85°C。
* **用量：** 3公克配150毫升 (1:50)。
* **茶具：** 玻璃杯 (觀賞茶舞, 观赏茶舞) 或白瓷蓋碗。
* **過程：** 注水，浸泡3分鐘。可沖泡3次。

**10. 儲存：**

* 密封包裝，冷藏0–5°C。保存期12個月。開封後1–2個月內飲用完畢。

**11. 價格與仿冒：**

* **價格區間：** 湖北綠茶中高檔。特級 — 500公克600元人民幣起。一級 — 300–500元人民幣。三級 — 大眾級。
* **防偽方法：** 購買標有地理標誌「邓村綠茶」的產品。檢查栗香 — 持久深沉。湯色 — 透明黃綠。

**12. 趣聞軼事：**

* **「茶經第一州」。** 陸羽將峽州 (包括鄧村) 列為「山南」茶葉第一 (山南，以峡州上)。歐陽修發展這一觀點，稱夷陵為「陸羽茶經第一州」(陆羽茶经第一州)。陸羽紀念碑至今仍立在鄧村。

* **蘇聯專家與命名。** 1956年，品嚐當地茶葉的蘇聯專家建議確定「鄧村綠茶」這個名稱 — 這是中國茶葉名稱由外國人「認定」的少數案例之一。

* **人均3畝。** 約30,000名鄧村居民中，每人平均擁有3畝 (約0.2公頃) 茶園 — 中國「茶園密度」最高的指標之一。該鄉茶園面積 (80,000畝) 相當於整個縣的面積。

* **「冷後渾」發源地。** 據研究人員資料，鄧村是首次描述「冷後渾」(冷后浑, lěnghòuhún) 現象的地方 — 紅茶冷卻時由於兒茶素與咖啡因形成複合物而產生的渾濁現象。這一現象後來成為紅茶品質的標誌之一。

* **森林覆蓋率70%，負離子10,000。** 鄧村空氣中負離子含量 — 10,000個/立方公分 — 比普通森林高6–7倍，比城市空氣高30–100倍。該鄉位於大老嶺國家森林公園和西陵峽之間 — 長江流域最生態潔淨的地區之一。

**13. 與湖北其他綠茶的比較：**

* **恩施玉露 (恩施玉露)：** 蒸汽殺青，「海味」。鄧村 — 炒製殺青，栗香，來自長江三峽。

* **採花毛尖 (采花毛尖)：** 湖北。茸毫豐富的「毛尖」，捲曲。鄧村 — 條形，強調栗香深度。

* **松峰綠茶 (松峰绿茶)：** 湖北。類似類別，但來自不同地區。鄧村 — 來自西陵峽，擁有80,000畝種植園和陸羽以來的歷史。

**結語：**

鄧村綠茶 — 一款茶葉，其產地聽起來像《茶經》中的詩句：西陵峽，長江三峽，陸羽稱為「第一」的土地。80,000畝茶園，被180天雲霧籠罩，在富含硒和鋅的花崗岩土壤上，產出具有如此深沉持久栗香的茶葉，以至於詩意的公式「栗香幽長持久」(栗香幽长持久) — 「栗香深沉、悠長而持久」— 看起來不像是描述，而像是配方。這是一款適合重視規模的人的茶 — 歷史的規模 (從三國到萬里茶道)，景觀的規模 (長江三峽) 和生產的規模 (一個鄉年產12,000噸)。
