# 新林玉露 (新林玉露, xīn línyùlù)

**新林玉露 (新林玉露, xīn línyùlù)**

![Синь Линьюйлу (新林玉露, xīn línyùlù)](images/Синь%20Линьюйлу%20(新林玉露,%20Xīn%20Línyùlù).png)

![Синь Линьюйлу (新林玉露, xīn línyùlù)-1](images/Синь%20Линьюйлу%20(新林玉露,%20Xīn%20Línyùlù)-1.png)

新林玉露 (新林玉露, xīn línyùlù) 是中國少數採用蒸汽殺青 (蒸青, zhēngqīng) 而非炒製方法生產的綠茶之一。這項古老技術在陸羽 (陆羽) 的《茶經》中已有記載（蒸之, 捣之 —「蒸之，搗之」），在中國本土幾乎消失，轉而傳入日本，成為煎茶和玉露的基礎。河南新林公司於1996年進口日本蒸汽殺青生產線，創造出獨特產品—「新林玉露」，結合日本技術與大別山原料：信陽群體種 (信阳群体种)，生長於海拔400–1000公尺的山區，一年中有200天霧氣瀰漫。2007年該茶榮獲世界綠茶競賽金獎，2015年獲得世界博覽會「金駱駝獎」。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵，綠茶, lǜchá）。殺青方法—**蒸汽** (蒸青, zhēngqīng)，溫度95°C。外形—「松針形」(松针形, sōngzhēn xíng)。

* **類別：** 地理標誌產品 (国家地理标志产品, 2010年起)；「河南省著名商標」(河南省著名商标)。現代中國工業化生產蒸青綠茶的極少數品種之一（與恩施玉露等少數其他品種並列）。

* **產地：** 中國；河南省 (河南, Hénán)；新縣 (新县, Xīn Xiàn)，隸屬信陽市 (信阳市, Xìnyáng Shì)。新縣位於大別山 (大别山, Dàbiéshān) 腹地—黃河與長江之間最大的山脈。核心產區—陳店鄉 (陈店乡, Chéndiàn Xiāng, 雲山茶場—云山茶场) 和蘇河鎮 (苏河镇, Sūhé Zhèn, 皇坡佬基地—皇坡佬高山野茶基地，40%茶樹樹齡超過30年)。

* **地理座標：** ~31°30′–32°00′北緯，114°30′–115°00′東經（新縣境內）。

**2. 歷史與文化意義：**

* **歷史：**

蒸汽殺青是中國最古老的茶葉加工方法。陸羽在《茶經》（八世紀）中描述了「蒸之, 捣之, 拍之, 焙之」—「蒸之，搗之，拍之，焙之」的標準生產流程。唐宋時期蒸青茶占主導地位；正是這項技術在十二至十三世紀被日本僧侶引進，發展成煎茶、玉露和抹茶的基礎。而在中國本土，自明代（十四世紀）起，蒸青被炒青 (炒青) 取代，到二十世紀，蒸青綠茶在中國已成稀有品種。

復興始於1996年，新林茶業有限公司 (新林茶业有限公司, Xīnlín Cháyè) 在新縣進口日本自動化蒸汽殺青生產線，並將其適應於當地原料—以高氨基酸含量著稱的信陽群體種。由此誕生了「新林玉露」—「新林的玉露」。名稱與著名的湖北恩施玉露 (恩施玉露) 相呼應，但產品在技術上獨立自主。

國際認可來得很快：2007年—世界綠茶競賽金獎 (世界绿茶评比金奖)；2010年—地理標誌產品地位；2015年—世界博覽會「金駱駝獎」(金骆驼奖)「百年世博中國名茶」(百年世博中国名茶)。產品出口至日本、韓國和美國。

* **名稱：** 新林 (Xīnlín) —創造公司名稱，同時寓意大別山茶園的「新森林」（新—「新」，林—「森林」）；玉露 (Yùlù) —「玉露」—蒸青綠茶最高品質的經典稱謂，借鑑自中日茶文化傳統。

* **文化意義：** 新林玉露象徵「回歸本源」—復興在日本保存並通過工業轉移回歸故土的古代中國技術。這是少數清楚展示文化循環的茶類之一：中國→日本→中國。新縣位於大別山「紅色」游擊區（原鄂豫皖蘇區），積極發展茶旅遊作為山區脫貧工具。雲山和皇坡佬山坡上的茶園是生態旅遊路線的一部分，連接革命紀念地與大別山自然景觀。到2020年代，新林玉露成為新縣主要高端茶產品，其自動化生產線成為河南茶業現代化的典範。

**3. 植物學描述與原料：**

* **品種/栽培種：** 信陽群體種 (信阳群体种, Xìnyáng Qúntǐzhǒng) —當地灌木型中葉種 *Camellia sinensis* var. *sinensis*，在大別山條件下經數百年自然選育形成。葉片橢圓形，厚實肉質—適應嚴酷山地條件。春茶生化成分：氨基酸≥4.2%，多酚類14.7%—獨特的高氨基酸/多酚比例（幾乎1:3.5），非常適合蒸青茶，比炒青茶更好地保存氨基酸。同一栽培種用於生產著名的信陽毛尖—中國「十大名茶」之一，但殺青技術根本不同。

* **採摘：** 春茶—三月底至四月。「茶頌」系列 (茶颂, 「茶之頌歌」) —清明前，單芽。「茶雅」(茶雅, 「茶之雅致」) —穀雨前，一芽一葉。「茶情」(茶情, 「茶之情懷」) —四月中旬，一芽二葉。

* **採摘標準：** 特級（特级, 「茶頌」系列）—純單芽。

* **原料要求：** 無損傷嫩芽。原料當日加工。皇坡佬核心茶園—30年以上老樹（占40%）。

**4. 風土條件與栽培特色：**

新縣位於大別山中部—黃河與長江之間最大山脈，河南、湖北、安徽三省交界處。

* **生長海拔：** 400–1000公尺。核心茶園—600公尺以上山坡。

* **氣候：** 亞熱帶北緣。年均溫15°C；降水量1100毫米/年；霧日≥200天—中國最「多霧」的茶區之一。明顯的晝夜溫差。冬季長期低溫迫使茶樹積累氨基酸作為抗凍劑—這是提高L-茶氨酸含量的自然機制。

* **土壤：** 紅黃壤 (红黄壤)，深厚，pH 4.5–6.5，有機質≥3%—中國茶區中最高指標之一。

* **生態：** 森林覆蓋率>85%。核心區—水源保護區 (水源保护区)，禁用化肥農藥。大別山是華中地區最重要的生物圈保護區之一，列入「全國28個重點生態區」。海拔600–1000公尺的茶園形成天然「雲島」—持續霧氣區域，濕度穩定保持80%以上，直射紫外線不足光照的30%。這些條件為茶葉中氨基酸積累和苦味抑制創造了理想環境。

**5. 生產工藝：**

新林玉露採用全自動化日式生產線，避免手工接觸和交叉污染。與炒青綠茶的關鍵區別—蒸汽殺青，能更好地保存葉綠素和氨基酸。

* **攤放 (摊放 — tān fàng)：** 6小時—比大多數綠茶更長，增強香氣前體物質。

* **蒸汽殺青 (蒸汽杀青 — zhēngqì shāqīng)：** 95°C蒸汽瞬間鈍化酶活性，「封存」葉綠素和顏色。與炒青（200+°C）不同，蒸青不產生「炒香」—香氣保持「綠色」和「海洋」特徵。

* **「葉打破形」(叶打破形 — yè dǎ pò xíng)：** 獨特工序，炒青茶無此環節：機械性破碎葉片，增加切面面積。這加速沖泡時的浸出並釋放營養成分。此工序借鑑日本「粗揉」(粗揉, 粗揉) 技術。

* **粗揉散水 (粗揉散水 — cū róu sàn shuǐ)：** 初步成型和快速去除表面水分。

* **揉捻造形 (揉捻造形 — róuniǎn zào xíng → 中揉成形 → 精揉整形)：** 三階段過程：從粗揉通過中揉到精細整形「松針形」。每個階段減小條索直徑並整齊外形。

* **乾燥 (干燥 — gānzào)：** 80°C至穩定含水量。

* **色選除雜與充氮包裝 (色选除杂 → 充氮包装)：** 光學分選去除不合格部分；氮氣包裝延長保鮮期至2年—比普通綠茶顯著更長。

**6. 感官特徵：**

* **乾茶外觀：** 細緊筆直的「松針形」(松针形, sōngzhēn xíng)，整齊均勻。色澤墨綠 (墨绿, mòlǜ) 帶光澤，明顯白毫。顏色比炒青綠茶明顯更深—蒸青最大程度保存葉綠素的結果。

* **乾茶香氣：** 清香純正，特有「海洋」/「海藻」香 (海藻香, hǎizǎo xiāng) —蒸青茶的標誌。春茶增添栗香 (栗香)。

* **茶湯香氣：** 「海洋綠色」，清新，淡淡「海苔」特徵。隨著冷卻顯現甘甜栗香底韻。

* **滋味：** 鮮爽 (鲜爽) —氨基酸的「活力」因溫和蒸汽殺青特別明亮。甘醇 (甘醇)，回甘生甜 (回甘生甜)。苦澀味極少—蒸青時多酚類比炒青「分解」更快。

* **湯色：** 碧綠透亮 (碧绿透亮)，清澈明亮。顏色比炒青茶明顯更「綠」—蒸青茶的診斷特徵。

* **葉底（沖泡後茶葉）：** 青綠勻整 (青绿匀整)，嫩葉舒展如蘭 (嫩叶舒展如兰)。

**7. 化學成分：**

* **氨基酸 (氨基酸)：** ≥4.2%（春茶特級）—河南茶類中最高指標之一。L-茶氨酸占主導。

* **多酚類 (茶多酚)：** 14.7%—明顯低於炒青綠茶（25–30%）。這是蒸汽殺青的結果：蒸青時部分多酚類轉化，解釋了滋味的溫和性。

* **葉綠素：** 含量提高—蒸青保存80%葉綠素（炒青50–60%）。正是這給出特有的「墨綠」色澤。

* **生物鹼：** 咖啡因、可可鹼、茶鹼。提神效果明顯。

* **氟：** 含量提高—15毫克/100克，有效預防蛀牙。

* **維生素：** 維生素C、B群維生素、維生素E—蒸青比高溫炒青更好地保存維生素。

**8. 保健功效：**

* **抗氧化保護：** 兒茶素+維生素C+維生素E—溫和蒸汽處理增強的綜合保護。

* **提神效果：** 咖啡因與L-茶氨酸協同—溫和持久的清醒。

* **口腔健康：** 氟含量（15毫克/100克）—茶類中最高之一，抑制90%致齲細菌。

* **心血管系統支持：** 茶氨酸強化血管壁。

* **認知功能：** L-茶氨酸刺激大腦α波活動。

* **重要提醒：** 上述功效基於一般資料，非醫療建議。不建議空腹飲用。新茶需放置15天以「退火」。開封後72小時內飲用以獲最佳香氣。

**9. 沖泡方法：**

* **水溫：** 85–90°C。不用沸水（>90°C破壞葉綠素，茶湯呈黃色而非綠色）。

* **茶量：** 3克配150毫升（1:50比例）。

* **茶具：** 玻璃杯—觀察「碧綠」茶湯；蓋碗—快沖法。

* **沖泡過程（玻璃杯，上投法）：**

	1. 熱水溫杯後倒出。
	2. 注入85–90°C水至7/10容量。
	3. 投茶入水（上投法）。
	4. 靜置1分鐘。首泡完成。

* **沖泡過程（蓋碗）：** 洗茶5秒→首泡~15秒→每泡遞增5秒。特級可沖8泡—綠茶中的卓越耐泡性，源於高浸出物濃度和「葉打破形」增加的水接觸面積。

**10. 儲存：**

* **溫度：** 0–5°C（冰箱）。
* **包裝：** 充氮包裝 (充氮包装) —「新林」品牌標準。未開封氮氣包裝保存期—2年。開封後—室溫72小時內飲用以獲最佳香氣。
* **光照：** 完全避光；葉綠素對紫外線特別敏感。
* **保質期：** 未開封—24個月（氮氣包裝使綠茶創紀錄保鮮期）。

**11. 價格與仿冒：**

新林玉露屬高價位茶品。特級（「茶頌」系列，單芽）—2000元/斤起；一級（「茶雅」）—800–1500元/斤；二級（「茶情」）—較親民。同時生產抹茶粉 (抹茶粉) 供食品工業使用。

* **避免仿冒方法：**

	* 主要標誌—「墨綠」(墨绿) 乾茶色澤和「碧綠」(碧绿) 茶湯。炒青綠茶絕不會有如此濃郁的綠色調。
	* 香氣—特有「海洋」/「海藻」香。缺乏此特徵表示非蒸青茶。
	* 外形—整齊的「松針形」。捲曲或無形茶條為替代品。
	* 「新林」標誌氮氣包裝—品牌標準。

**12. 趣聞軼事：**

* **技術循環：中國→日本→中國：** 唐代發明的蒸汽殺青，十二世紀傳入日本，發展成煎茶和玉露技術，1996年通過工業進口日本生產線回歸中國。新林玉露是這種文化循環的生動展示。

* **「葉打破形」—非缺陷而是工藝：** 「葉打破形」（機械破碎葉片）工序對完整葉片愛好者可能顯得粗暴，但正是它增加切面面積並加速浸出—葉片更快更充分地「釋放」滋味和香氣。

* **世博會金駱駝2015：** 「百年世博中國名茶金駱駝獎」(百年世博中国名茶金骆驼奖) 紀念1915年巴拿馬世界博覽會，中國茶首次獲得國際認可。新林玉露進入「世紀名茶」精英俱樂部。

* **年200霧日：** 新縣是中國最「多霧」的茶區之一。持續雲霧提供散射光，刺激氨基酸合成並抑制苦味。

* **大別山抹茶：** 除葉茶外，「新林」生產抹茶 (抹茶粉, mǒchá fěn) —同樣原料的超細粉末，採用超微粒研磨法。這是少數按日本技術生產的正宗中國抹茶之一，用於中國食品工業和出口。

* **出口日本—最高評價：** 新林玉露出口至日本—自認為世界蒸青綠茶標準的國家—證明產品高品質。日本茶專家稱讚其「獨特甘甜」和「溫和」，這是日本煎茶所不具備的特徵。

**13. 與其他蒸青綠茶比較：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北。中國現存最古老的蒸青綠茶（十九世紀）。外形—「松針形」，類似新林玉露。香氣—更多「麵包」和「穀物」味，「海洋」香較淡。滋味—較「厚重」和「溫暖」。新林玉露—更「清新」和「海洋」，得益於現代自動化生產線和更高氨基酸含量（4.2% vs ~3%）。

* **日本煎茶 (煎茶, Sencha)：** 蒸青，但使用日本栽培種（薮北等）。香氣—明顯「鮮味」和「海洋」；滋味—更「厚重」和「海藻」味。新林玉露—更溫和，帶大別山原料特有的甘甜栗香底韻。煎茶外形更「扁平」，新林玉露截面更「圓潤」。

* **日本玉露 (玉露, Gyokuro)：** 蒸青加遮陰。氨基酸含量極高（達6%）。滋味—濃郁「鮮味」，幾乎「湯汁」般。新林玉露—更輕盈清新，無「厚重」鮮味；無需遮陰即可憑藉山地微氣候達到高氨基酸含量。

* **信陽毛尖 (信阳毛尖, Xìnyáng Máo Jiān)：** 同一地區（信陽），同一栽培種，但炒青 (炒青)。香氣—栗綠香，無「海洋」香。滋味—更「乾爽」和澀。新林玉露—更溫和，顏色和香氣更「綠」，具獨特「海藻」風味。

**結語：**

新林玉露是連接古代中國與現代日本、八世紀與二十一世紀的橋樑茶品。陸羽描述的技術經過千年跨越東海，以自動化生產線形式回歸故土，與日本不知的原料相遇：來自大別山雲霧的厚葉山地栽培種。結果—具有「碧綠」茶湯、「海洋」香氣和無需人工遮陰創造的甘甜的茶品。以85°C上投法沖泡—觀察「松針」緩緩沉入水中，將水染成那種給這款獨特茶品命名的「玉露色澤」。
