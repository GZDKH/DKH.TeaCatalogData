# 洞庭碧螺春 (洞庭碧螺春, Dòngtíng bìluóchūn)

**洞庭碧螺春 (洞庭碧螺春, Dòngtíng bìluóchūn)**

**1. 分類與產地：**

* **類型：** 綠茶（未發酵茶，綠茶，lǜchá）。氧化程度低於5%。
* **類別：** 中國十大名茶（中國十大名茶，Zhōngguó shí dà míngchá）。中國十大名茶之一，地理標誌產品（地理標誌產品，dìlǐ biāozhì chǎnpǐn），中華人民共和國非物質文化遺產。
* **產地：** 中國江蘇省（江蘇，Jiāngsū）蘇州市（蘇州，Sūzhōu）吳中區（吳中區，Wúzhōng qū）。僅產於洞庭山（洞庭山，Dòngtíng shān）——洞庭東山（洞庭東山，Dòngtíng Dōngshān，「洞庭東山」，半島，現為東山鎮）和洞庭西山（洞庭西山，Dòngtíng Xīshān，「洞庭西山」，太湖中的島嶼，現為金庭鎮，金庭鎮）。根據國家標準，在這兩個地區以外生產的茶葉不能稱為「洞庭碧螺春」。
* **地理座標：** 約北緯31°03′，東經120°22′（太湖地區，太湖，Tàihú）。

**2. 歷史與文化意義：**

* **歷史：** 洞庭山的茶葉種植歷史可追溯至六朝時期（六朝，Liùcháo，西元3-6世紀），傳統已逾千年。唐代（唐，Táng，618-907）時，洞庭茶在陸羽的經典著作《茶經》（茶經，Cháijīng）中被提及，詩人皮日休（皮日休，Pí Rìxiū）和陸龜蒙（陸龜蒙，Lù Guīméng）歌頌太湖茶山。宋代（宋，Sòng，960-1279）時，洞庭的「水月茶」（水月茶，Shuǐyuè chá）已作為貢茶（貢茶，gòngchá，「茶葉貢品」）進獻朝廷。明代（明，Míng，1368-1644）時，洞庭出產「雲霧茶」和「雨前茗芽」。至清代（清，Qīng，1644-1912）初期，茶葉已接近現代形態，民間稱為「嚇煞人香」（嚇煞人香，Xiàshàrénxiāng），意為「香得嚇人」。據王應奎（王應奎，Wáng Yìngkuí）《柳南隨筆》（柳南隨筆，Liǔnán suíbǐ）記載，康熙三十八年（康熙三十八年，西元1699年），康熙皇帝南巡（南巡，nánxún）時，品嚐了巡撫宋犖（宋犖，Sòng Luò）進獻的洞庭茶。康熙認為民間名稱「嚇煞人香」過於粗俗，賜名「碧螺春」（碧螺春，Bìluóchūn）——「碧綠的螺旋春茶」。此後，該茶每年作為貢茶進獻朝廷。
* **名稱：**
    * **碧（bì）**——翡翠綠色、玉石般的綠色。反映茶湯和嫩葉的顏色，據一說也指洞庭東山的碧螺峰（碧螺峰，Bìluó fēng）。
    * **螺（luó）**——螺旋、蝸牛。指茶葉緊密捲曲的形狀，如同微小的貝殼。
    * **春（chūn）**——春天。象徵早春採摘，此時採集最嫩最珍貴的芽葉。
* **文化意義：** 碧螺春是江南（江南，Jiāngnán，「長江以南」）茶文化最具代表性的象徵之一。清代思想家龔自珍（龔自珍，Gōng Zìzhēn），本身是龍井故鄉杭州人，曾寫道：「茶以洞庭山之碧螺春為天下第一，古人未之知也」。2011年，碧螺春手工製作技藝被列入國家級非物質文化遺產名錄（國家級非物質文化遺產，Guójiājí fēi wùzhì wénhuà yíchǎn）。2020年，洞庭茶果複合系統被認定為中國重要農業文化遺產（中國重要農業文化遺產，Zhōngguó zhòngyào nóngyè wénhuà yíchǎn），茶葉本身獲得「農產品地理標誌」認證。

**3. 植物學描述與原料：**

* **品種/栽培種：** 使用當地小葉種群體——洞庭山群體小葉種（洞庭山群體小葉種，Dòngtíngshān qúntǐ xiǎoyèzhǒng，「洞庭山群體小葉種」），屬於*Camellia sinensis* var. *sinensis*。這是在島嶼和半島湖泊微氣候的獨特條件下，經過數世紀自然和民間選育的結果。葉片細小、柔嫩，氨基酸含量高，具有獨特的果香花香特徵。
* **採摘：** 從春分（春分，Chūnfēn，「春分」，約3月20-21日）開始，至穀雨（穀雨，Gǔyǔ，「穀雨」，約4月20日）結束。最珍貴的是「明前茶」（明前茶，míngqián chá），即清明節（清明，Qīngmíng，約4月5日）前採摘的茶葉。4月20日後，當地傳統不再稱為「碧螺春」，而稱為「炒青」（炒青，chǎoqīng，「炒製綠茶」）。
* **採摘標準：** 採摘最嫩的、剛剛展開的芽葉和一至兩片最上層的葉子，覆蓋著銀白色絨毛（白毫，白毫，báiháo）。芽葉長度1.6-2.0公分，形狀稱為「雀舌」（雀舌，quèshé，「麻雀舌頭」）。製作500克高級碧螺春需要68,000至74,000個芽葉；歷史記載中曾有每500克含90,000個芽葉的批次。
* **原料要求：** 極為嚴格。只使用均勻、完整的芽葉；在乾燥天氣、涼爽的早晨時分採摘。採摘三原則：「摘得早」、「採得嫩」、「揀得淨」。

**4. 風土條件與種植特色：**

* **獨特微氣候：** 洞庭山——東山（半島）和西山（島嶼）位於太湖中，三面或四面環水。湖霧繚繞、冬暖夏涼、散射光充足、濕度高，蘇州人形容為「冬暖夏涼」。空氣清新，水質達國家一級標準。
* **土壤：** 該地區特有的紅土壤（紅土壤，hóngtǔrǎng），有機質和礦物質含量高，微酸性（pH 4.5-6.0），排水良好，即使濕潤時仍保持透氣性。這是茶樹的理想基質。
* **茶果間作（茶果間種，cháguǒ jiānzhòng）：** 洞庭風土的關鍵特色。茶樹與果樹間種——枇杷（枇杷，pípá）、柑橘（柑桔，gānjú）、楊梅（楊梅，yángméi）、梅（梅，méi）、桃、杏、柿、石榴、銀杏。根系相互交織，果樹樹冠提供稀疏遮蔭，保護茶樹免受直射陽光，促進氨基酸積累。花果香氣被茶葉吸收，形成著名的「花香果味」（花香果味，huāxiāng guǒwèi）——這是洞庭碧螺春的標誌，所有外地仿製品都無法複製。
* **生長海拔：** 洞庭山不高——海拔300-350公尺；湖泊微氣候和濃霧彌補了海拔的不足。
* **種植面積與產量：** 生產區域極為有限；東山和西山的茶園總面積不大。正宗洞庭碧螺春年產量約100-120噸——僅占中國以「碧螺春」名義銷售茶葉總量的0.2%左右。

**5. 製作工藝：**

正宗洞庭碧螺春的生產是完全手工的過程，被認定為中國非物質文化遺產。其工藝特點是炒製、揉捻、做形和乾燥在同一口鍋（鍋，guō）中幾乎不間斷地完成。

* **採摘（採摘，cǎizhāi）：** 如上所述。在涼爽的早晨進行。
* **揀剔（揀剔，jiǎntī）：** 仔細挑選原料，去除受損和不符合標準的葉片。目標是絕對均勻的品級。
* **攤放（攤放，tānfàng）：** 將挑選好的原料薄薄攤放在竹篩上，置於陰涼處3-5小時，去除表面水分，輕微萎凋並開始形成香氣。
* **高溫殺青（高溫殺青，gāowēn shāqīng）：** 關鍵步驟。在加熱的傾斜鍋中（鍋底溫度約190-200°C）投入500-700克萎凋原料。製茶師傅赤手翻炒茶葉，停止酵素活性並固定綠色。青草味消失，花香開始顯現。
* **揉捻整形（揉捻整形，róuniǎn zhěngxíng）：** 不取出茶葉，稍微降低溫度。師傅翻炒、抖動和揉捻茶葉，使其初步形成螺旋狀。動作必須輕柔，以保持絨毛完整，避免葉片破碎。持續約12-15分鐘。
* **搓團顯毫（搓團顯毫，cuōtuán xiǎnháo）：** 碧螺春最具特色的步驟。溫度降至120-150°C。師傅分批將茶葉握在手中，搓成緊密的螺旋球狀，然後攤開在鍋壁上，再次搓團——如此反覆多次。正是在此步驟中，茶葉獲得最終的緊密螺旋形（「蜜蜂腿」，蜜蜂腿，mìfēng tuǐ），白毫顯露。原則：「先輕後重再輕」——壓力過大絨毛脫落，壓力不足螺旋不成形。持續約10分鐘。
* **文火乾燥（文火乾燥，wénhuǒ gānzào）：** 溫度降至50-60°C。輕輕翻動茶葉，直到含水量達到約7%。此時固定形狀並鎖定香氣。
* **冷卻與包裝：** 成品茶從鍋中取出，冷卻後立即密封包裝。

**6. 感官特徵：**

* **乾茶外觀：** 纖細優雅的茶葉，緊密捲曲成螺旋狀，形似蝸牛殼或「蜜蜂腿」（銅絲條，銅絲條，tóngsītiáo，「銅絲」）。覆蓋豐富的銀白色絨毛（白毫顯露）。色澤為銀綠隱翠（銀綠隱翠，yínlǜ yǐncuì）。
* **乾茶香氣：** 濃郁、多層次：前調為清新花香（茉莉、庭園花卉），中調為果香（桃、杏、枇杷），後調為淡淡的奶油蜂蜜香。這種天然的「花香果味」（花香果味）是正宗洞庭產地的標誌。
* **茶湯香氣：** 明亮、純淨、甜美花香，帶有蜂蜜和水果的細微差別；香氣從第一泡持續到最後一泡。
* **滋味：** 細膩、柔和、鮮爽（鮮爽，xiānshuǎng）、微甜，無粗糙苦澀感。明顯的氨基酸甜味（回甘，huígān）和生津效果（生津，shēngjīn）。特點：第一泡滋味細膩精緻，第二、三泡飽滿濃郁；「花香果味」貫穿整個品飲過程。
* **湯色：** 嫩綠帶淡黃色調（嫩綠，nèn lǜ），明亮、透明、清澈。高級品帶有微妙的翡翠光澤。
* **葉底（沖泡後的葉片）：** 嫩、完整、有彈性的芽葉；色澤均勻淺綠（嫩黃綠明亮）；可清楚看到單個芽葉。

**7. 化學成分：**

洞庭碧螺春的特點是氨基酸含量高，香氣複合物豐富，這是由於小葉品種、果樹遮蔭和早春採摘所致。

* **多酚類（茶多酚，chá duōfēn）：** 總含量約占乾重的15-20%（根據個別分析高達11.1克/100克）。主要兒茶素：表沒食子兒茶素沒食子酸酯（EGCG）、表兒茶素沒食子酸酯（ECG）、表兒茶素（EC）。多酚與氨基酸的比例低於大多數綠茶，確保了口感的柔和。
* **氨基酸（氨基酸，ānjīsuān）：** 總含量約2.9克/100克（根據實驗室研究為2916.29毫克/100克），超過黃山毛峰和廬山雲霧。以L-茶氨酸（L-茶氨酸，L-chá ānjīsuān）為主，負責甜味、「鮮味」和放鬆效果。
* **生物鹼：** 咖啡鹼（咖啡鹼，kāfēi jiǎn）占乾重的2-4%，符合早春綠茶的標準水平；可可鹼和茶鹼含量極少。
* **精油和芳香化合物：** 研究在洞庭碧螺春的香氣中鑑定出42種成分，包括11種醇類、7種含氮雜環化合物、6種醛類、5種萜烯、3種酸、2種酮和其他物質。正是這種複雜的芳香組合將正宗洞庭茶與仿製品區分開來。
* **維生素：** 抗壞血酸（維生素C）、B群維生素、維生素A（類胡蘿蔔素）——維生素A含量高於紅茶和黑茶。
* **礦物質：** 氟（F）、鉀（K）、鎂（Mg）、鋅（Zn）、錳（Mn）、硒（Se，微量）。

**8. 保健功效：**

* **抗氧化作用：** 高含量的兒茶素（特別是EGCG）有效中和自由基；茶多酚的抗氧化能力是維生素E的6-7倍。
* **提神和認知效果：** 咖啡鹼和L-茶氨酸的協同作用提供溫和但持久的專注力和工作效率提升，沒有咖啡特有的「高峰」和「低谷」。
* **支持心血管系統：** 多酚有助於降低低密度脂蛋白膽固醇和三酸甘油酯，改善血管彈性，抑制血小板聚集。
* **改善消化：** 咖啡鹼和兒茶素刺激胃液分泌，加速脂肪代謝；傳統上飯後飲用碧螺春以助消化。
* **增強免疫力：** 維生素C和氨基酸共同支持免疫功能；經常飲用綠茶與感冒發病率降低有關。
* **保護口腔健康：** 茶葉中的氟含量強化牙齒琺瑯質，降低蛀牙風險；多酚抑制口腔細菌生長。
* **支持皮膚健康：** 多酚的抗氧化特性減緩光老化，減少皮膚細胞的氧化壓力。
* **清新效果：** 明顯的止渴生津（生津止渴，shēngjīn zhǐkě）特性——在江南炎熱潮濕的氣候中最受重視的特質之一。

**9. 沖泡方法：**

碧螺春是少數傳統上採用「上投法」（上投法，shàngtóufǎ）的茶葉之一：先注水，後投茶。這是因為茶葉極其細嫩。

* **水溫：** 75-80°C。沸水會「燙傷」嫩葉，產生苦味並破壞細膩香氣。
* **茶量：** 3-5克配150-200毫升水。
* **茶具：** 最佳選擇——玻璃杯或玻璃壺（以觀察茶葉「舞動」）、薄瓷蓋碗（蓋碗，gàiwǎn）或青瓷茶具。
* **沖泡過程（上投法）：**
    1. 用開水預熱玻璃杯或蓋碗，倒掉水。
    2. 將杯子注入約2/3的水（75-80°C）。
    3. 輕輕將茶葉撒在水面上。觀察茶葉緩緩下沉至杯底，展開並釋放香氣——這是品茶的美學享受之一。
    4. 第一泡——1.5-2分鐘。香氣細膩，滋味精緻。
    5. 第二泡——2-3分鐘。滋味達到飽滿。
    6. 第三泡——3-4分鐘。滋味和香氣逐漸柔和。
    7. 優質洞庭碧螺春可沖泡3-4次。
* **功夫泡法（功夫泡法，gōngfū pàofǎ）：**
    1. 預熱蓋碗。
    2. 投茶：4-5克配100-120毫升。
    3. 洗茶：新鮮碧螺春通常不需要。
    4. 第一泡：75-80°C水溫，10-15秒。
    5. 後續沖泡：5-7泡，每次增加5-10秒。
* **特點：** 高級洞庭碧螺春甚至可以用室溫的泉水或礦泉水沖泡（冷泡）——這是芽葉強韌多汁的標誌，仿製品無法達到。

**10. 儲存方法：**

* **條件：** 乾燥、陰暗、涼爽的地方，遠離異味。
* **容器：** 密封——瓷罐、密封鐵罐或真空包裝。避免透明容器（光線加速葉綠素氧化）。
* **溫度：** 最佳——冰箱0-5°C。碧螺春是早春綠茶，氨基酸含量高且吸濕性強，在室溫下比炒製綠茶更快失去新鮮度。
* **保質期：** 理想情況下在生產後6-12個月內飲用。當年新茶（新茶，xīnchá）價值最高；隔年茶失去特有的「活力」。
* **茶葉的敵人：** 濕氣、光線、高溫、異味、氧氣。

**11. 價格與仿冒品：**

* **價格等級：** 正宗洞庭碧螺春是中國最昂貴的綠茶之一。最高等級（特一、特二）的價格可達每500克數千元人民幣。成本因素：微小的生產區域、完全手工勞動、單位重量所需的大量芽葉、有限的年產量（全中國約100-120噸）。
* **仿冒問題：** 在中國和海外銷售的絕大多數「碧螺春」都是在洞庭以外生產的——在浙江、福建、四川、雲南。這些茶葉外觀相似，但缺乏特有的「花香果味」，口感較粗糙，帶有青草苦味。
* **如何避免仿冒品：**
    * **檢查包裝上的標準：** 正宗茶葉標有國家標準GB/T 18957「地理標誌——洞庭（山）碧螺春」。如果標示行業標準（NY/T）或企業標準（Q/）——這是來自其他地區的茶葉。
    * **評估外觀：** 正宗洞庭碧螺春極其纖細，「如銅絲」；絨毛天然，分布均勻。仿冒品通常較大較粗，絨毛可能是人工添加的（例如枇杷絨毛）。
    * **聞乾茶：** 天然洞庭碧螺春具有明顯的花香果味。仿冒品聞起來像「生青草」或「土味」。
    * **評估茶湯：** 正宗茶湯清澈、透明、嫩綠；假茶湯混濁、偏黃。
    * **注意價格：** 可疑的低價（「最高級」每500克低於500元人民幣）——幾乎可以肯定是洞庭以外生產的茶葉。

**12. 趣聞軼事：**

* 生產500克最高級洞庭碧螺春需要68,000至74,000個單獨的芽葉——每個芽葉都是手工採摘的。歷史記載中有每500克乾茶含90,000個芽葉的記錄。
* 正宗洞庭碧螺春用「上投法」沖泡時會沉到杯底，而不是漂浮在水面——這是傳統的真偽測試，在民國時期的記載中就有歌頌。
* 洞庭碧螺春是中國唯一一種其生產生態系統（茶果複合園）和加工技術同時被列入文化遺產名錄的茶葉——這是江蘇茶業獨有的雙重地位。
* 正宗洞庭碧螺春的年產量（約100-120噸）僅占中國以此名稱銷售的所有茶葉的0.2%左右。其餘99.8%是其他地區的產品。
* 科學家在香氣成分中鑑定出42種揮發性化合物——這是中國綠茶中最複雜的香氣特徵之一。

**13. 與其他綠茶的比較：**

* **西湖龍井（西湖龍井，Xīhú Lóngjǐng）：** 兩者都是十大名茶，都是春茶、小葉種。主要區別：龍井扁平，壓製成「雀舌」形，無絨毛，高溫炒製，呈現「栗豆香」；碧螺春螺旋狀，密布絨毛，花香果味。龍井更「油潤」厚重，碧螺春更細膩「輕盈」。
* **黃山毛峰（黃山毛峰，Huángshān Máofēng）：** 相似的絨毛和細嫩，但毛峰呈「花蕾」形（非螺旋），香氣更「高山」，帶蘭花和鮮綠香，無明顯果香成分。毛峰可承受稍高的沖泡溫度。
* **銀螺（銀螺，Yín Luó）：** 形式上最接近的「形態」類似物（捲曲成螺旋，有絨毛）。然而，銀螺不是地理標誌茶，在多個省份用不同品種生產，不具備「花香果味」，通常葉片較大，風味特徵較不明顯。價格明顯較低。
* **信陽毛尖（信陽毛尖，Xìnyáng Máojiān）：** 同樣是有絨毛的春季綠茶，但產自河南。葉片更直更細，香氣帶栗香和草香，無果香。口感更澀。
* **太平猴魁（太平猴魁，Tàipíng Hóukuí）：** 形態上的對立面——巨大扁平的葉片長達6-7公分。蘭花香，口感柔和，但美學和感官體驗完全不同。

**結語：**

洞庭碧螺春是江南春天的精華，濃縮在微小的銀綠色螺旋中。千年風土、太湖島嶼上的茶果複合園、稀有的小葉品種和完全手工的技藝，共同造就了這款真正獨一無二的茶葉。它的花香果味、氨基酸甜味和極其細膩的茶湯質地——不是行銷辭藻，而是經實驗室證實的深層生化特性的結果。碧螺春是為那些珍視精緻、願意放慢腳步、觀察微小茶葉在透明水中旋轉下沉、釋放香氣的人而生的茶——四個世紀前，採茶女們給了它最準確的名字——「香得嚇人」。

這款茶不僅是飲品，更是文化的載體、歷史的見證、自然與人類智慧的結晶。每一杯洞庭碧螺春都訴說著太湖的晨霧、果園的花香、製茶師傅的巧手，以及延續千年的茶文化傳統。在快節奏的現代生活中，品飲一杯正宗的洞庭碧螺春，不啻是一次心靈的回歸，一場與江南春天的美麗邂逅。
