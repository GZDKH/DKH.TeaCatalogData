# 蒙頂山綠毛峰 (蒙顶山绿毛峰, Méngdǐngshān lǜ máo fēng)

**蒙頂山綠毛峰 (蒙顶山绿毛峰, Méngdǐngshān lǜ máo fēng)**

蒙頂山綠毛峰 (蒙顶山绿毛峰, Méngdǐngshān lǜ máo fēng) —「蒙頂山綠色毛峰」— 來自四川省雅安市 (雅安市, Yǎ'ān Shì) 蒙頂山 (蒙顶山, Méngdǐng Shān, 1456公尺) 的精緻烘青綠茶，此山為傳說中的「世界茶業搖籃」，相傳道士吳理真 (吴理真, Wú Lǐzhēn) 於西漢時期 (西元前二世紀) 在此種植了七株茶樹。自此蒙頂山被譽為「西蜀漏天，蒙頂仙茶」—「西蜀如漏天，蒙頂出仙茶」。蒙頂山茶作為「貢茶」(贡茶, gòngchá，「進貢茶」) 長達1169年 — 自西元742年 (唐玄宗天寶年間) 至清朝末年 — 歷經五個朝代，因此獲得「五朝貢茗」(Wǔ Cháo Gòng Míng，「五朝貢茶」) 的稱號 — 中國歷史上最長的連續貢茶傳統。蒙頂山綠毛峰與其他蒙頂山茶的區別在於其明顯的筆直條索形狀 (紧细匀直, jǐnxì yúnzhí) 和明代的「三炒三揉」(sān chǎo sān róu，「三次炒製三次揉捻」) 工藝。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵。精緻烘青綠茶 (细嫩烘青绿茶, xìnèn hōngqīng lǜchá)。形狀 — 筆直條索 (紧细匀直, jǐnxì yúnzhí)。發酵程度 — 0%。

* **類別：** 蒙頂山茶 (蒙顶山茶, Méngdǐngshān Chá) 的代表。「五朝貢茗」(五朝贡茗, 742年 — 清末, 1169年)。蒙頂山茶列入「全國十大名茶」(全国十大名茶, 1959年)。行業標準《蒙頂山茶 第2部分：綠茶》(《蒙顶山茶 第2部分：绿茶》) 於2020年頒布。核心產區 —「古皇茶園」(古皇茶园, Gǔ Huáng Cháyuán) 位於蒙頂山五峰：上清峰 (上清峰)、甘露峰 (甘露峰)、靈隱峰 (灵隐峰) 等。

* **產地：** 中國四川省 (四川省, Sìchuān Shěng) 雅安市 (雅安市, Yǎ'ān Shì) 名山區 (名山区, Míngshān Qū) 及部分雨城區 (雨城区, Yǔchéng Qū)。該地區位於「北緯30°黃金產茶帶」(北纬30°黄金产茶带, Běiwěi 30° Huángjīn Chǎnchá Dài) — 連接世界許多著名茶區的緯度走廊。

* **地理座標：** 約北緯30°05′，東經103°12′。

**2. 歷史與文化意義：**

* **西漢 — 吳理真。** 道士吳理真 (吴理真) 在蒙頂山頂種植七株茶樹 — 世界歷史上第一次有記載的茶樹栽培。這七株茶樹被命名為「聖揚花」(圣扬花, Shèng Yánghuā，「神聖綻放之花」) 和「吉祥蕊」(吉祥蕊, Jíxiáng Ruǐ，「吉祥花蕊」)。吳理真被尊為「茶祖」(茶祖, Cházǔ) 和「茶神」(茶神, Cháshén)。種植地至今仍存在「古皇茶園」(古皇茶园)，以石牆圍繞 — 茶史的活博物館。

* **唐代 —「貢茶」(742年)。** 自唐玄宗天寶 (天宝) 年間起，蒙頂山茶成為「貢茶」— 皇室進貢茶。由此開始了1169年不間斷的「貢茶」傳統 — 中國歷史上最長的，跨越五個朝代：唐 (唐, 618–907) → 宋 (宋, 960–1279) → 元 (元, 1271–1368) → 明 (明, 1368–1644) → 清 (清, 1644–1912)。

* **明代 —「三炒三揉」。** 明代朱元璋 (明太祖) 頒布「罷造龍團」(停止製作緊壓茶) 詔令後，蒙頂山形成了「三炒三揉」(三炒三揉, sān chǎo sān róu) 工藝，至今仍用於綠毛峰的製作。此工藝確保了深厚「濃醇」(浓醇, nóngchún) 的口感，這是單次加工無法達到的。

* **1959年 — 中國十大名茶。** 蒙頂山茶列入「全國十大名茶」— 權威的「十大名茶」之一，確立了蒙頂山作為國家主要茶葉產區之一的地位。

* **李時珍與「溫性」綠茶。** 在《本草綱目》(《本草纲目》, Běncǎo Gāngmù，《藥物大全》, 1578年) 中，偉大的藥物學家李時珍 (李时珍) 記載：「惟雅州蒙山出者溫而主疾」—「只有雅州蒙山所產者性溫且治病」。這是獨特的特徵：絕大多數綠茶在傳統中醫 (TCM) 中被歸類為「寒性」(寒性, hánxìng) 或「涼性」(凉性, liángxìng)，而蒙頂茶為「溫性」(温性, wēnxìng)，適合「寒性」體質的人飲用。

* **茶名解析。** 蒙頂山 (Méngdǐng Shān) —「霧罩山頂」：蒙 —「覆蓋、籠罩」(指常年雲霧)；頂 —「山頂」；山 —「山」。綠 (Lǜ) —「綠色」— 指茶類。毛峰 (Máo Fēng) —「毛茸山峰」：毛 —「茸毛、絨毛」(芽上的白毫)；峰 —「峰、山頂」(茶葉尖端如山峰)。完整詩意：「來自雲霧籠罩山頂的綠色毛茸山峰」。

* **文化意義。** 蒙頂山不僅是產茶區，更是茶業起源的象徵。雅安地區被稱為「雨城」(雨城, Yǔchéng) — 中國最潮濕的地方之一，因此有「西蜀漏天」(「西蜀如漏天」) 的說法。蒙頂山茶文化與道教密不可分：吳理真是道士，山上的茶一直被視為「仙茶」(仙茶, xiānchá，「仙人之茶」)。每年的蒙頂山茶節吸引數萬朝聖者和愛好者。

**3. 植物學描述與原料：**

* **品種：** *Camellia sinensis* var. *sinensis*。

* **栽培品種：** 蒙山群體種 (蒙山群体种, Méngshān Qúntǐzhǒng) — 當地小葉種群體品種，在蒙頂山高海拔、超高濕度氣候條件下經過數百年形成。特點是抗寒性強、芽上白毫豐富、氨基酸含量高 — 這是對常年雲霧和散射光適應的結果。芽葉嫩綠、緊實，具有特徵性的「毛茸」外觀。

* **採摘：** 早春採摘，在春分 (春分, Chūnfēn，「春季分點」，約3月20-21日) 之前或期間。採摘標準：一芽一葉 (一芽一叶, yī yá yī yè)。芽長 ≤2.5公分。特級茶需要40,000–50,000個芽製成500克成品茶 (80,000–100,000個/公斤)，顯示其極致的嫩度和工藝要求。

* **等級：**
  * **特級 (特级, tèjí)：** 完整芽或一芽一葉初展 (一芽一叶初展)。茶條筆直、細嫩、嫩綠色帶濃密白毫。香氣 — 栗香 (栗香)。價格 — 500–800元人民幣/500克。
  * **一級 (一级, yī jí)：** 一芽一葉 (一芽一叶)。茶條均勻，白毫較少。
  * **二級 (二级, èr jí)：** 一芽二葉 (一芽二叶)。形狀稍鬆散，香氣較淡，口感更濃郁。

**4. 風土與栽培特色：**

* **氣候：** 北亞熱帶濕潤氣候。年平均溫度 — 15.5°C。年降雨量 — ≥1500毫米 — 中國最潮濕的茶區之一。創紀錄的霧日：每年超過280天霧日 (某些資料顯示達300+天)，提供幾乎持續的散射光並抑制紫外線輻射。相對濕度 — 82%。正是這種微氣候造就了著名的「漏天」(「漏天」) 稱號。

* **海拔：** 1000–1400公尺。核心區 — 蒙頂山五峰和山頂的「古皇茶園」(古皇茶园)。海拔差異和持續雲霧創造垂直氣候帶，即使在同一座山內也能提供多樣的風味特徵。

* **土壤：** 黃棕壤土 (黄棕壤土, huángzōng rǎngtǔ)。pH 4.5–5.6 — 微酸性，最適合茶樹。土壤富含有機物質和微量元素 — 硒 (Se) 和鋅 (Zn)，這是由蒙頂山岩石地質成分決定的，為茶葉獨特的礦物質特徵做出貢獻。

* **「李時珍公式」。** 蒙頂茶的「溫性」(温性) 特徵 — 獨特性質，由風土因子綜合作用解釋：持續雲霧降低多酚類的光合積累 (負責「寒性」特徵)，柔和散射光促進氨基酸和糖類的增加積累，高濕度創造次生代謝物的特殊平衡。結果 — 茶葉「內在溫暖」而非像大多數綠茶那樣「清涼」。

**5. 製作工藝：**

明代「三炒三揉」(sān chǎo sān róu，「三次炒製三次揉捻」) 工藝 — 蒙頂山綠毛峰的名片，區別於該地區其他茶類 (蒙頂甘露、蒙頂黃芽)：

* **攤青 (摊青, tānqīng)：** 新鮮採摘的原料薄攤在竹篩上，置於通風室內。開始輕微蛋白質水解，釋放游離氨基酸，降低草腥醛含量。

* **第一輪：** 殺青 (杀青, shāqīng) — 快速高溫固定，使氧化酶失活並固定綠色。→ 揉捻 (揉捻, róuniǎn) — 機械破壞細胞壁，釋放細胞汁液。

* **第二輪：** 再次炒製 → 再次揉捻。深化香氣特徵，進一步緊實形狀。

* **第三輪：** 最終炒製 → 最終揉捻。固定「濃醇」(浓醇, nóngchún) 口感，形成最終緊密、細嫩的條索結構。

* **做形 (做形, zuòxíng)：** 使茶葉形成筆直、細嫩、均勻的形狀 (紧细匀直)。

* **烘乾 (烘干, hōnggān)：** 低溫最終乾燥至含水量 ≤6%。固定香氣並確保儲存穩定性。

三次重複「炒製 — 揉捻」循環 — 工藝的關鍵區別。與大多數綠茶的單次處理不同，三輪循環確保：條索形狀的最大緊實；沖泡時更深層和「層次化」的風味萃取；特徵性「濃醇」(浓醇，「濃厚醇和」) 風味特徵，具有明顯的深度和厚度。

**6. 感官特徵：**

* **乾茶外觀：** 緊密、細嫩、筆直的條索 (紧细匀直, jǐnxì yúnzhí)。顏色 — 嫩綠 (嫩绿, nèn lǜ) 帶明顯白毫。茶葉大小形狀均勻，略尖 —「山峰」(峰)。

* **乾茶香氣：** 清雅、甘甜，帶栗香和草香調。淡淡「玉米香」(玉米香, yùmǐ xiāng) — 蒙頂山特有的風土香調。

* **茶湯香氣：** 「嫩栗香」(嫩栗香, nèn lìxiāng) — 主導香調。「青葉香」(青叶香, qīngyè xiāng) — 背景清新。「玉米香」(玉米香) — 蒙頂山特有的風土香調，幾乎不見於其他地區的綠茶。香氣持久，逐漸展開。

* **口感：** 鮮醇 (鲜醇, xiānchún)。濃醇 (浓醇, nóngchún) — 關鍵特徵，由「三炒三揉」工藝造就。甘爽 (甘爽, gānshuǎng) 帶持久回甘 (回甘, huígān)。苦味極少，因多酚含量較低 (280+天雲霧的結果)。餘韻 — 悠長，帶淡淡「溫暖」調，大多數綠茶所沒有的。

* **湯色：** 黃亮明淨 (黄亮明净, huáng liàng míng jìng)。不是許多綠茶的綠色，而是黃色 — 烘青 (烘青) 工藝和三次炒製的結果。

* **葉底 (沖泡後茶葉)：** 綠黃勻亮 (绿黄匀亮, lǜhuáng yún liàng)。葉片完全展開，保持完整性和柔軟度。

**7. 化學成分：**

* **多酚類 (茶多酚, chá duōfēn)：** 含量適中 — 低於低海拔陽光充足地區的茶葉。原因 — 280+天雲霧抑制兒茶素的光合積累。較低的多酚含量確保茶葉「溫性」(温性) 特徵和極少苦味。主要成分 — 兒茶素 (EGCG, ECG, EGC)，具有抗氧化作用。

* **氨基酸 (氨基酸, ānjīsuān)：** 含量較高 — 創紀錄雲霧天數 (280+天) 的直接結果：散射光減緩氨基酸分解並促進其積累。主要成分 — L-茶氨酸 (茶氨酸, chá ānjīsuān)，約占游離氨基酸總量的50%。L-茶氨酸賦予清雅鮮味、放鬆效果和「溫暖」特徵。多酚/氨基酸比 (酚氨比, fēn'ān bǐ) — 中國綠茶中最低之一，使綠毛峰成為「鮮而不苦」的典範。

* **礦物質：** 硒 (Se) 和鋅 (Zn) — 含量較高，由蒙頂山岩石礦物成分決定。硒 — 強抗氧化劑，參與穀胱甘肽過氧化酶作用。鋅 — 免疫功能和蛋白質合成必需。鉀、鈣、鎂、鐵、錳 — 綠茶標準含量。

* **咖啡鹼 (咖啡碱, kāfēi jiǎn)：** 含量適中 (乾重的2–3%)。與L-茶氨酸結合提供溫和提神效果，無明顯興奮感。

* **黃酮類 (黄酮类, huángtóng lèi)：** 含量顯著，對抗氧化潛力和茶湯黃色有貢獻。

* **維生素：** 維生素C (含量顯著，因最少發酵而保存)、B族維生素 (B1, B2)、維生素E。

* **精油 (芳香物质)：** 負責特徵性「栗香玉米香」。主要成分 — 芳樟醇、橙花醇、糠醛 (負責栗香)、己醛和順-3-己烯醇 (草香調)。

**8. 健康功效：**

* **抗氧化作用。** 兒茶素和黃酮類中和自由基。硒通過穀胱甘肽過氧化酶增強抗氧化防護。

* **「溫性」特徵 — 適合「寒性」體質。** 李時珍記載的獨特性質：與大多數綠茶不同，蒙頂山綠毛峰不「清涼」身體，而是溫和「溫暖」，適合胃敏感和在中醫術語中傾向「寒性」狀態的人。

* **提神效果。** 咖啡鹼與L-茶氨酸結合提供溫和、持久的清醒和專注力提升，無焦慮和心悸。

* **消化支持。** 適量多酚刺激胃液分泌和蠕動，不刺激黏膜。「溫性」特徵對消化系統額外有益。

* **抗菌作用。** 兒茶素抑制胃腸道和口腔病原微生物生長。

* **免疫支持。** 鋅和維生素C協同增強免疫功能。硒參與免疫反應調節。

* **認知功能支持。** L-茶氨酸刺激大腦α波，改善記憶、學習能力和專注力。

* **心血管系統支持。** 多酚和黃酮類有助於降低LDL膽固醇水平並改善血管彈性。

**9. 沖泡方法：**

* **水溫：** 80–90°C。特級 — 80°C (更嫩的原料需要更低溫度)。一級和二級 — 至90°C。

* **茶量：** 3克/150毫升水 (1:50比例)。

* **茶具：** 玻璃杯 (欣賞茶葉舒展的視覺享受)、瓷蓋碗 (盖碗, gàiwǎn) 或瓷茶壺。為最大程度展現「濃醇」口感，建議用蓋碗 — 能更精確控制浸泡時間。

* **沖泡過程：**
  1. 用熱水溫杯，倒掉。
  2. 投入3克茶葉。
  3. 注水至1/3容量，讓茶葉「醒茶」20–30秒。
  4. 注滿水至全容量。
  5. 第一泡 — 浸泡60–90秒 (玻璃杯) 或10–15秒 (蓋碗，功夫茶法)。
  6. 後續沖泡 — 每泡增加15秒。茶葉可沖泡3–4次，逐漸展現甘甜和栗香深度。

**10. 儲存：**

* **製作後「醒茶」。** 新茶建議在室溫下放置10–15天 (退火, tuìhuǒ) 以穩定香氣。

* **主要儲存。** 密封包裝，冰箱0–5°C保存。避光、防潮、防異味和防氧化。

* **長期儲存。** 超過6個月儲存 — 真空包裝冷凍於−18°C。從冷凍庫取出時 — 讓包裝在室溫下完全回溫後再開封，避免茶葉凝露。

* **保質期。** 適當條件下 — 至18個月。開封後 — 1–2個月內飲用完畢。

**11. 價格與假冒：**

* **價格範圍。** 特級 (特级) — 500–800元人民幣/500克 (≈70–110美元)。一級 — 200–500元人民幣/500克。二級 — 100–200元人民幣/500克。影響價格因素：採摘時間 (春分前顯著更貴)、原料等級、茶園海拔、茶場聲譽。

* **如何避免假冒：**
  * **購買標有「蒙頂山茶」的產品。** 正宗茶葉標有地理標誌。注意2020年行業標準的存在。
  * **外觀評估。** 正宗綠毛峰 — 均勻、細嫩、筆直的條索 (紧细匀直) 帶白毫，嫩綠色。假冒品常有不均勻形狀、暗淡顏色或粗糙質地。
  * **香氣評估。** 特徵性「嫩栗香」(嫩栗香) 帶玉米調 — 正宗茶的名片。缺乏玉米香或有刺激性「烘烤」味 — 值得懷疑。
  * **茶湯檢查。** 茶湯應為黃亮明淨 (黄亮明净)，而非渾濁或綠灰色。
  * **價格檢查。** 聲稱「特級」但價格低於300元人民幣/500克 — 值得警惕。

**12. 有趣事實：**

* **1169年「貢茶」— 連續5個朝代。** 中國歷史上最長的連續「貢茶」傳統：唐 → 宋 → 元 → 明 → 清。沒有其他茶區能與此記錄相比。

* **每年280+天霧日。** 中國所有茶區中的創紀錄數字。「漏天」(漏天) — 雅安的古稱，反映幾乎不間斷的雨霧。這種氣候極端反而為茶樹創造了理想條件。

* **唯一的「溫性」綠茶。** 李時珍在《本草綱目》中將蒙頂茶歸類為「溫性」— 中醫中唯一被賦予「溫性」特徵的綠茶。所有其他綠茶都是「寒性」或「涼性」。

* **吳理真的七株茶樹 — 世界首次茶樹栽培。** 蒙頂山頂的「古皇茶園」(古皇茶园) — 傳說中人類茶道開始的地方。古茶樹周圍的七個石圍至今保存，作為活博物館。

* **「三炒三揉」(三炒三揉)。** 明代工藝 — 三輪炒製和揉捻循環 — 綠茶中獨有。大多數綠茶經過一次炒製；雙重處理罕見；三重處理極其特殊，提供單次處理無法達到的口感深度。

**13. 與其他蒙頂山茶和綠茶「毛峰」的比較：**

* **蒙頂甘露 (蒙顶甘露, Méngdǐng Gānlù)。** 蒙頂山最著名的茶。區別：「甘露」— 半捲曲螺旋形 (卷曲)，「三炒三揉」+ 額外揉捻工藝；綠毛峰 — 筆直條索形 (紧细匀直)。「甘露」香氣 — 更「甘露」(字面意思「甘露」= 「甜露」)；綠毛峰 — 更「栗香玉米香」。「甘露」被視為產品線旗艦，價格更高。

* **蒙頂黃芽 (蒙顶黄芽, Méngdǐng Huángyá)。** 同山黃茶。根本區別 —「悶黃」(闷黄，「黃悶」) 工序，賦予茶葉「黃色」發酵 (~10–15%)。口感更溫和甘甜，帶蜂蜜調；香氣較不清新但更深厚。綠毛峰 — 純綠茶，無發酵。

* **黃山毛峰 (黄山毛峰, Huángshān Máo Fēng)。** 主要「同名」— 安徽著名「毛峰」。區別：「黃山」— 芽葉帶特徵性「金魚葉」(金鱼叶, jīnyú yè)；蒙頂山 — 無「金魚葉」。「黃山」香氣 — 蘭花香，更花香；蒙頂山 — 栗香玉米香，更「穀物」。風土：「黃山」— 花崗岩酸性土壤；蒙頂山 — 黃棕壤，富含Se和Zn。

* **蒙頂石花 (蒙顶石花, Méngdǐng Shíhuā)。** 蒙頂山最古老的茶，早期「貢茶」之一。形狀 — 扁平，「石花」(石花)。工藝不同於「三炒三揉」：單次炒製加做形。口感更細膩清雅，但無綠毛峰的「濃醇」深度。

**結論：**

蒙頂山綠毛峰 — 來自茶業起源之山的茶：吳理真的七株茶樹、1169年貢茶歷史、280天雲霧和傳統中醫中唯一的「溫性」綠茶。明代「三炒三揉」工藝賦予其「濃醇」口感，這是單次處理無法達到的，而風土特有的「玉米栗香」無可替代。對於那些不只是喝茶，而是品味茶業歷史 — 近兩千年歷史 — 的人來說，蒙頂山綠毛峰不是飲品，而是通往茶文明根源的旅程。
