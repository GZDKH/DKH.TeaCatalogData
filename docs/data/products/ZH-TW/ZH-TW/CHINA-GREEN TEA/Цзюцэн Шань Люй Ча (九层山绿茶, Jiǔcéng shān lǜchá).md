# 九層山綠茶 (九层山绿茶, Jiǔcéng shān lǜchá)

**九層山綠茶 (九层山绿茶, Jiǔcéng shān lǜchá)**

![Цзюцэн Шань Люй Ча (九层山绿茶, Jiǔcéng shān lǜchá)-1](images/Цзюцэн%20Шань%20Люй%20Ча%20%28九层山绿茶,%20Jiǔcéng%20Shān%20Lǜchá%29-1.png)

九層山綠茶 (九层山绿茶, Jiǔcéng shān lǜchá) 是中國貴州省的高山綠茶，產於六盤水市六枝特區的九層山山脈。此茶具有獨特的「低緯度、高海拔、寡日照」(低纬度、高海拔、寡日照) 風土條件，自2017年起受中華人民共和國地理標誌產品保護，以「乾淨茶」(干净茶) 的聲譽著稱，通過歐盟標準超過400項農藥殘留檢測。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。有兩種形式：扁形 (扁形, biǎnxíng) 和捲曲形 (卷曲形, juǎnqū xíng)。
* **類別：** 貴州高山綠茶；中國地理標誌區域綠茶。屬於省級品牌「貴州綠茶」(贵州绿茶, Guìzhōu Lǜchá) 體系。
* **產地：** 中國貴州省 (贵州省, Guìzhōu Shěng) 六盤水市 (六盘水市, Liùpánshuǐ Shì) 六枝特區 (六枝特区, Liùzhī Tèqū)。地理標誌保護區域涵蓋六枝特區全境。
* **地理座標：** 約北緯26°12′，東經105°28′（該區域位於北緯26度「茶葉生產黃金帶」）。

**2. 歷史與文化意義：**

* **歷史：** 九層山山脈的茶業始於1960年代，當時建立了城市「知青農場」(知青农场, zhīqīng nóngchǎng)，奠定了茶園基礎。在半坡村 (半坡村, Bànpō Cūn) 保存著千年古茶樹 (千年古茶树)，見證了茶樹在此地區的千年存在。2008年成立「六枝特區九層山土特產開發有限公司」(六枝特区九层山土特产开发有限公司)，註冊「九層山」品牌並開始大規模開發該地區茶葉資源。2017年獲得國家地理標誌保護產品 (国家地理标志保护产品) 地位。2024年「S9」系列在國際茶葉展覽會上榮獲銀獎，「九層山茶」品牌價值評估為256.7億元人民幣。

* **名稱：** 九層山 (九层山) — 字面意思為「九層之山」（九 jiǔ — 九；層 céng — 層、階；山 shān — 山）。據當地傳說，山脈中約有一百座山峰，層層疊疊如多層樓塔，故得「九層」之名。綠茶 (绿茶, lǜchá) — 綠茶。

* **文化意義：** 九層山綠茶成為六枝特區經濟復興和「綠色發展」(绿色发展) 的象徵——這個前煤礦區轉向生態農業和茶旅遊。六枝特區是具有獨特「特區」(特区, tèqū) 地位的行政區劃，創建於1960年代管理煤炭資源，現已成為可持續發展的典範。茶產品積極推廣作為貴州山區脫貧攻堅和創造就業機會戰略的一部分。六枝特區有七個鄉鎮擁有超過萬畝的茶園，茶園總面積超過107,000畝（超過7,100公頃）。

**3. 植物學描述與原料：**

* **品種/栽培種：** 種植基礎（約60%）為福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbáichá) 栽培種——特早生品種（*Camellia sinensis* var. *sinensis*），芽葉萌發早，多酚含量高，適合生產香氣濃郁的扁形綠茶。此外還栽培本地群體種 (本地群体种, běndì qúntǐ zhǒng)，包括大葉種和中葉種，葉片厚實多肉，富含果膠物質——是製作毛尖 (毛尖) 類捲曲茶的理想原料。該地區有超過300,000株百年以上古茶樹。
* **採摘：** 主要採摘季節為春季，以早春明前 (明前, míngqián) 為最高等級。標準：純芽——T9系列；一芽一葉——高品質茶；一芽二葉——大宗生產。
* **原料要求：** 嫩度均勻的原料，無粗老、受損或病變葉片。清潔度通過每年481項歐盟標準農藥殘留檢測確認。

**4. 風土與栽培特點：**

* **氣候與地形：** 茶園位於郎岱鎮 (郎岱镇, Lángdài Zhèn) 和牂牁鎮 (牂牁镇, Zāngkē Zhèn) 交界處，典型的河谷地帶，受牂牁江 (牂牁江) 水體影響。年平均氣溫14°C，年降雨量超過1,200毫米。霧日超過180天，散射光佔總光照的70%以上，日夜溫差超過10°C。年平均日照時數僅1,107小時，遠低於東部產茶省份——正是這種「寡日照」(寡日照, guǎ rìzhào) 因子促進氨基酸積累，形成溫和甘甜的口感。
* **海拔高度：** 1,000-1,500公尺。
* **土壤：** 微酸性黃壤 (微酸性黄壤, wēi suānxìng huáng rǎng)，pH值5.0-6.5，有機質含量≥2%。土壤富含硒、鋅等微量元素。森林覆蓋率78.8%，無工業污染。水源由牂牁江提供，水質符合國家一級標準。
* **栽培特點：** 高海拔位置減緩茶樹生長，有利於嫩芽營養物質積累。據資料顯示，春茶原料氨基酸含量為3.5-5.0%（比低海拔同類產品高20%），多酚含量超過40%，形成特有的「高山冷韻」(高山冷韵, gāoshān lěngyùn)。茶園按生態標準管理：病蟲害防治僅使用物理方法（燈光誘捕），完全禁用化學農藥。

**5. 製作工藝：**

九層山綠茶的生產工藝包括綠茶製作的經典步驟，重要特色是雙重成型路徑，決定最終產品類型：扁形或捲曲形。

1. **攤青 (摊青, tānqīng)：** 鮮葉攤放在竹篩上4-10小時，失水約25%。此階段啟動香氣物質的初步發展。
2. **殺青 (杀青, shāqīng)：** 在滾筒機中於180-240°C處理。滾筒快速旋轉確保均勻受熱和「鎖翠」(锁翠, suǒcuì)。
3. **揉捻 (揉捻, róuniǎn)：** 持續25-30分鐘；過程中釋放果膠物質，賦予茶葉緊實度和「體感」。
4. **做形 (做形, zuòxíng)：** 此階段工藝分為兩路：
   * *扁形* (扁形, biǎnxíng)：手工或機械壓製成「翠芽」(翠芽) 風格，形成扁平直條，類似龍井；
   * *捲曲形* (卷曲形, juǎnqū xíng)：傳統手工成型，被認定為非物質文化遺產，形成緊實捲曲條索。
5. **乾燥 (干燥, gānzào)：** 分2-3階段進行，直至最終含水量≤7.0%。

製作特色——全程生態清潔：從茶園到包裝僅採用物理防護方法，產品每年通過歐盟標準481項農藥殘留檢測。

**6. 感官特徵：**

* **乾茶外形：** 依形狀而異：扁形茶（特別是「明前翠芽」系列，明前翠芽）——扁平筆直、整齊的翠綠色茶條；捲曲茶——緊實沉重的捲曲條索。兩種類型均勻整潔。
* **乾茶香氣：** 扁形茶具嫩香 (嫩香, nèn xiāng)；捲曲茶具栗香 (栗香, lì xiāng)。春茶樣品帶有淡雅花香。
* **茶湯香氣：** 持久清純：高等級茶具嫩鮮感，捲曲葉茶具深沉栗香調。冷杯留香 (冷杯留香, lěng bēi liú xiāng) 持續10分鐘以上。
* **滋味：** 鮮爽 (鲜爽, xiānshuǎng)、醇厚 (醇厚, chúnhòu)。氨基酸和諧提供明顯甜味，多酚帶來輕微澀感，迅速轉為甘甜回味 (回甘, huígān)。茶多糖創造飽滿口感和柔和包覆感。
* **湯色：** 高等級茶呈嫩綠明亮 (嫩绿明亮, nèn lǜ míngliàng)；大宗茶呈黃綠透明。
* **葉底（沖泡後葉片）：** 嫩黃綠色葉片，整齊有彈性，芽頭厚實多肉。越冬芽 (越冬芽, yuèdōng yá) 葉片可能有輕微紅邊——特有的自然特徵。

**7. 化學成分：**

* **茶多酚 (茶多酚, chá duōfēn)：** ≥14.5%（某些資料顯示季節高峰可達40%）。主要為兒茶素，提供抗氧化活性。明顯比平地綠茶高15%。
* **氨基酸 (氨基酸, ānjīsuān)：** 3.5-5.0%，包括L-茶氨酸。高含量是高海拔和寡日照的結果。
* **水浸出物 (水浸出物, shuǐ jìnchū wù)：** ≥42%，表明極其豐富的風味特徵和高萃取性。
* **生物鹼：** 咖啡因、可可鹼、茶鹼——綠茶標準濃度。
* **硒 (硒, xī)：** 有機硒含量顯著——該地區土壤的獨特地球化學特徵。
* **維生素：** C（鮮葉中）、B群、K。
* **礦物質：** 鋅、鉀、錳等微量元素。
* **精油：** 形成栗香花香特徵，因高山緩慢生長而特別突出。

**8. 保健功效：**

* **高抗氧化活性：** 多酚複合物配合有機硒，提供增強的自由基中和效果——據資料顯示比綠茶平均水準高20%。
* **支持脂質代謝：** 兒茶素（特別是EGCG）抑制脂肪吸收，高水浸出物含量（≥42%）在規律飲用時增強代謝效果。
* **心血管保護：** 有機硒與多酚結合有助於心血管系統保護和血管彈性維持。
* **溫和提神效果：** 咖啡因配合L-茶氨酸提供平穩提神，無急劇波動——清晰專注狀態。
* **消化支持：** 多酚和果膠物質對胃腸道功能有益。
* **免疫增強：** 維生素、礦物質和生物活性多酚複合物支持機體防禦功能。

**9. 沖泡方法：**

* **水溫：** 80-85°C（T9系列高等級茶75°C）。建議將水煮沸後冷卻約2分鐘。
* **茶量：** 3克配150毫升水（1:50比例）。
* **茶具：** 玻璃杯 (玻璃杯, bōli bēi)——觀察茶形和葉片展開；白瓷蓋碗 (白瓷盖碗, báicí gàiwǎn)——聚集香氣。
* **沖泡過程：**
  1. 用熱水溫潤茶具。
  2. 推薦「上投法」(上投法, shàngtóu fǎ)：先注水入杯，再輕放茶葉於水面。
  3. 第一泡：浸泡30秒。
  4. 倒出茶湯或開始品飲。
  5. 後續沖泡：每次增加10-15秒浸泡時間。
  6. 可沖泡3-4次。

**10. 儲存：**

* 密封包裝，完全避光、防潮、防異味。
* 最佳溫度：0-5°C（冰箱）。開封前需在室溫下密封靜置，防止凝露。
* 保質期——適當儲存條件下24個月。開封後建議儘快飲用以保持香氣和新鮮度。
* 茶葉天敵：潮濕、直射光、異味、高溫。

需注意，高多酚含量的綠茶不建議空腹飲用，因單寧物質可能刺激胃黏膜。最佳時間為餐後一小時。建議每日飲用量不超過600毫升。服用藥物時（特別是鐵劑和抗生素）應與茶飲間隔至少一小時。

**11. 價格與仿冒：**

* **價格範圍：** T9系列特級 (特级, tèjí) 純芽茶——超過1,000元人民幣/斤（500克），零售400-600元/100克。高品質「一芽一葉」綠茶——50-100元/斤（這是貴州高山茶中最實惠的之一，因大規模生產）。大宗茶——顯著更便宜。
* **價格因素：** 採摘季節（早春顯著更貴）、原料等級、形狀（扁形比捲曲形貴）、歐洲認證。
* **避免仿冒：**
  * 購買有地理標誌確認的茶葉——標示「九層山茶」及相應標誌。
  * 注意葉片清潔度和均勻性：扁形茶應平整翠綠，捲曲茶應緊實沉重。
  * 評估香氣：持久清純，無霉味、異味化學氣息。
  * 檢查茶湯：應透明，鮮綠色調。
  * 索取品質證書——品牌產品通過歐盟標準農藥殘留檢測。

**12. 有趣事實：**

* 「九層山」之名詩意地描述當地景觀：站在高處可見約一百座山峰，層層遞進如巨大階梯。
* 半坡村 (半坡村) 集中了一群千年古茶樹，包括被專家認定為「貴州最具推介價值古茶樹」(贵州最具推介价值古茶树) 的樣本。
* 九層山茶是少數系統性通過歐盟標準檢測（481項農藥殘留指標）的中國綠茶之一，使其成為貴州茶品牌體系中「乾淨茶」(干净茶) 概念的典範。
* 六枝特區是中國唯一保持「特區」(特区) 地位的行政區，承襲自1960年代煤礦開發時期。從煤炭到茶葉的轉變是區域經濟轉型的象徵和現實。
* 「九層山茶」品牌價值在2024年達到256.7億元人民幣——對於幾十年前在貴州以外幾乎無人知曉的區域茶而言，這是令人印象深刻的成就。

**13. 與其他綠茶比較：**

* **都勻毛尖 (都匀毛尖, Dūyún Máojiān)：** 貴州最著名的綠茶，列入中國十大名茶。產於黔南布依族苗族自治州，使用當地小葉種。與九層山綠茶相比，都勻毛尖香氣更細膩，針形更精細，鮮味特徵更突出。九層山茶更厚重、濃郁，「體感」更強。
* **湄潭翠芽 (湄潭翠芽, Méitán Cuìyá)：** 另一款主要的貴州扁形綠茶，產於湄潭縣。外形和風格最接近九層山綠茶扁形，但生長海拔較低（600-900公尺），口感更溫和但缺少「冷韻」。
* **西湖龍井 (西湖龙井, Xīhú Lóngjǐng)：** 浙江經典扁形綠茶。九層山綠茶扁形在視覺上類似龍井，但栗香更突出，體感更厚重，具有平地龍井所沒有的特有「高山冷韻」。
* **水城春茶 (水城春茶, Shuǐchéng Chūnchá)：** 同屬六盤水市鄰近水城區的綠茶。風土和海拔特徵相似，但九層山茶生態控制更嚴格，古茶樹群體遺傳基礎更豐富。
* **鳳岡富鋅富硒茶 (凤冈富锌富硒茶, Fènggāng Fùxīn Fùxī Chá)：** 貴州鳳岡縣綠茶，同樣富含鋅硒並有地理標誌地位。與九層山茶相比，鳳岡茶海拔稍低，口感更細膩溫和，而九層山茶在體感厚重度和栗香深度上更勝一籌。

**結論：**

九層山綠茶是貴州新一代高山綠茶的傑出代表，結合了千年自然資源與現代生態標準。「低緯度、高海拔、寡日照」——這一風土因子三合一賦予茶葉罕見的厚重飽滿體感與溫和氨基酸甜味組合，而毫不妥協的清潔生產控制使其成為貴州「乾淨茶」的典範。這款茶值得那些欣賞濃郁、有體感的綠茶，具有明顯礦物特徵，並希望探索經典茶類之外中國西南茶葉財富的茶友品嚐。
