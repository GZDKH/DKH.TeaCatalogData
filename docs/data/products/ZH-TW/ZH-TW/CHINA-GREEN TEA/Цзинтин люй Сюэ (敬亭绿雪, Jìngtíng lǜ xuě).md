# 敬亭綠雪 (敬亭绿雪, Jìngtíng Lǜ Xuě)

**敬亭綠雪 (敬亭绿雪, Jìngtíng Lǜ Xuě)**
![Цзинтин Люй Сюэ (敬亭绿雪, Jìngtíng Lǜ Xuě)](images/Цзинтин%20Люй%20Сюэ%20(敬亭绿雪,%20Jìngtíng%20Lǜ%20Xuě).png)

![Цзинтин Люй Сюэ (敬亭绿雪, Jìngtíng Lǜ Xuě)-1](images/Цзинтин%20Люй%20Сюэ%20(敬亭绿雪,%20Jìngtíng%20Lǜ%20Xuě)-1.png)

敬亭綠雪是中國最古老的歷史綠茶之一，創製於明代，並在明清兩朝獲得貢茶地位。這款茶的製作技術在清末失傳，直到1978年才得以恢復，與黃山毛峰 (黄山毛峰) 和六安瓜片 (六安瓜片) 並列為安徽三大名茶。其詩意的名稱——「敬亭山上的綠色雪花」——源於茶葉上的白毫 (白毫)，沖泡時在茶杯中旋轉如雪花飛舞。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵），烘青 (烘青, hōngqīng) — 採用烘焙乾燥法（烘籠乾燥）。
* **類別：** 中國歷史名茶 (历史名茶, lìshǐ míngchá)。
* **產地：** 中國安徽省 (安徽, Ānhuī) 宣城市 (宣城, Xuānchéng) 宣州區 (宣州区, Xuānzhōu Qū) 敬亭山 (敬亭山, Jìngtíng Shān)。核心產區集中在主峰一峰 (一峰, Yī Fēng) 及周邊茶園——茗香台 (茗香台)、上十壩 (上十坝) 等陰坡地段，海拔300–500公尺。
* **地理座標：** 約北緯31°00′，東經118°40′。

**2. 歷史與文化意義：**

* **歷史：** 敬亭綠雪創製於明朝（1368–1644）。明清時期被列入皇室貢茶名錄。據《宣城縣志》(《宣城县志》, Xuānchéng Xiànzhì) 記載，明清時期每年向朝廷進貢此茶300斤（約150公斤）。康熙年間詩人學者施閏章 (施闰章, Shī Rùnzhāng) 曾以詩詞讚頌此茶，奠定了其文學聲譽。清末製作技術失傳。1972年敬亭山茶場 (敬亭山茶场) 開始復原工作，1978年成功恢復製作技術。1976年著名文學家社會活動家郭沫若 (郭沫若, Guō Mòruò) 親筆題寫「敬亭綠雪」，成為此茶的標誌。1983年獲得中華人民共和國對外經濟貿易部榮譽證書。2010年敬亭綠雪製作技藝被列入安徽省非物質文化遺產名錄。此茶的採製方法已納入高等院校茶學專業正式教材。

* **名稱：** 名稱的每個組成部分都具有深刻含義。「敬亭」(敬亭) — 山名；原名昭亭山 (昭亭山, Zhāotíng Shān)，西晉初年（西元266年）為避司馬昭 (司马昭) 名諱而改名。「綠」(绿, lǜ) — 「綠色」，指茶芽茶葉的顏色。「雪」(雪, xuě) — 「雪花」，形容茶葉上豐富的白毫，沖泡時脫落在水中旋轉，營造雪花飛舞的效果。民間還有一個傳說：綠雪是一位製茶姑娘的名字，她製作的茶葉非常出色。為躲避當地官員的騷擾，她跳崖自盡，籃中散落的茶葉在整個山坡上生根發芽。人們以她的名字命名此茶以紀念她。

* **文化意義：** 敬亭山是中國南方最著名的「詩山」(诗山, Shī Shān) 之一。南齊詩人謝朓 (谢朓, Xiè Tiǎo, 464–499) 使其聞名，偉大的李白 (李白, Lǐ Bái) 七次登臨此山，留下45首詩作，包括不朽的《獨坐敬亭山》(《独坐敬亭山》)。一千多年來——從六朝到清代——超過300位文人墨客讚頌此山，包括白居易、杜牧、歐陽修、黃庭堅、蘇軾和文天祥。誕生於這座傳奇山峰斜坡上的茶葉，從一開始就與文學傳統密不可分。清代詩人施閏章寫道：「倒入白瓷中難以分辨[與清水]，如花香被山溪帶走」。畫家梅庚 (梅庚, Méi Gēng) 也讚美此茶，指出其「色如秋水，味似蘭花」。敬亭綠雪與黃山毛峰、六安瓜片構成經典的「安徽三大名茶」(安徽三大名茶)。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種為宣城尖葉 (宣城尖叶, Xuānchéng Jiānyè) — 國家級有性系良種 (国家级有性系良种)。這是灌木型 (*Camellia sinensis* var. *sinensis*) 中葉種，植株半開張。嫩梢黃綠色，茸毛密度高。「一芽三葉」標準的百芽重約64克。該栽培種具有較強的抗寒抗旱能力，以及持續的嫩度保持性 (持嫩性强, chí nèn xìng qiáng)。
* **採摘：** 每年清明 (清明, Qīngmíng, 四月初) 至穀雨 (谷雨, Gǔyǔ, 四月中下旬) 期間採摘。採摘標準為「一葉抱一芯」(一叶抱一芯)：未展開或剛展開的芽頭配一片嫩葉，長度約一寸 (寸, ~3.3公分)。
* **採摘標準：** 特級 (特级) — 純芽頭或芽頭配第一片未展開葉片；一級 — 一芽一葉；二級 — 一芽二初展葉。採摘要求用四個字概括：「嫩、均、淨、齊」(嫩、均、净、齐) — 嫩度、均勻度、潔淨度、整齊度。
* **原料要求：** 僅採用嫩綠、無損傷的芽葉，不含粗老葉片，大小均勻，無異味和雜質。

**4. 風土條件與栽培特點：**

* **氣候與地形：** 敬亭山位於皖南山區 (皖南山区) 與長江沿岸平原的過渡地帶。氣候為亞熱帶季風氣候，四季分明。年平均氣溫15–16.8°C。年降水量1500–2000毫米。無霜期超過220天。年平均濕度超過80%。敬亭山森林覆蓋率達96.3%，使茶樹主要接受散射光照，有利於氨基酸積累並抑制粗纖維形成。
* **海拔高度：** 主要茶園位於海拔300–500公尺。最優質的原料採自主峰附近的陰坡（北坡）。
* **土壤：** 土壤由砂岩風化形成。呈弱酸性反應（pH 4.5–5.0），腐殖質和礦物元素含量豐富，包括硒和碘。這種土壤組成為茶葉提供豐富的礦物質特徵，有助於形成獨特香氣。

**5. 製作工藝：**

敬亭綠雪的製作包括六個連續步驟。關鍵特色是手工造型技法「搭攏理條」(搭拢理条, dā lǒng lǐ tiáo)，形成特徵性的「雀舌形」，以及最終的槐炭烘籠 (槐炭烘笼, huáitàn hōnglóng) 乾燥，固定香氣並確保儲存穩定性。

1. **鮮葉攤放 (鲜叶摊放, xiānyè tānfàng)：** 採摘的原料攤放在竹盤上2–3小時，自然去除多餘水分並喚醒香氣。
2. **殺青 (杀青, shāqīng)：** 在鍋中進行，溫度130–140°C。每次投葉200–250克。前2分鐘大力抖動 (抖, dǒu)，然後交替抖動和短暫悶炒 (闷, mèn)。溫度控制至關重要：過熱產生焦味，不足則有青草味。殺青後攤涼。
3. **做形 (做形, zuòxíng)：** 在約60°C下進行。使用手工「搭攏理條」(搭拢理条) 技法：師傅同時用四指和拇指操作，將茶葉握在掌中形成光滑、筆直的「舌形」。壓力按「輕-重-輕」(轻-重-轻) 原則調節，防止芽頭變黑和破碎。
4. **毛烘 (毛烘, máo hōng)：** 從110°C開始，逐漸降溫 (梯度降温, tīdù jiàngwēn)。此階段固定外形並去除大部分剩餘水分。
5. **足烘 (足烘, zú hōng)：** 在60°C下用暗火慢烘 (暗火慢烘, ànhuǒ mànhōng) 至含水量≤5%。正是在此階段使用槐木炭烘籠，賦予茶葉特徵性的栗香調。
6. **分級 (分级, fēnjí)：** 成品茶按大小、完整度和茸毛量分為四個等級。

**6. 感官特徵：**

* **乾茶外形：** 茶葉呈「雀舌形」(雀舌形, quèshé xíng) — 筆直、緊實、略扁平，手感有彈性。色澤翠綠鮮活，白毫豐富，局部帶金黃色澤（特級, 特级）。茶葉不含碎片和茶末。高級茶葉緊結度特別好，光澤明亮。
* **乾茶香氣：** 主調為板栗香 (板栗香, bǎnlì xiāng)，伴有蘭花香 (兰花香, lánhuā xiāng)。根據產地微氣候不同，也可能呈現金銀花香 (金银花香, jīnyínhuā xiāng)。香氣清新、純淨、持久。
* **茶湯香氣：** 濃郁持久，栗香突出，花香層次豐富，綠茶清香為基調。茶杯放涼後展現蜜香和堅果調。
* **滋味：** 鮮爽 (鲜爽, xiānshuǎng)、醇和 (醇和, chúnhé)，甘甜明顯 (甘, gān)，回甘 (回甘, huígān) 悠長。湯質中等厚度，正確沖泡無苦澀味。特級茶額外呈現鮮豆味和奶香調。
* **湯色：** 嫩綠明亮 (嫩绿明亮, nèn lǜ míng liàng)，透明清澈。在玻璃杯中沖泡可清楚看到白毫從茶葉上脫落並在水中旋轉，形成著名的「綠雪」效果。
* **葉底（沖泡後茶葉）：** 嫩綠、有彈性、色澤鮮亮的葉片和芽頭，聚集成緊密的「花朵狀」(成朵状, chéng duǒ zhuàng)。色澤均勻和葉片完整證明原料品質和精心加工。

**7. 化學成分：**

由於富含腐殖質的土壤和最佳散射光條件，此茶富含生物活性物質。

* **多酚類（兒茶素）：** 茶多酚含量31.1% — 顯著高於綠茶平均水準。主要兒茶素包括表沒食子兒茶素沒食子酸酯（EGCG）、表兒茶素沒食子酸酯（ECG）、表沒食子兒茶素（EGC）。兒茶素總含量14.7%。據評估，此茶多酚的抗氧化活性是維生素E的18倍。
* **氨基酸：** 游離氨基酸含量4.3%，為綠茶中的高含量。主要成分為L-茶氨酸 (L-茶氨酸, L-chá'ānjīsuān)，提供特徵性甜味、鮮味飽滿度，並與咖啡因產生協同效應實現「溫和提神」。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēi jiǎn) — 主要提神成分，與L-茶氨酸協同作用。還含有可可鹼和茶鹼。
* **維生素：** 維生素C（抗壞血酸） — 鮮葉中含量高，通過溫和烘焙乾燥部分保留。B族維生素（B₁、B₂）。維生素K。
* **礦物質：** 鉀、錳。氟含量在200–300 ppm範圍內，有助於強化牙釉質。土壤成分確保含有微量元素硒和碘。
* **精油：** 香氣特徵在殺青和最終炭火烘焙過程中形成。主要香氣化合物包括吡嗪類（栗香調）、芳樟醇和香葉醇（花香調）。

**8. 健康功效：**

* **抗氧化保護：** 高含量兒茶素（31.1%多酚）提供強大的自由基中和作用，支持細胞健康並延緩氧化衰老過程。
* **溫和提神效果：** 咖啡因與L-茶氨酸結合產生平靜專注狀態，無咖啡特有的能量峰谷現象。
* **支持認知功能：** L-茶氨酸刺激大腦α波產生，改善注意力集中、反應速度和思維清晰度。
* **支持消化：** 多酚和單寧物質刺激消化酶分泌，幫助消化油膩食物。傳統上此茶在餐後飲用。
* **心血管支持：** 兒茶素有助於降低氧化膽固醇（LDL）水準，支持血管壁彈性，規律飲用有助於血壓正常化。
* **強化牙釉質：** 200–300 ppm的氟含量結合兒茶素的抗菌作用（抑制致齲菌*變形鏈球菌*生長）有助於預防蛀牙。
* **支持新陳代謝：** 兒茶素和咖啡因共同激活產熱並刺激脂肪氧化，有助於維持健康體重。

**9. 沖泡方法：**

* **水溫：** 80°C（沸水冷卻3分鐘）。重要的是不要超過85°C — 過熱會破壞葉綠素，導致茶湯變黃和產生苦味。
* **茶量：** 3克配150毫升水（1:50比例）。功夫泡法 — 5克配120毫升。
* **茶具：** 玻璃杯（推薦初次品嚐 — 可觀察「飛雪」效果）；白瓷蓋碗 (盖碗, gàiwǎn) — 最適合評估香氣。不推薦宜興紫砂壺：多孔陶土可能掩蓋精緻的栗香。
* **沖泡過程：**
  1. 用熱水溫杯或蓋碗，然後倒掉。
  2. 將茶葉放入器皿底部（下投法, 下投法, xiàtóu fǎ）。
  3. 對於嫩綠茶，洗茶非必需。如需要可進行快速「喚醒」洗茶：倒入1/3水量，輕搖杯子，讓茶葉吸水10–15秒（搖香潤茶, 摇香润茶, yáo xiāng rùn chá），然後加滿水。
  4. 第一泡 — 1–2分鐘。倒出茶湯或開始品飲，杯中留三分之一茶湯。
  5. 第二、三泡 — 每次增加30秒。優質敬亭綠雪可沖泡3次，保持香氣和甜味。

**10. 儲存：**

敬亭綠雪如同大多數精緻綠茶，對氧化、濕氣和異味敏感。最佳條件是密封包裝（真空袋或密閉鐵罐）在0–5°C冰箱中保存。從冰箱取出包裝後需在室溫下放置1–2小時再開封 — 這可防止茶葉上凝結水分。開封後建議在2–4週內飲用完畢。密封包裝在適當溫度下的總保存期限可達12個月。值得注意的是，師傅建議製作後等待約兩週再進行首次品嚐：這段時間可消散「火氣」(火气, huǒqì)，使滋味更加柔和圓潤。

**11. 價格與仿冒品：**

敬亭綠雪是產量有限且具有明顯地域特色的茶葉，這決定了其相對較高的價格。特級茶 (特级) 價格從每斤（500克）1200元人民幣起；一級和二級茶價格相對親民。影響價格的主要因素：採摘時間（清明前原料更貴）、等級、手工比例和生產商聲譽。

**如何避免仿冒品：**

* **葉形：** 正宗敬亭綠雪具有特徵性「雀舌形」— 筆直、緊實、無變形的茶條，白毫豐富。仿冒品通常外觀鬆散，大小不均，有破碎片段。
* **香氣：** 天然栗香配花香層次是關鍵特徵。缺乏栗香、有「乾草」味或霉味表明品質低劣或用其他產區原料替代。
* **茶湯：** 透明、嫩綠，無渾濁。有「飛毫」是好兆頭。渾濁或深黃色茶湯表明原料陳舊或品質不佳。
* **價格：** 可疑的低價（標稱「特級」每斤少於300–400元）幾乎肯定是替代品。
* **來源：** 建議從可靠商家購買，要有透明的產地和採收日期資訊。自2010年起製作技藝受安徽省非物質文化遺產地位保護。

**12. 有趣事實：**

* 出產此茶的敬亭山被稱為「江南詩山」(江南诗山)。偉大詩人李白為其創作了45首詩歌，一生中七次登臨此山。他最後一次造訪是在761年，距離去世僅一年。
* 在透明玻璃杯中沖泡敬亭綠雪會產生獨特景象：白毫從茶葉上脫落並在水中旋轉，如綠色森林中的雪花飛舞。正是這種視覺效果賦予了此茶詩意的名稱。
* 敬亭山原名昭亭 (昭亭)，但在266年因避西晉開國皇帝司馬昭名諱而改名。這是中國地名學中最著名的歷史「避諱」(避讳, bìhuì) 例子之一。
* 「敬亭綠雪採製技藝」被納入全國高等院校茶學專業教學大綱 — 宣城地區唯一獲此殊榮的茶葉。
* 敬亭山森林覆蓋率達96.3%，創造了獨特的微氣候：豐富的散射光和穩定的高濕度使茶樹在多酚形成緩慢的同時積累氨基酸 — 這正是此茶特徵性甜味和嫩度的來源。

**13. 與安徽其他綠茶的比較：**

* **黃山毛峰 (黄山毛峰, Huángshān Máo Fēng)：** 產自安徽南部黃山山脈，海拔700–1200公尺。茶葉呈微捲曲形，帶金黃色「魚葉」(鱼叶金黄)。香氣為蘭花香，比敬亭綠雪的栗香更輕盈高雅。滋味以鮮爽清淨著稱，湯質密度較低。
* **六安瓜片 (六安瓜片, Liù'ān Guā Piàn)：** 獨特之處在於僅用葉片製作（無芽頭和葉梗）。外形扁平，似葵花籽。香氣為炒制堅果香，「火功」更明顯。滋味更加厚重濃郁。與敬亭綠雪不同，此茶無白毫視覺效果。
* **太平猴魁 (太平猴魁, Tàipíng Hóu Kuí)：** 產自太平縣（現黃山區）的大葉種綠茶。茶葉長大（達7公分）、扁平，有特徵性網紋。香氣為持久蘭香。在大小和形狀上與敬亭綠雪的迷你「雀舌」完全相反。
* **九華毛峰 (九华毛峰, Jiǔhuá Máo Fēng)：** 產自佛教聖山九華山附近。採摘標準和工藝與敬亭綠雪相近，但香氣更偏草本，栗香不明顯。文化聯想為佛教，與敬亭綠雪的「詩意」身份形成對比。

**14. 敬亭綠雪等級分類：**

* **特級 (特级, tèjí)：** 純芽頭或「芽頭+第一片未展葉」標準。外形為筆直、細嫩、完美的「雀舌」。金黃色茸毛覆蓋不少於80%表面。香氣為明亮栗香配蘭花層次。滋味最為鮮爽甘甜，回甘明顯。湯色嫩綠透明。價格每斤1200元起。
* **一級 (一级, yī jí)：** 「一芽一葉」標準。茶條翠綠帶毫。香氣純正持久，栗香明顯。滋味醇和諧調。
* **二級 (二级, èr jí)：** 「一芽二初展葉」標準。茸毛較少。香氣純淨但較內斂。滋味柔和，回甘強度較低。
* **三級 (三级, sān jí)：** 以成熟葉片為主。適合日常飲用和拼配。

**總結：**

敬亭綠雪是一款血統罕見的茶葉：誕生於中國南方最著名的「詩山」斜坡，從謝朓到郭沫若的文人墨客為其吟詠，曾經失傳又重獲新生。其栗香配蘭花層次、清甜滋味伴悠長回甘，以及在玻璃杯中令人著迷的「雪花飛舞」，創造出令人難忘的體驗。這款茶適合那些不僅追求味覺享受，更尋求文化深度的品茶者：每一杯敬亭綠雪背後都承載著一千五百年的詩歌、山川和工藝傳承。
