# 青城雪芽 (青城雪芽, Qīngchéng xuě yá)

**青城雪芽 (青城雪芽, Qīngchéng xuě yá)**

青城雪芽（青城雪芽，Qīngchéng xuě yá——「青城之雪芽」）——著名的四川綠茶，產於道教聖山青城山（青城山，Qīngchéng Shān）——「道教發源地」（道教發源地，dàojiào fāyuándì）之一，聯合國教科文組織世界遺產。其名詩意精準：芽葉密覆白毫，確實如山間嫩芽覆雪。此茶自唐代即負盛名：陸羽《茶經》載「青城縣有散茶、貢茶」（青城縣有散茶、貢茶），五代毛文錫（毛文錫）《茶譜》（茶譜，Chápǔ）描述其精緻形態——「雀舌」（雀舌，quèshé）與「麥顆」（麥顆，màikē）。宋代青城山設有**皇家茶場**（皇家茶場），35座道觀須向朝廷進貢茶葉。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵）。工藝為炒青烘青結合：「三炒三揉」（三炒三揉，sān chǎo sān róu）+ 竹籠炭烘（竹籠炭烘）。外形為直條形，秀麗微曲（直條形，秀麗微曲）。

* **類別：** 國家地理標誌產品（國家地理標誌產品）。1982年獲「四川省優質產品」（四川省優質產品）。歷史上為唐宋「貢茶」（貢茶）。

* **產地：** 中國四川省（四川，Sìchuān）都江堰市（都江堰市，Dūjiāngyàn Shì）青城山（青城山）。核心產區為海拔600-1200公尺的古茶園，位於楓坑（楓坑）、盤坑（盤坑）、石井坑（石井坑）一帶，以及丈人峰（丈人峰，「老人峰」）周邊——此處有原生茶樹群落。

* **地理座標：** 東經103°33′，北緯30°54′。

**2. 歷史與文化意義：**

* **歷史：** 青城山為「道教發源地」，西元二世紀張道陵（張道陵，Zhāng Dàolíng）在此創立天師道（天師道，Tiānshī Dào）。茶葉種植始見於唐代：陸羽《茶經》載青城縣產散茶與貢茶。五代（五代，十世紀）毛文錫《茶譜》詳述青城茶形態。

	宋代（960-1279）青城山設**皇家茶場**（皇家茶場）：「沙坪茶」（沙坪茶）列入貢品。據《永康軍貢目》（永康軍貢目，「永康軍貢品清單」），**35座道觀**須向朝廷進貢茶葉——此為中國歷史上最大規模的「制度化」貢茶機制之一。

	現代歷史：1958年國營灌縣茶場（國營灌縣茶場）依據歷史記載創製「青城雪芽」。1982年獲省級獎項。2010年代獲地理標誌保護。

* **名稱釋義：**

	* 「青城」（青城）——「青色之城」（或「翠綠堡壘」）：道教聖山之名。「青」指山巒終年蒼翠，植被如城牆環繞。
	* 「雪」（雪）——比喻芽葉上豐富的銀白色茸毫。
	* 「芽」（芽）——嫩芽、芽葉。

* **文化意義：** 青城雪芽具雙重靈氣：佛教茶與禪修相連，而青城雪芽是**著名綠茶中唯一與道教密不可分者**。青城山為「道教發源地」，道教主要流派發源處。此地之茶為道教「養生」（養生，yǎngshēng，「滋養生命」）實踐之一部，而非佛教禪修。此為根本不同的哲學框架：道教茶為與自然和諧、延年益壽，佛教茶為開悟證道。

**3. 植物學描述與原料：**

* **品種/栽培種：** **福鼎大白茶（福鼎大白茶）**——主要栽培種，茸毫豐富。**名山特早213（名山特早213）**——小葉種早生品種。另有**四川中葉群體種（四川中葉群體種）**。核心產區丈人峰周邊保存有**原生茶樹群落**（原生茶樹群落）。

* **採摘：** 嚴格於清明（清明）前。標準為一芽一葉初展（一芽一葉初展），長度≤2.5公分。製作500克特級茶需40,000-50,000個芽頭。

* **等級：** 分三級：
	* **特級（特級）：** 全芽或一芽一葉。茸毫覆蓋率≥90%。嫩栗香。每斤800元人民幣起。
	* **一級（一級）：** 一芽一葉。勻整微曲。滋味醇和。
	* **二級（二級）：** 含少量一芽二葉。香氣純正，回甘明顯。

**4. 風土條件與栽培特點：**

* **聖山環境：** 青城山為聯合國教科文組織世界遺產（2000年，與都江堰灌溉系統共同列入）。山巒植被茂密。

* **氣候：** 亞熱帶濕潤氣候。年均溫15.2°C，年降雨量1225.2毫米。**年均霧日>280天**——為世界茶區最高值之一。濕度≥80%。散射光充足。

* **海拔：** 600-1200公尺。核心區800-1200公尺。

* **土壤：** 酸性黃棕紫泥土（酸性黃棕紫泥），土層深厚，肥力高，富含有機質。森林覆蓋率**93%**。水資源為都江堰灌溉系統（都江堰）支流，水質達國家一級標準。

* **胺基酸含量：** 原料中含**484.29毫克/100克**（=4.84%）——高於綠茶平均值。成因：280+天霧日+散射光+酸性土壤。

**5. 製作工藝：**

青城雪芽工藝——「三炒三揉」（三炒三揉）+炭火烘乾——為綠茶中最複雜工序之一。

* **殺青（殺青——shāqīng）：** 140°C平鍋。「抖悶」（抖悶，dǒumèn）結合。

* **初揉（初揉——chūróu）：** 輕度成形。

* **二炒（二炒——èrchǎo）：** 80-100°C——發展香氣，進一步乾燥。

* **復揉（復揉——fùróu）：** 緊實條索。

* **三炒（三炒——sānchǎo）：** 最終定型。

* **攤涼（攤涼——tānliáng）：** 水分重新分布。

* **整形提毫（整形提毫——zhěngxíng tíháo）：** 溫度「先高後低」（鍋溫先高後低）。製茶師20小時（！）手工「提取」銀毫，形成特有「白毫似雪」（白毫似雪，báiháo sì xuě）。此為四川綠茶中最長的成形工序。

* **竹籠炭烘（竹籠炭烘——zhúlóng tànhōng）：** 竹籠木炭烘乾。

* **揀選復火（揀選復火——jiǎnxuǎn fùhuǒ）：** 最終分級與短暫烘焙固香。

**9. 沖泡方法：**

* **水溫：** 80-85°C。

* **茶量：** 3克配150毫升（比例1:50）。

* **茶具：** 玻璃杯——上投法（上投法）。

* **步驟：**
	1. 注水至七分滿，投茶，潤茶30秒。
	2. 加滿水。首泡30秒。
	3. 續泡每次增加10秒。可沖泡4次。

**10. 保存方法：**

* 密封、避光、陰涼處。冰箱0-5°C。
* 新購茶葉靜置15天「退火」。
* 保質期12個月。開封後1-2個月內飲用。

**12. 趣聞軼事：**

* 青城山——**「道教發源地」**，世界遺產，天師道創立處（西元二世紀）。青城雪芽為著名綠茶中唯一與道教而非佛教傳統密不可分者。

* 宋代**35座道觀**須向朝廷進貢茶葉——歷史上最大規模的「制度化」貢茶機制。

* **年均280+天霧日**——世界茶區最高值之一。青城山幾乎全年「不見天日」。

* 20小時手工「提毫」（提毫）——四川綠茶中最長成形工序。成就「白毫似雪」（「茸毫如雪」）。

* 都江堰灌溉系統（都江堰）位於青城山麓——世界現存最古老仍在使用的灌溉系統（西元前三世紀，共同世界遺產）。茶園用水源自此系統。

**13. 與其他四川綠茶比較：**

* **峨眉竹葉青（峨眉竹葉青）：** 產自佛教名山峨眉。扁平，翠綠。竹葉青較「輕盈」、「竹韻」；雪芽較「雪白」、栗香。

* **蒙頂甘露（蒙頂甘露）：** 產自蒙頂山。捲曲，蘭花甜潤。甘露較花香；雪芽較栗香清新，「雪毫」明顯。

* **竹葉青（竹葉青）vs. 雪芽（雪芽）：** 四川兩大名茶——佛山「竹葉青」與道山「雪芽」。佛教vs.道教，扁平vs.直條，峨眉vs.青城。

**結語：**

青城雪芽——道教聖山之茶。作為著名綠茶中唯一誕生於道觀而非佛寺者，它在「雪白茸毫」與栗香清新中蘊含著「養生」——道教與自然和諧、延年益壽之道。年均280天雲霧、宋代皇家茶場、35座道觀貢茶與丈人峰原生茶樹——這一切都凝聚在每一枚「雪芽」中，銀毫在翠綠底色上閃爍，如聖山初霜，彷彿當年張道陵得道成仙之處。

青城山的雲霧不僅孕育了獨特的茶葉，更承載著千年道教文化的精髓。每一杯青城雪芽，都是對這座聖山的致敬，對道教養生智慧的傳承，對人與自然和諧共生理念的詮釋。
