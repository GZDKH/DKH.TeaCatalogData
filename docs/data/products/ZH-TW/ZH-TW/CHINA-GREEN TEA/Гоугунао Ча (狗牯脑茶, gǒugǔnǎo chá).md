# 狗牯腦茶 (狗牯脑茶, gǒugǔnǎo chá)

**狗牯腦茶 (狗牯脑茶, gǒugǔnǎo chá)**
![Гоугунао Ча (狗牯脑茶, gǒugǔnǎo chá)](images/Гоугунао%20Ча%20(狗牯脑茶,%20Gǒugǔnǎo%20Chá).png)

![Гоугунао Ча (狗牯脑茶, gǒugǔnǎo chá)-1](images/Гоугунао%20Ча%20(狗牯脑茶,%20Gǒugǔnǎo%20Chá)-1.png)

狗牯腦茶是江西省珍貴的名茶之一，1915年巴拿馬太平洋國際博覽會金獎得主，也是地理標誌保護產品（GI）。這款擁有兩百多年歷史的綠茶，以其乾茶外形優雅、清純的蘭花栗香以及清爽持久的甘甜滋味而備受推崇。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá) — 不發酵茶，氧化程度低於5%。酵素固定透過加熱進行 (杀青, shāqīng)。
* **類別：** 中國區域性名綠茶；地理標誌保護產品 (国家地理标志产品, Guójiā Dìlǐ Biāozhì Chǎnpǐn)；列入中華人民共和國商務部「中華老字號」(中华老字号, Zhōnghuá Lǎo Zìhào) 名錄（2011年）。製作技藝入選江西省非物質文化遺產名錄（2008年）。
* **產地：** 中國江西省 (江西, Jiāngxī) 吉安市 (吉安, Jí'ān) 遂川縣 (遂川, Suìchuān)。核心產區為湯湖鎮 (汤湖镇, Tānghú Zhèn) 附近的狗牯腦山 (狗牯脑山, Gǒugǔnǎo Shān)。保護區涵蓋遂川縣全境；全縣茶園總面積約20萬畝（≈ 13,300公頃）。
* **地理座標：** ≈ 北緯26.3°，東經114.5°（以遂川縣城為準；主要茶園位於縣西南部山區，羅霄山脈腳下）。

**2. 歷史與文化意義：**

* **歷史：** 狗牯腦茶的起源可追溯至清朝 (清, Qīng)。嘉慶年間 (嘉庆, Jiāqìng)，約1796年，放排工梁為鎰 (梁为镒, Liáng Wéiyì) 因木排失事漂至南京，娶當地精通茶藝的楊氏 (杨氏) 為妻。夫婦倆將南京茶種帶回遂川，在狗牯腦山安家並開闢茶園，從此開創了當地的茶葉傳統。製茶技藝在梁家世代相傳，長期作為「家傳秘技」。

  1915年，當地茶商李玉山 (李玉山, Lǐ Yùshān) 用狗牯腦山茶葉製作了三個品種——「銀針」(银针, yínzhēn)、「雀舌」(雀舌, quèshé) 和「圓珠」(圆珠, yuánzhū)，各一公斤，送往舊金山巴拿馬太平洋國際博覽會 (巴拿马太平洋国际博览会)。茶葉榮獲金獎，被譽為「頂上綠茶」(顶上绿茶)。1930年，李玉山之孫李文龍 (李文龙, Lǐ Wénlóng) 以「玉山茶」(玉山茶，「翡翠山茶」) 之名參加浙贛聯合物產展覽會，榮獲一等獎。後來歷史名稱「狗牯腦茶」重新回歸。1964年成立遂川縣國營茶廠。1982年榮獲「江西省名茶」稱號；1988年獲第一屆全國食品博覽會金獎；1992年獲香港國際食品博覽會金獎。2004年國家質檢總局批准狗牯腦茶原產地保護。2010年入選江西館上海世博會展品，獲綠茶類金獎。2015年獲米蘭世博會金獎。

* **名稱：**
  * **狗 (gǒu)** — 狗；
  * **牯 (gǔ)** — 公牛（方言，亦指「石丘」）；
  * **腦 (nǎo)** — 頭，頂部。
  此山形似狗頭，故名狗牯腦 (狗牯脑)。茶以產地命名。曾一度稱為「狗牯腦石山茶」(狗牯脑石山茶，「狗牯腦山石山茶」)。

* **文化意義：** 遂川縣有「三寶」(三宝) 之說：金桔 (金桔, jīnjú)、板鴨 (板鸭, bǎnyā) 和狗牯腦茶。茶葉是該縣的名片，是當地經濟和鄉村振興的重要元素：茶園遍布羅霄山脈各個山坡，銷售既有傳統管道，也透過電子商務進行。早在北宋時期，詩人蘇軾 (苏轼, Sū Shì) 路過遂川時，曾在詩中讚美當地茶葉，留下「石鼎煮石泉」的詩句。

**3. 植物學描述與原料：**

* **品種/栽培種：** 最高品質的基礎是當地群體種狗牯腦 (狗牯脑本地群体种, Gǒugǔnǎo běndì qúntǐ zhǒng) — *Camellia sinensis* var. *sinensis*，灌木型 (灌木型, guànmù xíng)，中葉類 (中叶类, zhōngyè lèi)，中生種 (中生种, zhōngshēng zhǒng)。茶樹特點是樹冠開張，分枝密集，幼芽多毛，抗寒性好。芽葉呈淺綠色，葉片橢圓形，深綠色，厚實緊密。從芽萌發到適採期為18-21天，確保香味物質的緩慢積累。
  除當地群體種外，縣內還栽培：福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbái Chá) — 芽呈淺綠色，葉片較薄，展開較快（12-15天），茶葉品質略遜於原生品種；以及早生種烏牛早 (乌牛早, Wūniú Zǎo) 和白毫早 (白毫早, Báiháo Zǎo) — 葉形鬆散，香氣較淡；不被視為狗牯腦茶高等級的傳統原料。
* **採摘：** 4月初開始。精品批次嚴格手工採摘。規則：不採露水茶，不採雨水茶，不採正午烈日茶。採摘後仔細挑選，剔除紫芽 (紫芽, zǐyá)、單片和魚葉 (鱼叶, yúyè)。
* **採摘標準：** 高等級 — 清明 (清明, Qīngmíng) 前單芽 (单芽, dānyá)；中等級 — 一芽一葉 (一芽一叶, yī yá yī yè)；大宗級 — 一芽二葉 (一芽二叶, yī yá èr yè)。每批次必須在嫩度、潔淨度和新鮮度上保持一致。

**4. 風土條件與栽培特色：**

* **地形與氣候：** 狗牯腦山海拔900多公尺，位於羅霄山脈 (罗霄山脉, Luóxiāo Shānmài) 南麓。南有五指峰 (五指峰, Wǔzhǐ Fēng)，北有老虎岩 (老虎岩, Lǎohǔ Yán)。氣候屬亞熱帶季風氣候，濕潤溫和。年平均氣溫約18.5°C（縣氣象站資料為19.1°C）；無霜期超過250天（平均約350天）；年平均降雨量約1525毫米。山區終年雲霧繚繞；日照時間短，散射光充足，有利於葉片中胺基酸、咖啡鹼和香氣物質的積累。
* **生長海拔：** 主要茶園位於海拔400-1000公尺。核心區為狗牯腦山坡（≈ 900公尺），茶園集中在山腰地帶。
* **土壤：** 微酸性（pH ≈ 5.2）砂質壤土 (麻沙泥土, máshā nítǔ)，由花崗岩風化形成；富含有機質。山腳有溫泉 — 獨特的風土要素，創造穩定的微氣候。
* **生態環境：** 遂川縣森林覆蓋率達78-79%。負氧離子濃度超過每立方公分5,600個。縣境內有著名的「千年鳥道」(千年鸟道, Qiānnián Niǎodào) — 世界級鳥類遷徙路線。無工業污染，高森林覆蓋率為高山有機茶業創造了條件。

**5. 製作工藝：**

狗牯腦茶採用獨特的「兩次殺青、兩次揉捻」(两次杀青、两次揉捻, liǎng cì shāqīng, liǎng cì róuniǎn) 技術，確保香氣的純淨和茶湯的細膩。高等級全程手工製作。

* **採摘 (采摘 — cǎizhāi)：** 清晨，嚴格按等級標準採摘。採摘的芽葉立即送至工廠。
* **攤青 (摊青 — tānqīng)：** 鮮葉均勻攤放在通風室內，平衡水分，初步降低「青草氣」。時間長短視天氣和原料含水量而定。
* **第一次殺青 (杀青 — shāqīng)：** 在柴火或炭火鍋中炒制。高溫使氧化酶失活，固定綠色和鮮香。關鍵在於精確控溫：過熱產生苦味和「焦味」，不足則留有「生青味」。
* **初揉 (初揉 — chūróu)：** 適度手工揉捻，使細胞汁液初步滲出至葉表。
* **第二次殺青 (杀二青 — shā èr qīng)：** 在稍低溫度下再次炒制 — 深化香氣，使葉片達到適宜的可塑性。
* **復揉 (复揉 — fùróu)：** 更充分的揉捻，形成最終外形，增強茶湯「厚度」。
* **整形提毫 (整形提毫 — zhěngxíng tíháo)：** 精細的手工操作：使茶葉呈現特有的「眉形」或微捲曲狀，同時顯露表面的銀毫 (白毫, báiháo)。
* **足乾 (足干 — zúgān)：** 緩慢乾燥至含水量 ≤ 6%，確保貯存穩定性。傳統上使用木炭或混合柴火，避免含樹脂木材產生異味。
* **分級包裝：** 剔除粗老片段，控制批次均勻性；包裝和貯存採用有別於標準的獨特方法 — 當地傳統的一部分。

**6. 感官特徵：**

* **乾茶外形：** 形狀為「眉形」(眉形, méixíng)，微捲曲；條索緊結秀麗 (紧结秀丽, jǐnjié xiùlì)。色澤深綠帶黛綠 (黛绿, dàilǜ) 色調；表面覆蓋細嫩銀毫 (白毫, báiháo)，使茶葉呈現特有光澤。
* **乾茶香氣：** 清高 (清高, qīnggāo)。以蘭花香 (兰花香, lánhuā xiāng) 和栗香 (栗香, lìxiāng) 為主；背景有淡雅的果香和木質調。
* **茶湯香氣：** 優雅清新；蘭花香更加豐富，伴有淡雅的花草香韻。
* **滋味：** 鮮爽 (鲜爽, xiānshuǎng)，甘甜 (甘, gān)，醇厚 (醇厚, chúnhòu)。水溫適當時幾乎無苦澀味。回甘 (回甘, huígān) 明顯 — 持久清純，帶有清涼感。
* **湯色：** 透明，從淺金黃到橙黃明亮 (橙亮清碧, chéng liàng qīng bì)。表面無泡沫；茶渣快速沉底。
* **葉底（沖泡後茶葉）：** 芽葉嫩整，鮮綠明亮，富有彈性；等級均勻證明採摘品質。

**7. 化學成分：**

狗牯腦茶的胺基酸與多酚類比例約為1:5，這在同類綠茶中較為突出，決定了其溫和甘甜的風味特徵。

* **茶多酚 (茶多酚, chá duōfēn)：** 春茶含量約28%；主要成分為兒茶素，包括表沒食子兒茶素沒食子酸酯（EGCG），提供抗氧化潛力。
* **胺基酸 (氨基酸, ānjīsuān)：** 含量約3.8%，包括L-茶胺酸 — 負責滋味甘甜 (甘, gān) 和與咖啡鹼協同「提神不焦慮」效果的關鍵成分。
* **咖啡鹼 (咖啡碱, kāfēi jiǎn)：** 含量適中；與茶胺酸結合提供平穩刺激，無急劇起伏。
* **維生素：** C、B₁、B₂、E、K；維生素C含量高於一般綠茶，得益於高山緩慢生長。
* **礦物元素：** 特色是微量元素含量較高。據江西省產品品質監督檢驗中心資料，狗牯腦茶含硒（Se）約0.2毫克/公斤，鋅（Zn）約54毫克/公斤，顯著超過多數同類綠茶。鈣（Ca）和鎂（Mg）含量也較高。
* **精油和香氣化合物：** 由於高山散射光環境和顯著的晝夜溫差，茶葉積累較多揮發性香氣物質，形成特有的蘭花栗香風味。

**8. 保健功效：**

* **提神醒腦 (提神醒脑, tíshén xǐngnǎo)：** 咖啡鹼與L-茶胺酸結合，提供溫和持久的清醒感，改善專注力，不引起焦慮。
* **抗氧化保護：** 高含量兒茶素（特別是EGCG）中和自由基，減緩細胞氧化壓力。
* **助消化 (消食去腻, xiāoshí qùnì)：** 傳統上餐後飲用，幫助消化油膩食物，消除飽脹感。
* **心血管支持：** 多酚類有助於強化微血管壁，可能有助於調節膽固醇和血壓。
* **認知功能：** L-茶胺酸刺激大腦α波產生，促進放鬆同時提高注意力。
* **美容抗衰老：** 抗氧化多酚維持肌膚彈性，延緩光老化過程。
* **微量元素支持：** 較高的硒和鋅含量 — 對免疫功能和抗氧化酶系統的重要元素。

*注意：* 狗牯腦茶是食品，非藥品。胃敏感者建議避免空腹飲用；睡眠障礙者應限制下午飲用。茶中單寧可能降低食物中鐵的吸收，不建議用餐時直接飲茶。

**9. 沖泡方法：**

* **水溫：** 80-90°C（沸水放涼2-3分鐘）。沸水禁用 — 會增加苦澀味，破壞細膩香氣。
* **用茶量：** 3公克配150毫升（玻璃杯或蓋碗）。
* **茶具：** 透明玻璃杯 (玻璃杯, bōlí bēi) — 欣賞「茶舞」；瓷蓋碗 (盖碗, gàiwǎn) — 更好地展現香氣。小瓷壺也可。

* **沖泡過程（玻璃杯/蓋碗）：**
  1. 熱水溫杯；倒掉。
  2. 投茶。讓茶葉在溫熱密閉容器中「醒茶」30秒 — 聞乾香。
  3. 第一泡：注入80-85°C水。浸泡30-60秒。
  4. 品飲：茶湯應呈金黃透明。
  5. 續泡：可沖3-5次；每次增加30秒。

* **功夫茶沖泡：**
  1. 溫蓋碗。
  2. 投茶：4-5公克配100-120毫升。
  3. 洗茶：通常不需要；如需要 — 快速2-3秒。
  4. 第一泡：80-85°C水，10-15秒。
  5. 續泡：可沖6-8次，逐漸延長時間。

* **建議：** 如茶湯苦澀 — 降低水溫和用茶量，而非「責怪茶葉」。嫩芽特級 (特级) 春茶最適宜80°C。

**10. 貯存：**

* 密封、避光包裝 — 防止光線、濕氣、氧氣和異味。
* 最佳溫度0-5°C（冰箱），需絕對密封。短期貯存可置於乾燥陰涼處（低於20°C）。
* 勿與香料、乾果等強香味食品同放。
* 開封後1-2個月內飲用以保最佳新鮮度。原包裝保質期12個月。
* 狗牯腦茶傳統包裝方法被視為非物質遺產的一部分：歷史上使用特殊包材和密封方式，有別於標準做法。

**11. 價格與仿冒：**

* **價格範圍：** 價格因等級和季節差異很大。早春特供特級 (特供特级, tègōng tèjí) 最昂貴。大宗等級 (壹级 / 统级, yījí / tǒngjí) 價格親民。
* **等級分類（由高到低）：**
  * **特供特級 (Tègōng Tèjí) — 「特供」：** 清明前單芽；外形完美均勻；蘭花香突出；最高價位。
  * **貢品特級 (Gòngpǐn Tèjí) — 「貢品特級」：** 穀雨 (谷雨, Gǔyǔ) 前單芽；條索緊實；栗香濃郁。
  * **珍品特級 (Zhēnpǐn Tèjí) — 「珍品特級」：** 立夏 (立夏, Lìxià) 前；一芽一葉初展；滋味鮮爽。
  * **特級 (Tèjí) — 「特級」：** 立夏前；一芽一葉展開；湯色透明。
  * **壹級/統級 (Yījí / Tǒngjí) — 「一級/統級」：** 清明至處暑 (处暑, Chǔshǔ)；一芽二葉；耐泡性強；價格親民。
* **常見仿冒：** 鄰縣或外省茶葉冒充「狗牯腦」；粗茶加毫；陳茶加香精「翻新」。
* **防偽方法：**
  * 向可靠商家購買，要求標明具體生產商和批次。
  * 評估葉片均勻性：正品條索緊結，等級一致，天然銀毫。
  * 檢查香氣：清純，無「香精」和「化學」味。
  * 評估茶湯：透明金黃，無混濁；茶渣快速沉底。
  * 價格過低需警惕仿冒。

**12. 趣聞軼事：**

* 狗牯腦茶是1915年巴拿馬太平洋博覽會獲金獎的少數中國綠茶之一，與信陽毛尖 (信阳毛尖) 等名茶並列。
* 狗牯腦茶傳統製作技藝作為梁氏家族秘傳，1943年由第五代傳人梁德梅 (梁德梅, Liáng Démèi) 「保護」，在包裝上蓋長印，標明「祖傳製造，貨真價實，請認標記」。
* 狗牯腦山腳下有湯湖溫泉 (汤湖温泉, Tānghú Wēnquán) — 茶園風土與地熱水共存的罕見案例。當地人相信溫泉蒸氣為茶樹「注入特殊能量」。
* 遂川縣位於中國綠茶生產「黃金緯度」最南端（約北緯25-26°），提供獨特的長生長期與高山涼爽的結合。
* 至2020年代，全縣茶園面積達約20萬畝（≈ 13,300公頃），年產量約3,500噸 — 從「深山珍寶」發展為完整的區域產業。

**13. 與江西省及華南其他綠茶比較：**

* **廬山雲霧 (庐山云雾, Lúshān Yúnwù)：** 江西最著名綠茶；產於省北部廬山。共同點是雲霧高山風土。廬山雲霧通常更「厚重」草本，而狗牯腦茶因栗香蘭韻更優雅甘甜。
* **井岡翠綠 (井冈翠绿, Jǐnggāng Cuìlǜ)：** 鄰近井岡山 (井冈山) 區綠茶；相似的羅霄山脈風土。風味較簡單草本；狗牯腦茶在香氣複雜度和回甘持久性上勝出。
* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南名綠茶。兩茶都以嫩芽多毫著稱；都獲1915年博覽會金獎。信陽毛尖常較澀「銳利」；狗牯腦茶更溫和甘甜，回甘更明顯。
* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北蒸青綠茶。工藝完全不同：蒸汽殺青vs炒青。恩施玉露有明亮「草本海味」（鮮味）；狗牯腦茶更花香栗香，炒青茶複雜香氣結構。
* **安吉白茶 (安吉白茶, Ānjí Báichá)：** 浙江綠茶，胺基酸含量異常高（5-10%）。安吉白茶極度溫和「湯鮮」甘甜；狗牯腦茶更有結構，栗香蘭韻突出。

**結語：**

狗牯腦茶有著奇妙的命運：從形似狗頭山上的家族秘方，到國際博覽會金獎和國家級珍寶。其主要優點 — 高雅的蘭花栗香、清澈金黃茶湯、持久溫和的回甘 — 只有在細心沖泡下才能完全展現：水溫不過熱，用茶不過量，浸泡不過久。狗牯腦茶適合欣賞清透香氣線條、細膩胺基酸甘甜和輕盈清新回味的茶友 — 帶著溫泉的礦物回響和羅霄山脈的山間清涼。
