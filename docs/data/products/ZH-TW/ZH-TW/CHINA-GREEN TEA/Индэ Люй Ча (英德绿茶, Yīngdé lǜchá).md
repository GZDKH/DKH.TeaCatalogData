# 英德綠茶 (英德绿茶, Yīngdé lǜchá)

**英德綠茶 (英德绿茶, Yīngdé lǜchá)**
![Индэ Люй Ча (英德绿茶, Yīngdé lǜchá)](images/Индэ%20Люй%20Ча%20(英德绿茶,%20Yīngdé%20Lǜchá).png)

![Индэ Люй Ча (英德绿茶, Yīngdé lǜchá)-1](images/Индэ%20Люй%20Ча%20(英德绿茶,%20Yīngdé%20Lǜchá)-1.png)

英德綠茶 (英德绿茶, Yīngdé lǜchá) — 廣東省最大的綠茶，產於該省北部的英德市。雖然英德以著名的紅茶英德紅茶 (英德红茶) 聞名於世，但其綠茶「兄弟」同樣令人矚目：採用雲南大葉種製作，具有極高的胺基酸含量（≥5% — 比平均水準高1.5倍）和獨特的「醫療」特性 — 1986年廣東科學家證實，英德茶中錳含量的增加與降低肝癌風險相關。到2025年，英德茶園面積超過141,000畝（9,400公頃），年產量11,000噸，總價值48億元人民幣：英德是廣東茶業無可爭議的領導者。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵）。屬於烘青綠茶 (烘青绿茶, hōngqīng lǜchá)。特點 — 採用大葉種原料 (大叶种, dàyè zhǒng) 製作，這使其有別於大多數著名綠茶使用小葉種品種。

* **類別：** 地理標誌產品。廣東省綠茶產業的旗艦。清朝歷史上的「貢茶」(贡茶)。

* **產地：** 中國廣東省 (广东, Guǎngdōng) 英德市 (英德市, Yīngdé Shì)。生產區域 — 整個英德市轄區。核心產區 — 沙堆鎮 (沙堆镇, Shāduī Zhèn) 和水邊鎮 (水边镇, Shuǐbiān Zhèn)，提供超過60%的總產量，其中沙堆是特級茶的主要來源 (特级)。

* **地理座標：** 約北緯24°10′，東經113°22′（北緯24度 — 「廣東茶葉帶」）。

**2. 歷史與文化意義：**

* **歷史：** 英德的茶葉種植可追溯至唐朝：陸羽 (陆羽) 在《茶經》(茶经) 中提到，韶州 (韶州，包括英德在內的古代地區名稱) 產茶且「其味極佳」(其味极佳)。英德是唐代韶州三個主要產茶縣之一。

	明朝時期，茶葉成為該地區重要的農產品。清朝時期 — 因「茶湯清亮、滋味鮮爽」(茶汤清亮、滋味鲜爽) 而被列入貢茶 (贡茶) 名單。

	現代歷史的轉捩點 — **1990年代**：「紅改綠」(红改绿，「從紅茶改為綠茶」) 計畫 — 大規模將茶園從紅茶（黑茶）生產轉換為綠茶。雲南大葉種 (云南大叶种)，原本專門用於紅茶生產，被改良用於生產「高香濃味綠茶」(高香浓味绿茶)。這一轉換將英德從廣東的「紅茶」之都變成了雙面巨人 — 既生產紅茶又生產綠茶。

	1986年，廣東省醫學院的科學家進行了研究，證實英德茶中**錳含量增加**與降低肝癌風險 (降低肝癌风险) 相關。這是中國首批將茶葉中特定微量元素與癌症預防聯繫起來的科學研究之一。

	2019年茶園面積達到141,000畝。到2025年，英德茶產業總價值達48億元人民幣。

* **名稱：**

	* 「英德」(英德) — 城市名稱。字面意思：「英國美德」— 歷史名稱，與英國無關，而是源於古代中國地名學。
	* 「綠茶」(绿茶) — 「綠茶」。

* **文化意義：** 英德是中國南方最大的茶葉生產城市之一，「廣東茶都」。英德綠茶體現了成功的「從紅到綠」轉型 — 這是世界茶業中罕見的整個地區從一種產品類型大規模轉換到另一種類型的例子。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要 — **雲南大葉種 (云南大叶种, Yúnnán Dàyè Zhǒng)** — 雲南大葉種 *Camellia sinensis* var. *assamica*。喬木型 (乔木型)，具有高抗寒性（在廣東條件下）。春季原料的化學成分：多酚類 ≥28.3%，胺基酸 ≥5% — **比標準小葉種綠茶高1.5倍**。正是大葉種原料提供了特有的「濃醇」(浓醇) 口感。雲南大葉種的種植在1990年代「紅改綠」計畫後開始佔主導地位。

* **採摘：** 四個季節：
	* **明前雀舌 (明前雀舌，「清明前雀舌」)：** 清明前採摘的完整芽頭。形狀 — 「雀舌」(雀舌)。茸毛豐富。特級 (特级)。
	* **雨前翠芽 (雨前翠芽，「穀雨前翠芽」)：** 穀雨前一芽一葉。一級 (一级)。
	* **夏暑茶 (夏暑茶，「夏季茶」)：** 一芽二葉。葉片厚實，耐泡。大宗茶。
	* **秋香茶 (秋香茶，「秋季香茶」)：** 白露 (白露，「白露」) 前。天然花香。高性價比。

* **原料要求：** 嫩度均勻的芽葉。當日採摘當日加工。

**4. 風土條件與栽培特點：**

* **位置：** 粵北 (粤北)，丘陵地帶。北緯24度。

* **氣候：** 南亞熱帶季風氣候 (南亚热带季风气候)。年平均氣溫 — 15.5–20.7°C，年降雨量 — 1,200–1,876毫米。年平均雲霧天數 — **>200天**。日夜溫差 — >10°C。散射光比例 — **70%** — 茶區中最高的指標之一。

* **土壤：** 喀斯特地貌 (喀斯特地貌) 酸性黃壤 (酸性黄壤)，pH 4.5–6.5。天然富含**硒和錳** (锰, měng)。森林覆蓋率 — 68.7%。無工業污染。

* **核心產區的獨特性：** 在沙堆和水邊鎮，原料中胺基酸含量達到**147.33毫克/克** — 極高的指標，解釋了顯著的鮮爽和回甘「鮮爽回甘」。多酚類和咖啡因的比例平衡，最大限度地減少了苦澀味。

**5. 生產工藝：**

英德綠茶是烘青綠茶 (烘青)，在殺青階段採用特有的「悶揚結合」(闷扬结合, mèn yáng jiéhé — 「悶炒和拋炒結合」) 技法。

* **採摘 (采摘 — cǎizhāi)：** 手工採摘「一芽一至三葉」標準的芽葉（根據季節和等級而定）。

* **輕萎凋 (轻萎凋 — qīng wěidiāo)：** 在竹篩上攤晾 (竹筛摊晾) 約4小時。含水量降至約70%。與標準攤放的區別 — 正是「萎凋」(萎凋)，而不僅僅是「攤放」(摊放)：原料經過受控的失水過程。

* **殺青 (杀青 — shāqīng)：** 在鐵鍋中於240°C進行。**關鍵技法 — 「悶揚結合」：** 師傅交替進行「悶炒」(闷, mèn — 葉片貼鍋底，在自身蒸汽中加熱，約2分鐘) 和「拋炒」(扬, yáng — 葉片大力拋起，釋放蒸汽，約6分鐘)。悶炒確保大葉片深度加熱；拋炒防止過熱並「鎖住翠綠色」(锁翠, suǒcuì)。這些技法的結合降低了大葉種原料的苦澀味 (降低多酚苦涩)。

* **揉捻 (揉捻 — róuniǎn)：** 按「輕 → 重 → 輕」(轻-重-轻) 原則，持續40分鐘。為特級茶塑造「雀舌形」(雀舌形)。碎茶率 (碎茶率) — <5%。

* **乾燥 (干燥 — gānzào)：** 兩階段：初烘至120°C至約70%乾度 (初烘至七成干)；複烘90°C → 60°C (复干)。

**6. 感官特徵：**

* **乾茶外形：** 特級 — 緊實細嫩的「雀舌形」(雀舌形, quèshé xíng)，翠綠色帶豐富的銀毫 (翠绿披毫)。大宗茶 — 捲曲形 (卷曲形)。葉片明顯比小葉種綠茶大 — 大葉種品種的結果。

* **乾茶香氣：** 清純、高揚、持久 (清香, qīngxiāng)。蘭花香 (兰花香) 和栗香 (栗香) 調性。空杯留香 — **>30分鐘** — 真品標誌。

* **茶湯香氣：** 高揚持久，具有栗香蘭花香特徵。

* **滋味：** 濃醇飽滿 (浓醇, nóngchún) — 大葉種原料多酚類含量高的結果。鮮爽 (鲜爽) — 胺基酸 ≥5%。回甘持久穩定 (回甘持久)。特色：**「微苦無澀」— 「輕微苦味無粗澀感」** — 獨特特徵，由多酚類和咖啡因的平衡比例所解釋。

* **湯色：** 黃綠明亮 (黄绿明亮)。

* **葉底：** 嫩勻成朵，綠黃鮮活 (嫩匀成朵，绿黄鲜活)。

**7. 化學成分：**

大葉種品種和富含錳的喀斯特土壤形成了獨特的成分特徵：

* **多酚類（兒茶素）：** 25–30% — 高於綠茶平均水準。EGCG — 主要成分。中和自由基的效力 — 比維生素E高10倍。

* **胺基酸：** **≥5% — 比綠茶平均水準高1.5倍**。在核心產區（沙堆）— 高達147.33毫克/克。正是胺基酸決定了顯著的鮮爽和「活潑」甜味。

* **錳 (锰, měng)：** **顯著高於**普通綠茶。根據廣東醫學院研究（1986年），錳含量增加與降低肝癌風險和增強自由基中和能力（比普通綠茶高30%）相關。

* **維生素C：** **比平均水準高1.5倍** — 溫和南方氣候和豐富散射光的結果。

* **生物鹼：** 咖啡因 — 適中含量，與多酚類平衡。

* **硒：** 天然高含量 — 來自喀斯特土壤。

* **礦物質：** 鉀、鎂、鋅、錳、硒。

**8. 健康功效：**

* **抗癌潛力 (防癌潜力)：** 錳含量增加 — 中和自由基的效力比普通綠茶高30%。1986年研究證實與降低肝癌風險的關聯。
* **強大抗氧化作用：** 兒茶素 — 效力比維生素E高10倍。維生素C — 比平均水準高1.5倍。
* **代謝調節：** 多酚類（25–30%）促進脂肪分解。
* **提神效果：** 咖啡因和L-茶胺酸。
* **改善消化：** 多酚類刺激酶活性。

* **重要提示：** 上述功效基於公開資料，不構成醫療建議。

**9. 沖泡方法：**

* **水溫：** 80–85°C；特級茶 (特级) — 75°C。

* **茶量：** 3克配150毫升水（1:50比例）。

* **茶具：** 玻璃杯（觀察「雀舌」）或白瓷蓋碗（評估香氣）。

* **沖泡過程：**

	1. 溫杯，倒掉水。
	2. 投茶。
	3. 快速洗茶 — 注水，立即倒出 (温润泡快速倒出)。
	4. 第一泡 — 30秒。
	5. 後續 — 每泡增加10秒。茶葉可沖泡3–4次。

* **注意：** 不要使用反覆煮沸的水 (忌反复烧开沸水) — 這會破壞多酚類結構。不建議空腹飲用 — 高單寧含量可能刺激黏膜。開封後 — 冷藏保存，3個月內飲用完畢。

**10. 儲存：**

* 密封容器，陰涼乾燥處保存。
* 最佳 — 冰箱0–5°C。
* 保質期 — 最多12個月。
* 開封後 — 3個月內飲用完畢。

**11. 價格與假冒：**

英德綠茶提供三個主要等級：特級 (特级) — 「明前雀舌」(全芽，600元人民幣/斤起)，一級 (一级) — 「雨前翠芽」(200–400元)，大宗茶 (大宗茶) — 用於茶包和餐廳。

* **如何避免假冒：**

	* **向可靠商家購買**，需有英德市產地證明。
	* **評估葉片大小：** 正宗英德綠茶採用大葉種品種，葉片明顯比小葉種綠茶大。小而「紙質」的葉片 — 可疑。
	* **檢查「冷杯香」：** 餘香>30分鐘 — 真品標誌。
	* **評估口感：** 「微苦無澀」特色（輕微苦味無澀感）。粗澀感 — 假冒。
	* **注意價格：** 可疑的低價 — 假冒跡象。

**12. 有趣事實：**

* 1990年代的「紅改綠」(「從紅茶改為綠茶」) 計畫 — 世界茶業中少數整個地區從紅茶大規模轉換為綠茶的例子之一。雲南大葉種品種，數世紀來僅用於紅茶生產，成功適應了綠茶生產 — 證明了 *Camellia sinensis* var. *assamica* 的多功能性。

* 1986年廣東醫學院證實英德茶中錳含量增加與降低肝癌風險的關聯 — 中國首批「茶葉 — 抗癌」科學研究之一。

* 胺基酸含量 ≥5% — 比平均水準高1.5倍。維生素C含量 — 也高1.5倍。兩個指標的雙重超標 — 綠茶中的罕見現象，由大葉種品種和南方氣候豐富散射光的結合所解釋。

* 陸羽早在8世紀就提到英德（作為韶州的一部分）：「其味極佳」(其味极佳) — 對廣東茶葉的早期正面評價之一。

* 到2025年英德茶產業規模 — 48億元人民幣，141,000畝種植園，年產11,000噸乾茶。英德是廣東茶業的絕對領導者。

**13. 與其他大葉種和廣東綠茶的比較：**

* **滇綠 (滇绿，雲南綠茶)：** 同樣採用雲南大葉種品種，但來自雲南。滇綠 — 更「生澀」；英德綠茶 — 更溫和「精製」，得益於「悶揚結合」工藝。

* **古勞茶 (古劳茶)：** 來自廣東（鶴山）。歷史悠久的綠茶，具有蘭花香。古勞茶 — 更小葉更精緻；英德 — 更厚重濃郁。

* **日照綠茶 (日照绿茶)：** 來自山東。同樣採用「引進」品種（但是小葉種），同樣胺基酸含量高。日照 — 「北方」；英德 — 「南方」。化學成分：兩者胺基酸都高於平均水準，但英德採用大葉種原料，提供更「厚重」的茶體。

* **西湖龍井 (西湖龙井)：** 採用小葉種品種，扁平形。龍井 — 精緻優雅；英德 — 更有力濃郁，具有大葉種特有的更明顯「苦味」。

**結論：**

英德綠茶 — 轉型誕生的茶葉。當1990年代英德的紅茶園「改色為綠」時，很少有人預期雲南大葉種品種，這個大自然為發酵而創造的品種，能夠產出世界級的綠茶。但富含錳和硒的喀斯特土壤、200天的霧日、70%的散射光和「悶揚結合」的工藝 — 在炙熱鐵鍋中的悶炒和拋炒 — 使不可能成為可能。結果 — 胺基酸含量比標準高1.5倍的綠茶，具有抗癌錳元素和「輕微苦味無澀感」的特色，其中大葉種的力量被加工工藝所馴服。英德綠茶 — 適合尋求有「茶體」綠茶的人：不是碧螺春的空靈嫩滑，而是南方濃郁厚重的綠色。
