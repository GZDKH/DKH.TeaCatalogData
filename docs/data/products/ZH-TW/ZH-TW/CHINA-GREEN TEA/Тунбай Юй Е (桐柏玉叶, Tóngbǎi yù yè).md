# 桐柏玉葉 (桐柏玉叶, Tóngbǎi yù yè)

**桐柏玉葉 (桐柏玉叶, Tóngbǎi yù yè)**

![Тунбай Юй Е (桐柏玉叶, Tóngbǎi yù yè)](images/Тунбай%20Юй%20Е%20%28桐柏玉叶,%20Tóngbǎi%20Yù%20Yè%29.png)

![Тунбай Юй Е (桐柏玉叶, Tóngbǎi yù yè)-1](images/Тунбай%20Юй%20Е%20%28桐柏玉叶,%20Tóngbǎi%20Yù%20Yè%29-1.png)

桐柏玉葉是河南省桐柏縣茶業的名片，屬扁形綠茶，在技術和美學上與西湖龍井相近。名稱意為「桐柏的玉葉」，這並非誇大其詞：光滑、扁平、翠綠油潤的茶葉，杏綠色透明的茶湯，持久的香氣和明顯的甘甜，使這款茶成為「中原」(中原) 最優秀的綠茶之一。受國家地理標誌 (农产品地理标志) 保護，並有專門的行業標準 GH/T 1445-2023。

**1. 分類與產地：**

* **類型：** 綠茶（不發酵），子類別——扁形 (扁形) 綠茶，炒青 (炒青) 加工。
* **類別：** 受保護原產地的地方名茶——國家地理標誌產品 (国家地理标志产品)。河南十大名茶 (河南十大名茶) 之一。列入中華人民共和國農業部「國家名特新農產品」名錄。自2022年起，「桐柏玉葉」品牌入選「全國茶葉百強公用品牌」(全国茶叶百强公用品牌)。
* **產地：** 中國河南省 (河南, Hénán)，南陽市 (南阳, Nányáng)，桐柏縣 (桐柏县, Tóngbǎi Xiàn)。核心產區——桐柏山主峰太白頂 (太白顶, Tàibái Dǐng, 1140公尺) 東坡，金台觀 (金台观) 寺廟、水簾洞 (水帘洞) 瀑布和桃花洞 (桃花洞) 地區。
* **地理座標：** 約北緯32.37°，東經113.42°。

**2. 歷史與文化意義：**

* **歷史：** 桐柏是中國中部最古老的茶葉產區之一。據《神農本草》(《神农本草》) 記載，神農 (神农)——傳說中的農業守護神，「嘗百草，日遇七十二毒，得茶而解」——而當地傳說將神農的活動地點定在桐柏山。茶聖陸羽 (陆羽) 在《茶經》(《茶经》) 中提到古義陽郡 (义阳郡, Yìyáng Jùn) 的茶，該郡包括現今桐柏地區，給予高度評價。民間諺語云：「借問陸君何處去？品茗只向太白峰」(借问陆君何处去？品茗只向太白峰)。唐朝 (唐, 618–907) 時期，該地區已是著名的茶區。宋朝 (宋, 960–1279) 時，桐柏山是十三個國家茶場 (茶场) 之一。宋代文人遊覽太白後，留下著名對聯：「閣樓三層讀書論世，泉飛九壑聽瀑烹茗」(阁楼三层读书论世，泉飞九壑听瀑烹茗)。

  現代「桐柏玉葉」茶創制於1985年，在桐柏茶種場 (桐柏茶种场)——最初名為「水簾玉葉」(水帘玉叶, 「水簾的玉葉」)，以瀑布命名。同年，新茶在南陽品鑑比賽中獲勝，並在省級展覽中獲獎。1986年獲「河南省優質茶」稱號。1988年浙江農業大學（現浙江大學）茶學系教授確認其最高品質水準。1997年和1999年——「河南名茶」稱號；1999年——昆明世界園藝博覽會銅獎。2001年——中國國際農業博覽會「名牌產品」地位。2002年——入選「河南十大名茶」。2006年——第六屆國際名茶評比銀獎。2015年12月——中華人民共和國農業部註冊為國家地理標誌產品。2023年通過行業標準 GH/T 1445-2023「桐柏玉葉茶」(桐柏玉叶茶)。

* **名稱：** 「桐柏」(桐柏)——縣名和山脈名，字面意思「梧桐柏樹」（歷史地名學）。「玉」(玉)——玉石、翡翠，純潔和高貴的象徵。「葉」(叶)——葉子。完整含義：「桐柏的玉葉」——詩意的隱喻，強調茶葉的翠綠色澤、光滑質地和珍貴價值。
* **文化意義：** 桐柏是「中原茶鄉」(中原茶乡)，河南省十個重點產茶縣之一。茶文化深深融入日常生活：不給客人奉茶的主人會感到失禮，客人也會感到不受歡迎。當地諺語：「寒夜客來茶當酒」(寒夜客来茶当酒)。自2016年起每年舉辦「春茶第一採」(春茶第一采)，以及桐柏玉葉手工製作比賽——全縣茶師聚集的節慶。桐柏茶業為超過50,000人提供就業機會。

**3. 植物學描述與原料：**

* **種類：** *Camellia sinensis* var. *sinensis*。
* **品種/栽培種：** 當地小葉種群體種 (小叶种群体种)。桐柏山保存著大面積野生和半野生小葉茶樹 (野生茶树) 群落——部分樹齡達百年。這是中國最大的有機認證野生小葉茶樹群落。自1962年起設立桐柏茶種場 (桐柏茶种场)——河南省唯一專業茶樹育種站，年產茶苗超過1000萬株。
* **採摘：** 春季（3月底至4月）；頂級批次——清明前 (清明前)。「桐柏玉葉」系列專用春茶原料。
* **採摘標準：** 特級——單芽或一芽一葉初展 (一芽一叶初展)，其中不少於80%的芽要比葉長。一級——一芽二葉初展 (一芽二叶初展)。
* **原料要求：** 不允許有損傷、病變、紫色葉片、單片葉、魚葉 (鱼叶) 和鱗片。芽葉長度應一致。原料用竹籃運輸（嚴禁使用塑膠袋）。

**4. 風土條件與栽培特點：**

桐柏縣位於河南與湖北交界，處於桐柏山 (桐柏山) 中心——秦嶺-淮河系統的東延，這是中國長江流域（亞熱帶）和黃河流域（溫帶）之間的主要氣候分水嶺。桐柏位於淮河 (淮河)——中國東部第三長河的源頭。

* **種植海拔：** 400–800公尺（主要茶園）；個別野生群落——達1000公尺。主峰太白——1140公尺。
* **氣候：** 過渡性亞熱帶-溫帶季風氣候。年均溫約15°C。年降水量約1168毫米，主要分布在春夏季。充沛的溫暖濕氣和頻繁的山霧為茶樹創造理想的「柔光」環境。
* **土壤：** 山地黃棕壤 (黄棕壤) 和山地棕壤，酸性至弱酸性（pH 4.5–6.0），有機質含量高。母岩為花崗岩和片麻岩。土壤排水良好，富含礦物質。
* **農業技術：** 桐柏是工業影響最小的地區，距最近大城市80公裡。空氣、土壤和水質符合生態清潔農業標準。100%茶園獲無公害 (无公害) 認證。2023年，縣內兩家企業率先獲得「生態低碳茶生產者」地位。

**5. 生產工藝：**

桐柏玉葉工藝是扁形綠茶 (扁形绿茶) 的經典工藝，接近龍井工藝，但適應當地原料。關鍵特點是「輝鍋」(辉锅) 工序，決定最終茶葉的扁平、光滑和光澤。完整流程：採摘 → 攤放 → 殺青 → 回潮 → 過篩 → 輝鍋（最終成型乾燥）→ 包裝。

* **採摘 (采摘 — cǎizhāi)：** 按上述標準手工採摘。原料立即送至工廠。
* **攤放 (摊放 — tānfàng)：** 原料在陰涼處攤放，厚度2–3公分，5–12小時，翻動兩次。水分降低25–30%，葉片變軟有彈性，出現淡雅清香。
* **殺青 (杀青 — shāqīng)：** 鍋溫100–110°C，預先用茶油潤鍋。投葉量約150克。茶葉入鍋發出輕微爆裂聲。手工技法分三階段：(1)「抖」(抖, 「抖動」)——1–2分鐘，每分鐘40–45次動作，手法輕快；(2)「抖+帶+甩」(抖 + 带 + 甩, 「抖動+帶動+甩動」)——2–3分鐘，壓力增加，茶葉開始扁平；(3)「捺+抓」(捺 + 抓, 「按壓+抓握」)——8–9分鐘，茶葉達約80%乾度並呈扁平形。出鍋。
* **回潮 (回潮 — huícháo)：** 熱茶葉薄攤在竹盤上，冷卻，蓋濕布30–40分鐘。茶葉「回復」內部水分，變軟，這對下一工序至關重要。
* **過篩 (过筛 — guòshāi)：** 茶葉通過竹篩（每英寸3.5孔）。大小不同規格在輝鍋時分別處理。
* **輝鍋——最終成型與乾燥 (辉锅 — huīguō)：** 決定性工序，決定扁平度、光滑度、光澤和最終風味特徵。鍋溫70–80°C（特級較低，一級較高），工序末期50–60°C。投葉量：250–300克（殺青後3–4批次）。特級處理約20分鐘，一級更長。茶葉以流暢動作貼鍋壁，獲得最終扁平、光滑、有光澤的形狀。
* **包裝貯藏 (包装贮藏 — bāozhuāng chǔcáng)：** 冷卻後的茶葉密封包裝以保持新鮮度。

**6. 感官特徵：**

* **乾茶外形：** 扁平 (扁平)、光滑 (光滑)、勻整。芽毫隱藏 (芽毫隐藏) 在葉片結構內。色澤翠綠油潤 (翠绿油润)，具特有健康光澤。
* **乾茶香氣：** 清新、純淨，明顯的栗香 (栗香) 和淡雅花香底韻。香氣持久——即使在敞開的杯中也能保持。
* **茶湯香氣：** 高揚、持久，清晰的栗香主調和柔和的植物甘甜背景。
* **滋味：** 鮮爽 (鲜)、醇厚 (醇厚)、飽滿。明顯甘甜、柔和厚度和純淨回味的結合。幾乎無苦澀。持久回甘 (回甘)。耐沖泡 (耐冲泡) 性好。
* **湯色：** 杏綠 (杏绿)，明亮透明，有活潑光澤。
* **葉底（沖泡後茶葉）：** 嫩綠 (嫩绿)，明亮均勻。葉片完整展開，有彈性，無破損。

**7. 化學成分：**

根據農業農村部茶葉品質監督檢驗測試中心 (农业农村部茶叶质量监督检验测试中心) 數據：

* **水溶性浸出物：** 44.7–49.5%——極高指標，確保滋味濃郁和多次沖泡。
* **胺基酸：** 4.6–6.4%——顯著高於綠茶平均值（典型值：2–4%）。正是這個指標賦予桐柏玉葉明顯的甘甜和鮮味。
* **多酚類（兒茶素）：** 適中含量，與高胺基酸水平結合，提供最佳「甘甜/澀味」比例。
* **生物鹼：** 咖啡因（約2.5–3.5%乾重），可可鹼——微量。
* **粗纖維和灰分：** 相對較低——高品質原料（嫩芽和幼葉）的標誌。
* **維生素：** C、B₁、B₂、E、K。維生素C含量高是溫和炒青扁形綠茶的特徵。
* **礦物質：** 鉀、磷、錳、鋅。硒——桐柏山山地土壤的重要微量元素。

**8. 保健功效：**

* **抗氧化保護：** 兒茶素和維生素C共同提供強效自由基中和作用。
* **免疫支持：** 茶多酚增強免疫細胞的抗病毒活性。
* **提神和專注：** 咖啡因和L-茶胺酸的溫和協同效應提供平穩的精神狀態，無「緊張感」。
* **消化支持：** 刺激酶分泌，溫和利膽效應。
* **心血管保護：** 規律飲用綠茶與維持健康膽固醇和血壓水平相關。
* **抗衰老作用：** 抗氧化劑複合物延緩細胞老化——在桐柏玉葉特有的高胺基酸含量下尤為明顯。
* **輻射防護：** 綠茶的傳統特性，與多酚類結合某些放射性核素的能力相關。

*重要提醒：需考慮個人對咖啡因的敏感性。胃酸過多者不建議空腹飲用綠茶。*

**9. 沖泡方法：**

* **水溫：** 75–85°C。特級（單芽）75–80°C；標準級可達85°C。
* **茶量：** 蓋碗3克配150毫升；玻璃杯5克配200–250毫升。
* **茶具：** 瓷蓋碗 (盖碗)——展現栗香和控制浸泡的最佳選擇。玻璃杯——日常品飲和視覺享受：桐柏玉葉的扁平茶葉在水中優美「漂浮」。
* **沖泡過程：**
  1. 用開水溫杯後倒出。
  2. 投茶，輕搖蓋碗，釋放乾茶香氣。
  3. 第一泡：80°C，沿壁注水，浸泡20–25秒。茶葉展開，茶湯呈淡雅杏綠色。
  4. 第二、三泡：25–35秒。滋味高峰——醇厚、栗香甘甜、持久回味。
  5. 第四至六泡：40–60秒，每次增加10–15秒。香氣轉向柔和花草調。
  6. 玻璃杯沖泡：2–3克配200毫升，2–3分鐘，可續水2–3次。

**10. 儲存方法：**

* 密封保存，避光、防潮、避熱、防異味。
* 最佳——冰箱0–5°C真空鋁箔包裝。開封前讓包裝回溫至室溫（15–20分鐘），避免結露。
* 建議飲用期：生產後6–12個月。扁形綠茶的栗香在前4–6個月最為明顯。
* 可冷凍保存（−18°C）長期儲存——18個月內品質無明顯下降。

**11. 價格與仿冒：**

* **價格區間：** 中等至高端。明前茶 (明前茶) 單芽早春批次價格顯著高於大宗等級。野生茶 (野生茶) 山地群落茶葉——獨立的更高價位區間。
* **價格因素：** 採摘季節和等級、茶樹年齡（野生更貴）、手工vs機械加工、具體生產者。
* **避免仿冒方法：**
  * 向有權使用「桐柏玉葉」地理標誌的生產者購買。
  * 檢查外觀：正宗桐柏玉葉——扁平、光滑、芽毫隱藏、油潤光澤。仿品常有不平整表面或粗糙質地。
  * 茶湯應為杏綠色透明，無渾濁和棕色調。
  * 香氣——純淨栗香，無化學「香精味」。
  * 「頂級春茶」價格異常低廉——可能是其他產區原料替代或使用夏秋茶葉的跡象。

**12. 有趣事實：**

* 茶葉最初名稱——「水簾玉葉」(水帘玉叶, 「水簾的玉葉」)——以太白峰腳下的水簾洞 (水帘洞) 瀑布命名。後來改名為「桐柏玉葉」以強化品牌的地域認同。
* 桐柏是中國中部最古老的茶區之一，聲稱「茶文化發源地」地位：當地傳說將神農的活動與桐柏山聯繫起來。
* 桐柏茶種場 (桐柏茶种场)，1962年成立，聘請河南農業大學科學家——中國中北部茶區最大的茶苗培育基地，年產超過1000萬株茶苗。
* 2017年桐柏引入湖南茶祖印象集團 (茶祖印象集团) 建設「茶祖小鎮」(茶祖小镇)——河南省最大的茶旅綜合體，整合生產、科研、文化和休閒，投資21億元人民幣。
* 品茶者將桐柏玉葉與西湖龍井比較，認為河南茶在香氣純淨度和滋味甘甜方面不遜於浙江「老大哥」，同時在浸出物濃度上更勝一籌。

**13. 與其他綠茶比較：**

* **西湖龍井 (西湖龙井, Xīhú Lóngjǐng)：** 浙江扁形綠茶標杆。兩茶都是扁平、光滑、栗香型。龍井通常更清淡，「豆香」調更明顯；桐柏玉葉稍厚重、更甘甜，浸出物含量更高（達49.5% vs 標準龍井約36–40%）。
* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南另一名茶，但為條形 (条形) 非扁形。更多茸毛，明顯清新「綠色」調。桐柏玉葉更光滑、栗香更濃、更油潤。
* **蒙頂甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 四川捲曲綠茶，同樣胺基酸含量高。風味更花香甘甜、「露水」感；桐柏玉葉調性更乾爽，栗香「核心」更明顯。
* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 安徽大葉扁形綠茶。完全不同美學——長大葉片，蘭花香。桐柏玉葉更緊湊、質地更密實，栗香而非蘭花香調。

**結論：**

桐柏玉葉是一款扁形綠茶，承載著桐柏山的古老歷史和現代「中原茶心」的雄心。其主要優勢——完美光滑的茶葉、極高的胺基酸和浸出物含量（近50%）、持久的栗香和柔和持久的甘甜。對於喜愛龍井但想探索新茶的人——桐柏玉葉將是認識中國境外鮮為人知的河南茶文化傳統的絕佳選擇。
