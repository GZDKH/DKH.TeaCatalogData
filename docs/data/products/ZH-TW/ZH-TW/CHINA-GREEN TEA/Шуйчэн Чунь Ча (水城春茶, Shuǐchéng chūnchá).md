# 水城春茶 (水城春茶, Shuǐchéng chūnchá)

**水城春茶 (水城春茶, Shuǐchéng chūnchá)**
![水城春茶 (水城春茶, Shuǐchéng chūnchá)](images/Шуйчэн%20Чунь%20Ча%20(水城春茶,%20Shuǐchéng%20Chūnchá).png)

![水城春茶 (水城春茶, Shuǐchéng chūnchá)-1](images/Шуйчэн%20Чунь%20Ча%20(水城春茶,%20Shuǐchéng%20Chūnchá)-1.png)

水城春茶（水城春茶，Shuǐchéng chūnchá——「水城的春茶」）——高山貴州綠茶，保持著**「貴州早春茶第一採」（貴州早春茶第一采）**的頭銜：其採收比主要貴州產區早10-15天，比浙江和江蘇茶早10-25天。矛盾之處：茶葉生長在**海拔1200-2200公尺**——貴州風土中最高的地區之一——但由於獨特的「低緯度＋高海拔＋少日照」（高海拔、低緯度、寡日照）微氣候，最早甦醒。水城的土壤是中國「硒含量」最高的地區之一：平均硒含量**3.24毫克/公斤**——比全國平均值高數十倍。清朝時期，水城的「木城茶」被列入皇家貢茶名錄（貢茶）。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵）。生產兩種形態：扁形（扁形，「明前翠芽」——明前翠芽，「貴州龍井」）和捲曲形（卷曲形，「清池毛尖」——清池毛尖）。工藝屬炒青烘青型。

* **類別：** 國家地理標誌產品（國家地理標誌產品，2015年）。「貴州早春茶第一採」（貴州早春茶第一采）。清朝（乾隆時期）歷史「貢茶」（貢茶）。國際茶葉大會銀獎（2024年，古樹紅茶）。至2025年——100,200畝（6,680公頃）茶園，200噸乾茶，總價值20億元以上。

* **產地：** 中國，貴州省（貴州，Guìzhōu），六盤水市（六盤水市，Liùpánshuǐ Shì），水城區（水城區，Shuǐchéng Qū）。地理標誌區域——21個鄉鎮，總面積2,370.8平方公里。核心產區：**楊梅鄉（楊梅鄉）**——姬官營古樹基地；**蟠龍鎮（蟠龍鎮）**——百年茶樹和清代「木城茶」發源地；**龍場鄉（龍場鄉）**——佔總產量60%以上。

* **地理座標：** 約北緯26°35′，東經104°50′。

**2. 歷史與文化意義：**

* **歷史：** **清朝「貢茶」（貢茶）：** 乾隆時期（乾隆，1736-1796年），「木城茶」（木城茶，來自蟠龍鎮）被列入皇家貢品名錄，確立了水城作為產茶區的地位。

	**1940年代：** 蟠龍和米籮（米籮）鎮出現首批工業化茶園。

	**1992年：** 楊梅林場（楊梅林場）開闢2,400畝茶園——首個大規模種植園。

	**2008年：** 生產區擴展至15個鄉鎮。

	**1998年至今：** 1998年成立水城茶葉公司，推出首個產品「春意」（春意，「春天的意境」）。2015年獲得地理標誌保護。至2025年——100,200畝。

* **名稱：**

	* 「水城」（水城）——「水之城」：六盤水市轄區名稱。該地區水資源豐富。
	* 「春」（春）——「春天」：強調關鍵優勢——超早春採收。
	* 「茶」（茶）——「茶」。

* **文化意義：** 水城春茶——「貴州第一春茶」，也是六盤水市的象徵，該市被稱為「中國涼都」（中國涼都，Zhōngguó Liángdū）——擁有獨特溫和夏季氣候的城市。

**3. 植物學描述與原料：**

* **品種/栽培種：** 當地群體種和引進品種，適應高山氣候。在核心區（楊梅、蟠龍）保存有**百年古茶樹**。

* **採摘：** 早春採收——貴州最早，比浙江早10-25天。特級——飽滿芽頭或一芽一葉。一級——一芽一葉。二級——一芽二葉。

* **等級：**
	* **特級（特級）：** 飽滿芽頭或一芽一葉。茸毛≥90%。香氣純正持久。1,000元/斤起。
	* **一級（一級）：** 一芽一葉。栗香。600-1,000元。
	* **二級（二級）：** 一芽二葉。持久。400-600元。

**4. 風土與栽培特點：**

* **生長海拔：** **1,200-2,200公尺**——貴州（及中國）綠茶中最高海拔風土之一。只有西藏林芝春綠（1,900-2,300公尺）超過此高度。

* **氣候：** 亞熱帶高山季風氣候。年均溫12.5-16.5°C，年降雨量1,050-1,650毫米。年均霧日**>180天**。散射光佔70%。日夜溫差>10°C。

* **風土公式：** 「高海拔、低緯度、寡日照、多雲霧」——獨特組合，解釋了極高海拔下超早採收的矛盾。

* **土壤：** 黃壤或黃棕壤（黃壤/黃棕壤），pH值4.5-6.5。有機質含量≥1.0%。**創紀錄的硒含量：平均3.24毫克/公斤**——比全國平均值（0.1-0.3毫克/公斤）高數十倍。森林覆蓋率**61.89%**。水質——國家I類標準。

* **「高山冷韻」（高山冷韻，gāoshān lěngyùn）：** 高度＋涼爽＋雲霧減緩生長，提高氨基酸含量（2.3-5.0%，比低海拔茶高20%）和茶多酚（25.2%）。

* **「越冬芽」特性（越冬芽，yuèdōng yá）：** 在海拔1,200-2,200公尺越冬的芽頭，獲得特有的「紅邊」（紅邊，hóngbiān）——葉片上細微的紅色邊緣，證明經過漫長冬季深度積累花青素和營養物質。

**5. 製作工藝：**

工藝包括高溫殺青（300°C）和非物質文化遺產——**「騎火炒」（騎火炒，qíhuǒ chǎo，「騎在火上炒製」）**技法。

* **採摘（採摘）：** 超早採收——12月至2月首批。
* **攤青（攤青——tān qīng）：** 竹製托盤上，4-6小時。
* **殺青（殺青——shāqīng）：** 滾筒式，**300°C**——快速「淬火」，鎖住翠綠色（鎖翠）。
* **揉捻（揉捻——róuniǎn）：** 手工熱揉（手工熱揉）。
* **乾燥（乾燥——gānzào）：** 三階段：初烘120°C（毛火），終烘80°C（足火）。含水量≤6.5%。

* **特色——「騎火炒」（騎在火上炒製）：** 非物質文化遺產——師傅依序使用三種火力：「武火」（武火，wǔhuǒ——高溫）、「文火」（文火，wénhuǒ——中溫）、「微火」（微火，wēihuǒ——低溫）——三重定香（三重定香，sānchóng dìngxiāng）。全程避光操作（全程避光操作）以保持花青素活性。

**6. 感官特徵：**

* **乾茶外觀：** 扁形——平整光滑的「雀舌」（扁形，似雀舌）。捲曲形——緊實厚重的螺旋（卷曲形，緊實重實）。高級品茸毛豐富。

* **乾茶香氣：** 栗香（栗香）——主調，高而持久（栗香高長）。嫩香（嫩香）——特級。花香（花香）——古樹茶。空杯留香**>15分鐘**。

* **茶湯香氣：** 栗香，持久，帶「高山冷韻」。

* **滋味：** 鮮爽（鮮爽）——氨基酸2.3-5.0%。醇厚（醇厚）——茶多酚25.2%。回甘持久（回甘持久）——高含量茶多糖（茶多糖，cháduōtáng）所致。

* **湯色：** 黃綠清澈明亮（黃綠清澈明亮）。

* **葉底：** 嫩綠勻亮，肥厚柔軟帶紅邊——越冬芽標誌（嫩綠勻亮，肥厚柔軟帶紅邊）。

**7. 化學成分：**

* **茶多酚（兒茶素）：** **25.2%**——含量顯著。與硒協同降低膽固醇30%——比浙江和江蘇綠茶更有效（根據比較研究）。
* **氨基酸：** 2.3-5.0%——範圍取決於等級。高級品比低海拔茶高20%。
* **有機硒（有機硒）：** **0.25-4.00毫克/公斤**（平均為普通綠茶的5.5倍以上）。阻斷致癌亞硝胺合成的效率比標準茶高40%。
* **茶多糖（茶多糖）：** 含量較高——提供持久回甘。
* **花青素（花青素）：** 存在於「越冬芽」中——形成紅色邊緣。
* **生物鹼：** 咖啡因——適中含量。
* **維生素：** 維生素C、類胡蘿蔔素。

**8. 保健功效：**

* **硒元素防癌（強效補硒）：** 有機硒（0.25-4.00毫克/公斤）阻斷亞硝胺合成，效率比普通茶高40%。
* **心血管保護（護心降脂）：** 茶多酚（25.2%）＋硒——降低膽固醇30%。
* **抗輻射（抗輻射損傷）：** 硒中和輻射自由基的效率是普通茶的1.5倍。
* **抗氧化作用：** 雙重效果——茶多酚＋硒。
* **提神效果：** 咖啡因和L-茶氨酸。

* **重要提示：** 非醫療建議。

**9. 沖泡方法：**

* **水溫：** 80-85°C；特級用75°C。
* **茶量：** 3克/150毫升（1:50）。
* **茶具：** 玻璃杯或白瓷蓋碗。
* **過程（上投法/上投法）：** 先注水，後投茶。首泡30秒。後續每泡增加10秒。可沖泡3-4次。

**10. 儲存方法：**

* 密封，冷藏0-5°C。保質期12個月。開封後3個月內飲用。

**11. 價格與防偽：**

三個等級：特級（≥1,000元/斤）、一級（600-1,000）、二級（400-600）。購買帶地理標誌認證產品；檢查栗香帶「高山冷韻」；評估越冬芽「紅邊」。

**12. 趣聞軼事：**

* **「貴州早春茶第一採」——「貴州第一春茶」：** 水城春茶是該省最早的茶，比浙江早10-25天。矛盾：在海拔1,200-2,200公尺——卻是第一。

* **土壤硒含量——3.24毫克/公斤（平均）**——比全國平均值高數十倍。這使水城春茶成為中國「硒含量」最高的茶之一——與紫陽綠茶和涇縣特尖並列。

* **「騎火炒」（騎在火上炒製）**——非物質文化遺產：三級火力（武火→文火→微火）——茶葉製作中最富詩意的工藝配方之一。

* 越冬芽的「紅邊」（紅邊）——視覺標記：葉片上細微的紅色邊緣，證明深冬花青素積累。這個特徵在溫暖地區的茶葉中不會出現。

* 六盤水——「中國涼都」（中國涼都）——擁有獨特溫和夏季氣候的城市（夏季平均溫度約19°C），使其成為中國少數茶葉與度假勝地結合的地方之一。

**13. 與其他貴州綠茶的比較：**

* **都勻毛尖（都勻毛尖）：** 「十大名茶」。捲曲「魚鉤」形，「三綠一黃」。都勻更著名；水城更高海拔（達2,200公尺對比約1,500公尺）且「硒含量」更高。

* **湄潭翠芽（湄潭翠芽）：** 扁形，95%自動化。湄潭更「科技化」；水城更「山地化」和手工化，有「騎火炒」。

* **綠寶石茶（綠寶石茶）：** 顆粒狀，用成熟原料。綠寶石更緊實耐泡（7泡以上）；水城更鮮爽和「早春」。

* **林芝春綠（林芝春綠）：** 西藏茶，1,900-2,300公尺。林芝更高海拔；水城更「早採」且更「富硒」（3.24毫克/公斤對比未明確）。

**結語：**

水城春茶——「涼都」的矛盾之茶：最高卻最早，最冷卻最富硒。其越冬芽的「紅邊」、帶「高山冷韻」的栗香、蟠龍的古樹和「騎在火上炒製」——這一切創造出一款茶，將貴州高原的嚴峻轉化為溫柔的春日甜美。這是該省春天的第一杯茶，在這裡春天從山上降臨——帶來硒元素、涼爽和希望。
