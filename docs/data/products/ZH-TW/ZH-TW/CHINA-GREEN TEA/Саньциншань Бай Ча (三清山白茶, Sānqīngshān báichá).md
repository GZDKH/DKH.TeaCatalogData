# 三清山白茶 (三清山白茶, Sānqīngshān báichá)

**三清山白茶 (三清山白茶, Sānqīngshān báichá)**

![Саньциншань Бай Ча (三清山白茶, Sānqīngshān báichá)-1](images/Саньциншань%20Бай%20Ча%20(三清山白茶,%20Sānqīngshān%20Báichá)-1.png)

三清山白茶 (三清山白茶, Sānqīngshān báichá) — 稀有的江西綠茶，採用白葉變異品種製作，栽培於神聖道教名山三清山 (三清山) 山麓 — 聯合國教科文組織世界遺產地。儘管名稱中有「白茶」(白茶，「白茶」) 一詞，此茶按綠茶工藝加工，屬於「白葉綠茶」類 — 類似著名的安吉白茶 (安吉白茶)。其主要價值在於非凡的胺基酸含量：6–10 %，比普通綠茶高2–3倍，而多酚含量異常偏低 (13–16 %)。這種獨特的比例創造出極致甘甜柔和的口感 — 「翡翠牛奶」般毫無苦澀。

**1. 分類與產地：**

* **類型：** 綠茶 (未發酵，绿茶, lǜchá)，採用白葉變異品種製作。儘管名稱為「白茶」，這並非傳統意義上的白茶 (屬於白茶類，加工程度極低)，而是具有特徵性白色/淡黃色葉片的綠茶。殺青方法 — 滾筒炒制，溫度130–140 °C。

* **類別：** 農產品地理標誌產品 (农产品地理标志，2011年，公告第1635號)；「江西名茶」(江西名茶，2013年)。稀有變異茶類 — 「高胺基酸低茶多酚」(高氨基酸低茶多酚)。

* **產地：** 中國；江西省 (江西, Jiāngxī)；玉山縣 (玉山县, Yùshān Xiàn)，隸屬上饒市 (上饶市, Shàngráo Shì)。保護區域 — 全縣境內。品質核心 — 三清鄉 (三清乡, Sānqīng Xiāng)、紫湖鎮 (紫湖镇, Zǐhú Zhèn) 和懷玉鄉 (怀玉乡, Huáiyù Xiāng) — 信江 (信江, Xìnjiāng) 源頭的高山茶園 (>700 公尺)，位於三清山山麓雲霧繚繞地帶。

* **地理座標：** 東經117°52′–118°25′，北緯28°30′–28°59′ (玉山縣境內)。

**2. 歷史與文化意義：**

* **歷史：**

玉山縣是江西最古老的茶區之一，曾列入「唐宋八大茶場」。同治年間 (同治, 1862–1874) 《玉山縣志》記載：「茶以三清山產者味特清冽……不及三清遠甚」(茶以三清山产者味特清冽……不及三清远甚)。這一十九世紀的記錄證實，三清山茶的獨特品質早在現代研究之前就已獲得認可。

1970年代，紫湖茶廠 (紫湖茶厂) 創製了兩個名茶產品：「三清雲霧茶」(三清云雾茶，「三清雲霧茶」)，被評為上饒最佳名茶，以及「紫湖春露」(紫湖春露，「紫湖春露」)，獲中國農業科學院茶葉研究所認證。然而1990年代茶廠停產，茶業陷入衰落。

復興始於二十一世紀，當地育種專家發現並培育了白葉變異 — 品種「白茶玉」(白茶玉，「白茶翡翠」)。這個品種成為當地對浙江安吉白茶成功的回應：相同原理 (白葉變異 + 綠茶加工)，但憑藉高山風土和花崗岩土壤實現了更極致的胺基酸特徵。2011年三清山白茶獲得農產品地理標誌 (中華人民共和國農業部公告第1635號)。2013年榮獲「江西名茶」稱號。至2023年品牌價值達24.45億元人民幣 — 對於相對年輕的品牌而言增長驚人，反映了對「胺基酸」類奢華茶品日益增長的需求。

* **名稱：** 三清山 (Sānqīngshān) — 「三清山」— 神聖的道教名山，道教四大名山 (四大道教名山) 之一，2008年列入聯合國教科文組織世界遺產 (標準VII — 「卓越自然美景」)。「三清」— 道教三位至高神祇 (元始天尊, 灵宝天尊, 道德天尊)；白茶 (Báichá) — 「白茶」，指變異品種葉片異常的白色/淡黃色。完整含義：「三清山的白葉茶」。

* **文化意義：** 三清山白茶與中國最壯觀的山地景觀之一密不可分。三清山以沉浸在雲海中的花崗岩石柱「手指峰」和豐富的生物圈 (超過2300種植物) 聞名。茶園位於聯合國教科文組織緩衝區，這施加了嚴格的生態限制並保證了環境純淨。玉山縣獲認證為「中國天然氧吧」(中国天然氧吧) — 空氣中負離子濃度比城市高50倍。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要 — 白茶玉 (白茶玉, Báichá Yù，「白茶翡翠」) — 當地選育的變異品種 *Camellia sinensis* var. *sinensis*。這是溫度敏感的白葉變異：當溫度低於約23 °C時，嫩葉失去葉綠素，呈現淡黃色或乳白色 (叶肉玉白，「葉肉翡翠白」)，而葉脈保持翠綠色 (叶脉翠绿) — 形成特徵性的雙色效果。溫度升高時葉片逐漸變綠。特點是萌芽早、抗寒耐熱、嫩度期延長 (比標準品種長7–10天)。另外使用福鼎大白茶 (福鼎大白茶)。樹齡超過30年。執行「五不採」標準 (五不采, wǔ bù cǎi)：不採雨水葉、紫色芽、病葉、蟲害葉和畸形葉。

* **採摘：** 春茶 — 三月底至四月中旬。最佳期間 — 溫度未超過約23 °C且葉片保持白色/淡黃色時。採摘延遲會使葉片變綠，失去獨特的「白葉」特徵。

* **採摘標準：** 特級 — 單芽或一芽一葉初展。一級 — 一芽一至二葉。

* **原料要求：** 具有特徵性白色/淡黃色葉片和綠色葉脈的嫩芽。無機械損傷和蟲害痕跡。

**4. 風土與栽培特點：**

玉山縣位於江西東北部，三清山山麓，懷玉山 (怀玉山) 和武夷山 (武夷山) 山脈交匯處。地形為陡峭山坡，有花崗岩露頭。

* **栽培海拔：** 758–1182 公尺 — 江西最高的茶區之一。

* **氣候：** 亞熱帶季風山地氣候。年均溫17–18 °C；降雨量1400–1724毫米/年；霧日>180天/年；日溫差>8 °C。涼爽的春夜 (低於23 °C) 對白葉變異的表現至關重要 — 正是山地微氣候確保了長期的「白葉」期，這在平地無法實現。

* **土壤：** 紅壤和黃壤 (红壤、黄壤)，pH 5.0–6.5。富含鋅 (76.2 毫克/公斤) 和硒 (0.82 毫克/公斤) — 三清山花崗岩體的地球化學標誌。

* **生態：** 森林覆蓋率81%。「中國天然氧吧」認證 (中国天然氧吧) — 負離子濃度比城市高50倍。茶園位於聯合國教科文組織三清山遺產地緩衝區，灌溉採用信江源頭泉水。完全排除工業污染。生態系統獨特：茶園梯田與原始針葉林交錯，三清山花崗岩石柱「手指峰」形成天然防風屏障和霧氣凝結增強的微區域。

**5. 製作工藝：**

三清山白茶採用獨特的複合工藝「晒青 + 轻揉捻」(日光萎凋 + 輕揉捻)，平衡香氣和外形。最終階段 — 炭火烘乾。

* **攤放/萎凋 (摊放 — tān fàng)：** 在通風室內4–6小時。

* **日光萎凋 (晒青 — shài qīng)：** 自然日曬 — 傳統綠茶不常見的步驟。溫和的陽光啟動初步香氣轉化和部分脫水，賦予茶葉額外的甘甜和花香，這是純「炒制」綠茶所不具備的。

* **殺青 (杀青 — shāqīng)：** 滾筒炒制，溫度130–140 °C — 適中溫度以保持精緻的胺基酸特徵。過高溫度會破壞L-茶胺酸和其他胺基酸。

* **輕揉捻 (轻揉捻 — qīng róuniǎn)：** 最小程度揉捻以塑形和釋放茶汁，同時保持芽頭完整性和白毫 (成条保毫，「成條保毫」)。

* **理條 (理条 — lǐ tiáo)：** 70 °C下 — 固定直條外形。

* **提毫 (提毫 — tí háo)：** 50 °C時師傅用手掌滾動茶葉，「提升」白毫至表面 — 決定特徵性銀色光澤的步驟。

* **炭火烘焙 (烘焙 — hōngbèi)：** 60 °C木炭烘乾至含水量≤6.5%。炭火加熱形成細膩甘甜背景，不掩蓋精緻香氣。

**6. 感官特徵：**

* **乾茶外觀：** 挺直緊結的條索 (挺直紧结)；嫩黃綠潤 (嫩黄绿润)，白毫顯露 (显毫)。「白肉綠脈」的雙色特徵在乾茶中也保持。

* **乾茶香氣：** 清香純正 (清香) 帶明顯花香背景 (花香)。精緻的「奶香」調 — 高胺基酸含量的結果。

* **茶湯香氣：** 濃郁持久 (馥郁, fùyù)。花香綠茶調帶「奶油」背景。非典型「炒制」特徵 — 在柔和度上更接近白茶，但更清新。

* **滋味：** 極其鮮爽 (鲜爽)，天然甘甜明亮 (甘醇) 和絲滑順口 (滑, huá)。苦澀幾乎不存在 — 極低多酚含量 (13–16 %) 的結果。甘甜回味悠長。「翡翠牛奶」— 源自十九世紀縣志的詩意描述。

* **湯色：** 透明，淺黃色 (銀針型) 或橙黃色 (壽眉型)。清澈度高。

* **葉底 (沖泡後茶葉)：** 翡翠白帶翠綠葉脈 (玉白翠绿) — 特徵性雙色「大理石」效果。鮮活，呈「花朵」狀展開 (鲜活成朵)。

**7. 化學成分：**

* **胺基酸 (氨基酸)：** 6–10 % — 非凡水平，比綠茶平均值 (2–3 %) 高2–3倍。L-茶胺酸占主導，提供甘甜、「奶油」質感和強效放鬆-提神效果。這是世界所有茶類中胺基酸濃度最高的之一。

* **多酚 (茶多酚)：** 13.2–15.6 % — 約為標準綠茶 (25–30 %) 的一半。低多酚水平解釋了幾乎完全沒有苦澀味。

* **生物鹼：** 咖啡因在標準範圍；可可鹼、茶鹼。

* **維生素：** 維生素A — 含量增高 (維生素A原達12.6毫克/100克)；維生素C、B族維生素、維生素E。

* **礦物質：** 鋅 — 76.2毫克/公斤；硒 — 0.82毫克/公斤 (三清山花崗岩土壤的地球化學標誌)。鉀、錳、氟。

* **葉綠素：** 白葉期含量降低 — 特徵性黃白色的原因。隨著升溫葉片變綠，但早期採摘固定了「白葉」階段。

**8. 保健功效：**

* **抗氧化保護：** 儘管多酚水平降低，兒茶素與維生素E和硒結合提供有效的氧化應激保護。

* **放鬆和認知功能：** 非凡的L-茶胺酸含量 (6–10 % 胺基酸) 刺激大腦α波活動 — 最強效的「放鬆-專注」茶類之一。

* **視力支持：** 維生素A原含量增高 (12.6毫克/100克) 有益視網膜健康，可減輕螢幕工作時的眼疲勞。

* **硒支持：** 天然硒 (0.82毫克/公斤) — 甲狀腺和免疫系統的重要微量元素。

* **胃部溫和：** 低多酚含量使此茶對胃黏膜比標準綠茶溫和得多。

* **心血管系統支持：** 兒茶素有助於調節血脂和血糖水平。

* **重要提醒：** 上述功效基於一般數據，非醫療建議。不建議空腹飲用 (儘管溫和) 和睡前4小時內飲用。內分泌疾病患者請諮詢醫師。

**9. 沖泡方法：**

* **水溫：** 80–85 °C。不使用沸水 — 超過85 °C會破壞L-茶胺酸並增加苦味。

* **茶量：** 3克配150毫升 (比例1:50)。

* **茶具：** 玻璃杯 — 觀察葉片雙色「大理石」效果；白瓷蓋碗 — 聚集花香。可用紫砂壺 (紫砂壶)。

* **沖泡過程：**

	1. 用熱水溫壺後倒掉。
	2. 投入3克茶葉。
	3. 注入80–85 °C水。
	4. 第一泡 — 1–2分鐘。
	5. 後續泡次稍微延長時間。可沖泡3–4次。

* **注意：** 第一泡展現最大甘甜和「奶香」— L-茶胺酸比多酚萃取更快的結果。第二、三泡口感更「綠茶」化，但保持特徵性柔和和無苦澀。

**10. 儲存：**

* **溫度：** 0–5 °C (冰箱) — 必須；白葉綠茶對氧化特別敏感。
* **光線：** 完全避光 — 葉綠素本已降低，額外氧化會使茶葉暗淡。
* **容器：** 密封真空或鋁箔包裝。
* **保質期：** 0–5 °C下可保存12個月。建議前6個月內飲用以獲最大新鮮度和甘甜 — 隨時間推移胺基酸比多酚降解更快，茶葉會失去獨特的「奶香」特徵。

**11. 價格與仿冒：**

三清山白茶屬高價位茶品，因變異品種種植面積有限和採摘窗口期短。特級 — 800元人民幣/斤起；一級 — 400–600元人民幣/斤。

* **如何避免仿冒：**

	* 檢查農產品地理標誌 (农产品地理标志) 標識。
	* 主要視覺標誌 — 葉底的「白肉綠脈」雙色效果 (叶肉玉白叶脉翠绿)。普通綠茶無此效果。
	* 口感應極其柔和甘甜，無明顯澀味。苦澀茶品非三清山白茶。
	* 香氣 — 花香奶油調，無「炒制」或「草腥」味。
	* 可疑低價 — 幾乎肯定是普通綠茶冒充。

**12. 有趣事實：**

* **6–10 % 胺基酸：** 三清山白茶是世界胺基酸含量紀錄保持者之一。比較：安吉白茶 — 5–7 %，標準龍井 — 2–3 %，日本玉露 (遮蔭) — 4–6 %。三清山變異種無需遮蔭即超越玉露。

* **聯合國教科文組織山麓茶：** 三清山2008年列入世界遺產名錄 (標準VII，卓越自然美景)。茶園位於遺產地緩衝區 — 施加最嚴格生態限制，但保證獨特環境純淨。

* **「天然氧吧」：** 玉山縣是中國茶區中少數「氧吧」之一。空氣負離子濃度比城市高50倍 — 原始森林特有指標。

* **「味特清冽……遠甚不及」：** 同治年間 (1862–1874) 縣志記錄三清山茶的優越性早在十九世紀 — 比科學解釋 (白葉變異 + 胺基酸特徵) 早一個半世紀。

* **維生素A原 — 12.6毫克/100克：** 茶類中最高值之一 — 葉綠素降低和白葉中類胡蘿蔔素積累增加的有益副作用。

* **從「雲霧」到「白茶翡翠」：** 三清山經歷了不尋常的品牌演變。1970年代製作傳統綠茶「雲霧」，1990年代產業崩潰，2000年代通過完全不同的產品復興：白葉變異。這是中國茶業中地區在一代人時間內完全改變「茶身份」的少數例子之一。

**13. 與其他白葉綠茶比較：**

* **安吉白茶 (安吉白茶, Ānjí Báichá)：** 浙江。相同原理 — 白葉變異 (白葉一號, 白叶一号)，按綠茶工藝加工。外形 — 扁平「蘭花形」(凤形/龙形)。胺基酸5–7 % vs 三清山6–10 %。安吉更「葉綠素」和「綠茶」口感，帶明顯栗香；三清山更甘甜、「奶香」，花香背景更明顯，花崗岩土壤帶來礦物質調。安吉知名度更高且更易獲得；三清山是極致胺基酸特徵的小眾奢華產品。

* **天目湖白茶 (天目湖白茶, Tiānmùhú Báichá)：** 江蘇天目湖地區。同樣白葉變異 (白葉一號品種)。口感更輕盈「水質」，「厚度」較少。三清山因炭火烘乾和高山風土 (758–1182公尺 vs 200–400公尺) 香氣更深厚。

* **資溪白茶 (资溪白茶, Zīxī Báichá)：** 江西資溪縣 (鄰近地區)。類似變異類型，但海拔較低 (400–800公尺)。胺基酸特徵不如明顯。三清山因聯合國教科文組織緩衝區具有更「極致」成分和更嚴格生態標準。

**結語：**

三清山白茶是茶中悖論：名為「白茶」卻是綠茶；葉片乳白色但升溫後變綠；口感奶油甘甜毫無苦澀，但既未加糖也未調味。這一切都源於最稀有的基因變異，在神聖道教名山的涼爽雲霧中甦醒，胺基酸濃度達到百分之十，空氣比城市清潔50倍。以80 °C水溫在玻璃杯中沖泡 — 觀察翡翠白芽頭帶翠綠葉脈在透明水中綻放，釋放出那種「味特清冽」的韻味，一個半世紀前令無名縣志作者驚嘆不已。
