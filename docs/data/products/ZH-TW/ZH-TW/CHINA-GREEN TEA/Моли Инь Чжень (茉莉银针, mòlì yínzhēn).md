# 茉莉銀針 (茉莉银针, mòlì yínzhēn)

**茉莉銀針 (茉莉银针, mòlì yínzhēn)**

**1. 分類與產地：**

* **類型：** 白茶，以茉莉花薰香。有時因茶湯顏色而被誤歸為綠茶，但基底確實是白茶。
* **類別：** 中國高品質薰香茶類。
* **產地：** 中國福建省 (福建, Fújiàn) - 傳統上被認為是最優質茉莉花茶和白茶的故鄉。福鼎縣 (福鼎, Fúdǐng) - 最可能的基底茶（白毫銀針）產地。
* **地理座標：** 依具體產地而定，福建省約為北緯26°，東經119°。

**2. 歷史與文化意義：**

* **歷史：** 茶葉薰香茉莉花是中國古老傳統，已有數百年歷史。使用白茶，特別是白毫銀針進行茉莉花薰香，是高品質和精英地位的象徵，因為這種白茶本身就極為珍貴。
* **名稱：**

	* "茉莉" (茉莉) - 茉莉花。表示茶葉經茉莉花薰香。
	* "銀針" (银针) - "銀色針狀"。指使用的原料類型 - 覆蓋銀色茸毛的茶芽，以及茶葉的形狀。
* **文化意義：** 茉莉銀針被認為是最精緻的茉莉花茶之一。因其精雅香氣、溫和口感和美麗外觀而備受推崇。這種茶常與奢華和富裕聯繫在一起。

**3. 植物學描述與原料：**

* **茶葉：** 茉莉銀針的製作使用**白毫銀針 (白毫银针, Báiháo Yínzhēn) - "覆蓋白色茸毛的銀針"**，這是一種頂級白茶，完全由未展開的芽頭組成，覆蓋著濃密的銀色茸毛。其原料採自**福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbáichá)** 或**政和大白茶 (政和大白茶, Zhènghé Dàbáichá)** 茶樹。
* **茉莉花：** 使用新鮮、剛剛綻放的**茉莉花 (Jasminum sambac，茉莉花)** 花朵，具有最濃郁和怡人的香氣。
* **茶葉採摘：** 茶葉（芽頭）採摘在早春進行，採摘期很短（幾天時間）。
* **採摘標準：** **僅**採摘嫩嫩的、未展開的芽頭（芽尖），密佈白色茸毛。
* **茉莉花採摘：** 茉莉花在傍晚或夜間採摘，此時花朵最大程度綻放並散發最濃郁的香氣。
* **原料要求：** 極高。僅使用精選的、無損傷的、飽滿的芽頭，覆蓋濃密銀色茸毛，以及新鮮、芳香的茉莉花朵。

**4. 風土條件與栽培特點：**

* **福建省：** 位於中國東南部，具有亞熱帶季風氣候，冬季溫暖，夏季炎熱。該地區地形多山，土壤肥沃，降雨充沛。為茶葉和茉莉花的種植提供了理想條件。
* **海拔高度：** 變化不一，但茶園通常位於海拔500至1000公尺之間。

**5. 製作工藝：**

茉莉銀針製作的主要特點是對最嫩的白茶進行**多次薰香**，使用新鮮茉莉花，同時**重要的是保持芽頭的完整性和外觀**。
* **茶葉加工（白毫銀針）：**

	1. **採摘 (采摘 - cǎi zhāi)：** 如上所述。
	2. **萎凋 (萎凋 - wěidiāo)：** 將採摘的芽頭薄薄攤放在戶外散射陽光下或通風良好的室內。過程持續24至72小時。目的是去除芽頭中的部分水分並啟動輕微的氧化過程。
	3. **乾燥 (干燥 - gānzào)：** 將萎凋的芽頭在陽光下或專用烘乾設備中低溫乾燥，直至完全去除水分。重要的是不要過度乾燥芽頭，以保持其香氣和有益特性。
* **茉莉花薰香 (窨花 - yìnhuā)：**

	1. **花朵準備：** 挑選新鮮採摘的茉莉花，去除損壞和未開放的花蕾。
	2. **混合：** 分層放置茶芽和茉莉花，讓茶葉吸收花朵的香氣。茶葉和茉莉花的比例可能因製造商而異。
	3. **多次薰香：** 混合、浸漬和分離花朵的過程重複數次（通常5至7次，有時更多），每次都使用新鮮的茉莉花。這確保了茶葉深度和持久的薰香。每個週期約持續6-8小時。
	4. **最終乾燥：** 最後一次薰香後，再次乾燥茶葉以去除水分並固定香氣。有時在此階段添加少量新鮮茉莉花以增強香氣（稱為"提花"）。
* **分級 (分级 - fēnjí)：** 成品茶按品質分級。

**6. 感官特徵：**

* **乾茶外觀：** 小巧、嫩嫩的、未展開的芽頭（芽尖），密佈銀白色茸毛。芽頭形狀筆直或略彎曲，如針狀。顏色為銀白色，帶有淡淡的珍珠光澤。
* **乾茶香氣：** 非常溫和、精緻，具有明顯的茉莉花香，與白茶的淡雅香氣交織。香氣應該自然、純淨，無人工香精。
* **茶湯香氣：** 精緻、細膩、茉莉花香，帶有蜂蜜和花香層次。
* **口感：** 非常柔和、溫和、純淨、清爽、微甜，白茶和茉莉花味道和諧結合。回甘悠長、花香，帶有輕微澀感。無苦味或苦味極少。
* **茶湯顏色：** 非常淺，淡黃色，幾乎透明，帶有淡淡的銀色調。
* **葉底（沖泡後的茶葉）：** 完整、有彈性的芽頭，保持形狀並覆蓋銀色茸毛，呈溫和的淺黃色或淡綠色。

**7. 化學成分：**

茉莉銀針結合了白茶和茉莉花的有益特性：
* **多酚類（兒茶素）：** 含量高，如優質白茶。強效抗氧化劑。
* **氨基酸：** 包括L-茶氨酸，具有鎮靜作用。
* **維生素：** C、B群維生素。
* **礦物質：** 鉀、氟、鎂。
* **茉莉花精油：** 賦予茶葉香氣，具有鎮靜、抗菌特性。

**8. 健康功效：**

* **抗氧化作用：** 保護免受自由基損害，延緩衰老，降低多種疾病風險。
* **增強免疫力：** 提高機體對感染的抵抗力。
* **改善消化：** 刺激消化。
* **提神效果：** 溫和提神，改善注意力集中。
* **鎮靜作用：** 茉莉花香氣和茶中的L-茶氨酸緩解壓力，放鬆身心，改善情緒，有助於治療失眠。
* **美膚功效：** 改善膚色，促進肌膚清潔，具有抗衰老效果。
* **清爽效果：** 很好地解渴，特別是在炎熱天氣。
* **抗炎和抗菌作用：** 茉莉花具有抗菌特性。
* **心血管系統益處：** 可能有助於降低膽固醇水平和強化血管。

**9. 沖泡方法：**

* **水溫：** 65-75°C。使用過熱的水可能會"燙傷"嫩芽並導致苦味出現。
* **茶量：** 3-5公克配150-200毫升水。
* **茶具：** 玻璃茶壺、蓋碗或瓷器。玻璃可以觀察芽頭綻放的美麗過程。
* **沖泡過程：**

	1. 用開水溫熱茶具。
	2. 將茶葉放入茶壺或蓋碗。
	3. 注入水後立即倒出第一泡（洗茶）。
	4. 再次注水，浸泡1-3分鐘（第一泡）。
	5. 將茶湯倒入茶杯。
	6. 重複沖泡3-5次，逐漸增加浸泡時間。

**重要細節：**

* **不要過度浸泡：** 過長的浸泡時間可能導致苦味出現。
* **觀察芽頭：** 在沖泡過程中觀察銀色芽頭如何在水中綻放和"舞蹈" - 這是非常美麗的景象。
* **實驗：** 不要害怕嘗試不同的浸泡時間和茶量，找到最適合自己的方式。

**10. 儲存：**

茉莉銀針應儲存在乾燥、涼爽、陰暗的地方，使用密封容器（瓷器、玻璃或鐵罐），遠離異味。理想情況下 - 在冰箱的獨立隔間中。

**11. 價格與仿品：**

茉莉銀針是精品薰香茶，價格通常較高。價格受以下因素影響：茶葉品質（芽頭數量和品質）、使用的茉莉花品質、薰香週期數量，以及製造商聲譽。

**如何辨別優質茉莉銀針：**

* **外觀：** 芽頭應完整、無損傷，密佈銀色茸毛。顏色均勻，無雜質。
* **香氣：** 香氣應明亮、自然的茉莉花香，無人工香精。茉莉花香應與白茶香氣和諧結合，而不是完全掩蓋。
* **口感：** 口感應柔和、平衡，帶有茉莉花香，無苦味。
* **價格：** 過低的價格應引起警覺。

**12. 有趣事實：**

* 茉莉銀針是最美麗和最具美感的中國茶之一。觀察銀色芽頭在沖泡過程中綻放是真正的視覺享受。
* 這種茶常被選作禮品，因為它結合了美麗、精緻的口感和香氣。
* 茉莉銀針非常適合"冷泡法"。

**13. 與其他茉莉花茶的比較：**

* **茉莉龍珠 (茉莉龙珠)：** 同樣是茉莉花薰香茶，但以綠茶葉為基底，捲成球狀（"珍珠"）。茉莉龍珠具有更濃郁的綠茶味，而茉莉銀針更精緻細膩，以芽香為主。
* **茉莉鳳眼 (茉莉凤眼)：** 同樣是茉莉花薰香茶，捲成"鳳眼"形狀。通常使用比茉莉銀針更成熟的原料製作。
* **茉莉花茶 (茉莉花茶)：** 茉莉花茶的總稱。茉莉銀針的區別在於使用純芽頭原料（白毫銀針）和多次薰香，使其口感和香氣更加精緻。

**結論：**

茉莉銀針是精緻的茉莉花茶，結合了白毫銀針白茶的溫和與茉莉花的迷人香氣。其銀色芽頭、包括多次薰香的多階段製作工藝，以及精緻的口感，使其成為真正的茶藝作品。品嚐正宗的茉莉銀針，意味著沉浸在和諧與享受的世界中，感受這種奇妙飲品的有益影響，並接觸中國茶道的古老傳統。這種茶非常適合那些重視美麗、精緻和天然的人，以及那些不僅想要美味飲品，還想從沖泡和品茶過程中獲得美學享受的人。
