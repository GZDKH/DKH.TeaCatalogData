# 磐安雲峰 (磐安雲峰, Pán'ān yúnfēng)

![磐安雲峰 (磐安雲峰, Pán'ān yúnfēng)](images/Паньань%20Юнь%20Фэн%20(磐安云峰,%20Pánān%20Yún%20Fēng).png)

![磐安雲峰 (磐安雲峰, Pán'ān yúnfēng)-1](images/Паньань%20Юнь%20Фэн%20(磐安云峰,%20Pánān%20Yún%20Fēng)-1.png)

磐安雲峰（磐安雲峰，Pán'ān yúnfēng）——「磐安縣的雲霧山峰」——產自浙江省金華市磐安縣的綠茶，位於大盤山脈——被譽為「群山之祖、諸水之源」：這裡是**四大河流水系**的發源地——錢塘江、甌江、靈江和曹娥江。磐安的茶葉傳統可追溯至晉代（3-5世紀）：相傳道教聖人**許遜**（許遜，「真君大帝」）——「四大天師」之一——雲遊磐安群山時，發現野生茶樹，教導當地農民製茶技術，創造了著名的**「婺州東白」**（婺州東白，「婺州的東白茶」）。陸羽在《茶經》中將「婺州東白」列為名茶，並記載「大盤山、東白山產者佳，列為貢品」——「產自大盤山和東白山的最佳，列入貢茶」。縣內有**玉山古茶場**（玉山古茶場，「玉山古代茶葉市場」）——**中國唯一完整保存的古代茶葉市場**（始建於宋，清代重建，面積3430平方公尺），**全國重點文物保護單位**。2022年，「趕茶場」（趕茶場，「茶場集市」——磐安民俗節慶）作為「中國傳統製茶技藝及其相關習俗」的一部分，被列入**聯合國教科文組織非物質文化遺產名錄**。

品質特徵：**「三綠一香」**（sān lǜ yī xiāng，「三種綠色一種香」）：綠色條索、綠色茶湯、綠色葉底＋蘭花香。土壤有機質含量——**3.5-14.8%**——中國所有茶區中最高的範圍之一。水浸出物——**47%**（國家標準36%——超出11個百分點）。森林覆蓋率——**80.1%**。品牌價值——**235.1億元人民幣**（2024年）。

**1. 分類與產地：**

* **類型：** 綠茶（綠茶，lǜchá），未發酵。半炒半烘（半炒半烘，bàn chǎo bàn hōng）。兩種形態：**條形**（條形，tiáo xíng——「針形」，主要形態）和**扁形**（扁形，biǎn xíng）。獨創成形技術**「抖捺理」（dǒu nà lǐ）**——「抖動、按壓、理順」——三種手工技法交替使用，保持芽葉完整並形成特有的「針形」結構。

* **類別：** 國家農產品地理標誌產品（國家農產品地理標誌產品，2020年）。「全國名茶」（全國名茶，商業部，1986年）。杭州茶葉節「文化名茶」（文化名茶，1991年）。縣級非物質文化遺產（縣級非遺，2021年——製作技藝）。「趕茶場」（趕茶場）——聯合國教科文組織遺產組成部分（2022年）。唐代「貢茶」（婺州東白）。玉山古茶場——全國重點文物保護單位。

* **產地：** 中國，浙江省，金華市，磐安縣。14個鄉鎮。大盤山脈，海拔600-1200公尺。座標：東經120°17′-120°47′，北緯28°49′-29°19′。

**2. 歷史與文化意義：**

* **歷史：**

磐安——「純山區縣」，浙江最多山的縣之一，「群山之祖、諸水之源」——「群山的始祖，眾水的源頭」。茶葉歷史始於晉代（3-5世紀）：道教聖人許遜（許遜，西元239-374年），被尊為「真君大帝」，「四大天師」之一——與張道陵、葛洪、薩守堅並列——雲遊磐安玉山時，發現野生茶樹，教導當地農民改良製茶技術。成果就是「婺州東白」（婺州東白）——聞名全中國的茶葉。許遜是真實的歷史人物，南昌人，以治理贛江水患聞名；道教淨明道派尊他為創始人。宋代玉山的茶農感念許遜，為他建立「茶場廟」（茶場廟，「茶場寺廟」），至今仍奉他為「茶神」——這是中國唯一將道教聖人奉為茶葉守護神的寺廟。

唐代陸羽將「婺州東白」收入《茶經》，記載：「大盤山、東白山產者佳，列為貢品」——茶葉成為「貢茶」。宋代，圍繞茶場廟形成了**玉山古茶場**——中國最古老的茶葉貿易市場之一，國家在此實行「榷茶」（榷茶，quèchá，茶葉專賣）。現存建築（清代，面積3430平方公尺）包括寺廟、行政用房和交易大廳——**中國唯一完整保存的古代茶葉市場遺址**。

1979年，在唐代「婺州東白」的基礎上，結合西湖龍井、太平猴魁和峨眉蕊的技術，創製了現代「磐安雲峰」。1983年——恢復磐安縣建制（原屬東陽）：大盤山和東白山被分屬兩縣，磐安「繼承」大盤山，東陽得東白山。這就需要新名稱——「雲峰」（「雲霧山峰」）。1986年——「全國名茶」（商業部）。2020年——地理標誌。2022年「趕茶場」（趕茶場）——玉山古茶場的民俗集市節慶——作為聯合國教科文組織遺產「中國傳統製茶技藝及其相關習俗」的組成部分。

* **名稱：** 磐安（Pán'ān）——縣名；雲峰（Yún Fēng）——「雲霧山峰」——雲霧繚繞山峰的意象。完整含義：「磐安的雲霧山峰」。品牌口號：**「道骨仙風，磐安峰」**——「道家風骨，仙人之風——磐安之峰」。

* **文化意義：** 磐安——「群山之祖、諸水之源」（「群山的始祖，眾水的源頭」）：浙江四大河流的分水嶺。與道教的聯繫（許遜——「四大天師」之一）、最古老的茶葉市場和聯合國教科文組織遺產，使磐安雲峰成為「自然、道教與貿易」交織1700年的茶葉。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要——**木荷群體種**——適應山地雲霧氣候的本地原生品種。輔助：龍井43號、烏牛早。最高等級：單芽或一芽一葉，芽葉長度≤3.5公分。

* **採摘：** 春季——清明前。標準因等級而異：特級——單芽（90%以上茸毛）；一級——一芽一葉（≥90%）；二級——一芽二葉。

* **等級：**
  * **特級：** 全芽，茸毛≥85%。**2000元人民幣/500克起。**
  * **一級：** 一芽一葉≥90%。800-2000元人民幣。
  * **二級：** 一芽二葉。300-800元人民幣。

**4. 風土與栽培特點：**

磐安——「純山區縣」：山地佔領土100%。大盤山（大盤山，1245公尺）——主峰，「四河之頂」。

* **生長海拔：** 600-1200公尺。核心區——800公尺以上。

* **氣候：** 亞熱帶季風氣候。年均溫——16.1°C。降雨量——1570毫米。**霧日——≥200天。** 濕度——≥85%。「山高水秀、林密霧重」——風土特徵。

* **土壤：** 紅黃壤（紅黃壤，pH 4.5-5.5）。土層深度——50-100公分。**有機質——3.5-14.8%**——中國茶區最高且範圍最廣之一。山泉水礦物質含量極低。

* **生態：** 森林覆蓋率——**80.1%**。磐安——**國家級生態文明建設示範縣**。山區零工業。2007年農業部實驗室分析顯示：**水浸出物——47%**（比國家標準36%高11個百分點），**13種農藥——未檢出**，鉛——0.74毫克/公斤（無害標準5毫克/公斤）。「典型山地茶特徵」——專家結論。

**5. 製作工藝：**

磐安雲峰技術繼承「婺州東白」並融合西湖龍井（扁形）、太平猴魁（大芽成形）和峨眉蕊（溫和烘乾）的要素。主要步驟：

* **攤放：** 3-6小時。失水率約15%。

* **殺青——斜鍋（斜鍋，xiégūo）：** 溫度——160°C。與標準平鍋不同，**斜鍋**同時實現拋葉和在重力作用下沿鍋壁滾動。工具——**竹炒手**（竹炒手，zhú chǎoshǒu，竹製炒茶工具）或棕帚（棕櫚掃帚）。原則：「葉翠不老，葉熟不黃，折梗不斷」——「葉片保持翠綠不老化；葉片炒熟不變黃；莖梗彎折不斷裂」。斜鍋＋竹炒手——磐安獨特技術，大多數其他浙江綠茶中未見。

* **做形——獨創技術「抖捺理」：** 在**平鍋**中，80°C。三種手法交替：
  * **抖（dǒu）——「抖動」：** 茶葉被拋起分散。嫩葉——多「抖」；粗葉——少「抖」。
  * **捺（nà）——「按壓」：** 茶葉被壓向鍋壁。嫩葉——**不按壓**（嫩葉不捺）；粗葉——加強按壓。
  * **理（lǐ）——「理順」：** 芽葉理成筆直的「針」。
  
  結果：筆直、秀麗的芽葉（緊直苗秀），保持完整——葉片不破碎。

* **烘乾——兩階段：**
  * **初烘：** 100-120°C。至含水量約30%（「刺手感」——「扎手感覺」）。
  * **復烘——「文火慢烘」：** 至含水量≤6%。目的——均勻乾燥並「顯現」香氣。芽葉搓揉時碎成粉末。

**6. 感官特徵：**

* **乾茶外觀：** 筆直、秀麗的「針」（緊直苗秀，jǐnzhí miáoxiù），「如立正的嫩芽」。翠綠鮮潤，有光澤。茸毛明顯但不過多。

* **乾茶香氣：** 清高持久（清高持久，qīng gāo chíjiǔ）。**蘭花清香**（蘭花清香，lánhuā qīngxiāng）——主調。

* **茶湯香氣：** 蘭花般花香，「空靈」。冷卻後——「蜜香」和「栗香」韻味。持久度——冷杯留香3分鐘以上。

* **滋味：** 鮮醇爽口（鮮醇爽口，xiān chún shuǎng kǒu）。甘醇。回味甘甜，持久。苦澀度——極低。

* **湯色：** 嫩綠明亮——某些資料稱「杏綠」（「杏黃帶綠」）。

* **葉底（沖泡後茶葉）：** 嫩綠成朵，鮮亮。芽葉——完整，不破碎。

* **品質特徵：** **「三綠一香」**（sān lǜ yī xiāng）——「三種綠色一種香」：條索翠綠＋湯色嫩綠＋葉底綠亮＋蘭花清香。

**7. 化學成分：**

* **水浸出物：** **47%**——比國家標準（36%）高11個百分點。這意味著近一半的乾物質溶入茶湯——浙江綠茶和全中國最高指標之一。高水浸出物——土壤有機質創紀錄（3.5-14.8%）和200天霧日的直接結果：接受最少直射陽光的茶樹合成更多可溶性物質（胺基酸、糖類）和較少不溶性物質（木質素、纖維素）。結果——茶湯「飽滿」同時保持柔和。

* **茶多酚：** 約19.2%——適中水平，確保柔和度。

* **胺基酸：** 較高——200天霧日、散射光和創紀錄土壤有機質（3.5-14.8%）的結果。EGCG活性高。

* **鉛（Pb）：** 0.74毫克/公斤——比無害標準（5毫克/公斤）低6.8倍。

* **農藥：** 13種——未檢出（2007年分析）。

* **咖啡因：** 綠茶標準含量。

**8. 保健功效：**

* **抗氧化保護：** EGCG＋維生素C——雙重抗氧化系統。茶多酚19.2%配合高水浸出物（47%）確保抗氧化物有效溶出。

* **提神效果：** 咖啡因與L-茶胺酸協同——溫和、持久的清醒感。

* **消化支持：** 兒茶素刺激胃腸蠕動和脂肪分解。

* **認知功能：** L-茶胺酸刺激大腦α波活動。

* **生態純淨：** 零農藥＋低鉛＝浙江最「純淨」的茶之一。

* **重要提示：** 所列功效基於一般資料，非醫療建議。不建議空腹飲用。每日不超過600毫升。

**9. 沖泡方法：**

* **水溫：** 85°C。不用沸水——嫩芽易「燙傷」。

* **茶量：** 3克/150毫升（比例1:50）。

* **茶具：** 玻璃杯——觀賞芽葉「舞動」：筆直的「針」緩緩垂直下沉，「成朵」綻放，宛如「蘭花瓣瓣」（形如蘭花瓣瓣）。這是浙江最具美感的茶藝之一。

* **方法——「三投法」（Sān tóu fǎ，「三次注水法」）：**
  1. 溫杯。
  2. 投茶。
  3. 注入1/3水（85°C）。
  4. 搖晃杯子30秒——「喚醒」。
  5. 注水至7/10滿。
  6. 浸泡1.5分鐘。

* **沖泡次數：** 3-4次。每次增加20-30秒。

**10. 儲存方法：**

* **溫度：** 0-5°C（冰箱），密封。磐安雲峰——嬌嫩的「半炒半烘」茶，對濕氣和異味敏感。
* **容器：** 密封、不透光。
* **光線：** 完全隔絕。
* **期限：** 建議6-8個月內飲用。開封後——30天。

**11. 價格與防偽：**

磐安雲峰——高價位茶葉。特級——**2000元人民幣/500克起**；一級——800-2000元；二級——300-800元。

* **如何避免假貨：**
  * 地理標誌——「磐安雲峰」。品牌價值——235.1億元。
  * 外形——筆直、秀麗的「針」（緊直苗秀），非捲曲或扁平。
  * 香氣——**蘭花香**（蘭花清香）。無蘭花香調——可能是假貨。
  * 「三綠一香」特徵——三種「綠」（條索、茶湯、葉底）＋一種香。

**12. 趣聞軼事：**

* **道教聖人創造了茶。** 許遜（許遜，西元239-374年）——道教「四大天師」之一——親自教導玉山農民製茶，創造「婺州東白」。茶場廟——他的聖殿。品牌口號：「道骨仙風」——「道家風骨，仙人之風」。

* **聯合國教科文組織（2022年）。** 「趕茶場」（趕茶場）——玉山茶場的民俗集市節慶——列入聯合國教科文組織遺產「中國傳統製茶技藝」。這是罕見的不是茶葉本身，而是**茶葉集市**獲得聯合國教科文組織地位的案例。

* **中國唯一的古代茶葉市場。** 玉山古茶場——**全國唯一保存完整的古代茶葉市場遺址**——中國唯一完整保存的古代茶葉貿易市場。面積——3430平方公尺。

* **四河之源。** 磐安——錢塘江、甌江、靈江和曹娥江的分水嶺。茶葉生長在四條流入太平洋的河流系統的「屋頂」。

* **有機質3.5-14.8%。** 中國茶區創紀錄的範圍。14.8%有機質——黑土地水平，但在山地酸性土壤上。

* **水浸出物47%。** 比標準（36%）高11個百分點。近一半茶葉溶入茶湯——最大「回報」。

* **斜鍋＋竹炒手。** 磐安獨有的炒製：鍋的傾斜使茶葉向下滾動，炒手向上拋起——「重力炒製」。

* **「三綠一香」——「三種綠色一種香」。** 品質公式，銘刻在每一杯磐安雲峰中。

**13. 與其他浙江綠茶的比較：**

* **西湖龍井（西湖龍井，Xīhú Lóngjǐng）：** 杭州。扁平，炒製。「栗香」/「豆香」。中國最著名的綠茶。磐安——「針形」，蘭花香，半炒半烘。兩者都產自浙江，但微氣候不同：西湖——湖區（30-100公尺），磐安——山區（600-1200公尺）。

* **開化龍頂（開化龍頂，Kāihuà Lóng Dǐng）：** 衢州（浙江）。山地茶，「拳形」。栗香。磐安——更「蘭花香」和「針形」。

* **涇縣特尖（涇縣特尖，Jīngxiàn Tè Jiān）：** 安徽。同樣「針形」。但來自不同省份，不同風土。磐安——來自「四河之頂」，有機質高達14.8%。

**結語：**

磐安雲峰——來自「四河之頂」的茶，由道教聖人許遜於3世紀創造，被陸羽在《茶經》中讚為「貢茶」，在中國唯一完整保存的古代茶葉市場交易，2022年通過其集市「趕茶場」進入聯合國教科文組織遺產。其「三綠一香」特徵——三種綠色和一種蘭花香——源自200天霧日、80%森林、高達14.8%的有機質和斜鍋配竹炒手。以85°C用「三投法」沖泡，觀賞筆直的「針」緩緩綻放成「蘭花朵朵」在晶瑩的水中——讓「道骨仙風」充盈您的茶杯。
