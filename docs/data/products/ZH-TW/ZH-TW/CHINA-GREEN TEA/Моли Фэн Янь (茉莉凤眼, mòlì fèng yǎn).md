# 茉莉鳳眼 (茉莉凤眼, mòlì fèng yǎn)

**茉莉鳳眼 (茉莉凤眼, mòlì fèng yǎn)**
![Моли Фэн Янь (茉莉凤眼, mòlì fèng yǎn)](images/Моли%20Фэн%20Янь%20(茉莉凤眼,%20Mòlì%20Fèng%20Yǎn).png)

![Моли Фэн Янь (茉莉凤眼, mòlì fèng yǎn)-1](images/Моли%20Фэн%20Янь%20(茉莉凤眼,%20Mòlì%20Fèng%20Yǎn)-1.png)

茉莉鳳眼是一種藝術性茉莉花茶，每片茶葉都經手工塑形成優雅的細長形狀，宛如神話鳳凰之眼。此茶屬於精品造型茉莉花茶 (工艺花茶, gōngyì huāchá) 的範疇，視覺美感與香氣、滋味密不可分。鳳凰 (凤凰, fènghuáng) 是中國文化的核心象徵之一，代表重生、繁榮與吉祥命運，賦予此茶深刻的象徵意義。

**1. 分類與產地：**

* **類型：** 花茶 (花茶, huāchá) — 綠茶（未發酵），以茉莉花薰製。屬於再加工茶類 (再加工茶, zàijiāgōng chá)。
* **類別：** 工藝茉莉花茶 (工艺花茶, gōngyì huāchá)。手工造型（束形）茶。福州茉莉花茶傳統工藝產品。
* **產地：** 中國福建省 (福建省, Fújiàn shěng) 福州市 (福州, Fúzhōu) — 公認的茉莉花茶製作搖籃。福州窨製工藝 (窨制工艺, yìnzhì gōngyì) 為中華人民共和國國家級非物質文化遺產（2014年起），並為聯合國教科文組織於2022年列入非物質文化遺產名錄的「中國傳統製茶技藝及其相關習俗」項目的一部分。廣西 (广西, Guǎngxī)、雲南 (云南, Yúnnán) 和四川 (四川, Sìchuān) 等省份亦有生產。
* **地理座標：** 福州 — 約北緯26°04′，東經119°18′。

**2. 歷史與文化意義：**

* **歷史：** 茉莉花薰茶傳統起源於中國宋朝 (宋朝, Sòng cháo, 960–1279年)，在明朝 (明朝, Míng cháo) 和清朝 (清朝, Qīng cháo) 達到鼎盛。茉莉花 (*Jasminum sambac*) 本身於2000多年前經海上絲綢之路從南亞傳入中國，在福建省文化中深深紮根。明代學者錢希言 (钱希言, Qián Xīyán) 如此描述繁忙的茉莉花茶市場：「茶鬥時節，買花成市，擇其大而乾者……南風十日，滿簾花香」。清代慈禧太后 (慈禧太后, Cíxǐ Tàihòu) 對茉莉花情有獨鍾，宣布其為宮廷御用花卉。福州茉莉花茶獲得貢茶 (贡茶, gòngchá) 地位，福州本身成為全國薰製中心 — 安徽、浙江和江蘇的茶葉基料運至此地製作頂級茉莉花茶。

    「鳳眼」造型的形成是較晚期工藝花茶傳統 (工艺花茶, gōngyì huāchá) 的元素，於20世紀在束茶技藝基礎上發展而成。此造型要求製茶師特別精準：茶葉必須紮成細長、略彎的橢圓形，尖端尖銳，精確重現神話鳳鳥特有的杏仁狀眼部輪廓。福州學派其他經典造型包括「玉蝶」(玉蝶, yùdié)、「龍珠」(龙珠, lóngzhū) 和「銀針」(银针, yínzhēn)。

* **名稱：**
    * **茉莉 (茉莉, Mòlì)** — 茉莉花。指以新鮮茉莉花薰製。
    * **鳳 (凤, Fèng)** — 鳳凰 (凤凰, fènghuáng)，神話鳥類，在中國文化中象徵重生、恩澤、繁榮與至高和諧。與龍 (龙, lóng) 配對時，鳳凰代表陰性原理與皇后。
    * **眼 (眼, Yǎn)** — 眼睛。合稱「茉莉鳳眼」。

* **文化意義：** 鳳凰是中國神話四靈 (四灵, sì líng) 之一，與龍、麒麟、龜並列。鳳眼被視為洞察力與清晰視野的化身。因此，茉莉鳳眼茶承載著智慧與福祉的祝願。由於結合了象徵深度、視覺美感與精緻口感，鳳眼茶作為禮品茶備受推崇，特別適合慶典場合和商務饋贈。

**3. 植物學描述與原料：**

* **茶坯 (茶坯, chápi)：** 優質烘青綠茶 (烘青, hōngqīng — 「熱風乾燥」)，採用福建省春季採摘製作。烘青基料具有多孔葉片結構，最適合在多次窨製過程中深度吸收茉莉花香。茶樹品種為當地小葉種 *Camellia sinensis* var. *sinensis* 群體種。
* **茉莉花：** 新鮮茉莉花 *Jasminum sambac* (L.) Ait.（茉莉花，茉莉花, mòlihuā）— 香氣最濃郁、純淨甜美的品種。福州茉莉花茶學派專用茉莉花品種。香氣成分由110多種揮發性化合物構成，主要包括芳樟醇、苯乙酸酯、苯甲酸甲酯和吲哚。
* **茶葉採摘標準：** 早春。一芽一葉或一芽二葉 (一芽一叶 / 一芽二叶, yī yá yī yè / yī yá èr yè)。原料完整、鮮嫩、無損傷。
* **茉莉花採摘：** 夏季（6-9月），下午14:00後，遵循「三不採」(三不采, sān bù cǎi) 原則：不採晨花、不採陰天花、不採雨後三天花。花蕾最佳開放程度由製茶師目測和手感判定。
* **原料要求：** 極高標準。僅使用精選芽頭，白毫 (白毫, báiháo) 明顯，茉莉花蕾芳香無損。

**4. 風土條件與栽培特色：**

* **茶園 — 福建：** 亞熱帶季風氣候：年均溫+19…+21°C，降雨量1200–1600毫米。茶園位於海拔200–800公尺，土壤為富含有機質的酸性紅黃壤 (红壤, hóng rǎng)。溫和濕潤的氣候和山地地形有利於形成滋味甘甜、氨基酸含量高的茶葉。
* **茉莉花園 — 閩江流域 (闽江, Mǐnjiāng)：** 沖積砂質土壤 (冲积土, chōngjī tǔ)，肥沃層深厚，位於低地平原（海拔5–50公尺）。炎熱白天與相對涼爽夜晚的溫差刺激花朵芳香物質合成。福州茉莉花以特有的「冰糖甜」(冰糖甜, bīngtáng tián) 聞名 — 獨特的甜香調，源於特殊的微氣候條件。「福州茉莉花與茶文化系統」自2014年起被聯合國糧農組織認定為全球重要農業文化遺產（GIAHS）。

**5. 製作工藝：**

茉莉鳳眼的製作工藝與其他福州學派造型茉莉花茶相似，包括兩個主要階段：茶坯製作（春季）和多次窨製後手工造型（夏季）。

* **茶坯製作 (茶坯制作, chápi zhìzuò)：**
    1. **採摘 (采摘, cǎizhāi)：** 手工採摘嫩芽一至二葉。
    2. **攤涼 (摊凉, tānliáng)：** 薄攤4–6小時，去除表面水分。
    3. **殺青 (杀青, shāqīng)：** 180–200°C快速熱處理，鈍化酶活性，保持綠色。
    4. **晾涼 (晾凉, liàngliáng)：** 冷卻至接近室溫。
    5. **揉捻 (揉捻, róuniǎn)：** 塑造細長形狀。
    6. **烘乾 (烘干, hōnggān)：** 乾燥至含水率4–4.5%。

* **茉莉花窨製 (窨花, yìnhuā)：**
    1. **鮮花養護 (鲜花养护, xiānhuā yǎnghù)：** 分揀、分層堆放，調節花堆溫度（32–37°C）促進開放和最大香氣釋放。
    2. **玉蘭打底 (玉兰打底, yùlán dǎdǐ)：** 可使用少量新鮮白玉蘭花 (*Michelia alba*，約佔茶重1%）創造香氣層次，增強茉莉花香的「清新感」，增添淡雅甜香背景。
    3. **多次窨製 (多窨, duō yìn)：** 循環工序：茶坯與新鮮茉莉花混合 → 靜置6–8小時 → 起花 (起花, qǐhuā) → 烘焙 (烘焙, hōngbèi)。重複5–7次（高檔茶可達8–10次），每次使用新鮮花朵。
    4. **提花 (提花, tíhuā)：** 最後短暫混合少量精選花朵（每100公斤茶用6–10公斤花），賦予明亮的茉莉花香「頭香」。
    5. **最終乾燥：** 至含水率不超過6%。

* **鳳眼造型 (造型, zàoxíng)：** 與其他造型茉莉花茶的關鍵區別。製茶師手工將數片窨製茶葉和芽頭紮成細長、略彎的橢圓形，尖端尖銳，重現鳳眼特有的杏仁狀輪廓。造型必須緊實、對稱，運輸過程中不易散開。此階段需要相當技藝：「鳳眼」造型比許多其他造型更難執行，因為需要精確控制比例和弧度。

* **分級 (分级, fēnjí)：** 成品「鳳眼」按大小、緊實度、對稱性和香氣品質分級。

**6. 感官特徵：**

* **乾茶外形：** 細長、略彎的杏仁狀造型，尖端尖銳，形似眼睛。長度3–5公分，寬度1–1.5公分。色澤銀綠至墨綠，芽頭可見銀白色茸毛 (白毫)。造型緊實，葉片牢固結合。可能有少量乾燥茉莉花瓣。
* **乾茶香氣：** 明亮濃郁的茉莉花香，帶綠茶草本底韻。香氣純淨自然，層次豐富 — 有新鮮花香、淡雅蜜甜和隱約綠茶背景。
* **茶湯香氣：** 濃郁包圍的茉莉花香，具有明顯「鮮靈度」(鲜灵度, xiānlíng dù) — 優質天然窨製的特徵標誌。芳樟醇（鈴蘭花香調）和苯乙酸酯（蜜甜）香調。隨著冷卻，顯現精緻的綠茶底韻。
* **滋味：** 柔和、細膩、清爽。天然甜味與精緻花香和諧交融。澀味極輕，無苦味。茶湯中等厚度，質感順滑絲潤。回甘 (回甘, huígān) 悠長純淨，花蜜香韻，帶愉悅清涼感和淡雅奶香。
* **湯色：** 淺黃帶金黃色調，透明清澈。每次沖泡後漸淺，呈現淡稻草色。
* **葉底：** 展開的葉片和芽頭呈嫩綠色，柔軟有彈性。「鳳眼」造型即使經過數次沖泡往往仍保持完整，展現造型技藝。原料品質清晰可見：葉片完整，色澤均勻。

**7. 化學成分：**

茉莉鳳眼的化學成分是綠茶生物活性物質與茉莉花芳香化合物的結合：

* **多酚類（兒茶素）：** 佔茶坯乾重15–25%。主要兒茶素為表沒食子兒茶素沒食子酸酯（EGCG），強效抗氧化劑。
* **氨基酸：** L-茶氨酸 — 佔乾重1–2%，負責鮮甜味和與咖啡因的協同作用，產生平靜專注效果。
* **生物鹼：** 咖啡因 — 佔乾重2–3%（每150毫升茶湯約20–35毫克）。可可鹼和茶鹼含量微量。
* **維生素：** 抗壞血酸（C）、核黃素（B₂）、硫胺素（B₁）、蘆丁（P）。
* **礦物質：** 鉀、鎂、氟、錳、鋅。
* **茉莉花精油：** 芳樟醇（鈴蘭調，花香基調）、苯乙酸酯（蜜甜）、苯甲酸甲酯（花果調）、吲哚（小劑量增強花香感知）、α-金合歡烯、鄰氨基苯甲酸甲酯（橙花調）、乙酸橙花酯（玫瑰蜜香調）。根據氣相色譜分析，四個主要產區的茉莉花茶中鑑定出145種以上揮發性化合物，其中約13種為品質特徵標誌物。
* **葉綠素：** 通過殺青工藝保存。

**8. 保健功效：**

* **抗氧化作用：** 綠茶兒茶素，特別是EGCG，有效中和自由基，保護細胞免受氧化損傷，延緩衰老過程。
* **提神效果：** 咖啡因與L-茶氨酸結合，提供溫和平穩的刺激，無緊張感，提高專注力和工作效率。
* **抗壓和鎮靜作用：** 茉莉花芳香成分 — 主要是芳樟醇 — 具有經證實的抗焦慮效果。吸入茉莉花香可降低心率，促進情緒放鬆。
* **改善消化：** 多酚類刺激胃液和消化酶分泌，促進新陳代謝正常化。
* **心血管系統支持：** 定期飲用綠茶與降低低密度脂蛋白膽固醇和血壓正常化相關。
* **抗菌作用：** 茉莉花精油對多種革蘭氏陽性菌表現出抗菌活性。
* **增強免疫力：** 維生素C、B群和兒茶素複合物支持機體免疫功能。
* **改善肌膚狀態：** 綠茶抗氧化劑結合茉莉花抗炎特性，有助改善膚色和整體肌膚狀態。

**9. 沖泡方法：**

* **水溫：** 75–85°C。過熱會產生苦味並破壞精緻芳香物質。
* **用茶量：** 1個「鳳眼」（約5–7克）配150–200毫升水。300–500毫升玻璃壺用2–3個造型。
* **茶具：** 玻璃壺 (玻璃壶, bōli hú) 或玻璃杯是觀察「鳳眼」展開的最佳選擇。白瓷蓋碗 (盖碗, gàiwǎn) 也適合。不建議紫砂壺：多孔陶土會吸收茉莉花香。
* **沖泡過程：**
    1. 用開水溫壺後倒掉。
    2. 輕放「鳳眼」入壺或蓋碗。
    3. 注入適溫水後立即倒掉第一泡（洗茶，洗茶, xǐ chá）。
    4. 再次注水，浸泡2–3分鐘（第一泡）。
    5. 分茶入杯。
    6. 重複3–5次，每次增加30–60秒浸泡時間。「鳳眼」會逐漸展開，釋放新的香氣和滋味層次。

**10. 儲存：**

茉莉鳳眼的儲存條件與其他綠茶基茉莉花茶要求相同：

* **溫度：** 最佳為冰箱0–5°C，密封容器中，與強味食品隔離。
* **容器：** 密封不透光容器 — 鐵罐、瓷罐或真空鋁箔袋。使用玻璃容器須完全避光。
* **茶葉大敵：** 濕氣、光線、異味、高溫、頻繁開啟包裝。
* **保存期限：** 適當條件下12–18個月。茉莉花香隨時間減弱，建議購買後一年內飲用。

**11. 價格與仿冒品：**

茉莉鳳眼是手工製作、多次天然茉莉花窨製的茶品，屬於中高端至頂級價位。價格取決於茶坯品質、窨製次數、手工造型複雜度和製茶師聲譽。

**優質茉莉鳳眼鑑別要點：**

* **造型：** 「鳳眼」應緊實、整齊、對稱，無破葉、茶末和雜質。鬆散不整齊的造型是大量生產或粗製的標誌。
* **香氣：** 天然茉莉花香 — 層次豐富、柔和深邃。刺鼻、「化學」、過甜香氣是人工香料的標誌。優質茶展現茉莉花香與綠茶香的和諧平衡。
* **滋味：** 柔和平衡，甜美收尾。苦澀、空洞或「平淡」口感是低品質指標。
* **湯色：** 純淨透明，淺金黃色。混濁或深色是警訊。
* **價格：** 明顯低於造型茉莉花茶市場水準的價格應引起警覺。

**12. 趣味知識：**

* 「鳳眼」造型是造型茉莉花茶中技術最複雜的之一。製茶師需要掌握細長橢圓和尖端的精確比例，需要多年經驗。
* 鳳凰 (凤凰, fènghuáng) 在中國神話中只在和平繁榮時代出現。贈送「鳳眼」茶是祝願福祉、智慧和好運的表達。
* 茉莉鳳眼非常適合冷泡：將造型放入冷水中，冰箱靜置4–8小時。低溫緩慢展開產生特別純淨、透明和細膩的香氣。
* 中國有句話「窨得茉莉無上味，列作人間第一香」— 「吸收茉莉花香後，[茶]獲得無與倫比的滋味；堪稱人間第一香」。這句話完全適用於鳳眼等高檔造型茉莉花茶。
* 福州市自2011年起擁有「世界茉莉花茶之都」非正式稱號，福州茉莉花系統於2014年列入聯合國糧農組織全球重要農業文化遺產名錄。
* 生產1公斤高檔茉莉花茶需要6–8公斤新鮮茉莉花，每朵都需手工採摘。整個夏季，熟練採花工每天只能採集約15–20公斤花蕾，使優質茉莉花茶成為最勞動密集的茶類之一。

**13. 與其他茉莉花茶比較：**

* **茉莉玉蝶 (茉莉玉蝶, Mòlì Yù Dié) — 「茉莉玉蝶」：** 同級別和工藝的最接近類比。區別在於造型：玉蝶呈扁平蝴蝶狀帶「翅膀」，而鳳眼為細長杏仁狀「眼睛」。香味特徵幾乎相同，差異在於象徵意義和視覺印象。
* **茉莉龍珠 (茉莉龙珠, Mòlì Lóngzhū) — 「茉莉龍珠」：** 葉片捲成緊實球狀。比鳳眼更緊密的造型確保更緩慢的展開和濃郁集中的茶湯。龍珠是最普遍的造型茉莉花茶。
* **茉莉銀針 (茉莉银针, Mòlì Yínzhēn) — 「茉莉銀針」：** 以白毫芽頭為基料。滋味比鳳眼更精緻清淡。屬於茉莉花茶最高價位段。
* **茉莉花茶 (茉莉花茶, Mòlì Huāchá) — 標準茉莉花茶：** 散茶無造型，通常3–4次窨製。鳳眼在原料品質、香氣深度和美感方面顯著超越。

**結語：**

茉莉鳳眼是古老象徵與工藝完美結合的茶品。手工塑造的「鳳眼」，以最嫩葉片製成，多次浸潤新鮮茉莉花香，既是飲品，也是藝術品，更是吉祥祝願。觀察杏仁狀造型在熱水中緩緩展開，釋放陣陣甜美花香的過程，近似冥想體驗。此茶適合那些在品茗中尋求不僅是滋味，更是美感、意義和覺知寧靜時光的人們。
