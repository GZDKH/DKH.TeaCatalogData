# 無錫毫茶 (无锡毫茶, Wúxī háochá)

**無錫毫茶 (无锡毫茶, Wúxī háochá)**

![Уси Хао Ча (无锡毫茶, Wúxī háochá)](images/Уси%20Хао%20Ча%20%28无锡毫茶,%20Wúxī%20Háochá%29.png)

![Уси Хао Ча (无锡毫茶, Wúxī háochá)-1](images/Уси%20Хао%20Ча%20%28无锡毫茶,%20Wúxī%20Háochá%29-1.png)

無錫毫茶是來自江蘇省的現代綠茶，於1970年代透過有目標的科學選育工作創製而成。其顯著特色是捲曲嫩芽表面覆蓋豐富的白毫 (毫, háo)，這是由從福建引進的大毫 (大毫) 品種特性所致。此茶與無錫悠久的茶文化緊密相連——無錫是著名惠山泉 (惠山泉) 的所在地，該泉被陸羽譽為「天下第二泉」(天下第二泉)。

**1. 分類與產地：**

* **類型：** 綠茶（不發酵茶）。屬於全炒 (全炒) 類別——完全在鍋中炒製（殺青）的特級優質綠茶。
* **等級：** 中國地方名茶 (名茶, míngchá)。自1986年起被中華人民共和國商務部列入全國名茶 (全国名茶)。自2019年起獲得中華人民共和國農業農村部農產品地理標誌 (农产品地理标志) 認證。自2016年起在國家工商行政管理總局註冊為地理標誌商標 (地理标志商标)。
* **產地：** 中國江蘇省 (江苏省, Jiāngsū shěng) 無錫市 (无锡市, Wúxī shì)。主要產區位於無錫市郊區：濱湖區 (滨湖区)——榮巷 (荣巷)、雪浪 (雪浪)、蠡園 (蠡园)、馬山 (马山) 街道，胡埭鎮 (胡埭镇)；錫山區 (锡山区)——錫北鎮 (锡北镇)；惠山區 (惠山区)——錢橋街道 (钱桥街道)。總計7個鄉鎮街道，32個行政村和社區。
* **地理座標：** 北緯31°22′–31°42′，東經120°04′–120°27′。產區位於太湖 (太湖, Tàihú) 西岸。

**2. 歷史與文化意義：**

* **歷史：** 無錫的茶文化根基深厚。早在明代 (明代, 1368–1644)，佛教僧侶就在惠山 (惠山) 山坡種植茶樹。據《無錫金匱縣志》(《无锡金匮县志》) 記載，西元1395年（洪武 (洪武) 二十八年），僧人普珍 (普珍) 向湖州工匠訂製竹爐 (竹炉) 用於以惠山泉水煮茶，並邀請文人參加茶會。這些活動被明代畫家王紱 (王绂) 的《竹爐煮茶圖》(《竹炉煮茶图》)，以及後來王問 (王问) 的《煮茶圖》(《煮茶图》) 和清代董誥 (董诰) 的《復竹爐煮茶圖》(《复竹炉煮茶图》) 所記錄。

  無錫毫茶的直接歷史始於1973年，當時無錫茶樹品種研究所 (无锡茶树品种研究所) 的科研人員開始基於1966年從福建引進的大毫品種創製新的高級綠茶。經過六年多的選育和技術工作，該茶於1979年正式發表並通過科技鑑定，獲得省市重大科技成果地位。隨後幾年該茶獲得眾多獎項：1984年獲江蘇省食品品質獎；1985、1986和1990年獲農業部和商務部全國名茶稱號；1988年在首屆全國食品博覽會獲銀牌；1991年在杭州國際茶文化節獲「中國文化名茶」(中国文化名茶) 稱號；1992年在首屆全國農業博覽會獲獎。在省級「陸羽杯」(陆羽杯) 競賽中連續八次獲得第一名。在國際競賽中連續兩次獲得金牌。

  2010年，無錫毫茶製作技藝獲得無錫市非物質文化遺產地位。該茶出口至美國、英國、加拿大等國家。

* **名稱：** 「無錫毫茶」(无锡毫茶) 字面意思是「來自無錫的毫茶」。第一部分——無锡 (Wúxī)——是地名，太湖岸邊城市的名稱。第二部分——毫 (háo)——意為「細毫」、「絨毛」，指覆蓋成品茶葉的豐富白毫 (茸毫, róngháo)。第三部分——茶 (chá)——「茶」。因此，名稱準確反映了地理位置和產品的主要外觀特徵。

* **文化意義：** 無錫毫茶與「無錫茶，二泉水」(无锡茶，二泉水) 的文化三位一體密不可分：著名的太湖、療癒的惠山泉 (惠山泉，「天下第二泉」，由唐代茶水專家張又新 (张又新) 如此命名) 和當地茶葉。該茶成為地區名片，參與當地節慶和品鑑競賽，是城市美食文化認同的重要元素。

**3. 植物學描述與原料：**

* **種類：** *Camellia sinensis* (L.) O. Kuntze.
* **品種/栽培種：** 大毫 (大毫, Dàháo)——無性（克隆）品種，1966年從福建省 (福建) 引進。屬於半喬木型 (半乔木型, bàn qiáomù xíng)，大葉型 (大叶型, dàyè xíng)，早芽種 (早芽种, zǎoyá zhǒng)。特點是萌芽能力強，萌芽整齊且早，嫩芽茸毛特多 (芽梢茸毛特多)，芽頭肥大緊實，產量高，抗逆性好。該品種很好地適應了太湖沿岸地區的條件，成為生產無錫毫茶的基礎。
* **採摘：** 主要是春季採摘；優質批次——早春（清明 (清明) 前，四月初）。夏季和秋季採摘也會加工，但價值較低。
* **採摘標準：** 原料分為四個等級。一級：一芽一葉初展 (一芽一叶初展)。二級：一芽一葉半開展 (一芽一叶半开展)。三級：一芽一葉開展 (一芽一叶开展)。四級：一芽二葉初展 (一芽二叶初展)。夏秋採摘——主要是一芽二葉開展。製作100公克一級乾茶需要16,000至20,000個芽頭和嫩梢。
* **原料要求：** 嫩梢必須新鮮、完整，無機械損傷和過熱。芽頭必須有豐富茸毛——這是該品種優質原料的關鍵標誌。

**4. 風土條件與栽培特點：**

* **地形與景觀：** 產區位於太湖西岸，為低山丘陵地貌，緩坡與水道、稻田交錯。中國東部最大淡水湖的影響提供了溫和的微氣候，經常有霧氣和適度的溫差。
* **海拔高度：** 茶園主要位於海拔30至300公尺。
* **氣候：** 亞熱帶季風氣候，具有海洋性影響特徵。四季分明。年平均氣溫約15.5°C。年降水量877–1438毫米。相對濕度超過80%。年日照約2064小時。氣候有利於芽頭早萌和氨基酸積累。
* **土壤：** 主要為酸性基礎上的黃棕壤 (黄棕壤) 和紅壤（pH 4.5–6.0），有機質含量高。良好的透水性和通氣性有利於根系發育和豐富礦物質成分的形成。
* **農業技術：** 茶園管理注重生態方法：及時修剪茶樹，有機施肥，生物防治害蟲。來自山丘和沿岸霧氣的自然遮蔭有利於L-茶氨酸積累和降低原料苦澀味。

**5. 製作工藝：**

無錫毫茶屬於全炒 (全炒) 類別——完全炒製的綠茶。其工藝旨在最大程度保持豐富茸毛，形成特徵性的捲曲嫩芽形狀，並創造持久純淨的香氣，無「生青」草味。主要步驟：

* **採摘 (采摘 — cǎizhāi)：** 根據等級標準手工選摘嫩芽。採摘的原料立即送到工廠，防止過熱和過早氧化。

* **攤晾 (摊晾 — tānliàng)：** 將新鮮嫩芽攤放在清潔托盤上，在陰涼室內鋪成3–5公分厚的薄層。持續約6小時（根據氣溫調整；夏季較短）。目的——平衡水分，透過蛋白質部分分解為游離氨基酸開始形成香氣化合物。

* **殺青 (杀青 — shāqīng)：** 關鍵步驟。在加熱的鍋中 (滚筒炒锅) 高溫進行。任務——快速鈍化氧化酶，停止多酚氧化，保持綠色並奠定香氣基礎。原則「高溫殺青，均勻一致」(高温杀青，均匀一致)。葉片變軟並略有黏性，草味消失，出現茶香。

* **揉捻 (揉捻 — róuniǎn)：** 輕柔揉捻以部分破壞細胞壁，使細胞汁液滲出到葉片表面。這確保沖泡時充分浸出。揉捻要輕，以免損傷茸毛。

* **搓毫 (搓毫 — cuōháo)：** 特殊步驟，賦予茶葉其名稱。手工或用專用設備進行：用特殊的手掌動作搓揉嫩芽，使白毫「豎立」並均勻覆蓋捲曲茶葉表面。此步驟需要高技能和細膩——過度壓力會折斷茸毛並損害茶葉外觀。

* **乾燥 (干燥 — gānzào)：** 在適中溫度下達到穩定水分（≤ 5–6%）。固定形狀，鎖定香氣，去除殘餘水分。成品茶獲得特徵性持久香氣和乾葉的脆性。

**6. 感官特徵：**

* **乾茶外觀：** 嫩芽緊密捲曲，捲成特徵性彈性螺旋 (条索卷曲)。葉片肥大飽滿，翠綠帶祖母綠色澤 (肥壮绿翠)。整個表面覆蓋豐富的銀白色茸毛 (白毫披覆)，這是品質的主要視覺標誌。
* **乾茶香氣：** 純淨、清新，具有明顯的栗香 (栗香, lìxiāng)，伴有淡雅花香。香氣持久且「高揚」(香高持久)。
* **茶湯香氣：** 清新、透明，以栗香為基調，帶有春綠的細膩香調和淡淡堅果餘韻。茸毛溶於茶湯中，為香氣增添幾不可察的奶香柔和感。
* **滋味：** 鮮醇濃厚 (鲜醇浓厚)。基調——純淨甘甜帶有明亮的鮮味成分，由高氨基酸含量所致。茶體中等濃度，無粗糙感和明顯苦澀。餘韻——持久、清爽，帶有甘甜回味 (回甘, huígān) 和純淨香氣尾韻。
* **湯色：** 碧綠澄清 (汤色碧绿澄清)，帶有淡雅祖母綠光澤，透明清澈。前幾泡可見飄浮的白色茸毛，這被認為是茶葉的美學特色。
* **葉底（沖泡後茶葉）：** 葉片整齊、嫩勻 (叶底嫩匀)。顏色——明亮淺綠色，葉片彈性、多汁、充分展開。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 含量約佔乾重的15–22%，這是基於大葉種的高品質綠茶的特徵。主要成分——兒茶素：表沒食子兒茶素沒食子酸酯 (EGCG)、表兒茶素 (EC)、表沒食子兒茶素 (EGC)。負責輕微澀味、抗氧化活性和茶湯顏色形成。
* **氨基酸 (氨基酸)：** 含量較高——這是無錫毫茶的特徵之一，在專業資料中被強調。關鍵成分——L-茶氨酸 (L-茶氨酸)，負責甘甜、鮮味和鎮靜效果。早春大毫品種原料中游離氨基酸含量超過江蘇地區綠茶平均水準，這既由品種遺傳特性決定，也由涼爽春季和高濕度條件所致。
* **生物鹼：** 咖啡鹼 (咖啡碱)——約佔乾重的2.5–3.5%，提供提神效果。可可鹼和茶鹼——微量，額外支持提神作用和利尿效果。
* **維生素：** 維生素C（抗壞血酸）——由於溫和的殺青工藝得到良好保存；B族維生素（B₁、B₂、B₃）；葉酸；β-胡蘿蔔素（維生素A原）。
* **礦物質：** 鉀、鎂、磷、錳、鋅、氟。由於太湖沿岸地區的酸性土壤，礦物質成分得到豐富。
* **精油和香氣化合物：** 芳樟醇、香葉醇、橙花叔醇、苯乙醛和其他揮發性化合物形成特徵性栗香花香風味。

**8. 健康功效：**

* **抗氧化保護：** 高兒茶素含量（特別是EGCG）提供強大的自由基中和作用，有助於延緩細胞老化。
* **溫和提神效果：** 咖啡鹼和L-茶氨酸的協同作用提供平衡的提神效果，無急劇波動：提高注意力集中，改善認知活動，同時保持平靜的情緒狀態。
* **消化支持：** 多酚和單寧刺激腸道蠕動，促進健康腸道菌群，幫助食物消化。
* **心血管系統：** 定期飲用綠茶與降低「壞」膽固醇（LDL）水準和維持血管彈性相關。
* **免疫力增強：** 維生素（C、B族）、礦物質（鋅、錳）和多酚的複合物具有整體強化作用。
* **口腔健康支持：** 茶中含有的氟和兒茶素具有殺菌特性，阻止蛀牙發展。
* **抗炎作用：** 兒茶素表現出適度的抗炎活性，對慢性炎症過程有益。

*注意：茶不是藥物。對咖啡鹼敏感者、懷孕和哺乳期間，以及患有胃腸道疾病時建議諮詢醫生。*

**9. 沖泡方法：**

* **水溫：** 80–90°C。一級（最嫩的原料）——80–85°C；二、三級——85–90°C。過熱的水會抑制精緻香氣並增強苦澀味。
* **茶量：** 3公克配150毫升水（蓋碗「快沖」法）；2–3公克配200毫升（玻璃杯或茶杯，浸泡法）。
* **茶具：** 玻璃杯 (玻璃杯)——視覺享受的理想選擇：茸毛在綠色茶湯中的舞蹈。蓋碗 (盖碗) 瓷製或玻璃——更精確控制浸出。瓷茶壺——日常沖泡。

* **沖泡過程：**
  1. **溫杯。** 用熱水沖洗玻璃杯或蓋碗，倒掉。
  2. **投茶。** 將3公克乾茶放入溫熱的容器中。
  3. **第一次沖水。** 倒入80–85°C的水約1/3容量。讓茶葉「甦醒」20–30秒，輕輕搖動容器。
  4. **加水和浸泡。** 加水至滿容量。浸泡40–60秒（玻璃杯中——1.5–2分鐘）。
  5. **倒茶。** 將茶湯倒入茶杯或直接從玻璃杯飲用，喝至2/3容量後再加水。
  6. **重複沖泡。** 茶可承受3–4次完整沖泡。每次後續沖泡增加15–20秒時間。

* **特別建議：** 無錫傳統規定用惠山泉水沖泡毫茶——或者，如無此泉水，用低礦化度軟水。「無錫茶，二泉水」(无锡茶，二泉水)——經典搭配，最好地展現茶的甘甜和純淨。

**10. 儲存：**

* **容器：** 密封、不透光包裝——鋁箔、鐵罐或帶緊密蓋子的陶瓷容器。額外建議真空包裝或使用帶拉鍊的鋁箔袋並排出空氣。
* **溫度：** 最佳——0–5°C（冷藏室）。嚴格密封下——可在室溫陰涼暗處保存，但保質期縮短。
* **茶的敵人：** 光線、濕氣、異味、高溫、氧氣。特別重要的是將茶與冰箱中食品異味隔離。
* **保質期：** 為最大程度展現風味——生產後6–12個月內飲用。專業冷藏保存——可達18個月而不顯著失去品質。

**11. 價格與假冒：**

* **價格類別：** 江蘇綠茶中的中等和中高檔。價格取決於等級（一級最貴）、採摘季節（早春較貴）和具體生產商。早春一級可比夏季四級貴3–5倍。
* **如何避免假冒：**
  * **向有執照的可靠供應商購買** 具有地理標誌 (地理标志) 使用許可。有「無錫毫茶——地理標誌產品」標識——真品的重要標誌。
  * **評估外觀：** 真正的毫茶具有豐富的銀白色茸毛，緊實肥大的翠綠色螺旋。假冒品通常葉片細小、茸毛稀少或顏色暗淡。
  * **檢查香氣：** 真正的毫茶具有純淨、持久的栗香花香。人工加香給出刺鼻、快速消散的氣味。
  * **檢查茶湯：** 真茶產生明亮、清澈、透明的綠色祖母綠茶湯。混濁、暗淡或黃色茶湯——陳茶或假茶的標誌。
  * **注意價格：** 可疑的低價——懷疑的理由。無錫毫茶——產區相對有限的產品。

*注意：市場上存在將無錫毫茶冒充洞庭碧螺春 (洞庭碧螺春) 銷售的做法，因為捲曲形狀外觀相似。關鍵區別：碧螺春有明顯的花果香 (花果香)，而毫茶有栗香毫香 (毫香, háoxiāng) 風味，無果香調。*

**12. 有趣事實：**

* 惠山泉 (惠山泉)，被唐宋詩人歌頌，在唐代被茶水專家張又新 (张又新) 認定為「天下第二泉」(天下第二泉)。盲人音樂家華彥鈞 (华彦钧, 阿炳) 創作的著名二胡曲《二泉映月》(《二泉映月》, Èrquán yìngyuè) 使這個泉水在世界文化中永垂不朽。
* 生產100公克最高等級無錫毫茶需要多達20,000個單獨採摘的嫩芽——這使其成為江蘇最勞動密集的綠茶之一。
* 作為茶葉基礎的大毫品種是來自福建的「移民」，完美適應了太湖沿岸條件。這個選育轉移的故事——是從亞熱帶南方成功「茶葉遷移」到較涼爽江南地區的罕見例子。
* 無錫毫茶是中國少數獲得三級地理保護的現代綠茶之一：省級名茶地位、國家地理標誌商標和國家農產品地理標誌認證。
* 在玻璃杯中沖泡時，白色茸毛從嫩芽分離並在祖母綠茶湯中飄浮——這種景象被稱為「碧湖飛雪」(碧湖飞雪)——被認為是品嚐此茶美學體驗的一部分。

**結論：**

無錫毫茶——命運奇特的茶：正式歷史雖短（不到半世紀），卻依託於太湖岸邊、惠山腳下、傳說中「天下第二泉」旁這座城市的悠久茶文化。其濃密的銀色茸毫、飽滿甘甜的茶體和透明祖母綠茶湯創造出豐富的印象——這款茶彷彿毫不吝嗇，在每次沖泡中傾其所有。無錫毫茶適合尋求有個性綠茶的人：不苦澀、不「水薄」，而是自信、濃厚且驚人純淨。用80–85°C軟水在玻璃杯中沖泡——觀看白色茸毛在翡翠茶湯中舞蹈，重現太湖上春雪飄舞的景象。
