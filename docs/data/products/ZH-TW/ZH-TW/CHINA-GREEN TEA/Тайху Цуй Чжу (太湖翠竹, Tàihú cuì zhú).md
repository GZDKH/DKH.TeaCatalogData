# 太湖翠竹 (太湖翠竹, Tàihú cuì zhú)

**太湖翠竹 (太湖翠竹, Tàihú cuì zhú)**

![Тайху Цуй Чжу (太湖翠竹, Tàihú cuì zhú)](images/Тайху%20Цуй%20Чжу%20%28太湖翠竹,%20Tàihú%20Cuì%20Zhú%29.png)

![Тайху Цуй Чжу (太湖翠竹, Tàihú cuì zhú)-1](images/Тайху%20Цуй%20Чжу%20%28太湖翠竹,%20Tàihú%20Cuì%20Zhú%29-1.png)

太湖翠竹是來自無錫的優雅綠茶，於1980年代末誕生於太湖湖畔。其特色是扁平、略微彎曲的茶葉，形似竹葉：在透明玻璃杯中沖泡時，茶葉展開並垂直立起，將茶杯變成一座微型竹林。這款茶是江南美學的精髓：清新、純淨、精緻。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵，綠茶, lǜchá）。

* **類別：** 江蘇省地方名茶（名茶, míngchá）。註冊地理標誌「斗山太湖翠竹」(斗山太湖翠竹) 於2011年獲得證明商標地理標誌（地理標志证明商標）地位（2006年提出申請）。多次競賽獲獎：連續八次在省級「陸羽杯」(陆羽杯) 競賽中獲得第一名，在全國「中國杯」(中国杯) 競賽中獲得第一名，兩次獲得國際名茶競賽金獎，兩次在中國農業展覽會上被評為「全國名優產品」。

* **產地：** 中國江蘇省 (江苏, Jiāngsū)，無錫市 (无锡市, Wúxī shì)。主要生產區域為錫北鎮 (锡北镇) 斗山地區 (斗山, Dǒushān)，以及八士 (八士)、雪浪 (雪浪)、藕塘 (藕塘)、張涇 (张泾)、胡埭 (胡埭) 等地區的茶園。茶園位於太湖 (太湖) 西北岸的丘陵地帶。

* **地理座標：** 約北緯31.49°，東經120.31°（以無錫市為參考）。斗山地區約北緯31.55°，東經120.37°。

**2. 歷史與文化意義：**

* **歷史：** 太湖翠竹是相對年輕的茶葉：由無錫茶業專家於1984-1989年間創制。項目始於1984年，當時從福建省引進福安大白茶 (福安大白茶) 和福鼎大毫茶 (福鼎大毫茶) 品種，以及從浙江引進安吉白茶 (安吉白茶)，進行試種並開發新的名茶。最初產品稱為「斗山青峰」(斗山青峰) 和「梁溪浪尖」(梁溪浪尖，其中「梁溪」是無錫的歷史名稱)；1989年確定了現在的詩意名稱「太湖翠竹」。

最初的批次完全手工製作。1994年開始引入多功能製茶機械，在不失去特色葉形的情況下實現規模化生產。2002年該茶在「陸羽杯」競賽中再次獲勝——連續第八次，鞏固了其作為無錫茶業旗艦的地位。自2003年起每年舉辦春季「無錫斗山太湖翠竹茶葉節」(无锡斗山太湖翠竹茶叶节)，大幅提升了品牌知名度。到2011年，斗山地區所有茶園（約3500畝）都通過了安全農業基地認證，部分茶園獲得了「綠色食品」(绿色食品) 和有機茶 (有机茶) 認證。產品出口至日本、西歐、東南亞和港澳地區。

* **重要日期：**
  * 1984年 — 開始引進福建品種並開發新茶。
  * 1989年 — 確定「太湖翠竹」名稱。
  * 1994年 — 轉向機械化生產。
  * 2002年 — 連續第八次在「陸羽杯」競賽中獲得第一名（江蘇省）。
  * 2003年 — 開始在斗山舉辦年度茶葉節。
  * 2011年 — 註冊「斗山太湖翠竹」地理標誌；有機茶園認證。

* **名稱：** 太湖 (太湖, Tàihú) —「大湖」，中國第三大淡水湖，茶園位於其湖畔。翠 (翠, cuì) —「翡翠綠、祖母綠」。竹 (竹, zhú) —「竹子」。全名「太湖翠竹」—「太湖的翡翠竹子」—準確傳達了地理來源和外觀：扁平、略微彎曲的嫩綠色茶葉形似嫩竹葉。

* **文化意義：** 太湖翠竹的誕生地斗山地區具有深厚的神話和生態根基。據傳說，正是在這裡，傳說中的舜帝 (舜帝, Shùn Dì) 耕種土地，宣揚「天人協和，萬物共榮」(天人协和，万物共荣) 的原則。十八世紀康熙皇帝 (康熙) 時期，該地區獲得自然保護區地位：實施了禁止狩獵、捕魚和砍伐森林的禁令 (禁渔禁猎，禁止开山)。現在斗山是江蘇省自然生態區的一部分，「太湖翠竹」已成為無錫優質地方農產品的象徵，作為太湖沿岸茶旅遊的元素推廣—與黿頭渚 (鼋头渚)、靈山大佛 (灵山大佛) 和其他景點並列。

**3. 植物學描述與原料：**

* **物種：** *Camellia sinensis* (L.) Kuntze.

* **品種/栽培種：** 專門為生產太湖翠竹而引進的主要栽培種：福安大白茶 (福安大白茶, Fú'ān Dà Bái Chá) — 來自福建省的大葉品系，產生均勻、易成形的芽頭；福鼎大毫茶 (福鼎大毫茶, Fúdǐng Dà Háo Chá) — 具有大型、密毛芽頭的品系，提供明顯的茸毛和較高的氨基酸含量。也使用安吉白茶 (安吉白茶)、龍井 (龙井) 品系和名山 (名山) 系列的本地適應品種。

* **採摘：** 春季採摘，主要在「清明」(清明, 四月初) 期間和之後幾天。優質批次 (明前茶) 在清明前採摘—三月底。

* **採摘標準：** 一芽一葉初展 (一芽一叶初展, yì yá yì yè chūzhǎn)。生產500克成品茶需要超過30,000個嫩芽。

* **原料要求：** 芽葉嫩綠、完整、大小均勻，無機械損傷和蟲害痕跡。新鮮採摘的原料立即送往加工。

**4. 風土條件與栽培特點：**

* **地形與地理：** 茶園位於太湖西北岸低矮丘陵 (丘陵) 的緩坡上，在斗山地區—「山勢雄偉，綿延數裡」(斗山雄峙，绵亘数里)。地勢三面環山，面向湖泊，形成天然的濕潤空氣循環。茶園遠離城市區域和工業區。

* **海拔高度：** 海拔30-191公尺。斗山最高海拔191公尺。按茶葉標準這是較低的海拔，但太湖巨大水面（面積約2400平方公裡）的鄰近彌補了海拔不足，創造了「湖泊高山效應」：穩定的高濕度、頻繁的霧氣和散射光。

* **氣候：** 北亞熱帶季風濕潤氣候 (北亚热带季风湿润性气候)。年平均溫度15.4°C。降雨充沛，無霜期長。由於太湖鄰近，空氣濕度較高。頻繁的晨霧和晚霧提供散射光，有利於茶葉中氨基酸和葉綠素的積累。

* **土壤：** 酸性和弱酸性（pH 4.5-6.0），壤土，有機質含量高。肥力由多世紀的森林腐殖質維持—斗山地區自康熙時期的保護令以來保持著茂密的森林覆蓋。

* **農業技術：** 自2011年起，斗山地區所有茶園採用病蟲害防治「五統一」(五统一) 系統：統一監測、統一允許使用的藥劑（僅生物性、對人體無害）、統一處理時間、統一培訓、統一技術規程。處理次數從每年13-14次減少到6-7次。部分茶園獲得有機認證。

**5. 製作工藝：**

太湖翠竹工藝結合炒制和烘乾 (烘炒结合)，包括特殊的最終階段「輝炒提香」(辉炒提香 —「光亮炒制提升香氣」)，賦予茶葉特有的栗色光澤和持久香氣。

* **攤放 (摊放 — tānfàng)：** 新鮮採摘的芽葉在通風室內薄攤3-5小時，以平衡水分並開始形成香氣。

* **萎凋 (萎凋 — wěidiāo)：** 輕度控制萎凋以降低濕度並提高葉片在殺青前的可塑性。對太湖翠竹而言，此階段較短—葉片不應失去新鮮度。

* **殺青 (杀青 — shāqīng)：** 在鍋中或機械化生產線上高溫炒制。目的：鈍化酶活性，固定綠色特徵並形成香氣基礎。標準：葉片柔軟，顏色深綠，香氣純淨清新。

* **整形 (整形 — zhěngxíng)：** 決定茶葉「竹葉」外觀的關鍵步驟。葉片被壓制和整理，獲得扁平、略微彎曲的形狀，形似竹葉 (扁似竹叶)。整形可手工進行或在多功能機器上以控制的壓力和溫度進行。

* **烘乾 (烘干 — hōnggān)：** 熱風將濕度降至中間水平，穩定形狀和顏色。

* **輝炒提香 (辉炒提香 — huīchǎo tíxiāng)：** 最終短時間中溫炒制。葉片獲得特有的油潤光澤 (油润) 和增強的栗香調。正是這個步驟區別了太湖翠竹與純烘乾綠茶，賦予其「溫暖」的調性。

* **精選 (精选 — jīngxuǎn)：** 去除破碎顆粒並均勻批次。

**6. 感官特徵：**

* **乾茶外觀：** 扁平如竹葉 (扁似竹叶)，略微彎曲，光滑，具油潤光澤。顏色翠綠油潤 (翠绿油润)。芽葉大小均勻，可見嫩芽。特級品—茶葉整齊「如用尺量」，光滑略有光澤 (扁平挺秀、平整光滑、稍弯曲)。

* **乾茶香氣：** 純淨清新，帶栗香 (栗香) 和淡雅花香。香氣持久，無「生青」草腥味。

* **茶湯香氣：** 高雅純淨持久 (清香持久 / 清高持久)。栗香調配以清新植物香和淡雅花香。香氣精緻—這是江南茶特有的「靜謐」之美，而非強烈「張揚」的香氣。

* **滋味：** 鮮醇爽口 (鲜醇爽口)。茶體輕盈優雅—無厚重和粗糙感。甘甜不突兀，帶柔和回甘。整體印象—「清雅甘醇」(清雅甘醇)（清淨、優雅、微甜醇厚）—江南風格的精髓。

* **湯色：** 清澈明亮 (清澈明亮)，嫩綠或綠黃色調。

* **葉底（沖泡後茶葉）：** 嫩綠勻整 (嫩绿匀整)，有彈性且「鮮活」。芽葉完全展開，展現芽葉的完整性。在玻璃杯中沖泡時的特殊視覺效果：芽葉垂直立起，緩慢搖擺—「如風中竹林」(似群山竹林)。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 江蘇優質綠茶的典型含量—18-25%。主要兒茶素—EGCG、EGC、ECG。適中的多酚含量（低於南方茶葉）確保了柔和性和無過度澀味。

* **氨基酸：** 由於使用福安大白和福鼎大毫品種以及有利的生長條件（散射光、濕度）而含量較高。L-茶氨酸是主要氨基酸，負責鮮味 (鲜) 和溫和放鬆效果。

* **生物鹼：** 咖啡因—適中含量（約3-4%），提供溫和提神效果。

* **維生素：** 維生素C（抗壞血酸）、B群維生素、維生素E。

* **礦物質：** 鉀、鎂、磷、錳、氟、鋅（適量）。

* **精油：** 揮發性化合物複合體，包括呋喃和吡嗪衍生物（在最終炒制時形成），負責栗子堅果調性。

**8. 健康功效：**

* **抗氧化保護：** 兒茶素（EGCG）中和自由基，延緩細胞氧化損傷。定期飲用綠茶與降低氧化應激標記物相關。

* **溫和提神和認知支持：** 咖啡因和L-茶氨酸的結合提供「溫和」清醒—無焦慮和急劇下降。茶氨酸有助於改善專注力和記憶力。

* **心血管系統支持：** 多酚有助於優化血脂譜並維持血管彈性。

* **抗菌和抗炎作用：** 兒茶素抑制多種病原菌生長，包括致齲鏈球菌。茶有助於保持口氣清新。

* **紫外線防護：** 綠茶抗氧化劑減少紫外線對皮膚的損傷，延緩光老化。

* **消化支持：** 適中的多酚含量刺激蠕動和分泌而不刺激黏膜。

* **代謝支持：** 咖啡因和兒茶素促進脂肪酸動員和產熱—對維持健康體重有適度貢獻。

* **重要提醒：** 對咖啡因敏感、胃腸道疾病急性期患者以及服用抗凝劑者建議適量飲用。

**9. 沖泡方法：**

* **水溫：** 75-85°C。特級品 (嫩芽) —75-80°C；一、二級—最高85°C。

* **茶量：** 3克配150-200毫升水。

* **茶具：** 高玻璃杯 (玻璃杯) —強烈建議初次品嚐使用：展開芽葉垂直立起的「舞蹈」—「杯中竹林」—是這款茶的主要美學享受。瓷蓋碗—用於更精確控制萃取和保持香氣。

* **沖泡過程：**
  1. 用開水溫杯或蓋碗，倒掉。
  2. 投入茶葉。
  3. 注入適溫水至1/3容量，等待15-20秒—芽葉開始展開。
  4. 續水至3/4容量。
  5. 浸泡：第一泡1.5-2分鐘。觀察茶葉上升、下沉、垂直立起—這是儀式的一部分。
  6. 飲用時不要喝完：留下1/3茶湯並加入新水。可續水2-3次。
  7. 蓋碗沖泡：第一泡20秒，後續每泡增加5-10秒。共3-5泡。

**10. 儲存：**

* **溫度：** 最佳0-5°C（冰箱，密封包裝）。室溫儲存—在陰涼避光處（不超過10°C）。

* **容器：** 真空鋁箔袋、密封鐵罐。必須避光和異味。

* **茶葉天敵：** 光線、濕氣、高溫、氧氣、異味。太湖翠竹是精緻香氣的嫩茶；儲存條件不當時，比粗老品種更快失去「綠色」新鮮度和油潤光澤。

* **最佳品嚐期：** 生產日期後6-12個月。春茶最好在同年秋季前飲完。

**11. 價格與仿冒：**

* **價格範圍：** 中等及中等以上。春茶特級 (明前特级) —每500克1000-1500元人民幣（2009-2011年數據），與優質江南綠茶同級，但低於龍井或碧螺春頂級產品。普通一、二級—價格便宜許多。

* **避免仿冒方法：**
  * 向有權使用地理標誌的斗山地區茶園或錫北鎮茶業協會 (锡北镇茶业协会) 專賣店購買。
  * 評估外形：正宗太湖翠竹—扁平「竹葉狀」，略微彎曲，光滑。粗糙捲曲或破碎茶葉是仿冒或低質量的標誌。
  * 檢查顏色：翠綠油潤。暗淡、發黃或不均勻顏色表示陳舊原料。
  * 評估香氣：純淨清新，帶栗香。尖銳「焦糊」或「腥味」是工藝問題的指標。
  * 沖泡觀察：正宗茶在杯中展開成「竹林」—芽葉垂直立起，茶湯清澈明亮。

**12. 有趣事實：**

* 「太湖翠竹」名稱於1989年確定，此前有兩個「工作」版本：「斗山青峰」(斗山青峰，「斗山綠峰」) 和「梁溪浪尖」(梁溪浪尖，「梁溪波尖」，其中「梁溪」是無錫歷史名稱)。最終選擇了同時指向太湖和茶葉竹子視覺形象的名稱—簡潔而富詩意。

* 生產500克乾太湖翠竹需要超過30,000個嫩芽—每個都是手工採摘。

* 斗山地區是中國最古老的自然保護區之一：禁止狩獵、捕魚和砍伐的禁令在康熙皇帝 (康熙, 1661-1722年在位) 時期就已實施，即300多年前。這種多世紀的保護制度保存了獨特的森林覆蓋，為茶園創造了天然生態系統。

* 太湖翠竹是少數專門為玻璃杯沖泡視覺效果而創制的綠茶之一。芽葉展開時垂直立起並輕微搖擺，創造微型竹林的錯覺 (似群山竹林) —這不是副作用，而是整形的有意結果。

* 儘管年輕（不到40年），太湖翠竹創造了「陸羽杯」競賽記錄—連續八次獲勝，使其成為江蘇省最多獲獎的綠茶。

**13. 與其他綠茶比較：**

* **西湖龍井 (西湖龙井, Xīhú Lóngjǐng)：** 扁形綠茶的標準。龍井完全扁平，具明顯「豆栗香」和油潤茶體。太湖翠竹也是扁平的，但有特色彎曲（「竹葉」），茶體更精緻輕盈，炒制味較淡，更具「綠色」特徵。

* **碧螺春 (碧螺春, Bìluóchūn)：** 來自蘇州的著名捲曲綠茶—與無錫相鄰的太湖沿岸地區。碧螺春螺旋捲曲，具花果香型（因與果樹間作）。太湖翠竹扁平，帶栗香調，較少「果味」，更具「竹韻」特色。

* **江蘇陽羨雪芽 (阳羡雪芽, Yángxiàn Xuěyá)：** 江蘇另一名茶—來自宜興（Yíxīng，也屬無錫）。陽羨雪芽捲曲，茸毛較多，更具「山地」特色（宜興海拔高於斗山）。太湖翠竹是平原丘陵型，更具「湖泊」特色，更柔和細膩。

* **南京雨花茶 (南京雨花茶, Nánjīng Yǔhuā Chá)：** 來自江蘇省會的針形綠茶。雨花茶緊直如針，帶松香；太湖翠竹扁平如「竹葉」，帶栗香。兩種完全不同的風格，統一於江蘇茶學派。

* **安吉白茶 (安吉白茶, Ānjí Bái Chá)：** 來自浙江的扁形綠茶，氨基酸含量創紀錄。太湖翠竹使用同樣的安吉白茶品種作為成分之一，但整體風格更柔和，更具「湖泊」特色，不如原產安吉白茶那樣「氨基酸甜味」突出。

**結語：**

太湖翠竹是懷著對美的自覺追求而創制的茶葉。其「竹葉」形狀不是工藝的偶然，而是藝術構思，將每次沖泡變成小型表演：玻璃杯成為微型竹林的窗口，嫩綠茶湯成為其湖泊。滋味和香氣同樣精緻：清新純淨、精緻栗香、柔和回甘—無多餘，無張揚。這是江南的精髓—不自我堅持的美，而是簡單的存在。對於那些在茶中重視優雅而非力量，願意花一分鐘觀看杯中竹林綻放的人來說，這是理想的選擇。
