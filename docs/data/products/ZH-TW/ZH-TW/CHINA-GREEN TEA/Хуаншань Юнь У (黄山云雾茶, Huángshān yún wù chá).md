# 黃山雲霧茶 (黄山云雾茶, Huángshān yún wù chá)

**黃山雲霧茶 (黄山云雾茶, Huángshān yún wù chá)**
![Хуаншань Юнь У (黄山云雾茶, Huángshān yún wù chá)](images/Хуаншань%20Юнь%20У%20(黄山云雾茶,%20Huángshān%20Yún%20Wù%20Chá).png)

![Хуаншань Юнь У (黄山云雾茶, Huángshān yún wù chá)-1](images/Хуаншань%20Юнь%20У%20(黄山云雾茶,%20Huángshān%20Yún%20Wù%20Chá)-1.png)

黃山雲霧茶是一款經典的高山綠茶，擁有數百年歷史，誕生於黃山神秘的雲霧之中。此茶被認為是著名黃山毛峰的歷史前身，屬於廣義的「雲霧茶」(云雾茶, yúnwù chá) 類別——即在雲霧繚繞的高山上種植的綠茶。其溫和清香的特質帶有草本堅果風味，直接反映了黃山獨特的風土條件，這裡的茶樹一年中大部分時間都被雲霧環抱。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。屬於雲霧 (云雾, yúnwù——「雲霧」) 類別——在持續雲霧環境中種植的高山綠茶特殊類別。
* **類別：** 中國名茶 (中国名茶, Zhōngguó míngchá)。安徽省歷史名茶。
* **產地：** 中國安徽省 (安徽省, Ānhuī shěng)，黃山山脈 (黄山, Huángshān——「黃山」)，主要在歙縣 (歙县, Shèxiàn)。黃山是聯合國教科文組織世界遺產 (1990年列入，雙重地位：自然與文化遺產)。
* **地理座標：** 黃山山脈位於大約北緯30°01′–30°18′，東經118°01′–118°17′之間。

**2. 歷史與文化意義：**

* **歷史：** 黃山地區的茶葉栽培歷史悠久。據《徽州府志》(《徽州府志》, Huīzhōu fǔzhì) 記載，黃山地區的茶葉種植始於北宋 (宋朝, Sòng cháo) 嘉祐年間 (嘉祐, Jiāyòu, 1056–1063年)，並在明朝 (明朝, Míng cháo) 隆慶年間 (隆庆, Lóngqìng, 1567–1572年) 得到顯著發展。明代文人許楚 (许楚, Xǔ Chǔ) 在其《黃山遊記》中留下了值得注意的記述：「蓮花庵石隙間種茶，清香冷韻，襲人斷腭，名黃山雲霧茶」。

    正是這種「黃山雲霧茶」被認為是著名黃山毛峰 (黄山毛峰, Huángshān Máofēng) 的祖先，毛峰於1875年由歙縣茶商謝正安 (谢正安, Xiè Zhènɡ'ān，字靜和 静和) 創制。謝正安創立了「謝裕大」(谢裕大, Xiè Yùdà) 茶號，為其新茶選用了比傳統雲霧茶更嫩的原料——「白毫似峰」——並命名為「毛峰」(毛峰，「毛茸茸的山峰」)。因此，雲霧茶代表了黃山製茶更古老的「農民」傳統，而毛峰則是其精緻的商業發展。

    清代作者江澄雲 (江澄云, Jiāng Chéngyún) 在《素壺便錄》(《素壶便录》, Sùhú biànlù) 中如此描述此茶：「黃山有雲霧茶，產於絕頂，煙雲繚繞，露霧滋培，其樹有數百年者，氣味恬雅，芳香撲鼻，絕無俗味，當為茶品中第一」。

* **名稱：**
    * **黃山 (黄山, Huángshān)** ——「黃山」，安徽省傳奇山脈，以神話中的黃帝 (黄帝, Huángdì) 命名，據傳說他曾在此煉丹並獲得永生。
    * **雲霧 (云雾, Yún Wù)** ——「雲霧」。指其生長的特有氣候條件：黃山峰頂一年中大部分時間被濃雲密霧籠罩。全名為「黃山雲霧茶」。

* **文化意義：** 黃山被詩人和畫家歌頌了千年，是中國文化的聖地之一。其奇幻的花崗岩峰、奇松和永恆的雲霧是徽州畫派和詩歌的經典主題。誕生於這些神秘景觀中的雲霧茶，被視為天地和諧、自然與精神統一的物質體現。黃山也是眾多名茶的故鄉：除了雲霧茶和毛峰，這裡還生產太平猴魁 (太平猴魁, Tàipíng Hóukuí) 和祁門紅茶 (祁门红茶, Qímén Hóngchá)。黃山因此被稱為「中國名茶之鄉」——中國十大名茶中有三種來自這個地區。

**3. 植物學描述與原料：**

* **品種/栽培種：** 使用當地小葉種茶樹 *Camellia sinensis* var. *sinensis* ——祁門小葉種 (祁门小叶种, Qímén Xiǎoyèzhǒng)，字面意思是「祁門小葉種」。這是一種低矮灌木，高約1公尺，樹冠緊湊，葉片小而橢圓，革質，長5–7公分。芽頭覆蓋豐富的銀白色茸毛 (白毫, báiháo)。
* **採摘：** 手工採摘在四月下半月進行，當嫩梢達到一定成熟度時。雲霧茶的採摘標準是一芽三四葉 (一芽三四叶, yī yá sān sì yè)，這與毛峰更早更嫩的採摘標準 (一芽一叶, yī yá yī yè) 有顯著不同。正是使用更成熟葉片賦予了雲霧茶特有的飽滿口感和明顯的草本堅果風味。
* **原料要求：** 選用健康、完整、無損傷的深綠色葉片，無霜凍、病害或機械損傷痕跡。芽頭具有明顯的銀白色茸毛。

**4. 風土條件與栽培特點：**

* **產區：** 安徽省黃山山脈是中國東部最美麗、生態最純淨的地區之一。黃山佔地約1200平方公裡，包括72座山峰，最高峰蓮花峰 (莲花峰, Liánhuā Fēng——「蓮花峰」) 海拔1864公尺。
* **種植海拔：** 茶園位於海拔600–1000公尺。
* **土壤：** 以黃棕壤 (黄棕壤, huáng zōng rǎng) 為主，形成於花崗岩基礎上。土壤呈酸性 (pH 4.5–5.5)，排水良好，富含有機物和礦物元素。
* **氣候：** 溫帶氣候，春季涼爽漫長。年平均氣溫+11…+15°C。年降水量約1800毫米。相對濕度穩定超過80%。
* **風土關鍵特徵——雲霧：** 黃山一年中有200–250天 (約70%的日子) 被雲霧籠罩。這種「雲霧環境」是茶葉品質的決定性因素：透過霧氣的散射光線減緩茶樹生長並緩和光合作用，導致兒茶素 (苦澀來源) 含量降低，同時積累游離胺基酸，特別是L-茶胺酸，負責甜味、鮮味和口感「深度」。正是胺基酸與兒茶素的高比例使雲霧茶如此溫和甘甜。
* **生態栽培：** 黃山茶園被松樹、竹子和樟樹的原始森林環繞。傳統栽培不使用農藥和除草劑，使用有機肥料 (發酵糞肥) 和人工除草。豐富的山坡生態系統——擁有多樣的動植物群——是害蟲的天然調節器。

**5. 製作工藝：**

黃山雲霧茶的製作遵循經典綠茶工藝，具有徽州茶學派的特色元素：

1. **採摘 (采摘, cǎizhāi)：** 手工採摘成熟嫩梢——一芽三四葉。在四月中下旬進行。
2. **萎凋 (萎凋, wěidiāo)：** 將採摘的葉片薄攤在陰涼處或通風良好的室內6–8小時，失水約15%。葉片變得柔軟，適合進一步加工。
3. **殺青 (杀青, shāqīng)：** 在180–200°C的熱鐵鍋 (铁锅, tiěguō) 中快速手工炒製。酶活性失活阻止氧化，固定葉片綠色和新鮮「活」香。這是關鍵步驟，需要師傅精確控制溫度和時間。
4. **揉捻 (揉捻, róuniǎn)：** 將葉片塑造成特有的輕微捲曲條索形狀，通常手工進行，用手掌輕壓。揉捻破壞細胞壁，釋放部分汁液，豐富茶湯口感。
5. **多階段乾燥 (烘干, hōnggān)：** 分幾個階段進行。初烘——用約90°C熱風至含水量約20%。然後葉片可能輕微壓製以緊實結構。最終烘乾——在較低溫度 (~60°C) 下，常使用傳統炭火烘焙 (炭火烘焙, tànhuǒ hōngbèi)，至殘餘水分不超過6%。炭火烘乾賦予茶葉特有的輕微焙火香。
6. **分級 (分级, fēnjí)：** 成品茶過篩，去除碎葉和茶梗，按品質分為：特級 (特级, tèjí)、一級 (一级, yījí) 等等級。

**6. 感官特徵：**

* **乾茶外形：** 輕微捲曲的細長條索 (長達5–7公分)，橄欖綠或灰綠色，芽頭有明顯的銀白色茸毛。形狀有時詩意地描述為「毫鋒」(毫锋, háo fēng)。葉片完整，均勻，無碎片。
* **乾茶香氣：** 清新草本香，帶有淡淡花香和微妙的焙火堅果香，這是炭火烘乾的特徵。
* **茶湯香氣：** 清雅新鮮，以草本花香調為主。第二、三泡中出現淡淡的炒籽、栗子或玉米香——徽州製茶工藝的標誌。
* **滋味：** 溫和清香，略帶甜味。主要風味輪廓——草本堅果味，帶有榛子、烤玉米和新鮮綠葉的調性。澀味適中或較弱，無苦味。茶湯中等厚度，質地順滑。回甘 (回甘, huígān) 清淨爽口，帶有溫和的回甜。
* **湯色：** 透明，淡黃色帶明顯綠色調，有時比作淡香檳色。明亮清澈。
* **葉底 (沖泡後葉片)：** 完整柔軟的葉片和芽頭，呈鮮綠或橄欖色，良好保持結構。葉片均勻展開，展現原料品質。

**7. 化學成分：**

* **多酚類 (兒茶素)：** 總多酚含量為乾重的18–25%。主要兒茶素是EGCG (表沒食子兒茶素沒食子酸酯)，強效抗氧化劑。由於高山雲霧風土條件，雲霧茶的兒茶素含量比平地綠茶略低，這造就了其溫和口感和較低澀味。
* **胺基酸：** L-茶胺酸含量較高——約為乾重的2–3%，顯著高於綠茶平均水準。高濃度L-茶胺酸是散射光照和雲霧環境中緩慢生長的直接結果。正是L-茶胺酸形成了甘甜的「鮮味」口感。
* **生物鹼：** 咖啡因——乾重的2–3%。由於使用較成熟葉片 (相比毛峰)，咖啡因含量可能適中。可可鹼和茶鹼——微量。
* **維生素：** 抗壞血酸 (C)、核黃素 (B₂)、硫胺素 (B₁)。
* **礦物質：** 鉀、氟、錳、鋅。花崗岩土壤基礎使茶葉富含微量元素。
* **葉綠素：** 由於殺青工藝含量較高，賦予葉片和茶湯特有的綠色調。
* **芳香化合物：** 順-3-己烯醇及其酯類 (草本香調)、2-乙醯吡咯和糠醛 (炭火烘乾的焙火香)、芳樟醇及其氧化物 (淡淡花香)。

**8. 健康功效：**

* **抗氧化作用：** 綠茶兒茶素 (EGCG) 有效中和自由基，保護細胞免受氧化損傷。
* **溫和提神效果：** 適中的咖啡因含量結合高水準L-茶胺酸，提供「平靜清醒」狀態——穩定專注而不緊張。這種協同效應被描述為「放鬆的專注」，是高山綠茶最珍貴的品質之一。
* **改善消化：** 多酚類刺激消化酶分泌，正常化新陳代謝，促進脂肪分解。
* **支持心血管系統：** 定期飲用綠茶與降低低密度脂蛋白膽固醇、改善血管彈性和正常化血壓相關。
* **增強免疫力：** 維生素C和兒茶素支持機體防禦功能，具有抗病毒活性。
* **認知功能：** L-茶胺酸促進大腦α波產生，改善專注力、學習能力和創造性思維。
* **口腔健康支持：** 氟和兒茶素具有抗菌作用，抑制引起蛀牙的細菌生長。

**9. 沖泡方法：**

* **水溫：** 75–85°C。沸水會損傷嫩葉並提取過多澀味。
* **用茶量：** 3–5克配150–200毫升水。
* **茶具：** 白瓷蓋碗 (盖碗, gàiwǎn) 是經典選擇，可以評估香氣並控制浸泡時間。玻璃茶壺或杯子也適合——透過玻璃可以觀察茶葉在水中的「舞蹈」。瓷茶壺——適合歐式沖泡。
* **沖泡過程：**
    1. 用沸水溫壺並倒出。
    2. 投入乾茶。
    3. 注入適溫水並立即倒出第一泡 (洗茶, 洗茶, xǐ chá)——這會喚醒葉片並去除茶塵。
    4. 再次注水，浸泡30–60秒 (第一泡)。
    5. 將茶湯倒入茶杯。
    6. 後續沖泡——每泡增加10–15秒。優質黃山雲霧茶可沖泡4–6次，每泡都展現新的風味層次。

**10. 儲存：**

* **溫度：** 最佳——冰箱中0–5°C，密封包裝，與強烈氣味食品分開。室溫儲存也可接受，在涼爽環境中 (不超過20°C)。
* **容器：** 密封、不透明——鐵罐、真空鋁箔袋、帶緊蓋的瓷罐。
* **茶葉天敵：** 濕氣、光線、氧氣、異味、高溫。
* **保存期限：** 在適當條件下12–18個月。綠茶不會隨時間改善；建議在採摘後一年內飲用以獲得最佳新鮮度。

**11. 價格與假冒：**

黃山雲霧茶的價格因等級 (特級、一級、二級)、採摘時間和加工方法 (手工vs.機械) 而異。來自主產區的高等級手工茶價格顯著較高。雲霧茶通常比高等級黃山毛峰便宜，這使其成為了解黃山茶傳統的有吸引力選擇。

**如何避免假冒：**

* **外觀：** 葉片應完整，輕微捲曲，有可見的銀白色茸毛。破碎、有粉塵的茶是低品質或替代品的標誌。
* **香氣：** 清新草本堅果香，無異味 (霉味、魚腥味)。人工染色茶可能有不自然的鮮綠色。
* **茶湯：** 透明，淡黃綠色。混濁、深色茶湯是警告信號。
* **產地：** 經常有其他地區的便宜綠茶冒充黃山雲霧茶。從可靠供應商購買，注意具體產地標示 (歙縣、黃山)。
* **價格：** 此類茶葉的可疑低價應引起警惕。

**12. 有趣事實：**

* 黃山雲霧茶被認為是黃山毛峰的歷史祖先。據茶史學家說，「毛峰本質上是雲霧茶的精緻版本，通過選用更嫩的原料和改進炒製工藝創造而成」。
* 「雲霧」之名不僅是詩意形象：黃山一年中有200–250天被雲霧覆蓋。當地有諺語：「晴時早晚遍地霧，陰雨成天滿山雲」。
* 黃山是中國唯一產生三種「十大名茶」的地方：毛峰、太平猴魁和祁門紅茶。雲霧茶作為它們的歷史前身在該地區茶譜系中佔有榮譽地位。
* 清代美食家江澄雲在18世紀稱黃山雲霧茶為「茶品中第一」(当为茶品中第一, dāng wéi chápin zhōng dì yī)——考慮到與龍井和碧螺春的競爭，這在當時是獨特的評價。
* 毛峰創制者茶商謝正安在上海銷售黃山茶非常成功，以至於其茶號所在街道被命名為漕溪路 (漕溪路)——以其歙縣老家漕溪村命名。漕溪路至今仍存在於上海。

**13. 與其他綠茶比較：**

* **黃山毛峰 (黄山毛峰, Huángshān Máofēng)：** 最親近的親戚。毛峰使用更嫩的原料 (一芽一葉)，產生更輕盈、精緻的口感和明顯花香。雲霧茶用其成熟葉片，更飽滿，有更明顯的堅果調和「鄉土」特色。毛峰更貴更有聲望；雲霧茶更深沉更「樸實」。
* **廬山雲霧茶 (庐山云雾茶, Lúshān Yún Wù Chá)：** 來自江西省 (江西, Jiāngxī) 的另一種著名雲霧茶。產於廬山 (庐山)——也是聯合國教科文組織世界遺產。與黃山雲霧茶相比，廬山版本通常有更明顯的草本調和更濃厚的茶湯質地，這與不同土壤類型和茶樹品種有關。
* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 來自河南省 (河南) 的綠茶，也有明顯的銀白色茸毛。比雲霧茶更細小尖銳；口感帶有特有的新鮮「綠色」澀味和高香氣。
* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 黃山的另一種偉大綠茶，但特色截然不同：大而扁平的葉片，濃厚「油潤」口感帶有蘭花香。雲霧茶是其更謙遜但同樣優秀的山鄰。

**14. 可能的禁忌：**

* 對綠茶成分的個體不耐受。
* 對咖啡因過敏：可能出現失眠、心動過速、焦慮。
* 懷孕和哺乳期：由於咖啡因含量建議限制攝入。
* 胃腸道疾病急性期 (胃炎、潰瘍病)：不應空腹飲用濃茶。
* 嚴重心血管疾病：建議定期飲用前諮詢醫生。
* 不建議用茶送服藥物或與酒精同飲。

**結語：**

黃山雲霧茶——帶有黃山靈魂的茶，是自然與時間創造完美事物的活見證。誕生於雲中，被霧露滋養，這款茶在每一口中都承載著山間空氣的溫和甘甜、古老森林的寧靜深度和炭火的淡淡堅果溫暖。雲霧茶不僅是飲品，更是與數百年徽州茶文化的聯繫，其根源可追溯至宋代。對於珍視有歷史和特色茶葉的人來說，黃山雲霧茶是進入安徽省偉大茶葉世界的絕佳入口：價格比毛峰更親民，但在深度和獨特性上毫不遜色。
