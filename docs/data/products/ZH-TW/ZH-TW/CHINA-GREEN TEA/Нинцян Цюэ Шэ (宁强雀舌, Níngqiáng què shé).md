# 寧強雀舌 (宁强雀舌, Níngqiáng què shé)

**寧強雀舌 (宁强雀舌, Níngqiáng què shé)**

![Нинцян Цюэ Шэ (宁强雀舌, Níngqiáng què shé)](images/Нинцян%20Цюэ%20Шэ%20(宁强雀舌,%20Níngqiáng%20Què%20Shé).png)

![Нинцян Цюэ Шэ (宁强雀舌, Níngqiáng què shé)-1](images/Нинцян%20Цюэ%20Шэ%20(宁强雀舌,%20Níngqiáng%20Què%20Shé)-1.png)

寧強雀舌 (宁强雀舌, Níngqiáng què shé) —「寧強縣的雀舌」— 綠茶，產自陝西省漢中市寧強縣 (宁强县, Níngqiáng Xiàn)，位於**漢江源頭** (汉江源头, Hànjiāng Yuántóu) — 長江最大支流，曾賦名於漢朝、漢族和漢字。寧強的茶葉傳統可追溯至唐代：陸羽在《茶經》中記載：「梁州生褒城、金牛二縣山谷」—「梁州褒城、金牛二縣山谷中產茶」。金牛 (金牛，「金牛」) — 寧強古名，與傳說中的「金牛道」(金牛道, Jīnniú Dào) 相關 — 連接關中（西安）與四川穿越秦嶺的最古老道路。此茶創製於1992年，**1996年列入「國家特需用茶」** (国家特需用茶, guójiā tèxū yòng chá) — 即供應**中南海** (中南海) — 中華人民共和國最高領導層駐地。2002年寧強成為**陝西省首個通過國家有機茶認證的縣**，縣域範圍內屬於**聯合國教科文組織生物圈保護區** (联合国世界人与自然生物圈保护区)。土壤含**硒 (Se) 0.653–3.853 ppm** — 比漢中平均值高1.5倍。

自2007年起，寧強雀舌歸入區域傘型品牌「漢中仙毫」(汉中仙毫，「漢中神仙毫毛」)，但保持自身工藝和香氣特色：若「漢中仙毫」為扁平形帶栗香，則「寧強雀舌」為**「雀舌形」** (雀舌形) 具**「高長馥郁」香氣** (高长馥郁) 且完全無澀味。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵。形狀 —**「雀舌形」** (雀舌形, quèshé xíng)：略扁、挺秀、帶銀毫 — 芽葉形似雀鳥吐出的「舌頭」。區域品牌「漢中仙毫」(汉中仙毫, 地理標誌 2007) 子類別，保持自身特色、工藝和名稱。

* **類別：** 地理標誌產品「漢中仙毫」(国家地理标志产品, 2007)。**「國家特需用茶」** (国家特需用茶, 1996 — 供應中南海)。陝西首個有機茶 (国家有机茶认证, 2002)。國家級有機茶標準化示範區 (国家级有机茶标准化示范区, 2003)。全國農業科技博覽會「后稷」金像獎 (中国农业科技博览会后稷金像奖, 1995)。西安國際貿易洽談會金獎 (西交会金奖, 2002)。聯合國教科文組織生物圈保護區。富硒茶。

* **產地：** 中國陝西省 (陕西省)，漢中市 (汉中市)，寧強縣 (宁强县)。核心產區：漢源鎮 (汉源镇，「漢江源頭」) 和高寨子鎮 (高寨子镇)，海拔800–1200公尺，位於漢江和嘉陵江源頭流域。

* **地理座標：** ~32°45′–33°15′ 北緯，105°55′–106°35′ 東經。陝西省最西北茶區，秦嶺與巴山交匯處。

**2. 歷史與文化意義：**

* **歷史：**

寧強是陝西省最古老的茶區之一。陸羽在《茶經》(八世紀) 將梁州 (梁州，漢中古名) 列入產茶地區，具體提及「褒城、金牛二縣山谷」(梁州生褒城、金牛二县山谷)。金牛 — 寧強古名，與傳說中的「金牛道」(金牛道) 相關 — 中國最古老道路之一，自戰國時期起連接關中與四川的貿易通道。因此寧強茶葉生產於這條已有2300多年商品（包括茶葉）運輸歷史的古道上。

現代「寧強雀舌」由縣茶葉技術團隊於1992年基於當地群體品種創製。1993年通過農業部專家評審。1995年獲全國農業科技博覽會「后稷金像獎」(后稷金像奖)。**1996年** — 列入「国家特需用茶」— 國家特需用茶：寧強雀舌供應**中南海** (中南海) — 中華人民共和國最高領導層駐地。這是中國少數幾個擁有確認「國家領導人用茶」地位的茶葉之一。2002年 — 西安國際貿易洽談會金獎 (西交会金奖)。同年2002年寧強成為**陝西省首個**獲得國家有機茶認證的縣，2003年成為**國家級有機茶標準化示範區**。2005年寧強雀舌與午子仙毫 (午子仙毫) 和定軍茗眉 (定军茗眉) 組成「漢中三大名茶」(汉中三大名茶)，2007年三個品牌統一於傘型品牌「漢中仙毫」(汉中仙毫) 之下。2017年「漢中仙毫」入選中國茶葉區域公用品牌價值50強 (中国茶叶区域公用品牌价值50强)。

* **名稱：** 宁强 (Níngqiáng) — 縣名；雀舌 (Què Shé) —「雀舌」— 描述成品茶形狀：小巧、略扁、挺秀的芽葉，形似雀鳥吐出的舌頭。「雀舌」是綠茶經典類別之一，與「毛尖」和「毛峰」並列。

* **文化意義：** 寧強 —「漢江源頭」(汉江源头)。漢江 (汉江) — 長江最大支流，長1577公裡，曾賦名於**漢朝** (汉朝, 西元前202年 — 西元220年)、**漢族** (汉族) 和**漢字** (汉字)。來自漢江源頭的茶葉 — 是「漢文明源頭」的茶。此外，寧強位於「金牛道」— 連接西安和成都穿越秦嶺的最古老交通幹線。品飲寧強雀舌 — 意味著同時觸及陸羽《茶經》、「漢」族認同的源頭和「金牛道」2300年歷史。

**3. 植物學描述與原料：**

* **品種/栽培種：** 當地群體品種 (本地群体种)，適應秦嶺-巴山交匯處北亞熱帶條件。小葉種，抗寒，芽葉銀毫豐富。部分茶園補植福鼎大白茶無性系以增加毫毛。

* **採摘：** 春茶 — 清明前後 (清明，四月初)。標準 — 一芽一葉初展 (一芽一叶初展)。芽葉須完整、挺秀、無缺陷 — 以形成特有的「雀舌」形狀。

* **原料生化成分：** 茶多酚 ≥24.73%，胺基酸 3.43%，土壤富含 Se + Zn。

**4. 風土條件與栽培特色：**

寧強位於兩大山系交匯處 — 北有秦嶺 (秦岭)，南有巴山 (巴山)，形成深邃山谷，溪流潺潺，雲霧繚繞。

* **海拔：** 800–1200公尺。核心區 — 1000±200公尺，位於有泉水溪流的山谷中。

* **氣候：** 北亞熱帶。年均溫 12.9°C（中國茶區中最低之一 —「寒冷」茶）。降雨量 900–1100毫米。日夜溫差 >10°C — 顯著，促進胺基酸積累。雲霧濕度 85%。秦嶺山脈形成天然屏障，阻擋北方寒風，同時允許太平洋濕潤氣流通過 —「天然溫室」環境。

* **土壤：** 花崗岩發育棕壤土 (花岗岩发育棕壤土, pH 4.8–5.5)。有機質 >2.0%。**硒 (Se) — 0.653–3.853 ppm** — 比漢中平均值高1.5倍。這是陝西省硒含量最高的茶園土壤之一。鋅含量也偏高。

* **生態環境：** 森林覆蓋率 >65%。縣域範圍屬於**聯合國教科文組織生物圈保護區** (联合国世界人与自然生物圈保护区)。90%以上茶葉產品產自生態核心區。自2002年（有機認證）起禁用化學農藥和肥料。秦嶺冰川溪流提供最純淨的灌溉水源。

**5. 製作工藝：**

寧強雀舌工藝結合「机械提效、手工定魂」—「機械化提高效率，手工賦予靈魂」。關鍵步驟：

* **攤放 (摊放, tān fàng)：** 室溫下4–6小時。失水約15%。發展香氣前體物質。

* **殺青 (杀青, shāqīng)：** 斜鍋炒制，溫度160–180°C。快速 — 保持「綠色」鮮爽和香氣。採用拋 (抛) 和悶 (闷) 交替 — 經典陝西技法。

* **做形「雀舌」(做形, zuòxíng)：** 關鍵步驟。用手掌將芽葉輕壓成特有「舌形」— 挺秀、略扁、尖端細緻。手法 — 輕柔「捺壓」(捺, nà)，不用力過度以免壓碎葉片和破壞銀毫。鍋溫降至80–90°C。此步驟是寧強雀舌的「靈魂」，也是與扁平「漢中仙毫」的主要區別：「舌形」保持立體感和「透氣性」，而扁平「仙毫」完全壓平。

* **烘乾 (烘干, hōnggān)：** 兩階段 — 初烘100–110°C，足烘70–80°C至含水量≤6.5%。溫和的足烘保持「高長馥郁」香氣 (高长馥郁)，這是寧強的特色。

* **現代元素：** 光譜分析儀控制硒和茶多酚含量；數位恆溫器控制鍋溫。但「舌形」塑造仍完全手工操作。

**6. 感官特徵：**

* **乾茶外形：** 略扁挺秀、勻齊似雀舌 (微扁挺秀、匀齐似雀舌)。翠綠顯毫 (翠绿显毫)。芽葉 — 整齊，大小均勻。

* **乾茶香氣：** 「嫩香」(嫩香, nèn xiāng)。「高長馥郁」(高长馥郁, gāo cháng fùyù) — 這是與標準「漢中仙毫」的關鍵區別，後者以「栗香」(栗香) 為主。寧強雀舌更具「花香」和「蘭花香」特質。

* **茶湯香氣：** 清香 (清香)，明顯的花果香調。冷卻後 —「蜜香」韻味。香氣持久性 — 冷杯留香3分鐘以上。

* **滋味：** 醇厚甘甜 (醇厚甘甜, chúnhòu gāntián)。回甘 (回甘) — 持久清純。**完全無澀味** (无涩感, wú sè gǎn) — 配方「醇厚甘甜，無澀感」— 與「漢中仙毫」和大多數其他綠茶的關鍵滋味區別。這是低茶多酚（24.73%）和高胺基酸（3.43%）平衡的結果 — 此平衡源於「寒冷」風土（12.9°C）和富硒土壤。

* **湯色：** 綠亮清澈 (绿亮清澈, lǜ liàng qīng chè)。

* **葉底（沖泡後葉片）：** 嫩綠成朵、肥厚柔軟 (嫩绿成朵、肥厚柔软)。芽葉 — 完整，未破碎，保持「舌形」。

**7. 化學成分：**

* **茶多酚 (茶多酚)：** ≥24.73% — 綠茶中等水平，確保柔和無澀味。

* **胺基酸 (氨基酸)：** 3.43% — 較高水平，源於「寒冷」風土（12.9°C）和顯著日夜溫差（>10°C）。以L-茶胺酸為主。

* **硒 (Se)：** 0.653–3.853 ppm — 比漢中平均值高1.5倍。來自花崗岩富硒土壤。

* **鋅 (Zn)：** 偏高 — 來自同樣土壤。

* **咖啡鹼 (咖啡碱)：** 綠茶標準水平。

* **維生素：** 維生素C，B群維生素。

* **茶多酚與胺基酸比值：** 約7.2 — 「柔和」北方綠茶特徵；相比之下南方茶為10–15。正是這個低比值創造了「無澀味」配方。

**8. 保健功效：**

* **富硒特性：** 硒含量高達3.853 ppm — 陝西茶葉中最高之一。硒是必需微量元素，谷胱甘肽過氧化酶 — 關鍵抗氧化酶的必需成分。定期飲用富硒茶有助維持免疫力和抗氧化防護。

* **抗氧化防護：** 茶多酚（24.73%）+ 維生素C + 硒 — 三重抗氧化系統。

* **提神效果：** 咖啡鹼與L-茶胺酸協同 — 溫和持久的清醒感，無緊張感。由於茶多酚與胺基酸低比值，效果特別溫和。

* **認知功能：** L-茶胺酸刺激大腦α波活動 —「放鬆專注」狀態。

* **消化支持：** 兒茶素溫和刺激腸胃蠕動。

* **重要提醒：** 以上功效基於一般資料，非醫療建議。不建議空腹飲用。每日用量不超過600毫升。

**9. 沖泡方法：**

* **水溫：** 75–85°C。不可用沸水 — 嫩「舌形」會被「燙傷」失去花香。高等級用75°C。

* **茶量：** 3克配150毫升（1:50比例）。

* **茶具：** 玻璃杯（推薦）— 觀賞「雀舌」在水中緩緩下沉的「舞蹈」。白瓷蓋碗 — 聚集香氣。不建議用宜興紫砂壺 — 多孔性會「吸收」細膩香氣。

* **沖泡法 —「中投法」(中投法, zhōng tóu fǎ)：**
	1. 沸水溫杯。
	2. 注入1/3的85°C水。
	3. 投茶。
	4. 搖動茶杯30秒 —「醒茶」。
	5. 注水至7/10滿。
	6. 浸泡1.5–2分鐘。

* **沖泡次數：** 3–4泡。每次增加30秒。三泡仍保持「高長馥郁」香氣 —「三泡餘韻綿長」(三泡余韵绵长)。

**10. 儲存方法：**

* **溫度：** 0–5°C（冰箱），密封包裝。寧強雀舌是嬌嫩的「芽茶」，對溫度、濕度和異味極為敏感。
* **包裝：** 密封、不透光。鋁箔 + 真空包裝最佳。
* **光線：** 完全避光 — 毫毛和葉綠素對光敏感。
* **保存期：** 建議6–8個月內飲用。新茶需「退火」7–10天。

**11. 價格與仿冒：**

寧強雀舌屬高價位茶葉，與其他「雀舌」類別相當。特級 — 600–1000元/500克；一級 — 300–600元；大眾級 — 100–300元。

* **避免仿冒方法：**
	* 標示 —「宁强雀舌」或「漢中仙毫」並註明「宁强」產地。
	* 形狀 — 挺秀「雀舌」(微扁挺秀)，非扁平或捲曲。
	* 香氣 —「高長馥郁」(高长馥郁)，無「栗香」調（標準「仙毫」特徵）。
	* 滋味 — **無澀味** (无涩感)。有澀味是其他茶冒充的徵象。

**12. 趣味知識：**

* **中南海用茶（1996）。**「国家特需用茶」— 國家特需用茶地位。寧強雀舌供應中華人民共和國最高領導層駐地 — 中國少數幾個確認「國家領導人用茶」地位的茶葉之一。

* **漢江源頭 = 「漢」文明源頭。** 漢江 — 賦名於漢朝 (汉朝)、漢族 (汉族) 和漢字 (汉字) 的河流。品飲其源頭茶葉 — 觸及聯繫14億人的「漢」族認同根源。

* **「金牛道」(金牛道)。** 穿越秦嶺的最古老道路，自戰國時期（西元前5世紀）起連接西安與成都的貿易通道。寧強（古金牛）— 此路要站。茶葉是這條已有2300年歷史路線上的商品之一。

* **聯合國教科文組織生物圈保護區。** 寧強縣域受聯合國教科文組織「人與生物圈」(MAB) 計畫保護。生物圈保護區種植的茶葉 — 稀有。

* **硒含量高達3.853 ppm。** 陝西省所有茶區中最高之一。相比：「一般」富硒茶含0.3–0.8 ppm。

* **陝西首個有機茶（2002）。** 寧強 — 省內有機認證先驅，為整個漢中設立標準。

* **「無澀味」— 品質配方。**「醇厚甘甜，無澀感」—「醇厚、甘甜、無澀味」。這是獨特平衡的結果：茶多酚24.73% / 胺基酸3.43% = 比值約7.2 — 中國綠茶中最低之一。

**13. 與其他「雀舌」及漢中茶葉比較：**

* **漢中仙毫 (汉中仙毫, Hànzhōng Xiān Háo)：** 傘型品牌，統合漢中茶葉。扁平形。「栗香」。寧強雀舌 —「舌形」(雀舌)，「高長馥郁」香氣，無澀味。寧強滋味更「花香」和「柔和」。

* **金壇雀舌 (金坛雀舌, Jīntán Què Shé)：** 江蘇。同樣「雀舌」，但來自華東，無富硒特性。香氣 —「清雅」(清雅)。寧強 — 更明顯「馥郁」和富硒。

* **定軍茗眉 (定军茗眉, Dìngjūn Míng Méi)：** 漢中，「定軍山茗眉茶」— 同屬「仙毫」品牌。形狀 — 細「眉」形。寧強 —「舌」形。兩者皆富硒，但來自漢中不同微產區。

**結語：**

寧強雀舌 —「漢江源頭的雀舌」，來自賦名整個文明的河流源頭。曾供應中南海（1996），種植於聯合國教科文組織生物圈保護區，位於「金牛道」— 穿越秦嶺的最古老道路上 — 陝西首個有機茶，中國富硒含量最高的茶葉之一（高達3.853 ppm）。其配方 —「醇厚甘甜，無澀感」— 非行銷口號，而是獨特風土的結果：年均溫12.9°C，花崗岩富硒土壤，秦嶺與巴山交匯的山谷。以85°C「中投法」沖泡 — 三泡之後餘韻仍綿長，如漢水從這些山脈流向偉大的長江。
