# 雲龍綠茶 (云龙绿茶, Yúnlóng lǜchá)

**雲龍綠茶 (云龙绿茶, Yúnlóng lǜchá)**
![Юньлун Люй Ча (云龙绿茶, Yúnlóng lǜchá)](images/Юньлун%20Люй%20Ча%20(云龙绿茶,%20Yúnlóng%20Lǜchá).png)

![Юньлун Люй Ча (云龙绿茶, Yúnlóng lǜchá)-1](images/Юньлун%20Люй%20Ча%20(云龙绿茶,%20Yúnlóng%20Lǜchá)-1.png)

雲龍綠茶是來自雲南省大理白族自治州雲龍縣的高山綠茶。採用雲南大葉種茶樹（*Camellia sinensis* var. *assamica*）的葉片，在海拔1800–2600公尺的高度種植——這是中國最高海拔的茶園之一。該產品自2016年起獲得中華人民共和國地理標誌保護。典型品質特徵：緊結壯實的茶葉帶有銀白色茸毫，淡綠色透明的茶湯，持久的炒栗子香氣，以及濃郁鮮爽的滋味和悠長的回甘。

**1. 分類與產地：:**

* **類型：** 綠茶 (绿茶, lǜchá) — 不發酵茶；氧化程度極低（少於5%）。工藝 — 鍋炒 (炒青, chǎoqīng)。
* **類別：** 中國地方綠茶；雲南大葉種綠茶 (云南大叶种绿茶, Yúnnán dàyèzhǒng lǜchá)。中華人民共和國地理標誌保護產品。
* **產地：** 中國雲南省 (云南省, Yúnnán shěng)，大理白族自治州 (大理白族自治州, Dàlǐ Báizú Zìzhìzhōu)，雲龍縣 (云龙县, Yúnlóng xiàn)。生產範圍涵蓋11個鄉鎮：諾鄧鎮 (诺邓镇, Nuòdèng zhèn)、功果橋鎮 (功果桥镇, Gōngguǒqiáo zhèn)、漕澗鎮 (漕涧镇, Cáojiàn zhèn)、白石鎮 (白石镇, Báishí zhèn)、寶豐鄉 (宝丰乡, Bǎofēng xiāng)、苗尾鄉 (苗尾乡, Miáowěi xiāng)、檢槽鄉 (检槽乡, Jiǎncáo xiāng)、長新鄉 (长新乡, Chángxīn xiāng)、關坪鄉 (关坪乡, Guānpíng xiāng)、團結鄉 (团结乡, Tuánjié xiāng) 和民建鄉 (民建乡, Mínjiàn xiāng)。
* **地理座標：** 東經98°52′–99°46′，北緯25°28′–26°23′。

**2. 歷史與文化意義：:**

* **歷史：**

雲龍縣的茶葉傳統起源相對較晚——始於20世紀初。1919年，縣長段韜 (段韬, Duàn Tāo) 從順寧縣 (顺宁, Shùnníng；現為鳳慶 (凤庆, Fèngqìng)) 購買茶籽，帶回舊州 (旧州, Jiùzhōu) 和湯澗 (汤涧, Tāngjiàn) 地區進行試種。實驗獲得成功：數百棵茶樹在山區氣候中成活。

產業化發展始於1987年，在寶豐、舊州等地區的茶園基礎上組織大規模種植；傳統手工製作方法得到統一和標準化，形成「雲龍綠茶」統一品牌。1996年，「大栗樹牌雲龍綠茶」(大栗树牌云龙绿茶, Dàlìshù pái Yúnlóng lǜchá) 在中國科技精品博覽會 (中国科技精品博览会) 上獲得金獎。

2016年，該茶獲得國家農產品地理標誌 (国家农产品地理标志) 註冊。到2024年，全縣茶園面積達到4000公頃，年產量超過3000噸。形成了多個知名品牌：「大栗樹」(大栗树, Dàlìshù)、「佬倵」(佬倵, Lǎomù)、「雲極」(云极)、「大山頭」(大山头)。茶葉成為全縣11個鄉鎮經濟發展的重要支柱。

* **名稱：**

云龍 (Yúnlóng) — 「雲龍」；地名，反映了當地山區「雲霧繚繞」的特色；綠茶 (lǜchá) — 「綠茶」。詩意的名稱恰當地傳達了來自「晴時早晚遍地霧，陰雨連天滿山雲」之地的茶葉形象。

* **文化意義：**

雲龍是大理白族自治州下轄的縣，主要居住著白族 (白族, Báizú) 人民。茶葉在這裡不僅是商品作物，更是山區社區生活方式的重要組成部分，許多社區居住在海拔2000公尺以上，歷史上以畜牧業和核桃種植為生。茶葉種植的引入為這些社區提供了穩定的收入來源，並與更廣泛的全國市場建立了聯繫。生產核心——寶豐鄉大栗樹村 (大栗树村, Dàlìshù cūn) 被認定為全國一村一品示範村 (全国一村一品示范村)。近年來發展茶葉農業旅遊：正在建設大栗樹生態茶旅莊園 (大栗树生态茶旅庄园)。

**3. 植物學描述與原料：:**

* **品種/栽培種：** 主要由雲南大葉種群體 (云南大叶种群体, Yúnnán dàyèzhǒng qúntǐ) 構成——有性系 (有性系, yǒuxìng xì) 來源。主要無性系品種：雲抗系列 (云抗系列, Yúnkàng xìliè) 和佛香系列 (佛香系列, Fóxiāng xìliè) — 無性繁殖的優良栽培種。還有當地古茶樹 (古茶树, gǔcháshù)，特別是在大山頭茶園核心區。所有栽培種均屬於 *Camellia sinensis* var. *assamica* — 雲南典型的大葉種變種。
* **採摘：** 春茶 (春茶, chūnchá) — 主要且最珍貴；清明 (清明, Qīngmíng，4月初) 後立即開始。夏茶和秋茶也有生產。
* **採摘標準：** 春茶：一芽二葉至一芽三葉 (一芽二叶至一芽三叶, yī yá èr yè zhì yī yá sān yè)。要求：整齊均勻的芽葉，芽葉比例協調。不允許魚葉 (鱼叶, yúyè)、雨水葉、對夾葉 (对夹叶, duìjiā yè)、病葉、凍傷葉或「紫芽」。
* **原料要求：** 鮮葉裝入竹簍 (竹篓, zhúlǒu) — 禁止使用塑膠袋以避免悶熱。茶多酚含量 ≥ 25.0%，氨基酸含量 ≥ 4.0%。

**4. 風土條件與栽培特色：:**

* **地形與氣候：** 雲龍縣位於瀾滄江縱谷區 (澜沧江纵谷区, Láncāngjiāng zònggǔ qū) — 湄公河上游 — 雲南西部。深谷、陡坡和七個水系（包括瀾滄江和沘江 (沘江)）形成獨特的微氣候。年平均氣溫16.2°C，相對濕度 ≥ 75%，霧日超過200天。晝夜溫差大有利於氨基酸和芳香化合物的積累。
* **種植海拔：** 1800–2600公尺 — 中國最高海拔的茶區之一。寶豐大山頭茶園核心區 — 2400–2500公尺。
* **土壤：** 黃棕壤 (黄棕壤, huáng zōng rǎng)，酸性反應（pH 4.5–6.0），土層深厚肥沃，排水良好，有機質含量高（≥ 6.01%）。土壤富含微量元素——硒（Se）和鋅（Zn）。核心生產區森林覆蓋率超過70%。核心區為水源保護區；禁止使用化學肥料和農藥。
* **特殊因子：** 從瀾滄江峽谷上升的溫暖濕潤氣流補償了高海拔的溫度降低，豐富的散射光 (漫射光, mànshè guāng) 刺激氨基酸的生物合成。因此，即使在海拔2500公尺，春茶中的氨基酸含量也能達到 ≥ 4.0%。

**5. 製作工藝：:**

雲龍綠茶是炒青綠茶 (炒青绿茶, chǎoqīng lǜchá)，製作過程中採用烘炒結合 (烘炒结合, hōng chǎo jiéhé) 的方法來固定香氣和外形。整個製程使用竹製和木製工具 — 避免茶葉與金屬接觸。製作500克特級茶需要約40,000個芽頭。

* **採摘 (采摘, cǎizhāi)：** 手工採摘，標準為一芽帶2–3葉。使用竹簍。
* **攤晾 (摊晾, tānliáng)：** 鮮葉攤放在竹篩上4–6小時，蒸發多餘水分並進行初步香氣轉化。
* **殺青 (杀青, shāqīng)：** 高溫拋炒 (高温抛炒, gāowēn pāochǎo) 法。鈍化酶活性，保持綠色並「封存」栗香。
* **揉捻 (揉捻, róuniǎn)：** 機械或手工；形成該品種特有的緊結條索。破壞細胞結構以利均勻浸出。
* **乾燥 (干燥, gānzào)：** 「炒制+烘乾」組合方法，溫度 ≤ 60°C。固定外形，確立香氣特徵 — 主要是栗香。
* **篩分 (筛分, shāifēn)：** 按大小和形狀分級，去除缺陷葉片和茶梗。

**6. 感官特徵：:**

* **乾茶外形：** 兩種主要類型。螺形 (螺形, luóxíng)：緊結厚重，螺旋狀，有光澤，綠色調 — 特級茶特徵。條形 (条形, tiáoxíng)：壯實整齊，灰綠色帶明顯銀毫 (灰绿上霜, huīlǜ shàngshuāng) — 一級和二級茶特徵。
* **乾茶香氣：** 明顯純淨，以栗香為主。高級茶帶有花果香韻。
* **茶湯香氣：** 持久的炒栗子香 (熟板栗香, shú bǎnlì xiāng) — 該品種的主要標誌。高雅持久，高級茶在前調中帶有淡雅的花果香（特別是大山頭核心區的茶）。
* **滋味：** 濃醇 (浓醇, nóngchún)，同時鮮爽 (鲜爽, xiānshuǎng)，回甘 (回甘, huígān) 明顯持久。苦澀平衡，快速轉化為甘甜。濃厚的「體感」源於高浸出物含量（≥ 44.0%）— 雲南大葉種茶的特色。
* **湯色：** 淡綠清澈明亮 (汤色淡绿清澈明亮, tāngsè dànlǜ qīngchè míngliàng)。
* **葉底（茶渣）：** 葉底完整勻整、嫩綠鮮活 (叶底完整匀整、嫩绿鲜活, yèdǐ wánzhěng yúnzhěng, nènlǜ xiānhuó)。

**7. 化學成分：:**

* **多酚類 (茶多酚, chá duōfēn)：** ≥ 25.0% — 顯著高於大多數小葉種綠茶。這是由於屬於大葉種變種 *var. assamica*，在遺傳上傾向於積累多酚類。提供強大的抗氧化作用。
* **氨基酸 (氨基酸, ānjīsuān)：** ≥ 4.0%（總量），L-茶氨酸 (茶氨酸, cháānsuān) ≥ 1.9%。高氨基酸含量同時伴隨高多酚 — 這種罕見的組合形成了特有的「濃厚+鮮爽」平衡。這是由極高海拔和豐富散射光造成的。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēijiǎn) ≥ 2.0%。可可鹼和茶鹼含量微量。
* **水浸出物 (水浸出物, shuǐjìnchūwù)：** ≥ 44.0% — 極高的指標，解釋了茶湯的濃厚體感和優秀的耐泡性。
* **維生素：** 由於 ≤ 60°C 的溫和乾燥，維生素C得到大量保存。還含有維生素B₁、B₂、E、β-胡蘿蔔素。
* **礦物質：** 鉀、錳、硒（Se）、鋅（Zn）— 源於山地土壤的地球化學特性。氟 — 有益牙齒健康。

**8. 健康功效：:**

* **強大抗氧化作用：** 多酚含量 ≥ 25% 提供綠茶中最高效的自由基清除能力之一。
* **提神醒腦效果：** 咖啡鹼（≥ 2.0%）提高注意力集中和思維清晰度；與L-茶氨酸結合，效果溫和持久，無「咖啡因崩潰」。
* **消化支持：** 多酚刺激胃腸蠕動，減少腹脹，促進油膩食物消化。
* **代謝支持：** 兒茶素加速脂肪分解，可能有助於正常化血脂譜。
* **認知功能：** L-茶氨酸（≥ 1.9%）促進放鬆而不困倦，改善記憶和注意力。
* **微量元素補充：** 天然富含硒和鋅，支持免疫系統和抗氧化防護。
* **口腔健康：** 氟抑制致齲細菌生長。
* **心血管支持：** 兒茶素和茶黃素促進血管壁彈性，定期飲用可能有助於正常化血壓。

**9. 沖泡方法：:**

* **水溫：** 80–85°C。不要使用超過85°C的沸水 — 會破壞L-茶氨酸，失去鮮爽感，產生苦味。
* **茶量：** 玻璃杯法3克配150毫升（1:50）；蓋碗功夫茶法5–7克配100–120毫升蓋碗。
* **茶具：** 玻璃杯（便於觀察葉片舒展），白瓷蓋碗 (盖碗, gàiwǎn)，小容量紫砂壺。

* **沖泡過程（玻璃杯法 — 上投法 (上投法)）：**
  1. 用熱水溫杯，倒掉。
  2. 注入80–85°C熱水至7/10滿。
  3. 投入茶葉 — 茶葉緩慢下沉並舒展。
  4. 浸泡2–3分鐘。
  5. 飲用時留1/3茶湯，續水；可續2–3次。

* **沖泡過程（蓋碗功夫茶法）：**
  1. 用沸水溫蓋碗和公道杯。
  2. 投入5–7克茶葉，聞乾茶香。
  3. 潤茶 (润茶, rùnchá)：5秒，倒掉。
  4. 第一泡：20秒，倒出。
  5. 每次增加10秒。可沖泡3–5次優質茶湯。

**10. 儲存方法：:**

* **條件：** 密封容器（真空袋、鋁箔袋、鐵罐）。避光、防潮、避熱、防異味。
* **溫度：** 最佳為冰箱冷藏，0–5°C。
* **保質期：** 最佳品質期為製作後6–12個月內。新茶建議在陰涼處放置15天以褪火氣 (褪火气, tuì huǒqì)。開封後7天內飲用完畢。

**11. 價格與防偽：:**

* **價格區間：**
  * 特級 (特级, tèjí)：600元/斤（500克）起 — 單芽（≥ 95%），螺形，濃郁栗香。
  * 一級 (一级, yījí)：300–500元 — 一芽一葉，緊結條索，鮮爽持久。
  * 二級 (二级, èrjí)：100–300元 — 一芽2–3葉，香氣純正，性價比高。
  價格因素：季節（春茶較貴）、產區（大山頭為頂級）、樹齡（古茶樹有溢價）、有機認證。

* **防偽方法：**
  * 向「大栗樹」品牌或其他註冊生產商的授權經銷商購買，確認地理標誌標識。
  * 檢查外觀：優質雲龍綠茶為緊結壯實的茶葉帶銀毫；細碎、斷裂或發黃的茶葉是次品標誌。
  * 評估香氣：品牌標誌是持久的栗香，無酸味、霉味或「魚腥味」。
  * 茶湯應透明淡綠 — 渾濁或深色表示有缺陷。
  * 聲稱「特級」但價格低於400元/斤值得懷疑。

**12. 有趣事實：:**

* 雲龍的茶葉歷史始於一位縣官的冒險實驗：1919年段韜從順寧（現鳳慶，著名雲南紅茶滇紅的故鄉）帶來茶籽。數百棵茶樹成活 — 一個世紀後，該縣年產茶葉3000噸。
* 寶豐鄉大山頭茶園 (大山头) 海拔2400–2500公尺，是中國最高海拔的商業茶園之一。相比之下：著名的臺灣高山烏龍茶種植在1600–2600公尺，而大多數雲南綠茶種植在1000–1800公尺。
* 雲龍綠茶的水浸出物達到 ≥ 44% — 這個指標更像是好普洱茶而非綠茶的特徵，解釋了多次沖泡時茶湯的卓越「耐力」。
* 該縣名為「雲龍」並非僅是比喻：當地俗語「晴時早晚遍地霧，陰雨連天滿山雲」描述了海拔2000+公尺生活的現實，茶樹真的生長在雲霧中。
* 生產由雲南農業大學茶學系 (云南农业大学茶学系) 指導 — 學術監督確保品質穩定和新農業技術標準的實施。
* 2024年，為高品質捲曲綠茶「佛香3號」系列 (佛香3号) 制定並批准了單獨的地方生產標準（大理佛香3號捲曲形綠茶加工技術規程），大理農科院、雲南省農科院茶葉研究所和寶豐「大栗樹」茶廠參與制定 — 突顯了該產業科學支持的嚴肅性。

**13. 與其他綠茶的比較：:**

* **永泰綠茶 (永泰绿茶, Yǒngtài Lǜchá)：** 福建綠茶；小葉種（*var. sinensis*），海拔500+公尺。體感較輕，栗花香溫和。雲龍綠茶因大葉種原料和極高海拔而更濃厚有力。
* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北蒸青 (蒸青) 綠茶；明顯的海草香調，高鮮味。雲龍綠茶為炒青，栗香主導，香氣特徵更「乾燥」。
* **滇綠/雲南綠茶 (滇绿, Diānlǜ)：** 雲南綠茶的總稱。雲龍綠茶以極高海拔茶園和瀾滄江峽谷獨特微氣候脫穎而出，形成平原雲南綠茶無法達到的多酚與氨基酸平衡。
* **廬山雲霧 (庐山云雾, Lúshān Yúnwù)：** 江西「雲霧」綠茶；海拔800–1200公尺，小葉種，清淡花香。雲龍綠茶高1000公尺，大葉種，更濃厚澀味，回甘明顯。
* **臺灣高山綠茶 (台灣高山綠茶)：** 相似海拔範圍，但不同栽培種（青心、TTES №12/13），不同工藝。雲龍綠茶因雲南大葉種茶樹的天性更有力更「乾燥」。

## 結語

雲龍綠茶是中國最高海拔的綠茶之一，有力證明了雲南大葉種茶樹不僅能產出偉大的普洱茶和滇紅，也能製作出卓越品質的綠茶。2000–2500公尺的極高海拔、豐富的雲霧、古老茶樹和純淨生態 — 這一切造就了具有「綠茶」中不尋常組合的茶：濃厚飽滿的體感，同時保持鮮爽、甘甜和悠長回甘。栗香 — 深沉持久 — 是該品種的標誌。雲龍綠茶適合尋求有個性綠茶的人：不是精緻「輕盈」的，而是有力、飽滿、令人難忘的。
