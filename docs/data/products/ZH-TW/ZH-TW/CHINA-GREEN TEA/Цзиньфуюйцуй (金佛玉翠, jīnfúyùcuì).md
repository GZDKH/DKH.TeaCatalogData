# 金佛玉翠 (金佛玉翠, jīnfúyùcuì)

**金佛玉翠 (金佛玉翠, jīnfúyùcuì)**
![Цзиньфуюйцуй (金佛玉翠, jīnfúyùcuì)](images/Цзиньфуюйцуй%20%28金佛玉翠,%20Jīnfúyùcuì%29.png)

![Цзиньфуюйцуй (金佛玉翠, jīnfúyùcuì)-1](images/Цзиньфуюйцуй%20%28金佛玉翠,%20Jīnfúyùcuì%29-1.png)

金佛玉翠 (金佛玉翠, jīnfúyùcuì) — 來自重慶市 (重庆, Chóngqìng) 南川區 (南川区, Nánchuān Qū) 的著名綠茶，為國家級農產品地理標誌產品 (国家农产品地理标志产品)。茶名意為「金佛山的翡翠綠茶」，直接指向聯合國教科文組織世界自然遺產金佛山 (金佛山, Jīnfóshān)，主要茶園即位於此山腳下。南川區擁有超過1700年的茶葉種植歷史，而金佛玉翠茶品本身創製於1993年，結合了該地區古老的茶葉傳統與現代加工技術。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。按殺青方式屬於炒青綠茶 (炒青绿茶, chǎoqīng lǜchá) — 經滾筒炒製的綠茶。
* **類別：** 中國地方綠茶；地理標誌保護產品。
* **產地：** 中國重慶市 (重庆市, Chóngqìng Shì) 南川區 (南川区, Nánchuān Qū)。生產區域涵蓋該區29個鄉鎮 — 東至水江鎮 (水江镇)，南至頭渡鎮 (头渡镇)，西至神童鎮 (神童镇)，北至太平場鎮 (太平场镇)。核心產區集中在金佛山國家風景名勝區及大觀園生態農業區 (大观园区) 周圍海拔750–1200公尺的雲霧帶。
* **地理座標：** 南川區 — 約北緯28°46′–29°30′，東經106°54′–107°27′。金佛山 — 北緯28°50′–29°20′，東經107°00′–107°20′。

**2. 歷史與文化意義：**

* **歷史：** 南川地區是中國西南最古老的產茶區之一。當地茶葉歷史可追溯至西周時期（西元前11–8世紀）：據《華陽國志》(《华阳国志》) 記載，巴國每年向周王室進貢茶葉等物產。晚唐時期（9世紀末–10世紀初），茶學家毛文錫 (毛文锡, Máo Wénxī) 在《茶譜》(《茶谱》) 中記載：「涪州出三般茶，賓化最佳」— 賓化 (宾化) 即指今日的南川。南宋時期（12世紀），《建炎雜記》(《建炎杂记》, 1162年) 提及「賓化早春」(宾化早春) — 「賓化早春茶」，在京城享有盛名。陸羽 (陆羽) 的《茶經》(《茶经》) 亦記載巴山峽川 (巴山峡川) 地區有古茶樹，金佛山區域即屬此範圍。

  近代以來，南川經歷了數個發展階段。1939年成立「金佛茶業公司」(金佛茶业公司)。1970年南川列入全國百個茶葉基地縣；1980年建成南川茶廠。1970–80年代，該區以「峨眉牌」(峨眉牌) 紅碎茶 (红碎茶, hóngsuìchá) 聞名，榮獲第25屆日內瓦國際食品博覽會金獎，並獲得上海港免檢出口資格。1979年，在著名茶學家吳覺農 (吴觉农, Wú Juénóng) 及西南農業大學教授參與下，在金佛山發現超過兩千株野生茶樹，確認該區為重要茶葉原產地之一。

  綠茶「金佛玉翠」本身創製於1993年，由南川茶技站 (南川茶技站) 研發。自2005年起，連續六次榮獲「三峽杯」(三峡杯) 競賽「重慶十大名茶」稱號。2005年獲國際「華茗杯」(华茗杯) 金獎及「聯合會杯」(联合会杯) 銀獎（中、日、韓、美四國參與的國際競賽）。2008年獲第七屆「三峽杯」第一名及首屆「重慶十大名茶」稱號。2010年獲第八屆「三峽杯」金獎，同時獲專家評審及觀眾評選雙料第一。2024年榮獲國際「鼎承茶王賽」(鼎承茶王赛) 綠茶類最高「六星特別金獎」(六星特别金奖)。「金佛玉翠」品牌價值評估為4.61億人民幣。

* **茶名：** 金 (jīn) — 「金、金色」；佛 (fó) — 「佛」；玉 (yù) — 「玉、翡翠」；翠 (cuì) — 「翠綠、綠色」。茶名詩意地結合了金佛山 (Jīnfóshān) 的意象與茶葉特徵 — 乾茶翡翠綠色、內蘊光澤的視覺特色。金佛山因夕陽西下時崖壁呈金光，如千佛閃耀而得名，宋代詩歌《望金佛山謠》(《望金佛山谣》) 詠道：「金佛何崔嵬，縹緲雲霞間」(金佛何崔嵬，缥缈云霞间 — 「金佛山何等巍峨，縹緲於雲霞之間」)。

* **文化意義：** 金佛玉翠是南川區的名片與「招牌茶」，為「重慶三大名茶」(重庆三大名茶) 之一。此茶與聯合國教科文組織世界遺產金佛山的文化景觀密不可分（屬「中國南方喀斯特」，2014年）。金佛山生長著野生古茶樹 (古茶树, gǔcháshù)，其中最大一株經西南大學鑑定超過1400年。這些古茶樹與方竹 (方竹)、銀杉 (银杉)、銀杏 (银杏)、杜鵑 (杜鹃) 並稱「金佛山五絕」(金佛山五绝)。特有種「南川茶」(*Camellia nanchuanica* H.T. Chang et Xiong) 由中山大學張宏達教授描述為獨立植物學種，列入「重慶十大優秀農業遺傳資源」。

  在當地生活文化中，「南川打油茶」(南川打油茶, Nánchuān dǎyóuchá) 傳統佔有特殊地位 — 這是一種用茶葉與油脂、調料炒製的濃稠飲品，當地人稱為「幹勁湯」(干劲汤 — 「能量湯」)，用以提神醒腦。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種為福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbáichá) 及巴渝特早 (巴渝特早, Bāyú Tèzǎo)，兩者均為國家級良種 (国家级良种)。輔助原料使用當地小葉群體種「川小葉群體種」(川小叶群体种, Chuān xiǎoyè qúntǐ zhǒng)。福鼎大白茶屬 *Camellia sinensis* var. *sinensis* 大芽種，特點是萌芽早、茸毛豐富、胺基酸含量高。巴渝特早為重慶選育的超早生品種，可比標準栽培種提早7–10天採摘。
* **採摘：** 主要採摘期為春季，集中在清明節 (清明) 前後（4月初）。春茶因冬季積累的胺基酸含量最高，品質最佳。夏、秋茶用於生產大宗茶類。
* **採摘標準：** 特級 (特级) — 一芽一葉初展 (一芽一叶初展, yī yá yī yè chūzhǎn)；一級 (一级) — 一芽二葉；二級 (二级) — 一芽三葉。
* **原料要求：** 鮮葉新鮮、均勻，無機械損傷，無粗老葉片。鮮葉茶多酚含量不低於25%，水浸出物不低於47.4%。

**4. 風土條件與栽培特色：**

* **氣候與地形：** 南川區位於四川盆地與雲貴高原交界處，屬亞熱帶濕潤季風氣候。年均溫16.6°C，年降雨量約1185毫米。年霧日超過200天。顯著的日夜溫差延緩芽葉生長，有利胺基酸積累：春梢游離胺基酸含量可達4.0%以上。散射光多於直射光的條件進一步提升原料的香氣和胺基酸潛力。
* **海拔高度：** 600–1200公尺；核心產區位於750–1200公尺的雲霧帶。
* **土壤：** 微酸性黃壤及紫色土 (紫色土, zǐsè tǔ)，pH值4.5–6.5，富含有機質。核心產區屬水源保護區，禁用化肥農藥。
* **栽培特色：** 茶園位於金佛山山坡，四周環繞天然森林，提供生物防護並形成獨特微氣候。金佛山為國家自然保護區及世界遺產地，生物多樣性豐富（記錄超過8000種動植物），直接保證茶葉原料的生態純淨度。

**5. 製作工藝：**

金佛玉翠採用炒青 (炒青, chǎoqīng) 工藝結合手工整形。完整流程包含28道工序，屬區域非物質文化遺產。總體原則：「高溫快殺鎖鮮，低溫慢烘塑形」(高温快杀锁鲜，低温慢烘塑形)。成品茶含水率不超過6.5%。

1. **鮮葉攤放 (鲜叶摊放 — xiānyè tānfàng)：** 採摘的鮮葉薄攤於通風室內4–6小時，適度失水。此步驟降低酯型兒茶素含量，為熱處理做準備，減少苦澀味，提升口感柔和度。
2. **殺青 (杀青 — shāqīng)：** 在滾筒機中於200–240°C進行酶活性固定。採用「高溫快殺」(高温快杀) 原則，瞬間停止氧化，鎖定鮮葉特性。
3. **揉捻 (揉捻 — róuniǎn)：** 適度壓力下輕揉10–15分鐘。目的是釋出細胞汁液，賦予茶葉初步形狀，但不過度破壞結構。
4. **理條 (理条 — lǐtiáo)：** 在80–100°C專用設備上處理，使茶條理直平行，形成金佛玉翠特有的「緊直」外形。
5. **做形提毫 (做形提毫 — zuòxíng tíháo)：** 手工操作：師傅用手掌搓揉茶葉，將細白茸毛 (毫, háo) 提至表面，賦予乾茶特有的銀白光澤。
6. **足乾 (足干 — zúgān)：** 60–80°C低溫慢烘至最終含水率≤6.5%。長時間溫和加熱固定栗香，防止儲存時發霉。
7. **整理去雜 (整理去杂 — zhěnglǐ qùzá)：** 去除茶梗、碎片及不合格部分，按大小均勻度分級。

**6. 感官特徵：**

* **乾茶外形：** 茶條緊直重實 (紧直重实, jǐnzhí zhòngshí)，色澤深綠油潤 (绿润, lǜrùn)，白毫顯露。形狀為緊直形 (紧直形, jǐnzhí xíng)，均勻度高。
* **乾茶香氣：** 明顯栗香 (栗香, lìxiāng) 伴清新綠茶香；高山批次具特有「冷香」(冷香, lěngxiāng)，茶涼後香氣更顯。
* **茶湯香氣：** 栗香主導且持久 (栗香持久)。上層香氣清香 (清香, qīngxiāng) 鮮嫩。香氣高揚 (高香, gāoxiāng)，無沉悶或青草味。
* **滋味：** 濃醇 (浓醇, nóngchún) — 茶湯「體感」超過一般綠茶水準。鮮爽 (鲜爽, xiānshuǎng) 明顯，胺基酸含量高所致。持久甘甜回味 — 回甘 (回甘, huígān)。苦澀味極輕。
* **湯色：** 嫩綠明亮 (嫩绿明亮, nèn lǜ míngliàng)，多次沖泡後轉為黃綠色。
* **葉底（茶渣）：** 黃綠明亮 (黄绿明亮)，芽葉完整平整，富彈性，大小均勻度佳。

**7. 化學成分：**

* **多酚類（兒茶素）：** 成品茶多酚含量不低於25%。主要成分為EGCG（表沒食子兒茶素沒食子酸酯）、EGC、ECG等兒茶素。多酚類提供抗氧化活性、澀味背景並形成茶湯「體感」。相較平地綠茶，金佛山高山條件促成更佳的兒茶素與胺基酸比例（較低的酚氨比），造就柔和濃郁而無過度澀味的口感。
* **胺基酸：** 游離胺基酸含量不低於4.0%（春茶原料），顯著高於綠茶平均水準（2.0–3.5%）。主要成分為L-茶胺酸 (L-茶氨酸, L-cháānjīsuān)，賦予「甘味」及「鮮味」感受。L-茶胺酸具溫和鎮靜專注效果，平衡咖啡因的刺激作用。
* **水浸出物：** 不低於47.4% — 顯示高萃取性及茶湯濃度。
* **生物鹼：** 咖啡鹼 (咖啡碱, kāfēi jiǎn)、可可鹼、茶鹼。高山茶咖啡因含量通常略高於平地茶，因生長期較長。
* **維生素：** 維生素C（抗壞血酸）— 綠茶鮮葉維C含量為食品中最高之一，但炒製時部分分解。另含B群維生素（B₁、B₂）、維生素K及維生素E。
* **礦物質：** 鉀、錳、鋅、氟、磷。火山成因紫色土壤豐富茶葉微量元素。
* **精油：** 負責栗香形成；主要揮發性成分包括芳樟醇、香葉醇、苯乙醛及炒製產生的吡嗪類化合物。

**8. 保健功效：**

1. **抗氧化支持：** 兒茶素為最強天然抗氧化劑之一；EGCG中和自由基，支持細胞保護。
2. **提神專注效果：** 咖啡因與L-茶胺酸結合提供溫和持續的清醒狀態，無咖啡特有的急升急降現象。L-茶胺酸提升注意力集中度，支持思維清晰。
3. **消化支持：** 綠茶多酚刺激消化酶分泌，可緩解餐後不適。
4. **心血管支持：** 規律飲用綠茶與改善血脂譜相關：兒茶素有助調節膽固醇水準。
5. **代謝支持：** 多酚與咖啡因共同啟動產熱作用，支持新陳代謝及輕盈感。
6. **抗菌消炎作用：** 兒茶素對多種口腔病原菌具抑菌活性，支持牙齦及牙齒健康。
7. **認知功能支持：** L-茶胺酸調節腦部α波節律，促進放鬆專注狀態。

**9. 沖泡方法：**

* **水溫：** 玻璃杯標準沖泡80–85°C；蓋碗功夫茶法首泡85°C。嚴禁使用90°C以上熱水 — 破壞鮮度並引發過度苦澀。
* **茶量：** 玻璃杯法3公克配150毫升（1:50比例）；蓋碗功夫茶法5–6公克配120毫升。
* **茶具：** 玻璃杯 — 觀賞「芽葉豎立」(芽叶竖立, yáyè shùlì) 的理想選擇：嫩芽垂直立於水中，景象壯觀。白瓷蓋碗 — 展現香氣，控制出湯時間。紫砂壺 — 可用，但可能「吸收」細膩上層香氣。
* **沖泡過程：**
  1. 熱水溫壺洗杯後倒掉。
  2. 投入茶葉。
  3. *玻璃杯（上投法 — shàngtóufǎ）：* 先注入1/3容量熱水，讓茶葉浸潤2–3分鐘，再注至7/8滿。靜置2–3分鐘，欣賞芽葉垂直景象。可續水三次。
  4. *蓋碗（功夫茶）：* 85°C熱水快速洗茶（5秒）後倒掉。第二泡20秒，第三泡起每次增加10秒。可泡4–6次。
  5. 最佳用水 — 軟質山泉水或過濾水。

**10. 儲存：**

* 密封包裝（真空或密實鋁箔袋附單向閥），避免異味、直射光及濕氣。
* 最佳溫度0–5°C（冰箱）。冷藏包裝開啟前需回溫至室溫，避免茶葉表面凝結水氣。
* 未開封包裝冰箱儲存可達18個月。開封後建議4–6週內飲用完畢。
* 新茶 (新茶) 建議製作後靜置7–15天再首次沖泡：此期間散去炒製殘留的「火氣」(火气, huǒqì)，口感更加圓潤。

**11. 價格與仿冒品：**

* **價格區間及影響因素：** 特級 (特级)：500–1000元人民幣/斤（500公克）— 一芽一葉初展，栗香濃郁，白毫顯著。一級 (一级)：300–500元/斤 — 一芽二葉，香氣清純，湯色明亮。二級 (二级)：100–300元/斤 — 一芽三葉，滋味濃厚持久，性價比優異。價格取決於採摘時間（早春最貴）、海拔高度、手工比例及具體茶園。

* **避免仿冒品方法：**
  * 向授權經銷商購買，要求透明的茶園、季節、等級資訊。地理標誌 (地理标志) 標識為重要參考。
  * 評估外觀：正宗金佛玉翠為緊直、厚重、深綠色茶條，白毫明顯。仿品常因大小形狀不均而露餡。
  * 栗香為招牌特徵：缺乏純正栗香或出現霉味、酸味、煙味，表示品質不良或仿冒。
  * 茶湯應嫩綠透明；混濁或深黃色表示原料陳舊或加工不當。
  * 可疑低價（聲稱一級品卻低於80–100元/斤）幾乎肯定是以廉價平地綠茶冒充。

**12. 趣味知識：**

* 金佛山生長17,712株野生古茶樹，最大一株樹幹直徑80公分。這是中國僅次於雲南的第二大野生茶樹群落。特有種 *Camellia nanchuanica* H.T. Chang et Xiong（「南川茶」），由中山大學張宏達教授描述，為珍貴的育種遺傳資源。

* 與南川茶相關的詩歌傳說云：「達摩金身降山巔，巧施佛法現茶園」(达摩金身降山巅，巧施佛法现茶园 — 「金身達摩降臨山頂，妙施佛法顯現茶園」)。傳說將金佛山茶葉起源與佛教傳統聯繫：相傳菩提達摩創造茶園以療癒眾生。

* 金佛玉翠完整製作流程包含28道工序，結合機械加工與手工技藝，列入區域非物質文化遺產名錄。

* 南川是中國西南唯一茶、竹、針葉林（銀杉）生態系統匯聚之地。大茶樹、方竹、銀杉的獨特組合形成「生態帶」— 中國獨有的植物學現象。

* 1980年代南川「峨眉牌」紅茶品質可比印度阿薩姆茶，出口英國、美國、新加坡、馬來西亞及西德。1990年代轉向綠茶生產的策略決定，使金佛玉翠成為該區主力產品。

**13. 與其他綠茶比較：**

* **永川秀芽 (永川秀芽, Yǒngchuān Xiùyá)：** 重慶另一著名綠茶，產於永川區。屬烘青 (烘青) 類型（熱風乾燥），有別於炒青的金佛玉翠。秀芽具更溫和的花草香，而金佛玉翠以濃郁栗香及更「厚重」的茶湯見長。

* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南省著名綠茶，「中國十大名茶」之一。毛尖為炒青綠茶，白毫豐富，清香略帶豆香。金佛玉翠栗香更突出，滋味濃醇 (浓醇)，而信陽毛尖偏向精緻清淡。

* **蒙頂甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 四川經典綠茶，來自蒙頂山，中國最古老的名茶之一。捲曲形，花香帶栗香。相較蒙頂甘露，金佛玉翠茶條更直，因金佛山喀斯特高山風土而具更明顯的礦物質感。

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北省唯一的傳統蒸青 (蒸青) 綠茶代表。玉露色澤深綠，具蒸青茶特有的「鮮海藻」香氣，與金佛玉翠栗香「炒製」風格截然不同。

**結語：**

金佛玉翠是千年傳統與現代工藝結合的產物，誕生於金佛山獨特的自然景觀中。其濃郁栗香、油潤飽滿的滋味及悠長甘甜回味，使其在中國西南眾多地方綠茶中獨樹一幟。對茶友而言，這是認識重慶茶文化的機會 — 在這個地區，野生古茶樹與銀杉、方竹比鄰而居，雲霧與喀斯特岩石創造出無法複製的風土條件。金佛玉翠特別適合那些在綠茶中追求的不是飄逸輕盈，而是深度、結構與令人難忘的栗香「簽名」的品茶者。
