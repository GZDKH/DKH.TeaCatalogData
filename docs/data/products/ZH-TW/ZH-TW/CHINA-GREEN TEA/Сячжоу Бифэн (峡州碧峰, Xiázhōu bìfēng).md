# 峽州碧峰 (峡州碧峰, Xiázhōu bìfēng)

**峽州碧峰 (峡州碧峰, Xiázhōu bìfēng)**

峽州碧峰 (峡州碧峰, Xiázhōu bìfēng) — 中國綠茶，屬半烘炒條形綠茶 (半烘炒条形绿茶, bàn hōngchǎo tiáoxíng lǜchá)，為湖北省宜昌市夷陵區特產。國家地理標誌產品 (国家地理标志产品, guójiā dìlǐ biāozhì chǎnpǐn)，於2017年獲得認證。此茶延續了古代峽州超過千年的茶葉傳統——「茶聖」陸羽 (陆羽, Lù Yǔ) 在其《茶經》(《茶经》, *Chájīng*) 中將峽州茶列為山南茶區之首。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵）。半烘炒類型 (半烘炒, bàn hōngchǎo) — 結合滾筒殺青 (杀青, shāqīng) 與最終炭火烘焙 (木炭烘焙, mùtàn hōngbèi)。
* **類別：** 中國地方茶；國家地理標誌產品（2017年）。
* **產地：** 中國湖北省 (湖北, Húběi) 宜昌市 (宜昌, Yíchāng) 夷陵區 (夷陵区, Yílíng Qū)。古代地名為峽州 (峡州, Xiázhōu)。
* **地理座標：** 約北緯30°42′–31°06′，東經110°51′–111°11′。

**2. 歷史與文化意義：**

* **歷史：** 峽州茶葉傳統為中國最古老之一。早在南北朝時期（420–589），峽州已是重要產茶區。該地區歷史上的關鍵時刻是《茶經》(《茶经》, *Chájīng*, 約780年) 中的評價，陸羽 (陆羽, Lù Yǔ) 記載：「山南，以峽州上，襄州、荊州次」(山南，以峡州上)。峽州茶因此獲得整個南方山區茶葉之首的地位。北宋文學家歐陽修 (欧阳修, Ōuyáng Xiū) 任夷陵縣令時，留下著名詩句：「楚國西偏境，陸羽茶經第一州」(陆羽茶经第一州)。詩人陸游 (陆游, Lù Yóu) 在三游洞 (三游洞) 品茶時讚美其色、香、味。王象晉 (王象晋) 《群芳譜》(《群芳谱》, *Qúnfāng Pǔ*) 中將峽州小江園 (小江园) 茶列入最高等級。

  現代峽州碧峰創製於1979年，由茶葉大師林作炎 (林作炎, Lín Zuòyán) 主持。技術完善工作歷時七年，1983年正式命名為「峡州碧峰」——取自明代詩人田鈞 (田钧) 詩句：「不見仙人空碧峰」(不见仙人空碧峰)。1985年獲農業部品質獎，並用作國禮茶 (国礼茶, guólǐ chá) 贈送外國賓客。

  2017年峽州碧峰獲地理標誌產品認證，品牌價值評估為5.63億元人民幣。茶葉出口至二十多個國家，包括英國、美國、日本和德國。

* **名稱：** 「峽州」(峡州) — 宜昌地區古名，可追溯至唐代，與長江峽谷相關。「碧峰」(碧峰) — 「翠綠山峰」— 來自明代詩歌的詩意意象，指向西陵峽 (西陵峡, Xīlíng Xiá) 的山景。

* **文化意義：** 峽州碧峰是古老茶葉傳統的現代傳承者，該傳統在《茶經》中有文獻記載。這是少數幾種歷史可直接追溯到陸羽的茶葉之一，在中國茶文化中具有特殊的象徵意義。唐代詩人皮日休 (皮日休, Pí Rìxiū) 讚美小江園茶：「吳僧漫說鴉山好，蜀叟休誇鳥嘴香」——這首九世紀的詩歌證明，早在盛唐時期當地茶葉就超越了江西和四川的名茶。對夷陵區而言，茶葉是地域品牌的名片，是長江峽谷文化不可分割的一部分。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種為宜昌大葉種 (宜昌大叶种, Yíchāng Dàyè Zhǒng)，註冊為華茶29號 (华茶29号) — 國家名錄中首批30個地方優良品種之一。輔助栽培種為宜紅早 (宜红早, Yíhóng Zǎo)，即鄂茶4號 (鄂茶4号)，以及鄂茶9號 (鄂茶9号)。樹齡超過30年；葉片橢圓形，長14.7 ± 2.4公分；100芽重約57公克；原料中多酚含量達35.8%。品種具有高抗寒性，適合在海拔1400公尺以下種植。

* **採摘：** 春季採摘（品質高峰期）。嚴格遵守「九不採」(九不采, jiǔ bù cǎi) 標準：不採紫色芽頭、蟲傷葉、病葉及其他缺陷部分。

* **採摘標準：** 特級 (特级) — 單芽或一芽一葉，芽長約3公分；芽頭比例 ≥ 90%；完全手工製作。一級 (一级) — 一芽一葉初展 (一芽一叶初展)，含量 ≥ 80%；允許部分機械揉捻。二級 (二级) — 主要為一芽二葉；可沖泡五次以上，葉底平整均勻。

**4. 風土條件與栽培特點：**

* **地形與位置：** 茶園位於長江西陵峽 (西陵峡, Xīlíng Xiá) 北岸，夷陵區半高山茶區 (半高山茶区, bàn gāoshān cháqū)。長江流經產區17.4公裡；當地河流溪澗眾多——百歲溪 (百岁溪)、太平溪 (太平溪) 等。森林覆蓋率超過70%，空氣中負離子含量比城市高50倍。

* **海拔：** 200–1000公尺；核心產區海拔600–900公尺（雲霧帶）。

* **氣候：** 年平均氣溫16.6°C；年降雨量1100–1300毫米；年霧日超過180天；日夜溫差 > 8°C。散射光有利於氨基酸積累（春茶氨基酸含量 ≥ 1.64%）。

* **土壤：** 黃棕壤 (黄棕壤, huáng zōng rǎng)，pH值4.5–6.5，有機質含量 ≥ 1.0%，富含鋅和硒。

* **核心產區：** 太平溪鎮 (太平溪镇)、樂天溪鎮 (乐天溪镇)、鄧村鄉 (邓村乡)、三斗坪鎮 (三斗坪镇)。所有核心區域位於西陵峽北岸，海拔600–900公尺的雲霧帶。鄧村鄉非正式稱為「湖北茶鄉」，在生產中發揮關鍵作用——這裡集中了樹齡超過30年的最古老茶園。霧氣、柔和散射光和富含硒鋅的礦化土壤相結合，形成了獨特的風土特徵，使碧峰有別於其他湖北綠茶。

**5. 製作工藝：**

峽州碧峰為半烘炒綠茶。工藝包括獨特的複揉理條 (复揉理条, fù róu lǐtiáo) 步驟，形成特有外形並顯露白毫。

1. **攤放 (摊放, tānfàng)：** 鮮葉薄攤4–6小時，初步失水並準備香氣前體物質。
2. **殺青 (杀青, shāqīng)：** 在旋轉滾筒中於180–200°C處理6–8分鐘。停止酶促氧化，固定綠色。
3. **攤涼 (摊凉, tānliáng)：** 攤開原料降溫，為揉捻做準備。
4. **初揉 (初揉, chū róu)：** 14–20分鐘，按「輕-重-輕」(轻-重-轻) 原則，釋放細胞汁液，形成葉片初始結構。
5. **初烘 (初烘, chū hōng)：** 溫度100–120°C，部分脫水。
6. **複揉理條 (复揉理条, fù róu lǐtiáo)：** 獨創步驟——手工「抓攏搓抖」(手工抓拢搓抖) 技法。正是在此步驟茶葉獲得特有的優雅外形和明顯的茸毛。
7. **足乾提毫 (足干提毫, zú gān tí háo)：** 炭火烘乾，溫度70–80°C。低溫和傳統木炭確保香氣的精緻固定和白毫的顯露。
8. **精製定級 (精制定级, jīngzhì dìngjí)：** 按等級分篩並確定商品等級。

**6. 感官特徵：**

* **乾茶外觀：** 條索形 (条索形, tiáosuǒ xíng)，緊細挺直，優雅，白毫顯露；色澤翠綠油潤 (翠绿油润)。
* **乾茶香氣：** 明顯的栗香 (栗香, lì xiāng) — 主調；背景為清純植物清香 (清香, qīng xiāng)。
* **茶湯香氣：** 栗香更深層展現，持久高揚；清香作為上層音符保持。
* **滋味：** 鮮爽 (鲜爽, xiān shuǎng)，由高氨基酸含量所致；明顯持久的甘甜回味——回甘 (回甘, huígān)。茶體輕盈但飽滿。
* **湯色：** 黃綠明亮 (黄绿明亮)。
* **葉底（沖泡後茶葉）：** 嫩綠色，勻整鮮活 (嫩绿匀整鲜活)。

**7. 化學成分：**

* **水浸出物 (水浸出物)：** ≥ 45.8% — 顯著高於綠茶平均水準，證明滋味濃度高。
* **茶多酚 (茶多酚)：** ≥ 30%（原料中達35.8%）；主要抗氧化群體為兒茶素（EGCG、EGC、ECG）。
* **咖啡鹼 (咖啡碱)：** 4.2%。
* **氨基酸：** 春茶中 ≥ 1.64%；酚氨比 (酚氨比, fēn'ān bǐ) 為12.1，顯示鮮爽度與茶體的良好平衡。
* **礦物質：** 鋅（Zn）和硒（Se）含量較高，由當地土壤地球化學特性所致。
* **維生素：** 維生素C（新鮮原料中）、B族維生素、維生素K。
* **精油：** 栗香特徵由滾筒殺青和炭火烘乾形成——主要為吡嗪類和呋喃類化合物。

**8. 保健功效：**

* **抗氧化保護：** 高多酚含量提供活性自由基清除。
* **提神效果：** 咖啡鹼與L-茶氨酸結合產生溫和持久的清醒感，無劇烈波動。
* **脂質代謝支持：** 兒茶素促進脂肪分解加速。
* **礦物質支持：** 天然富含鋅和硒，有益於免疫系統和抗氧化保護。
* **消化支持：** 適量單寧有助於餐後胃腸道正常功能。
* **思維清晰：** L-茶氨酸支持專注、平靜的意識狀態。
* **心血管系統支持：** 定期適量飲用高兒茶素綠茶與血管彈性維護相關。
* **抗輻射保護：** 據某些資料，綠茶多酚有助於減少家用設備電磁輻射引起的氧化應激。

*重要提示：這是一般資訊，非醫療建議。*

**9. 沖泡方法：**

* **水溫：** 80°C（絕對不建議沸水——溫度超過80°C會破壞葉綠素並產生苦味）。
* **茶量：** 3公克配150毫升（比例1:50）。
* **茶具：** 玻璃杯（觀察茶葉舞動）；薄瓷蓋碗。

* **過程（玻璃杯，中投法 — 中投法, zhōng tóu fǎ）：**
  1. 用沸水溫杯。
  2. 注入1/3容量的水（80°C）。
  3. 投入3公克茶葉，輕搖杯子潤茶搖香 (润茶摇香)。
  4. 高沖注水至滿杯。
  5. 浸泡2–3分鐘。待降至60°C時飲用——此溫度鮮爽感最為明顯。

* **過程（蓋碗）：**
  1. 溫蓋碗。
  2. 投茶，首泡30秒。
  3. 每次增加10秒。
  4. 可沖泡3次。

* **注意事項：** 沿杯壁緩慢注水——急流會打散茸毛使茶湯渾濁。新茶建議購買後避光存放約7天進行醒茶 (醒茶, xǐng chá)。開封後冷藏保存，一個月內飲用完畢。

**10. 儲存：**

* 密封包裝，避免異味。
* 最佳溫度：0–5°C（冰箱）長期保存。
* 冷藏後開封前室溫平衡至無凝露。
* 避免直射陽光、高濕度和熱源。
* 開封後一個月內飲用完畢以保持新鮮。

**11. 價格與仿冒：**

* **價格區間：** 特級 (特级)：600–800元/斤（手工製作，單芽）；一級 (一级)：250–400元/斤；二級 (二级) — 價格親民，定位為日常茶，沖泡耐久性好。
* **價格因素：** 原料等級、採摘季節（早春較貴）、手工比例、地理標誌認證。

* **避免仿冒：**
  * 向有「峡州碧峰」品牌使用權的經銷商購買（品牌屬湖北鄧村綠茶集團）。
  * 核實地理標誌標識。
  * 真正的峽州碧峰具有細緻緊實的條索，白毫明顯，油潤光澤——粗糙暗淡的葉片表明替代品。
  * 香氣應為純正栗香，無霉味和「魚腥」味。
  * 聲稱等級的可疑低價是偽造品的主要標誌。

**12. 趣味知識：**

* 陸羽在《茶經》中不僅將峽州列為山南茶區之首，還注意到巴山峽谷中有茶樹粗到需要兩人合抱。
* 「碧峰」之名借自明代詩人田鈞詩句：「不見仙人空碧峰」(不见仙人空碧峰)。這是少數幾種名稱直接源於詩歌文本的茶葉之一。
* 現代茶的前身是「太平毛尖」(太平毛尖)；現名在1912年東湖縣撤銷後確立。
* 1985年峽州碧峰成為少數幾種用作國禮贈送外國代表團的湖北綠茶之一。
* 2017年品牌價值評估為5.63億元人民幣——對一個區域性茶葉而言是可觀數字。
* 「九不採」(九不采) 技術特色是湖北綠茶中最嚴格的原料品質控制系統之一：禁採紫色芽頭、機械損傷葉、蟲害葉、病葉、過度展開葉及其他缺陷部分。
* 宜昌是中國最古老的茶葉中心之一：早在三國時期（3世紀）《廣雅》(《广雅》) 就記錄了荊巴地區（包括峽州）的茶餅生產。

**13. 與其他綠茶比較：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 同樣來自湖北，但採用日式蒸青 (蒸青) 工藝——具有更多草本「海洋」特徵。峽州碧峰採用滾筒殺青配炭火烘乾，呈現更深層的栗香。
* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南鄰省茶，茸毛相似，但滋味更圓潤，豆香明顯。峽州碧峰的回甘更明顯，具有鋅硒土壤特有的礦物質韻味。
* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 安徽大葉扁形茶——在視覺和質感上與碧峰的細嫩茸毛條索完全相反。兩者都以高浸出率著稱，但猴魁傾向蘭花香，碧峰則為栗香。
* **蒙頂甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 四川茶，工藝相似（炒製+烘乾）。甘露更彎曲，花香豆香明顯；碧峰更挺直，堅果香主導，西陵峽特有礦物質感明顯。
* **廬山雲霧 (庐山云雾, Lúshān Yúnwù)：** 江西高山綠茶，來自類似的雲霧風土。雲霧呈現更柔和溫潤的特徵，花香層次豐富，而碧峰因高浸出率（≥ 45.8%）展現更濃郁的滋味密度和更明顯的烘炒栗香特徵。

**結語：**

峽州碧峰是一款具有文學血統和千年風土歷史的茶葉。它站在《茶經》中「山南第一茶」古老榮耀與湖北茶人1979年創製的現代工藝的交匯點，創造了配得上這一傳承的茶品。栗香、鮮活清爽和悠長甘甜回味，使其成為重視經典中國綠茶品格與深度的茶友的絕佳選擇。

這款茶在春季月份特別出色，此時首批採摘展現氨基酸甘甜和嫩毫茶湯的全部豐富性。用軟水沖泡，勿過熱——西陵峽的翠綠山峰將在您的茶杯中綻放。對於地方中國綠茶愛好者，峽州碧峰是最佳發現之一：它不如湖北恩施玉露那樣廣為人知，但在滋味深度和歷史價值方面不遜於任何著名同類茶品。
