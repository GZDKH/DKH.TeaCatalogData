# 信陽毛尖 (信陽毛尖, Xìnyáng máo jiān)

**信陽毛尖 (信陽毛尖, Xìnyáng máo jiān)**

![信陽毛尖 (信陽毛尖, Xìnyáng máo jiān)](images/Синьян%20Мао%20Цзянь%20(信阳毛尖,%20Xìnyáng%20Máo%20Jiān).png)

![信陽毛尖 (信陽毛尖, Xìnyáng máo jiān)-1](images/Синьян%20Мао%20Цзянь%20(信阳毛尖,%20Xìnyáng%20Máo%20Jiān)-1.png)

信陽毛尖是中國最北方的著名綠茶之一，產於遠離南方省份傳統「茶葉帶」的地區。其故鄉是河南省信陽市的山區，這裡涼爽的過渡性氣候、長時間的霧氣和富含腐殖質的土壤，孕育出氨基酸含量異常高且帶有鮮明板栗香氣的茶葉。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵茶）。茶葉在鍋中以高溫固定（「殺青」），氧化作用幾乎完全停止。

* **類別：** 中國名茶（中國十大名茶，Zhōngguó Shí Dà Míng Chá）。自1958年起列入「中國十大名茶」。

* **產地：** 中國，河南省（河南省，Hénán Shěng），信陽市（信陽市，Xìnyáng Shì）。地理標誌保護區域涵蓋八個行政區：浉河區（浉河區）和平橋區（平橋區），以及羅山縣（羅山縣）、光山縣（光山縣）、新縣（新縣）、商城縣（商城縣）、固始縣（固始縣）和潢川縣（潢川縣）。

* **核心產區——「五雲兩潭一寨」（五雲兩潭一寨，Wǔ Yún Liǎng Tán Yī Zhài）：** 車雲山（車雲山）、集雲山（集雲山）、雲霧山（雲霧山）、天雲山（天雲山）、連雲山（連雲山）；黑龍潭（黑龍潭）和白龍潭（白龍潭）；何家寨（何家寨）。新的關鍵微產區——浉河港鎮（浉河港鎮）和董家河鎮（董家河鎮），高山茶園位於海拔500公尺以上。

* **地理座標：** 約北緯32°07′，東經114°04′。

**2. 歷史與文化意義：**

* **歷史：**

信陽地區的茶葉種植歷史超過兩千年，可追溯至東周時期（東周，Dōng Zhōu）。西元760-780年間，茶聖陸羽（陸羽，Lù Yǔ）在《茶經》（茶經，Chájīng）中將此地列入淮南茶區（淮南茶區），並將義陽郡（義陽郡，今信陽）的茶葉列為上品。北宋（北宋）詩人蘇東坡（蘇東坡）留下名句：「淮南茶信陽第一」（淮南茶信陽第一）。

清朝（清朝）末期，1905-1909年間，企業家蔡竹賢（蔡竹賢）創立八大茶社——元貞、廣益、裕申、宏濟、博厚、森林、龍潭和廣生——開闢約30公頃茶園，並系統化炒製技術。1913年，此茶正式命名為「信陽毛尖」（信陽毛尖）。1915年在舊金山巴拿馬太平洋萬國博覽會上榮獲金獎。

1958年信陽毛尖入選「中國十大名茶」。1990年獲得國家質量金獎。2007年在日本世界綠茶大會上再次榮獲金獎。2008年國家標準GB/T 22737-2008生效，確立地理標誌保護地位。2014年信陽毛尖製作技藝被列入國家級非物質文化遺產名錄。

* **名稱：**

	* **信陽（信陽）** ——地名，指產地：河南省信陽市。
	* **毛（毛）** ——「絨毛、細毛」。指茶芽和嫩葉上豐富的白色絨毛（白毫，bái háo）。
	* **尖（尖）** ——「尖端、頂端、峰」。形容成品茶葉的尖細針狀外形。
	* 「毛尖」（Máo Jiān）整體意為「毛絨尖端」——詩意地描繪了乾茶的外觀和觸感。

* **文化意義：** 信陽毛尖是河南省的驕傲和名片，河南是中華文明的發源地之一。此茶在當地的饋贈文化和官方接待中占有重要地位。信陽每年舉辦國際茶文化節（信陽國際茶文化節），專門推廣此茶。自1994年起，所有符合統一工藝和品質標準的信陽地區茶葉都統一使用「信陽毛尖」品牌，使該地區成為中國最大的名茶生產基地。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要品種（約70%）為當地群體種——信陽群體種（信陽群體種，Xìnyáng Qúntǐ Zhǒng），屬於*Camellia sinensis* var. *sinensis*的灌木型中葉種。植株抗寒性強；芽葉淺綠色，肉質厚實，密布絨毛。一芽一葉百芽重約32克。輔助品種包括福鼎大白茶（福鼎大白茶，Fúdǐng Dà Bái Chá）、烏牛早（烏牛早，Wūniú Zǎo）等早生引進品種，可延長採摘期。

* **採摘：** 主要採摘季為春季，從3月底至5月初。傳統季節分級：
	* **明前茶（明前茶）** ——清明節（清明，Qīngmíng，4月初）前：主要為單芽，鮮度最高，「毫香」最顯。
	* **穀雨茶（穀雨茶）** ——穀雨節（穀雨，Gǔyǔ，約4月20日）前：標準為「一芽一葉」（一芽一葉），滋味更飽滿濃郁。
	* **春尾茶（春尾茶）** ——立夏（立夏，5月初）前：性價比較高。
	* **夏秋茶（夏秋茶）** ——夏秋季採摘：較苦澀，常用於製作花茶。

* **採摘標準：** 最高等級——「珍品」（珍品）和「特級」（特級）——僅採單芽或「一芽一葉初展」（一芽一葉初展）。製作1公斤「珍品」級茶葉需採摘超過10萬個茶芽。一級茶為「一芽二葉初展」。二、三級為「一芽二三葉」。採摘須在晴天，露水乾後進行。

* **原料要求：** 芽葉須嫩、完整、飽滿、大小均勻、覆有白毫。不得有粗老葉、茶梗、紫色芽葉和雜質。鮮葉含水量約70%。

**4. 風土條件與栽培特點：**

核心產區位於北亞熱帶向暖溫帶過渡地帶（北亞熱帶向暖溫帶過渡），賦予茶葉獨特的品質特徵，有別於典型的南方綠茶。

* **生長海拔：** 300-800公尺。最佳茶園海拔500公尺以上（浉河港鎮和董家河鎮）。

* **氣候：** 年均溫15.1°C。年降雨量約1200毫米。相對濕度76%。日夜溫差大。年均雲霧天數約200天，提供充足的漫射光（漫射光），抑制粗纖維形成。冬季相對溫和，夏季炎熱潮濕，具明顯季風特徵。

* **土壤：** 黃棕壤（黃棕壤，huáng zōng rǎng），pH值4.5-6.0，富含腐殖質（有機質含量≥2.5%）。高腐殖質含量促進氮素代謝，提高葉片氨基酸含量。

* **風土特色：** 信陽地區素有「山水之鄉」（山水之鄉）美譽。涼爽的高山夜晚、持久的霧氣、酸性土壤和較長的生長期，使茶葉游離氨基酸含量高而茶多酚適中，形成北方綠茶特有的甘甜鮮爽滋味。

**5. 製作工藝：**

信陽毛尖的製作是典型的綠茶加工工藝，特色在於「生熟雙鍋」（生熟雙鍋，shēng shú shuāng guō）技術：「生鍋」（生鍋）用於殺青，「熟鍋」（熟鍋）用於做形。傳統工藝包含九道手工工序。

* **攤放（攤放——tān fàng）：** 鮮葉薄攤（約3公分厚）於竹匾上，置於陰涼處4-10小時。過程中失去部分水分（至約70%含水量），茶多酚輕微氧化，蛋白質水解為氨基酸，澱粉分解為可溶性糖；部分青草味揮發物散失。這改善了香氣並減少苦澀味。

* **生鍋殺青（生鍋殺青——shēng guō shā qīng）：** 傾斜鐵鍋（30-35°角）加熱至約140°C（某些批次可達160-200°C）。採用「手工抓條」（手工抓條，shǒugōng zhuā tiáo）技術——用手掌和手指有力地拋擲攪拌茶葉。目的是快速鈍化酶（多酚氧化酶），保持綠色並固定鮮香。

* **熟鍋做形（熟鍋做形——shú guō zuò xíng）：** 鍋溫降至80-100°C。製茶師依序進行技術動作：「裹條」（裹條，guǒ tiáo）、「扇條」（扇條，shàn tiáo）、「趕條」（趕條，gǎn tiáo）以拉直茶條並打散結塊，最後「理條」（理條，lǐ tiáo）——反覆抓拋茶條，賦予其特有的細、直、圓、光形態。此工序決定了「細圓光直」——細、圓、光滑、直的外形。出鍋時含水量33-35%。

* **初烘（初烘——chū hōng）：** 用木炭烘焙，初溫約120°C逐漸降至約90°C。烘至含水量約15%。

* **攤涼（攤涼——tān liáng）：** 攤放茶葉使表層與內部水分均勻。時間不少於40分鐘。

* **復烘（復烘——fù hōng）：** 約60°C緩慢溫和烘乾至含水量≤6%。茶葉擠壓時應碎成粉末，色澤翠綠，白毫顯露。

* **揀剔（揀剔——jiǎn tī）：** 手工剔除不合格茶葉、茶梗和雜質。

* **工藝特點：** 關鍵特色是「生熟雙鍋」（兩個溫度和功能不同的連續鍋）。完整工序包含九道手工操作。核心技術——「提毫保翠」（提毫保翠，tí háo bǎo cuì）——「提升白毫，保持翠綠」：做形時的特殊手法使白毫浮現於茶葉表面，同時保護葉綠素不被破壞。

**6. 感官特徵：**

* **乾茶外形：** 細緊、挺直、尖銳的針芽狀（細直針芽狀）。按標準——「直、細、挺、勻、嫩」（直、細、挺、勻、嫩）。色澤鮮翠綠（翠綠）帶豐富的銀白色絨毛（白毫顯露），呈現淡灰色光澤。茶葉完整，未展開。

* **乾茶香氣：** 主調為明顯的炒板栗香（板栗香，bǎnlì xiāng），輔以純淨的清香（清香）和「毫香」（毫香），類似嫩玉米或煮豆香。高等級茶可能呈現淡雅花香（蘭花）和細膩的奶油香韻。

* **茶湯香氣：** 純淨、高揚、持久。板栗香仍為主調但更柔和；花草香調配合輕微炒製香氣凸顯。香氣從杯中明顯升起，久久不散。

* **滋味：** 顯著的鮮爽（鮮爽，xiān shuǎng），源於高氨基酸含量。湯感醇厚圓潤（醇厚，chún hòu）。特有的回甘生津（回甘生津，huí gān shēng jīn）。口感融合板栗、綠葉和淡雅花香。高等級茶澀感極低，夏秋茶澀感增加。正確沖泡時無苦味。

* **湯色：** 淺綠帶明亮黃色調，清澈透明，富有光澤。沖泡時可見因懸浮絨毛造成的輕微乳光——這是正常現象，非缺陷。

* **葉底（沖泡後茶葉）：** 嫩、完整、有彈性的芽葉，呈鮮綠均勻色澤，質地柔軟。充分舒展但不過度。高品質原料明顯：芽葉均勻，無粗老葉和茶梗。

**7. 化學成分：**

信陽毛尖的化學成分已由信陽農業學院和多個實驗室的專家研究。根據科學文獻：

* **茶多酚（兒茶素）：** 茶多酚總含量20-28%（不同來源數據：核心產區春季一級茶樣品為20.02-21.87%；各等級廣泛範圍為25.97-27.87%）。總兒茶素含量117.71-184.18毫克/克，以EGCG（表沒食子兒茶素沒食子酸酯）為主——主要抗氧化劑。作為北方綠茶，茶多酚含量相對較高，在保持適度苦澀的同時提供良好的抗氧化活性。

* **氨基酸：** 游離氨基酸含量2.95-4.34%，在中國綠茶中屬高水平。根據HPLC分析，濃度為53.21-61.07毫克/克。L-茶氨酸（L-theanine）是主要氨基酸，賦予明顯的「鮮」（鮮/鮮味）特徵。高氨基酸含量源於北方涼爽氣候和長期雲霧，抑制了氨基酸向茶多酚的光合轉化。

* **生物鹼：** 咖啡因4.06-4.73%（37.59-45.19毫克/克）；還含有可可鹼和茶鹼。檢測到顯著含量的GABA（γ-氨基丁酸）——有助於血管平滑肌鬆弛。

* **維生素：** 富含維生素C（尤其春茶）、B群維生素（B₁、B₂）以及維生素P、PP（菸鹼酸）、K。

* **礦物質：** 含氟（200-400 ppm，對預防蛀牙有益）、鉀、鎂、鋅、錳、硒等微量元素。水浸出物43-46.5%，超過國標≥39%的要求（GB/T 22737）。

* **精油和芳香成分：** GC-MS分析鑑定出85種揮發性化合物。關鍵香氣成分（按ROAV——相對氣味活性值）：芳樟醇、萘、δ-杜松烯、香葉醇、β-紫羅蘭酮、順式茉莉酮、苯甲醛、β-環檸檬醛和2-正戊基呋喃。

* **註：** 數值因採摘季節（春季vs.夏季）、等級、茶園海拔和具體年份而有顯著差異。春季高山茶的氨基酸與茶多酚比值最高，與優異的口感品質相關。

**8. 保健功效：**

* **抗氧化作用：** 兒茶素（尤其EGCG）有效中和自由基，減緩氧化壓力和細胞老化。

* **提神和認知效果：** 咖啡因與L-茶氨酸協同作用，提供溫和持久的清醒狀態，無劇烈起伏；改善注意力、記憶力和反應速度。

* **心血管系統：** 兒茶素和GABA有助降低「壞」膽固醇（LDL），增強血管壁彈性，調節血壓。

* **輻射防護：** 茶多酚能結合某些放射性元素（如鍶-90等），加速其排出體外。

* **護齒：** 高氟含量（200-400 ppm）強化牙釉質，抑制致齲細菌活動。

* **改善消化：** 刺激胃液分泌和腸道蠕動，幫助分解油膩食物。

* **增強免疫力：** 維生素C、茶多酚和微量元素（鋅、硒）支持免疫反應和抗感染能力。

* **清涼解渴：** 通過刺激唾液分泌和體溫調節，在炎熱天氣中極佳解渴。

* **重要提示：** 這是關於茶葉成分特性的一般資訊，非醫療建議。如有疾病應諮詢醫生。

**9. 沖泡方法：**

* **水溫：** 80-85°C。絕不可用沸水——會破壞維生素C，使茶湯混濁並增加苦澀。

* **茶量：** 3克茶葉配150毫升水（比例1:50）。若要更濃郁可用4-5克配150-200毫升。

* **茶具：** 玻璃杯或蓋碗（蓋碗，gàiwǎn）薄瓷器。玻璃可觀賞茶葉「舞動」和葉片舒展——這是品飲毛尖的美學享受之一。瓷壺亦可。

* **步驟：**

	1. 用熱水溫杯後倒掉。
	2. 投入茶葉。
	3. 注入三分之一容量的水（80-85°C）輕柔潤濕茶葉——這是「潤茶」（潤茶，rùn chá），洗茶/喚醒茶葉；等待約10秒後倒掉。
	4. 高沖（高沖，gāo chōng）注水至七分滿。
	5. 浸泡1-2分鐘（第一泡）。
	6. 飲用約三分之一後續水（留根法，liú gēn fǎ——「留根法」）。
	7. 可重複沖泡3-4次，逐漸延長浸泡時間。

* **品飲注意事項：**
	* 避免長時間浸泡（超過3分鐘）——過度萃取單寧會使口感澀而粗糙。
	* 新購茶葉建議在冰箱中存放約一週，以去除「火氣」（退火氣）。
	* 腸胃敏感者宜避免空腹飲用，並配以清淡點心。

**10. 儲存方法：**

信陽毛尖如同所有嬌嫩綠茶，對儲存條件極為敏感。

* **溫度：** 0-5°C（冰箱，獨立密封隔間）。長期儲存（超過3個月）——冷凍庫。
* **容器：** 密封、不透光容器。理想選擇為密封鐵罐、鋁箔真空包裝袋或雙層密封袋。內可放矽膠吸收殘餘水分。
* **茶葉大敵：** 光線（破壞葉綠素和維生素）、濕氣（引發黴變和氧化）、高溫（加速香氣降解）、異味（茶葉易吸附）。
* **保存期限：** 正確條件下（冰箱、密封容器）——12-18個月。開封後建議1-2個月內飲用完畢。隨時間推移，板栗香減弱，葉色變暗——這是老化跡象。

**11. 價格與防偽：**

信陽毛尖屬於高價綠茶類別。價格受多項因素影響：等級（珍品和特級最貴）、採摘季節（明前>穀雨>春尾>夏秋）、茶園海拔（高山>低地）、產地（核心產區「五雲...」>周邊縣區）、手工vs.機製以及是否有地理標誌認證。

* **如何避免假貨：**

	* **向可靠商家購買：** 專業茶葉店、信陽大型茶葉合作社授權經銷商。地理標誌（地理標誌）標識是重要指標。
	* **評估外觀：** 真正高品質毛尖——細、直、勻的針狀，白毫豐富，翠綠鮮亮。暗淡、不均勻的顏色，粗大葉片，缺乏絨毛——是低品質或假貨標誌。
	* **檢查香氣：** 乾茶應有新鮮炒板栗香和純淨綠香。霉味、「煮熟」的草味、異味——都是不良跡象。
	* **檢驗茶湯：** 顏色應透明、淺綠帶明亮光澤。混濁、深色或黃褐色茶湯表明原料劣質或儲存不當。
	* **價格過低：** 如果價格明顯低於市場價，幾乎肯定是假貨（其他地區以類似工藝製作的茶葉）或冒充高級的低等級原料。

**12. 趣聞軼事：**

* 信陽毛尖是中國最北方的著名綠茶之一。河南省位於「茶葉帶」（主要產茶省份——浙江、福建、雲南、安徽）以北很遠，使信陽風土獨一無二。

* 製作一公斤頂級「珍品」（珍品）需要採摘和加工超過10萬個茶芽——完全手工作業。

* 近千年前的詩人蘇東坡就已將信陽茶譽為淮河以南地區之冠。

* 信陽舉辦中國最大的茶文化節之一——信陽國際茶文化節，吸引全國及海外參與者和採購商。

* 優質信陽毛尖樣品的水浸出物（可溶性物質）達46.5%——遠超39%的最低標準，證明其滋味的非凡飽滿度。

**13. 與其他「毛尖」和「毛峰」類綠茶的比較：**

信陽毛尖屬於「毛尖」（毛尖，「毛絨尖端」）茶系，但每種都有獨特的風土和風格。

* **都勻毛尖（都勻毛尖，Dūyún Máo Jiān）：** 貴州省。同為「十大名茶」。原料為小葉種貴州品種。茶葉形狀更捲曲（鉤狀），花香更明顯，板栗香較淡。口感溫和細膩，不如信陽毛尖「厚實」。

* **黃山毛峰（黃山毛峰，Huángshān Máo Fēng）：** 安徽省黃山。「毛峰」（「毛絨山峰」）——另一種做形方式：茶葉較寬，微彎，似「雀舌」。香氣花香蘭花型，較少「炒製」感。口感較輕盈細膩，甜度明顯，無板栗調。

* **西湖龍井（西湖龍井，Xīhú Lóngjǐng）：** 浙江省。「龍井」——扁形茶（扁形），無絨毛。香氣豆香，「炒豌豆」味。相比信陽毛尖——口感更輕盈「順滑」，湯感較薄，但餘韻優雅悠長。

* **安吉白茶（安吉白茶，Ānjí Bái Chá）：** 浙江省。名為白茶實為綠茶，採用白化品種，氨基酸含量創紀錄（達6-7%）。口感極其溫和鮮爽（鮮），無板栗香。相比信陽毛尖——較不飽滿，較不澀，更「透明」。

**結語：**

信陽毛尖是一款矛盾的茶：誕生於茶葉種植的最北邊界，卻擁有驚人的飽滿度和複雜性。河南涼爽的山地氣候、持久的霧氣和酸性腐殖土壤，賦予它南方地區無法複製的特質：異常高的氨基酸濃度創造出鮮明的甘甜，而獨特的「雙鍋」技術和手工「理條」鎖定了標誌性的板栗香和銀色絨毛。

這是一款為欣賞內在力量而非表面精緻的人準備的茶：第一口的清新綠意之後，展現出飽滿、油潤的茶體，隨後是悠長的甜美餘韻——千年前蘇東坡將其列為偉大淮河茶葉之首。用80-85°C的軟水在玻璃杯中沖泡，觀賞銀針的「舞蹈」——這款茶必將向您展現其魅力。
