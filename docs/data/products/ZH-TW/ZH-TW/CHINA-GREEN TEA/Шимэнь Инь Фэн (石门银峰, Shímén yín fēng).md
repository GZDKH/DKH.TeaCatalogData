# 石門銀峰 (石门银峰, Shímén yín fēng)

**石門銀峰 (石门银峰, Shímén yín fēng)**
![Шимэнь Инь Фэн (石门银峰, Shímén yín fēng)](images/Шимэнь%20Инь%20Фэн%20(石门银峰,%20Shímén%20Yín%20Fēng).png)

![Шимэнь Инь Фэн (石门银峰, Shímén yín fēng)-1](images/Шимэнь%20Инь%20Фэн%20(石门银峰,%20Shímén%20Yín%20Fēng)-1.png)

石門銀峰——「石門的銀峰」——是一款年輕但迅速聲名鵲起的綠茶，產自湖南省西北部的石門縣。該茶於1991年在湖南農業大學朱先明教授的指導下，以古代皇室貢茶「牛抵茶」(牛抵茶) 為基礎創製而成。三十年來，石門銀峰獲得了一百多項國內外獎項，包括「中國馳名商標」地位（2012年）和2010年上海世博會「唯一官方茶葉」稱號。該茶生長在「黃金」北緯三十度，雲霧繚繞的武陵山脈——全國生態最潔淨的地區之一，富含硒元素。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)。未發酵；氧化程度極低（少於5%）。
* **類別：** 「湖南十大名茶」(湖南十大名茶, Húnán shí dà míng chá，自2005年起)。地理標誌保護產品——註冊為地理證明商標 (地理证明商标, dìlǐ zhèngmíng shāngbiāo，2007年)。「中國馳名商標」(中国驰名商标, Zhōngguó Chímíng Shāngbiāo，2012年)。
* **產地：** 中國湖南省 (湖南省, Húnán Shěng)，常德市 (常德市, Chángdé Shì)，石門縣 (石门县, Shímén Xiàn)。12個鄉鎮，3個農林場。
* **地理座標：** 東經110°29′–111°32′，北緯29°16′–30°08′。

**2. 歷史與文化意義：**

* **歷史：**

石門縣的茶葉歷史超過1700年。西晉（西晋，265–316）地理著作《荊州土地記》(《荆州土地记》, Jīngzhōu Tǔdìjì) 記載：「武陵七縣通出茶，最好」(武陵七县通出茶，最好)。石門屬於武陵郡 (武陵郡, Wǔlíng Jùn) 轄區。

唐代（唐朝，618–907）著名文學家劉禹錫 (刘禹锡, Liú Yǔxī) 任朗州（今常德）司馬 (司马) 時，寫下《西山蘭若試茶歌》(《西山兰若试茶歌》)，記錄了當地綠茶炒製技術。位於石門的夾山寺 (夹山寺, Jiāshān Sì) 誕生了著名的「茶禪一味」(茶禅一味, chá chán yī wèi) 理念，成為日本茶道的哲學基礎。

石門夾山寺出產的「牛抵茶」(牛抵茶, Niú Dǐ Chá——「頂牛茶」)，從宋代（宋，960–1279）到清朝末年（清，1644–1912）一直是皇室貢茶 (贡茶, gòng chá)。這在蔡襄 (蔡襄, Cài Xiāng) 的《茶錄》(《茶录》, Chá Lù) 和陳宗懋 (陈宗懋, Chén Zōngmào) 院士的《中國茶經》(《中国茶经》, Zhōngguó Chá Jīng) 中均有記載。

現代歷史：1991年，湖南農業大學 (湖南农业大学) 朱先明 (朱先明, Zhū Xiānmíng) 教授率領團隊，與石門茶葉公司合作改良歷史上的「牛抵茶」工藝，創製了新茶——「石門銀峰」。1993年即被認定為「湖南省名茶」。1994年榮獲亞太國際貿易博覽會金獎。2005年入選「湖南十大名茶」。2010年成為上海世博會國際茶文化節唯一官方茶葉。2012年獲得「中國馳名商標」地位。2021年成功通過省級專家評審，申報國家農產品地理標誌。截至2023年，品牌價值評估為24.45億元人民幣（另有評估為18億元）。

* **名稱：**

    * **石門 (石门, Shímén)** ——字面意思「石門」——湖南省西北部、武陵山脈東端的縣名。
    * **銀 (银, Yín)** ——「銀色」：指茶葉上覆蓋的濃密銀色茸毛。
    * **峰 (峰, Fēng)** ——「峰」、「山峰」：比喻茶葉的山地出身和芽尖的尖銳形狀。

書法家史穆 (史穆) 以藏頭詩讚頌此名：「石鼎烹泉活，門庭散倚霞，銀針初茁蕊，峰翠育新芽」——「石鼎中煮著活泉，門庭散發霞光；銀針初發嫩芽，翠峰孕育新葉」。

* **文化意義：** 石門銀峰是石門縣的主要茶葉品牌，也是湖南茶葉的旗艦之一。與夾山寺和「茶禪一味」理念的聯繫賦予了它特殊的「茶禪」地位。石門縣擁有「中國名茶之鄉」(中国名茶之乡) 和「中國茶禪之鄉」(中国茶禅之乡) 稱號。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要品種——石門群體種 (石门群体种, Shímén Qúntǐ Zhǒng)——當地百年群體品種，以及白毫早 (白毫早, Báiháo Zǎo)——國家優良早生品種，具有濃密茸毛。輔助品種：福鼎大白茶 (福鼎大白茶)、槠葉齊 (槠叶齐, Zhūyè Qí) 等無性系品種。*Camellia sinensis* var. *sinensis*。主要茶樹樹齡——30年以上。百芽重（一芽一葉）——約45克。
* **採摘：** 春季，清明 (清明，4月初) 前後。由於種植園海拔較高，茶樹萌發較晚，採摘時間比平原地區晚。對於「銀峰王」(银峰王，「銀峰之王」)——特級品種——採摘限制在清明前後2-3天，嚴格在晴天上午9:00至下午15:00進行。
* **採摘標準：** 特級——純芽頭 (单芽, dān yá)，長度不超過2.5公分。一級——一芽一葉初展（≥80%）。二級——一芽二葉。嚴格遵守「四不採」(四不采, sì bù cǎi) 原則：不採雨天芽葉，不採帶露芽葉，不採紫色芽葉，不採病蟲害芽葉。同時不允許魚葉 (鱼叶)、鱗片和葉柄。
* **原料要求：** 「嫩、勻、淨、齊」(嫩、匀、净、齐)。

**4. 風土條件與栽培特色：**

* **地形：** 石門縣位於武陵山脈 (武陵山脉, Wǔlíng Shānmài) 東端，處於四省（湖南、湖北、貴州、重慶）交界地帶。壺瓶山 (壶瓶山, Húpíng Shān) 是湖南最高峰（2098.7公尺），位於縣西部，是國家級自然保護區核心。該地區被認為是「第三紀動植物避難所」——歐亞大陸同緯度生態最完整的地區之一。
* **海拔：** 300–1200公尺。優質原料核心區——海拔600公尺以上的高山茶園，常年雲霧繚繞，山泉灌溉。
* **氣候：** 亞熱帶季風氣候，具明顯垂直梯度。年平均氣溫13–17°C（因海拔而異）。年降雨量1300–1900毫米。霧日超過180天。日夜溫差超過8°C。充足的散射光有利於氨基酸積累（春茶：氨基酸≥3.32%）。
* **土壤：** 黃棕壤 (黄棕壤, huáng zōng rǎng) 和紅壤 (红壤, hóng rǎng)，呈酸性（pH 5.0–6.5）。富含硒（0.82毫克/公斤）和鋅（1.6毫克/公斤）——石門屬於湖南「富硒帶」。
* **生態：** 森林覆蓋率81%。負氧離子濃度比城市高50倍。該地區獲得「中國天然氧吧」(中国天然氧吧) 認證。部分茶園通過歐盟有機認證。

**5. 製作工藝：**

石門銀峰工藝是改良的「炒青」(炒青) 技術，繼承自歷史上的「牛抵茶」並經朱先明教授團隊改良。特色是九道工序的多階段製程，包含獨創的「提毫」(提毫——提升茸毛) 技術和木炭烘焙。

* **攤青 (摊青 — tān qīng)：** 鮮葉攤放厚度不超過3公分，在清潔通風的室內放置4–10小時。目的是初步失水和形成香氣。

* **殺青 (杀青 — shā qīng)：** 平口鍋溫度140°C。投葉量500克。雙手炒製，交替使用「悶」(闷, mèn) 和「抖」(抖, dǒu) 手法。以失去光澤、葉質變軟、出現清香為準。

* **清風 (清风 — qīng fēng)：** 殺青葉在竹篩中拋動10次以上，快速散熱，防止悶黃 (闷黄)。

* **炒坯 (炒坯 — chǎo pī)：** 溫度85°C，雙手輕柔翻炒至含水量約40%。

* **緊條 (紧条 — jǐn tiáo)：** 溫度60°C。輕度單向揉捻至含水量約30%。動作由輕到重，間歇散團防止結塊。

* **理條 (理条 — lǐ tiáo)：** 溫度50°C。右手向前理直茶條，形成筆直的「針狀」外形。動作極其輕柔。乾燥度達80%。

* **攤涼 (摊凉 — tān liáng)：** 短暫（約4分鐘）晾涼，使水分均勻分布。

* **提毫 (提毫 — tí háo)：** 溫度50°C。有節奏地摩擦茶葉，顯露和整理銀色茸毛，在茶條表面形成特有的「銀毫」。這是石門銀峰的獨特工序。

* **烘焙 (烘焙 — hōng bèi)：** 木炭烘焙，溫度70°C，至含水量不超過5%。木炭烘焙 (木炭焙, mùtàn bèi) 技術能溫和地固定香氣，保護茸毫不受損傷。

**6. 感官特徵：**

* **乾茶外形：** 細緊勻直 (紧细匀直, jǐn xì yún zhí)，鋒苗顯露 (锋苗, fēng miáo)。銀毫滿披 (银毫满披, yín háo mǎn pī)。色澤翠綠油潤 (翠绿油润, cuì lǜ yóu rùn)。

* **乾茶香氣：** 嫩香 (嫩香, nèn xiāng) 清純高雅，帶栗香 (栗香, lì xiāng)——特級品尤為明顯。特有溫和花香，帶鈴蘭香韻。

* **茶湯香氣：** 清高持久 (清高持久, qīng gāo chíjiǔ)。主要特色是層次豐富：「頭泡清香 (清香)，二泡味濃 (味浓)，三四泡幽香猶存 (幽香犹存)」。據品茶師證實，優質樣品香氣可持續至第七泡。

* **滋味：** 鮮爽 (鲜爽, xiān shuǎng)，同時醇厚 (醇厚, chún hòu)。回甘 (回甘, huí gān) 持久明顯，回味甘甜 (回味甘甜, huí wèi gān tián)。澀味極輕。茶湯飽滿，有「鮮潤」感。

* **湯色：** 嫩綠明亮 (亮绿, liàng lǜ / 嫩绿明亮, nèn lǜ míng liàng)，清澈透明。

* **葉底：** 嫩綠鮮活勻整 (嫩绿鲜活匀整)。芽葉完整，舒展良好。

**7. 化學成分：**

* **茶多酚 (茶多酚, chá duōfēn)：** 不少於30%（一級品）。高含量歸因於高山風土和溫差。
* **兒茶素 (儿茶素, ér chá sù)：** EGCG含量創紀錄達8.22%，據生產商數據，這是中國綠茶中的最高水準之一。EGCG是最具生物活性的兒茶素，決定抗氧化潛力。
* **氨基酸 (氨基酸, ānjīsuān)：** 不少於3.32%（春茶）。L-茶氨酸提供鮮味和溫和的提神效果。
* **水浸出物 (水浸出物, shuǐ jìnchūwù)：** 不少於45%（特級）——高於綠茶平均水準，解釋了優異的耐泡性。
* **生物鹼：** 咖啡因約25–35毫克/克。可可鹼、茶鹼含量微量。
* **維生素：** 維生素C、B群維生素、維生素E。
* **礦物質：** 硒（土壤中0.82毫克/公斤，大部分轉移到茶葉中）、鋅（1.6毫克/公斤）、鉀、磷、鎂。
* **精油：** 栗香由吡嗪和呋喃酮形成；溫和花香由芳樟醇和香葉醇形成。

**8. 保健功效：**

* **強效抗氧化作用：** 創紀錄的EGCG含量（8.22%）提供綠茶中最高的抗氧化活性之一。多酚類化合物中和自由基，延緩細胞老化。
* **心血管系統支持：** 兒茶素加速脂肪代謝，有助降低低密度脂蛋白膽固醇，降低動脈硬化風險。
* **牙齒保護和抗菌效果：** 兒茶素抑制致齲細菌活性，減少牙菌斑形成。
* **提神和精神效果：** 咖啡因和L-茶氨酸的結合提供穩定持久的提神效果和思維清晰。
* **硒元素支持：** 由於富硒土壤，茶葉是生物可利用硒的天然來源——這種微量元素對免疫力和甲狀腺功能很重要。
* **改善消化：** 適度刺激消化酶分泌。
* **免疫支持：** 多酚類和維生素C共同增強機體防禦功能。

**9. 沖泡方法：**

* **水溫：** 80–85°C。絕對不能使用沸水：溫度超過85°C會破壞L-茶氨酸並導致過度苦澀。
* **茶量：** 3克配150毫升（1:50比例）。
* **茶具：** 透明玻璃杯 (透明玻璃杯)——最適合觀察銀芽舒展。優選山泉水；避免鹼性水，會破壞湯色。
* **沖泡過程：**
  1. **溫杯：** 用熱水沖洗玻璃杯。
  2. **投茶：** 上投法 (上投法, shàng tóu fǎ)：先注水，再輕輕投入茶葉。
  3. **首泡：** 1–2分鐘。
  4. **續泡：** 每泡增加30秒。優質石門銀峰可沖泡3–4次；頂級樣品可達5次以上。
  5. **注意：** 沿杯壁緩慢注水——防止茸毫散落和湯色渾濁。

**10. 儲存：**

* **條件：** 密封包裝，避光、防潮、防異味。最佳條件——冰箱0–5°C保存。
* **保質期：** 新鮮茶葉。最佳香氣和滋味——製作後前6個月。
* **建議：** 新茶在室溫陰涼處放置5–7天「退火」(褪火)。開封後冷藏保存，一個月內飲用完畢。

**11. 價格與假冒：**

* **價格區間：** 特級——1000–3000元人民幣/斤（500克）；一級——600–1200元；二級——400–800元。「銀峰王」(银峰王)——精品品種：2003年拍賣會上100克售價創紀錄的53,000元。
* **價格因素：** 等級、採摘時間（清明前茶更貴）、具體茶園（壺瓶山高山茶最貴）、手工vs機械加工、有機認證。
* **避免假冒方法：**
    * 從授權經銷商購買，帶有地理標誌「石門銀峰」標識。
    * 評估外觀：細直勻整的茶條，銀毫豐富完整。粗糙不勻的茶葉是假冒跡象。
    * 檢查香氣：清純高雅，無異味。有「陳味」或「煙味」不正常。
    * 測試耐泡性：正宗銀峰香氣至少持續到第三、四泡。
    * 警惕一級品低於400元/斤的價格。

**12. 趣聞軼事：**

* **皇室貢茶傳人：** 石門銀峰是「牛抵茶」(牛抵茶，「頂牛茶」) 的直接傳承，後者從宋代到清代一直是皇室貢茶——約800年的宮廷「生涯」。
* **「茶禪一味」發源地：** 石門夾山寺是著名「茶禪一味」(茶禅一味) 理念的誕生地，唐代僧人善會 (善会) 首次提出，宋代禪師圓悟克勤 (圆悟克勤) 書寫並傳入日本，成為日本茶道的基石。
* **53,000元100克：** 2003年石門名茶拍賣會上，100克「銀峰王」以53,000元成交——創紀錄價格，使品牌聞名全國。
* **「第三紀避難所」：** 核心茶園所在的壺瓶山是古代第三紀動植物的「避難所」——保存了在歐亞大陸其他地區已消失的物種。這是北緯30度線上生態最純淨的地區之一。
* **富硒天堂：** 石門縣土壤屬於湖南「富硒帶」——中國少數幾個硒元素以生物可利用形式天然存在於土壤中的地區之一。

**13. 與其他綠茶比較：**

* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南名茶，同樣茸毫豐富。毛尖外形更「修長」，花香清雅。石門銀峰茶湯更飽滿，栗香更突出，耐泡性更佳。
* **上饒白眉 (上饶白眉, Shàngráo Bái Méi)：** 江西「眉茶」，茸毫豐富。外觀相似，但銀峰使用不同品種（山地群體vs大面白），栗香更深，茶湯更厚重。
* **古丈毛尖 (古丈毛尖, Gǔzhàng Máojiān)：** 另一款湖南武陵山區名茶，但來自山脈西部。更清淡，「綠味」和草本味明顯。石門銀峰更厚重，回甘更持久。
* **安化松針 (安化松针, Ānhuà Sōngzhēn)：** 湖南「針形」茶。外形更細長，「松香」清雅。銀峰更濃郁，突出栗香和耐泡性。

**結語：**

石門銀峰是一款融合武陵地區1700年茶葉傳統與現代科學方法的茶葉。在其「銀毫」背後，蘊含著皇室「牛抵茶」的傳承、夾山寺禪茶哲學、壺瓶山生態純淨和創紀錄的EGCG含量，使每一杯茶不僅是享受，更是健康投資。「頭泡清香，二泡味濃，三四泡幽香猶存」——這個成為品牌座右銘的公式，準確描述了這款從第一滴到最後一滴都能帶來驚喜的茶葉特性。
