# 綠寶石茶 (绿宝石茶, lǜbǎoshí chá)

**綠寶石茶 (绿宝石茶, lǜbǎoshí chá)**
![Люйбаоши Ча (绿宝石茶, lǜbǎoshí chá)](images/Люйбаоши%20Ча%20(绿宝石茶,%20Lǜbǎoshí%20Chá).png)

![Люйбаоши Ча (绿宝石茶, lǜbǎoshí chá)-1](images/Люйбаоши%20Ча%20(绿宝石茶,%20Lǜbǎoshí%20Chá)-1.png)

綠寶石茶 (绿宝石茶, lǜbǎoshí chá —「綠翡翠茶」) — 創新的貴州綠茶，於西元2003年由「貴州茶業之父」牟應書 (牟应书, Mù Yìngshū) 創制，這位茶葉科學家實現了概念性突破：他證明了使用中等嫩度原料（一芽二三葉）— 傳統上被認為是「低等級」的材料 — 可以生產出優質茶葉。這個原則 —「以中低端原料制高端茶」— 顛覆了傳統範式，該範式將品質完全與芽葉嫩度掛鉤。綠寶石茶成為貴州茶葉在國際舞台上的名片：出口至德國、美國和日本。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵）。按形狀 — 顆粒珠茶 (珠茶/盘花茶, zhūchá / pánhuāchá)，捲成緊密圓形「螺旋」。工藝 — 炒青 (炒青) 配合最終烘乾。

* **類別：** 「貴州十大名茶」(贵州十大名茶) 之一。國家生態原產地保護產品 (国家生态原产地保护产品)。西元2014年 — 國家地理標誌保護。自西元2006年起 — 多次獲得國際茶葉大會金獎。

* **產地：** 中國貴州省 (贵州, Guìzhōu)。精確的地理標誌保護區域涵蓋該省多個專門從事高山有機茶業的縣。原料來自老樹茶園 (树龄多超50年) — 樹齡超過50年的茶樹。

* **地理座標：** 貴州省，約北緯26–28°，東經106–108°。

**2. 歷史與文化意義：**

* **歷史：** 綠寶石茶是中國最年輕的「名茶」之一：誕生日期 — 西元2003年。其創制者 — **牟應書 (牟应书)**，傳奇的貴州茶葉科學家，數十年致力於研究當地茶葉資源。西元2003年，利用當地茶樹群體種和創新的顆粒化技術，牟應書創制了與傳統綠茶截然不同的茶葉：他不使用最嫩的芽頭，而是使用「一芽二三葉」的芽葉，將其轉化為緊密、厚重的「翡翠」顆粒，具有卓越的耐泡性（7次以上沖泡）。

	認可來得很快：自西元2006年起 — 在國際競賽中連續獲得金獎。西元2013年 — 「貴州著名商標」。西元2014年 — 地理標誌保護。2010年代，茶葉開始出口至德國、美國、日本和香港，成為少數幾個具有國際銷售網路的貴州茶葉之一。

* **名稱：**

	* 「綠」(绿) — 「綠色」：茶葉的顏色。
	* 「寶石」(宝石) — 「寶石、珍寶」：比喻描述緊密、光亮的顆粒，如同拋光的翡翠。

* **文化意義：** 綠寶石茶是貴州茶業「新浪潮」的象徵：不是模仿華東茶葉風格（如湄潭翠芽受龍井啟發），而是創造自己獨特的茶葉類型。牟應書的哲學 —「品質不是由原料嫩度決定，而是由加工技藝決定」— 成為貴州茶業前衛派的宣言。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要栽培種 — **福鼎大白茶 (福鼎大白茶, Fúdǐng Dà Bái Chá)** — *Camellia sinensis* var. *sinensis* 的無性系中葉品種。另外 — **貴州本地群體種 (贵州本地群体种)** 灌木型，許多茶樹樹齡超過50年。葉片 — 橢圓形，肉質，厚實易碎；百芽三葉重量 — 約70公克。

* **採摘：** 採摘標準 — **一芽二三葉** (一芽二三叶, yī yá èr sān yè) — 與大多數使用最嫩芽頭的「名茶」綠茶根本不同。這是牟應書的概念性決定：較成熟的原料提供更多浸出物，確保卓越的耐泡性和豐富的口感。

* **原料要求：** 新鮮、健康的芽葉，來自通過生態認證的茶園。當日採摘當日加工。

**4. 風土與栽培特點：**

* **氣候：** 貴州高原。亞熱帶濕潤季風氣候。雲霧豐富，散射光充足，日夜溫差顯著。這些條件是所有貴州茶葉的特徵：氨基酸和芳香物質含量較高。

* **海拔高度：** 主要在海拔700–1200公尺。

* **土壤：** 酸性黃壤 (黄壤)，貴州特有，天然含硒和鋅。

* **生態：** 貴州是中國生態最清潔的省份之一。森林覆蓋率高，工業污染極少。許多茶園為有機茶園。

* **特點：** 使用老樹 (树龄超50年) — 不是行銷手段，而是有意識的選擇：老茶樹的根系深入土壤更深，提取更多礦物質，形成更複雜的風味特徵。

**5. 製作工藝：**

綠寶石茶工藝是牟應書的原創開發，結合貴州傳統茶業原則與創新的成型方法。

* **進料檢驗 (鲜叶抽检 — xiānyè chōujiǎn)：** 加工前對原料品質進行實驗室檢驗。

* **攤放與萎凋 (摊青/自然萎凋 — tān qīng / zìrán wěidiāo)：** 自然萎凋4–6小時。

* **殺青 (杀青 — shāqīng)：** 在180–200°C溫度下，採用「抖悶結合」(抖闷结合, dǒu mèn jiéhé) — 師傅交替拋撒茶葉（排出水蒸氣）和壓制（深度加熱）。這確保較粗原料的均勻殺青。

* **揉捻 (揉捻 — róuniǎn)：** 機械分段加壓 (机械分段加压)。比嫩茶更強烈 — 為了從成熟原料形成緊密顆粒所必需。

* **乾燥提香 (干燥提香 — gānzào tíxiāng)：** 低溫慢烘 (80°C, 低温慢烘) — 溫和過程，保持栗香。含水量 — ≤5%。

* **精製 (精制 — jīngzhì)：** 篩分 (筛分)，色選 (色选)，勻堆 (匀堆)。高度標準化。

* **特點：** 全程使用竹木器具避金屬氧化 (全程竹木器具避金属氧化) — 牟應書的哲學，旨在防止氧化並保持香氣純淨。

**6. 感官特徵：**

* **乾茶外觀：** 緊密、厚重的螺旋形顆粒 (颗粒盘花状, kēlì pánhuā zhuàng —「螺旋花狀顆粒」)。顏色 — 深綠色帶油潤光澤 (绿润显毫)。茸毛 — 明顯，銀白色。顆粒 — 均勻，手感厚重 (紧实重实)。

* **乾茶香氣：** 栗香 (栗香, lì xiāng) — 主導香調。新茶有花果香 (花果香) 韻味。炒米香 (炒米香, chǎomǐ xiāng) — 傳統「火工」處理的香調。

* **茶湯香氣：** 栗香，持久深沉。在多次沖泡中持續展現。

* **滋味：** 鮮爽 (鲜爽, xiānshuǎng) — 氨基酸特徵。醇厚 (醇厚, chúnhòu) — 茶湯黏稠感強 (粘稠感强)。回甘 (回甘, huígān) — 持續而悠長。主要特點 — **卓越的耐泡性：7次以上沖泡**，保持滋味和香氣。這明顯高於大多數綠茶（3–5次沖泡），這是由於使用較成熟原料，浸出物含量較高。

* **湯色：** 黃綠明亮 (黄绿明亮)。

* **葉底（沖泡後茶葉）：** 嫩綠鮮活、柔軟勻整 (嫩绿鲜活、柔软匀整)，從顆粒展開成完整的「芽葉」。葉片 — 完整，均勻。

**7. 化學成分：**

* **多酚類（兒茶素）：** 含量 — ≥27.5% 乾重。清除自由基能力 — 比維生素E高18倍（根據研究資料）。

* **氨基酸（包括L-茶氨酸）：** 含量顯著 — 提供鮮爽和「鮮味」。較成熟原料（一芽二三葉）的氨基酸含量比最嫩芽頭少，但工藝通過優化殺青和乾燥過程來補償。

* **生物鹼：** 咖啡因 — 適中含量。

* **氟：** 含量明顯 — 有助於保護牙釉質 (护齿防龋)。

* **礦物質：** 鉀、鎂、鋅、錳、氟 — 由貴州土壤決定。

* **維生素：** 維生素C，B群維生素。

**8. 保健功效：**

* **強效抗氧化作用：** 多酚類（≥27.5%）清除自由基效率比維生素E高18倍。
* **體重控制和血脂調節：** 兒茶素促進脂肪分解，效率比一般綠茶高30%。
* **護齒防蛀 (护齿防龋)：** 氟抑制致蛀菌並強化牙釉質。
* **提神效果：** 咖啡因和L-茶氨酸。
* **改善消化：** 多酚類刺激脂肪分解。

* **重要提醒：** 上述功效基於公開資料，不構成醫療建議。

**9. 沖泡方法：**

* **熱泡法 (热泡法, rè pào fǎ)：**
	* **水溫：** 85–90°C。
	* **茶量：** 3公克配150毫升水（比例1:50）。
	* **茶具：** 白瓷蓋碗 — 控制香氣和出湯時間的最佳選擇。
	* **過程：** 10–15秒出湯（1–3泡）；20–25秒（4–7泡）。茶葉可沖泡**7次以上**。

* **冷泡法 (冷泡法, lěng pào fǎ)：**
	* 1公克茶配50毫升冷水。
	* 冰箱冷藏30分鐘。
	* 結果 — 鮮爽度提升 (鲜爽度提升)。

* **注意：** 不使用沸水（>90°C）— 破壞生物活性物質並加重苦澀。新茶建議醒茶 (醒茶) 7天置於陰暗處。開封後 — 10天內飲用完畢。

**10. 儲存：**

* 密封容器，陰涼乾燥處保存。
* 最佳 — 冰箱0–5°C。
* 保存期限 — 12個月。
* 開封後 — 10天內飲用完畢以保持最佳新鮮度。

**11. 價格與仿冒：**

綠寶石茶具有平民化的價格結構：「以中低端原料制高端茶」的哲學意味著即使基礎等級也提供卓越的性價比。

價格參考：特級上等 — 500元人民幣/斤起；特級 — 200–400元；一級 — 200元以下。

* **如何避免仿冒：**

	* **向可信賴的經銷商購買**，有生態原產地或地理標誌認證標識。
	* **評估外形：** 緊密、厚重的「螺旋」顆粒，帶油潤光澤。鬆散、輕飄的球狀 — 仿冒跡象。
	* **檢驗耐泡性：** 7次以上沖泡 — 真品標誌。如果茶葉3–4泡後就「失去活力」— 可疑。
	* **評估香氣：** 栗香，持久。微弱或「化學」味 — 仿冒品。

**12. 有趣事實：**

* 牟應書 (牟应书) — 「貴州茶業之父」，其「以中低端原料制高端茶」概念顛覆了傳統觀念，即優質綠茶只能用最嫩芽頭製作。他的方法為合理利用貴州茶葉資源開闢了道路。

* 耐泡性 — 7次以上沖泡 — 使綠寶石茶在「持久性」方面與烏龍茶和普洱茶並列。對綠茶而言這是極其罕見的指標。

* 綠寶石茶多酚類的抗氧化效率 — 比維生素E高18倍 — 是綠茶中最令人印象深刻的指標之一。

* 完全不使用金屬器具 (全程竹木器具) — 不是復古風格，而是有意識的技術選擇：竹木不催化多酚類氧化，保持新鮮度和綠色的鮮豔。

* 綠寶石茶是少數成功進入已開發國家市場（德國、美國、日本）的貴州茶葉之一，證實其符合國際品質和安全標準。

**13. 與其他貴州茶和顆粒綠茶的比較：**

* **都勻毛尖 (都匀毛尖)：** 捲曲針形茶，使用純芽頭。都勻更細膩「精緻」；綠寶石更厚重、耐泡且價格「平民化」。

* **湄潭翠芽 (湄潭翠芽)：** 扁形茶，使用純芽頭。湄潭在視覺上更精美；綠寶石在滋味上更濃郁且耐泡。

* **涌溪火青 (涌溪火青)：** 來自安徽。同樣是顆粒茶，同樣極其耐泡。涌溪 — 20小時炭焙產生杏黃湯色；綠寶石 — 更經典的綠茶特徵和栗香調。

* **浙江珠茶 (浙江珠茶)：** 經典出口「珠茶」(Gunpowder)。浙江珠茶 — 更大眾化簡單；綠寶石 — 明顯更複雜芳香，具有鮮明的風土特徵。

**結論：**

綠寶石茶是宣言茶，是對千年教條「只有最嫩芽頭才能製出偉大茶葉」的抗議茶。牟應書證明了加工技藝可以將「普通」原料轉化為非凡品質的產品：緊密的翡翠顆粒，在七次以上沖泡中展現，帶著栗香和油潤甘甜，— 這不是妥協，而是獨立風格，不遜色於嫩芽茶，在某些方面甚至超越它們。如果都勻毛尖是貴州的優雅，湄潭翠芽是其技術性，那麼綠寶石茶就是其叛逆精神：從異端和技藝中誕生的綠色翡翠。
