# 松峰綠茶 (松峰绿茶, Sōngfēng lǜchá)

**松峰綠茶 (松峰绿茶, Sōngfēng lǜchá)**

![Сунфэн Люй Ча (松峰绿茶, Sōngfēng lǜchá)](images/Сунфэн%20Люй%20Ча%20(松峰绿茶,%20Sōngfēng%20Lǜchá).png)

![Сунфэн Люй Ча (松峰绿茶, Sōngfēng lǜchá)-1](images/Сунфэн%20Люй%20Ча%20(松峰绿茶,%20Sōngfēng%20Lǜchá)-1.png)

松峰綠茶 (松峰绿茶, Sōngfēng lǜchá) — 湖北綠茶，由明朝開國皇帝朱元璋 (朱元璋) 於西元1368年親自命名。正是在這裡，在松峰山 (松峰山，「松樹山峰」) 山腳下，在古老的茶葉村落羊樓洞 (羊楼洞, Yánglóudòng)，實現了著名的西元1391年皇帝詔書「罷造龍團，唯採茶芽以進」(罢造龙团，唯采茶芽以进) — 這道詔令結束了緊壓茶時代，開啟了散葉綠茶的紀元。羊樓洞後來成為長達14,000公裡的萬裡茶道 (万里茶道) 起點 — 從湖北經蒙古到莫斯科，再延伸至歐洲。西元2015年，國際茶葉委員會授予羊樓洞「世界茶業第一古鎮」(世界茶业第一古镇) 稱號。松峰綠茶是著名赤壁磚茶 (青砖茶) 的綠茶「雙胞胎」，誕生於同樣的山脈和同樣的千年傳統。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，帶有黃茶加工元素：生產週期中包含「初包悶黃」(chūbāo mènhuáng，「第一次包裹發黃」，40–48小時) 工序 — 介於綠茶和黃茶之間的邊界技術。正式分類為綠茶。外形 — 「松針形」(松针形, sōngzhēn xíng)，捲曲。

* **類別：** 歷史名茶，由皇帝朱元璋命名 (1368年)。入選「全國十大名茶」(全国十大名茶, 1958年)。兩度榮獲「部優產品」(部优产品, 1982年、1985年) 稱號。榮獲湖北「楚茶杯金獎」(楚茶杯金奖, 2023年)。出口至亞洲和非洲10多個國家。

* **產地：** 中國；湖北省 (湖北, Húběi)；赤壁市 (赤壁市, Chìbì Shì，古稱蒲圻，蒲圻)，隸屬咸寧市 (咸宁市, Xiánníng Shì)。核心產區 — 羊樓洞鎮 (羊楼洞镇, Yánglóudòng Zhèn)，位於松峰山 (松峰山) 山腳下，湖南、湖北、江西三省交界處。羊樓洞 — 「世界茶業第一古鎮」(世界茶业第一古镇，自2015年)，萬裡茶道源頭 (万里茶道源头)。

* **地理座標：** ~29°30′–30°00′ 北緯，113°30′–114°00′ 東經 (赤壁地區，「黃金緯度」北緯30°)。

**2. 歷史與文化意義：**

* **歷史：**

松峰茶的歷史就是散葉綠茶作為類別誕生的歷史。赤壁地區 (古蒲圻) 的茶葉傳統已有2000多年歷史。早在唐代，該地區就被劃為「茶園戶」(园户, yuánhù)，羊樓洞的「洞茶」(洞茶, dòngchá) 成為貢茶 (贡茶)。

與松峰茶皇帝命名相關的傳奇事件發生在元朝末年：朱元璋 (朱元璋, 1328–1398)，未來的明朝開國皇帝，在軍事征戰期間與軍隊在蒲圻駐紮。士兵們飯後腹痛；羊樓洞當地茶農向他們提供蒲圻綠茶 — 疼痛停止了。西元1368年明朝建立後，感激的朱元璋賜予茶葉「松峰」(松峰，「松樹山峰」) 之名，並將茶樹生長的山命名為松峰山。

但最重要的事件發生在西元1391年 (洪武24年)：朱元璋頒布著名詔書「罢造龍團，唯采茶芽以進」— 「廢除龍團餅，只進獻茶芽」。這道詔令結束了數百年的緊壓茶 (团茶, tuánchá) 傳統，開啟了散葉綠茶 (散茶, sǎnchá) 時代，這個時代延續至今。羊樓洞成為首批實現新形式 — 炒製散葉綠茶 — 的地方之一。

清代，羊樓洞發展成國際茶葉貿易中心：在不到2平方公裡的面積上，有200多家茶葉公司，人口超過40,000人，城市贏得「小漢口」(小汉口) 美譽。從這裡開始萬裡茶道 (万里茶道, Wànlǐ Chádào) — 長達14,000公裡的貿易路線，從湖北經蒙古到俄國，再延伸至歐洲。卡爾·馬克思在《論中俄茶葉貿易》中提到羊樓洞磚茶，列寧親自接見羊樓洞茶商劉峻周 (刘峻周)。西元1956年，羊樓洞入選全國保留的三家茶廠之一。2015年，國際茶葉委員會授予其「世界茶業第一古鎮」(世界茶业第一古镇) 稱號。

* **名稱：** 松峰 (Sōngfēng) — 「松樹山峰」— 朱元璋親自賜予山和茶的名字。松 (sōng) — 松樹，象徵堅韌和長壽；峰 (fēng) — 山峰、頂峰；綠茶 (Lǜchá) — 「綠茶」。完整含義：「松峰綠茶」。

* **文化意義：** 松峰綠茶是少數由王朝開國皇帝賜名的茶葉之一。羊樓洞不僅是生產地，更是「世界茶葉遺產」：其石板街道，數百年來獨輪車「雞公車」(鸡公车) 載著磚茶滾過，已列入萬裡茶道聯合國教科文組織世界遺產申報預備名單 (万里茶道申遗)。松峰茶廠旁設有面積2600平方公尺的中國青磚茶博物館 (中国青砖茶博物馆)。赤壁是著名赤壁之戰 (赤壁之战, 西元208年) 發生地，這是三國時期的關鍵戰役，被羅貫中在小說中歌頌。因此，一座城市與中國歷史上兩場最偉大的「戰役」相關：軍事戰役 (西元208年) 和茶葉戰役 (西元1391年)。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要 — 羊樓洞群體種 (羊楼洞群体种, Yánglóudòng Qúntǐzhǒng) — 百年本地中小葉種 *Camellia sinensis* var. *sinensis*。樹齡30年以上。輔助 — 鄂茶1號 (鄂茶1号, Èchá 1 Hào) 和福鼎大白茶 (福鼎大白茶) — 早熟無性系品種，具有高發芽能力。100個「芽加一葉」芽頭重量 — ~45公克。持嫩期 (持嫩期, chí nèn qī) 延長 — 羊樓洞品種的特徵之一。

* **採摘：** 春季 — 3月底至4月。採用嚴格的「九不採」(九不采, jiǔ bù cǎi) 標準：不採雨水、露水、紫色、病蟲害、蟲蛀、畸形、空心、開裂芽頭和頂端受損芽頭。

* **採摘標準：** 特級 — 單芽 (≥90%)。一級 — 芽加一個半開葉 (≥80%)。二級 — 芽加兩葉。

* **原料要求：** 嫩芽完整，符合「九不採」標準。

**4. 風土條件與栽培特點：**

赤壁位於幕阜山脈 (幕阜山, Mùfùshān) 北麓，三省交界處 (湖南、湖北、江西)，「黃金茶葉緯度」北緯30°。

* **生長海拔：** 300–600公尺 (松峰山南坡)。

* **氣候：** 北亞熱帶地區。年平均氣溫16.8°C；降雨量1560毫米/年；雲霧天數佔相當比例；散射光充足，有利於氨基酸合成。

* **土壤：** 黃棕壤 (黄棕壤)，pH 4.5–6.5，富含鋅和硒。天然灌溉 — 松峰山山泉。

* **生態：** 森林覆蓋率 >72%。該地區獲得「中國天然氧吧」(中国天然氧吧) 認證。空氣中負離子含量高。茶園被松林環繞 — 因此得名松峰山。茶園供水來自松峰山坡流下的山泉，未受人為污染。幕阜山脈的接近形成天然氣候屏障，阻擋濕氣並形成持續霧氣 — 氨基酸積累的理想條件。

**5. 製作工藝：**

松峰綠茶採用獨特工藝，包含「悶黃」(闷黄, mènhuáng，「包裹發黃」) 工序，這是黃茶特有工序，但在此用於形成栗香，而非經典的「發黃」。

* **攤放 (摊放 — tān fàng)：** 標準工序。

* **殺青 (杀青 — shāqīng)：** 在斜鍋 (斜锅, xiéguō) 中進行 — 溫度在炒製過程中從100–120°C降至80°C。斜鍋是罕見技法，提供梯度處理：葉片上部受熱較強，下部較溫和。

* **攤涼 (摊凉 — tān liáng)：** 乾燥前停止熱處理。

* **初焙 (初焙 — chū bèi)：** 50–60°C。

* **初包悶黃 (初包闷黄 — chū bāo mèn huáng)：** 40–48小時 — 關鍵工序！茶葉用布或紙包裹，在溫暖潮濕環境中保持。微生物和酶促過程形成特有栗香 (栗香) 並軟化澀味。此工序使松峰與黃茶 (蒙頂黃芽、君山銀針) 相似，但此處目的不同 — 為了香氣而非顏色。

* **復焙 (复焙 — fù bèi)：** 固定初次「悶黃」結果。

* **復包 (复包 — fù bāo)：** 20小時 — 較短時間，最終穩定香氣。

* **足火三焙 (足火三焙 — zú huǒ sān bèi)：** 50–55°C，三次連續加熱 — 確保含水量 ≤5%，形成「熟栗香」(熟栗香)。

* **儲存 — 石膏貯藏法 (石膏贮藏法, shígāo zhùcáng fǎ)：** 羊樓洞傳統方法：茶葉儲存在密封容器中，放入石膏塊吸收濕氣並穩定環境。此方法確保香氣的卓越保存。

**6. 感官特徵：**

* **乾茶外觀：** 細長、均勻、捲曲的「松針」(针芽状, zhēnyá zhuàng)，緊實均勻 (紧细匀整)。顏色 — 翠綠油潤 (翠绿油润)。

* **乾茶香氣：** 栗香 (栗香, lì xiāng) — 主導香調，高銳。清香 (清香) — 輔助背景。「熟栗香」(熟栗香) — 三次最終烘焙的結果。

* **茶湯香氣：** 栗香綠香，高而穩定。冷卻後 — 甜美「麵包」香調。

* **滋味：** 鮮爽 (鲜爽) 同時醇厚 (醇厚, chúnhòu) — 不尋常的組合，標準綠茶不典型，由「悶黃」工序解釋。黏稠度和「體感」高於平均水準。持久甘甜回味 (回甘, huígān)。

* **湯色：** 透明、清澈明亮 (清澈明亮)。

* **耐泡性：** 二級茶5+泡 — 良好指標，由高水浸出物含量 (≥45%) 決定。特級和一級茶因原料濃縮度更高而更耐泡。

* **葉底 (沖泡後茶葉)：** 嫩綠勻齊鮮活 (嫩绿匀齐鲜活)。

**7. 化學成分：**

* **多酚類 (茶多酚)：** ≥30% (一級) — 高含量，提供強大抗氧化活性。

* **水浸出物 (水浸出物)：** ≥45% (特級) — 綠茶中最高指標之一，證明茶湯「濃度」卓越。

* **氨基酸 (氨基酸)：** 充足水準確保鮮爽度。L-茶氨酸佔主導。

* **生物鹼：** 咖啡因 — 適中，提供提神效果。

* **礦物質：** 鋅和硒 — 幕阜山脈地球化學標誌。鉀、錳、氟。

* **維生素：** 維生素C、B群維生素、維生素E。

**8. 保健功效：**

* **消化支持：** 朱元璋傳說中記錄的歷史功效：茶葉緩解士兵腹痛。多酚和單寧刺激胃腸蠕動。

* **抗氧化保護：** 多酚 (≥30%) 提供強大自由基中和作用。

* **提神效果：** 咖啡因 + L-茶氨酸 — 溫和、持久的清醒。

* **認知功能：** L-茶氨酸刺激大腦α波活動。

* **重要提醒：** 所列功效基於一般數據，非醫療建議。不建議空腹飲用。新茶需放置7天以「降火」。開封後10天內飲用完畢。

**9. 沖泡方法：**

* **水溫：** 80–85°C。不使用沸水 (>85°C「燙傷」茶葉，產生混濁茶湯)。

* **茶量：** 3公克配150毫升 (比例1:50)。

* **茶具：** 玻璃杯 (上投法, 上投法) 或白瓷蓋碗 (中投法, 中投法)。

* **沖泡過程 (玻璃杯)：** 注入80–85°C水，然後投入茶葉。第一泡30秒。每次增加10秒。

* **沖泡過程 (蓋碗)：** 注入1/3水量，投茶，搖香 (摇香, yáo xiāng) 喚醒香氣，加滿水。浸泡2–3分鐘。

* **注意：** 注水時沿杯壁緩流 (沿杯壁缓流)，而非直衝中心 — 防止「沖散」茸毛和茶湯混濁。

**10. 儲存方法：**

* **溫度：** 0–5°C (冰箱) 或傳統石膏貯藏法 (石膏贮藏法)。
* **容器：** 密封；傳統方法 — 陶瓷罐配天然石膏塊 (石膏, shígāo)。石膏作為天然乾燥劑：其晶體結構吸收容器內空氣中的濕氣，維持阻止氧化但不過度乾燥茶葉的濕度水準。這種羊樓洞茶農數百年前開發的方法，在效果上可與現代真空包裝相媲美。
* **光線：** 完全隔離。
* **保存期：** 開封後10天內享受最佳香氣。密封狀態下可保存12個月。

**11. 價格與仿冒：**

松峰綠茶屬中等至中高價位茶葉。特級 (單芽) — 500元/斤起；一級 — 200–400元/斤；二級 — 更實惠。

* **避免仿冒方法：**

	* 檢查產地 — 羊樓洞、赤壁、湖北。
	* 栗香 (栗香/熟栗香) — 「悶黃」工藝的名片。缺乏此香 — 普通綠茶替代品跡象。
	* 外形 — 「松針」，緊實均勻。無形茶葉 — 非松峰茶。
	* 滋味 — 同時鮮爽且醇厚 (鲜爽 + 醇厚)。僅「鮮爽」無「醇厚」— 普通綠茶。

**12. 有趣事實：**

* **改變茶世界的詔書：** 朱元璋西元1391年詔書「罢造龍團，唯采茶芽以進」— 茶業史上最具影響力的文件之一。它結束了500年緊壓茶時代，開啟散葉茶紀元。羊樓洞是首批實現這一轉變的地方之一。

* **馬克思和列寧：** 卡爾·馬克思在《論中俄茶葉貿易》中提到羊樓洞磚茶。列寧親自接見羊樓洞茶商劉峻周。松峰茶和磚茶是同一地方的「綠色和黑色雙胞胎」。

* **14,000公裡茶路：** 始於羊樓洞的萬裡茶道，從湖北經漢口，沿漢水上行至襄陽，然後經張家口 (或包頭) 進入蒙古，再經恰克圖到俄國、莫斯科、聖彼得堡和歐洲。總長度14,000+公裡。

* **「小漢口」：** 高峰期 (19世紀)，小小的羊樓洞 (面積<2平方公里) 容納200多家茶廠和40,000居民 — 世界茶業史上無與倫比的茶業密度。在皇家地圖《大清皇輿全圖》(《大清皇舆全图》) 上，羊樓洞與漢口、武昌使用相同字體標註 — 對一個村鎮來說是前所未有的榮譽。

* **綠茶中的「悶黃」：** 「初包悶黃」工序 (40–48小時！) 使松峰與黃茶相似，但此處用於形成栗香而非改變顏色。這是綠茶和黃茶之間的技術「橋樑」。

**13. 與其他湖北綠茶比較：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北。蒸青 (蒸青)，「松針」形。香氣 — 「海洋味」，無栗香。松峰 — 炒青配「悶黃」，明顯栗香特徵和更強「體感」。兩者都是「松針」，但工藝根本不同。

* **三峽龍井 (三峡龙井, Sānxiá Lóng Jǐng)：** 湖北宜昌。扁平，炒青。竹炭栗香 (三峡辉锅)。松峰 — 捲曲，「悶黃」栗香。兩者都有「溫暖」特徵，但香氣形成機制不同。

* **鄧村綠茶 (邓村绿茶, Dèngcūn Lǜchá)：** 湖北宜昌。經典炒青綠茶，捲曲形。特徵更「標準」，無「悶黃」。松峰 — 滋味更有「深度」和「醇厚」。

* **蒙頂黃芽 (蒙顶黄芽, Méngdǐng Huáng Yá)：** 四川。經典黃茶，完全「悶黃」。香氣 — 「穀物」、「麵包」；滋味 — 更溫和「乾爽」。松峰 — 正式為綠茶，更明顯「綠色」鮮爽，但部分「悶黃」帶來栗香「深度」。

**結論：**

松峰綠茶 — 誕生於歷史轉折點的茶葉：在這裡，在松峰山上，千年「龍團餅」被埋葬，散葉綠茶誕生，成為我們今天所知的形式。皇帝賜名，「悶黃」工藝在綠色鮮爽中創造栗香深度，以及地址 — 羊樓洞，「世界茶業第一古鎮」，從這裡14,000公裡茶路通向莫斯科和倫敦的餐桌，— 這一切使松峰不僅是茶葉，更是茶文明的活化石。以80°C沖泡，沿杯壁注水，— 品飲六個半世紀前治癒未來皇帝士兵的那款茶。
