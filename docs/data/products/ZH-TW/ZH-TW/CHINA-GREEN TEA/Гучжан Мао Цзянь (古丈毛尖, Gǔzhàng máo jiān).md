# 古丈毛尖 (古丈毛尖, Gǔzhàng máo jiān)

**古丈毛尖 (古丈毛尖, Gǔzhàng máo jiān)**

![Гучжан Мао Цзянь (古丈毛尖, Gǔzhàng máo jiān)-1](images/Гучжан%20Мао%20Цзянь%20(古丈毛尖,%20Gǔzhàng%20Máo%20Jiān)-1.png)

古丈毛尖 (古丈毛尖, Gǔzhàng máo jiān) 是來自湖南省湘西土家族苗族自治州古丈縣的著名綠茶，該縣位於武陵山脈 (武陵山脉) 的心臟地帶，處於「黃金三十度」——這條傳奇緯度孕育了中國最優質的茶葉。這款茶以兩種罕見結合的品質而聞名：對多次沖泡的卓越耐受性（可達15泡）以及土壤中豐富的硒含量。纖細挺拔的芽葉覆蓋著銀白色茸毛，被譽為「綠茶中的珍品」(绿茶中的珍品)。

**1. 分類與產地：**

* **類型：** 綠茶（未發酵茶）。屬於炒青綠茶 (炒青绿茶, chǎoqīng lǜchá)，具有直針狀外形。

* **等級：** 國家地理標誌保護產品 (国家地理标志保护产品, 2007年)。中國馳名商標 (中国驰名商标, 2011年)。1982年入選「中國十大名茶」（按其中一個版本的排名）。

* **產地：** 中國湖南省 (湖南, Húnán)，湘西土家族苗族自治州 (湘西土家族苗族自治州, Xiāngxī Tǔjiāzú Miáozú Zìzhìzhōu)，古丈縣 (古丈县, Gǔzhàng Xiàn)。核心產區——古陽鎮 (古阳镇)，東方 (东方)、龍天坪 (龙天坪) 和牛角山 (牛角山) 茶山。

* **擴展區域：** 沿武陵山脈北緯三十度「黃金茶帶」，橫跨湖南、湖北、重慶和貴州諸省。

* **地理座標：** 約北緯28°37′，東經109°57′。

**2. 歷史與文化意義：**

* **歷史：** 古丈縣的茶葉種植可追溯至東漢時期（25–220年）：在古代藥物學著作《桐君錄》(桐君录, Tóngjūn Lù) 中，古丈地區已被標註為產茶區域。唐代（618–907年）當地茶葉被列入皇室貢品 (贡品, gòngpǐn)。

	國際認可始於20世紀：1929年古丈毛尖在巴黎國際博覽會 (法国国际博览会) 上獲得「國際名茶」獎項；1957年在萊比錫國際博覽會上獲得金獎。1982年入選「中國十大名茶」。

	經過數十年的衰落後，生產於1980年代得到復興。2007年獲得國家地理標誌保護，2011年獲得「中國馳名商標」地位。

* **名稱：**

	* 「古丈」(古丈) ——縣名，源於土家 (土家) 和苗 (苗) 族古代地名，這些民族是湘西山區的原住民。
	* 「毛」(毛) ——「茸毛、絨毛」：指芽頭上豐富的銀白色茸毛。
	* 「尖」(尖) ——「尖端」：描述芽葉纖細尖銳的形狀。

* **文化意義：** 古丈毛尖是湘西的驕傲，該地區被稱為「山水之鄉」(山水之乡)，是作家沈從文和畫家黃永玉的故鄉。茶葉與土家族和苗族文化密不可分——這些民族千年來居住在武陵山區。對於茶園佔農業用地相當比例的古丈縣而言，茶葉是經濟和文化認同的基礎。

**3. 植物學描述與原料：**

* **品種/栽培種：** 當地原生中葉種和小葉種灌木型 *Camellia sinensis* var. *sinensis* ——**古丈群體種 (古丈群体种)**，以槠葉種 (槠叶种, Zhūyè zhǒng) 亞型為主。特點是持嫩性強 (持嫩性强) 和茸毛豐富 (茸毛多)。正是這些適應武陵山區山地條件的當地栽培種群，確保了茶葉對多次沖泡的特有耐受性。

* **採摘：** 清明前早春採摘 (清明, ~4月5日)。特級品——純芽或一芽一葉初展 (一芽一叶初展)。芽葉須肥壯、多毫、大小均勻。

* **採摘標準：** 三個等級：
	* **特級 (特级)：** 純芽或一芽一葉。白毫豐富，滋味鮮爽。
	* **一級 (一级)：** 一芽一葉。條索勻整挺直，香氣持久。
	* **二級 (二级)：** 略帶嫩莖。滋味醇厚，耐泡性卓越。

* **原料要求：** 嫩度一致、完整無損的芽葉。當日採摘當日加工。

**4. 風土條件與栽培特色：**

* **氣候：** 亞熱帶山地季風氣候。年平均氣溫16°C。雲霧繚繞，直射陽光少，以漫射光 (漫射光) 為主。這些條件減緩了多酚類物質的合成，促進氨基酸積累——結果：滋味甘醇，苦澀味不明顯。

* **海拔：** 主要茶園——海拔1000公尺以上。森林覆蓋率80%。山區溪流瀑布眾多，無工業污染。

* **土壤：** 獨特之處——發育於紫色板頁岩 (紫色板页岩, zǐsè bǎnyèyán) 的土壤，天然富含磷和**硒** (硒, xī) ——硒含量顯著高於中國大多數茶區。含氮量>1%的土壤佔83%——肥力的卓越指標。

* **生態環境：** 古丈縣位於北緯30度「黃金茶帶」——這條緯線連接著世界最優秀的產茶區（大吉嶺、宇治、黃山）。空氣、水質和土壤的純淨使這一風土成為中國生態環境最優良的地區之一。

**5. 製作工藝：**

古丈毛尖的工藝特點是多階段：八道工序，三次炒製，兩次揉捻。這比大多數綠茶複雜，解釋了其特有的耐泡性。

* **攤青 (摊青 — tān qīng)：** 短時間攤放，去除多餘水分。

* **第一次炒製/殺青 (杀青 — shāqīng)：** 180°C——停止酶氧化，固定鮮香。

* **第一次揉捻 (揉条 — róutiáo)：** 重力揉捻 (揉捻力度重) ——比大多數綠茶更強，確保細胞結構深度破壞，解釋了對多次沖泡的卓越耐受性。

* **第二次炒製 (炒二青 — chǎo èrqīng)：** 120°C——中間乾燥和進一步香氣發展。

* **復揉 (复揉 — fùróu)：** 進一步緊實結構。

* **第三次炒製 (炒三青 — chǎo sānqīng)：** 最終炒製以固定形狀。

* **做條 (做条 — zuòtiáo)：** 師傅手工塑形，結合「理」(理, lǐ)、「拉」(拉, lā) 和「搓」(搓, cuō) 手法，使茶葉呈直挺針狀。

* **提毫收鍋 (提毫收锅 — tí háo shōuguō)：** 最後階段——在精確控制的溫度下，師傅將銀白茸毛「提」到茶葉表面，不讓其脫落。這需要極高精確度——稍微過熱會破壞茸毛，溫度不足則茸毛「內陷」。

**6. 感官特徵：**

* **乾茶外形：** 條索緊細圓直 (条索紧细圆直)，鋒苗挺秀 (锋苗挺秀)。色澤翠潤 (翠润) 帶油光，顯毫豐富 (显毫)。

* **乾茶香氣：** 清香 (清香, qīngxiāng) 純正高揚。嫩香高悅 (嫩香高悦, nènxiāng gāoyuè) ——「令人愉悅的嫩香」。冷杯留香 (冷杯留香) 持久。

* **茶湯香氣：** 持久高揚，帶純淨花香調。在多次沖泡中逐漸展現。

* **滋味：** 鮮爽 (鲜爽, xiānshuǎng)，醇厚 (醇厚, chúnhòu)，回甘生津 (回甘生津, huígān shēngjīn) 明顯。主要特色——**卓越的耐泡性**：優質古丈毛尖可沖泡15次，保持香氣和滋味——這對普通綠茶幾乎不可能。品鑑公式：「前3泡——純淨鮮爽；第4泡起——深度和回甘」。

* **湯色：** 嫩綠清澈透亮，有活力光澤。

* **葉底（沖泡後茶葉）：** 嫩綠勻整，柔軟有彈性。完整無破損。

**7. 化學成分：**

富硒紫色板岩土壤、高山產地和多階段加工形成獨特成分：

* **多酚類（兒茶素）：** 含量豐富——提供強大抗氧化潛力和滋味結構深度。

* **氨基酸（包括L-茶氨酸）：** 含量較高——漫射光充足和富氮土壤（83%區域含氮量>1%）的結果。正是氨基酸確保了鮮爽甘醇。

* **硒 (硒, xī)：** 天然高含量——古丈毛尖屬於「富硒茶」(富硒茶, fùxī chá) 類別。硒是強效抗氧化劑和免疫調節劑。

* **生物鹼：** 咖啡因——含量適中。可可鹼、茶鹼。

* **維生素：** 維生素C、B群維生素、類胡蘿蔔素。

* **礦物質：** 鉀、鎂、磷、鋅、錳、硒。

**8. 保健功效：**

* **抗氧化作用：** 多酚類與硒的高含量結合，提供增強的氧化應激保護。
* **提神醒腦 (清头目)：** 咖啡因和L-茶氨酸提供溫和專注的清醒感。
* **助消化 (消食)：** 多酚類促進脂肪分解——據研究，此效果特別顯著。
* **清熱消暑 (清热消暑)：** 解渴，清除內熱。
* **增強免疫力：** 硒和多酚類支持免疫功能。
* **止渴生津 (止渴生津)：** 茶湯刺激唾液分泌。

* **重要提醒：** 上述功效基於公開資料，非醫療建議。

**9. 沖泡方法：**

* **水溫：** 90°C（沸水稍涼）。古丈毛尖是少數建議相對高溫沖泡的綠茶，因其葉質緊實。

* **茶量：** 3–4克配120–150毫升水（比例1:40）。

* **茶具：** 白瓷蓋碗 (白瓷盖碗) 或玻璃杯。

* **沖泡過程（上投法 / 上投法）：**

	1. 熱水溫杯，倒掉。
	2. 注入90°C水至杯子7/10滿。
	3. 投入茶葉。
	4. 第一泡——5秒。倒出。
	5. 後續各泡——每次增加5秒。
	6. 茶葉可沖泡15次——這是其主要特色。

* **注意：** 前3泡展現純淨鮮爽和香氣；第4泡起——深度、厚度和回甘。避免悶泡 (闷泡) ——會增加澀味。不宜空腹飲用；最佳時機——餐後30分鐘。

**10. 儲存：**

* 密封容器，陰涼乾燥避光處保存。
* 最佳溫度——0–5°C（冰箱），密封包裝。
* 保質期——適當條件下12–18個月。
* 開封後——1–2個月內飲用完畢。

**11. 價格與仿冒品：**

古丈毛尖是人氣上升、產量有限的高山茶。價格取決於等級、採摘時間和核心產區來源。

* **避免仿冒品的方法：**

	* **選擇可信賣家** 有古丈縣地理標誌認證。
	* **評估外形：** 纖細挺直的「針」形，茸毛豐富。鬆散不勻的茶葉——仿冒跡象。
	* **測試耐泡性：** 正宗古丈毛尖可沖泡10+次。2–3泡後「失味」——明確的仿冒標誌。
	* **評估香氣：** 純淨高揚，無刺激性。陳味或「草腥味」——值得懷疑。
	* **注意價格：** 可疑的低價——仿冒跡象。

**12. 趣味知識：**

* 15泡的耐泡性——綠茶中的卓越表現。大多數知名綠茶可沖泡3–5次；15次是通常與烏龍茶或普洱茶相關的水準。秘訣——強力雙重揉捻和風土賦予的豐富可溶物。

* 1929年古丈毛尖成為首批在亞洲以外獲得國際獎項的中國綠茶之一——巴黎國際博覽會。這遠早於大多數中國茶葉進入世界市場的時期。

* 古丈縣位於「黃金三十度」北緯——連接世界最佳產茶區的緯線：從印度大吉嶺到日本宇治和安徽黃山。

* 83%的縣域土壤含氮量>1%——這是中國茶區中天然肥力最高的指標之一，直接確保了茶葉中氨基酸的高含量。

* 古丈是土家族和苗族聚居地，茶葉在其待客禮儀和婚禮儀式中佔中心地位。

**13. 與其他「毛尖」類茶葉的比較：**

* **信陽毛尖 (信阳毛尖)：** 來自河南省。條索更緊實筆直，栗香明顯。信陽——更「北方」特色，澀味強勁；古丈——更甘醇，耐泡性顯著更佳。

* **都勻毛尖 (都匀毛尖)：** 來自貴州省。彎曲如「魚鉤」，「三綠透黃」。都勻——多酚類更豐富；古丈——外形更挺直，耐泡性更佳。

* **五峰毛尖 (五峰毛尖)：** 來自湖北省。武陵山脈的鄰居。更溫和細膩；古丈——更厚重，具明顯的「硒」特色。

**結語：**

古丈毛尖是耐力型茶葉：其主要優勢不在於第一口的驚豔，而在於從一泡到一泡的層次展現，從鮮爽到深厚，從純淨到回甘——如此直到第十五泡。武陵山高海拔、富硒紫色板岩土壤、雲霧涼爽和土家苗族千年傳統——這一切都凝聚在每根纖細的銀毫「針」中。如果您尋找的不是一杯茶，而是能陪伴整個夜晚的綠茶——古丈毛尖正是為此而生。
