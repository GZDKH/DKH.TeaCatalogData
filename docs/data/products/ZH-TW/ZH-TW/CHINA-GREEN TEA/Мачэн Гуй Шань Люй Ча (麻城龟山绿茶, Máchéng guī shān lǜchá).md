# 麻城龜山綠茶 (麻城龟山绿茶, Máchéng guī shān lǜchá)

**麻城龜山綠茶 (麻城龟山绿茶, Máchéng guī shān lǜchá)**

麻城龜山綠茶 (麻城龟山绿茶, Máchéng guī shān lǜchá) —「麻城市龜山綠茶」— 產自龜峰山 (龟峰山, Guīfēngshān,「龜峰」, 1320公尺) 的綠茶，該山位於大別山 (大别山) 山脈，隸屬湖北省麻城市 (麻城市)。此茶早在陸羽 (陆羽, Lù Yǔ) 的《茶經》(《茶经》, Chájīng) 中即有記載：「黄州山谷茶生麻城縣」—「黃州山谷茶產於麻城縣」。唐代太宗皇帝 (太宗, Tàizōng, 李世民, Lǐ Shìmín) 曾以詩讚美此茶：「龟涎煮龟茶，天下第一家」—「以龜山泉水沖泡龜茶，天下第一家」。西元1959年，基於歷史名茶「龜山雲霧」(龟山云雾,「龜山雲霧」) 創製了「龜山岩綠」(龟山岩绿,「龜山岩綠」)。西元1962年，董必武 (董必武, Dǒng Bìwǔ, 1886–1975) — 中國共產黨創始人之一及中華人民共和國副主席 — 為此茶賦詩讚美。西元1980年此茶被列入「全國五大名茶」(全国五大名茶)，西元2012年獲得中華人民共和國地理標誌產品地位。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。屬於條形綠茶 (条形绿茶, tiáoxíng lǜchá)，採用炒青工藝殺青。茶葉外形為緊細圓直，鋒毫顯露 (紧细圆直、锋毫显露)。獨特工藝為「小火長炒配合旺火提香」(小火长炒+旺火提香, xiǎohuǒ chángchǎo + wànghuǒ tíxiāng)。

* **等級：** 中華人民共和國地理標誌產品 (国家地理标志产品, 2012年)。全國五大名茶之一 (全国五大名茶, 1980年)。農牧漁業部優質獎得主 (农牧渔业部优质奖, 1988年)。收錄於《中國名茶研究選集》(《中国名茶研究选集》)。唐代貢茶 (唐代贡茶)。宋代御品 (宋代御品)。陸羽《茶經》記載。截至2024年，茶園面積達35,000畝（約2,333公頃），年產量150噸，總產值6,000萬人民幣。

* **產地：** 中國湖北省 (湖北省, Húběi Shěng) 黃岡市 (黄冈市, Huánggāng Shì) 麻城市 (麻城市, Máchéng Shì)。大別山山脈 (大别山)，龜峰山 (龟峰山, 1320公尺)。產區包括五個茶區：龜山茶場 (龟山茶场,「龜山茶場」)、龜山鎮 (龟山镇)、三河口鎮 (三河口镇)、木子店鎮 (木子店镇)、張家畈鎮 (张家畈镇)。

* **地理座標：** 約北緯31°10′，東經115°10′。

**2. 歷史與文化意義：**

* **歷史：**

麻城的茶葉歷史是湖北省有文獻記載最古老的歷史之一。最早的文字記載來自陸羽 (陆羽, 733–804年)，他在《茶經》(《茶经》) 中記載：「黄州山谷茶生麻城縣」—「黃州山谷茶產於麻城縣」。這一直接記載使麻城成為「茶聖」在其經典著作中個別提及的少數地區之一。同樣在唐代 (618–907年)，唐太宗 (太宗, 李世民) 曾遊覽龜峰山並留下詩句讚美：「龟涎煮龟茶，天下第一家」—「以龜山泉水沖泡龜茶，天下第一家」。「龟涎」(龜涎) 的意象是對從龜峰山岩石中湧出的最純淨泉水的比喻。

宋代 (960–1279年)，麻城茶葉由專門的國家機構榷茶使司 (榷茶使司, Quèchá Shǐsī,「榷茶使司」) 管理，負責向朝廷供應。宋史記載《宋史·食貨志》(《宋史·食货志》) 證實「海州榷茶之所…茶善而易售」— 海州茶葉 (麻城作為更廣泛貿易網絡的一部分) 備受推崇且「易於銷售」。茶葉以「御品」(御品,「御品」) 地位供應朝廷。

西元1958年，在龜峰山坡建立了國營龜山茶場 (国营龟山茶场)。其技術人員基於「龜山雲霧」(龟山云雾,「龜山雲霧」) 的傳統製作工藝，於1959年開發了改良茶品「龜山岩綠」(龟山岩绿,「龜山岩綠」)，該茶在1960年代初即躋身湖北省四大名茶。

西元1962年，中華人民共和國副主席董必武 (董必武, 1886–1975年)，中國共產黨創始人之一，遊覽龜峰山並以詩歌讚美此茶：「昔日游击地，今為產茶區。龟峰名久著，牯嶺德不孤…」—「昔日游擊地，今為產茶區。龜峰名久著，牯嶺德不孤…」。董必武稱龜峰山為「第二廬山」(第二庐山)，為茶葉賦予了「紅色」光環，將革命歷史與茶葉歷史交織在一起。

西元1980年，在全國性評比中，此茶被列入「全國五大名茶」(全国五大名茶) — 當時的最高國家榮譽。西元1988年獲得「農牧漁業部優質獎」(农牧渔业部优质奖)。西元2005年在第六屆「中茶杯」全國名優茶評比中榮獲一等獎 (第六届"中茶杯"全国名优茶评比一等奖)。西元2012年，國家質量監督檢驗檢疫總局批准「龜山岩綠」(龟山岩绿) 地理標誌。截至2024年，「龜山岩綠」系列產品共獲得19項國家級和國際級獎項。

* **名稱：**

「麻城」(麻城) — 城市名稱，可追溯至南北朝時期 (420–589年)。「龜山」(龟山) —「龜山」，龜峰山 (龟峰山) 的簡稱；該山因其形狀酷似一隻長達16公裡的巨龜而得名 — 從「頭部」到「尾部」。「綠茶」(绿茶) —「綠茶」。

* **文化意義：**

龜峰山是麻城的名片之一，因其形狀被譽為「天下第一龜」(天下第一龟)。該山是大別山世界地質公園 (大别山世界地质公园) 的一部分，並於2024年獲得國家5A級旅遊景區地位。龜 (龟, guī) 在中國文化中是四靈 (四灵, sìlíng) 之一，象徵長壽、智慧和穩定。生長在「龜背」上的茶葉，每片葉子都承載著這種象徵意義。太宗的詩意表達 —「龜涎 + 龜茶 = 天下第一」— 是古代「茶葉命名」的完美範例，將地名、神話和感官體驗融為一體。此外，龜峰山是重要的「紅色旅遊」景點：在內戰期間，該地區是游擊隊基地，董必武的詩歌將茶葉轉化為象徵 — 從革命過去到和平茶業的傳承。

**3. 植物學描述與原料：**

* **品種/栽培種：** 原料基礎主要由當地群體栽培種**龜茶一號** (龟茶一号, Guīchá Yīhào) 和**龜茶二號** (龟茶二号, Guīchá Èrhào) 組成 — 中小葉種 (*Camellia sinensis* var. *sinensis*)，具有高抗寒性和肥厚芽頭，具有顯著的「持嫩性強」(持嫩性强)。龜峰山坡上還保存著樹齡超過100年的野生古茶樹。鮮葉生化成分：多酚類 — 17.69%，氨基酸 — 4.36%（比湖北平原綠茶高約20%）。

* **採摘：** 主要季節為春季。特級在清明 (清明, Qīngmíng, 約4月5日) 前採摘；一級在穀雨 (谷雨, Gǔyǔ, 約4月20日) 前採摘。特級採摘標準為單芽或一芽一葉初展。

* **等級：**
	* **特級 (特级, tèjí)：** 單芽或一芽一葉，清明前採摘。茸毛豐富。栗香帶嫩香 (嫩香)。價格 — 每500克450元人民幣起。
	* **一級 (一级, yījí)：** 一芽一葉，穀雨前採摘。價格 — 每500克200–300元人民幣。
	* **二級 (二级, èrjí)：** 一芽二葉。大眾等級，也用於包裝產品。

* **核心產區：** 海拔800公尺以上的羅百川村 (罗百川村)、龜尾村 (龟尾村,「龜尾村」) 和柿餅山村 (柿饼山村,「柿餅山村」)。這些村莊產出高達70%的特級茶。

**4. 風土條件與栽培特點：**

* **氣候：** 該地區位於湖北、河南、安徽三省交界處，大別山脈中部。氣候為北亞熱帶與溫帶過渡性氣候。年平均氣溫16°C。年降水量1200–1300毫米。雲霧天數超過180天。散射光比例超過70%。日夜溫差超過8°C，有利於氨基酸積累並減緩L-茶氨酸分解，確保茶葉高度「鮮爽」(鲜爽, xiānshuǎng)。

* **海拔：** 茶園位於海拔600–1000公尺。核心產區海拔800公尺以上。山脈最高點為1320公尺（險刀峰, 險刀峰）。

* **土壤：** 黃棕壤 (黄棕壤, huángzōngrǎng)，pH值4.0–6.5。有機質含量≥1%。土壤富含鐵 (Fe) 和鋅 (Zn)，有利於多酚類生物合成。土層厚度不少於60公分。

* **生態環境：** 龜峰山森林覆蓋率95%（中國茶區中最高之一）。負離子濃度高達每立方公分200,000個。水資源為一級清潔；舉水 (举水) 和巴水 (巴水) 河流源頭直接位於茶園區域。龜峰山擁有國家生態旅遊示範區 (国家生态旅游示范区) 地位。

**5. 製作工藝：**

獨特工藝「小火長炒 + 旺火提香」(xiǎohuǒ chángchǎo + wànghuǒ tíxiāng) —「小火長炒 + 旺火提香」— 雙階段乾燥，是製作的特色：

* **攤放 (摊放, tānfàng)：** 室溫20–25°C下攤放3–4小時。攤放厚度不超過5公分。葉片失去多餘水分（含水量降至60–65%）並開始散發「草青」香氣。

* **殺青 —「殺青」(杀青, shāqīng)：** 鍋溫120–150°C。溫和炒製，鈍化氧化酶但保持葉片嫩度。通過顏色變化和觸感彈性手工控制。

* **揉捻 (揉捻, róuniǎn)：** 按「輕→重→輕」(轻→重→轻) 原則。目的是破壞細胞結構以便後續浸出，但不損害茶葉完整性。

* **初乾 (初干, chūgān)：** 80–120°C。將濕度降至中間水平。

* **手工整形 (手工整形, shǒugōng zhěngxíng)：** 「掌心搓揉」(掌心搓揉, zhǎngxīn cuōróu) 方法 — 師傅純用手掌形成緊細圓直的條索。標準嚴格禁止機械壓制 — 只有手工操作才能確保特有的「紧细圆直」(緊細圓直) 結構。

* **「小火長炒」(小火长炒, xiǎohuǒ chángchǎo)：** 溫度60°C。緩慢長時間炒製直至水分降至約20%。此階段形成栗香的「主體」。

* **「旺火提香」(旺火提香, wànghuǒ tíxiāng)：** 溫度120°C。快速最終加熱，將栗香「鎖定」在茶葉內部。緩慢積累香氣化合物與高溫瞬間固定之間的對比 — 是區別龜山岩綠與其他湖北綠茶的關鍵技術特色。

**6. 感官特徵：**

* **乾茶外觀：** 緊細圓直的條索 (紧细圆直)，銀白茸毫顯露 (锋毫显露)。色澤翠綠油潤 (翠绿油润)。茶葉大小均勻，無碎片和「茶末」。

* **乾茶香氣：** 清純高雅，具有明顯的栗香 (栗香, lìxiāng)。特級茶另有「嫩香」(嫩香, nènxiāng) — 鮮嫩綠葉和嫩玉米的精緻香調。

* **茶湯香氣：** 栗香持久 (栗香持久)，為主要特徵。春茶具有「清香」(清香, qīngxiāng)，帶淡雅花香。茶杯冷卻後香氣保持10分鐘以上 — 顯示揮發性香氣化合物含量高。

* **滋味：** 醇厚 (醇厚, chúnhòu) 且鮮爽 (鲜爽, xiānshuǎng)。回甘 (回甘, huígān) 明顯持久。特有的滋味公式為「澀中泛甜」(涩中泛甜, sè zhōng fàn tián)：初始輕微澀味轉化為純淨礦物甜味。餘韻悠長，帶有「栗香回響」。

* **湯色：** 碧綠清亮 (碧绿清亮, bìlǜ qīngliàng)。透明度高，無渾濁和沉澱。

* **葉底（茶渣）：** 黃綠嫩勻 (黄绿嫩匀)。葉片在3–4次沖泡後仍保持彈性和韌性，證明耐泡性強。

**7. 化學成分：**

* **多酚類 (茶多酚)：** 17.69% — 綠茶的中等水平，低於平均水平（20–30%），這是由於高山產地和散射光比例高所致。多酚類含量低意味著苦澀味較少。

* **氨基酸 (氨基酸)：** 4.36% — 比湖北平原綠茶高約20%。氨基酸含量高，主要是L-茶氨酸，確保明顯的「鮮爽」(鲜爽) 和茶湯甜味。多酚類/氨基酸比例約為4:1 — 「醇厚鮮爽」(醇厚鲜爽) 滋味特徵的最佳比例。

* **咖啡鹼 (咖啡碱)：** 約4.5% — 高於綠茶平均水平（典型水平2–4%）。提供增強的提神效果。

* **水浸出物 (水浸出物)：** 39.93% — 高水平（國標≥39%），證明內含物質豐富。

* **可溶性糖 (可溶性总糖)：** 2.65% — 有助於「回甘」和滋味圓潤。

* **維生素：** 維生素C、B族維生素（B₁、B₂）、維生素E — 優質綠茶的典型組合。

* **礦物質：** Fe、Zn、K、Mg、Mn。鋅 (Zn) 含量較高 — 大別山山地土壤的特徵。

**8. 保健功效：**

* **抗氧化作用：** 兒茶素（EGCG、ECG）和維生素C共同中和自由基，延緩細胞老化。

* **增強提神效果：** 咖啡鹼（約4.5%）與L-茶氨酸結合產生「溫和提神」— 提高專注力而無劇烈神經興奮。

* **支持肌肉恢復：** 咖啡鹼加速運動後肌肉中乳酸的排出，促進更快恢復。

* **支持心血管系統：** 多酚類有助於降低「壞」膽固醇（LDL）水平並維持血管彈性。

* **助消化：** 單寧和可溶性糖刺激消化酶分泌並調節腸道蠕動。

* **抗菌作用：** 兒茶素對廣譜病原菌具有抑菌活性。

* **認知支持：** L-茶氨酸促進大腦α波產生，改善注意力集中和工作記憶。

* **增強免疫力：** 維生素C、鋅和多酚類複合物支持機體免疫反應。

**9. 沖泡方法：**

* **水溫：** 一級和二級茶85–90°C；特級茶80°C（更精細的原料需要較低溫度以防止苦澀）。

* **茶量：** 3克茶配150毫升水（比例1:50）。

* **茶具：** 玻璃杯 (玻璃杯) — 觀察茶葉「舞蹈」；蓋碗 (盖碗) 容量120–150毫升；瓷壺 — 日常沖泡。

* **過程：**
	1. 用熱水溫潤茶具，倒掉。
	2. 投入3克茶葉。
	3. 採用「中投法」(中投法, zhōngtóufǎ)：注入⅓容量的水，讓茶葉「甦醒」1分鐘。
	4. 加水至容器7/10容量。
	5. 第一泡 — 浸泡1–2分鐘。
	6. 重複沖泡 — 3–4次。每次增加15–20秒浸泡時間。建議每次加水時保留¼茶湯在容器中。

**10. 儲存：**

* **包裝：** 鋁箔真空密封包裝 — 最佳選擇。可用密封蓋鐵罐儲存。
* **溫度：** 冰箱，0–5°C。長期儲存 — 冷凍庫（−18°C）不透明密封包裝。
* **保質期：** 適當條件下12個月。開封後建議1個月內飲用。
* **茶葉天敵：** 濕氣、光線、異味、高溫。不要與強香味產品（香料、咖啡）一起儲存。

**11. 價格與仿冒品：**

* **價格範圍：** 特級（特级，明前茶）— 每500克450元人民幣起。一級 — 每500克200–300元人民幣。二級 — 每500克80–150元人民幣。

* **價格因素：** 採摘季節（明前茶最貴）、原料等級、生長海拔（800公尺以上村莊的茶最珍貴）、手工vs機械加工。

* **避免仿冒品：**
	* 購買帶有地理標誌標識「麻城龜山綠茶」或「龜山岩綠」的茶葉 — 受法律保護的產品。
	* 評估外觀：正宗龜山岩綠為細直均勻的條索帶茸毫，無碎片。仿冒品通常看起來鬆散不均勻。
	* 檢查香氣：真正的栗香持久純正，無「焦糊」或「草青」味。香氣微弱或異味 — 仿冒品。
	* 評估茶湯：正宗茶湯呈碧綠清亮。渾濁或黃褐色茶湯 — 劣質原料的標誌。
	* 對可疑低價保持警惕：特級茶低於每500克300元人民幣 — 值得懷疑。

**12. 有趣事實：**

* **陸羽點名麻城。**「黄州山谷茶生麻城縣」— 《茶經》中對麻城的直接記載。這是少數獲得「茶聖」個別提及的城市之一，證明早在8世紀茶葉品質就已卓越。

* **皇帝與「龜涎」。** 太宗的公式「龟涎煮龟茶，天下第一家」— 古代中國「茶葉行銷」的範例，將地名（龜山）、動物學比喻（泉水如「龜涎」）和評價判斷（「天下第一」）融合在一個對句中。

* **革命家的詩歌。** 董必武不僅是政治家，還是中共一大13名代表之一（1921年）。他1962年在龜峰山的詩意造訪將茶葉與大別山的「紅色」歷史聯繫起來 — 該地區培養了36位將軍，包括徐向前元帥和麻城籍王樹聲將軍。

* **1980年「五大名茶」。** 入選「全国五大名茶」— 在中國數千種綠茶中只有少數獲得的最高國家認可。

* **龜山與千萬杜鵑。** 龜峰山不僅是茶山，也是世界最大古杜鵑花群落 (杜鹃花) 的生長地：1000萬株，樹齡估計數十萬年。每年春天山坡被火紅花毯覆蓋 — 這一景觀為麻城帶來第二個品牌：「人間四月天，麻城看杜鵑」(人间四月天，麻城看杜鹃)。

* **雙階段火候。** 工藝「60°C（緩慢，至20%水分）→ 120°C（快速，鎖定栗香）」— 龜山茶場師傅的獨創技術，在其他湖北綠茶中沒有直接類似工藝。

* **宋代國家專賣。** 麻城茶是少數歷史包含「榷茶使司」(榷茶使司) — 國家茶葉貿易專賣管理機構的茶葉之一，證明該產品在帝制時代的卓越價值。

**13. 與其他綠茶比較：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 同樣產於湖北，但採用日式蒸青 (蒸青) 而非炒青。滋味更「綠」，帶鮮味；香氣海洋性，「海藻味」。龜山岩綠為炒青茶，栗香明顯，茶體更厚重。

* **信陽毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 產於鄰省河南，同樣在大別山脈。相似風土，但揉捻和乾燥工藝不同。毛尖更「尖銳」，茸毫明顯，香氣帶鮮綠調。龜山更「圓潤」且栗香突出。

* **廬山雲霧 (庐山云雾, Lúshān Yúnwù)：** 「廬山雲霧茶」— 鄰省江西名茶。董必武曾將龜峰山比作廬山。兩者都是高山「雲霧」茶，但廬山雲霧具有更明顯的「豆香」(豆香)，而龜山為栗香。

* **黃山毛峰 (黄山毛峰, Huángshān Máofēng)：** 安徽茶，同樣產於大別山系統（東延部分）。相當的海拔和雲霧。毛峰更清淡，花草香氣帶「蘭花」調。龜山更厚重，栗香基調更明顯。

* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 安徽大葉茶。形狀根本不同（大型扁平葉片），但相似的風土原理 — 高山雲霧，酸性土壤。猴魁「蘭花香」，優雅；龜山更「厚實」，結構明顯。

**結論：**

麻城龜山綠茶 — 陸羽提及其名，太宗皇帝讚美其味，革命家董必武確立其聲譽，「1980年五大名茶」確認其地位的茶葉。細直條索帶栗香，由雙階段火候「鎖定」— 從緩慢的60°C到迅速的120°C — 生長在「龜背」上，龜峰山，其形狀象徵長壽，其山坡每年春天被千萬杜鵑點燃，其泉水按太宗之言使茶成為「天下第一」。以4.36%氨基酸和17.69%多酚類，龜山岩綠達到「醇厚鮮爽」滋味的理想比例 — 適合那些珍視歷史深度和山間雲霧純淨的每一杯茶的人們。
