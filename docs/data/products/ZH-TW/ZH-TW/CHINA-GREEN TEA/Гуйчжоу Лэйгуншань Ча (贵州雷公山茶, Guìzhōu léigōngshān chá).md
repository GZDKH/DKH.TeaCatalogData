# 貴州雷公山茶 (贵州雷公山茶, Guìzhōu léigōngshān chá)

**貴州雷公山茶 (贵州雷公山茶, Guìzhōu léigōngshān chá)**

![Гуйчжоу Лэйгуншань Ча (贵州雷公山茶, Guìzhōu léigōngshān chá)](images/Гуйчжоу%20Лэйгуншань%20Ча%20(贵州雷公山茶,%20Guìzhōu%20Léigōngshān%20Chá).png)

![Гуйчжоу Лэйгуншань Ча (贵州雷公山茶, Guìzhōu léigōngshān chá)-1](images/Гуйчжоу%20Лэйгуншань%20Ча%20(贵州雷公山茶,%20Guìzhōu%20Léigōngshān%20Chá)-1.png)

雷公山茶是高山綠茶的集合名稱，產於雷公山山坡及貴州省雷山縣鄰近地區。這些茶葉共同擁有霧氣瀰漫的山地風土條件，來自國家級自然保護區，具有溫和的胺基酸特徵和純淨、「透明」的茶湯品質，反映了中國工業化程度最低的茶葉產區之一的生態聲譽。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá) — 未發酵茶，氧化程度低於5%。以加熱方式固定酶活性 (杀青, shāqīng)。
* **類別：** 貴州高山生態綠茶；公共品牌「雷公山茶」(雷公山茶) 包含數個產品線 — 雷公山清明茶 (雷公山清明茶, Léigōngshān Qīngmíng Chá)、雷山銀球茶 (雷山银球茶, Léishān Yínqiú Chá — 具有地理標示保護的球形茶) 及其他產品。在貴州省，雷公山茶與都勻毛尖 (都匀毛尖) 和湄潭翠芽 (湄潭翠芽) 並列為十大名綠茶。
* **產地：** 中國，貴州省 (贵州, Guìzhōu)，黔東南苗族侗族自治州 (黔东南苗族侗族自治州, Qiándōngnán Miáozú Dòngzú Zìzhìzhōu)，雷山縣 (雷山县, Léishān Xiàn)。茶園位於雷公山 (雷公山, Léigōngshān) 山地的山坡和山谷中，雷公山是苗嶺 (苗岭, Miáolǐng) 山脈的主峰，苗嶺是貴州東南部最大的山脈。
* **地理座標：** ≈ 26.4° 北緯，108.2° 東經（雷山縣；茶園分佈在全縣範圍內海拔900–1,400公尺的高度帶）。

**2. 歷史與文化意義：**

* **歷史：** 雷山地區擁有悠久但文獻記載較少的茶葉傳統。根據《雷山縣志》記載，早在清朝時期這些地方就建立了茶園，但由於山區交通不便，許多茶園隨時間推移而荒廢。有組織的茶葉發展始於1970年代，當時縣政府開始大規模恢復和擴展茶園。

  轉捩點是1980年：縣科技委員會工作人員毛克翕 (毛克翕, Máo Kèxǔ) 自願前往丹江鎮 (丹江镇, Dānjiāng Zhèn) 覺散村 (觉散, Juésàn) 的山村，在兩年內恢復了500畝（≈ 33公頃）荒廢的茶園並新建了200畝。通過多年實驗，毛克翕發現海拔1,200–1,400公尺的春季嫩芽含有極高的天然果膠，可以在不使用膠水和添加劑的情況下將茶葉形成緊密的球狀。1988年他推出了「銀球茶」(银球茶, Yínqiú Chá —「銀球」)，獲得第一屆全國食品博覽會金獎，1991年獲得獨特成型技術的國家專利。

  21世紀，雷山的茶葉經濟成為該縣的關鍵產業：到2024年，茶園面積超過16萬畝（≈ 10,870公頃），年產量達到約6,580噸，產品總價值超過11.6億元人民幣。銀球茶和清明茶多年來作為工作用茶供應給中國中央政府機構；銀球茶也是中國外交部的官方禮品茶。2015年，特級銀球茶在貴州首屆春茶大賽中獲得「茶王」(茶王, chá wáng) 稱號。2023年，雷山銀球茶入選全國「土特產」(土特产, tǔtèchǎn) 名錄。

* **名稱：**
  * **贵州 (Guìzhōu)** — 貴州省。
  * **雷公山 (Léigōngshān)** — 字面意思「雷神之山」：雷公 (Léigōng) — 雷公，中國神話中的雷神；山 (shān) — 山。具有神話色彩的地名，營造出雄偉、雲霧繚繞的山峰形象。
  * **茶 (chá)** — 茶。

  與銀球茶並行發展的還有雷公山清明茶產品線 — 採用傳統技術製作的早春原料葉形綠茶。兩種產品都成為該縣的公共品牌，2010年代制定了一系列地方標準（《雷公山銀球茶、清明茶綜合標準體系》）。縣政府每年投資約300萬元人民幣補貼鮮葉收購和生產設施現代化。貴州整體將雷公山綠茶視為「黔東優質出口綠茶產業帶」的中心，目標是國際市場。

* **文化意義：** 雷山縣是苗族 (苗族, Miáozú) 的主要文化區域之一；這裡有世界最大的苗族聚落 — 西江千戶苗寨 (西江千户苗寨, Xījiāng Qiānhù Miáozhài) —「西江千戶苗寨」，AAAA級旅遊遺產景點。茶在這些地方不僅是農業作物，更是日常生活、節慶待客和苗族集市貿易的一部分。銀球茶的銀色球狀，讓人聯想到苗族傳統飾品上的微型銀鈴，成為該地區的美食和文化象徵。山茶是雷山民族文化和自然景點遊客的熱門紀念品。

**3. 植物學描述與原料：**

* **品種/栽培種：** 該地區呈現種植材料的多樣性：當地的中華茶樹 *Camellia sinensis* var. *sinensis* 群體（適應高山環境的灌木和半木本形態）以及引進的品種，專注於早期和嫩春採摘。對於具體批次，建議向生產者確認栽培種。雷公山高山群體的特色是嫩芽中天然果膠含量較高，使得成型為球狀（用於銀球茶）成為可能。
* **採摘：** 主要價值在於早春採摘（清明節前後，清明, Qīngmíng，4月初）。高山環境延遲了植物生長期的開始：嫩芽出現比低地晚，但緩慢的生長確保了胺基酸和芳香物質的更高濃度。採摘主要以手工進行；據統計，超過90%的採茶工是苗族婦女。
* **採摘標準：** 對於銀球茶和清明茶的最高等級 —「一芽一葉」(一芽一叶, yī yá yī yè) 或「一芽二葉初展」(一芽二叶初展, yī yá èr yè chūzhǎn)。對於大宗產品 — 較成熟的葉片。
* **原料要求：** 完整、有彈性的葉片，無黃化和老化跡象；均勻的規格；無異味（煙味、燃料味、家用香料味）。

**4. 風土條件與栽培特色：**

* **地形與氣候：** 雷公山是苗嶺山脈的主峰，最高海拔2,178.8公尺。氣候屬中亞熱帶季風氣候，溫和濕潤：年平均氣溫14–15.6°C，無霜期超過240–282天；年平均降雨量1,250–1,375毫米；相對濕度約80%；年平均日照約1,136–1,225小時（寡日照，「少陽光」— 山雲自然遮蔽）。低日照和充足的散射光是確保葉片胺基酸積累和降低粗澀味的關鍵因素。
* **生長海拔：** 最佳茶葉帶 — 海拔1,200–1,400公尺。正是在這些海拔高度集中了最好的茶園，包括腳堯村 (脚尧村, Jiǎoyáo Cūn) 周圍 — 銀球茶生產的核心區。
* **土壤：** 微酸性（pH 4–6）砂質黏土黃壤 (黄壤, huángrǎng)，具有顆粒結構；深厚、疏鬆、肥沃的土層。有機物含量3.47–7.22%，氮含量2.02–3.42克/公斤。重金屬未超標；未發現有毒元素。表層礦物資源微量，最大程度降低了人為污染風險。
* **生態環境：** 雷山縣近三分之一的面積屬於雷公山國家級自然保護區 (雷公山国家级自然保护区)。植被覆蓋率94%，森林覆蓋率約72.8%。無工業企業，鄰近保護區和高海拔位置創造了「生態茶」的聲譽。然而，在百科全書的語境下，應區分具體生產者的實際農業技術（有無有機認證、農藥/肥料使用情況）與行銷形象。

**5. 製作工藝：**

標準雷公山茶（清明茶及類似產品）的特點是「純淨」的綠茶風格：溫和的殺青、細緻的成型、保持清新風味。

* **採摘 (采摘 — cǎizhāi)：** 清晨，手工採摘；快速冷卻並運送原料到生產地。
* **攤放 (摊放 — tānfàng / 摊晾 — tānliàng)：** 在通風室內的竹盤上均勻攤放；根據天氣和葉片濕度30–90分鐘。目的 — 平衡水分，初步降低「草青味」並為殺青做準備。
* **殺青 (杀青 — shāqīng)：** 停止酶促氧化。在貴州既有鍋炒 (锅炒, guōchǎo) 也有滾筒殺青。關鍵是不要過熱嫩嫩的高山葉片：過熱產生苦味和「炒焦味」，不足則有「生草味」。
* **揉捻 (揉捻 — róuniǎn)：** 適度，為茶湯增加「體感」而不將葉片破碎成粉末。
* **做形 (做形 — zuòxíng)：** 必要時 — 賦予商品形狀：直條、輕微彎曲的捲曲。對於銀球茶 — 獨特操作：利用天然果膠手工搓成球狀 (搓球, cuōqiú)，直徑18–20毫米；每個球重約2.5克。
* **烘乾 (烘干 — hōnggān)：** 分階段：初步固定水分和最終乾燥至穩定的5–6%。對於銀球茶需要特別小心：球狀必須均勻乾燥，無「外乾內濕」現象，否則可能發霉。
* **揀剔/包裝 (拣剔 / 包装 — jiǎntī / bāozhuāng)：** 去除粗糙碎片，控制批次均勻性。

**6. 感官特徵：**

* **乾茶外觀：** 緊結的條索 (条索紧结, tiáosuǒ jǐnjié)，從亮綠色到墨綠色，帶有油潤光澤 (墨绿油润, mòlǜ yóurùn)。對於銀球茶 — 均勻的銀綠色球狀，帶有輕微茸毛。
* **乾茶香氣：** 清新、草花香 (清香, qīngxiāng)；較溫和的殺青 — 溫和的堅果味或栗香 (栗香, lìxiāng)。
* **茶湯香氣：** 純淨，帶有栗香背景和花香層次；銀球茶 — 更濃郁，帶有「嫩乳香」和「果膠」風味。
* **滋味：** 溫和、鮮爽 (鲜爽, xiānshuǎng)，具有適度的醇厚感 (醇厚, chúnhòu)。苦味通常短暫，快速轉為持久的回甘 (回甘, huígān)，帶有礦物清涼感。由於葉片厚實和高浸出物含量 — 耐沖泡 (耐冲泡, nài chōngpào)。
* **湯色：** 從淡綠色和稻草金黃色到黃綠明亮 (黄绿明亮, huánglǜ míngliàng)；透明清澈。
* **葉底（沖泡後的葉片）：** 嫩綠、完整的葉片 — 亮綠色、新鮮、有彈性；規格均勻。銀球茶 — 球狀在杯中緩慢而優雅地「綻放」，「宛若荷苞初綻」(宛若荷苞初绽)。

**7. 化學成分：**

典型的高山綠茶特徵，強調胺基酸的溫和性和微量元素的豐富性。

* **茶多酚 (茶多酚, chá duōfēn)：** 主要是兒茶素，包括EGCG；含量因規格和季節而異。
* **胺基酸 (氨基酸, ānjīsuān)：** L-茶胺酸含量較高，這是由於在「少陽光」和高山涼爽條件下緩慢生長所致。正是茶胺酸負責特有的感官溫和性和茶湯的「鮮甜」味。
* **咖啡鹼 (咖啡碱, kāfēi jiǎn)：** 適度含量；與茶胺酸協同作用提供平穩的刺激。
* **果膠質 (果胶质, guǒjiāo zhì)：** 海拔1,200–1,400公尺春季嫩芽中含量異常高 — 這是能夠將銀球茶成型為緊密球狀而無需人工黏合劑的關鍵特性。
* **微量元素：** 特色 — 硒含量較高：根據分析數據，銀球茶中含量為2.00–2.02微克/克，約為綠茶平均水平的15倍。還含有鋅和其他礦物元素。
* **維生素：** C、B₁、B₂、E — 綠茶的典型組合；高山產地有助於維生素C的保存。

**8. 健康功效：**

* **抗氧化保護：** 多酚（兒茶素）中和自由基，支持細胞健康。
* **溫和刺激：** 咖啡鹼與L-茶胺酸結合 — 提神而不焦慮，改善專注力和注意力。
* **消化支持：** 傳統上飯後飲用綠茶以幫助消化。
* **心血管支持：** 多酚有助於維持血管彈性和正常膽固醇水平。
* **認知功能：** L-茶胺酸刺激大腦α波，促進平靜專注。
* **微量元素支持（硒）：** 高硒含量 — 抗氧化輔因子，對免疫系統和甲狀腺功能重要。

* **皮膚與抗衰老：** 兒茶素的抗氧化特性有助於保護皮膚免受光老化並支持其彈性。
* **新陳代謝調節：** 綠茶傳統上與對脂肪和碳水化合物代謝的溫和影響相關；臨床前研究數據表明兒茶素能夠刺激產熱。

*注意：* 這是食品，不是藥品。濃茶和大劑量可能刺激敏感人群的胃部。不建議空腹飲用。有睡眠問題者 — 限制下午飲用。

**9. 沖泡方法：**

* **水溫：** 75–80°C。更高溫度會增強苦味並「掩蓋」高山茶葉精緻的胺基酸甜味。
* **茶葉用量：** 3–4克配150毫升（杯子或蓋碗）。對於銀球茶 — 1個球（≈ 2.5克）配150–200毫升。
* **茶具：** 透明玻璃杯 (玻璃杯) — 用於視覺欣賞球狀「綻放」；瓷蓋碗 (盖碗) — 用於最大程度展現香氣。

* **工夫茶沖泡法（分次沖泡）：**
  1. 溫杯。
  2. 投茶：4–6克配100毫升。
  3. 洗茶：通常不需要；如果茶葉有粉塵 — 快速沖泡2秒。
  4. 第一泡：75–80°C水溫，10–20秒。
  5. 出湯。評估茶湯透明度和顏色。
  6. 後續沖泡：5–8泡，逐漸增加時間。

* **悶泡法（西式）：**
  * 2–3克配250毫升，75–80°C，2–3分鐘。

* **建議：** 如果茶湯偏苦 — 降低水溫和減少用量，而不要「責怪茶葉」。貴州高山綠茶通常在80°C以下的溫度表現最佳 — 這樣保持其「甜美透明」的特質。

**10. 儲存：**

* 密封、不透明包裝；乾燥、陰暗、無異味的地方。
* 最佳 — 冰箱保存（0–5°C）完全密封，特別是嫩春茶。
* 最佳新鮮期 — 生產日期起6–12個月（無真空或惰性包裝時）。
* 開封後 — 1–2個月內飲用完畢。
* 對於銀球茶，均勻乾燥很重要：密封破損時球狀可能吸濕並失去形狀。

**11. 價格與假冒：**

* **價格：** 由茶園海拔、採摘時間（清明前早春茶最貴）、芽葉比例和具體生產者聲譽決定。銀球茶 — 需要費工手工成型的高端產品，因此比普通葉形雷公山茶貴得多。大宗「清明茶」較為實惠。
* **典型假冒：** 原料替換：來自鄰近縣市甚至其他省份的廉價綠茶，以「雷公山」品牌銷售。對於銀球茶 — 嘗試用果膠不足的低品質原料模仿球形（沖泡時球狀散開）。
* **如何避免假冒：**
  * 尋找具體信息：包裝上的縣/村/合作社/企業名稱。
  * 評估葉片完整性和香氣純淨度：正宗雷公山茶無「香精」和「化學」味。
  * 對於銀球茶：球狀應緊實、均勻，輕壓不散；沖泡時 — 緩慢展開，而非瞬間散開。
  * 選擇標明季節和生產日期的批次。
  * 可疑的低價 — 可能假冒的信號。

**12. 有趣事實：**

* 銀球茶是世界上唯一使用純天然葉片果膠黏合的球形綠茶。該技術已獲專利（中國國家專利，1991年）。
* 「銀球」(银球，「銀球」) 名稱的選擇有兩個原因：球狀讓人聯想到苗族飾品上的銀鈴，而在茶葉創制年份（1988年）中國慶祝乒乓球奧運勝利 —「球」(球, qiú) 成為幸運象徵。
* 雷公山 —「雷神之山」— 不僅是茶葉地名：這是貴州最大的保護區，擁有原始森林和獨特動物群。創造茶葉「溫和性」的高山雲霧 — 正是使雷公山成為中國西南最多雨地區之一的現象。
* 銀球茶的硒含量（約2微克/克）是中國綠茶中最高的之一，這歸因於雷山山地土壤的特性。
* 雷山縣每年約有27萬人次參與茶葉採摘；超過90%的採茶工是苗族婦女。茶產業成為78,000名當地居民的主要收入來源，人均年收入增加約3,500元人民幣。

**13. 與貴州及中國西南其他綠茶的比較：**

* **都勻毛尖 (都匀毛尖, Dūyún Máojiān)：** 貴州最著名的綠茶，「中國十大名茶」之一。產於更南的都勻地區。更「經典」的特徵：明顯茸毛、尖銳清香、輕微澀味。雷公山茶通常更溫和、更「透明」，強調胺基酸甜味。
* **湄潭翠芽 (湄潭翠芽, Méitán Cuìyá)：** 來自貴州北部（遵義地區）的扁形綠茶。更「堅果味」和濃郁；工藝上更接近龍井。雷公山茶 — 外形更豐滿，香氣更「花草味」。
* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 來自鄰近湖北省的蒸青綠茶。相似的「溫和」方向，但完全不同的工藝特徵：蒸汽殺青產生明亮的「海洋」味和鮮味，而雷公山茶 — 炒青，具有花栗香特徵。
* **雲南高山綠茶 (滇绿, Diānlǜ)：** 雲南大葉種 *var. assamica* 高山綠茶通常更濃郁、更「草本」，澀味明顯。雷公山茶來自小葉種 *var. sinensis* — 更輕盈、更透明、口感更細膩。

**結論：**

貴州雷公山茶是進入中國西南高山綠茶世界的理想和友好入門。它不需要品茶者「培養」對澀味的接受度，也不會以複雜性壓倒人：它的優勢在於純淨、溫和的胺基酸甜味、透明的香氣和清涼山地新鮮感。嘗試用稍低的溫度沖泡 — 75°C或更低 — 茶葉會展現「雷神之山」的主要優點：輕盈、透明和持久的礦物甜回甘，如同苗嶺無盡綠色山坡上雲霧的回響。
