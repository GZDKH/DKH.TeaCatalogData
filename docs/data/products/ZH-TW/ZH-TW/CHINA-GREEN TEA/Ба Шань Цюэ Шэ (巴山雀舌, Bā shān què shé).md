# 巴山雀舌 (巴山雀舌, Bā shān què shé)

**巴山雀舌 (巴山雀舌, Bā shān què shé)**
![Ба Шань Цюэ Шэ (巴山雀舌, Bā shān què shé)](images/Ба%20Шань%20Цюэ%20Шэ%20%28巴山雀舌,%20Bā%20Shān%20Què%20Shé%29.png)

![Ба Шань Цюэ Шэ (巴山雀舌, Bā shān què shé)-1](images/Ба%20Шань%20Цюэ%20Шэ%20%28巴山雀舌,%20Bā%20Shān%20Què%20Shé%29-1.png)

巴山雀舌 (巴山雀舌, Bā shān què shé) — 來自四川東北部巴山 (大巴山) 深山的著名扁形綠茶，為「四川十大名茶」(四川十大名茶) 之一。此茶享有雙重美譽：「富硒茶王、雀舌珍品」(富硒茶王、雀舌珍品, fùxī chá wáng, quèshé zhēnpǐn)。扁平整齊的茶葉，確實如同山雀的小舌頭，生長在天然富硒的土壤上 — 萬源市屬於中國三大富硒帶之一。憑藉超過千年的茶葉種植歷史和80多項國內外獎項，巴山雀舌已確立為四川「硒茶帶」的旗艦產品。

**1. 分類與產地：**

* **類型：** 綠茶 (绿茶, lǜchá)，未發酵茶。屬於扁形綠茶 (扁形绿茶, biǎnxíng lǜchá) — 「雀舌」(雀舌, 「麻雀舌頭」) 子類型，以微型扁平葉形為特色。工藝上結合炒製與烘乾 (半烘半炒, bàn hōng bàn chǎo)。

* **等級：** 「四川十大名茶」(四川十大名茶) 之一。國家地理標誌保護產品 (国家地理标志保护产品)。擁有「中國馳名商標」(中国驰名商标, Zhōngguó Chímíng Shāngbiāo) 地位。1991年在杭州國際茶文化節上獲得「中國文化名茶」(中国文化名茶) 稱號 — 全國排名第五。通過歐盟SGS標準486項參數檢測。獲得80多項各級獎項。

* **產地：** 中國四川省 (四川省, Sìchuān Shěng) 萬源市 (万源市, Wànyuán Shì)，隸屬達州市 (达州市, Dázhōu Shì)。萬源位於四川、陝西省與重慶市交界處，大巴山 (大巴山, Dà Bā Shān) 山脈腹地。生產區域涵蓋13個鄉鎮。

* **地理座標：** 東經107°28′–108°31′，北緯30°39′–32°20′。

**2. 歷史與文化意義：**

* **歷史：** 萬源的茶葉傳統有著深厚的歷史根基。

	**古代時期。** 據史學家常璩 (常璩) 於四世紀編撰的巴國史書《華陽國志·巴志》(《华阳国志·巴志》, Huáyáng Guózhì — Bā Zhì) 記載，萬源地區在西周時期 (西元前11–8世紀) 已納入茶葉種植區域，茶葉是向統治者進獻的「貢品」(贡品) 之一。萬源位於茶馬古道 (茶马古道) 北方路線上，自宋代 (960–1279) 起在中國中部與西北地區茶葉貿易中扮演重要角色。

	**重要歷史文物。** 在萬源境內石窩鎮 (石窝镇) 保存著獨特的文物 — 石刻銘文《紫雲坪植茗靈園記》(《紫云坪植茗灵园记》)，可追溯至1109年 (北宋大觀三年)。記述了王敏 (王敏) 從建溪 (建溪，今福建南平地區) 移植茶苗並在自宅種植的事蹟。銘文中有詩句：「築成小圃凝蒙頂，分得靈根自建溪」(筑成小圃凝蒙顶，分得灵根自建溪)。據現有資料，這是中國最古老的記錄茶葉栽培的石刻銘文；其拓片 (拓片) 收藏於杭州中國茶葉博物館 (中国茶叶博物馆)。

	**現代時期。** 現今形式的「巴山雀舌」茶於1985年由萬源茶葉專家創制，開發出獨特的加工工藝。1987年國營草壩茶場 (国营草坝茶场, Guóyíng Cǎobà Cháchǎng) 開始工業化生產。1989年註冊商標。1991年在杭州國際茶文化節上榮獲「中國文化名茶」(中国文化名茶) 稱號 — 全國排名第五。著名軍事將領張愛萍 (张爱萍, Zhāng Àipíng) 親筆題寫「巴山雀舌」，此書法至今仍用作品牌標誌。2000年實現全機械化生產。2010年中華人民共和國農業部註冊地理標誌「萬源富硒茶」(万源富硒茶)。截至2024年，萬源茶園面積達24.7萬畝 (万亩, ~165,000公頃)，年產乾茶6,590噸，總產值約160億元人民幣。

* **名稱：**
	* 「巴山」(巴山) — 「巴[國]之山」。指古代巴國 (巴国, Bāguó)，西元前1千年存在於四川東部和重慶西部。大巴山 (大巴山, 「大巴山脈」) — 茶葉生長地 — 名稱中承載相同的歷史淵源。
	* 「雀舌」(雀舌) — 「麻雀舌頭」。形象描述茶葉形狀：扁平、微型、略尖，確實如同山鳥的小舌頭。在中國茶葉傳統中，「雀舌」(雀舌) 名稱代表最高品質的原料和加工；只有最嫩、最精選的芽頭才能如此命名。

* **文化意義：** 萬源擁有「中國富硒茶都」(中国富硒茶都) 和「中國名茶之鄉」(中国名茶之乡) 稱號。巴山雀舌不僅是萬源，也是整個達州地區的名片，以及「中國硒部茶園走廊」(中国硒部茶园走廊) 的象徵，該市致力於將其推廣為統一品牌。清代官員王夢庚 (王梦庚) 兩次擔任萬源縣令 (1808, 1826)，其詩中有著名詩句：「雀舌芒欺蒙頂撷，龍團餅壓臨邛研」(雀舌芒欺蒙顶撷，龙团饼压临邛研) — 證明萬源茶葉在兩百年前已與四川最偉大的茶葉並列。

**3. 植物學描述與原料：**

* **品種/栽培種：** 主要品種 — **四川中葉群體種** (四川中叶群体种, Sìchuān Zhōngyè Qúntǐzhǒng) — 種子繁殖的 *Camellia sinensis* var. *sinensis*。具有高抗寒性 (耐寒性强)，這對大巴山冬季氣溫降至–10°C的山區條件至關重要。葉片中等大小，茸毛豐富。30年以上的老茶樹集中在草壩鎮 (草坝镇)。「一芽一葉」標準的百芽重約45克。

* **採摘：** 春茶採摘為主。最珍貴的是「明前茶」(明前茶) — 清明前 (~4月5日) 採摘的茶：不少於90%的「一芽一剛展葉」標準芽頭。「雨前茶」(雨前茶) — 穀雨前 (~4月20日)：「一芽一開葉」芽頭，不少於80%。實行「五不採」(五不采, wǔ bù cǎi) 標準：雨天不採、露水葉不採、紫色葉不採、損傷葉不採、不符標準葉不採。

* **採摘標準：**
	* 特級 (特级)：單芽或芽帶剛展小葉。芽長不超過2公分。
	* 一級 (一级)：一芽一葉。長度不超過2.5公分。
	* 二級 (二级)：一芽二小葉。

* **原料要求：** 極高標準。芽頭大小一致，完整，無機械損傷。手工採摘。鮮葉立即送廠加工。

**4. 風土條件與栽培特色：**

* **氣候：** 萬源位於亞熱帶濕潤季風氣候區，具明顯垂直氣候帶 — 最低海拔 (335公尺) 與最高海拔 (2412公尺) 相差超過2000公尺。年平均氣溫14.7°C。無霜期237天。年降雨量約1170毫米。雲霧天數超過200天。散射光為主促進胺基酸積累：春茶胺基酸含量不少於3.0%。

* **生長海拔：** 海拔800–1200公尺 — 所謂「大巴山黃金產茶帶」(大巴山黄金产茶带)。

* **土壤：** 黃棕壤 (黄棕壤, huáng zōng rǎng) 和「爛石地」(烂石地, làn shí dì) — 正是陸羽 (陆羽) 在《茶經》(茶经) 中認為最適合茶葉的土壤類型。pH值4.5–6.0，有機質含量超過1.0%。關鍵獨特性 — **天然硒含量**：0.28–0.51微克/克 — 萬源屬於中國三大富硒帶 (全国三大富硒带) 之一。

* **生態環境：** 森林覆蓋率60.2%。萬源市位於渠江 (渠江) 上游，工業污染極少。該地區獲得「中國天然氧吧」(中国天然氧吧) 認證。茶園由山澗溪流和丹江 (丹江) 上游泉水灌溉。

* **核心產區 (核心产区)：**
	* **草壩鎮 (草坝镇, Cǎobà Zhèn)：** 歷史生產核心 — 最古老的國營茶廠 (1987年) 所在地。30年以上老樹。
	* **八台鄉 (八台乡, Bātái Xiāng)：** 海拔1000–1200公尺高山茶園，終年雲霧繚繞。
	* **石塘鎮 (石塘镇)：** 3000多畝茶園進入盛產期。

**5. 製作工藝：**

巴山雀舌採用獨創工藝，結合炒製與烘乾 (半烘半炒, bàn hōng bàn chǎo)。此方法既保持「雀舌」扁平形狀，又展現栗香。茶葉通過歐洲標準認證 (SGS，486項參數)。

* **採摘 (采摘 — cǎi zhāi)：** 按「五不採」(五不采) 標準手工採摘。芽葉當日送廠。

* **攤放 (摊放 — tān fàng)：** 採摘芽葉薄攤4–6小時輕度萎凋。此期間水分降低，葉片變柔軟，開始形成香氣。

* **殺青 (杀青 — shāqīng)：** 在滾筒中約180°C處理。快速鈍化酶活性，固定綠色和栗香調性。

* **理條壓扁 (理条压扁 — lǐtiáo yā biǎn)：** 決定特色「雀舌」形狀的關鍵工序。採用手工「拖帶」(手工拖带, shǒugōng tuōdài) 技法：師傅在加熱表面拖拉葉片，同時壓扁和拉直每個芽頭。正是在此形成扁平、整齊、略尖的「舌頭」輪廓。

* **攤晾 (摊晾 — tān liáng)：** 攤放葉片平衡水分，在最終加工前「休息」。

* **隔離整形 (隔离整形 — gélí zhěngxíng)：** 進一步修正形狀：去除不平整處，確保均勻性。

* **輝乾 (辉干 — huīgān)：** 在竹篩上輕柔搓揉 (竹筛轻揉) 最終成形表面並部分去除殘餘水分。

* **烘焙提香 (烘焙提香 — hōngbèi tíxiāng)：** 最終60–70°C低溫乾燥。緩慢、溫和加熱將栗香「鎖」在茶葉內，不過度加熱葉片。

**6. 感官特徵：**

* **乾茶外形：** 扁平勻直 (扁平匀直, biǎnpíng yúnzhí)，翠綠色 (翠绿, cuìlǜ) 帶明顯白毫 (显毫, xiǎn háo)。形狀微型 — 如山雀舌頭。葉片極其均勻。

* **乾茶香氣：** 嫩栗香 (嫩栗香, nèn lìxiāng) — 巴山雀舌品質的特色標誌。伴有清香 (清香, qīngxiāng)。

* **茶湯香氣：** 栗香為主調，持久綿長。高等級茶品有額外細膩花香層次。

* **滋味：** 鮮爽 (鲜爽, xiānshuǎng) 突出 — 高胺基酸含量 (≥3.0%) 的結果。明顯甘甜 (甘, gān)。持久穩定的回甘生津 (回甘生津, huígān shēngjīn)。茶體適中飽滿。在適當溫度下幾乎無苦澀味。

* **湯色：** 杏綠 (杏绿)，明亮 (明亮, míngliàng) 透澈。

* **葉底 (沖泡後葉片)：** 嫩綠鮮活 (嫩绿鲜活, nèn lǜ xiānhuó)，均勻。芽頭完整展開如「花朵」(匀整成朵)。

**7. 化學成分：**

* **多酚類 (兒茶素)：** 特級≥28%。水浸出物≥45% — 中國綠茶中最高指標之一，證明內含物質極其豐富。

* **胺基酸 (包括L-茶胺酸)：** 一級及以上≥3.0%。高胺基酸含量是決定鮮爽「多汁」口感的關鍵因素。

* **硒 (Se)：** 獨特特色 — 茶葉中硒含量顯著高於全國平均水準。萬源土壤含硒0.28–0.51微克/克；茶葉有效積累此微量元素。據生產商聲稱，巴山雀舌乾茶硒含量比普通綠茶高數倍。

* **生物鹼：** 咖啡因 (適中含量)、可可鹼、茶鹼。

* **維生素：** 維生素C、B群維生素 (B1, B2)、維生素E。

* **礦物質：** 鉀、鎂、鋅、錳、氟，以及此風土條件獨有的硒。

* **精油：** 提供特色持久栗香。海拔800–1200公尺的日夜溫差加強芳香化合物合成。

**8. 健康功效：**

* **強效抗氧化作用：** 高多酚含量 (≥28%) 與天然硒的結合產生協同抗氧化效果 — 硒是穀胱甘肽過氧化酶的組成部分，這是機體抗氧化防護的關鍵酶之一。
* **補充硒缺乏：** 硒是甲狀腺、免疫系統和生殖健康正常功能所必需的微量元素。富硒土壤茶葉是有機硒的安全來源。
* **提神效果：** 咖啡因與L-茶胺酸的結合提供溫和、平衡的清醒感和注意力改善。
* **心血管系統支持：** 兒茶素有助於脂質代謝正常化和維持血管彈性。
* **消化支持：** 多酚和兒茶素刺激消化過程並正常化腸道菌群。
* **牙齒保護：** 高氟含量抑制致齲細菌活性。
* **清涼作用：** 在傳統中醫中，綠茶屬於「涼性」食品，有助於散發內熱。

**9. 沖泡方法：**

* **水溫：** 80–85°C。特級茶 (特级) 可降至75°C — 原料越嫩，溫度越低。

* **茶量：** 3克配150毫升 (玻璃杯，1:50比例) 或4–5克配100–120毫升 (蓋碗，快沖法)。

* **茶具：** 玻璃杯 (玻璃杯) — 最佳選擇：可欣賞扁平「雀舌」在水中直立，如微型扇子般展開。也適合白瓷蓋碗 (白瓷盖碗)。

* **沖泡過程：**
	1. 用熱水溫杯或溫蓋碗。
	2. 投入茶葉。
	3. 「中投法」(中投法)：注入1/3容量的水 (80–85°C)，輕搖杯子「醒香」(摇香)，然後注滿水。
	4. 第一泡：30秒。
	5. 後續沖泡：每次增加10秒。
	6. 茶葉可沖泡3–4次 (二級茶可達5次以上)。

**10. 儲存：**

* 密封包裝 (鋁箔或真空)，避光、防潮、防異味。
* 最佳儲存 — 冰箱0–5°C密封保存。建議預先取出1–2週用量，避免頻繁開啟主包裝。
* 如在2個月內飲用完畢 — 可在室溫陰涼乾燥處保存。
* 新茶 (新茶) 建議「醒茶」10–15天後首次沖泡，以減輕加工後的「火氣」(火气)。
* 開封後1個月內飲用完畢以保持最佳新鮮度。

**11. 價格與仿冒品：**

* **價格區間：** 四川綠茶中高檔次。參考零售價：特級 (特级，明前茶) — 每斤 (500克) 800元人民幣起；一級 (一级，雨前茶) — 400–600元；二級 (二级) — 較親民，適合日常飲用。

* **避免仿冒品方法：**
	* 向「巴山雀舌」(巴山雀舌) 品牌官方經銷商購買。主要生產商為萬源國營企業及其授權合作夥伴。
	* 檢查包裝上「萬源富硒茶」(万源富硒茶) 地理標誌和/或「巴山雀舌」(巴山雀舌) 商標。
	* 外觀檢查：正品茶葉扁平整齊，茸毫明顯，翠綠均勻。仿品常形狀不整、色澤暗淡、茸毫稀少。
	* 香氣應為純正栗香，無「香精」或陳腐氣味。
	* 茶湯透澈杏綠。混濁或偏黃茶湯為品質低劣或陳茶標誌。

**12. 趣味知識：**

* **中國最古老茶葉石刻銘文。** 石窩鎮《紫雲坪植茗靈園記》(《紫云坪植茗灵园记》，1109年) 銘文是中國最古老記錄茶葉栽培的石刻文字。其拓片收藏於杭州中國茶葉博物館。值得注意的是，銘文記錄了從福建向大巴山移植茶苗的事蹟 — 證明11–12世紀之交跨地區「茶葉交流」的存在。

* **將軍書法。** 「巴山雀舌」由將軍張愛萍 (张爱萍，1910–2003) 親筆題寫 — 中華人民共和國傑出軍事將領之一，四川人。其書法至今用作品牌官方標誌。

* **硒王。** 萬源是四川唯一被認定為「天然富硒區」(天然富硒区) 的地區。土壤硒含量0.28–0.51微克/克，個別區域達12ppm。此土壤茶葉是補充硒缺乏最便利愉悅的方式之一，據世界衛生組織評估，硒缺乏影響40多個國家的人口。

* **清代官員詩句。** 王夢庚 (王梦庚) 兩次擔任萬源縣令 (1808, 1826)，坦承第二次回任是為了當地茶葉：「雀舌芒欺蒙頂撷...」— 官員為茶而非職責賦詩的罕見例子。

* **486項參數。** 巴山雀舌通過歐洲SGS標準486項參數檢測 — 少數中國綠茶能達到的成就。

**13. 與其他「雀舌」及四川-陝西茶葉比較：**

* **寧強雀舌 (宁强雀舌)：** 陝西。同為「雀舌」，但來自漢中 (富硒土壤)。巴山來自達州，同樣富硒。兩者均溫和，低澀味。

* **蒲江雀舌 (蒲江雀舌)：** 四川。扁平，「豆香」，大規模生產。巴山更「花香」，以栗香為基調且富硒。

* **金壇雀舌 (金坛雀舌)：** 江蘇。東部「雀舌」，無硒。巴山富硒，栗香調更明顯。

**結語：**

巴山雀舌是獎勵耐心探索者的茶葉。在樸實的「雀舌」名稱背後，隱藏著千年歷史、中國最古老的茶葉栽培石刻記錄、富硒土壤的獨特風土，以及不惜筆墨題字的將軍。杯中呈現溫和清香伴栗香溫暖，純淨甘甜無苦澀，每一口都有綿長回味。此茶適合尋求完美口感與真正健康益處結合的人 — 珍視背後有真實而非虛構歷史的茶葉。

**13. 與其他「雀舌」及四川-陝西茶類的比較：**

* **寧強雀舌（宁强雀舌）：** 陝西。同樣是「雀舌」，但產自漢中（富硒土壤）。巴山產自達州，同樣富含硒。兩者都柔和，澀度低。

* **蒲江雀舌（蒲江雀舌）：** 四川。扁平，「豆香」，產量大。巴山更偏「花香」，帶栗香基調和硒元素。

* **金壇雀舌（金坛雀舌）：** 江蘇。東部「雀舌」，不含硒。巴山富含硒，栗香調性更明顯。

巴山雀舌是那種會獎賞耐心探索者的茶。在樸素的「雀舌」名稱背後，隱藏著千年歷史、中國最古老的茶葉種植岩畫記錄、富含硒元素的獨特風土，以及一位不吝揮毫題字的將軍。杯中是帶著栗香溫潤的柔和鮮爽、無苦的純淨甘甜，以及隨著每一口而漸增的悠長回甘。這款茶適合那些尋求完美口感與實際健康益處結合的人——以及珍視擁有真實而非虛構歷史的茶葉的人。
