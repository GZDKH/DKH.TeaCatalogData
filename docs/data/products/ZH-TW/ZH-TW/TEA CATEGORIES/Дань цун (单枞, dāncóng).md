# 單叢 (单枞, dāncóng)

**單叢 (单枞, dāncóng)**

![Дань цун (单枞, dāncóng)-1](images/%D0%94%D0%B0%D0%BD%D1%8C%20%D1%86%D1%83%D0%BD%20%28%E5%8D%95%E6%9E%9E%2C%20d%C4%81nc%C3%B3ng%29-1.png)

**單叢，翻譯為「孤獨的灌木」或「單獨的灌木」** — 這不是單一種類的茶，而是一**組烏龍茶**，以其廣泛的天然香氣而聞名，這些香氣模仿各種花朵和水果的氣味。

**1. 分類與產地：**

* **類型：** 烏龍茶（半發酵茶）。發酵和焙火程度可以變化，但通常為中等（20-60%）。
* **類別：** 高品質烏龍茶。單叢屬於**閩北烏龍茶（北福建烏龍茶）**，但有時因其獨特特性而被歸為單獨類別。
* **產地：** 中國，廣東省 (广东, Guǎngdōng)，潮州市 (潮州, Cháozhōu)，鳳凰山 (凤凰山, Fènghuáng Shān) - 「鳳凰山」。具體村莊：烏岽 (乌岽)、大庵 (大庵)、獅頭 (狮头)、竹坑 (竹坑) 等。
* **地理座標：** 約北緯24°，東經116°。

**2. 歷史與文化意義：**

* **歷史：** 鳳凰山的茶葉種植歷史超過900年。據推測，第一批茶樹在宋朝（960-1279年）時期就已在此種植。最初，這個地區的所有茶葉可能都屬於單叢類別。後來由於選育和雜交，出現了許多現在歸入單叢組的品種。
* **名稱：**

	* 「單」(单) - 單一、個別、分離。
	* 「叢」(丛) - 灌木、叢林。
	* 「單叢」(单丛) - 「單獨的灌木」或「分離的灌木」。這個名稱反映了**傳統的採摘和加工方法**：理想情況下，每種單叢的葉子都是**從每個茶樹（或同一品種的一組茶樹）分別採摘和加工**，這樣可以保持**每株植物的個別特性**。現在這種做法並不總是嚴格遵守，但「單叢」一詞仍然表示**高品質**和**品種/變種的獨特性**。
* **文化意義：** 單叢是鳳凰山的珍寶。它們是當地居民的驕傲，在全世界的茶葉鑑賞家中備受推崇。單叢的生產是真正的藝術，需要對茶葉的深刻理解和高超的技藝。品飲單叢不僅僅是解渴，而是沉浸在奇妙的香氣和味道世界中。

**3. 植物學描述與原料：**

* **品種：** 單叢不是單一品種茶。這是**一組以產地和生產特點結合的茶葉**。鳳凰山生長著**多種茶樹變種**，按**香氣**、**葉形**、**採摘時間**和其他特徵分類。其中一些變種：
	* **蜜蘭香 (蜜兰香, Mì Lán Xiāng)：** 「蜂蜜蘭花香」- 最著名和最受歡迎的單叢。
	* **鴨屎香 (鸭屎香, Yā Shǐ Xiāng)：** 「鴨糞香」- 儘管名稱如此，卻具有宜人的花香。
	* **杏仁香 (杏仁香, Xìngrén Xiāng)：** 「杏仁香」。
	* **芝蘭香 (芝兰香, Zhī Lán Xiāng)：** 「芝蘭蘭花香」。
	* **肉桂香 (肉桂香, Ròu Guì Xiāng)：** 「肉桂香」。
	* **玉蘭花香 (玉兰花香, Yù Lán Huā Xiāng)：** 「玉蘭花香」。
	* **夜來香 (夜来香, Yè Lái Xiāng)：** 「夜來香」（晚香玉）。
	* **薑母香 (姜母香, Jiāng Mǔ Xiāng)：** 「薑根香」。
	* **宋種 (宋种, Sòng Zhǒng)：** 「宋種」，被認為是最古老的品種之一。
	* **八仙 (八仙, Bā Xiān)：** 「八仙」
	* 以及許多其他品種。
* **茶樹年齡：** 鳳凰山有許多古老的茶樹，年齡可達數百年。它們被稱為**「老叢」(老枞, Lǎo Cōng) - 「老茶樹」**。來自這些茶樹的茶葉特別珍貴。也有年輕的種植園。
* **採摘：** 採摘主要在春季進行，但也可能在夏季和秋季進行。春季採摘被認為是最珍貴的，特別是早春採摘。
* **採摘標準：** 通常採摘芽和兩到三片頂葉。對於精品單叢，可能只使用芽或芽加一片葉子。
* **原料要求：** 要求很高。只使用健康、未受損的葉子，在乾燥天氣中採摘。

**4. 風土條件與種植特點：**

* **鳳凰山（鳳凰山）：** 位於廣東省東北部，靠近南中國海海岸。山脈主要由花崗岩和火山岩組成。以風景如畫的景觀、清潔的空氣和豐富的霧氣而著稱。
* **種植海拔：** 茶園位於海拔400至1500公尺之間。高山茶（1000公尺以上）特別珍貴。
* **土壤：** 鳳凰山的土壤呈酸性，富含有機物和礦物質，排水良好。它們為茶葉帶來特有的礦物質味道。
* **氣候：** 亞熱帶季風氣候，冬季溫暖潮濕，夏季炎熱多雨。年平均氣溫約21°C。高濕度、頻繁的霧氣、充足的陽光和顯著的日夜溫差為茶葉種植創造了理想條件。
* **特點：** 鳳凰山的許多茶樹都非常古老，年齡達數百年。它們被稱為「老叢」(老枞) - 「老茶樹」。據信來自這些茶樹的茶具有更深層和複雜的味道，以及特殊的能量 - 「茶氣」(茶氣)。傳統上，鳳凰山的許多茶園不是通常意義上的種植園，而是生長在岩石和森林中的單獨茶樹或小群茶樹。

**5. 生產工藝：**

單叢的生產工藝結合了傳統烏龍茶製作方法和潮州地區的特色。關鍵要點是**細心但強烈的搖青**和**多次焙火**。
* **採摘 (采摘 - cǎi zhāi)：** 如上所述。手工採摘。
* **萎凋 (萎凋 - wěidiāo)：** 將採摘的葉子攤放在露天（陽光或陰涼萎凋）或室內數小時。目的是去除葉子中的部分水分，使其更柔軟並開始發酵過程。
* **搖青 (摇青 - yáo qīng)：** **烏龍茶製作中最重要的階段**，特別是單叢。在竹盤上細心但強烈地搖動和翻動葉子，以啟動氧化過程。這個階段進行多次（有時多達10-12次），中間有葉子的「靜置」(静置 - jìngzhì) 休息時間。「靜置」可能持續30分鐘到數小時。搖青和「靜置」總共可能需要一天或更長時間。正是在這個階段形成了單叢豐富多樣的香氣。師傅必須具有豐富經驗，才能正確「感受」茶葉並達到所需的發酵程度。
* **發酵 (发酵 - fājiào)：** 在搖青和葉子「靜置」期間發生的氧化過程。單叢的發酵程度可以變化，但通常為中等（30-60%）。
* **殺青 (杀青 - shā qīng)：** 高溫炒制以停止發酵過程。
* **揉捻 (揉捻 - róuniǎn)：** 將葉子塑造成縱向捲曲的形狀。揉捻可以是手工或機械的。
* **乾燥 (烘干 - hōnggān)：** 乾燥茶葉以去除水分。
* **焙火 (焙火 - bèihuǒ)：** 單叢經常進行多次不同溫度的焙火（加熱）。這為它們增添了額外的風味層次，並促進其在儲存期間的進一步成熟。焙火程度可以從輕到重變化。焙火可以在電烤箱中進行，或用傳統方法 - 用木炭。
* **分級 (分级 - fēnjí)：** 將成品茶按大小和品質分級。

**6. 感官特徵：**

* **乾茶外觀：** 相對較大、縱向捲曲的葉子，深棕色，有時幾乎是黑色，帶有紅色調。可能有茶梗。有時可以注意到輕微的灰色霜，這是焙火的結果。
* **乾茶香氣：** 非常明亮、強烈、複雜，具有明顯的**花香、果香、蜂蜜香、香料香調**。每種單叢都有其獨特的香氣組合。可能有焙火香調、木質香調。
* **茶湯香氣：** 濃郁、包圍性、甜美，以花果香調為主，帶有蜂蜜、香料的色調，有時帶有輕微的煙燻味和礦物質香調。香氣非常持久和「活潑」，從一泡到另一泡都在變化。
* **滋味：** 飽滿、濃郁、油潤、甘甜，帶有輕微的澀味和清新的酸味。香氣組合以花香和果香為主，帶有蜂蜜、香料、焦糖、堅果的香調。回甘悠長、甘甜，帶有花香和果香色調。滋味根據發酵和焙火程度以及具體的單叢品種而變化。
* **茶湯顏色：** 從金黃色到琥珀紅色或棕色，透明、清澈，有明亮光澤。顏色取決於發酵和焙火程度。
* **葉底（沖泡後的葉子）：** 完整、有彈性的葉子，沖泡後展開，呈棕綠色，經常有明顯的「紅邊」- 氧化的結果。

**7. 化學成分：**

單叢富含：
* **多酚類（兒茶素）：** 強效抗氧化劑。
* **胺基酸：** 包括L-茶胺酸，負責茶的甘甜味道並具有鎮靜作用。
* **生物鹼：** 咖啡因、可可鹼、茶鹼。咖啡因含量可能相當高。
* **精油：** 極其豐富的精油成分，造就了每種單叢多面和獨特的香氣。
* **維生素：** C、B群、E、K。
* **礦物質：** 鉀、氟、鎂、錳、鐵、硒。

**8. 健康功效：**

* **提神效果：** 單叢具有明顯的提神效果，振奮精神，消除疲勞，提高工作效率，改善注意力集中和記憶力。
* **抗氧化作用：** 保護細胞免受自由基損害，延緩衰老過程，降低多種疾病的風險。
* **改善消化：** 刺激消化，促進食物吸收，特別是油膩食物。
* **溫暖作用：** 在寒冷季節很好地溫暖身體，改善血液循環。
* **心血管系統：** 可能有助於降低「壞」膽固醇水平，強化血管壁，正常化血壓。
* **排毒：** 有助於清除體內廢物和毒素。
* **提升情緒：** 帶來和諧、快樂和愉悅的感覺。有助於放鬆和緩解壓力。
* **改善新陳代謝：** 可能有助於正常化新陳代謝和減重。

**9. 沖泡方法：**

* **水溫：** 90-95°C（對大多數品種而言）。
* **茶量：** 5-7克配150-200毫升水。
* **茶具：** 理想的是**蓋碗**（傳統中國帶蓋茶杯）或**宜興紫砂壺**。宜興紫砂多孔且透氣性好，讓茶葉完全展現。宜興紫砂壺會「積累」茶香，因此建議只用於單叢。
* **沖泡過程：**

	1. **溫壺：** 用開水沖洗蓋碗或茶壺，溫熱茶具並為沖泡做準備。
	2. **洗茶（快速沖泡）：** 將茶放入蓋碗，倒入少量熱水並立即倒掉。這個步驟可以洗去葉子上的灰塵，同時「喚醒」茶葉，為其展開做準備。
	3. **第一次沖泡：** 用熱水（90-95°C）沖泡茶葉，浸泡幾秒到1-2分鐘（第一泡）。第一泡時間可以很短，僅5-15秒，特別是高品質茶葉。根據您的口味和喜好的濃度調整。
	4. **倒出茶湯：** 將茶湯完全從蓋碗或茶壺倒入公道杯，然後分到茶杯中。這是為了讓所有茶杯都得到相同濃度的茶湯。使用蓋碗時，也可以不用公道杯直接分茶，但要避免茶葉進入茶杯。
	5. **重複沖泡：** 單叢可以多次沖泡（5-7次，有時更多），每次後續沖泡逐漸增加10-30秒的浸泡時間。每一泡的味道和香氣都會變化，展現新的層面。單叢的特點是能夠承受多次沖泡而不失去風味品質。

**重要細節：**

* **不要過度浸泡：** 過長的浸泡時間可能使茶味變得澀苦。
* **聆聽茶葉：** 根據自己的感受調整沖泡時間，以達到所需的茶湯濃度。
* **觀察茶葉：** 注意茶湯顏色、香氣、茶葉的展開。這將幫助您更好地理解茶的特性並找到最佳沖泡方法。
* **實驗：** 不要害怕嘗試不同的沖泡方法、水溫、浸泡時間，以找到您的理想選擇。

**10. 儲存：**

單叢應儲存在乾燥、涼爽、陰暗的地方，密封容器中（最好是瓷器或陶瓷罐），遠離異味。焙火程度高的單叢通常比輕焙火的保存得更好。

**11. 價格與假冒：**

單叢，特別是來自老茶樹（老叢）的，屬於**昂貴、精品茶葉**類別。高價格是由於：
* **有限的產量：** 來自單獨茶樹和老茶樹的原料數量有限。
* **採摘和加工的複雜性：** 手工採摘和多階段加工需要大量勞動。
* **獨特的風味香氣特徵：** 單叢具有獨特的香氣和味道，受到鑑賞家高度評價。
* **高需求：** 對單叢的需求，特別是稀有品種，持續增長。

由於高價格和受歡迎程度，市場上不幸存在許多假冒和仿製品。
**如何避免假冒：**

* **只從可信賴的賣家購買：** 尋找聲譽卓越的專業茶葉商店，他們珍惜客戶並能提供關於茶葉來源、採摘年份、生產者的可靠資訊。他們也應該保證其真實性和品質。
* **警惕過低的價格：** 可疑的低價格幾乎總是假冒的確切標誌。真正的單叢不可能便宜。記住，奇蹟是不存在的。
* **仔細檢查外觀：** 注意葉子的形狀、顏色、完整性。它們應該符合特定單叢品種的描述。大量破碎葉子、粉末、雜質的存在是低品質或假冒的標誌。
* **評估香氣：** 乾茶應具有濃郁、複雜的香氣，符合該單叢品種的特徵。避免香氣微弱、不明顯、陳腐或有異味的茶葉。不良商家有時使用的人工香料通常會暴露出過於刺鼻、不自然的氣味。
* **檢查茶湯和葉底：** 茶湯顏色應符合特定品種的描述，透明清澈。葉底應由完整、有彈性的葉子組成。
* **購買標明茶樹年齡（「老叢」）或具體採摘地點的單叢時要特別小心：** 驗證這些資訊極其困難，因此只信任可靠來源。

**12. 有趣事實：**

* **「單獨的灌木」：** 「單叢」這個名稱強調了鳳凰山每個茶樹的獨特性，其傳遞給茶葉的個別特徵。
* **「活化石」：** 鳳凰山的一些茶樹非常古老，年齡達數百年。它們被稱為茶葉世界的「活化石」。
* **香氣多樣性：** 鳳凰山有超過一百種單叢品種，每種都具有其獨特的香氣。這種多樣性是自然選擇和獨特生長條件的結果。
* **變色龍茶：** 單叢以其從一泡到另一泡改變味道和香氣的能力而聞名，展現新的層面。
* **冥想茶：** 由於其豐富的味道、香氣和提神效果，單叢經常用於茶藝和冥想。

**13. 飲用文化：**

* **功夫茶：** 單叢非常適合用功夫茶法沖泡 - 傳統中國茶藝。這種方法能最大限度地展現茶的味道和香氣，同時享受沖泡過程本身。
* **茶具：** 沖泡最好使用蓋碗或宜興紫砂小壺。宜興紫砂被認為是烏龍茶的理想選擇，因為它多孔且讓茶葉「呼吸」，同時「記住」茶香，隨著時間推移改善茶湯味道。
* **與食物搭配：** 單叢通常不與食物搭配，以免干擾其精緻的味道和香氣。這種茶最好單獨品飲，享受每一口。
* **飲用時間：** 單叢可以在一天中的任何時間飲用，但特別適合下午和晚上品茶，因為它們具有提神效果，同時有助於放鬆。

**結論：**

單叢是一組令人驚嘆的烏龍茶，是鳳凰山的真正珍寶。每種單叢都具有其獨特的特性，獨特的風味香氣特徵，這是由於特定茶樹品種的特點、風土條件和生產者的技藝而形成的。品嚐真正的單叢意味著為自己打開豐富的味道和香氣世界，接觸古老的茶葉種植傳統，體驗無與倫比的茶葉享受。這是為那些珍視真實性、獨特性並準備踏上鳳凰山坡尋找理想單叢的迷人旅程的人而準備的茶。
