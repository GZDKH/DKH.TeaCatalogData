# 柑桔茶 (柑桔茶, gān jié chá)

**柑桔茶 (柑桔茶, gān jié chá)**

**1. 分類與產地：**

* **類型：** 取決於基底茶。可以是**黑茶**、**紅茶**、**綠茶**、**白茶**或**普洱茶（熟普或生普）**，加入柑橘類，或將茶葉置於水果內部。此外，也可以是由乾燥柑橘皮製成的**草本茶飲**。
* **類別：** 調味茶、水果茶、添加物茶。
* **產地：** 在茶中添加柑橘類的傳統在中國有著悠久的歷史。黑茶與柑橘的組合在**廣東省 (广东, Guǎngdōng)**特別受歡迎。此外，將茶葉置於挖空水果內部是一個受歡迎的茶類別，包括例如「大紅柑」(大红柑) - 置於挖空橙子內的紅茶，以及「小青柑」(小青柑) - 置於挖空青桔內的茶葉。
* **地理座標：** 取決於基底茶和柑橘類的產地。

**2. 歷史與文化意義：**

* **歷史：** 確切不知中國何時開始在茶中添加柑橘類，但這種做法可能已有數個世紀的歷史。乾燥的柑橘皮（陳皮 - 陳皮）自古以來就在傳統中醫中使用。
* **名稱：**

	* 「柑桔」(柑桔/柑橘) - 柑橘類的總稱，包括桔子、橙子、香櫞等。
	* 「茶」(茶) - 茶。
* **文化意義：** 柑橘茶是中國的熱門飲品，特別是在南方地區。它被認為具有清爽、有益消化和富含維生素C的特性。置於水果內部的茶也具有美學功能。

**3. 植物學描述與原料：**

* **茶葉：** 用作基底的茶葉種類可以不同：
	* **黑茶：** 最常見的是熟普洱，較少見的是生普洱。
	* **紅茶：** 各種紅茶品種。
	* **綠茶：** 較少見。
	* **白茶：** 也有使用，但較少見。
	* **其他：** 有時會遇到添加柑橘類的烏龍茶。
* **柑橘類：** 用於調味和作為茶葉「容器」的各種柑橘類：
	* **桔子 (Citrus reticulata)：** 不同品種，包括青桔（未成熟）和成熟桔子。
	* **橙子 (Citrus sinensis)：** 不同品種，包括「大紅」(大红)，意為「大紅」，但指的是橙子品種。
	* **佛手柑 (Citrus medica var. sarcodactylis)：** 使用較少，但為茶帶來獨特香氣。
	* **柚子 (Citrus maxima)：** 有時用於製作大型「茶彈」。
	* **佛手柑 (Citrus bergamia)：** 較少見，但可用於調味。
	* **檸檬 (Citrus limon)：** 通常以乾燥形式使用（片狀、皮屑）。
* **採收：** 茶葉採收時間取決於其種類。柑橘類按成熟度採收。
* **原料要求：** 茶葉和柑橘類的品質都很重要。

**4. 風土與栽培特點：**

* **地區：** 取決於基底茶和柑橘類的產地。中國南方省份（廣東、福建、雲南）是茶葉和柑橘類的主要供應地。
* **氣候：** 亞熱帶或熱帶氣候，冬季溫暖，夏季炎熱。

**5. 製作工藝：**

柑桔茶有幾種主要製作工藝：
* **水果內茶葉：**

	1. **水果準備：** 從柑橘類（桔子、橙子、柚子）切掉頂部，小心移除果肉，將果皮晾乾。
	2. **填充：** 在準備好的水果中填入茶葉（最常見的是熟普洱，較少見其他種類）。
	3. **封閉：** 用切下的頂部封閉開口或保持開放。
	4. **乾燥/發酵：** 含茶的水果進行進一步處理：
		* **乾燥：** 可在陽光下、烘爐中或乾燥櫃中進行。在乾燥過程中，茶葉吸收水果香氣。
		* **發酵：** 對於置於水果內的熟普洱，發酵過程繼續進行，受到水果內部微氣候的影響。
	5. **儲存：** 成品儲存在乾燥、涼爽的地方。有時茶葉會特意陳放數年以改善味道和香氣。
* **添加乾燥柑橘類的茶：**

	1. **柑橘類準備：** 清洗水果，切成片狀、圓片或取皮屑並乾燥。
	2. **混合：** 將乾燥的片狀、圓片或皮屑按一定比例添加到成品茶（紅茶、綠茶、白茶、烏龍茶）中。
	3. **儲存：** 混合物儲存在密封容器中，讓茶葉吸收柑橘香氣。
* **調味：** 在某些情況下，茶葉可能用柑橘精油或其他天然香料調味。

**6. 感官特徵：**

柑桔茶的感官特徵取決於基底茶類型、柑橘類種類、製作工藝和陳放程度（如有）。
* **外觀：**

	* **水果內茶葉：** 完整的乾燥水果（桔子、橙子、柚子），內含茶葉。水果的大小和顏色取決於其種類。
	* **添加物茶：** 茶葉與乾燥柑橘皮屑或柑橘片的混合物。
* **香氣：** 明亮、濃郁的柑橘香氣（桔子、橙子、佛手柑、檸檬等），與茶香混合。可能有甜味、香料味、木質調。
* **味道：** 取決於茶葉種類。通常有明顯的柑橘酸味、甜味，以及基底茶特有的味道層次（澀味、木質、堅果、花香調等）。
* **湯色：** 取決於茶葉種類。可從淺黃色（綠茶）到深棕色（熟普洱）。通常有橙色或紅色調，來自柑橘皮。
* **葉底：** 取決於茶葉種類。

**7. 化學成分：**

除了基底茶特有的物質（多酚、胺基酸、生物鹼、維生素、礦物質）外，柑桔茶還含有：
* **柑橘精油：** 檸檬烯、檸檬醛等，賦予特有香氣。
* **維生素C：** 柑橘皮含有大量維生素C。
* **類黃酮：** 茶葉和柑橘類中都含有。
* **有機酸：** 檸檬酸、蘋果酸等。

**8. 健康功效：**

柑桔茶的健康功效由基底茶和添加柑橘類的特性組合決定：
* **提神效果：** 取決於基底茶種類。
* **抗氧化作用：** 延緩老化過程，保護細胞免受損傷。
* **增強免疫力：** 由於維生素C和其他有益物質。
* **改善消化：** 柑橘類和某些茶類（如普洱）刺激消化。
* **溫暖/清爽作用：** 取決於基底茶種類。
* **抗感冒作用：** 由於維生素C和精油。
* **對呼吸系統的益處：** 柑橘精油可能有助於呼吸。
* **改善情緒：** 明亮的柑橘香氣提神並提升情緒。

**9. 沖泡方法：**

* **水溫：** 取決於基底茶種類。綠茶為75-85°C，紅茶為90-95°C，熟普洱為95-100°C。
* **茶量：** 取決於茶葉種類和個人喜好。通常每150-200毫升水用3-5克茶。水果內茶葉用1個水果配一個茶壺/蓋碗。
* **茶具：** 蓋碗、紫砂壺、玻璃壺、瓷器。
* **沖泡過程：**

	* **水果內茶葉：** 類似熟普洱（或內部其他茶類）的沖泡方法。可以悶泡，也可以快速出湯。有時預先將水果弄破以便沖泡。
	* **添加物茶：** 取決於茶葉種類。通常像普通茶（綠茶、紅茶、白茶）一樣沖泡，但浸泡時間稍短，以免柑橘味掩蓋茶味。

**10. 儲存：**

柑桔茶應儲存在乾燥、涼爽、陰暗的地方，密封容器中，遠離異味。水果內茶葉可直接在水果中儲存，但最好放入密封容器中。

**11. 價格與假冒：**

柑桔茶的價格可能因基底茶種類、柑橘品質、製作工藝、品牌知名度和購買地點而大幅變化。水果內茶葉通常比添加物茶更昂貴。

**如何避免假冒：**

* **向可靠商家購買：** 尋找有良好聲譽的專業茶葉店。
* **注意價格：** 過低的價格應引起警覺。
* **仔細檢查外觀：** 茶葉和水果的品質應該很高。
* **評估香氣：** 香氣應該天然，無人工香精。

**12. 有趣事實：**

* **桔子/橙子內茶葉是熱門禮品：** 這種茶看起來非常吸引人，經常作為節日禮品。
* **味道多樣性：** 不同茶類與不同柑橘類的組合提供了巨大的味道和香氣多樣性。
* **創意方法：** 一些生產商進行實驗，不僅將茶葉放入桔子和橙子中，還放入其他水果，如葡萄柚、青檸甚至南瓜中。

**結論：**

柑桔茶是一整類多樣化的茶產品，以使用柑橘類作為添加物或「包裝」為共同特點。這可以是挖空水果內的茶葉，也可以是茶葉與乾燥片狀或皮屑的簡單混合。這種茶的味道、香氣、沖泡方法和健康功效將取決於具體的基底茶種類和使用的柑橘類。但無論如何，柑桔茶都是一種有趣且美味的產品，值得茶葉和柑橘愛好者嘗試。
