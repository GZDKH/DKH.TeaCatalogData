# 英德绿茶 (英德绿茶, Yīngdé lǜchá)

**英德绿茶 (英德绿茶, Yīngdé lǜchá)**
![Индэ Люй Ча (英德绿茶, Yīngdé lǜchá)](images/Индэ%20Люй%20Ча%20(英德绿茶,%20Yīngdé%20Lǜchá).png)

英德绿茶 (英德绿茶, Yīngdé lǜchá) — 广东省最大的绿茶，产于该省北部的英德市。虽然英德以著名的红茶英德红茶 (英德红茶) 闻名于世，但其绿茶「兄弟」同样令人瞩目：采用云南大叶种制作，具有极高的胺基酸含量（≥5% — 比平均水准高1.5倍）和独特的「医疗」特性 — 1986年广东科学家证实，英德茶中锰含量的增加与降低肝癌风险相关。到2025年，英德茶园面积超过141,000亩（9,400公顷），年产量11,000吨，总价值48亿元人民币：英德是广东茶业无可争议的领导者。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。属于烘青绿茶 (烘青绿茶, hōngqīng lǜchá)。特点 — 采用大叶种原料 (大叶种, dàyè zhǒng) 制作，这使其有别于大多数著名绿茶使用小叶种品种。

* **类别：** 地理标志产品。广东省绿茶产业的旗舰。清朝历史上的「贡茶」(贡茶)。

* **产地：** 中国广东省 (广东, Guǎngdōng) 英德市 (英德市, Yīngdé Shì)。生产区域 — 整个英德市辖区。核心产区 — 沙堆镇 (沙堆镇, Shāduī Zhèn) 和水边镇 (水边镇, Shuǐbiān Zhèn)，提供超过60%的总产量，其中沙堆是特级茶的主要来源 (特级)。

* **地理座标：** 约北纬24°10′，东经113°22′（北纬24度 — 「广东茶叶带」）。

**2. 历史与文化意义：**

* **历史：** 英德的茶叶种植可追溯至唐朝：陆羽 (陆羽) 在《茶经》(茶经) 中提到，韶州 (韶州，包括英德在内的古代地区名称) 产茶且「其味极佳」(其味极佳)。英德是唐代韶州三个主要产茶县之一。

	明朝时期，茶叶成为该地区重要的农产品。清朝时期 — 因「茶汤清亮、滋味鲜爽」(茶汤清亮、滋味鲜爽) 而被列入贡茶 (贡茶) 名单。

	现代历史的转捩点 — **1990年代**：「红改绿」(红改绿，「从红茶改为绿茶」) 计划 — 大规模将茶园从红茶（黑茶）生产转换为绿茶。云南大叶种 (云南大叶种)，原本专门用于红茶生产，被改良用于生产「高香浓味绿茶」(高香浓味绿茶)。这一转换将英德从广东的「红茶」之都变成了双面巨人 — 既生产红茶又生产绿茶。

	1986年，广东省医学院的科学家进行了研究，证实英德茶中**锰含量增加**与降低肝癌风险 (降低肝癌风险) 相关。这是中国首批将茶叶中特定微量元素与癌症预防联系起来的科学研究之一。

	2019年茶园面积达到141,000亩。到2025年，英德茶产业总价值达48亿元人民币。

* **名称：**

	* 「英德」(英德) — 城市名称。字面意思：「英国美德」— 历史名称，与英国无关，而是源于古代中国地名学。
	* 「绿茶」(绿茶) — 「绿茶」。

* **文化意义：** 英德是中国南方最大的茶叶生产城市之一，「广东茶都」。英德绿茶体现了成功的「从红到绿」转型 — 这是世界茶业中罕见的整个地区从一种产品类型大规模转换到另一种类型的例子。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要 — **云南大叶种 (云南大叶种, Yúnnán Dàyè Zhǒng)** — 云南大叶种 *Camellia sinensis* var. *assamica*。乔木型 (乔木型)，具有高抗寒性（在广东条件下）。春季原料的化学成分：多酚类 ≥28.3%，胺基酸 ≥5% — **比标准小叶种绿茶高1.5倍**。正是大叶种原料提供了特有的「浓醇」(浓醇) 口感。云南大叶种的种植在1990年代「红改绿」计划后开始占主导地位。

* **采摘：** 四个季节：
	* **明前雀舌 (明前雀舌，「清明前雀舌」)：** 清明前采摘的完整芽头。形状 — 「雀舌」(雀舌)。茸毛丰富。特级 (特级)。
	* **雨前翠芽 (雨前翠芽，「谷雨前翠芽」)：** 谷雨前一芽一叶。一级 (一级)。
	* **夏暑茶 (夏暑茶，「夏季茶」)：** 一芽二叶。叶片厚实，耐泡。大宗茶。
	* **秋香茶 (秋香茶，「秋季香茶」)：** 白露 (白露，「白露」) 前。天然花香。高性价比。

* **原料要求：** 嫩度均匀的芽叶。当日采摘当日加工。

**4. 风土条件与栽培特点：**

* **位置：** 粤北 (粤北)，丘陵地带。北纬24度。

* **气候：** 南亚热带季风气候 (南亚热带季风气候)。年平均气温 — 15.5–20.7°C，年降雨量 — 1,200–1,876毫米。年平均云雾天数 — **>200天**。日夜温差 — >10°C。散射光比例 — **70%** — 茶区中最高的指标之一。

* **土壤：** 喀斯特地貌 (喀斯特地貌) 酸性黄壤 (酸性黄壤)，pH 4.5–6.5。天然富含**硒和锰** (锰, měng)。森林覆盖率 — 68.7%。无工业污染。

* **核心产区的独特性：** 在沙堆和水边镇，原料中胺基酸含量达到**147.33毫克/克** — 极高的指标，解释了显著的鲜爽和回甘「鲜爽回甘」。多酚类和咖啡因的比例平衡，最大限度地减少了苦涩味。

**5. 生产工艺：**

英德绿茶是烘青绿茶 (烘青)，在杀青阶段采用特有的「闷扬结合」(闷扬结合, mèn yáng jiéhé — 「闷炒和抛炒结合」) 技法。

* **采摘 (采摘 — cǎizhāi)：** 手工采摘「一芽一至三叶」标准的芽叶（根据季节和等级而定）。

* **轻萎凋 (轻萎凋 — qīng wěidiāo)：** 在竹筛上摊晾 (竹筛摊晾) 约4小时。含水量降至约70%。与标准摊放的区别 — 正是「萎凋」(萎凋)，而不仅仅是「摊放」(摊放)：原料经过受控的失水过程。

* **杀青 (杀青 — shāqīng)：** 在铁锅中于240°C进行。**关键技法 — 「闷扬结合」：** 师傅交替进行「闷炒」(闷, mèn — 叶片贴锅底，在自身蒸汽中加热，约2分钟) 和「抛炒」(扬, yáng — 叶片大力抛起，释放蒸汽，约6分钟)。闷炒确保大叶片深度加热；抛炒防止过热并「锁住翠绿色」(锁翠, suǒcuì)。这些技法的结合降低了大叶种原料的苦涩味 (降低多酚苦涩)。

* **揉捻 (揉捻 — róuniǎn)：** 按「轻 → 重 → 轻」(轻-重-轻) 原则，持续40分钟。为特级茶塑造「雀舌形」(雀舌形)。碎茶率 (碎茶率) — <5%。

* **干燥 (干燥 — gānzào)：** 两阶段：初烘至120°C至约70%干度 (初烘至七成干)；复烘90°C → 60°C (复干)。

**6. 感官特征：**

* **干茶外形：** 特级 — 紧实细嫩的「雀舌形」(雀舌形, quèshé xíng)，翠绿色带丰富的银毫 (翠绿披毫)。大宗茶 — 卷曲形 (卷曲形)。叶片明显比小叶种绿茶大 — 大叶种品种的结果。

* **干茶香气：** 清纯、高扬、持久 (清香, qīngxiāng)。兰花香 (兰花香) 和栗香 (栗香) 调性。空杯留香 — **>30分钟** — 真品标志。

* **茶汤香气：** 高扬持久，具有栗香兰花香特征。

* **滋味：** 浓醇饱满 (浓醇, nóngchún) — 大叶种原料多酚类含量高的结果。鲜爽 (鲜爽) — 胺基酸 ≥5%。回甘持久稳定 (回甘持久)。特色：**「微苦无涩」— 「轻微苦味无粗涩感」** — 独特特征，由多酚类和咖啡因的平衡比例所解释。

* **汤色：** 黄绿明亮 (黄绿明亮)。

* **叶底：** 嫩匀成朵，绿黄鲜活 (嫩匀成朵，绿黄鲜活)。

**7. 化学成分：**

大叶种品种和富含锰的喀斯特土壤形成了独特的成分特征：

* **多酚类（儿茶素）：** 25–30% — 高于绿茶平均水准。EGCG — 主要成分。中和自由基的效力 — 比维生素E高10倍。

* **胺基酸：** **≥5% — 比绿茶平均水准高1.5倍**。在核心产区（沙堆）— 高达147.33毫克/克。正是胺基酸决定了显著的鲜爽和「活泼」甜味。

* **锰 (锰, měng)：** **显著高于**普通绿茶。根据广东医学院研究（1986年），锰含量增加与降低肝癌风险和增强自由基中和能力（比普通绿茶高30%）相关。

* **维生素C：** **比平均水准高1.5倍** — 温和南方气候和丰富散射光的结果。

* **生物碱：** 咖啡因 — 适中含量，与多酚类平衡。

* **硒：** 天然高含量 — 来自喀斯特土壤。

* **矿物质：** 钾、镁、锌、锰、硒。

**8. 健康功效：**

* **抗癌潜力 (防癌潜力)：** 锰含量增加 — 中和自由基的效力比普通绿茶高30%。1986年研究证实与降低肝癌风险的关联。
* **强大抗氧化作用：** 儿茶素 — 效力比维生素E高10倍。维生素C — 比平均水准高1.5倍。
* **代谢调节：** 多酚类（25–30%）促进脂肪分解。
* **提神效果：** 咖啡因和L-茶胺酸。
* **改善消化：** 多酚类刺激酶活性。

* **重要提示：** 上述功效基于公开资料，不构成医疗建议。

**9. 冲泡方法：**

* **水温：** 80–85°C；特级茶 (特级) — 75°C。

* **茶量：** 3克配150毫升水（1:50比例）。

* **茶具：** 玻璃杯（观察「雀舌」）或白瓷盖碗（评估香气）。

* **冲泡过程：**

	1. 温杯，倒掉水。
	2. 投茶。
	3. 快速洗茶 — 注水，立即倒出 (温润泡快速倒出)。
	4. 第一泡 — 30秒。
	5. 后续 — 每泡增加10秒。茶叶可冲泡3–4次。

* **注意：** 不要使用反复煮沸的水 (忌反复烧开沸水) — 这会破坏多酚类结构。不建议空腹饮用 — 高单宁含量可能刺激黏膜。开封后 — 冷藏保存，3个月内饮用完毕。

**10. 储存：**

* 密封容器，阴凉干燥处保存。
* 最佳 — 冰箱0–5°C。
* 保质期 — 最多12个月。
* 开封后 — 3个月内饮用完毕。

**11. 价格与假冒：**

英德绿茶提供三个主要等级：特级 (特级) — 「明前雀舌」(全芽，600元人民币/斤起)，一级 (一级) — 「雨前翠芽」(200–400元)，大宗茶 (大宗茶) — 用于茶包和餐厅。

* **如何避免假冒：**

	* **向可靠商家购买**，需有英德市产地证明。
	* **评估叶片大小：** 正宗英德绿茶采用大叶种品种，叶片明显比小叶种绿茶大。小而「纸质」的叶片 — 可疑。
	* **检查「冷杯香」：** 余香>30分钟 — 真品标志。
	* **评估口感：** 「微苦无涩」特色（轻微苦味无涩感）。粗涩感 — 假冒。
	* **注意价格：** 可疑的低价 — 假冒迹象。

**12. 有趣事实：**

* 1990年代的「红改绿」(「从红茶改为绿茶」) 计划 — 世界茶业中少数整个地区从红茶大规模转换为绿茶的例子之一。云南大叶种品种，数世纪来仅用于红茶生产，成功适应了绿茶生产 — 证明了 *Camellia sinensis* var. *assamica* 的多功能性。

* 1986年广东医学院证实英德茶中锰含量增加与降低肝癌风险的关联 — 中国首批「茶叶 — 抗癌」科学研究之一。

* 胺基酸含量 ≥5% — 比平均水准高1.5倍。维生素C含量 — 也高1.5倍。两个指标的双重超标 — 绿茶中的罕见现象，由大叶种品种和南方气候丰富散射光的结合所解释。

* 陆羽早在8世纪就提到英德（作为韶州的一部分）：「其味极佳」(其味极佳) — 对广东茶叶的早期正面评价之一。

* 到2025年英德茶产业规模 — 48亿元人民币，141,000亩种植园，年产11,000吨干茶。英德是广东茶业的绝对领导者。

**13. 与其他大叶种和广东绿茶的比较：**

* **滇绿 (滇绿，云南绿茶)：** 同样采用云南大叶种品种，但来自云南。滇绿 — 更「生涩」；英德绿茶 — 更温和「精制」，得益于「闷扬结合」工艺。

* **古劳茶 (古劳茶)：** 来自广东（鹤山）。历史悠久的绿茶，具有兰花香。古劳茶 — 更小叶更精致；英德 — 更厚重浓郁。

* **日照绿茶 (日照绿茶)：** 来自山东。同样采用「引进」品种（但是小叶种），同样胺基酸含量高。日照 — 「北方」；英德 — 「南方」。化学成分：两者胺基酸都高于平均水准，但英德采用大叶种原料，提供更「厚重」的茶体。

* **西湖龙井 (西湖龙井)：** 采用小叶种品种，扁平形。龙井 — 精致优雅；英德 — 更有力浓郁，具有大叶种特有的更明显「苦味」。

**结论：**

英德绿茶 — 转型诞生的茶叶。当1990年代英德的红茶园「改色为绿」时，很少有人预期云南大叶种品种，这个大自然为发酵而创造的品种，能够产出世界级的绿茶。但富含锰和硒的喀斯特土壤、200天的雾日、70%的散射光和「闷扬结合」的工艺 — 在炙热铁锅中的闷炒和抛炒 — 使不可能成为可能。结果 — 胺基酸含量比标准高1.5倍的绿茶，具有抗癌锰元素和「轻微苦味无涩感」的特色，其中大叶种的力量被加工工艺所驯服。英德绿茶 — 适合寻求有「茶体」绿茶的人：不是碧螺春的空灵嫩滑，而是南方浓郁厚重的绿色。
