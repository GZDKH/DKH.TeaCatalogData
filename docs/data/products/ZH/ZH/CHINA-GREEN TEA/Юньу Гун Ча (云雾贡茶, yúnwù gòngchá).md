# 云雾贡茶 (云雾贡茶, yúnwù gòngchá)

**云雾贡茶 (云雾贡茶, yúnwù gòngchá)**
![Юньу Гун Ча (云雾贡茶, yúnwù gòngchá)](images/Юньу%20Гун%20Ча%20(云雾贡茶,%20Yúnwù%20Gòngchá).png)

![Юньу Гун Ча (云雾贡茶, yúnwù gòngchá)-1](images/Юньу%20Гун%20Ча%20(云雾贡茶,%20Yúnwù%20Gòngchá)-1.png)

云雾贡茶是贵州省最古老的历史名茶之一，是该省唯一具有石碑记载的宫廷贡茶地位的茶叶。采用当地鸟王种 (鸟王种) 制作，产于云雾山高山地区——苗岭主峰，此处云雾缭绕茶园超过200天。这款绿茶以其独特的鱼钩状叶形、浓郁的栗香蜜韵和持久的回甘著称。

**1. 分类与产地：**

* **类型：** 绿茶 (绿茶, lǜchá) — 不发酵茶，氧化程度极低（少于5%）。
* **类别：** 中国历史名茶；入选贵州十大名茶 (贵州十大名茶)。属于特种绿茶 (特种绿茶, tèzhǒng lǜchá) 类别。
* **产地：** 中国贵州省 (贵州省)，贵定县 (贵定县)，云雾镇 (云雾镇)。茶名源自云雾山 (云雾山) — 苗岭 (苗岭) 主峰，为乌江 (乌江)、沅江 (沅江) 和盘江 (盘江) 三大水系的分水岭。山峰最高点海拔1583.6公尺。
* **地理座标：** 约东经106°51′–107°22′，北纬26°05′–26°47′（贵定县境内）。
* **别名：** 鸟王茶 (鸟王茶, Niǎowáng Chá) — 以鸟王村命名；鱼钩茶 (鱼钩茶, Yúgōu Chá) — 因干茶形状似鱼钩；白云茶 (白云茶, Báiyún Chá) — 据传说命名；贵定雪芽 (贵定雪芽, Guìdìng Xuěyá) — 「贵定雪芽」。当地苗族（海葩苗，海葩苗）称此茶为「不老几」(不老几, bùlǎojī)。
* **保护地位：** 国家农产品地理标志 (国家农产品地理标志)（2010年认证）。生产受地方标准DB52/T 547—2008《贵定云雾贡茶》规范。

**2. 历史与文化意义：**

* **历史：**

贵定是中国西南地区最古老的产茶县之一，其茶叶历史超过两千年。早期茶业发展与云雾山苗族社区有关，他们在文字记载之前就已驯化野生高干茶树。斗篷山 (斗篷山) 山坡至今仍保存著树干直径达48公分、树龄超过千年的野生茶树。

贵定茶叶向朝廷进贡的最早可靠记录可追溯至元朝泰安二年 (泰安二年, 1325年)。明朝洪武五年 (洪武五年, 1372年) 开始，云雾山茶叶定期列入「贡方物」(贡方物) 名册。《明实录》(《明实录》) 记载了明朝276年间贵定茶叶和马匹进贡27次。

《康熙贵州通志》(《康熙贵州通志》, 1673年) 记载，在全省茶叶中，贵定云雾茶最为著名。《续遵义府志》(《续遵义府志》) 载：「云雾茶为贵州茶之冠，岁以充贡」。

关键历史文物是乾隆五十五年 (乾隆五十五年, 1790年) 立的「云雾贡茶碑」(云雾贡茶碑) 石碑。此碑共228字，记录了固定进贡数量、禁止官员欺压苗族茶农、拨银420两支持茶业的诏令。1982年该碑被列为省级文物保护单位。嘉庆十年 (嘉庆十年, 1805年) 立第二块石碑，划定贡茶生产区域界限。

清朝时期，贵定茶入选全国「八大名茶」。光绪年间（1904–1905年），贵州巡抚林绍年 (林绍年) 向皇帝和慈禧太后各进献一盒贵定茶叶——此文件保存于中国第一档案馆 (中国第一档案馆)。

现代发展：1987年开始系统化种植；1990年在全国评比中获最高分，荣获商务部「名茶之冠」称号；2002年在第四届国际名茶评比中获金奖；2010年获国家地理标志。截至2024年底，全县茶园面积达255,600亩（约17,040公顷），年产量11,800吨，品牌价值超过350亿元（2023年）。

* **名称：**

「云雾」(云雾) 意为「云彩和雾气」——直接指向山峰常年云雾缭绕的特点。「贡」(贡) 意为「进贡（皇帝）」，「茶」(茶) 即「茶叶」。因此全名意为「云雾中的贡茶」。别名「鸟王茶」源自位于茶区核心的鸟王村 (鸟王村)；「鸟王」字面意思是「鸟中之王」。

* **文化意义：**

云雾贡茶与海葩苗 (海葩苗) 文化密不可分——海葩苗是居住在云雾山周围18个山村的苗族分支。每年春季开茶时节，都会举行传统仪式，表演芦笙长鼓舞 (芦笙长鼓舞)——祈求茶叶丰收。附近的佛教圣地阳宝山 (阳宝山)，僧人自明代起就种植茶叶；「白云茶」品种即源于此地，1997年献给中国佛教协会会长赵朴初 (赵朴初)，他品尝后赞不绝口，亲笔题写「佛茶」(佛茶)。著名茶学家陈椽 (陈椽) 为此茶题诗：「贵在定钩，清明贡秀。云海雾度，质良兼优」——赞美其叶形和无与伦比的品质。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要品种为鸟王群体种 (鸟王群体种, Niǎowáng qúntǐ zhǒng)，当地小叶种群体品种 *Camellia sinensis* var. *sinensis*。2014年被贵州省品种审定委员会认定为省级优良品种 (省级优良品种)。特点是持嫩性 (持嫩性) 强，叶片厚实肉质，椭圆形，茸毛丰富。一芽一叶 (一芽一叶) 标准100芽重约45克。辅助品种使用福鼎大白茶 (福鼎大白茶)。
* **采摘：** 主要采摘期为春季（3月至4月初）；最佳期间为清明节 (清明, 通常4-5月4日) 前。夏秋茶也有生产，但价值明显较低。
* **采摘标准：** 依等级而定：珍贡 (珍贡, Zhēngòng) 级仅采芽头 (单芽)；特级 (特级) 为芽头加一片初展叶 (一芽一叶初展)；一级 (一级) 为一芽二叶 (一芽二叶)；二级 (二级) 为一芽二叶完全展开。
* **原料要求：** 纯手工采摘；使用竹制和木制工具（避免金属接触防止氧化）。原料必须新鲜，无机械损伤。春茶芽头氨基酸含量较高（春茶≥3.32%）。

**4. 风土条件与栽培特点：**

云雾山——贵定县南部苗岭主峰——代表了中国西南地区典型的高山茶风土。

* **海拔高度：** 主要茶园位于海拔1200–1500公尺；山峰最高点达1583.6公尺。
* **气候：** 亚热带高山季风气候。年平均气温约15°C；冬季温和无严寒，夏季无酷热。雾日超过200天，提供充足散射光——积累氨基酸和L-茶氨酸的理想条件。显著的日夜温差进一步促进芳香化合物合成。年降水量约1107毫米，相对湿度约80%。无霜期300–340天。
* **土壤：** 黄壤和黄红砂壤 (黄壤, 黄红砂壤)，发育于黄灰色页岩。pH值4.5–6.0——茶树最适宜范围。肥沃土层深度不少于80公分。有机质含量5.63–18.87%。土壤富含锌和硒。
* **生态环境：** 森林覆盖率70.93%（云雾镇数据）；主要茶园区域达87.6%。禁用化肥和农药；许多农场获欧洲有机认证。水源为山泉，符合一级水质标准。

* **核心区域：** 云雾镇鸟王村 (鸟王村) — 历史贡茶产地；主峰脚下梅子冲茶场 (梅子冲茶场)，面积3000亩；佛教区阳宝山；斗篷山古茶树群落。

**5. 制作工艺：**

云雾贡茶采用苗族茶农世代传承的「三炒三揉」(三炒三揉, sān chǎo sān róu) 传统工艺。现代生产结合手工和机械加工，但最珍贵的批次仍完全手工制作。该工艺已列入省级非物质文化遗产名录。

* **采摘 (采摘, cǎizhāi)：** 晨间手工采摘；标准依等级而定（见第3节）。使用竹篮运输。

* **摊青 (摊青, tān qīng)：** 鲜叶薄摊于通风室内4小时，部分失水并开始发展香气。

* **杀青 (杀青, shāqīng)：** 使用滚筒杀青 (滚筒杀青)，温度180–240°C。目的是钝化多酚氧化酶，停止氧化，固定绿色和鲜香。手工制作使用铸铁锅柴火加热。

* **揉捻 (揉捻, róuniǎn)：** 轻揉破坏细胞壁，形成初步叶形结构。压力轻柔——保持白毫 (毫, háo)。

* **二炒二揉、三炒三揉 (二炒二揉, 三炒三揉)：** 循环重复三次：每次炒制固定形状，揉捻加强卷曲。正是多次循环形成特有的钩状叶形。

* **搓团提毫 (搓团提毫, cuō tuán tí háo)：** 关键独特工序：将茶叶搓成团块后展开，提取白色茸毛到表面。此技法使成茶茸毫丰富，视觉上呈「银白色」。

* **足干 (足干, zú gān)：** 80°C慢速干燥至含水量≤7%。采用「先武火后文火」(先武火后文火) 原则，固定香气而不过度干燥。

* **筛分归类 (筛分归类, shāi fēn guī lèi)：** 成茶过筛，按大小和品质分选，分级包装。

**6. 感官特征：**

* **干茶外形：** 细紧卷曲成鱼钩形 (鱼钩形, yúgōu xíng)，白毫丰富。色泽翠绿 (翠绿)，叶质匀整；高等级茶叶有光泽，油润。

* **干茶香气：** 清新纯正，明显栗香 (栗香, lìxiāng) 和淡淡蜜韵。珍贡级有浓郁毫香 (毫香, háoxiāng)，似嫩玉米香。

* **茶汤香气：** 持久高雅，香气层次丰富。主调为栗香；中段为蜜香 (蜜香, mìxiāng)；茶杯放凉后显现淡雅花香，似兰花。冷杯香 (冷杯香) 持续时间长。

* **滋味：** 醇厚 (醇厚, chúnhòu) 鲜爽 (鲜爽, xiānshuǎng)。茶汤中等厚度，有明显「黏稠感」(粘稠感)。回甘 (回甘, huígān) 明显持久。正确冲泡无苦涩。经典评价：「一杯香，二杯浓，三杯甘又醇，四杯五杯韵犹存」(一杯香，二杯浓，三杯甘又醇，四杯五杯韵犹存)。

* **汤色：** 嫩绿明亮 (嫩绿明亮) 带黄色调，清澈透明。珍贡级更浅，如嫩玉色。

* **叶底：** 嫩匀鲜活 (嫩匀鲜活)；黄绿明亮 (黄绿明亮)。叶片完全展开，显示采摘完整性。

**7. 化学成分：**

云雾贡茶在绿茶中以多酚含量高、氨基酸平衡好著称，既有浓郁滋味又有特有甘甜。

* **多酚 (茶多酚)：** ≥33.81%（检测数据）；地理标志标准≥34.4%。相比之下：中国绿茶平均含量20–30%。高含量源于小叶种品种和高山风土结合。主要成分为儿茶素：EGCG、EGC、ECG、EC；总儿茶素含量约125.21毫克/%。

* **氨基酸 (氨基酸)：** ≥4.65%（地标标准）；一芽二叶标准为2.18毫克/克。L-茶氨酸高含量与长期散射光照射和显著日夜温差有关。

* **生物碱：** 咖啡碱 (咖啡碱) 约2.89–4.23%；可可碱和茶碱含量为绿茶典型水平。

* **水浸出物 (水浸出物)：** ≥41.69%，显示茶汤极其浓郁「厚重」。一级茶≥40%。

* **维生素：** 维生素C含量丰富（高山绿茶特点），还有B族维生素、维生素E。

* **矿物质：** 锌(Zn)和硒(Se)含量较高，源于当地土壤矿物成分。

* **精油和芳香化合物：** 栗香和蜜香特征由吡嗪、呋喃酮和芳樟醇形成，在「三炒」过程中产生。

**8. 保健功效：**

* **抗氧化作用：** 高儿茶素含量（特别是EGCG）提供强效自由基清除能力。

* **提神效果：** 咖啡碱和L-茶氨酸结合产生温和平稳的刺激，无急剧起伏——典型的「茶醒感」(清醒感)。

* **代谢支持：** 儿茶素促进脂肪氧化，可能有助于体重控制。据当地资料，脂解效率比普通绿茶高30%——可能因多酚浓度较高。

* **心血管系统：** 定期饮用高多酚绿茶与降低「坏」胆固醇(LDL)水平相关。

* **微量元素补充：** 锌参与免疫系统和组织再生；硒是天然抗氧化剂，支持甲状腺功能。

* **清热解暑 (清热解暑)：** 咖啡碱配合多酚传统上用于炎热季节解渴和缓解热不适。

* **认知功能：** L-茶氨酸改善专注力，促进平静专注状态。

* **抗菌作用：** 茶中儿茶素和单宁具有温和抑菌效果。

**9. 冲泡方法：**

* **水温：** 80–85°C。珍贡级（纯芽）接近80°C；一、二级可达85°C。绝对不建议90°C以上沸水：高温破坏茶氨酸并加重苦涩。
* **茶量：** 3克配150毫升（1:50比例）；100–120毫升盖碗用5–7克。
* **茶具：** 玻璃杯 (玻璃杯) — 观察茶叶「舞蹈」和汤色；白瓷盖碗 (盖碗) — 展现香气和多次冲泡；瓷壶 — 欧式泡法。
* **冲泡过程：**

1. 热水温杯，倒掉。
2. 珍贡级采用上投法 (上投法, shàngtóufǎ)：先注水，后投茶。其他等级用中投法 (中投法, zhōngtóufǎ)：注入1/3水，投茶，加满。
3. 沿杯壁缓慢注水，避免直接冲击茶叶——防止茸毫脱落造成浑浊。
4. 首泡30秒。后续每泡增加5–10秒。
5. 耐泡性强 (耐泡性强)，可冲7泡以上。

* **冷泡法 (冷泡法)：** 1克茶配50毫升冷水；冰箱静置30分钟。此法增强甜味，降低咖啡碱萃取。

**10. 储存：**

* **条件：** 密封避光包装；远离异味和湿气。最佳为冰箱0–5°C保存。
* **期限：** 制作后6–12个月内最佳。新茶建议开封后「醒茶」(醒茶) 7天——置于通风避光处，去除残留「火味」。真空包装开封后10天内饮完。
* **茶叶天敌：** 光线、湿气（储存含水量≤7%）、高温、异味、氧气。

**11. 价格与假冒：**

* **价格参考（中国国内市场，2023–2024年）：**
  * 珍贡 (珍贡)：≥800元/斤（500克）— 纯芽，栗香浓郁。
  * 特级 (特级)：500–800元/斤 — 一芽一叶，蜜韵甘甜。
  * 一级 (一级)：200–500元/斤 — 一芽二叶，香气纯正。
  * 二级 (二级)：200元/斤以下 — 日常饮用性价比最佳。

* **避免假冒方法：**
  * **检查叶形：** 正宗云雾贡茶有特征性钩状卷曲和丰富白毫。仿品常为直条或卷曲不明显。
  * **评估香气：** 真茶具持久栗香蜜韵，无「草腥」、霉味或酸味。
  * **检查汤色：** 透明嫩绿，无浑浊。浑浊茶汤表示原料粗糙或储存不当。
  * **注意价格：** 可疑低价（声称「特级」却低于100元/斤）是替换征象。
  * **选择商家：** 优选有国家农产品地理标志 (国家农产品地理标志) 标识和贵定县知名合作社产品。检查包装上DB52/T 547—2008标准。

**12. 趣闻轶事：**

* **「云中佛陀」：** 据传说，云雾山茶首次冲泡时，盖下升起白色蒸汽柱，先成伞状，后成云朵，慢慢消散空中。正因此视觉效果，茶得「云雾」之名和「白云茶」(白云茶) 别称。灵药禅师 (灵药禅师) 据说在蒸汽中看到佛陀身影而礼拜——由此有「佛茶」(佛茶) 之说。

* **唯一「茶碑」：** 贵定云雾贡茶是贵州省唯一（也是中国少数）以现存石碑证实贡茶地位的茶叶——1790年228字石碑至今屹立山坡。

* **茶代赋税：** 清代云雾山苗族社区免交粮食税，以茶代粮纳税——反映产品价值的独特财政关系。

* **千年古树：** 斗篷山区保存著树龄超千年、树干直径达48公分、树冠高逾40公尺的野生茶树——贵定是茶树原产地中心之一的活证据。

* **欧洲纯净标准：** 云雾贡茶通过400多种农药残留的欧洲标准检测——中国绿茶中最严格的控制之一。

**13. 与其他绿茶比较：**

* **都匀毛尖 (都匀毛尖, Dūyún Máojiān)：** 「近亲」——同样来自贵州省（都匀县），入选「中国十大名茶」。两茶都使用当地小叶种，茸毫丰富，滋味鲜爽。但都匀毛尖形状更细直如针，云雾贡茶为独特钩状卷曲。毛尖香气更精致花香；贡茶更浓郁，栗香蜜韵突出。

* **庐山云雾 (庐山云雾, Lúshān Yúnwù)：** 江西省经典「云雾」茶。相似生态条件（高山、常雾），但庐山云雾采用炒青工艺（锅炒）无多次「三炒」循环；形状扁曲，滋味更温和草香，无明显栗香调。

* **信阳毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南省名茶，同样茸毫丰富。形状直针状；滋味纯正鲜爽，栗豆香调。信阳风土（800–1000公尺）比云雾（1200–1500公尺）低，氨基酸积累较少。

* **蒙顶甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 四川古代「贡茶」。与云雾贡茶同样具有多世纪「贡茶」历史。形状细嫩卷曲；滋味更精致「甘甜」（「甘露」意为「甘甜露水」）。多酚含量低于云雾贡茶。

* **绿宝石 (绿宝石, Lǜ Bǎoshí)：** 贵州现代名茶，面向大众市场。叶片明显较大（一芽二三叶），栗香明显，但比云雾贡茶精致度和复杂性较低。

**结语**

云雾贡茶是中国绿茶中文献记载「履历」最长的茶叶之一：从元代首次进贡到现代地理标志。其钩状叶形、栗香蜜韵和厚重多层次甘甜，使其成为贵州茶学派的代表。此茶特别推荐给寻求浓郁「厚实」绿茶并具历史底蕴的爱好者——比更知名但有时较内敛的华东经典茶更具选择性。山间云雾的香气保存在卷曲叶中，每泡都有展现，印证古老公式：「一杯香，五杯韵犹存」。
