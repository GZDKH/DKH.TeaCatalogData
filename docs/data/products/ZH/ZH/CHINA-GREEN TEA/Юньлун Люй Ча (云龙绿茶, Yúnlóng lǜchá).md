# 云龙绿茶 (云龙绿茶, Yúnlóng lǜchá)

**云龙绿茶 (云龙绿茶, Yúnlóng lǜchá)**
![Юньлун Люй Ча (云龙绿茶, Yúnlóng lǜchá)](images/Юньлун%20Люй%20Ча%20(云龙绿茶,%20Yúnlóng%20Lǜchá).png)

![Юньлун Люй Ча (云龙绿茶, Yúnlóng lǜchá)-1](images/Юньлун%20Люй%20Ча%20(云龙绿茶,%20Yúnlóng%20Lǜchá)-1.png)

云龙绿茶是来自云南省大理白族自治州云龙县的高山绿茶。采用云南大叶种茶树（*Camellia sinensis* var. *assamica*）的叶片，在海拔1800–2600公尺的高度种植——这是中国最高海拔的茶园之一。该产品自2016年起获得中华人民共和国地理标志保护。典型品质特征：紧结壮实的茶叶带有银白色茸毫，淡绿色透明的茶汤，持久的炒栗子香气，以及浓郁鲜爽的滋味和悠长的回甘。

**1. 分类与产地：:**

* **类型：** 绿茶 (绿茶, lǜchá) — 不发酵茶；氧化程度极低（少于5%）。工艺 — 锅炒 (炒青, chǎoqīng)。
* **类别：** 中国地方绿茶；云南大叶种绿茶 (云南大叶种绿茶, Yúnnán dàyèzhǒng lǜchá)。中华人民共和国地理标志保护产品。
* **产地：** 中国云南省 (云南省, Yúnnán shěng)，大理白族自治州 (大理白族自治州, Dàlǐ Báizú Zìzhìzhōu)，云龙县 (云龙县, Yúnlóng xiàn)。生产范围涵盖11个乡镇：诺邓镇 (诺邓镇, Nuòdèng zhèn)、功果桥镇 (功果桥镇, Gōngguǒqiáo zhèn)、漕涧镇 (漕涧镇, Cáojiàn zhèn)、白石镇 (白石镇, Báishí zhèn)、宝丰乡 (宝丰乡, Bǎofēng xiāng)、苗尾乡 (苗尾乡, Miáowěi xiāng)、检槽乡 (检槽乡, Jiǎncáo xiāng)、长新乡 (长新乡, Chángxīn xiāng)、关坪乡 (关坪乡, Guānpíng xiāng)、团结乡 (团结乡, Tuánjié xiāng) 和民建乡 (民建乡, Mínjiàn xiāng)。
* **地理座标：** 东经98°52′–99°46′，北纬25°28′–26°23′。

**2. 历史与文化意义：:**

* **历史：**

云龙县的茶叶传统起源相对较晚——始于20世纪初。1919年，县长段韬 (段韬, Duàn Tāo) 从顺宁县 (顺宁, Shùnníng；现为凤庆 (凤庆, Fèngqìng)) 购买茶籽，带回旧州 (旧州, Jiùzhōu) 和汤涧 (汤涧, Tāngjiàn) 地区进行试种。实验获得成功：数百棵茶树在山区气候中成活。

产业化发展始于1987年，在宝丰、旧州等地区的茶园基础上组织大规模种植；传统手工制作方法得到统一和标准化，形成「云龙绿茶」统一品牌。1996年，「大栗树牌云龙绿茶」(大栗树牌云龙绿茶, Dàlìshù pái Yúnlóng lǜchá) 在中国科技精品博览会 (中国科技精品博览会) 上获得金奖。

2016年，该茶获得国家农产品地理标志 (国家农产品地理标志) 注册。到2024年，全县茶园面积达到4000公顷，年产量超过3000吨。形成了多个知名品牌：「大栗树」(大栗树, Dàlìshù)、「佬倵」(佬倵, Lǎomù)、「云极」(云极)、「大山头」(大山头)。茶叶成为全县11个乡镇经济发展的重要支柱。

* **名称：**

云龙 (Yúnlóng) — 「云龙」；地名，反映了当地山区「云雾缭绕」的特色；绿茶 (lǜchá) — 「绿茶」。诗意的名称恰当地传达了来自「晴时早晚遍地雾，阴雨连天满山云」之地的茶叶形象。

* **文化意义：**

云龙是大理白族自治州下辖的县，主要居住著白族 (白族, Báizú) 人民。茶叶在这里不仅是商品作物，更是山区社区生活方式的重要组成部分，许多社区居住在海拔2000公尺以上，历史上以畜牧业和核桃种植为生。茶叶种植的引入为这些社区提供了稳定的收入来源，并与更广泛的全国市场建立了联系。生产核心——宝丰乡大栗树村 (大栗树村, Dàlìshù cūn) 被认定为全国一村一品示范村 (全国一村一品示范村)。近年来发展茶叶农业旅游：正在建设大栗树生态茶旅庄园 (大栗树生态茶旅庄园)。

**3. 植物学描述与原料：:**

* **品种/栽培种：** 主要由云南大叶种群体 (云南大叶种群体, Yúnnán dàyèzhǒng qúntǐ) 构成——有性系 (有性系, yǒuxìng xì) 来源。主要无性系品种：云抗系列 (云抗系列, Yúnkàng xìliè) 和佛香系列 (佛香系列, Fóxiāng xìliè) — 无性繁殖的优良栽培种。还有当地古茶树 (古茶树, gǔcháshù)，特别是在大山头茶园核心区。所有栽培种均属于 *Camellia sinensis* var. *assamica* — 云南典型的大叶种变种。
* **采摘：** 春茶 (春茶, chūnchá) — 主要且最珍贵；清明 (清明, Qīngmíng，4月初) 后立即开始。夏茶和秋茶也有生产。
* **采摘标准：** 春茶：一芽二叶至一芽三叶 (一芽二叶至一芽三叶, yī yá èr yè zhì yī yá sān yè)。要求：整齐均匀的芽叶，芽叶比例协调。不允许鱼叶 (鱼叶, yúyè)、雨水叶、对夹叶 (对夹叶, duìjiā yè)、病叶、冻伤叶或「紫芽」。
* **原料要求：** 鲜叶装入竹篓 (竹篓, zhúlǒu) — 禁止使用塑胶袋以避免闷热。茶多酚含量 ≥ 25.0%，氨基酸含量 ≥ 4.0%。

**4. 风土条件与栽培特色：:**

* **地形与气候：** 云龙县位于澜沧江纵谷区 (澜沧江纵谷区, Láncāngjiāng zònggǔ qū) — 湄公河上游 — 云南西部。深谷、陡坡和七个水系（包括澜沧江和沘江 (沘江)）形成独特的微气候。年平均气温16.2°C，相对湿度 ≥ 75%，雾日超过200天。昼夜温差大有利于氨基酸和芳香化合物的积累。
* **种植海拔：** 1800–2600公尺 — 中国最高海拔的茶区之一。宝丰大山头茶园核心区 — 2400–2500公尺。
* **土壤：** 黄棕壤 (黄棕壤, huáng zōng rǎng)，酸性反应（pH 4.5–6.0），土层深厚肥沃，排水良好，有机质含量高（≥ 6.01%）。土壤富含微量元素——硒（Se）和锌（Zn）。核心生产区森林覆盖率超过70%。核心区为水源保护区；禁止使用化学肥料和农药。
* **特殊因子：** 从澜沧江峡谷上升的温暖湿润气流补偿了高海拔的温度降低，丰富的散射光 (漫射光, mànshè guāng) 刺激氨基酸的生物合成。因此，即使在海拔2500公尺，春茶中的氨基酸含量也能达到 ≥ 4.0%。

**5. 制作工艺：:**

云龙绿茶是炒青绿茶 (炒青绿茶, chǎoqīng lǜchá)，制作过程中采用烘炒结合 (烘炒结合, hōng chǎo jiéhé) 的方法来固定香气和外形。整个制程使用竹制和木制工具 — 避免茶叶与金属接触。制作500克特级茶需要约40,000个芽头。

* **采摘 (采摘, cǎizhāi)：** 手工采摘，标准为一芽带2–3叶。使用竹篓。
* **摊晾 (摊晾, tānliáng)：** 鲜叶摊放在竹筛上4–6小时，蒸发多余水分并进行初步香气转化。
* **杀青 (杀青, shāqīng)：** 高温抛炒 (高温抛炒, gāowēn pāochǎo) 法。钝化酶活性，保持绿色并「封存」栗香。
* **揉捻 (揉捻, róuniǎn)：** 机械或手工；形成该品种特有的紧结条索。破坏细胞结构以利均匀浸出。
* **干燥 (干燥, gānzào)：** 「炒制+烘干」组合方法，温度 ≤ 60°C。固定外形，确立香气特征 — 主要是栗香。
* **筛分 (筛分, shāifēn)：** 按大小和形状分级，去除缺陷叶片和茶梗。

**6. 感官特征：:**

* **干茶外形：** 两种主要类型。螺形 (螺形, luóxíng)：紧结厚重，螺旋状，有光泽，绿色调 — 特级茶特征。条形 (条形, tiáoxíng)：壮实整齐，灰绿色带明显银毫 (灰绿上霜, huīlǜ shàngshuāng) — 一级和二级茶特征。
* **干茶香气：** 明显纯净，以栗香为主。高级茶带有花果香韵。
* **茶汤香气：** 持久的炒栗子香 (熟板栗香, shú bǎnlì xiāng) — 该品种的主要标志。高雅持久，高级茶在前调中带有淡雅的花果香（特别是大山头核心区的茶）。
* **滋味：** 浓醇 (浓醇, nóngchún)，同时鲜爽 (鲜爽, xiānshuǎng)，回甘 (回甘, huígān) 明显持久。苦涩平衡，快速转化为甘甜。浓厚的「体感」源于高浸出物含量（≥ 44.0%）— 云南大叶种茶的特色。
* **汤色：** 淡绿清澈明亮 (汤色淡绿清澈明亮, tāngsè dànlǜ qīngchè míngliàng)。
* **叶底（茶渣）：** 叶底完整匀整、嫩绿鲜活 (叶底完整匀整、嫩绿鲜活, yèdǐ wánzhěng yúnzhěng, nènlǜ xiānhuó)。

**7. 化学成分：:**

* **多酚类 (茶多酚, chá duōfēn)：** ≥ 25.0% — 显著高于大多数小叶种绿茶。这是由于属于大叶种变种 *var. assamica*，在遗传上倾向于积累多酚类。提供强大的抗氧化作用。
* **氨基酸 (氨基酸, ānjīsuān)：** ≥ 4.0%（总量），L-茶氨酸 (茶氨酸, cháānsuān) ≥ 1.9%。高氨基酸含量同时伴随高多酚 — 这种罕见的组合形成了特有的「浓厚+鲜爽」平衡。这是由极高海拔和丰富散射光造成的。
* **生物碱：** 咖啡碱 (咖啡碱, kāfēijiǎn) ≥ 2.0%。可可碱和茶碱含量微量。
* **水浸出物 (水浸出物, shuǐjìnchūwù)：** ≥ 44.0% — 极高的指标，解释了茶汤的浓厚体感和优秀的耐泡性。
* **维生素：** 由于 ≤ 60°C 的温和干燥，维生素C得到大量保存。还含有维生素B₁、B₂、E、β-胡萝卜素。
* **矿物质：** 钾、锰、硒（Se）、锌（Zn）— 源于山地土壤的地球化学特性。氟 — 有益牙齿健康。

**8. 健康功效：:**

* **强大抗氧化作用：** 多酚含量 ≥ 25% 提供绿茶中最高效的自由基清除能力之一。
* **提神醒脑效果：** 咖啡碱（≥ 2.0%）提高注意力集中和思维清晰度；与L-茶氨酸结合，效果温和持久，无「咖啡因崩溃」。
* **消化支持：** 多酚刺激胃肠蠕动，减少腹胀，促进油腻食物消化。
* **代谢支持：** 儿茶素加速脂肪分解，可能有助于正常化血脂谱。
* **认知功能：** L-茶氨酸（≥ 1.9%）促进放松而不困倦，改善记忆和注意力。
* **微量元素补充：** 天然富含硒和锌，支持免疫系统和抗氧化防护。
* **口腔健康：** 氟抑制致龋细菌生长。
* **心血管支持：** 儿茶素和茶黄素促进血管壁弹性，定期饮用可能有助于正常化血压。

**9. 冲泡方法：:**

* **水温：** 80–85°C。不要使用超过85°C的沸水 — 会破坏L-茶氨酸，失去鲜爽感，产生苦味。
* **茶量：** 玻璃杯法3克配150毫升（1:50）；盖碗功夫茶法5–7克配100–120毫升盖碗。
* **茶具：** 玻璃杯（便于观察叶片舒展），白瓷盖碗 (盖碗, gàiwǎn)，小容量紫砂壶。

* **冲泡过程（玻璃杯法 — 上投法 (上投法)）：**
  1. 用热水温杯，倒掉。
  2. 注入80–85°C热水至7/10满。
  3. 投入茶叶 — 茶叶缓慢下沉并舒展。
  4. 浸泡2–3分钟。
  5. 饮用时留1/3茶汤，续水；可续2–3次。

* **冲泡过程（盖碗功夫茶法）：**
  1. 用沸水温盖碗和公道杯。
  2. 投入5–7克茶叶，闻干茶香。
  3. 润茶 (润茶, rùnchá)：5秒，倒掉。
  4. 第一泡：20秒，倒出。
  5. 每次增加10秒。可冲泡3–5次优质茶汤。

**10. 储存方法：:**

* **条件：** 密封容器（真空袋、铝箔袋、铁罐）。避光、防潮、避热、防异味。
* **温度：** 最佳为冰箱冷藏，0–5°C。
* **保质期：** 最佳品质期为制作后6–12个月内。新茶建议在阴凉处放置15天以褪火气 (褪火气, tuì huǒqì)。开封后7天内饮用完毕。

**11. 价格与防伪：:**

* **价格区间：**
  * 特级 (特级, tèjí)：600元/斤（500克）起 — 单芽（≥ 95%），螺形，浓郁栗香。
  * 一级 (一级, yījí)：300–500元 — 一芽一叶，紧结条索，鲜爽持久。
  * 二级 (二级, èrjí)：100–300元 — 一芽2–3叶，香气纯正，性价比高。
  价格因素：季节（春茶较贵）、产区（大山头为顶级）、树龄（古茶树有溢价）、有机认证。

* **防伪方法：**
  * 向「大栗树」品牌或其他注册生产商的授权经销商购买，确认地理标志标识。
  * 检查外观：优质云龙绿茶为紧结壮实的茶叶带银毫；细碎、断裂或发黄的茶叶是次品标志。
  * 评估香气：品牌标志是持久的栗香，无酸味、霉味或「鱼腥味」。
  * 茶汤应透明淡绿 — 浑浊或深色表示有缺陷。
  * 声称「特级」但价格低于400元/斤值得怀疑。

**12. 有趣事实：:**

* 云龙的茶叶历史始于一位县官的冒险实验：1919年段韬从顺宁（现凤庆，著名云南红茶滇红的故乡）带来茶籽。数百棵茶树成活 — 一个世纪后，该县年产茶叶3000吨。
* 宝丰乡大山头茶园 (大山头) 海拔2400–2500公尺，是中国最高海拔的商业茶园之一。相比之下：著名的台湾高山乌龙茶种植在1600–2600公尺，而大多数云南绿茶种植在1000–1800公尺。
* 云龙绿茶的水浸出物达到 ≥ 44% — 这个指标更像是好普洱茶而非绿茶的特征，解释了多次冲泡时茶汤的卓越「耐力」。
* 该县名为「云龙」并非仅是比喻：当地俗语「晴时早晚遍地雾，阴雨连天满山云」描述了海拔2000+公尺生活的现实，茶树真的生长在云雾中。
* 生产由云南农业大学茶学系 (云南农业大学茶学系) 指导 — 学术监督确保品质稳定和新农业技术标准的实施。
* 2024年，为高品质卷曲绿茶「佛香3号」系列 (佛香3号) 制定并批准了单独的地方生产标准（大理佛香3号卷曲形绿茶加工技术规程），大理农科院、云南省农科院茶叶研究所和宝丰「大栗树」茶厂参与制定 — 突显了该产业科学支持的严肃性。

**13. 与其他绿茶的比较：:**

* **永泰绿茶 (永泰绿茶, Yǒngtài Lǜchá)：** 福建绿茶；小叶种（*var. sinensis*），海拔500+公尺。体感较轻，栗花香温和。云龙绿茶因大叶种原料和极高海拔而更浓厚有力。
* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北蒸青 (蒸青) 绿茶；明显的海草香调，高鲜味。云龙绿茶为炒青，栗香主导，香气特征更「干燥」。
* **滇绿/云南绿茶 (滇绿, Diānlǜ)：** 云南绿茶的总称。云龙绿茶以极高海拔茶园和澜沧江峡谷独特微气候脱颖而出，形成平原云南绿茶无法达到的多酚与氨基酸平衡。
* **庐山云雾 (庐山云雾, Lúshān Yúnwù)：** 江西「云雾」绿茶；海拔800–1200公尺，小叶种，清淡花香。云龙绿茶高1000公尺，大叶种，更浓厚涩味，回甘明显。
* **台湾高山绿茶 (台湾高山绿茶)：** 相似海拔范围，但不同栽培种（青心、TTES №12/13），不同工艺。云龙绿茶因云南大叶种茶树的天性更有力更「干燥」。

## 结语

云龙绿茶是中国最高海拔的绿茶之一，有力证明了云南大叶种茶树不仅能产出伟大的普洱茶和滇红，也能制作出卓越品质的绿茶。2000–2500公尺的极高海拔、丰富的云雾、古老茶树和纯净生态 — 这一切造就了具有「绿茶」中不寻常组合的茶：浓厚饱满的体感，同时保持鲜爽、甘甜和悠长回甘。栗香 — 深沉持久 — 是该品种的标志。云龙绿茶适合寻求有个性绿茶的人：不是精致「轻盈」的，而是有力、饱满、令人难忘的。
