# 香草兰绿茶 (香草兰绿茶, xiāngcǎo lán lǜchá)

**香草兰绿茶 (香草兰绿茶, xiāngcǎo lán lǜchá)**

![Сянцао Лань Люй Ча (香草兰绿茶, xiāngcǎo lán lǜchá)-1](images/Сянцао%20Лань%20Люй%20Ча%20(香草兰绿茶,%20Xiāngcǎo%20Lán%20Lǜchá)-1.png)

香草兰绿茶 (香草兰绿茶, xiāngcǎo lán lǜchá) — 世界上唯一以天然香草兰 (*Vanilla planifolia*) 薰香的绿茶，被誉为「世界天然食品香料之王」(世界天然食品香料之王)。此产品诞生于海南岛 (海南) — 中国唯一能够种植香草兰的热带地区，作为中国热带农业科学院香料饮料研究所 (中国热带农业科学院香料饮料研究所) 的科学项目。创制于1993年，采用革命性解决方案：取代传统花卉「窨制」(窨制，以茉莉花或玫瑰花瓣薰香) 技术，使用发酵香草兰豆荚 — 此产品需要120天发酵过程，含有超过250种芳香化合物。成果是一款具有巧克力香草香气、「兰花」甜味和抗焦虑效果的茶品，在中国乃至世界都无类似产品。2024年，其制作技艺被列入万宁市非物质文化遗产名录。

**1. 分类与产地：**

* **类型：** 添香型绿茶 (添香型绿茶, tiān xiāng xíng lǜchá)。基茶 — 海南大叶种绿茶。薰香剂 — 天然发酵香草兰豆荚 (*Vanilla planifolia*)。薰香方法 — 「免花窨制」(免花窨制, miǎn huā yìnzhì) — 25–45 °C低温吸附，有别于传统茉莉花瓣「窨制」。

* **类别：** 科学研发新型茶类 (1993)。万宁市级非物质文化遗产 (万宁市级非遗, 2024)。世界茶叶竞赛获奖者 (2022年世界红茶大赛金奖 — 红茶版本「雨林春芽」)。

* **产地：** 中国；海南省 (海南, Hǎinán)；万宁市 (万宁市, Wànníng Shì)。生产核心 — 兴隆热带植物园 (兴隆热带植物园, Xīnglóng Rèdài Zhíwùyuán)，位于香料饮料研究所内 (18°44′ 北纬，110°11′ 东经)。兴隆 — 东南亚归侨历史聚居地，归侨带来热带作物，包括香草兰。

* **地理座标：** 18°44′ 北纬，110°11′ 东经 (兴隆，万宁)。


**2. 历史与文化意义：**

* **历史：**

香草兰绿茶是中国最「年轻」的名茶之一：仅有30多年历史。1993年，海南香圣天然食品有限公司 (海南香圣天然食品有限公司) 与西南农业大学 (西南农业大学) 合作，首次在中国将香草兰用于茶叶薰香。突破在于取代传统花卉「窨制」(窨制) — 茶叶吸收新鲜茉莉花或玫瑰花瓣香气的技术 — 开发出「免花窨制」(免花窨制) 方法：茶叶以发酵香草兰豆荚薰香，而非花朵。这是根本性创新：香草兰豆荚不是花朵，而是果实，其香气 (香草醛和250+伴随化合物) 不是在开花时形成，而是通过120天发酵过程产生。

同年1993年，产品通过海南省科学技术委员会科技鉴定，填补国内薰香茶类空白 (填补国内空白)。2000年代起，兴隆热带植物园扩大生产规模，创建产品线：绿茶、红茶、乌龙茶香草兰系列。2022年，红茶版本「雨林春芽」(雨林春芽) 荣获世界红茶大赛金奖。2024年，「兴隆香草兰茶制作技艺」(兴隆香草兰茶制作技艺) 被列入万宁市非物质文化遗产名录。

* **名称：** 香草兰 (Xiāngcǎo Lán) — 「香草兰」— *Vanilla planifolia* 的中文名称，兰科热带藤本植物，其发酵豆荚是天然香草醛的来源；绿茶 (Lǜchá) — 「绿茶」。完整含义：「香草兰绿茶」。

* **文化意义：** 香草兰绿茶是热带农业与茶业独特交汇的产物，仅在海南 — 中国唯一的热带岛屿 — 才可能实现。兴隆由1950年代从东南亚归国的华侨建立，成为热带作物的「熔炉」：咖啡、可可、胡椒、香草兰 — 正是这种文化环境孕育了香草兰与茶结合的创意。兴隆植物园是中国最受欢迎的热带植物园之一 (42公顷，超过3000种植物)。香料饮料研究所是中国唯一专门研究热带香料和饮料作物的科研机构；香草兰研究自1983年开始。海南被纳入「中国自由贸易港」(中国自由贸易港) 计划，为香草兰茶出口国际市场开辟前景。


**3. 植物学描述与原料：**

* **基茶 — 海南大叶种 (海南大叶种)：** *Camellia sinensis* var. *assamica*，海南本土品种 — 中国最南端的茶树栽培品种之一 (北纬18°)。叶片大而肉厚，多酚类 ≥28 %，提供「厚重」基底，能够「承载」浓郁香草兰香气而不失茶叶本色。在海拔400公尺以下的热带雨林条件下种植。部分批次另用奇兰 (奇兰) 品种，带来花果香调。

* **薰香剂 — 香草兰 (*Vanilla planifolia*)：** 兰科热带藤本植物，原产墨西哥，1960年代起在海南栽培 (香料饮料研究所)。豆荚长15–25公分，绿色采收后经120天发酵 (杀青 → 干燥 → 40–50 °C发酵)，过程中β-葡萄糖苷酶将葡萄糖香草醛分解为香草醛和葡萄糖，形成特有巧克力香草香气及250+伴随芳香化合物。发酵豆荚切段用于薰香。

* **附加变化：** 存在「香兰糯米香」(香兰糯米香) 版本，添加西双版纳野生「糯米香叶」(糯米香叶, nuòmǐ xiāng yè) — 创造「香草兰 + 糯米」双重香气。

* **茶叶采收：** 全年采收 (热带气候允许年采10次 — 远超亚热带地区)。特级 — 单芽或一芽一叶，手工采摘。大宗 — 一芽二三叶，可机械采收。海南热带茶与大陆同类相比，多酚类含量较高，胺基酸含量较低，成为薰香的理想「厚重」基底。


**4. 风土与栽培特色：**

万宁位于海南岛东南沿海，处于热带地区 (北纬18°) — 这是中国唯一能够露地种植香草兰的地区。

* **气候：** 热带季风气候。年均温 >25 °C；降雨量2200毫米/年；云雾天数 >200天/年；森林覆盖率86 %。香草兰生长需要温度 ≥25 °C 和遮荫度 ≥50 %。

* **土壤：** 微酸性红壤 (pH 5.0–6.5)，富含硒元素。有机质 ≥15 g/kg。

* **特色：** 香草兰是世界上最劳动密集的农作物之一：每朵花需在开放后4–6小时内人工授粉 (自然界中香草兰由墨西哥特有的无刺蜂授粉，海南没有)；豆荚成熟需9个月；发酵需120天。这使天然香草兰成为最昂贵香料之一 (250–560美元/公斤干豆荚)。海南研究所开发「单元式热空气发酵生香法」(单元式热空气发酵生香法)，达到ISO品质标准，并首创国家香草兰种苗和产品标准 (《香荚兰种苗》, 《香荚兰》)。海南香草兰种植园产量超过世界平均水准，得益于独特的人工遮荫棚架技术。


**5. 制作工艺：**

香草兰绿茶生产分两阶段：先制作基础绿茶，再进行香草兰薰香。

* **第一阶段 — 基础绿茶制作：** 标准海南工艺：采摘 → 萎凋 → 杀青 (炒制) → 揉捻 → 干燥。基茶须「厚重」(多酚类 ≥28 %) — 以承受薰香而不失本色。

* **第二阶段 — 香草兰豆荚发酵 (120天)：** 绿色豆荚采收，热水处理 (杀青，「烫漂」)，然后在40–50 °C发酵约120天。β-葡萄糖苷酶分解葡萄糖香草醛 → 香草醛 + 葡萄糖。发酵豆荚切段。

* **第三阶段 — 「免花窨制」薰香 (免花窨制)：**

	* 基础绿茶与切段发酵香草兰豆荚混合 (特级品 ≥5 %)。
	* 混合物密封于铁罐 (铁罐密封)。
	* 25–45 °C静置5–30天 — 「低温吸附」(低温吸附, dī wēn xīfù)。茶叶吸附香草兰豆荚挥发性芳香化合物。温度至关重要：>45 °C香草醛降解，<25 °C吸附过慢。
	* 取出豆荚，茶叶补充干燥。

* **与茉莉花「窨制」的关键差异：** 茉莉花茶用新鲜花朵薰香，花朵凋谢时「释放」香气；香草兰茶用发酵豆荚 (果实) 薰香，其香气由120天酵素过程形成。这不是「花香」而是「果香」薰制 — 机制根本不同。


**6. 感官特征：**

* **干茶外形：** 紧结卷曲条索 (条索紧结卷曲)，色泽绿润显霜 (色泽绿润显霜)。有时可见香草兰豆荚细片 (褐黑色斑点)。

* **干茶香气：** 主调为巧克力香草香 (巧克力香, qiǎokèlì xiāng) — *Vanilla planifolia* 标志性香气。背景为「兰花香」(兰花香) 和草本清香 (草本清香)。香气持久：冷杯保持 >5分钟。

* **茶汤香气：** 浓郁「糖果香」：香草兰 + 巧克力 + 淡淡绿茶清香。任何其他绿茶都无此香型 — 最接近 (但相距甚远) 的类比是茉莉花茶，但香型完全不同。冷却后显现「奶香」和「焦糖」调 — 香草醛与茶胺基酸在50 °C以下相互作用的结果。

* **滋味：** 醇厚 (醇厚, chúnhòu) — 高多酚基底 (≥28 %) 的结果。回甘 (回甘) 带香草兰甜味。顺滑油润 (润滑, rùnhuá)。涩味极轻 — 香草醛掩盖单宁苦味。

* **汤色：** 黄绿明亮 (黄绿明亮)。

* **叶底 (冲泡后茶叶)：** 嫩匀黄绿 (嫩匀黄绿)；有时可见香草兰豆荚片段。


**7. 化学成分：**

* **多酚类 (茶多酚)：** ≥28 % (基茶) — 高含量，提供「厚重」基底。

* **香草醛 (香兰素, xiānglán sù)：** 香草兰豆荚主要芳香化合物 (3-甲氧基-4-羟基苯甲醛)。与250+伴随芳香分子共同创造「天然香草兰香韵」，合成香草醛无法复制。

* **胺基酸：** 基茶中存在，提供鲜味。

* **咖啡因：** 绿茶标准含量。

* **维生素：** 维生素C，B群维生素。

* **矿物质：** 硒 (来自海南土壤)，钾，锰。


**8. 保健功效：**

* **抗焦虑效果：** 香草醛刺激γ-胺基丁酸 (GABA) 活性 — 主要抑制性神经传导物质。据资料显示，缓解焦虑效果比普通绿茶高40 %。

* **助消化：** 香草兰芳香化合物增强胃蛋白酶活性。

* **抗氧化保护：** 多酚类 (≥28 %) 提供绿茶标准抗氧化活性。

* **提神效果：** 咖啡因 + L-茶胺酸 — 温和提神。

* **助眠：** 香草兰精油对中枢神经系统有温和镇静效果 — 下午饮用时 (无咖啡因形式 — 冷泡茶)。

* **重要提醒：** 上述功效基于一般资料，非医疗建议。不建议空腹饮用。日用量不超过600毫升 (避免咖啡因过量)。


**9. 冲泡方法：**

* **水温：** 80 °C (特级品 — 75 °C)。勿用沸水 — 破坏香草醛并产生苦味。

* **茶量：** 3公克配150毫升 (比例1:50)。

* **茶具：** 玻璃杯 (观察汤色) 或白瓷盖碗 (聚集香气)。

* **冲泡：** 80 °C水冲泡，静置3分钟。可冲泡3次。

* **冷泡法 (冷泡法, lěng pào fǎ)：** 4公克茶 + 500毫升冷水 → 冰箱，4小时。冷泡香草兰茶是中国最佳冷泡茶之一：香草兰巧克力香气更加鲜明，咖啡因萃取最少。


**10. 储存方法：**

* **温度：** 室温。**勿冷藏！** (绿茶中的例外) — 冰箱凝结水破坏香草兰香气，食品异味瞬间被茶叶吸附。这是与大多数需要冷藏的绿茶的根本差异。
* **容器：** 密封，避光。
* **光照：** 完全隔离 — 香草醛对光敏感。
* **保质期：** 18个月内。


**11. 价格与仿品：**

香草兰绿茶是平价茶品 (相比昂贵茉莉花茶的优势之一)。特级 — 80–100元人民币/100公克 (礼盒铁罐装)；一级 — 30–50元人民币/100公克；大众级 (袋装) — 20–30元人民币/66公克。

* **避免仿品方法：**

	* 香气 — 天然香草兰巧克力香，持久 (冷杯 >5分钟)。合成香草醛 — 刺鼻，「单调」，快速消失。
	* 产地 — 兴隆，万宁，海南。品牌：「兴隆」(兴隆)，「香圣」(香圣)。
	* 茶中香草兰豆荚片段 — 天然薰香标志。


**12. 趣味知识：**

* **世界唯一香草兰茶：** 无论中国、日本、印度还是斯里兰卡，都没有香草兰绿茶的类似产品。这直接源于海南独特地理位置：中国唯一能露地种植香草兰的热带岛屿。

* **「免花窨制」：** 传统中国「窨制」(窨制) 是花卉薰香：茉莉、玫瑰、桂花。香草兰茶是首个也是唯一的「果实」窨制：用发酵豆荚而非花瓣薰香。这需要完全重新设计技术：取代6–8小时花香「爆发」— 5–30天缓慢低温吸附。

* **120天发酵：** 香草兰豆荚是世界上最「劳动密集」的芳香产品之一。每朵花人工授粉 (清晨，开放后4–6小时内)；豆荚成熟9个月；发酵120天。一批香草兰茶使用的豆荚，其「生命史」始于一年多前。

* **兴隆 — 归侨村：** 兴隆建于1950年代，安置从东南亚 (马来西亚、印尼、越南) 归国的华侨。他们带来热带农业文化：咖啡、可可、胡椒、香草兰。正是这个文化「熔炉」孕育了香草兰与茶结合的创意。

* **冷泡理想选择：** 香草兰巧克力香气在冷泡时比热泡更加鲜明，同时咖啡因萃取最少 (约为热泡的30 %)。这使冷泡香草兰茶成为热带海南的理想夏日饮品 — 更广泛地说，适合冷茶文化远比大陆中国发达的整个东南亚地区。

* **槟榔林中的香草兰：** 海南香草兰种植的前景方法之一 — 与槟榔 (槟榔, bīnglang) 间作 (间作, jiānzuò)：香草兰是需要支撑和遮荫的藤本；槟榔是提供两者的高大乔木。因此，两种海南「特产」— 槟榔果和香草兰 — 在同一种植园生长，节约土地和资源。


**13. 与其他薰香绿茶比较：**

* **茉莉花茶 (茉莉花茶, Mòlì Huāchá)：** 经典茉莉绿茶。薰香 — 新鲜茉莉花瓣 (花卉「窨制」，6–8小时 × 3–7轮)。香气 — 花香，「清白」；滋味 — 清鲜，「轻盈」。香草兰茶 — 「深沉」，巧克力香草香；滋味 — 更「厚重」和「糖果感」。技术根本不同：花卉 vs 果实薰香。

* **桂花绿茶 (桂花绿茶, Guìhuā Lǜchá)：** 桂花绿茶。香气 — 「杏蜜香」。比茉莉花茶更「温暖」，但无香草兰茶的巧克力深度。

* **伯爵茶 (Earl Grey)：** 西方对应品 — 佛手柑油薰香红茶。两者都是「异国香型薰香茶」，但机制完全不同：伯爵茶用精油 (萃取物)；香草兰茶用整个发酵豆荚 (天然吸附)。结果 — 香草兰茶香气更复杂「深层」，因为豆荚含250+芳香化合物 vs 佛手柑单一主导芳樟醇。此外，香草兰茶是绿茶，伯爵茶是红茶，提供根本不同的「背景」风味：清香植物感 vs 麦芽单宁感。


**结语：**

香草兰绿茶是茶中异数：它诞生于热带植物园而非高山；用果实而非花朵薰香；冷饮胜过热饮；虽是绿茶却不宜冷藏。这一切都源于两位「王者」的相遇：海南大叶种茶 (多酚类28 %) 和墨西哥香草兰 (250+芳香分子，120天发酵)，在中国唯一热带岛屿上的东南亚归侨村落中实现。80 °C冲泡 — 或者更佳，冷水浸泡冰箱4小时：巧克力香草兰香气的绽放，是世界上任何其他绿茶都无法企及的。
