# 白毛猴 (白毛猴, bái máo hóu)

**白毛猴 (白毛猴, bái máo hóu)**

![Бай Мао Хоу (白毛猴, bái máo hóu)](images/Бай%20Мао%20Хоу%20(白毛猴,%20Bái%20Máo%20Hóu).png)

白毛猴 (白毛猴, bái máo hóu) —「白毛猴」— 历史名称，统称来自福建省的**两种根本不同的茶**，仅以「白毛猴」这个共同名称相连：

**(1) 政和白毛猴** (政和白毛猴) — 绿茶 (绿茶) 来自政和县 (政和县)，福建北部。由茶商**范昌义** (范昌义) 于**1910年**创制。工艺描述为「介于红茶绿茶之间」—「介于红茶与绿茶之间」：长时间（16–18小时）萎凋 (萎凋) 伴随部分酵素氧化，然后杀青、揉捻和干燥。当地暱称—「白绿」(白绿，「白绿茶」)。最初出口至广东、香港和澳门。原料—著名品种**政和大白茶** (政和大白茶) — 与制作白茶「政和白毫银针」相同的品种。政和自北宋时期即以生产银针闻名；正是这个县给了宋徽宗政和年间（1111–1118）的名称，据传说，皇帝对当地茶叶如此赞赏，以至于以县名命名整个统治时期。

**(2) 安溪白毛猴** (安溪白毛猴) — 乌龙茶 (乌龙茶) 来自安溪，福建南部。来自石竹岩山 (石竹岩, 700+ 公尺) 的原创品种，由兄弟**谢驾和谢冰** (谢驾、谢冰) 于清末开发。加工—完整的乌龙茶工艺，包含发酵和「窨制」(窨制，额外的「薰制」)。以「安溪药茶」(「安溪药茶」) 闻名，出口至台湾、日本、东南亚和欧洲。目前几乎**失传**—石竹岩山上仅剩7–8棵老树。

本文专门介绍**政和白毛猴**（绿茶/「白绿茶」），因为正是这种茶在收藏中呈现且更容易获得。安溪版本在「比较」章节中提及。

**1. 分类与产地：**

* **类型：** 绿茶 (绿茶, lǜchá)，形式上为不发酵茶，然而具有**长时间萎凋**（16–18小时）—「制法介于红茶绿茶之间」（「工艺介于红茶与绿茶之间」）。当地暱称—「白绿」(白绿，「白绿茶」)。重点—在**「保毫」(保毫, bǎo háo) 和「做形」(做形, zuò xíng)**。

* **类别：** 历史名茶 (历史名茶)。创制于1910年。在《中国茶经》(《中国茶经》) 中分类为绿茶，尽管工艺包含白茶元素（长时间萎凋）甚至轻乌龙（萎凋时部分氧化）。

* **产地：** 中国，福建省 (福建省)，政和县 (政和县, Zhènghé Xiàn)，福建北部。政和—福建「三大茶县」之一，与福鼎、建瓯并列。座标：约北纬27°22′，东经118°51′。

**2. 历史与文化意义：**

* **历史：**

政和—福建北部茶叶历史最悠久的县。银针 (银针) 生产自北宋时期（960–1127）即已开始。据传说，宋徽宗 (宋徽宗, 1100–1126)，著名的艺术赞助者和《大观茶论》(《大观茶论》，「大观茶论」) 的作者，对这个县的白茶如此赞赏，以至于在1115年将其统治时期重新命名为「政和」(政和，「政治和谐」)—以县名命名。这是中国历史上统治者以茶命名统治时期的极罕见案例之一。

在这个千年茶叶基础上，**1910年**当地茶商**范昌义** (范昌义, Fàn Chāngyì) 创制了新型茶—白毛猴。范昌义开发了独特工艺，结合白茶的长时间萎凋（16–18小时）与绿茶的「杀青」。结果—「介于红茶与绿茶之间」的茶，具有毛茸茸、蓬松的「蜷缩猴子」形状和温和、「香清味醇」的口感。白毛猴迅速征服了广东、香港和澳门市场—这些地区重视毛茸茸的「毫」茶和温和口感。

* **名称：** 白毛 (白毛) —「白毫」（芽和嫩叶上丰富的白色茸毛）；猴 (猴) —「猴子」。弯曲、卷曲的茶叶，密布白毫，像蜷缩的小猴子。当地暱称—「白绿」（「白绿茶」）—反映工艺的中间特性。特别优秀的批次获得后缀「王」(王，「王」)。

* **文化意义：** 白毛猴—「混合」茶：诞生于给整个宋朝统治时期命名的县，在白茶和绿茶传统的交汇点。其工艺—「介于红茶与绿茶之间」—反映了福建实验精神，孕育了中国茶的全部六大类别。

**3. 植物学描述与原料：**

* **品种/栽培种：** **政和大白茶** (政和大白茶, Zhènghé Dàbáichá) —「政和大白茶」。福建最珍贵的栽培种之一：大型、肉质芽叶，密布白毫。灌木型 (灌木型)，中叶类 (中叶类)，中芽种 (中芽种)。正是用这个品种生产著名白茶「政和白毫银针」(政和白毫银针) 和「政和白牡丹」(政和白牡丹)。

* **采摘：** 清明后 (清明后)，春茶。标准—一芽二三叶 (一芽二三叶)，密布白毫。只采摘干燥天气下未受损的嫩芽。嫩芽必须大型、「肥壮」(肥壮，「肥壮结实」)。

* **原料要求：** 绝对不能损伤毫毛—「保毫」(保毫，「保护毫毛」)—主要原则。嫩芽松散地放在竹篮中，避免挤压。

**4. 风土条件与栽培特点：**

* **生长海拔：** 600–900公尺（福建北部山区）。政和县位于武夷山脉 (武夷山脉) 支脉，福建与浙江交界—省内最「山地」的茶县之一。

* **气候：** 亚热带季风气候。年平均温度约18°C。年降雨量1600–1800毫米。高湿度，频繁雾气—一年超过180天。显著的日夜温差（>8°C）—刺激胺基酸和芳香物质的积累。云雾的散射光—确保政和大白茶嫩芽丰富毫毛的关键因素之一。

* **土壤：** 红壤、黄壤 (红壤, 黄壤)，酸性（pH 4.5–5.5）。排水良好，深厚（>60公分），富含有机质和矿物质。母岩—花岗岩和砂岩，提供口感的矿物质「骨架」。典型的福建北部山地土壤—与武夷岩茶生长的土壤相同，尽管海拔和微气候不同。

* **生态：** 政和—森林覆盖率高（>70%）的山县。竹林、松树和阔叶林在茶园上方形成「绿色天篷」。水资源—闽江流域的溪流和河流。山区无工业污染。

**5. 制作工艺：**

白毛猴工艺—独特的「混合」，结合白茶、绿茶和部分红茶元素。主要原则：**「保毫」(保护毫毛)**和**「做形」(塑造「猴子」)**。「内质重萎凋适度」(内在品质依赖适度萎凋)—关键公式。

* **萎凋 (萎凋, wěidiāo)：** **16–18小时**—比标准绿茶（2–4小时）长得多。叶子薄层摊放在竹盘或「水筛」(水筛，「水筛」) 上，在阴凉或通风室内。16–18小时内发生部分酵素氧化多酚：失水25–30%，叶子变暗至深绿色，叶柄变皱，毫毛显现「如银」(白毫显露, 毫毛如银)，质地「柔软如棉」(叶质柔软如棉)，叶柄弯曲但不断裂 (梗折不脆断)。过度萎凋→发红；萎凋不足→过度「绿色」苦涩。

* **杀青 (杀青, shāqīng)：** 在锅中140–150°C。先「扬炒」(扬炒, yáng chǎo)，然后「闷炒」(闷炒, mèn chǎo)。直到颜色转为「青」(青，蓝绿色)，叶柄黄绿色，香气「清香」(清香)，无草腥味。

* **揉捻做形 (揉捻/做形)：** 独特阶段—结合揉捻和塑造「猴子」。师傅交替**「揉搓」**(揉搓, róucuō) 和**「茶团旋转」**(茶团旋转, chá tuán xuánzhuǎn)：叶子卷成圆形、弯曲形状，像「蜷缩的小猴子」。操作「轻巧温和，注意保毫」(操作轻巧，注意保毫)。5–8分钟。湿润的「茶团」(湿茶团)—每个50–500克。

* **初烘 (初烘)：** 100–110°C。每个「焙笼」(焙笼，竹制烘干篮) 放10–15个小「茶团」。至50–60%干燥度（茶叶不黏手指）。

* **复焙整形 (复焙整形)：** 50–60°C。同时继续塑形：茶叶「完善」至完整的「小猴」(小猴) 形状。毫毛不应脱落（温度不太高），颜色不应「闷」成黄褐色（温度不太低）。至80–90%干燥度—降至40–50°C。烘至完全干燥。

* **拣剔 (拣剔)：** 去除叶柄和有缺陷的茶叶。包装。

**6. 感官特征：**

* **干茶外观：** 大型、弯曲茶叶，「蜷缩的猴子」(犹如毛猴静伏，「如猴子静卧安息」)。密布银白色毫毛。形状蓬松，「肥壮卷曲」(肥壮卷曲，「肥壮卷曲」)。颜色从银绿色到深绿色带银色「霜」。

* **香气：** 「毫香」(毫香, háo xiāng)—清新、甜美。清香 (清香)。比典型绿茶复杂—带有「奶香」和「蜜香」层次，由长时间萎凋产生。

* **滋味：** 温和，「醇和微甘」(醇和微甘, chúnhé wēigān，「和谐温和，略带甜味」)。最小涩味。明显的「回甘」(回甘，回甜)。茶体中等，「丝滑」。

* **汤色：** 「清绿泛黄」(清绿泛黄, qīnglǜ fànhuáng，「清绿带黄色调」)。某些资料中为「橙黄」(橙黄, chénghúang，「琥珀黄」)—取决于萎凋程度。

* **叶底：** 嫩绿、完整、匀净、无杂 (嫩绿、完整、匀净、无杂)。嫩芽保持形状—「保毫」和精心加工的指标。

**7. 化学成分：**

* **多酚类 (茶多酚)：** 中等—长时间萎凋（16–18小时）部分氧化儿茶素，降低涩味。多酚含量白毛猴更接近白茶（15–20%）而非标准绿茶（20–30%）。

* **胺基酸 (氨基酸)：** 较高—政和大白茶，福建最富胺基酸的栽培种之一，加上山地风土（雾气>180天，日夜温差>8°C）=丰富的胺基酸谱。L-茶胺酸占主导—负责特有的「丝滑」甜味和「毫香」(毫香，毫毛香气)。

* **EGCG：** 存在，但以更「温和」形式—16–18小时萎凋时的部分氧化将部分EGCG转化为茶黄素和茶红素，创造出纯绿茶所没有的「双重」抗氧化谱。

* **咖啡因：** 中等。

* **维生素：** C、B群。

* **矿物质：** 氟、钾、镁、锌。

**8. 保健功效：**

* **抗氧化保护：** 多酚类 + EGCG + 维生素C。长时间萎凋部分将儿茶素转化为茶黄素，提供「双重」抗氧化谱（绿茶 + 部分氧化）。

* **温和提神效果：** 咖啡因 + L-茶胺酸—提神而不紧张。特别温和—由于多酚类较低和胺基酸较高。

* **消化支持：** 中等儿茶素刺激胃肠蠕动，不刺激黏膜（与更「强烈」的绿茶不同）。

* **认知功能：** L-茶胺酸刺激大脑α波活动。

* **重要：** 所列功效基于一般资料，不构成医疗建议。

**9. 冲泡方法：**

* **水温：** 75–80°C。嫩「猴子」对沸水敏感—毫毛「烫伤」会使茶汤混浊。

* **茶量：** 3–5克/150–200毫升。

* **茶具：** 玻璃杯或瓷盖碗—观察「猴子」在水中展开和银色毫毛。

* **过程：**
	1. 温杯。
	2. 投茶。
	3. 洗茶—注水，3秒后倒出。白毛猴建议洗茶—「展开」蓬松卷曲的「猴子」并去除第一泡的细毫。
	4. 第一泡—1–2分钟（75–80°C）。
	5. 3–5泡，每次增加15–20秒。
	6. 观察叶底：如果嫩芽完整、嫩绿、保持毫毛—茶叶真实且冲泡正确。破碎的嫩芽和脱落的毫毛—过度干燥或假冒的迹象。

* **特点：** 由于长时间萎凋（16–18小时），白毛猴比标准绿茶更「耐温」：即使在85°C也不苦涩，反而展现「蜜香」层次。然而为了最大「毫香」(毫香，毫毛香气)，75–80°C更佳。

**10. 储存：**

* **温度：** 0–5°C，密封。白毛猴对湿度特别敏感—茶叶上丰富的毫毛如「海绵」，比大多数绿茶更快吸收大气湿度和异味。
* **容器：** 密封、不透明。铝箔 + 真空包装—理想。开封前—恢复至室温，避免毫毛凝露。
* **光线：** 完全隔离—毫毛和叶绿素对光敏感。
* **保存期：** 0–5°C下6–8个月。最好在当年采摘年份内饮用。不建议长期存放：白毛猴以新鲜度和「毫香」著称，随时间会消散。

**11. 价格与假冒：**

白毛猴—稀有且昂贵的茶：有限产区（政和）、手工劳动、要求高的原料（政和大白茶）、复杂工艺（16–18小时萎凋 + 手工塑造「猴子」）。

* **如何避免假冒：**
	* 形状—蓬松、弯曲的「猴子」，密布白毫。不是扁平、不是「针」。
	* 香气—「毫香 + 清香」(毫香 + 清香，毫毛香 + 清香)。无「草腥」或「焦糊」味。
	* 茶汤—「清绿泛黄」(清绿泛黄，清绿带黄色调)。混浊茶汤—过度干燥的迹象。
	* 叶底—完整、嫩绿的嫩芽，保持毫毛。

**12. 趣闻轶事：**

* **给统治时期命名的县。** 政和—中国唯一一个宋朝皇帝宋徽宗以其命名整个统治时期（1111–1118）的县：他对当地白茶如此印象深刻，以至于将时代重新命名为「政和」(政和)。白毛猴—这个千年传统的延续者。

* **「介于红茶与绿茶之间」(介于红茶绿茶之间)。** 独特工艺：16–18小时萎凋（如白茶）+ 杀青和揉捻（如绿茶）。结果—形式上是绿茶，但口感更接近白茶。

* **范昌义 (范昌义, 1910)。** 白毛猴创制者—政和茶商，结合白茶和绿茶传统为广东和香港出口市场。

* **两个白毛猴。** 政和（绿茶/白绿茶，政和大白茶）和安溪（乌龙茶，来自石竹岩山）。安溪白毛猴，由谢驾和谢冰兄弟在清末创制，以「安溪药茶」(「安溪药茶」) 闻名，经台湾出口至日本、东南亚和欧洲。谢冰兄弟致富后，1878年山东发生干旱时，组织运粮队响应皇帝救灾号召—因此获得清廷「奉政大夫」(奉政大夫) 头衔和佩戴「花翎」(花翎，孔雀羽毛) 的权利。目前安溪白毛猴几乎**失传**—石竹岩山上仅剩7–8棵「碗口粗」(「碗口粗」，树干如碗口粗) 的老树。

* **「静卧的猴子」(毛猴静伏)。** 茶叶形状—名片：蓬松、卷曲、覆盖银色毫毛，确实像蜷缩成团的小猴子。

* **政和大白茶—万能品种。** 用同一品种制作：白毫银针、白牡丹、红茶「政和工夫」(政和工夫红茶)，和白毛猴。一个栽培种四类茶—记录。

**13. 与其他政和和福建茶的比较：**

* **政和白毫银针 (政和白毫银针)：** 来自同一品种（政和大白茶）的白茶。仅芽头。萎凋更长（24–48小时），无杀青。白毛猴—有杀青和揉捻，芽 + 2–3叶，「绿茶」特性。

* **安溪白毛猴 (安溪白毛猴)：** 来自石竹岩山的乌龙茶。完整乌龙发酵 + 「窨制」。「安溪药茶」。几乎失传。完全不同类型—乌龙，非绿茶。

* **太平猴魁 (太平猴魁)：** 安徽。也是「猴」茶，但扁平、大叶、无毫毛。白毛猴—卷曲、蓬松、丰富毫毛。两者都是绿茶，但来自不同省份，工艺不同。

* **政和白牡丹 (政和白牡丹)：** 来自政和大白茶的白茶。芽 + 1–2叶。无杀青。白毛猴—有杀青和塑造「猴子」。两者都「毛茸茸」，但不同类别。

**结论：**

白毛猴—具有千年血统的茶：给宋朝皇帝统治时期命名的政和县，在1910年孕育了这个白茶和绿茶的「混合体」。其公式—16–18小时萎凋 + 杀青 + 塑造「蜷缩猴子」—创造出形式上是绿茶，但精神上是「白绿」(「白绿茶」) 的茶：毫香、温和口感「介于」类别之间，杯中银色「猴子」。用同一政和大白茶品种制作四种茶—从白毫银针到红茶「工夫」；白毛猴—其中最「过渡」的，白茶与绿茶之间的桥梁茶，诞生于福建实验精神。
