# 磐安云峰 (磐安云峰, Pán'ān yúnfēng)

![磐安云峰 (磐安云峰, Pán'ān yúnfēng)](images/Паньань%20Юнь%20Фэн%20(磐安云峰,%20Pánān%20Yún%20Fēng).png)

![磐安云峰 (磐安云峰, Pán'ān yúnfēng)-1](images/Паньань%20Юнь%20Фэн%20(磐安云峰,%20Pánān%20Yún%20Fēng)-1.png)

磐安云峰（磐安云峰，Pán'ān yúnfēng）——「磐安县的云雾山峰」——产自浙江省金华市磐安县的绿茶，位于大盘山脉——被誉为「群山之祖、诸水之源」：这里是**四大河流水系**的发源地——钱塘江、瓯江、灵江和曹娥江。磐安的茶叶传统可追溯至晋代（3-5世纪）：相传道教圣人**许逊**（许逊，「真君大帝」）——「四大天师」之一——云游磐安群山时，发现野生茶树，教导当地农民制茶技术，创造了著名的**「婺州东白」**（婺州东白，「婺州的东白茶」）。陆羽在《茶经》中将「婺州东白」列为名茶，并记载「大盘山、东白山产者佳，列为贡品」——「产自大盘山和东白山的最佳，列入贡茶」。县内有**玉山古茶场**（玉山古茶场，「玉山古代茶叶市场」）——**中国唯一完整保存的古代茶叶市场**（始建于宋，清代重建，面积3430平方公尺），**全国重点文物保护单位**。2022年，「赶茶场」（赶茶场，「茶场集市」——磐安民俗节庆）作为「中国传统制茶技艺及其相关习俗」的一部分，被列入**联合国教科文组织非物质文化遗产名录**。

品质特征：**「三绿一香」**（sān lǜ yī xiāng，「三种绿色一种香」）：绿色条索、绿色茶汤、绿色叶底＋兰花香。土壤有机质含量——**3.5-14.8%**——中国所有茶区中最高的范围之一。水浸出物——**47%**（国家标准36%——超出11个百分点）。森林覆盖率——**80.1%**。品牌价值——**235.1亿元人民币**（2024年）。

**1. 分类与产地：**

* **类型：** 绿茶（绿茶，lǜchá），未发酵。半炒半烘（半炒半烘，bàn chǎo bàn hōng）。两种形态：**条形**（条形，tiáo xíng——「针形」，主要形态）和**扁形**（扁形，biǎn xíng）。独创成形技术**「抖捺理」（dǒu nà lǐ）**——「抖动、按压、理顺」——三种手工技法交替使用，保持芽叶完整并形成特有的「针形」结构。

* **类别：** 国家农产品地理标志产品（国家农产品地理标志产品，2020年）。「全国名茶」（全国名茶，商业部，1986年）。杭州茶叶节「文化名茶」（文化名茶，1991年）。县级非物质文化遗产（县级非遗，2021年——制作技艺）。「赶茶场」（赶茶场）——联合国教科文组织遗产组成部分（2022年）。唐代「贡茶」（婺州东白）。玉山古茶场——全国重点文物保护单位。

* **产地：** 中国，浙江省，金华市，磐安县。14个乡镇。大盘山脉，海拔600-1200公尺。座标：东经120°17′-120°47′，北纬28°49′-29°19′。

**2. 历史与文化意义：**

* **历史：**

磐安——「纯山区县」，浙江最多山的县之一，「群山之祖、诸水之源」——「群山的始祖，众水的源头」。茶叶历史始于晋代（3-5世纪）：道教圣人许逊（许逊，西元239-374年），被尊为「真君大帝」，「四大天师」之一——与张道陵、葛洪、萨守坚并列——云游磐安玉山时，发现野生茶树，教导当地农民改良制茶技术。成果就是「婺州东白」（婺州东白）——闻名全中国的茶叶。许逊是真实的历史人物，南昌人，以治理赣江水患闻名；道教净明道派尊他为创始人。宋代玉山的茶农感念许逊，为他建立「茶场庙」（茶场庙，「茶场寺庙」），至今仍奉他为「茶神」——这是中国唯一将道教圣人奉为茶叶守护神的寺庙。

唐代陆羽将「婺州东白」收入《茶经》，记载：「大盘山、东白山产者佳，列为贡品」——茶叶成为「贡茶」。宋代，围绕茶场庙形成了**玉山古茶场**——中国最古老的茶叶贸易市场之一，国家在此实行「榷茶」（榷茶，quèchá，茶叶专卖）。现存建筑（清代，面积3430平方公尺）包括寺庙、行政用房和交易大厅——**中国唯一完整保存的古代茶叶市场遗址**。

1979年，在唐代「婺州东白」的基础上，结合西湖龙井、太平猴魁和峨眉蕊的技术，创制了现代「磐安云峰」。1983年——恢复磐安县建制（原属东阳）：大盘山和东白山被分属两县，磐安「继承」大盘山，东阳得东白山。这就需要新名称——「云峰」（「云雾山峰」）。1986年——「全国名茶」（商业部）。2020年——地理标志。2022年「赶茶场」（赶茶场）——玉山古茶场的民俗集市节庆——作为联合国教科文组织遗产「中国传统制茶技艺及其相关习俗」的组成部分。

* **名称：** 磐安（Pán'ān）——县名；云峰（Yún Fēng）——「云雾山峰」——云雾缭绕山峰的意象。完整含义：「磐安的云雾山峰」。品牌口号：**「道骨仙风，磐安峰」**——「道家风骨，仙人之风——磐安之峰」。

* **文化意义：** 磐安——「群山之祖、诸水之源」（「群山的始祖，众水的源头」）：浙江四大河流的分水岭。与道教的联系（许逊——「四大天师」之一）、最古老的茶叶市场和联合国教科文组织遗产，使磐安云峰成为「自然、道教与贸易」交织1700年的茶叶。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要——**木荷群体种**——适应山地云雾气候的本地原生品种。辅助：龙井43号、乌牛早。最高等级：单芽或一芽一叶，芽叶长度≤3.5公分。

* **采摘：** 春季——清明前。标准因等级而异：特级——单芽（90%以上茸毛）；一级——一芽一叶（≥90%）；二级——一芽二叶。

* **等级：**
  * **特级：** 全芽，茸毛≥85%。**2000元人民币/500克起。**
  * **一级：** 一芽一叶≥90%。800-2000元人民币。
  * **二级：** 一芽二叶。300-800元人民币。

**4. 风土与栽培特点：**

磐安——「纯山区县」：山地占领土100%。大盘山（大盘山，1245公尺）——主峰，「四河之顶」。

* **生长海拔：** 600-1200公尺。核心区——800公尺以上。

* **气候：** 亚热带季风气候。年均温——16.1°C。降雨量——1570毫米。**雾日——≥200天。** 湿度——≥85%。「山高水秀、林密雾重」——风土特征。

* **土壤：** 红黄壤（红黄壤，pH 4.5-5.5）。土层深度——50-100公分。**有机质——3.5-14.8%**——中国茶区最高且范围最广之一。山泉水矿物质含量极低。

* **生态：** 森林覆盖率——**80.1%**。磐安——**国家级生态文明建设示范县**。山区零工业。2007年农业部实验室分析显示：**水浸出物——47%**（比国家标准36%高11个百分点），**13种农药——未检出**，铅——0.74毫克/公斤（无害标准5毫克/公斤）。「典型山地茶特征」——专家结论。

**5. 制作工艺：**

磐安云峰技术继承「婺州东白」并融合西湖龙井（扁形）、太平猴魁（大芽成形）和峨眉蕊（温和烘干）的要素。主要步骤：

* **摊放：** 3-6小时。失水率约15%。

* **杀青——斜锅（斜锅，xiégūo）：** 温度——160°C。与标准平锅不同，**斜锅**同时实现抛叶和在重力作用下沿锅壁滚动。工具——**竹炒手**（竹炒手，zhú chǎoshǒu，竹制炒茶工具）或棕帚（棕榈扫帚）。原则：「叶翠不老，叶熟不黄，折梗不断」——「叶片保持翠绿不老化；叶片炒熟不变黄；茎梗弯折不断裂」。斜锅＋竹炒手——磐安独特技术，大多数其他浙江绿茶中未见。

* **做形——独创技术「抖捺理」：** 在**平锅**中，80°C。三种手法交替：
  * **抖（dǒu）——「抖动」：** 茶叶被抛起分散。嫩叶——多「抖」；粗叶——少「抖」。
  * **捺（nà）——「按压」：** 茶叶被压向锅壁。嫩叶——**不按压**（嫩叶不捺）；粗叶——加强按压。
  * **理（lǐ）——「理顺」：** 芽叶理成笔直的「针」。
  
  结果：笔直、秀丽的芽叶（紧直苗秀），保持完整——叶片不破碎。

* **烘干——两阶段：**
  * **初烘：** 100-120°C。至含水量约30%（「刺手感」——「扎手感觉」）。
  * **复烘——「文火慢烘」：** 至含水量≤6%。目的——均匀干燥并「显现」香气。芽叶搓揉时碎成粉末。

**6. 感官特征：**

* **干茶外观：** 笔直、秀丽的「针」（紧直苗秀，jǐnzhí miáoxiù），「如立正的嫩芽」。翠绿鲜润，有光泽。茸毛明显但不过多。

* **干茶香气：** 清高持久（清高持久，qīng gāo chíjiǔ）。**兰花清香**（兰花清香，lánhuā qīngxiāng）——主调。

* **茶汤香气：** 兰花般花香，「空灵」。冷却后——「蜜香」和「栗香」韵味。持久度——冷杯留香3分钟以上。

* **滋味：** 鲜醇爽口（鲜醇爽口，xiān chún shuǎng kǒu）。甘醇。回味甘甜，持久。苦涩度——极低。

* **汤色：** 嫩绿明亮——某些资料称「杏绿」（「杏黄带绿」）。

* **叶底（冲泡后茶叶）：** 嫩绿成朵，鲜亮。芽叶——完整，不破碎。

* **品质特征：** **「三绿一香」**（sān lǜ yī xiāng）——「三种绿色一种香」：条索翠绿＋汤色嫩绿＋叶底绿亮＋兰花清香。

**7. 化学成分：**

* **水浸出物：** **47%**——比国家标准（36%）高11个百分点。这意味著近一半的干物质溶入茶汤——浙江绿茶和全中国最高指标之一。高水浸出物——土壤有机质创纪录（3.5-14.8%）和200天雾日的直接结果：接受最少直射阳光的茶树合成更多可溶性物质（胺基酸、糖类）和较少不溶性物质（木质素、纤维素）。结果——茶汤「饱满」同时保持柔和。

* **茶多酚：** 约19.2%——适中水平，确保柔和度。

* **胺基酸：** 较高——200天雾日、散射光和创纪录土壤有机质（3.5-14.8%）的结果。EGCG活性高。

* **铅（Pb）：** 0.74毫克/公斤——比无害标准（5毫克/公斤）低6.8倍。

* **农药：** 13种——未检出（2007年分析）。

* **咖啡因：** 绿茶标准含量。

**8. 保健功效：**

* **抗氧化保护：** EGCG＋维生素C——双重抗氧化系统。茶多酚19.2%配合高水浸出物（47%）确保抗氧化物有效溶出。

* **提神效果：** 咖啡因与L-茶胺酸协同——温和、持久的清醒感。

* **消化支持：** 儿茶素刺激胃肠蠕动和脂肪分解。

* **认知功能：** L-茶胺酸刺激大脑α波活动。

* **生态纯净：** 零农药＋低铅＝浙江最「纯净」的茶之一。

* **重要提示：** 所列功效基于一般资料，非医疗建议。不建议空腹饮用。每日不超过600毫升。

**9. 冲泡方法：**

* **水温：** 85°C。不用沸水——嫩芽易「烫伤」。

* **茶量：** 3克/150毫升（比例1:50）。

* **茶具：** 玻璃杯——观赏芽叶「舞动」：笔直的「针」缓缓垂直下沉，「成朵」绽放，宛如「兰花瓣瓣」（形如兰花瓣瓣）。这是浙江最具美感的茶艺之一。

* **方法——「三投法」（Sān tóu fǎ，「三次注水法」）：**
  1. 温杯。
  2. 投茶。
  3. 注入1/3水（85°C）。
  4. 摇晃杯子30秒——「唤醒」。
  5. 注水至7/10满。
  6. 浸泡1.5分钟。

* **冲泡次数：** 3-4次。每次增加20-30秒。

**10. 储存方法：**

* **温度：** 0-5°C（冰箱），密封。磐安云峰——娇嫩的「半炒半烘」茶，对湿气和异味敏感。
* **容器：** 密封、不透光。
* **光线：** 完全隔绝。
* **期限：** 建议6-8个月内饮用。开封后——30天。

**11. 价格与防伪：**

磐安云峰——高价位茶叶。特级——**2000元人民币/500克起**；一级——800-2000元；二级——300-800元。

* **如何避免假货：**
  * 地理标志——「磐安云峰」。品牌价值——235.1亿元。
  * 外形——笔直、秀丽的「针」（紧直苗秀），非卷曲或扁平。
  * 香气——**兰花香**（兰花清香）。无兰花香调——可能是假货。
  * 「三绿一香」特征——三种「绿」（条索、茶汤、叶底）＋一种香。

**12. 趣闻轶事：**

* **道教圣人创造了茶。** 许逊（许逊，西元239-374年）——道教「四大天师」之一——亲自教导玉山农民制茶，创造「婺州东白」。茶场庙——他的圣殿。品牌口号：「道骨仙风」——「道家风骨，仙人之风」。

* **联合国教科文组织（2022年）。** 「赶茶场」（赶茶场）——玉山茶场的民俗集市节庆——列入联合国教科文组织遗产「中国传统制茶技艺」。这是罕见的不是茶叶本身，而是**茶叶集市**获得联合国教科文组织地位的案例。

* **中国唯一的古代茶叶市场。** 玉山古茶场——**全国唯一保存完整的古代茶叶市场遗址**——中国唯一完整保存的古代茶叶贸易市场。面积——3430平方公尺。

* **四河之源。** 磐安——钱塘江、瓯江、灵江和曹娥江的分水岭。茶叶生长在四条流入太平洋的河流系统的「屋顶」。

* **有机质3.5-14.8%。** 中国茶区创纪录的范围。14.8%有机质——黑土地水平，但在山地酸性土壤上。

* **水浸出物47%。** 比标准（36%）高11个百分点。近一半茶叶溶入茶汤——最大「回报」。

* **斜锅＋竹炒手。** 磐安独有的炒制：锅的倾斜使茶叶向下滚动，炒手向上抛起——「重力炒制」。

* **「三绿一香」——「三种绿色一种香」。** 品质公式，铭刻在每一杯磐安云峰中。

**13. 与其他浙江绿茶的比较：**

* **西湖龙井（西湖龙井，Xīhú Lóngjǐng）：** 杭州。扁平，炒制。「栗香」/「豆香」。中国最著名的绿茶。磐安——「针形」，兰花香，半炒半烘。两者都产自浙江，但微气候不同：西湖——湖区（30-100公尺），磐安——山区（600-1200公尺）。

* **开化龙顶（开化龙顶，Kāihuà Lóng Dǐng）：** 衢州（浙江）。山地茶，「拳形」。栗香。磐安——更「兰花香」和「针形」。

* **泾县特尖（泾县特尖，Jīngxiàn Tè Jiān）：** 安徽。同样「针形」。但来自不同省份，不同风土。磐安——来自「四河之顶」，有机质高达14.8%。

**结语：**

磐安云峰——来自「四河之顶」的茶，由道教圣人许逊于3世纪创造，被陆羽在《茶经》中赞为「贡茶」，在中国唯一完整保存的古代茶叶市场交易，2022年通过其集市「赶茶场」进入联合国教科文组织遗产。其「三绿一香」特征——三种绿色和一种兰花香——源自200天雾日、80%森林、高达14.8%的有机质和斜锅配竹炒手。以85°C用「三投法」冲泡，观赏笔直的「针」缓缓绽放成「兰花朵朵」在晶莹的水中——让「道骨仙风」充盈您的茶杯。
