# 松峰绿茶 (松峰绿茶, Sōngfēng lǜchá)

**松峰绿茶 (松峰绿茶, Sōngfēng lǜchá)**

![Сунфэн Люй Ча (松峰绿茶, Sōngfēng lǜchá)](images/Сунфэн%20Люй%20Ча%20(松峰绿茶,%20Sōngfēng%20Lǜchá).png)

![Сунфэн Люй Ча (松峰绿茶, Sōngfēng lǜchá)-1](images/Сунфэн%20Люй%20Ча%20(松峰绿茶,%20Sōngfēng%20Lǜchá)-1.png)

松峰绿茶 (松峰绿茶, Sōngfēng lǜchá) — 湖北绿茶，由明朝开国皇帝朱元璋 (朱元璋) 于西元1368年亲自命名。正是在这里，在松峰山 (松峰山，「松树山峰」) 山脚下，在古老的茶叶村落羊楼洞 (羊楼洞, Yánglóudòng)，实现了著名的西元1391年皇帝诏书「罢造龙团，唯采茶芽以进」(罢造龙团，唯采茶芽以进) — 这道诏令结束了紧压茶时代，开启了散叶绿茶的纪元。羊楼洞后来成为长达14,000公里的万里茶道 (万里茶道) 起点 — 从湖北经蒙古到莫斯科，再延伸至欧洲。西元2015年，国际茶叶委员会授予羊楼洞「世界茶业第一古镇」(世界茶业第一古镇) 称号。松峰绿茶是著名赤壁砖茶 (青砖茶) 的绿茶「双胞胎」，诞生于同样的山脉和同样的千年传统。

**1. 分类与产地：**

* **类型：** 绿茶 (绿茶, lǜchá)，带有黄茶加工元素：生产周期中包含「初包闷黄」(chūbāo mènhuáng，「第一次包裹发黄」，40–48小时) 工序 — 介于绿茶和黄茶之间的边界技术。正式分类为绿茶。外形 — 「松针形」(松针形, sōngzhēn xíng)，卷曲。

* **类别：** 历史名茶，由皇帝朱元璋命名 (1368年)。入选「全国十大名茶」(全国十大名茶, 1958年)。两度荣获「部优产品」(部优产品, 1982年、1985年) 称号。荣获湖北「楚茶杯金奖」(楚茶杯金奖, 2023年)。出口至亚洲和非洲10多个国家。

* **产地：** 中国；湖北省 (湖北, Húběi)；赤壁市 (赤壁市, Chìbì Shì，古称蒲圻，蒲圻)，隶属咸宁市 (咸宁市, Xiánníng Shì)。核心产区 — 羊楼洞镇 (羊楼洞镇, Yánglóudòng Zhèn)，位于松峰山 (松峰山) 山脚下，湖南、湖北、江西三省交界处。羊楼洞 — 「世界茶业第一古镇」(世界茶业第一古镇，自2015年)，万里茶道源头 (万里茶道源头)。

* **地理座标：** ~29°30′–30°00′ 北纬，113°30′–114°00′ 东经 (赤壁地区，「黄金纬度」北纬30°)。

**2. 历史与文化意义：**

* **历史：**

松峰茶的历史就是散叶绿茶作为类别诞生的历史。赤壁地区 (古蒲圻) 的茶叶传统已有2000多年历史。早在唐代，该地区就被划为「茶园户」(园户, yuánhù)，羊楼洞的「洞茶」(洞茶, dòngchá) 成为贡茶 (贡茶)。

与松峰茶皇帝命名相关的传奇事件发生在元朝末年：朱元璋 (朱元璋, 1328–1398)，未来的明朝开国皇帝，在军事征战期间与军队在蒲圻驻扎。士兵们饭后腹痛；羊楼洞当地茶农向他们提供蒲圻绿茶 — 疼痛停止了。西元1368年明朝建立后，感激的朱元璋赐予茶叶「松峰」(松峰，「松树山峰」) 之名，并将茶树生长的山命名为松峰山。

但最重要的事件发生在西元1391年 (洪武24年)：朱元璋颁布著名诏书「罢造龙团，唯采茶芽以进」— 「废除龙团饼，只进献茶芽」。这道诏令结束了数百年的紧压茶 (团茶, tuánchá) 传统，开启了散叶绿茶 (散茶, sǎnchá) 时代，这个时代延续至今。羊楼洞成为首批实现新形式 — 炒制散叶绿茶 — 的地方之一。

清代，羊楼洞发展成国际茶叶贸易中心：在不到2平方公里的面积上，有200多家茶叶公司，人口超过40,000人，城市赢得「小汉口」(小汉口) 美誉。从这里开始万里茶道 (万里茶道, Wànlǐ Chádào) — 长达14,000公里的贸易路线，从湖北经蒙古到俄国，再延伸至欧洲。卡尔·马克思在《论中俄茶叶贸易》中提到羊楼洞砖茶，列宁亲自接见羊楼洞茶商刘峻周 (刘峻周)。西元1956年，羊楼洞入选全国保留的三家茶厂之一。2015年，国际茶叶委员会授予其「世界茶业第一古镇」(世界茶业第一古镇) 称号。

* **名称：** 松峰 (Sōngfēng) — 「松树山峰」— 朱元璋亲自赐予山和茶的名字。松 (sōng) — 松树，象征坚韧和长寿；峰 (fēng) — 山峰、顶峰；绿茶 (Lǜchá) — 「绿茶」。完整含义：「松峰绿茶」。

* **文化意义：** 松峰绿茶是少数由王朝开国皇帝赐名的茶叶之一。羊楼洞不仅是生产地，更是「世界茶叶遗产」：其石板街道，数百年来独轮车「鸡公车」(鸡公车) 载著砖茶滚过，已列入万里茶道联合国教科文组织世界遗产申报预备名单 (万里茶道申遗)。松峰茶厂旁设有面积2600平方公尺的中国青砖茶博物馆 (中国青砖茶博物馆)。赤壁是著名赤壁之战 (赤壁之战, 西元208年) 发生地，这是三国时期的关键战役，被罗贯中在小说中歌颂。因此，一座城市与中国历史上两场最伟大的「战役」相关：军事战役 (西元208年) 和茶叶战役 (西元1391年)。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要 — 羊楼洞群体种 (羊楼洞群体种, Yánglóudòng Qúntǐzhǒng) — 百年本地中小叶种 *Camellia sinensis* var. *sinensis*。树龄30年以上。辅助 — 鄂茶1号 (鄂茶1号, Èchá 1 Hào) 和福鼎大白茶 (福鼎大白茶) — 早熟无性系品种，具有高发芽能力。100个「芽加一叶」芽头重量 — ~45公克。持嫩期 (持嫩期, chí nèn qī) 延长 — 羊楼洞品种的特征之一。

* **采摘：** 春季 — 3月底至4月。采用严格的「九不采」(九不采, jiǔ bù cǎi) 标准：不采雨水、露水、紫色、病虫害、虫蛀、畸形、空心、开裂芽头和顶端受损芽头。

* **采摘标准：** 特级 — 单芽 (≥90%)。一级 — 芽加一个半开叶 (≥80%)。二级 — 芽加两叶。

* **原料要求：** 嫩芽完整，符合「九不采」标准。

**4. 风土条件与栽培特点：**

赤壁位于幕阜山脉 (幕阜山, Mùfùshān) 北麓，三省交界处 (湖南、湖北、江西)，「黄金茶叶纬度」北纬30°。

* **生长海拔：** 300–600公尺 (松峰山南坡)。

* **气候：** 北亚热带地区。年平均气温16.8°C；降雨量1560毫米/年；云雾天数占相当比例；散射光充足，有利于氨基酸合成。

* **土壤：** 黄棕壤 (黄棕壤)，pH 4.5–6.5，富含锌和硒。天然灌溉 — 松峰山山泉。

* **生态：** 森林覆盖率 >72%。该地区获得「中国天然氧吧」(中国天然氧吧) 认证。空气中负离子含量高。茶园被松林环绕 — 因此得名松峰山。茶园供水来自松峰山坡流下的山泉，未受人为污染。幕阜山脉的接近形成天然气候屏障，阻挡湿气并形成持续雾气 — 氨基酸积累的理想条件。

**5. 制作工艺：**

松峰绿茶采用独特工艺，包含「闷黄」(闷黄, mènhuáng，「包裹发黄」) 工序，这是黄茶特有工序，但在此用于形成栗香，而非经典的「发黄」。

* **摊放 (摊放 — tān fàng)：** 标准工序。

* **杀青 (杀青 — shāqīng)：** 在斜锅 (斜锅, xiéguō) 中进行 — 温度在炒制过程中从100–120°C降至80°C。斜锅是罕见技法，提供梯度处理：叶片上部受热较强，下部较温和。

* **摊凉 (摊凉 — tān liáng)：** 干燥前停止热处理。

* **初焙 (初焙 — chū bèi)：** 50–60°C。

* **初包闷黄 (初包闷黄 — chū bāo mèn huáng)：** 40–48小时 — 关键工序！茶叶用布或纸包裹，在温暖潮湿环境中保持。微生物和酶促过程形成特有栗香 (栗香) 并软化涩味。此工序使松峰与黄茶 (蒙顶黄芽、君山银针) 相似，但此处目的不同 — 为了香气而非颜色。

* **复焙 (复焙 — fù bèi)：** 固定初次「闷黄」结果。

* **复包 (复包 — fù bāo)：** 20小时 — 较短时间，最终稳定香气。

* **足火三焙 (足火三焙 — zú huǒ sān bèi)：** 50–55°C，三次连续加热 — 确保含水量 ≤5%，形成「熟栗香」(熟栗香)。

* **储存 — 石膏贮藏法 (石膏贮藏法, shígāo zhùcáng fǎ)：** 羊楼洞传统方法：茶叶储存在密封容器中，放入石膏块吸收湿气并稳定环境。此方法确保香气的卓越保存。

**6. 感官特征：**

* **干茶外观：** 细长、均匀、卷曲的「松针」(针芽状, zhēnyá zhuàng)，紧实均匀 (紧细匀整)。颜色 — 翠绿油润 (翠绿油润)。

* **干茶香气：** 栗香 (栗香, lì xiāng) — 主导香调，高锐。清香 (清香) — 辅助背景。「熟栗香」(熟栗香) — 三次最终烘焙的结果。

* **茶汤香气：** 栗香绿香，高而稳定。冷却后 — 甜美「面包」香调。

* **滋味：** 鲜爽 (鲜爽) 同时醇厚 (醇厚, chúnhòu) — 不寻常的组合，标准绿茶不典型，由「闷黄」工序解释。黏稠度和「体感」高于平均水准。持久甘甜回味 (回甘, huígān)。

* **汤色：** 透明、清澈明亮 (清澈明亮)。

* **耐泡性：** 二级茶5+泡 — 良好指标，由高水浸出物含量 (≥45%) 决定。特级和一级茶因原料浓缩度更高而更耐泡。

* **叶底 (冲泡后茶叶)：** 嫩绿匀齐鲜活 (嫩绿匀齐鲜活)。

**7. 化学成分：**

* **多酚类 (茶多酚)：** ≥30% (一级) — 高含量，提供强大抗氧化活性。

* **水浸出物 (水浸出物)：** ≥45% (特级) — 绿茶中最高指标之一，证明茶汤「浓度」卓越。

* **氨基酸 (氨基酸)：** 充足水准确保鲜爽度。L-茶氨酸占主导。

* **生物碱：** 咖啡因 — 适中，提供提神效果。

* **矿物质：** 锌和硒 — 幕阜山脉地球化学标志。钾、锰、氟。

* **维生素：** 维生素C、B群维生素、维生素E。

**8. 保健功效：**

* **消化支持：** 朱元璋传说中记录的历史功效：茶叶缓解士兵腹痛。多酚和单宁刺激胃肠蠕动。

* **抗氧化保护：** 多酚 (≥30%) 提供强大自由基中和作用。

* **提神效果：** 咖啡因 + L-茶氨酸 — 温和、持久的清醒。

* **认知功能：** L-茶氨酸刺激大脑α波活动。

* **重要提醒：** 所列功效基于一般数据，非医疗建议。不建议空腹饮用。新茶需放置7天以「降火」。开封后10天内饮用完毕。

**9. 冲泡方法：**

* **水温：** 80–85°C。不使用沸水 (>85°C「烫伤」茶叶，产生混浊茶汤)。

* **茶量：** 3公克配150毫升 (比例1:50)。

* **茶具：** 玻璃杯 (上投法, 上投法) 或白瓷盖碗 (中投法, 中投法)。

* **冲泡过程 (玻璃杯)：** 注入80–85°C水，然后投入茶叶。第一泡30秒。每次增加10秒。

* **冲泡过程 (盖碗)：** 注入1/3水量，投茶，摇香 (摇香, yáo xiāng) 唤醒香气，加满水。浸泡2–3分钟。

* **注意：** 注水时沿杯壁缓流 (沿杯壁缓流)，而非直冲中心 — 防止「冲散」茸毛和茶汤混浊。

**10. 储存方法：**

* **温度：** 0–5°C (冰箱) 或传统石膏贮藏法 (石膏贮藏法)。
* **容器：** 密封；传统方法 — 陶瓷罐配天然石膏块 (石膏, shígāo)。石膏作为天然干燥剂：其晶体结构吸收容器内空气中的湿气，维持阻止氧化但不过度干燥茶叶的湿度水准。这种羊楼洞茶农数百年前开发的方法，在效果上可与现代真空包装相媲美。
* **光线：** 完全隔离。
* **保存期：** 开封后10天内享受最佳香气。密封状态下可保存12个月。

**11. 价格与仿冒：**

松峰绿茶属中等至中高价位茶叶。特级 (单芽) — 500元/斤起；一级 — 200–400元/斤；二级 — 更实惠。

* **避免仿冒方法：**

	* 检查产地 — 羊楼洞、赤壁、湖北。
	* 栗香 (栗香/熟栗香) — 「闷黄」工艺的名片。缺乏此香 — 普通绿茶替代品迹象。
	* 外形 — 「松针」，紧实均匀。无形茶叶 — 非松峰茶。
	* 滋味 — 同时鲜爽且醇厚 (鲜爽 + 醇厚)。仅「鲜爽」无「醇厚」— 普通绿茶。

**12. 有趣事实：**

* **改变茶世界的诏书：** 朱元璋西元1391年诏书「罢造龙团，唯采茶芽以进」— 茶业史上最具影响力的文件之一。它结束了500年紧压茶时代，开启散叶茶纪元。羊楼洞是首批实现这一转变的地方之一。

* **马克思和列宁：** 卡尔·马克思在《论中俄茶叶贸易》中提到羊楼洞砖茶。列宁亲自接见羊楼洞茶商刘峻周。松峰茶和砖茶是同一地方的「绿色和黑色双胞胎」。

* **14,000公里茶路：** 始于羊楼洞的万里茶道，从湖北经汉口，沿汉水上行至襄阳，然后经张家口 (或包头) 进入蒙古，再经恰克图到俄国、莫斯科、圣彼得堡和欧洲。总长度14,000+公里。

* **「小汉口」：** 高峰期 (19世纪)，小小的羊楼洞 (面积<2平方公里) 容纳200多家茶厂和40,000居民 — 世界茶业史上无与伦比的茶业密度。在皇家地图《大清皇舆全图》(《大清皇舆全图》) 上，羊楼洞与汉口、武昌使用相同字体标注 — 对一个村镇来说是前所未有的荣誉。

* **绿茶中的「闷黄」：** 「初包闷黄」工序 (40–48小时！) 使松峰与黄茶相似，但此处用于形成栗香而非改变颜色。这是绿茶和黄茶之间的技术「桥梁」。

**13. 与其他湖北绿茶比较：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 湖北。蒸青 (蒸青)，「松针」形。香气 — 「海洋味」，无栗香。松峰 — 炒青配「闷黄」，明显栗香特征和更强「体感」。两者都是「松针」，但工艺根本不同。

* **三峡龙井 (三峡龙井, Sānxiá Lóng Jǐng)：** 湖北宜昌。扁平，炒青。竹炭栗香 (三峡辉锅)。松峰 — 卷曲，「闷黄」栗香。两者都有「温暖」特征，但香气形成机制不同。

* **邓村绿茶 (邓村绿茶, Dèngcūn Lǜchá)：** 湖北宜昌。经典炒青绿茶，卷曲形。特征更「标准」，无「闷黄」。松峰 — 滋味更有「深度」和「醇厚」。

* **蒙顶黄芽 (蒙顶黄芽, Méngdǐng Huáng Yá)：** 四川。经典黄茶，完全「闷黄」。香气 — 「谷物」、「面包」；滋味 — 更温和「干爽」。松峰 — 正式为绿茶，更明显「绿色」鲜爽，但部分「闷黄」带来栗香「深度」。

**结论：**

松峰绿茶 — 诞生于历史转折点的茶叶：在这里，在松峰山上，千年「龙团饼」被埋葬，散叶绿茶诞生，成为我们今天所知的形式。皇帝赐名，「闷黄」工艺在绿色鲜爽中创造栗香深度，以及地址 — 羊楼洞，「世界茶业第一古镇」，从这里14,000公里茶路通向莫斯科和伦敦的餐桌，— 这一切使松峰不仅是茶叶，更是茶文明的活化石。以80°C冲泡，沿杯壁注水，— 品饮六个半世纪前治愈未来皇帝士兵的那款茶。
