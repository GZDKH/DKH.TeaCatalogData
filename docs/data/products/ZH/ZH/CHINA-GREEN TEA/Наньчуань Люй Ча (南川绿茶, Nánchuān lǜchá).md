# 南川绿茶 (南川绿茶, Nánchuān lǜchá)

**南川绿茶 (南川绿茶, Nánchuān lǜchá)**
![Наньчуань Люй Ча (南川绿茶, Nánchuān lǜchá)](images/Наньчуань%20Люй%20Ча%20(南川绿茶,%20Nánchuān%20Lǜchá).png)

![Наньчуань Люй Ча (南川绿茶, Nánchuān lǜchá)-1](images/Наньчуань%20Люй%20Ча%20(南川绿茶,%20Nánchuān%20Lǜchá)-1.png)

南川绿茶 (南川绿茶, Nánchuān lǜchá) —「南川区绿茶」— 来自重庆市直辖市南川区 (南川区, Nánchuān Qū) 的绿茶，生长于金佛山 (金佛山, Jīnfóshān，「金佛山」，2238公尺) 山坡上 — 联合国教科文组织世界遗产 (自2014年起，属于「中国南方喀斯特地貌」)、国家自然保护区及中国最大生物多样性中心之一 (超过5000种植物、500多种脊椎动物)。南川茶叶传统已有1700多年历史 — 从巴人 (巴人, Bā rén) 向周朝廷进贡到五代《茶谱》(《茶谱》, Chá Pǔ, 五世纪) 记载：「涪州出三般茶，宾化最上」— 「涪州产三种茶；宾化县 [=南川] 的最佳」。胺基酸含量 — 4.3% (是一般绿茶的两倍)，硒 (Se) — 特级品≥0.3毫克/公斤，全年雾日260天。特级品 —「金佛玉翠」(金佛玉翠, Jīnfó Yùcuì，「金佛的翡翠绿」)，5000–10000元人民币/公斤。四度荣获「重庆市十大名茶」称号 (重庆市十大名茶, 自2008年起)。截至2024年 — 134500亩茶园，产业总值 — 12.3亿元人民币。

**1. 分类与产地：**

* **类型：** 绿茶 (绿茶, lǜchá)，未发酵茶。工艺 — 炒制后木炭烘干。有三种形态：**紧直形** (紧直形, jǐn zhí xíng) — 用于特级「金佛玉翠」，**眉形** (眉形, méi xíng) — 用于一级茶，**大宗炒青** (大宗炒青, dàzōng chǎoqīng) — 用于二级茶及基础产品。

* **类别：** 中国地理标志产品 (全国农产品地理标志产品, 2012年)。四度荣获「重庆市十大名茶」(重庆市十大名茶, 自2008年起)。日内瓦国际食品博览会金奖 (日内瓦国际食品博览会金奖, 1987年 — 南川红茶)。全国外销红茶第一名 (全国外销第一名, 1980年)。该区获得「全国茶叶基地县」地位 (全国茶叶基地县, 1970年)。

* **产地：** 中国重庆市直辖市南川区 (南川区)。29个乡镇。地理座标：东经106°54′–107°27′，北纬28°46′–29°30′。核心区：北部乡镇 — 大观镇 (大观镇, Dàguān Zhèn)、兴隆镇 (兴隆镇, Xīnglóng Zhèn)、木凉镇 (木凉镇, Mùliáng Zhèn)、河图镇 (河图镇, Hétú Zhèn)、干丰镇 (干丰镇, Qiánfēng Zhèn) — 海拔750–1200公尺，位于云雾带，土壤含硒 (Se)。

* **核心区地理座标：** 约北纬29°10′，东经107°10′。

**2. 历史与文化意义：**

* **历史：**

**商周 — 巴人 (商周, ~西元前十一至三世纪)。** 居住在长江上游流域的巴人 (巴人) 向周朝廷进贡茶叶作为贡品。南川位于巴人聚居区 — 世界最古老产茶区之一。

**陆羽与《茶经》(陆羽, 《茶经》, 八世纪)。** 「茶圣」陆羽 (陆羽, Lù Yǔ) 记载：「巴山峡川有两人合抱」— 「巴山峡谷中有需要两人才能合抱的 [茶树]」。推测指的是金佛山的古茶树 — 至今在其山区仍保存约2000株野生茶树 (*Camellia nanchuanica*)，其中最大的树龄推测达2700年。

**《茶谱》(《茶谱》, 五世纪/五代)。** 毛文锡 (毛文锡, Máo Wénxī) 记载：「涪州出三般茶，宾化最上」— 「涪州产三种茶；宾化县 [=南川] 的最佳」。这是该地区作为产茶区最古老的文字记录之一。

**1920年代 — 出口。** 南川红茶出口至俄国和日本，绿茶出口至欧洲和美国。该区已融入国际茶叶贸易。

**1970–1987年 — 巅峰期。** 1970年：南川获得「全国茶叶基地县」地位 (全国茶叶基地县)。1980年：南川红茶在中国出口红茶中排名第一。1984年：商业部第一名。1987年 — 国际认可的巅峰：**日内瓦国际食品博览会金奖**红茶。

**1990年 —「金佛玉翠」。** 创立高端产品线「金佛玉翠」(金佛玉翠, Jīnfó Yùcuì，「金佛的翡翠绿」) — 特级绿茶，采用紧直形、手工加工和含硒原料。

**2008–2024年。** 四度荣获「重庆市十大名茶」称号 (自2008年起)。地理标志 (2012年)。截至2024年：134500亩茶园 (~8967公顷)，总价值 — 12.3亿元人民币。诗意别名：「金佛山珍」(Jīnfó Shānzhēn，「金佛山的珍宝」)。

* **名称：**
	* 「南川」(南川, Nánchuān) —「南川」— 重庆市的行政区，位于金佛山南麓。
	* 「绿茶」(绿茶, Lǜchá) —「绿茶」— 直接指明类别。
	* 特级品：「金佛玉翠」(金佛玉翠) —「金佛的翡翠绿」— 直接引用金佛山和嫩芽的翡翠绿色。

* **文化意义：** 金佛山 —「金佛山」— 得名于夕阳西下时：夏秋傍晚，当阳光照射在多层岩石峭壁上时，山峰闪耀金光，「如金佛放射无数光芒」。联合国教科文组织世界遗产 (自2014年起)，金佛山属于「中国南方喀斯特地貌」系列，被认为是地球上最重要的生物多样性中心之一。来自这座山的茶 —「金佛的圣饮」，融合了佛教象征「茶禅一味」(茶禅一味，「茶与禅 — 一味」)。金佛山还生长著「五绝」(金佛山五绝)：野生茶树、方竹 (方竹)、银杉 (银杉, *Cathaya argyrophylla*)、银杏和树杜鹃。

**3. 植物学描述与原料：**

* **品种/栽培种：** 两个主要品系：**福鼎大白茶 (福鼎大白茶, Fúdǐng Dàbái Chá)** — 标准高产栽培种，和**巴渝特早 (巴渝特早, Bāyú Tèzǎo，「超早巴渝」)** — 为重庆条件培育的无条纹超早熟品系。另外 — 当地群体种 (群体种, qúntǐ zhǒng)，古茶树的后代。生化特征：胺基酸 — 4.3%，多酚类 — 30.35%。金佛山境内还保存约2000株野生 *Camellia nanchuanica* — 特有的大叶种，属国家一级保护植物。

* **等级：**
	* **「金佛玉翠」(金佛玉翠, 特级)：** 一芽一叶初展 (一芽一叶初展)。采用「十大手法」(十大手法, shí dà shǒufǎ) 手工加工。「竹叶形」(竹叶形)。栗香带「嫩香」。Se ≥0.3毫克/公斤。价格：**5000–10000元人民币/公斤**。
	* **一级 (一级)：** 一芽二叶。眉形。浓厚持久口感。夏秋茶主体。
	* **二级 (二级)：** 一芽二至三叶。大宗炒青形。

**4. 风土条件与栽培特色：**

* **气候：** 亚热带湿润季风气候 (亚热带湿润季风气候)。年平均气温 — 15.1°C (金佛山顶夏季仅约17°C，重庆约28°C)。年平均降雨量 — 1185毫米。关键指标 — **全年雾日260天** (雾日260天)。相对湿度 — **90%**。散射光为主。日夜温差 — 超过10°C。这些条件创造了芽叶极慢生长和胺基酸密集累积的环境。

* **海拔：** 600–1200公尺。生产核心 — 750–1200公尺，位于「云雾带」(云雾带, yúnwù dài)。金佛山 — 大娄山脉 (大娄山) 东端，主峰风吹岭 (风吹岭) — 2238公尺 (某些资料为2251公尺)。

* **土壤：** 弱酸性**紫色土** (紫色土, zǐsè tǔ) 和**黄泥土** (黄泥土, huángnítǔ)，pH 4.5–6.5。母岩 — 二叠纪石灰岩。有机质丰富。关键特色：**局部硒 (Se) 含量超过1 ppm** — 南川区被认为是重庆最大的硒土壤区 (超过一半领土)。硒 — 大多数茶区缺乏的必需微量元素。

* **水资源：** 河流发源于金佛山，流入长江和乌江。水质优良，部分水源含锶 (Sr) 达矿泉水标准。

* **生态：** 森林覆盖率 — 51%。核心区零工业污染。金佛山生物多样性 (超过5000种植物，包括181种特有种，仅此地存在；500多种脊椎动物) 提供天然生物保护，免受害虫侵害，特级品农药需求降至零。

**5. 制作工艺：**

工艺结合特级品手工技法与大宗产品机械化。「金佛玉翠」采用「十大手法」(shí dà shǒufǎ，「十种手法：抓、抖、垫、压、拉…」)，类似连云港云雾茶方法。现代生产整合AI温度控制炒制。

* **摊放 (摊放, tānfàng)：** 长达8小时 — 明显长于大多数绿茶。在高湿度 (90%) 下长时间摊放有助于蛋白质的预酶解和游离胺基酸累积。

* **杀青 (杀青, shāqīng)：** 130°C。采用「抛抖法」(抛抖, pāo dǒu) — 高频翻动防止焦糊，确保均匀固定。

* **揉捻 (揉捻, róuniǎn)：** 轻压，4–8分钟。破坏细胞膜以利萃取，形成初步外形。

* **初烘 (初烘, chū hōng)：** 热风。降低水分至约15–20%。

* **复烘 (复烘, fù hōng)：** 中间阶段，平衡水分。

* **木炭烘焙 (木炭烘焙, mùtàn hōngbèi)：** 至含水量≤7%。木炭提供温和红外线加热，增强栗香而无直火的「焦味」。

**「金佛玉翠」**：严禁农药和化肥。AI系统即时监控温度，根据湿度和叶温感测器数据调整炒制模式。

**6. 感官特征：**

* **干茶外观：** **特级品**：紧直、紧实茶条，绿色带丰富茸毛，「如竹叶」(紧直绿润, jǐn zhí lǜ rùn)。**一级**：眉形 (眉形)，紧实。**大宗**：炒青绿色茶末。

* **干茶香气：** 栗香 (栗香, lìxiāng) — 主调，温暖持久。特级品 — 额外的「嫩香」(嫩香, nèn xiāng)，清新略带乳香。大宗品 — 清香 (清香, qīng xiāng)。

* **茶汤香气：** 栗香温暖，分两阶段展开：第一阶段 — 明亮「坚果冲击」，第二阶段 — 温和草本波浪带轻微甜味。香气持久性 — 特级品5–6泡。

* **滋味：** 鲜爽 (鲜爽, xiān shuǎng) — 4.3%胺基酸 (是平均值两倍！) 提供明显「鲜味」特征。醇厚 (醇厚, chún hòu) — 高水浸出物含量 (≥35%)。回甘 — 持久，带长久「乳甜」尾韵。苦涩味 — 因多酚类/胺基酸比例低而极少。

* **汤色：** 嫩绿明亮 (嫩绿明亮, nèn lǜ míng liàng)。透明，光照下带极淡黄调。

* **叶底 (冲泡后茶叶)：** 黄绿色，均匀，叶片展开成「花朵」(芽叶成朵, yá yè chéng duǒ) — 细心采摘和加工的标志。

**7. 化学成分：**

* **胺基酸 (氨基酸)：** **4.3%** — 是绿茶平均值 (~2–3%) 的两倍。主要成分 — L-茶胺酸 (L-茶氨酸)。高含量归因于三个因素：260天雾日 (散射光抑制茶胺酸光合转化为儿茶素)、日夜温差>10°C (低夜温减缓胺基酸分解代谢) 和含硒土壤 (Se参与胺基酸代谢)。

* **多酚类 (茶多酚)：** 30.35% — 中等偏高。儿茶素特征平衡：足够「厚度」而无过度涩味。

* **水浸出物 (水浸出物)：** ≥35% — 茶汤高「浓度」，可多次冲泡。

* **硒 (Se, 硒)：** 特级品≥0.3毫克/公斤。来自金佛山含硒土壤 (局部Se >1 ppm)。硒 — 具抗氧化、免疫调节和甲状腺保护作用的必需微量元素。

* **咖啡碱 (咖啡碱)：** 约3.0–4.0% — 中等，绿茶典型值。

* **维生素：** C、B₁、B₂、E、K。维生素C因温和加工而保存。

* **矿物质：** K、Mg、Mn、Zn、F、Se。锶 (Sr) — 来自山区水源的微量水平。

**8. 保健功效：**

* **抗氧化作用：** 多酚类 (30.35%) + 硒 — 双重抗氧化系统。硒是谷胱甘肽过氧化酶的辅因子 — 抗氧化防护的关键酶之一。

* **提神效果：** 咖啡碱 + L-茶胺酸 — 温和、持久的提神而无焦虑感。

* **免疫支持：** 胺基酸 (4.3%) + 硒。茶胺酸刺激γ-δ T淋巴细胞增殖。硒支持胸腺功能和抗体产生。

* **甲状腺支持：** 硒 — 脱碘酶成分，将甲状腺素 (T₄) 转化为三碘甲状腺原氨酸 (T₃) 的酶。定期饮用含硒茶对缺硒地区可能有益。

* **脂质代谢支持：** 儿茶素 (主要是EGCG) 促进脂肪氧化和胆固醇谱正常化。

* **认知功能：** L-茶胺酸改善注意力和工作记忆，刺激大脑α波。

* **牙齿保护：** 茶叶中的氟具有防蛀牙作用。

**9. 冲泡方法：**

* **水温：** 一、二级茶80–85°C。特级「金佛玉翠」— 75°C (嫩料需温和处理)。

* **茶量：** 3公克配150毫升 (1:50比例)。

* **茶具：**

	* **玻璃杯 (玻璃杯)：** 最适合「金佛玉翠」— 观察「竹叶」展开。
	* **盖碗 (盖碗)：** 用于一级和大宗茶 — 控制短时间冲泡。

* **过程 (玻璃杯，特级品)：**
	1. 用开水温杯，倒掉。
	2. 注入7/10容量的水 (75°C)。
	3. 采用「上投法」(上投法, shàng tóu fǎ — 先水后茶) 投茶。
	4. 第一泡 — 30秒。
	5. 每次后续 — +15秒。
	6. 可冲4–5次。

* **过程 (盖碗，一级茶)：**
	1. 温盖碗和茶海。
	2. 投入3–4公克茶。洗茶 — 5秒，倒掉。
	3. 第一泡 — 15–20秒。
	4. 每次后续 — +5秒。
	5. 可冲6–8次。

**10. 储存：**

* **条件：** 密封包装，冰箱0–5°C。最好 — 真空或氮气包装。
* **新茶：** 制作后「醒茶」1–2周以稳定香气。
* **茶敌：** 湿气、光线、热度、异味。含硒茶对氧化特别敏感 — 必须严格密封储存。
* **保质期：** 密封包装在0–5°C下 — 最多18个月。

**11. 价格与仿冒：**

* **价格参考：**
	* **「金佛玉翠」(金佛玉翠, 特级)：** 5000–10000元人民币/公斤 — 高端水平，可比拟顶级龙井。
	* **一级 (一级)：** 800–2000元人民币/公斤。
	* **大宗 (二级)：** 平价，日常饮用。

* **如何避免仿冒：**
	* 购买带有地理标志「南川绿茶」标记的产品 — 官方保护标志。
	* 正宗「金佛玉翠」— 紧直、紧实的「竹叶形」茶条，绿色带茸毛。粗大或松散茶条 — 替代品迹象。
	* 检查香气：持久栗香带「嫩香」。无香气或「草腥」刺激 — 可疑。
	* 评估茶汤：嫩绿明亮透明。浑浊或深色茶汤 — 异常。
	* 可疑低价：「金佛玉翠」不可能远低于5000元人民币/公斤。

**12. 趣味知识：**

* **联合国教科文组织山峰。** 金佛山 —「金佛山」— 联合国教科文组织世界遗产 (喀斯特地貌，2014年)。超过5000种植物，181种特有种，包括「五绝」(五绝)：野生茶树、方竹、银杉、银杏、树杜鹃。南川绿茶 — 世界上少数生长在联合国教科文组织名录山峰上的茶之一。

* **陆羽与「两人合抱」。** 「巴山峡川有两人合抱」—《茶经》中对巨大茶树的描述，推测指金佛山，至今仍生长约2000株野生 *Camellia nanchuanica*，其中最大的树龄达2700年。

* **「宾化最上」(五世纪)。** 毛文锡在五代《茶谱》(《茶谱》) 中记载南川 (=宾化) 为涪州三个茶区中最佳 — 该地区茶叶最古老的文字证据之一。

* **日内瓦金奖 (1987年)。** 南川红茶荣获日内瓦国际食品博览会金奖 — 国际认可的巅峰。有趣的是，成名的是红茶，而品牌后来「转向」绿茶。

* **4.3%胺基酸 —「雾的秘密」。** 胺基酸含量是绿茶平均水平的两倍 — 260天雾日、90%湿度和含硒土壤的结果。散射光抑制茶胺酸转化为儿茶素，保持「鲜味」特征。

* **AI控制 +「十大手法」。** 现代「金佛玉翠」生产结合传统手工技法 (十大手法，承袭连云港师傅) 与即时监控炒制温度的人工智慧 — 中国西南地区最先进的茶叶生产线之一。

* **「金佛山珍」—「金佛山的珍宝」。** 南川绿茶的诗意别名，反映其来自世界级山峰的出身。

**13. 与其他绿茶比较：**

* **蒙顶甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 经典四川绿茶，同样来自山区 (蒙顶山，1456公尺)。形状 — 卷曲，香气 — 栗香。南川绿茶的区别：硒含量更高，特级品形状更直，260天雾日 (对比蒙顶约300个晴天)。

* **重庆永川秀芽 (重庆永川秀芽, Yǒngchuān Xiùyá)：** 另一著名重庆绿茶。永川的「秀芽」— 针形，清新特征。南川绿茶更厚重，栗香更明显，有硒优势。

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 中国唯一大型蒸青绿茶 (来自邻近湖北)。同样是硒茶 (恩施 — 世界最大硒区之一)。区别 — 工艺：恩施玉露蒸青 (蒸青)，而南川炒青 (炒青) 并木炭烘干，产生更温暖的「栗香」特征而非「草本清新」。

* **凤冈富锌富硒茶 (凤冈富锌富硒茶, Fènggāng Fùxīn Fùxī Chá)：** 贵州硒锌茶，「硒」领域的直接竞争对手。两者都是含硒山区绿茶，但南川绿茶历史更丰富 (陆羽、《茶谱》) 和联合国教科文组织风土。

**结论：**

南川绿茶 — 来自「金佛山」的茶，联合国教科文组织名录山峰：260天雾日、含硒土壤、4.3%胺基酸、木炭烘干和人工智慧控制下的「十大手法」。从五代《茶谱》五世纪到1987年日内瓦金奖，从对俄红茶到每公斤10000元人民币的绿茶「金佛玉翠」— 一千五百年茶叶传统，冠以现代别名「金佛山珍」—「金佛山的珍宝」。

这款茶 — 适合寻求有「厚度」绿茶的人：醇厚、鲜爽，明显「鲜味」特征和有机硒额外优势。也适合那些重视手中茶杯是世界公认值得保护山峰上260天雾日结果的人。
