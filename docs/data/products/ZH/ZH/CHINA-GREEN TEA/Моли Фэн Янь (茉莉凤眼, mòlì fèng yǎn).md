# 茉莉凤眼 (茉莉凤眼, mòlì fèng yǎn)

**茉莉凤眼 (茉莉凤眼, mòlì fèng yǎn)**
![Моли Фэн Янь (茉莉凤眼, mòlì fèng yǎn)](images/Моли%20Фэн%20Янь%20(茉莉凤眼,%20Mòlì%20Fèng%20Yǎn).png)

![Моли Фэн Янь (茉莉凤眼, mòlì fèng yǎn)-1](images/Моли%20Фэн%20Янь%20(茉莉凤眼,%20Mòlì%20Fèng%20Yǎn)-1.png)

茉莉凤眼是一种艺术性茉莉花茶，每片茶叶都经手工塑形成优雅的细长形状，宛如神话凤凰之眼。此茶属于精品造型茉莉花茶 (工艺花茶, gōngyì huāchá) 的范畴，视觉美感与香气、滋味密不可分。凤凰 (凤凰, fènghuáng) 是中国文化的核心象征之一，代表重生、繁荣与吉祥命运，赋予此茶深刻的象征意义。

**1. 分类与产地：**

* **类型：** 花茶 (花茶, huāchá) — 绿茶（未发酵），以茉莉花薰制。属于再加工茶类 (再加工茶, zàijiāgōng chá)。
* **类别：** 工艺茉莉花茶 (工艺花茶, gōngyì huāchá)。手工造型（束形）茶。福州茉莉花茶传统工艺产品。
* **产地：** 中国福建省 (福建省, Fújiàn shěng) 福州市 (福州, Fúzhōu) — 公认的茉莉花茶制作摇篮。福州窨制工艺 (窨制工艺, yìnzhì gōngyì) 为中华人民共和国国家级非物质文化遗产（2014年起），并为联合国教科文组织于2022年列入非物质文化遗产名录的「中国传统制茶技艺及其相关习俗」项目的一部分。广西 (广西, Guǎngxī)、云南 (云南, Yúnnán) 和四川 (四川, Sìchuān) 等省份亦有生产。
* **地理座标：** 福州 — 约北纬26°04′，东经119°18′。

**2. 历史与文化意义：**

* **历史：** 茉莉花薰茶传统起源于中国宋朝 (宋朝, Sòng cháo, 960–1279年)，在明朝 (明朝, Míng cháo) 和清朝 (清朝, Qīng cháo) 达到鼎盛。茉莉花 (*Jasminum sambac*) 本身于2000多年前经海上丝绸之路从南亚传入中国，在福建省文化中深深扎根。明代学者钱希言 (钱希言, Qián Xīyán) 如此描述繁忙的茉莉花茶市场：「茶斗时节，买花成市，择其大而干者……南风十日，满帘花香」。清代慈禧太后 (慈禧太后, Cíxǐ Tàihòu) 对茉莉花情有独钟，宣布其为宫廷御用花卉。福州茉莉花茶获得贡茶 (贡茶, gòngchá) 地位，福州本身成为全国薰制中心 — 安徽、浙江和江苏的茶叶基料运至此地制作顶级茉莉花茶。

    「凤眼」造型的形成是较晚期工艺花茶传统 (工艺花茶, gōngyì huāchá) 的元素，于20世纪在束茶技艺基础上发展而成。此造型要求制茶师特别精准：茶叶必须扎成细长、略弯的椭圆形，尖端尖锐，精确重现神话凤鸟特有的杏仁状眼部轮廓。福州学派其他经典造型包括「玉蝶」(玉蝶, yùdié)、「龙珠」(龙珠, lóngzhū) 和「银针」(银针, yínzhēn)。

* **名称：**
    * **茉莉 (茉莉, Mòlì)** — 茉莉花。指以新鲜茉莉花薰制。
    * **凤 (凤, Fèng)** — 凤凰 (凤凰, fènghuáng)，神话鸟类，在中国文化中象征重生、恩泽、繁荣与至高和谐。与龙 (龙, lóng) 配对时，凤凰代表阴性原理与皇后。
    * **眼 (眼, Yǎn)** — 眼睛。合称「茉莉凤眼」。

* **文化意义：** 凤凰是中国神话四灵 (四灵, sì líng) 之一，与龙、麒麟、龟并列。凤眼被视为洞察力与清晰视野的化身。因此，茉莉凤眼茶承载著智慧与福祉的祝愿。由于结合了象征深度、视觉美感与精致口感，凤眼茶作为礼品茶备受推崇，特别适合庆典场合和商务馈赠。

**3. 植物学描述与原料：**

* **茶坯 (茶坯, chápi)：** 优质烘青绿茶 (烘青, hōngqīng — 「热风干燥」)，采用福建省春季采摘制作。烘青基料具有多孔叶片结构，最适合在多次窨制过程中深度吸收茉莉花香。茶树品种为当地小叶种 *Camellia sinensis* var. *sinensis* 群体种。
* **茉莉花：** 新鲜茉莉花 *Jasminum sambac* (L.) Ait.（茉莉花，茉莉花, mòlihuā）— 香气最浓郁、纯净甜美的品种。福州茉莉花茶学派专用茉莉花品种。香气成分由110多种挥发性化合物构成，主要包括芳樟醇、苯乙酸酯、苯甲酸甲酯和吲哚。
* **茶叶采摘标准：** 早春。一芽一叶或一芽二叶 (一芽一叶 / 一芽二叶, yī yá yī yè / yī yá èr yè)。原料完整、鲜嫩、无损伤。
* **茉莉花采摘：** 夏季（6-9月），下午14:00后，遵循「三不采」(三不采, sān bù cǎi) 原则：不采晨花、不采阴天花、不采雨后三天花。花蕾最佳开放程度由制茶师目测和手感判定。
* **原料要求：** 极高标准。仅使用精选芽头，白毫 (白毫, báiháo) 明显，茉莉花蕾芳香无损。

**4. 风土条件与栽培特色：**

* **茶园 — 福建：** 亚热带季风气候：年均温+19…+21°C，降雨量1200–1600毫米。茶园位于海拔200–800公尺，土壤为富含有机质的酸性红黄壤 (红壤, hóng rǎng)。温和湿润的气候和山地地形有利于形成滋味甘甜、氨基酸含量高的茶叶。
* **茉莉花园 — 闽江流域 (闽江, Mǐnjiāng)：** 冲积砂质土壤 (冲积土, chōngjī tǔ)，肥沃层深厚，位于低地平原（海拔5–50公尺）。炎热白天与相对凉爽夜晚的温差刺激花朵芳香物质合成。福州茉莉花以特有的「冰糖甜」(冰糖甜, bīngtáng tián) 闻名 — 独特的甜香调，源于特殊的微气候条件。「福州茉莉花与茶文化系统」自2014年起被联合国粮农组织认定为全球重要农业文化遗产（GIAHS）。

**5. 制作工艺：**

茉莉凤眼的制作工艺与其他福州学派造型茉莉花茶相似，包括两个主要阶段：茶坯制作（春季）和多次窨制后手工造型（夏季）。

* **茶坯制作 (茶坯制作, chápi zhìzuò)：**
    1. **采摘 (采摘, cǎizhāi)：** 手工采摘嫩芽一至二叶。
    2. **摊凉 (摊凉, tānliáng)：** 薄摊4–6小时，去除表面水分。
    3. **杀青 (杀青, shāqīng)：** 180–200°C快速热处理，钝化酶活性，保持绿色。
    4. **晾凉 (晾凉, liàngliáng)：** 冷却至接近室温。
    5. **揉捻 (揉捻, róuniǎn)：** 塑造细长形状。
    6. **烘干 (烘干, hōnggān)：** 干燥至含水率4–4.5%。

* **茉莉花窨制 (窨花, yìnhuā)：**
    1. **鲜花养护 (鲜花养护, xiānhuā yǎnghù)：** 分拣、分层堆放，调节花堆温度（32–37°C）促进开放和最大香气释放。
    2. **玉兰打底 (玉兰打底, yùlán dǎdǐ)：** 可使用少量新鲜白玉兰花 (*Michelia alba*，约占茶重1%）创造香气层次，增强茉莉花香的「清新感」，增添淡雅甜香背景。
    3. **多次窨制 (多窨, duō yìn)：** 循环工序：茶坯与新鲜茉莉花混合 → 静置6–8小时 → 起花 (起花, qǐhuā) → 烘焙 (烘焙, hōngbèi)。重复5–7次（高档茶可达8–10次），每次使用新鲜花朵。
    4. **提花 (提花, tíhuā)：** 最后短暂混合少量精选花朵（每100公斤茶用6–10公斤花），赋予明亮的茉莉花香「头香」。
    5. **最终干燥：** 至含水率不超过6%。

* **凤眼造型 (造型, zàoxíng)：** 与其他造型茉莉花茶的关键区别。制茶师手工将数片窨制茶叶和芽头扎成细长、略弯的椭圆形，尖端尖锐，重现凤眼特有的杏仁状轮廓。造型必须紧实、对称，运输过程中不易散开。此阶段需要相当技艺：「凤眼」造型比许多其他造型更难执行，因为需要精确控制比例和弧度。

* **分级 (分级, fēnjí)：** 成品「凤眼」按大小、紧实度、对称性和香气品质分级。

**6. 感官特征：**

* **干茶外形：** 细长、略弯的杏仁状造型，尖端尖锐，形似眼睛。长度3–5公分，宽度1–1.5公分。色泽银绿至墨绿，芽头可见银白色茸毛 (白毫)。造型紧实，叶片牢固结合。可能有少量干燥茉莉花瓣。
* **干茶香气：** 明亮浓郁的茉莉花香，带绿茶草本底韵。香气纯净自然，层次丰富 — 有新鲜花香、淡雅蜜甜和隐约绿茶背景。
* **茶汤香气：** 浓郁包围的茉莉花香，具有明显「鲜灵度」(鲜灵度, xiānlíng dù) — 优质天然窨制的特征标志。芳樟醇（铃兰花香调）和苯乙酸酯（蜜甜）香调。随著冷却，显现精致的绿茶底韵。
* **滋味：** 柔和、细腻、清爽。天然甜味与精致花香和谐交融。涩味极轻，无苦味。茶汤中等厚度，质感顺滑丝润。回甘 (回甘, huígān) 悠长纯净，花蜜香韵，带愉悦清凉感和淡雅奶香。
* **汤色：** 浅黄带金黄色调，透明清澈。每次冲泡后渐浅，呈现淡稻草色。
* **叶底：** 展开的叶片和芽头呈嫩绿色，柔软有弹性。「凤眼」造型即使经过数次冲泡往往仍保持完整，展现造型技艺。原料品质清晰可见：叶片完整，色泽均匀。

**7. 化学成分：**

茉莉凤眼的化学成分是绿茶生物活性物质与茉莉花芳香化合物的结合：

* **多酚类（儿茶素）：** 占茶坯干重15–25%。主要儿茶素为表没食子儿茶素没食子酸酯（EGCG），强效抗氧化剂。
* **氨基酸：** L-茶氨酸 — 占干重1–2%，负责鲜甜味和与咖啡因的协同作用，产生平静专注效果。
* **生物碱：** 咖啡因 — 占干重2–3%（每150毫升茶汤约20–35毫克）。可可碱和茶碱含量微量。
* **维生素：** 抗坏血酸（C）、核黄素（B₂）、硫胺素（B₁）、芦丁（P）。
* **矿物质：** 钾、镁、氟、锰、锌。
* **茉莉花精油：** 芳樟醇（铃兰调，花香基调）、苯乙酸酯（蜜甜）、苯甲酸甲酯（花果调）、吲哚（小剂量增强花香感知）、α-金合欢烯、邻氨基苯甲酸甲酯（橙花调）、乙酸橙花酯（玫瑰蜜香调）。根据气相色谱分析，四个主要产区的茉莉花茶中鉴定出145种以上挥发性化合物，其中约13种为品质特征标志物。
* **叶绿素：** 通过杀青工艺保存。

**8. 保健功效：**

* **抗氧化作用：** 绿茶儿茶素，特别是EGCG，有效中和自由基，保护细胞免受氧化损伤，延缓衰老过程。
* **提神效果：** 咖啡因与L-茶氨酸结合，提供温和平稳的刺激，无紧张感，提高专注力和工作效率。
* **抗压和镇静作用：** 茉莉花芳香成分 — 主要是芳樟醇 — 具有经证实的抗焦虑效果。吸入茉莉花香可降低心率，促进情绪放松。
* **改善消化：** 多酚类刺激胃液和消化酶分泌，促进新陈代谢正常化。
* **心血管系统支持：** 定期饮用绿茶与降低低密度脂蛋白胆固醇和血压正常化相关。
* **抗菌作用：** 茉莉花精油对多种革兰氏阳性菌表现出抗菌活性。
* **增强免疫力：** 维生素C、B群和儿茶素复合物支持机体免疫功能。
* **改善肌肤状态：** 绿茶抗氧化剂结合茉莉花抗炎特性，有助改善肤色和整体肌肤状态。

**9. 冲泡方法：**

* **水温：** 75–85°C。过热会产生苦味并破坏精致芳香物质。
* **用茶量：** 1个「凤眼」（约5–7克）配150–200毫升水。300–500毫升玻璃壶用2–3个造型。
* **茶具：** 玻璃壶 (玻璃壶, bōli hú) 或玻璃杯是观察「凤眼」展开的最佳选择。白瓷盖碗 (盖碗, gàiwǎn) 也适合。不建议紫砂壶：多孔陶土会吸收茉莉花香。
* **冲泡过程：**
    1. 用开水温壶后倒掉。
    2. 轻放「凤眼」入壶或盖碗。
    3. 注入适温水后立即倒掉第一泡（洗茶，洗茶, xǐ chá）。
    4. 再次注水，浸泡2–3分钟（第一泡）。
    5. 分茶入杯。
    6. 重复3–5次，每次增加30–60秒浸泡时间。「凤眼」会逐渐展开，释放新的香气和滋味层次。

**10. 储存：**

茉莉凤眼的储存条件与其他绿茶基茉莉花茶要求相同：

* **温度：** 最佳为冰箱0–5°C，密封容器中，与强味食品隔离。
* **容器：** 密封不透光容器 — 铁罐、瓷罐或真空铝箔袋。使用玻璃容器须完全避光。
* **茶叶大敌：** 湿气、光线、异味、高温、频繁开启包装。
* **保存期限：** 适当条件下12–18个月。茉莉花香随时间减弱，建议购买后一年内饮用。

**11. 价格与仿冒品：**

茉莉凤眼是手工制作、多次天然茉莉花窨制的茶品，属于中高端至顶级价位。价格取决于茶坯品质、窨制次数、手工造型复杂度和制茶师声誉。

**优质茉莉凤眼鉴别要点：**

* **造型：** 「凤眼」应紧实、整齐、对称，无破叶、茶末和杂质。松散不整齐的造型是大量生产或粗制的标志。
* **香气：** 天然茉莉花香 — 层次丰富、柔和深邃。刺鼻、「化学」、过甜香气是人工香料的标志。优质茶展现茉莉花香与绿茶香的和谐平衡。
* **滋味：** 柔和平衡，甜美收尾。苦涩、空洞或「平淡」口感是低品质指标。
* **汤色：** 纯净透明，浅金黄色。混浊或深色是警讯。
* **价格：** 明显低于造型茉莉花茶市场水准的价格应引起警觉。

**12. 趣味知识：**

* 「凤眼」造型是造型茉莉花茶中技术最复杂的之一。制茶师需要掌握细长椭圆和尖端的精确比例，需要多年经验。
* 凤凰 (凤凰, fènghuáng) 在中国神话中只在和平繁荣时代出现。赠送「凤眼」茶是祝愿福祉、智慧和好运的表达。
* 茉莉凤眼非常适合冷泡：将造型放入冷水中，冰箱静置4–8小时。低温缓慢展开产生特别纯净、透明和细腻的香气。
* 中国有句话「窨得茉莉无上味，列作人间第一香」— 「吸收茉莉花香后，[茶]获得无与伦比的滋味；堪称人间第一香」。这句话完全适用于凤眼等高档造型茉莉花茶。
* 福州市自2011年起拥有「世界茉莉花茶之都」非正式称号，福州茉莉花系统于2014年列入联合国粮农组织全球重要农业文化遗产名录。
* 生产1公斤高档茉莉花茶需要6–8公斤新鲜茉莉花，每朵都需手工采摘。整个夏季，熟练采花工每天只能采集约15–20公斤花蕾，使优质茉莉花茶成为最劳动密集的茶类之一。

**13. 与其他茉莉花茶比较：**

* **茉莉玉蝶 (茉莉玉蝶, Mòlì Yù Dié) — 「茉莉玉蝶」：** 同级别和工艺的最接近类比。区别在于造型：玉蝶呈扁平蝴蝶状带「翅膀」，而凤眼为细长杏仁状「眼睛」。香味特征几乎相同，差异在于象征意义和视觉印象。
* **茉莉龙珠 (茉莉龙珠, Mòlì Lóngzhū) — 「茉莉龙珠」：** 叶片卷成紧实球状。比凤眼更紧密的造型确保更缓慢的展开和浓郁集中的茶汤。龙珠是最普遍的造型茉莉花茶。
* **茉莉银针 (茉莉银针, Mòlì Yínzhēn) — 「茉莉银针」：** 以白毫芽头为基料。滋味比凤眼更精致清淡。属于茉莉花茶最高价位段。
* **茉莉花茶 (茉莉花茶, Mòlì Huāchá) — 标准茉莉花茶：** 散茶无造型，通常3–4次窨制。凤眼在原料品质、香气深度和美感方面显著超越。

**结语：**

茉莉凤眼是古老象征与工艺完美结合的茶品。手工塑造的「凤眼」，以最嫩叶片制成，多次浸润新鲜茉莉花香，既是饮品，也是艺术品，更是吉祥祝愿。观察杏仁状造型在热水中缓缓展开，释放阵阵甜美花香的过程，近似冥想体验。此茶适合那些在品茗中寻求不仅是滋味，更是美感、意义和觉知宁静时光的人们。
