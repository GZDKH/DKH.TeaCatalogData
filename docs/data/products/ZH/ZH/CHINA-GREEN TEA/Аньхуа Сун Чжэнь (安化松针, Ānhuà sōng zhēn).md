# 安化松针 (安化松针, Ānhuà sōng zhēn)

**安化松针 (安化松针, Ānhuà sōng zhēn)**
![Аньхуа Сун Чжэнь (安化松针, Ānhuà sōng zhēn)](images/Аньхуа%20Сун%20Чжэнь%20(安化松针,%20Ānhuà%20Sōng%20Zhēn).png)

![Аньхуа Сун Чжэнь (安化松针, Ānhuà sōng zhēn)-1](images/Аньхуа%20Сун%20Чжэнь%20(安化松针,%20Ānhuà%20Sōng%20Zhēn)-1.png)

安化松针 (安化松针, Ānhuà sōng zhēn —「安化松针」) — 著名的湖南绿茶，与南京雨花茶和恩施玉露并列为「中国三针」(中国三针, Zhōngguó Sānzhēn)。创制于1959年 — 与其湖南同乡高桥银峰一样 — 作为「庆祝中华人民共和国成立十周年的献礼」。此茶以其完美的针形著称：纤细、笔直、挺秀的芽叶，形似山松针叶，在玻璃杯中垂直竖立，营造出微型「松林」景观。成形工序 — 40分钟手工搓揉每根芽叶达到「圆、紧、直」(圆、紧、直) 标准 — 工艺如此复杂，以至于师傅们称之为「绝世武功」(绝世武功, juéshì wǔgōng)。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。属于特种炒青绿茶 (特种炒青绿茶, tèzhǒng chǎoqīng lǜchá) 针形茶 (针形茶, zhēnxíng chá)。中国绿茶针形成型的典范。

* **等级：** 「中国三针」之一 (中国三针)。「湖南省三大名茶」之一 (湖南省三大名茶, 1962年)。多次获得国际奖项，包括乌兰巴托国际博览会金奖 (乌兰巴托国际博览会金奖)。2021年制作技艺列入湖南省非物质文化遗产名录。

* **产地：** 中国湖南省 (湖南, Húnán) 安化县 (安化县, Ānhuà Xiàn)。核心产区 — **云台山 (云台山, Yúntái Shān)** — 海拔500公尺以上的山地，常年云雾缭绕。云台山上的褒家冲茶场 (褒家冲茶场, Bāojiāchōng Cháchǎng) 正是安化松针的诞生地。

* **地理座标：** 约北纬28°23′，东经111°13′。

**2. 历史与文化意义：**

* **历史：** 安化县是湖南最古老的产茶区之一，历史可追溯至唐代。唐代文献中提及「渠江薄片」(渠江薄片, Qújiāng Bópián) — 早期的安化茶。宋代时安化茶被列入皇室贡茶 (贡茶) 名单。

	现代安化松针创制于**1959年**，由安化县褒家冲茶场研制 — 作为庆祝中华人民共和国成立十周年的献礼。茶学专家以传统贡茶工艺为基础，开发出独创的40分钟手工针形成型技术。1962年工艺最终标准化，此茶被认定为「湖南省三大名茶」之一 — 与君山银针、高桥银峰齐名。

	随后获得国际认可：乌兰巴托国际博览会金奖及众多国内奖项。2021年 — 制作技艺列入湖南省非物质文化遗产名录。

* **名称：**

	* 「安化」(安化) — 县名。字面意思：「安定教化」。
	* 「松」(松) — 「松树」：描述茶叶形状，似松针。
	* 「针」(针) — 「针」：强调芽叶的纤细、笔直和尖锐。

* **文化意义：** 安化 — 这个县份为世界贡献了两大茶类：著名的**安化黑茶 (安化黑茶,「安化黑茶」)** — 中国六大茶类之一 — 和安化松针 — 针形绿茶的典范。同一县份的两种茶，茶界的两个极端 — 从深度发酵的黑茶到最嫩绿的绿茶。有趣的是，安化正是采用**「茶果间作」(茶果间作)** 系统 — 茶树与果树（枇杷、杨梅）混种 — 类似著名的洞庭山碧螺春系统。

**3. 植物学描述与原料：**

* **品种/栽培种：** 两种类型：
	* **安化群体种 (安化群体种)** — 当地原生品种 *Camellia sinensis* var. *sinensis*。叶质肥厚，持嫩性强 (叶质肥厚持嫩性强)。产生更复杂、多层次的风味。
	* **湘波绿 (湘波绿, Xiāngbōlǜ)** 和**白毫早 (白毫早, Báiháo Zǎo)** — 从云台山群体种中选育的改良品种。萌芽早，氨基酸含量高。

* **采摘：** 严格在清明节前 (清明, ~4月5日)。标准 — **一芽一叶初展**。剔除：虫伤叶、紫叶、露水叶 (露水叶)。只选用新鲜、嫩绿、完美的原料。

* **原料要求：** 极高标准 — 「无虫伤叶、紫叶、露水叶」（无虫伤、紫叶、露水叶）。当日采摘当日加工。

**4. 风土条件与栽培特色：**

* **地形：** 雪峰山脉 (雪峰山脉, Xuěfēng Shānmài) 北坡。山地地形，深谷众多，溪流密布。

* **气候：** 中亚热带季风气候。年平均气温16.2°C，年降水量1682毫米。云雾丰富，昼夜温差显著。

* **海拔：** 500公尺以上。核心区 — 云台山，常年云雾缭绕（因此得名：云台 — 「云台」）。

* **土壤：** 红壤、黄壤 (红壤、黄壤)，由板页岩风化发育 (板页岩风化)，富含磷和硒。酸性适中。土层深厚，透水性良好。

* **间作系统 (茶果间作)：** 如同洞庭山（碧螺春产地），安化茶树历史上与果树混植 — 枇杷 (枇杷) 和杨梅 (杨梅)。茶树吸收花果芳香物质，形成特有的「天然花果香韵」(天然花果香韵)。

**5. 制作工艺：**

安化松针工艺 — 九道工序，关键的40分钟手工成形。这是绿茶中最费工的工艺之一。

* **摊放 (鲜叶摊放 — xiānyè tānfàng)：** 短时间摊放萎凋。

* **杀青 (杀青 — shāqīng)：** 铁锅140°C或滚筒杀青。快速停止氧化。

* **揉捻 (揉捻 — róuniǎn)：** 三段式 — 「轻→重→轻」(轻-重-轻)。

* **炒坯 (炒坯 — chǎopēi)：** 70–80°C — 蒸发水分，开始形成香气。

* **摊凉 (摊凉 — tānliáng)：** 中间冷却，重新分布水分。

* **整形 — 「绝世武功」(整形 — zhěngxíng)：** **关键且最复杂的工序。** 在50–70°C下，师傅**40分钟**手工搓揉每根芽叶，使其达到完美形状，标准为**「圆、紧、直」— 「圆（截面）、紧、直」**。此工序被师傅称为「绝世武功」— 「绝世武功」，因为需要绝对精准：压力过重会折断芽叶，过轻则无法成直形；温度过高会破坏香气，过低则留有水分。

* **干燥 (干燥 — gānzào)：** **35–40°C** 低温慢速烘干 — 绿茶中最低的最终干燥温度之一。慢速是持久深层香气的保证。

* **筛拣 (筛拣 — shāijiǎn)：** 最终分级，去除不合格芽叶。

**6. 感官特征：**

* **干茶外形：** 笔直、纤细、挺秀的芽叶，形状完美如「松针」(细直挺秀似松针)。长度均匀。「长、直、圆、浑、紧、细」(长直圆浑紧细, cháng zhí yuán hún jǐn xì) — 完整的形状标准。色泽翠绿匀整 (翠绿匀整)。白毫显露 (白毫显露)。

* **干茶香气：** 清香高雅持久 (清香高雅持久)。栗香 (栗香) — 在炒坯阶段形成。嫩香 (嫩香)。花果香调 — 间作系统的结果。

* **茶汤香气：** 清香馥郁 (香气馥郁, xiāngqì fùyù)。栗香清新风格。持久。

* **滋味：** 鲜爽 (鲜爽, xiānshuǎng) — 氨基酸含量高。甜醇 (甜醇, tiánchún) — 回甘明显 (回甘明显)。醇厚 (醇厚) — 内含物丰富。特有品评公式：**初始微苦 (微苦) 迅速转甘甜 (迅速转甘甜)** — 绿茶中最快的「转换」之一。

* **汤色：** 嫩绿透亮 (嫩绿透亮)。

* **冲泡视觉效果：** 在玻璃杯中注水时，芽叶缓慢下沉并**垂直竖立**，形成微型「松林」— 针形茶中最具视觉震撼力的效果之一。

* **叶底：** 嫩绿、挺秀、匀整的芽叶，色泽鲜绿。叶片完整，无破损。

**7. 化学成分：**

高山产地、云雾微气候、富硒板岩土壤和间作系统形成特有成分：

* **多酚类（儿茶素）：** 含量丰富。提供抗氧化能力和滋味结构。
* **氨基酸（包括L-茶氨酸）：** 含量较高 — 高山和丰富散射光的结果。提供鲜爽和快速回甘。
* **生物碱：** 咖啡因 — 适中含量。可可碱、茶碱。
* **类胡萝卜素和维生素A：** 含量较高 — 提供明目功效 (明目)。
* **维生素：** 维生素C、B群维生素、维生素A。
* **矿物质：** 钾、镁、磷、硒、锰。

**8. 保健功效：**

* **清热消暑 (清热消暑)：** 传统上推荐于炎热时节饮用。
* **抗氧化作用：** 儿茶素中和自由基。
* **降脂减肥 (降脂减肥)：** 儿茶素促进脂肪代谢。
* **明目 (明目)：** 富含类胡萝卜素和维生素A。
* **提神效果：** 咖啡因和L-茶氨酸。
* **改善消化：** 多酚类刺激消化酶。

* **重要提醒：** 上述功效基于公开资料，不构成医疗建议。

**9. 冲泡方法：**

* **水温：** 80–85°C。

* **茶量：** 3公克配150毫升水（比例1:50）。

* **茶具：** 玻璃杯 — **必选** 用于观赏「松林」：垂直竖立的针叶 — 此茶的主要视觉表演。

* **冲泡过程：**

	1. 温杯，倒掉。
	2. 投茶。
	3. 注水1/3，摇杯 — 「摇香润茶」(摇香润茶)。
	4. 注水至7/10。浸泡1–2分钟。
	5. 后续冲泡 — 每次增加10秒。可冲泡3–4次。

* **注意：** 沿杯壁缓流 (沿杯壁缓流) 注水，不要直接冲茶叶 — 否则针叶会「倒伏」而不能垂直竖立。观察芽叶缓慢下沉并竖立成「松林」— 这景象值得等待数分钟。

**10. 储存：**

* 密封容器，阴凉干燥处保存。
* 最佳 — 冰箱0–5°C。
* 保存期限 — 12个月。
* 开封后 — 1–2个月内饮用完毕。

**11. 价格与仿冒：**

安化松针 — 限量生产茶：核心产区 — 褒家冲茶场和云台山。三个等级（特级, 一级, 二级）。

* **避免仿冒的方法：**

	* **向可靠商家购买**，最好是褒家冲茶场 (褒家冲茶场) 或「三十九铺」(三十九铺) 品牌产品。
	* **评估外形：** 完美笔直的「针」— 「长、直、圆、滑」。弯曲、折断或不均匀 — 仿冒迹象。
	* **检验「松林」：** 玻璃杯冲泡时针叶应垂直竖立。
	* **评估香气：** 清香、栗香清新，带花果香调。
	* **注意价格：** 过低价格 — 仿冒迹象。

**12. 有趣事实：**

* 安化松针 — 「中国三针」之一 (中国三针)：南京雨花茶（南京）、恩施玉露（湖北）和安化松针（湖南）。三「针」— 三省，三种不同风土和三种完全不同工艺（炒青、蒸青和半烘青）。

* 40分钟手工成形 — 「绝世武功」（「绝世武功」）— 绿茶中最长的手工成形工序之一。比较：龙井揉捻约10分钟，碧螺春成形约15分钟。

* 安化县为世界贡献了两大茶类：安化黑茶 (安化黑茶,「黑茶」— 中国六大基本茶类之一) 和安化松针（绿茶）。同一风土的两个极端 — 从多年发酵到瞬间新鲜。

* 「茶果间作」（茶果间作）系统 — 茶树与枇杷、杨梅混种 — 类似著名的洞庭山碧螺春系统。安化是江苏以外少数使用此方法的地区之一。

* 35–40°C干燥 — 绿茶中最低的最终干燥温度之一。比较：大多数茶类在60–80°C干燥。超低温最大程度保存氨基酸和芳香物质。

**13. 与其他「针形」绿茶比较：**

* **南京雨花茶 (南京雨花茶)：** 来自南京。同样是「针」，同样是「三针」之一。雨花茶 — 更花香，带「革命」象征；松针 — 更栗香果香，带「松」意象。

* **恩施玉露 (恩施玉露)：** 来自湖北。同样是「针」，同样是「三针」之一。玉露 — 蒸青 (蒸青)，带「日式」特色；松针 — 炒青 (炒青)，带湖南栗香。

* **高桥银峰 (高桥银峰)：** 湖南同乡。同样1959年，同样「十周年献礼」。银峰 — 更「银白」和卷曲；松针 — 更「松绿」和笔直。

* **信阳毛尖 (信阳毛尖)：** 来自河南。同样针形，同样栗香。信阳 — 更紧实「北方」，双锅技术；松针 — 更挺秀，40分钟手工成形和果香调。

**结语：**

安化松针 — 形状成为艺术的茶。四十分钟不间断手工搓揉，师傅的「绝世武功」，将嫩绿春芽转化为完美「松针」，— 结果：纤细绿色针叶在玻璃杯中垂直竖立，如同云台山云雾峰顶的微型山林。清香栗果香气（间作遗产），从微苦迅速转甘甜，「三针」身份证 — 这一切使安化松针成为重视视觉完美不亚于味觉体验者的理想茶品 — 准备在品尝第一口前先欣赏杯中「松林」。
