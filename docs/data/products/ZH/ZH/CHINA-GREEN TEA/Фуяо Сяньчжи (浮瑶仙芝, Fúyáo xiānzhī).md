# 浮瑶仙芝 (浮瑶仙芝, Fúyáo xiānzhī)

浮瑶仙芝是来自江西省景德镇市浮梁县的高级绿茶。浮梁是中国茶叶栽培的摇篮之一：唐代时期，这里集中了帝国最大的茶叶市场，提供了全国茶税的八分之三。白居易在《琵琶行》中永恒地记录了浮梁：「商人重利轻别离，前月浮梁买茶去」。「仙芝」之名源于杨贵妃的传说，她将浮梁茶的滋味比作灵芝的香气。如今，浮瑶仙芝是浮梁茶品牌的旗舰产品之一，代表著江西省茶业名片「四绿一红」中的一员。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵茶，绿茶，lǜchá）。按工艺属烘青绿茶，最后经文火轻烤。
* **类别：** 地方名茶（名茶，míngchá）。浮梁茶品牌于2010年获得中国农业部颁发的农产品地理标志认证。浮瑶仙芝是此「伞型」品牌下的主导产品之一；2003年被国务院选为特选礼品茶。
* **产地：** 中国江西省（江西省，Jiāngxī Shěng）景德镇市（景德镇市，Jǐngdézhèn Shì）浮梁县（浮梁县，Fúliáng Xiàn）。茶园主要分布在占全县70%以上的山地和丘陵地区，以瑶里镇、西湖及周边山区为中心。
* **地理座标：** 约北纬29.35°，东经117.23°（以浮梁县为参考）。

**2. 历史与文化意义：**

* **历史：** 浮梁的茶叶栽培始于汉代（西元前2世纪至西元3世纪），当时佛教僧侣开始在山坡上种植茶树。据《中国商业简史》记载，南北朝时期（5-6世纪）「浮梁茶最好」。唐代（7-10世纪）是鼎盛时期：据《元和郡县志》记载，浮梁年茶叶交易量达七百万驮，茶税超过十五万贯，相当于全国茶税收入的八分之三。王敷在《茶酒论》中写道：「浮梁歙州，万国来求」。白居易（772-846）于816年在著名的《琵琶行》中提到浮梁，永远将该县与繁荣的茶叶贸易联系在一起。据宫廷传说，唐玄宗品尝浮梁茶后大为赞赏，杨贵妃说：「有灵芝的香和味，只配神仙所用也」，皇帝遂赐名「仙芝」。唐代浮梁茶中还有「嫩蕊」、「福合」和「禄合」。《牡丹亭》作者汤显祖（1550-1616）评价：「浮梁之茗，冠于天下」。清代，红茶制作技术传入浮梁，该县成为著名的浮梁红茶产地。1915年，严台村的「天祥」茶号在旧金山巴拿马太平洋国际博览会上获得金奖——「茅台镇的酒，严台村的茶」成为流行语。现代，从1991年开始创制浮瑶仙芝（「浮梁」+「瑶里」+「仙芝」的组合名），1992年在农业部展览会上获得「最佳产品」奖。1997年浮梁县获「中国红茶之乡」称号。2003年浮瑶仙芝被国务院选为礼品茶，同年在上海国际茶文化节获金奖。2005年该县成为「全国无公害茶生产示范基地县」；另一种浮梁绿茶「野兰芝」在第五届「中茶杯」获一等奖。2010年浮梁茶获地理标志；至2014年其市场价值评估为9.28亿元。2019年浮梁茶入选「中国农业品牌目录」。

* **名称：** 浮瑶——「浮梁」和「瑶里」（高品质茶叶产地）的组合词。仙芝——「神仙灵芝」：仙（xiān）——「神仙、神圣」；芝（zhī）——「灵芝」（*Ganoderma lucidum*）。名称源于唐玄宗的传说，传达「天赐」、崇高滋味的理念。*注：* 某些资料中出现仙枝（xiānzhī，「神仙枝条」）的写法——这是书写错误；权威资料一致使用仙芝。

* **文化意义：** 浮梁与中国文化的两大象征——茶和瓷器密不可分。历史上的景德镇「世界瓷都」最初属于浮梁；该县理所当然地拥有「瓷都之源，茶国之地」的称号。这种「瓷茶一体」的结合始于汉代。与庐山云雾、婺源绿茶、狗牯脑茶并列为「四绿」——江西茶业的名片。

**3. 植物学描述与原料：**

* **品种：** *Camellia sinensis* var. *sinensis*。
* **品种/栽培种：** 适应浮梁县亚热带丘陵和山麓条件的当地群体种（群体种，qúntǐzhǒng）。大规模生产也使用区域化的无性系品种。
* **采摘：** 主要时期——早春。最珍贵的批次——「谷雨尖」：在谷雨（约4月20日）来临前采摘，此时第一批嫩芽刚刚出现。后续采摘产出「细茶」和「粗茶」。
* **采摘标准：** 一芽一二叶。浮瑶仙芝最高等级——只采嫩芽和第一叶，不含叶柄；标准绿茶允许较成熟的原料。
* **原料要求：** 完整、新鲜采摘、无机械损伤。采摘后去掉叶梗；茶叶迅速送往加工。

**4. 风土与栽培特点：**

* **生长海拔：** 海拔100-800公尺；山地（500公尺以上）占全县面积41.7%，山麓（100-500公尺）占30.6%。最佳茶园位于瑶里镇周围的高山区——浮梁（江西）、婺源（江西）、祁门（安徽）和休宁（安徽）县交界处，海拔600-800公尺。
* **地形：** 丘陵山地：山脉与昌江、东河、西河的河谷交错，这些河流从南向北贯穿全县。森林覆盖率79.4%。
* **气候：** 中亚热带季风气候。年平均气温14-17°C。有效积温5000-6000°C。年平均降雨量1700-1900毫米——中国茶区最高之一，确保充足的水分。无霜期约247天。日照率约45%。相对湿度约79%。
* **微气候：** 频繁的雾气和云雾是浮梁山区茶园的特征。当地谚语说：「晴天早晚遍地雾，阴雨之时满山云」。
* **土壤：** 红壤和黄壤山地土壤，酸性（pH 4.3-5.5），有机质含量极高——达14.5%。这是中国茶区最高的指标之一，确保矿物质丰富和滋味浓郁。
* **生态：** 浮梁县工业污染极少：经济历来以瓷器、茶叶和林业为主，而非重工业。2005年获「全国无公害茶生产示范基地县」地位，2007年获「国家级茶叶标准化示范县」。水资源丰富：昌江、东河、西河三大河流为茶园提供稳定的水分平衡。

**5. 制作工艺：**

浮瑶仙芝是烘青绿茶，结合手工处理和最后的文火轻烤。工艺旨在保持嫩绿、纯净香气和最大甜度。

* **采摘（采摘 — cǎizhāi）：** 早晨手工挑选嫩芽。原料快速送至车间。

* **摊放（摊放 — tānfàng）：** 鲜叶薄摊以平衡水分并启动初步酶促反应，丰富未来的香气。

* **杀青（杀青 — shāqīng）：** 高温炒制以灭活氧化酶。形成香气基础——纯净、新鲜，带轻微花香。

* **摊凉（摊凉 — tānliáng）：** 短暂通风以防止「闷熟」。

* **揉捻（揉捻 — róuniǎn）：** 轻柔揉捻释放细胞汁液，形成紧密、细致的叶形，确保冲泡时充分萃取。

* **初烘（初烘 — chūhōng）：** 适温烘焙，降低叶片水分。

* **摊凉与复揉（摊凉 — 复揉 — fùróu）：** 必要时——额外揉捻以获得更紧的形状（用于标准等级）。

* **文火轻烤（文火轻烤 — wénhuǒ qīngkǎo）：** 最后的细致加热，将水分降至稳定水平（≤6.5%），固定香气并赋予茶汤纯净度。这一步骤是浮瑶仙芝的标志：温和的火候展现细腻的花香和甜味细节，而无「焦味」。

* **拣剔与分级（拣剔 / 分级 — jiǎntī / fēnjí）：** 去除瑕疵叶；按等级分类。

**6. 感官特征：**

* **干茶外观：** 细紧的条索（条索紧细）；颜色——嫩绿色带丰富白毫（白毫显露）。叶片匀整，大小一致，有油润光泽。
* **干茶香气：** 纯净、持久，带细腻花香，类似野生兰花（兰花香，lánhuā xiāng）。背景有轻微甜香。
* **茶汤香气：** 高扬、持久；以兰花香为主，伴有新鲜绿意的细微背景和柔和蜜香。香气被描述为「清香持久」。
* **滋味：** 鲜爽醇正。茶体中等，具明显甜度和圆润感。苦涩度极低。回甘纯净，有持久的回甘和「甘滑」的口感。
* **汤色：** 黄绿色，透明，明亮（嫩绿 / 嫩黄，清澈无浊）。
* **叶底（冲泡后的叶片）：** 嫩叶和芽完整；颜色——浅黄带绿（嫩黄明亮显毫）。茸毛清晰可见。

**7. 化学成分：**

* **茶多酚：** 该地区绿茶典型含量22-30%。主要成分——儿茶素（EGCG、ECG），决定抗氧化活性和轻微涩味。
* **氨基酸：** 含量较高——浮梁茶的特征，得益于高湿度和丰富降雨刺激L-茶氨酸的积累。具体数据因等级而异，但通常高于江西绿茶的平均水平。
* **水浸出物：** 高指标（通常>40%），源于土壤有机质丰富（达14.5%）和充沛降雨（1700-1900毫米/年），确保茶树代谢旺盛和叶片中萃取物质的积累。
* **生物碱：** 咖啡因——该地区绿茶典型含量2.5-4%。可可碱和茶碱——微量。
* **维生素：** 维生素C（抗坏血酸）、B群维生素（B₁、B₂）、类胡萝卜素。
* **矿物质：** 硒——江西南部土壤天然富含（资料中提及）。还有钾、镁、锰、锌、氟。
* **精油和芳香化合物：** 兰花香由芳樟醇、香叶醇和乙酸橙花酯形成；柔和的「蜜香」由辛醛和苯甲醛产生。

**8. 保健功效：**

* **抗氧化保护：** 儿茶素，特别是EGCG，中和自由基并减缓细胞氧化过程。
* **认知支持：** L-茶氨酸和咖啡因的组合提供温和、平稳的提神效果——集中注意力而不紧张。
* **心血管系统支持：** 茶多酚有助于调节胆固醇水平和血压。
* **助消化：** 温和刺激肠道蠕动和酶分泌；非常适合搭配清淡餐食。
* **免疫系统支持：** 硒和维生素C增强身体的抗氧化防御。
* **口腔健康：** 氟和儿茶素抑制致龋细菌的生长。
* **皮肤健康维护：** 抗氧化剂和维生素C有助于防止光老化和胶原蛋白合成。
* **代谢支持：** 茶多酚和咖啡因促进新陈代谢和脂肪氧化，结合均衡饮食可能有助于体重控制。
* **重要提示：** 对咖啡因敏感者应限制下午饮用。不建议空腹饮用。

**9. 冲泡方法：**

* **水温：** 75-85°C。最高等级浮瑶仙芝——75-80°C；标准绿茶——80-85°C。
* **茶量：** 150-200毫升（玻璃杯）用3-4克或100-120毫升（盖碗）用4-5克。
* **茶具：** 透明玻璃杯——观赏茸毫芽叶的「舞蹈」；瓷盖碗——最大程度展现兰花香。特别有韵味的是用景德镇瓷器冲泡：「瓷茶一体」在一杯中相会。
* **步骤：**
  1. 用开水温杯后倒掉。
  2. 投茶。对于嫩茶——采用「上投法」：先注水至杯子七分满，然后轻轻加入茶叶。
  3. 无需洗茶。
  4. 第一泡——浸泡1.5-2分钟（玻璃杯）或20-30秒（盖碗）。
  5. 倒出；玻璃杯——喝掉三分之一时续水。
  6. 重复冲泡：玻璃杯3-4泡，盖碗5-6泡（每次增加5-10秒）。

**10. 储存方法：**

* 存放在密封、不透光的容器中（真空包装袋、锡罐），避光、防潮、隔热和异味。
* 最佳温度——冰箱0-5°C，双重密封。
* 室温储存——凉爽、干燥、阴暗处；开封后2-3个月内饮用。
* 为获得最佳品质——生产后6-12个月内饮用。浮瑶仙芝是新鲜茶；陈放对其无益。

**11. 价格与防伪：**

* **价格区间：** 高级早春浮瑶仙芝——800-3,000元/公斤。标准浮梁绿茶——200-600元/公斤。主要价格因素：采摘季节（谷雨尖最贵）、茶园海拔（瑶里山茶更贵）、等级（芽茶比叶茶贵）和具体子品牌。

* **如何避免假货：**
  * 向有权使用浮梁茶地理标志品牌的认证企业购买。
  * 评估香气：正宗浮瑶仙芝具有天然兰花香，无人工香精。
  * 检查外观：细紧条索带丰富白毫；不匀整、暗淡的叶片是假冒的迹象。
  * 评估茶汤：透明、黄绿色、无浑浊。
  * 价格过低令人怀疑，特别是声称「礼品」等级时。

**12. 趣闻轶事：**

* 浮梁是两大世界珍宝的故乡：瓷器和茶叶。历史上的「瓷茶一体」早在汉代就在这里诞生：僧侣们同时烧制陶瓷和栽培茶树。

* 白居易《琵琶行》中的诗句——「商人前往浮梁买茶」——进入中国文学的黄金宝库，成为浮梁茶业的非官方「口号」，而现代的「浮梁买茶节」直接呼应这首诗。

* 唐代浮梁茶税占全国的八分之三——这一比例没有任何其他县达到过，使浮梁成为中世纪中国最大的茶叶市场。

* 浮梁是两个伟大茶县的「母亲」：740年从浮梁（当时称新昌）分出婺源，766年分出祁门。婺源以中国最佳出口绿茶闻名，祁门是「十大名茶」之一祁门红茶的故乡。因此，浮梁是两个茶叶传奇的共同历史根源。

* 1915年巴拿马太平洋博览会上，严台村的浮梁红茶与茅台酒并列获得金奖——由此产生「茅台镇的酒，严台村的茶」的说法。严台和沧溪村的古老茶号建筑保存至今，是茶叶贸易的历史见证。

* 生产浮瑶仙芝的瑶里镇同时也是景德镇瓷器的摇篮：正是从这里开采的高岭土，给世界带来了「china」这个词。今天瑶里是热门旅游路线，结合了瓷器和茶叶遗产。

* 汤显祖（1550-1616）——中国最伟大的剧作家，「中国的莎士比亚」——在关于浮梁学派的文章中写道：「浮梁之茗，冠于天下，帷清帷馨」，记录了该地区在16-17世纪之交的声誉。

**13. 与其他绿茶的比较：**

* **瑶里崖玉（瑶里崖玉，Yáolǐ Yáyù）：** 另一种高级浮梁绿茶，产于瑶里海拔600-1000公尺的山地茶园。崖玉更加细腻，带银色茸毫和细致香气；浮瑶仙芝稍微厚实和「圆润」。

* **庐山云雾（庐山云雾，Lúshān Yúnwù）：** 「中国十大名茶」之一，与浮瑶同为江西茶。云雾卷曲，叶片厚实多毫，具强烈「山韵」；仙芝更细腻，兰花香更明显，茶体更柔和。

* **婺源绿茶（婺源绿茶，Wùyuán Lǜchá）：** 浮梁茶的历史「兄弟」（婺源740年前属于浮梁）。婺源绿茶以极高的水溶性儿茶素和浓郁萃取著称；浮瑶仙芝更嫩更花香。

* **黄山毛峰（黄山毛峰，Huángshān Máo Fēng）：** 邻近安徽的绿茶，风土相似（江西安徽交界的山区）。毛峰更多毫，具更细腻的「铃兰」香；仙芝稍厚实「圆润」，具兰花香。

* **靖安白茶（靖安白茶，Jìng'ān Báichá）：** 江西邻县靖安的独特「白叶」绿茶，由白化品种制成。白茶明显更轻盈「清凉」，具明显鲜味；浮瑶仙芝更饱满，花香更发达，甜度「温暖」。

浮瑶仙芝是一款血统深入中国历史深处的茶：唐代商人就来浮梁「买茶」，从这里出发的商队以茶税供养帝国。这款绿茶吸收了江西千年雨水的滋润、红色山地土壤的酸性和雾霭峡谷的纯净空气。今天的浮瑶仙芝不追求响亮的纪录——它以其他特质取胜：兰花香气、纯净甜度、柔和茶体，以及千年前杨贵妃察觉到的那种难以捉摸的「灵芝」感觉。这是适合悠闲早晨和沉思傍晚的茶，为那些珍视细腻与深度的人而生。用景德镇瓷器冲泡它——浮梁的两大恩赐将在您的杯中相遇。
