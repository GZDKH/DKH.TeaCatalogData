# 岳西翠兰 (岳西翠兰, Yuèxī cuì lán)

**岳西翠兰 (岳西翠兰, Yuèxī cuì lán)**
![Юэси Цуй Лань (岳西翠兰, Yuèxī cuì lán)](images/Юэси%20Цуй%20Лань%20(岳西翠兰,%20Yuèxī%20Cuì%20Lán).png)

岳西翠兰 (岳西翠兰, Yuèxī cuì lán —「岳西的翠绿兰花」) — 安徽绿茶，拥有创纪录的高胺基酸含量 (≥6.3%) 和创纪录的低多酚含量 (≤19.5%) — 此比例确保了卓越的甘甜和柔和，在中国绿茶中几乎无与伦比。此茶创制于1980年代初期，位于大别山 (大别山, Dàbié Shān) 山脉的心脏地带 — 长江和淮河流域的主要分水岭 — 并于1985年入选「全国新创十大名茶」(全国新创十大名茶)。在2010–2012年间四次作为**国宾礼茶馈赠外国元首** (国宾礼茶馈赠外国元首) — 最高外交认可。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。按形状 — 「展开的花蕾」：芽叶相连的嫩芽，冲泡时展开成「兰花朵」(芽叶相连舒展成朵，形似兰花)。按工艺 — 炒青烘干配合炭火烘焙。

* **类别：** 国家地理标志产品 (国家地理标志产品, 2013年)。「全国新创十大名茶」之一 (1985年)。四次「国宾礼茶」(2010–2012)。农业展览会金奖 (1996)。2020年 — 安徽省非物质文化遗产。品牌价值 — 196.3亿元人民币 (2024)。

* **产地：** 中国安徽省 (安徽, Ānhuī) 岳西县 (岳西县, Yuèxī Xiàn) — 位于大别山山脉的正中心。地理标志区域 — 该县24个乡镇和182个行政村。三个核心微区：**姚河香炉冲 (姚河香炉冲)** (姚河、头陀、黄尾乡镇)；**包家石佛寺 (包家石佛寺)** (包家、古坊、河图乡镇)；**田头闵山 (田头闵山)** (田头、五河、菖蒲乡镇)。

* **地理座标：** 约北纬30°50′，东经116°20′。

**2. 历史与文化意义：**

* **历史：** 岳西翠兰 — 年轻的茶品，但声誉迅速攀升。1980年代初期，茶艺师**程东明 (程东明)** 和**查道生 (查道生)** 基于传统的「小兰花」工艺 (小兰花茶, xiǎo lánhuā chá) — 安徽绿茶的历史风格 — 在两个核心区域创制了新型茶品：包家乡的石佛寺 (石佛寺,「石佛寺」) 和姚河乡的竹山 (竹山)。

	1985年在安庆地区竞赛中，两个版本分别获得第一和第二名。同年，此茶获得全国认可 — 成为「全国新创十大名茶」之一。

	随后迅速崛起：1996年 — 农业展览会金奖；2010–2012年 — **四次作为国宾礼茶赠送外国领导人**；2013年 — 地理标志保护；2020年 — 列入安徽非物质文化遗产名录。到2024年 — 18万亩（1.2万公顷）茶园，总产值 — 20亿元人民币，品牌价值 — 196.3亿元人民币。

* **名称：**

	* 「岳西」(岳西) — 「岳山之西」：大别山中岳山（大岳, Dayu）以西的县名。
	* 「翠」(翠) — 「翠绿」：茶叶的颜色。
	* 「兰」(兰) — 「兰花」：既指香气 (兰花香)，也指形状 — 展开的嫩芽如兰花朵。

* **文化意义：** 岳西翠兰 — 山区岳西县和整个皖西地区的骄傲。四次外交使用 — 在「新」绿茶中创下绝对纪录。此茶体现了「三绿」(三绿) 哲学：绿叶、绿汤、绿底 — 视觉纯净，成为该地区的标志。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要 — **地方群体种 (地方群体种, dìfāng qúntǐzhǒng)** — 本土小叶种 *Camellia sinensis* var. *sinensis* 有性繁殖品种，以显著的兰花香 (兰花香显著) 著称。辅助品种：**石佛翠 (石佛翠, Shífócuì)**、**舒茶早 (舒茶早, Shūchá Zǎo)**、**龙井43 (龙井43)**、**乌牛早 (乌牛早)** — 无性系品种，提高产量和稳定性。春季原料化学成分：**胺基酸 ≥6.3%，多酚 ≤19.5%** — 绿茶中最有利的比例之一；维生素C — **121毫克/100克** — 比平均值高一倍。

* **采摘：** 早春采摘。三个等级：
	* **翠芽/特级 (翠芽/特级)：** 纯芽头 (单芽, ≥95%)。500克需4万个芽头。挺直、秀美，茸毛丰富。
	* **翠尖/一级 (翠尖/一级)：** 一芽一叶 (≥90%)。卷曲「花蕾」形状。持久纯净香气。
	* **翠兰/二级 (翠兰/二级)：** 一芽二叶。「展开花蕾」。柔和持久口感。

**4. 风土与栽培特色：**

* **位置：** 大别山腹地 (大别山腹地) — 长江和淮河流域的主要分水岭，华东地区最生态纯净的区域之一。

* **海拔：** 400–1000公尺。

* **气候：** 年平均气温 — 16.4°C，相对湿度 — ≥77%，年平均雾日 — **>200天**。日夜温差 — 显著。丰富的漫射光 (漫射光) — 创纪录胺基酸含量的关键因素。

* **土壤：** 红黄壤 (红黄壤)，pH 4.5–6.5，有机质含量高，天然富含**硒和锌**。森林覆盖率 — **74%**。核心区域 — **水源保护区** (水源保护区)：**禁止**使用化学肥料和农药。

* **创纪录的化学成分：** 胺基酸 ≥6.3%，多酚 ≤19.5% — 这种「酚氨比」(酚氨比, fēn'ān bǐ) 约为3:1，而大多数绿茶为5–8:1。结果：最小的苦涩味，最大的甘甜和鲜爽。

**5. 制作工艺：**

岳西翠兰工艺 — 炒青烘干配合炭火烘焙，完全不使用金属工具。

* **摊放 (鲜叶摊放 — xiānyè tānfàng)：** 短时间摊放。

* **杀青 (杀青 — shāqīng)：** **竹帚抛炒** (竹帚抛炒, zhúzhǒu pāochǎo) 在240°C。使用竹制工具而非金属 — 原则性选择：竹子不催化多酚氧化，保持最大鲜度和绿色。

* **手工理条 (手工理条 — shǒugōng lǐtiáo)：** 手工整理和塑形嫩芽。

* **初烘/炭烘 (毛火/炭烘 — máohuǒ / tànhōng)：** 在80°C用木炭烘焙。

* **复摊 (复摊 — fùtān)：** 冷却和重新分布水分。

* **足火/炭烘 (足火/炭烘 — zúhuǒ / tànhōng)：** 在60°C — 缓慢「暗火」烘焙 (暗火慢烘)。正是炭火烘焙形成特有的「冷韵」(冷韵, lěngyùn) — 电烘无法重现的细腻韵味。

* **特色：** 全程 — 竹木器具 (全程竹木器具避金属氧化)。任何阶段茶叶都不接触金属。

**6. 感官特征：**

「三绿」(三绿, sān lǜ)：绿色干茶、绿色茶汤、绿色叶底 — 名片特色。

* **干茶外观：** 芽叶相连 (芽叶相连)，覆盖翠绿茸毛 (翠绿显毫)。冲泡时展开成「花蕾」，如兰花朵 (舒展成朵，形似兰花)。

* **干茶香气：** 清香高雅 (清香)。兰花香 (兰花香, lánhuā xiāng) — 来自本土原生栽培种的招牌香气。嫩栗香 (嫩栗香) — 来自春季采摘和炭火烘焙。

* **茶汤香气：** 兰花香，持久优雅。「冷韵」(冷韵) — 炭火烘焙的细腻韵味。

* **口感：** 鲜爽 (鲜爽, xiānshuǎng) — 创纪录的胺基酸含量 (≥6.3%)。醇厚。回甘绵长 — 持久「丝滑」(回甘绵长)。**苦涩度极低** (苦涩度低) — 创纪录低多酚含量的结果 (≤19.5%)。

* **茶汤颜色：** 碧绿明亮 (碧绿明亮)。

* **叶底：** 嫩绿匀整，「展开成朵」(嫩绿匀整成朵)。

**7. 化学成分：**

200+雾日、水源保护无化学品和大别山山地微气候创造独特的「超柔和」特性：

* **胺基酸：** **≥6.3%** — 中国绿茶中最高指标之一。对比：大多数著名绿茶 — 3–5%。创纪录的胺基酸水平确保卓越的鲜爽和「鲜味」。

* **多酚（儿茶素）：** **≤19.5%** — 显著低于平均值（通常20–30%）。正是低多酚含量解释了无苦涩味的特点。

* **维生素C：** **121毫克/100克** — 比绿茶平均值高一倍。结果：强大的抗氧化保护。

* **生物碱：** 咖啡因 — 2.8–3.5%。提神效果 — 比平均值高30%。

* **硒和锌：** 天然高含量 — 来自大别山山地土壤。

* **矿物质：** 钾、镁、锰、磷。

**8. 健康功效：**

* **强大抗氧化作用：** 维生素C（121毫克/100克 — ×2）结合儿茶素。
* **利尿解毒作用 (清热利尿)：** 儿茶素加速新陈代谢和水分排出。
* **提神醒脑效果 (提神醒脑)：** 咖啡因（2.8–3.5%）— 比平均值高30%。
* **改善消化：** 多酚刺激酶活性。
* **增强免疫力：** 硒、锌、维生素C。

* **重要提醒：** 上述功效基于公开资料，不构成医疗建议。

**9. 冲泡方法：**

* **玻璃杯法 (上投法)：**
	* 水温：85–90°C。注水至7/10，投入3克茶叶。
	* 浸泡2–3分钟。观察「兰花花蕾」的展开。

* **盖碗法：**
	* 洗茶 — 5秒。首泡 — 20秒。
	* 茶叶可冲泡**4–6次**。

* **注意：** 不要使用沸水（>90°C）— 破坏胺基酸并产生黄色。开封后 — 7天内饮用以获得最佳香气。

**10. 储存：**

* 密封，置于阴凉干燥处。冰箱0–5°C。
* 新购茶叶 — 放置15天「退火」。
* 保存期限 — 最多12个月。开封后 — 7天内饮用以获得最佳香气。

**11. 价格与仿冒：**

三个等级：翠芽/特级（600元人民币/斤起），翠尖/一级，翠兰/二级。

* **避免仿冒：** 购买带有岳西县GI标识的产品；评估冲泡时的「兰花花蕾」；检查兰花香气；价格过低 — 仿冒迹象。

**12. 有趣事实：**

* 四次作为国宾礼茶赠送外国领导人（2010–2012）— 在「新」（1949年后创制）绿茶中创下绝对纪录。

* 胺基酸 ≥6.3%，多酚 ≤19.5% — 「酚氨比」~3:1 对比通常的5–8:1。这使岳西翠兰成为世界上最「甘甜」和最少「涩味」的绿茶之一。

* 维生素C — 121毫克/100克 — **比平均值高一倍**。在此指标上，岳西翠兰是绿茶中的「冠军」之一。

* 完全不使用金属工具 (全程竹木器具) 和炭火烘焙的「冷韵」(冷韵) — 罕见组合，赋予茶叶独特品格。

* 核心区域 — 水源保护区，禁止化学肥料和农药。茶叶 — 事实上的有机茶，无需正式认证。

**13. 与其他安徽绿茶的比较：**

* **黄山毛峰 (黄山毛峰)：** 烘青，「雀舌」，兰花栗香。毛峰 — 更「优雅」；翠兰 — 更「甘甜」柔和（胺基酸6.3% vs. ~3–5%）。

* **六安瓜片 (六安瓜片)：** 纯叶片，无芽头。瓜片 — 更草本浓郁；翠兰 — 更嫩滑「兰香」。

* **泾县特尖 (泾县特尖)：** 安徽同乡。特尖 — 更栗香矿物，富硒纪录；翠兰 — 更兰香甘甜，胺基酸纪录。

* **安吉白茶 (安吉白茶)：** 来自浙江。同样创纪录胺基酸（6–7%），同样最小苦味。安吉 — 「纯甘甜」无茸毛；翠兰 — 「兰香甘甜」有茸毛和「花蕾」形状。

**总结：**

岳西翠兰 — 追求绝对柔和的茶品。创纪录胺基酸（≥6.3%），创纪录低多酚（≤19.5%），双倍维生素C，本土原生栽培种的兰花香，炭火烘焙的「冷韵」和在翠绿茶汤中展开的「兰花花蕾」，— 这一切使「翠绿兰花」成为那些视苦涩为敌、以甘甜鲜爽为绿茶唯一美德的人的理想茶品。四次赠送国家元首 — 实至名归：这朵「兰花」中没有任何粗糙音符。
