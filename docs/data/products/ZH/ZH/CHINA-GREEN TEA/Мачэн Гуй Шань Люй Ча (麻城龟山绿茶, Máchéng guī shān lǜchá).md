# 麻城龟山绿茶 (麻城龟山绿茶, Máchéng guī shān lǜchá)

**麻城龟山绿茶 (麻城龟山绿茶, Máchéng guī shān lǜchá)**

麻城龟山绿茶 (麻城龟山绿茶, Máchéng guī shān lǜchá) —「麻城市龟山绿茶」— 产自龟峰山 (龟峰山, Guīfēngshān,「龟峰」, 1320公尺) 的绿茶，该山位于大别山 (大别山) 山脉，隶属湖北省麻城市 (麻城市)。此茶早在陆羽 (陆羽, Lù Yǔ) 的《茶经》(《茶经》, Chájīng) 中即有记载：「黄州山谷茶生麻城县」—「黄州山谷茶产于麻城县」。唐代太宗皇帝 (太宗, Tàizōng, 李世民, Lǐ Shìmín) 曾以诗赞美此茶：「龟涎煮龟茶，天下第一家」—「以龟山泉水冲泡龟茶，天下第一家」。西元1959年，基于历史名茶「龟山云雾」(龟山云雾,「龟山云雾」) 创制了「龟山岩绿」(龟山岩绿,「龟山岩绿」)。西元1962年，董必武 (董必武, Dǒng Bìwǔ, 1886–1975) — 中国共产党创始人之一及中华人民共和国副主席 — 为此茶赋诗赞美。西元1980年此茶被列入「全国五大名茶」(全国五大名茶)，西元2012年获得中华人民共和国地理标志产品地位。

**1. 分类与产地：**

* **类型：** 绿茶 (绿茶, lǜchá)，未发酵茶。属于条形绿茶 (条形绿茶, tiáoxíng lǜchá)，采用炒青工艺杀青。茶叶外形为紧细圆直，锋毫显露 (紧细圆直、锋毫显露)。独特工艺为「小火长炒配合旺火提香」(小火长炒+旺火提香, xiǎohuǒ chángchǎo + wànghuǒ tíxiāng)。

* **等级：** 中华人民共和国地理标志产品 (国家地理标志产品, 2012年)。全国五大名茶之一 (全国五大名茶, 1980年)。农牧渔业部优质奖得主 (农牧渔业部优质奖, 1988年)。收录于《中国名茶研究选集》(《中国名茶研究选集》)。唐代贡茶 (唐代贡茶)。宋代御品 (宋代御品)。陆羽《茶经》记载。截至2024年，茶园面积达35,000亩（约2,333公顷），年产量150吨，总产值6,000万人民币。

* **产地：** 中国湖北省 (湖北省, Húběi Shěng) 黄冈市 (黄冈市, Huánggāng Shì) 麻城市 (麻城市, Máchéng Shì)。大别山山脉 (大别山)，龟峰山 (龟峰山, 1320公尺)。产区包括五个茶区：龟山茶场 (龟山茶场,「龟山茶场」)、龟山镇 (龟山镇)、三河口镇 (三河口镇)、木子店镇 (木子店镇)、张家畈镇 (张家畈镇)。

* **地理座标：** 约北纬31°10′，东经115°10′。

**2. 历史与文化意义：**

* **历史：**

麻城的茶叶历史是湖北省有文献记载最古老的历史之一。最早的文字记载来自陆羽 (陆羽, 733–804年)，他在《茶经》(《茶经》) 中记载：「黄州山谷茶生麻城县」—「黄州山谷茶产于麻城县」。这一直接记载使麻城成为「茶圣」在其经典著作中个别提及的少数地区之一。同样在唐代 (618–907年)，唐太宗 (太宗, 李世民) 曾游览龟峰山并留下诗句赞美：「龟涎煮龟茶，天下第一家」—「以龟山泉水冲泡龟茶，天下第一家」。「龟涎」(龟涎) 的意象是对从龟峰山岩石中涌出的最纯净泉水的比喻。

宋代 (960–1279年)，麻城茶叶由专门的国家机构榷茶使司 (榷茶使司, Quèchá Shǐsī,「榷茶使司」) 管理，负责向朝廷供应。宋史记载《宋史·食货志》(《宋史·食货志》) 证实「海州榷茶之所…茶善而易售」— 海州茶叶 (麻城作为更广泛贸易网络的一部分) 备受推崇且「易于销售」。茶叶以「御品」(御品,「御品」) 地位供应朝廷。

西元1958年，在龟峰山坡建立了国营龟山茶场 (国营龟山茶场)。其技术人员基于「龟山云雾」(龟山云雾,「龟山云雾」) 的传统制作工艺，于1959年开发了改良茶品「龟山岩绿」(龟山岩绿,「龟山岩绿」)，该茶在1960年代初即跻身湖北省四大名茶。

西元1962年，中华人民共和国副主席董必武 (董必武, 1886–1975年)，中国共产党创始人之一，游览龟峰山并以诗歌赞美此茶：「昔日游击地，今为产茶区。龟峰名久著，牯岭德不孤…」—「昔日游击地，今为产茶区。龟峰名久著，牯岭德不孤…」。董必武称龟峰山为「第二庐山」(第二庐山)，为茶叶赋予了「红色」光环，将革命历史与茶叶历史交织在一起。

西元1980年，在全国性评比中，此茶被列入「全国五大名茶」(全国五大名茶) — 当时的最高国家荣誉。西元1988年获得「农牧渔业部优质奖」(农牧渔业部优质奖)。西元2005年在第六届「中茶杯」全国名优茶评比中荣获一等奖 (第六届"中茶杯"全国名优茶评比一等奖)。西元2012年，国家质量监督检验检疫总局批准「龟山岩绿」(龟山岩绿) 地理标志。截至2024年，「龟山岩绿」系列产品共获得19项国家级和国际级奖项。

* **名称：**

「麻城」(麻城) — 城市名称，可追溯至南北朝时期 (420–589年)。「龟山」(龟山) —「龟山」，龟峰山 (龟峰山) 的简称；该山因其形状酷似一只长达16公里的巨龟而得名 — 从「头部」到「尾部」。「绿茶」(绿茶) —「绿茶」。

* **文化意义：**

龟峰山是麻城的名片之一，因其形状被誉为「天下第一龟」(天下第一龟)。该山是大别山世界地质公园 (大别山世界地质公园) 的一部分，并于2024年获得国家5A级旅游景区地位。龟 (龟, guī) 在中国文化中是四灵 (四灵, sìlíng) 之一，象征长寿、智慧和稳定。生长在「龟背」上的茶叶，每片叶子都承载著这种象征意义。太宗的诗意表达 —「龟涎 + 龟茶 = 天下第一」— 是古代「茶叶命名」的完美范例，将地名、神话和感官体验融为一体。此外，龟峰山是重要的「红色旅游」景点：在内战期间，该地区是游击队基地，董必武的诗歌将茶叶转化为象征 — 从革命过去到和平茶业的传承。

**3. 植物学描述与原料：**

* **品种/栽培种：** 原料基础主要由当地群体栽培种**龟茶一号** (龟茶一号, Guīchá Yīhào) 和**龟茶二号** (龟茶二号, Guīchá Èrhào) 组成 — 中小叶种 (*Camellia sinensis* var. *sinensis*)，具有高抗寒性和肥厚芽头，具有显著的「持嫩性强」(持嫩性强)。龟峰山坡上还保存著树龄超过100年的野生古茶树。鲜叶生化成分：多酚类 — 17.69%，氨基酸 — 4.36%（比湖北平原绿茶高约20%）。

* **采摘：** 主要季节为春季。特级在清明 (清明, Qīngmíng, 约4月5日) 前采摘；一级在谷雨 (谷雨, Gǔyǔ, 约4月20日) 前采摘。特级采摘标准为单芽或一芽一叶初展。

* **等级：**
	* **特级 (特级, tèjí)：** 单芽或一芽一叶，清明前采摘。茸毛丰富。栗香带嫩香 (嫩香)。价格 — 每500克450元人民币起。
	* **一级 (一级, yījí)：** 一芽一叶，谷雨前采摘。价格 — 每500克200–300元人民币。
	* **二级 (二级, èrjí)：** 一芽二叶。大众等级，也用于包装产品。

* **核心产区：** 海拔800公尺以上的罗百川村 (罗百川村)、龟尾村 (龟尾村,「龟尾村」) 和柿饼山村 (柿饼山村,「柿饼山村」)。这些村庄产出高达70%的特级茶。

**4. 风土条件与栽培特点：**

* **气候：** 该地区位于湖北、河南、安徽三省交界处，大别山脉中部。气候为北亚热带与温带过渡性气候。年平均气温16°C。年降水量1200–1300毫米。云雾天数超过180天。散射光比例超过70%。日夜温差超过8°C，有利于氨基酸积累并减缓L-茶氨酸分解，确保茶叶高度「鲜爽」(鲜爽, xiānshuǎng)。

* **海拔：** 茶园位于海拔600–1000公尺。核心产区海拔800公尺以上。山脉最高点为1320公尺（险刀峰, 险刀峰）。

* **土壤：** 黄棕壤 (黄棕壤, huángzōngrǎng)，pH值4.0–6.5。有机质含量≥1%。土壤富含铁 (Fe) 和锌 (Zn)，有利于多酚类生物合成。土层厚度不少于60公分。

* **生态环境：** 龟峰山森林覆盖率95%（中国茶区中最高之一）。负离子浓度高达每立方公分200,000个。水资源为一级清洁；举水 (举水) 和巴水 (巴水) 河流源头直接位于茶园区域。龟峰山拥有国家生态旅游示范区 (国家生态旅游示范区) 地位。

**5. 制作工艺：**

独特工艺「小火长炒 + 旺火提香」(xiǎohuǒ chángchǎo + wànghuǒ tíxiāng) —「小火长炒 + 旺火提香」— 双阶段干燥，是制作的特色：

* **摊放 (摊放, tānfàng)：** 室温20–25°C下摊放3–4小时。摊放厚度不超过5公分。叶片失去多余水分（含水量降至60–65%）并开始散发「草青」香气。

* **杀青 —「杀青」(杀青, shāqīng)：** 锅温120–150°C。温和炒制，钝化氧化酶但保持叶片嫩度。通过颜色变化和触感弹性手工控制。

* **揉捻 (揉捻, róuniǎn)：** 按「轻→重→轻」(轻→重→轻) 原则。目的是破坏细胞结构以便后续浸出，但不损害茶叶完整性。

* **初干 (初干, chūgān)：** 80–120°C。将湿度降至中间水平。

* **手工整形 (手工整形, shǒugōng zhěngxíng)：** 「掌心搓揉」(掌心搓揉, zhǎngxīn cuōróu) 方法 — 师傅纯用手掌形成紧细圆直的条索。标准严格禁止机械压制 — 只有手工操作才能确保特有的「紧细圆直」(紧细圆直) 结构。

* **「小火长炒」(小火长炒, xiǎohuǒ chángchǎo)：** 温度60°C。缓慢长时间炒制直至水分降至约20%。此阶段形成栗香的「主体」。

* **「旺火提香」(旺火提香, wànghuǒ tíxiāng)：** 温度120°C。快速最终加热，将栗香「锁定」在茶叶内部。缓慢积累香气化合物与高温瞬间固定之间的对比 — 是区别龟山岩绿与其他湖北绿茶的关键技术特色。

**6. 感官特征：**

* **干茶外观：** 紧细圆直的条索 (紧细圆直)，银白茸毫显露 (锋毫显露)。色泽翠绿油润 (翠绿油润)。茶叶大小均匀，无碎片和「茶末」。

* **干茶香气：** 清纯高雅，具有明显的栗香 (栗香, lìxiāng)。特级茶另有「嫩香」(嫩香, nènxiāng) — 鲜嫩绿叶和嫩玉米的精致香调。

* **茶汤香气：** 栗香持久 (栗香持久)，为主要特征。春茶具有「清香」(清香, qīngxiāng)，带淡雅花香。茶杯冷却后香气保持10分钟以上 — 显示挥发性香气化合物含量高。

* **滋味：** 醇厚 (醇厚, chúnhòu) 且鲜爽 (鲜爽, xiānshuǎng)。回甘 (回甘, huígān) 明显持久。特有的滋味公式为「涩中泛甜」(涩中泛甜, sè zhōng fàn tián)：初始轻微涩味转化为纯净矿物甜味。余韵悠长，带有「栗香回响」。

* **汤色：** 碧绿清亮 (碧绿清亮, bìlǜ qīngliàng)。透明度高，无浑浊和沉淀。

* **叶底（茶渣）：** 黄绿嫩匀 (黄绿嫩匀)。叶片在3–4次冲泡后仍保持弹性和韧性，证明耐泡性强。

**7. 化学成分：**

* **多酚类 (茶多酚)：** 17.69% — 绿茶的中等水平，低于平均水平（20–30%），这是由于高山产地和散射光比例高所致。多酚类含量低意味著苦涩味较少。

* **氨基酸 (氨基酸)：** 4.36% — 比湖北平原绿茶高约20%。氨基酸含量高，主要是L-茶氨酸，确保明显的「鲜爽」(鲜爽) 和茶汤甜味。多酚类/氨基酸比例约为4:1 — 「醇厚鲜爽」(醇厚鲜爽) 滋味特征的最佳比例。

* **咖啡碱 (咖啡碱)：** 约4.5% — 高于绿茶平均水平（典型水平2–4%）。提供增强的提神效果。

* **水浸出物 (水浸出物)：** 39.93% — 高水平（国标≥39%），证明内含物质丰富。

* **可溶性糖 (可溶性总糖)：** 2.65% — 有助于「回甘」和滋味圆润。

* **维生素：** 维生素C、B族维生素（B₁、B₂）、维生素E — 优质绿茶的典型组合。

* **矿物质：** Fe、Zn、K、Mg、Mn。锌 (Zn) 含量较高 — 大别山山地土壤的特征。

**8. 保健功效：**

* **抗氧化作用：** 儿茶素（EGCG、ECG）和维生素C共同中和自由基，延缓细胞老化。

* **增强提神效果：** 咖啡碱（约4.5%）与L-茶氨酸结合产生「温和提神」— 提高专注力而无剧烈神经兴奋。

* **支持肌肉恢复：** 咖啡碱加速运动后肌肉中乳酸的排出，促进更快恢复。

* **支持心血管系统：** 多酚类有助于降低「坏」胆固醇（LDL）水平并维持血管弹性。

* **助消化：** 单宁和可溶性糖刺激消化酶分泌并调节肠道蠕动。

* **抗菌作用：** 儿茶素对广谱病原菌具有抑菌活性。

* **认知支持：** L-茶氨酸促进大脑α波产生，改善注意力集中和工作记忆。

* **增强免疫力：** 维生素C、锌和多酚类复合物支持机体免疫反应。

**9. 冲泡方法：**

* **水温：** 一级和二级茶85–90°C；特级茶80°C（更精细的原料需要较低温度以防止苦涩）。

* **茶量：** 3克茶配150毫升水（比例1:50）。

* **茶具：** 玻璃杯 (玻璃杯) — 观察茶叶「舞蹈」；盖碗 (盖碗) 容量120–150毫升；瓷壶 — 日常冲泡。

* **过程：**
	1. 用热水温润茶具，倒掉。
	2. 投入3克茶叶。
	3. 采用「中投法」(中投法, zhōngtóufǎ)：注入⅓容量的水，让茶叶「苏醒」1分钟。
	4. 加水至容器7/10容量。
	5. 第一泡 — 浸泡1–2分钟。
	6. 重复冲泡 — 3–4次。每次增加15–20秒浸泡时间。建议每次加水时保留¼茶汤在容器中。

**10. 储存：**

* **包装：** 铝箔真空密封包装 — 最佳选择。可用密封盖铁罐储存。
* **温度：** 冰箱，0–5°C。长期储存 — 冷冻库（−18°C）不透明密封包装。
* **保质期：** 适当条件下12个月。开封后建议1个月内饮用。
* **茶叶天敌：** 湿气、光线、异味、高温。不要与强香味产品（香料、咖啡）一起储存。

**11. 价格与仿冒品：**

* **价格范围：** 特级（特级，明前茶）— 每500克450元人民币起。一级 — 每500克200–300元人民币。二级 — 每500克80–150元人民币。

* **价格因素：** 采摘季节（明前茶最贵）、原料等级、生长海拔（800公尺以上村庄的茶最珍贵）、手工vs机械加工。

* **避免仿冒品：**
	* 购买带有地理标志标识「麻城龟山绿茶」或「龟山岩绿」的茶叶 — 受法律保护的产品。
	* 评估外观：正宗龟山岩绿为细直均匀的条索带茸毫，无碎片。仿冒品通常看起来松散不均匀。
	* 检查香气：真正的栗香持久纯正，无「焦糊」或「草青」味。香气微弱或异味 — 仿冒品。
	* 评估茶汤：正宗茶汤呈碧绿清亮。浑浊或黄褐色茶汤 — 劣质原料的标志。
	* 对可疑低价保持警惕：特级茶低于每500克300元人民币 — 值得怀疑。

**12. 有趣事实：**

* **陆羽点名麻城。**「黄州山谷茶生麻城县」— 《茶经》中对麻城的直接记载。这是少数获得「茶圣」个别提及的城市之一，证明早在8世纪茶叶品质就已卓越。

* **皇帝与「龟涎」。** 太宗的公式「龟涎煮龟茶，天下第一家」— 古代中国「茶叶行销」的范例，将地名（龟山）、动物学比喻（泉水如「龟涎」）和评价判断（「天下第一」）融合在一个对句中。

* **革命家的诗歌。** 董必武不仅是政治家，还是中共一大13名代表之一（1921年）。他1962年在龟峰山的诗意造访将茶叶与大别山的「红色」历史联系起来 — 该地区培养了36位将军，包括徐向前元帅和麻城籍王树声将军。

* **1980年「五大名茶」。** 入选「全国五大名茶」— 在中国数千种绿茶中只有少数获得的最高国家认可。

* **龟山与千万杜鹃。** 龟峰山不仅是茶山，也是世界最大古杜鹃花群落 (杜鹃花) 的生长地：1000万株，树龄估计数十万年。每年春天山坡被火红花毯覆盖 — 这一景观为麻城带来第二个品牌：「人间四月天，麻城看杜鹃」(人间四月天，麻城看杜鹃)。

* **双阶段火候。** 工艺「60°C（缓慢，至20%水分）→ 120°C（快速，锁定栗香）」— 龟山茶场师傅的独创技术，在其他湖北绿茶中没有直接类似工艺。

* **宋代国家专卖。** 麻城茶是少数历史包含「榷茶使司」(榷茶使司) — 国家茶叶贸易专卖管理机构的茶叶之一，证明该产品在帝制时代的卓越价值。

**13. 与其他绿茶比较：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 同样产于湖北，但采用日式蒸青 (蒸青) 而非炒青。滋味更「绿」，带鲜味；香气海洋性，「海藻味」。龟山岩绿为炒青茶，栗香明显，茶体更厚重。

* **信阳毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 产于邻省河南，同样在大别山脉。相似风土，但揉捻和干燥工艺不同。毛尖更「尖锐」，茸毫明显，香气带鲜绿调。龟山更「圆润」且栗香突出。

* **庐山云雾 (庐山云雾, Lúshān Yúnwù)：** 「庐山云雾茶」— 邻省江西名茶。董必武曾将龟峰山比作庐山。两者都是高山「云雾」茶，但庐山云雾具有更明显的「豆香」(豆香)，而龟山为栗香。

* **黄山毛峰 (黄山毛峰, Huángshān Máofēng)：** 安徽茶，同样产于大别山系统（东延部分）。相当的海拔和云雾。毛峰更清淡，花草香气带「兰花」调。龟山更厚重，栗香基调更明显。

* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 安徽大叶茶。形状根本不同（大型扁平叶片），但相似的风土原理 — 高山云雾，酸性土壤。猴魁「兰花香」，优雅；龟山更「厚实」，结构明显。

**结论：**

麻城龟山绿茶 — 陆羽提及其名，太宗皇帝赞美其味，革命家董必武确立其声誉，「1980年五大名茶」确认其地位的茶叶。细直条索带栗香，由双阶段火候「锁定」— 从缓慢的60°C到迅速的120°C — 生长在「龟背」上，龟峰山，其形状象征长寿，其山坡每年春天被千万杜鹃点燃，其泉水按太宗之言使茶成为「天下第一」。以4.36%氨基酸和17.69%多酚类，龟山岩绿达到「醇厚鲜爽」滋味的理想比例 — 适合那些珍视历史深度和山间云雾纯净的每一杯茶的人们。
