# 峡州碧峰 (峡州碧峰, Xiázhōu bìfēng)

**峡州碧峰 (峡州碧峰, Xiázhōu bìfēng)**

![Сячжоу Бифэн (峡州碧峰, Xiázhōu bìfēng)-1](images/Сячжоу%20Бифэн%20(峡州碧峰,%20Xiázhōu%20Bìfēng)-1.png)

峡州碧峰 (峡州碧峰, Xiázhōu bìfēng) — 中国绿茶，属半烘炒条形绿茶 (半烘炒条形绿茶, bàn hōngchǎo tiáoxíng lǜchá)，为湖北省宜昌市夷陵区特产。国家地理标志产品 (国家地理标志产品, guójiā dìlǐ biāozhì chǎnpǐn)，于2017年获得认证。此茶延续了古代峡州超过千年的茶叶传统——「茶圣」陆羽 (陆羽, Lù Yǔ) 在其《茶经》(《茶经》, *Chájīng*) 中将峡州茶列为山南茶区之首。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。半烘炒类型 (半烘炒, bàn hōngchǎo) — 结合滚筒杀青 (杀青, shāqīng) 与最终炭火烘焙 (木炭烘焙, mùtàn hōngbèi)。
* **类别：** 中国地方茶；国家地理标志产品（2017年）。
* **产地：** 中国湖北省 (湖北, Húběi) 宜昌市 (宜昌, Yíchāng) 夷陵区 (夷陵区, Yílíng Qū)。古代地名为峡州 (峡州, Xiázhōu)。
* **地理座标：** 约北纬30°42′–31°06′，东经110°51′–111°11′。

**2. 历史与文化意义：**

* **历史：** 峡州茶叶传统为中国最古老之一。早在南北朝时期（420–589），峡州已是重要产茶区。该地区历史上的关键时刻是《茶经》(《茶经》, *Chájīng*, 约780年) 中的评价，陆羽 (陆羽, Lù Yǔ) 记载：「山南，以峡州上，襄州、荆州次」(山南，以峡州上)。峡州茶因此获得整个南方山区茶叶之首的地位。北宋文学家欧阳修 (欧阳修, Ōuyáng Xiū) 任夷陵县令时，留下著名诗句：「楚国西偏境，陆羽茶经第一州」(陆羽茶经第一州)。诗人陆游 (陆游, Lù Yóu) 在三游洞 (三游洞) 品茶时赞美其色、香、味。王象晋 (王象晋) 《群芳谱》(《群芳谱》, *Qúnfāng Pǔ*) 中将峡州小江园 (小江园) 茶列入最高等级。

  现代峡州碧峰创制于1979年，由茶叶大师林作炎 (林作炎, Lín Zuòyán) 主持。技术完善工作历时七年，1983年正式命名为「峡州碧峰」——取自明代诗人田钧 (田钧) 诗句：「不见仙人空碧峰」(不见仙人空碧峰)。1985年获农业部品质奖，并用作国礼茶 (国礼茶, guólǐ chá) 赠送外国宾客。

  2017年峡州碧峰获地理标志产品认证，品牌价值评估为5.63亿元人民币。茶叶出口至二十多个国家，包括英国、美国、日本和德国。

* **名称：** 「峡州」(峡州) — 宜昌地区古名，可追溯至唐代，与长江峡谷相关。「碧峰」(碧峰) — 「翠绿山峰」— 来自明代诗歌的诗意意象，指向西陵峡 (西陵峡, Xīlíng Xiá) 的山景。

* **文化意义：** 峡州碧峰是古老茶叶传统的现代传承者，该传统在《茶经》中有文献记载。这是少数几种历史可直接追溯到陆羽的茶叶之一，在中国茶文化中具有特殊的象征意义。唐代诗人皮日休 (皮日休, Pí Rìxiū) 赞美小江园茶：「吴僧漫说鸦山好，蜀叟休夸鸟嘴香」——这首九世纪的诗歌证明，早在盛唐时期当地茶叶就超越了江西和四川的名茶。对夷陵区而言，茶叶是地域品牌的名片，是长江峡谷文化不可分割的一部分。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要栽培种为宜昌大叶种 (宜昌大叶种, Yíchāng Dàyè Zhǒng)，注册为华茶29号 (华茶29号) — 国家名录中首批30个地方优良品种之一。辅助栽培种为宜红早 (宜红早, Yíhóng Zǎo)，即鄂茶4号 (鄂茶4号)，以及鄂茶9号 (鄂茶9号)。树龄超过30年；叶片椭圆形，长14.7 ± 2.4公分；100芽重约57公克；原料中多酚含量达35.8%。品种具有高抗寒性，适合在海拔1400公尺以下种植。

* **采摘：** 春季采摘（品质高峰期）。严格遵守「九不采」(九不采, jiǔ bù cǎi) 标准：不采紫色芽头、虫伤叶、病叶及其他缺陷部分。

* **采摘标准：** 特级 (特级) — 单芽或一芽一叶，芽长约3公分；芽头比例 ≥ 90%；完全手工制作。一级 (一级) — 一芽一叶初展 (一芽一叶初展)，含量 ≥ 80%；允许部分机械揉捻。二级 (二级) — 主要为一芽二叶；可冲泡五次以上，叶底平整均匀。

**4. 风土条件与栽培特点：**

* **地形与位置：** 茶园位于长江西陵峡 (西陵峡, Xīlíng Xiá) 北岸，夷陵区半高山茶区 (半高山茶区, bàn gāoshān cháqū)。长江流经产区17.4公里；当地河流溪涧众多——百岁溪 (百岁溪)、太平溪 (太平溪) 等。森林覆盖率超过70%，空气中负离子含量比城市高50倍。

* **海拔：** 200–1000公尺；核心产区海拔600–900公尺（云雾带）。

* **气候：** 年平均气温16.6°C；年降雨量1100–1300毫米；年雾日超过180天；日夜温差 > 8°C。散射光有利于氨基酸积累（春茶氨基酸含量 ≥ 1.64%）。

* **土壤：** 黄棕壤 (黄棕壤, huáng zōng rǎng)，pH值4.5–6.5，有机质含量 ≥ 1.0%，富含锌和硒。

* **核心产区：** 太平溪镇 (太平溪镇)、乐天溪镇 (乐天溪镇)、邓村乡 (邓村乡)、三斗坪镇 (三斗坪镇)。所有核心区域位于西陵峡北岸，海拔600–900公尺的云雾带。邓村乡非正式称为「湖北茶乡」，在生产中发挥关键作用——这里集中了树龄超过30年的最古老茶园。雾气、柔和散射光和富含硒锌的矿化土壤相结合，形成了独特的风土特征，使碧峰有别于其他湖北绿茶。

**5. 制作工艺：**

峡州碧峰为半烘炒绿茶。工艺包括独特的复揉理条 (复揉理条, fù róu lǐtiáo) 步骤，形成特有外形并显露白毫。

1. **摊放 (摊放, tānfàng)：** 鲜叶薄摊4–6小时，初步失水并准备香气前体物质。
2. **杀青 (杀青, shāqīng)：** 在旋转滚筒中于180–200°C处理6–8分钟。停止酶促氧化，固定绿色。
3. **摊凉 (摊凉, tānliáng)：** 摊开原料降温，为揉捻做准备。
4. **初揉 (初揉, chū róu)：** 14–20分钟，按「轻-重-轻」(轻-重-轻) 原则，释放细胞汁液，形成叶片初始结构。
5. **初烘 (初烘, chū hōng)：** 温度100–120°C，部分脱水。
6. **复揉理条 (复揉理条, fù róu lǐtiáo)：** 独创步骤——手工「抓拢搓抖」(手工抓拢搓抖) 技法。正是在此步骤茶叶获得特有的优雅外形和明显的茸毛。
7. **足干提毫 (足干提毫, zú gān tí háo)：** 炭火烘干，温度70–80°C。低温和传统木炭确保香气的精致固定和白毫的显露。
8. **精制定级 (精制定级, jīngzhì dìngjí)：** 按等级分筛并确定商品等级。

**6. 感官特征：**

* **干茶外观：** 条索形 (条索形, tiáosuǒ xíng)，紧细挺直，优雅，白毫显露；色泽翠绿油润 (翠绿油润)。
* **干茶香气：** 明显的栗香 (栗香, lì xiāng) — 主调；背景为清纯植物清香 (清香, qīng xiāng)。
* **茶汤香气：** 栗香更深层展现，持久高扬；清香作为上层音符保持。
* **滋味：** 鲜爽 (鲜爽, xiān shuǎng)，由高氨基酸含量所致；明显持久的甘甜回味——回甘 (回甘, huígān)。茶体轻盈但饱满。
* **汤色：** 黄绿明亮 (黄绿明亮)。
* **叶底（冲泡后茶叶）：** 嫩绿色，匀整鲜活 (嫩绿匀整鲜活)。

**7. 化学成分：**

* **水浸出物 (水浸出物)：** ≥ 45.8% — 显著高于绿茶平均水准，证明滋味浓度高。
* **茶多酚 (茶多酚)：** ≥ 30%（原料中达35.8%）；主要抗氧化群体为儿茶素（EGCG、EGC、ECG）。
* **咖啡碱 (咖啡碱)：** 4.2%。
* **氨基酸：** 春茶中 ≥ 1.64%；酚氨比 (酚氨比, fēn'ān bǐ) 为12.1，显示鲜爽度与茶体的良好平衡。
* **矿物质：** 锌（Zn）和硒（Se）含量较高，由当地土壤地球化学特性所致。
* **维生素：** 维生素C（新鲜原料中）、B族维生素、维生素K。
* **精油：** 栗香特征由滚筒杀青和炭火烘干形成——主要为吡嗪类和呋喃类化合物。

**8. 保健功效：**

* **抗氧化保护：** 高多酚含量提供活性自由基清除。
* **提神效果：** 咖啡碱与L-茶氨酸结合产生温和持久的清醒感，无剧烈波动。
* **脂质代谢支持：** 儿茶素促进脂肪分解加速。
* **矿物质支持：** 天然富含锌和硒，有益于免疫系统和抗氧化保护。
* **消化支持：** 适量单宁有助于餐后胃肠道正常功能。
* **思维清晰：** L-茶氨酸支持专注、平静的意识状态。
* **心血管系统支持：** 定期适量饮用高儿茶素绿茶与血管弹性维护相关。
* **抗辐射保护：** 据某些资料，绿茶多酚有助于减少家用设备电磁辐射引起的氧化应激。

*重要提示：这是一般资讯，非医疗建议。*

**9. 冲泡方法：**

* **水温：** 80°C（绝对不建议沸水——温度超过80°C会破坏叶绿素并产生苦味）。
* **茶量：** 3公克配150毫升（比例1:50）。
* **茶具：** 玻璃杯（观察茶叶舞动）；薄瓷盖碗。

* **过程（玻璃杯，中投法 — 中投法, zhōng tóu fǎ）：**
  1. 用沸水温杯。
  2. 注入1/3容量的水（80°C）。
  3. 投入3公克茶叶，轻摇杯子润茶摇香 (润茶摇香)。
  4. 高冲注水至满杯。
  5. 浸泡2–3分钟。待降至60°C时饮用——此温度鲜爽感最为明显。

* **过程（盖碗）：**
  1. 温盖碗。
  2. 投茶，首泡30秒。
  3. 每次增加10秒。
  4. 可冲泡3次。

* **注意事项：** 沿杯壁缓慢注水——急流会打散茸毛使茶汤浑浊。新茶建议购买后避光存放约7天进行醒茶 (醒茶, xǐng chá)。开封后冷藏保存，一个月内饮用完毕。

**10. 储存：**

* 密封包装，避免异味。
* 最佳温度：0–5°C（冰箱）长期保存。
* 冷藏后开封前室温平衡至无凝露。
* 避免直射阳光、高湿度和热源。
* 开封后一个月内饮用完毕以保持新鲜。

**11. 价格与仿冒：**

* **价格区间：** 特级 (特级)：600–800元/斤（手工制作，单芽）；一级 (一级)：250–400元/斤；二级 (二级) — 价格亲民，定位为日常茶，冲泡耐久性好。
* **价格因素：** 原料等级、采摘季节（早春较贵）、手工比例、地理标志认证。

* **避免仿冒：**
  * 向有「峡州碧峰」品牌使用权的经销商购买（品牌属湖北邓村绿茶集团）。
  * 核实地理标志标识。
  * 真正的峡州碧峰具有细致紧实的条索，白毫明显，油润光泽——粗糙暗淡的叶片表明替代品。
  * 香气应为纯正栗香，无霉味和「鱼腥」味。
  * 声称等级的可疑低价是伪造品的主要标志。

**12. 趣味知识：**

* 陆羽在《茶经》中不仅将峡州列为山南茶区之首，还注意到巴山峡谷中有茶树粗到需要两人合抱。
* 「碧峰」之名借自明代诗人田钧诗句：「不见仙人空碧峰」(不见仙人空碧峰)。这是少数几种名称直接源于诗歌文本的茶叶之一。
* 现代茶的前身是「太平毛尖」(太平毛尖)；现名在1912年东湖县撤销后确立。
* 1985年峡州碧峰成为少数几种用作国礼赠送外国代表团的湖北绿茶之一。
* 2017年品牌价值评估为5.63亿元人民币——对一个区域性茶叶而言是可观数字。
* 「九不采」(九不采) 技术特色是湖北绿茶中最严格的原料品质控制系统之一：禁采紫色芽头、机械损伤叶、虫害叶、病叶、过度展开叶及其他缺陷部分。
* 宜昌是中国最古老的茶叶中心之一：早在三国时期（3世纪）《广雅》(《广雅》) 就记录了荆巴地区（包括峡州）的茶饼生产。

**13. 与其他绿茶比较：**

* **恩施玉露 (恩施玉露, Ēnshī Yùlù)：** 同样来自湖北，但采用日式蒸青 (蒸青) 工艺——具有更多草本「海洋」特征。峡州碧峰采用滚筒杀青配炭火烘干，呈现更深层的栗香。
* **信阳毛尖 (信阳毛尖, Xìnyáng Máojiān)：** 河南邻省茶，茸毛相似，但滋味更圆润，豆香明显。峡州碧峰的回甘更明显，具有锌硒土壤特有的矿物质韵味。
* **太平猴魁 (太平猴魁, Tàipíng Hóukuí)：** 安徽大叶扁形茶——在视觉和质感上与碧峰的细嫩茸毛条索完全相反。两者都以高浸出率著称，但猴魁倾向兰花香，碧峰则为栗香。
* **蒙顶甘露 (蒙顶甘露, Méngdǐng Gānlù)：** 四川茶，工艺相似（炒制+烘干）。甘露更弯曲，花香豆香明显；碧峰更挺直，坚果香主导，西陵峡特有矿物质感明显。
* **庐山云雾 (庐山云雾, Lúshān Yúnwù)：** 江西高山绿茶，来自类似的云雾风土。云雾呈现更柔和温润的特征，花香层次丰富，而碧峰因高浸出率（≥ 45.8%）展现更浓郁的滋味密度和更明显的烘炒栗香特征。

**结语：**

峡州碧峰是一款具有文学血统和千年风土历史的茶叶。它站在《茶经》中「山南第一茶」古老荣耀与湖北茶人1979年创制的现代工艺的交汇点，创造了配得上这一传承的茶品。栗香、鲜活清爽和悠长甘甜回味，使其成为重视经典中国绿茶品格与深度的茶友的绝佳选择。

这款茶在春季月份特别出色，此时首批采摘展现氨基酸甘甜和嫩毫茶汤的全部丰富性。用软水冲泡，勿过热——西陵峡的翠绿山峰将在您的茶杯中绽放。对于地方中国绿茶爱好者，峡州碧峰是最佳发现之一：它不如湖北恩施玉露那样广为人知，但在滋味深度和历史价值方面不逊于任何著名同类茶品。
