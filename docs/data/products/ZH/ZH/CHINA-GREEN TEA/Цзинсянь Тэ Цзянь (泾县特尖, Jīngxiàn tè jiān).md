# 泾县特尖 (泾县特尖, Jīngxiàn tè jiān)

**泾县特尖 (泾县特尖, Jīngxiàn tè jiān)**
![Цзинсянь Тэ Цзянь (泾县特尖, Jīngxiàn tè jiān)](images/Цзинсянь%20Тэ%20Цзянь%20(泾县特尖,%20Jīngxiàn%20Tè%20Jiān).png)

![Цзинсянь Тэ Цзянь (泾县特尖, Jīngxiàn tè jiān)-1](images/Цзинсянь%20Тэ%20Цзянь%20(泾县特尖,%20Jīngxiàn%20Tè%20Jiān)-1.png)

泾县特尖 (泾县特尖, Jīngxiàn tè jiān) 是来自安徽省泾县东南山区的历史烘青绿茶，与著名的涌溪火青同乡，承继古老的安徽「尖茶」(尖茶, jiānchá —「尖形茶」) 传统。此茶有两个独特特征：首先，土壤中硒含量极高，比普通茶叶高5.5倍；其次，持久的兰花香气，结合栗香温暖与蜜糖甜味。乾隆皇帝 (乾隆, Qiánlóng) 南巡时，亲自品尝了泾县汀溪 (汀溪) 谷地的茶叶，赞誉道：「此茶甚好」。

**1. 分类与产地：**

* **类型：** 绿茶（不发酵茶）。属于烘青绿茶 (烘青绿茶, hōngqīng lǜchá) — 最终干燥采用烘焙法 (烘焙, hōngbèi)，包括炭火烘焙。形状方面 — 特级和一级为卷曲形 (卷曲形)，提魁 (提魁) 等级为针芽状 (针芽状)。

* **等级：** 安徽历史名茶 (安徽历史名茶)。1983年获外贸部「荣誉证书」(外贸部"荣耀证书")。1985年获「部优茶」(部优茶)。1986年获「中国名茶」(中国名茶)。地理标志产品。

* **产地：** 中国安徽省 (安徽, Ānhuī) 泾县 (泾县, Jīngxiàn)。生产区域涵盖全县，但核心产区为东南山区，特别是爱民乡 (爱民乡)、南容乡 (南容乡)、汀溪乡 (汀溪乡) 和铜山乡 (铜山乡)。90%以上的高等级茶叶来自这四个乡镇。

* **地理座标：** 约北纬30°30′，东经118°25′。

**2. 历史与文化意义：**

* **历史：** 泾县茶叶种植在唐宋时期已有记载，当时当地茶叶在区域内已享有盛名。清代时泾县茶叶被称为「洋尖」(洋尖) 并出口至东南亚国家 — 这是安徽茶叶早期出口的例子之一。

	重要时刻 — 乾隆皇帝 (乾隆) 南巡时，在汀溪 (汀溪) 谷地停留并品尝了当地的「贡尖」(贡尖，「进贡尖茶」)。他的评语「此茶甚好」被载入地方史册。

	二十世纪：1983年获外贸部出口产品品质证书。1985年获部门奖项。1986年获「中国名茶」地位，复兴了「尖茶」(尖茶) 传统，确立泾县为安徽主要产茶县之一。

* **名称：**

	* 「泾县」(泾县) — 安徽南部县名，位于黄山余脉。
	* 「特」(特) — 「特殊、优秀」：表示最高品质等级。
	* 「尖」(尖) — 「尖端、顶端」：描述芽头的尖锐形状。「尖」字是传统安徽茶叶术语，指用嫩尖芽头制作的茶类。

* **文化意义：** 泾县是具有深厚茶文化传统的县份，孕育了两款著名茶叶：涌溪火青 (涌溪火青) 和泾县特尖。如果涌溪火青是经过20小时炭火干燥的珍珠，那么特尖就是安徽烘青绿茶的「经典」，延续著古老的「尖茶」传统。

**3. 植物学描述与原料：**

* **品种/栽培种：** **泾县群体种 (泾县群体种, Jīngxiàn Qúntǐzhǒng)** — 当地原生 *Camellia sinensis* var. *sinensis* 品种，被认定为「国家级地方良种」(国家级地方良种)。灌木型，叶片从椭圆形到柳叶形。特点是萌芽早，对不良环境抗性强。鲜叶化学成分：多酚类 — 30.35%，胺基酸 — 3.08%，硒含量 — 中国绿茶中最高之一。

* **采摘：** 早春采摘。标准依等级而定：
	* **提魁 (提魁，「最高级」)：** 纯芽或一芽一叶。笔直针状形。
	* **特级 (特级)：** 一芽一叶初展（≥60%）。卷曲形，茸毛丰富。
	* **一级 (一级)：** 一芽二叶。浓郁口感，回甘明显。
	* **二级 (二级)：** 一芽二叶带嫩茎。蜜糖兼兰花香气。

* **原料要求：** 嫩度均匀，无损伤的芽叶。当日采摘当日加工。

**4. 风土条件与栽培特色：**

* **地形：** 黄山东南余脉 (黄山余脉)。山谷间溪流丰富。

* **海拔：** 600公尺以上。

* **气候：** 年均温约15°C，年降雨量1200–1800毫米。全年云雾缭绕。散射光 (漫射光) 充足。

* **土壤：** 花岗岩发育的酸性红黄壤 (红黄壤)，pH 4.5–6.5。腐殖质层深厚，有机质含量高。**关键特色 — 硒 (硒, xī) 含量极高：0.70–3.85 ppm — 比普通茶园土壤高5倍以上。** 这种天然富硒特性决定了泾县特尖独特的矿物质成分。

**5. 制作工艺：**

泾县特尖采用经典安徽烘青 (烘青) 工艺配合炭火干燥。十道工序。

* **摊青 (摊青 — tān qīng)：** 短时间摊放。

* **杀青 (杀青 — shāqīng)：** 锅炒 — 停止氧化，固定兰花栗香。

* **轻揉 (轻揉 — qīngróu)：** 轻压保持芽头完整。

* **炒坯 (炒坯 — chǎopēi)：** 中间炒制。

* **复揉 (复揉 — fùróu)：** 紧实结构。

* **初烘 (初烘 — chū hōng)：** 100–120°C炭火烘焙。

* **理条 (理条 — lǐtiáo)：** 手工整形 — 轻压做形 (轻压做形) 不用粗暴压力。

* **复烘 (复烘 — fù hōng)：** 70–90°C — 温和烘焙展现香气。

* **提毫 (提毫 — tí háo)：** 特殊手法将银毫「提」到表面。

* **足干 (足干 — zúgān)：** 干燥至含水量≤4% — 绿茶中最低含水量之一，确保长期保存。

**6. 感官特征：**

* **干茶外形：** 提魁等级 — 笔直挺秀的针状芽头 (紧直显毫)。特级等级 — 紧密卷曲，茸毫显露 (细曲披毫)。色泽翠绿油润。

* **干茶香气：** 清雅持久。兰花香 (兰花香, lánhuā xiāng) — 主要香气特征。栗香 (栗香) — 浓郁，特级等级尤为明显。

* **茶汤香气：** 兰香清高持久 (兰香清高持久)。栗香层次。

* **滋味：** 鲜爽 (鲜爽)，回甘明显 (回甘)。醇厚 (醇厚)。多酚类含量超过30% — 烘青绿茶中最高之一，赋予口感结构深度。同时胺基酸（3.08%）平衡涩味，创造和谐风味。

	二级等级有独特风味：蜜糖兼兰花香 (蜜糖兼兰花香, mìtáng jiān lánhuā xiāng) — 绿茶中罕见的组合。

* **汤色：** 嫩绿清亮 (嫩绿清亮)，透明「如翡翠」(清澈如翡翠)。

* **叶底（冲泡后茶叶）：** 嫩黄匀整，肥壮成朵 (嫩黄匀整，肥壮成朵)。叶片弹性佳，富有活力。

**7. 化学成分：**

富硒花岗岩土壤和高山微气候决定了独特成分：

* **多酚类（儿茶素）：** 含量超过干重30% — 烘青绿茶中最高之一。提供强大抗氧化潜力和结构深度。

* **胺基酸（包括L-茶胺酸）：** 3.08% — 提供鲜爽和甜味，平衡高多酚含量。

* **硒 (硒, xī)：** **0.70–3.85 ppm — 比茶园土壤平均值高5.5倍。** 硒是强效抗氧化剂和免疫调节剂。泾县特尖是中国最富硒的绿茶之一。

* **生物碱：** 咖啡因约3%。可可碱、茶碱。

* **维生素：** 维生素C、B群维生素。

* **矿物质：** 钾、镁、锌、锰、硒、磷。

**8. 保健功效：**

* **抗氧化作用：** 创纪录的多酚含量（>30%）结合硒（比平均值高5.5倍）提供双重抗氧化效果。
* **心血管系统支持：** 硒与儿茶素结合有助于血压和血脂正常化。
* **提神效果：** 咖啡因和L-茶胺酸。
* **清热消暑 (清热消暑)：** 生津止渴。
* **助消化 (消食)：** 促进脂肪分解。

* **重要提醒：** 以上功效基于公开资料，不构成医疗建议。

**9. 冲泡方法：**

* **水温：** 80°C。

* **茶量：** 3公克配150毫升水（1:50比例）。

* **茶具：** 玻璃杯 — 下投法 (下投法)。

* **冲泡过程：**

	1. 温杯，倒掉温水。
	2. 投入茶叶。
	3. 注入热水。静置2分钟。
	4. 剩余1/3时续水。可冲泡3–4次。

* **注意：** 咖啡因敏感者减少用量。不建议空腹饮用或睡前饮用。

**10. 储存方法：**

* 密封容器，阴凉干燥处保存。
* 最佳条件 — 冰箱0–5°C。
* 成品茶含水量≤4% — 比含水量较高的茶叶保存性更佳。
* 保存期限 — 12–18个月。
* 开封后 — 1–2个月内饮用完毕。

**11. 价格与仿冒品：**

泾县特尖价格适中，性价比良好，特别是特级和一级等级。提魁等级价格较高。

* **避免仿冒品方法：**

	* **向可靠商家购买**，确认来自泾县产地。
	* **评估香气：** 兰花香是招牌特征。缺乏兰香令人怀疑。
	* **检查汤色：** 「如翡翠」— 清澈明亮的嫩绿色。
	* **注意产地：** 90%以上高等级茶来自四个核心乡镇（爱民、南容、汀溪、铜山）。

**12. 趣味知识：**

* 泾县土壤硒含量（0.70–3.85 ppm）比茶区平均值高5.5倍。泾县特尖是世界上最富硒的绿茶之一。

* 乾隆皇帝品尝汀溪谷地茶叶后说：「此茶甚好」。这是少数有文献记载的皇帝对非「十大名茶」安徽茶叶的具体评价。

* 泾县孕育了两款截然不同的著名绿茶：珠形涌溪火青 (涌溪火青) 具杏黄汤色和20小时干燥工艺，以及卷曲针状特尖具兰花香气和创纪录硒含量。同一县份两款茶 — 两种完全不同的风格。

* 残余水分≤4% — 绿茶中最低之一（通常5–7%）。这是双重炭火干燥的结果，确保更佳保存性。

* 清代泾县茶以「洋尖」(洋尖，「海外尖茶」) 名义出口东南亚 — 远早于大多数安徽茶叶进入国际市场。

**13. 与其他安徽绿茶比较：**

* **涌溪火青 (涌溪火青)：** 泾县同乡。珠形，杏黄汤色，20小时炭火干燥。火青更「温暖」、圆润、「火气」；特尖更兰香、矿物质丰富。

* **黄山毛峰 (黄山毛峰)：** 来自黄山市。烘青「雀舌」具兰花栗香。毛峰更「知名」、商业成功；特尖硒含量更高、多酚含量更高。

* **太平猴魁 (太平猴魁)：** 来自黄山市。大型扁平叶片，深沉兰花香。猴魁更「宏伟」；特尖更紧实、「矿物质」。

* **六安瓜片 (六安瓜片)：** 来自皖西。纯叶片制成的扁平「瓜子」。瓜片更草本浓郁；特尖更兰香甜美。

**结语：**

泾县特尖是一款土地比工艺师更有发言权的茶。黄山余脉富硒花岗岩土壤、云雾缭绕的溪谷和百年「尖茶」传统，造就了这款兰花香气和矿物质深度在其他地方无法复制的茶叶。这不是最响亮、最著名的安徽茶 — 但可能是最慷慨的之一：慷慨的硒含量、兰花香气、二级的蜜糖甜味和一级的持久栗香温暖。如果涌溪火青是泾县的「火」，那么特尖就是它的「土」：深沉、矿物质丰富、疗愈。
