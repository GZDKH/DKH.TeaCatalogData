# 三峡碧螺春 (三峡碧螺春, Sānxiá bìluóchūn)

**三峡碧螺春 (三峡碧螺春, Sānxiá bìluóchūn)**
![三峡碧螺春 (三峡碧螺春, Sānxiá bìluóchūn)](images/Санься%20Би%20Ло%20Чунь%20(三峡碧螺春,%20Sānxiá%20Bìluóchūn).png)

![三峡碧螺春 (三峡碧螺春, Sānxiá bìluóchūn)-1](images/Санься%20Би%20Ло%20Чунь%20(三峡碧螺春,%20Sānxiá%20Bìluóchūn)-1.png)

三峡碧螺春是以独特的本土品种青心柑仔（青心柑仔）为原料制成的台湾绿茶，此品种仅生长于三峡地区。这是一款拥有260年茶叶栽培历史的茶品，历经乌龙茶时期、大英帝国红茶时期、日本「日东红茶」时期以及战后复兴时期，最终在20世纪末透过独特的绿豆仁香绿茶找到了真正的身份认同。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵茶，氧化程度低于5%）。杀青方式为炒青（炒菁, chǎoqīng）。
* **类别：** 台湾绿茶。列入「台湾十大经典名茶」（台湾十大经典名茶）之一。
* **产地：** 台湾，新北市（新北市, Xīnběi Shì），三峡区（三峡区, Sānxiá Qū）。古地名为三角涌（三角涌, Sānjiǎoyǒng）——「三条涌动的溪流」，因大汉溪（大汉溪）、三峡溪（三峡溪）和横溪（横溪）三条河流在此汇流而得名。主要茶区：西南部（弘道里、五寮、大埔、金敏、插角、有木）和东北部（礁溪、白鸡、溪南、成福、竹仑）——后者产出品质较高的原料。
* **地理座标：** 约北纬24°55′，东经121°22′。

**2. 历史与文化意义：**

* **历史：**

三峡是台湾最古老的茶叶产区之一。此地的茶叶栽培历史可追溯至1763年（乾隆28年），当时来自安溪（安溪）的林理完（林理完）开始在横溪南岸的溪南（溪南）地区种植茶叶。来自福建茶乡安溪（安溪）和永春（永春）的移民带来了茶苗和技术，奠定了北台湾最早茶区之一的基础。

1868年，英国商人约翰·陶德（John Dodd）组织将「福尔摩沙乌龙茶」从台湾出口到纽约，引起国际轰动——这些首批出口茶叶的原料有相当部分来自三峡。

在日本殖民统治时期（1895-1945），三井合名会社（三井合名会社，后来的台湾农林公司）在三峡设立了大豹和大寮（1924年）茶厂，转向以阿萨姆品种大量生产红茶，以「日东红茶」（日东红茶）品牌行销全球。大寮茶厂成为东亚最大的茶叶加工厂之一。

1945年后，随著习惯饮用绿茶（龙井、碧螺春、茉莉花茶）的大陆军政人员来台，三峡茶农转向生产炒青绿茶（炒菁绿茶）。这一时期的主要产品是龙井、碧螺春、香片（香片，茉莉花茶）和包种——三峡四大茶（三峡四大茶）。当时的绿茶以朴实的「海山绿茶」（海山绿茶）之名销售。

1980年代，台湾茶业经历危机：成本上升、出口停滞和建筑业繁荣导致茶农大量弃茶。三峡的茶园面积从超过1000公顷锐减至仅100多公顷。许多茶农转向种植更有利可图的槟榔。

1990年代，随著绿茶作为「健康饮品」的流行，产业迎来复兴。三峡农会推广股长王清松（王清松）发挥了关键作用，他提议使用包种茶设备来生产碧螺春。1998年举办了首届「三峡优良茶碧螺春比赛」（三峡优良茶碧螺春比赛），开始以「三峡碧螺春」品牌取代原有的「海山绿茶」积极推广当地绿茶。十年间，茶青价格上涨六倍，年轻茶农开始回流。

到2018年，三峡茶园面积恢复至约200公顷；该区茶叶产量约占全台湾总产量的8%。

* **名称：**

    * 「三峡」（三峡）——1920年日本当局根据古台湾地名「三角涌」（三角涌）的谐音而定的区名。字面意思为「三个峡谷」。
    * 「碧螺春」（碧螺春）——「翠绿的春天螺旋」。此名称借自江苏省的同名中国著名绿茶，康熙皇帝（康熙）约在1678年因欣赏其香气而赐名（原名「吓煞人香」——吓煞人香，「香得吓死人」）。台湾版本因外观相似（紧结的绿色螺旋状带白毫）和香气浓郁而得此名，但在风味特征上是独立的产品。

* **文化意义：** 三峡碧螺春是台湾茶业复兴和韧性的象征。这款茶生动地展示了一个经历多次茶叶典范转变（乌龙→红茶→绿茶→衰退→复兴）的地区，如何透过结合本土品种和改良技术找到独特的身份认同。三峡是台湾唯一实行「见芽就采」（见芽就采）原则的地区，强调当地原料的特殊细嫩。茶叶在三峡美食中广泛应用：茶馅点心、茶叶蛋糕卷、冷泡茶已成为该区的特色伴手礼。

**3. 植物学描述与原料：**

* **品种／品系：** *Camellia sinensis* var. *sinensis*。主要品种为**青心柑仔（青心柑仔, Qīngxīn Gānzǎi）**，也简称「柑仔」（柑仔）。这是小叶种早生种（早生种, zǎoshēngzhǒng），为三峡地区特有——在台湾其他地方或海外都没有栽培。品种来源仍是谜团：当地耆老和地方研究者都无法确定它何时从何处引进；没有书面记录。灌木中等或大型，分枝较少，略呈直立型。叶片大，形状类似柑橘叶（柑叶, gānyè）——因此得名。特征是叶缘明显向上弯曲。嫩芽绿色，带有丰富的白毫（白毫, báiháo）。花朵花瓣数量异常多。品种非常适合制作绿茶（特别是龙井和碧螺春）；也用于制作具有特殊蜜香和柑橘香的红茶（蜜香红茶, mìxiāng hóngchá）。近年来三峡也开始种植金萱（金萱，台茶12号），这引起了爱茶人士的担忧——新品种的推广可能导致失去青心柑仔独有的「绿豆」香气特征。
* **采摘：** 主要季节为春季（春茶, chūnchá，3月至4月初清明节前——最珍贵的原料）和冬季（冬茶, dōngchá）。夏季原料（5-8月）常用于制作红茶（蜜香红茶），当叶片被小绿叶蝉（Jacobiasca formosana）叮咬时。采摘为手工，遵循「见芽就采」（见芽就采）原则——看到嫩芽就采，不等大量展叶。
* **采摘标准：** 高级茶采一芽一叶（一心一叶, yī xīn yī yè）或一芽二叶（一心二叶, yī xīn èr yè）。嫩芽必须覆盖丰富的白毫。
* **原料要求：** 必须是极其细嫩、完整无损的嫩梢。精品批次实行「一日标准」：从采摘到最终干燥的整个周期在一天内完成。

**4. 风土条件与栽培特点：**

* **地区：** 三峡区位于台湾西北部，地处三条河流汇流处的丘陵地带，四周环山。茶园分布在缓坡和河谷地带。
* **海拔高度：** 海拔200-400公尺。西南部部分地区可达500公尺。
* **土壤：** 主要为酸性红壤（pH 4.5-5.5），由砂岩风化形成，铁含量和矿物质含量较高。河流系统提供良好的排水。
* **气候：** 亚热带季风气候，空气湿度高（年平均>80%），降雨充沛（约2000-2500毫米/年），冬季温和（1月平均温度+15°C），夏季炎热。特征是早晚多雾，形成自然的光线散射，减少叶片的紫外线负荷。
* **特点：** 实行茶树与果树（桃树、柿子）混种，果树作为天然防风屏障，当地茶农认为还能为茶叶增添细微的果香。部分茶园已转向有机栽培（不使用农药，最少化肥），得到当地农会支持。到2018年，产业积极发展生态茶业方向。

**5. 制作工艺：**

三峡碧螺春的制作工艺基于炒青法（炒菁, chǎoqīng），但包含重要的地区特色——短暂萎凋，这在传统绿茶制作中并不常见，使茶叶比中国同类产品更甜润柔和。

* **采摘（采摘, cǎi zhāi）：** 清晨露水干后手工采摘嫩梢。

* **摊放／萎凋（萎凋, wěidiāo）：** 鲜叶在室内短暂萎凋。这个步骤是台湾工艺的特色：叶片与空气接触，开始温和失水和初步香气转化。正是这个萎凋过程使三峡碧螺春比典型的中国绿茶更柔和甘甜。

* **杀青（杀菁, shāqīng）：** 在旋转滚筒锅中高温（约180°C）炒制。酵素失活，氧化停止，形成特有的炒香。由于设备与包种茶（包种茶）生产设备相似，三峡茶厂拥有成熟的技术基础。

* **揉捻（揉捻, róuniǎn）：** 使用专门的揉捻机以控制压力将叶片揉成紧结的螺旋状。过程模仿手工揉捻，但确保形状均匀。嫩芽上的白毫得以保留。

* **干燥（干燥, gānzào）：** 在适中温度下最终干燥，将水分降至稳定水平（<5%），固定形状和香气特征。

* **分级（分级, fēnjí）：** 根据螺旋大小、叶片完整度和嫩芽含量进行手工和机械分级。

**6. 感官特征：**

* **干茶外观：** 小而紧结的翠绿色（碧绿, bìlǜ）螺旋状，覆盖明显的白毫（白毫）。叶片在视觉上比中国洞庭碧螺春稍大且更紧实。
* **干茶香气：** 明亮、高扬，带有特征性的绿豆仁香（绿豆仁香, lǜdòurén xiāng）——三峡碧螺春的主要香气「名片」。附加香气：海苔（海苔, hǎitái）、栗子（栗子, lìzi）、甘蔗（甘蔗, gānzhè）、牧草（牧草, mùcǎo）、甘草（甘草, gāncǎo）。香气被描述为属于「草本调」（草本调, cǎoběn diào）。
* **茶汤香气：** 保持「绿豆」特征，辅以轻微的花香和甜香。香气高扬、纯净、持久。
* **滋味：** 鲜爽（鲜爽, xiānshuǎng），具有明显的天然甘醇（甘醇, gānchún）和饱满圆润的茶体（质厚, zhìhòu）。特征是轻微的收敛鲜爽感，转化为带有柑橘色调的持久甜韵——青心柑仔独特的「品种特性」。与中国和日本绿茶相比，三峡碧螺春明显更饱满、浓郁，耐泡度更高。
* **茶汤颜色：** 透明，淡绿色或绿黄色（淡绿, dànlǜ），清澈明亮。多次冲泡仍保持透明。
* **叶底（泡开的茶叶）：** 细嫩、完整的嫩叶和芽呈绿色，经过几次冲泡后完全展开。叶底的均匀性是原料品质的指标。

**7. 化学成分：**

* **多酚类（儿茶素）：** 含量高，包括EGCG——绿茶的主要抗氧化剂。
* **胺基酸：** 由于春季采摘和多雾的微气候，L-茶胺酸含量较高。L-茶胺酸负责甜味（鲜味）和放松效果。
* **生物碱：** 咖啡因、可可碱、茶碱含量适中。
* **叶绿素：** 含量高，赋予干茶和茶汤翠绿色。
* **糖类：** 游离糖含量较高（春季采摘和青心柑仔品种特性的结果），增强天然甜度。
* **矿物质：** 铁、钾、锰、氟——来自该地区的酸性红壤。
* **维生素：** 维生素C（高达200-250毫克/100克）、B群维生素、维生素E。

**8. 保健功效：**

* **抗氧化保护：** 高浓度儿茶素提供强大的抗氧化应激保护。
* **改善认知功能：** L-茶胺酸和咖啡因的协同作用有助于温和提高专注力和思维清晰度，而不会造成神经过度兴奋。
* **支持新陈代谢：** 咖啡因和儿茶素的结合有助于加速代谢。
* **放松效果：** L-茶胺酸有助于减少焦虑和改善情绪。
* **增强免疫力：** 维生素C和矿物质复合物支持身体的防御功能。
* **理想的冷泡茶：** 由于胺基酸和糖类含量高，三峡碧螺春非常适合冷泡（冷泡, lěng pào）技术，可以享受保健功效而不会因热破坏维生素。

**9. 冲泡方法：**

* **水温：** 85-90°C。三峡碧螺春比许多绿茶更耐高温，但不建议使用沸水。
* **茶叶用量：** 功夫泡法（功夫泡法）5克茶叶配200毫升水；杯泡1克配100毫升；冷泡1克配130毫升。
* **茶具：** 瓷盖碗（盖碗）是最佳选择；瓷茶壶和玻璃壶也适合。优先顺序：瓷器 > 陶器 > 玻璃。
* **冲泡过程（功夫泡法）：**
    1. 用热水温润茶具。
    2. 投入干茶。
    3. 快速洗茶（20秒，85°C）——可行且推荐。
    4. 第一、二泡——各20秒，85°C。
    5. 第三泡——20秒，88°C。
    6. 后续冲泡——逐渐延长时间。茶叶可冲泡5-7次。
* **冷泡（冷泡, lěng pào）：** 3克茶叶配400毫升冷水，在冰箱中浸泡4-8小时。这种方法展现三峡碧螺春特有的清新和甜度，是该地区的招牌饮用方式。

**10. 储存方法：**

三峡碧螺春是需要妥善储存的绿茶。建议使用密封、不透光的包装（真空铝箔袋、锡罐或陶罐），存放在阴凉、干燥处，远离阳光和强烈气味。为最大程度保持新鲜度——在0-5°C的冰箱中以可靠密封的包装保存。最佳饮用期为采摘后6-12个月；一年后香气和新鲜度明显下降。正是因为保存期短，三峡农会开始尝试用青心柑仔的秋冬原料制作白茶——白茶与绿茶不同，适合长期陈放。

**11. 价格与防伪：**

三峡碧螺春属于高价位茶品。价格高昂是由于茶园面积有限（约200公顷）、手工采摘、采摘季节短和品种的特有性。在台湾市场，优质春季三峡碧螺春售价为每150克700至2000新台币（NT$）。在国际市场上——每100克25至50美元。

* **如何避免假货：**

    * 向专业的台湾茶商购买。可靠的指标——在年度「三峡优良茶碧螺春比赛」（由三峡农会举办）中获奖的茶叶。
    * 评估外观：正宗的三峡碧螺春具有紧结的翠绿螺旋状，带有明显的白毫。过于细小、均匀、鲜绿色且无白毫的茶叶——可能是南中国茶叶的替代品。
    * 检查关键香气：特征性的「绿豆仁香」——绿豆香气——是青心柑仔品种的名片。缺少这种香气是严重的怀疑理由。
    * 测试耐泡度：真正的三峡碧螺春比大多数中国绿茶更饱满、更耐泡；它不会在2-3泡后「失味」。
    * 警惕可疑的低价：有限的产量和手工劳动使得廉价的「三峡碧螺春」不切实际。

**12. 趣闻轶事：**

* 青心柑仔（青心柑仔）品种是台湾植物学之谜之一：它不存在于岛上任何其他茶区，其来源无法确定。一些研究者推测它是在18-19世纪通过淡水（淡水）港从福建引进的，但没有文献证据。
* 三峡是台湾唯一实行「见芽就采」原则的地区。在台湾其他茶区，采摘与叶片展开阶段相关，而不是与嫩芽出现相关。
* 用同样的青心柑仔品种，三峡不仅生产碧螺春，还生产台湾龙井（龙井），以及「蜜香红茶」（蜜香红茶）——用被小绿叶蝉（Jacobiasca formosana）叮咬的叶片制成的蜜香红茶。因此，一个品种产生三种根本不同的产品。
* 1924年成立的大寮制茶场（大寮制茶场）是东亚最大的茶叶加工厂之一。旁边保存著1944年建造的厂长宿舍，为昭和时期日式建筑风格，现为大寮茶文馆（大寮茶文馆），自2015年起对外开放。
* 冷泡（冷泡）——三峡碧螺春的招牌饮用方式：在炎热的台湾夏天，冷泡茶瓶几乎成为该地区的象征，就像热茶一样。

**13. 与其他绿茶的比较：**

* **洞庭碧螺春（洞庭碧螺春, Dòngtíng Bìluóchūn）：** 来自江苏省的中国「同名茶」。由小叶种洞庭山小叶（洞庭山小叶）制成。香气为果香花香型，带有桃子和杏子的香气；口感轻盈、细腻，略带涩感。台湾三峡版本更饱满，具有豆类草本香气，甜度更明显，耐泡度显著更高。
* **台湾龙井（三峡龙井, Sānxiá Lóngjǐng）：** 由同样的青心柑仔品种制成，但呈扁平状。三峡龙井以「色绿、香郁、味甘、形美」闻名——「绿色、浓香、甜味、美形」。与碧螺春的区别在于形状（扁平vs螺旋）和更突出的先苦后甘（先苦后甘）。
* **西湖龙井（西湖龙井, Xīhú Lóngjǐng）：** 中国经典扁平绿茶，具有栗子坚果香。与台湾碧螺春不同，龙井更「干」和「矿物质」，没有明显的豆香。
* **日本煎茶（煎茶, Sencha）：** 蒸制（蒸制, zhēngzhì）绿茶，具有海洋、海藻香气。三峡碧螺春是炒制（炒制），具有草本豆类特征；它不那么「海洋」而更「大地」，而且明显更耐泡。

**结语：**

三峡碧螺春不是对著名中国「同名茶」的复制或模仿，而是具有自身特色的台湾绿茶，其特色来自神秘的本土品种和该地区260年的茶叶栽培历史。其标志性的绿豆香气、鲜活的清新感、饱满的甜度和惊人的耐泡性使其在亚洲绿茶中脱颖而出。对于寻求有个性的绿茶的人——浓郁、令人难忘、能在热泡和夏日冷泡中展现风采的茶——三峡碧螺春将是一个精彩的发现。
