# 乐昌白毛茶 (乐昌白毛茶, Lèchāng bái máochá)

**乐昌白毛茶 (乐昌白毛茶, Lèchāng bái máochá)**
![乐昌白毛茶 (乐昌白毛茶, Lèchāng bái máochá)](images/Лэчан%20Бай%20Мао%20Ча%20(乐昌白毛茶,%20Lèchāng%20Bái%20Máochá).png)

乐昌白毛茶（乐昌白毛茶，Lèchāng bái máochá——「乐昌的白色绒毛茶」）——古老的广东绿茶，拥有独特的血统：「茶圣」陆羽（陆羽）亲自造访乐昌研究当地茶叶，并在西石岩洞的岩壁上留下「枢室」（shūshì，「轴心之室」）题字——这是陆羽造访特定茶区的少数文献证据之一。此茶采用**乐昌白毛茶群体种（乐昌白毛茶群体种）**制成——这是原生的大叶种乔木型品种，1985年被认定为**国家级良种**（全国茶树良种，编号GS13015-1985）。其特色是创纪录的茶多酚含量（**38.0%**），为中国绿茶中最高之一，以及独特的**荔枝木炭**（lìzhīmùtàn——「荔枝木炭」）最终烘干工艺，赋予茶叶独特的「蜜韵」（蜜韵，mìyùn）。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。属于烘青绿茶（烘青绿茶，hōngqīng lǜchá）。外形——肥壮紧结的条索，披毫显著（肥壮紧结，披毫显著）。

* **类别：** 国家地理标志产品（国家地理标志产品，2009年）。全国名特优新农产品（2020年）。品种——国家级良种（全国茶树良种，GS13015-1985）。核心产区森林覆盖率——90%。

* **产地：** 中国广东省（广东，Guǎngdōng）乐昌市（乐昌市，Lèchāng Shì）。地理标志保护区——整个乐昌市辖区。核心产区——**九峰山脉（九峰山脉，Jiǔfēng Shānmài）**，南岭山区：大源镇（大源镇）、九峰镇（九峰镇）、两江镇（两江镇）——产量超过60%。沿溪山茶场（沿溪山茶场，Yánxīshān Cháchǎng）——母树发现地和地理标志保护核心。至2025年——23,900亩茶园，1,233吨干茶，总产值25.5亿元。

* **地理座标：** 北纬25度（北纬25°），南岭山脉地区。

**2. 历史与文化意义：**

* **历史：** **唐代（789年）：** 「茶圣」陆羽（陆羽）在《茶经》（茶经）中记载，韶州（韶州，包括乐昌）产茶且「其味极佳」（其味极佳）。更重要的是，陆羽**亲自造访**乐昌研究当地白毛茶，并在西石岩洞的岩壁上留下「枢室」题字——这是中国少数保存有陆羽亲临实物证据的地方之一。

	**近代历史：** 1943年，茶叶专家**罗博柔（罗博柔，Luó Bóróu）**在乐昌山区发现野生茶树，并命名为「乐昌白毛茶」——「乐昌的白色绒毛茶」。1956年，在**陶铸（陶铸，Táo Zhù）**——广东省委书记的指示下，成立国营沿溪山茶场（沿溪山茶场），开始工业化开发野生资源。

	1985年——认定为国家级良种（GS13015-1985）。2009年——获得地理标志保护。2020年——「全国名特优新农产品」。

* **名称：**

	* 「乐昌」（乐昌）——「快乐繁荣」：广东北部城市名。
	* 「白毛」（白毛）——「白色绒毛」：形容极其丰富的银白色绒毛覆盖嫩芽。
	* 「茶」（茶）——「茶」。

* **文化意义：** 乐昌白毛茶是少数能将历史「连结」到陆羽亲访的茶叶——陆羽是世界茶史上最伟大的人物。岩壁上的「枢室」题字不是传说，而是实物文物。对乐昌和广东北部而言，茶叶是经济支柱和文化象征。

**3. 植物学描述与原料：**

* **品种/品系：** **乐昌白毛茶群体种（乐昌白毛茶群体种）**——**国家级良种**（全国茶树良种，GS13015-1985）。乔木型（乔木型，qiáomù xíng），大叶类（大叶类），早生种（早生种）。嫩芽——黄绿色，绒毛——特别丰富（茸毛特多）。百芽三叶重——**130克**——中国品种中最高指标之一。春季原料化学成分：**茶多酚——38.0%，氨基酸——1.6%，咖啡因——3.9%，儿茶素——22.6%。**

	茶多酚38.0%——创纪录的高指标，解释了强大的抗氧化潜力和味道的结构深度。

	改良品种：**白毛2号（白毛2号）**——广东省茶叶研究所选育，香气更浓，抗病性更强。

	原生品种分为三个亚型：
	* **大叶白毛（大叶白毛）：** 肥壮芽头，绒毛最丰富。品质最佳。约占80%种植面积。
	* **中叶白毛（中叶白毛）：** 优雅香气。用于大宗绿茶和红茶。
	* **尖叶白毛（尖叶白毛）：** 抗寒性强。用于白茶和黄茶。

* **采摘：** 春季。特级——一芽一叶。一级——一芽二叶。二级——一芽三叶。

**4. 风土条件与栽培特点：**

* **位置：** 南岭山区（南岭山区），北纬25度——广东最北部。

* **海拔高度：** 400-800公尺。核心区——九峰山地区。

* **气候：** 年均温度——19.6°C，年降雨量——1500-1900毫米，年均雾日——>180天。日夜温差——>10°C。散射光比例——70%。

* **土壤：** 花岗岩风化红黄壤（花岗岩风化红黄壤），pH 4.5-6.5。有机质含量——≥3%。硒含量——0.018-0.066毫克/公斤。森林覆盖率——**90%**——茶区中最高指标之一。

* **特点：** 高山「云雾带」减缓嫩芽生长——氨基酸累积比一般高20%。日夜温差刺激茶多酚和芳香物质合成。

**5. 制作工艺：**

乐昌白毛茶——烘青绿茶，有两个关键特点：**「重杀青」（zhòng shāqīng——「重度杀青」）**在300°C和**荔枝木炭最终烘干（荔枝木炭焙，lìzhī mùtàn bèi）**。

* **采摘（采摘）：** 手工春季采摘「一芽一至三叶」标准嫩芽。

* **摊放（摊放——tānfàng）：** 在竹筛（竹筛）上，4-16小时——绿茶中最长的摊放时间之一。

* **「重度」杀青（杀青——shāqīng）：** 在滚筒杀青机中**300°C**——绿茶中最高的杀青温度之一（相比之下：标准温度——140-240°C）。极高温度瞬间灭活酵素，「锁住翠绿」（锁翠，suǒcuì）并降低大叶原料创纪录茶多酚含量（38.0%）的苦涩。

* **揉捻（揉捻——róuniǎn）：** 按「轻→重→轻」（轻-重-轻）原则。

* **初步烘干（毛火烘干——máohuǒ hōnggān）：** 120°C。

* **摊凉回潮（摊凉回潮——tānliáng huícháo）：** 内部水分重新分布。

* **荔枝木炭最终烘干（足火烘干/荔枝木炭焙——zúhuǒ hōnggān / lìzhī mùtàn bèi）：** 95°C，至含水量≤6%。**关键且独特的步骤：**使用**荔枝木炭**（荔枝木，lìzhī mù）烘干——这是非物质文化遗产核心技艺（非遗技艺核心）。荔枝木燃烧时释放特殊芳香化合物，被茶叶吸收，形成特有的**「蜜韵」（蜜韵，mìyùn）**——此茶独有。

**6. 感官特征：**

* **干茶外观：** 肥壮、紧结、坚实的条索（肥壮紧结）。丰富的银色绒毛（披毫显著）——「名片」：白毛茶是中国「绒毛最多」的绿茶之一。

* **干茶香气：** 清纯、高扬、持久（清香馥郁，qīngxiāng fùyù）。兰花香（兰花香）。栗香（栗香）。空杯留香——**>30分钟**。陈放后显现**「蜜韵」（蜜韵）**——荔枝木炭烘干的独特标志。

* **茶汤香气：** 高扬，兰花栗香，持久。「蜜」的深度。

* **滋味：** 鲜爽（鲜爽）。醇和（醇和，chúnhé）——尽管茶多酚含量创纪录（38.0%），300°C的「重度」杀青中和了粗糙的涩味。回甘——稳定持久，带有**「高山韵清凉感」（高山韵清凉感，gāoshān yùn qīngliáng gǎn）**。

* **汤色：** 嫩绿明亮（嫩绿明亮）——特级；黄绿色——标准级。

* **叶底：** 嫩绿、肥壮、坚实的嫩芽（嫩绿肥壮），匀整明亮（匀整明亮）。

**7. 化学成分：**

大叶乔木品种、花岗岩土壤和「云雾带」形成强大的成分：

* **茶多酚（儿茶素）：** **38.0%**——中国绿茶中最高指标之一。清除自由基效率——比一般绿茶高30%。

* **儿茶素（总量）：** **22.6%**——抑制脂肪合成酵素活性；根据研究，降脂效果——显著高于红茶。

* **氨基酸：** 1.6%——低于平均值（大叶品种和充足阳光的结果）。然而300°C的「重度」杀青弥补了这点，软化了涩味。

* **咖啡因：** 3.9%。

* **氟：** 高含量——在牙齿珐瑯质上形成「氟磷灰石」（氟磷灰石，fú línhuī shí）保护层，提高抗酸侵蚀能力40%。

* **硒：** 0.018-0.066毫克/公斤——来自花岗岩土壤的天然富集。

* **维生素：** 维生素C、类胡萝卜素。

**8. 保健功效：**

* **强大的抗氧化作用：** 茶多酚38.0%——效果比一般绿茶高30%。
* **护齿防龋（护齿防龋）：** 高氟含量——珐瑯质抗酸性提高40%。
* **调节血脂（降脂护心）：** 儿茶素22.6%抑制脂肪合成——比红茶更有效。
* **抗菌作用：** 创纪录的茶多酚凝固细菌蛋白质，效率比一般高30%。
* **提神效果：** 咖啡因3.9%。

* **重要提示：** 非医疗建议。

**9. 冲泡方法：**

* **水温：** 80-85°C；特级——75°C。
* **茶量：** 3克/150毫升（1:50）。
* **茶具：** 玻璃杯或白瓷盖碗。
* **过程：** 洗茶→第一泡30秒→每泡+10秒。3-4泡。

**10. 储存方法：**

* 密封，冰箱0-5°C。保质期——12个月。开封后——3个月。

**11. 价格与辨伪：**

三个等级：特级、一级、二级。购买有地理标志标识的；评估绒毛丰富度和「蜜韵」；检查留香时间（>30分钟）。

**12. 趣闻轶事：**

* 陆羽**亲自**造访乐昌并在岩壁留下「枢室」题字——「茶圣」造访的实物证据，世界罕见。

* 茶多酚**38.0%**——绿茶中的纪录。相比之下：大多数——20-30%。

* **荔枝木炭**烘干（荔枝木炭焙）——独特技术，别处未见。荔枝木赋予「蜜韵」，电烘或其他木材烘干无法复制。

* 百芽重——**130克**——品种中最高指标之一：嫩芽「沉重」、肥壮、饱满。

* 品种被认定为国家级良种（GS13015-1985）——中国茶树选育最高地位。

* 核心区森林覆盖率——**90%**——几乎是绝对纪录。

**13. 与其他广东及大叶绿茶的比较：**

* **英德绿茶（英德绿茶）：** 广东同乡。同样来自大叶品种。英德——来自云南引进种；乐昌——来自本地原生种。英德——较「温和」（茶多酚25-30%）；乐昌——更「强劲」（38.0%）。

* **滇绿（滇绿，云南绿茶）：** 同样大叶种。滇绿——较「生」；乐昌——经300°C杀青和荔枝炭烘更「熟」。

* **惠明茶（惠明茶）：** 来自浙江。同样绒毛丰富。惠明茶——小叶种，乳白色芽头；乐昌——大叶种，银色绒毛和「蜜韵」。

**结语：**

乐昌白毛茶——陆羽亲自前来品尝的茶。创纪录的茶多酚（38.0%）、独特的荔枝木炭烘干、肥壮的嫩芽配上丰富的银色绒毛和任何其他技术都无法复制的「蜜韵」——这一切使乐昌的「白毛」成为中国南方最不寻常的绿茶之一。这不是精致的龙井，也不是细嫩的碧螺春——这是一款强劲、深沉、矿物质丰富的茶，具有大叶乔木品种的个性，经过300度杀青的淬炼，并被荔枝香气「加持」。对于寻求有体感、有力度、有陆羽亲笔历史的绿茶爱好者而言，这是不二之选。

乐昌白毛茶不仅是一款茶，更是一段活著的历史。从唐代陆羽的亲访题字，到现代的国家级良种认定，这款茶见证了中国茶文化的千年传承。其独特的荔枝木炭烘焙工艺和创纪录的茶多酚含量，使其在众多绿茶中独树一帜，成为广东茶叶的骄傲和中国茶文化遗产的重要组成部分。
