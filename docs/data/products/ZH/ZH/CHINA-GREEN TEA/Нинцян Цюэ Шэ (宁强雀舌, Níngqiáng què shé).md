# 宁强雀舌 (宁强雀舌, Níngqiáng què shé)

**宁强雀舌 (宁强雀舌, Níngqiáng què shé)**

![Нинцян Цюэ Шэ (宁强雀舌, Níngqiáng què shé)](images/Нинцян%20Цюэ%20Шэ%20(宁强雀舌,%20Níngqiáng%20Què%20Shé).png)

![Нинцян Цюэ Шэ (宁强雀舌, Níngqiáng què shé)-1](images/Нинцян%20Цюэ%20Шэ%20(宁强雀舌,%20Níngqiáng%20Què%20Shé)-1.png)

宁强雀舌 (宁强雀舌, Níngqiáng què shé) —「宁强县的雀舌」— 绿茶，产自陕西省汉中市宁强县 (宁强县, Níngqiáng Xiàn)，位于**汉江源头** (汉江源头, Hànjiāng Yuántóu) — 长江最大支流，曾赋名于汉朝、汉族和汉字。宁强的茶叶传统可追溯至唐代：陆羽在《茶经》中记载：「梁州生褒城、金牛二县山谷」—「梁州褒城、金牛二县山谷中产茶」。金牛 (金牛，「金牛」) — 宁强古名，与传说中的「金牛道」(金牛道, Jīnniú Dào) 相关 — 连接关中（西安）与四川穿越秦岭的最古老道路。此茶创制于1992年，**1996年列入「国家特需用茶」** (国家特需用茶, guójiā tèxū yòng chá) — 即供应**中南海** (中南海) — 中华人民共和国最高领导层驻地。2002年宁强成为**陕西省首个通过国家有机茶认证的县**，县域范围内属于**联合国教科文组织生物圈保护区** (联合国世界人与自然生物圈保护区)。土壤含**硒 (Se) 0.653–3.853 ppm** — 比汉中平均值高1.5倍。

自2007年起，宁强雀舌归入区域伞型品牌「汉中仙毫」(汉中仙毫，「汉中神仙毫毛」)，但保持自身工艺和香气特色：若「汉中仙毫」为扁平形带栗香，则「宁强雀舌」为**「雀舌形」** (雀舌形) 具**「高长馥郁」香气** (高长馥郁) 且完全无涩味。

**1. 分类与产地：**

* **类型：** 绿茶 (绿茶, lǜchá)，未发酵。形状 —**「雀舌形」** (雀舌形, quèshé xíng)：略扁、挺秀、带银毫 — 芽叶形似雀鸟吐出的「舌头」。区域品牌「汉中仙毫」(汉中仙毫, 地理标志 2007) 子类别，保持自身特色、工艺和名称。

* **类别：** 地理标志产品「汉中仙毫」(国家地理标志产品, 2007)。**「国家特需用茶」** (国家特需用茶, 1996 — 供应中南海)。陕西首个有机茶 (国家有机茶认证, 2002)。国家级有机茶标准化示范区 (国家级有机茶标准化示范区, 2003)。全国农业科技博览会「后稷」金像奖 (中国农业科技博览会后稷金像奖, 1995)。西安国际贸易洽谈会金奖 (西交会金奖, 2002)。联合国教科文组织生物圈保护区。富硒茶。

* **产地：** 中国陕西省 (陕西省)，汉中市 (汉中市)，宁强县 (宁强县)。核心产区：汉源镇 (汉源镇，「汉江源头」) 和高寨子镇 (高寨子镇)，海拔800–1200公尺，位于汉江和嘉陵江源头流域。

* **地理座标：** ~32°45′–33°15′ 北纬，105°55′–106°35′ 东经。陕西省最西北茶区，秦岭与巴山交汇处。

**2. 历史与文化意义：**

* **历史：**

宁强是陕西省最古老的茶区之一。陆羽在《茶经》(八世纪) 将梁州 (梁州，汉中古名) 列入产茶地区，具体提及「褒城、金牛二县山谷」(梁州生褒城、金牛二县山谷)。金牛 — 宁强古名，与传说中的「金牛道」(金牛道) 相关 — 中国最古老道路之一，自战国时期起连接关中与四川的贸易通道。因此宁强茶叶生产于这条已有2300多年商品（包括茶叶）运输历史的古道上。

现代「宁强雀舌」由县茶叶技术团队于1992年基于当地群体品种创制。1993年通过农业部专家评审。1995年获全国农业科技博览会「后稷金像奖」(后稷金像奖)。**1996年** — 列入「国家特需用茶」— 国家特需用茶：宁强雀舌供应**中南海** (中南海) — 中华人民共和国最高领导层驻地。这是中国少数几个拥有确认「国家领导人用茶」地位的茶叶之一。2002年 — 西安国际贸易洽谈会金奖 (西交会金奖)。同年2002年宁强成为**陕西省首个**获得国家有机茶认证的县，2003年成为**国家级有机茶标准化示范区**。2005年宁强雀舌与午子仙毫 (午子仙毫) 和定军茗眉 (定军茗眉) 组成「汉中三大名茶」(汉中三大名茶)，2007年三个品牌统一于伞型品牌「汉中仙毫」(汉中仙毫) 之下。2017年「汉中仙毫」入选中国茶叶区域公用品牌价值50强 (中国茶叶区域公用品牌价值50强)。

* **名称：** 宁强 (Níngqiáng) — 县名；雀舌 (Què Shé) —「雀舌」— 描述成品茶形状：小巧、略扁、挺秀的芽叶，形似雀鸟吐出的舌头。「雀舌」是绿茶经典类别之一，与「毛尖」和「毛峰」并列。

* **文化意义：** 宁强 —「汉江源头」(汉江源头)。汉江 (汉江) — 长江最大支流，长1577公里，曾赋名于**汉朝** (汉朝, 西元前202年 — 西元220年)、**汉族** (汉族) 和**汉字** (汉字)。来自汉江源头的茶叶 — 是「汉文明源头」的茶。此外，宁强位于「金牛道」— 连接西安和成都穿越秦岭的最古老交通干线。品饮宁强雀舌 — 意味著同时触及陆羽《茶经》、「汉」族认同的源头和「金牛道」2300年历史。

**3. 植物学描述与原料：**

* **品种/栽培种：** 当地群体品种 (本地群体种)，适应秦岭-巴山交汇处北亚热带条件。小叶种，抗寒，芽叶银毫丰富。部分茶园补植福鼎大白茶无性系以增加毫毛。

* **采摘：** 春茶 — 清明前后 (清明，四月初)。标准 — 一芽一叶初展 (一芽一叶初展)。芽叶须完整、挺秀、无缺陷 — 以形成特有的「雀舌」形状。

* **原料生化成分：** 茶多酚 ≥24.73%，胺基酸 3.43%，土壤富含 Se + Zn。

**4. 风土条件与栽培特色：**

宁强位于两大山系交汇处 — 北有秦岭 (秦岭)，南有巴山 (巴山)，形成深邃山谷，溪流潺潺，云雾缭绕。

* **海拔：** 800–1200公尺。核心区 — 1000±200公尺，位于有泉水溪流的山谷中。

* **气候：** 北亚热带。年均温 12.9°C（中国茶区中最低之一 —「寒冷」茶）。降雨量 900–1100毫米。日夜温差 >10°C — 显著，促进胺基酸积累。云雾湿度 85%。秦岭山脉形成天然屏障，阻挡北方寒风，同时允许太平洋湿润气流通过 —「天然温室」环境。

* **土壤：** 花岗岩发育棕壤土 (花岗岩发育棕壤土, pH 4.8–5.5)。有机质 >2.0%。**硒 (Se) — 0.653–3.853 ppm** — 比汉中平均值高1.5倍。这是陕西省硒含量最高的茶园土壤之一。锌含量也偏高。

* **生态环境：** 森林覆盖率 >65%。县域范围属于**联合国教科文组织生物圈保护区** (联合国世界人与自然生物圈保护区)。90%以上茶叶产品产自生态核心区。自2002年（有机认证）起禁用化学农药和肥料。秦岭冰川溪流提供最纯净的灌溉水源。

**5. 制作工艺：**

宁强雀舌工艺结合「机械提效、手工定魂」—「机械化提高效率，手工赋予灵魂」。关键步骤：

* **摊放 (摊放, tān fàng)：** 室温下4–6小时。失水约15%。发展香气前体物质。

* **杀青 (杀青, shāqīng)：** 斜锅炒制，温度160–180°C。快速 — 保持「绿色」鲜爽和香气。采用抛 (抛) 和闷 (闷) 交替 — 经典陕西技法。

* **做形「雀舌」(做形, zuòxíng)：** 关键步骤。用手掌将芽叶轻压成特有「舌形」— 挺秀、略扁、尖端细致。手法 — 轻柔「捺压」(捺, nà)，不用力过度以免压碎叶片和破坏银毫。锅温降至80–90°C。此步骤是宁强雀舌的「灵魂」，也是与扁平「汉中仙毫」的主要区别：「舌形」保持立体感和「透气性」，而扁平「仙毫」完全压平。

* **烘干 (烘干, hōnggān)：** 两阶段 — 初烘100–110°C，足烘70–80°C至含水量≤6.5%。温和的足烘保持「高长馥郁」香气 (高长馥郁)，这是宁强的特色。

* **现代元素：** 光谱分析仪控制硒和茶多酚含量；数位恒温器控制锅温。但「舌形」塑造仍完全手工操作。

**6. 感官特征：**

* **干茶外形：** 略扁挺秀、匀齐似雀舌 (微扁挺秀、匀齐似雀舌)。翠绿显毫 (翠绿显毫)。芽叶 — 整齐，大小均匀。

* **干茶香气：** 「嫩香」(嫩香, nèn xiāng)。「高长馥郁」(高长馥郁, gāo cháng fùyù) — 这是与标准「汉中仙毫」的关键区别，后者以「栗香」(栗香) 为主。宁强雀舌更具「花香」和「兰花香」特质。

* **茶汤香气：** 清香 (清香)，明显的花果香调。冷却后 —「蜜香」韵味。香气持久性 — 冷杯留香3分钟以上。

* **滋味：** 醇厚甘甜 (醇厚甘甜, chúnhòu gāntián)。回甘 (回甘) — 持久清纯。**完全无涩味** (无涩感, wú sè gǎn) — 配方「醇厚甘甜，无涩感」— 与「汉中仙毫」和大多数其他绿茶的关键滋味区别。这是低茶多酚（24.73%）和高胺基酸（3.43%）平衡的结果 — 此平衡源于「寒冷」风土（12.9°C）和富硒土壤。

* **汤色：** 绿亮清澈 (绿亮清澈, lǜ liàng qīng chè)。

* **叶底（冲泡后叶片）：** 嫩绿成朵、肥厚柔软 (嫩绿成朵、肥厚柔软)。芽叶 — 完整，未破碎，保持「舌形」。

**7. 化学成分：**

* **茶多酚 (茶多酚)：** ≥24.73% — 绿茶中等水平，确保柔和无涩味。

* **胺基酸 (氨基酸)：** 3.43% — 较高水平，源于「寒冷」风土（12.9°C）和显著日夜温差（>10°C）。以L-茶胺酸为主。

* **硒 (Se)：** 0.653–3.853 ppm — 比汉中平均值高1.5倍。来自花岗岩富硒土壤。

* **锌 (Zn)：** 偏高 — 来自同样土壤。

* **咖啡碱 (咖啡碱)：** 绿茶标准水平。

* **维生素：** 维生素C，B群维生素。

* **茶多酚与胺基酸比值：** 约7.2 — 「柔和」北方绿茶特征；相比之下南方茶为10–15。正是这个低比值创造了「无涩味」配方。

**8. 保健功效：**

* **富硒特性：** 硒含量高达3.853 ppm — 陕西茶叶中最高之一。硒是必需微量元素，谷胱甘肽过氧化酶 — 关键抗氧化酶的必需成分。定期饮用富硒茶有助维持免疫力和抗氧化防护。

* **抗氧化防护：** 茶多酚（24.73%）+ 维生素C + 硒 — 三重抗氧化系统。

* **提神效果：** 咖啡碱与L-茶胺酸协同 — 温和持久的清醒感，无紧张感。由于茶多酚与胺基酸低比值，效果特别温和。

* **认知功能：** L-茶胺酸刺激大脑α波活动 —「放松专注」状态。

* **消化支持：** 儿茶素温和刺激肠胃蠕动。

* **重要提醒：** 以上功效基于一般资料，非医疗建议。不建议空腹饮用。每日用量不超过600毫升。

**9. 冲泡方法：**

* **水温：** 75–85°C。不可用沸水 — 嫩「舌形」会被「烫伤」失去花香。高等级用75°C。

* **茶量：** 3克配150毫升（1:50比例）。

* **茶具：** 玻璃杯（推荐）— 观赏「雀舌」在水中缓缓下沉的「舞蹈」。白瓷盖碗 — 聚集香气。不建议用宜兴紫砂壶 — 多孔性会「吸收」细腻香气。

* **冲泡法 —「中投法」(中投法, zhōng tóu fǎ)：**
	1. 沸水温杯。
	2. 注入1/3的85°C水。
	3. 投茶。
	4. 摇动茶杯30秒 —「醒茶」。
	5. 注水至7/10满。
	6. 浸泡1.5–2分钟。

* **冲泡次数：** 3–4泡。每次增加30秒。三泡仍保持「高长馥郁」香气 —「三泡余韵绵长」(三泡余韵绵长)。

**10. 储存方法：**

* **温度：** 0–5°C（冰箱），密封包装。宁强雀舌是娇嫩的「芽茶」，对温度、湿度和异味极为敏感。
* **包装：** 密封、不透光。铝箔 + 真空包装最佳。
* **光线：** 完全避光 — 毫毛和叶绿素对光敏感。
* **保存期：** 建议6–8个月内饮用。新茶需「退火」7–10天。

**11. 价格与仿冒：**

宁强雀舌属高价位茶叶，与其他「雀舌」类别相当。特级 — 600–1000元/500克；一级 — 300–600元；大众级 — 100–300元。

* **避免仿冒方法：**
	* 标示 —「宁强雀舌」或「汉中仙毫」并注明「宁强」产地。
	* 形状 — 挺秀「雀舌」(微扁挺秀)，非扁平或卷曲。
	* 香气 —「高长馥郁」(高长馥郁)，无「栗香」调（标准「仙毫」特征）。
	* 滋味 — **无涩味** (无涩感)。有涩味是其他茶冒充的征象。

**12. 趣味知识：**

* **中南海用茶（1996）。**「国家特需用茶」— 国家特需用茶地位。宁强雀舌供应中华人民共和国最高领导层驻地 — 中国少数几个确认「国家领导人用茶」地位的茶叶之一。

* **汉江源头 = 「汉」文明源头。** 汉江 — 赋名于汉朝 (汉朝)、汉族 (汉族) 和汉字 (汉字) 的河流。品饮其源头茶叶 — 触及联系14亿人的「汉」族认同根源。

* **「金牛道」(金牛道)。** 穿越秦岭的最古老道路，自战国时期（西元前5世纪）起连接西安与成都的贸易通道。宁强（古金牛）— 此路要站。茶叶是这条已有2300年历史路线上的商品之一。

* **联合国教科文组织生物圈保护区。** 宁强县域受联合国教科文组织「人与生物圈」(MAB) 计划保护。生物圈保护区种植的茶叶 — 稀有。

* **硒含量高达3.853 ppm。** 陕西省所有茶区中最高之一。相比：「一般」富硒茶含0.3–0.8 ppm。

* **陕西首个有机茶（2002）。** 宁强 — 省内有机认证先驱，为整个汉中设立标准。

* **「无涩味」— 品质配方。**「醇厚甘甜，无涩感」—「醇厚、甘甜、无涩味」。这是独特平衡的结果：茶多酚24.73% / 胺基酸3.43% = 比值约7.2 — 中国绿茶中最低之一。

**13. 与其他「雀舌」及汉中茶叶比较：**

* **汉中仙毫 (汉中仙毫, Hànzhōng Xiān Háo)：** 伞型品牌，统合汉中茶叶。扁平形。「栗香」。宁强雀舌 —「舌形」(雀舌)，「高长馥郁」香气，无涩味。宁强滋味更「花香」和「柔和」。

* **金坛雀舌 (金坛雀舌, Jīntán Què Shé)：** 江苏。同样「雀舌」，但来自华东，无富硒特性。香气 —「清雅」(清雅)。宁强 — 更明显「馥郁」和富硒。

* **定军茗眉 (定军茗眉, Dìngjūn Míng Méi)：** 汉中，「定军山茗眉茶」— 同属「仙毫」品牌。形状 — 细「眉」形。宁强 —「舌」形。两者皆富硒，但来自汉中不同微产区。

**结语：**

宁强雀舌 —「汉江源头的雀舌」，来自赋名整个文明的河流源头。曾供应中南海（1996），种植于联合国教科文组织生物圈保护区，位于「金牛道」— 穿越秦岭的最古老道路上 — 陕西首个有机茶，中国富硒含量最高的茶叶之一（高达3.853 ppm）。其配方 —「醇厚甘甜，无涩感」— 非行销口号，而是独特风土的结果：年均温12.9°C，花岗岩富硒土壤，秦岭与巴山交汇的山谷。以85°C「中投法」冲泡 — 三泡之后余韵仍绵长，如汉水从这些山脉流向伟大的长江。
