# 西山茶 (西山茶, Xīshān chá)

**西山茶 (西山茶, Xīshān chá)**

![Сишань Ча (西山茶, Xīshān chá)](images/Сишань%20Ча%20(西山茶,%20Xīshān%20Chá).png)

![Сишань Ча (西山茶, Xīshān chá)-1](images/Сишань%20Ча%20(西山茶,%20Xīshān%20Chá)-1.png)

西山茶 (西山茶, Xīshān chá) — 著名的广西绿茶，诞生于神圣的佛教名山西山 (西山，「西山」)，寺庙、疗愈之泉乳泉 (乳泉，「乳泉」) 与千年茶树构成不可分割的整体。茶树由唐代僧侣种植，清代时列入「全国二十四名茶」(全国二十四名茶) 并作为贡茶进贡朝廷。1950年代，宽能与昌慧两位比丘尼三次向毛泽东赠送茶叶，获得回复：「好茶，可与龙井媲美」(好茶，可与龙井媲美)。2021年西山茶列入中欧地理标志互认名录 — 贵港市唯一获此地位的茶叶。独特的「四季四香」(四季四香) 现象：春茶 — 清香，夏茶 — 梨香，秋茶 — 醇香，冬茶 — 荷香。

**1. 分类与产地：**

* **类型：** 绿茶 (不发酵，绿茶, lǜchá)。杀青 — 铁锅 (铁锅, tiěguō) 炒制，温度180°C，采用「抖闷结合」(抖闷结合, dǒu mèn jiéhé) 方法。外形 — 条索形 (条索形, tiáosuǒ xíng)。

* **类别：** 国家地理标志保护产品 (国家地理标志保护产品，2010年起)；列入中欧地理标志互认名录 (中欧地理标志互认名录，2021年)。贵港市非物质文化遗产 (2019年)。两度「全国名茶」(全国名茶，1986年、1990年)。中国21种历史名茶之一，清代「二十四名茶」之一。

* **产地：** 中国；广西壮族自治区 (广西壮族自治区, Guǎngxī)；桂平市 (桂平市, Guìpíng Shì)，隶属贵港市 (贵港市, Guìgǎng Shì)。保护区域 — 桂平市全境 (26个乡镇，4074平方公里)。品质核心区 — 西山 (西山，古称思灵山, Sīlíng Shān)、棋盘石 (棋盘石, Qípán Shí) 区域、乳泉井 (乳泉井, Rǔquán Jǐng) 与洗石庵 (洗石庵, Xǐshí Ān) — 海拔700公尺以下，原始森林区域。

* **地理座标：** 约北纬23°10′–24°10′，东经109°30′–110°30′ (桂平市境内)。

**2. 历史与文化意义：**

* **历史：**

西山茶是广西最古老的有文献记载的茶叶之一。据《桂平县志》(《桂平县志》) 记载：「西山茶，出西山棋盘石、乳泉井、观音岩下，矮株散植，根吸石髓，叶映朝暾，故味甘腴，而气芬芳」(西山茶，出西山棋盘石、乳泉井、观音岩下，矮株散植，根吸石髓，叶映朝暾，故味甘腴，而气芬芳)。栽培始于唐代 (七至九世纪)，高僧李明远 (李明远) 从江南带来茶种，种植于棋盘石旁 — 传说中仙人下棋饮天茶之地，路过的樵夫与仙人共度一日，回到人间已过百年。

明代时茶叶闻名华南 — 广东、湖南、福建。清代西山茶列入「全国二十四名茶」，成为贡茶 (贡茶)。光绪年间《浔州府志》(《浔州府志》) 记载：「西山茶，色清而味芬芳，不减龙井」(西山茶，色清而味芬芳，不减龙井)。

二十世纪茶业衰落，但在佛教比丘尼努力下重获新生。1949年比丘尼宽能 (释宽能, Shì Kuānnéng) 受著名佛教改革家巨赞 (释巨赞) 邀请，来到西山，与比丘尼昌慧 (释昌慧, Shì Chānghuì) 共同恢复棋盘石与洗石庵的茶园。1950年代她们三次向毛泽东赠送精品茶叶，主席回复：「好茶，可与龙井媲美」(好茶，可与龙井媲美)，并呼吁扩大生产。

国际认可：1986年、1990年 — 「全国名茶」(商务部)；2010年 — 中国地理标志；2021年 — 列入中欧地理标志互认名录 (贵港市首个且唯一入选茶叶)。至2023年茶园面积20,500亩 (约1367公顷)，年产量1069吨。

* **名称：** 西山 (Xīshān) — 「西山」— 桂平著名佛教名山，南方四大佛教名山 (南方四大佛教名山) 之一。古称思灵山 (思灵山，「思灵山」)；茶 (Chá) — 「茶」。别名：棋盘茶 (棋盘茶，「棋盘茶」) 与棋盘仙茗 (棋盘仙茗，「棋盘仙茗」) — 源于仙人传说。

* **文化意义：** 西山茶是少数与佛教文化密不可分的中国茶叶之一。「禅茶一味」(禅茶一味, chán chá yī wèi，「禅茶一味」) 传统自唐代延续至今：僧尼不仅种植加工茶叶，更将其作为禅修工具。乳泉 — 广西著名自然景观：泉水白如牛乳，被视为冲泡西山茶的理想用水。当地俗语：「山有好景，茶有佳味」(山有好景，茶有佳味)。1950年代茶叶曾赠送摩洛哥国王 — 中华人民共和国早期「茶叶外交」实例之一。

**3. 植物学描述与原料：**

* **品种/栽培种：** 桂平西山茶群体种 (桂平西山茶群体种, Guìpíng Xīshān Chá Qúntǐzhǒng) — 当地灌木型小叶种 *Camellia sinensis* var. *sinensis*。叶片椭圆形，深绿色，茸毛丰富。百芽三叶重约46克。生化成分：胺基酸≥3.2%，多酚≥27.5% (春季原料)。水浸出物约50% — 极高指标。

* **采摘：** 几乎全年 — 二月底至十一月，每季20-30批次。四季产生四种不同香型：春 — 清香 (清香，清淡绿香)，夏 — 梨香 (梨香，梨香)，秋 — 醇香 (醇香，深厚/奶香)，冬 — 荷香 (荷香/莲香，莲香)。

* **采摘标准：** 特级 — 单芽或一芽一叶初展。长度≤4公分。要求大小、长度、色泽均匀。

* **原料要求：** 嫩度佳、完整、新鲜的芽叶。纯手工采摘 — 仅用手，不用剪刀。竹篮运输，避免损伤挤压。

**4. 风土与栽培特色：**

桂平位于广西南部，浔江 (浔江) 河畔，西江 (西江) 支流。西山为平原中的孤立山体，形成独特小气候。

* **种植海拔：** 700公尺以下 (核心区 — 棋盘石、乳泉、观音岩)。主要茶园 — 200-700公尺。

* **气候：** 南亚热带。年均温21.4°C；降雨量1700-2400毫米/年 (中国最湿润茶区之一)；雾日>200天/年；无霜期354天。特色 — 「夏不热而秋热，冬不寒而春寒」(夏不热而秋热，冬不寒而春寒) — 逆转温度模式，刺激芳香物质的特殊累积。

* **土壤：** 黄红砂壤 (黄红砂壤)，花岗岩风化形成。pH 4.5-6.5，有机质2-3%。排水良好，富含花岗岩基质矿物。核心区为水源保护区 (水源保护区)，禁用化肥农药。

* **生态：** 核心茶园位于西山原始森林中，邻近乳泉。高湿度、散射光 (>70%光通量) 与持续雾气为胺基酸与芳香化合物合成创造理想条件。灌溉水源自乳泉，富含钙镁矿物质。西山为华南重要佛教圣地，严格生态管控：国家保护区限制经济活动。这是地理标志标准外的天然环境纯净保证。

**5. 制作工艺：**

西山茶采用传统工艺，列入贵港市非物质文化遗产名录 (2019年)。关键特色 — 「二炒二揉」(二炒二揉, èr chǎo èr róu) 方法，专用竹木器具，最终「轻揉显毫」(轻揉显毫)。

* **采摘 (采摘)：** 手工，标准「一芽一二叶」，≤4公分。

* **摊青 (摊青 — tān qīng)：** 竹盘摊放4-6小时。

* **第一次炒制/杀青 (杀青 — shāqīng)：** 铁锅180°C，「抖闷结合」(抖闷结合) 方法 — 交替抛叶入空中与压贴热面。结合「扬、抛、抖、甩」(扬、抛、抖、甩) 手法。

* **第一次揉捻 (揉捻)：** 轻度手工揉捻 — 形成条索结构。

* **第二次炒制 (初炒 — chū chǎo)：** 定形。

* **竹架烘干 (烘焙 — hōngbèi)：** 80°C传统竹焙笼 (竹焙笼, zhú bèi lóng) 烘干。

* **最终炒制/提香 (复炒 — fù chǎo)：** 最后「精制」炒制，形成栗香与花果香。含水量≤5%。

* **备注：** 全程不用金属工具 (除杀青铁锅) — 竹木器具避免与氧化性金属接触。此原则与信阳毛尖的「三不原则」(三不原则) 相呼应：两个传统独立得出相同结论 — 茶叶与金属接触会降低多酚品质。

**6. 感官特征：**

* **干茶外观：** 细紧微曲条索 (紧细微曲)，黛绿显毫 (黛绿显毫)。「龙卷状」(龙卷状, lóng juǎn zhuàng) — 特征性尖端形状。银毫覆盖芽尖。

* **干茶香气：** 依季节而异。春 — 清香 (清香)；夏 — 梨香 (梨香)；秋 — 醇香 (醇香)；冬 — 莲香 (莲香/荷香)。持久多层次。

* **茶汤香气：** 花果香 (花果香) — 独特「四季四香」特征，大多数绿茶所无。冷却后显现栗香底韵。

* **滋味：** 醇和 (醇和, chún hé)，鲜爽明显 (鲜爽)，回甘持久 (回甘持久)。饱满但无粗涩。耐泡3-6次。

* **汤色：** 碧绿清澈 (碧绿清澈)，透亮。

* **叶底 (茶渣)：** 嫩绿匀整鲜活 (嫩绿匀整鲜活)。

**7. 化学成分：**

* **多酚 (茶多酚)：** ≥27.5%；核心批次可达35.6%。与胺基酸平衡良好，确保柔和无过度涩味。EGCG含量高。

* **胺基酸 (氨基酸)：** ≥3.2% (春茶)。L-茶胺酸为主。

* **水浸出物 (水浸出物)：** 约50% — 绿茶中最高指标之一，证明茶汤极其浓郁「厚实」。

* **咖啡碱 (咖啡碱)：** ≥3.9% — 高于平均，提供明显提神效果。

* **维生素：** 维生素C、B群维生素、维生素E。类黄酮。

* **矿物质：** 钾、锰、氟；花岗岩土壤与乳泉矿泉水中的钙镁微量元素。

**8. 健康功效：**

* **抗氧化保护：** 多酚 (≥27.5%) + 类黄酮 + 维生素C — 全面抗氧化压力保护。

* **提神效果：** 咖啡碱 (≥3.9%) 与L-茶胺酸协同 — 温和持久的清醒感。

* **心血管健康：** 儿茶素有助调节胆固醇与血管弹性。

* **肌肤健康：** 类黄酮与维生素C共同抑制氧化，减少紫外线伤害 — 对广西南部强烈日照地区特别重要。

* **认知功能：** L-茶胺酸刺激大脑α波活动。

* **重要提醒：** 上述功效基于一般资料，非医疗建议。不建议空腹饮用。新茶需放置7天「退火」。开封后10天内饮用完毕。

**9. 冲泡方法：**

* **水温：** 80-85°C。理想用水 — 接近乳泉成分的矿泉水 (软水，弱碱性)。勿用沸水 (>85°C破坏活性物质并产生苦味)。

* **茶量：** 3克配150毫升 (1:50比例)。

* **茶具：** 玻璃杯 — 观察「芽叶竖立」(芽叶竖立)；白瓷盖碗。

* **冲泡程序 (玻璃杯)：**

	1. 温杯，倒掉。
	2. 投茶，80-85°C注水。
	3. 首泡1-2分钟。观察「竖立芽叶」展开。

* **冲泡程序 (盖碗)：** 洗茶5秒 → 首泡20秒 → 每次增加10秒。可泡3-6次。

**10. 储存：**

* **温度：** 0-5°C (冰箱)。
* **容器：** 密封，无异味。
* **光线：** 完全避光。
* **期限：** 建议6个月内饮用，以保持「四季香型」最佳表现。新茶需放置7天「醒茶」(醒茶) 退火。冬季批次 (莲香) 因含水量较低，比春茶保存更久。

**11. 价格与假冒：**

西山茶属中高价位茶叶。特级 — 500元/斤起；一级 — 200-400元/斤；二级 — 200元/斤以下。

* **防伪方法：**

	* 检查地理标志「西山茶」标识。
	* 外观标志 — 「黛绿银尖」(黛绿银尖)。无茸毫即为替代品。
	* 香气 — 花果香，持久。「草腥」或「焦糊」味非西山茶。
	* 用乳泉水 (或软质矿泉水) 冲泡，茶汤展现独特甘甜。

**12. 趣闻轶事：**

* **「不减龙井」：** 此评语出现两次 — 清代县志 (十九世纪) 与毛泽东 (1950年代)。相隔两世纪，评价一致：西山茶达到中国顶级绿茶水准。

* **四季四香：** 独特特征：春 — 清香 (清淡绿香)，夏 — 梨香，秋 — 醇香 (深厚奶香)，冬 — 莲香。此为西山特殊逆转温度模式的结果。

* **棋盘石传说：** 相传樵夫在山上遇见两仙人下棋，品尝仙茶，陪伴一日 — 回到人间发现已过百年。遗忘斧头处长出茶树。故茶又名「棋盘茶」(棋盘茶)。

* **乳泉：** 乳泉 (乳泉) — 从西山花岗岩中涌出的白色「乳状」泉水。被视为冲泡西山茶的理想用水；矿物成分接近欧洲顶级山泉。

* **茶叶外交：** 1950年代西山茶曾赠送摩洛哥国王 — 中华人民共和国早期「茶叶外交」实例。2021年列入中欧名录，开启欧洲市场。

* **比丘尼茶师：** 西山茶复兴归功于两位佛教比丘尼：宽能与昌慧。战后年代她们恢复荒废茶园，开发「轻揉显毫」(轻揉显毫) 技术，成为现代西山茶标志。

**13. 与其他广西绿茶比较：**

* **三江绿茶 (三江绿茶, Sānjiāng Lǜchá)：** 广西柳州。地理位置较「北」。栗香兰花香型，耐泡5-7次。西山更偏「花果香」，具独特冬季莲香。三江规模大 (215,000亩)；西山精品化 (20,500亩)，地位更高。

* **凌云白毫 (凌云白毫, Língyún Báiháo)：** 广西百色。高海拔 (800-1500公尺)，凌云白毫栽培种。更精致的「银毫」茶，茸毛丰富。西山香型更「温暖」，香气更复杂，佛教文化底蕴更深。

* **西湖龙井 (西湖龙井, Xīhú Lóngjǐng)：** 浙江。比较不可避免：两部志书 — 清代与民国 — 均称西山「不减龙井」，毛泽东于二十世纪确认此评价。龙井 — 扁平，栗香坚果香；西山 — 条索，花果香。龙井 — 价格更高，知名度更广；西山 — 较少宣传，但具独特「四季香型」与龙井所无的佛教历史。

**结语：**

西山茶 — 诞生于祈祷与云雾中的茶叶：唐代僧侣将其种植于棋盘石旁，传说仙人在此下棋，乳泉自此以白色矿泉水滋养茶根。十三世纪来，它曾是清代贡茶，获毛泽东赞誉，列入欧洲名录 — 但其真正奇迹不在奖项，而在「四季四香」：春之清香，夏之梨香，秋之醇香，冬之莲香。以80°C软水冲泡 — 让西山茶为您讲述始于两仙人棋局的千年故事。
