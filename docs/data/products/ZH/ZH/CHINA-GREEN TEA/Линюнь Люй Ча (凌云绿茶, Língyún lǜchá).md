# 凌云绿茶 (凌云绿茶, Língyún lǜchá)

**凌云绿茶 (凌云绿茶, Língyún lǜchá)**
![凌云绿茶 (凌云绿茶, Língyún lǜchá)](images/Линюнь%20Люй%20Ча%20(凌云绿茶,%20Língyún%20Lǜchá).png)

凌云绿茶（凌云绿茶，Língyún lǜchá）是产自广西壮族自治区百色市凌云县（凌云县，Língyún Xiàn）的绿茶，采用独特的**凌云白毫种（凌云白毫种，Língyún Báiháo Zhǒng）**制成，该品种又称**华茶26号（华茶26号）**——1965年中华人民共和国农业部认定的首批21个国家级茶树良种之一。这是小乔木大叶种（小乔木大叶种，xiǎo qiáomù dàyè zhǒng）：茶树高达6-9公尺，芽头肥大、肉质丰厚，覆满白色茸毛——因此得名「白毫」（báiháo，「白色茸毛」）。生化成分令人印象深刻：茶多酚含量**35.6%**，儿茶素**182.92毫克/克**——这些指标可与云南大叶种茶相媲美，是中国绿茶中最高的之一。凌云白毫种的独特之处在于可以制作六大茶类（六大茶类）——「一茶千化」（一茶千化，yī chá qiān huà）。1915年该茶在巴拿马国际博览会（巴拿马国际食品博览会二等奖）获得二等奖，2005年获得中国地理标志产品保护。至2024年，该县茶园面积达112,500亩（约7,500公顷），年产量8,326吨，总产值7.16亿元人民币。

**1. 分类与产地：**

* **类型：** 绿茶（绿茶，lǜchá），未发酵茶。有多种形态：**白毫王（白毫王，Báiháo Wáng，「白毫之王」）**——直条针形（针形，zhēnxíng）；**凌螺春（凌螺春，Língluóchūn）**——螺旋形（螺形）；**白毫银针（白毫银针，Báiháo Yínzhēn）**——单芽。制作工艺采用独创的「低温慢炒三青」（低温慢炒三青，dīwēn mànchǎo sān qīng）技术，最后复香（复香，fùxiāng）提升香气。

* **级别：** 国家地理标志产品（国家地理标志产品，2005年；商标2016年）。华茶26号——首批21个国家级茶树良种之一（1965年）。国家级茶树良种（国家级茶树良种，1984年）。巴拿马国际食品博览会二等奖（巴拿马国际食品博览会二等奖，1915年）。多次国际大赛获奖：红茶和黄茶类金奖，绿茶类银奖（第二、三届国际名茶评比）。凌云是首批20个全国无公害茶叶生产示范基地县之一（全国无公害茶叶生产示范基地县，2001年）。

* **产地：** 中国广西壮族自治区（广西壮族自治区，Guǎngxī Zhuàngzú Zìzhìqū），百色市（百色市，Bǎisè Shì），凌云县（凌云县）和乐业县（乐业县，Lèyè Xiàn）。核心产区——岑王老山（岑王老山，Cénwáng Lǎoshān）和青龙山（青龙山，Qīnglóng Shān）山脉，海拔800-1,500公尺。

* **地理座标：** 约北纬24°06′-24°36′，东经106°23′-106°55′。

**2. 历史与文化意义：**

* **300多年栽培历史。** 凌云的茶叶种植记载始于清朝初期（清，1644-1912）。该地区拥有数百年树龄的野生和半野生茶树群，岑王老山保护区内的个别野生茶树高达8.5公尺，据估计树龄超过1,000年。乾隆年间（乾隆，1736-1795），当地茶叶成为「贡茶」（贡茶，gòngchá）——皇室御用茶，证明其早期声誉。

* **1915年——巴拿马国际博览会。** 在著名的1915年巴拿马博览会——20世纪初最大的国际贸易盛会之一——凌云茶获得二等奖（二等奖）。这是广西茶叶首次获得国际认可，将凌云白毫推上世界茶叶舞台。

* **1965年——华茶26号。** 中华人民共和国农业部和中国农业科学院茶叶研究所（中国农业科学院茶叶研究所）对全国茶树品种进行系统评估。凌云白毫被列入首批21个国家级茶树良种，获得华茶26号注册编号——是少数入选这一精英名单的广西品种之一。

* **1984年——国家一级良种。** 中国茶树良种审定委员会（中国茶树良种审定委员会）授予凌云白毫国家级茶树良种（国家级茶树良种）一级地位。同年，该品种在全国农业区划会议上展示，并收录于《中国茶树优良品种集》（《中国茶树优良品种集》）。

* **2005年——地理标志。** 国家质检总局（国家质检总局）批准「凌云白毫茶」为地理标志保护产品。

* **文化意义。** 百色是「红色」（革命）历史之城：1929年12月，邓小平（邓小平，Dèng Xiǎopíng）在此领导百色起义（百色起义，Bǎisè Qǐyì），成为中国共产党斗争史上的关键事件。凌云绿茶是这片「红色」土地的「绿色」象征。凌云是广西最贫困的县之一，茶产业成为脱贫攻坚的关键工具：至2024年，茶园面积112,500亩，为数万户壮族和瑶族（瑶族）少数民族家庭提供就业。凌云白毫种能够制作六大茶类，被誉为「一茶千化」（一茶千化）——这种独特的多功能性在亚洲无与伦比。

**3. 植物学描述与原料：**

* **品种：** **凌云白毫种（凌云白毫种，Língyún Báiháo Zhǒng）**，又称**华茶26号（华茶26号）**——*Camellia sinensis* var. *sinensis*（根据另一分类可能为var. *pubilimba*，待确认）。小乔木型（小乔木，xiǎo qiáomù）。大叶类（大叶类，dàyè lèi），中生种（中生种，zhōngshēng zhǒng）。有性繁殖系（有性繁殖系，yǒuxìng fánzhí xì）。植物学特征：
  * **高度**——自然生长可达6-9公尺；茶园中通过修剪保持在1-1.5公尺。
  * **树型**——半张开型（半张开，bàn zhāngkāi），分枝相对稀疏。
  * **叶片**——大型，椭圆形，水平或略下垂。深绿色，无光泽，叶面隆起（叶面隆起）。叶脉明显。背面（下表面）密被茸毛。
  * **芽头**——肥大、肉质，覆满长而密的白色茸毛（茸毫长而密）。一芽三叶百芽重（一芽三叶百芽重）平均99克——显著高于中叶种。
  * **春梢**——在凌云于3月中旬开始生长；采摘高峰期（一芽三叶）为3月底至4月初。
  * **花期**——11-12月，结实稀少。

* **生化成分（据中国农科院茶叶研究所数据）：**
  * 氨基酸——**3.36%**。
  * 茶多酚——**35.6%**——绿茶中最高之一。
  * 儿茶素——**182.92毫克/克**——卓越的抗氧化潜力。
  * 咖啡碱——**4.91%**。
  * 水浸出物——高。

* **等级：**
  * **特级：** 一芽一叶，茸毛≥90%，板栗香（板栗香）。白毫王形态——直条针形。
  * **一级：** 一芽二叶，紧结。
  * **二级：** 成熟叶，大众产品线。

**4. 风土条件与栽培特点：**

* **地形。** 凌云是位于云贵高原与广西盆地交界处的山区县。地形为喀斯特（喀斯特，kāsītè）山地、深峡谷、山涧。茶园主要分布在岑王老山和青龙山山脉的坡地上，海拔800-1,500公尺——云雾带。典型地貌类型为「峡谷溪涧茶园」：茶树沿著狭窄峡谷中的山涧种植，获得自然遮荫和较高湿度。

* **气候：** 亚热带季风气候（亚热带季风气候）。年均温19-23°C。年降雨量1,700-1,800毫米。云雾天数超过100天/年。日夜温差大，特别是在1,000-1,500公尺区域。云雾带空气湿度≥90%。

* **土壤：** 发育于岑王老山和青龙山的**火山岩风化土**（火山岩风化土，huǒshān yán fēnghuà tǔ）。酸性（pH 4.5-5.5）。有机质含量3.0-4.5‰。富含矿物元素。因岩石基底排水良好。

* **生态：** 森林覆盖率**76.7%**——广西茶区中最高之一。水资源——山涧，一级水质。核心产区——玉洪乡（玉洪乡，Yùhóng Xiāng）和加尤镇（加尤镇，Jiāyóu Zhèn）。生物多样性高：喀斯特森林支持丰富的动植物群，茶叶害虫的天敌充足，可最大限度减少化学防治。

* **规模。** 至2024年：茶园112,500亩（约7,500公顷），年产干茶8,326吨，总产值7.16亿元。凌云是广西最大的产茶县，也是华南重要产茶县之一。

**5. 制作工艺：**

独创技术「低温慢炒三青」（低温慢炒三青，dīwēn mànchǎo sān qīng）——「低温慢速三次炒青以最大程度保留茸毛香气」。全程使用竹木器具加工（全程竹木器具加工，quánchéng zhú mù qìjù jiāgōng）——金属不接触茶叶，防止氧化变黑和金属味。

* **摊青（摊青，tān qīng）：** 摊放时间较长（摊放时间较长）——比大多数绿茶更长。让肥大、肉质的芽头均匀失水并形成香气前体。

* **杀青（杀青，shā qīng）：** 温度180-200°C。杀青程度偏重（杀青程度偏重）：彻底破坏大叶种高多酚含量（35.6%）原料中的残余酶活性，防止褐变。杀青后立即**扇风散热**（扇风散热，shàn fēng sàn rè）防闷黄（防闷黄，fáng mèn huáng）——关键步骤，防止自发转化为黄茶类别。

* **揉捻（揉捻，róuniǎn）：** 初步成形——白毫王为直条针形，凌螺春为螺旋形。

* **「慢速三青」（炒干——三青，chǎo gān — sān qīng）：** 关键步骤。低温、慢速、三次循环：炒制→冷却→再炒→冷却→最后炒制。「小火长时间多次慢抛」（小火长时间多次慢抛）。目的——最大程度保留白毫（白毫）并形成深厚的板栗香（板栗香）而不焦糊。这种方法使凌云绿茶有别于其他大叶种绿茶。

* **复香（复香，fùxiāng）：** 温和加热以「提升」香气并将水分降至≤6%。

**6. 感官特征：**

* **干茶外形：** 因形态而异。**白毫王**——直条针形，覆满白毫，形似银针（形似银针，xíng sì yínzhēn）。**凌螺春**——紧结螺旋形。色泽——绿色覆满白毫（条索紧结、白毫显露，tiáosuǒ jǐnjié, báiháo xiǎnlù）。五大特色：「色翠、毫多、香高、味浓、耐泡」——「翠绿、毫多、香高、味浓、耐泡」。

* **干茶香气：** 板栗香（板栗香，bǎnlì xiāng）——主要且最具特色。清香（清香，qīngxiāng）。「毫香」（毫香，háoxiāng）——细腻的「玉米」香调，为多毫茶叶特有。

* **茶汤香气：** 板栗香——深沉、温暖、持久。「毫香」——冷却时出现的甜香。香气在杯中持续很长时间，即使冲泡后的叶底几天后仍保留香气——罕见特性。

* **滋味：** 鲜爽（鲜爽，xiānshuǎng）。浓醇（浓醇，nóngchún）——茶汤饱满，结构明显，这在绿茶中不常见，归因于高多酚含量（35.6%）。回甘（回甘）——稳定、持久。耐泡性——4-5次完整冲泡不失特色，这也不寻常，可归因于大叶种原料。

* **汤色：** 嫩绿清澈（嫩绿清澈，nèn lǜ qīngchè）。第二、三泡时略带黄色调。

* **叶底（冲泡后的茶叶）：** 青橄榄色、匀嫩亮润（青橄榄色、匀嫩亮润，qīng gǎnlǎn sè, yún nèn liàng rùn）。叶片大、完整，可见茸毛。在玻璃杯中冲泡时——叶片「站立」下沉，如春笋——视觉效果佳。

**7. 化学成分：**

* **茶多酚（茶多酚）：** **35.6%**——中国绿茶中最高之一，可与云南大叶种茶（滇绿、普洱生茶）相媲美。高含量归因于大叶种品种、亚热带气候的强烈日照和刺激酚类化合物合成的火山土壤。

* **儿茶素（儿茶素，ér chá sù）：** **182.92毫克/克**——卓越的抗氧化潜力。主要成分——EGCG（表没食子儿茶素没食子酸酯）、EGC、ECG、EC。相比之下：典型的中叶种绿茶含80-120毫克/克儿茶素。

* **氨基酸（氨基酸）：** **3.36%**——适中水平，为南方大叶种特征。酚氨比（酚氨比）高（约10.6:1），决定了「浓厚、有结构」的风味特征，具有明显的涩感和持久的余韵。

* **咖啡碱（咖啡碱）：** **4.91%**——偏高，提供明显的提神效果。

* **维生素：** 维生素C——因「低温慢速三青」而得以保留。B族维生素（B₁、B₂、B₃）。维生素E。

* **矿物质：** 钾、镁、锰、锌、铁、氟。火山基质提供丰富的矿物质谱。

* **独特特点：** 大叶种、35.6%茶多酚和182.92毫克/克儿茶素与3.36%氨基酸的组合，创造出绿茶中不典型的特征，接近「普洱型」——强调茶体、结构和耐泡性，而非轻盈和「鲜爽」。

**8. 保健功效：**

* **强大的抗氧化作用。** 儿茶素182.92毫克/克——绿茶中最高浓度之一。EGCG——主要儿茶素——被认为是最强的天然抗氧化剂之一，可中和自由基并延缓细胞老化。

* **明显的提神效果。** 咖啡碱4.91%——高于绿茶平均水平。提供活力、改善注意力和工作记忆。与L-茶氨酸结合——无紧张感。

* **支持消化。** 高茶多酚含量（35.6%）刺激消化酶分泌，支持健康的肠道菌群。

* **心脏保护。** 儿茶素和茶多酚参与调节胆固醇水平，减少低密度脂蛋白氧化，维持血管弹性。

* **支持脂质代谢。** EGCG激活产热和脂肪酸氧化——潜在的代谢支持。

* **抗菌作用。** 儿茶素抑制多种病原菌生长，包括变形链球菌（蛀牙）和幽门螺杆菌。

* **增强免疫力。** 茶多酚和维生素C、E复合物具有免疫调节作用。

* **认知支持。** L-茶氨酸刺激大脑α波，改善注意力和放松而不镇静。

**9. 冲泡方法：**

* **水温：** 80-85°C。高多酚含量的大叶种原料需要适中的温度以避免过度苦涩。

* **茶叶用量：** 3克/150毫升（比例1:50）。

* **茶具：** 玻璃杯（观察茸毛针叶的「舞动」）。盖碗。瓷壶。

* **步骤：**
  1. 用热水温杯，倒掉。
  2. 投入茶叶。
  3. **上投法**（上投法，shàng tóu fǎ）：先注水，后投茶。凌云白毫的肥大、肉质芽头缓慢下沉并在杯中「站立」——视觉效果如春笋。
  4. 第一泡——30-45秒。
  5. 倒出茶汤。
  6. 重复冲泡——**4-5次**，每次增加15-30秒。耐泡性——「五大特色」之一（耐泡）：即使第五泡仍保持特色。

**10. 储存方法：**

* 密封包装（铝箔袋真空包装），冰箱0-5°C。保质期——最长12个月。

* 长期储存——冷冻（-18°C）真空包装。

* 茶叶的敌人：湿气、异味、光线、高温。广西亚热带气候——湿度高；不建议在冰箱外储存。

**11. 价格与防伪：**

* **价格区间：** 特级（白毫王，一芽一叶）——500-1,500元/500克，视生产商和年份而定。凌螺春——200-600元/500克。大众级——100元/500克起。

* **如何避免假货：**
  * **地理标志标记。** 正品标有「凌云白毫茶」或「凌云绿茶」。
  * **芽头大小。** 正宗凌云白毫——**肥大、肉质**芽头，**覆满白毫**。细小芽头——可能是中叶种替代品。
  * **香气。** 板栗香（板栗香）+「毫香」（毫香）。无焦味、酸味。
  * **耐泡性。** 正品可冲泡4-5次。两泡后失味——可能是假货。
  * **价格。** 特级不可能低于300元/500克。以「凌云白毫」名义的廉价产品值得怀疑。

**12. 趣闻轶事：**

* **1915年——巴拿马奖章。** 凌云白毫是广西首批获得国际认可的茶叶之一。1915年巴拿马博览会也让西湖龙井、信阳毛尖等中国名茶声名远播。

* **华茶26号——树高达9公尺。** 小乔木型——对绿茶来说异常高大。中国大多数绿茶产自1-1.5公尺高的中叶种灌木；凌云白毫自然生长可达三层楼高。

* **「一茶千化」（一茶千化）。** 凌云白毫是少数（据某些资料，是亚洲唯一）可制作六大茶类的品种：绿茶、红茶、白茶、黄茶、黑茶和乌龙茶。凌云红茶曾获国际比赛金奖。

* **峡谷茶园。** 凌云的茶园沿峡谷山涧（峡谷溪涧）分布——独特的地貌类型，提供自然遮荫、高湿度和防风保护。

* **儿茶素182.92毫克/克。** 中国绿茶中最高之一——可与传统上被认为是「儿茶素冠军」的云南大叶种茶相媲美。相比之下：典型的龙井含100-130毫克/克儿茶素。

**13. 与其他绿茶的比较：**

* **云南大叶种绿茶（滇绿，滇绿，Diānlǜ）。** 两者都是大叶种，茶多酚和儿茶素含量高。滇绿——来自云南大叶种（*C. sinensis* var. *assamica*）；凌云——来自凌云白毫（*C. sinensis* var. *sinensis*，小乔木型）。滇绿——更涩、「粗犷」；凌云——有明显的板栗香和「毫」的柔和。两者都具有强大的抗氧化潜力。

* **桂平西山茶（桂平西山茶，Guìpíng Xīshān Chá）。** 同为广西绿茶，来自小叶种。西山——轻盈、花香；凌云——浓厚、板栗香。西山——来自平原丘陵（200-500公尺）；凌云——来自山地峡谷（800-1,500公尺）。规模不同：凌云——112,500亩，西山——显著较小。

* **洞庭碧螺春（洞庭碧螺春，Dòngtíng Bìluóchūn）。** 凌螺春（凌螺春）——凌云的螺旋形——特意以碧螺春命名。两者都是螺旋形、多毫。碧螺春——小叶种，花果香；凌螺春——大叶种，板栗香。碧螺春——60,000-80,000芽/公斤；凌螺春——因芽头大而显著较少。

* **白沙绿茶（白沙绿茶，Báishā Lǜchá）。** 海南绿茶，独特地质（陨石坑）。两者都来自非标准地质区（凌云的火山岩，白沙的陨石坑）。白沙——热带，全年生长；凌云——亚热带，春季高峰。白沙——小叶种；凌云——大叶种。

**结语：**

凌云绿茶——来自「红色」百色的茶叶，诞生于青龙山峡谷的火山土壤上，来自高达九公尺的小乔木茶树，儿茶素含量182.92毫克/克，拥有1915年巴拿马奖章。华茶26号品种——中国首批国家级茶树良种之一——产出具有浓厚板栗香、「毫」的柔和感和可与云南巨人媲美的抗氧化潜力的茶叶。五大特色——翠绿、毫多、香高、味浓、耐泡——以及一棵树能制作六大茶类的能力，使凌云白毫成为中国最多功能且被低估的茶树品种之一。
