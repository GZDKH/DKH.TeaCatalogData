# 金坛雀舌 (金坛雀舌, Jīntán què shé)

**金坛雀舌 (金坛雀舌, Jīntán què shé)**
![Цзиньтань Цюэ Шэ (金坛雀舌, Jīntán què shé)](images/Цзиньтань%20Цюэ%20Шэ%20(金坛雀舌,%20Jīntán%20Què%20Shé).png)

金坛雀舌 (金坛雀舌, Jīntán què shé) — 来自江苏省常州市金坛区的著名绿茶，其名称「金坛的雀舌」准确描述了其形状：微小扁平的茶叶，修长而尖锐，宛如细小的鸟舌。「雀舌」(què shé，「雀舌」) 这一名称最早在十一世纪由学者沈括 (沈括, Shěn Kuò) 在其著名著作《梦溪笔谈》(梦溪笔谈，「梦溪笔谈」) 中记载。此茶产于神圣道教名山茅山 (茅山, Máoshān) 东坡 — 中国最重要的道教中心之一。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵茶）。属于扁形炒青绿茶 (扁形炒青绿茶, biǎnxíng chǎoqīng lǜchá)。

* **类别：** 中国国家地理标志产品 (中国国家地理标志产品, 2012年) — 常州市第一个获得此保护的茶叶产品。1986年获得中华人民共和国商业部「全国名茶」(商业部"全国名茶") 称号。

* **产地：** 中国江苏省 (江苏, Jiāngsū) 常州市 (常州市, Chángzhōu Shì) 金坛区 (金坛区, Jīntán Qū)。地理标志保护区域涵盖薛埠镇 (薛埠镇)、指前镇 (指前镇)、朱林镇 (朱林镇) 和直溪镇 (直溪镇)。

* **核心产区：** 方麓茶场 (方麓茶场) 和茅山东麓 (茅山东麓) 古茶园，面积约6000亩（400公顷）。最高品质来自茅山风景区内的茶园。

* **地理座标：** 约北纬31°33′，东经119°32′。

**2. 历史与文化意义：**

* **历史：** 金坛地区的茶叶种植有隋朝（581–618）的文献记载：当时当地茶叶已被列入地方特产名录。北宋时期（960–1127），学者百科全书编纂者沈括 (沈括, 1031–1095) 在《梦溪笔谈》(梦溪笔谈, Mèngxī Bǐtán) 中记录了「雀舌」(què shé，「雀舌」) 这一名称 — 这是文献中最早提及此茶叶术语的记载之一。

	金坛雀舌的工业化历史始于1905年，当时成立了「茅麓明农树艺公司」(茅麓明农树艺公司，「茅麓明农树艺公司」)，开始在茅山坡系统性种植茶叶。1919年，企业家纪振纲 (纪振纲, Jì Zhèngāng) 扩大生产，创建了农业企业「茅麓农林场」(茅麓农林场)。

	1982年，金坛县多管局 (金坛县多管局) 启动科学项目，基于当地原料和历史传统创造新型扁形绿茶。1985年，该茶正式命名为「金坛雀舌」并通过省级认证。1986年获得商业部「名茶」称号。2012年获得地理标志保护。

* **名称：**

	* 「金坛」(金坛) — 「金坛」：地区名称，与茅山道教传统相关。
	* 「雀」(雀) — 「雀」：指茶叶的微小尺寸。
	* 「舌」(舌) — 「舌」：描述扁平、略尖的形状，宛如鸟舌。

* **文化意义：** 金坛雀舌是少数与道教名山茅山相关的茶叶之一 — 茅山是上清派 (上清派, Shàngqīng Pài) 的中心，这是最具影响力的道教传统之一。茅山是江苏南部道教的「精神首都」，来自其山坡的茶叶具有象征性的「道教」纯净和长寿光环。

**3. 植物学描述与原料：**

* **品种/栽培种：** 生产中使用几种中叶和小叶型 *Camellia sinensis* var. *sinensis* 栽培种：
	* **龙井43 (龙井43)** — 早熟克隆品种，叶片整齐修长。
	* **祁门槠叶种 (祁门槠叶种)** — 来自安徽南部的中叶型品种。
	* **鸠坑种 (鸠坑种)** — 传统浙江栽培种。
	* **浙农113 (浙农113)** — 高产品种。
	所有栽培种均为灌木型，芽叶肥厚，叶片中等大小，茸毛较少，胺基酸含量高。

* **采摘：** 早春采摘。特级 (特级) 仅采用纯芽 (全单芽, quán dān yá)。生产500克特级茶需要40,000–45,000个芽头。一级采一芽一叶初展。二级采一芽一叶。

* **原料要求：** 嫩度一致、芽头肥厚、无损伤。当日采摘当日加工。

**4. 风土条件与栽培特点：**

* **地形：** 茅山东坡 — 丘陵地带，坡度平缓，非常适合茶园种植。

* **海拔高度：** 海拔300–800公尺。

* **气候：** 北亚热带季风气候。年平均气温15.3°C，年降水量≥1600毫米。年平均雾日120天。丰富的散射光 (漫射光) 有利于胺基酸和芳香化合物的积累。

* **土壤：** 黄棕壤 (黄棕壤)，pH值4.5–6.0，土层深厚（≥1公尺），有机质含量≥1.5%，富含矿物质。无工业污染。森林覆盖率超过30%。众多水库和池塘确保稳定湿度。

**5. 制作工艺：**

金坛雀舌工艺结合机械和手工技法，关键的整形阶段为手工操作。

* **摊放与萎凋 (鲜叶摊放 — xiānyè tānfàng)：** 摊放4–7小时，去除多余水分，启动香气形成过程。

* **杀青 (杀青 — shāqīng)：** 在滚筒 (滚筒) 中于100–120°C进行 — 温和杀青，保持原料嫩度。

* **摊凉 (摊凉 — tānliáng)：** 中间冷却。

* **整形 (整形 — zhěngxíng)：** 关键工序。师傅运用「搭」(dā，「搭」)、「压」(yā，「压」) 和「抓」(zhuā，「抓」) 手法，塑造扁平的「雀舌」形状。锅温按「高—低—高」(高—低—高) 变化，确保形状和香气的最佳平衡。

* **干燥/辉干 (干燥/辉干 — gānzào / huīgān)：** 在70–80°C温和干燥至含水量≤6%。固定栗香。

**6. 感官特征：**

* **干茶外形：** 扁平挺秀 (扁平挺秀, biǎnpíng tǐngxiù)，形如雀舌 (状如雀舌)：尖端锐利，「如鸟喙」。色泽翠绿油润 (绿润)。显毫 (显毫) 轻微。高级品有金毫隐现 (金毫隐现)。

* **干茶香气：** 清高持久 (清高)。栗香明显持久 (栗香明显持久, lì xiāng míngxiǎn chíjiǔ) — 浓郁而持久。清新绿色调。某些批次带有淡雅花香 (花香)，这是整形过程中轻微发酵所致。

* **茶汤香气：** 栗香持久，清香基调。层次渐进展开。

* **滋味：** 鲜爽 (鲜爽, xiānshuǎng) — 高胺基酸含量带来明显的「鲜味」。甘甜 (甘, gān)。醇厚 (醇厚, chúnhòu)。回甘持久 (回甘持久, huígān chíjiǔ)。涩味极轻。

* **汤色：** 明亮 (明亮) 透澈，黄绿色。

* **叶底（冲泡后茶叶）：** 嫩匀成朵 (嫩匀成朵)。色泽嫩绿鲜活。

**7. 化学成分：**

* **多酚类（儿茶素）：** 含量≥25% — 高于扁形绿茶平均水准。提供强大抗氧化潜力。

* **胺基酸（包括L-茶胺酸）：** 含量显著高于绿茶平均水准 — 这是丰富散射光（120个雾日）和有机质含量≥1.5%的肥沃土壤的结果。

* **生物碱：** 咖啡因 — 含量高于平均水准（研究显示显著高于普通绿茶）。可可碱、茶碱。

* **茶黄素 (茶黄素)：** 含量显著 — 有助于调节血脂。

* **矿物质：** 锰 (锰, měng) 含量高 — 有助于骨骼强化。钾、镁、锌、氟。

* **维生素：** 维生素C、B群维生素。

**8. 保健功效：**

* **抗氧化作用：** 多酚类（≥25%）有效中和自由基。
* **胆固醇控制：** 茶黄素调节血脂水准。
* **骨骼强化：** 高锰含量有助于维持骨密度。
* **抗菌作用：** 儿茶素抑制口腔和肠道病原菌。
* **提神效果：** 咖啡因和L-茶胺酸提供温和提神。

* **重要提醒：** 上述功效基于公开资料，不构成医疗建议。

**9. 冲泡方法：**

* **水温：** 80°C（沸水放凉）。

* **茶量：** 3克配150–180毫升水（比例1:50–1:60）。

* **茶具：** 玻璃杯或白瓷盖碗。

* **冲泡过程：**

	1. 温杯，倒掉水。
	2. 投入3克茶叶。
	3. 注水至1/3容量，「润茶」30秒。
	4. 续水至7/10容量。
	5. 浸泡1–2分钟。
	6. 饮用至剩余1/3时续水。可冲泡3次。

* **注意：** 新购茶叶建议放置约2周以去除「火气」。避免长时间浸泡 — 单宁会增加苦涩味。

**10. 储存方法：**

* 密封保存于阴凉干燥处。
* 最佳条件 — 冰箱0–5°C。
* 保存期限 — 最多12个月。
* 开封后 — 1–2个月内饮用完毕。

**11. 价格与仿冒：**

金坛雀舌价格差异很大。特级 (特级) 纯芽茶 — 每斤（500克）1000元人民币以上。一级和二级相对便宜许多。

* **避免仿冒的方法：**

	* **向可信赖商家购买**，要有金坛地区地理标志标识。
	* **评估外形：** 特征性「雀舌」— 扁平、修长、尖锐。圆形或不规则茶叶是其他茶类。
	* **评估香气：** 持久栗香是标志性特征。缺乏栗香值得怀疑。
	* **检查茶汤：** 明亮透澈。混浊是仿冒迹象。
	* **注意价格：** 异常低价是仿冒迹象。

**12. 趣味知识：**

* 「雀舌」(què shé，「雀舌」) 这一名称是茶叶术语中最富诗意的之一。它由沈括在《梦溪笔谈》（十一世纪）中记录 — 这是中世纪中国最伟大的科学著作之一，包含天文学、数学、药理学和植物学观察。

* 茅山是江苏南部道教三大「圣峰」之一，上清派中心。来自其山坡的茶叶不仅是饮品，更是道教「养生」(养生, yǎngshēng) 文化的一部分。

* 生产500克特级茶需要40,000–45,000个纯芽 — 每个都是手工采摘并按大小筛选。

* 1905年「茅麓明农树艺公司」成为中国最早的现代茶业企业之一 — 远早于行业大规模现代化。

* 金坛雀舌成为常州市第一个获得国家地理标志保护（2012年）的茶叶产品 — 象征茅山风土在国家层面的认可。

**13. 与其他「雀舌」类茶叶比较：**

* **蒙顶甘露 (蒙顶甘露)：** 来自四川。卷曲（非扁平）叶片，花香兰韵。甘露更「甘露」甜美；金坛更栗香结构化。

* **蒲江雀舌 (蒲江雀舌)：** 来自四川。同样是「雀舌」，但来自不同风土。蒲江更温和；金坛多酚含量更高，栗香更持久。

* **西湖龙井 (西湖龙井)：** 扁形茶，豆香栗香。龙井更「油润」宽阔；金坛雀舌更精巧尖锐，栗香更「纯净」。

* **宁强雀舌 (宁强雀舌)：** 来自陕西。同样是「雀舌」，但来自西北山区风土。宁强更清淡；金坛更醇厚。

**结语：**

金坛雀舌是一款茶，千年道教圣地茅山、中世纪典籍中的诗意「雀舌」形象与现代制茶工艺在微小扁平茶叶中完美结合，形状无可挑剔。其持久的栗香、鲜甜滋味和修长轮廓 — 这些都是茅山东坡的恩赐，那里一年有120天云雾缭绕，土壤蕴藏著千年的矿物财富。这是一款适合欣赏含蓄优雅的茶：不是碧螺春的果香明亮，也不是龙井的豆香强劲，而是安静、自信的栗香甘甜 — 如山坡上的晨光，道教隐士曾在此寻求不朽。
