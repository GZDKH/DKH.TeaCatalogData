# 茉莉飘雪 (茉莉飘雪, mòlì piāo xuě)

**茉莉飘雪 (茉莉飘雪, mòlì piāo xuě)**
![Моли Пяо Сюэ (茉莉飘雪, mòlì piāo xuě)](images/Моли%20Пяо%20Сюэ%20(茉莉飘雪,%20Mòlì%20Piāo%20Xuě).png)

![Моли Пяо Сюэ (茉莉飘雪, mòlì piāo xuě)-1](images/Моли%20Пяо%20Сюэ%20(茉莉飘雪,%20Mòlì%20Piāo%20Xuě)-1.png)

茉莉飘雪是四川茉莉花茶学派的名片，这款茶在名称中就蕴含著诗意：「飘雪」——雪白的茉莉花瓣在深绿色茶叶间飞舞，如同翡翠池塘上空的初雪。与大多数经典茉莉花茶在薰制后完全去除花瓣不同，飘雪刻意在成品中保留干燥的茉莉花瓣，将品茶变成一场视觉盛宴。这款茶是四川「飞雪」风格 (飘雪, piāo xuě——「飞雪」) 的典型代表，茶香胜过花香，茶汤具有特有的厚度和清新的涩味。

**1. 分类与产地：**

* **类型：** 花茶 (花茶, huāchá)，以绿茶（不发酵茶）为基础，用茉莉花薰制。特色是成品中保留干燥茉莉花瓣。
* **类别：** 中国高品质花茶。四川茉莉花茶学派 (四川茉莉花茶, Sìchuān mòlìhuāchá) 的代表，与福建学派不同，使用当地茶树品种和炒青工艺 (炒青, chǎoqīng——炒制杀青)。
* **产地：** 中国四川省 (四川, Sìchuān)。最珍贵的飘雪产于峨眉山 (峨眉山, Éméishān) 周边——中国四大佛教圣山之一——以及夹江县 (夹江, Jiājiāng)。茶叶基础来自峨眉山、蒙顶山 (蒙顶山, Méngdǐngshān) 和四川南部地区的高山茶园。薰制用的茉莉花传统上种植于四川犍为县 (犍为, Qiánwéi)，现代大规模生产中也使用广西横县 (横县, Héngxiàn) 的茉莉花。
* **地理座标：** 峨眉山地区——约北纬29°35′，东经103°20′。夹江——北纬29°44′，东经103°34′。蒙顶山——北纬30°04′，东经103°12′。
* **别名：** 茉莉「飘雪」、Jasmine Snow Flurry（英文）。「飘雪」(飘雪) 已成为整个四川茉莉花茶系列的通用名称：有「碧潭飘雪」(碧潭飘雪, Bìtán Piāo Xuě——「翡翠潭上飘雪」)、「林湖飘雪」(林湖飘雪)、「峨顶飘雪」(峨顶飘雪) 等品种。「碧潭飘雪」是最知名的商标，由茶艺师徐金华 (徐金华, Xú Jīnhuá) 于1980年代创制，后来转让给竹叶青公司 (竹叶青, Zhúyèqīng)。

**2. 历史与文化意义：**

* **历史：** 四川茉莉花茶薰制传统根深蒂固，虽然在规模和知名度上历史上不及福建学派。四川是中国最古老的产茶区之一：早在晋代编纂的《华阳国志》(华阳国志, Huáyáng Guózhì) 中就记载，南安（今乐山，乐山）和峨眉山地区出产著名茶叶。然而「飘雪」作为独立的茉莉花茶类型，直到20世纪末才形成。关键人物是茶艺师徐金华 (徐金华, Xú Jīnhuá)，四川新津县 (新津, Xīnjīn) 人，他在1980年代基于明代历史薰制技术（包括徐渭 (徐渭) 的「浪园」方法和闵汶水 (闵汶水) 大师的「明茶」传统），创制了「碧潭飘雪」——用峨眉山春芽制成的茉莉花茶，保留茉莉花瓣。1993年在四川省茶叶学会常务理事会上正式发表并命名为「碧潭飘雪」。1995年通过四川省茶叶标准化委员会认证，确认品质达到特级茉莉花茶水准。后来在2008年，品牌转让给竹叶青茶业 (竹叶青茶业)，实现生产现代化。徐金华被尊称为「徐公」(徐公, Xú Gōng——「徐先生」)，他的茶被称为「徐公茶」(徐公茶, Xú Gōng Chá——「徐先生茶」)。
* **名称：**
    * 「茉莉」(茉莉, Mòlì)——茉莉花。
    * 「飘」(飘, Piāo)——飘浮、在空中飞舞、旋转。
    * 「雪」(雪, Xuě)——雪。
    * 完整意象：雪白的茉莉花瓣在茶汤中飞舞飘浮，如同雪花。名称来源于画家邓岱昆 (邓岱昆, Dèng Dàikūn) 为徐金华的茶创作的藏头诗：「碧岭拾毛尖，潭底汲清泉，飘飘何所似，雪梅散人间」——「翡翠山岭采毛尖，/潭底汲取清泉水，/飘飘何物可比拟？/雪梅散落满人间」。每行首字组成「碧潭飘雪」(碧潭飘雪)。
    * 作者对「碧潭飘雪」的诠释：「碧」(bì)——茶色；「潭」(tán)——茶杯（如潭水）；「飘」(piāo)——飘浮散香的花瓣；「雪」(xuě)——雪白茉莉。
* **文化意义：** 飘雪成为四川茶文化的象征和地方骄傲。四川是茶馆 (茶馆, cháguǎn) 的发源地，「飘雪」风格的茉莉花茶传统上与成都 (成都, Chéngdū) 的悠闲生活方式相关，那里的茶馆至今仍是社交生活的中心。诗人画家黄纯尧 (黄纯尧, Huáng Chúnyáo) 品尝徐金华的茶后，为其赋诗，称其为「初春嫩芽天赐礼」。

**3. 植物学描述与原料：**

* **茶叶原料（茶坯，茶坯, chápī）：** 飘雪使用四川当地茶树品种 *Camellia sinensis* var. *sinensis* 的嫩春芽和幼叶。主要品种——福选早 (福选早)，来自峨眉山、蒙顶山和四川南部（宜宾，宜宾）地区的品种。茶叶基础采用炒青工艺 (炒青, chǎoqīng——锅炒杀青) 处理，这是四川飘雪与福建烘青基础茉莉花茶的根本区别。炒制基础赋予茶叶更明显的「茶味」特色、厚度和特有涩味，平衡茉莉花的甜味。
* **采摘标准：** 一芽一二叶 (一芽一二叶, yī yá yī èr yè)。专用春茶（清明前，清明）或早春茶（明前，míngqián）采摘。海拔600-1500公尺的峨眉山茶园，由于「华西雨屏」(华西雨屏, Huáxī Yǔpíng——四川西部常年云雾露水的微气候) 条件下植被早期萌发，产出特别嫩的芽叶。
* **茉莉花：** 夏季采摘的新鲜 *Jasminum sambac* (L.) Ait. 花朵（伏季，fújì）。传统四川茉莉花来自犍为县 (犍为)，现代大规模生产中广泛使用广西横县茉莉花。花蕾在下午开始绽放时采摘。
* **原料要求：** 茶芽——挺直、笔直，有明显白毫 (白毫, báiháo)，无损伤。徐金华的原始配方规定「5斤（2.5公斤）花配1斤（0.5公斤）茶坯」的比例，以达到深度薰制。

**4. 风土与栽培特色：**

* **四川省峨眉山：** 峨眉山 (峨眉山, Éméishān) 是中国四大佛教圣山之一，联合国教科文组织世界遗产（自然和文化）。位于四川盆地西南部，从盆地向青藏高原的过渡地带。主峰万佛顶 (万佛顶, Wànfódǐng) 海拔3099公尺。茶园分布在海拔600-1500公尺。气候为湿润亚热带，特征是「华西雨屏」现象：丰富降水（年降水量1500-2000毫米）、常年云雾、长时间散射光照、温和冬季。年平均气温17°C。土壤为酸性山地黄壤和棕壤，富含有机质。这些条件非常适合形成高氨基酸、适中儿茶素含量的嫩芽，确保茶叶基础的甜美柔和口感。
* **雅安地区蒙顶山：** 历史名茶山，以经典名句著称：「扬子江中水，蒙山顶上茶」(扬子江中水，蒙山顶上茶)。海拔1000-1400公尺，类似气候条件。
* **夹江：** 乐山市 (乐山) 下辖县，位于峨眉山南部。飘雪绿茶基础的传统产区。
* **犍为（茉莉花）：** 乐山市下辖县，种植四川茉莉花。亚热带气候，岷江 (岷江) 流域肥沃冲积土壤。

**5. 制作工艺：**

茉莉飘雪的生产结合四川茶叶基础加工技术（炒青——炒制杀青）与多次茉莉花薰制，最终在成品茶中保留茉莉花瓣。

* **茶叶加工（制作茶坯）：**

    1. **采摘 (采摘, cǎizhāi)：** 手工采摘嫩春芽和一二片叶。
    2. **摊凉 (摊凉, tānliáng)：** 在竹盘上摊放4-6小时。
    3. **炒青杀青 (炒青杀青, chǎoqīng shāqīng)：** 与福建工艺的关键区别。茶叶在200-250°C的热锅中炒制。炒制杀青赋予茶叶基础更明显的「烘烤」香气、厚度和特有风味轮廓，区别于福建茉莉花茶的烘青基础。
    4. **晾凉 (晾凉, liàngliáng)。**
    5. **揉捻 (揉捻, róuniǎn)：** 轻度纵向揉捻，不过紧——茶叶呈细长「舌状」或徐金华描述的「雀嘴」(鹊嘴, quèzuǐ) 形状。
    6. **干燥 (干燥, gānzào)：** 最终干燥至适合后续薰制的含水量。

* **茉莉花薰制——窨制 (窨制, xūnzhì)：**

    1. **伺花 (伺花, sìhuā)：** 标准程序——控制花蕾绽放。
    2. **分层窨花 (分层窨花, fēncéng xūnhuā)：** 飘雪工艺的特色。在专用容器或竹盘中，茶叶和绽放的茉莉花交替分层：茶层——花层——茶层——花层。这种分层配置确保香气均匀分布，赋予成品茶「雪花飞舞」的视觉效果。
    3. **静置窨花 (静置窨花, jìngzhì xūnhuā)：** 每个周期6-8小时。
    4. **通花 (通花, tōnghuā)：** 必要时进行。
    5. **起花 (起花, qǐhuā)——部分：** 与完全去除花朵的福建工艺不同，飘雪生产中刻意在茶中保留部分花朵。这种做法是「飘雪」风格的决定性特征。
    6. **多次窨制：** 大众产品通常4-6次，高级产品6-9次（徐金华原始配方为「九窨一提」，九窨一提, jiǔ xūn yī tí，45天周期）。最高等级「论道」(论道, Lùndào——「论道之路」，竹叶青高端系列) 达到9次窨制。
    7. **提花 (提花, tíhuā)：** 最终添加新鲜花朵，短暂薰制后保留在茶中，确保「鲜活」的顶级香气和视觉效果。
    8. **最终干燥：** 小心干燥至含水量6-8%，茉莉花瓣保持白色和形状。

* **分级 (分级, fēnjí)：** 成品茶按茶叶品质、茉莉花瓣数量和状态、混合均匀度分级。

**6. 感官特征：**

* **干茶外观：** 细长、轻度卷曲的深绿色茶叶，有明显白毫，其中清楚可见干燥的白色或淡奶油色茉莉花瓣。正是绿色茶叶与白色花瓣的对比创造了著名的「雪花飞舞效果」。茶叶完整，无碎末和粉尘。
* **干茶香气：** 浓郁，茉莉花香突出，底层有烘烤绿茶香——特有的「双层」香气，花香不掩盖茶香。
* **茶汤香气：** 浓郁茉莉花香，伴有新鲜绿叶、淡栗子香（来自炒制基础）和蜂蜜甜香。香气持久，但不如顶级福建茉莉花茶那样「鲜灵」(鲜灵)——这由深度和「温暖」的烘烤底调补偿。
* **口感：** 厚实、清新，结构良好，比福建茉莉花茶有更明显的「茶味」特色。茶体中等至饱满。天然甜味，但克制，由轻微涩味和特有清新感平衡。回甘 (回甘, huígān) 纯净，有茉莉花香和温和甜味。四川风格：「茶香比花香重」(茶香比花香重, cháxiāng bǐ huāxiāng zhòng——「茶香胜过花香」)。
* **汤色：** 黄绿色，透明，明亮。由于炒制基础，比福建茉莉花茶稍浓。
* **叶底（冲泡后茶叶）：** 展开的嫩绿茶叶和芽头，伴随失去香气但保持形状的湿润茉莉花瓣。在玻璃杯中冲泡时，茶叶沉底，茉莉花瓣在水中和表面飘浮——飘雪的标志性视觉特征。

**7. 化学成分：**

* **多酚类 (茶多酚, chá duōfēn)：** 儿茶素——EGCG、EC、EGC、ECG。总含量为干重的18-28%。炒制杀青相比烘青基础略微降低不稳定儿茶素含量，但通过美拉德反应和焦糖化形成更复杂的香气轮廓。
* **氨基酸 (氨基酸, ānjīsuān)：** L-茶氨酸——干重的1.5-2.5%。来自峨眉山在云雾遮蔽条件下生长的高山原料，氨基酸含量较高——达484毫克/100克，四川茶叶原料分析研究证实。
* **生物碱：** 咖啡因——2-4%（每150毫升杯30-50毫克）。可可碱和茶碱——微量。
* **茉莉花精油：** 特有芳香化合物组合，研究显示四川茉莉花茶中「绿色」(青草气) 化合物——如(Z)-3-己烯-1-醇、1-辛烯-3-醇、戊醛和己醛——的相对含量高于福建和广西茉莉花茶。这赋予四川飘雪更清新的「绿色」香气特色，而广西茉莉花茶富含花香成分（苯甲酸苄酯、顺式茉莉酮、香叶醇、α-法呢烯）。
* **维生素：** C、B₁、B₂、E、P（芸香苷）。
* **矿物质：** 钾、镁、氟、锌、锰。

**8. 保健功效：**

* **抗氧化作用：** 绿茶基础的儿茶素和多酚提供显著抗氧化效果，保护细胞免受自由基损害，延缓氧化衰老。
* **提神醒脑效果：** 咖啡因和L-茶氨酸的结合产生温和、持久的清醒感，提高专注力，无紧张感。炒青基础比烘青茶提供稍强的提神效果。
* **抗压作用：** 芳樟醇和其他茉莉花油成分降低焦虑，改善情绪，促进放松。
* **助消化：** 多酚和萜类化合物刺激消化酶，帮助消化油腻和重食物。在四川菜中，茉莉花茶传统上是辛辣菜肴的理想搭配。
* **心血管系统：** 经常饮用绿茶有助于降低低密度脂蛋白胆固醇，维持血管弹性。
* **抗菌作用：** 茉莉花油成分具有证实的抗微生物活性。
* **清热解渴效果：** 优秀的夏季饮品，特别是冷泡版本。
* **促进新陈代谢：** 绿茶儿茶素适度激活产热和脂肪代谢。

**9. 冲泡方法：**

* **水温：** 80-85°C。不要用沸水——会破坏芳香化合物并产生过度涩味。
* **茶量：** 3-5克配150-200毫升水。功夫茶法——5-6克配100-120毫升盖碗。
* **茶具：** 玻璃杯是飘雪的绝对首选：透明壁面可观察标志性「雪花飞舞效果」——白色花瓣飘浮在绿色茶叶上方。也适合白瓷盖碗 (盖碗)。徐金华本人推荐盖碗，强调盖子能聚集香气。
* **冲泡过程：**

    1. 用沸水温热玻璃杯或盖碗，倒掉水。
    2. 放入茶叶。
    3. 注入80-85°C水立即倒掉——快速洗茶 (润茶, rùnchá)。
    4. 第一泡：注水，浸泡2-3分钟。观察「雪花飞舞」——茉莉花瓣浮到表面，茶叶沉底，形成双层画面。
    5. 将茶汤倒入茶杯。
    6. 续泡：3-5泡，每次延长30-60秒。炒青基础在多次冲泡中保持良好的味道持久性。

* **冷泡：** 3-5克配500毫升冷水，冰箱中6-8小时。在透明瓶中视觉效果特别吸引人——白色花瓣在绿叶间飘浮。

**10. 储存：**

* **条件：** 干燥、阴凉、避光处，远离异味。
* **容器：** 密封不透明包装——铁罐、铝箔袋。
* **温度：** 冰箱（0-5°C）密封包装。开封前——恢复至室温。
* **保质期：** 12-18个月。茉莉花香随时间减弱；成分中的干茉莉花瓣不会增强香气，只是视觉提醒新鲜度，因此茶叶最好在生产后6-12个月内饮用。
* **茶叶天敌：** 湿气、光线、高温、异味。

**11. 价格与假冒：**

* **价格范围：** 飘雪价格范围很广，从日常实惠到高端。基础级（3-4次窨制，标准基础）——100-300元/500克。中级（5-6次窨制，峨眉山基础）——400-1000元/500克。高端级（「碧潭飘雪」品牌，「论道」系列，9次窨制）——2000-5000元/500克以上。主要价格因素：茶叶基础产地（峨眉山vs标准平原茶园）、窨制次数、品牌（竹叶青、林湖等）。
* **如何避免假冒：**
    * **外观：** 茶叶——完整、细长，有白毫。茉莉花瓣——白色、完整、未破碎。茶叶与花瓣比例——和谐，不过量（过多花瓣常掩盖基础品质低劣）。
    * **香气：** 天然、双层：茶香+花香。人工香精表现为刺鼻、单一、「香水味」，快速消散。
    * **口感：** 厚实、清新，有明显「茶味」。如果尽管花瓣丰富但口感空洞水薄——是基础品质低劣的标志。
    * **产地：** 真正飘雪是四川产品。其他地区的「飘雪」可能品质良好，但不是正宗风格代表。
    * **价格：** 低于80元/500克却声称「多次窨制」——值得怀疑。

**12. 趣闻轶事：**

* 徐金华的原始配方规定5:1比例——5斤（2.5公斤）新鲜茉莉花配1斤（0.5公斤）茶叶基础。9次窨制时，500克成品茶总共消耗12-15公斤新鲜花朵。
* 峨眉山是中国最古老的产茶山之一，有3000多年茶叶种植历史。南宋诗人陆游 (陆游) 赞叹：「雪芽近自峨眉得，不减红囊顾渚春」(雪芽近自峨眉得，不减红囊顾渚春)。
* 在四川茶馆 (茶馆) 中，「飘雪」风格茉莉花茶用传统瓷盖碗盛装——成都茶文化的特色器具。盖碗能在盖下聚集香气，优雅地推开盖子饮茶。
* 「飘雪」(飘雪——「飘雪」) 名称已成为整类保留花瓣的四川茉莉花茶的通称：「林湖飘雪」、「峨顶飘雪」、「玉心飘雪」等——都使用「见茶见花」(见茶见花) 原则，与福建「见茶不见花」(见茶不见花) 相对。

**13. 与其他茉莉花茶比较：**

* **福建茉莉花茶 (福建茉莉花茶)：** 基于烘青基础 (烘青——热风干燥)，而飘雪基于炒青 (炒青——炒制)。福建茶更「花香」，茉莉花香主导，有特色「冰糖甜」(冰糖甜)。飘雪更「茶味」，厚实，有涩味清新感和明显基础特色。福建——无花瓣 (见茶不见花)，飘雪——有花瓣 (见茶见花)。
* **茉莉龙珠 (茉莉龙珠, Mòlì Lóngzhū)：** 「龙珠」——福建茉莉花茶球形颗粒。形状不同（球状vs散茶配花瓣）、基础不同（烘青vs炒青）、无可见花瓣。龙珠更甜美「圆润」，飘雪更有结构和清新。
* **茉莉女儿环 (茉莉女儿环, Mòlì Nǚ'ér Huán)：** 「少女环」——手工造型茉莉花茶环形。与飘雪同属「工艺茶」，但形状和技术完全不同。女儿环——无可见花瓣，制作更费工。
* **茉莉银针 (茉莉银针, Mòlì Yínzhēn)：** 「银针」——白芽基础的高端茉莉花茶。更精致、飘逸，口感细腻。相对风格极端：简约纯净vs飘雪的视觉丰富。
* **黄山毛峰茉莉花茶：** 如果飘雪有其他地区版本，使用不同茶叶基础，但保持「保留花瓣」原则。结果与四川原版口感差异显著。

**结语：**

茉莉飘雪是一款每杯都诉说产地故事的茶。其中有——在峨眉圣山云雾中生长的四川绿茶深度，夏季茉莉花的慷慨，以及决定在茶中保留白色花瓣的茶师诗意眼光，将每次冲泡变成小型表演。「飘雪」不仅是饮品，更是邀请人们放慢脚步，在简单事物中发现美：绿白对比、花瓣在水面的舞蹈、既提神又安神的香气。这款茶非常适合不仅重视口感，也欣赏品茶视觉诗意的人，以及想要探索四川茉莉花茶作为经典福建茉莉花茶替代选择的人。
