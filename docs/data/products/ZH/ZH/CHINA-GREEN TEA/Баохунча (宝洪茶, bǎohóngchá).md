# 宝洪茶 (宝洪茶, bǎohóngchá)

**宝洪茶 (宝洪茶, bǎohóngchá)**
![Баохунча (宝洪茶, bǎohóngchá)](images/Баохунча%20(宝洪茶,%20Bǎohóngchá).png)

![Баохунча (宝洪茶, bǎohóngchá)-1](images/Баохунча%20(宝洪茶,%20Bǎohóngchá)-1.png)

宝洪茶 (宝洪茶, bǎohóngchá) — 来自云南省宜良县的历史绿茶，是云南省唯一的小叶种绿茶 (云南唯一的小叶种茶)，在以大叶种普洱茶和滇红茶闻名的省份中独树一帜。这款茶获得了诗意的美称「云南龙井茶」(云南龙井茶)，以其极为浓郁的香气著称：当地俗语说「屋内炒茶院外香」(屋内炒茶院外香, wūnèi chǎochá yuànwài xiāng)。明代学者和农学家徐光启 (徐光启, Xú Guāngqǐ) 在其著作《农政全书》(农政全书, 「完整农业指南」) 中称宝洪茶为「茶之极品」。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。属于扁形炒青绿茶 (扁形炒青绿茶, biǎnxíng chǎoqīng lǜchá) — 与龙井茶相同的成形工艺。

* **等级：** 中国历史名茶 (中国历史名茶)。明清时期的贡茶 (贡茶, gòngchá) — 从嘉靖三十六年 (嘉靖, 1557) 到咸丰年间 (咸丰, 1851–1861)。2016年制作技艺被列入云南省非物质文化遗产名录。2024年获得「中茶杯」(中茶杯) 特别奖。

* **产地：** 中国云南省 (云南, Yúnnán) 昆明市 (昆明, Kūnmíng) 宜良县 (宜良县, Yíliáng Xiàn)。生产区域涵盖该县15个乡镇。核心产区 — 宝洪山 (宝洪山, Bǎohóng Shān)，宝洪寺 (宝洪寺, Bǎohóng Sì) 佛教寺院周围。核心茶园占产量的90%，集中在江头村 (江头村) 和大村子 (大村子)。

* **地理座标：** 约北纬24°55′，东经103°10′。

**2. 历史与文化意义：**

* **历史：** 宝洪茶的起源与建在同名山上的佛教寺院宝洪寺有关。据当地史料记载，唐代 (618–907) 寺院开山和尚 (开山和尚, kāishān héshàng) 从福建省带来小叶种茶苗，种植在寺院周围的山坡上。最初这茶被称为「相国寺茶」(相国寺茶, 「相国寺茶」)。

	明代 (1368–1644) 和清代 (1644–1912)，宝洪茶被列入皇室贡茶 (贡茶) 名单。进贡期间 — 从1557年（嘉靖三十六年）到19世纪中期（咸丰年间）— 是云南茶中最长的进贡期之一。学者徐光启在其农学著作中给予了最高评价。

	20世纪：1934年创制了独特的「橘红」(橘红, 「蜜橘红」) 工艺 — 具有丰富金毫的高级品种。1958年在宝洪山建立国营茶场，开始恢复工业化生产。2016年宝洪茶制作技艺被列入云南省非物质文化遗产名录。

* **名称：**

	* 「宝洪」(宝洪) — 「珍贵的急流」：山名和佛教寺院名。
	* 「茶」(茶) — 「茶」。

* **文化意义：** 宝洪茶是跨地区文化交流的独特产物：一千多年前佛教僧侣将福建茶树带到云南，适应了云南高山风土，产生了既不同于福建茶也不同于经典云南茶风格的茶品。宝洪茶是中国东部和西南部茶文化之间的独特「桥梁」。宝洪寺至今仍是该地区茶文化传统的中心。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要栽培种 — **云南中小叶群体种 (云南中小叶群体种, Yúnnán zhōngxiǎoyè qúntǐzhǒng)** — 当地原生小叶种和中叶种 *Camellia sinensis* var. *sinensis*，是福建引入种群的后代。叶片厚实，肉质，具有高「持嫩性」(持嫩性强)。另外使用**福鼎大白茶 (福鼎大白茶)** — 引入的大叶种，毫毛丰富，提高白毫密度和口感清新度。

* **采摘：** 早春采摘。特级 (特级) — 完整芽头或一芽一叶初展。一级 — 一芽一叶。二级 — 一芽二叶。

* **原料要求：** 嫩度均匀，无损伤的芽叶。当天采摘当天加工。

**4. 风土与栽培特点：**

* **海拔高度：** 1550–1750公尺 — 明显高于中国大多数小叶种绿茶。高海拔位置确保了缓慢生长和胺基酸的增加积累。

* **气候：** 亚热带季风气候。年平均温度16°C，年降雨量1200–1400毫米。云雾覆盖茶园80%的时间。日夜温差显著。丰富的散射光有助于形成柔和、甘甜的风味特征。

* **土壤：** 花岗岩基础上的黄棕壤 (黄棕壤)，pH值4.8–5.5，富含有机物、钾和硒。良好的通气性和透水性。

**5. 制作工艺：**

宝洪茶采用借鉴并适应了「龙井八法」(龙井"八法") 的工艺 — 在热锅中进行抛、抓、压等手工技法。整个过程完全手工制作。

* **鲜叶摊放 (鲜叶摊放 — xiānyè tānfàng)：** 短时间摊放以去除多余水分。

* **杀青 (杀青 — shāqīng)：** 在铸铁锅中140–200°C手工炒制。师傅徒手操作 (徒手炒, túshǒu chǎo) — 不使用金属工具，以免损害香气。此阶段停止氧化并固定著名的浓郁香气。

* **摊凉回潮 (摊凉回潮 — tānliáng huícháo)：** 中间冷却以重新分配水分。

* **成形和「辉锅」(𪸩锅 — huīguō)：** 决定形状和外观的关键阶段。在低温（50–60°C）下，师傅将茶叶压贴并抛光锅壁，使其呈现扁平、光滑的形状，类似杉松叶 (似杉松叶)。同时将毫毛「拉」到表面。整个阶段不使用金属工具 — 仅用手工。

* **筛拣 (筛拣 — shāijiǎn)：** 最终分级，去除碎片和不标准茶叶。成品茶含水量≤7%。

**6. 感官特征：**

* **干茶外形：** 扁直平滑 (扁直平滑) 的茶条，类似杉松针叶 (似杉松叶)。色泽翠绿油润 (绿润)。「橘红」(橘红) 品种具有丰富的金色毫毛 (金毫密布)。

* **干茶香气：** 极为浓郁、高锐馥郁 (高锐馥郁, gāoruì fùyù)。主要特征是强度：「屋内炒茶院外香」(「在屋内炒茶院外都能闻到香味」)。以清香 (清香)、嫩香 (嫩香) 和豆香/栗香 (豆香/栗香) 为主。

* **茶汤香气：** 持久、浓郁，具有相同的高香特征。冷杯留香 (冷杯留香) 持久，是真品的标志。

* **滋味：** 鲜爽 (鲜爽, xiānshuǎng) — 高胺基酸含量确保了明亮的清新感。甘甜 (甘甜, gāntián) — 快速回甘。醇厚 (醇厚, chúnhòu) — 茶汤具有明显的「油润感」。涩味极少。

* **汤色：** 黄绿清澈 (黄绿清澈)。

* **叶底（冲泡后茶叶）：** 嫩绿均匀，柔软有弹性。

**7. 化学成分：**

云南高海拔风土（1550–1750公尺）结合小叶种栽培种和手工加工形成了特征性成分：

* **多酚类（儿茶素）：** 含量显著。提供抗氧化潜力和清淡的口感结构。
* **胺基酸（包括L-茶胺酸）：** 含量较高 — 明显清新感和甘甜味的关键因素。
* **生物碱：** 咖啡因 — 中等含量。可可碱、茶碱。
* **矿物质：** 钾、硒、锌、锰 — 由花岗岩土壤决定。
* **维生素：** 维生素C、类胡萝卜素。
* **芳香化合物：** 极其丰富的挥发性复合物 — 正是它确保了著名的香气强度。

**8. 保健功效：**

* **清热解暑 (清热)：** 传统上属于「凉性」茶类。
* **明目 (明目)：** 类胡萝卜素和维生素C有助于保护眼睛。
* **抗氧化作用：** 儿茶素中和自由基。
* **助消化 (消食)：** 多酚类刺激脂肪分解。
* **提神效果：** 咖啡因和L-茶胺酸提供温和的清醒感。

* **重要提醒：** 上述功效基于公开资料，不构成医疗建议。

**9. 冲泡方法：**

* **水温：** 80–85°C。

* **茶量：** 3公克配150毫升水（比例1:50）。

* **茶具：** 白瓷杯或盖碗 — 用于评估黄绿色茶汤和香气。玻璃杯 — 用于观察扁平茶叶的舒展。

* **冲泡过程（下投法）：**

	1. 用热水温杯，倒掉。
	2. 投入3公克茶叶。
	3. 注入1/3容量的水，「润茶」10秒，倒掉（洗茶）。
	4. 注水至7/10容量。
	5. 第一泡 — 10–15秒。
	6. 后续冲泡 — 每次增加5–10秒。可冲泡4–5次。

* **注意：** 最佳饮用温度约60°C：此时最能感受到清新感和甘甜味。

**10. 储存：**

* 密封保存，置于阴凉干燥处。
* 最佳储存 — 冰箱0–5°C。
* 保存期限 — 最多12个月。
* 开封后 — 1–2个月内饮用完毕。

**11. 价格与仿冒品：**

宝洪茶产量有限：90%来自宝洪山狭窄区域。价格取决于等级、采摘时间和类型（传统vs.「橘红」）。

* **如何避免仿冒品：**

	* **向可靠商家购买**，需有宜良县产地证明。
	* **评估香气：** 标志性强度 — 「屋内炒茶院外香」。香气微弱、不明显是仿冒品的征象。
	* **检查冷杯留香：** 持久的冷香 (冷杯留香) 是真品标志。
	* **评估外形：** 扁直茶条，类似松针。卷曲或不规则是其他茶类。
	* **注意价格：** 价格过低是仿冒品征象。

**12. 有趣事实：**

* 宝洪茶是云南唯一的小叶种绿茶：在以大叶种普洱茶和滇红茶闻名的省份中，它独树一帜 — 云南茶界的「白乌鸦」。

* 俗语「屋内炒茶院外香」(「在屋内炒茶院外都能闻到香味」) 不是文学比喻，而是真实观察：宝洪茶芳香化合物的强度如此之高，炒制时香气确实能传播数十公尺。

* 徐光启 (1562–1633) — 明末最伟大的学者之一，天文学家、数学家和农学家，《农政全书》作者 — 称宝洪茶为「茶之极品」。来自如此重要人物的评价是极高的赞誉。

* 宝洪茶的炒制工艺直接借鉴了「龙井八法」（抛、抓、压等），并适应云南原料。这使宝洪茶成为龙井茶的独特「云南表亲」。

* 1934年创制的「橘红」(橘红, 「蜜橘红」) 品种 — 具有金色毫毛的独特高级品，在扁形绿茶中无与伦比。

**13. 与其他扁形绿茶的比较：**

* **西湖龙井 (西湖龙井)：** 原型和「老师」。龙井更温和，具有豆栗香，明显的鲜味。宝洪茶香气更浓郁，具有强烈、「穿透性」的高香。

* **湄潭翠芽 (湄潭翠芽)：** 来自贵州。同样扁形，同样「受龙井启发」。湄潭更「技术化」（95%自动化）；宝洪茶完全手工，历史背景更深厚。

* **峨眉竹叶青 (峨眉竹叶青)：** 来自四川。扁形，翠绿，「竹叶」状。竹叶青更清淡「纯净」；宝洪茶香气更浓郁。

**结论：**

宝洪茶是一款矛盾之茶：小叶种福建「客人」在大叶种云南巨人的土地上，它不仅在宝洪山高海拔坡地上扎根 — 它创造了自己独特、无与伦比的风格。其令人难以置信的香气浓郁度 — 「屋内炒茶院外香」— 在扁形绿茶中无与伦比。清新、甘甜、醇厚的滋味，「松针」般的扁平外形和千年佛教历史，使宝洪茶成为中国最独特和被低估的绿茶之一 — 适合寻求真正稀有茶品的人士。
