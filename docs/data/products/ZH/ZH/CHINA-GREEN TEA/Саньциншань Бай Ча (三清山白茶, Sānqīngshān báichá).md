# 三清山白茶 (三清山白茶, Sānqīngshān báichá)

**三清山白茶 (三清山白茶, Sānqīngshān báichá)**

![Саньциншань Бай Ча (三清山白茶, Sānqīngshān báichá)-1](images/Саньциншань%20Бай%20Ча%20(三清山白茶,%20Sānqīngshān%20Báichá)-1.png)

三清山白茶 (三清山白茶, Sānqīngshān báichá) — 稀有的江西绿茶，采用白叶变异品种制作，栽培于神圣道教名山三清山 (三清山) 山麓 — 联合国教科文组织世界遗产地。尽管名称中有「白茶」(白茶，「白茶」) 一词，此茶按绿茶工艺加工，属于「白叶绿茶」类 — 类似著名的安吉白茶 (安吉白茶)。其主要价值在于非凡的胺基酸含量：6–10 %，比普通绿茶高2–3倍，而多酚含量异常偏低 (13–16 %)。这种独特的比例创造出极致甘甜柔和的口感 — 「翡翠牛奶」般毫无苦涩。

**1. 分类与产地：**

* **类型：** 绿茶 (未发酵，绿茶, lǜchá)，采用白叶变异品种制作。尽管名称为「白茶」，这并非传统意义上的白茶 (属于白茶类，加工程度极低)，而是具有特征性白色/淡黄色叶片的绿茶。杀青方法 — 滚筒炒制，温度130–140 °C。

* **类别：** 农产品地理标志产品 (农产品地理标志，2011年，公告第1635号)；「江西名茶」(江西名茶，2013年)。稀有变异茶类 — 「高胺基酸低茶多酚」(高氨基酸低茶多酚)。

* **产地：** 中国；江西省 (江西, Jiāngxī)；玉山县 (玉山县, Yùshān Xiàn)，隶属上饶市 (上饶市, Shàngráo Shì)。保护区域 — 全县境内。品质核心 — 三清乡 (三清乡, Sānqīng Xiāng)、紫湖镇 (紫湖镇, Zǐhú Zhèn) 和怀玉乡 (怀玉乡, Huáiyù Xiāng) — 信江 (信江, Xìnjiāng) 源头的高山茶园 (>700 公尺)，位于三清山山麓云雾缭绕地带。

* **地理座标：** 东经117°52′–118°25′，北纬28°30′–28°59′ (玉山县境内)。

**2. 历史与文化意义：**

* **历史：**

玉山县是江西最古老的茶区之一，曾列入「唐宋八大茶场」。同治年间 (同治, 1862–1874) 《玉山县志》记载：「茶以三清山产者味特清冽……不及三清远甚」(茶以三清山产者味特清冽……不及三清远甚)。这一十九世纪的记录证实，三清山茶的独特品质早在现代研究之前就已获得认可。

1970年代，紫湖茶厂 (紫湖茶厂) 创制了两个名茶产品：「三清云雾茶」(三清云雾茶，「三清云雾茶」)，被评为上饶最佳名茶，以及「紫湖春露」(紫湖春露，「紫湖春露」)，获中国农业科学院茶叶研究所认证。然而1990年代茶厂停产，茶业陷入衰落。

复兴始于二十一世纪，当地育种专家发现并培育了白叶变异 — 品种「白茶玉」(白茶玉，「白茶翡翠」)。这个品种成为当地对浙江安吉白茶成功的回应：相同原理 (白叶变异 + 绿茶加工)，但凭借高山风土和花岗岩土壤实现了更极致的胺基酸特征。2011年三清山白茶获得农产品地理标志 (中华人民共和国农业部公告第1635号)。2013年荣获「江西名茶」称号。至2023年品牌价值达24.45亿元人民币 — 对于相对年轻的品牌而言增长惊人，反映了对「胺基酸」类奢华茶品日益增长的需求。

* **名称：** 三清山 (Sānqīngshān) — 「三清山」— 神圣的道教名山，道教四大名山 (四大道教名山) 之一，2008年列入联合国教科文组织世界遗产 (标准VII — 「卓越自然美景」)。「三清」— 道教三位至高神祇 (元始天尊, 灵宝天尊, 道德天尊)；白茶 (Báichá) — 「白茶」，指变异品种叶片异常的白色/淡黄色。完整含义：「三清山的白叶茶」。

* **文化意义：** 三清山白茶与中国最壮观的山地景观之一密不可分。三清山以沉浸在云海中的花岗岩石柱「手指峰」和丰富的生物圈 (超过2300种植物) 闻名。茶园位于联合国教科文组织缓冲区，这施加了严格的生态限制并保证了环境纯净。玉山县获认证为「中国天然氧吧」(中国天然氧吧) — 空气中负离子浓度比城市高50倍。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要 — 白茶玉 (白茶玉, Báichá Yù，「白茶翡翠」) — 当地选育的变异品种 *Camellia sinensis* var. *sinensis*。这是温度敏感的白叶变异：当温度低于约23 °C时，嫩叶失去叶绿素，呈现淡黄色或乳白色 (叶肉玉白，「叶肉翡翠白」)，而叶脉保持翠绿色 (叶脉翠绿) — 形成特征性的双色效果。温度升高时叶片逐渐变绿。特点是萌芽早、抗寒耐热、嫩度期延长 (比标准品种长7–10天)。另外使用福鼎大白茶 (福鼎大白茶)。树龄超过30年。执行「五不采」标准 (五不采, wǔ bù cǎi)：不采雨水叶、紫色芽、病叶、虫害叶和畸形叶。

* **采摘：** 春茶 — 三月底至四月中旬。最佳期间 — 温度未超过约23 °C且叶片保持白色/淡黄色时。采摘延迟会使叶片变绿，失去独特的「白叶」特征。

* **采摘标准：** 特级 — 单芽或一芽一叶初展。一级 — 一芽一至二叶。

* **原料要求：** 具有特征性白色/淡黄色叶片和绿色叶脉的嫩芽。无机械损伤和虫害痕迹。

**4. 风土与栽培特点：**

玉山县位于江西东北部，三清山山麓，怀玉山 (怀玉山) 和武夷山 (武夷山) 山脉交汇处。地形为陡峭山坡，有花岗岩露头。

* **栽培海拔：** 758–1182 公尺 — 江西最高的茶区之一。

* **气候：** 亚热带季风山地气候。年均温17–18 °C；降雨量1400–1724毫米/年；雾日>180天/年；日温差>8 °C。凉爽的春夜 (低于23 °C) 对白叶变异的表现至关重要 — 正是山地微气候确保了长期的「白叶」期，这在平地无法实现。

* **土壤：** 红壤和黄壤 (红壤、黄壤)，pH 5.0–6.5。富含锌 (76.2 毫克/公斤) 和硒 (0.82 毫克/公斤) — 三清山花岗岩体的地球化学标志。

* **生态：** 森林覆盖率81%。「中国天然氧吧」认证 (中国天然氧吧) — 负离子浓度比城市高50倍。茶园位于联合国教科文组织三清山遗产地缓冲区，灌溉采用信江源头泉水。完全排除工业污染。生态系统独特：茶园梯田与原始针叶林交错，三清山花岗岩石柱「手指峰」形成天然防风屏障和雾气凝结增强的微区域。

**5. 制作工艺：**

三清山白茶采用独特的复合工艺「晒青 + 轻揉捻」(日光萎凋 + 轻揉捻)，平衡香气和外形。最终阶段 — 炭火烘干。

* **摊放/萎凋 (摊放 — tān fàng)：** 在通风室内4–6小时。

* **日光萎凋 (晒青 — shài qīng)：** 自然日晒 — 传统绿茶不常见的步骤。温和的阳光启动初步香气转化和部分脱水，赋予茶叶额外的甘甜和花香，这是纯「炒制」绿茶所不具备的。

* **杀青 (杀青 — shāqīng)：** 滚筒炒制，温度130–140 °C — 适中温度以保持精致的胺基酸特征。过高温度会破坏L-茶胺酸和其他胺基酸。

* **轻揉捻 (轻揉捻 — qīng róuniǎn)：** 最小程度揉捻以塑形和释放茶汁，同时保持芽头完整性和白毫 (成条保毫，「成条保毫」)。

* **理条 (理条 — lǐ tiáo)：** 70 °C下 — 固定直条外形。

* **提毫 (提毫 — tí háo)：** 50 °C时师傅用手掌滚动茶叶，「提升」白毫至表面 — 决定特征性银色光泽的步骤。

* **炭火烘焙 (烘焙 — hōngbèi)：** 60 °C木炭烘干至含水量≤6.5%。炭火加热形成细腻甘甜背景，不掩盖精致香气。

**6. 感官特征：**

* **干茶外观：** 挺直紧结的条索 (挺直紧结)；嫩黄绿润 (嫩黄绿润)，白毫显露 (显毫)。「白肉绿脉」的双色特征在干茶中也保持。

* **干茶香气：** 清香纯正 (清香) 带明显花香背景 (花香)。精致的「奶香」调 — 高胺基酸含量的结果。

* **茶汤香气：** 浓郁持久 (馥郁, fùyù)。花香绿茶调带「奶油」背景。非典型「炒制」特征 — 在柔和度上更接近白茶，但更清新。

* **滋味：** 极其鲜爽 (鲜爽)，天然甘甜明亮 (甘醇) 和丝滑顺口 (滑, huá)。苦涩几乎不存在 — 极低多酚含量 (13–16 %) 的结果。甘甜回味悠长。「翡翠牛奶」— 源自十九世纪县志的诗意描述。

* **汤色：** 透明，浅黄色 (银针型) 或橙黄色 (寿眉型)。清澈度高。

* **叶底 (冲泡后茶叶)：** 翡翠白带翠绿叶脉 (玉白翠绿) — 特征性双色「大理石」效果。鲜活，呈「花朵」状展开 (鲜活成朵)。

**7. 化学成分：**

* **胺基酸 (氨基酸)：** 6–10 % — 非凡水平，比绿茶平均值 (2–3 %) 高2–3倍。L-茶胺酸占主导，提供甘甜、「奶油」质感和强效放松-提神效果。这是世界所有茶类中胺基酸浓度最高的之一。

* **多酚 (茶多酚)：** 13.2–15.6 % — 约为标准绿茶 (25–30 %) 的一半。低多酚水平解释了几乎完全没有苦涩味。

* **生物碱：** 咖啡因在标准范围；可可碱、茶碱。

* **维生素：** 维生素A — 含量增高 (维生素A原达12.6毫克/100克)；维生素C、B族维生素、维生素E。

* **矿物质：** 锌 — 76.2毫克/公斤；硒 — 0.82毫克/公斤 (三清山花岗岩土壤的地球化学标志)。钾、锰、氟。

* **叶绿素：** 白叶期含量降低 — 特征性黄白色的原因。随著升温叶片变绿，但早期采摘固定了「白叶」阶段。

**8. 保健功效：**

* **抗氧化保护：** 尽管多酚水平降低，儿茶素与维生素E和硒结合提供有效的氧化应激保护。

* **放松和认知功能：** 非凡的L-茶胺酸含量 (6–10 % 胺基酸) 刺激大脑α波活动 — 最强效的「放松-专注」茶类之一。

* **视力支持：** 维生素A原含量增高 (12.6毫克/100克) 有益视网膜健康，可减轻萤幕工作时的眼疲劳。

* **硒支持：** 天然硒 (0.82毫克/公斤) — 甲状腺和免疫系统的重要微量元素。

* **胃部温和：** 低多酚含量使此茶对胃黏膜比标准绿茶温和得多。

* **心血管系统支持：** 儿茶素有助于调节血脂和血糖水平。

* **重要提醒：** 上述功效基于一般数据，非医疗建议。不建议空腹饮用 (尽管温和) 和睡前4小时内饮用。内分泌疾病患者请咨询医师。

**9. 冲泡方法：**

* **水温：** 80–85 °C。不使用沸水 — 超过85 °C会破坏L-茶胺酸并增加苦味。

* **茶量：** 3克配150毫升 (比例1:50)。

* **茶具：** 玻璃杯 — 观察叶片双色「大理石」效果；白瓷盖碗 — 聚集花香。可用紫砂壶 (紫砂壶)。

* **冲泡过程：**

	1. 用热水温壶后倒掉。
	2. 投入3克茶叶。
	3. 注入80–85 °C水。
	4. 第一泡 — 1–2分钟。
	5. 后续泡次稍微延长时间。可冲泡3–4次。

* **注意：** 第一泡展现最大甘甜和「奶香」— L-茶胺酸比多酚萃取更快的结果。第二、三泡口感更「绿茶」化，但保持特征性柔和和无苦涩。

**10. 储存：**

* **温度：** 0–5 °C (冰箱) — 必须；白叶绿茶对氧化特别敏感。
* **光线：** 完全避光 — 叶绿素本已降低，额外氧化会使茶叶暗淡。
* **容器：** 密封真空或铝箔包装。
* **保质期：** 0–5 °C下可保存12个月。建议前6个月内饮用以获最大新鲜度和甘甜 — 随时间推移胺基酸比多酚降解更快，茶叶会失去独特的「奶香」特征。

**11. 价格与仿冒：**

三清山白茶属高价位茶品，因变异品种种植面积有限和采摘窗口期短。特级 — 800元人民币/斤起；一级 — 400–600元人民币/斤。

* **如何避免仿冒：**

	* 检查农产品地理标志 (农产品地理标志) 标识。
	* 主要视觉标志 — 叶底的「白肉绿脉」双色效果 (叶肉玉白叶脉翠绿)。普通绿茶无此效果。
	* 口感应极其柔和甘甜，无明显涩味。苦涩茶品非三清山白茶。
	* 香气 — 花香奶油调，无「炒制」或「草腥」味。
	* 可疑低价 — 几乎肯定是普通绿茶冒充。

**12. 有趣事实：**

* **6–10 % 胺基酸：** 三清山白茶是世界胺基酸含量纪录保持者之一。比较：安吉白茶 — 5–7 %，标准龙井 — 2–3 %，日本玉露 (遮荫) — 4–6 %。三清山变异种无需遮荫即超越玉露。

* **联合国教科文组织山麓茶：** 三清山2008年列入世界遗产名录 (标准VII，卓越自然美景)。茶园位于遗产地缓冲区 — 施加最严格生态限制，但保证独特环境纯净。

* **「天然氧吧」：** 玉山县是中国茶区中少数「氧吧」之一。空气负离子浓度比城市高50倍 — 原始森林特有指标。

* **「味特清冽……远甚不及」：** 同治年间 (1862–1874) 县志记录三清山茶的优越性早在十九世纪 — 比科学解释 (白叶变异 + 胺基酸特征) 早一个半世纪。

* **维生素A原 — 12.6毫克/100克：** 茶类中最高值之一 — 叶绿素降低和白叶中类胡萝卜素积累增加的有益副作用。

* **从「云雾」到「白茶翡翠」：** 三清山经历了不寻常的品牌演变。1970年代制作传统绿茶「云雾」，1990年代产业崩溃，2000年代通过完全不同的产品复兴：白叶变异。这是中国茶业中地区在一代人时间内完全改变「茶身份」的少数例子之一。

**13. 与其他白叶绿茶比较：**

* **安吉白茶 (安吉白茶, Ānjí Báichá)：** 浙江。相同原理 — 白叶变异 (白叶一号, 白叶一号)，按绿茶工艺加工。外形 — 扁平「兰花形」(凤形/龙形)。胺基酸5–7 % vs 三清山6–10 %。安吉更「叶绿素」和「绿茶」口感，带明显栗香；三清山更甘甜、「奶香」，花香背景更明显，花岗岩土壤带来矿物质调。安吉知名度更高且更易获得；三清山是极致胺基酸特征的小众奢华产品。

* **天目湖白茶 (天目湖白茶, Tiānmùhú Báichá)：** 江苏天目湖地区。同样白叶变异 (白叶一号品种)。口感更轻盈「水质」，「厚度」较少。三清山因炭火烘干和高山风土 (758–1182公尺 vs 200–400公尺) 香气更深厚。

* **资溪白茶 (资溪白茶, Zīxī Báichá)：** 江西资溪县 (邻近地区)。类似变异类型，但海拔较低 (400–800公尺)。胺基酸特征不如明显。三清山因联合国教科文组织缓冲区具有更「极致」成分和更严格生态标准。

**结语：**

三清山白茶是茶中悖论：名为「白茶」却是绿茶；叶片乳白色但升温后变绿；口感奶油甘甜毫无苦涩，但既未加糖也未调味。这一切都源于最稀有的基因变异，在神圣道教名山的凉爽云雾中苏醒，胺基酸浓度达到百分之十，空气比城市清洁50倍。以80 °C水温在玻璃杯中冲泡 — 观察翡翠白芽头带翠绿叶脉在透明水中绽放，释放出那种「味特清冽」的韵味，一个半世纪前令无名县志作者惊叹不已。
