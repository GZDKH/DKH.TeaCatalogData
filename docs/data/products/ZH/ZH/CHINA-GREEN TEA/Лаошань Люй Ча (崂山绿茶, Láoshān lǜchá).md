# 崂山绿茶 (崂山绿茶, Láoshān lǜchá)

**崂山绿茶 (崂山绿茶, Láoshān lǜchá)**
![崂山绿茶 (崂山绿茶, Láoshān lǜchá)](images/Лаошань%20Люй%20Ча%20(崂山绿茶,%20Láoshān%20Lǜchá).png)

![崂山绿茶 (崂山绿茶, Láoshān lǜchá)-1](images/Лаошань%20Люй%20Ча%20(崂山绿茶,%20Láoshān%20Lǜchá)-1.png)

崂山绿茶——中国广泛栽培的绿茶中最北的品种，诞生于道教圣山崂山的花岗岩峭壁与黄海咸风交汇之处。此茶是1950年代开始的大规模农业实验「南茶北引」(南茶北引, Nán chá běi yǐn)成功的活见证，推翻了数百年来北纬30度以北无法进行商业茶叶种植的定论。

**1. 分类与产地：**

* **类型：** 绿茶（未发酵）。杀青方式——炒青（炒青, chǎoqīng）。
* **类别：** 华北绿茶。被誉为「江北第一名茶」（江北第一名茶, Jiāngběi dìyī míngchá）——「长江以北第一名茶」。
* **产地：** 中国，山东省（山东省, Shāndōng Shěng），青岛市（青岛市, Qīngdǎo Shì），崂山区（崂山区, Láoshān Qū）。地理标志保护区域涵盖四个街道办事处（街道办事处, jiēdào bànshìchù）：王哥庄（王哥庄, Wánggēzhuāng）、沙子口（沙子口, Shāzǐkǒu）、中韩（中韩, Zhōnghán）和北宅（北宅, Běizhái）。这是世界上最北的商业茶区之一——约北纬36度。
* **地理座标：** 北纬36°03′–36°20′，东经120°24′–120°43′。

**2. 历史与文化意义：**

* **历史：**

崂山（崂山）——中国最古老的道教中心之一，被称为「海上第一名山」——「海上第一名山」。然而，这里的茶叶种植历史却出人意料地年轻，始于二十世纪下半叶。

1957年，青岛市园林管理处发起了将南方茶树北移的计划。第一批——从黄山（安徽省）运来的5000株两年生茶苗——被种植在太平山南坡的中山公园（中山公园）苗圃中。由于挖掘时间不当和根部受损，所有茶苗在第二年春天前全部死亡。

尽管失败，尝试仍在继续。1959年，「南茶北引」（南茶北引，「南方茶叶北移」）计划被正式认定为成功：来自安徽、浙江和福建省的茶苗在崂山山坡、道教寺庙太清宫（太清宫, Tàiqīnggōng）——「太清宫」附近成功存活。这一成果推翻了科学界关于北纬30度以北无法种茶的既定观点。

初期阶段存活率仍然很低，未进行大规模种植。1972年，《人民日报》（《人民日报》）发表了关于山东「南茶北引」成功的报导。1973年，农业部在日照县（日照）召开了六省「南茶北引西迁」（南茶北引西迁）——「南茶北移西迁」经验交流会。自1990年代起，青岛市政府采取了一系列支持措施：补贴、技术支援、鼓励农民从传统粮食种植转向茶叶种植。到2004年，崂山区茶园面积超过12,000亩（约800公顷），同年5月举办了首届「崂山茶节」（崂山茶节），展出了60多个品种。

2004年通过了首批地方标准：崂山绿茶的「生产技术规程」和「加工技术规程」——青岛首批茶叶标准。2006年，国家质检总局（国家质检总局）授予「崂山绿茶」地理标志产品保护地位（地理标志产品保护，第161号令）。2011年批准了国家标准GB/T 26530-2011「地理标志产品——崂山绿茶」。

* **名称：**

    * 「崂山」（崂山）——黄海边的圣山，道教重要中心之一，以山泉清水闻名。「崂」字独特，几乎只用于此地名。
    * 「绿茶」（绿茶）——「绿茶」。

* **文化意义：** 尽管茶叶种植历史仅有半个多世纪，崂山绿茶已成为青岛和整个山东省不可或缺的「名片」。它已深深融入当地待客文化：向客人奉上崂山绿茶是尊重的表现，而茶叶本身是胶东半岛（胶东半岛）节日和婚礼的必备礼品。与崂山道教遗产的深厚联系赋予茶叶特殊的精神内涵，尽管历史上道教僧侣并未在此种茶。现代生产者积极发展「茶旅融合」（茶旅融合），邀请游客在山景和道教寺庙的背景下参与采茶和炒茶。

**3. 植物学描述与原料：**

* **品种／栽培种：** *Camellia sinensis* var. *sinensis*。使用适应山东寒冷气候的中叶和小叶抗寒栽培种。根据国家标准，主要品种包括：黄山群体种（黄山群体种, Huángshān qúntǐzhǒng）——来自黄山的群体种，鸠坑（鸠坑, Jiūkēng）——浙江最古老的栽培种之一，祁门（祁门, Qímén），龙井43（龙井43, Lóngjǐng 43）——早熟、高产的无性系，以及福鼎大白（福鼎大白, Fúdǐng Dà Bái）。茶树形态紧凑，叶片小而有光泽，呈圆形，叶绿素含量高，叶片增厚——这是适应北方气候的结果。
* **采摘：** 三个主要季节：春茶（春茶, chūnchá，4月中旬至5月初，被认为最佳）、夏茶（夏茶, xiàchá）和秋茶（秋茶, qiūchá）。由于气候寒冷，生长期比南方晚得多——当地茶叶常常是其纬度地区最早的茶叶之一。主要采用手工采摘。
* **采摘标准：** 高级茶——一芽一叶或一芽二叶（一芽一叶, yī yá yī yè；一芽二叶, yī yá èr yè）。叶片必须完整、嫩、大小均匀。
* **原料要求：** 新鲜、未受损的嫩芽，具有崂山特有的叶片厚度。不允许枯萎、冻伤或成熟度不均的叶片。

**4. 风土条件与栽培特点：**

* **地区：** 崂山区位于山东半岛东南部，崂山山脉（崂山）山麓，直接濒临黄海（黄海, Huánghǎi）。崂山是中国最高的滨海山峰（主峰巨峰，巨峰，1132.7公尺）。茶园位于缓坡（坡度30°以内）和丘陵地带，背风向阳（背风向阳, bèifēng xiàngyáng）。
* **海拔高度：** 主要在低山坡——从海平面到200-400公尺。靠近海洋和花岗岩峭壁创造了独特的微气候——一种「阳光陷阱」，白天岩石吸收热量，夜晚释放，缓和温差。
* **土壤：** 棕壤（棕壤, zōng rǎng），发育于花岗岩母岩。深厚肥沃的土层（不少于60公分），pH值4.5-6.5，有机质含量超过1%。花岗岩成因确保了丰富的矿物质成分：石英、铁、锌、硒、锰——赋予茶叶特有的矿物质风味。
* **气候：** 温带海洋性季风气候（温带海洋性季风气候, wēndài hǎiyángxìng jìfēng qìhòu）。夏季温和湿润（7月平均温度+25°C），冬季寒冷（1月平均温度约-1°C，最低可达-15°C）。年降水量约700-800毫米。频繁的海雾和海风确保了高空气湿度和自然光线散射。
* **特点：** 主要农业技术特点——需要保护茶树免受冬季霜冻。使用聚碳酸酯温室、塑胶薄膜覆盖和覆盖物。寒冷的冬季不仅是挑战，也是优势：长达约5个月的休眠期和在凉爽气候中极其缓慢的生长，使叶片能够积累比南方同类更多的胺基酸（特别是L-茶胺酸）、矿物质和叶绿素。实行大豆间种（大豆间种, dàdòu jiānzhòng）以自然富集土壤氮素——这是当地农民开发的创新方法。灌溉使用崂山山泉清水，其矿物质成分也影响茶叶的味道。

**5. 制作工艺：**

崂山绿茶的制作工艺旨在最大程度保持叶片的新鲜度，并发展出特有的「豌豆香」（豌豆香, wāndòu xiāng）——煮豆（毛豆）香气，这是此茶的标志。生产两种主要风格：**卷曲形**（卷曲形, juǎnqū xíng）——卷曲（螺旋形）和**扁形**（扁形, biǎn xíng）——扁平（龙井风格）。

* **采摘（采摘, cǎi zhāi）：** 早晨手工采摘嫩芽。高级茶——一芽一叶或二叶。

* **摊凉（摊凉, tān liáng）：** 新鲜采摘的叶片薄摊在竹盘上，在凉爽通风的室内放置3-4小时，以去除表面水分并开始温和的生化转变，为炒制做准备。

* **杀青（杀青, shāqīng）：** 关键步骤。叶片在旋转加热的滚筒（滚筒杀青, gǔntǒng shāqīng）中快速炒制，温度约180°C。高温使氧化酶失活，停止发酵过程，并赋予茶叶特有的炒制香气（豆类、栗子、烤面包）。

* **揉捻／做形（揉捻/做形, róuniǎn / zuò xíng）：** 赋予叶片形状。根据风格：
    * *卷曲形（卷曲形）：* 叶片卷成螺旋状，类似碧螺春。这种风格更常见。
    * *扁形（扁形）：* 叶片压平，赋予扁平形状，模仿龙井。这种风格突出叶片的光滑度和翡翠色。

* **烘干（烘干, hōnggān）：** 在适中温度下最终干燥，去除残余水分，稳定形状并固定香气特征。

* **分级（分级, fēnjí）：** 手工分级成品茶：去除受损叶片、茎梗，确保均匀性。根据国家标准GB/T 26530-2011，成品茶分为四个等级：特级（特级, tèjí，有两个子类别）、一级（一级）、二级（二级）和三级（三级）。

**6. 感官特征：**

* **干茶外观：** 取决于风格。卷曲形（卷曲形）：紧结的小螺旋，绿色带明显的银白色茸毛。扁形（扁形）：光滑、扁平的翡翠绿色叶片。两种风格的共同特点——叶片明显比南方绿茶更厚更密实。
* **干茶香气：** 明亮、新鲜，带有特有的煮毛豆或炒毛豆香（豌豆香, wāndòu xiāng）、栗子香（栗香, lì xiāng）、温暖面包香，有时带甜玉米香。轻微的花香或草本香调。
* **茶汤香气：** 保持豆类栗子香调，辅以绿豆、菠菜的新鲜感，奶油和矿物质的细微差别。香气持久，逐渐展开。
* **滋味：** 主要特征——明显的天然甜味（甘醇, gānchún）、柔和度和浓厚的茶体（滋味浓, zīwèi nóng）。毛豆、炒瓜子、奶油香调，轻微矿物质感。正确冲泡时几乎没有苦味。回甘（回甘, huígān）——持久、甜美、清新，带矿物质余韵。茶叶耐冲泡（耐冲泡, nài chōngpào）——可达5-7次完整冲泡。
* **汤色：** 明亮、清澈、浅黄或黄绿色（黄绿明亮, huánglǜ míngliàng），常带轻微「黄头」（带黄头, dài huángtóu）——这是区别于南方绿茶的特征。有时可见白色茸毛微粒造成的轻微乳光。
* **叶底（泡开的叶片）：** 鲜绿、嫩、完整的叶片和芽。叶片展开缓慢且不完全——即使经过多次冲泡仍保持轻微皱褶，证明叶片的厚度。

崂山绿茶的关键感官特征可用公式表达：**「叶片厚、豌豆香、滋味浓、耐冲泡」**——「厚叶、豆香、浓郁味、耐泡」。

**7. 化学成分：**

* **多酚类（儿茶素）：** 含量高，特别是表没食子儿茶素没食子酸酯（EGCG），提供强大的抗氧化特性。
* **胺基酸：** L-茶胺酸（L-茶氨酸, L-chá ānjīsuān）含量高——崂山绿茶的主要特点之一，由凉爽气候中的缓慢生长造成。L-茶胺酸赋予茶叶天然甜味（鲜味）和温和的放松效果。由于植物的低温胁迫，可能还含有微量的GABA（γ-胺基丁酸）。
* **生物碱：** 适量的咖啡因（咖啡因）、可可碱和茶碱，提供温和的提神效果。
* **叶绿素：** 由于适应北方高纬度长日照的厚实致密叶片，叶绿素b含量高。确保饱和的绿色。
* **矿物质：** 富含锌、硒、锰、铁、钾、氟——这是崂山花岗岩土壤和山泉水独特矿物质成分的结果。
* **维生素：** 含有维生素C、E、B群、胡萝卜素。

**8. 保健功效：**

* **抗氧化保护：** 高含量的儿茶素和矿物质（硒、锌）提供有效的细胞氧化应激保护。
* **改善认知功能：** L-茶胺酸和咖啡因的有利比例有助于提高注意力集中和思维清晰度，而不会过度兴奋。
* **支持心血管系统：** 绿茶多酚有助于正常化脂质代谢和改善血管弹性。
* **放松和抗压效果：** 高含量的L-茶胺酸有助于降低压力水平和改善情绪。
* **支持免疫力：** 来自独特花岗岩土壤的矿物质复合物（硒、锌、锰）增强身体的防御功能。
* **改善新陈代谢：** 促进脂肪代谢和整体身体活力。

**9. 冲泡方法：**

* **水温：** 70-80°C。过热的水会增加苦味并「烫伤」嫩叶。
* **茶量：** 功夫泡法（功夫泡法, gōngfu pàofǎ）每150-200毫升水用5-7克；玻璃杯泡法每200毫升用2-3克。
* **茶具：** 瓷盖碗（盖碗, gàiwǎn）——经典选择。玻璃杯或玻璃壶——可观察厚实致密叶片的展开。瓷茶壶。不建议使用宜兴紫砂——它会吸收细腻的豆香。
* **过程（功夫泡法）：**
    1. 用开水温润茶具。
    2. 将干茶放入盖碗或茶壶。
    3. 快速洗茶：注水后立即倒出（崂山绿茶因叶片厚度可承受洗茶）。
    4. 第一泡——15-30秒。
    5. 后续冲泡——逐渐增加时间（20、30、45秒等）。
    6. 茶叶可冲泡5-7次，展现不同层次的味道：从开始的豆类甜味到最后的矿物质清新感。

**10. 储存方法：**

崂山绿茶对光、湿、热和异味敏感。建议储存在密封、不透光的包装中（真空铝箔袋、密封盖的锡罐），置于阴凉干燥处，远离热源和强烈气味。长期储存（12-18个月）——在0-5°C的冰箱中，使用可靠密封的包装，以防止吸收异味和取出时形成冷凝水。室温下——应在2-3个月内饮用。

**11. 价格与假冒：**

崂山绿茶属于高价位茶叶。价格高昂的原因包括产量有限（茶园总面积约16,000亩，年产量约900吨）、严苛的种植条件（需要温室、防霜保护）、高比例的手工劳动和巨大需求。春茶（特别是「头春」，头春）最贵。在中国国内市场，优质春季崂山绿茶价格为每500克500-3000元人民币。国际市场——每100克25-60美元。

* **如何避免假冒：**

    * 从有权使用「崂山绿茶」地理标志的可靠卖家购买。注意GI标记和生产商资讯。
    * 评估外观：真正的崂山绿茶叶片明显比南方绿茶更大更厚，带有特征性的绿黄色调和白色茸毛。过于细小、薄、完美均匀的亮绿色茶叶——值得怀疑。
    * 检查香气：必须有「豌豆香」——毛豆/栗子香气。冒充崂山的南方茶通常缺乏这种特征香气。
    * 测试耐泡性：真正的崂山绿茶可冲泡5-7次而不失味道。假冒品在2-3次冲泡后就「失味」。
    * 价格过低——主要警讯：由于生产的客观限制，真正的崂山绿茶不可能便宜。

**12. 趣闻轶事：**

* 崂山绿茶生动地反驳了陆羽在《茶经》（《茶经》）中「茶者，南方之嘉木也」的论述。「南茶北引」的成功证明，在正确的农业技术下，茶叶可以在传统南方地区之外的地方种植。
* 迫使茶树「休眠」约5个月的寒冷冬季——矛盾的是——是崂山的主要优势：长期休眠和缓慢生长确保了异常高的胺基酸含量，使茶叶异常甜美浓郁。
* 道教圣山崂山以其泉水闻名，当地的水（崂山矿泉水, Láoshān kuàngquán shuǐ）被认为是中国最适合泡茶的水之一。水的矿物质成分是独特味道的另一个因素。
* 2006年在地方层面实施了「地理标志+区块链」系统——使用区块链技术追踪茶叶从茶园到柜台的真实性，这是中国茶叶产业的先驱项目之一。
* 崂山茶叶种植的当地农民先驱除了绿茶外，还尝试用同样的原料生产红茶（黑茶/红茶）、黄茶、白茶甚至乌龙茶。最著名的是崂山红茶（崂山红茶）——带有明显黑巧克力和焦糖香调的红茶。

**13. 与其他绿茶的比较：**

* **西湖龙井（西湖龙井, Xīhú Lóngjǐng）：** 龙井——来自浙江的扁平炒青（炒青）茶，带有坚果、栗子香气。崂山绿茶的扁形风格外观类似龙井，但叶片明显更厚，有明显的豆香（而非坚果香），耐泡性显著更强。
* **洞庭碧螺春（洞庭碧螺春, Dòngtíng Bìluóchūn）：** 崂山的卷曲形风格在形状上类似碧螺春，但味道和香气截然不同：碧螺春——花果香、轻盈、优雅；崂山——豆香、浓厚、油润。
* **日照绿茶（日照绿茶, Rìzhào Lǜchá）：** 最近的「邻居」和竞争对手——同样来自山东省（日照市）的北方绿茶。两种茶都是「南茶北引」的产物，但崂山由于独特的滨海微气候和花岗岩土壤，具有更明显的矿物质特征和更细腻的香气。
* **黄山毛峰（黄山毛峰, Huángshān Máo Fēng）：** 来自安徽的细嫩春季烘青（烘青）绿茶，带有花香兰花香气。毛峰——更轻盈花香；崂山——更浓厚、甜美和「土味」。

**结语：**

崂山绿茶是活生生的证明，伟大的茶可以违背所有规则和典范而诞生。不是由数百年的传统创造，而是由科学的大胆、农民的坚持和独特风土的慷慨——在道教圣山的花岗岩峭壁与黄海咸风相遇之处——这种茶在半个世纪内从农业实验发展成为「江北第一名茶」。其浓郁的豆香、油润的甜味、矿物质的深度和惊人的耐泡性——这一切都是漫长寒冷冬季的馈赠，它迫使每片叶子吸收最大的味道。崂山绿茶将为那些欣赏饱满、浓郁、有个性的绿茶并寻求与众不同味道的人带来享受。
