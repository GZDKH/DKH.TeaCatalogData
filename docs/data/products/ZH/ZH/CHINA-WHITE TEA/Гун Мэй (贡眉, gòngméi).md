# 贡眉 (贡眉, gòngméi)

**贡眉 (贡眉, gòngméi)**
![贡眉 (贡眉, gòngméi)](images/Гун%20Мэй%20(贡眉,%20Gòngméi).png)

![贡眉 (贡眉, gòngméi)-1](images/Гун%20Мэй%20(贡眉,%20Gòngméi)-1.png)

**1. 分类与产地：**

* **类型：** 白茶（轻发酵茶，氧化程度约5-10%）。采用传统白茶工艺制作——不杀青、不揉捻——仅萎凋、干燥和分级。
* **类别：** 中国传统白茶。在白茶等级中位列第三，仅次于白毫银针（白毫银针，Báiháo Yínzhēn）和白牡丹（白牡丹，Bái Mǔdān）。为国家标准GB/T 22291-2017《白茶》所规定的四大白茶品类之一。
* **产地：** 中国福建省（福建，Fújiàn）。主要产区包括：
	* **建阳县（建阳，Jiànyáng）：** 贡眉的历史发源地，具体为漳墩镇（漳墩，Zhāngdūn）及其周边地区，包括南坑村（南坑，Nánkēng）。清代时期，当地以菜茶（菜茶）品种创制了现代贡眉的雏形。
	* **政和县（政和，Zhènghé）：** 现代白茶主要产区之一，包括贡眉和寿眉。政和白茶以茶汤醇厚、花果香显著而闻名。
	* **松溪县（松溪，Sōngxī）和建瓯县（建瓯，Jiàn'ōu）：** 与建阳、政和共同构成传统贡眉的核心产区。
	* **福鼎市（福鼎，Fúdǐng）：** 虽然福鼎历史上更以白毫银针和白牡丹闻名，但也生产贡眉。
* **地理座标：** 约北纬27°00'-27°30'，东经117°30'-120°00'（建阳-政和-福鼎区域）。

**2. 历史与文化意义：**

* **历史：** 贡眉与白茶的历史渊源深厚。其前身——「小白」（小白，xiǎo bái）或「南坑白」（南坑白，Nánkēng bái）——于清朝乾隆年间（清，Qīng），约1772至1782年间，由建阳县漳墩镇肖氏家族（肖）创制。采用当地菜茶品种，以简单的萎凋和干燥工艺制成。著名茶学家张天福（张天福，Zhāng Tiānfú）曾言：「先有小白，后有大白，再有水仙白」（先有小白，后有大白，再有水仙白），强调了「小白」（即贡眉前身）在福建白茶史上的先驱地位。历史上，菜茶也曾用于制作白毫银针，但19世纪末大叶种（福鼎大白、政和大白）推广后，分工明确：大叶种专制「银针」和白牡丹，菜茶专制贡眉。「贡眉」之名出现较晚——20世纪中叶：1940年代的文献中，水吉（水吉）分水岭地区的白茶仅分为「白牡丹」和「寿眉」，尚无「贡眉」类别。据传，漳墩优质寿眉被清朝宫廷采购作为贡品（贡品，gòngpǐn），故得名「贡」。1984年，漳墩白茶荣获「中国名茶」称号，后「贡眉」商标正式注册，确立了这一名称在茶业中的地位。
* **名称释义：**
	* 「贡」（贡）——进贡、贡品。在帝制中国，优质茶叶作为贡品（贡品）进献朝廷。「贡」字彰显茶叶的上乘品质。
	* 「眉」（眉）——眉毛。形容茶叶修长微弯的形态，宛如眉毛。此字亦见于寿眉（寿眉，「长寿之眉」）。
* **文化意义：** 贡眉在福建日常茶文化中占有重要地位。作为性价比最高的白茶，其风味介于清雅花香的白牡丹与浓郁醇厚的寿眉之间。在福鼎，贡眉（与寿眉）历来是茶农的「口粮茶」（口粮茶，kǒuliáng chá）。因含有较多成熟叶片和叶梗，贡眉极适合长期存放陈化，随著时间推移，会发展出更复杂深邃的风味——在适当储存条件下，会呈现干果、枣香、香料和蜂蜜的香气。正如俗语所言：「一年茶，三年药，七年宝」（一年茶，三年药，七年宝）。

**3. 植物学描述与原料：**

* **品种/栽培种：** 贡眉与其他白茶的关键区别在于使用菜茶（菜茶，càichá），又称「群体种」（群体种，qúntǐ zhǒng）。这是*Camellia sinensis var. sinensis*的地方半野生品种，通过种子（有性繁殖）繁衍，不同于无性繁殖的栽培品种。菜茶为矮小灌木，叶片较小，种群内基因多样性丰富，赋予成茶更丰富多层次的风味。根据国家标准GB/T 22291-2017定义，贡眉是以群体种茶树嫩梢制成的白茶。然而实际上，由于菜茶种植面积减少，现代商业生产中常使用大叶种：福鼎大白茶（福鼎大白茶，Fúdǐng Dàbáichá）、福鼎大毫茶（福鼎大毫茶，Fúdǐng Dàháochá）、政和大白茶（政和大白茶，Zhènghé Dàbáichá）以及福安大白茶（福安大白茶，Fú'ān Dàbáichá）。传统菜茶贡眉现主要保存于建阳。茶树种质资源研究者虞富莲（虞富莲）指出，近亲繁殖的群体品种具有更强的生命力，个体间的生化互补性赋予茶叶更饱满的滋味和更高的耐泡度。
* **采摘：** 春茶为主，采摘时间晚于白毫银针和白牡丹，通常在3月底至4月。秋茶采摘（约9-10月）也很常见，秋茶香气更显（称为「秋香」，秋香，qiū xiāng），而春茶滋味更醇厚圆润（「春水」，春水，chūn shuǐ）。这种特性被概括为「春水秋香」。
* **采摘标准：** 一芽二三叶（一芽二三叶）。不同于白毫银针（纯芽）和白牡丹（一芽一二叶），贡眉允许使用更成熟、叶片更多的原料。须有芽头（毫心明显），但芽头相对叶片的比例较高等级白茶为小。
* **原料要求：** 叶片须完整无损，无机械损伤，在干燥天气采摘。嫩梢手工采摘，选取大小和成熟度一致的嫩梢。

**4. 风土与栽培特点：**

* **福建省：** 贡眉产区位于福建西北部的丘陵和低山地带，亚热带季风气候带来温暖的冬季和炎热潮湿的夏季。年均温17-19°C，年降雨量1400-1800公厘，相对湿度约78-82%。
* **建阳（建阳）：** 贡眉生产的历史核心。位于建溪（建溪）流域，武夷山脉东南侧。地形为丘陵，主要茶园海拔200-600公尺。土壤以黄壤和红壤为主，呈酸性（pH 4.5-5.5），富含有机质，排水良好。
* **政和（政和）：** 海拔较高的产区，茶园海拔400-900公尺。气候较建阳凉爽，减缓嫩梢生长速度，有利于氨基酸积累。土壤为酸性红壤和黄壤。
* **福鼎（福鼎）：** 滨海地区，茶园平均海拔300-700公尺，因靠海湿度较高。含火山矿物质的红壤赋予茶叶特有的甜味。
* **生长海拔：** 海拔200-900公尺，因具体产区而异。相较于白毫银针，海拔因素对贡眉的影响较小，因为使用的是更成熟的原料。

**5. 制作工艺：**

贡眉的制作工艺体现了白茶哲学的精髓：对自然过程的最小干预。这是茶叶加工中最「温和」的工艺之一——无炒青、无揉捻、无强烈发酵。仅有两个关键步骤加上最后的分级。

* **采摘（采摘，cǎi zhāi）：** 手工采摘「一芽二三叶」标准的嫩梢。在干燥天气的早晨，露水散去后进行。鲜叶放入竹篮，避免损伤和挤压嫩叶。
* **萎凋（萎凋，wěidiāo）：** 制作的核心环节，决定成茶品质。采摘的嫩梢均匀薄摊在竹匾或竹筛上。萎凋采用两种方式之一（或结合）：
	* **自然日光萎凋：** 竹匾置于阳光下（散射光或直射光，视日照强度而定）。此法使叶片缓慢失水。
	* **室内萎凋（室内萎凋）：** 竹匾置于通风良好的室内。此法用于不利天气条件（雨天、湿度过高）。
	萎凋时间36-72小时，视天气条件、摊叶厚度和原料类型而定。萎凋过程中水分缓慢流失（从75-78%降至20-25%），启动轻微的酶促反应——多酚氧化、叶绿素和蛋白质分解、芳香化合物形成。正是在此阶段形成白茶特有的甜味、花香和果香，同时儿茶素含量降低，氨基酸含量增加。
* **干燥（干燥，gānzào）：** 萎凋后进行最终干燥，将残余水分降至4-6%。采用两种方法：
	* **日晒干燥（晒干，shàigān）：** 传统方法，萎凋叶在阳光下继续干燥。
	* **烘干（烘干，hōnggān）：** 在专用烘箱或炭火上的竹匾中低温（40-55°C）干燥。此法效果更稳定。
* **拣剔与分级（拣剔，jiǎntī / 分级，fēnjí）：** 成茶经分级，剔除瑕疵叶、茎梗和杂质。根据GB/T 22291-2017标准，贡眉分为四个等级：特级（特级，tèjí）、一级（一级，yī jí）、二级（二级，èr jí）和三级（三级，sān jí）。特级芽头比例最高，原料更嫩，叶色更浅。

**6. 感官特征：**

* **干茶外观：** 芽叶混合，银白色茸毛（白毫）明显。叶片修长微弯，形似眉毛（故名「眉」）。色泽灰绿至褐绿带银色芽头。与白牡丹相比，贡眉成熟叶片较多，芽头比例较少。叶片略厚实肥嫩（叶张稍肥嫩），叶梗可见。
* **干茶香气：** 清新甘甜，花香明显，伴有蜜香和果香。与白牡丹相比，香气更「成熟」——草本和木质香更明显，有时带干草和落叶香。
* **茶汤香气：** 饱满多层次：新茶呈花蜜香，带果香和绿意；陈年贡眉（老贡眉）呈温暖包容的香气，带枣香、龙眼、蜜饯、肉桂和陈木香。
* **滋味：** 比白牡丹更饱满、醇厚、有「茶体」，但比寿眉更柔和优雅。甘甜清爽，略有怡人的涩感，回甘（回甘，huígān）持久包容。新贡眉以花香、蜜香和果香为主，带青草韵味。陈化3年以上后，滋味更深邃，出现枣香和干果味，甜味更「成熟」、「蜜糖般」，涩感几乎消失。冲泡得当的老贡眉可耐泡10-15次以上而滋味不减。
* **茶汤颜色：** 新茶——浅黄带绿，透亮清澈；随陈化年份增加，汤色加深至琥珀色、蜜色，老茶呈红琥珀色。
* **叶底（冲泡后茶叶）：** 完整有弹性的嫩梢，保持原形——一芽二三叶连枝。色泽灰绿至褐绿，透光可见红色叶脉（优质贡眉的特征）。

**7. 化学成分：**

贡眉与其他白茶一样，因加工程度最低，保留了叶片的天然成分，生物活性物质含量高。由于原料较成熟，某些成分含量比纯芽白茶更高。

* **多酚类（儿茶素）：** 白茶多酚总含量为干重的18-26%，高于某些绿茶。主要儿茶素包括EGCG（表没食子儿茶素没食子酸酯）、ECG（表儿茶素没食子酸酯）、EGC（表没食子儿茶素）和EC（表儿茶素）。白茶的EGCG含量仅次于绿茶。长期储存时儿茶素含量逐渐降低，但黄酮类化合物含量增加。
* **氨基酸：** 白茶的游离氨基酸含量在所有茶类中最高——研究显示，比同原料制成的其他茶类高1-2倍。L-茶氨酸（茶氨酸）约占游离氨基酸总量的70%，赋予茶叶特有的甜味和柔和口感。
* **生物碱：** 咖啡因含量约为干重的2.5-4%。贡眉的咖啡因含量略低于白毫银针和白牡丹，因为成熟叶片的咖啡因含量低于嫩芽。还含有少量可可碱和茶碱。
* **黄酮类化合物：** 白茶的黄酮类含量极高——8.5-13毫克/克，显著高于其他茶类。特别值得注意的是二氢杨梅素（二氢杨梅素，èrqīng yángméisù）——天然护肝成分。随著储存时间延长，黄酮类含量增加，这解释了陈年白茶的价值提升。
* **茶多糖：** 由于使用含叶梗和茎的成熟原料，贡眉的茶多糖含量高于白毫银针。
* **维生素：** C、B₁、B₂、PP以及类胡萝卜素。因无高温处理，白茶的维生素C保存优于绿茶。
* **矿物质：** 钾、钙、镁、磷、锌、氟、锰、铁。矿物质组成取决于特定风土的土壤成分。
* **精油：** 香气由挥发性化合物复合体形成：芳樟醇、香叶醇、顺式茉莉酮、β-紫罗兰酮、苯甲醛等。陈化过程中香气特征显著转变。

**8. 保健功效：**

* **抗氧化保护：** 高含量的多酚和黄酮类提供强大的抗氧化活性，有助于中和自由基，延缓细胞老化。研究显示，室温冲泡的白茶比热水冲泡表现出更高的抗氧化活性。
* **增强免疫力：** 多酚、氨基酸和维生素C共同刺激免疫系统，提高机体抗感染能力。研究证实白茶具有抗病毒和抗菌特性。
* **护肝作用：** 二氢杨梅素（白茶特有的黄酮类）对肝细胞有保护作用，促进其修复并减少酒精和其他有害物质的毒性影响。
* **调节代谢：** 白茶含有活性酶，有助于脂肪分解和碳水化合物代谢正常化。多酚和咖啡因共同刺激脂质代谢，可能有助于体重控制。
* **支持心血管系统：** 白茶中的儿茶素和黄酮类有助于降低「坏」胆固醇（低密度脂蛋白），正常化血压，改善血管弹性。
* **镇静与提神效果：** 白茶中的L-茶氨酸具有独特特性——同时温和提神和放松，刺激α脑电波产生。这提供了平静专注的状态，而无过度兴奋。
* **口腔保健：** 白茶中的氟化物和儿茶素在口腔中表现出显著的抗菌作用，降低蛀牙风险，维护牙龈健康。
* **改善皮肤状况：** 白茶的抗氧化剂保护皮肤免受光损伤，延缓衰老迹象。传统中医建议白茶用于「内热」和炎症过程。

**9. 冲泡方法：**

* **水温：** 85-95°C。新贡眉（1-2年内）宜用85-90°C水温，避免「烫伤」嫩叶；陈年贡眉（3年以上）可用90-95°C甚至沸水——较高水温更能释放深沉的「陈韵」。
* **茶量：** 功夫泡法5-7克/100-150毫升水；茶壶或大杯泡法3-5克/200-300毫升。
* **茶具：** 白瓷盖碗（盖碗，gàiwǎn）——最佳选择，可控制浸泡时间并充分欣赏香气。玻璃茶壶（观赏叶片「舞动」）和陶瓷器皿也适合。陈年贡眉可用紫砂壶（紫砂壶，zǐshā hú）——多孔陶土突显老茶的柔和与深度。老贡眉也极适合在玻璃或陶瓷壶中煮饮（煮，zhǔ）。
* **冲泡步骤：**
	1. 用沸水温润盖碗或茶壶，倒掉水。
	2. 将干茶放入温热的茶具。闻温热干茶的香气。
	3. 注入适温的水，立即倒出第一泡（润茶，润茶，rùn chá）。这能唤醒茶叶并去除灰尘。
	4. 第二泡——浸泡15-20秒（功夫泡）或2-3分钟（茶壶）。
	5. 将茶汤倒入茶杯。
	6. 重复冲泡5-8次（新贡眉）或10-15次（陈年），每泡增加5-10秒浸泡时间。
	7. 贡眉也极适合冷泡（cold brew）：5克茶叶加500毫升冷水，冰箱冷藏4-8小时。

**10. 储存方法：**

贡眉（及白茶整体）的重要特性之一是能够长期储存并随时间改善。国家标准GB/T 22291-2017明确指出，在适当条件下白茶可长期储存。

* **陈化储存条件：** 干燥环境，相对湿度40-65%，避免阳光直射，无剧烈温度变化（最佳18-28°C），远离异味。在此条件下，茶叶缓慢进行后发酵——黄酮类含量增加，滋味柔和，香气复杂化。
* **容器：** 长期储存建议三层包装：内层铝箔、中层牛皮纸、外层纸箱。可用带密封盖的陶瓷或陶土容器。重要的是不使用塑胶和聚乙烯——它们产生温室效应，阻碍茶叶「呼吸」。
* **茶叶的敌人：** 过度湿气（导致发霉）、阳光直射（破坏叶绿素和芳香物质）、异味（茶叶易吸收）、剧烈温度变化。
* **陈化潜力：** 优质贡眉可储存并改善10-20年以上。3-5年茶被视为「年轻陈茶」，风味柔和宜人；7-10年——经典「老白茶」，带深沉的枣香和龙眼味；10年以上——具独特个性的收藏级茶。

**11. 价格与辨伪：**

贡眉是福建知名白茶中最实惠的，是认识白茶类别的绝佳起点。其价格显著低于白毫银针和白牡丹，但高于寿眉。影响价格的因素：年份（陈年贡眉比新茶贵数倍）、等级（特级、一级、二级、三级）、采摘季节（春茶价高于秋茶）、产区（建阳和政和为优质产区）以及品种类型（传统菜茶比大叶种贡眉贵）。标准品质的新贡眉约100-400元人民币/500克；高品质传统菜茶贡眉或陈年茶价格可能显著更高。

**如何避免假货：**

* **向可靠商家购买：** 有信誉的专业茶叶店或福建的可靠供应商。注意是否有产地和生产年份资讯。
* **评估外观：** 真正的贡眉应在成熟叶片中含有可见的带白毫的芽头。完全没有芽头表明可能是寿眉或低级原料。叶片应完整，不破碎。
* **检查香气：** 干茶应有清新宜人的香气——花香、蜜香、干草香。霉味、酸味或发霉气味表明储存条件不当。
* **评估茶汤：** 汤色从浅黄到琥珀色（视年份而定），必须透明清澈。浑浊的茶汤是劣质茶或工艺不当的标志。
* **警惕「老茶」低价：** 陈年白茶市场充斥假货——经湿热快速处理的「人工做旧」茶。真正的陈年贡眉口感纯净「透明」，无霉味和潮味。

**12. 趣闻轶事：**

* 茶学家张天福以「小白→大白→水仙白」概括福建白茶的「谱系」：正是「小白」——菜茶贡眉的前身——是历史上最早的白茶，比「银针」和白牡丹的出现早了数十年。
* 贡眉和寿眉合计约占福建省白茶总产量的50%，使它们成为产业的「主力军」，而白毫银针和白牡丹仍是类别的「门面」。
* 老贡眉（老贡眉）——唯一传统上不仅冲泡还煮饮的白茶：将压制茶饼的一块放入玻璃或陶瓷壶，加冷水煮沸。这种方法最大程度地释放陈茶的深度和甜味。
* 在东南亚，特别是越南，白茶（包括贡眉）传统上被视为退烧药，在民间医学中用于降低儿童体温。
* 「一年茶，三年药，七年宝」这句话最适用于贡眉和寿眉：由于含有富含多糖的成熟叶片和叶梗，这些茶在储存时转化特别明显且可预测。

**13. 与其他白茶的比较：**

* **白毫银针（白毫银针，Báiháo Yínzhēn）：** 白茶最高等级。纯芽，白毫最多。口感极其细腻、「丝滑」，以奶香、奶油香和鲜草香为主。价格比贡眉高3-10倍。耐泡度较低（3-5泡）。陈化潜力较低。
* **白牡丹（白牡丹，Bái Mǔdān）：** 一芽一二叶。花香（牡丹、铃兰）比贡眉更明显，但滋味不如贡眉饱满和「有茶体」。价格比贡眉高1.5-3倍。中等陈化潜力。
* **牡丹王（牡丹王，Mǔdān Wáng）：** 白牡丹最高等级，芽头特别大。介于白毫银针和标准白牡丹之间。花香更浓郁，奶油香明显。
* **寿眉（寿眉，Shòu Méi）：** 白茶最「粗老」类别——成熟叶片，芽头最少。口感更厚重、「土味」、草本味，带木质香。价格低于贡眉。陈化和煮饮潜力极佳。
* **月光白（月光白，Yuèguāng Bái）：** 云南大叶种（*Camellia sinensis var. assamica*）白茶。与福建白茶明显不同：口感更「强劲」，蜜果香明显，带特有的蜂蜜、干杏和开花草本香。不同的风土和品种创造了完全不同的茶体验。

**结语：**

贡眉是一款集实惠、丰富口感和惊人时间发展潜力于一身的白茶。数百年前在福建西北部山村以朴素的半野生菜茶品种创制，从无名的「小白」发展成为国家标准认定的独立类别。贡眉是认识白茶世界的绝佳起点：其饱满甘甜的口感带花蜜香韵，即使初学者也能欣赏，而其多年陈化并逐渐展现新层次——枣香、香料、木质香——的能力使其对资深品茗者而言趣味无穷。无论冲泡或煮饮，新茶或陈茶，贡眉始终带来温暖柔和的茶汤，让人一再回味。
