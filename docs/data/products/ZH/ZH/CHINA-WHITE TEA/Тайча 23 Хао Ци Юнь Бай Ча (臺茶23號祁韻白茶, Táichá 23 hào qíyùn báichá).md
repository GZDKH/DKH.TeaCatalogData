# 台茶23号祁韵白茶 (台茶23号祁韵白茶, Táichá 23 hào qíyùn báichá)

**台茶23号祁韵白茶 (台茶23号祁韵白茶, Táichá 23 hào qíyùn báichá)**

台茶23号祁韵白茶是台湾新一代白茶，采用TTES №23「祁韵」(祁韵, Qíyùn)品种制成。此品种源自中国著名红茶祁门(Qímén)的种子培育而成。尽管具有「红茶」血统，此品种在白茶制程中展现出卓越潜力，呈现复杂的花果香气与优雅甜味——是台湾「茶叶复兴」最耀眼的代表之一。

**1. 分类与产地：**

* **类型：** 白茶（轻发酵茶，氧化程度低于12%）。
* **类别：** 实验性台湾白茶，采用小叶种红茶品种制成。极限量生产的小众产品。
* **品种：** TTES №23 (台茶23号, Táichá 23 hào)，商业名称——祁韵 (祁韵, Qíyùn)。小叶种 *Camellia sinensis* var. *sinensis*，源自中国安徽省祁门 (祁门, Qímén) 茶区植物种子培育。
* **产地：** 台湾南投县 (南投县, Nántóu xiàn)。主要产区为名间乡 (名间乡, Míngjiān xiāng) 及鱼池乡 (鱼池乡, Yúchí xiāng) 周边地区。
* **地理座标：** 约北纬23.84°，东经120.70°（名间地区）。

**2. 历史与文化意义：**

* **历史：** TTES №23品种的血统始于西元1938年（昭和27年），当时台北帝国大学 (台北帝国大学, Táiběi Dìguó Dàxué) 的山本亮 (山本亮, Yamamoto Ryō) 教授从安徽省祁门茶区带回种子，并交给鱼池红茶试验支所（台湾总督府中央研究所鱼池红茶试验支所），即现今的台湾茶业改良场鱼池分场（TRES鱼池分场）。祁门种子被选定用于开发能与世界标准竞争的台湾红茶。经过数十年的观察与实验工作，在西元2001-2002年间，从累积的材料中选出具有潜力的「祁办1」品系，展现出优异品质。西元2015-2017年进行与对照品种青心乌龙 (青心乌龙, Qīngxīn wūlóng) 的比较试验，证实新品种在多项指标上的优越性。西元2018年（民国107年）正式注册为TTES №23。商业名称「祁韵」（祁韵，「祁门的韵味」/「祁门的回响」）于西元2019年5月18日TRES 116周年庆典上经投票选出，从「红悦」(红悦)、「祁玉」(祁玉)、「祁韵」(祁韵) 和「红祁」(红祁) 四个选项中胜出。

    虽然此品种原本是为生产红茶而开发，但将嫩叶和芽以白茶技术进行实验性加工，发现其在此类别中具有卓越潜力。祁韵白茶因其优雅的花果香气迅速获得品茶人士的认可。

* **名称：**「台茶23号」(台茶23号) ——「台湾茶第23号」，品种注册编号。「祁韵」(祁韵) ——「祁门的韵味」或「祁门的回响」：「祁」(祁) 字指向祁门——母本种子的原产地，而「韵」(韵) 意为「韵律」、「节奏」、「余韵」——台湾茶美学的核心概念，代表味道的深度与共鸣。「白茶」(白茶) ——「白茶」。
* **文化意义：** 祁韵白茶象征台湾茶叶复兴的新浪潮——从大量生产转向创造独特、高品质的小众产品。此茶体现跨洲际基因传承的理念：中国祁门的种子经过数十年台湾选育，获得全新的身份认同。祁韵白茶年产量极少——约200公斤，使其成为最稀有的台湾茶之一。

**3. 植物学描述与原料：**

* **品种：** *Camellia sinensis* var. *sinensis*，品种TTES №23（祁韵）。小叶种灌木型 (灌木型, guànmù xíng) 品种，中等高度，枝条直立生长。具有强劲的生长活力、高密度的芽头和单位面积高产量。对疾病和干旱具有显著抗性。
* **形态学：** 枝条高度可达120公分。成熟叶片呈长椭圆形，长8-10公分，宽3-4公分，边缘细锯齿状，表面略呈波浪状，呈雾面灰绿色。嫩芽具有特征性的白黄红渐层色彩；芽头密布银色绒毛（茸毛），厚度达0.2公厘——高品质白茶原料的关键特征。春季萌芽比对照品种青心乌龙早约2周。
* **原料：** 仅使用首批春茶——「明前」(明前, Míngqián，清明节前)。采摘期为2月底至3月初（南投地区为4月初）。采摘标准——手工采摘的嫩梢（一芽一二叶），长度不超过4公分，无机械损伤。芽与嫩叶的最佳比例约为70/30（按重量计）。成品率约为鲜叶重量的25%。

**4. 风土与栽培特点：**

* **地区：** 南投县名间乡 (名间乡)，台湾中部。也在鱼池乡周边及邻近地区栽培。
* **海拔：** 海拔300-350公尺。
* **土壤：** 酸性红土（pH 4.5-5.5），形成于浊水溪 (浊水溪, Zhuóshuǐ xī)——台湾最大河流的花岗岩冲积层上。这些土壤提供优良排水、适度的有机质含量和影响茶叶风味的特殊矿物质组成。
* **气候：** 亚热带气候，降雨丰沛（约1800公厘/年，夏季最多），年均温约21°C。特色是频繁且浓厚的雾气（每年超过150天），减缓茶树生长，降低光合作用强度，促进嫩芽中氨基酸和芳香物质的累积。日夜温差达8-10°C，进一步刺激叶片中的生化过程。
* **特点：** 茶树在半遮荫环境下生长，由天然的山茶花和其他树木提供遮荫，模拟野生茶的森林生长条件，有助于L-茶氨酸的累积。使用有机肥料（堆肥），虽然农场可能没有正式的有机认证。TTES №23白茶栽培的先驱是余姓农民，管理不到0.5公顷的此品种茶园。茶园位于浊水溪谷，确保稳定的水源供应，河谷雾气额外保护嫩芽免受直射阳光照射，这是白茶原料品质的重要因素。

**5. 制作工艺：**

工艺旨在以最少干预最大程度保持叶片的原始外观和化学成分。关键原则——「不揉不炒」(不揉不炒, bù róu bù chǎo)。

* **日光萎凋 (日光萎凋, rìguāng wěidiāo)：** 采摘的叶片在散射阳光下薄层摊放（≤5公分），温度约28°C，持续约45分钟，直到水分降至约65%。此阶段启动初始氧化过程并形成香气基础。
* **室内萎凋 (室内萎凋, shìnèi wěidiāo)：** 茶叶移至温度（约25°C）和湿度（约85%）受控的室内，在旋转竹筒中继续萎凋约18小时。轻微的机械搅拌确保均匀萎凋。当多酚氧化程度达到低于12%时停止。
* **烘干 (烘干, hōnggān)：** 茶叶在约105°C的热风下进行两次干燥，直到最终含水量不超过5%。双重干燥确保储存时风味稳定并完全停止酵素活动。
* **无揉捻：** 关键特点——完全省略揉捻 (揉捻, róuniǎn) 步骤，保持叶片、银芽的完整性和最大量的绒毛。
* **品质控制：** 每批次都经过实验室控制，使用红外光谱法检查儿茶素含量（应>干重的18%）。这保证了生物活性物质的稳定高含量。

**6. 感官特征：**

* **干茶外观：** 未揉捻、完整、针状的银绿色芽，长度15公厘以上，碎片极少。银色绒毛浓密且明显。
* **干茶香气：** 复杂、多层次。主要是茉莉花和新鲜割草（梯牧草）的香气，辅以细致的矿物质气息，让人联想到雨后湿润花岗岩的气味。成熟叶片中也有木质柑橘调，但早期芽头的白茶原料以花香为主。
* **茶汤香气：** 发展并深化干茶的花香——茉莉花、木兰花、淡淡的果香。冷却后呈现新鲜苹果（富士品种）、郁金香和花蜜的香气。
* **口感：** 多面向，层层展开。第一层——木兰花的花香甜味与细致的矿物质基调。中段——榛果的坚果奶油调，柔软的质地。尾韵——姜根的轻微辛辣感和让人联想到未熟柿子的细腻涩感。质地——光滑、包覆感，对白茶而言具有明显的厚度。回甘 (回甘, huígān) ——持久，带有甜美的花蜜余韵。
* **茶汤颜色：** 透明、明亮的黄色，带有淡金色光泽。
* **叶底 (叶底, yèdǐ)：** 完整、有弹性的浅绿色叶片和芽头，保持良好形状。芽头上的绒毛仍然可见。

**7. 化学成分：**

早期采摘和温和加工确保了极高的生物活性物质含量：

* **多酚类：** 儿茶素含量高，特别是EGCG（表没食子儿茶素没食子酸酯）——约占干重的14%。总儿茶素含量超过18%（实验室控制）。相比之下：福建银针的EGCG含量通常为8-12%。
* **氨基酸：** L-茶氨酸含量显著高——超过干重的5%，大大超过白茶的平均水平。温和干燥可保留鲜叶中80%以上的原始氨基酸。氨基酸与多酚的高比例决定了明显的甜味和柔和度。
* **生物碱：** 咖啡因含量适中——约占干重的2%。可可碱和茶碱——微量。
* **维生素：** 维生素C含量显著——每100克干物质超过150毫克，高于大多数绿茶。含有B群维生素。
* **矿物质：** 氟化物含量——约0.03%。钾、镁、锰、锌。由于浊水溪谷冲积红土的特性，矿物质成分丰富。
* **芳香化合物：** 白茶加工最能充分展现祁韵的品种香气——挥发性化合物谱包括芳樟醇（花香）、橙花叔醇（果香）、乙酸苄酯（茉莉花），以及祁门基因遗产特有的调性，创造出可识别的「斯里兰卡」前调香气。

**8. 保健功效：**

* **抗氧化作用：** 高抗氧化活性——ORAC值约980 µmol TE/g，超过许多传统上被认为是「超级食品」的产品（蓝莓、石榴）。EGCG提供显著的自由基中和作用。
* **血糖调节：** 抑制α-淀粉酶的能力可能有助于减缓淀粉分解并降低餐后血糖水平。对预防代谢症候群具有意义。
* **神经保护作用：** 初步研究表明L-茶氨酸和多酚复合物对神经细胞具有潜在保护作用，可能对预防神经退化性疾病有重要意义。
* **无嗜睡的放松效果：** 高含量的L-茶氨酸（>5%）有助于产生大脑α波，提供「平静警觉」的状态——放松专注而无镇静作用。
* **免疫系统支持：** 儿茶素、多糖和维生素C的复合作用具有整体强化和免疫刺激作用。
* **对皮肤的有益影响：** 高含量的维生素C和抗氧化多酚有助于胶原蛋白生成并保护皮肤免受紫外线损伤。

**9. 冲泡方法：**

* **水温：** 80-85°C。更高的温度可能萃取过多苦味并破坏细致的芳香化合物。
* **茶量：** 3-5克配150-200毫升水。
* **茶具：** 瓷盖碗 (盖碗, gàiwǎn) ——最佳选择，可控制浸泡时间并观察叶片展开。玻璃茶壶也适合。不建议使用吸收细致香气的无釉陶器。
* **用水：** 软水、过滤水，矿物质含量低。
* **步骤：**
    1. 用热水温润茶具。
    2. 投入茶叶，让芽头预热10-15秒。
    3. 第一泡——15-20秒，倒掉。这是洗茶 (洗茶, xǐ chá)：唤醒叶片并去除细小的绒毛颗粒。
    4. 第二泡——45-60秒。享受第一波香气。
    5. 后续冲泡——每次逐渐增加10-15秒浸泡时间。
    6. 茶叶可冲泡5-7次。味道从花香茉莉演变为坚果和蜂蜜味。

**10. 储存方法：**

* 储存在密封、不透光的包装中，置于干燥、阴凉处，远离气味强烈的产品和直射阳光。最佳湿度——不超过45%。
* 如同其他白茶，具有陈化潜力：在适当储存条件下（密封包装、稳定温度15-25°C、防潮），茶叶随时间会发展出更深的蜜饯干果调。
* 为保持新鲜的花香特征，建议在冰箱（0-5°C）密封包装中储存，并在第一年内饮用。
* 陈化时（1至5年）——储存在干燥通风处，避免温度剧烈变化；预期蜂蜜和坚果调会增强，同时花香会柔化。一些行家指出，一年的陈化让祁韵白茶展现额外的深度，当初始的花香亮度转化为更复杂的「秋季」特征，带有干果和栗子的香气。然而，此茶的主要价值正是新鲜的春季特征，大多数品茶者偏好在第一年饮用。

**11. 价格与仿冒品：**

* **价格类别：** 超高档。由于产量极其有限（约200公斤/年）、手工采摘和最高品质，价格大大超过其他台湾白茶和许多乌龙茶。零售价——每100克60至120美元以上，取决于批次和销售管道。
* **成本因素：** 品种极度稀有（先驱生产者的茶园不到0.5公顷）、手工劳动、超小批量、品种新颖、实验室检验的品质控制。
* **如何避免仿冒品：**
    * 仅向与台湾茶农有直接联系的可靠专业供应商购买。目前祁韵白茶的生产者圈子极小。
    * 要求产地资讯：标明南投县名间或鱼池地区。
    * 评估外观：完整、未揉捻、银绿色芽头，绒毛浓密。碎片或揉捻叶片的存在表示仿冒或替代品。
    * 检查香气和味道：特征性的茉莉花、草地和矿物质香气；无薄荷调（与TTES №18的区别）和烟熏味。
    * 提防以更便宜的台湾或中国白茶替代。

**12. 趣闻轶事：**

* 祁韵的基因谱系反映了茶叶选育的复杂历史：西元1938年日本教授在中国安徽省收集的祁门种子被带到台湾，经过80年的选育工作，孕育出全新的品种，创造出白茶——这个类别在历史上既不是祁门也不是台湾的特色。
* 尽管有「红茶」血统，在白茶加工中祁韵展现的芳香谱包括富士苹果、郁金香、檀香、青梅、花蜜、月桂和紫罗兰的香气——这是完全发酵时无法获得的调性。
* 在TTES №23商业名称竞赛中，「祁韵」击败了「红祁」(红祁)、「红悦」(红悦) 和「祁玉」(祁玉)。选择「韵」(韵——韵律、回响、余韵) 字强调了台湾茶业的美学取向，对其而言「韵」的概念是茶的最高特征之一。
* TRES最初将TTES №23定位为「四季皆宜」的红茶品种（一年四季均可产制高香型红茶）：全年都可采收和加工。然而，白茶仅使用早春原料，这进一步限制了产量。

**13. 与其他白茶的比较：**

* **台茶18号红玉白茶 (台茶18号红玉白茶)：** 采用大叶种杂交种TTES №18制成的台湾白茶。明亮的薄荷樟脑香气、果香甜味、油润质地。相比之下，祁韵——花香、优雅，具有矿物质基调，无薄荷调。如果红玉是异国情调和个性，那么祁韵就是精致和深度。
* **福鼎白毫银针 (福鼎白毫银针)：** 福建银针的标准。新鲜、矿物质感，带有竹子和干草堆的香气。祁韵的特点是更明显的花香甜味、坚果余韵和「温暖」的整体特征，而福鼎银针更「凉爽」和简洁。
* **祁韵红茶 (祁韵红茶)：** 同一品种TTES №23制成的红茶——该品种培育的主要产品。完全氧化，茶汤橙红色，甜美的花果香气和适度的涩感。根据TRES的官方描述，祁韵红茶具有「橙红明亮的茶汤，精致的甜花果香，甜润饱满的口感与宜人的涩味」。与白茶版本相比——更厚实、结构化，但少了「轻盈感」，失去了最少加工所保留的部分细致花香细节。
* **云南大理茶银针 (云南大理茶银针)：** 野生 *C. taliensis* 制成的白茶。性格更加柔和和「森林感」，带有兰花和木兰花的香气。祁韵——更有结构，中段有坚果味，尾韵有姜的辛辣感。

**14. 禁忌事项：**

* **个人不耐受：** 如同任何茶，可能引起个人反应。
* **咖啡因：** 尽管含量适中（约2%），对咖啡因敏感、睡眠障碍或心律不整的人应控制饮用量。
* **怀孕和哺乳：** 建议咨询医生；适度饮用（每天1-2杯）通常被认为是可接受的。
* **与药物的相互作用：** 茶多酚可能影响某些药物的代谢。服用处方药时最好咨询专家。

**结语：**

台茶23号祁韵白茶浓缩了台湾茶叶选育的全部历史和潜力。从日本学者在殖民时代带来的著名祁门种子，经过八十年的耐心工作——到没有人计划但却成为新品种最诗意表达的白茶。优雅的花香甜味、带有茉莉花和榛果香气的多层次芳香、持久的余韵和最高含量的有益物质，使这款茶成为寻求每一杯绝对品质和独特性的鉴赏家的理想选择。祁韵——杯中的「韵律」，而且它的声音是全新的。
