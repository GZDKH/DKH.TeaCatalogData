# 霍山黄芽 (霍山黄芽, Huòshān huáng yá)

**霍山黄芽 (霍山黄芽, Huòshān huáng yá)**

![Хошань Хуан Я (霍山黄芽, Huòshān huáng yá)](images/Хо%20Шань%20Хуан%20Я%20(霍山黄芽,%20Huò%20Shān%20Huáng%20Yá).png)

![Хошань Хуан Я (霍山黄芽, Huòshān huáng yá)-1](images/Хошань%20Хуан%20Я%20(霍山黄芽,%20Huòshān%20Huáng%20Yá)-1.png)

**霍山黄芽 (霍山黄芽, Huòshān huáng yá)** — 中国四大传统黄茶之一，也可能是有文献记载最古老的黄茶：其踪迹可在司马迁的《史记》中发现——这是中华文明的基础文献之一。这款茶诞生于大别山脉的核心地带——这座山系分隔著中国南北，而在这个地理位置中蕴藏著它的特性：霍山黄芽是一款处于交汇点的茶，是一座桥梁之茶，北方的矿物质刚劲在此遇见南方的温柔甘甜。其工艺建立在「摊放黄变」(摊放黄变, 「摊放使其黄变」) 基础上——这是黄茶中最从容不迫、最具禅意的闷黄方法，茶叶不在堆中「闷」，也不用纸包裹，而是简单摊放，让其自然黄变，无压力无急躁——一至两天，个别师傅甚至长达十天。招牌特色是「板栗香」(板栗香, 炒栗子香气)，因此霍山黄芽被称为「三鲜茶」(三鲜, sān xiān)：香气鲜、滋味鲜、汤色鲜。

**1. 分类与产地：**

* **类型：** 黄茶 (黄茶, huángchá)，轻发酵茶。属于「黄芽茶」(黄芽茶, huáng yá chá) 子类别——原料品质最高的等级。
* **类别：** 中国四大传统黄茶 (中国四大传统黄茶) 之一——与君山银针、蒙顶黄芽、平阳黄汤并列。历史宫廷茶。地理标志保护产品 (2006年)。制作技艺列入安徽省非物质文化遗产名录。2024年——列入《中国农耕农品记忆索引名》。
* **产地：** 中国安徽省 (安徽, Ānhuī) 六安市 (六安, Lù'ān) 霍山县 (霍山县, Huòshān Xiàn)。霍山位于大别山脉 (大别山, Dàbié Shān) 中部——这座山系分隔长江与淮河流域，是中国南北的天然分界线。核心产区——大化坪镇 (大化坪镇, Dàhuàpíng Zhèn)：金鸡山 (金鸡山, 「金鸡山」)、金鸡凼 (金鸡凼)、金竹坪 (金竹坪) 和乌米尖 (乌米尖)，以及磨子潭镇 (磨子潭镇) 的高山林区。这些地方在茶界被称为「三金一乌」(三金一乌, sān jīn yī wū)——按地名首字而得名，正是在这里生产最高品质的茶叶。
* **地理座标：** 约北纬31°，东经116°。

**2. 历史与文化意义：**

* **历史：**
    * **西汉 (西汉, 西元前206年—西元8年) — 首次记载：** 司马迁《史记》载：「寿春之山有黄芽焉，可煮而饮，久服得仙」(寿春之山有黄芽焉，可煮而饮，久服得仙)。霍山当时隶属寿州 (寿州)，「寿春之山」即今霍山县境内群山。这是中国文献中最早的黄茶记载之一——距今2000多年。需要说明的是，当时的「黄芽」可能仅指黄色茶芽，而非经闷黄工艺处理的茶叶。
    * **唐代 (唐, 618–907年) — 宫廷茶地位：** 李肇在《国史补》(《国史补》) 中将「寿州霍山黄芽」(寿州霍山黄芽) 列入十四种宫廷茶名单。书中还记载了著名的外交插曲：「常鲁公使西蕃，烹茶帐中。赞普问：'此为何物？'鲁公曰：'茶也，涤烦疗渴，所谓茶也。'赞普曰：'我亦有此。'命出示，曰：'此寿州者，此顾渚者。'」霍山茶因此在八至九世纪已传至西藏。唐代黄芽以饼茶 (饼茶, 饼茶) 和小团 (小团, 小团) 形式生产。《膳夫经手录》(《膳夫经手录》) 记载：「寿州有霍山小团，盖仿龙牙小饼；数甚少。」
    * **宋代 (宋, 960–1279年) — 重要贸易中心：** 设立「霍山茶场」(霍山茶场)，年销售量266,154.5斤 (~133吨)。黄芽逐渐从饼茶形式转为散茶 (散茶)，虽然杀青仍采用蒸青 (蒸青) 工艺。
    * **明代 (明, 1368–1644年) — 鼎盛期与现代工艺诞生：** 黄芽列入皇室贡品。《六安州志》(《六安州志》) 记载：初始贡额200袋茶叶；霍山析县后 (1496年) 六安25袋，霍山175袋 (87.5%！)，显示「六安茶」实际主要产自霍山。官员曹琥 (曹琥) 在《注黄芽茶疏》(《注黄芽茶疏》) 中抱怨：「岁贡芽茶仅二十斤……而正德十年 (1515年) 征收芽茶一千二百斤，细茶六千斤……一斤芽茶索银一两，且不易得。」县令王毗翁 (王毗翁) 亲自督制茶叶，留下《黄芽焙茗诗》(《黄芽焙茗诗》)：「露蕊纤纤才吐碧，即防叶老采须忙，家家篝火山窗下，每到春来一县香」(露蕊纤纤才吐碧，即防叶老采须忙，家家篝火山窗下，每到春来一县香)。明代发生决定性技术转变：杀青由蒸改为炒 (改蒸为炒)，出现闷黄 (闷黄, 闷黄) 工序——现代意义的黄茶诞生。明代作者许次纾 (许次纾) 在《茶疏》(《茶疏》) 中写道：「江北则霍山茶最多……山西、陕西人皆用之。南方谓其能消垢腻，去积滞，亦甚宝爱。」
    * **清代 (清, 1644–1911年) — 宫廷「内用茶」：** 霍山黄芽被定为「内用」(内用) ——供皇室成员个人饮用，地位高于一般宫廷贡茶。光绪《霍山县志》(《霍山县志》)：「南乡乌米尖、瓜芦尖诸峰所产，为全县茶品之冠；制法精工，价倍他乡。」另一处记载：「茶为山货之首。上者曰银针，次曰雀舌，又次曰梅花片……」
    * **1915年 — 巴拿马博览会：** 霍山茶「抱儿钟秀」(抱儿钟秀, 「怀抱婴儿，钟灵毓秀」) 品牌获巴拿马万国博览会金奖——获奖黄茶中的唯一代表。
    * **1940–1960年代 — 失传：** 战争和经济动荡导致黄芽生产实际停止。技术未有文字记录，仅存于少数老茶农记忆中。据某些资料，复原前曾为山东商人生产名为「米茶」(米茶, 「米茶」) 的茶叶。
    * **1971–1972年 — 复原：** 霍山县茶叶局组织乌米尖考察。三名技术人员与三名70–80岁老茶农共同重建工艺。1972年4月27–30日制作试验批次14斤 (7公斤)。6斤装入白铁罐直接送国务院审评——这是极其罕见的事实：复原第一年即「呈报政府」的茶叶。次年开始在三个点正规生产：金鸡山 (主要)、乌米尖和金竹坪。1972年样品用于国家级外宾接待。产量逐步增长：1973年178公斤，1980年644公斤，1985年3700公斤。
    * **2006年 — 地理标志。** 2019年——中国茶叶流通协会授予霍山「中国黄茶之乡」(中国黄茶之乡) 称号。至2022年霍山茶园面积达206,400亩 (~13,760公顷)。

* **名称：**
    * 「霍山」(霍山) — 霍山，即霍山县。「霍」(霍) 字意为「迅速」、「突然」——可能指山势陡峭。
    * 「黄芽」(黄芽) — 「黄色芽头」。在最古老的文献中可能仅指「黄色嫩芽」，但自明代起指经闷黄工艺处理的茶叶。
    * 历史地名：寿州 (寿州) — 霍山茶在《史记》和《国史补》中以此名出现。

* **文化意义：** 霍山黄芽是少数几款历史可由连续文献链证实的茶叶，从西汉至今：司马迁 → 陆羽 → 李肇 → 许次纾 → 曹琥 → 王毗翁 → 县志 → 巴拿马博览会 → 1972年国家复原。金鸡山金鸡传说——关于神奇茶树散发无法形容的香气，由一对金鸡守护，每年仅在谷雨前黎明鸡鸣时显现一次——是安徽茶文化神话中最诗意的故事之一。霍山位于大别山「茶叶走廊」(茶叶走廊) 上，这条走廊从豫南经皖西延伸——历史茶叶贸易路线之一。

**3. 植物学描述与原料：**

* **品种：** 主要栽培品种——霍山金鸡种 (霍山金鸡种, 「霍山金鸡品种」) — 当地群体品种，省级认定。特征：茶多酚14.9%，胺基酸4.97% — 「双高」(双高, shuāng gāo) 的特殊组合，同时提供涩味和甘甜。持嫩性强 (持嫩性强)，适应山地条件良好。辅助使用：槠叶齐 (槠叶齐) 和黄山大叶种 (黄山大叶种) — 增强香气和变化性。
* **采摘：** 谷雨 (谷雨, ~4月20日) ± 2–3天为主要采摘期。核心产区「三金一乌」因海拔较高，采摘期较晚，接近四月底。总采摘期约一个月，3–4批春茶。
* **采摘标准：** 特一级 (特一级) — 一芽一叶初展 (一芽一叶初展)，金色茸毛，形如「雀舌」。特二级 (特二级) — 一芽一叶至一芽二叶初展 (一芽一叶至一芽二叶初展)。一级 — 一芽二叶 (一芽二叶)。二级 — 成熟「对夹叶」(对夹叶)。
* **原料要求：** 实行「三个一致，四不采」(三个一致，四不采) 原则：形状、大小、颜色一致；不采开口芽 (开口芽)、虫伤芽 (虫伤芽)、霜冻芽 (霜冻芽)、紫色芽 (紫色芽)。全程使用竹制工具；严禁接触铁器 (全程忌铁器防腥, 「全程避免铁器防止腥味」)。

**4. 风土条件与栽培特点：**

* **地区：** 霍山位于大别山脉 (大别山) 中心，处于安徽、湖北、河南三省交界。主峰白马尖 (白马尖, 1774公尺)。县境被西南—东北走向山脉切割；特殊地质现象——「霍山弧」(霍山弧, Huòshān Hú) — 山脉急剧弯曲，形成众多具有独特气候条件的微谷地。冰碛岩 (冰碛岩, 冰川砾岩) — 约6亿年前的古老岩石，在核心产区露出，为土壤提供独特矿物成分。
* **海拔：** 标准原料≥600公尺。核心产区金鸡凼 ~720公尺。金鸡凼茶园面积仅约3亩 (~0.2公顷)，年产量不足50公斤，解释了该微产区茶叶的极度稀有和高价格。
* **土壤：** 黄棕壤 (黄棕壤)，由冰碛岩 (冰碛岩) 形成。pH 5.0–6.5。有机质含量 ~2.5%。富含硒 (Se) — 霍山土壤的特色。所谓「乌砂土」(乌沙土, 「黑砂土」) — 砂质成分与黄壤混合。结构疏松，排水极佳。
* **气候：** 亚热带与温带过渡性气候。年均温 ~15.1°C。年降水量1100–1600毫米。相对湿度≥80%。雾云天数达181天。昼夜温差8–10°C — 积累芳香物质和胺基酸的重要因子。森林覆盖率75.1%。佛子岭 (佛子岭) 和磨子潭 (磨子潭) 水库位于淠河上游，进一步调节微气候。
* **特点：** 霍山弧形成众多微气候，解释了口感的显著变化：金鸡山 (金鸡山) 茶 — 更浓厚、更油润；乌米尖 (乌米尖) 茶 — 更具矿物质感、更严谨；磨子潭 (磨子潭) 茶 — 更温柔、更花香。霍山位于中国东部茶区北缘 (我国东部茶叶产区的北缘)，进一步强化「边界」特性 — 生长缓慢，萌发较晚，风味物质积累最大化。

**5. 制作工艺：**

霍山黄芽工艺与其他黄茶的黄变方法不同：这里使用的不是包纸闷黄 (如蒙顶黄芽) 也不是堆积闷黄 (如海马宫茶)，而是「摊放黄变」(摊放黄变, 「摊放使其黄变」) — 茶叶简单摊放成薄层，在室温下缓慢黄变。这是最「禅意」的闷黄方法。完整工序包括：

* **摊放 (摊放 — tān fàng)：** 鲜叶摊放在竹筛上 (竹制簸箕)。时间1–2小时。部分水分蒸发，开始香气形成。
* **杀青 (杀青 — shā qīng)：** 两阶段炒制：
    * **生锅 (生锅, 「生锅」)：** 温度 ~150°C。快速高温处理以钝化酶活性。
    * **熟锅 (熟锅, 「熟锅」)：** 温度 ~130°C。做形 — 使茶叶呈现特有的「雀舌形」(雀舌形, quèshé xíng)：笔直、略微展开的条索。传统师傅使用青冈木炭 (青杠木炭) 柴火 — 认为这种火给出更纯净的香气，无烟熏味。
* **初烘 (初烘 — chū hōng)：** 温度 ~100°C。烘至约70%干度。
* **摊放黄变 (摊放黄变 — tān fàng huáng biàn)：** 关键独特工序。烘干的茶叶摊放成薄层，在室温下放置1–2天。这是「干闷」(干闷, gān mèn)：茶叶缓慢黄变，无强制加热和高湿度。发生叶绿素逐渐分解，酯型儿茶素非酶氧化，形成黄色色素和特有柔和度。某些师傅将此工序延长至10天以上，以获得最大「黄茶特性」深度。其他师傅交替「干闷」和「湿闷」(湿闷) — 茶叶在杀青后趁热堆积。
* **复烘 (足火 — zú huǒ)：** 温度 ~90°C。烘至约90%干度。
* **二次摊放 (摊放)：** 再次均匀水分和完善黄变。
* **拣剔 (拣剔 — jiǎn tī)：** 去除不合格叶片、茶梗、杂质。
* **足火 (复火 — fù huǒ)：** 温度100–120°C。烘至完全干燥。之后茶叶可用「踩筒」(踩筒, cǎi tǒng) 方法装入竹篮 — 压实以便紧凑储存。

**6. 感官特征：**

* **干茶外形：** 笔直、略微展开的条索，似雀舌 (形似雀舌, xíng sì quèshé)。大小均匀，聚成整齐「朵状」(匀齐成朵)。色泽嫩绿带黄，油润有光 (嫩绿披毫, nèn lǜ pī háo)。白毫或金毫丰富。
* **干茶香气：** 清纯持久，明显的炒栗香 (板栗香, bǎnlì xiāng) — 霍山黄芽的主要香气特征。还有淡淡的煮玉米香 (毫香, máo xiāng — 茸毛香气)、花香和蜜香层次。
* **茶汤香气：** 「清香持久」(清香持久) — 清纯、持久、优雅。栗香为基调；后续冲泡展现花香和果香层次。霍山黄芽香气更「北方」，更内敛矿物质感，较南方黄茶。
* **滋味：** 「鲜醇浓厚」(鲜醇浓厚) — 鲜爽、醇和、浓郁、油润。甘甜清爽。特有双重性：初始涩味 (比蒙顶黄芽或平阳黄汤更明显) 转为深沉持久的回甘 (回甘)。苦味极轻。淡淡「清凉感」(清凉感)，归因于土壤中硒含量高。滋味是四大黄茶中最具「矿物质感」和「结构感」的。胺基酸含量≥5.2%，多酚类≥28%。
* **汤色：** 「黄绿清澈」(黄绿清澈) — 黄绿色，透明，明亮有金色光泽。高等级茶 — 纯净发光。
* **叶底 (冲泡后茶叶)：** 嫩黄明亮，整齐成朵 (嫩黄明亮，匀齐成朵)。完整、柔嫩、饱满。

**7. 化学成分：**

* **多酚类：** ≥28% 干物质 — 黄茶中的高含量。长时间「干燥」黄变将部分酯型儿茶素转化为更温和形式，但保留大量原生化合物，解释了比其他黄茶更明显的涩味。
* **胺基酸：** ≥5.2% 干物质。L-茶胺酸为主要成分。提供明显甘甜和回味中的「鲜味」。金鸡种栽培品种胺基酸4.97% — 在原料阶段就为柔和度奠定基础。
* **生物碱：** 咖啡因 — 标准含量。与L-茶胺酸协同作用提供温和提神效果。
* **维生素：** 维生素C、B族维生素、维生素E。
* **矿物质：** 钾、镁、锌。氟 — 含量达246毫克/公斤 (高含量，对牙齿和骨骼健康有意义)。硒 (Se) — 冰碛岩霍山土壤的特色。
* **茶多糖 (茶多糖, cháduōtáng)：** 含量丰富，具免疫调节活性。

**8. 健康功效：**

* **脂质代谢支持：** 高多酚含量 (≥28%) 有效促进脂肪分解。效果评估比同等原料绿茶高约1.8倍。
* **牙齿和骨骼健康：** 高氟含量 (246毫克/公斤) 有助牙釉质强化。
* **免疫调节：** 茶多糖激活巨噬细胞 — 免疫系统细胞。
* **抗氧化保护：** 双重系统 — 多酚类 + 硒 — 提供强大抗氧化活性。
* **温和提神：** L-茶胺酸 + 咖啡因 — 平静清醒的经典组合。促进放松和减压而不困倦。
* **胃部温和作用：** 尽管多酚含量较高，长时间黄变 (1–2天以上) 软化儿茶素的刺激性。但因残留单宁，不建议空腹饮用。
* **视力支持：** 传统中医认为黄茶对眼睛有益。

**9. 冲泡方法：**

* **水温：** 80–90°C。建议将水煮沸后晾凉约2分钟。过热的水可能「烫伤」嫩芽并引起苦味。
* **茶量：** 3克配150毫升水。
* **茶具：** 玻璃杯 — 观察汤色和芽叶展开的美感。白瓷盖碗 — 最大程度展现香气。
* **过程：**
    1. 用开水温杯，倒掉。
    2. 投入3克茶叶。
    3. 注入80–90°C水至1/3容量。润湿所有茶叶，等待30秒。第一泡不要倒掉 — 其中含有最多茸毛香 (毫香) 和栗香；倒掉意味著失去主要的「第一鲜」。
    4. 续水至7/10容量。浸泡1–2分钟。
    5. 观察「春笋出土」(春笋出土)：芽头垂直沉入杯底，如破土而出的竹笋。茶汤应呈黄绿色，透明。
    6. 重复冲泡：可冲3次，每次增加15–20秒。

**10. 储存：**

最佳 — 铝箔袋或铁罐/瓷罐密封包装。冰箱 (0…+5°C) 或冷冻库 (−10…−18°C)。室温 — 阴凉干燥处，远离异味；6个月内饮用。茶叶天敌：光线、热量、湿气、异味、氧气。传统储存 — 竹制容器；不宜接触金属 (特别是铁制) 器具。

**11. 价格与假冒：**

霍山黄芽是稀有昂贵茶叶，价格范围很广。核心产区金鸡凼特一级 (~720公尺，面积 ~3亩，年产不足50公斤) — 2000元人民币/斤 (500克) 起，甚至更高。大化坪特一级 — 800–1500元。一级和二级 — 日常饮用的平价类别。

* **如何避免假冒：**
    * 主要问题：市场上大量「霍山黄芽」实际是未经完整闷黄 (闷黄) 工序的绿茶。真正的黄芽有明显黄色 (非鲜绿色) 茶叶和汤色，以及带玉米味的栗香。「绿色」版本 — 更清鲜、尖锐，缺乏「黄茶」柔和度。
    * 外形 — 「雀舌」(雀舌)：笔直、略微展开的条索，茸毛丰富，非卷曲或扁平。
    * 汤色 — 黄绿色 (黄绿)，透明，金黄光泽，非鲜绿色。
    * 从可靠供应商购买，有「国家地理标志」标识和具体生产乡镇标注。
    * 价格过低 — 假冒或未经完整黄变工序的确切标志。

**12. 趣闻轶事：**

* 霍山黄芽是唯一在司马迁《史记》中提及 (古名「寿春黄芽」) 的黄茶。这使其成为有文献记载最长谱系的茶叶之一 — 超过2000年。
* 霍山黄芽被称为「三鲜茶」(三鲜茶)，因其三重鲜爽：香气鲜、滋味鲜、汤色鲜。这既是行销定义，也是感官体验的准确描述。
* 1972年技术复原时，6斤实验茶直接送国务院 — 历史上茶叶在复原第一年即「呈报政府」的罕见案例之一。
* 金鸡传说：金鸡山上长著神奇茶树，由一对金鸡守护。这棵树对大多数人不可见，但每年谷雨前黎明鸡鸣时显现一次，只有一个幸运者能采摘其叶片。一次，一个来迁葬祖先的青年追赶金鸡 — 金鸡落入溪中，从此溪名「落鸡河」(落鸡河)，草坪名「金鸡凼」(金鸡凼)。
* 霍山黄芽是唯一获巴拿马万国博览会金奖 (1915年) 的黄茶。获奖品牌「抱儿钟秀」(抱儿钟秀) 至今存在。
* 明代宫廷贡茶配额中，霍山175袋，总数200袋 — 即87.5%的「六安茶」实际产自霍山。这是历史悖论：茶叶以他人之名闻名 (六安茶, 「六安茶」)，直到霍山析县后公正才部分恢复。
* 「栗香」(板栗香, 板栗香) — 霍山黄芽的招牌香气，在所有黄茶中独树一帜。由金鸡种栽培品种特性和长时间「干燥」黄变两个因素结合形成。
* 整个生产过程避免接触铁器 (全程忌铁器) — 仅使用竹、木、陶瓷。这是少数几款将禁用金属作为现行规则而非博物馆细节保持的茶叶之一。

**13. 与其他黄茶比较：**

* **蒙顶黄芽 (蒙顶黄芽)：** 两者都是芽茶类「黄芽茶」，都是有古老谱系的历史宫廷茶。蒙顶 — 更甜、更蜜香、剑形，「三炒三闷纸包」工艺。霍山 — 更具矿物质感、更有结构、「雀舌」形和「干摊」工艺。蒙顶 — 浪漫主义者，有道士吴理真传说；霍山 — 知识分子，有司马迁引文。
* **平阳黄汤 (平阳黄汤)：** 平阳 — 海洋性、玉米香、杏黄色、卷曲形。霍山 — 山地性、栗香、黄绿色、笔直形。平阳 — 「九烘九闷」72小时；霍山 — 「干摊」1–2天 (有时达10天)。平阳 — 顺滑包覆；霍山 — 有「骨架」和矿物结构。
* **君山银针 (君山银针)：** 两者都是「黄芽茶」，都在「四大」之列。君山 — 油润、丝滑、针形；霍山 — 更干爽、涩感、「舌形」。君山 — 湖泊性，洞庭湖湿润气候；霍山 — 山地性，温差剧烈。
* **霍山黄大茶 (霍山黄大茶)：** 同县「兄长」黄芽。黄大茶 — 大叶黄茶 (一芽四五叶)，焦香、「面包」特性和锅巴香 (锅巴香)。当地茶农谚语：「叶大能包盐，梗长能撑船」(叶大能包盐，梗长能撑船)。黄芽 — 嫩雅、栗香、芽茶；黄大茶 — 粗犷、焦香、民间茶。

**结语：**

霍山黄芽是一款具有其生长山脉特性的茶叶。大别山分隔中国南北，在霍山黄芽茶杯中能听到两面的声音：北方矿物质的直率和南方温柔的甘甜，首口的涩味和回味的蜜香，栗香的严谨和玉米香的温柔。其工艺是黄茶中最「从容」的：茶叶不包裹、不压制、不堆闷 — 只是摊放等待，日复一日，直到它按自己的节奏自然黄变。「三鲜茶」— 香气鲜、滋味鲜、汤色鲜 — 同时也是记忆最长的茶：从司马迁到巴拿马博览会，从西藏帐篷到中国国务院。也许正因如此，《史记》中说：「寿春之山有黄芽焉，久服得仙」。长生不老 — 问题有争议。但霍山黄芽教导的耐心 — 是完全真实的东西。
