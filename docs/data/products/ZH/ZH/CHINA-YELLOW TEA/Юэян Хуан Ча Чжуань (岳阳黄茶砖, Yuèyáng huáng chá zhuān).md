# 岳阳黄茶砖 (岳阳黄茶砖, Yuèyáng huáng chá zhuān)

**岳阳黄茶砖 (岳阳黄茶砖, Yuèyáng huáng chá zhuān)**
![Юэян Хуан Ча Чжуань (岳阳黄茶砖, Yuèyáng huáng chá zhuān)](images/Юэян%20Хуан%20Ча%20Чжуань%20(岳阳黄茶砖,%20Yuèyáng%20Huáng%20Chá%20Zhuān).png)

![Юэян Хуан Ча Чжуань (岳阳黄茶砖, Yuèyáng huáng chá zhuān)-1](images/Юэян%20Хуан%20Ча%20Чжуань%20(岳阳黄茶砖,%20Yuèyáng%20Huáng%20Chá%20Zhuān)-1.png)

岳阳黄茶砖是中国茶叶传统中最不寻常且最具矛盾性的茶类之一。这是一种紧压黄茶，含有「金花」(金花, jīn huā)——有益真菌*Eurotium cristatum*的菌落，此前仅与黑茶(黑茶)相关联。实质上，岳阳黄茶砖是连接两个茶叶世界的「桥梁」：黄茶的温和与清新，以及黑茶特有的深度、长期储存和「陈化」能力。其配方为「金砖玉叶、杏汤醇厚、陈韵绵长」(金砖玉叶、杏汤醇厚、陈韵绵长)。

**1. 分类与产地：**

* **类型：** 紧压黄茶 (紧压黄茶, jǐnyā huángchá)。正式归属于黄茶 (黄茶, huángchá) 类别，但在工艺上包含黑茶特有的元素——主要是参与*Eurotium cristatum*的「发花」(发花, fā huā)。这使其成为两个茶类交界处的独特茶品。
* **类别：** 再加工茶 (再加工茶, zài jiāgōng chá)，以岳阳黄毛茶为基础。是受地理标志保护产品「岳阳黄茶」(2014年注册)的重要子类别。
* **产地：** 中国湖南省 (湖南, Húnán) 岳阳市 (岳阳, Yuèyáng) 君山区 (君山区, Jūnshān Qū)。
* **地理座标：** 东经112°18′–114°09′，北纬28°25′–29°51′(在「岳阳黄茶」生产区域范围内)。

**2. 历史与文化意义：**

* **历史：**

**清代前史：皇室贡茶砖。** 岳阳紧压黄茶的根源可追溯到清代贡茶 (贡茶, gòngchá) 传统。据光绪年间 (光绪, Guāngxù, 1875–1908)《巴陵县志》记载：「君山茶砖，岁贡十八斤」(君山茶砖，岁贡十八斤)。这证明岳阳茶的紧压形式在19世纪已经存在，可能更早：唐代的「灉湖含膏」(灉湖含膏) 就是紧压的「茶膏」。

**1980年代：现代工艺。** 现代岳阳黄茶砖的形式确立于1980年代，当时君山茶厂 (君山茶厂, Jūnshān Cháchǎng) 开发了创新的「梯次增湿发酵法」(梯次增湿发酵法, tīcì zēng shī fājiào fǎ)，能够控制黄茶中的「发花」过程——这项技术此前仅用于黑茶(茯砖茶)。该方法确保了*Eurotium cristatum*在紧压黄茶内部的可重现形成，以及储存过程中可控的风味演变。

**21世纪：获得认可。** 2001年，紧压黄茶被纳入「君山黄金系列」(君山黄金系列)——高端产品线。2014年，随著「岳阳黄茶」地理标志的获得，紧压形式被正式确立为独立子类别。2023年，岳阳紧压黄茶在中国黄茶斗茶大赛 (中国黄茶斗茶大赛) 中获得金奖，成为独立类别。

* **名称：**
    * 「岳阳」(岳阳) — 洞庭湖畔的城市。
    * 「黄茶」(黄茶) — 黄茶。
    * 「砖」(砖) — 砖。指紧压形式。
    * 也使用商业名称：「岳阳茗片」(岳阳茗片 —「岳阳茶片」)、「黄金砖」(黄金砖 —「金砖」)。
* **文化意义：**

岳阳黄茶砖被定位为「可收藏的液态古董」(可收藏的液态古董, kě shōucáng de yètài gǔdǒng) 和「黄茶陈化标杆」(黄茶陈化标杆)。如果散装岳阳黄茶体现清新与春天，那么茶砖就是「透过时间棱镜看到的同样岳阳特色」。两种形式——散茶和紧压茶——的存在创造了岳阳茶人所说的「黄茶品类的完整价值生态链」(黄茶品类的完整价值生态链)，其中清新与陈化相互补充。

**3. 植物学描述与原料：**

* **品种/栽培种：** 使用与整个岳阳黄茶系列相同的栽培种：槠叶齐 (槠叶齐)、碧香早 (碧香早)、黄金茶 (黄金茶) 等。但对于茶砖，偏好大叶种，包括桃源大叶 (桃源大叶)。
* **采摘：** 春季、夏季、秋季。紧压茶允许使用比散茶更晚期和成熟的叶片。
* **采摘标准：** 主要为一芽三至四叶 (一芽三至四叶) — 比散装岳阳茶 (一芽一叶) 粗糙得多。大型成熟叶片积累更多茶多糖 (茶多糖) 和矿物质，这对形成茶汤体感和陈化潜力至关重要。
* **原料半成品：** 黄大茶 (黄大茶 —「黄色大叶茶」) 类别的黄毛茶，经过标准的「双闷」处理。因此，茶砖是已完成黄茶的二次加工产品。

**4. 风土与栽培特点：**

岳阳紧压黄茶的原料来自与散装岳阳黄茶相同的生产区域：洞庭湖周边、君山区、临湘、平江、华容。红壤和黄壤 (pH 4.0–6.0)，有机质含量≥1.5%，硒含量较高 (0.82 mg/kg)。

但对于茶砖，不仅栽培风土重要，**「陈化风土」**也很重要——紧压块内部的微环境。压制时形成致密结构，内部建立自己的微气候：温度~25±3°C，湿度~70±5%。正是这些条件确保了*Eurotium cristatum*的发育和储存过程中物质的缓慢转化——这在散茶中是不可能的过程。

**5. 制作工艺：**

岳阳紧压黄茶的工艺是多阶段过程，结合了经典黄茶处理与从茯砖茶 (茯砖茶) 生产中借鉴(并适应)的压制和「发花」技术。

**阶段I：黄毛茶生产 (黄毛茶)**

标准的「双闷黄」(双闷黄) 处理：萎凋 → 杀青 → 第一次闷黄 → 揉捻 → 第二次闷黄 → 干燥。成品毛茶进入二次加工。

**阶段II：精制 (精制, jīng zhì)**

毛茶经过筛分 (筛分, shāi fēn)、切轧 (切轧, qiē yà)、风选 (风选, fēng xuǎn)、手工挑拣 (拣剔, jiǎn tī) 和拼堆 (拼堆, pīn duī)。目标是达到均匀性、纯净度和预定的批次特征。

**阶段III：「梯次增湿发酵」(梯次增湿发酵)**

君山茶厂1980年代的创新。精制毛茶分阶段增湿，为*Eurotium cristatum*生长创造条件。温度和湿度分阶段控制，升高和降低——因此称为「梯次」(梯次, tīcì)。这种方法与黑茶中使用的「冲击式」增湿根本不同：温和渐进的过程保持黄茶特色 (温和、甘甜、无苦涩)，同时启动「金花」生长。

**阶段IV：紧压定型 (紧压定型, jǐnyā dìng xíng)**

准备好的原料经蒸茶 (蒸茶, zhēng chá) 软化，然后放入模具压制成砖或饼。压制密度是关键参数：太紧——「花」不会生长；太松——砖会散开且无法创造所需的微气候。

**阶段V：发花 (发花, fā huā)**

新压制的茶砖放入温度控制的专门房间 (~28–30°C) 和湿度 (~75–85%) 7–15天。茶砖内部开始生长*Eurotium cristatum*菌落——肉眼可见的金黄色圆点，劈开茶砖时可见。这种真菌分泌多种酶：淀粉酶 (将淀粉转化为糖)、氧化酶 (温和氧化多酚，去除粗涩味)、纤维素酶。其代谢产物形成独特的「菌花香」(菌花香, jūn huā xiāng)，散茶中没有。*Eurotium cristatum*是强势「主导者」：其存在抑制不良微生物生长，确保产品微生物安全。

**阶段VI：烘干 (烘干, hōng gān)**

茶砖在适中温度下干燥至水分含量≤8%，固定形状并将微生物群「封存」在内部。

**6. 感官特征：**

* **外观：** 砖形，表面平整，棱角清晰。表面致密、光滑，有金毫显露 (金毫显露)。劈开茶砖时内部可见金黄色点状物——*Eurotium cristatum*菌落(「金花」)。其丰富程度和亮度是品质指标：特级品「发花」浓密均匀。
* **干茶香气：** 主导「酵花香」(酵花香, jiào huā xiāng) — 复杂香调，由茶砖内*Eurotium cristatum*代谢产生。另外还有焦糖香 (焦糖香, jiāo táng xiāng)，在梯次烘干过程中产生。三年以上样品出现「陈香」(陈香) — 陈化香气，带有干果和珍贵木材香调。
* **滋味：** 配方为「醇厚浓强」(醇厚浓强, chún hòu nóng qiáng —「醇和厚重，浓郁强劲」)。这比散装岳阳茶更「厚重」和浓郁的滋味。「骨架」由多酚形成 (≥30.5% — 显著高于散茶的~24.7%)。「填充」— 多糖的焦糖甜味 (7.2%)。矿物「清新感」— 来自压制时从湖泊沉积物中「挤出」的元素。回甘持久，有明显的甜味。
* **汤色：** 深杏黄 (深杏黄) 至琥珀色 (琥珀色) — 比散装岳阳茶 (杏黄至橙黄) 明显更深更浓。茶汤透明、明亮。
* **叶底(冲泡后茶叶)：** 褐黄色 (褐黄)，均匀，有光泽。叶片肉厚、柔软、有弹性 (比散装岳阳茶明显更大更厚)。

**7. 化学成分：**

由于*Eurotium cristatum*的作用和压制，岳阳黄茶砖的化学特征与散茶有显著差异。

* **多酚：** ≥30.5% 干物质 — 高于散装岳阳黄茶 (~24.7%)。这个矛盾现象的解释是大叶原料 (黄大茶) 本身含有更多多酚，而*Eurotium cristatum*只是部分修饰它们，并不破坏。
* **多糖：** 7.2% — 非常高的指标，显著超过散茶标准。正是多糖形成浓厚、包覆性的茶汤体感并带来焦糖甜味。
* ***Eurotium cristatum*代谢产物：** 淀粉酶、氧化酶、纤维素酶 — 在储存过程中继续缓慢转化茶叶成分的酶。洛伐他汀 (lovastatin) — 天然HMG-CoA还原酶抑制剂，可能参与脂质代谢调节。真菌胞外多糖 — 具有自身生物活性。
* **生物碱：** 咖啡因 — 干物质的2.5–4.5% (含量与散茶相当)。
* **胺基酸：** 含量低于细嫩原料散茶 (大叶积累较少胺基酸)，但足以形成甘甜和温和口感。
* **矿物质：** 硒、锌、钾含量较高 — 反映洞庭湖湖泊土壤的地球化学特征，因压制时的「矿物挤压」效应而增强。

**8. 保健功效：**

* **消化支持：** *Eurotium cristatum*分泌淀粉酶和蛋白酶，有助于食物分解。高多糖含量包覆胃黏膜，减少刺激。
* **脂质代谢调节：** 综合作用：多酚、洛伐他汀 (真菌代谢产物) 和*Eurotium cristatum*胞外多糖 — 脂质调节潜力的三个「支点」。
* **抗氧化作用：** 高多酚含量提供强大的抗氧化保护，可与优质绿茶相媲美。
* **血糖调节：** 茶多糖和多酚共同影响胰岛素敏感性。
* **肠道菌群支持：** *Eurotium cristatum*具有益生元潜力，有助于形成有益的肠道环境。
* **温和提神效果：** 咖啡因与胺基酸结合提供清醒感而无剧烈波动。

**9. 冲泡方法：**

* **水温：** 95–100°C (沸水) — 紧压茶需要比散茶更热的水。
* **茶量：** 5–7克/150毫升。
* **茶具：** 瓷盖碗 (盖碗) 或紫砂壶 (紫砂壶, zǐshā hú) — 厚壁保温，紧压叶展开所需。
* **过程：**
    1. 用茶刀 (茶刀, chá dāo) 或茶针 (茶针, chá zhēn) 从茶砖撬取所需量。尽量不要弄碎叶片。
    2. 沸水温壶。
    3. 投茶。第一泡 — 沸水，8–10秒，倒掉。这是紧压叶的「醒茶」(醒茶, xǐng chá)。
    4. 第二泡 — 同样8–10秒，倒掉。两次洗茶去除压制粉尘并「唤醒」茶叶。
    5. 第三泡 (第一泡正式) — 10–15秒。茶汤深杏黄至琥珀色，有焦糖甜味和菌花香调。
    6. 后续各泡 — 时间递增5–10秒。
    7. 耐泡度 — 8–12泡以上。
    8. **煮饮 (煮饮, zhǔ yǐn)：** 泡完后 (或代替冲泡) 紧压黄茶很适合煮饮。5–7克/500毫升水，煮沸后再煮3–5分钟。煮出的茶汤浓厚、甘甜，有深沉的陈化香气。

**10. 储存：**

与散装黄茶的根本区别：岳阳茶砖**不需要冷藏**且**随时间改善** (越陈越香, yuè chén yuè xiāng —「越陈越香」)。

* **条件：** 干燥、阴凉、避光、通风的环境。温度20–30°C。湿度50–70%。避免直射阳光和异味。
* **包装：** 棉纸或纸板；可无包装储存 (茶砖需要「呼吸」)。**不要使用**密封塑胶袋 —「金花」需要最小的气体交换。
* **储存演变：**
    * **新茶 (1年内)：** 主导菌花香。滋味明亮，略有涩味。
    * **1–3年：** 香气加深，出现干果调。涩味消失，甜味增加。
    * **3–7年：** 成熟「陈香」— 陈化香气。茶汤浓厚、蜜甜，有焦糖「底韵」。口感丝滑。
    * **7年以上：** 药香 (药香, yào xiāng)。滋味极其温和、包覆，有持久温暖回甘。
* **保存期限：** 在适当条件下无限期。紧压岳阳黄茶是少数允许多年 (潜在多十年) 陈化的黄茶之一。

**11. 价格与假冒：**

* **价格区间：** 中高端。紧压岳阳黄茶通常比散装岳阳黄叶贵，但比君山银针便宜。陈年样品 (5年以上) — 高端收藏级别。
* **如何避免假冒：**
    * **「金花」在内部，不在外部。** 真正的*Eurotium cristatum* (金花) 是金黄色圆点，生长在茶砖*内部*，在茶叶表面。劈开时可见。霉菌 (黄曲霉) 是茶砖*表面*的绒毛状「毛发」，容易擦掉和吹掉。
    * **茶汤透明，无浑浊。** 深杏黄至琥珀色是正常的。浑浊、深棕色茶汤是霉菌感染或工艺问题的征象。
    * **香气纯净，无霉味。** 应该是「菌花香」(菌花香) 带焦糖调。酸味、霉味或「鱼腥味」是次品。
    * **茶砖表面平整、紧实。** 散碎、松散的茶砖是压制问题，这样的茶无法创造正确的「发花」微环境。
    * **文件证明。** 真正的岳阳黄茶砖应有标识，确认来自「岳阳黄茶」地理标志保护区。

**12. 有趣事实：**

* **带有黑茶「花」的黄茶。** 岳阳黄茶砖是黑茶类别之外极少数有意使用*Eurotium cristatum*的茶类之一。「金花」是湖南茯砖茶 (茯砖茶) 的「名片」，被认为在黑茶之外无法重现。岳阳师傅证明了*Eurotium cristatum*也可以「嫁接」到黄茶上 — 在严格遵守梯次发酵参数的条件下。

* **「液态古董」。** 紧压岳阳黄茶是世界上少数可以像生普洱或老白茶一样收藏和陈化多年的黄茶之一。在贸易中获得了诗意的别名「液态古董」(液态古董)。

* **复原「灉湖含膏」。** 现代岳阳茶人将紧压黄茶工艺视为「复原」古代唐代「灉湖含膏」(灉湖含膏) — 曾被认为失传的紧压茶。黄毛茶压制后「发花」— 据茶史学家认为，这是对唐茶长期储存过程的现代诠释。

* **两种形式 — 一个「生态系统」。** 岳阳茶人强调，散装和紧压黄茶不是竞争对手，而是彼此的「时间补充」：散茶用于享受清新，茶砖用于时间之旅。它们共同形成黄茶的「完整价值生态系统」。

**13. 与其他茶类比较：**

* **散装岳阳黄茶 (岳阳黄茶 —散茶)：** 相关产品，但使用更嫩的原料 (一芽一至二叶)。香气 —「酵香」和「嫩栗香」(发酵香和嫩栗香)。滋味 — 清淡、清新、「甘醇鲜爽」。汤色 — 浅杏黄。储存 — 冷藏，12–18个月。主要区别：无「金花」且无法长期陈化。

* **安化茯砖茶 (安化茯砖茶, Ānhuà Fúzhuān Chá)：** 因含*Eurotium cristatum*而在工艺上最相近。但这是黑茶：原料是经过渥堆 (渥堆) 的黑毛茶 (3级)。滋味更粗糙、更「厚重」，有明显的「土味」。汤色红褐色。岳阳黄茶砖更温和、更浅、更甜，保持「黄茶」清新感。

* **千两茶 (千两茶, Qiān Liǎng Chá)：** 湖南「千两茶」— 巨型柱状紧压黑茶。也含微生物群，但组成不同 (较少控制的「发花」)。滋味强劲、「烟熏味」。完全不同的风格世界。

* **老白茶 (老白茶, Lǎo Báichá)：** 在「储存陈化」理念上相似，但无*Eurotium cristatum*参与。老白茶的演变纯粹是氧化性的。岳阳黄茶砖是微生物性的。风味特征不同：老白茶「蜜枣味」；岳阳茶砖「焦糖菌香」。

* **生普洱 (生普洱, Shēng Pǔ'ěr)：** 也是随年份改善的紧压茶。但生普洱按分类是云南大叶种「晒青」绿茶，无有意「发花」。生普洱的微生物群在储存过程中自然形成。岳阳茶砖有「受控」的*Eurotium cristatum*。风味世界不同，但「茶随时间成长」的哲学相同。

**结论：**

岳阳黄茶砖是两个茶叶世界边界上大胆而成功的实验。将黄茶的温和甘甜与黑茶的深度和时间前景结合，岳阳师傅创造了无直接类比的产品：今天可以饮用的茶 — 二十年后会发现其中完全不同的饮品。对于喜爱「活茶」理念 — 在储藏室中继续发展的茶 — 的人来说，岳阳金砖将是发现和长久伴侣。
