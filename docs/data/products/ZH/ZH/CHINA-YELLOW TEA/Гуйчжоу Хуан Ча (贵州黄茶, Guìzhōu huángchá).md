# 贵州黄茶 (贵州黄茶, Guìzhōu huángchá)

**贵州黄茶 (贵州黄茶, Guìzhōu huángchá)**

贵州黄茶是对产于中国西南部贵州省 (贵州省) 山区黄茶的总称。主要且最著名的代表是**海马宫茶** (海马宫茶, Hǎimǎ Gōng Chá)，产于大方县 (大方县, Dàfāng Xiàn) 海马宫乡。除此之外，贵州黄茶还包括来自茅洞村 (茅洞, Máodòng) 的「老贡香」(老贡香, Lǎo Gòng Xiāng)——这是一种采用所谓「唐朝古法」(唐朝古法, Tángcháo gǔfǎ) 制作的茶，闷黄时间长达50天。贵州黄茶在中国黄茶中以特别长的闷黄时间、使用木炭加热以及浓郁的「谷物」香气而闻名。

贵州省是世界上最古老的茶叶产区之一：1980年在普安县 (普安县) 发现了约164万年前的茶树种子化石——这是地球上茶属 (*Camellia*) 最古老的古植物学证据。陆羽 (陆羽, Lù Yǔ) 在《茶经》(《茶经》, 八世纪) 中记载：「黔中生思州、播州、费州、夷州……往往得之，其味极佳」(黔中生思州、播州、费州、夷州……往往得之，其味极佳)。

**1. 分类与产地：**

* **类型：** 黄茶 (黄茶, huángchá) — 轻度发酵（5-25% 氧化）。关键工艺步骤是闷黄 (闷黄, mèn huáng)，这决定了茶类。

* **类别：** 贵州省黄茶的区域性群体。在按原料嫩度分类中占中间位置：海马宫茶更接近黄小茶 (黄小茶，「小黄茶」——由芽头和1-2片叶子制成)，而使用较成熟原料的老贡香则接近黄大茶 (黄大茶，「大黄茶」)。一些分类学者将「贵州唐朝古法黄茶」(贵州唐朝古法黄茶) 列为独立的子类别——其工艺如此独特。

* **产地：** 中国贵州省 (贵州省, Guìzhōu Shěng)，中国西南部。主要产区：毕节市 (毕节市, Bìjié Shì) 大方县 (大方县) — 海马宫茶的故乡；同县茅洞村 (茅洞) — 老贡香的故乡。地理上位于贵州高原西北部，乌江 (乌江) 和赤水河 (赤水) 流域交汇处。

* **地理座标：** 约北纬27°07′，东经105°36′（海马宫地区）。

**2. 历史与文化意义：**

* **历史：**

贵州是茶叶文明的「摇篮」之一。早在唐代，陆羽就将来自「黔中」(黔中，贵州的历史名称) 的茶列为天下优质茶之一。宋代，贵州茶参与了「茶马古道」(茶马古道, Chámǎgǔdào) 体系——在边境贸易中以茶换马。明代 (1368-1644)，贵州成为五个「茶叶布政司」(布政司) 之一，每年向朝廷进贡茶叶；贵州的进贡量在全国排名第二。

黄茶作为独立的工艺类别在贵州形成，据不同估计，约在明清时期。清代，贵州黄茶获得了贡茶 (贡茶，「宫廷进贡」) 地位。嘉庆皇帝 (嘉庆，1796-1820年在位) 赐予茅洞村李氏 (李) 家族为朝廷生产黄茶的专营权——他们的后代至今仍以「老贡香」(老贡香，「古老进贡香气」) 品牌延续这项事业。来自大方县同名乡的海马宫茶也是宫廷进贡茶，当地史志有记载。

20世纪，贵州黄茶的生产如同中国大多数黄茶一样，经历了深度衰落。闷黄工艺的复杂性、低盈利性和工匠的流失导致这门手艺几乎完全消失。复兴始于2000年代，随著对稀有茶叶兴趣的兴起；茅洞和海马宫的家庭作坊恢复了传统工艺，吸引了收藏家和专业出口商的关注。

* **名称：** 贵州 (Guìzhōu) — 省名，字面意思是「珍贵的地区」；黄茶 (Huángchá) — 「黄茶」。全称为「来自贵州的黄茶」。

* **文化意义：** 贵州黄茶是中国最鲜为人知和最稀有的黄茶之一。在贵州山区地理隔离的条件下，这里保存了其他省份已失传的古老工艺：长时间闷黄（长达50天）配合木炭加热，多次循环「闷黄 — 干燥 — 再次闷黄」。这使得贵州黄茶不仅是饮品，更是茶叶工艺的活化石。

**3. 植物学描述与原料：**

* **品种：** *Camellia sinensis* var. *sinensis* — 小叶和中叶型。

* **品种/栽培种：** 主要是当地群体种 (群体种, qúntǐ zhǒng)，适应贵州西北部山区条件。海马宫茶使用中叶型灌木，嫩枝上有丰富绒毛——「背有白毛」(背有白毛, bèi yǒu bái máo) 被视为品质标志。也使用小叶福鼎 (小叶福鼎, Xiǎoyè Fúdǐng) 栽培种——来自福建的小叶选育品种，在贵州适应良好。

* **采摘：** 春茶为主。海马宫茶在谷雨 (谷雨，「谷雨」，约4月20日) 期间采摘；老贡香也可秋采。标准为一芽一二叶 (一芽一二叶)。高级品种采用有丰富绒毛的嫩芽。

* **原料要求：** 新鲜、完整、无损伤。在露水干后的上午时分采摘。当地群体种的深根系深入石质土壤，确保原料具有较高的矿物质含量（锌、硒）。

**4. 风土与栽培特点：**

* **产区：** 贵州高原西北部 (黔西北)，毕节地区。茶园位于交通不便的偏远山村，这有助于保存传统工艺，但限制了生产规模。

* **海拔高度：** 海拔1000-1500公尺。海马宫乡约海拔1200公尺；茅洞村约海拔1100-1300公尺。

* **气候：** 亚热带山地季风气候，具有明显的垂直地带性。年平均气温约15°C——比中国大多数茶区凉爽。年平均降雨量约1100-1200毫米。相对湿度约80%。频繁的雾气和云层——每年超过200天。显著的日夜温差（10-15°C）减缓嫩芽生长，有利于胺基酸、糖类和微量元素的积累。

* **土壤：** 贫瘠、石质、酸性（pH 4.5-5.5），以石英岩和页岩为基础。富含锌 (Zn) 和硒 (Se) — 毕节地区属于贵州「硒带」。山坡排水良好。

* **生态：** 人口密度低，工业负荷最小，森林丰富。许多种植园采用有机方法——不使用杀虫剂，使用发酵菜籽饼作为肥料。一些农场拥有国际有机认证（USDA Organic、EU Organic）。

**5. 制作工艺：**

贵州黄茶的工艺是中国所有黄茶中最费工和最耗时的之一。「古法」老贡香的完整周期可长达**50天**，这在其他黄茶中无与伦比（相比之下：君山银针72小时，蒙顶黄芽三个周期6-8小时）。

* **采摘 (采摘, cǎi zhāi)：** 手工采摘嫩原料。

* **杀青 (杀青, shā qīng)：** 在约300°C的热铁锅中炒制。时间短暂（比绿茶短）——以钝化酶活性，但保持足够水分供后续闷黄。时间2-3分钟。

* **揉捻 (揉捻, róu niǎn)：** 轻度手工揉捻以塑形并释放细胞汁液。均匀分布水分是均匀变黄的关键。

* **闷黄 (闷黄, mèn huáng)：** 决定黄茶类别的关键步骤。将略湿的揉捻叶包在厚布或特殊纸中，在控制温度（约40-50°C）和湿度下放置。海马宫茶闷黄**8-12小时**，一到两个周期。老贡香长达**50天**（！），多次循环：闷黄 → 木炭烘干 → 再次闷黄。加热使用当地阔叶树种的燃烧木炭——这赋予茶叶特有的淡淡烟熏味 (焦香, jiāo xiāng)。闷黄过程中发生非酶性多酚氧化：叶绿素部分分解并转化为去镁形式 (脱镁叶绿素)，绿色转为黄色；同时游离胺基酸和挥发性芳香化合物含量增加。

* **烘干 (烘干, hōng gān)：** 在木炭上低温（70-80°C）最终干燥 (文火慢炕, wénhuǒ màn kàng —「文火慢烘」)。海马宫茶这个阶段长达**10-12小时**——按茶叶生产标准极其漫长。海马宫茶的总加工时间超过30小时。

**6. 感官特征：**

* **干茶外观：** 紧结卷曲 (紧结卷曲, jǐnjié juǎnqū) 的茶条，有明显的银色绒毛。颜色从黄绿色到橄榄色，带金色条纹。老贡香颜色更深，带铜栗色调。

* **干茶香气：** 复杂：甜香，带干草、烤玉米、淡淡柑橘皮和兰花香调。炭焙品种带有细腻的烟熏尾韵 (焦糖香，焦糖烟熏调)。

* **汤色：** 透明，从浅金黄到浓郁琥珀色 (杏黄, xìng huáng —「杏黄色」)。经过长时间闷黄的老贡香呈深蜜琥珀色。

* **滋味：** 醇和 (醇和, chúnhé)、滑润 (滑, huá)、油润，几乎无苦涩味。甜香调，类似烤玉米或甜蔬菜。淡淡矿物感——「岩韵」(岩韵, yán yùn)，源于石质土壤。清新酸味，类似梨子。回甘 (回甘, huígān) 悠长、甘甜，带「谷物」余韵。

* **叶底：** 柔软、有弹性的黄绿色叶片，颜色均匀。变黄的均匀性是闷黄品质的主要标志。

**7. 化学成分：**

贵州黄茶的化学特征反映了风土特色（富含微量元素的山地酸性土壤）和长时间闷黄的影响：

* **多酚类：** 中等含量（约15-20%）。闷黄过程中部分儿茶素（特别是EGCG和ECG）经历非酶性氧化，与同样原料制成的绿茶相比，减少了苦涩味。平衡的多酚复合物确保了滋味的醇和。
* **胺基酸：** 含量较高（约3-4%）。L-茶胺酸占主导。凉爽的山地气候配合大的日夜温差和长时间云雾有利于胺基酸积累。酚氨比 (酚氨比, fēn'ān bǐ) 低于绿茶，这解释了滋味的甘甜和「鲜爽」。
* **咖啡因：** 约2.5-3.5%。中等水平——低于大多数绿茶。
* **可溶性糖：** 因长时间闷黄而增加——微生物和非酶性水解增加了单糖含量。
* **矿物质：** 锌 (Zn)、硒 (Se)、铁 (Fe) 含量较高——毕节土壤地球化学特征的结果。硒是天然抗氧化剂；其存在是贵州茶叶的竞争优势之一。
* **色素：** 闷黄过程中叶绿素部分降解，形成脱镁叶绿素 (脱镁叶绿素)；同时类胡萝卜素的可见度增加——因此叶片和茶汤呈黄色。

**8. 健康功效：**

* **对胃肠道温和：** 由于长时间闷黄，贵州黄茶比绿茶对胃部更温和。未转化儿茶素含量降低，减少了对黏膜的刺激作用。传统认为黄茶能「暖脾胃」(暖脾胃, nuǎn pí wèi)。
* **消化支持：** 闷黄过程中产生消化酶，有助于食物分解。黄茶传统上推荐用于腹胀、餐后沉重感、食欲不振。
* **抗氧化保护：** 残余多酚 + 土壤中的硒——双重抗氧化复合物。
* **认知支持：** L-茶胺酸与适量咖啡因结合，提供平静、专注的清醒感。
* **心血管支持：** 黄酮类和多酚有助于正常化血脂和血压。
* **重要提醒：** 这是食品，不是药物。对咖啡因敏感者——晚上不宜饮用。胃炎急性期——需谨慎，尽管温和。每日5-8克。

**9. 冲泡方法：**

* **水温：** 75-85°C。嫩原料不耐沸水——温度超过90°C会出现不应有的苦味。
* **用量：** 功夫茶法4-5克/150毫升；欧式方法3克/200毫升。
* **茶具：** 瓷盖碗 (盖碗) 最理想：不吸收香气，便于观察汤色。玻璃杯——用于视觉享受。不推荐紫砂壶——多孔陶土会吸收细腻香调。

* **冲泡过程：**
	1. 用热水温壶。倒出。
	2. 投茶。
	3. 洗茶——快速冲泡3-5秒（高级品种可省略）。
	4. 第一泡——30-40秒。
	5. 后续每泡增加10-15秒。
	6. 茶叶可冲泡**5-7次**，展现从明亮清新到温和甘甜的演变。

* **特点：** 贵州黄茶是黄茶中最「宽容」的之一：长时间闷黄已经「软化」了多酚复合物，即使冲泡一分钟也不会出现苦味。

**10. 储存：**

* **容器：** 密封、不透光——铝箔袋配拉链、陶瓷罐、铁罐。
* **温度：** 最佳0-5°C（冰箱）密封包装。经过长时间闷黄的老贡香，可在干燥阴凉处室温保存——其特征对氧化不太敏感。
* **天敌：** 光线、湿气、异味、高温。
* **保质期：** 密封冷藏可保存18-24个月。开封后1-2个月。经过深度闷黄的老贡香能够有限陈化——滋味在1-2年内「圆润」，类似轻发酵乌龙茶。

**11. 价格与仿冒：**

贵州黄茶属于高价位茶类，因为制作费工（老贡香需50天）、手工采摘、产量小且稀有。李氏家族正宗老贡香50克起价75美元；顶级批次50克可达200+美元。海马宫茶较便宜，但仍比贵州大宗绿茶贵得多。

* **如何避免仿冒：**
	* 向专营稀有中国茶的可靠供应商购买。
	* 评估外观：正宗贵州黄茶有明显绒毛，颜色均匀的黄绿或橄榄色。
	* 检查香气：特有的「谷物」调（玉米、大米）和淡淡烟熏味。缺乏这些香调或有刺鼻「霉味」是仿冒或储存不当的标志。
	* 评估茶汤：透明、金琥珀色，无浑浊。
	* 警惕用储存不当的普通绿茶冒充，以产生黄色调。

**12. 有趣事实：**

* **贵州化石茶。** 1980年在普安县 (普安县) 发现了约164万年前的茶树种子化石——地球上茶属 (*Camellia*) 最古老的古植物学证据。贵州完全有资格被称为「茶树的摇篮」。

* **李氏家族专营权。** 清朝嘉庆皇帝 (嘉庆) 赐予茅洞村李氏家族为朝廷生产黄茶的专有权。李氏第八、九代后人继续这门手艺，生产「老贡香」——世界上少数拥有超过200年不间断家族传承的茶叶之一。

* **50天闷黄。** 老贡香的闷黄长达50天——这是世界黄茶的绝对记录。相比之下：君山银针闷黄约72小时，蒙顶黄芽6-8小时（三个周期），霍山黄芽1-2天。贵州黄茶是茶界的「慢食」。

* **木炭。** 与大多数其他黄茶在杀青「余热」或热水下进行闷黄不同，贵州工艺使用**燃烧木炭**维持温湿度——这是追溯到前工业时代的古老方法。

* **硒茶。** 毕节地区是贵州「硒带」的一部分。这个地区的茶叶天然富含硒 (Se) ——一种抗氧化微量元素。这使得贵州黄茶成为中国矿物质最丰富的茶叶之一。

* **「隐藏巨人」。** 贵州是中国茶业的「隐藏巨人」：全省茶园面积超过470万亩（约31.3万公顷），包括世界最大的连续茶园「中国茶海」(中国茶海，「中国茶海」) 位于湄潭县 (湄潭县) ——超过4万亩（约2670公顷）的统一茶园景观。

**13. 与其他黄茶的比较：**

* **君山银针 (君山银针, 湖南)：** 「芽茶」黄茶的标准——纯芽头，闷黄约72小时，蜜甜。贵州黄茶——原料较粗，闷黄时间显著更长，具有明显的「谷物」和烟熏特征。
