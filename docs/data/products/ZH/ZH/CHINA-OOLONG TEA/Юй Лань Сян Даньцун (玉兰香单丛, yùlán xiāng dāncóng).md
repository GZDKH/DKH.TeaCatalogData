# 玉兰香单丛 (玉兰香单丛, yùlán xiāng dāncóng)

**玉兰香单丛 (玉兰香单丛, yùlán xiāng dāncóng)**

**玉兰香单丛 (玉兰香单丛, yùlán xiāng dāncóng)** — 凤凰单丛十大经典花蜜香型之一，有别于其他「兄弟茶」的强劲与浓烈，以精致优雅见长。黄栀香以栀子花的辛香取胜，蜜兰香以浓郁蜂蜜包裹，玉兰香则另辟蹊径 — 以轻盈、清凉的玉兰花香从茶杯中缓缓升起，一泡接一泡地持续绽放。这款茶常被称为单丛中的「贵族」：其茶汤不苦不涩，如室内乐般轻柔，与其他高香型品种的管弦乐般轰鸣形成鲜明对比。

**1. 分类与产地：**

* **类型：** 乌龙茶（半发酵茶，乌龙茶, wūlóngchá）。广东乌龙（广东乌龙, Guǎngdōng wūlóng）。发酵程度 — 中等，通常为30–50%。
* **类别：** 凤凰单丛（凤凰单丛, Fènghuáng Dāncóng），十大经典花蜜香型之一（十大花蜜香型, shí dà huā mì xiāng xíng）。地理标志保护产品（2010年 — AQSIQ；2020年 — 纳入欧盟-中国保护名录）。
* **产地：** 中国广东省（广东省, Guǎngdōng shěng）潮州市（潮州市, Cháozhōu shì）潮安区（潮安区, Cháo'ān qū）凤凰镇（凤凰镇, Fènghuáng zhèn），凤凰山（凤凰山, Fènghuáng Shān）山脉。最优质的样品产自乌岽山（乌岽山, Wūdǒng Shān）及周边高山村落。
* **地理座标：** 约北纬23°55′，东经116°38′（乌岽山区域，凤凰山顶峰海拔1,497.8公尺）。

**2. 历史与文化意义：**

* **历史：** 玉兰香是凤凰单丛十大经典香型之一，其分类于1996年由华南农业大学（华南农业大学）戴素贤教授（戴素贤, Dài Sùxián）的研究团队科学确立。为期三年的研究识别出不同单丛中的104种芳香化合物，确定十大香型各具独特的挥发性物质特征。

然而，孕育玉兰香品系的茶树历史更为悠久。母树最初于1961年通过扦插繁殖。后来凤凰镇茶农魏立民（魏立民, Wèi Lìmín）发现该株茶树生长势强、发芽能力高且品质稳定。魏氏进行再次扦插，后代迅速在全村推广，获得民间称号「立民种」（立民种, Lìmín zhǒng）— 「立民的品种」。值得注意的是，玉兰香是少数能在相对较低海拔种植仍能产出高香茶叶的凤凰单丛之一，这决定了它在乌岽精英山带之外的广泛传播。

更广泛的背景：凤凰山茶业已有900多年历史。传说最初的茶树与1278年逃避蒙古追击的南宋皇帝有关 — 凤凰鸟为他带来了茶树枝条。最早的文献记载可追溯至明代：弘治年间（弘治, 1488–1505），待诏山（待诏山）的茶叶成为宫廷贡品。「一株一香」（单株采制, dānzhū cǎizhì）的原则在清代同治-光绪年间（1875–1908）形成，当时茶农开始选育和繁殖优异的单株茶树。

* **名称：** 玉兰（Yùlán）— 「玉兰花」（字面意思为「玉石兰花」）；香（xiāng）— 「香气」；单丛（dāncóng）— 「单独的茶丛」，指个别选育和加工的原则。完整含义：「具有玉兰花香的单丛」。植物学上，「玉兰」在华南和台湾文化语境中通常指白玉兰/白兰花（*Michelia × alba*），而非望春玉兰（*Magnolia denudata*）— 一种具有浓郁甜凉香气的花朵，在东南亚常由街头小贩贩售供佩戴。

* **文化意义：** 玉兰香在十大经典香型中占据特殊地位：如果说黄栀香、蜜兰香和芝兰香代表凤凰派的「强劲」与「浓郁」，那么玉兰香则代表其「优雅」的一面。这款茶深受潮州工夫茶（潮州工夫茶, Cháozhōu gōngfū chá）爱好者推崇，因其能够逐渐展开：前几泡内敛，但每一泡香气都在增强，可持续十多泡。单丛整体被称为「茶中香水」（茶中香水, chá zhōng xiāngshuǐ），而玉兰香是其最精致的音符。

**3. 植物学描述与原料：**

* **品种/栽培种：** 玉兰香是从凤凰水仙（凤凰水仙, Fènghuáng Shuǐxiān）群体中选育的无性系品种，属国家标准品种（*Camellia sinensis* var. *sinensis*）。小乔木型（小乔木, xiǎo qiáomù），中叶种，晚芽种（迟芽种, chí yá zhǒng）。叶片椭圆形，具特征性叶柄弯曲，锯齿边缘稀疏而尖锐。芽活力强，产量良好。最著名的亚品系包括：官目石玉兰香（官目石玉兰香）、金玉兰（金玉兰, Jīn Yùlán — 「金色玉兰」，因嫩叶呈黄绿色而得名）、字茅玉兰（字茅玉兰）、娘仔伞（娘仔伞, Niángzǐ Sǎn — 「女儿伞」，因树冠形状而得名）。

* **采摘：** 玉兰香属晚熟品种。主要采摘期为晚春，从谷雨（谷雨, 4月20日）到立夏（立夏, 5月初）。秋茶（秋茶, qiūchá）也有采制，但品质较低。采摘严格手工进行：等待驻芽（驻芽, zhù yá）出现，然后以「骑马采」（骑马采, qímǎ cǎi）手法采摘2–5叶的嫩梢。有三个严格禁忌：烈日下不采、雨天不采、露水未干不采。

* **采摘标准：** 带驻芽的嫩梢加2–3片成熟叶 — 高级茶的标准。过嫩的嫩梢会产生过多苦味（儿茶素含量高），过老则口感粗糙、纤维质重。

* **原料要求：** 完整嫩梢无机械损伤，成熟度一致。叶片松散放置在茶箩（茶罗, cháluó）中，不压实，立即送至加工厂。玉兰香的特点：该品种即使在中等海拔（400–600公尺）种植也能产出明显花香的茶叶，而大多数单丛需要海拔≥800公尺才能表现出「山韵」。

**4. 风土条件与栽培特点：**

* **地区与地形：** 潮州东北部的凤凰山山脉，最高海拔1,497.8公尺（凤凰顶, 凤凰顶）。地形为陡峭山坡、深谷，被山涧切割。凤凰镇位于山脉东南坡，当地人称之为「潮汕屋脊」（潮汕屋脊）。森林覆盖率85.1%，总绿化率96.4%。

* **种植海拔：** 玉兰香的最佳区域为800–1,000公尺（乌岽的顶级样品）。次级区域为400–600公尺，该品种与许多「娇贵」的单丛不同，在此海拔仍能形成高香茶叶。平原和冲积地不适合种植。

* **气候：** 南亚热带季风气候。具有三个创造独特条件的特点：早冬寒冷、春季凉爽绵长、夏季无极端高温。乌岽山「十日三日晴，百步无平地」（山高高，雾蒙蒙，天无三日晴，地无百步坪）：全年云雾缭绕，降雨充沛，直射阳光短暂。这些条件减缓嫩梢生长，有利于芳香前体物质和胺基酸的积累。

* **土壤：** 海拔>400公尺为黄壤、红壤（黄壤, 红壤），pH 4.5–6.5，土层深厚，有机质含量高（3.8–4.3%），微量元素丰富。矿物营养通过根系运输至叶片，参与芳香化合物的生物合成。茶园无需人工灌溉 — 山地土壤因云雾和森林覆盖保持充足水分。

**5. 制作工艺：**

凤凰单丛与福建和台湾乌龙茶有两个根本区别：条索型（而非球型）揉捻和深度最终焙火加「退火熟化」（退火熟化, tuìhuǒ shúhuà）。玉兰香等晚熟品种从采摘到成品的整个周期约15天。

* **采摘 / 采摘 — cǎizhāi：** 在驻芽出现时手工采摘嫩梢，多在午后进行。

* **晒青 / 晒青 — shàiqīng：** 鲜叶薄摊于阳光下。目的是初步失水、软化细胞壁、启动酶促过程。玉兰香的晒青适中：该品种需要温和处理，以免破坏精致的芳香前体物质。

* **晾青 / 晾青 — liàngqīng：** 将叶片移至通风室内，平衡水分分布，让组织在下一阶段前「休息」。

* **做青（摇青与静置交替）/ 做青 — zuòqīng：** 关键阶段，包括摇青（摇青, yáoqīng）、浪青（浪青, làngqīng）和碰青（碰青, pèngqīng）与静置的节奏性交替。每个周期加强叶缘氧化，释放芳香化合物。对于玉兰香，师傅追求均匀但不过度的氧化，保持玉兰花香的纯净和透明。正是在此阶段形成特征性的「绿蒂绿腹红镶边」（青蒂绿腹红镶边, qīngdì lǜfù hóng xiāngbiān），发生决定性的香气转化。

* **杀青（杀青）/ 杀青 — shāqīng：** 在圆筒形炒茶机中高温处理，停止酶促氧化，固定已形成的香气特征。

* **揉捻 / 揉捻 — róuniǎn：** 条索型揉捻形成广东乌龙茶特有的长、紧、直的条索（条索, tiáosuǒ）— 有别于台湾和闽南乌龙的球形。揉捻适中：过度压力会破坏叶片完整性，影响单丛美观。

* **烘焙 / 烘焙 — hōngbèi：** 多阶段干燥和焙火 — 决定香气深度和稳定性的最终阶段。玉兰香通常比黄栀香或肉桂香焙火更轻：目标是保持透明的花香，而非增强「火韵」深度。

* **退火熟化 / 退火熟化 — tuìhuǒ shúhuà：** 焙火后静置约15天，让「火气」沉淀，香气成熟。只有经过这个过程才能展现玉兰香的真正特色。年底会出现「返春」（返春, fǎnchūn）现象：香气再次增强，滋味更加柔和。

**6. 感官特征：**

* **干茶外形：** 条索紧结（条索紧结, tiáosuǒ jǐnjié），粗壮整齐，深褐色（褐色）带油润光泽，具特征性红色「朱砂红点」（朱砂红点, zhūshā hóngdiǎn）。金玉兰颗粒更大更紧结，呈灰褐色调。

* **干茶香气：** 细腻、清凉、纯净。不会从茶荷中「喊叫」— 需温壶后才展现。可识别的主题：白玉兰花带绿色、略带水感的韵味。

* **茶汤香气：** 主要特征是纯净和「空灵」。天然白玉兰花香（玉兰花香清幽馥郁）：清凉甘甜、轻盈，随每泡递增。香气不压迫，而是飘浮在茶杯上方，渗透口腔，在空杯底留香（杯底香, bēidǐ xiāng）。正确焙火后无丝毫「火味」。经15天退火后出现淡淡蜂蜜韵味。

* **滋味：** 玉兰香在单丛中的特色是精致而不空洞。茶汤不苦不涩（色黄而不苦涩, sè huáng ér bù kǔsè），滋味清净而不寡薄（味清而不寡薄, wèi qīng ér bù guǎbó）。汤质轻盈丝滑，带柔和甘甜和明显回甘（回甘, huígān）。余韵持久，花蜜香型，具特征性「喉韵」（喉韵, hóuyùn）。茶叶可冲泡10多次，香气和滋味保持稳定。

* **汤色：** 淡黄明亮（淡黄明亮, dàn huáng míngliàng），清澈透明。无混浊和红色调（与氧化程度更深的蜜兰香或黄栀香不同）。

* **叶底（冲泡后茶叶）：** 完整大叶带嫩梢。经典特征：「青蒂绿腹红镶边」（青蒂绿腹红镶边）。叶片弹性好，油润有光泽，锯齿边缘有明显红点。

**7. 化学成分：**

* **多酚类：** 凤凰单丛总含量可达30%。玉兰香因晚采和适度氧化，多酚复合物平衡：儿茶素足以构建滋味结构，但无过度涩味。

* **胺基酸：** L-茶胺酸和其他游离胺基酸提供柔和、甘甜和特征性「透明」质感。高山样品（>800公尺）因云雾和散射光条件下生长缓慢，胺基酸含量更高。

* **芳香化合物：** 根据戴素贤团队的GC/MS分析，玉兰香的特征是法呢醇（法呢醇, fǎ ní chún）、吲哚（吲哚, yǐnduǒ）、法呢烯（法呢烯, fǎ ní xī）、芳樟醇及其氧化物（芳樟醇及其氧化物）和香叶醇（香叶醇, xiāngyè chún）的主导地位。正是法呢醇和香叶醇的高含量形成了区别玉兰香与其他单丛的特征性清凉花香。总共识别出40多种芳香成分。

* **生物碱：** 咖啡因 — 中等水平（广东半发酵乌龙茶的典型水平），可可碱和茶碱 — 微量。

* **维生素：** C、B₁、B₂、P（芸香苷）；深度焙火会降低维生素C含量。

* **矿物质：** 钾、镁、锰、锌、氟；凤凰山古老火山土壤（有机质3.8–4.3%）丰富了矿物质组成。

* **特点：** 玉兰香与其他单丛一样，含有相对较高的吲哚 — 低浓度时产生愉悦的茉莉花香，高浓度时则呈现厚重的「香水味」。加工技艺在于将吲哚保持在「花香」范围内。

**8. 保健功效：**

* **温和刺激无焦虑：** 咖啡因和L-茶胺酸的协同作用提供持续的专注力和思维清晰，无「咖啡因峰值」和心跳加速。

* **抗氧化保护：** 多酚复合物（儿茶素、茶黄素）中和自由基，减少氧化应激。单丛的多酚含量是乌龙茶中最高的之一。

* **消化支持：** 半发酵茶对胃黏膜的刺激比绿茶温和。焙火单丛对敏感消化系统特别舒适，有助于餐后脂肪分解。

* **脂质代谢益处：** 儿茶素有助于调节胆固醇水平，降低低密度脂蛋白。

* **心血管系统支持：** 系统性饮用茶多酚与血压正常化相关。

* **口腔健康：** 氟和多酚具抗菌作用，防止蛀牙和牙菌斑形成。

* **抗辐射特性：** 研究表明茶多酚能减少电磁辐射的有害影响。

* **冥想茶饮：** 潮州工夫茶是中国最古老、最仪式化的茶道传统之一。玉兰香的缓慢多泡冲泡和香气逐渐展开是减压、回归当下的修行。

**9. 冲泡方法：**

* **水温：** 95–100°C。沸水充分展现玉兰花香；非常嫩的春茶可用95°C。
* **茶量：** 8公克配100–120毫升（工夫泡）。单丛「宁多勿少」— 浓投茶确保香气饱满。
* **茶具：** 潮州瓷盖碗（盖碗, gàiwǎn）是理想选择：薄瓷不吸香，可精确控制时间。宜兴紫砂壶（紫砂壶）可用，但对玉兰香这类透明花香单丛，盖碗更佳。茶杯小型（30–50毫升），潮州工夫茶传统用三个。
* **流程：**
  1. 用沸水温热盖碗、公道杯和茶杯。
  2. 投茶入热盖碗，盖上摇晃2–3次 — 从盖上闻温热干茶香。
  3. 温润泡（温润泡, wēnrùn pào）：注水3–5秒后倒出。
  4. 第一泡：5–8秒。
  5. 通过公道杯分茶入杯。
  6. 后续冲泡：第2泡5秒，第3–5泡5–8秒，之后每泡增加5秒。可冲10–15泡。
* **注意事项：** 单丛不耐闷泡：稍微「闷」一下就会从精致纯净变成苦涩。宁可早一秒出汤，不可晚一秒。泡间稍开盖碗盖 — 防止茶叶「闷熟」。

**10. 储存：**

* 密封不透光容器（铝箔袋真空封装或密盖铁罐）。干燥阴凉处，避免直射光和异味。
* 焙火单丛储存性好 — 正确条件下可存2–3年。每6–12个月建议低温「复烘」（复烘, fù hōng）稳定水分。
* 清香型轻焙火茶 — 冰箱（0–5°C）密封保存，6个月内饮用。
* 特点：年底优质玉兰香会出现「返春」（返春, fǎnchūn）现象 — 储存几个月后香气自然增强。
* **茶叶天敌：** 湿气、高温、光照、异味。勿与香料、香水、清洁用品同存。

**11. 价格与假冒：**

* **价格范围：** 价格变化很大。乌岽山玉兰香（海拔>800公尺，老树）— 根据具体茶树和师傅，每公斤数千至数万元人民币。中山样品（400–600公尺）— 价格亲民得多，每公斤500–2,000元。价格因素：种植海拔、树龄、季节（春茶比秋茶贵）、师傅声誉、手工程度。

* **避免假冒方法：**
  * 向有透明来源链的商家购买；理想情况是凤凰茶农直供。
  * 评估外观：正宗单丛条索整齐挺直，油润有光泽。细碎、断裂、粉尘多是低级或假冒茶的标志。
  * 评估香气：玉兰香应具纯净清凉的玉兰花香，无「化学香水味」、刺鼻烟味或霉味。
  * 检查茶汤：透明清黄，无混浊。滋味柔和纯净，正确冲泡不苦。假冒和低级「单丛」常第一泡就苦。
  * 注意耐泡性：正宗玉兰香香气可持续10多泡。如果第三四泡香气就消失，品质可疑。

**12. 趣味知识：**

* **「立民种」— 民间育种家：** 玉兰香是少数获得「双重命名」的单丛之一：官方名（按香气）和民间名（按1960年代繁殖推广此品系的农民魏立民命名）。这种双重命名是凤凰传统的典型特征，茶树按香气、主人、树形甚至邻近石头命名。

* **不怕山谷的单丛：** 大多数凤凰单丛只有在800公尺以上才能真正「发声」。玉兰香是罕见的例外：其遗传特性允许即使在中等海拔（400–600公尺）也能形成明显的高香特征，使其成为乌岽精英区外最广泛种植的品系之一。

* **一山104香：** 戴素贤教授（1996）的研究在不同凤凰单丛中识别出104种挥发性芳香化合物。仅玉兰香就发现40多种成分 — 比许多香水配方还多。

* **「返春」：** 返春（返春）现象 — 茶叶香气在制作几个月后自然增强 — 许多单丛都有，但玉兰香表现特别明显。凤凰茶农说：「春茶冬饮」（春茶冬饮）。

* **茶中香水：** 单丛常被称为「茶中香水」（茶中香水）。刘仲华院士（刘仲华）品尝后留下经典评价：「第一杯 — 一见钟情；第二杯 — 难以忘怀；第三杯 — 不离不弃」。玉兰香在凤凰十大「香水音符」中是白玉兰音符：最安静、最持久、最私密。

**13. 与其他单丛比较：**

* **黄栀香单丛（黄枝香单丛, Huángzhī Xiāng Dāncóng）：** 栀子花香 — 强劲、辛香、「张扬」。茶汤更厚重浓郁，汤质更深，涩味明显。玉兰香是其完全对立面：以精致代替强劲，以清凉代替辛香。如果黄栀香是管弦乐，玉兰香就是独奏长笛。

* **蜜兰香单丛（蜜兰香单丛, Mìlán Xiāng Dāncóng）：** 最普及和「亲民」的单丛，浓郁蜂蜜兰花香。茶汤更厚更甜，香气更「温暖」包裹。玉兰香更干净透明；其甘甜不是蜂蜜式，而是「花香式」，带特征性清凉。

* **芝兰香单丛（芝兰香单丛, Zhīlán Xiāng Dāncóng）：** 紫兰花香 — 精致悠长，在「优雅度」上最接近玉兰香。区别在性格：芝兰香更辛香深沉，有明显「老丛韵味」（老丛特韵）；玉兰香更轻盈清凉，更「透明」。

* **桂花香单丛（桂花香单丛, Guìhuā Xiāng Dāncóng）：** 桂花香 — 温暖、辛甜、持久。茶汤更厚，有坚果韵味。玉兰香音调更「冷」，不那么辛香，更偏花香绿调。

* **鸭屎香单丛（鸭屎香单丛, Yāshǐ Xiāng Dāncóng，又名银花香 / 银花香）：** 近年最「时尚」的单丛，浓郁金银花香。更强劲、甘甜、「厚重」，蜂蜜韵明显。玉兰香在风格上是其对立面：鸭屎香以强势取胜，玉兰香以宁静制胜。

**结语：**

玉兰香单丛是为那些已经超越「响亮」单丛初识阶段，在香气中寻求宁静的人而生。其他凤凰乌龙以第一口就震撼，玉兰香则邀请耐心聆听：其玉兰韵不是倾泻而下，而是渐次显现 — 一杯接一杯，一泡接一泡，从最初羞涩的花香暗示到第六七泡充满整个房间的饱满立体香气，直到夜晚仍不离喉韵。

这款茶是单丛世界中力量与精致并非一回事的最佳证明。有时最轻的声音响得最久。
