# 金萱 (金萱, jīn xuān)

**金萱 (金萱, jīn xuān)**

![Цзинь Сюань (金萱, jīn xuān)-1](images/Цзинь%20Сюань%20(金萱,%20Jīn%20Xuān)-1.png)

**金萱 (金萱, jīn xuān)** — 台湾最具辨识度的乌龙茶之一，以其天然奶香 (奶香, nǎi xiāng) 闻名——这种罕见特性源于品种本身的遗传特质，而非人工香料添加。官方选育名称为台茶12号 (台茶12号, Táichá 12 hào)，民间也称为「27仔」(27仔) ——源自实验代号2027。按种植面积计算，金萱在台湾排名第二，仅次于青心乌龙 (青心乌龙, Qīngxīn Wūlóng)，在年轻消费者和女性群体中享有特别的人气。

**1. 分类与产地：**

* **类型：** 乌龙茶（半发酵茶）。多数以轻发酵（20–30%）、低温烘焙方式制作，形成半球型包种 (包种, bāozhǒng) 乌龙茶类别。此品种也用于制作绿茶和红茶，但经典且最普遍的产品仍是乌龙茶。
* **类别：** 台湾乌龙茶 (台湾乌龙茶, Táiwān Wūlóng chá)。在海拔1,000公尺以上种植时——台湾高山乌龙茶 (高山乌龙茶, gāoshān wūlóng chá)。
* **产地：** 台湾。主要产区为南投县 (南投县, Nántóu xiàn) 和嘉义县阿里山乡 (嘉义县阿里山乡, Jiāyì xiàn Ālǐshān xiāng)。高山产区核心——南投竹山镇 (竹山镇, Zhúshān zhèn) 和阿里山茶园，海拔高达1,600公尺。此外，金萱在台湾其他1,600公尺以下茶区广泛种植，自1988年起也在福建省（中国大陆）种植。
* **地理座标：** 阿里山——约北纬23°30′，东经120°43′；竹山——约北纬23°40′，东经120°41′。

**2. 历史与文化意义：**

* **历史：** 金萱是台湾省茶业改良场 (台湾省茶业改良场, Táiwān shěng Cháyè Gǎiliáng Chǎng) 有目的选育工作的成果。杂交育种计划始于1950年代：以硬枝红心 (硬枝红心, Yìng zhī Hóng xīn) 为父本，台农8号 (台农8号, Táinóng 8 hào) 为母本。经过四十多年的选拔和试验，实验代号2027的样本被认定具有发展前景。1981年，台湾农林厅正式命名为台茶12号。

  品种培育者为吴振铎 (吴振铎, Wú Zhènduó, 1918–2000)，台湾茶业改良场首任场长、台湾大学教授，被誉为「战后台茶之父」(战后台茶之父)。他在职业生涯中培育了15个新品种，但台茶12号（金萱）和台茶13号（翠玉, 翠玉, Cuì Yù）成为对台湾茶业最重要的贡献。

  1988年，吴振铎将金萱和翠玉苗木带回故乡——中国大陆福建省。2011年，该品种正式通过福建省品种认证，开始在中国东南部茶区（龙岩、宁德、三明）大规模推广。

* **命名：** 汉字金 (jīn) 意为「黄金」，萱 (xuān) 指「萱草」（萱草属 *Hemerocallis* 植物），在中国文化中象征母爱。据最普遍的说法，吴振铎以祖母的闺名 (闺名, guīmíng) 命名此品种，而台茶13号——翠玉——则以母亲之名命名。因此，这位育种家的两大杰作承载著对家中至亲女性的怀念。

* **文化意义：** 金萱在台湾茶向国内外消费者推广中发挥了关键作用。其温和的奶香特质吸引了新的受众——那些原本对传统乌龙茶不感兴趣的人群。这款茶成为台湾选育学派的象征，证明了有目的的品种培育能够创造出全新的风味特征。近年来，金萱因冷泡 (冷泡, lěng pào) 风潮而获得额外人气——其天然甜味和奶香在冷泡中完美展现。

**3. 植物学描述与原料：**

* **品种/栽培种：** 台茶12号 (台茶12号)，*Camellia sinensis* cv. Taicha 12。无性系 (无性系, wúxìng xì)，灌木型 (灌木型, guànmù xíng)，中叶类 (中叶类, zhōng yè lèi)，中生偏早种。二倍体。植株中等大小，树冠开张型 (开张型, kāizhāng xíng)。分枝密集均匀，枝条比青心乌龙和铁观音更粗壮。叶片中等或略大于中等，接近椭圆形，厚实肉质，淡绿色，有明显光泽。叶面平整，叶缘波浪状，锯齿细小不规则，叶尖钝尖。嫩梢绿色带紫色调，下表面有短密绒毛。一芽二叶 (一芽二叶) 百芽重44–67克。
* **采摘：** 主要商品茶采摘期——春季（4月至5月中旬）和冬季（10月至11月）。夏季和秋季也有采摘，但品质较低。春芽在2月底开始活跃生长；最佳春茶采摘期在4月中旬。采摘期较长——这是该品种的商业优势之一。
* **采摘标准：** 一个顶芽带两到三片展开叶 (一芽二三叶, yī yá èr sān yè)。高级茶要求「一芽二叶」比例不少于95%。
* **原料要求：** 完整、成熟度均匀的芽叶，无机械损伤。叶片应肉质、有弹性，无异味。金萱产量比青心大冇 (青心大冇) 和青心乌龙高20–50%，对茶农具有经济吸引力。

**4. 风土与栽培特点：**

* **地区与地形：** 茶园位于台湾中央山脉的山坡上。主要梯田式茶园集中在两个区域：南投县竹山—鹿谷—杉林溪地区和嘉义县阿里山—梅山地区。地貌为陡峭山坡，有天然森林植被（森林覆盖率达93%），茶园梯田穿插其间。
* **海拔高度：** 高山金萱种植于海拔1,000–1,600公尺。平地金萱 (平地金萱, píngdì Jīn Xuān) 种植于1,000公尺以下，茶汤更浓厚但缺少「高山韵味」(山韵, shān yùn)。按海拔分类：
  * 高山金萱 (高山金萱, gāoshān Jīn Xuān)：≥ 1,200公尺——明显的矿物清凉感，纯净细腻的奶香，占据高端市场主要份额。
  * 平地金萱：< 1,000公尺——茶汤更浓厚扎实，但缺少高山特有的山韵 (山韵, shān yùn)。
* **气候：** 亚热带高山气候。年平均气温低于18°C，雾日超过200天，相对湿度超过80%。显著的日夜温差（达10–15°C）减缓芽叶生长，有利于氨基酸和芳香物质积累，直接影响甜味和香气的复杂性。
* **土壤：** 酸性红黄壤 (红黄壤, hóng huáng rǎng)，pH值4.5–6.5，富含矿物元素。山坡良好的排水性防止积水。据台湾茶业改良场资料，世界最优质茶园集中在北回归线50公里范围内——阿里山地区完全符合这一条件。

**5. 制作工艺：**

金萱主要制作为轻发酵半球型乌龙茶。工艺关键在于保持和展现品种天然的奶香花香，采用「轻手法」策略：温和氧化（20–30%）、低温长时间干燥和手工塑形成半球状。特色工序——包揉 (包揉, bāoróu)，赋予茶叶特有的紧结球状外形。

* **采摘 / 采摘 — cǎizhāi：** 顶芽（一芽2–3叶）在露水蒸发后的上午时段手工或机械采摘。原料立即送至加工厂。
* **日光萎凋 / 日光萎凋 — rìguāng wěidiāo：** 茶叶在散射阳光下摊放25–30分钟。目的是启动初步酵素作用并降低叶片含水量。
* **室内萎凋 / 室内萎凋 — shìnèi wěidiāo：** 在温湿度可控的室内（通常有空调）持续约4小时。茶叶变得柔软，开始形成香气基础。
* **浪青 / 浪青 — làngqīng (摇青 — yáoqīng)：** 三轮轻柔摇青，与静置期交替进行。机械作用破坏叶缘细胞，启动可控氧化。正是在此阶段开始展现特有的奶香花香特征。
* **杀青 / 炒青 — chǎoqīng：** 在约280°C的滚筒中加热，停止酵素作用并固定香气方向。对金萱而言，关键是不过度加热，以保持脆弱的奶香。
* **揉捻 / 揉捻 — róuniǎn：** 初步揉捻破坏细胞结构并形成初步外形。
* **初烘 / 初烘 — chū hōng：** 在80°C下加热，在塑形前部分脱水。
* **包揉 / 包揉 — bāoróu：** 半球型乌龙茶的关键工序。茶叶用布包裹并反复揉压，形成紧结球状。此过程重复多次，中间穿插干燥。正是通过此工序，干茶获得特有的紧实半球状颗粒外观。
* **复烘 / 复烘 — fù hōng：** 在约60°C低温下按「低温慢焙」(低温慢焙, dī wēn màn bèi) 原则进行最终干燥。这种方式封存奶香并将含水量稳定至储存标准（≤ 5%）。

**6. 感官特征：**

* **干茶外观：** 紧实半球状 (半球状, bànqiú zhuàng) 颗粒，圆润紧结，卷曲均匀。色泽墨绿带砂绿 (砂绿, shā lǜ) 色调，表面略有光泽。
* **干茶香气：** 明显奶香——许多人比作奶糖或奶油太妃糖的香气。背景是精致的花香，类似桂花 (桂花, guìhuā)。高山茶样增加了雾区茶特有的清凉感。
* **茶汤香气：** 奶香奶油特征配以明亮花香成分：桂花、淡淡香草暗示。香气持久，从冲泡到冲泡保持良好，逐渐从奶香转为温和甜花香。
* **滋味：** 顺滑丝绸般，有明显天然甜味。茶体中等，无粗涩感。味觉层次：入口奶油柔和，中段清新淡果香，持久回甘 (回甘, huígān) 伴随清凉「喉韵」(喉韵, hóu yùn)。氨基酸含量（≥ 1.2%）提供明显的鲜甜成分，相对较低的多酚含量（12.1%）确保无明显苦涩。
* **汤色：** 蜜绿透金黄 (蜜绿透金黄, mì lǜ tòu jīn huáng)——透亮明澈。氧化程度较高时——偏向绿黄色。
* **叶底（冲泡后茶叶）：** 展开的完整叶片——肉厚柔软，有明亮光泽。品质的经典标志——「绿叶红镶边」(绿叶红镶边, lǜ yè hóng xiāng biān)，证明部分氧化处理得当。

**7. 化学成分：**

* **多酚类：** 春茶含量（干茶，一芽二叶）约12.1%。明显低于大多数传统乌龙茶（15–25%），这解释了金萱的温和性和无明显苦涩味。主要成分——儿茶素：表没食子儿茶素 (EGC)、表没食子儿茶素没食子酸酯 (EGCG)、表儿茶素 (EC)、表儿茶素没食子酸酯 (ECG)。在轻度氧化下，儿茶素大部分保留，提供抗氧化潜力。
* **氨基酸：** 总含量约1.2%，在乌龙茶中属较高水平。L-茶氨酸——主要氨基酸——负责茶汤甜味、鲜味感受和与咖啡因的协同放松效果。高山茶样通常含有更多氨基酸，这得益于雾气和低温条件下的缓慢生长。
* **生物碱：** 咖啡因约占干重2.4%。热水冲泡时，100毫升茶汤咖啡因含量约25–55毫克，将金萱归入中等咖啡因茶类。冷泡可将咖啡因萃取量减少约一半。可可碱和茶碱含量微量。
* **芳香化合物：** 金萱独特的奶香花香源于该品种特有的挥发性物质组合：内酯（内部酯类化合物，形成奶香奶油香调）、双乙酰/丁二酮（奶油香调）、橙花叔醇 (橙花叔醇, chéng huā shū chún — 花香木质香气，桂花典型香气)、芳樟醇氧化物 (芳樟醇氧化物, fāng zhāngnǎo yǎnghuà wù — 甜花香调)。这些化合物由遗传决定，只有在轻度氧化和低温干燥的正确工艺条件下才会显现。
* **总氮：** 约4.9%——与高蛋白质和氨基酸含量相关的指标。
* **维生素：** 维生素C、B₂、E、K——轻发酵乌龙茶的典型组合，由于温和处理，维生素C保存良好。
* **矿物质：** 钾、镁、锰、锌、氟——微量，来源于富含矿物质的红黄色山地土壤。

**8. 保健功效：**

* **抗氧化作用：** 金萱的儿茶素有效中和自由基。据某些资料，茶多酚的抗氧化活性比维生素E高数倍。
* **心血管系统支持：** 儿茶素和类黄酮有助于降低「坏」胆固醇（LDL）水平并维持血管弹性。
* **温和提神效果：** 咖啡因和L-茶氨酸的协同作用提供平衡刺激：清醒而不紧张，改善专注力和认知功能。
* **消化支持：** 多酚和轻度氧化使金萱对胃部温和。儿茶素促进脂肪分解——据个别研究，金萱的脂质分解效率比一般乌龙茶高30%。
* **抗菌作用：** 多酚抑制口腔病原菌生长，降低蛀牙风险并改善口气清新。
* **血糖控制：** 类黄酮和儿茶素有助于减缓葡萄糖吸收——对血糖偏高倾向者有潜在支持作用。
* **敏感胃部的温和选择：** 冷泡将咖啡因和单宁萃取量减少约一半，使茶适合消化系统敏感人群。
* **放松和减压：** 温和奶香和L-茶氨酸有助于降低皮质醇水平，创造平静舒适感。

**9. 冲泡方法：**

* **水温：** 热泡（功夫茶）90–95°C；冷泡用常温饮用水或冰箱冷水。
* **茶量：** 功夫茶7–8克配150–200毫升；杯泡3–5克配250–300毫升；冷泡5克配1,500毫升。
* **茶具：** 瓷盖碗 (盖碗, gàiwǎn) ——展现奶香花香的最佳选择，不会被器壁吸收香气。可用小瓷壶或玻璃壶。紫砂壶（宜兴紫砂）对轻香型乌龙茶较不适宜，因器壁多孔性可能削弱精致香气。
* **冲泡过程（热泡——功夫茶法）：**
  1. 用开水温热盖碗和公道杯，倒掉水。
  2. 投入7–8克茶叶，盖上盖子，闻加热后干茶香气。
  3. 第一泡洗茶（可选）——快速注水立即倒出。茶叶开始舒展。
  4. 第一泡正式冲泡：注入90–95°C水，浸泡40–45秒，出汤。
  5. 后续各泡：每泡增加10秒浸泡时间。
  6. 茶可冲泡6–8泡，高品质者可达10泡。
* **冷泡法 (冷泡法, lěng pào fǎ)：** 5克茶叶放入1,500毫升冷水中，置于冰箱4–5小时。茶汤甜润，苦涩味减弱，奶香增强。可重复冲泡2–3次。成品冷泡茶在冰箱保存，4–5天内饮用完毕。

**10. 储存：**

* **条件：** 密封包装（真空铝箔袋或密封铁罐），阴凉避光处。轻香型乌龙茶对异味、潮湿和光照极其敏感。
* **温度：** 新鲜（未开封）金萱建议冰箱保存于0–5°C——可保持奶香和新鲜度1–2年。真空铝箔包装在常温下可保存2年。
* **开封后：** 开封茶叶最好在72小时内饮用完毕，因芳香物质快速散失。如无法做到——转入密封不透光容器并冰箱保存。
* **茶叶天敌：** 潮湿、高温、异味（特别是冰箱中的食品气味）、直射阳光、氧气。

**11. 价格与仿冒：**

* **价格区间：** 根据产地差异价格范围很大。平地金萱是最平价的台湾乌龙茶之一，作为入门茶广受欢迎。高山阿里山金萱冬茶 (冬茶, dōng chá) 价格显著较高，与其他高山台湾乌龙茶相当。影响价格因素：茶园海拔、采摘季节（冬茶和春茶较贵）、手工比例、茶园声誉。
* **避免仿冒方法：**
  * **识别人工香精茶 (香精茶, xiāngjīng chá)：** 这是金萱市场的主要问题。香精茶容易辨别：第一泡就有刺鼻浓烈奶香，第二泡迅速消失。天然奶香更精致但持久——在多泡中保持。
  * **评估叶底：** 天然金萱冲泡后叶片厚实有弹性，叶缘有红边。香精茶叶片往往薄软，无特有红边。
  * **香气检验：** 天然奶香绝不会有「香水」或「化学」味。如香气像速食店奶昔——必是香精。
  * **价格指标：** 高山金萱不可能与平地绿茶同价。标榜「阿里山高山」却价格可疑偏低——几乎确定是仿冒或香精茶。
  * **选择可靠供应商：** 优选有透明产地链、标明具体茶园和海拔高度的销售商。

**12. 趣闻轶事：**

* 金萱是世界上少数几种「奶香」完全天然、由品种遗传决定的茶叶之一。关键芳香化合物——内酯和双乙酰，正是形成奶油和牛奶香气的物质。
* 「27仔」(27仔) 这个暱称至今仍在台湾茶友中使用，源自试验阶段的代号2027。这个非正式术语成为行家间的暗号。
* 品种创造者吴振铎在整个职业生涯中培育了15个新品种，但正是金萱和翠玉使他名垂青史。值得注意的是，两个品种都以他家中女性长辈命名——祖母和母亲。
* 1990年，吴振铎访问武夷山，在与大陆分离40多年后，凭记忆准确说出收藏园中168个茶树品种的名称和特征——树上没有标牌，只有编号。
* 金萱成为台湾茶在大众饮品领域的主要「大使」之一：瓶装「金萱乌龙」几乎在每家台湾商店都有售，冷泡形式使其成为受欢迎的夏季饮品，与茶饮店奶茶竞争。

**13. 与其他台湾乌龙茶比较：**

* **青心乌龙 (青心乌龙, Qīngxīn Wūlóng)：** 最普遍的台湾品种。特征——纯花香（兰花、栀子花）无奶香。更精致优雅，但缺少金萱的识别「名片」。叶片更薄嫩，需要更谨慎处理。
* **翠玉/台茶13 (翠玉, Cuì Yù)：** 金萱的「亲兄弟」——同为吴振铎创作。特征——明显茉莉栀子花香 (玉兰花香, yùlán huā xiāng)，无奶香。茶汤更轻盈透明，涩味稍明显。
* **四季春 (四季春, Sìjì Chūn)：** 「四季春」——高产品种，有栀子花香。结构比金萱简单，层次较少，但因价格亲民和全年产茶能力而受欢迎。
* **冻顶乌龙 (冻顶乌龙, Dòngdǐng Wūlóng)：** 来自鹿谷 (鹿谷) 的经典台湾乌龙，通常采用更深度氧化和烘焙。特征——焦糖坚果香，有木质调。与金萱清新奶香特质截然不同。
* **阿里山高山乌龙 (阿里山高山乌龙, Ālǐshān Gāoshān Wūlóng)：** 常用青心乌龙在与金萱相同海拔制作。特征——花香奶油香，但奶香更精致淡雅，无金萱特有的「焦糖感」。直接比较时，阿里山金萱通常被认为更「温暖」和「包覆感」强。

**结语：**

金萱是一款打破成见的茶。在推崇百年传统和野生茶树的乌龙茶世界中，这个在实验室培育、以科学家祖母命名的品种，凭借纯粹的香气力量征服了数百万人的心。其奶香柔和并非行销手段，而是遗传、风土和精确工艺的结果。正是这种不添加任何香精的天然甜美，使金萱成为整整一代茶爱好者的「第一款乌龙茶」——同时也是经验丰富的品茶者在想要品尝纯粹美好时会回归的茶款。

对于初识台湾茶传统的人来说，金萱是理想的入门选择：易懂、迷人、宽容冲泡失误。而对于资深茶友，高山阿里山金萱冬茶能够以隐藏在看似简单奶香背后的深度和复杂性带来惊喜。
