# 四季春 (四季春, sìjìchūn)

**四季春 (四季春, sìjìchūn)**
![Сы Цзи Чунь (四季春, sìjìchūn)](images/Сы%20Цзи%20Чунь%20(四季春,%20Sìjìchūn).png)

![Сы Цзи Чунь (四季春, sìjìchūn)-1](images/Сы%20Цзи%20Чунь%20(四季春,%20Sìjìchūn)-1.png)

四季春是最具生产力且最易取得的台湾乌龙茶之一，因其不挑剔的品种特性而广泛传播，该品种能够每年提供六至八次采收。此茶以明亮的花香特征著称，以栀子花香为主导，口感温和清新，使其成为台湾茶饮产业的基础原料，也是传统茶艺中受欢迎的日常乌龙茶。

**1. 分类与产地：**

* **类型：** 乌龙茶（半发酵茶）。氧化程度为15–30%，多数为轻发酵（清香型, qīngxiāng xíng — 约20%）或中度发酵（浓香型, nóngxiāng xíng — 约30%）。烘焙通常极轻；制作风格著重保持清新花香。
* **类别：** 台湾中低海拔乌龙茶。不属于高山茶（高山茶, gāoshān chá）；主要种植园位于海拔1000公尺以下。商业上常定位为「松柏长青茶」（松柏长青茶, Sōngbǎi Chángqīng Chá）— 「松柏常青茶」。
* **产地：** 台湾（台湾, Táiwān），南投县（南投县, Nántóu Xiàn），名间乡（名间乡, Míngjiān Xiāng）— 主要生产中心。也种植于嘉义县（嘉义县, Jiāyì Xiàn）和花莲县（花莲县, Huālián Xiàn）。该品种最初在木栅（木栅, Mùzhà，现今台北）地区被发现。
* **地理座标：** 约北纬23°50'，东经120°40'（名间茶区核心，八卦山脉南端的柏岭台地）。

**2. 历史与文化意义：**

* **历史：** 四季春是相对年轻的品种，于1980年代初期被发现。来自木栅地区的茶农张文辉（张文辉, Zhāng Wénhuī）在其茶园中注意到一株茶树，该树是自然杂交（自然有性繁殖, zìrán yǒuxìng fánzhí）的结果，展现出异常旺盛的生长力和全年发芽特性。最初这株植物被称为「辉仔茶」（辉仔茶, Huī Zǎi Chá — 「辉的茶」），以发现者命名，也称为「六季香」（六季香, Liù Jì Xiāng — 「六季之香」）。当苗木被引入南投县名间乡时，当地农民高度评价这个新品种的高产量，并给它命名为「四季春」— 「四季之春」。1988年，种植材料被引入福建省（中国大陆）。2000年后，四季春实现了「产业突破」：由于适合机械采收和高产量，它成为台湾蓬勃发展的茶饮产业（手摇茶饮, shǒuyáo cháyǐn）的主要茶叶原料，据估计占岛上中低海拔茶区面积的30%。

* **名称：**
    * 「四季」（四季）— 「四季」，指品种全年生长和采收的能力。
    * 「春」（春）— 「春天」，强调茶叶无论何时采收都保持新鲜春天般的香气特性。
    * 完整名称「四季之春」— 永恒新鲜的诗意隐喻：每次采收都带来令人想起最佳春茶的香气。

* **文化意义：** 四季春在台湾茶文化中占据独特地位。一方面，它是茶产业的「工作马」— 大众化、民主化的产品，是奶茶鸡尾酒和香味乌龙茶的基础。另一方面，来自精品农场的优质春茶和冬茶受到行家珍视，因其纯净的花香特征和和谐的口感。正是四季春成为数百万人进入台湾乌龙茶世界的「引路人」，他们首次以冷泡茶饮的形式品尝到它。此茶与松柏（松柏, Sōngbǎi）茶区和名间乡深度相关— 台湾最大的茶叶批发中心，茶店沿街延伸数百公尺。

**3. 植物学描述与原料：**

* **品种/栽培种：** 四季春（四季春, sìjìchūn）— 台湾地方品种（地方性品种, dìfāngxìng pǐnzhǒng），通过自然杂交产生。**并非**台湾茶业改良场（TTES）正式注册的「台茶」（台茶, Táichá）系列品种。属于*Camellia sinensis* var. *sinensis*。植物学特征：灌木型（灌木型, guànmù xíng），小叶种（小叶种, xiǎoyè zhǒng），早生种（早生种, zǎoshēng zhǒng）。叶形为纺锤形（纺锤形, fǎngchuí xíng），两端尖锐。叶片颜色为浅绿色带黄色调。叶缘覆盖细小尖锯齿。幼芽呈淡红紫色。植物具有开张树型（树型披张, shùxíng pīzhāng），芽叶密集分布，生长势强，对病害和寒冷抗性高。几乎没有休眠期，确保全年多次采收。

* **采收：** 品种的主要优势是每年可采收6–8次（某些资料显示4–5次完整采收加上额外的夏季采收）。最珍贵的是春茶（春茶, chūnchá，3–4月）：一芽一叶，栀子花香明显，口感清新纯净。冬茶（冬茶, dōngchá，11–12月）因茶汤浓度、甘蔗糖甜味和特有的「冷香」而受到重视。夏秋采收主要用于商业拼配和茶饮。

* **采收标准：** 主要为一芽二三叶（一芽二三叶, yī yá èr-sān yè）。精品批次为一芽二叶，比例不少于95%。大量生产采用机械采收。

* **原料要求：** 与高山乌龙茶相比，对原料的要求较不严格，这是该品种商业吸引力的一部分。允许使用较成熟的叶片，但最佳批次从成熟度均匀的嫩芽中挑选。

**4. 风土与栽培特点：**

* **地区与地形：** 生产核心为名间乡的柏岭台地（柏岭台地, Bǎilǐng Táidì），位于八卦山脉（八卦山脉, Bāguà Shānmài）南端。这是丘陵地带，坡度平缓，有广阔的梯田种植园。额外的种植区包括松柏坑（松柏坑, Sōngbǎi Kēng）和赤水（赤水, Chìshuǐ）地区，那里保存著百年茶园。也在嘉义县和花莲县种植。

* **海拔高度：** 海拔200–500公尺— 典型的低中海拔带。一些农场在海拔800公尺处种植四季春，但大部分产品来自海拔500公尺以下的种植园。

* **气候：** 亚热带，温暖潮湿。年平均温度18–22°C。年降雨量超过2000毫米。空气湿度80%以上。雾日超过200天。名间开阔台地的充足阳光确保茶树旺盛生长和频繁采收，但与高山阴凉地区相比，氨基酸积累较少。

* **土壤：** 红壤和红黄壤（红黄壤, hóng huáng rǎng），八卦山台地特有。酸度pH 4.5–6.5。土壤富含铁和镁，排水良好，有利于茶树深根。茶园周围森林覆盖率约85%。

**5. 制作工艺：**

四季春工艺遵循经典台湾乌龙茶制法，重点在于轻发酵和「锁住」花香。关键特点是「低温慢焙」（低温慢焙, dīwēn mànbèi），保持精致花香。制作过程中在对氧化敏感的阶段不使用金属器具。整个过程在技术上比高山乌龙茶更简单快速，这是由于大量生产的特性。

* **采摘 / 采摘 — cǎizhāi：** 嫩芽手工采摘（精品批次）或机械采摘（大量生产）。采摘的原料立即送到加工厂。

* **日光萎凋 / 日光萎凋 — rìguāng wěidiāo：** 叶片在阳光下摊放约30分钟，进行初步失水和活化酶促过程。

* **室内萎凋 / 室内萎凋 — shìnèi wěidiāo：** 在通风室内持续约4小时。叶片变得柔软，形成香气基础。

* **摇青 / 浪青 — làngqīng (摇青 — yáoqīng)：** 三轮在竹盘上轻柔摇动，启动叶缘部分氧化，形成特有的「绿叶红镶边」（绿叶红镶边, lǜyè hóng xiāngbiān）图案。四季春的摇青强度通常低于冻顶或单丛。

* **杀青 / 炒青 — chǎoqīng (杀青 — shāqīng)：** 约280°C炒制停止酶促过程并固定香气方向。

* **揉捻 / 揉捻 — róuniǎn：** 揉捻叶片以形成外观并提高浸出性。

* **初烘 / 初烘 — chūhōng：** 80°C干燥以稳定。

* **包揉塑形 / 包揉塑形 — bāoróu sùxíng：** 手工布包后压制，赋予叶片经典半球状（半球状, bànqiú zhuàng）形状，这是台湾乌龙茶的典型特征。

* **复烘 / 复烘 — fùhōng：** 60°C烘干至适合储存的稳定湿度。采用「低温慢焙」模式以最大程度保持花香。

* **分级 / 分级 — fēnjí：** 成品茶按颗粒大小、均匀度和品质分级。

**按加工类型分为两种风格：**
* **清香型（清香型, qīngxiāng xíng）** — 轻发酵（~20%），重点在清新花香。
* **浓香型（浓香型, nóngxiāng xíng）** — 中度发酵（~30%），更浓郁的蜜香口感，甜味明显。

**6. 感官特征：**

* **干茶外观：** 紧密卷曲的半球状颗粒（半球状, bànqiú zhuàng），紧实紧凑。颜色为墨绿油润（墨绿油润, mòlǜ yóurùn），带有该品种特有的淡黄色调。颗粒大小均匀。

* **干茶香气：** 浓郁纯净的花香，明显的栀子花香（栀子花香, zhīzihuā xiāng）主导— 四季春的名片。次要香调：玉兰香（玉兰香, yùlán xiāng）、野姜花香（野姜花香, yě jiānghuā xiāng）、草本果香细节。香气活跃，「迎面而来」— 在冲泡前就能在远处感受到。

* **茶汤香气：** 浓郁花香谱系，甜味渐增。热茶汤中— 饱满油润的花果香调。冷茶汤中— 更纯净透明的花香调，重点在栀子花。中度发酵版本出现蜜香和焦糖调。

* **口感：** 鲜爽（鲜爽, xiānshuǎng），甘滑（甘滑, gānhuá）明显，回甘（回甘, huígān — 回甜）持久，喉韵（喉韵, hóuyùn）清凉。茶汤体感从轻到中等，比高山乌龙茶油润性和复杂性低，但花香成分更明亮「响亮」。氨基酸含量≥ 4.2%确保明显的鲜味和鲜味。过度浸泡可能出现轻微苦味，这是由于儿茶素含量相对较高，茶氨酸水平比青心乌龙低。

* **茶汤颜色：** 轻发酵时— 蜜绿透金黄（蜜绿透金黄, mìlǜ tòu jīnhuáng），透明纯净。中度发酵时— 橙黄明亮（橙黄明亮, chénghuáng míngliàng）。

* **叶底（冲泡后叶片）：** 完整有弹性的肥厚叶片，具有特有的「绿叶红镶边」图案— 绿橄榄色中心和红褐色氧化边缘。叶片肥厚软亮（肥厚软亮, féihòu ruǎnliàng），经过几次冲泡后充分展开。

**7. 化学成分：**

* **多酚类（茶多酚, chá duōfēn）：** 总含量为干重的15–25%。儿茶素总含量约101–121毫克/克（根据台湾茶业改良场数据），与金萱（台茶12号）相当，略低于青心乌龙（约124毫克/克）。主要儿茶素：表没食子儿茶素没食子酸酯（EGCG）、表儿茶素没食子酸酯（ECG）、表儿茶素（EC）。四季春多酚类显示高抗氧化活性。

* **氨基酸：** 游离氨基酸总含量≥ 4.2%（春茶）。主要成分— L-茶氨酸（L-茶氨酸, L-chá ānjīsuān），提供口感中的鲜味和鲜味感。四季春的茶氨酸含量明显低于青心乌龙（青心乌龙），这是由于低海拔种植园阳光充足，导致光合作用过程中茶氨酸加速分解为儿茶素。

* **生物碱：** 咖啡碱（咖啡碱, kāfēi jiǎn）— 干重的2–4%。冲泡茶汤中约25–55毫克/100毫升（取决于浓度和冲泡方法），约为等量咖啡中咖啡因含量的四分之一。还含有微量可可碱和茶碱。

* **维生素：** 维生素C（抗坏血酸）、B群维生素（B₁、B₂、叶酸）。轻发酵批次维生素C含量较高。

* **矿物质：** 钾、镁、锰、氟、铁、锌— 生理意义的微量含量。名间土壤中铁和镁含量高反映在茶叶的矿物质特征中。

* **精油（挥发性芳香化合物）：** 决定四季春特有的栀子花香。挥发性成分的主要组成：芳樟醇、香叶醇、橙花醇、苯乙酸酯、水杨酸甲酯。正是挥发性物质的独特平衡使四季春即使在盲品中也容易识别。

* **特点：** 四季春化学特征的特点是儿茶素含量相对较高，氨基酸水平中等。这提供更「响亮」、清爽的口感，但长时间浸泡可能出现苦涩味。

**8. 健康功效：**

* **抗氧化保护：** 茶多酚含量高确保中和自由基。一些台湾资料显示，四季春多酚类的抗氧化效力比维生素E高数倍。

* **提神效果：** 咖啡因和L-茶氨酸的结合提供温和但持久的提神作用— 提高注意力和工作能力，没有急剧的「咖啡因峰值」和随后的下降。

* **代谢支持：** 多酚类促进脂肪分解和加速代谢过程。四季春在台湾传统上被认为是适合日常饮用的「轻」茶，作为均衡饮食的一部分。

* **消化帮助：** 有机酸和多酚类刺激消化酶分泌，帮助消化食物。建议餐后饮用；空腹饮用浓茶可能刺激胃黏膜。

* **心血管支持：** 抗氧化成分有助于降低氧化胆固醇水平并维持血管弹性。

* **利尿作用：** 适量咖啡因有助于排出多余水分和减少水肿。

* **免疫增强：** 茶多酚具有抗菌活性并支持机体免疫功能。

* **心理舒适：** L-茶氨酸促进放松而不困倦，降低压力荷尔蒙水平。四季春冷泡— 获得温和舒缓饮品的流行方法，单宁提取最少。

**9. 冲泡方法：**

* **水温：** 热泡90–95°C。冷泡（冷萃）— 冷水（~5°C），在冰箱中浸泡4–5小时。

* **茶量：** 功夫茶法：8克/200毫升（茶水比1:25）。日常方法：3–5克/250–300毫升。冷泡：5克/1000毫升。

* **茶具：** 盖碗（盖碗, gàiwǎn）瓷器— 展现花香特征的最佳选择。宜兴紫砂壶（宜兴紫砂壶, Yíxīng zǐshā hú）也适合，虽然对于轻乌龙茶瓷器更佳，因为不吸收香气。玻璃茶具— 观察叶片展开的好选择。

* **过程（功夫茶法）：**
    1. 用沸水温热盖碗和茶杯。
    2. 在200毫升盖碗中放入8克茶。
    3. 用两次快速冲洗（各5秒）洗茶— 这「唤醒」叶片。
    4. 第一泡：用95°C水冲泡，浸泡45秒，倒出。
    5. 通过茶滤或公道杯（公道杯, gōngdào bēi）将茶汤分到茶杯中。
    6. 后续冲泡：每次增加10秒浸泡时间。
    7. 优质四季春可冲泡5–7次。

* **冷泡：** 5克茶叶加1000毫升冷水，放入冰箱4–5小时。此方法减少单宁和咖啡因提取，产生特别纯净、甜美清爽的饮品，花香明亮— 在炎热天气中展现四季春特性的最佳方法之一。

**10. 储存：**

四季春作为轻乌龙茶，对外界影响敏感，需要小心储存：

* **容器：** 密封、避光包装— 多层铝箔真空袋或密封盖铁罐。
* **温度：** 长期储存建议冷藏（5–10°C），特别是清香型批次。烘焙版本（浓香型）要求较低，可室温储存。
* **茶叶天敌：** 湿气、热量、异味和直射光— 香气和口感劣化的主要因素。
* **期限：** 新鲜四季春最好在6–12个月内饮用。台湾资料建议新茶在阴暗处放置约15天以「褪火」（褪火, tuìhuǒ）最终干燥后的火气，开封后72小时内使用以保持最大香气。
* **不适合陈放：** 与深色乌龙茶或普洱茶不同，四季春不会随年份改善。其价值在于新鲜度。

**11. 价格与仿品：**

* **价格类别：** 四季春属于最易取得的台湾乌龙茶。价格范围明显低于高山茶（阿里山、梨山、大禹岭）。价格取决于采收季节（春冬茶较贵，夏秋茶较便宜）、采收方法（手工较贵，机械较便宜）、具体农场和年份。台湾特级品质批次参考价格— 从600元（≈ 2700卢布）/斤（500克）起；大宗批次价格明显更低。

* **如何避免仿品：**
    * **向专业销售商购买**，提供透明的产地和农场资讯。
    * **评估干茶香气：** 正宗四季春具有纯净明亮的栀子花香— 无化学香精味和异味。缺乏明显花香或「人工」感— 警告信号。
    * **检查外观：** 颗粒应均匀卷曲、紧实，无过多茎梗和茶末。
    * **测试茶汤：** 颜色纯净透明，无混浊。口感清新滑顺，有回甘，无酸败或陈味。
    * **对异常低价保持警惕：** 过分便宜的「四季春」可能由低质原料制成，或为中国大陆产品（该品种自1988年也在大陆种植）冒充台湾茶。

**12. 有趣事实：**

* **茶名中的农民姓名。** 在木栅地区和台湾老一代茶师中，此茶有时仍被称为「辉仔茶」（辉仔茶）— 「辉的茶」，以发现者张文辉命名。这是少数在非正式名称中保留具体人名的台湾茶之一。

* **冷泡之王。** 四季春被认为是冷萃法的最佳茶之一：冷浸泡时其花香展现特别纯净明亮，苦涩味几乎不出现。正是以冷饮形式首次征服了台湾大众消费者。

* **珍珠奶茶产业引擎。** 2000年代台湾手摇茶饮（手摇茶饮）产业的爆炸性增长很大程度上依赖四季春作为基础原料。高产量、低价格和明亮香气「穿透」牛奶和糖的特性，使其成为奶茶和果茶饮品的理想基础。

* **不要与金萱混淆。** 某些资料错误地将四季春与「台茶」12号（台茶12号），即金萱（金萱, Jīnxuān）等同。这是完全不同的品种：金萱是TTES正式注册的选育品种，具有明显的奶香，而四季春是自然起源的地方品种，以栀子花香为主。

* **「四季」非夸张。** 在名间温和亚热带气候条件下，该品种确实不进入休眠状态，继续全年发芽，提供6–8次采收— 这是世界茶树品种中的记录之一。

**13. 与其他台湾乌龙茶比较：**

* **阿里山乌龙（阿里山乌龙, Ālǐshān Wūlóng）：** 高山乌龙茶（1000–1500公尺），主要由青心乌龙品种制成。更复杂、油润深厚的口感，带奶香和果香调。茶汤更浓厚「厚重」。四季春更轻盈、明亮、花香，价格明显更易接受。

* **冻顶乌龙（冻顶乌龙, Dòngdǐng Wūlóng）：** 中海拔乌龙茶（500–800公尺），中重度烘焙。口感特征— 坚果焦糖味，蜜甜和明显回甘。冻顶在特性上明显更深厚「温暖」，而四季春清新「凉爽」。

* **金萱（金萱, Jīnxuān，台茶12号）：** 台湾选育品种，以天然奶香著称。口感温和奶香，无明显花香。四季春和金萱常在名间同一种植园种植，但产生完全不同的香气特征：四季春的栀子花和野花对比金萱的奶香温和。

* **文山包种（文山包种, Wénshān Bāozhǒng）：** 轻氧化台湾乌龙茶，纵向卷曲（非半球形）。包种更优雅精致，具有铃兰和水仙的细腻香气。四季春在花香上更「响亮」直接，同时更易理解。

* **翠玉（翠玉, Cuìyù，台茶13号）：** 选育品种，具有特有的茉莉玉兰香。特征比四季春更精致「温柔」。两种茶都属于中低海拔带，常在同一价格段竞争，但香气不同：翠玉的茉莉对比四季春的栀子花。

**14. 四季春品种与等级：**

**按采收季节：**
* **春茶（春茶, chūnchá，3–4月）：** 一芽一叶，栀子花香明显，口感清新明亮。被认为是最佳季节。
* **冬茶（冬茶, dōngchá，11–12月）：** 叶片较厚，多糖含量高，「冷香」和甘蔗糖甜味。第二珍贵季节。
* **夏秋采收：** 主要用于商业批次和茶饮。口感较简单，涩味较高。

**按等级：**
* **特级（特级, tèjí）：** 一芽二叶比例≥ 95%。颗粒紧实，颜色墨绿带沙色调。栀子花香强烈持久穿透。价格从600元/斤起。
* **一级（一级, yī jí）：** 以一芽二叶为主。香气纯净，茶汤蜜黄透明。
* **二级（二级, èr jí）：** 混合采收，包括夏秋叶。口感纯净但较不复杂，耐泡性较低。

**结论：**

四季春是一款矛盾的茶：作为最简单易得的台湾乌龙茶之一，它同时具有惊人的表现力和令人难忘的特性。其栀子花香— 明亮、欢快、几乎大胆— 无法与任何其他茶混淆，正是这种「花香直率」为它赢得了数百万人的喜爱，从台湾街头茶摊到欧洲茶叶精品店。对于刚开始接触台湾乌龙茶的人来说，四季春是理想的入门点：冲泡简单，容错性高，冷泡效果极佳，同时诚实地展示了什么是台湾乌龙茶。对于经验丰富的茶爱好者，来自名间的优质春茶四季春提醒我们，茶的特性不仅由种植园海拔和价格决定，还由品种、风土和工艺的成功结合决定— 那种神奇的巧合，台湾人诗意地称之为「永恒之春」。
