# 单丛蜜兰香 (单丛蜜兰香, dān cóng mì lán xiāng)

**单丛蜜兰香 (单丛蜜兰香, dān cóng mì lán xiāng)**

**1. 分类与产地：**

* **类型：** 乌龙茶（通常为中度发酵，30-60%）。焙火程度可从轻焙到重焙不等。
* **类别：** 高品质乌龙茶。属於单丛茶类 (单丛, Dān Cóng)，意指「单株」或「来自单一茶树」。
* **产地：** 中国广东省 (广东, Guǎngdōng) 潮州市 (潮州, Cháozhōu) 凤凰山 (凤凰山, Fènghuáng Shān)，亦称凤凰山脉。
* **地理座标：** 约北纬23-24度，东经116-117度。

**2. 历史与文化意义：**

* **历史：** 凤凰山的茶叶种植文化已有900多年历史。单丛茶作为独立茶类，约在100-150年前开始被区分出来。
* **名称：**

	* 「单丛」(单丛) - 「单株」或「来自单一茶树」。此名称强调单丛茶的独特之处：理想情况下，茶叶从每株茶树单独采摘和加工，以保持每株植物的个别特性。在现代实践中，这并非总是严格遵循，但「单丛」仍意味著茶叶来自同一小型种植园内特定品种/变种的茶树。
	* 「蜜兰香」(蜜兰香) - 「蜜兰花香」。指这种单丛茶的特征香气，以兰花和蜂蜜香调为主。
* **文化意义：** 单丛茶，特别是蜜兰香，因其丰富多层次的香气、浓郁的滋味和耐泡性而备受推崇。它们被视为资深茶叶爱好者的茶品，是潮州地区的骄傲。

**3. 植物学描述与原料：**

* **品种：** 蜜兰香不仅是茶名，更是生长在凤凰山的**品种变异**茶树名称。单丛茶在严格植物学意义上并非品种，而是在自然选择和独特生长条件下形成的**地方变种**。每种单丛都有其特点：
	* **叶形与大小：** 蜜兰香的叶片通常较大，呈椭圆形。
	* **叶色：** 深绿色，边缘带红褐色调（表明中度发酵）。
	* **香气：** 蜜兰香品种以极强烈的特征香气著称，以兰花、蜂蜜、果香为主调。
* **采摘：** 采摘通常在春季进行，但也可在夏季和秋季采摘。春茶被认为最珍贵。
* **采摘标准：** 采摘一芽二至三叶，有时也采四叶。
* **原料要求：** 要求很高。只使用健康、无损伤的叶片。

**4. 风土条件与栽培特色：**

* **凤凰山（凤凰山脉）：** 位于广东省东北部的山脉。山体主要由花岗岩和火山岩构成。以秀丽景色、清新空气和丰富云雾著称。
* **海拔高度：** 茶园分布在海拔400至1500公尺之间。高山茶（1000公尺以上）特别珍贵。
* **土壤：** 红壤和黄壤，富含矿物质，排水良好。山地土壤，常有石砾。
* **气候：** 亚热带季风气候，冬季温暖潮湿，夏季炎热多雨。年平均气温约21°C。高湿度和频繁云雾为茶叶生长创造理想条件。
* **特色：** 凤凰山许多茶树非常古老，树龄数百年。称为「老丛」(老枞)。据信这些古树茶具有更深层复杂的滋味。

**5. 制作工艺：**

单丛蜜兰香的制作工艺结合传统乌龙茶制法和潮州地区特色。
* **采摘 (采摘 - cǎi zhāi)：** 如上所述。
* **萎凋 (萎凋 - wěidiāo)：** 将采摘的叶片摊放在户外（日光或阴凉萎凋）或室内数小时。目的是去除部分水分，使叶片变软并开始发酵过程。
* **摇青 (摇青 - yáo qīng)：** 在竹筛上轻柔摇动和翻动叶片。此步骤进行数次，中间让叶片「静置」。对蜜兰香而言，摇青通常较轻柔，以保持叶片完整性并发展花香。
* **发酵 (发酵 - fājiào)：** 在摇青和静置过程中发生的氧化过程。蜜兰香的发酵程度通常为中度（30-60%），但可因制茶师而异。
* **杀青 (杀青 - shā qīng)：** 高温炒制以停止发酵过程。
* **揉捻 (揉捻 - róuniǎn)：** 将叶片塑造成纵向卷曲形状。可手工或机械揉捻。
* **烘干 (烘干 - hōnggān)：** 干燥茶叶以去除水分。此阶段可进行轻度或较强烈的焙火，视所需风味而定。
* **分级 (分级 - fēnjí)：** 成品茶按大小和品质分级。

**6. 感官特征：**

* **干茶外观：** 相对较大的纵向卷曲叶片，深棕色、褐色带红褐色调。可能有茶梗。有时可见复盖金色茸毛的芽头。
* **干茶香气：** 非常明亮浓郁，具有显著的蜜兰花香（蜜兰），果香（桃子、荔枝、热带水果）、蜂蜜、花香。可能有焙火、香料的细腻层次。
* **茶汤香气：** 浓郁、包覆性强、甘甜，以兰花和蜂蜜香调为主，伴有果香、花香，有时带轻微酸味。
* **滋味：** 丰富、浓郁、油润、甘甜，带轻微涩味和清新酸味。香韵以蜜兰花调为主，伴有果香（桃子、荔枝、芒果）、花香、香料细腻层次。回甘持久、甘甜，花果香韵悠长。
* **汤色：** 从金黄色到琥珀橙色，透明清澈。汤色取决于发酵程度和焙火程度。
* **叶底（冲泡后叶片）：** 完整、有弹性的叶片，冲泡后展开，呈棕绿色带红褐色边缘。

**7. 化学成分：**

单丛蜜兰香富含：
* **多酚类（儿茶素）：** 抗氧化剂。
* **胺基酸：** 包括L-茶胺酸。
* **生物碱：** 咖啡因、可可碱、茶碱。
* **精油：** 精油含量高，特别是花香和蜜香成分。
* **维生素：** C、B群、E、K。
* **矿物质：** 钾、氟、镁、锰、铁。

**8. 保健功效：**

* **提神效果：** 提神醒脑、消除疲劳、提高工作效率、改善注意力集中。
* **抗氧化作用：** 保护细胞免受自由基损害，延缓衰老过程。
* **改善消化：** 刺激消化、促进食物吸收。
* **温暖作用：** 在寒冷季节有很好的暖身效果。
* **心血管系统：** 可能有助于降低「坏」胆固醇水平，强化血管壁。
* **放松效果：** 尽管有提神作用，茶香和L-茶胺酸成分有助于放松和缓解压力。
* **提升情绪：** 带来和谐、愉悦和满足感。

**9. 冲泡方法：**

* **水温：** 90-95°C（大多数品种）。对于轻焙火和较绿的品种可使用85-90°C的水。
* **茶量：** 5-7公克配150-200毫升水。
* **茶具：** 最适合使用**盖碗**（传统中式带盖茶杯）或**宜兴紫砂壶**。也可使用瓷器茶具。
* **冲泡过程：**

	1. **温壶：** 用沸水冲洗盖碗或茶壶。
	2. **洗茶（快速冲泡）：** 将茶叶放入盖碗，注入少量热水后立即倒掉。
	3. **第一泡：** 注入热水（90-95°C）浸泡数秒至1分钟（第一冲）。第一泡时间可以很短，仅5-15秒，特别是高品质茶叶。
	4. **倒茶：** 将茶汤完全从盖碗或茶壶倒入公道杯，再分倒至茶杯。
	5. **续泡：** 单丛蜜兰香可多次冲泡（7-10次，有时更多），每次逐渐增加浸泡时间10-30秒。每一泡的滋味和香气都会变化，展现新的层次。

**重要细节：**

* **勿过度浸泡：** 浸泡时间过长可能使茶汤变得苦涩。
* **感受茶性：** 根据个人感受调整冲泡时间以达到理想浓度。
* **观察茶叶：** 注意汤色、香气、茶叶展开情况。

**10. 储存：**

单丛蜜兰香应储存在干燥、阴暗、凉爽处，使用密封容器（陶瓷、瓷器、玻璃或铁罐），远离异味。

**11. 价格与仿冒品：**

单丛蜜兰香属于昂贵的精品茶类。价格变化范围很大，从每100公克数十美元到数百美元，有时更高，取决于：
* **树龄：** 老丛茶价格远高于一般茶树。
* **海拔高度：** 高山茶（1000公尺以上）价格较高。
* **原料品质：** 是否使用精选芽叶或较成熟原料。
* **制茶师技艺：** 制茶师的经验和声誉对价格影响很大。
* **焙火程度与品质：** 经验丰富师傅的手工炭焙大幅提升茶价。
* **稀有性：** 某些稀有蜜兰香品种价格极高。
* **市场需求：** 单丛茶的高需求也影响价格。

由于高价格和受欢迎程度，市场上不幸存在许多仿冒品和模仿品。

**如何避免仿冒品：**

* **只向可信赖商家购买：** 寻找有良好声誉的专业茶叶店，能提供茶叶产地、采摘年份、制茶师等可靠资讯。
* **警惕过低价格：** 异常低价几乎总是仿冒品的明确标志。真正的单丛蜜兰香不可能便宜。
* **仔细检查外观：** 注意叶片形状、颜色、完整性。应符合上述描述。
* **评估香气：** 干茶应具有非常浓郁、复杂的香气，带有特征性蜜兰花、果香和花香。
* **检查茶汤和叶底：** 汤色应为金黄至琥珀橙色，清澈透明。

**12. 有趣事实：**

* **「单株」：** 「单丛」名称强调凤凰山每株茶树的独特性，其个别特征会传递给茶叶。
* **「活化石」：** 凤凰山某些茶树非常古老，树龄数百年。被称为茶界「活化石」。
* **香型多样：** 凤凰山有超过百种单丛品种，各具独特香气。包括杏仁、肉桂、茉莉、栀子花等各种花果香型的单丛。

**13. 其他知名单丛：**

除蜜兰香外，广为人知的还有：
* **鸭屎香 (鸭屎香, Yā Shǐ Xiāng)：** 「鸭屎香」。尽管名称挑衅，实际具有怡人花香。
* **芝兰香 (芝兰香, Zhī Lán Xiāng)：** 「芝兰香」。
* **杏仁香 (杏仁香, Xìng Rén Xiāng)：** 「杏仁香」。
* **肉桂香 (肉桂香, Ròu Guì Xiāng)：** 「肉桂香」。
* **玉兰花香 (玉兰花香, Yù Lán Huā Xiāng)：** 「玉兰花香」。
* **夜来香 (夜来香, Yè Lái Xiāng)：** 「夜来香」（晚香玉）。
* **姜母香 (姜母香, Jiāng Mǔ Xiāng)：** 「姜母香」。

还有许多其他品种，各具独特特色。

**14. 与其他乌龙茶比较：**

* **铁观音 (铁观音)：** 铁观音通常具有更明显的奶香花调，而单丛，特别是蜜兰香，以花果香和蜜香为主。铁观音通常呈较圆的卷曲形状，单丛则较细长。
* **大红袍 (大红袍)：** 大红袍是重发酵重焙火的岩茶，具有浓郁「火香」滋味和香气。蜜兰香相对具有更细腻甘甜的滋味，以花果香为主，焙火程度通常较轻。
* **水仙 (水仙)：** 水仙，特别是武夷水仙，具有更明显的矿物质和「岩韵」，以及奶香调。蜜兰香则以明亮的花香、蜜兰花香著称。

**结论：**

单丛蜜兰香是具有独特个性的精致乌龙茶，是凤凰山的名片。其明亮浓郁的蜜兰花香、丰富甘甜的滋味伴随果花香调，以及持久清新的回甘，不会让任何茶叶爱好者无动于衷。这款茶是真正的艺术品，在与自然和谐中创造，经多世纪传统淬炼而成。品尝真正的单丛蜜兰香，意味著为自己开启乌龙茶的奇妙世界，感受山岳能量，享受真正的茶艺完美。这是特殊场合的茶品，适合悠闲、深思的品茗时光，当人们想要沉浸在精致滋味和香气的世界中，感受和谐并获得难忘体验时。
