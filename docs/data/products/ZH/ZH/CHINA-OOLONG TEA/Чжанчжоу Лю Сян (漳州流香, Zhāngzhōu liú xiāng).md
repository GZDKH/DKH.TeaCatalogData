# 漳州流香 (漳州流香, Zhāngzhōu liú xiāng)

**漳州流香 (漳州流香, Zhāngzhōu liú xiāng)**
![Чжанчжоу Лю Сян (漳州流香, Zhāngzhōu liú xiāng)](images/Чжанчжоу%20Лю%20Сян%20%28漳州流香,%20Zhāngzhōu%20Liú%20Xiāng%29.png)

![Чжанчжоу Лю Сян (漳州流香, Zhāngzhōu liú xiāng)-1](images/Чжанчжоу%20Лю%20Сян%20%28漳州流香,%20Zhāngzhōu%20Liú%20Xiāng%29-1.png)

漳州流香是漳州国营茶厂三大经典茶品之一，与色种 (色种, Sèzhǒng) 和一枝春 (一枝春, Yī Zhī Chūn) 并列。这是现代中国罕见的「拼配」乌龙茶典范：来自福建不同地区的原料经过统一的精制工序 (精制, jīngzhì) ——精制、拼配和炭焙，最终诞生出具有独特「火香」风格和深厚浓郁滋味的茶品。

**1. 分类与产地：**

* **类型：** 乌龙茶（半发酵茶，乌龙茶, wūlóngchá）。风格——重发酵 (重发酵, zhòng fājiào)、重摇青 (重摇, zhòng yáo)、炭焙 (炭焙, tànbèi)。发酵程度约为40–60%。
* **类别：** 按精制地点归类为闽南乌龙 (闽南乌龙, Mǐnnán wūlóng)。然而主要原料来自闽北 (闽北, Mǐnběi)，这使得流香成为连接福建南北乌龙茶传统的独特「桥梁」。属于「漳州传统茶」类别。
* **产地：** 中国 (中国, Zhōngguó) 福建省 (福建省, Fújiàn shěng)。精制和拼配——漳州市 (漳州市, Zhāngzhōu shì) 芗城区 (芗城区, Xiāngchéng qū) 小坑头生产基地 (小坑头, Xiǎokēngtóu)。原料产区——武夷山 (武夷山, Wǔyíshān)、安溪 (安溪, Ānxī) 及福建南北其他乌龙茶产区。
* **地理座标：** 漳州：约北纬24°31′，东经117°39′。原料产区（武夷山核心区）：约北纬27°43′，东经117°57′。

**2. 历史与文化意义：**

* **历史：**

漳州精制乌龙茶的传统可追溯至清代。同治年间（同治, 1862–1874），企业家林心傅 (林心傅) 在漳州创办「林奇苑」(林奇苑) 茶庄，开始从武夷山采购毛茶，然后在漳州进行精制 (精制, jīngzhì) ——拼配、拣剔、复焙、匀堆和包装。「林奇苑」以「三印水仙」(三印水仙) 品牌生产的产品在东南亚成为福建乌龙茶的象征。同时期还有张源美 (张源美)、源崇美 (源崇美)、奇苑 (奇苑) 等茶庄，各有其拼配秘方。

1956年，在私营企业国有化过程中，所有主要漳州茶庄合并为国营漳州茶厂 (漳州茶厂, Zhāngzhōu cháchǎng，成立于1953年)。各家茶庄的拼配技艺融为一体，1957年茶厂正式创制「流香」(流香) 产品。1960年代，流香、色种 (色种) 和一枝春 (一枝春) 成为茶厂三大旗舰茶品。当时民间流传「流香色种一枝春，乘风飞马大前门」(流香色种一枝春，乘风飞马大前门)，将这些茶品与最知名的香烟品牌并列，突显其奢侈品地位。

在鼎盛时期的1970–1990年代，漳州乌龙茶出口量达到福建乌龙茶总出口量的40%。茶厂是中国三家拥有自主出口代码 (唛号, màhào) 的企业之一，其S2系列产品在日本和东南亚广为人知。1996年，流香在中国科技精品博览会 (中国科技精品博览会) 上获得金奖。2019年，漳州传统乌龙茶精制技艺入选福建省第六批省级非物质文化遗产名录 (福建省非物质文化遗产)。2022年，流香入选中国茶叶谱系代表品 (中国茶叶谱系代表品) 名录。2024年，为庆祝茶厂成立70周年，推出纪念版——特级流香 (特级流香)，采用手工纸包装，延续半世纪传统。

* **名称：**

「漳州」(漳州) ——精制地；「流」(流) ——「流动、流淌」；「香」(香) ——「香气」。字面意思：「来自漳州的流淌香气」。名称借用武夷山岩区著名的流香涧 (流香涧, Liúxiāng Jiàn)，这是历史上重要原料的来源地。这是武夷山茶叶风土的经典「灵气之地」之一，位于正岩区 (正岩区, zhèngyán qū) 核心地带。

* **文化意义：**

流香是漳州茶文化和闽南、潮汕工夫茶 (工夫茶, gōngfu chá) 文化的象征。漳州是最早有文献记载的工夫茶中心之一：彭光斗《闽琐记》(闽琐记, Mǐn Suǒ Jì) 记载于1766年（乾隆31年），描述了漳州使用武夷茶的仪式性茶饮。对于该地区世代居民以及马来西亚、新加坡和印尼的海外华人社群而言，流香一直是「故乡的味道」——乡愁和文化认同的物质体现。

**3. 植物学描述与原料：**

* **品种/栽培种：** 主要栽培种——福建水仙 (福建水仙, Fújiàn Shuǐxiān)，又称水吉水仙 (水吉水仙) 或武夷水仙 (武夷水仙)。这是半乔木型 (小乔木型, xiǎo qiáomù xíng) 大叶类 (大叶类, dàyè lèi) 晚生种，为天然三倍体。入选中国国家级茶树良种 (国家级茶树良种) 名录，注册号GS13009-1985，「华茶9号」(华茶9号)。学名：*Camellia sinensis* cv. Fujian-shuixian。辅助栽培种——肉桂 (肉桂, Ròuguì) 和闽北菜茶 (菜茶, càichá) 群体种。
* **采摘：** 春茶 (春茶, chūnchá) 为主，能提供最高的氨基酸浓度和香气深度。春季原料主要用于高等级产品。
* **采摘标准：** 一芽二至三叶 (一芽二至三叶, yī yá èr zhì sān yè)，成熟，无机械损伤。
* **原料要求：** 鲜叶茶多酚含量不少于25%，氨基酸不少于4.0%（按茶厂规格）。叶片应完整、成熟均匀，无异味。

**4. 风土与栽培特点：**

* **产区与地形：** 主要原料来自武夷山 (武夷山, Wǔyíshān) 高山岩石区——著名的丹霞地貌 (丹霞地貌, Dānxiá dìmào) 山体，具有陡峭峡谷、岩壁和狭窄山谷，为茶树创造独特的微气候。补充原料来自福建南部乌龙茶产区（安溪等）。
* **海拔高度：** 海拔600–800公尺（武夷山核心原料区）。
* **气候：** 湿润亚热带气候。采摘区年平均气温16–20°C，相对湿度≥75%，年雾日超过200天。昼夜温差显著。丰富的散射光 (漫射光, mànshèguāng) 有利于叶片中氨基酸和香气前体物质的积累。
* **土壤：** 丹霞砂岩-砾岩基底上的酸性红壤 (红壤, hóng rǎng)，pH值4.5–6.0。富含矿物质和有机质。核心岩区的茶园属于水源保护区，禁用化学肥料和农药，实行生态种植。

**5. 制作工艺：**

流香是双重工艺链的产物：原料首先在产地按传统乌龙茶工艺进行初制 (初制, chūzhì)，然后在漳州茶厂进行精制 (精制, jīngzhì)。精制阶段被认定为非物质遗产项目，是形成流香特色的关键环节。

**初制 (初制, chūzhì)：**

* **采摘 / 采摘 — cǎizhāi：** 采摘「一芽+二三叶」标准的顶梢。采摘的原料迅速送至加工车间。
* **晒青 / 晒青 — shàiqīng：** 比轻发酵乌龙茶更重 (较重, jiào zhòng)。叶片失去大量水分，细胞壁软化，开始形成香气基础。
* **晾青 / 晾青 — liàngqīng：** 叶片在阴凉处「休息」，水分从叶柄向叶缘重新分布。
* **做青（摇青）/ 做青 — zuòqīng (摇青 — yáoqīng)：** 摇青和静置的交替循环。重摇重发酵 (重摇重发酵, zhòng yáo zhòng fājiào) 启动叶缘的强烈氧化，形成花果香型和特征性的「红边」(红边, hóng biān)。
* **杀青 / 杀青 — shāqīng：** 热处理停止酶促反应，固定达到的氧化程度。
* **揉捻 / 揉捻 — róuniǎn：** 塑造外形，破坏细胞结构以改善冲泡时的浸出。
* **初焙 / 烘焙 — hōngbèi：** 稳定毛茶 (毛茶, máochá) 的含水量。

**精制 (精制, jīngzhì) ——非物质遗产：**

* **人工拼样 / 人工拼样 — réngōng pīnyàng：** 师傅手工制作控制样品，确定批次风格。
* **手工筛分 / 手工筛分 — shǒugōng shāifēn：** 按大小和形状分离茶叶。
* **拣剔 / 拣剔 — jiǎntī：** 去除茶梗、叶脉和缺陷片段。
* **官堆（拼配）/ 官堆 — guānduī：** 关键步骤。师傅将不同山头、季节、品种和加工程度的原料混合成统一和谐的拼配。拼配配方在师傅与学徒间传承五代。
* **炭焙 / 炭焙 — tànbèi：** 在不超过60°C的温度下用木炭进行多次缓慢焙火。正是这一步骤形成了标志性的「火香」(火香, huǒxiāng) ——流香的名片。
* **匀堆 / 匀堆 — yúnduī：** 最终混合以确保批次均匀性。
* **包装 / 包装 — bāozhuāng：** 传统上采用手工包装，内层白色竹纸，外层印有品牌图案的牛皮纸。红色图案——「红流香」(红流香, Hóng Liú Xiāng)，等级较高；深色图案——「黑流香」(黑流香, Hēi Liú Xiāng)。

**6. 感官特征：**

* **干茶外形：** 条索紧结卷曲 (条索紧结卷曲, tiáosuǒ jǐnjié juǎnqū)，相当沉重饱满。色泽乌褐带黄似香蕉色 (乌褐带黄似香蕉色)。表面油润，有适度光泽。高等级产品可见金毫 (金毫, jīnháo)。
* **干茶香气：** 深沉浓郁的「火香」(火香, huǒxiāng)，带有木炭和焦糖香调。陈年版本有「陈香」(陈香, chénxiāng) 韵味。背景中有细腻的兰花香 (兰香, lánxiāng)，特别是特制版本。
* **茶汤香气：** 强劲、温暖、多层次。主调为焦糖化、坚果、烘烤香调；展开后显现果香花香泛音。香气持久悠长，在空杯壁上留香。
* **滋味：** 醇厚 (醇厚, chúnhòu)，浓度与柔和度平衡。口中有包覆性甜味，类似蔗糖甜感 (蔗糖甜感)，转为持久的回甘 (回甘, huígān)。有清新感 (鲜爽, xiānshuǎng)，源于高氨基酸含量。涩味极少，无苦味。滋味平衡：「浓淡适中」(浓淡适中)。
* **汤色：** 橙黄 (橙黄, chénghuáng) 至红橙 (红橙, hóngchéng)，透明明亮，有良好深度。
* **叶底（冲泡后茶叶）：** 棕褐肥壮 (棕褐肥壮)，有弹性，红边明显 (红边显, hóng biān xiǎn)，证明重度氧化。中脉清晰。

**7. 化学成分：**

流香的化学特征主要由水仙栽培种的高浸出物含量和强化加工（重氧化+多次炭焙）决定。

* **多酚类：** 鲜叶含量≥25%（按茶厂规格）。成品茶中部分儿茶素在氧化和热处理过程中转化为茶黄素、茶红素和更复杂的聚合物。根据研究，福建水仙栽培种（福安，福建样品）鲜叶一芽二叶多酚含量约17.6%，低于茶厂标准，这是由于样品采集海拔较低；武夷山高山原料通常显示更高数值。
* **氨基酸：** 鲜叶≥4.0%（茶厂规格）。L-茶氨酸为主要成分（占总氨基酸含量50%以上）。正是它提供鲜爽感 (鲜爽) 和柔和甜味，并调节咖啡因作用，提供平静的提神效果。
* **生物碱：** 咖啡因——鲜叶约4.0%（福安水仙数据）。焙火后成品茶含量略有下降。还含有微量可可碱和茶碱。
* **水溶性浸出物：** 水仙栽培种高达50.5%——极高指标，解释了茶汤的浓度和饱满度。
* **维生素：** B族维生素（B₁、B₂）、维生素C（焙火时大部分被破坏）、维生素E、维生素K。
* **矿物质：** 钾、镁、锰、氟、锌、微量硒。武夷山丹霞红土为茶叶提供额外微量元素。
* **精油和香气化合物：** 形成复杂香气特征。炭焙促进美拉德反应和焦糖化，产生呋喃、吡嗪和吡咯化合物，负责「火香」。保留的萜烯醇（芳樟醇、橙花醇、香叶醇）提供花香背景。

**特点：** 水仙的多酚与氨基酸比值 (酚氨比, fēn'ān bǐ) 约5.33，被认为是制作平衡涩味与甜味乌龙茶的最佳比例。

**8. 保健功效：**

* **温和持续的提神效果：** 咖啡因与L-茶氨酸的结合提供无剧烈起伏的清醒感。L-茶氨酸刺激大脑α波产生，促进专注平静。
* **抗氧化活性：** 茶多酚（儿茶素及其氧化产物——茶黄素和茶红素）有效中和自由基，帮助保护细胞免受氧化压力。
* **支持脂质代谢：** 儿茶素及其转化产物促进脂肪代谢加速。据资料显示，重发酵乌龙茶的这种效果比轻发酵乌龙茶更明显。
* **助消化：** 焙火乌龙茶传统上被认为是「温性」茶，对肠胃蠕动有益，温和刺激消化。特别推荐餐后饮用，尤其是油腻食物后。
* **利尿作用：** 咖啡因和矿物成分促进温和的利尿效果，帮助排出多余水分。
* **支持心血管系统：** 定期适量饮用多酚茶与降低「坏」胆固醇（LDL）水平和维持血管弹性相关。
* **强化牙齿珐瑯质：** 茶叶中的氟有助于珐瑯质再矿化；多酚抑制致龋细菌生长。
* **正念茶饮与减压：** 流香的工夫茶冲泡仪式，通过多次冲泡和观察香气展开，是有助于放松和情绪舒缓的实践。

**9. 冲泡方法：**

* **水温：** 95–100°C。热水对于充分展现焙火乌龙茶的「火香」和浓厚茶体是必需的。不建议使用超过100°C的沸水——这可能破坏炭焙的细腻泛音。
* **茶量：** 工夫茶法5–8克/100毫升，或欧式冲泡法3–4克/200–250毫升。
* **茶具：** 瓷质盖碗 (盖碗, gàiwǎn) 或宜兴紫砂壶 (紫砂壶, zǐshā hú)。紫砂壶特别推荐用于陈年 (陈年, chénnián) 版本流香——多孔结构增强「陈香」表现，为茶汤增添丝滑感。
* **冲泡过程：**
  1. 用沸水温热茶具并倒掉。
  2. 将茶叶放入盖碗或茶壶。
  3. 快速洗茶 (润茶, rùnchá)：注水3–5秒后倒掉。这能「唤醒」干茶并去除表面尘埃。
  4. 第一泡：注入95–100°C水，浸泡15–20秒。倒入公道杯 (公道杯, gōngdào bēi)。
  5. 分杯品饮。
  6. 续泡：可冲泡6–10次。每次增加5–10秒浸泡时间。高等级产品可冲泡8–10次。

* **注意：** 新焙火茶建议在阴凉处存放约15天后饮用，以消散过重的火气 (火气, huǒqì)。开封后建议7天内饮完，以免失去香气。不建议空腹饮用——单宁可能刺激胃黏膜；最佳时间是餐后一小时。

**10. 储存：**

* **条件：** 密封包装，干燥阴凉处，远离热源和异味。理想储存温度15–25°C，湿度不超过60%。
* **容器：** 密封铁罐、真空袋或茶厂原装纸包装。陶罐 (陶罐, táoguàn) 也适合。
* **茶叶天敌：** 潮湿、高温、异味、直射光、氧气。
* **保质期：** 焙火乌龙茶是最稳定的茶类之一。正确储存下流香可保持品质2–3年，陈年版本 (陈年, chénnián) 在定期复焙条件下随年份增加「陈香」韵味。

**11. 价格与仿冒：**

* **价格区间：** 取决于原料等级、采摘季节、拼配技艺和年份。参考价格（茶厂数据）：特级 (特级, 高等级) ——700元/斤（500克）起；一级 (一级) ——中等价位；二级 (二级) ——160–240元/斤，性价比最佳。纪念版和限量版价格可能显著更高。
* **避免仿冒方法：**
  * 从漳州茶厂官方经销商或专卖店购买（漳州、潮州、汕头共29个点）。
  * 检查原装包装：传统牛皮纸配品牌红色或深色图案，内层白色竹纸。品牌「霞漳」(霞漳, Xiázhāng) 或「芝山」(芝山, Zhīshān)。
  * 评估干茶香气：真正的流香有纯净深沉的「火香」，无刺鼻化学味。
  * 检查茶汤：颜色应为橙黄色，透明；滋味浓厚，有持久回甘，无「空洞感」。
  * 过低价格配合声称的高等级应引起警惕。

**12. 趣味知识：**

* 漳州茶厂是福建四大国营乌龙茶厂中唯一存续至今的。其余三家在市场化改革中关闭。
* 漳州乌龙茶精制技艺是中国唯一以「精制」(精制) 而非初制 (初制) 阶段为非物质遗产对象的案例。这突显了拼配和焙火技艺的独特性。
* 流香是「桥梁茶」：其配方有意结合闽北（浓厚、深沉）和闽南（花香、柔和）乌龙茶优点于一杯，使其在各地区「纯正」乌龙茶中独树一帜。
* 茶厂至今仍采用手工包装：女工将每份（一两——约50克）用白色竹纸和牛皮纸包装，约30秒完成，保持半世纪不变的方法。
* 流香和其他漳州乌龙茶的出口沿海上丝绸之路 (海上丝绸之路, Hǎishang Sīchóu zhī Lù)，这种茶的味道成为东南亚数百万华侨的文化标志。

**13. 与其他乌龙茶比较：**

* **武夷水仙 (武夷水仙, Wǔyí Shuǐxiān)：** 同一栽培种的单一成分岩茶，但无拼配和闽南添加。通常有更明显的「岩韵」(岩韵, yányùn) 和矿物感。流香因拼配而有更柔和「圆润」的特征。
* **武夷肉桂 (武夷肉桂, Wǔyí Ròuguì)：** 明亮、辛香，有特征性「桂皮」香和更明显的辛辣感。流香更平静、深沉、甘甜。
* **铁观音传统 (铁观音传统, Tiěguānyīn chuántǒng)：** 安溪传统重焙火铁观音——风格上最接近的闽南类比。但铁观音用同名栽培种制作，有更多花蜜特征，而流香基于水仙，茶体更浓厚。
* **漳平水仙 (漳平水仙, Zhāngpíng Shuǐxiān)：** 同一栽培种的闽南乌龙茶，制成压饼形式。风格更轻，花香奶香特征。流香显著更「火香」和浓缩。
* **凤凰单丛 (凤凰单丛, Fènghuáng Dāncóng)：** 广东乌龙茶，有明显的花果单一香型。单丛是「单株茶」，而流香是「拼配茶」；两者理念截然相反，但都在潮汕工夫茶文化中受到重视。

**结语：**

漳州流香是百年漳州茶艺传统的活化石：从清代商号到唯一存续的国营乌龙茶厂。其特殊价值在于拼配艺术：掌握「官堆」秘诀的第五代传人，将武夷山岩茶的山野力量与闽南乌龙茶的柔和在一杯中融合，通过多次炭焙直至诞生标志性的「火香」。

这款茶适合珍视深度与温暖的人：浓厚、温暖，有持久甘甜回味和在杯壁上久久不散的香气。流香不仅是饮品，更是文化记忆的片段，连接漳州与武夷山，福建与东南亚，以及一代代茶艺师傅。
