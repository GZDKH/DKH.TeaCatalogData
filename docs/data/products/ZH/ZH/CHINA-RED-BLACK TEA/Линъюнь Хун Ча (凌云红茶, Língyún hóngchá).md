# 凌云红茶 (凌云红茶, Língyún hóngchá)

**凌云红茶 (凌云红茶, Língyún hóngchá)**
![Линъюнь Хун Ча (凌云红茶, Língyún hóngchá)](images/Линъюнь%20Хун%20Ча%20%28凌云红茶,%20Língyún%20Hóngchá%29.png)

![Линъюнь Хун Ча (凌云红茶, Língyún hóngchá)-1](images/Линъюнь%20Хун%20Ча%20%28凌云红茶,%20Língyún%20Hóngchá%29-1.png)

凌云红茶是来自广西西北部高山县份凌云 (凌云) 的红茶，采用独特的凌云白毫 (凌云白毫, Língyún Báiháo) 品种制作——这是中国30个国家级优良茶树品种之一，也是亚洲唯一能够生产六大茶类产品的品种。如果说「绿茶」版本的凌云白毫以银白茸毫和清新感著称，那么「红茶」版本则展现出完全不同的风貌——浓郁的蜜果香味、明亮透澈的茶汤和特有的悠长甘甜回味。这个品种的座右铭是「一茶千化」(yī chá qiān huà)——「一茶千变」。

**1. 分类与产地：**

* **类型：** 红茶 (红茶, hóngchá)——完全发酵/氧化茶。
* **类别：** 中国地方红茶；大叶种原料制成的工夫红茶 (工夫红茶)。
* **产地：** 中国广西壮族自治区 (广西壮族自治区, Guǎngxī Zhuàngzú Zìzhìqū)；百色市 (百色市, Bǎisè Shì)；凌云县 (凌云县, Língyún Xiàn)。主要生产区域集中在岑王老山 (岑王老山, Cénwáng Lǎoshān) 和青龙山 (青龙山, Qīnglóng Shān) 山脉，以及玉洪乡 (玉洪乡, Yùhóng Xiāng) 和加尤镇 (加尤镇, Jiāyóu Zhèn)。
* **地理座标：** ≈ 24.35° 北纬，106.56° 东经（以凌云县城为参考点）。

**2. 历史与文化意义：**

* **历史：** 凌云白毫茶树在岑王老山和青龙山坡上生长已超过千年。《中国名茶志》记载：「凌云白毛茶为历史名茶，创于清乾隆以前」——凌云白毛茶是历史名茶，创于清乾隆以前。县内首次有目的性的茶树种植可追溯至1488年（明朝弘治 (弘治) 年间），当时当地居民建立了第一个茶园。清朝时期，凌云茶成为宫廷贡茶 (贡茶)。《广西特产物品志》（1937年）描述当地茶树：「白毛茶，树大者高约二丈，小者七尺，嫩叶如银针，老叶尖长如龙眼树叶而薄，皆有白色茸毛，故名，概属野生」——树木高达二丈（≈ 6公尺），嫩叶如银针，覆盖白色茸毛。

  凌云原料制成的红茶是相对年轻的发展方向。历史上凌云白毫主要加工成绿茶。红茶生产的有目的性发展始于2000年代，当时当地企业掌握了适用于大叶种原料的工夫红茶技术。结果令人印象深刻：2015年凌云红茶在米兰世界博览会Expo 2015的「红茶」类别中获得金奖。2010年凌云茶被列入广西名茶品牌名录。2016年「凌云白毫」获得国家工商总局地理标志证明商标 (国家工商总局地理标志证明商标) 地位。2021年凌云白毫茶制茶技艺 (凌云白毫茶制茶技艺) 被列入自治区非物质文化遗产名录。目前凌云县拥有超过120,000亩（≈ 8000公顷）茶园，包括23,000亩有机茶园；茶产业涉及县内四分之一人口，为一般家庭提供年收入超过30,000元人民币。

* **名称：** 「凌云」(Língyún)——县名；字面意思「凌」——「上升、翱翔」，「云」——「云」，准确反映了当地高山多云的特色。「红茶」(hóngchá)——红茶。全称——「凌云红茶」。替代商品名称——凌云白毫红茶 (凌云白毫红茶)——「凌云白毫原料红茶」。

* **文化意义：** 凌云是官方认定的「中国名茶之乡」(中国名茶之乡, Zhōngguó Míngchá zhī Xiāng) 之一，并入选「中国十大生态产茶县」。茶叶是当地经济和文化认同的支柱。有一个关于茶圣陆羽 (陆羽) 造访凌云的传说：据说陆羽品尝当地白毛茶后印象深刻，将自己的茶艺秘诀传授给凌云人。为纪念此事，在先锋岭 (先锋岭) 建立了「茶圣传艺亭」(茶圣传艺亭)。这个能够转化为六大茶类（绿、红、白、黄、黑、青）任何一种的独特品种，赋予凌云白毫中国茶业「万能茶兵」的地位。

**3. 植物学描述与原料：**

* **品种/栽培种：** 凌云白毫 (凌云白毫, Língyún Báiháo)，也称凌云白毛茶 (凌云白毛茶)——「凌云白毛茶」。官方编号——华茶26号 (华茶26号, Huáchá 26 Hào)。1984年被认定为中国首批30个国家级优良茶树品种之一。属于小乔木大叶种 (小乔木大叶种, xiǎo qiáomù dàyè zhǒng) *Camellia sinensis* 类型。树木可长至9公尺；树冠直立，分枝稀疏；叶片大、厚、肉质，水平排列或下垂。特征——叶背和芽上有丰富的白毫 (白毫, báiháo)，这是品种名称的由来。发芽能力强；百芽三叶重约99克。开花较晚（11-12月），结果不多。
* **采摘：** 春季——主要季节；芽在3月中旬萌发，「一芽三叶」采摘高峰期——3月底至4月初。红茶也使用夏秋茶叶，能提供更浓郁和涩味的风味特征。
* **采摘标准：** 一芽一叶 (一芽一叶) 用于「金钩红条」(金钩红条, Jīngōu Hóng Tiáo——「金钩」) 等高级批次；一芽二叶 (一芽二叶) 用于标准级；出口红碎茶 (红碎茶, hóng suì chá) 使用较成熟的叶片。
* **原料要求：** 完整、新鲜的嫩梢，白毫明显；无机械损伤和粗老叶片。

**4. 风土条件与栽培特色：**

* **海拔高度：** 800-2000公尺。主要茶园位于海拔800-1500公尺，但个别野生茶树可见于更高处——岑王老山坡上高达2000公尺。
* **气候：** 高山亚热带气候；年平均温度19-23°C；夏季不炎热，冬季温和。年降雨量1700-1800毫米。特点是持续的高云量和雾气：「晴时早晚遍山雾，阴雨成天满山云」——「晴天早晚满山雾，阴雨整天满山云」。散射光和短暂直射阳光刺激氨基酸和芳香化合物的积累。
* **土壤：** 红壤、红黄壤和黄壤山地土壤，pH值4.5-6.0。由于岑王老山森林的存在，有机质含量高。肥沃层深度超过40公分。土壤疏松、排水良好，富含铁和锰。
* **水资源：** 凌云县位于广西西北部，靠近云南-贵州高原。地形高耸起伏，有众多山间溪流。生态环境优良——无工业污染。
* **农业技术：** 大部分茶园获得有机认证（超过23,000亩）。采用绿色农业方法：使用有机肥料（泥炭堆肥、油饼），生物防虫（灯光诱捕、植物杀虫剂），高级茶叶手工采摘。

**5. 制作工艺：**

凌云红茶采用适用于大叶种高茸毫原料的工夫红茶技术制作。

* **采摘 (采摘, cǎizhāi)：** 手工挑选嫩梢；高级批次专用春季原料。
* **萎凋/摊青 (摊青/萎凋, tānqīng/wěidiāo)：** 鲜叶摊放在通风良好室内的竹制托盘上。凌云品种的大型肉质叶片需要长时间萎凋——自然方法下达14-18小时。失水至60-65%，叶片变软，香气从草香演变为花果香。
* **揉捻 (揉捻, róuniǎn)：** 萎凋叶揉捻以破坏细胞壁并释放酶类。由于叶片较大，采用适度、逐渐增加压力的方式，避免完全撕裂叶片，而是形成特有的卷曲形状。
* **发酵/氧化 (发酵, fājiào)：** 揉捻叶在温度控制（25-28°C）和湿度（≥95%）的发酵室中分层放置。持续时间——2-4小时。通过颜色变化（从绿色到红铜色）和香气变化（从草香到果蜜香）来控制。
* **初烘 (初烘, chūhōng)：** 100-110°C热风——固定发酵过程。
* **摊凉 (摊凉, tānliáng)：** 茶叶摊放均匀冷却和水分重新分布。
* **造型、提毫 (造型、提毫, zàoxíng, tíháo)：** 凌云茶特有工序——额外轻揉和轻摇，使茶叶表面显现金毫 (金毫)。
* **足烘 (足烘, zúhōng)：** 干燥至残余水分5-6%。
* **分级：** 按等级分类；去除缺陷叶片和叶柄。

**6. 感官特征：**

* **干茶外观：** 紧结卷曲、略弯的茶条（「金钩」——因此得名「金钩红条」）。颜色——深褐色带油润光泽。表面有丰富的金毫 (金毫, jīnháo)，由白毫在氧化过程中转化而成。
* **干茶香气：** 浓郁、蜜甜香带温暖的栗子、干果调和淡雅花香。白毫原料赋予香气特殊的「柔和」调性，这是小叶种红茶所没有的。
* **茶汤香气：** 浓郁、多层次——蜂蜜、熟果（杏、柿子）、焦糖。随著冲泡展现温暖的面包和坚果调性。香气持久，在杯壁上久留不散。
* **滋味：** 浓厚、饱满，带明显圆润甘甜和柔和涩味。茶体——浓醇 (浓醇, nóng chún)、「丝绒般」。回味悠长，带栗子蜜香和淡雅辛香。特色——「回甘」(huígān)——饮后渐增的回甜。
* **汤色：** 红琥珀色，明亮透澈 (红艳明亮, hóngyàn míngliàng)。杯壁有金圈。
* **叶底（冲泡后叶片）：** 大型、完全展开的红铜色叶片；肉质、有弹性、均匀。叶背可见白毫痕迹——凌云原料的识别特征。

**7. 化学成分：**

根据中国农业科学院茶叶研究所数据，凌云白毫绿茶含有：咖啡因——4.91%，氨基酸——3.36%，茶多酚——35.6%，儿茶素总含量——182.92毫克/克。完全氧化（红茶）后儿茶素组成显著改变：

* **多酚类：** 儿茶素氧化为茶黄素和茶红素。高初始多酚含量（绿茶35.6%）为红茶提供浓厚茶体、明亮汤色和浓郁滋味。
* **氨基酸：** 含量（绿茶3.36%）——对大叶种而言相对较高。L-茶氨酸提供柔和甘甜和放松效果。红茶中部分氨基酸参与美拉德反应，形成焦糖和面包调性。
* **生物碱：** 咖啡因——约4.9%（高于中国茶叶平均水平，这是由于大叶种和高山生长环境）。可可碱和茶碱——微量。
* **维生素：** B族维生素、维生素C（完全发酵时显著降低）、芸香苷。
* **矿物质：** 钾、镁、锰、锌、氟、硒。凌云红色山地土壤富含铁和锰。
* **精油和挥发性化合物：** 芳樟醇、香叶醇、苯乙醇、糠醛。丰富的芳香化合物形成多层次蜜果香气和花香调性。

**8. 健康功效：**

* **提神醒脑：** 高咖啡因含量（≈ 4.9%）结合L-茶氨酸提供强劲但温和的提神效果——精神振奋、注意力集中和思维清晰，无焦虑感。
* **抗氧化保护：** 茶黄素和茶红素——多酚氧化产物——保持明显的抗氧化活性。
* **消化支持：** 单宁物质刺激消化酶分泌；红茶特别推荐在丰盛油腻餐后饮用。
* **温暖作用：** 在中医营养学中，大叶种红茶属于「温性」食品——温胃和促进血液循环。
* **心血管系统支持：** 适量饮用红茶与维持血管弹性相关。
* **免疫调节特性：** 矿物质复合物（锌、硒、锰）结合多酚支持天然防御功能。
* **疲劳和过劳缓解：** 咖啡因、茶氨酸和B族维生素的组合使凌云红茶成为良好的「恢复性」饮品。

**9. 冲泡方法：**

* **水温：** 90-95°C。
* **茶量：** 4-5克/100-120毫升（工夫泡法）；3-4克/200-250毫升（西式泡法）。
* **茶具：** 白瓷盖碗 (盖碗)——最适合展现香气。宜兴紫砂壶——增加柔和度。玻璃茶壶——便于观察汤色和大叶展开。
* **工夫泡法流程：**
  1. 热水温盖碗，倒出。
  2. 投茶，盖上盖子10秒，闻香。
  3. 洗茶——快速冲泡（2-3秒），倒出。推荐用于紧结大叶茶。
  4. 第一泡：8-10秒。
  5. 第二至四泡：10-15秒。
  6. 第五泡起——增加5-10秒。
  7. 凌云红茶通常可冲泡7-10次。

**10. 储存：**

* 密封、不透光容器（铁罐、铝箔袋）。
* 温度15-25°C；干燥、阴暗处；湿度低于60%。
* 最佳饮用期——12-24个月。优质春茶批次可温和陈化2-3年。
* 不要冷藏保存；避免与有强烈气味的产品邻近。

**11. 价格与仿冒：**

* **价格区间：** 中等及中高价位：标准批次——150-400元人民币/500克；高级「金钩红条」——500-1500+元人民币/500克。价格取决于采摘标准、芽头比例、季节和认证（有机生产——更贵）。
* **避免仿冒方法：**
  1. 注意「凌云白毫」地理标志证明商标 (地理标志证明商标, 2016年注册) 或「凌云白毫茶」(地理标志产品保护, 2005年起) 标识。
  2. 评估茶叶：丰富的金毫 (金毫)——凌云原料的名片；声称为凌云红茶却缺乏金毫是警讯。
  3. 检查冲泡后叶底：大型、肉质叶片背面有白毫残留——凌云白毫品种的可靠特征。
  4. 评估香气：纯净、蜜甜，无焦糊、酸败或霉味。
  5. 谨慎对待无生产商和产区标示的产品——正宗凌云红茶由县内有限的认证企业生产。

**12. 趣闻轶事：**

* 凌云白毫品种是亚洲唯一能制作六大茶类的品种：绿茶 (绿茶)、红茶 (红茶)、白茶 (白茶)、黄茶 (黄茶)、黑茶 (黑茶) 和乌龙茶 (青茶)。这种现象源于叶片中多酚、氨基酸和酶的独特平衡，使其能「响应」任何加工技术。
* 1915年凌云茶与传奇中国烈酒茅台一同在巴拿马万国博览会 (巴拿马万国博览会) 上展出，获得高度评价。
* 凌云白毫最著名的红茶产品是「金钩红条」(金钩红条)——「金钩」——因茶条略弯、密布金毫的特征形状得名。另一个受欢迎的等级是「红螺王」(红螺王, Hóng Luówáng)——「红螺王」——紧结螺旋状茶条。
* 2015年凌云白毫红茶在米兰世界博览会Expo 2015获得金奖——这是广西茶产品在「红茶」类别的首个国际奖项。
* 凌云县是中国少数以茶产业为农村经济主导产业的县份之一，涉及约25%人口。每四个县民中就有一个与茶叶种植、加工或销售相关。

**13. 与其他红茶比较：**

* **滇红 (滇红, Diānhóng)：** 云南大叶种红茶，来自var. *assamica*。与凌云红茶在「大叶种」特性上相似——浓厚茶体、蜜甜、丰富金毫。但滇红通常更强劲、麦芽味更重，而凌云更优雅，带花香调和更纯净、「透明」的甘甜。
* **宜红工夫 (宜红工夫, Yíhóng Gōngfū)：** 湖北小叶种工夫红茶。条索明显更细，焦糖面包风味更突出，外观较少「茸毛」。凌云红茶——更粗壮、「丰满」，茶体更强劲。
* **九红工夫 (九红工夫, Jiǔhóng Gōngfū)：** 江西九江县红茶。中叶种，风味精致。凌云红茶——明显更粗壮浓郁。
* **龙脊红茶 (龙脊红茶, Lóngjǐ Hóngchá)：** 广西另一款红茶（龙胜县），但基于当地野生大叶种。龙脊——「野性」、森林感，带木质坚果调；凌云——更「文雅」，明显的茸毫芽头和花蜜风味。两款茶都是广西「隐藏茶潜力」的亮丽代表，至今在区域外仍鲜为人知。

**结语：**

凌云红茶是一个优秀品种如何在完全意想不到的维度中展现的绝佳例子。凌云白毫，数世纪来以银毫绿茶闻名，在红茶版本中转化为浓郁、蜜甜的饮品，具有丝绒般的茶体和引人注目的金毫。对于那些厌倦了「标准」红茶、寻求有个性茶品的人——大叶种、芳香、有历史底蕴且每片茶叶都带金光——凌云红茶将是愉快的发现。最好在宁静环境中慢慢品饮，不急不躁，一泡接一泡地观察肉质叶片的展开和茶汤蜜甜的加深。
