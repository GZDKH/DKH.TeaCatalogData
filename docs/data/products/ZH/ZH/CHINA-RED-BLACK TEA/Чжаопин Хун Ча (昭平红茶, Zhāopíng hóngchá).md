# 昭平红茶 (昭平红茶, Zhāopíng hóngchá)

**昭平红茶 (昭平红茶, Zhāopíng hóngchá)**
![Чжаопин Хун Ча (昭平红茶, Zhāopíng hóngchá)](images/Чжаопин%20Хун%20Ча%20(昭平红茶,%20Zhāopíng%20Hóngchá).png)

![Чжаопин Хун Ча (昭平红茶, Zhāopíng hóngchá)-1](images/Чжаопин%20Хун%20Ча%20(昭平红茶,%20Zhāopíng%20Hóngchá)-1.png)

昭平红茶是来自广西壮族自治区昭平县的工夫红茶，以绿茶品种福云六号 (福云六号, Fúyún Liùhào) 为原料，采用红茶工艺制作。该产品成为该地区的标志性茶品，首次打破了昭平千年绿茶垄断的局面，并于2013年获得「昭平茶」(昭平茶) 地理标志产品保护。

**1. 分类与产地：**

* **类型：** 中国红茶 (红茶, hóngchá)，全发酵茶。
* **类别：** 工夫红茶 (工夫红茶, gōngfū hóngchá) — 多阶段生产工艺的精制红茶。广西地方茶。
* **产地：** 中国广西壮族自治区 (广西壮族自治区, Guǎngxī Zhuàngzú Zìzhìqū)，贺州市 (贺州市, Hèzhōu Shì)，昭平县 (昭平县, Zhāopíng Xiàn)。主要产区：昭平镇、走马镇 (走马镇)、南岭镇 (南岭镇)，以及象棋山茶园。主要创建品牌的企业是象棋山茶叶有限公司 (象棋山茶叶有限公司)。
* **地理座标：** 约北纬24°10′，东经110°48′。

**2. 历史与文化意义：**

* **历史：** 昭平是中国南方最古老的产茶县之一：明清时期的县志中已有当地茶叶的文字记载；20世纪初，县内茶叶在《昭平县志》(《昭平县志》) 中有详细描述。2009年之前，该地区仅以绿茶闻名；1988年「绿茶2号」(绿茶2号) 品种荣获「陆羽杯」(陆羽杯, Lùyǔ Bēi) 金奖，并被收录于《中国茶经》(《中国茶经》)。红茶「昭平红」于2009年7月由象棋山公司首次成功制作，以福云六号品种为原料。2010年该新产品荣获上海国际茶叶展览会金奖，专家学者陈宗懋 (陈宗懋, Chén Zōngmào) 给予高度评价。2013年「昭平茶」(包括红茶和绿茶) 获得中华人民共和国国家质量监督检验检疫总局地理标志产品保护 (地理标志产品保护) 地位。至2017年，全县茶园面积达2.15万亩 (约14,300公顷)，年产干茶1.16万吨。
* **名称：** 「昭平」(昭平) — 地名，字面意思为「光明平坦」；县名起源于宋代 (12世纪)：1124年 (宣和6年)，徽宗皇帝 (徽宗, Huīzōng) 将龙平县 (龙平) 改名为昭平，将「招」(招, 「召唤、招募」) 字改为「昭」(昭, 「光明、明亮」)，认为前者不吉利。「红茶」(红茶) — 红茶。「工夫」(工夫) 在茶叶制作中意指「精细、费工的工作」，指需要复杂多阶段工艺，每个环节都需要特殊技艺 — 从发酵控制到最终精制。
* **文化意义：** 2009年昭平红的创制成为该县茶业的转捩点，打破了多世纪绿茶垄断的局面。昭平拥有「中国名茶之乡」(中国名茶之乡, 「中国著名茶叶之乡」)、「中国有机茶之乡」(中国有机茶之乡, 「中国有机茶之乡」) 称号，并列入「全国重点产茶县」。该县同时以「中国长寿之乡」(中国长寿之乡, 「中国长寿之乡」) 闻名，在行销上与当地茶园的生态纯净相结合。

**3. 植物学描述与原料：**

* **品种/栽培种：** 昭平红的主要栽培种为福云六号 (福云六号, Fúyún Liùhào)，即「福云6号」，属于 *Camellia sinensis* var. *sinensis*。这是小叶和中叶型栽培种，原产福建省，很好地适应了广西的亚热带气候。也使用当地群体种 (群体种, qúntǐzhǒng) 和其他适应当地的小叶和中叶型栽培种 — 乌牛早 (乌牛早, Wūniú Zǎo)、黄金茶 (黄金茶, Huángjīn Chá)、元宵绿 (元宵绿, Yuánxiāo Lǜ)。
* **采摘：** 昭平以早春开采闻名 — 采摘从2月底开始，因此当地茶叶被称为「中国大陆早春第一茶」(中国大陆早春第一茶)。春茶 (3-4月) 提供最高品质的原料；夏茶也有采摘，但风味特征不如春茶。
* **采摘标准：** 一芽一叶或一芽二叶 (一芽一叶, 一芽二叶)。高等级茶品偏好芽头较多的比例。
* **原料要求：** 完整、新鲜的叶片，无机械损伤和粗梗；采摘与萎凋开始之间的延迟时间最短，以保持酶活性。

**4. 风土与栽培特色：**

* **地形与生态：** 昭平县位于山区：山地占总面积87%以上，森林覆盖率超过87.6%。该地区位于南岭 (南岭, 「南岭山脉」) 范围内，特征是河谷深切，湿度高，雾气频繁。
* **海拔高度：** 主要茶园位于海拔400-1000公尺；象棋山有机茶园约700公尺。
* **气候：** 亚热带季风气候，温和湿润。年平均气温19.8-19.9°C；无霜期超过310天；年平均降雨量约2046毫米 — 广西最多雨的地区之一。年平均日照约1440小时，特征是经常多云和高湿度。
* **土壤：** 以黄壤和红黄壤 (黄壤, huáng rǎng) 为主，酸性，pH 4.5-6.5，由于丰富的森林落叶，有机质含量高。
* **水资源：** 桂江 (桂江, Guìjiāng) 穿越境内，内部水道总长1624公里。水质清洁，无工业污染 — 该县因工业存在极少而入选「深呼吸小城百佳」(深呼吸小城)。
* **生态认证：** 「绿色食品」(绿色食品) 认证茶园面积7900亩；16家企业通过有机生产认证或转换。大型工业缺乏和高森林覆盖率确保了空气清洁和无酸雨。

**5. 生产工艺：**

昭平红按照经典工夫红茶工艺生产，分为初制 (初制, chūzhì) 和精制 (精制, jīngzhì)。关键技术特点是成功克服了用绿茶品种制作红茶时常见的发酵不均、汤色淡薄、香气贫乏和明显苦涩等问题。

* **采摘 (采摘, cǎizhāi)：** 清晨手工采摘「一芽一至二叶」标准。
* **萎凋 (萎凋, wěidiāo)：** 将叶片薄摊在通风室内；水分含量降至约60%，叶片变软有弹性。持续4-6小时，视空气湿度而定。
* **揉捻 (揉捻, róuniǎn)：** 机械揉捻破坏细胞壁，使细胞汁液流到表面，确保均匀氧化。叶片呈现特有的紧密螺旋形。
* **氧化/发酵 (发酵, fājiào)：** 在控制温湿度下进行。儿茶素的酶促氧化形成茶黄素和茶红素，产生红色和甜味特征。福云六号需要特别仔细的控制，以避免发酵不均。
* **初烘 (初烘, chūhōng)：** 用100-110°C热风停止氧化。
* **精制 (精制, jīngzhì)：** 包括筛分 (筛分, shāifēn)、风选 (风选, fēngxuǎn)、手工挑拣 (挑拣, tiāojiǎn)、复火 (复火, fùhuǒ) 和拼装 (拼装, pīnzhuāng)。正是这个阶段的多步骤性质赋予了「工夫」— 「精细工作」的名称。

**6. 感官特征：**

* **干茶外形：** 细紧螺旋卷曲 (卷曲成螺, juǎnqū chéng luó)，颗粒均匀紧实。色泽乌润 (乌润, wūrùn) 有油光，金毫显露 (金毫显露, jīnháo xiǎnlù)。
* **干茶香气：** 清纯、高扬持久；以花果香调为主，伴有蜂蜜和温暖焦糖香。高等级茶品有明显的花果香 (花果香, huāguǒ xiāng)。
* **茶汤香气：** 浓郁多层次，甜蜜蜂蜜香带明显花香顶调；随著冷却出现果干和淡焦糖香调。
* **滋味：** 醇厚甘爽 (醇厚甘爽, chúnhòu gānshuǎng)，前几泡甜味明显，中段柔滑如丝，回甘 (回甘, huígān) 悠长。涩味极轻，余韵清净持久。
* **汤色：** 红艳明亮 (红艳明亮, hóngyàn míngliàng)，杯缘有特征性金圈 (金圈, jīnquān) — 茶黄素含量高的标志。
* **叶底 (茶渣)：** 红匀明亮 (红匀明亮, hóngyún míngliàng)，叶片肥嫩，完整芽头丰富。

**7. 化学成分：**

* **多酚类：** 发酵过程中90%以上的原始儿茶素转化为茶黄素 (茶叶「金色」色素，负责汤色亮度和金圈)、茶红素 (形成红色深度和滋味「丝滑感」) 和茶褐素。
* **胺基酸：** L-茶胺酸含量显著，提供滋味柔和甜美，并与咖啡因产生协同「平静提神」效果。
* **生物碱：** 咖啡因 (约占干重2-4%) 提供提神作用；可可碱和茶碱含量微量。
* **维生素：** 维生素C含量显著 (红茶比一般认为的更好地保存维生素C，得益于多酚类的保护作用)。
* **矿物质：** 钙、铜、钠、磷、锌和其他微量元素。
* **精油和挥发性芳香化合物：** 芳樟醇、香叶醇、苯甲醛和其他萜烯类和醛类形成特征性花果香。热量：每100克干茶少于1244千焦。

**8. 健康功效：**

* **提神效果：** 咖啡因与茶胺酸结合提供温和持久的专注力提升，无剧烈「峰值」兴奋。
* **抗氧化保护：** 茶黄素和茶红素具有显著的自由基清除能力，支持细胞健康。
* **消化支持：** 多酚氧化产物温和刺激消化酶分泌，促进舒适消化，特别是餐后。
* **心血管系统：** 适量规律饮用红茶与维持血管弹性和胆固醇水平正常化相关。
* **利尿效果：** 咖啡因与芳香物质结合温和刺激利尿。
* **温暖作用：** 红茶在中医饮食学中属于「温性」饮品；对胃部舒适，推荐寒冷季节饮用。
* **心理情绪放松：** 温暖甜美的香气和柔和滋味产生明显的「感官」镇静效果，有助缓解紧张。

**9. 冲泡方法：**

* **水温：** 标准茶品90-95°C；精致高芽头等级85-90°C。
* **用茶量：** 工夫泡法4-6克/100-120毫升；欧式泡法2-3克/200-250毫升。
* **茶具：** 瓷盖碗 (盖碗, gàiwǎn) 100-120毫升容量最佳，能完全展现香气；瓷壶或玻璃壶可视觉欣赏汤色。紧实茶品可用紫砂壶。
* **冲泡过程：**
  1. 用沸水温壶，倒掉水。
  2. 投茶，盖上盖子几秒钟，闻加热干茶香气。
  3. 洗茶 (可选)：注水1-2秒后倒掉 — 这是「唤醒」茶叶。
  4. 第一泡：5-8秒。
  5. 后续各泡：每泡增加3-5秒。
  6. 冲泡次数：优质茶品6-8泡，紧实高芽头等级可达10泡。

**10. 储存：**

密封、避光容器 (铁罐、真空铝箔袋) 储存，远离湿气、异味和直射阳光。最适温度10-25°C。最佳品质期：生产后12-24个月。一些高芽头含量的紧实茶品在适当储存2-3年后会有良好发展 — 蜂蜜和果香加深，涩味进一步柔化。

**11. 价格与仿冒：**

价格类别：中等。昭平红比著名红茶如金骏眉或祁门红茶便宜得多；基本茶品每100克几十元起，高等有机等级200-400元。影响价格因素：采摘季节 (春茶较贵)、芽头比例、有机认证和获奖地位。

* **如何避免仿冒：**
  1. 检查产地：真正的昭平红仅产于昭平县；寻找包装上的地理标志 (地理标志) 标识。
  2. 评估香气：纯净花蜜香无化学刺激、酸败或霉味。
  3. 检查汤色：应透明、红艳带金圈；混浊表示发酵或储存问题。
  4. 品尝滋味：天然甜味和清净余韵；酸味或陈腐味需警惕。
  5. 谨防低价：如果「获奖」昭平红售价明显低于市价 — 很可能是其他地区原料替代。需注意部分福建茶企从昭平采购原料制作金骏眉和正山小种，可能造成混淆。

**12. 趣闻轶事：**

* 昭平红成为该县首款红茶 — 2009年前该地区仅生产绿茶。技术突破在于将红茶发酵工艺适应绿茶品种福云六号，需要数年实验。
* 昭平县拥有「中国长寿之乡」(中国长寿之乡) 地位：百岁居民比例超过联合国「长寿区」标准两倍以上。当地茶文化与生态纯净形象密不可分。
* 昭平有机茶园认证面积超过3500亩 (约2330公顷)，按有机标准管理的茶园面积达3600亩；该县是广西最大的有机茶中心之一。
* 昭平被称为「中国大陆早春第一茶之乡」：得益于亚热带气候，采摘从2月底开始 — 比浙江早一个月。
* 除绿茶和红茶外，该县也是六堡茶 (六堡茶) 的重要产地和茉莉花茶原料供应地，使其成为广西最多元化的茶区之一。
* 民间俗语「昭平不平」(昭平不平, 「昭平一点也不平」) 巧妙运用县名中「平」(平, 「平坦」) 字的含义，反映真实地形：山地占近90%面积，几乎没有平原。
* 昭平是广西最多雨的地点之一：年平均降雨量超过2000毫米，该县位于该地区雷暴和暴雨活动中心，为形成丰富雾气和云层创造理想条件，这对嫩茶叶发育至关重要。

**13. 与其他红茶比较：**

* **英红 (英红, Yīng Hóng)** — 广东英德县红茶。用大叶种栽培种 (英红九号) 制作，茶体更厚重，麦芽香明显，「巧克力」余韵。昭平红相比更轻盈优雅，以花果香调为主。
* **祁门红茶 (祁门红茶, Qímén Hóngchá)** — 安徽著名红茶，「中国十大名茶」之一。祁门以独特的「祁门香」— 香料兰花香闻名。昭平红香气较简单，但以明显甜味、柔和和多次冲泡耐力补偿。
* **滇红 (滇红, Diān Hóng)** — 云南大叶种阿萨姆栽培种红茶。滇红茶体强劲，蜂蜜胡椒香，金毫含量高。昭平红明显更精致轻盈，茶体接近中等。
* **桂红 (桂红, Guìhóng)** — 广西红茶总称。昭平红实际上是这个群体的领先代表，为该地区红茶设定了新的品质标准，获得学者陈宗懋高度评价。
* **金骏眉 (金骏眉, Jīn Jùn Méi)** — 福建武夷山高端红茶，仅用芽头制作。有趣的是，一些福建公司从昭平采购原料制作金骏眉，间接证实了当地茶叶的高品质。然而，真正的昭平红具有自己的特色 — 更明显的螺旋卷曲和独特的地域香气特征。

**结论：**

昭平红茶是创新方法如何在具有多世纪茶叶传统的地区创造全新产品的雄辩例证。用绿茶品种按红茶工艺制作，这款茶以纯净明亮的花蜜香、精致甜味和美丽的红宝石汤色配金圈令人惊艳。昭平红非常适合寻求柔和、芳香红茶而无过度涩味的人 — 来自中国南方最生态纯净角落之一的日常而又独特的饮品。
