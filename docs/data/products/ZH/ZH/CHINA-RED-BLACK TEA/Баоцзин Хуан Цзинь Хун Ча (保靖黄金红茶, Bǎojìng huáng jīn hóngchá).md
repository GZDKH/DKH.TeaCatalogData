# 保靖黄金红茶 (保靖黄金红茶, Bǎojìng huáng jīn hóngchá)

**保靖黄金红茶 (保靖黄金红茶, Bǎojìng huáng jīn hóngchá)**

保靖黄金红茶是以传奇品种保靖黄金茶 (保靖黄金茶, Bǎojìng Huángjīn Chá) 为原料制作的红茶，这是一个来自湖南省西部湘西 (湘西) 山区深处的古老且基因独特的品种。传统上此原料用于制作绿茶，然而其极高的胺基酸含量（高达7.47%，是平均值的两倍）和多酚含量使其成为制作红茶的绝佳原料，在红茶中品种的天然甜味得到特别鲜明的展现。

**1. 分类与产地：**

* **类型：** 红茶 (红茶, hóngchá) — 完全发酵（氧化）。
* **类别：** 地方中国红茶，工夫红茶 (工夫红茶, gōngfu hóngchá)。属于「湖南红茶」(湖南红茶, 「湖南红茶」) 系列，为此制定了专门的技术规程——「保靖黄金茶工夫红茶技术规程」(《保靖黄金茶工夫红茶技术规程》, 「保靖黄金茶工夫红茶技术规程」)。
* **产地：** 中国湖南省 (湖南省, Húnán Shěng)，湘西土家族苗族自治州 (湘西土家族苗族自治州, Xiāngxī Tǔjiāzú Miáozú Zìzhìzhōu)，保靖县 (保靖县, Bǎojìng Xiàn)。核心产区——葫芦镇 (葫芦镇, Húlu Zhèn)、夯沙乡 (夯沙乡, Hāngshā Xiāng) 和水田河镇 (水田河镇, Shuǐtiánhé Zhèn)。历史「零公里」——黄金村 (黄金村, Huángjīn Cūn)，位于吕洞山 (吕洞山, Lǚdòng Shān) 山脚。
* **地理座标：** 东经109°12′–109°33′，北纬28°24′–28°36′（根据地理标志登记资料）。

**2. 历史与文化意义：**

* **历史：** 保靖的茶叶传统可追溯至远古时代。早期史书《荆州土地记》(《荆州土地记》) 中记载：「武陵七县通出茶，最好」(武陵七县通出茶，最好)。唐代学者杜佑 (杜佑) 在百科全书式著作《通典》(《通典》, 西元801年) 中记录，溪州 (溪州, 大致相当于现今保靖地区) 进贡茶芽作为贡品。更早的证据——里耶 (里耶) 的发现，在36,000片秦简 (秦简, 西元前三世纪) 中发现了来自迁陵 (迁陵, 现今保靖) 的经济运输记录，推测包括茶叶原料。

关键传说：西元1539年（明嘉靖十八年, 明嘉靖十八年），总督陆杰 (陆杰, Lù Jié) 巡视湘西山区边防哨所时，他和随从在鲁旗 (鲁旗, 现今葫芦) 附近的偏僻峡谷中感染了瘴气 (瘴气, zhàngqì)。一位向姓 (向) 苗族老妇救了虚弱的士兵：她用自家门前百年茶树的叶子泡茶给病人喝。半小时后热病退去。陆杰感激地给老妇一两 (两) 黄金，并将此茶列入宫廷贡品。从此民间流传：「一两黄金一两茶」(一两黄金一两茶)。村庄得名黄金寨 (黄金寨, 「黄金要塞」)，茶叶被称为黄金茶 (黄金茶, 「黄金茶」)。

近代史：1993–1994年农艺师张湘生 (张湘生, Zhāng Xiāngshēng) 实现突破——首次成功以扦插法 (扦插, qiānchā) 繁殖黄金茶，克服了多年的繁殖「瓶颈」。这为品种的大规模推广开辟了道路。2009年文化遗产普查中发现2057棵古茶树（树干周长超过30公分），其中718棵可追溯至明代，1339棵至清代。最古老的——「保靖黄金茶树王」(保靖黄金茶树王) ——超过400年。2010年该茶获得中华人民共和国农业部农产品地理标志 (农产品地理标志) 保护。2020年黄金茶园被列入中国重要农业文化遗产 (中国重要农业文化遗产) 第五批名录。同年品牌纳入《中欧地理标志协定》(中欧地理标志协定)，在27个国家获得国际法律保护。2025年保靖黄金茶1号 (保靖黄金茶1号) 品种被列入国家骨干型茶树品种 (国家骨干型茶树品种) 名录。

红茶版本——黄金红茶——在2010年代与绿茶版本并行积极发展，当时工夫红茶技术在省级标准中正式化。

* **名称：** 保靖 (Bǎojìng) — 县名，字面意思「受保护的安宁」；黄金 (Huáng Jīn) — 「黄金」，指陆杰为神奇茶叶付出一两黄金的传说；红茶 (Hóngchá) — 红茶，指加工方式。

* **文化意义：** 保靖黄金茶不仅是农业文化，更是湘西苗族和土家族文化的活遗产。黄金村古茶园被宣布为省级文化遗产保护对象，「茶树王」本身是湖南唯一的活体非物质文化遗产 (活体非物质文化遗产)。当地苗族尊崇「茶花仙子」(茶花仙子) 为守护神，茶文化主题贯穿传统苗鼓 (苗鼓) 节奏和歌谣创作 (苗歌)。对保靖县而言茶叶是主导经济产业：截至2023年种植面积达15.5万亩（≈ 10,300公顷），茶产业综合产值231.6亿元，品牌价值评估为409.3亿元。

**3. 植物学描述与原料：**

* **品种/栽培种：** 保靖黄金茶 (保靖黄金茶) — 独特的地方品种，在湘西偏僻山区条件下经长期自然选择形成。属于群体种 (群体种)，具有高度遗传多样性。根据何士华 (何士华) 教授研究，黄金茶古树属于乔木型大叶类品种 (乔木型大叶类品种)，与3500万年前的景谷宽叶木兰 (景谷宽叶木兰, *Magnolia latifolia*) 化石显示一定遗传亲缘关系。中大型灌木，叶片椭圆形或披针形，嫩枝柔软，芽密度高，对不良条件抗性强。关键植物学特征——嫩芽中胺基酸含量极高。
* **采摘：** 春季为主要季节：明前 (明前, míngqián) 早春采摘特别珍贵。黄金茶特点是发芽早 (发芽早) 且芽叶密集、整齐。红茶也使用夏季原料，多酚含量更高。
* **采摘标准：** 一芽一叶至一芽二叶 (一芽一叶 — 一芽二叶)。顶级批次使用单芽 (单芽)：制作一斤（500克）需要32,000–35,000次手工采摘。
* **原料要求：** 完整、嫩绿、新鲜的叶片，无损伤。手工采摘遵循「十不采」(十不采) 原则，包括拒绝采摘虫害、过熟、露水湿润等不合标准的嫩芽。

**4. 风土条件与栽培特点：**

* **海拔高度：** 海拔280–1500公尺。主要优质原料区——500–800公尺。黄金村位于海拔约635公尺。
* **气候：** 亚热带季风山地气候。年均温15–17°C，夏季温和无极端炎热，冬季无严寒。云雾几乎是常伴：该地区位于云贵高原与武当山交汇处。年降雨量1200–1500毫米。显著的日夜温差促进芳香化合物和胺基酸积累。
* **土壤：** 形成于母岩——页岩和砂岩 (板页岩和砂岩)。弱酸性（pH 4.5–5.5），排水良好，深厚，有机质含量高。矿物组成——武陵山 (武陵山) 数亿年地质过程的结果——为特有的风味「深度」做出贡献。
* **生态：** 湘西地区生态环境极其纯净：远离工业中心，森林覆盖率高，山泉清澈。保靖茶园通过有机认证面积超过4200亩。许多种植园毗邻原始森林，形成天然农林系统。

**5. 制作工艺：**

按工夫红茶标准生产，并根据黄金茶原料高胺基酸含量特点进行调整：

* **采摘 (采摘, cǎizhāi)：** 手工选摘嫩芽。采摘时间——露水散去后的上午时段。
* **萎凋 (萎凋, wěidiāo)：** 自然或复合式。持续12–20小时，失水35–40%。目的——为揉捻准备叶片，启动初步芳香转化。对于高胺基酸含量的黄金茶，萎凋过程温和，温度适中，以保持甜味基础。
* **揉捻 (揉捻, róuniǎn)：** 形成紧密卷曲并破坏细胞膜使汁液渗出到叶面。可手工揉捻 (手工揉捻) 或机械揉捻。
* **发酵/氧化 (发酵, fājiào)：** 关键步骤。温度26–30°C，湿度90–95%，持续3–5小时。由于黄金茶原料多酚含量高（达25%）和胺基酸含量高（达7.47%），创造理想平衡：茶黄素提供亮度和活力，茶红素提供深度和「体感」，未氧化胺基酸提供明显天然甜味。
* **干燥 (烘干, hōnggān / 干燥, gānzào)：** 初步干燥在100–120°C停止氧化，然后在较温和温度（60–80°C）最终干燥以稳定香气特征。
* **分级 (分级, fēnjí)：** 按叶片标准、芽头比例、颗粒大小分类。

**6. 感官特征：**

* **干茶外观：** 紧密卷曲，整齐的条索 (条索紧细)，深棕色或黑色，金毫 (金毫) 丰富。叶片均匀，分级良好。
* **干茶香气：** 甜美、蜂蜜味，带有成熟南方水果香调和淡雅花香背景。表现出特有的「黄金香」(黄金香) ——复杂的甜蜜蜂蜜香气，是品种的「名片」。
* **茶汤香气：** 多层次且持久：顶调——蜂蜜和干果（杏干、枣子）；中调——焦糖、饼干、麦芽；底调——精致的木质香料调。每次冲泡香气都在演变，展现新的层面。
* **滋味：** 浓厚、饱满同时惊人地柔和。主导纯净、深沉的甜味 (回甘)，源于原料中创纪录的胺基酸含量。涩味极少，快速消散于持久温暖的回甘中。口感「油润」(油润)，包覆性强。
* **汤色：** 红琥珀色，明亮透澈，有明显的金圈 (金圈, jīn quān) ——茶黄素含量高的标志。
* **叶底（冲泡后叶片）：** 叶片舒展有弹性且均匀；颜色——铜棕色至红栗色。结构完整，芽叶清晰可辨。

**7. 化学成分：**

* **胺基酸：** 黄金茶的主要「超能力」。新鲜春季原料中游离胺基酸含量达7.47%（某些资料显示达7.76%），是普通绿茶的两倍。以L-茶胺酸为主。完全氧化后大部分胺基酸得以保留，确保红茶的特殊甜味和「鲜味」。
* **多酚：** 新鲜原料含量约20–25%。红茶中以氧化形式为主：茶黄素（TF）和茶红素（TR），形成汤色和「体感」。TF/TR平衡决定特有的「活泼」明亮度和同时的风味深度。
* **水溶性物质：** 高达50% — 极高指标，解释了多次冲泡时汤色的浓郁和持久。
* **生物碱：** 咖啡因 — 4.3%（原生原料资料）。可可碱和茶碱 — 微量。
* **叶绿素：** 含量比对照品种高50%，有助于提高光合作用活性，间接促进次级代谢物积累。
* **挥发性芳香化合物：** 丰富的萜烯、醛类和美拉德反应产物复合体。特有的「黄金香」由芳樟醇、香叶醇、β-紫罗兰酮和特定胺基酸衍生物组合形成。
* **维生素和矿物质：** B群维生素、抗坏血酸（部分）、钾、镁、锰、锌、硒。

**8. 健康功效：**

* **温和提神与认知支持：** L-茶胺酸与咖啡因的高含量组合提供「放松专注」状态——清醒无焦虑，改善记忆和注意力。
* **强效抗氧化保护：** 茶黄素、茶红素和残留儿茶素具有显著中和自由基和降低氧化压力的能力。就抗氧化活性而言，高多酚黄金茶原料制成的红茶在中国红茶中占据高位。
* **代谢支持：** 多酚化合物和咖啡因刺激产热并促进脂肪分解，使茶成为体重管理的良好伴侣。
* **消化舒适：** 温和的红茶传统上建议餐后饮用。适度的单宁物质有助于胃液分泌正常化。
* **心血管健康：** 研究显示茶黄素能够支持血管弹性和血脂正常化。
* **免疫力增强：** 多酚具有适度抗菌和免疫调节作用。
* **延缓细胞老化：** 茶多酚的高抗氧化特性在研究中与延缓细胞年龄相关变化有关。
* **温暖效应：** 完全氧化的茶在寒冷季节温暖身体，缓解主观疲劳感。

**9. 冲泡方法：**

* **水温：** 90–95°C。
* **茶量：** 4–5克配100–120毫升。顶级芽茶批次稍少（3–4克），因为物质浓度更高。
* **茶具：** 瓷盖碗 (盖碗) — 最精确展现香气特征；宜兴紫砂壶 (宜兴紫砂壶) — 更加圆润油滑；玻璃茶壶 — 欣赏金芽舒展「舞蹈」的美学享受。
* **过程：**
1. 用沸水温热茶具并倒掉水。
2. 投茶，盖上盖子轻摇 — 闻温热茶叶的「黄金」香气。
3. 洗茶非必需；如需要 — 短暂（1–2秒）过水。
4. 第一泡：8–10秒。从第一杯就能感受到特有的蜂蜜果香甜味。
5. 2–4泡：10–15秒。
6. 从第5泡开始：增加5–10秒。
7. 优质批次可承受7–9次完整冲泡，以稳定性和「耐泡」(耐冲泡) 著称 — 品种公认优点之一。

**10. 储存：**

* 密封不透明容器 — 金属罐、真空铝箔袋、陶瓷容器。
* 干燥、阴暗、凉爽处（15–25°C，湿度低于60%），远离异味。
* 最佳期限 — 6–18个月保持鲜活度。良好干燥的批次能在2–3年内温和「圆润化」。
* 避免潮湿、直射光、温度变化和与芳香食品邻近。

**11. 价格与仿冒：**

* **价格类别：** 保靖黄金红茶是顶级地方产品。价格由采摘标准（芽茶批次最贵）、季节（早春原料最受重视）、来源（古树vs.年轻种植园）和获奖证书决定。俗语「一两黄金一两茶」反映了这种茶作为奢侈品的历史认知，尽管现代市场价格当然远非字面意思的「一两黄金换一两茶」。
* **如何避免仿冒：**
1. 向可追溯至保靖具体农场的可信卖家购买。注意地理标志 (地理标志) 标识。
2. 评估外观：均匀、紧密卷曲，金毫丰富，无粉尘和杂质。
3. 检查香气：应该纯净、蜂蜜甜香，带特有「黄金」调。无焦糊、霉味、鱼腥味。
4. 评估汤色：明亮、红琥珀色、透澈，有明显金圈。混浊是警告信号。
5. 测试「耐泡性」：正宗黄金茶以多次冲泡耐久著称。如果3–4泡后滋味急剧「衰落」 — 可能表示原料替换。

**12. 有趣事实：**

* 保靖黄金茶被称为「可以喝的文物」(可以拿来喝的文物, 「可以拿来喝的文物」)：黄金村保存著2057棵古茶树，最古老的超过400年。七个历史茶园 — 龙颈坳 (龙颈坳)、格者麦 (格者麦)、德让拱 (德让拱)、库鲁 (库鲁)、团田 (团田)、冷寨河 (冷寨河) 和夯纳乌 (夯纳乌) — 作为露天活博物馆保存。

* 「黄金茶之母」— 农艺师张湘生 (张湘生)，文化大革命后第一批大学毕业生。从1993年起她专注於单一任务 — 学会扦插繁殖黄金茶。1994年实验成功：3.16亩幼苗（约500,000株）开启了品种在整个湘西的大规模推广。

* 院士刘仲华 (刘仲华, Liú Zhōnghuá)，世界领先茶科学专家之一和中华人民共和国国家奖获得者，公开赞赏黄金茶并亲自每年采购超过100公斤茶叶。据他说，他从未见过胺基酸含量如此高的茶树品种。

* 保靖位于北纬28° — 所谓「茶叶生产黄金带」。县境与著名国家公园「悬浮山」张家界 (张家界, Zhāngjiājiè) 和作家沈从文故乡「凤凰古城」凤凰 (凤凰, Fènghuáng, 「凤凰城」) 接壤。

* 保靖茶不仅生产红茶和绿茶版本：当地师傅尝试用同样原料进行黑茶、乌龙茶、白茶和黄茶加工。年产量（2020年资料）：绿茶 — 581吨，红茶 — 309吨，黑茶 — 270吨，整个茶产业链综合价值 — 125亿元。

**13. 与其他红茶比较：**

* **古丈毛尖红茶 (古丈毛尖红茶)：** 来自邻近古丈县 (古丈) 的红茶，同样位于湘西。以著名绿茶古丈毛尖原料制作。结构上是「亲缘」产品，但没有黄金茶创纪录的胺基酸含量。古丈红茶通常稍微「干燥」，甜味较少。

* **湖南红茶 (湖南红茶, 总体类别)：** 涵盖湖南省红茶的伞形品牌，包括湖红 (湖红, Hú Hóng)。传统湖南红茶使用标准品种制作，具有更均匀的「工业」特征。黄金红茶以独特品种特性带来的特殊甜味和「油润」质感脱颖而出。

* **滇红 (滇红, Diān Hóng)：** 云南大叶种阿萨姆型红茶。滇红 — 强劲、饱满，带明显巧克力、胡椒和麝香调。黄金红茶 — 更柔和、更甜，香气特征更精致，胺基酸含量显著更高。

* **金骏眉 (金骏眉, Jīn Jùn Méi)：** 武夷山（福建）顶级红茶，用野生茶树纯芽制作。两种茶都是芽茶，蜂蜜甜味，金黄汤色。差异在于风土：金骏眉带有武夷山「岩韵」矿物调，黄金红茶带有湘西果香深度和特有「黄金」甜味。

**结论：**

保靖黄金红茶是具有特殊血统的红茶。背后有四个多世纪古茶树的生命，一两黄金的传说，胺基酸含量创纪录的独特品种，湘西山区及其苗族土家族文化和生态原始性。在茶杯中它提供了大多数红茶难以同时达到的：深沉蜂蜜甜味而无丝毫腻味，饱满「油润」体感，持久的黄金干果香调 — 以及多次冲泡的惊人「耐久性」，这正是黄金茶获得「黄金」声誉的原因。
