# 台湾烟小种红茶 (台湾烟小种红茶, Táiwān yān xiǎozhǒng hóngchá)

**台湾烟小种红茶 (台湾烟小种红茶, Táiwān yān xiǎozhǒng hóngchá)**

台湾烟小种是台湾对著名烟熏红茶拉普山小种的诠释，在国际贸易中以Tarry Lapsang Souchong之名闻名。与福建原产地茶品不同之处在于强烈的热熏制程并添加松脂，使用大叶种阿萨姆茶菁，以及鲜明的树脂烟熏特色，因而获得「tarry」——「树脂状、焦油状」的称谓。

**1. 分类与产地:**

* **类型：** 红茶 (红茶, Hóngchá) — 完全氧化，经过松木热熏 (烟熏, Yānxūn) 并添加松脂处理。属于烟熏茶类 (烟茶, Yānchá)。
* **类别：** 台湾烟熏红茶顶级品。为福建正山小种 (正山小种, Zhèngshān Xiǎozhǒng) 技术的地方改良版本。
* **产地：** 名间乡 (名间乡, Míngjiān Xiāng)，南投县 (南投县, Nántóu Xiàn)，台湾中部。现代「tarry」形式（含树脂烟熏）于1980年代形成。
* **地理座标：** 约北纬23°54'，东经120°41'。
* **别名：** Tarry Lapsang Souchong (英)、Formosa Lapsang (英)、台湾拉普山小种、Lapsang Souchong Crocodile (部分市场商品名)。

**2. 历史与文化意义:**

**历史。** 所有烟熏茶的始祖——福建正山小种 (正山小种) ——于十七世纪在武夷山区创制，成为最早进入欧洲的中国茶之一，由荷兰商人引进。根据历史记载，这款茶在欧洲贵族中赢得众多爱好者；据知温斯顿·邱吉尔也是其爱好者之一。

台湾烟熏茶的生产始于日治时期 (1895–1945)，当时岛上积极发展茶业出口。然而，特殊的「tarry」版本——强烈烟熏并添加松脂——是在更晚的1980年代开发，以回应欧洲市场对更浓郁烟熏风味的需求。生产集中在南投县名间乡——一个具有悠久茶业传统且能取得台湾特有松树的地区。

**名称。** 「烟」(烟) — 「烟熏」。「小种」(小种) — 「小品种」，历史称谓，源自福州方言，其中「拉桑」(拉桑) 意指「松木」，而「小种」指特定的叶型。在国际贸易中，「tarry」(英语「树脂状、焦油状」) 特指台湾版本的强烈烟熏，有别于更精致的福建原版。

**文化意义。** 台湾烟小种在当地茶文化中占有特殊地位。在台湾客家 (客家, Kèjiā) 传统中，烟熏茶用于祭祖仪式，并在家庭庆典中作为世代传承的象征。在名间乡，松木烟熏茶的技艺被视为珍贵的手工艺，在制茶家族内部传承。掌握「tarry」版本完整制程——从木材选择到烟熏温度控制及松脂添加——的茶农极少，使这款茶成为真正的手工艺品。在国际市场上，台湾Tarry Lapsang始终以鲜明、毫不妥协的烟熏茶代表而引起关注。

**3. 植物学描述与原料:**

* **品种：** 使用基于阿萨姆变种 (*Camellia sinensis* var. *assamica*) 的杂交品种，特别是茶业改良场的台湾选育品系。阿萨姆亚种产生大型、厚实的叶片，最适合强烈烟熏处理。
* **茶树描述：** 高大茶树 (高达2公尺)，具有大型、厚实的倒卵形叶片，长10–15公分，浅绿色。嫩芽沿中脉可能有轻微茸毛。
* **采摘：** 主要在夏季进行，此时叶片中多酚含量最高，确保足够的风味浓度以抵抗强烈烟熏。允许机械采摘。
* **原料要求：** 「tarry」版本选用成熟的大叶片——芽尖下第三、四对叶片 (所谓「老小种」)。较粗老厚实的叶片更能承受长时间烟熏，并更积极吸收烟熏香气化合物。

**4. 风土条件与栽培特色:**

* **产区：** 名间乡 (名间乡)，南投县，台湾中部。名间位于中央山脉西侧丘陵。
* **海拔高度：** 茶园位于中等海拔——海拔200–500公尺。
* **土壤：** 浊水溪 (浊水溪, Zhuóshuǐ Xī) 沿岸冲积土，富含矿物质，包括来自玉山 (玉山) 山脉的石英颗粒。土壤酸碱值pH 4.8–5.2。
* **气候：** 亚热带湿润气候。年平均温度22–25°C。年降雨量1500–2000毫米。日夜温差显著，白天受热与夜间山区冷却形成的温差有助于茶叶香气物质积累。
* **特有松树：** 烟熏使用两种台湾特有松树：*Pinus morrisonicola* (台湾五叶松, 台湾五叶松) 和 *Pinus taiwanensis* (台湾二叶松, 台湾二叶松)。其木材和树脂具有特殊香气特征，不同于福建马尾松 (*Pinus massoniana*)，这决定了台湾拉普山小种的独特特色。

**5. 制作工艺:**

台湾烟小种的生产是复杂的多阶段过程，与福建原版的关键差异在于热熏 (热熏, Rèxūn) 并添加松脂：

* **萎凋 (萎凋, Wěidiāo)：** 采摘的叶片摊放进行初步失水。特色：萎凋在新砍伐台湾五叶松 (*Pinus morrisonicola*) 冷烟上进行，温度约60°C，在此阶段就开始形成烟熏风味。
* **揉捻 (揉捻, Róuniǎn)：** 萎凋后的叶片手工或机械揉捻，破坏细胞壁并释放细胞液。
* **发酵 (发酵, Fājiào)：** 茶叶在控制条件下完全氧化。此阶段形成茶黄素和茶红素，构成红茶风味和色泽的基础。
* **杀青 (杀青, Shāqīng)：** 在锅中快速炒制 (传统方法) 以停止氧化过程。
* **造型：** 茶叶可塑造成各种形状，包括小颗粒。
* **热熏 (热熏, Rèxūn)：** 主要决定性阶段。茶叶在高温 (约110°C) 下于闷烧松木上烟熏。「tarry」版本此阶段约持续9小时——比福建版本 (~6小时) 显著更长。
* **加树脂 (加树脂, Jiā Shùzhī)：** 在热熏阶段添加松脂 (松香)，燃烧时释放强烈香气化合物，赋予茶叶特征性「tarry」——树脂焦油——风味。这是与传统福建正山小种的关键技术差异，后者不添加树脂。
* **最终干燥 (干燥, Gānzào)：** 水分含量降至最低——少于3%，确保长期储存稳定性。
* **陈化 (陈化, Chénhuà)：** 成品茶陈放至少三个月以稳定和协调香气。期间最激烈的烟熏味调会软化，风味变得更圆润。

**6. 感官特征:**

* **干茶外观：** 大型、卷曲的深棕色或黑色叶片，有时呈颗粒状。可能有轻微油亮光泽，由树脂燃烧产物沉积所致。
* **干茶香气：** 强烈、主导性的烟熏、松脂、焦油和烟熏味。联想到松木篝火、烟熏火腿、冒烟壁炉。
* **茶汤香气：** 强烈、多层次：基调为烟熏树脂味，带有愈创木酚 (烟熏木质) 和杂酚油气息。展开时可能出现意外的甜味调，如干龙眼、香草和黑焦糖。
* **滋味：** 浓厚、饱满、油润。烟熏味主导，伴随轻微天然甜味和深沉温暖的回甘。涩味适中。茶汤体感——饱满、厚实，质地如天鹅绒。品茶师注意到黑糖、焦糖、雪松、肉桂和烤大麦的味调。
* **汤色：** 浓郁深红色、红宝石色或白兰地色，具有深沉温暖感。
* **叶底 (冲泡后茶叶)：** 大型、厚实的深棕色叶片。由于强烈处理，展开不完全。

**7. 化学成分:**

化学特征由完全氧化茶的标准成分和烟熏特有产物决定：

* **多酚类：** 总含量高。儿茶素氧化产物——**茶黄素**——浓度比传统福建拉普山小种更高，确保茶汤的鲜活明亮。
* **烟熏挥发性香气化合物：** 关键标记物：**愈创木酚** (2-甲氧基苯酚) — 主要烟熏木质调；**杂酚油** — 深沉树脂调；**长叶烯** — 来自松木的倍半萜烯，台湾版本特有。与福建版本不同，可能缺少甲基胡椒酚 (草蒿脑)。
* **咖啡因：** 含量符合阿萨姆亚种红茶典型水准——约3.5–4.5%，提供明显提神效果。
* **茶红素：** 形成茶汤体感和深色。高浓度由阿萨姆大叶种完全氧化所致。
* **矿物质：** 钾、镁、钙、磷、铁。矿物质特征因名间地区冲积土而丰富。
* **维生素：** B₁、B₂、PP — 适量。

**8. 保健功效:**

* **抗氧化活性：** 高多酚和茶黄素含量提供细胞保护，免受自由基损害。
* **提神效果：** 显著的咖啡因含量刺激中枢神经系统，提高专注力和精神状态。这款茶传统上被认为是良好的晨间饮品。
* **温暖作用：** 由于完全氧化、厚实体感和烟熏特性，传统上推荐在寒冷季节饮用。在台湾民间医学中，烟熏茶被认为具有「温性」(温性, Wēnxìng)。
* **抗炎效果：** 一些研究指出红茶多酚具有抗炎特性，能抑制特定促炎酶。
* **消化支持：** 红茶传统上用于改善消化，特别是在重油腻食物后。烟熏茶浓厚油润的质地与肉类菜肴搭配良好。
* **心血管支持：** 茶黄素有助于降低胆固醇水准并维持血管弹性。

*注意：台湾烟小种特定保健功效的研究有限；大多数资料基于红茶整体研究。*

**禁忌与注意事项。** 支气管哮喘或呼吸道高敏感者应谨慎：烟熏挥发性成分 (愈创木酚) 在吸入热茶蒸气时可能引发支气管痉挛。由于咖啡因含量相对较高，不建议孕妇、哺乳期妇女大量饮用，12岁以下儿童也不宜。可能与香豆素类抗凝血剂 (华法林) 产生交互作用。

**9. 冲泡方法:**

* **水温：** 95–100°C。需要沸水才能完全展现烟熏茶浓厚的烟熏风味和香气。
* **茶量：** 5–7公克配150–200毫升水 (快冲法) 或3–4公克配250–300毫升 (欧式冲泡法)。
* **茶具：** 宜兴紫砂壶 (宜兴紫砂壶) — 绝佳选择，因为多孔陶土会吸收并随时间回馈烟熏香气，增强风味深度。瓷盖碗或茶壶也适合。建议为烟熏茶专用一把宜兴壶，避免烟熏香气「污染」其他茶类。
* **冲泡过程 (快冲法)：**
    1. 用沸水温壶，倒掉水。
    2. 投入干茶。可进行快速洗茶 (注水立即倒出) 唤醒茶叶。
    3. 第一泡：注入沸水，浸泡45–60秒。
    4. 第二泡及后续：逐渐增加时间——60秒、75秒、90秒。
    5. 茶叶可冲泡3–5次，保持特征烟熏香气。
* **欧式冲泡法：** 3–4公克配250–300毫升沸水，浸泡3–5分钟。大多数品茶师推荐此法为这类茶的最佳冲泡方式——大叶碎茶快速释放风味，无需多次冲泡。
* **补充：** 茶与牛奶搭配良好——其强烈烟熏特性即使在奶茶中也不会失去。也用于烹饪：作为高汤、酱汁和腌料的基底。

**10. 储存:**

烟熏茶具有强烈自身香气，在储存时既有优势也有限制：

* **容器：** 密封、不透光容器——必要条件。密封金属罐、陶瓷茶罐或真空包装。重要的是既要防止烟熏香气散失，也要避免传递给邻近产品。
* **温度和湿度：** 干燥、凉爽处，温度不超过15°C，相对湿度低于35%。避免剧烈温度变化。
* **光线：** 避免直射阳光。
* **氧气：** 长期储存最好使用真空包装或脱氧剂。
* **保存期限：** 正确储存下茶叶可长期保持品质——3至5年以上。陈年样品 (10年以上) 可获得额外复杂性和柔和度，受收藏家珍视。

**11. 价格与仿冒品:**

* **价格等级：** 正宗台湾烟小种是顶级茶品，因烟熏过程复杂手工、使用特殊原料、台湾特有松树资源有限及产量小。零售价约28–45美元/100公克优质产品。
* **仿制品：** 其他地区生产 (包括大陆福建) 仿台湾风格的茶，或用「液体烟熏」(人工香料) 而非天然烟熏处理的红茶，价格显著较低——12–18美元/100公克。
* **如何避免仿冒品：**
    * 向有确认声誉的台湾茶专业供应商购买。
    * 注意价格：可疑的低价是仿冒品第一征象。
    * 评估香气：正宗茶具有复杂、深沉、多层次的烟熏树脂香气。仿冒品气味刺鼻、单调，带化学或「平板」调性。
    * 化学标记：正宗台湾烟小种含有特征性倍半萜烯长叶烯和特殊愈创木酚特征，不同于福建版本。实验室方法 (FTIR光谱) 可精确确定产地。

**12. 趣闻轶事:**

* **间谍与航海者之茶。** 拉普山小种的强烈香气历史上用于掩盖其他气味。据一种说法，俄国商人在这种茶的箱子里运输贵重毛皮，其烟熏味驱赶蛀虫并掩盖毛皮气味。
* **收藏样品。** 1970年代陈年台湾烟小种在专业茶叶拍卖会上以可观价格售出——其烟熏特征经过数十年转化为高雅、复杂的组合，带有陈年威士忌和老木头的调性。
* **生态依赖性。** 这款茶的生产严重依赖台湾特有松树——*Pinus taiwanensis* 和 *Pinus morrisonicola* 的存在。这种生产的长期可持续性与保护台湾天然松林相关，使这款茶作为与当地生态系统密不可分的产品具有额外价值。
* **波士顿茶叶事件。** 在著名的1773年「波士顿茶叶事件」中，东印度公司被毁货物中有35箱小种茶——现代拉普山小种的前身。因此，烟熏茶在美国独立斗争中发挥了微小作用。
* **后发酵烟熏。** 与许多烟熏产品 (烟熏鱼、肉类) 在生鲜或部分处理产品上烟熏不同，台湾烟小种在完全发酵后烟熏——这称为「后发酵烟熏」(post-oxidation smoking)。这种方法允许完全发展红茶基本风味，然后用烟熏调性丰富，而不压制茶叶本身特性。

**13. 与其他烟熏茶和红茶比较:**

* **正山小种 (正山小种, Zhèngshān Xiǎozhǒng)：** 来自武夷山桐木关 (桐木关) 地区的原始福建拉普山小种。使用当地品种原料 (*var. sinensis*，即「武夷」)。烟熏更精致——在闷烧松木上不添加树脂。氧化度更高 (~92%)，烟熏时间更短 (~6小时)。风味更优雅：龙眼、干果、淡烟熏。经典正山小种是优雅；台湾Tarry是力量。
* **Formosa Lapsang (台湾拉普山小种，无「tarry」标示)：** 台湾烟熏茶的通称，烟熏较温和，不添加树脂。氧化度可能较低 (~78%)，烟熏时间较短 (~4小时)。风味更柔和，花香和果香更明显。
* **祁门红茶 (祁门红茶, Qímén Hóngchá)：** 安徽著名红茶。不经烟熏。风味——精致、兰花香，带干果调性和淡烟熏味。比较突显烟熏过程如何转化红茶基本特性。
* **无烟熏正山小种 (新式正山小种)：** 来自桐木关村的现代无烟熏版本，称为「金骏眉」(金骏眉, Jīn Jùn Méi) 及相关品种。与「tarry」版本完全相反——温和、花蜜香，无烟熏调性。展现同一传统的两个极端。

## 结语

台湾烟小种是为追求极致浓郁、深度和特色者而生的茶品。这不是悠闲午后茶时光的精致饮品，而是强劲、富有魅力的茶，从第一次闻香就宣告其存在。优质阿萨姆原料、名间特殊风土条件及其冲积土壤，加上独特的台湾松脂热熏技术的结合，创造出具有难忘烟熏树脂特性、温暖滋味和丰富历史的饮品。这款茶是寒冷夜晚、重乳酪拼盘和烟熏肉类美食的绝佳伴侣，也是烹饪和调酒中意外有趣的食材。
