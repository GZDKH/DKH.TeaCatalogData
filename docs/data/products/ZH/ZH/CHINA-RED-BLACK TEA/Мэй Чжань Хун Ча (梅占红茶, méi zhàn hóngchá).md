# 梅占红茶 (梅占红茶, méi zhàn hóngchá)

**梅占红茶 (梅占红茶, méi zhàn hóngchá)**
![Мэй Чжань Хун Ча (梅占红茶, méi zhàn hóngchá)](images/Мэй%20Чжань%20Хун%20Ча%20(梅占红茶,%20Méi%20Zhàn%20Hóngchá).png)

![Мэй Чжань Хун Ча (梅占红茶, méi zhàn hóngchá)-1](images/Мэй%20Чжань%20Хун%20Ча%20(梅占红茶,%20Méi%20Zhàn%20Hóngchá)-1.png)

梅占红茶是采用中国最具多用途性的品种之一——梅占 (梅占) 「梅花夺魁」制作的红茶。这个来自安溪的传奇品种是「安溪六大名茶」(安溪六大名茶) 之一，以「茶界变色龙」闻名：可制作出优质的乌龙茶、红茶、绿茶和白茶。然而，正是在红茶的表现中，梅占展现了其最亮丽的一面——高雅的「兰花香」(兰花香) 和深厚、浓郁、甜润带辛香的滋味，带有梅花的韵味。

**1. 分类与产地：**

* **类型：** 红茶 (红茶, hóngchá)，全发酵茶。
* **类别：** 品种红茶／地方红茶，采用梅占品种制作。也可归类为高等级白琳工夫 (白琳工夫) 的原料。
* **产地：** 梅占品种原产于福建省安溪县 (安溪县, Ānxī Xiàn)——具体来自芦田镇 (芦田镇, Lútián Zhèn) 三洋村 (三洋村, Sānyáng Cūn)，银瓶山。作为红茶，梅占主要在两个地区生产：**武夷山 (武夷山)** ——用于制作高级红茶（包括「金梅冰芽」——金梅冰芽）；**福鼎 (福鼎)** ——传统上作为白琳工夫的优质原料；以及安溪本地和福建其他产茶区。
* **地理座标：** 约北纬25°03′，东经117°57′（安溪，品种发源地）。

**2. 历史与文化意义：**

* **历史：** 梅占品种有超过200年的历史。关于其起源有两个主要传说。第一个：约1810年（嘉庆十五年）三洋村农民杨奕糖 (杨奕糖) 遇到一位带著茶苗的流浪老人，用粥招待了他；老人感激之余赠送了三株茶苗。杨将它们种在「玉树厝」(玉树厝) 旁，两三年后茶树繁茂。当地举人 (举人) 杨飞文 (杨飞文) 见这种茶的花叶如梅花瓣般绽放，散发腊梅香气 (腊梅)，便命名为「梅占」——「梅花夺魁」。第二个传说：1821年（道光元年）西坪王氏族人在芦田见到一棵无名茶树，不知其名，注意到门楣上的题字「梅占百花魁」（「梅花在百花中称王」）——于是取名「梅占」。

  1960年代，梅占开始在中国各产茶省份推广。1985年，国家农作物品种审定委员会授予其**国家品种** (国家品种, GS13004-1985) 地位。在铁观音热潮期间（1990-2000年代），许多安溪茶农砍掉梅占，改种铁观音。幸存的老树 (老枞, lǎo cóng)——50-100+年——成为制作高级红茶的珍贵资源。

* **名称：** 「梅」(梅) ——梅花；「占」(占) ——「占据」、「夺取首位」。完整含义：「梅花在百花中夺魁」(梅占百花魁) ——指这种茶树的花朵像梅花花瓣，香气如腊梅。

* **文化意义：** 梅占是「安溪六大名茶」(安溪六大名茶) 之一，与铁观音、黄金桂、本山、毛蟹、大叶乌龙并列。清末「福建八大诗人」之一、芦田人林鹤年 (林鹤年) 写道：「种梅三万株，终老吾何悔」(种梅三万株，终老吾何悔)。

**3. 植物学描述与原料：**

* **品种／栽培种：** 梅占 (梅占)，又称大叶梅占 (大叶梅占，「大叶梅占」)、高脚乌龙 (高脚乌龙，「高脚乌龙」)。国家品种（GS13004-1985），*Camellia sinensis*。无性系 (无性系)，小乔木型 (小乔木型)，中叶类 (中叶类)，中芽种 (中芽种)。植株高大（可达1.6公尺），主干直立，主枝明显，节间较长。叶片——长椭圆形，深绿色，厚实，边缘内卷。梅占是适应性冠军：「百变茶青」(百变茶青)。适合制作乌龙茶、红茶、绿茶和白茶。
* **原料化学成分（春茶，一芽二叶）：** 胺基酸——3.6%；茶多酚——27.5%；儿茶素——18.1%；咖啡因——4.4%。
* **采摘：** 春茶（最珍贵）、夏茶、秋茶。四月中旬——春茶采摘高峰期。手工采摘。
* **采摘标准：** 一芽一至二叶用于高级等级（包括「金梅」——纯芽茶）；一芽二至三叶用于标准等级。
* **原料要求：** 必须嫩采、早采——梅占芽叶持嫩性较差 (持嫩性较差)，需要及时采摘。

**4. 风土与栽培特点：**

* **原产地——安溪芦田：** 闽南山区。海拔400-1,200公尺。亚热带季风气候，雨量充沛，云雾缭绕，肥沃的弱酸性红壤。银瓶山 (银瓶山, 1,200公尺) ——品种发源地。
* **武夷山：** 梅占红茶也在武夷山生产，该品种于20世纪初（民国时期）引入。武夷山风土赋予其特有的「岩韵」(岩韵，「岩石韵味」)。
* **福鼎：** 白琳工夫的传统产区，梅占在此作为优质原料使用。
* **推广：** 自1960年代起，梅占在全中国栽培——广东、江西、浙江、安徽、湖南、湖北、江苏、广西，以及台湾。由于高适应性和产量（200-350公斤／亩），成为最广泛种植的多用途品种之一。

**5. 制作工艺：**

梅占红茶按标准全发酵红茶工艺制作：萎凋 → 揉捻 → 发酵 → 干燥 → 分级。特点——注重「激发」兰花香气。

* **采摘 (采摘 — cǎizhāi)：** 手工采摘。早采嫩采 (早采嫩采) ——芽叶易老化。
* **萎凋 (萎凋 — wěidiāo)：** 日光或室内萎凋，12-18小时。控制程度——「不过干」，保持鲜度并启动香气形成。
* **揉捻 (揉捻 — róuniǎn)：** 机械或手工揉捻。形成紧结、细长的条索。梅占因叶肉厚实出汁丰富。
* **发酵／氧化 (发酵 — fājiào)：** 3-5小时，温度约25-28°C。关键指标——出现兰花梅香。过度发酵产生「平淡」味道；发酵不足则有苦涩味。
* **干燥 (烘干 — hōnggān)：** 热风或炭火烘干。安溪版本有时采用额外烘焙 (提香, tíxiāng — 「提升香气」)。
* **分级 (分级 — fēnjí)：** 手工最终筛选。

**6. 感官特征：**

* **干茶外形：** 紧结细长的条索 (条索紧结细长)。色泽——乌黑油润 (乌黑油润)，高等级带金毫。条索匀整，完整。
* **干茶香气：** 招牌特色——高锐香气 (香气高锐)：兰花香 (兰花香)、梅花香 (梅花香)、蜂蜜、果干。武夷山版本——额外的矿物质「岩石」韵味。香气持久，辨识度高。
* **茶汤香气：** 层次丰富——兰花、梅花、蜂蜜、焦糖、淡淡辛香。随冲泡次数增加——甜蜜蜂蜜果香加强。
* **滋味：** 浓厚饱满，「丝滑」(味厚)。主调——兰花、梅花、蜂蜜、焦糖、淡淡巧克力韵味。涩味——适中，「丝绸般」。天然甜味。回味——持久，带兰花梅香。回甘 (回甘) ——明显。
* **汤色：** 橙黄至红宝石色，明亮透澈，高等级带「金圈」(金圈)。
* **叶底：** 大片完整有弹性的红铜色叶片。叶缘显红边 (红边显)。

**7. 化学成分：**

* **多酚类：** 原料中27.5%（高于平均），发酵过程中——活跃形成茶黄素和茶红素，构成浓郁色泽和「体感」。
* **胺基酸：** 3.6%——提供甜味和柔和口感。
* **咖啡因：** 4.4%——中等偏高水平，有明显提神效果。
* **儿茶素：** 18.1%——发酵时转化为茶黄素、茶红素。
* **芳香化合物：** 芳樟醇、橙花醇、香叶醇——形成特有的「兰花梅香」复合体。

**8. 保健功效：**

* 温和提神（咖啡因 + L-茶胺酸）。抗氧化作用（高多酚含量）。助消化。温暖身体。抗菌效果。抗压力作用。

**9. 冲泡方法：**

* **水温：** 90-95°C。
* **茶量：** 4-5克／100-120毫升（功夫茶）；3克／200-250毫升（西式冲泡）。
* **茶具：** 瓷盖碗——完美展现兰花香气。
* **冲泡：** 温杯 → 投茶 → 洗茶（可选）→ 首泡10-15秒 → 5-8泡，每泡增加5-10秒。

**10. 储存：**

密封容器，干燥阴凉处，10-25°C，可保存18个月。无需冷藏。

**11. 价格与仿冒：**

梅占红茶属中等价位：标准级——200-600元人民币／500克；老树 (老枞) ——800-2,000元；「金梅」（纯芽）——可达3,000+元。

**如何避免仿冒：**
* 寻找特有的「兰花梅香」——正宗梅占的标志。
* 评估条索：细长紧结，乌黑有光泽。
* 汤色——橙黄透亮。混浊——品质低劣的征象。
* 确认产地：安溪、武夷山、福鼎——主要产区。

**12. 趣闻轶事：**

* **「茶界变色龙」：** 梅占是少数可制作任何茶类的品种之一：乌龙茶（清香型和浓香型）、红茶、绿茶、白茶。在每种表现中都展现特有的「梅香」特色。
* **「不夺他茶香」：** 在武夷山，梅占被视为拼配的珍贵成分——它丰富拼配茶，而不掩盖其他品种 (拼茶不夺他茶香)。
* **三万株茶树：** 清末芦田诗人林鹤年，「福建八大诗人」之一，为梅占题诗：「种梅三万株，终老吾何悔」——「种梅三万株，终老吾何悔」。
* **铁观音热潮的牺牲品：** 1990-2000年代大规模以铁观音替代梅占，导致安溪大部分老茶园消失。幸存的老树（50-100+年）——珍贵资源。
* **白琳工夫的优质原料：** 历史上最好的白琳工夫都用梅占制作——这个事实突显了其卓越的「红茶」品质。

**13. 与其他红茶比较：**

* **金骏眉 (金骏眉)：** 纯芽茶，采用桐木菜茶。更甜润、蜜香、「轻盈」。梅占红茶——更浓厚，兰花梅香突出，「体感」更强。
* **正山小种 (正山小种)：** 「鼻祖」——带龙眼香和（烟熏版本）烟味。梅占红茶——无烟味，香气谱完全不同（兰花、梅花）。
* **祁门红茶 (祁门红茶)：** 「祁门香」——细腻、「内敛」、玫瑰蜜香。梅占——更明亮、「响亮」，兰花香突出。
* **梅占乌龙 (梅占乌龙)：** 同一品种，乌龙工艺制作。乌龙——更「清香」、花香，回甘明显。红茶——更饱满、浓郁，带焦糖巧克力深度。

**结语：**

梅占红茶是一款有个性的茶：明亮、自信、丰富。来自银瓶山传奇品种的兰花梅香，不隐藏不低语——从第一泡开始就全力绽放。这是为那些在红茶中不仅欣赏柔和，也珍视力量；不仅追求甜味，也渴望深度的人而生的茶。

「种梅三万株，终老吾何悔」——这位19世纪安溪诗人的诗句今天听来如预言：梅占，经历了铁观音热潮并几乎消失，如今正在回归——尤其是在红茶领域，其声音比以往任何时候都更响亮、更有说服力。
