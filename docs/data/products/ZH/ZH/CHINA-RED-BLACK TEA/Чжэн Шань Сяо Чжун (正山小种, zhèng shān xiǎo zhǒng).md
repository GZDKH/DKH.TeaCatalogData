# 正山小种 (正山小种, zhèng shān xiǎo zhǒng)

**正山小种 (正山小种, zhèng shān xiǎo zhǒng)**
![Чжэн Шань Сяо Чжун (正山小种, zhèng shān xiǎo zhǒng)](images/Чжэн%20Шань%20Сяо%20Чжун%20(正山小种,%20Zhèng%20Shān%20Xiǎo%20Zhǒng).png)

![Чжэн Шань Сяо Чжун (正山小种, zhèng shān xiǎo zhǒng)-1](images/Чжэн%20Шань%20Сяо%20Чжун%20(正山小种,%20Zhèng%20Shān%20Xiǎo%20Zhǒng)-1.png)

正山小种是世界上第一款红茶，是全球所有红茶（黑茶）的鼻祖。四百多年前在桐木山中偶然诞生，从乡村制茶师的意外之作，发展成为中国在欧洲的象征、英国茶文化的启发者，以及后来祁门、滇红、阿萨姆、大吉岭和世界上所有红茶的基础。在西方，它以**拉普山小种（Lapsang Souchong）**之名闻名。

**1. 分类与产地：**

* **类型：** 红茶 (红茶, hóngchá)，完全氧化。按欧洲分类属于黑茶。发酵度为80–95%。存在两个主要品种：传统烟熏 (传统烟熏正山小种, chuántǒng yānxūn) — 用马尾松 (马尾松, Pinus massoniana) 木材烘干/烟熏；以及无烟熏/「新工艺」(新工艺正山小种, xīn gōngyì) — 不经烟熏，强调天然蜜果香型。
* **类别：** 中国名茶 (中国名茶)。被认定为世界所有红茶的鼻祖。自2010年起受地理标志保护 (地理标志, dìlǐ biāozhì)。
* **产地：** 中国福建省 (福建省, Fújiàn Shěng) 南平市 (南平市, Nánpíng Shì) 武夷山市 (武夷山市, Wǔyíshān Shì) 桐木村 (桐木村, Tóngmù Cūn) — 武夷山国家级自然保护区 (武夷山国家级自然保护区, 565平方公里) 的核心区。桐木是正山小种的历史发源地和唯一正宗产地。「正山」(正山) 意为「正宗山区」，即桐木及相同海拔带的邻近地区。在此范围外生产的茶称为「外山」(外山, 「外部山区」)，不能使用「正山」名称。
* **地理座标：** 东经117°38′6″–117°44′30″，北纬27°41′35″–27°49′00″。
* **别名：** 拉普山小种（Lapsang Souchong）— 欧洲贸易名称；桐木关小种 (桐木关小种) — 「桐木的小种」；星村小种 (星村小种) — 以历史上茶叶出口的贸易集镇命名。在17–18世纪欧洲文献中称为BOHEA（来自武夷Wuyi — 「博希」）。

**2. 历史与文化意义：**

* **历史：** 正山小种是人类历史上第一款红茶，已有400多年历史。确切创制日期未有记载，但根据综合资料（邹新球研究 — 邹新球，《武夷正山小种红茶》— 《武夷正山小种红茶》），其起源可追溯至16世纪中后期（约1567–1600年）。广为流传的传说是：明朝末年 (明, 1368–1644)，战乱期间，一支从江西前往福建的军队在庙湾村（庙湾，现为桐木一部分）的制茶作坊过夜。士兵们直接睡在摊放待加工的茶叶上。到了早晨，茶叶开始发酵变红。为了挽救茶叶，制茶师（据正山堂 — 正山堂 — 江氏祖先的说法）将变色的茶叶揉捻，并用当地马尾松 (马尾松, 松树马松) 的火烘干。制成的茶叶呈黑色，带有浓郁的松烟香味，在桐木当地不受欢迎，但当运到45公里外的星村 (星村) 集市时，一位闽南商人以低价收购。次年他回来时出价高出两三倍 — 这种特殊茶叶的需求很大。红茶的历史由此开始。

* **出口欧洲：** 约1604年，荷兰商人将正山小种带到欧洲 — 这是第一款到达西方的中国红茶。在荷兰和英国，它最初在药房作为药物销售。1662年，葡萄牙公主凯瑟琳·布拉干萨嫁给英国国王查理二世时，在嫁妆中带了几箱「正山小种」— 红茶由此进入英国宫廷。安妮女王后来在上流社会推广茶饮，奠定了下午茶传统的基础。在17–18世纪，BOHEA（来自「武夷」— Wuyi）一词在欧洲是「中国茶」的同义词。英国浪漫主义诗人拜伦在诗作《唐璜》(1819–1824) 中提到了「博希红茶」(BOHEA)。19世纪，正山小种在中国红茶出口中的份额达到85%（嘉庆年间, 嘉庆, 1796–1820）。

* **技术窃取：** 1848年，英国植物间谍罗伯特·福琼（Robert Fortune）受东印度公司委派，伪装成中国人潜入武夷山，窃取了茶树样本和红茶加工技术的秘密。这一工业间谍行为开启了印度（阿萨姆、大吉岭）和斯里兰卡的茶业。

* **20世纪 — 危机与拯救：** 在第二次世界大战和内战期间，正山小种的生产几乎停止。1980年代，在出口停滞和面临完全停产威胁的情况下，杰出的中国茶学家张天福 (张天福, 1910–2017) 向福建省政府呼吁：「正山小种是福建独特的出口产品，必须保护。」茶叶得以保存。2005年，基于正山小种传统创制了金骏眉，启动了「红茶复兴」，使红茶在中国国内市场重新获得声望。

* **名称：**
  * 「正山」(正山) — 「正宗/真正的山区」。意指茶叶产于正宗产区 — 桐木及具有相似海拔、气候和传统的周边地区。「正」(正) 含有「正确、真正、正宗」之意，与「外山」(外山, 「外来山区」) 相对。
  * 「小种」(小种) — 「小品种」。双重含义：(1) 茶树属于小叶种 (*var. sinensis*)；(2) 产量和产区有限，受高山小气候限制。
  * 「拉普山小种」(Lapsang Souchong) — 欧洲贸易名称。「Lapsang」词源有争议：可能来自「腊松」(腊松, 「蜡松」?) 或地名；「Souchong」来自「小种」(小种)。

* **文化意义：** 正山小种不仅是茶叶，更是世界级的文化遗产。它是欧洲茶文化传统的起源，启发了下午茶的出现，影响了殖民时代的地缘政治（鸦片战争、「茶叶竞赛」），至今仍是东西方四个世纪茶文化交流的象征。

**3. 植物学描述与原料：**

* **品种/栽培种：** 当地小叶种茶树群体 — 奇种 (奇种, Qízhǒng) / 菜茶 (菜茶, Càichá)，*Camellia sinensis* var. *sinensis*。在桐木山中生长数百年的有性繁殖异质群体。每株茶树基因独特。小叶型提供更高的氨基酸与多酚比例，确保特有的甜味和龙眼香。
* **采摘：** 由于高海拔位置（寒冷气候），季节开始较晚：春茶 — 从5月初（约立夏, 立夏）至5月底；夏茶 — 6月底。一年两季。手工采摘。
* **采摘标准：** 一芽二三叶 (一芽二三叶, yī yá èr sān yè)。高等级为「一芽二叶」；标准级允许较成熟的叶片。与金骏眉（仅芽头）和银骏眉（芽头+1叶）不同，正山小种使用较成熟的原料，提供厚度和「体感」。
* **原料要求：** 完整、洁净的叶片，无粗老叶柄。采摘后快速送至加工厂。

**4. 风土与栽培特点：**

* **武夷山保护区：** 565平方公里，联合国教科文组织世界遗产（1999年）。红砂岩和火山岩山脉；陡峭峡谷、瀑布、河流、亚热带森林，生物多样性极其丰富。森林覆盖率96.3%。
* **桐木村：** 「正山」的历史核心。位于保护区深处，海拔700–1800公尺。茶树在森林覆盖下生长，处于半野生和野生状态，常与竹子和松树混生。「正山」界限：东至麻栗 (麻栗)，西至挂墩 (挂墩)，南至皮坑 (皮坑)、古王坑 (古王坑)，北至桐木关 (桐木关)。
* **生长海拔：** 海拔700–1800公尺。主要区域为1200–1500公尺。
* **气候：** 亚热带山地气候。年平均气温山脚约18°C，山顶约11°C。降雨量超过2300毫米/年。相对湿度80–85%。雾日超过100天/年。大气中CO₂浓度0.026%（显著低于城市水平）。日照时间短，霜期长。这些因素都减缓生长并促进芳香物质和氨基酸的积累。
* **土壤：** 山地红壤和山地黄壤，弱酸性（pH 4.5–5.5），富含腐殖质，疏松，排水良好。土层厚度30–90公分。

**5. 制作工艺：**

正山小种有两个工艺方向：**传统烟熏**（在专用建筑青楼, 青楼中用松烟烘干/萎凋）和**无烟熏「新式」**（不使用松烟，强调原料天然蜜果香）。两种都是正宗桐木产品；区别在于干燥阶段。

* **采摘 (采摘 — cǎizhāi)：** 手工采摘「一芽二三叶」。
* **萎凋 (萎凋 — wěidiāo)：** 关键工序，决定茶叶风格。
  * *传统烟熏：* 萎凋在青楼 (青楼) 中进行 — 二或三层石/木建筑。底层用湿润的马尾松 (马尾松) 木材生火；烟雾和热气通过地板缝隙上升到上层，茶叶摊放在竹制筛子上。温度约30°C，叶层厚度3–7公分。每20分钟翻动一次。茶叶同时失水、软化并吸收松香。
  * *无烟熏：* 在通风良好的室内或露天萎凋（在少有的晴天）。无烟熏。
* **揉捻 (揉捻 — róuniǎn)：** 传统上用脚揉 (脚揉, jiǎo róu)：制茶师在木桶中用脚揉捻茶叶；现在多使用揉捻机，但手工/脚工揉捻在高端生产商中仍有保留。目的是挤出细胞汁液，为发酵做准备。
* **发酵/氧化 (发酵 — fājiào)：** 揉捻后的茶叶放入竹篮，盖上湿布，在25–28°C下保持至出现铜红色和特有果香。制茶师凭感官控制过程。
* **「过红锅」(过红锅 — guò hóng guō)：** 正山小种独有工序，20世纪中期失传，由梁骏德 (梁骏德) 大师恢复。发酵后的茶叶在热锅中短时间炒制，停止发酵并固定香气。并非所有生产商都采用此工序；多见于高端批次。
* **烟熏和烘干 (熏焙 — xūn bèi / 烘干 — hōnggān)：**
  * *传统：* 茶叶回到青楼 — 现在放在下层 (焙架, bèijià)，靠近火源。用慢燃的马尾松木材烟熏6–8小时。茶叶呈现炭黑色和浓郁的松烟香 (松烟香, sōng yān xiāng)。
  * *无烟熏：* 在竹篮中用炭火烘干（类似骏眉系列）或热风烘干。无松烟。
* **分级 (分级 — fēnjí)：** 手工最终分拣，按大小和品质分级。

**6. 感官特征：**

烟熏和无烟熏版本的特征差异显著。

***传统烟熏正山小种：***
* **干茶外观：** 紧实、紧结的条索，比金骏眉粗大。色泽炭黑带油润光泽。茸毛较少（叶片成熟）。
* **干茶香气：** 浓郁、复杂 — 松烟香 (松烟香)、桂圆干 (桂圆干)、话梅、皮革、樟脑。深沉持久。
* **茶汤香气：** 松烟背景，上层是桂圆甜香、蜂蜜、果干。随冲泡次数增加，烟味退去，显露果甜。
* **滋味：** 厚实、饱满，具特有的「焦糖丝滑感」。主调是桂圆汤味 (桂圆汤味, guìyuán tāng wèi) — 正宗正山的标志性特征。轻微涩感、蜂蜜、果干。回甘悠长、甘甜，带「高山」清凉喉韵 (喉韵, hóuyùn)。
* **汤色：** 深琥珀色、红宝石色，深邃透亮。
* **叶底：** 大片完整叶片，铜红色，有弹性，光泽。

***无烟熏（新工艺）：***
* **干茶外观：** 紧结条索，深棕色带红褐色调（非炭黑色）。
* **干茶香气：** 花果香、蜜香，带桂圆和果干香。无烟味。
* **茶汤香气：** 纯净、花蜜香、桂圆、蜂蜜、淡薯香 (薯香)。接近骏眉系列风格。
* **滋味：** 比烟熏版柔和，但有明显「厚度」。桂圆甜、蜂蜜、焦糖、淡果香。涩感较少。
* **汤色：** 金黄琥珀色、橙红色，透亮。
* **叶底：** 铜红色、完整、有弹性的叶片。

**7. 化学成分：**

* **多酚类 (茶多酚)：** 干重的10–20%。儿茶素发酵成茶黄素（0.4–2%）、茶红素（5–11%）和茶褐素（3–9%），负责汤色、「厚度」和「丝滑感」。
* **氨基酸 (氨基酸)：** 干重的1.5–3%。L-茶氨酸是主要成分，提供甜味（龙眼香）和放松效果。
* **生物碱：** 咖啡因3–5%。可可碱、茶碱含量较少。
* **维生素：** C（部分保留）、B₁、B₂、B₃、E、K。
* **矿物质：** 约30种元素 — 钾、磷、钙、镁、铁、锰、氟、锌、铜、硒。
* **精油 (芳香油)：** 约0.02%。烟熏型另含松脂挥发性成分（α-蒎烯、β-蒎烯、莰烯），赋予特有烟香。
* **其他：** 可溶性糖2–4%，果胶1–2%，有机酸约1%。

**8. 保健功效：**

* **温热作用：** 完全发酵的红茶按中医理论属「温性」。适合寒冷季节，特别是烟熏版 — 产生「内在温暖」感。
* **温和提神：** 咖啡因和L-茶氨酸的结合提供平稳、持久的提神效果，无焦虑感。
* **助消化：** 温和刺激胃液分泌，餐后有助消化。
* **抗氧化作用：** 茶黄素和茶红素是强效抗氧化剂。
* **心血管保健：** 多酚类改善血管弹性。
* **抗菌作用：** 茶多酚和单宁抑制病原菌。
* **抗压效果：** L-茶氨酸促进放松专注。

**9. 冲泡方法：**

* **水温：** 90–100°C。烟熏正山小种用沸水冲泡效果好；无烟熏版用90–95°C。
* **茶量：** 功夫泡法5–6克/100–120毫升；欧式泡法3–4克/200–250毫升。
* **茶具：** 瓷盖碗 (盖碗) 100–120毫升是无烟熏版的最佳选择。烟熏版可用宜兴紫砂壶 (宜兴紫砂壶)，能柔化烟味强度。必备公道杯 (公道杯)。
* **冲泡过程：**
  1. **温壶：** 用沸水冲烫盖碗、公道杯和茶杯。
  2. **投茶：** 将5–6克茶放入温热的盖碗。
  3. **洗茶 (润茶)：** 快速冲泡2–3秒 — 特别推荐烟熏版（柔化初始烟味）。
  4. **第一泡：** 10–15秒。
  5. **出汤：** 完全倒入公道杯，再分入茶杯。
  6. **续泡：** 5–8泡。每次增加5–10秒。烟熏版的烟味在第3–4泡后消退，显露果甜。

**10. 储存：**

* **容器：** 密封、不透光 — 铁罐、铝箔袋、陶瓷罐。
* **条件：** 干燥、阴凉、避光，远离异味。10–25°C，湿度60%以下。
* **期限：** 最佳风味期12–24个月。烟熏版储存期更长（可达3年），随时间烟味柔化，显露果香基调。部分爱好者专门陈放烟熏正山小种2–5年。
* **注意：** 无需冷藏 — 红茶在室温下储存良好。

**11. 价格与仿冒：**

正宗桐木正山小种价格因等级而异：标准级300–800元/500克；高级800–2000元；老枞 (老枞, lǎo cóng) 收藏级可达3000+元。烟熏型通常比无烟熏高端版便宜。

市场充斥仿冒品和模仿品 — 「外山小种」(外山小种, 保护区外茶叶)、「烟小种」(烟小种, 「烟熏小种」— 人工香精调味的廉价红茶) 和以「拉普山小种」名义销售的普通廉价红茶。

**如何避免仿冒：**

* **检查产地：** 正宗正山小种仅产自武夷山保护区（桐木）。要求提供生产商和产区资讯。
* **评估龙眼味：** 标志性特征是桂圆汤味 (桂圆汤味)。如果缺乏龙眼香而只有「粗糙烟味」，很可能是「外山」或人工烟熏茶。
* **检查香气：** 正宗烟熏版有柔和、包覆性的松香，与果甜融合。仿冒品有刺鼻、化学、「焦糊」烟味。
* **评估汤色：** 透亮、深琥珀红宝石色。混浊、深暗或「脏」的汤色是低品质标志。
* **警惕异常低价：** 50–100元/500克的正山小种几乎肯定不是桐木产。

**12. 趣闻轶事：**

* **400年无文献：** 正山小种的确切创制日期未见任何历史文献记载。首次提及「小种」(小种) 作为红茶品种见于《清代通史》(《清代通史》)：「崇祯十三年[1640年]，红茶（工夫茶、武夷茶、小种茶、白毫等）开始从荷兰进入英国」。
* **嫁妆茶：** 1662年，葡萄牙公主凯瑟琳带了几箱「正山小种」作为嫁给英国宫廷的结婚礼物。据传，英国女王每天早晨都以这种茶开始一天。
* **青楼 — 「绿色楼房」：** 萎凋和烟熏茶叶的独特建筑。桐木现存最古老的青楼约100年历史，但这种结构被认为更加古老。
* **所有红茶的祖先：** 红茶发酵技术从桐木传播：到江西（祁门，1876年 — 通过县令余干臣），到云南（滇红），到印度（通过罗伯特·福琼，1848年），到斯里兰卡，进而传遍全世界。
* **张天福的拯救：** 1980年代生产几乎停止时，正是茶界泰斗张天福向福建省政府的呼吁拯救了正山小种免于消失。

**13. 正山小种品种：**

* **传统烟熏 (传统烟熏正山小种)：** 经典拉普山小种 — 浓郁松烟香。在欧洲市场和「老派」爱好者中受欢迎。
* **无烟熏/「新工艺」(新工艺正山小种)：** 无松烟，强调原料天然蜜果香。自2005年起在中国国内市场受欢迎（与骏眉系列发展同步）。
* **老枞正山小种 (老枞正山小种)：** 采用老树茶叶（50–100+年）。更深沉、「矿物质」口感，明显的「高山」清凉感。高端等级。
* **骏眉系列 (骏眉)：** 形式上属于「新工艺」分支，但按采摘标准独立成系：金骏眉（纯芽头）、银骏眉（芽头+1叶）、铜骏眉/赤甘（芽头+2叶）。三者都是正山小种传统的衍生品。
* **外山小种 (外山小种)：** 桐木保护区外生产的茶叶。不是正宗「正山」，但市场上广泛存在。品质从可接受到低劣不等。

**结语：**

正山小种不仅是茶叶，更是连接四个世纪茶叶历史的活线索。从桐木山中的偶然和松脂香诞生，它为世界所有红茶开辟了道路 — 从祁门到阿萨姆，从滇红到伯爵茶。其烟熏版 — 强劲、深沉、烟香与桂圆甜香交织 — 仍是世界茶叶风味谱中最具辨识度的味道之一。而无烟熏「新工艺」则展现了桐木原料天然的果蜜本性，让人理解在松烟背后隐藏了数百年的真味。

品尝真正的正山小种，就是触摸源头：触摸那个茶叶意外发酵和燃烧松香为人类开启全新饮品世界的时刻。我们至今仍生活在这个世界中。
