# 碧螺红茶 (碧螺红茶, bìluó hóngchá)

**碧螺红茶 (碧螺红茶, bìluó hóngchá)**
![Було Хун Ча (碧螺红茶, bìluó hóngchá)](images/Було%20Хун%20Ча%20%28碧螺红茶,%20Bìluó%20Hóngchá%29.png)

![Було Хун Ча (碧螺红茶, bìluó hóngchá)-1](images/Було%20Хун%20Ча%20%28碧螺红茶,%20Bìluó%20Hóngchá%29-1.png)

碧螺红茶是一种红茶，采用与著名绿茶洞庭碧螺春 (洞庭碧螺春, Dòngtíng Bìluóchūn) 相同的原料和产区，但以全发酵工艺制作。这是一个相对年轻的产品，于2010年代出现，作为合理利用不适合制作精品绿茶原料的方式，并迅速在爱好者中赢得了自己的市场地位。

**1. 分类与产地：**

* **类型：** 红茶 (红茶, hóngchá) — 全发酵茶。
* **类别：** 中国地方红茶；工夫红茶 (工夫红茶, gōngfū hóngchá)。基于历史风土的现代创新茶品。
* **产地：** 中国江苏省 (江苏, Jiāngsū)，苏州市 (苏州市, Sūzhōu shì)，吴中区 (吴中区, Wúzhōng qū)；洞庭山 — 洞庭东山半岛 (洞庭东山, Dòngtíng Dōngshān) 和洞庭西山岛 (洞庭西山, Dòngtíng Xīshān)，位于太湖 (太湖, Tàihú)。
* **地理座标：** ≈ 31.07° 北纬，120.38° 东经（东山 — 西山地区，苏州西南部）。

**2. 历史与文化意义：**

* **历史：** 洞庭山地区产茶已有千余年历史：唐朝 (唐, 618–907) 时期史书中已提及「洞庭茶」，宋朝 (宋, 960–1279) 时期西山的「水月茶」(水月茶) 被列入贡茶 (贡茶) 名单。据王应奎《柳南随笔》(《柳南随笔》) 记载，茶叶于1699年（康熙38年）获得现今「碧螺春」(碧螺春) 之名，当时巡抚宋荦 (宋荦) 将当地茶叶进献给康熙皇帝 (康熙)，此茶原本被称为俗名「吓煞人香」(吓煞人香，「令人惊艳的香气」)。皇帝认为民间名称不雅，遂赐名「碧螺春」— 「翠绿螺旋的春天」。

  然而，用相同原料制作红茶是新兴现象。对碧螺春红茶的兴趣约始于2010年代，原因有几个：首先，洞庭山每年在清明 (清明) 和谷雨 (谷雨) 之后采摘的茶叶量增加，这些茶叶不符合精品绿茶碧螺春的要求；其次，手工炒制绿茶的师傅数量减少；第三，红茶更易储存和运输。因此，碧螺春红茶从副产品转变为独立商品。自2019年起，碧螺红茶价格开始上涨，显示其形成了自己的市场地位。2025年，吴中区发布了《洞庭山红茶团体标准》(《洞庭山红茶团体标准》)，确立了生产要求。

* **名称：** 碧 (bì) — 「翠绿、碧玉色」；螺 (luó) — 「螺旋、蜗牛」（指茶叶特有的螺旋卷曲）；红茶 (hóngchá) — 「红茶」。因此，名称直接表明：这是用碧螺春风土制作的红茶，承袭了绿茶原型特有的螺旋卷曲。

* **文化意义：** 碧螺红茶是著名茶区创意多元化的典范：不是绿茶的替代品，而是其「红色镜像」，让相同的风土 — 特别是独特的茶果复合系统 (茶果复合系统) — 透过完全不同的风味特征得以展现。这款茶成为新一代洞庭茶农的名片。

**3. 植物学描述与原料：**

* **品种/栽培种：** 洞庭山群体小叶种 (洞庭山群体小叶种, Dòngtíngshān qúntǐ xiǎoyè zhǒng)，属于 *Camellia sinensis* var. *sinensis*。俗称包括「柳叶条」(柳叶条，「柳叶条」) 和「酱板头」(酱板头)。这是制作绿茶碧螺春所用的相同正宗基因库 — 在太湖条件下经过千余年栽培自然选育的群体。植株为低矮灌木（0.7–2.5公尺），叶片小而密实，芳香物质含量高。
* **采摘：** 高等级红茶使用清明 (清明, ≈ 4月5日) 前采摘的原料 — 「明前」(明前)；主要产量来自清明后至谷雨 (谷雨, ≈ 4月20日) 及更晚期间 — 四月底至五月初。
* **采摘标准：** 精品批次 — 一芽一叶或一芽二叶 (一芽一叶 / 一芽二叶, yī yá yī yè / yī yá èr yè)。标准红茶允许更成熟的采摘 — 一芽二三叶 (一芽二三叶)，包括晚季采摘的「大叶子」(大叶子)。
* **原料要求：** 新鲜完整叶片，无机械损伤。红茶的关键要求 — 在开始控制萎凋阶段前不得有「红头」(红头)，即茶叶在运输过程中不应开始自发氧化。

**4. 风土与栽培特色：**

* **海拔高度：** 海拔50–293公尺。主要茶园位于东山斜坡（主峰莫厘峰 (莫厘峰, 293.5公尺)）和西山丘陵。
* **气候：** 亚热带季风气候，受太湖水体显著调节 — 长江三角洲最大淡水湖（面积 ≈ 2,250平方公里）。冬季温和，夏季不炎热，年均温约15.5–16.5°C，年降水量1,000–1,200毫米，空气湿度75–85%。来自湖面的云雾是茶园的常伴，提供散射光线，有利于茶叶中胺基酸的积累。
* **土壤：** 黄棕色森林土和红黄壤 (红黄壤, hónghuáng rǎng)，酸度pH 4.5–6.0；因数百年果树落叶而富含有机质和磷。
* **农业技术：** 关键特色 — 茶果复合系统 (茶果复合系统, chágǔo fùhé xìtǒng)，2020年被中国农业部认定为「中国重要农业文化遗产」。茶树生长在枇杷 (枇杷, pípa)、杨梅 (杨梅, yángméi)、柑橘 (柑桔, gānjú)、桃树、李树、栗树等果树荫下。果树与茶树根系交织；落花落果滋养土壤，茶叶吸收果香 — 正是这种方式形成了著名的「花香果味」(花香果味, huāxiāng guǒwèi) 特征，使洞庭原料区别于中国任何其他产区。森林覆盖率达80%，提供天然遮荫。

**5. 制作工艺：**

与绿茶碧螺春四个关键步骤（杀青 — 揉捻 — 搓团显毫 — 干燥）在一个锅中30–40分钟完成不同，红茶制作是长期过程，需要3–4天，每个阶段都需要精确控制温度、湿度和时间。

* **采摘 (采摘, cǎizhāi)：** 手工采摘，选择嫩芽；红茶允许比绿茶碧螺春更成熟的叶片。

* **萎凋 (萎凋, wěidiāo)：** 将新鲜采摘的茶叶薄摊在通风良好、无直射阳光的室内。专业师傅使用特制萎凋架 (萎凋池)，控制温度以均匀降低湿度。目标是使茶叶柔软有弹性（握拳时茶叶应有弹性，不折断，如面团状）。定期翻动茶叶，防止过早变红。持续时间 — 根据天气条件8–16小时。

* **揉捻 (揉捻, róuniǎn)：** 使用机械滚筒机（与绿茶碧螺春手工卷曲不同）。压力原则：从轻到重再回到轻，总时长约一小时。目标 — 破坏细胞壁，释放汁液和酶，形成碧螺春风格特有的紧密螺旋卷曲。过度压力会导致茶叶破碎。

* **发酵/氧化 (发酵, fājiào)：** 将揉捻后的茶叶置于温暖潮湿环境（温度 ~25–28°C，湿度 ≥ 90%）4–8小时。儿茶素氧化成茶黄素 (金黄) 和茶红素 (红褐)，形成红琥珀色茶汤和特有的甜蜜蜂蜜果香。发酵停止时机通过感官判断：茶叶呈现红铜色泽，带有明显果香。

* **干燥/烘焙 (烘干, hōnggān)：** 在适中温度（80–110°C）下进行，以固定酵素特征并将残余水分降至 ≤ 6%。两阶段干燥：初次高温干燥停止氧化，然后在较温和条件下最终「熟成」，发展香气深度。

* **分级 (分级, fēnjí)：** 将成品茶按等级分类 — 从富含金毫的芽茶到较大叶片的茶叶。

**6. 感官特征：**

* **干茶外观：** 细长、紧密卷曲的「螺旋」，长1–2公分；颜色从深栗色到黑色，散布金色芽毫 (金毫, jīnháo)。高等级展现丰富绒毛和明显「青铜」光泽。
* **干茶香气：** 明亮、甜美，带有明显蜂蜜、干杏和温暖焦糖香调；背景有淡雅花香，承袭自与果树相邻的环境。
* **茶汤香气：** 层次丰富：初泡时 — 蜂蜜、荔枝、熟杏；随著冲泡发展出烤栗子、面包皮和淡可可香调。特有的「果花香」(花果香) — 正宗洞庭产地的主要标志。
* **滋味：** 醇厚、圆润，带有明显天然甜味 (回甘, huígān) 和温和、不粗糙的涩味。茶汤质感「丝滑」，带有愉悦的温暖厚度。回甘持久、甜美，带果香调。
* **汤色：** 红琥珀色，透明明亮，杯缘有明显「金圈」(金圈, jīnquān) — 茶黄素含量高的标志。
* **叶底（冲泡后茶叶）：** 叶片舒展有弹性且均匀；色泽从铜红到栗色；优质批次茶叶完整，叶脉清晰，无烧焦痕迹。

**7. 化学成分：**

* **多酚类：** 在完全氧化过程中，主要儿茶素（EGCG、EGC、ECG）转化为茶黄素（TF，约占干重0.5–1.5%）和茶红素（TR，约6–12%）。茶黄素负责「金圈」和茶汤明亮度，茶红素负责滋味厚度和「体感」。总多酚含量约占干重10–15%（低于绿茶碧螺春的20–30%）。
* **胺基酸：** L-茶胺酸是关键胺基酸，提供温和、甜味和放松效果。由于小叶种和太湖雾气微气候，洞庭原料的胺基酸含量较高。
* **生物碱：** 咖啡因（约占干重2.5–4.0%）、可可碱、茶碱。咖啡因与L-茶胺酸的协同作用提供温和、专注的提神效果，无「跳跃」和随后的疲劳。
* **维生素：** 微量B群维生素（B₁、B₂、B₆）、维生素E；维生素C在氧化过程中部分破坏（与绿茶保存最大量不同）。
* **矿物质：** 钾、镁、锰、锌、氟、磷 — 源于洞庭土壤的丰富性。
* **挥发性芳香化合物：** 芳樟醇、香叶醇、顺式茉莉酮、β-紫罗兰酮、苯乙醛，以及最终干燥时形成焦糖蜂蜜香调的美拉德反应产物。洞庭原料的特色 — 果香类萜烯化合物含量较高（来自「吸收」的枇杷和柑橘香气）。

**8. 健康功效：**

* **温和提神：** 咖啡因与L-茶胺酸复合物提供持续清醒和专注力提升，无咖啡特有的急剧「峰值」。
* **抗氧化保护：** 茶黄素和茶红素是强效抗氧化剂，有助中和自由基；某些研究显示，红茶的抗氧化活性可与绿茶相媲美。
* **消化支持：** 红茶传统上被认为「温性」（中医术语），对胃部温和；适合餐后饮用，有助舒适消化。
* **心血管系统：** 适量饮用红茶与维持血管弹性和降低「坏」胆固醇（LDL）水平相关。
* **温暖效果：** 寒冷季节，碧螺红茶是理想的温暖饮品，减少主观疲劳感。
* **免疫支持：** 多酚具有抗菌和抗病毒特性；系统性饮茶增强整体抵抗力。
* **口腔卫生：** 氟化物和多酚有助预防蛀牙和减少口腔病原菌。

**9. 冲泡方法：**

* **水温：** 90–95°C。沸水（100°C）可能增强涩味；芽茶批次最好使用88–90°C。
* **茶量：** 4–5克配100–120毫升（工夫泡法）；3–4克配200毫升（杯泡法）。
* **茶具：** 盖碗 (盖碗, gàiwǎn) 瓷器 — 通用选择；瓷茶壶；宜兴紫砂壶 (宜兴紫砂壶) — 如偏好更圆润、「天鹅绒」般的口感。
* **冲泡过程：**
  1. 用热水温热茶具，倒出。
  2. 投茶；闻「苏醒」干茶香气。
  3. 洗茶：快速冲泡1–2秒（新鲜批次非必需，但紧结茶可洗茶）。
  4. 第一泡：8–10秒。
  5. 第2–4泡：10–15秒。
  6. 之后每泡增加5–10秒。
  7. 优质碧螺红茶可冲泡6–8次，层次展现：从明亮果蜜香调到更深层的焦糖面包香调。

**10. 储存：**

* 密封不透光容器（铁罐、铝箔袋配单向阀）。避免异味、直射光、潮湿和高温。
* 最佳条件：15–25°C，湿度 ≤ 60%，阴凉干燥处。无需冷藏（与绿茶碧螺春不同）。
* 建议饮用期：6–18个月以展现完整香气；优质紧结批次可「圆熟」至2–3年而不失特色。

**11. 价格与仿冒：**

* **价格区间：** 根据2025年数据，正宗洞庭原料碧螺红茶价格范围：
  - 特一级 (特一级, tè yī jí)：≈ 1,500元人民币/斤（500克）；
  - 特二级 (特二级)：≈ 1,200元人民币/斤；
  - 一级 (一级)：≈ 750元人民币/斤；
  - 二级 (二级)：≈ 550元人民币/斤；
  - 三级 (三级)：≈ 350元人民币/斤；
  - 普通红茶：约250元人民币/斤起。
  相比之下：高等级绿茶碧螺春可达8,000–56,000元人民币/公斤，因此红茶版本明显更实惠。
* **如何避免仿冒：**
  1. **购买可追溯产品：** 寻找「洞庭山」(洞庭山) 标识和吴中区地理标志 (地理标志)；要求具体茶园资讯。
  2. **评估茶叶：** 正宗碧螺红茶保持小叶种原料的螺旋卷曲 — 细密紧实；粗大螺旋表明非本地产。
  3. **检查香气：** 关键标志 — 果花香 (花果香)，承袭自茶果系统；浙江或云南仿品缺乏此特征。
  4. **观察茶汤：** 茶汤应透明、红琥珀色，无混浊；杯壁「金圈」是好兆头。
  5. **对低价保持怀疑：** 如「洞庭」红茶售价低于200元人民币/斤，很可能是其他省份原料或秋茶冒充春茶。

**12. 趣闻轶事：**

* 洞庭茶果复合系统 (茶果复合系统) 于2020年入选「中国重要农业文化遗产」(中国重要农业文化遗产)，2022年绿茶碧螺春制作技艺成为联合国教科文组织非物质文化遗产名录的组成部分。
* 制作500克精品绿茶碧螺春需要60,000–80,000个芽头。允许更成熟叶片的红茶在原料使用上更「经济」，使其成为环保合理的产品 — 物尽其用。
* 名称悖论：「碧」(bì) — 「翠绿」，而茶是红色的。这是对原始绿茶碧螺春和共同风土的有意致敬，而非成品颜色。
* 碧螺红茶是中国少数用小叶种 (小叶种) 原料制作的红茶之一，而大多数著名红茶（滇红、英红）用大叶种 (大叶种) 制作。小叶种带来更精致、「丝滑」的茶汤特性。
* 在盲品中，专家注意到碧螺红茶前两泡的香气强度可与武夷山红茶（正山小种 (正山小种)、金骏眉 (金骏眉)）竞争，尽管洞庭茶在多次冲泡的耐泡性上略逊。

**13. 与其他红茶比较：**

* **正山小种 (正山小种, Zhèngshān Xiǎozhǒng)：** 福建武夷山经典红茶。以烟松香气（传统版本）和更「深沉」、矿物质风味著称。碧螺红茶相对展现明亮果花香和承袭自洞庭风土的温和特质。
* **祁门红茶/祁红 (祁门红茶, Qímén Hóngchá)：** 著名安徽工夫红茶，具特有「兰花香」(祁门香)。祁门更干爽、「酒香」；碧螺红茶更「蜜甜」圆润，果香明显。
* **滇红 (滇红, Diānhóng)：** 云南大叶种阿萨姆亚种红茶；浓郁、麦芽味，金毫丰富。碧螺红茶更精致细腻，具小叶种原料的「丝滑」质感。
* **九曲红梅 (九曲红梅, Jiǔqū Hóngméi)：** 杭州（浙江）红茶，风格上最接近的「邻居」：同样小叶种原料，精致卷曲，细腻风味。主要区别 — 红梅缺乏洞庭「果花香」，这源于茶果系统。

**结语：**

碧螺红茶可能是著名洞庭风土在21世纪给茶界最意外的礼物。在绿茶碧螺春数百年来独领风骚的地方，出现了一款不模仿而是重新诠释传统的红茶。它将茶园东山和西山著名的果花氛围 — 橙花丛荫、枇杷和柑橘树影 — 转移到完全不同的味觉维度：温暖、蜜甜、暖心。这是秋冬悠闲茶时光的理想茶品，适合那些珍视温和、纯净甜味和每一杯中地域感的人们。
